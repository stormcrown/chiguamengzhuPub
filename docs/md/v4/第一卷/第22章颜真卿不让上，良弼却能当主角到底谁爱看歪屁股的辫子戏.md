# 第22章  颜真卿不让上，良弼却能当主角 到底谁爱看歪屁股的辫子戏

为何演艺圈中有那么多的满人？关于这个问题，我以前可能讲过。哈哈，也许吧。没关系，可以再讲一次。实际上，在整个文艺圈中，当年的八旗后裔确实数量不少，这是事实。他们有几个来源：第一，我们之前说过，清代普通人的识字率几乎跌到零，但整个中国还有一点识字率。哪个群体的受教育程度较高呢？自然是八旗群体。他们的识字率和受教育比例较高，这使他们在文化和文艺方面更有工作能力。第二，辛亥革命成功后，八旗及其后代的社会地位急转直下，从原来骑在别人脖子上拉屎、提笼遛鸟，变得必须出去打工，还受人排挤，成为社会上看不起的一群人，社会地位极低。此时，他们的就业渠道非常狭窄。他们能做哪些工作？一部分人干体力活，比如有些八旗成员保留了一些财产，在社会上正常运转，便包下车行，招募找不到工作的八旗子弟当人力车夫。因此，旗人后裔中很多男性做了苦力，而女性则有不少流落风尘。互联网上有些人玩梗提到的“八大胡同”，正是源于辛亥革命后一部分旗人妇女流落风尘，男性则干苦力。但除此之外，他们还有一个就业渠道——走文化文艺路线，比如唱戏。因为自古以来，唱戏被视为贱籍，归入下九流。尽管旗人在就业上遭到群体性排挤，但这些行业本身是贱籍，所以他们可以进入。同时，他们识字率较高、文化水平较高，便能在这些行业中崭露头角。这是一个基础。

到了建国时期，我们的文盲率很高。尽管经过民国几十年的推动，比清末好很多，但文盲率仍超过80%，需要不断推行扫盲运动。普遍受教育程度较差，但国家安稳后，需要许多能工作、能念书、能认字的人参与文化工作。谁有这种能力呢？是那些经过改造、表示愿成为共和国公民、不再怀念昔日耀武扬威的日子、承认过去错误、改邪归正的八旗后裔。他们有一批进入了文艺和文化工作岗位。所以今天你会发现文艺圈里似乎他们的人特别多。这是因为一代代传下来，导致他们在特定职业范围内形成了传帮带系统，拉拔同类、朋友、亲戚。内部圈层就这样固化下来。所以你才会发现怎么那么多满人，就是这个缘故。当然，由于已经是新社会，他们在正常做文艺工作时，我们不应挑剔；除非他们在文艺创作中掺沙子注水，那我们就批评其行为。有没有这种行为了？有，坦然承认。所以从80年代开始，文艺文化创作领域受监管程度逐渐降低，逐渐宽松自由。之后你会发现，这个群体开始陆陆续续写各种小故事、演各种影片，为满清朝廷洗白，把那个时代的人描述成有规矩、讲老理儿、有传统，这些东西逐渐回来了。

关于清宫剧：80年代以前，涉及满清时代的电视剧、电影，你可以回看，凡涉及这个时代的都有一个滤镜。不光是今天的人知道滤镜影响情绪，我们的先辈也知道。所有搞艺术创作的人都明白，滤镜、色彩、节奏、音乐影响情绪。80年代之前拍摄有关这个时代的影片，滤镜中都有黑暗、阴森的东西，于是大家形成的直观印象是那个时代很烂。哪怕拍英雄，英雄也会直面反派，依然起到批判作用。对特定时期的黑暗现象进行全方位批判，这是80年代以前影视作品、文艺作品走的路。但从80年代开始，陆续出现一些影片，先从清末讲细枝末节的故事，然后不断吹嘘帝王将相。正剧里出现康熙、雍正等剧，再加上小燕子系列等插科打诨的喜剧，会让人们对那个时代产生较好共鸣，觉得那时候不错，甚至认为皇帝勤政爱民。这些资讯通过一部又一部影片渗入人心，时间久了，大家对那个时代就开始正常化，觉得没什么毛病，就算有错也是具体个人的错，而非时代整体有问题。这个进程从80年代开始一直演进到2019年，差不多40年时间。40年是两代人，所以有两代人的成长伴随着满清帝王将相的吹捧。而且在满清帝王将相被吹得天花乱坠时，其他朝代的正片影视作品却播不出来了。大家可以看唐国强老师的采访，他当年拍好的影片出不来、不让播。就在辫子戏横生的时期，其他朝代的正向推动的剧播不出来，于是铺天盖地都是娘娘格格宫斗这类子虚乌有的东西。满清实际上没有宫斗，因为不值得，也没能力——宫斗需要实力。但宫斗剧都穿着满清衣服、打着它的背景，于是你对那个时代有了新生的想象。宫斗会让女性觉得那时还有话语权，但事实上满清内部女性完全没有话语权，没什么可争的。然而影视作品扭转了你的思想，让你觉得那个时代还不错。正向洗白、侧面洗白、反向洗白都有，你才会形成这种印象。

到2019年，出现了一个历史转折点。因为之前几十年他们做得太顺风顺水，以至于有人幻想在中国国内有了群众基础，大家都爱他们的康雍乾盛世。于是以前在境外肆虐的满族组织跑到国内来落地生根，整大活儿，结果整个组织被扒掉，之后发了一套通文：从2019年起，正规渠道审批的清宫戏每年不能超过一部。当规定这个类目的剧不能超过一部时，大概率这几年一部都过不了，偶尔有背书的能过，但有限额。所以现在能看到的清宫剧变得很少。国家知道这些情况，当然知道。国际著名人士还办了大会，有正经组织，背后有资金流。他们在美国、隔壁、日本等地都设有机构，是真正的线下有活动的组织。这就解释了为什么40多年看不到大明的影视作品？也不是完全看不到，他们把之前用在满清身上的黑暗滤镜用到了明王朝身上。过去二三十年影视时代成长起来的人，如果看片多，会产生一种印象：想到明王朝时，会联想到灰蒙蒙的滤镜，脑中天然出现这种滤镜，这不是你上的，而是各种影视作品帮你上的。过去几十年，对明王朝形象的打造，不管影片正面还是负面，都会给它灰蒙蒙的黑暗滤镜，配上紧凑的音乐，让人感到神经紧绷、压迫感、山雨欲来风满楼，并不断强调锦衣卫、东西厂里邪恶的公公们，看起来都像变态，好像割一刀后直接从正常人变成变态——这不存在。无论是东西厂还是锦衣卫，影视作品中的形象与历史实际完全不同。他们夸张描写这些场景，用黑暗滤镜，让你形成一种直觉：那时高压、日子过不下去、恐惧。好，欢迎来到直播间。

关于《极简明史》小人书：这是一部绘画作品，按常规明史绘画，作为一般性科普阅读没问题。虽然我们知道明史注水量较大，但暂时没有形成新明史，依然可以看这套书。画风硬朗，我个人喜欢这种画风，会看绘图版本的简史，小学生也可做基本常识阅读。绘画没有丑化朱元璋，这点很重要，画的挺帅。关于宫斗剧影响：他们推崇宫斗，是想让女人变成每天靠色相、心计获取权力、取悦男人，不是真正的女英雄。这有侧面影响。宫斗这种东西在什么情况下存在？其实是在女性拥有一定话语权时才会搞宫斗，包括宅斗也一样。女性有一定话语权，能支使一些人、事，有财力掌控东西，才能做宫斗和宅斗。它明面上的负面影响是展示勾心斗角，让你模仿借鉴，甚至搞厚黑学；但背地里，它会直接影响你的直觉，让你误以为这个时期女性拥有话语权和社会地位。因为没有权利的人是做不了宫斗和宅斗的。你看宫斗剧、宅斗剧中，哪一部不需要收买周边人群（太监、太医、宫女等），调动资源为我办事？这需要一定话语权和尊重才能做。所以会让你感觉那时女性地位还可以。同时，清代没有宫斗，因为宫斗是一种竞争，像职场狂卷，可能涉及非法手段。你什么时候会卷？前面有大利诱才卷。满清后宫里无利可图，皇后的权力打折到几乎看不见，你图啥？皇后说“臣妾做不到”，她真做不到。那慈禧太后怎么算？她是太后，能掌握权力靠的是当妈。关于为什么不推现代汉服作为民族服饰？现在我们没有精力处理这个问题。目前能在民间推动且没有官方阻力，你都要谢天谢地了。你如果真的说推现代汉服，猜会设计出什么？你自己到某宝搜“现代汉服”，看能搜出什么；搜“国风”“新国风”，看能出什么。零一或零二年，当时要设计传统民族服饰，因为国际大会上有赠送传统民族服饰的惯例。设计出来的是以西装为底板、蜈蚣扣的唐装——以西装为底板、满洲元素为装饰的一套唐装。除了“唐”这个字，跟我们的历史传统有多少关系？大家可以在网上搜图，当时小布什来串门时也穿了。他脸有点猴像，穿上那套衣服非常离谱。你猜如果设计现代汉服会设计出什么？官方主导设计过，要求展现民族形象和传统，招聘专业设计师投稿，最后成果就是那东西。你想再来一次吗？为什么03年会有汉服复兴运动？就是因为被这件事恶心到了，刺激出来的。如果传统服饰就长那样，谁都不想要。于是有了汉服复兴的星星之火，如今已燎原。然后有人下山摘桃子，说“清汉服也是汉服”，已进入新阶段。这时你说弄现代汉服，能变成什么样？现在皇后大礼服在准官方设计里已经出现资金了。你真推？推完了又得剃头。唐装为什么叫“唐装”？跟唐王朝一毛钱关系都没有。我猜这个名字是拿来忽悠领导的，让他们觉得“唐”就是传统，领导不懂就签字过关了。实际上它源于“唐人街”，因为唐人街上有些人保留了清代穿马褂的习惯（他们是那时出去的，留下了当时的衣裳）。基于马褂，结合西装剪裁，做出了那套衣服叫“唐人装”，跟唐半毛钱关系都没有。
