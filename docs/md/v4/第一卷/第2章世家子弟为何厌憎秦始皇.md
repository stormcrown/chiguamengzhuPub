# 第2章  世家子弟为何厌憎秦始皇

司马迁在《史记》中编造故事，这是确凿的。他特意针对始皇帝编造了不少内容。为什么要这样诋毁始皇帝呢？实际上，在相当长的时间里，整个汉代有许多人厌憎始皇帝。先不提其他事，始皇帝最重要的一项举措就是推行了非常彻底的郡县制。请注意，汉朝建立后其实推行了一次妥协——西汉并不是完全的郡县制，而是郡县与分封杂糅在一起。开国功臣会得到一定的封地和封侯的权利，这种操作其实是分封制的一种残余。汉代是分封与郡县杂糅的时代，而秦朝是彻底的郡县制国家。这中间会触及一部分人的利益：纯粹的郡县制和官僚录用方法会影响到谁？世家。原本的旧贵族、旧世家，他们的根本利益与始皇帝推行的管理方法之间是矛盾的。所以不仅六国贵族一直在反对始皇帝，实际上还包括大量世家也在反对。因为纯粹的郡县制以及军功授爵这样的制度，对之前的贵族制度、分封制度、承袭制度是一种完全碎裂式的打击。

那么司马迁的家族是什么人？他是史官家族。如果你认为史官是竞聘上岗的岗位，那就错了。史官是世家，史官这种职业从来都是世家传承的。为什么？因为史官从设立一开始就是一个很特殊的职位。他做的是工作档案登记，比如在工厂里工作过的朋友都知道，工作成果登记、工作日志编写这件事重不重要？一般这样的岗位掌握在谁手里？现在实现了电子化，造假的空间变小了，但之前不是这样。在还没有电子化的时候，工厂里专门负责档案登记、记工分、计件的岗位都是谁在做？这些岗位重不重要？需不需要极大的信任感？是不是管理者的亲信？史官行业做的就是这件事，他是记工作档案的人，要求必须是自己人。长期的自己人会形成一个圈子，成为小圈子，后来就是世家。所以史官这个角色从设立一开始，就是王者的亲信，而且是天下王者的绝对亲信。正因为这样特殊的身份，史官世家子弟一定会不喜欢始皇帝。因为始皇帝无论是严格的军功授爵管理方式，还是拒绝承袭的郡县制，都是与世家、贵族过不去的手法。所以他被恨是很正常的。那司马迁为什么会受宫刑？如果他不是世家，就不是宫刑了。司马迁本身很喜欢世家，他的书里对各种源远流长的世家非常友善，而打击世家的则会遭到他狠辣的批判。受到他最残酷批判的就是始皇帝。他甚至帮始皇帝改了氏。我曾说过，始皇帝嬴姓，但很多人认为他是赵氏，之所以人们信誓旦旦说始皇帝嬴姓赵氏应该叫赵政，谣言的源头就是司马迁。他自己在写书时把始皇帝称为嬴姓赵氏。这种命名法在先秦时期就是指着鼻子骂人。今天我们看书不会直觉地感受到，但先秦的人很懂：当你给一国之主起一个氏，既不是他的国名，又不是他的姓，而是隔壁某宗主的氏，这等于把他从大宗改成了小宗，是在说始皇帝没有资格供奉嬴姓的祖先，是依附于赵氏而存在的。所以司马迁在书里写“始皇帝是嬴姓赵氏”这一句，看起来像客观评价，实际上翻译成今天粗糙的话，就是“始皇帝是个龟孙儿”。

（关于长城的评价）秦始皇修长城算不得大兴土木，因为它实际上是连通原有的长城，不是平地而起——把几段原有的长城拼合在一起。为何要这样做？因为他完成了一统，需要把原有的长城连起来。秦始皇的秦长城不光修了长城，还拆了很多长城，拆的是六国之间的长城。必须弄清楚长城是干什么的：长城是一个哨所、一个关卡。它修在山上的关键点位，作用是关卡，可以是海关，也可以是瞭望所，但唯独不是边界线。所以你不能一看到就说“国境线那么长，他修了一整条国境线”——不是这样的。真正的大兴土木应该是阿房宫和复刻的六国宫殿。阿房宫到最后也没建完，它的地基算一部分工程量。还有他的陵寝——骊山陵寝是巨大的工作量，以及咸阳宫的扩建，即复刻六国宫殿。这相当于今天你们搜集手办一样，开了服务器，地图铺开了，拿个纪念品不为过。他确实设置了一些宫殿，还有重要的大型土木工程就是修驰道——那个时代的高速公路。为什么要修驰道？因为他要搞郡县制管理，要传递信息、人员往来，不能让中间的交通阻断。一旦阻断，中央怎么掌控地方？所以大兴土木有没有？有。无论是河道的疏通巩固，还是驰道的修建，或者包括陵寝、宫殿以及长城的连接，加在一起才叫大兴土木。你不能单独说哪个项目就是大兴土木。那个时代之所以出现大兴土木的工程，大部分是非常重要的东西，但量太大了，钱花超了，这是真的，不需要为他遮掩。因为在秦朝刚统一天下时，人心未定——各地依然有六国的残部存在，整个天下没有完全归心。一些刚需的建设确实要做，但一些并非刚需的建设投入过大，就会使劳役过重。劳役过重很容易变成六国残部手里的把柄。所以直接说他行的全是善政是不对的。虽然从竹简分析出秦代规则其实蛮人性，劳役制度分层的、有量的、按时间区划，但规则是这样定的，帮你算算账就知道，一统天下后确实花超了，所以肯定存在一定范围内的暴政。但暴政到了什么程度？我个人认为不会达到后世汉代点评的那种程度。司马迁对始皇帝非常厌憎，所以他的描述不可能客观。秦的问题不是出在大事上，规则并不残暴，但因为一统天下后想做的事太多，花超了，就会出现临时征敛。所以当时人的体感中存在一定程度和范围内的暴政，而这些操作又被六国贵族利用，形成了残暴的始皇帝形象。

（关于赵秦是一家）他们的血缘确实有联系。我们之前讲过，先秦时期姓和氏是分开用的。姓是血缘来历，来自上古母系氏族部落时期——我们当年是哪个部落的，这是姓。所以很多先秦的姓都包含女字旁，这是母系氏族时期的烙印。姓单纯讲血缘关系，你归在哪一群人里，你的父亲是哪个家族。氏则不同。氏是男子用的，男子通常不称姓，见面称呼会用氏。氏是一种社会身份，这种划分法上古时期就存在，比如轩辕氏——轩辕氏是造车的人，是一种职业划分法，指社会分工、社会岗位。在先秦，公子、王孙们都有自己的氏。有的氏来自官职，例如司马迁的“司马”，祖上就有司马氏的身份，司马是一个官职；司空也是官职，所以司空也是一个氏。还有的氏叫“王子”，因为他爹是周天子，他是王的儿子；还有“王孙”、“公孙”——为什么叫公孙？因为他是公的孙子。这种社会身份定位就叫氏。氏当中还包括封地，比如一个人被封在鲁地，长期成为管理者，他的氏就可以改称鲁氏；封在齐称齐氏。这些都是氏的命名方法。所以看到“赵氏”时，它跟封国有关，跟自己的社会公共身份有关。嬴政的祖上确实与赵氏是同宗关系。但要知道氏是可以改的，赵氏自己也是改过的——他封到这个地方拥有了身份才叫赵氏，之前可以不叫赵氏。赵氏旗下子弟到别处受分封并拥有足够实力，就可以单独立宗，变成新小宗，不叫赵氏而叫邯郸氏。邯郸氏都可以从赵氏中分宗出去，那么嬴政这样已经拥有一国、可以称王称霸称皇帝的角色，他有没有新的氏？必然有。按照正规方法，氏代表他的祭祀权，他可以祭谁。当他执掌天下时，自动成为嬴姓的大宗，称谓中就可以不称氏了，因为他直接供奉嬴姓最老的老祖宗，此时他就是嬴姓大宗。如果按照封地、国号这种身份，也可以称氏。那这时候他应该是什么氏？他的国叫秦，所以无论怎么命名，他可以叫嬴政，也可以叫秦政。但称他为赵政，就是在骂他是个龟孙儿，还是指着鼻子骂的那种。
