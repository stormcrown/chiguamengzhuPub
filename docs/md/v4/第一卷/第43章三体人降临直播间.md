# 第43章  三体人降临直播间 #抽象 #晋商 #山西人 #君王

嗯，什么东西？邪恶呀。我看看啊。我从来不看任何历史书籍，但是我知道，三千年来华夏大地上所有的帝王都是邪恶的，只是程度高低罢了。

呃，如果你从来不看历史书籍的话，就不要大放厥词。因为你说的这个方式呢，就和……嗯，算了，不专门针对批判了，很没有意思。

呃，第一呢，帝王这个角色——如果你再说皇帝的话——皇帝的诞生历史没有三千年那么长，这是第一。第二呢，华夏大地上的君王如果都是邪恶的，我们是怎么在满清之前的漫长数千年间，都成为当时翘楚的？如果说我们自古以来的君王都是邪恶的，那周边的那些地区为什么那么蛮荒？是因为他们缺一个邪恶的君王去拯救他们吗？

什么东西叫正义？什么东西叫邪恶？什么是好的？什么是坏的？有定义吗？因为他是统治者，所以他就是坏的？那么你是觉得大家就应该像山里的猴子一样？那不行——山里的猴子都有个猴王。当蚂蚁也不行，蚂蚁还有带队的呢。

所以呢，自然界里的所有小生灵都知道，当大家要集群生活的时候，需要有一个带队的人。所以什么叫做帝王？帝王就是那个带队的人，就是那个举旗帜的人。你觉得不需要？你觉得只要他存在就不好？那意味着你是散兵游勇。别人有旗帜、有组织的人过来说，怎么扬了你就怎么扬了。你连生存权都不存在，你还在这里谈什么美好？

所以，一个君王在需要有君王的时候出现，承担君王的责任，他就是优秀的；在不需要有君王的时候出现，并且高压统治，他就是卑劣的。一个人的贡献取决于他对这个社会的影响，一个人的价值取决于他对这个群体的影响。

中国人自古以来的君王，引领着这个国家向上，开疆拓土，同时安宁度日，使我们一直处在文明的最前沿、科技的最前沿、组织力的最前沿。然后你说他们都是烂的？那好，那些被我们远远甩在身后的蛮荒部落呢？他们的统治者是什么？

嗯，一面说君王都是坏的，一面说华夏土地上的君王都是坏的，那就是想说其他地方的统治者是好的呗？统治者那么好，吃不饱穿不暖？啊，你说人家都过得可好了？怎么好的那么好？来到我们这个这么不好的地方打劫？我们这儿过得那么苦，你过来打劫？你那儿过得那么美，你过来打劫？你天天吃饱饭，你为什么要打劫自己治下的人民？只能靠打劫来维持尊严与生活的时候，你说他对君王老好了？因为我们真的有粮啊，但是他说君王是邪恶的。

那其他地方帝王也是邪恶的？就希望所有人类全都是散兵游勇呗？不是，你是不是三体派过来的代表啊？你是外星人前哨吧？你就说全世界所有的君王都是烂的，都是邪恶的，就是要让人类进入无组织、全员分散的地步呗？全世界原子化，全员大分散，完全无组织无调度，一杆旗帜都没有，要集结的时候不知道往哪跑，然后这叫什么呢？

不是，你肯定是外星人的前哨，这就不想让人类活着啊。那就厉害了啊。这啥玩意儿？这是降临派是吧？啊，好的好的，我意识到了啊，我知道这是啥玩意儿了。恨人类恨成这样？我的天雷啊！暴露了啊，暴露了。下一次收着一点。

嗯，我们是草原行，就喜欢打劫，你怎么了？不服学习？刘彻来北伐呀，户口减半？笑死人了。所以呢，刘彻北伐之后开疆拓土，你以为的草原雄鹰跟你的祖上没有任何关系，因为刘彻北伐之后的那批人跑了啊，跑了。你要喜欢找他们的话，可能需要到西欧或者西亚，你在中国本土找不着他们的。而且户口减半怎么了？花钱花超了吗？导致一批人破产，不愿意交税了。因此呢，户口减半不是人口减半啊，不要偷换概念。所谓的“户籍减半”就是老子不干了，老子不交税了啊，老子销户，就这，申购破产的意思。

今天人气不旺，是不是小黑子不够？我也是觉得啊，我这边缺黑子啊。不行的话你们帮我去招募一批啊。啊，我有斯德哥尔摩综合症啊？你有反人类综合症，我已经看出来了啊。理解不了的事情呢，就不要犟嘴。嗯，阅读理解有问题的话呢，我这里的学历起征点是小学五年级起步啊，低于这个学历的，在我这个直播间里面没有办法谈话，知道吧？而且呢，狡辩这件事情呢，也没有意义。

那个啥，嗯，有人说晋商，很多攻击晋商的。你是说我吗？啊，我就是带头攻击晋商的那个啊，我就是带头攻击晋商的那个。所以呢，想问什么问题？说“你不要攻击我们山西人”？谁跟你说晋商是山西人呢？晋商是晋商，山西人是山西人，晋商跟山西人之间有什么关系吗？有晋商第一个敲诈勒索、第一个出卖、第一个蹂躏、第一个虐待的，就是山西人。山西人首当其冲，第一波受害者。所以我喷晋商怎么了？结果啊，最大的那批受害者出来了，替加害方叫委屈？随意啊，随意随意。晋商能是什么好玩意儿啊？啊，闲的没事干，在这个地方替晋商找补吗？犯不上，你知道吗？

晋商是干什么？为什么在那个地方发迹起来？因为山西这个地方的地形特别宝贵，所以它有通道，可以在当年汉人被圈禁在汉地十八省的时候，它有通道能够抵达草原。所以这个地方可以形成一个贸易集结点。当他要搞走私贸易的时候，他要勾连敌国，出卖国家的基本资产，出卖战略物资的时候，必须要从这儿过。所以他才会在这个地方发达，因此他才是晋商。他跟山西人，就是毛线关系都没有。

满山啊，你们都是山西人是吧？哼，我时不时地就有人跳出来说“那你不要骂山西人”。哪个区、哪句话骂山西人了？晋商，晋商跟你山西有啥关系？没关系的事情，不要在那里闲的没事捡黑锅背。我可见不得这样的事情。好好过日子的农民的后代，闲来无事，把人家奴隶主座下的走狗、鹰犬的这口黑锅顶头顶上了，然后谁让你撤下来你都不乐意。我不懂这样的人。

嗯，咱们不要在我的公屏上面聊别的UP主，聊得那么热火朝天的啊。你们要是很爱很爱他，就去他那儿吵嘴好吧。老在我的直播间里面去吵别人，我会怀疑说我热度是要起来了吗？呵呵呵呵呵。哎，最近我觉得自己热度不够啊，那肯定是骂我的人有点少。不行的话，你们出去串串门子吧，啊，递两个话头出去，让他多积极多努力努力，再骂几回街啊。我这边呢，如今啊，如今我这边日子过得也是稀松平常了啊。啊，再骂几波，再骂几波。肯定是最近骂我的人少了，播骂量不够，以至于热度跟不上趟儿啊。加热紧，加热紧。你们那么热爱谈论他的话题，你去他的那个地方，去他的地头上面递点话茬啊，地点画瓣，让他接着出来喷一喷什么的。

那好的好的啊，黑粉都少了。可不是嘛，之前的那波黑粉都已经开始转化了。好吧，那个从一开头被他弄过来的一批黑粉，然后逐渐一批一批地转化。一批呢，变成了死忠粉，一批呢，变成了路人粉。反正吧，现在残存的黑子啊，黑子库存不够了，需要对方加紧生产。有没有人可以去催个单？就这么一件事啊，属于一个建议，我是不付费的。你们要是愿意的话可以去。

嗯，蛇灵和胖子都没来是吗？完了，我黑粉头子不见了。我最近嘴太毒了吗？我连黑粉头子都没有了，难怪日子不好过啊。啊，裹脚啊？嘿嘿嘿嘿嘿嘿，我很好。我很高兴这些各路的黑子开始对我进行断章取义和移花接木式的诽谤造谣。为什么？因为当他们的冲击波开始大规模围剿的时候，就意味着它疼了啊。说明他更上一层，开始进行系统性的发单了啊。我这个地方一旦遭到这样的互联网势力的围剿，就意味着相对来说，至少是中层开始对他们进行罚单，这就说明我又戳到他痛处了呀。亲，他不疼，他怎么会做这件事情呢？

但是你有没有注意到一件事，有没有点儿纳闷？为什么这些人会那么在意“断足”与“裹脚”的问题？发现了吗？他们编造的清代、明代和宋代的假历史特别的多，胡搅蛮缠、颠倒黑白的东西特别的多。但是他们非常非常执着于强调说掰脚丫子这事是汉人的传统，特别的执念。谁把东西扒出来，然后发现他们在撒谎，他们就疯了一样，乌泱泱的一群蛆就过来了。为什么这么着急啊？因为断脚这件事情是他们的很重要一颗棋子，这颗棋子下在了棋盘的棋眼上面。

要知道，所谓的满洲思想复辟这个玩意儿，他们已经搞了四十年了。从像《戏说乾隆》这样的电视剧开始啊，他们就一直在做满洲思想的复辟这件事情。这个文化上的反攻倒算搞了四十多年，但是为什么在断脚这件事情上面他们却特别特别有执念？因为如果你真的相信了中国人从宋明开始就断脚虐待女性——至少说如果在舆论上面制造出这种氛围——他就天然地把中国汉民族群体当中占比一半、实际影响力超过一半的女性群体推到了敌对的那一边去。他是为了搞社会撕裂，所以他们极为执念，他一定要大家相信他们编造的那个谎言的牢笼，一定要让你相信所谓的掰脚丫子、小裹脚就是你们汉人的传统，“我们救了你两百多年，硬是救不下来呀，挽救不了呀”。

这种赤裸裸的谎言，他用无数的水军不停地撒谎，铺天盖地地、始终如一地坚持，运用形形色色的手法，在你脑子里面种种子，埋钢印。为什么要这样做？因为这个钢印只要打成功了，我们的群体就天然地被分成了两个阵营，一个男一个女。因为断脚这种残酷的刑罚，断脚这种残酷的刑罚谁都接受不了。一旦你相信说这是我们的传统文化，那么很有可能会被断脚的女性就会开始本能地厌恶传统文化，并且开始里通外国。他的目的就是这个。所以他铺天盖地地运用各种各样的技法，偷换概念，就要让你去相信他撒的谎。为什么他们过来围剿我？因为我好歹直播间里面还有千把人，而很多其他那些纠正这些事情的人，被他们打击得想说话找不着听众。所以他们特别恨我啊。

中国人自古以来，断脚、跺脚趾头这样的就叫做刑罚，而且是肉刑。说明中国人从来非常清楚这件事情的残酷性，它是用在奴隶身上防止逃跑的。肉刑叫做“刖”，月亮的月加上一个利刀旁。大家可以自己去检索。它是先秦时期普遍运用在隶臣妾——也就是奴婢——身上的刑法，轻微的是跺脚趾头，重的就会一直一节一节地跺上去，直到膝盖。他干嘛要做这件事情？掰脚这种行为是针对奴隶的一种行为。而中国人自古就没有在自己家正常人身上施以刑罚的这种习惯。我们讲究“身体发肤，受之父母，不敢毁伤，孝之始也”。掰断脚丫子这种行为是不孝的行为，是败德的行为，是违背祖先的行为，是破坏身体健康的行为。所以为什么会断脚丫子？因为我们遭遇了奴隶制的统治。

这种裹小脚，你所看见的那些可怕的、走不了路的小脚，是在清朝的时候才集中性地出现的，而且是集中性、批量性地出现。在那之前，在那之前就不这样。仅有的发现的所谓的断脚墓葬，号称是明代的，那是时间落在明代，但是他的墓葬不是汉人的，是满洲志士。所以断脚这种东西就是拿来应对逃奴的。那么请问有清两百多年，谁是那个奴？谁被视为奴隶？谁被圈禁？心里有点数。身为奴隶，不要跟奴隶主共情，这是一个底线。
