# 第53章 环保组织搞了好几年的碳排放，最近怎么不提了？ #哈佛 #演讲 #环保 #全球变暖

嗯，河南能养大象吗？你没有学过《黄河象》那篇课文吗？啊，在义务教育教材里头啊。我们的那个什么，我们的早期，无论是西北地区还是中原、河南、河洛之地啊，气候呢，都比今天更加的温暖。说到这儿的时候，咱们可以唠一下丁院士。呵呵呵呵呵呵，我这一杆子戳的远吧？那一杆子戳的老远了啊！那个，那个啥啊，我们开始从丁院士讲起啊。哈哈哈哈哈！丁院士当年不是参加那个关于全球变暖的气候大会吗？在那个上面的话，嗯，最后他和柴大记者之间的那个对话，是不是大家都记忆犹新？就是中国人到底是不是一个人？你算账的时候，一个中国人算不算一个人头的账啊？就这么一个核心的问题。这个核心的议题讲的是啥？讲的就是关于碳排放的问题。其实这个碳排放呢，也是一个碳勒索，是吧？它也是一个碳勒索。但是啊，但是他之所以能够出来搞碳勒索，是因为他假设了一种前提条件，并且是用科学的手法进行的假设。这个在采访当中，丁院士也讲过，说那个主流学界啊——这个是柴记者的话——说主流学界认为，说那个，呃，碳排放会造成什么全球变暖，然后会造成什么什么什么东西，所以我们要为此而负责。然后丁院士说：“主流的说的就是对的吗？那个科学家有主流吗？”就这么插嘴。

那这个时候的所谓的主流科学家认为，碳排放造成全球气候变暖，全球气候变暖会造成巨大的灾变，会造成大家都民不聊生。这一条逻辑链条本身其实就是有问题的。为什么？因为人类社会的这个气候，原本就是有高有低的。实际上今天啊，今天我们所处的这个气候的时期，属于相对较寒冷的一个时期，相对较寒冷的一个期间啊。但是呢，在我们发迹的那个时代，就是先秦时期，先秦时期我们发迹发达起来的那个时代，气候比今天暖和，暖和老多老多的了。所以你看我们发端的时候在哪里？在西北地区；然后迁徙的时候迁到哪里？迁到河洛之地。这都是当时最丰饶的土地。为什么？因为这个地方气候温暖湿润，同时呢，又没有热到那种热带丛林的那种程度，所以我们才能够开发出我们的早期文明。如果说那个西北之地和今天一样相对干燥的话，你觉得像什么西周，呃，包括之前的那些时代，在那里能够耕作出足够丰富的成果吗？没有足够丰富的成果，他怎么可能实现人口的爆发？没有人口的爆发，它怎么可能实现自身族群的扩张？没有扩张，你怎么能够形成一个庞大的、有影响力的文明体？所以其实很重要的一个前提条件就是，在那个时代，我们所发迹、所拥有的那片土地，其实是很温暖的啊，温暖而且湿润。那意味着什么啊？这意味着什么？在这个时代里面啊，温暖而湿润的这个时代里面，我们在这个温暖而湿润的地区发达了。那么相对来说，其他的地方是什么样子啊？相对今天的长江以南的地区，在那个时候呢，就是差不多就是亚热带和热带的气候，然后很炎热，瘴疠之地瘟疫横生，蛇虫鼠蚁四窜啊，这样的一个地方。那么同样的，在这样的气候条件下，北美是一个什么环境呢？遍地的沼泽啊。北美是这样的，它在这种气候上面偏于温暖的时期，它会是一个遍地沼泽的地方，气候条件相当的恶劣，基本上没有自保之地。

所以你看为什么欧美在前些年的时候，特别喜欢鼓吹说，只要全球变暖了，大家都遭殃。实际上全球变暖了之后是谁遭殃？是一部分的地方遭殃，就是今天的一些看上去还比较适宜耕作的地方，他就会遭殃。但是中国啊，中国这块地方，我们南北距离非常非常的遥远啊，我们的这个土地的纵深非常的深。在这样的情况下，实际上气候的温暖和湿润，对于我们这里的，呃，北方地区来说是非常友好的一种状态，它可以让我们的土地质量得到提升，土地的保水量得到提升，耕作效率得到提升。事实上，在历朝历代当中啊，那些非常繁盛、充满了扩张欲的这些时代，就是气候变暖的时代。例如西周的大扩张，也例如盛唐的大扩张，它都处在一个气候温暖的时期。但是有一个例外啊，明其实它也是大扩张的，但是明呢，是处在一个相对寒冷的区间，所以他其实在一开始做事的时候就极难，他的君主是拿着一个碗头饭讨起来的。那么在整个的王朝的发展的过程当中，气候也非常的不利于农耕群体的发展，但是呢，还是硬扛着起来，并且实现了领土上面、疆域上面的扩张。当然了，现在很多人说啊，他巅峰才300，胡说八道啊，胡说八道。你要是按照这些人胡说的来说的话，按照他们胡说八道的标准来判定的话，那么满清末年的土地，嗯，大概半张龙床那么大啊。

嗯嗯，但是明中后期，北方60年连旱农业，嗯，就是明代的气候条件和其他的时代并不一样啊。其他的时代在它盛世扩张期的时候，其实气候条件是非常有利的，有利农耕。所以呢，啊，有利农耕会有一个什么表现？它除了我们自己的粮食种的会比较多，人会比较日子过得比较好之外啊，它还有一个很重要的东西，就是北边的那些居住于草原地区的这些游牧的群体，他的生活条件也会好一点。当你整体的气候是比较温暖湿润的时候，北边也会相对的温暖湿润，他的生活条件就会好一些，遭遇雪灾的风险就会下降一些，于是呢，他南下劫掠的意愿也会下降一些。他的那个基本原理是这样的：就是你什么时候面对北方的压力较大？其实就是当你寒冷的时候。你哪一年比较寒冷，在北边的这个边界附近，他的军事压力就会比较沉重。因为草原地区它是一旦遇到寒冷的天气，就会遭遇灭顶之灾的地方，他的自我防范能力非常的差。一旦遇到这种极寒的天气，他就牲畜啊、人丁都会大规模的折损。这个时候他怎么弥补自己？就会南下劫掠，所谓的打草谷是抢粮食，讲抢财富。但是呢，名义上是抢粮食、抢财富，除此之外，他们还会抢奴隶，就是抢人丁啊。因为什么？因为他那里的话，只要遭遇到这种灾难，不光是财产上面的巨大折损，还会出现人丁上面的巨大折损。所以呢，他还会掳掠丁口，掳掠奴隶，这也是一个常态的操作。所以什么时候边境线上面的压力比较大？那就是气候较为寒冷的时候，边境的压力就会比较大。实际上他的这个就是捕奴，捕奴隶啊，抓奴隶、抓人丁的这件事情当中，他抓的更多的其实是青壮年的男性。就是北边的，北边的这些游牧的群体，他抓奴隶、抓丁口、抢人的这件事情，他抢的大部分啊，大部分是青壮年男性，和你想象当中呢，他要能够养成他自己人的小孩儿啊，他会去抓女人，生孩子完全不是一个样子。抓不抓女人？也抓啊，但是没有你想象的那么多的女性。为什么这样子？无论你抓男的还是抓女的，其实都要费一个力气，对吧？你去抓一个人，然后呢，下一个步骤就是要支付成本，把他带回草原上去。这个过程当中，你打算怎么关照这个孩子和女人？你怎么照顾这些孩子和女人？它中间的折损量会非常非常的巨大的。所以他们，他们最喜欢的，其实捕奴的对象是青壮年的男性和工匠。因为北边儿的这个地方啊，最缺的，他最缺的就是劳动力和技术人才。这些工匠被抓过去之后，在那边人家会当地就地给他安排媳妇的，安排工作岗位的，就是这样的一种方式。然后那些壮丁就是拿，拿过去之后就直接啊，戴上链子就开始干活的。有的时候你居住的那个地方非常偏僻，连链子都用不着，因为你跑出去就就是个死嘛，旁边你又不认识路，你走也走不了的时候，他连绳子都不需要给你拴，你只能留在那。这个是什么？这个就是奴工奴隶。他要选什么样的人？带女人带孩子过去，折损率非常的高，成本就会非常的高。但是抓壮丁过去的时候呢，他就可以让他跟在马后面，拴在马后面跟着跑，只要你的马速别太离谱，把人拽死，他就可以带回草原去。然后带回去之后就可以直接干活。对方那边要的是低成本的劳动力。
