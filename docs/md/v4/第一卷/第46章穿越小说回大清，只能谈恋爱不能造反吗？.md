# 第46章  穿越小说回大清，只能谈恋爱不能造反吗？ #小说 #穿越 #清朝 #穿越小说

写一本反清的穿越小说肯定火。你有没有听说过，在这个写小说的行当里，有句话叫“穿清不造反，菊花套电钻”。所以反清的穿越小说量级极大。其实在早期女频当中，穿清也是要造反的，但后来为什么似乎就没有那么多了呢？因为有一些人对这件事不太满意，不太高兴。所以写穿清造反小说的作者，常常很容易违规、限售、下架，甚至封号。于是这些类型的创作后来就慢慢变少了。

而你所以为的，很多女频作者一谈到穿清就开始找光头老爷们谈恋爱，也是在这波事情之后才慢慢兴盛起来的。也就是说，当你去找那些秃头的人谈恋爱时，他就给你推流，说你是写作业的大大，说你创作能力极好，内容极丰富、极动人、极浪漫。但如果你穿清之后写了造反，大部分人看不见。用这样的驱动力，就让创作群体中穿清造反的比例飞速下降。于是你看到现在这种模样：似乎在穿越小说中，穿清造反的姿态不算如火如荼，但在重重打击下，依然有很多人不断穿清造反。后来啊，也不再穿清了，开始直接穿明，直接穿明。

那个人是谁？这么大能力？这怎么可能只有一个人呢？这很典型是一个团伙。你说他们是团队，是不是这个词有点过于中性？所以这是一个团伙，是一个靠利益勾连起来的团伙。这些团伙倚仗的是谁呢？倚仗的是境外势力。为什么要倚仗境外势力？这就要说起“列强无不爱我大清”了。这是正经话，这里面一点都不包含反讽，一点都不阴阳怪气。真的是世界列强无不爱我大清。为什么呢？因为只要大清在，列强就在；只要大清不亡，列强就不亡。诸般列强靠大清的供养，失了满清的列强很难当列强。所以列强无不爱我大清。因此也有无数列强及其后代，努力供养那些吹捧大清、试图让大清卷土重来的小团伙，在中国国内兴风作浪。

然后兴出什么浪呢？如果说今日之中国，在中国人努力了数十年、近百年之久之后，终于又开始登上世界顶尖级的殿堂了。那如果这时候大清又回来了，又开始骑在大家的脖子上拉屎，你说这一小团伙，这一小撮反动分子，他们会干什么？你说他们会激励生产？别闹了。当他们激励生产时，绝大多数人就会把他们掀走，有生产能力的普通人会把他们轰走。所以他们要干什么？他们要挟洋自重。

目前而言，西方人手里应该还有很多利益关联，与当年满清的赔款、合约、供奉甚至联姻相关。至少包含古董收藏、文物交易等行当，都与当年满清八旗贵胄脱不开干系。要知道，在文物贸易行当中，中国人的态度和其他西方人完全不一样。西方人早年间在中国国内搞一些所谓的考古，甚至有人大言不惭地说，中国的考古是西方人教我们的。对，虽然他们没什么像样的古迹，也弄不清文物的造假流程，但他可以指导中国人如何考古。这些所谓的过来指导中国人考古、不认得古董的考古学家是干什么的呢？他们就是专业的文物贩子。他们为什么那么热衷于教中国人考古？因为中国人这里的古董是真古董，在国际市场上真有价值、真抢手，真的可以当硬通货。而这些硬通货中，除了那些连坑带骗刨出来的东西之外，更大量走正常渠道进行贸易的国际走私贩、国际文物贩子最喜欢的藏品来自哪里呢？来自八旗子弟的监守自盗。他们之间有持续性的买卖关系。

好，欢迎大家来到直播间。什么？宋的金石学就是考古？对呀对呀，金石学就是考古啊。李清照、赵明诚夫妻两人就是金石学的大家。他们所收藏和整理汇编的文物，在靖康之变时，大部分都流散了。那啥，现在还在进行暗地里的文化战争吗？你是不是有点瞧不起人？这怎么可能是暗地里的？你就往后看两条，镜头后的那个11（指代）。那个说我是造谣、破坏民族团结的，就这玩意儿它不叫文化战争，叫什么？你只要说那个罪恶的王朝做了多少罪恶的事情，他就说你破坏团结。破坏的是什么团结？破坏了团结敌对的满清势力，破坏了团结试图瓜分中国的列强。那他站在哪一边？他当然站在反动势力和列强那一边。我破坏的是谁？破坏了这些反动分子的复国迷梦。所以他们恨我，恨得要死。你说这算不算文化战争？所以当我说历史真相时，他们痛心疾首、哭爹喊娘，铺天盖地用各种封号、小号在那边说：“哎，这娘们撒谎，这娘们不是个好人。”你到互联网上搜我的ID，会发现无数这样的信息：“他又撒谎了，他又造谣了，他根本就不是个好人，他九真一假，哪个假不知道，但他九真一假藏得极深。”到处都是这些东西，跟我这种光在水嘴上叨叨的带货主播进行争端的，这玩意儿不叫文化战争，难道叫热战吗？

那啥，那个谁阴阳怪气，知道吧？貌似说：“哎呀，你别闹了啊，那个全屋智能……”他什么时候阴阳怪气过我呀？他不都是直着骂吗？一个菜市场骂架的架势，怎么可能搞什么阴阳怪气？他都是直着骂的呀。这个人呢，因为他自己撒的谎，上一句和下一句之间能打起来，要是前一场和后一场直播，那直接可以自己拽自己耳光，就到了这样的地步。而我对这个人没有任何意见，让他去骂去。没有他骂我，哪里来的粉丝？全指着这些反动分子抬杠。没有他们，能显示出我是个正常人吗？

我为什么老黑清朝？我老黑清朝吗？有吗？没有吧？我从来不黑清朝啊。我从来不黑清朝，没有问题吧？犯得上吗？完全啊，这个叫什么来着？互联网上有个词，叫“啊对对对”，不吹不黑。我仅仅只是没吹而已，你们就不满意了，就觉得我黑了？是不是？我完全没有黑清朝啊，我只是没吹他。

来来来，我们带波货吧。不行是吧？那个都喜欢。对，我这里是专业带货直播间，那就必须得带货。今天带这个什么啊？有望大白除螨仪。有望大白除螨仪来！家里头有床的、有沙发的，就需要备一个。为什么呢？因为大家的家里头都有小螨虫。谁家里还能没点小螨虫呢？为什么要有小螨虫？因为这东西进化成了一个小废物，猎食能力基本为零，它必须依附人类生活，是一个寄生虫类的东西。所以只要有人活动的地方，又有细软围绕着你身边，那个螨虫就会寄居其中，它就靠我们掉落的皮屑什么的为生。这是它的性命所在。然后这玩意儿还很阴暗，它怕见太阳。所以勤晒被子也是有帮助的。但是勤晒被子去除不了里面那些深层次的粉尘，所以还是需要一个除螨仪。它是一个专用吸尘器，用拍打震荡的方式把这些粉尘、碎沫沫都给你敲出来，然后嘶嘶吸走。同时呢，你知道那玩意儿怕太阳，所以它还有热风加上紫外线的功能，那紫光呲啦一照就糊了。所以这个除螨仪还是很好的一个商品。家里有床、有沙发、有宠物、有小孩的，完全可以走一波；皮肤容易过敏的，完全可以走一波；不排除脸上身上长痘痘，这事实际上是因为被子里头有螨虫。我这一带除螨仪，咔嚓咔嚓人跑了一两百；我卖别的货的时候，怎么就不掉人呢？一卖除螨仪，咔嚓人没了。所以嘛，神奇啊，神奇啊。

好，时间差不多了。我们说个再见，就下播了。下班，再见。
