# 第70章  大明为何金光闪闪，前后两屎山的衬托。 自轻自贱的磕头不叫礼仪。#磕头 #张居正 #明粉 #清粉

明朝之所以在今天这样金光闪闪，是因为他前面是一个小粪坑的元朝，后面是一个大粪坑的清朝。而且，活着的清朝，这是啥玩意儿啊？全靠同行衬托啊！如果它不是夹在这两个中间的话，明显显不出数来。你说一说满清烂，同学就说你是明粉——怎么回击？你先认了呗，当明粉很丢人吗？你还得告诉他：“我不只是明粉，我还是宋粉，我还是唐粉，我还是汉粉。明粉又不丢人，当清粉才丢人呢！”你粉一个正常王朝有什么可丢人的？最丢人的是粉一个粪坑好吗？他当清粉他都不丢人，你还嫌当明粉丢人了吗？

诶，好，我……嗯？啥？我适合穿清汉女装？你恨我你直接说呗，能不能不恶心我？人家那些坦坦荡荡的小黑子，冲上来都是直接开骂、直接造谣的。你能不拿这些鸡零狗碎的东西恶心我吗？啊，对，我家的祖上曾经穿过那种衣裳，为什么呢？为了活着，为了苟且，为了今天我们能够不穿那件衣裳，保住一条性命，保住一个机会，为了今天我们能够活回来，我们能够复兴，留下这样的一条血脉。所以他们当年曾经穿着过那些屈辱的衣裳。所以呢，在苟且的过程当中，我们也没有认野爹；在苟且的过程当中，留下了火种，没有屈服。即使事隔数百年，今天我们依然知道，清民女装那种东西是耻辱的象征。

嗯，生活不只是诗和远方，还有苟且和忍耐。其实这世上所有的人都会苟且地活下来的啊。全世界现在啊，全世界现在所有会喘气的人，他的祖上都曾经苟且地活过——全部啊，每一个人，全部他的祖上都曾经苟且地活过，都曾经在失败之后忍耐过，一定如此。但是只有我们还记得祖先是谁。在苟且的过程当中，我们也没有认野爹；在苟且的过程当中，留下了火种，没有屈服。即使事隔数百年，今天我们依然知道，清民女装那种东西是耻辱的象征。

说的明朝好像天堂一样？明朝不是天堂啊。明朝之所以现在地位那么高，并不是因为他自己是天堂，不是因为他伟大，不是因为他什么了不起，到了一种地上天国的程度。明朝之所以在今天这样金光闪闪，因为他前面是一个小粪坑的元朝，后面是一个大粪坑的清朝。而且，活着的清朝，这是啥玩意儿啊？全靠同行衬托啊！如果它不是夹在这两个中间的话，明显显不出数来。

他是一个正常的王朝。他身上有优点也有缺点。他的进步是踩在前人的肩膀上面，循序渐进的进步，并不是突然之间一下子崛起的。他是一个自然而然的进程。所以这个王朝呢，他有他的优势，也有他的劣势。例如他有点过于的文明，过于的文明，以至于欠缺某一种维度上面的组织能力。他在探索一种新的生产关系的组织形式，新的上层建筑，以至于自己的内部非常的脆弱。所以这个时代他遭遇到的最后那些悲惨的境遇，实际上是由于在转型期他很脆，非常的脆皮儿，经历了外来的冲击之后，非常难以抗住。所以它有没有缺陷？它有缺陷。它是一个完美的时代吗？它不是一个完美的时代。它是地上天国吗？它绝对不是地上天国。但是架不住，后来接手的那个是地狱里面的18层地狱啊！当你跟地下18层比的时候，地上的小二楼就变成了天国。全靠同行衬托。

嗯，什么张居正坏的？我觉得有些时候吧，咱们不能够直接去说一个人是好还是坏。但是可以说张居正他的这种——你说它是什么呢？改制、改良啊。总之呢，张居正的这一波改动，实际上让这个王朝走向了末路。至于说他根本上是不是怀着这样的目标啊，我不认为他必然怀着这个目标，但是呢，他做的事情——他做的事情本身造成了这个王朝走向末路啊。

嗯，速度快一点儿啊，来人，速度快一点儿。我们这现在进场最快的居然是清粉儿啊——这叫啥热度呀？我的天呐，进场最快的居然是清粉儿。你们能跟上吗？啊，速度能冲起来吗？哎呀，不是，就不是。还还得给自己贴个标签，说“我不是”。呵呵呵呵呵……嗯，摇人儿了是吧？对，嗯，赶紧啊，快点摇人儿，摇人儿。

嗯，啥？你说一说满清烂，同学就说你是明粉——怎么回击？你先认了呗，当明粉很丢人吗？你还得告诉他：“我不只是明粉，我还是宋粉，我还是唐粉，我还是汉粉。明粉又不丢人，当清粉才丢人呢！”你粉一个正常王朝有什么可丢人的？最丢人的是粉一个粪坑好吗？他当清粉他都不丢人，你还嫌当明粉丢人了吗？直接就接下来啊。

美白下蛋了吗？没有啊，因为正在学习化妆，所以今天下午的时候抹了一点粉底啊。呵呵呵呵，你们适应适应吧，我现在不好，我现在不好调啊。

嗯，啥？嗯，非常不适应啊。你不适应的话，你来那么早干啥呢？不行的话你歇一歇，等我化妆术大成之后你再来呗。我不着急啊。

嗯，有人举报我是吧？那就对了呀，他们已经举报我一年了。有人记得去年他们是打什么时候开始整我的吗？已经满了一周年了吧？再怎么算也应该满了一周年了吧。什么举报呀，然后什么报警呀，打各种轮番的各种电话呀，网络围剿呀，互联网的网军大冲击啊，已经搞了一年了啊。所以谢谢你们提醒啊，但是我已经比较适应了。

明的时候不用跪拜皇帝吗？不用啊。正常的王朝都不需要跪拜皇帝的。百姓不需要跪拜皇帝。什么时候需要啊？什么时候需要？你知道什么时候需要就行了。天地君亲师，在践行他天地君亲师职责的那一刻，你需要跪拜，这个是行大礼啊。什么是他践行职责？天子授权的时候啊，拜相啊，封侯拜相——这个是什么呢？授权，以天子的身份授权臣子需要跪拜。其他的时候不用，正常工作、上下级关系的时候不用。包括说父母这种关系，啊，这是典型的亲。什么时候要跪拜？过节的时候啊，过大节的时候，大年节的时候需要跪拜；长期在外回家的时候需要跪拜。啊，这个呢，是特殊场合。特殊场合的时候，行大礼仪的时候才需要跪拜。正常的往来关系是不需要跪拜的。连对老师啊，拜师的时候需要行跪拜礼，但是平日里见面作揖就可以了。所以这种见了高等级就要磕头的，这种远非我们的习俗，也不是我们的礼仪啊。自轻自贱从来不是礼仪的范围。礼仪是什么？礼仪表达的是我们彼此之间的关系与秩序，而不是一种自轻自贱。

呃，对，到了清的时候，磕头就变得非常的普及了。为什么？因为清是奴隶制啊。为什么在清的时候磕头变得非常非常的普遍？因为清是奴隶制。奴隶制是有强人身依附关系的。奴隶与奴隶主之间的关系，是一种完全不对等的人身依附关系。一个人当他属于另一个人的时候，用最重的礼仪表示谦卑，表示“我还是你的附庸”，就变成了一个正常的选择。所以到了清的时候，才出现了普遍性的啊，普遍性的磕头，以及普遍性的称大人。

之前在直播的时候，我也聊过“大人”这个词。在中国的语境当中，它的意思是尊长，基本上只有自己的啊，父母、至亲、长辈、恩人、恩人，然后有的时候呢，也会尊称自己的老师。这种情况下才会称对方为大人。大人是什么？大人是可以为我做决断、可以为我负责任的那个人。就好像你的老师说：“哎，你在学校里打架了，把你家大人叫过来啊。”大人就是这样的一个角色啊，他是可以为你负责、可以为你做决定的一个人，才叫做大人。但是到了清的时候啊——其实到了明已经开始出这个味儿了，为什么呢？因为元也这样。明的时候已经出现了这种大人泛称的这种味儿啊。这个习俗呢，是从元开始流传出来的。但是到清，它变成了一个非常普遍的行为，就是所有的这些啊，比你高等级的，你都要称他为大人。这个语境当中，其实就是视对方为可以决断自己人生的人。这是什么？这是诞生于奴隶依附关系的哈。

啊啥？今天好几个满姨。我日常这边都靠满姨来撑着的啊。你不要看我的直播间里面人数，人数似乎还可以，但实际上大部分时候都是靠那个小青粉、小满姨他们来撑着的，靠黑子来给我撑流量啊。所以大家对那个，嗯，对黑子呢，该怼的怼，但是要注意自己的言辞，以免被他人举报成功哈。啊，就稍微地容忍一下。我这边没有房管，为什么没有房管？因为一旦有房管的话，房管会爱我啊，房管喜欢我，就会把这些黑子咔嚓一下子拉黑了。这所有的黑子都被拉黑了之后，我这儿能少好几百个人。你想一想，流量是啥？流量是一个网红的命啊。啊，我是谁？我是一个励志要当网红的人啊。虽然还没红的时候就开黑了，但是不重要，黑红也是红嘛啊。励志要当网红的人，要容得下黑子。所以大家呢，一定要注意一下啊，跟他们对线的时候呢，保持一下自己的冷静，以免被他人举报成功。就完事儿了，就到这儿哈。至于剩下的啊，无所谓，他们是一天到晚蹲在这儿了，很正常。我的公屏上面有大量的清粉满意，而且其中有一些是收费过来的，那些人会格外地执着啊。大家要心里头有点数，很正常，很合理，没有问题。
