# 第52章  因为都没考满分，所以0分和90分都是差学生？ #传统 #现代

中医在过去是能够治病的，不能治的那些会被踢出去，能治的那些会留下档案留下来。我们是可以治病的，是可以解决瘟疫的。而西方人一直到近代都还在那里让他的理发师割血管。所以你说，“哎，那都是传统医学”——你这一个“传统”，就把会医学的和不会医学的并到了一起，把能治病的和不能治病的并到了一起，把救人的和害人的并到了一起，全部都叫做“传统医学”。他用打包的方式，把一个一整个完全不同内核的大梯队都包到一起去，然后说他都不行，就好像他很公平一样——哪个地方公平了？

中医黑之所以让你觉得数量很多，是因为啊，制造一个群体潜意识，他需要掌控媒体的通道，并且持之以恒，不停地宣传谎话。这个工作量是很大的啊。相较于影响整个的传媒思想而言的话，买通几个原本就利益相关的专家，是一件过于简单的事情，甚至可以说是理所当然的事情。嗯，啥？嗯，商朝就有针灸了，嗯，对对对。有的医学呢，医学这个东西它是要一点一点积累的。说实话，人类所有的知识都是需要一点一点积累的啊。所有的这些医学类的东西，其实呢，都是从0到1，一点一点迭代进去的。那么如何迭代？如何才能够从0到1？从0到1的过程是最艰难的一件事情。所以呢，唉，有一些地方说，他们的那个医学怎么样的横空出世啊，发明了什么显微镜，然后呢他就整出了什么一套医学。所以什么是医学？真的是我研究出了一个什么物理的或者是化学的知识，我的医学就会跟着进步吗？啊，我不认为如此。医学和他的这些内容是不一样的。医学是研究人，研究的是人的身体，而人的身体它不是一个死物件儿，它不是一个零件儿。实际上一直到现在为止，我们对于人身体的各种运作，还没有能够完全达成一个逻辑完全自洽的一套系统。也就是在医学领域上面，我们现在都还没有达到一个对自身身体的充分的理解。所以什么样子算医学要先尝试，一定是先尝试。所以我们的医学的故事当中包含了什么？神农尝百草。为什么要尝百草？因为你不知道呀，亲，因为你不知道，所以你要尝试。包括说我们的这个医学典籍当中，其启蒙观当中是有一个内观系统的。什么是内观？内观就是观察自身的身体运转。你说观得准不准，相对于今天的观察来说是否科学，这个都可以摆开不提。但是最重要的是，这个思路是对的。这个思路本身是对的，就是用各种各样的方式尝试了解自己的身体，并且是在活着的情况下了解自己的身体运转，而不是在身故之后拆解零件啊——零件与一个活着的人完全不一样，运转的机制模型完全不一样。所以呢，哎，什么是医学？医学就是要靠一个又一个活着的人，不断地积累经验，并且总结规律，才会产生出最终的医学。这就叫做从0到1。

而现在号称说那个什么“没有中医、西医，只有传统医学与现代医学”的这种其实很荒谬的说法啊，就是在忽略着中国传统医学上面在过去数千年间从0到1、从1到2、从2到5中间的这个不断迭代的过程。他忽略了所有的这些真实的数据积累和技术的迭代，直接按照今天的所谓科学的标准去审视过去，然后我说：“那都叫做传统医学。能治病的和不能治病的都叫做传统医学，公平吗？”中医在过去是能够治病的，不能治的那些会被踢出去，能治的那些会留下档案留下来。我们是可以治病的，是可以解决瘟疫的啊。你说有没有今天好？那不一定。但是呢，无论是外内附耳，所有的科目都有人研究，并且都有医书传世，都有案例、成方，是有成果的。而西方人一直到近代都还在那里让他的理发师割血管。所以你说：“哎，那都是传统医学。”你这一个“传统”，就把会医学的和不会医学的并到了一起，把能治病的和不能治病的并到了一起，把救人的和害人的并到了一起，全部都叫做“传统医学”。逗闷子的吗？这个思路啊，这种思路就和今天的那些人在那里喊说“所有的封建时代都是垃圾的时代，所有的封建帝王都是垃圾的帝王”一样的。他用打包的方式，把一个一整个完全不同内核的一个大梯队都包到一起去，然后说他都不行，就好像他很公平一样。哪个地方公平了？对，600分成绩不高，但是他比100分的成绩高了，他比零分的成绩高了。结果你把那个600分的和零分的放到一起，说他们都不是满分，成绩都可差了。就是这样的思路。我们在历史领域上面，包括在这些什么文化传承领域上面，全都是这样的思路。动辄冲起来就是：“哎呀，那些东西都是垃圾，都是那个已经过去的，都是落后的，都是闭塞的啊，都是老土老帽。”但是他把文明和野蛮并称在一起，都叫做土老帽儿，因为他不如今天的我们。这个事儿公平吗？第一口饭和最后一口饭放到一起来比较，然后把饿肚子的和吃饱了饭的放到一起，还说都不行啊，都没吃上鲍参翅肚。啥玩意儿？

不要说自己不懂得医学，尤其中医。换个话题吧，维持你在我心中的最美。你不需要相信我最美，你要是不喜欢我就可以出去，我不要求你觉得我最美，我也不需要维护在你心目当中我的形象。中国的传统医学，中国的中医，与其他任何地方的那些假医学、那些非医学、那些不能够救人的东西完全不同。我们是有自己的系统的，它从0到1，从无到有地开创了整个人类世间绝顶的医学文明。你要是觉得这个东西影响到了你的唯科学论，让你觉得我是一个伪科学者，是一个民科，是一个思密达、精神入脑的人，你可以出门左转。我不需要维护在你心目当中我的形象，因为我的形象在我的手里，不在你的心里。谢谢。溜溜球。

嗯嗯呃。中医黑多到底是因为什么？是因为人口基数大吗？不是。中医黑之所以让你觉得数量很多，是因为他们掌控了一种通道——传媒通道。中医黑让你觉得数量非常多，势力非常强，甚至呢他会假冒各种各样的名义去做这件事情。原因是什么？原因在于我们的传媒通道，我们的话语权不在自己的手里面，而在敌人的手里面去。包括说你在互联网上老是会觉得亲粉啊、小满姨啊数量特别多，真的是他的客观实力很强、他的人很多吗？不是的。他们才是反动者，他们才是分离势力，他们人很少，也得不到大家的支持，他们也没有真正的道理。他们所说的话，脚打后脑勺，自相矛盾，自己打自己的耳光，都是日常的现象。但是为什么你会觉得他声势浩大呢？因为他借由某一些资金的流量，使得掌控了某一些维度上面的话语权。他会用大量的水军，反反复复地重复他的谎言；他会一遍又一遍地把谎言用不同的人、不同的面目、不同的身份、不同的ID去拼命地传播出来，而且呢无孔不入。甚至其中包含了大量的电子生物参与其中，触发了关键词，立刻开始出来发疯啊。就会出现这样的情形。这种情形一旦出现，会发生什么现象？那就是你不管在听什么，都会有人插嘴说这玩意儿是伪科学。然后某一天的时候，你就会有一种潜意识，有一种直觉，觉得他就是一个伪科学，因为大家都这样说了。至于说其中的哪一些专家声音特别大的出来说，那就更简单了啊：制造一个群体潜意识，他需要掌控媒体的通道，并且持之以恒，不停地宣传谎话，这个工作量是很大的。但是呢，买通几个本来就利益相关的专家，便宜很多，非常便宜啊。相较于影响整个的传媒思想而言的话，买通几个原本就利益相关的专家，是一件过于简单的事情，甚至可以说是理所当然的事情。
