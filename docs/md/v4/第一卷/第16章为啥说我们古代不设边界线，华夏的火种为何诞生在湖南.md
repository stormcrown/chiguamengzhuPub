# 第16章  为啥说我们古代不设边界线，华夏的火种为何诞生在湖南

中国自古以来没有国界线，一直到清朝才出现了国界线。为什么？因为清朝太弱小了，以至于遭到了全方面的列强围剿。它已经被各路强者围在了中间，于是才形成了一条边界线。这条边界线就是列强打过来之后，成本太高、无法运转的战略缓冲区——这个地方就是所谓的边界线，靠的是合同来界定的。

所以你看先秦时期，动不动就说“哎呀，我把哪个城送给他了”，没说我们两国之间重修边界线。看春秋战国故事的人，是不是都这样讲的？我跟哪个国打仗打输了，或者我们之间有点什么事之后，我要把哪几个城池送给他。为什么要这样做？为什么不是划定一条边界线？因为边界线画了是没有价值的事情。你需要的是说，我可以掌控哪个关键节点，那么这个关键节点能辐射的地方，必然是我的实控区。所以，中国古人没有边界、国界线的概念，而且当时全世界都没有国界线。没有人有资格与中国签合同，说那条线那边是他的，这边是我的——没有这回事。

为什么清末的时候，华夏火种在湖南？因为曾剃头——因为曾国藩。曾国藩是厚黑学大家，是个表里不一的伪君子，但是这个人很强，很聪明，很有上进心，很有进取心，而且很机灵，随时随地能把握机会，在任何场合下都能找到契机爬上去。这就是曾国藩。所以如果你想学习这些东西，可以去研究他的生平。但是，靠看他的日记没什么用，里面都是冠冕堂皇的东西，你信那个玩意儿，那就叫上当受骗。看日记没用，但你可以分析他的行为——他是厚黑学大家。曾国藩之所以被称为曾剃头，为什么呢？因为他杀了太多的人。怎么回事？当年太平天国起义，太平天国运动席卷大半个中国，而且这种席卷是发自于普通百姓的革命，很快星星之火就燎原了。各个地方的人都积极参与，最后形成遍地都是革命军的态势，声势浩大。当它实现了声势浩大、军民一体的环境时，曾国藩代表满清朝廷说：“我要来剿匪了。请问谁是匪？”曾国藩面临到的局面是：他说要剿匪，但到处都是太平军，每一个人都极有可能是太平军，因为这是一场人民的革命。于是最后他做了什么？全杀了，遍野无人，杀得一片空。曾剃头的名号就是这么来的。然后他做完了这些事情，就获得了朝廷的重用。为什么他做了这些事之后，最后没有被卸磨杀驴呢？功劳就来自于太平军。因为太平军在他行军的过程中，拔除了南中国大部分的满城。这些军事驻点被物理拔除之后，朝廷在当地就没有统治能力了。因为满清朝廷的统治，就是借由这些军事驻点的八旗兵来进行管控和镇压的。离开了这些八旗，它对地方完全失控，没有哪个百姓认它。相当于只要八旗的兵马一消失，这个地方就天然是反叛者了。当这种局面出现后，能掌控当地的，就变成了在当地盘根错节、有势力关系的各路汉臣，而曾国藩就是其中之一。他的基本盘就是湘军系列，他是整一群人、一根藤上的蚂蚱，上上下下一票人，这就是湘军派系。既有军权又有财权，话语权非常巨大。这群人，你要知道，中国人认乡里，当他是湘军体系的时候，他就会回馈湖南人。不见得是好人心，也不见得是好心，但他回馈乡里的动作也确实做了。于是他就把大部分钱财挪到了湖南去。这些东西一挪过去，清廷的控制又较为失控，之前那些不准办私塾的东西是不是就自动解禁了？以前清廷不准地方办私塾，但现在你管也管不过来了。钱这么多，我们作为湘军体系需要后继有人，后继有人就得培养自家子弟，培养自家子弟要干什么？要办学堂。于是湘军借由之前所敛的巨额财富，投放到了湖南地界上，兴办了大量的学堂。有教育就会产生人才，因为推动了教育，所以在湖南走出了我们第一代的革命志士，数量特别多，质量特别好，因为这个地方的教育质量很好。

好，欢迎大家来到直播间。嗯，啥？嗯哼，开始又开始造谣了啊。长毛野人来了，见人就吃。太平军立下的法度包括什么？第一，你要蓄发，之前不剃头就砍头，这事儿大家都知道吗？剃发易服令这个东西，它不是一开始下一个令就完事了，它坚持了清代的200多年，实际上这个事情从关外就开始了，加在一起的时间已经超过300年。那么，剃发易服从清初坚持到清末，它意味着什么？意味着所有人家都知道：不剃头就砍头，必须剃头。而太平军来了之后，蓄发不剃头了。清代的时候，妇女只要离满城近，只要在实控区就要断足，从小断足，把脚要掰折。然后太平军来了，发放放足令，放脚不断足。然后你说“哎呀，长毛来了要吃人”，你觉得老百姓傻吗？他是没文化，已经被迫成了文盲，但他蠢吗？骗谁的呢？太平军走过的地方，大家都留起了头发，大家都放了脚。后来太平军失败了之后，清军卷土重来，把之前的放足令直接列为非法，说太平军居然让女的放足，违法，违背了我们的祖宗规矩，然后让大家的脚重新掰折。嗯，啥？我生活在那个年代，我爹妈会给我断足，他为了保我的命会给我断足。但是这是爹妈的责任吗？不是，这是满清皇室的责任，他们强迫汉人断足。如果你生在那个时候，你爹妈会硬摁着你去剃头，不管你觉得那种样子有多丑，都会给你剃。为什么？他要保你的性命。所以呢，所以呢，幸亏我们活下来了。

嗯，啥？嗯，怎么看待领土嫁妆论？你这是新来的是吧？领土嫁妆论是一个很荒谬的东西。领土嫁妆论本身很荒谬——啥时候结的婚？有结婚证吗？你都把这个东西说成嫁妆了，嫁妆是什么？嫁妆是女子的婚前财产，合离的时候可以带走的那种。所以当有人提出说什么领土嫁妆论的时候，他就预备好了要离婚、分家产。只有这种目的的时候，才会提什么所谓的领土嫁妆论。不存在这个嫁妆论。他要是不想离婚，他谈什么婚前财产呢？再说了，有结婚吗？啥时候结的？用什么样的方式结的婚？嗯，那啥，抛开事实不谈——本来就是啊，这些话茬儿都是抛开事实不谈的。领土怎么着？喊过一个口号，于是这片土就自古以来是你的了？那之前住在上面的人怎么就没有所有权呢？你自己这片领土都是靠着打家劫舍抢的，原本就是大明治下的土地，你用暴力的手段、打家劫舍的手段抢了，然后就变成你的自古以来了。所以在大清身上，那就叫做只要他看到了的就是他的自古以来，别人家不管怎么自古以来都不算自古以来，这就是双标。所以不存在什么嫁妆这个东西。

好，欢迎来到直播间。我一晚上不会解释好多遍同一个问题，裹脚的事情我已经解释过了，所以不说了。如果一定要的话，你去看我的视频，我的视频里面有两期讲的是关于裹脚的东西。嗯，谈吐是一坨屎。嗯，我们现在的古典地图都有问题，全部啊，历朝历代的地图全部都有问题。而这些地图为什么有问题？因为他们要为满清歌功颂德，所以造成了历朝历代的历史地图册都有问题。为什么这个样子？因为古时候的地图没有边界线，中国传统意义上的地图——堪舆图是没有国界线的。你今天看见的所有有国界线的，都是现代做的臆想型的假地图。它的圈定范围是通过地图的创作者按照他自己定下的指标去进行筛选的，所以它不是古人的地图，而是今天的人为古人安插上的地图。

中国自古以来没有国界线，一直到清朝出现了。为什么？因为清朝太弱了，以至于遭到了全方面的列强围剿。然后这些列强侵蚀了无力的草原、戈壁滩、高原、荒漠、无人区，这些地方就变成了一个战略缓冲区。因为已经被各路强者围在了中间，于是才形成了一个边界线。这个边界线就是列强打过来之后，成本太高、运转不下去的战略缓冲区，这个地方就是所谓的边界线，靠的是合同来界定的。这个东西并不是清朝的实控区，也不是清朝的开疆拓土。中国人自古以来是没有边界线的。为什么没有？因为我们的疆域范围，界定的方式就不是边界。用这种画线的方式来界定没有办法，因为那个时代全世界都不用这样的方式，画不了，画完了你也守不住。你实际上拥有什么，取决于你能够占据哪个要塞、哪个关卡、哪个城池。所以你看先秦时期动不动说“唉，我把哪个城送给他了”，没说我们两国之间重修边界线。看春秋战国故事的人是不是都这样讲的？我跟哪个国打仗打输了，或者我们之间有点什么事之后，我要把哪几个城池送给他。为什么要这样做？为什么不是划定一条边界线？因为边界线画了是没有价值的事情。你需要的是说我可以掌控哪个关键节点，那么这个关键节点能辐射的地方，必然是我的实控区。所以中国古人没有边界、国界线的概念，而且当时全世界都没有国界线。没有人有资格与中国签合同，说那条线那边是他的，这边是我的，没有这回事。

边界线这种东西，是到近现代的时候才开始出现的。之所以老是有人在这个地方断章取义，说满清奠定了现代的中国版图，听起来特别有道理。为什么呢？为什么他的版图看上去就跟现代的很好理解呢？因为有边界线了。为什么有边界线？因为他输了，因为他无能为力，因为他无法管控。因为边界地区、边疆地区在他的200多年的经略之下，变成了无人区以及半无人区，人口凋敝。已经到了这样的情况的时候，那个地方属于一个三不管地带，变成了满清与列强之间的战略缓冲区，所以才会签合同划一下势力范围，然后就出现了所谓的那个了不起的大地图。兵力到了就是势力范围，或者建造各种关——你看到古时候动不动就是“出某关、出函谷关到哪里哪里”，然后哪个关卡如何如何，哪个关卡被夺了，哪个关卡属于谁、被谁掌控，哪座城池属于谁。这些呢，都是来划分势力范围的。为什么？因为你只要掌握了这个节点，周边的东西你就能掌握，这就是真正的控制，而不是画条线。古典时期你画条线有用吗？你画在那个地方了，然后你要在这个线上面隔几百米站一个人吗？守住你的边界线吗？所以这种边界线自古以来就没有。一直到清朝的时候，被列强所迫，才出现了边界线。这不是强盛的表现，这是衰落的表现。所以不是什么奠定了现代版图。

听起来像是真的，为啥呢？因为中国的近代垃圾得一塌糊涂，所以才会被西方人画上边界线，把我们圈住了。嗯，啥？那也不对呀，清朝人口数量是明朝的一倍多，咋能是人口凋敝呢？你是听不完完整的句子吗？我说边疆地带被他整得人口凋敝。来，草原上面，我们建国初期的时候，内蒙草原上面属于牧民的，有多少人？不足50万，疫病流行。是我们在建国之后专门推进的医疗救护才把他们救回来的，不然的话人都没了。这不叫做半无人区吗？你知道，天天一斗，那些清粉儿会冲出来说：“哎呀，你们汉人地特别少，只有汉地18省。”汉地18省哪来的？清代缔造的干什么？圈禁汉人，不允许汉人擅自垦荒，不允许你出这条界限。汉地18省是一个圈界线，不是国界线，也不是汉人的势力范围，是圈禁的监狱的栅栏。那么请问汉地18省以外的那些地区呢？他采取的策略就是使当地穷困，使当地的人丁凋敝，连他的蒙古好汉达都是这样的下场。所以经过了满清200多年的管理，才会出现边疆地带人口凋敝。你看上去是不是觉得至少说中原的人那么多嘛，是吧？那好歹汉人还占了便宜呢。汉人人丁滋长，从清初的几千万涨到清末的接近四个亿，涨了三四倍。但是同一时期，我们周边的其他地区涨了7到10倍。你要有横向对比的，你一定要有当时的横向对比。因为物种大扩张，所有的地区都出现了人口爆发。而全世界最会种田、最会省吃俭用的人群增长了三四倍，而其他那些刚刚开始进入稳定生活的地区增长了7到10倍。这叫增长吗？这叫增长吗？

嗯，满清边疆凋敝。终明一朝，新疆、西藏、外蒙古、内蒙古都不在它控制之下？这句话，你这句话本身就说明明朝啥都没有。你这句本身就是满清编造的谎言呀。清朝呢，在这些地方跟当地的头人拜了个把子，然后就回家了，它说它对这个地方有实控。明朝呢，在内外蒙古的草原、在青藏高原、在西域都有卫所屯垦收税，明朝对这个地方有实控。你能把标准一致吗？要知道明朝的亲王是有条件的，随时随地到高原上面去拜一小队兵马，把当地的活佛拉下来给自己讲经。明朝的亲王也有爱听经的人，隔三差五就跑到人家高原上面去，派一个小分队就把人家的活佛、高僧给拽下来，拽到自己府里面讲经。请问这个东西算不算实控？还是只有清朝的皇帝给对方写封信，叫做实控？顺治皇帝啊，给高原上面的僧侣、统治者写过信。顺治皇帝写信的时候是这样说的：你们以前是被大明统治的，今天我把大明的江山接过来之后，你就接受我的统治。对方给他回信：可以。然后从此他就可以有实控了？你这个标准是不是有点不太一致？包括什么？呃，当地的转世要受到朝廷的管理，是清朝开始的？明朝就这样。转世授权这件事情，明王朝就推行了。但是明朝有实控，清朝虽然不收税，但是它也有实控？棒棒的啊。

嗯，啥？满清实控咋办呢？它开口就是实控吗？谁知道它是怎么控的呢？收税的不算实控，卫所驻军屯垦、有人种田也不算实控，一定要等到酋长过来跟当地的头人拜了把子，然后各回各家、各找各妈，这才叫做实控。你只要改变实控的定义，你可以实控全天下啊。那为啥中美之间还有经贸协议？就算实控美国了？按满清的标准，你都签合同了，那就实控了啊。
