# 第6章  终于明白有些中国学者，因为西方历史造假而破防的原因

这些优秀的学生、社会的精英、科研人员，他们的成长履历当中，不管学什么科目，都有一个精神支柱——来自于西方的先贤们。他们通过知识的积累，从而获得了社会地位，获得了巨大的正向反馈，以至于说什么话别人都听。这是什么？这是巨大的心理满足感，是一个巨大的心理满足感。

也就是这些人、这些学霸、这些优秀的学生、优秀的人才，在他的成长过程当中，不断获得正反馈。小时候，别人会说：“哎呀，你分儿好高呀！”长大了之后，别人会说：“诶，你好会学习啊！”甚至说：“居然这个你都能研究出来，太厉害了！”所以，当一个人成长的过程当中，他的心理支柱非常重要的一环，就来自于西方的这些哲学的、科学的、技术的先贤时，你说西方的那个历史造假，那些先贤都是编造出来的，他们的著作都是抄袭的——你知道这是什么吗？你要拆毁他心灵当中最稳定的三角形当中的一个角，你要卸掉它车的四个轮子当中的一个轮子。这种崩溃感，普通人是很难察觉到的。这是一个很朴素的道理，但是呢，不代表容易接受。

嗯，怎么感觉没昨天白了？可能是眼睛的问题。为什么要这样说？因为昨天晚上刚做了美白面膜。今天晚上做了美白面膜，今天说我没有昨天白，我认为是你的眼睛需要修正一下啊。嗯，啥？嗯，别做卖面膜智商税。哎，咱们知道一件事啊——嗯，谢谢小笨孩儿的仰望——咱们得弄明白一件事：心理需求也是需求的一种。听我直播听得久的人，心里头应该有这个概念：心理诉求对于一个人来说是很重要的一件事情。它跟那个东西实质上面的效力到底如何？当然这个效率有待考证啊，有可能好，有可能不好。但是呢，最重要的一件事情就是，哪怕它没效果，心理抚慰也是重要的价值。为什么？因为它让我快乐。嗯嗯，啥？起码便宜。那是的啊，它比其他的心理抚慰要便宜多了啊。

于老师又在批评嘲讽西史辨伪了。为什么研究唐史的要这样做？嗯嗯，研究唐史不代表……嗯，这个跟族群身份呢，也没有什么关系。咱们得知道一件事儿啊，咱们得知道一件事情：其实呢，反对西史辨伪的当中，不全都是研究希腊、古希腊史、古罗马史的，不全都是这些人。他还包括史学界的各方面的人才，同时呢，也包括了理工类的多方面的人才。大家一定会注意到一件事情：严重的反对西史辨伪，并且认为西史辨伪这件事情就是九漏鱼，就是民科，就是愚蠢，就是无知无能的表现的，其中有相当一部分是理工界人士。所以他们不全都是直接利益相关方啊。这里面呢，是包含了很多间接利益相关方的。这种利益当中包含了一个项目，就是心理价值。

很多人容易忽视这件事。就像我们刚开始进直播间的时候说，哎呀，不要做美白面膜，那玩意儿就是智商税。你跟我说它是一个智商税，会造成一个什么效果？我本来呢，只是几块钱一张贴上去的一个美白面膜，但是你跟我说那玩意儿就是智商税，你用这个东西就没脑，我是不是心情直接就下降了？但是呢，贴了美白面膜本身这件事情，是不是会让我快乐？哎，心情会上升。这个呢，就是情绪价值。人呢，是有心理价值诉求的。而这个心理价值当中，包含一个很重要的维度是什么？我们身边多多少少交际圈当中都会出现，哪怕再少啊，也会有个板——学历很高的人。大家应该多少都认识几个啊。学历很高，社会的资历很高、地位不错的人，你总是能遇到一部分。那么这些人有一个什么样的特点？他通过知识的积累，从而获得了社会地位，获得了巨大的正向反馈，以至于他说什么话别人都听。这是什么？这是巨大的心理满足感，这是一个巨大的心理满足感。

也就是这些人、这些学霸、这些优秀的学生、优秀的人才，在他的成长过程当中，不断获得正反馈。小时候，别人会说：“哎呀，你分儿好高呀！”长大了之后，别人会说：“诶，你好会学习啊！”甚至说：“居然这个你都能研究出来，太厉害了！”这些都叫做正反馈。正反馈会树立他的价值观，会树立他的自信，甚至这些东西就是他精神当中不可或缺的支柱。我们知道，产业当中有产业支柱，心理当中也有心理支柱。每一个人的生命当中都存在心理支柱。这个东西为啥要绕这么远？西史辨伪跟这个有关系吗？有关系。

你要知道，我们的生活当中有很多的高知型的人才，他在成长的履历过程当中，学习了很多榜样。任何人在前行的过程当中，都需要引领者，都需要精神导师，都需要榜样的力量。而这些优中选优的社会精英，他们的榜样是谁啊？大家记得吗？自己读小学、读初中、读高中的时候，教学走廊里面贴的历代先贤，是不是绝大多数都是洋人？对这些优秀的学生、社会的精英、科研人员，他们的成长履历当中，不管他学什么科目，都有一个精神支柱来自于西方的先贤们。所以，当一个人成长的过程当中，他的心理支柱非常重要的一个项目，就来自于西方的这些哲学的、科学的、技术的先贤时，你说西方的那个历史造假，那些先贤都是编造出来的，他们的著作都是抄袭的——你知道这是什么吗？你要拆毁他心灵当中最稳定的三角形当中的一个角，你要卸掉它车的四个轮子当中的一个轮子。这种崩溃感，嗯，普通人是很难察觉到的。

当然了，你可以去跟他说这个东西啊，他的作者是谁，和这个定律本身没有关系。他的真实作者如果是中国人的话，你不是应该更骄傲吗？这是一个很朴素的道理，但是呢，不代表容易接受。为什么这个样子？因为这些人在他不断学习的过程当中，已经刻下非常清晰的烙印：这个定理是某一个洋人发明的，那个定理是某一个洋人发现的，那个定理又是某一个洋人总结出来的，这套规则是某一个洋人撰写的。而这些东西在他当年成长的过程当中，他就已经不断地拷问过自己了：我这么聪明，为什么这个事儿不是中国人干的呢？于是他已经给自己自圆其说过了。他们会给自己安排各种各样的理由，让这件事情能够圆满起来。它包括但是不限于以下的几种理由，例如说，虽然当年的洋人不错，但是今天的中国人更强；例如说，虽然洋人啊，在那个时候突然迸发出了很多的科学家，但是他们以前穷啊，人生总有起起伏伏，他们当年曾经发达过，现在我们想办法追上就是了嘛。各色各样的让自己的心灵变得丰满的假设，他们在自己学习和成长的过程当中，都已经完成建设了。也就是说，他有一根支柱是“洋人最伟大”，对他来说已经不那么要紧，甚至在是他精神稳定、心灵稳定很重要的基础。结果这个时候你突然拆台，你说不对，那些洋人就是编的，那些洋人就是抄的，那些历史都是假的，你会把他逼疯的。

所以别说于老师了，他好歹还是历史学界的，身边呢，还有很多历史学家是和他是同类啊，彼此之间晚上会一起吃饭的关系，写论文的时候会互相提携的关系。他还是在历史学界圈子里面的，本身就是一个间接利益相关方。就连那些玩并非利益相关方的理工类的学者当中，都有大把的人认为，说思考西史辨伪，是不是没有上过学。唉，好，欢迎大家来到直播间啊。
