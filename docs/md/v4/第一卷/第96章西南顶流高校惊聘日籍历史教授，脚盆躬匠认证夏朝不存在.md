# 第96章  西南顶流高校惊聘日籍历史教授，脚盆躬匠认证夏朝不存在

嗯，好的，我们继续下一段。大家最近有没有看这么一件事？有个大学，在西南地区，我国西南地区的一所高校，想聘请一位外教，一位外来的历史老师，而且是搞考古的历史老师。这个老师是个什么人呢？是个日本人。对，他们找了一个日本人过来，要当历史老师。带感吧？嗯，对对对。

我们西南那里的顶流大学，不是什么三流大学，不是什么下三流的大学，是正正经经的顶流大学。具体是哪个学校，公屏上有人打了，他们说的就是对的。请了一个日本人过来给大家教历史。然后呢，我们现在来讲讲这茬事。

请过来的这个所谓的得过奖的历史教授、历史老师，他在我们这里做过演讲，做过报告。什么样的报告呢？研究的是啥呢？研究的就是青铜器的“西来说”。就是说，这个他们计划聘请的教授，他的研究项目当中，就包括了中国青铜器的“西来说”。然后是怎么来的？是通过草原传来的。从草原上面传播了青铜器给我们。就是说中亚、西亚那个地方发明了青铜器，然后传播给了我们。他们使用的证据，就是在蒙古高原上的墓葬中有青铜器陪葬。所以就是这样的学者，然后你把他请过来教我们的学生，你让我们的学生学什么？就学“西来说”呗，就学日本人治学严谨呗，是吧？他们的学术成就比我们强是吧？啊，棒棒的。

为什么呀？能讲根源吗？我就问你，这样的教授来这里讲课的时候，他会不会告诉你说“我的祖上有罪呀，抢了太多的东西了，所以我那边研究得贼好，就是因为他们是劫匪”？他会这样说话吗？他要是这样做的话，我敬他是条汉子。不可能啊！啥蒙古那地方连锅都没有，给我们传青铜器？人家说了青铜器是通过草原传给我们的。至于草原上有没有矿，有没有定居生活，有没有能够烧窑的地方，别人不在乎。别人那里有一根针，针就是他发明的；别人那里有个勺子，勺子就是他发明的。一直到传播给了我们，我们永远是一个学习者。我们这里不管发现了什么，都不是原生的，都一定是外头来的。这就是万物“西来说”。

对，传达一下。这个学校是不是前两年的时候还出过新闻事件啊？新闻系的，一个女性，在地铁上欺负一个……算是农民工吗？啊，反正是一个工人阶级，欺负一个工人阶级的大叔，硬要诬陷别人，在互联网上写小作文。那个也是他们家出来的。然后呢，后来好像，嗯，对啊，我就不叫人家名了。后来是不是雪也照常上？所以呢，这个地儿啊，本身有一点什么小问题哈。就是这个学校，来来来，记得了吧？弄清楚了吗？对，就这个学校。现在呢，聘用一个日本人给他们当历史老师。然后互联网上掀起了腥风血雨之后，现在说“不聘了，不聘了”，不聘了就怂了。但是不管他怎么说，聘或者是不聘，他把人家找过来，讲过课、开过讲座，这个事儿已经是实现了的，已经是板上钉钉操作过了的事情。所以不是你今天恐惧于流言，然后说“哎，我不聘了”，我不聘了就完事了啊。不能够这个样子。做过的事情咱得认账。所以呢，咱们还是要把这个事儿捋出来。

就是这里他聘请一个日本人来当历史老师，这件事情本身，你要是说合不合规，那肯定合规啊，那老合规了。但是为什么不行？为什么不合适？为什么不可以？这和对方的这个研究的水平、获过多少奖，关系并不大。主要的原因是什么？历史是一个什么科目？历史是归在文科里面的。而文科是什么？文科的第一要务是什么？文科的第一件事情解决的是立场——我是谁的问题，你站在哪一边去看待这件事情的问题。看历史的时候，你要是立场站在某英身上，那当年中国是个什么玩意儿啊？活该被他抢吗？但是你站在中国的百姓这边呢？凭什么抢？凭什么欺负我？这个东西叫历史的立场，这就是视角。所以历史第一件事情要解决的就是“我是谁”。

而日本人和其他的那种地方的人一样吗？不一样。为什么不一样？你就自己想一想。日本人那里的老师、历史老师，他能够研究的历史是哪个期间的历史？就日本岛上面，他自身的历史实在是乏善可陈。真的让他们去好好研究，都研究不出来几本书。所以他那个地方的历史教授、历史研究员，研究的都是中国历史，他们那儿都是研究中国历史的。然后啊，取得的成就是确实很厉害啊。但是咱们得问一问：请问他为什么有那么好的研究的基础？为什么他那里的研究做得就这么棒呢？是因为他们方法特别棒吗？是因为他们特别聪明吗？不是。是因为他们当年侵华的时候，劫掠了大量的历史文献。中国的历史文献被他抢的，我们自己都没剩下仨瓜俩枣的了，就已经抢到了这种地步啊。无论是盗墓——西方人盗墓，你当他没参与吗？日本人那边的人，考古学家参与到我们这边的那些人的盗墓行为当中，去窃取、大规模地窃取文物，然后去把我们的历史典籍扫荡一空，直接搬过去。

这些年是不是有很多人跑到日本去买什么汉方药？有吧。你们出门旅行的时候，很多人都会介绍，你说你去买点汉方药，那可好了。请问他为什么会有那么多的汉方药？哪来的？怎么弄的啊？因为他劫掠了大量的医书，然后把里面的成方翻出来做药啊。好几十年的时间，是不是很多人说，日本人那里的音乐家做东方音乐特别的牛，比别人厉害，中国的音乐家都没有，人家做中国风的音乐做得好。为什么？我们的音乐书、音乐曲谱都被他们劫掠一空，我们自己这边都快没有了。中国这里的古典雅乐系统的文献被劫掠一空啊。所以这是实力的问题吗？不是实力的问题，这是立场的问题。

这个日本的研究学者，他自己就是持青铜“西来说”的。全世界青铜考古发掘量最大的中国，有大规模铜矿和锡矿开采痕迹的中国，有高温炉窑塑造历史和经验的中国，有大规模冶炼场所的中国，不是发源地。要到那些没水没柴、没矿没定居的地方，才是青铜的发源地，我们只能是学生。嗯，为什么我们的文化界这么软弱？你看这个，我为什么要讲这个聘用的这件事情？这种聘用的这件事情本身就代表了什么？就代表了话语权啊。聘用这个动作本身就代表了话语权。什么样的履历、什么样的学历，能够在中国的学术圈里面说得上话？这个叫做学术话语权。而现在很明显的，我们的顶流高校按照什么样的标准来聘用教授老师？是按照国际影响力、国际大奖、国际论文数量来聘用老师。当你用这种指标来聘用老师的时候，对方需要掌控的是什么？他只要掌控几本杂志就可以了，他只要控制几本杂志，让这几本杂志上面刊登的文章，控制几个奖项，让这个颁奖的项目受到他的审批。那么凡是在这个地方能够刊登的，在他这里能够拿奖的，都是符合他的思想流程的，就完事了。凡是在这个地方经过了一轮筛选的，在我们这里就能够升官加职，就可以——讲师评副教授，副教授评教授啊。如果是这个样子的话，我们的文化产业、我们的学术领域会不会受到对方的影响？我们这里的任免、我们这里的升职，要受到他的奖项的影响，要受到他的论文的影响的时候，话语权在谁手里啊？谁说了算？只有相信青铜“西来说”的人才会在这个地方独占鳌头的时候，那么青铜“西来说”不管他是证据多么单薄，逻辑多么的不清晰，它都变成了学术主流。但是学术有主流吗？人中国人是人吗？问题追根究底就是这个样子的。
