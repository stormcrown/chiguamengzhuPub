# 第18章   爱反思的公知，凭啥教化大众？

不是单纯的自我怀疑，也不是单纯的自卑。单纯的自卑不会导致逆向民族主义。逆向民族主义的表现是自我否定，而且是从根儿上否定——照照镜子就觉得自己这个品种长得不好，做做题就觉得自己这个品种狭隘了，跟别人聊聊天，就觉得自己这个品种不够浪漫。这种类型的人就叫做逆向民族主义。而过去的几十年间，包括民国期间的相当一部分公知的先辈们，他们的表现特点就是这种自我否定。

正是因为这些号称要教化众生的公共知识分子，在教化众生的时候，没有让众生产生希望和自立自强，反而让这些众生产生了媚外的、自卑的、自我贬低的、自我否定的，甚至自我毁灭的思想。这种方式极具破坏性，所以“公知”现在才会变成一个恶臭的名字，提起它时才会出现阴阳怪气。

嗯，什么是公知？现在还需要问这个问题吗？啊，什么是公知？谢谢观自在。一定要说什么是公知的话，公知全称叫做“公共知识分子”，干什么的呢？教化世人用的。你听名字就知道了：公共知识分子，就是说话说给大众看的人。这种公共身份是他身上最重要的元素。所以公知当中，文艺类的人物特别多，因为他们创作的内容直面大众。公知的读音，重音不在“知”上，而在前面的“公”上。他的特征就是要把所谓的知识传播给大众，教化众生——这是他们所坚持的。他们自身的自我价值问题就在于，他到底有没有资格来教导大众。

之所以现在公知变成了一个嘲讽型的、可以随时随地阴阳怪气的词汇，就是因为过去的40年中，社会上反复不断活跃的各路公知，以及被吹捧得更加古老的民国时期的大师们——这些公知的鼻祖们，他们统一的特点就是特别的逆向民族主义。我有一个视频专门讲解逆向民族主义。什么是逆向民族主义？就是特别的看不起自己。有兴趣的小伙伴可以去揣摩一下：一个人如何生存于天地之间？非常非常重要的一个东西，就是你觉得你这个人还行。所有在座的小伙伴，你们在任何时候会完全彻底地否定自己吗？学习不好的人会不会说我运动还挺好的？学习和运动都不好的人，有没有说我人缘挺好的？智商不够的人有没有说我情商很高？有没有？有吧。甚至做事每做每败的人都会说，我只是运气不好。是这个样子的吧？大部分的正常人都是这样的操作方式。为什么？因为这是人类生存的一个根本，这是人类存活的根本。不是说你把，没说你。没说你，你帅得不得了。

为什么会这样？因为人类会思考。思考会带来一个毛病——所有东西都有正面作用和反面作用。思考让人先进发达，同时它带有一个缺陷：思考着，思考着，想多了，最后就开始说“我是谁”、“我从哪里来”、“我往哪里去”。所谓的哲学三问，就诞生在人类思考的终极上面。只要你去想事，闲着没事干想啊想，想得越来越多，最后得出来的问题就是：我是谁？我在哪儿？我在干什么？我干啥呢？想怎么样呢？啊，这个问题就出现了。这种问题一旦出现，你就会发现自己对自己产生自我质疑、自我怀疑：干嘛在这儿喘气儿呢？多不多余啊？很容易产生这种想法。于是，在人类的思想上就有一个弥补机制——机制有缺陷、有漏洞、有bug就有补丁。人类的补丁机制就是找借口、甩锅、找理由。所有人身上都有。我成绩不好，那就是我人缘好；我跳得不高，那就是我跑得快；我腿短，那就是我智商好；如果说我什么都不行，那就是老天爷对不起我，嗯，就是我的运气还在后头呢。甚至你会包括说，我们在传统文化当中的心理疗法、心理疗愈法这些各种各样的先生们，他会干什么？哎，这个东西克你，那个东西克你，你要调整你们家房屋的坐向、朝向，你的桌子的摆的位置、床摆的位置、颜色。干什么？都是来弥补内心上面的这个小bug的。人类身上都有这种bug，所以特别需要进行一个弥补。你说怎么办呢？我开始自我怀疑了，我应该如何？哎，我就把它串台串回来说：你很好，你不错，错的是你旁边那棵树。这个呢，就是一个正常的思维方式，所有人身上都必须有这个。

但是逆向民族主义，他就是缺了那个补丁，就是不限制地怀疑自己，甚至看见自己身上的优点时也要怀疑。怀疑到了什么程度？如果说自己很讲礼貌，那就是没出息的表现——因为没有突破界限的能力；如果发现自己很会读书，那么自己就是一个情商不足、灵巧性不够、人缘不好、前程不显、只能给人打工的只会死读书的书呆子。这种思维方式就是自我否定式的思维方式，在逆向民族主义当中非常普遍。它的特点不是单纯的自我怀疑，不是单纯的自卑，单纯的自卑不会出现逆向民族主义。他的表现是自我否定，而且是从根儿上否定：照照镜子就觉得自己这个品种长得不好，做做题就觉得自己这个品种狭隘了，跟别人聊聊天，就觉得自己这个品种不够浪漫。这种类型的人就叫做逆向民族主义。

而过去的几十年间，包括民国期间的相当一部分公知的先辈们，他们的表现特点就是这种自我否定。而且他不是单纯地否定他自己，他在否定自己的国家、自己的文化、自己的传统、自己的族群——他在否定自己这一个类别。并且，以公共知识分子的身份，努力地把自己的这种非常拥有自毁倾向的思想传递给社会公众。本来，对于社会公众来说，哪怕在最苦的日子里面，也会安抚自己：好死不如赖活着，活着活着、苟着苟着，总有一天能够得见天日。这是一个非常正常甚至非常本能的思想，也是任何一个群体能够活下来的基本盘。但是这些公知喜欢什么呢？你错了，你不对，你不好，从根儿上就坏了。所以你看过去的几十年间，这些公共知识分子的著作当中就包含了大量这种内容，说什么“你不可能是汉人啊”、“秦始皇是说蒙古语的女真人啊”、“中国人的劣根性”诸如此类的书，都是非常典型的逆向民族主义的书籍。他试图让中国人相信自己不行，自己不可以，自己错了。而且这种错不是经过学习可以改变的，它不可更改，它刻在骨血之中，除非哪一天你长得也不像你了，你做事也不像你了，你彻头彻尾地变成了一个外国人，除此之外无可改变。这种思想就是公知当年曾经传递给我们的逆向民族主义思想。

之所以公知这个名字变臭了，就是因为这些号称要教化众生的公共知识分子，在教化众生的时候，没有让众生产生希望、产生自立自强，反而让这些众生产生了媚外的、自卑的、自我贬低的、自我否定的、甚至自我毁灭的思想。这种方式极具破坏性，所以公知现在才会变成一个恶臭的名字，所以提起它时才会出现阴阳怪气。自我怀疑，要把人整疯的。对啊，人是不能够这样从头到尾、完完整整地自我否定的。人类的思想机制不能够接纳这件事情，一旦你陷入到这种完全的自我否定，你之后走的所有的道路都会毁灭，自己一定会这样。这个就是能够思考的智慧族群会面临到的客观问题，这是一个客观的短板。

哎，好，欢迎大家来到直播间。要说到这个中国人身上的这种客观的短板啊，那就是中国人非常容易受到知识分子的引领，非常容易受到知识分子的影响。为什么呢？因为中国人看重知识、看重文化。因为看重这些东西，因为在意这些东西，所以中国人格外容易受到知识分子的影响。嗯，“天下皆下品，唯有读书高”。因为中国人开启文明太早了。我们古人的时候，我们的古人常常把我们周围的人统称为蛮夷，这个不是专门指谁，他叫做“我不是特指你们的哪一位啊，在座的各位都是啊”。中国古人就是这种姿态。他为什么有这种姿态？中国的古人为什么有这种傲慢的、狂放的姿态？因为我们进入文明的时间过于早，太早了，早得有点离谱。因为崛起得太早，所以当我们穿上衣服的时候，发现别人还光着；当我们开始使用精致的耕作的工具，然后每顿饭都能吃上东西的时候，别人还在那里风餐露宿；当我们这里已经开始讲究家庭的秩序，谁可以做什么，谁不可以做什么，走路的时候都知道要礼让，要分一下先后秩序，而不会你争我夺的时候，别人那里还在疯抢。这种情形就会导致我们的祖先抬眼一看，遍地蛮荒，所以他就将自己称为文明之后，将其他的所有一切都统称为蛮夷，连分都懒得分，都到了这样的地步。

因为发生了这样的过于早的、太早太早的文明开端，于是造成了一个问题：在这个文明演进的过程当中，我们在文明这件事情上、在秩序这件事情上获得了很多的好处。因为当你拥有很稳定的秩序、内部很好的分配机制，不再争抢，可以进行细化分工，在这种情况之下，你的生产会不断地进阶，不断地扩大。这就像流水线生产一样，就像我们在企业里面的流水线工作，任何一个工作岗位上面拧螺丝的人，都不知道产品的全貌什么样子，你让他做他也做不出来。这就是社会分工精细化。你实现了这个社会分工的精细化，会造成什么问题呢？就是每一个人都能够发挥自己的长才。跑步跑不过别人的人，在丛林当中会死去，但是在中国这里你跑步不行，你手灵巧，你就可以过得比别人还要好。最后产生的一个结果是，我们就会越来越尊重秩序，越来越尊重稳妥，越来越尊重那些能够开创知识与秩序还有技术的、先进的文化人。而我们是特别特别尊重文化人的，因为我们自古以来，几千年以来，就在这事上拿到了无数的好处。所以中国人极其在乎文化人说了什么。

而且我们想一想，我们两千多年前到处都是贵族，结果呢，出来了一个孔夫子说“有教无类”，谁都能读书，谁都能学习怎么当管理者。以至于后面一代一代的思想迭代之后，最后就呈现出了“王侯将相，宁有种乎”。这些思想都是一点一点地萌发，一点一点地出现，循序渐进产生的。当中国人这里已经出现了这些东西的时候，我们就会格外格外地尊重、格外格外地在乎文化人。但是过于在乎文化人的思想，就在上个世纪的时候，导致中国人在接受了一些文化人的错误思想之后，也受到了最严重的冲击。像这种“哎呀，混血才好”、“别人家多高级”、“你看他白人长得俊男靓女的老漂亮了”这些思想，你以为他只教给了中国人吗？不是的，这些“洋人从头到尾都很好”的思想他交给了全世界，但是中国人受到的影响最剧烈，就是因为我们自古以来就在乎这些文明之崛起，就在乎这些文化人说了什么，就在乎社会上面谁最体面、谁最妥帖，我得向他学。因为我们在乎它传递的讯息，所以受到的影响最严重。而那些文明开化较短暂、较后期、较晚的这些人群，反而没有受到严重的冲击。就像这两天大家都在那里看的那个阿三啊，巴拉特那边，有没有发现他们就不在乎这事儿？你以为当年那些殖民者去的时候跟他们客气吗？不客气的。殖民者什么时候跟殖民地的人客气过？没客气过呀。还不是吹的，好像那个英伦岛上面的人就是最顶级的人，都是这样整的。但是为什么隔壁的人就不是那么在意这件事情呢？他们为什么拥有那么强烈的自信心呢？因为他们自古以来连成文史都不记载，他们压根儿不考虑这个问题。谁的神话故事讲得好，谁就是扛把子——是靠这个东西来的。所以人家压根儿不在意你这个引领。谁最强？打了砸了的，行，我就跟着你干吧。妥协可以妥协，但是他并不是追随，也并没有想要用别人的思想来改造自己，他们就不存在这样的欲望，压根儿没有这样考虑过，也没有尝试过这样的想法，所以他反而没有受到这些东西的影响。以前就没有考虑过这件事情，因此甚至于学习了别人的语言之后，他还可以嫌弃人家宗主国的语言不正宗——就是因为自信。而这个自信是什么？这个自信其实是人类在自然情况下必须拥有的一种品性。就是你不受到任何文明的熏陶的情况下，你天然地就会拥有一种盲目的自信。而文化的开化程度较低的群体，普遍性拥有近乎于猖獗的自信，大家只管到世界上去看，全这样。因此，这些人相较于中国人，他们反而没有受到上世纪那些所谓公共知识分子所传递的信息的影响。

嗯，这种狂信也是对付公知的好方法。就是这样的呀。因为公知是什么？公知是一种认为自己很有文化，于是要去教化众生。但是如果他真的很有文化，他就知道他教的那些东西是坨屎了。所以这些人都是一瓶子不满半瓶子咣当的东西，他们当中没有真正的大智慧的人。有文化啊，有文化也可能很有文采，但是没有大智慧，也没有大心胸，更没有大格局，然后他才会成为上世纪的那种类型的公知——自我贬低、自我否定，并且想要把这个东西传递给所有其他的人。那么什么才是应对这些人的防护盾呢？要么你要超过于他的智慧，超过于他的格局，戳破他的谎言；要么你就变成一个压根儿听不懂他在说什么的这样一片白纸，白纸也不会被他欺辱的。当你什么都不想，根本对他接受无能的时候，他就无能为力了。

嗯，啥？哎呀，我已经讲了这么长时间了吗？对对对，啊，我要开始带货了。这里是一个带货直播间，咱们不能搞岔了事啊，责任不能偏了。我们这里是一个带货直播间，来，走起走起啊，嗯，带什么东西呢？
