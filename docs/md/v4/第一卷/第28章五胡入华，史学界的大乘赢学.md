# 第28章  五胡入华，史学界的大乘赢学

我们经历过太多太多的苦难。五胡乱华的时候，北方千里无人呐。剃发易服的时候，很多人都不记得祖先衣冠长什么样子，从认字变成了文盲。就经过了这么多、这么多，最后我们还是能够再支棱起来，遇到阳光雨露的时候，能够再重新地爬起来。在最直的时候能屈，在背负了之后能够生，是很了不起的事情，绝无仅有的伟大。

就像宋金之争。为什么居然会觉得金国掳掠宋国帝姬，并且百般蹂躏，这件事情叫做“融合”？“太好了，干得漂亮”？军事的威压，战无不胜，女真不满万，满万无人敌。为什么他能够这样牛哄哄地开始吹这个东西？因为他把屁股坐到了战胜者的那一边，他只决定“谁赢我帮谁”。“谁赢帮谁”这件事情是对的吗？这是投机行为，这是押宝行为，这是赌徒心态，唯独它离正义背道而驰。

如果说你站在宋的那一边，你当然可以批判赵宋皇室的无耻、软弱、无能，但是同时更要批判侵略者的残暴、野性、无德。这个东西就是你的站位问题——你到底站在哪一边。如果站在了渔猎部落的那一边，那抢赢了就是最厉害的事情，而且抢赢了之后还把对方羞辱了，让对方居然跪服了，那就太高兴了，能够欢欣雀跃。这意味着他站在了渔猎部落的那一边。但是按照他自己的身份，他不可能是一个渔猎部落，所以干什么？他站在了赢家的那边，甚至没有站在自己的这一边。

一个人要么你选择站在正义的这一边，这叫做顺应天道；要么你站在自己的这一边，这是维护自身利益。但是像这一种，打着政治正确的旗号，站在赢家的那一边，这叫做什么？这叫做奴婢心理——谁赢谁就是最好的主子。他是没有自我意识，没有自我认知，也不知道自己是谁。他只选择谁赢我帮谁，谁赢谁是我的同伙，谁赢了谁是我的同类。他已经预备好了，随时随地认野爹。谁都可以是他的爹，除了他的亲爹。这种行为的方式，即使放到野兽界也是令人不齿的。但是当他愿意甚至自甘为奴的时候，这就很正常，因为对于奴隶来讲，这种心态太正常了。

就像你刚才说的那个什么，呃，契丹啊。辽实际上是剃发易服的祖宗，中国人所遭遇的剃发易服的第一波就是辽人做的。虽然你看墓地的墓志里面的绘画的时候会发现，辽国的贵族也会使用汉制的服装，但实际上在辽的内部，是发行过剃发易服的相关政令的，要求剃发，要求服装改左衽。但是他比后来的清要柔和一些。为什么？因为他说的不是砍头，而是“不照这样子改，要依律而为”，就是依法办事。你不按照我的服装改，我要依法惩治你，这个法比后世的砍头要轻一些——这样的一个区别。但是剃发易服原本就是辽推行的。那你说当他都已经开始改变你的服装，改变你的发型，甚至契丹人还专门设计了一款单独的契丹文的时候，他要改变你的文字系统；他在官僚的录取上面，用不同的录取方法和标准。结果你硬说是一样的，因为有汉奸家庭是吗？那汉奸家庭在这种时代的汉奸家庭是什么身份？无论是辽、金还是后来的清、元，都是一样的。这种能够在高层获得利益分成的这些家庭，看上去是豪门，实际是奴婢和依附者，只是早早投靠的奴婢，所以给的优待比较多一点点。放到轻的话语上面，这个就叫做“抬旗”。

所以实际上我大部分时候是非常不喜欢“政治正确”的。我不喜欢“政治正确”，因为它不唯物。唯物主义要求的是，不管这个事儿好不好听，我们都真实地面对他；如果他对我不利，我就想办法改变他，这个才叫做唯物主义。实实在在地面对世界，是什么就是什么，不是什么就不是什么，这就叫做唯物主义。看待历史也要用这样的方式。但是当你规定了标准答案，它“必须”是什么的时候，这就不唯物了。

那那个啥，连麦是哪个平台？我怎么没看着？好好好，hello 盟主你好哦，喂哟。我很喜欢你。好，谢谢。那个就是以前我在喜欢在网上搜一些论文或者看社评，看到一些文章会涉及到在中原建立的政权，过分倾向于强调、美化了，甚至是有些粉饰。当然我没有全盘否定他，但是这样一来，就是有一些真正悲惨的事实就被掩盖了。

我印象比较深的是有几件事情。一个是在2015年的时候，我在微博上看到的，黑龙江很早以前出土的一个古墓，然后里面是应该是金朝的，里面有一个殉葬的姑娘，应该是宋朝被掳过去的公主，死得很惨。当时有一个学者——我很印象很深的是一个老奶奶——她很和颜悦色的，然后用赞美的，甚至是有一些自我陶醉的语气说“这个是民族融合”。还有一次就是我那时候不知道是“齐民通婚”还是什么，我当时是搜的那个关于满汉通婚的那个文章，看到文章有说之前是有过那种隔离，但是越到后面就越来越没有满汉的隔离啦，还把乾隆嫁女儿嫁给孔家的例子，作为了一个例子。还有吹捧辽，说他是“中华正统”，夸赞他“南北两制”，给他加了很多的滤镜。看了这些文章，就跟您刚才说的那个 AI 一样，确实是会影响我们的判断的，但是也会让人隐隐地觉得哪儿不对劲。然后慢慢的随着这个阅历的提升，也会通过自己成长过程当中形成的一些逻辑，再重新去看那个问题。

比如说是以我刚才说的辽为例，尽管有人他强调当时多么“契汉一体”，契丹人管契丹人，汉人管汉人，多么先进多么好，但是甭看这些大道理，从多方面来看，仍然是有很多的问题。首先就是看钱的来源——我这个人是比较俗的——当时那个契丹族说是他不纳税，那么财政从哪里来？那我看了一下，主要是来自燕云十六州的汉地，或者是和那个渤海地区，因为其中渤海它可能体量比较小，所以绝大部分应该是来自于燕云的汉地。所以纳税就是从钱的来源的，这个制度是不平等的。然后就是看从政的机会嘛，他们也是举了一些，比如说有一些韩德让的家族，但是我是觉得这些人的他们的家族，他不能代表真正的汉族，因为他们从一开始就是与契丹的皇权紧密绑定的权贵，所以举这个例子是没有意义的。真正要看的是汉地的、燕云地区的寒门，有没有上升的通道？如果说被堵塞了，那么政治权利其实也是不平等的。最后看社会层面，那有没有民族通婚？有没有自由婚嫁的这个政策大幅度推广？这个也很重要。那也会举，比如说像萧家与韩德让家他们的联姻，那还有什么乾隆家女儿的例子，这个不能算算是民族通婚，因为只能算是贵族通婚，因为他是以政治利益的权贵之间的联姻。所以就是说如果说平民的通婚被阻止的话，那也是意味着是存在种族隔离的。当然这个是我自己总结的逻辑啊，肯定有很多不全面的。

最后我关注你，是因为你有两个观点，解决了我一直以来困扰我的问题。一个是在讲丛林法则的时候，你说正义的那个才是正确的，哪怕失败的英雄也是英雄，这个话我印象一直很深，所以我对你的逻辑就产生了那种天然的、比较很亲切的感觉。或许你讲“清明女装”的观点，我是很容易接受的，因为很早之前我是不明白为什么喜欢汉服的人会反对这个作为民族服饰。我相信很多人，现在有很多人和我一样，非常听不进去别人跟我讲形制。当然他那个“清明女装”丑是一方面，但我那个时候是觉得穿过了就穿过了，为什么不能作为民族服饰？但后来就很容易接受你的逻辑——看到这个事情，你对这个事情的点评才知道，我们是要站在我们祖先的立场上去看待问题，他们是被迫的，而且是以性命为要挟的。所以穿这个东西的，而且穿这个他只能穿一半，是没有资格穿整套的，这个事它是带有侮辱性的。所以就很容易接受为什么要反对“清明女装”来作为民族服饰了。

再夸一夸你、表白一下：你看问题的角度，还有用的方法，你的逻辑都是很让人接受的，所以我们大家听了以后也很愿意被你说服。那你本身也是很有知识，非常一个有趣的人。这也就是为什么你并没有很激进的言辞，那些小黑子找了一年多也没找出什么硬伤，但是他们就是很恨你的原因。好的，谢谢啊，我我我我准备都说完了，稿子念完了。

好的。其实认真地说起来啊，在我们的学界是有标准答案的。尤其是当现在还专门以准官方的身份——有很多人误以为是官方，其实是准官方的身份——确实授了权，但他并不是官方。所谓的什么，那本书叫什么来着？刚才还有人打了那个全称，怪长的，算了。讲五胡——什么“五胡入华”，讲这个东西的那些，包括讲少数民族南迁，使用各种各样的中性词汇去描述历史事件的这种方式是什么？这属于是一个标准答案，这是中国人的“政治正确”。我们一定要弄明白一件事情：政治正确的反面不是政治错误，政治正确的反面叫做事实正确。所以在这种政治正确的定义下面，也就是说我给你一个钢印，我给你一个标准答案，不管你识别出来的证据是否支持这个答案，最终你必须要证明这个答案。这相当于是我们出了一道数学题，给了答案，再让你去填写运算过程。尽管大家发现怎么运算都算不出来这个数，但是不管你怎么算的，最终你必须给出这个数，只有这个数是标准答案。

所以你就会发现，很多学界的人，有学术背书的人，反而特别热衷于给你背标准答案。那这个标准答案当中就包含了所谓的“融合论”。按照今天的分子生物学的溯源来看，这个融合论的说法本身就是有问题的。因为如果真的如他们所说，打过一仗就容易融合，中国人的父系血缘不会那么干净。我们的父系血缘能够那么的单纯，就意味着它不是他们所说的婚娶关系——这个所谓的融合与婚娶背道而驰，实际上是战胜与战败之间的关系。战败了之后，彻头彻尾的战败，除了顶层签约的极个别的人，其他的人实际上都会面临到社会层面的排挤，而且主流的团队一定会跑路，这个才是我们历史上面的基本的规律。就像是当年匈奴，南匈奴南归了之后，实际上真正的南匈奴南下的部分，只是它当中整个匈奴本部当中的一小部分，剩下的大部分战败了之后，他会选择什么？西逃。包括契丹也是这个样子，战败了之后大部就会选择西逃，真正留下来的是其中的极少数。

那当它定义这个东西叫“融合”的时候，那么“何处不可容”？这种言辞、这种标准答案背后是贻患无穷的。毫无疑问，贻患无穷。但是因为它是标准答案，所以你在哪儿都能听见它。但是为什么会觉得不舒服？包括说哪怕没有学习过历史的人，也会意识到这一点，尽管他背的是标准答案，说的话好像还挺好听的，还特别和气，还特别正规，而且背后的背书、学术背书头衔一老长排，一张名片印不下，但是我就是不舒服。这个不舒服来自于哪里？这个不适感就来自于：他没有站在你这一边说话。这个就叫做历史观——站在哪边说话就叫做历史观。他屁股坐在哪个板凳上面，人站在哪个队伍里面，决定了它的是非价值判断。
