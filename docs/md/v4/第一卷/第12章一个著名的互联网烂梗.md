# 第12章  一个著名的互联网烂梗

包括你认为的那种甩肩膀的动作啊，那个蒙古舞当中有一个经典的甩肩膀的动作，这个是汉族的舞蹈艺术家帮他们设计的。所以你说大家在一起好好地唱歌、跳舞，努力地让自己有特殊的风情、有独到的韵味，行不行？可以，因为你过去的条件不好，没有总结出这个东西。甚至我还可以帮你总结你以前的歌谣，就是随便哼唱。我甚至可以帮你设计音律，没有问题。好好地唱，唱得越来越好听，越来越美好，可以。但是唱完了之后翻脸就说“我能歌善舞，你啥也不会”，这是什么呀？升米恩斗米仇是吗？

嗯嗯，啥舞蹈？哦，你们在说那个著名的互联网梗是吧？56个民族，55个能歌善舞。是不是舞蹈诞生在什么样的环境之下啊？还有，你对舞蹈的标准是什么？这个事儿很重要。其实按照什么天分、按照天资来说，大家都差不多。真的是按照天资的话，打那个什么56个民族55个能歌善舞，他是在刻意地打击中国主体民族的文化自信啊。这个呢，是几十年前别人已经用了数十年之久的一个什么，算是互联网梗。你要是问他的话，他肯定说就是开玩笑。但是呢，这种玩笑铺天盖地地推广了之后，就会形成群体潜意识，进而打击到一大批人的信心。但事实真的如此吗？那当然是完全背道而驰的啊。

因为舞蹈这个东西，如果你讲的就是扭动身体、根据节奏来扭动身体就算作舞蹈的话，那么所有人都会能歌善舞。你肯唱、肯哼、肯扭就算能歌善舞的情况下，全世界每一个人都是能歌善舞的。如果说你对舞蹈、对音乐有更高的要求，那就完全不同了哈。当你有要求的时候，什么样的情况才算是能歌善舞？需不需要有音律的识别能力啊？需不需要你有音乐相关的学说？要不要这些东西？你要的话，那汉民族传统的乐师曾经是最重要的律法制定者之一，他们是统一度量衡的重要技术门类。自古以来，乐师就是我们整个社会管理制度当中不可或缺的一环，是超高等级的国家顶流技术人才。所以在音乐方面，我们自古以来就极度重视，甚至把它作为国家礼制当中不可或缺的一环。然后你说：“哎呀，汉民族不善歌”，那什么算？没事儿。因为和别人离得太远，所以说话必须扯着嗓子嚎，就叫做能歌是吗？你按这种标准的话，哪个地方空旷，哪个地方的人就能歌；人口密度高、生活条件好，说话的时候可以凑近了说，你就不能歌。这算什么呢？这是指标吗？

然后包括舞蹈也是这个样子的。如果说只是围着火堆能够扭动一下就算善舞，那么所有人都有这个能力，全部都有。但是当你要说舞蹈是需要有仪式感的，是需要有肢体上特殊的协调、特殊的节奏，要能够有意境、要能够表达思想，那请问我们自古以来所有的王侯将相，他们在自身的理智范围之内，甚至每一餐饭有几个人可以给他跳舞都已经列入标准指标了，我们对舞蹈重不重视？重视到了这样的程度，算不算善舞啊？都不算是吧？好的。那就是说今天大家哪一个群体更善于跳舞呢？很好，你所认为的那些少数民族舞蹈当中，大量的都由汉族的舞蹈艺术家参与创作。包括你认为的那种甩肩膀的动作啊，那个蒙古舞当中有一个经典的甩肩膀的动作，这个是汉族的舞蹈艺术家帮他们设计的。所以你说大家在一起好好地唱歌、跳舞，努力地让自己有特殊的风情、有独到的韵味，行不行？可以，因为你过去的条件不好，没有总结出这个东西。甚至我还可以帮你总结你以前的歌谣，就是随便哼唱。我甚至可以帮你设计音律，没有问题。好好地唱，唱得越来越好听，越来越美好，可以。但是唱完了之后翻脸就说“我能歌善舞，你啥也不会”，这是什么呀？升米恩斗米仇是吗？

哎，好，欢迎大家来到直播间啊。嗯，啥玩意儿？嗯，是汉族人打工人的自嘲和羡慕少数民族的生活罢了。你怎么知道的呢？你怎么知道的呢？你以为少数民族不工作吗？少数民族也是共和国公民，他们也工作，他们也纳税，他们也上学，所以你们在这里说的都是什么意思呢？哎呀，我就是开个玩笑，就是吐槽一下刚才说的是什么。当这种互联网梗铺天盖地地传播，不断地强调什么血脉天赋啊、血缘优势啊，这是干什么？这是在强调差异性，而且是凭空创造差异性来进行强调。然后指出来之后说：“哎呀，就是玩个梗，就是开个玩笑。”如果你的这个玩笑伤害到了中国90%以上的人，你这个玩笑就不应该开。找这种借口有什么价值呢？嗯，你身边的同事、同学和你没啥区别。本来嘛，你能够生活在一起的区别都不大啊。特意在互联网上把人家的那些什么其实日常生活当中根本用不上的东西，硬给搬出来，然后说：“哎呀，血脉天赋，血脉压制”。你到互联网上一搜，就有“血脉压制”这一个词，你能搜出来多少？视频铺天盖地呢。这是吐槽吗？这是成系统的、成组织的互联网攻击行为等等。

穿汉服被满人说成贩毒，呵呵啊。在全世界范围内没有汉族这个组织，就这样啊。其他的这毒那毒的组织是存在的，但是汉毒组织是不存在的啊，从来没有过。只有在那些分裂分子的嘴里头才存在这样的群体，现实生活当中没有，理论群体当中也没有，就不存在。为啥？你想一想，一个国家里面啊，理论上90%，实际上大概率是超过了93%，甚至超过了95%的人群，它不是分裂。什么样的是分裂？分裂是指极少数的反动群体与其他的人群背道而驰，这才叫做分裂。如果说95%的人都认为自己应该有另一条道路，这叫做革命，这就叫做正义，多数人的正义，这就叫做人民的力量啊。哼哼，各色奇怪人种都已经出现了，对吧？啊，你们来了就对了。嗯，那个蛇灵哈，你再多说几句话的话就要暴露了，知道吧？蛇灵你再说两句话就要暴露了，到时候你星号又被我拉黑了啊，就很尴尬。省一省，好吧，都差不多要被看出来了。嗯，他不是每天都暴露吗？呵呵呵呵呵呵呵。关键是，嗯，关键是蛇灵，每天吧，坚持不懈，他甚至呢，在灯牌上面等级都不低啊，很认真地在那里刷贡献值，相当努力哈。每天蹲着作为黑粉、作为反对者，能够这样地持之以恒，还是很了不起的啊。不容易不容易，天天来。
