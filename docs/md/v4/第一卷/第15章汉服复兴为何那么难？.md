# 第15章  汉服复兴为何那么难？ #汉服 #汉服复兴 #汉服小姐姐 #汉服之美在华夏

当他产生了这种情感链接的时候，你跟他说这个东西是假的，这个东西他骗了你，他会反抗，因为你让他痛苦了啊。人都是这样的啊，你戳穿了某一种痛点，他就会痛苦，他痛苦他就会跟你对峙。所以现在就是你可以直接冲上去说“你这个穿错了”，你不能啊。我坦荡地跟大家说，你不能跟任何人去直接说这句话，除了那些专业的搞传媒的人，他们是故意为之，他们是收钱在这里做黑单。但是对于穿着衣裳的普通人，你不能说这件事情。他们是受市场环境，甚至是服装杂志的影响，去穿着的这件衣裳。你说他错了，你说他有伤害性，最后受到伤害的是这个消费者本人，而他只是想过好日子的一个人，凭什么被你这样攻击呢？所以你只要指出来，他就会生气；只要指出来，他就会愤怒，最后就说你是一个汉服警察。

哎，好，欢迎大家来到直播间啊。那个叫毒瘤的小伙伴啊，你有看我视频的更新吗？啊，我专门讲了这个视频，我专门做视频讲了的。唉，那也不是某一位格格一人之功，是群体性的创作啊，是群体性的功劳。甚至在某一些地方的这种准官方的汉服走秀当中，给明制的皇后嫡衣大礼服做了一套厂字领啊。所以怎么说呢，就是目前有一群人，对于厂字领和蜈蚣扣有着极致的执念，一定要把这个符号，一定要把这个不兼容的、不受喜爱的、极其有敏感度的符号塞进到汉服系统来。而且这一两年他们表现得极为的急切，重金出击，买通了相当大一批的汉服博主、汉服 up 主，帮助他们去穿着厂字领蜈蚣扣的清民女装，进行汉服推广啊。

在这个领域当中，那些穿了清民女装就坦然承认这是清民女装的人，已经算自己人了啊。大部分呢，其实就按照甲方的要求，直接就在那里开始蒙事儿了啊。所以在这个事上头，很明显对方是成组织的刻意推动，而且借用了很多的力量。这个很多的力量呢，除了大家能看到的这些互联网营销的系列之外，还包含了这种服装设计工作室、服装厂家，它当中是包含了这些很正经的、看上去是自由市场手段的东西的。哪怕那个衣服设计成蜈蚣扣之后，销量大打折扣，他也要大规模地生产，然后呢，打很多很多的折扣做清仓处理，让你用极便宜的价格去买到它啊。这是什么？这种行为啊，如果跨国，它就叫做倾销。用超低廉的价格，贴补式地将他的商品大规模地塞进市场里面，这是一种倾销行为。

所以厂字领配蜈蚣扣的这些产品，它不光是在传媒上面发力，它还在市场上面进行了大规模的倾销。大家到那些商场、卖场的清仓区去看，都有蜈蚣扣。你们只管去啊，我已经逛过很多这样的地方了。看出来我穷了吗？老是看特价区啊。这些特价区里面充斥着蜈蚣扣、厂字领。那这些衣服都是谁去购买？社会基层的普通人为了省吃俭用，就会在这些特价的、甩卖的啊，甩卖的卖场里面去购买这些价格低廉、面料和品质很好的衣服，他们不会去介意这个服装上面的文化符号，于是就用这样的方式倾销出去了。

然后同时啊，同一时间，他们呢，还侵蚀了很多的出版行业。大家很多时候都非常信赖正规出版物，因为正规出版物是有审核的。但是呢，在这个服装的定义领域当中，没有审核，任何的编辑、任何出版社都不会审核这个东西的。不会有人说你塞了清的元素，说它是传统服装，他就说你这是错的。因为在历史上确实有两三百年的时间，尽管大家不喜欢，尽管是被迫的，可确实穿了。于是呢，啊，研究服装发展史的人、专业的学者们一定会认为它就是我们的历史进程当中的一个段落。所以呢，他们可以进军正规出版物这个系列当中。

好，嗯，当用这样的方式去进行大规模的推广的时候，普通人对这个东西就会产生，逐渐地产生熟悉感。而这种熟悉感在下一个步骤产生的就是亲切感。亲切感的下一个步骤是什么？觉得他就是自己的一部分。当他产生了这种情感链接的时候，你跟他说这个东西是假的，这个东西他骗了你，他会反抗，因为你让他痛苦了啊。人都是这样的啊，你戳穿了某一种痛点，他就会痛苦，他痛苦他就会跟你对峙。所以现在就是你可以直接冲上去说“你这个穿错了”，你不能啊。我坦荡地跟大家说，你不能跟任何人去直接说这句话，除了那些专业的搞传媒的人，他们是故意为之，他们是收钱在这里做黑单。但是对于穿着衣裳的普通人，你不能说这件事情。他们是受市场环境，甚至是服装杂志的影响，去穿着的这件衣裳。你说他错了，你说他有伤害性，最后受到伤害的是这个消费者本人，而他只是想过好日子的一个人，凭什么被你这样攻击呢？所以你只要指出来，他就会生气；只要指出来，他就会愤怒，最后就说你是一个汉服警察。所以你指出他的错误，实际上败坏的是自己的名声，以至于这件事情其实是不能做的。

感觉有的西装领像汉服领。前两天我在那个图片上面啊，看那个唐圆领，然后他们看到就说：“哎，你这个领子好奇怪。”为什么呢？因为他知道我在看汉服，然后看了唐圆领的翻领之后，就觉得这汉服是不是不正宗。为什么？因为他不了解，于是就觉得它有翻领之后就不正宗。所以西服当中有一部分的领子看上去像汉服很正常，因为领子是什么？领子是服装的一个装饰件啊。汉服当中不光是有翻领、有圆领、有立领，那这些领口都存在，因为领子是服装的一个装饰性的配件，它不涉及到服装的结构。而一件衣服到底是什么，取决于它的结构本体啊。
