# 第55章  真实的唐三藏有多逗，道心破碎念头不通 #唐三藏 #佛教 #孙悟空 #西游记

在那个时代，玄奘所做的事情叫做“倒行逆施”——他在把我们已经掌控得差不多的释经权拱手让给他人。所以在这种情况下，皇帝不可能支持他。然后他就那样“蹭蹭蹭”自己跑了，开始学习，跟别人讨论。结果他就发现，他所向往的那些原教旨的原始版、初代版、真实的经典，其实跟玩儿似的，远不如我们这边后来二创、三创、四创、五创的东西。这就是出门留个学，然后把人家场子给砸了；这就是一出国就爱国。玄奘法师身上真是这样的：一出国就爱国。

所以皇帝要求他清晰地描述：“你经历了哪些国？那里都有什么？地形如何？有多少人？当地的风貌如何？”这是干啥呢？就是把他当探子用啊。

什么？直播间里怎么吵上了？看你的灯牌也有15级了，不至于从来没来过吧？我们直播间里不是日常吵架的吗？而且公平吵架通常跟我讲的东西没什么关系，这也是常态。以至于到后来我基本上都不怎么看了。为什么呢？因为我也不知道你们在吵什么啊，日常就这样，不知道在吵什么。

听说玄奘当初把天竺干翻了？是的啊，是的啊。怎么跑题到八宝粥了？没跑题啊，八宝粥才是正题呀！别的东西才是跑题。你们要分得清主次：八宝粥才是正题，因为我这里是带货直播间，带货直播间的货才是主题，其他的都是附加的，都是赠品。

玄奘法师的成就还是很独特的，因为他是我们隔壁巴拉特（印度）那里最重要、甚至可以说是唯一可靠的成文信史的口述撰写人。玄奘是一个挺了不起的人物。玄奘法师当年西行这件事情本身是没有受到官方支持的。咱们得先确认一件事：我们在《西游记》里看到玄奘西行受到君王支持，甚至有送行，电视剧里还特别给他送了行。但现实生活当中，玄奘法师的西行没有真正受到支持。为啥呀？那简直见了鬼好吗？你支持他干啥呀？换你你都不支持他。

要知道当时中国是什么境况？我们当时已经不可一世，到觉得自己是天下至尊的程度，看别处都跟看儿戏一样。结果有一个号称得道高僧、信众还不少的高知人才说：“我们这儿的经书不好，我得出去求经，我得去学习别人的东西。”你知道这个有多么打击文化自信吗？换你当皇帝，你也不许他出去。出去取经取真经，认为自己家的书不好，得去跟别人家学一学，而且那个隔壁的别人家文化水平、经济水平还都不咋地。换谁谁也不让他去啊。所以他其实并没有真正受到支持。但是他为什么要出去呢？他有点原教旨主义倾向。

释经权很重要啊！不管你是什么学问，最终解释权在谁手上，这事非常重要。而以李唐王朝当时的不可一世，你觉得皇帝会允许释经权放到老远地方的一个境外小邦国里去吗？绝无可能。我不管你是从什么时候、从哪里传来的经文，既然在我家里，就由我来掌握它的最终解释权。这就叫释经权。我们当时掌握释经权已经很久了——从汉代传进来开始到唐，中间历经漫长岁月，一直有人不断进行解释和再创作，甚至有人托古自己编经书。所以玄奘之所以觉得我们这里有很多伪经，那是因为真的有，因为我们已经掌握了释经权之后，自己创作，自己写。他相当于什么？就相当于洪秀全说他是上帝的二儿子那种操作——看上去是西方的一神教，但已经经过二创、三创、四创，进行了本土化解释，已经四通八达，并且直接在我们这个地方登记造册。这就是我决定了它是什么就是什么。所以在这种情况下，玄奘做的事情就是完全的倒行逆施。

相信我，在那个时代，玄奘所做的事情叫做倒行逆施。他在把我们已经掌控得差不多的释经权拱手让给他人。所以在这种时候，皇帝不可能支持他。然后他就那样“蹭蹭蹭”自己跑了。他就这样自己跑了。结果跑出去之后，发现一出国就爱国。玄奘法师身上真是这样的：一出国就爱国。和今天那种瞎吹牛的“润人”不一样。玄奘当年经历了这一波三折之后，真的做到了一出国就爱国。为什么这样？因为他当年在国内、在长安城的时候，老是觉得我们这儿的东西都是自己创作的——你看这一部经，说是外头传来的，其实和我们以前的经典没啥区别，这玩意儿完全是自己编的。不行，我得找那个真正的真经，以前的、真正的原始版本、真正的佛传真经。我要找着这个东西去。然后他就去到发源地了。去了之后去留学，那个叫什么？那烂陀寺是吧？他去留学去了。

到了那儿之后就发现问题了。佛学讲究辩经，就是辩论、讲道理——一个讲道理的教派还是挺好的，咱们聊一聊，把这个理理顺，把事情讲清楚，谁说得通谁对，基本上是这个原理，到哪都是这样通行的。然后他就去了。去留学之后，先是过语言关——谁出门留学都是这样。过了语言关，时间不长就掌握了这门语言，流利了。之后参与到了当地的经文梳理等基本工作中去学习，当然要不断整理档案资料。他就开始学习，然后跟别人讨论。结果他就发现，那边的和尚没有哪一个能吵得过他的。在理论知识和逻辑的完整性上面，玄奘高过对方18层楼，大家就不在一个世界里面。对方在那里辩经时是一种言辞上的死循环，但玄奘这里内生逻辑非常完善。结果他就发现，他所向往的那些原教旨的原始版、初代版、真实的经典，跟玩儿似的，远不如我们这边后来二创、三创、四创、五创的东西。这就是出门留个学，然后把人家场子给砸了；这就是一出国就爱国。

后来他就回来了。回来之后又翻译了几本经书，但这个成就其实一般。他后来非常重要的一个成就就是《大唐西域记》。这本书的由来是：反正你已经出去了，周游列国转了那么大一圈，既然你回来了，就应该为这个国家做点贡献吧——归国留学生，总得为国家建设做点什么吧。就是这样一种诉求。所以皇帝就要求他清晰地描述：你经历了哪些国？那里都有什么？地形如何？有多少人？当地的风貌如何？这是干啥呢？就是把他当探子用，直接把他当探子用了，让他画地图就这么一件事。玄奘法师心里有数，他知道皇帝要干啥，所以不是很乐意。但是，被迫、被迫啊，被逼被压迫着，最后他就自己口述，由别人来撰写，讲了一下他那边到底有多少国，哪个国啥样，他经历了哪，往西边走了多少碰见一个啥，又往南边走了多少碰见了个啥。这就画了一套地图。嗯，然后写外国人都“歪瓜裂枣”？他用的词叫“鄙陋”，玄奘法师对于他途经的西域诸国里面的人的面貌描述叫做“鄙陋”。还是开地图有用。他沿途肯定是留下了一定的笔迹的——中国人都有这样的习惯，不管你出门去哪儿，你途经经历了什么东西，总要留下一些记录、留下一些笔记。所以这些玩意儿到了还是起到了不少的作用。

嗯，估计唐王要的重点和他想写的重点不同。大家的目的是截然不同的。你觉得皇帝是真的好奇别人的那些寺庙里面是怎么讲故事的吗？不可能啊。皇帝好奇的肯定是土地的范围、物产的丰富程度，还有地形地貌，包括什么军械、装备，这些东西才是皇帝感兴趣的。呃，佛经写在树叶上？你是说贝叶经吗？这些东西上面残存了一部分，听说我没有专门去看这个东西，但听说残存了一部分。但即便有这些残存的，其实影响也不大，因为这种计数方式本身就有问题，这种记录方式就不稳定。而且，就是我们隔壁巴拉特（印度）那个地方，它有一个很鲜明的特点：不管做什么事情都不精准。中国人是在意精准的，数数数到一就是一，二就是二，我们在乎这个事情。但是我们隔壁巴拉特不在乎这个事儿，他们不光今天的人不在乎，他们以前的人也不在乎。
