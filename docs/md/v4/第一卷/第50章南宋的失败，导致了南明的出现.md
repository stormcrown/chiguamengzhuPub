# 第50章  南宋的失败，导致了南明的出现 #西方 #南明 #南宋 #历史

蒙元统治的时间并不长，他在文化上的破坏有，但是也有限。可是他留下了一个很重要的烂摊子：有相当一部分中国人，尤其是以文人士大夫这个群体——那些日子过得还不错的群体为典范——他们开始觉得谁当主子都行。蒙元留下的最大祸根，就是在中国的文人群体当中留下了“奴婢心理”。这种“谁来都可以当我主子”的奴婢心理，对于南明在遭遇外敌时内部不停地撕裂，成了一个很重要的前置条件。因为内部的朝臣对真正的危机、真正的危险没有清晰的判断，他们觉得来了无非就是再来一个蒙古大汗。因为经历过一次彻头彻尾的失败，所以觉得再失败一次没关系，因为底线已经被跌破了。所以才会出现南明那样垃圾一样的操作方式。

好，欢迎大家来到我直播间。请把《光从中华来》放得离《南明史》远一点。好，在这儿啊，好，我知道了，可以了啊，满足你的心愿。放到一起——这两件事是一回事。南明史的失败使得我们失去话语权；失去话语权之后，我们就失去了自身的科技叙事权；失去了科技叙事权之后，这个世界的科技与文明的源头，就变成了那个之前还在树上住的地方。这两件事情之间是有内因关联的。为什么西方的历史大面积造假，我们却不可以说？一说就是“民科”，一说就是“伪科学”，一说就是“九漏鱼”。明摆着他们那个地方编造的故事漏洞百出、逻辑不自洽——在一片荒原上面拥有那些匪夷所思的伟大建筑，而在真正拥有耕地的地区却没有这些东西。这些混乱的、造假的历史，你只要一戳破它，它就开始发疯、发飙：“唉，你个民科！你个九漏鱼！你个思密达入脑的世界中心主义者！”为什么会发生这种现象？因为南明的风雨飘摇，使得我们失去了自身的话语权。所以这两样东西之间还真有关系，就放到一起，很应该放到一起来。

我把《南明史》给你们弹一下啊。知道自己怎么栽的跟头，知道自己为什么后来倒霉了300年，知道为什么推翻满清之后，哪怕是屎一样的民国都开始走上坡路——要从哪开始入手呢？从《南明史》开始入手。好，欢迎大家来到直播间。但是一定要提出来的一样东西：《南明史》这个书不是很好看啊，做好心理预备。《南明史》并没有多好看，事实上它非常难看。看了之后着急上火的，满脸都是包。所以呢，慢慢看，不要着急。一次性地从头看到尾的话，我怕你会疯球了。所以要慢慢看，蓄力完成之后看两篇，再蓄力完成一次再看两篇，保证自己的心理健康依然很重要啊。

所以刚才那个鬼人在那里说什么“你把《光从中华来》和《南明史》放远一点”？能是为了什么呀？不就是想说《光从中华来》是一个伪史吗？西方历史造假不许说是吗？那西方历史为什么居然有本事造那么大的假？怎么就有本事在中国倒退的这些年里头飞黄腾达？之前的底子什么都没有的情况下，怎么就在中国直线倒退的时候他直线腾飞？这个中间的转折点、关联点在哪里？就在南明。南明才是需要为这些事情负责任的。我从来不要求满清负这个责任，为什么？因为他是南明失败的一个结果。你把自己作散黄了，被一个痛恨你的、想要摧毁你的群体摧毁了。最重要的原因是什么？是内因而非外因啊。为啥换成南宋的结果都要开心一整天？哈哈哈哈。但是南宋最后的失败，也是促成了满清入关很重要的一个原因。

蒙元统治的时间并不长，他在文化上的破坏有，但是也有限。可是他留下了一个很重要的烂摊子：相当一部分中国人，尤其是以文人士大夫这个群体——日子过得还不错的群体为典范——他们开始觉得谁当主子都行。蒙元留下的最大祸根，就是在中国的文人群体当中留下了“奴婢心理”。而这种奴婢心理，对于南明在遭遇外敌时内部不停地撕裂，成了一个很重要的前置条件。因为内部的朝臣对真正的危机、真正的危险没有清晰的判断，他们觉得来了无非就是再来一个蒙古大汗。因为经历过一次彻头彻尾的失败，所以觉得再失败一次没关系，因为底线已经被跌破了。所以才会出现南明那样垃圾一样的操作方式。

所以认真的说起来，南宋那个局面相较于南明是不是好很多？但是南宋的失败导致了南明的出现，这两样东西之间是有关联的。南宋失败之后，蒙古人南下，他们的统治管理方法非常粗糙，粗糙到了一个约等于没管的程度：各种各样的分包、承包商，找代理人代为收税、代为管理、代为制约。这种代理的管理方式既残暴又宽松，两种东西同时存在，就造成了大量的中间实力阶层。因为最顶层的人对这个国家毫无责任感，他拿这里当成他的势力范围，没有责任感，也不想让这里有序，只需要横征暴敛——你只要把东西和账都给我交齐了就行。这样的操作方式会培养一个权力极大、利润也极高的中间实力阶层。而这个中间实力阶层获利过甚，以至于他们老想去当这样的中间实力者。这个实力者是什么样的人呢？就有点像阿三那边的中等种姓。你说他的地位高不高？不高。上面的王公蔑视他，不许他加盟；但是他下面有一大堆他可以盘剥的对象。于是只要上层没有踩他的那一刻，他就是土皇帝。

在蒙元的四等人制（实际细分起来超过四等）基础上，就形成了中间阶层，已经出现了阿三的心态：一边对上只管去跪舔——这个跪舔永远都是短暂的，因为上层数量少；而对下面呢，他想怎么收拾就怎么收拾，没人会去在意。这样的人群过得是非常舒适的。所以蒙元时期就形成了这样一个稳定的实力阶层。而这个阶层在明代、在朱明王朝特别吃亏，因为朱元璋最恨的就是这些人。他是农民的子弟，当年的农民底层下层人，受到这些中间实力阶层的盘剥，是一个极致的——不光是盘剥过剩，而且毫无道理。要知道其他的时代，哪怕是遇到了横征暴敛，就算是遇上了昏君横征暴敛，他还编也要给你编个借口、编个理由出来——尽管粗糙，但总要找个理由才出来敛财，总要装出有规矩的样子。但是到了蒙元时期，由于这种分层的管理，使得他把所有的道德、伦理规则、天道伦常、人心全都弃之不用了，不在乎，没有人在意这件事情，也没有人能够用这些东西来制约它。所以这个时期底层所受到的盘剥与其他的时代完全不一样：不光是在经济上面盘剥过剩，而且是那种一点道理都不想给你讲的程度，就是一群匪徒一样肆无忌惮的、这种性质的剥削。用这样的方式来进行管理之后，底层的感知是截然不同的。所以朱元璋有明一代，最恨的就是这群人。为什么朱元璋也好、朱棣也好，都是暴君？朱元璋和朱棣在名声上面都不好，都算作暴君。为什么？因为他们最恨的、打击最甚的，就是这些写故事、写书、写史书的人。

好，欢迎大家来到直播间。然后刚才那个说什么“南明史不要和《光从中华来》放在一起来”？我们开始带这个《光从中华来》啊。这套书非常具有节点意义。你说这个书已经完美的叙事了吗？我个人认为还没到那个程度，不完美。但是呢，不需要它完美——完美这件事情对于现阶段的质疑、批判来说不重要。因为你不需要拿出所有的证据，把别人家的历史理得清清楚楚才能够讲话。没有那个义务挖掘他们的历史本源到底什么样子，那是他们自己的责任。而中国人由于受到了西方中心主义、西方缘起主义的影响，影响了我们自身的文化崛起、文化复兴，所以我们现在迫切需要的是推翻他的西方绝对权威。那么怎么开始呢？就是去质疑他的历史造假。质疑对方的历史造假，你是不需要知道对方真实历史是什么样子的，为什么？因为编造谎言这件事情，每一个谎言都需要100个谎来圆，他的逻辑上面先天就有缺陷。一个逻辑上有缺陷的东西，你把他的逻辑戳破了就行了。至于说“那它真实的历史啥样？难不成他没历史了吗？”——凡是活到今天的人都有过去，但是他的过去有没有记载，这不归我管，归他自己管；他记载的东西是真是假，也不归我管，也是他自己管。如果说他编造的故事不影响我，那其实他是不是编故事我都不稀得管。例如说，其实现在非洲那边有一些小国也在给自己编历史，我不知道有多少人知道这事。非洲那里有很多小国现在正在编造历史，但是为什么我们无论是民科还是官科，没有人去质疑这件事情？为什么？因为当一个国家的人群想要树立自己的精神、想要让自己团结起来的时候，它需要有一个共同的叙事。既然他需要有一个共同的叙事，那他给自己编一个传说、编一个伟大的祖先，其实就是很合情合理的一个操作。那别人的这个东西——用来凝聚他自己共同意志的东西——我们干涉这玩意儿干嘛呢？多余，没有那个必要。你们想怎么玩怎么玩，我们可以在旁边看着，就静静地观察就好了。

但是为什么对于西方的历史，我们不可以这样做？因为西方的历史严重干扰到了中国历史发展进程，就因为这个原因。所以对于西方的假历史，是需要把它撕开画皮的，你得让大众都知道他们编了太多故事，这是一个刚需。你必须知道对方编造了哪些故事、哪个东西是真的、哪个东西是假的。所以这个《光从中华来》这一套书在这个地方就很有用，它是对西方历史的一个系统性的质疑。这个书一共是上下两册，是一个美术史研究学者黄河清老师的作品。研究美术史也能够发现对方的历史造假，为什么连画画的画作他们也会造假？是的，他们给人画肖像画，但是那个肖像画里面配置的各种东西、物件、人物、人的装束，有造假的内容。那发现了可不可以指出来？当然可以。所以他说一幅画，500年前画的，结果你一考证里面的东西——无论是染料还是服装的织纹——一考证之后发现200年前才画出来。请问这个造假有没有揭露的价值？当然有。因为不同的绘画水平、绘画作品当中的一些物品的特征，其实可以体现特定时代的时代风貌，也能够体现那个时代的生产力发展水平。如果他在这些东西当中都造假，你会以为西方人500年前就发明了天文学的科研仪器，但他没有。所以当这些东西都铺天盖地出现的时候，如何质疑对方的历史就变得非常重要。

要知道我们现在啊，如果说你不是特意在乎这个领域，只是普普通通地平时看看电视、看看视频，你会刷到相应的内容。例如说青铜器的锡来说，青铜器是西方人研究出来然后传播给我们的，我们是学生。你说当学生行不行？当学生没有问题啊，可以当学生，但是它是真的吗？它是真的吗？西方人是怎么在自己家里面没有充分的燃料、没有充分的冶炼场所、也没有充分的锡矿的情况下，研发出来了青铜器的？这些年来，西方在各种各样的地方去建设、去挖掘新的考古遗迹、去发现各色各样的青铜器，他不停地去发现崭崭新的这种古典遗迹。这个地儿它不全都是在西欧，你记得：西方文明体系、西方文化缘起这个东西，它不一定是在西欧，甚至不一定是在欧洲。为什么？因为所谓的西方缘起，它指的是“非中国缘起”。你一定会发现一件事，我给你总结出来你就知道了：西方人在宣传世界文明遗址和世界人类的重要文化源头的时候，你会意识到他对中国人和对非中国地区是两套班底、两个标准、两套完全不同的标准。在中国以外的地方，只要不管是发现的还是创作的，只要有人类聚居痕迹，他就说这是一个文明。你可以去看，就是那个古印度文明的发源地——很小的一个村落，一个莫名其妙的井。因为那个井的建设方法显然不对劲，那就不是古典遗迹当中应该存在的水井的建设方法，那是凭空出来的一个东西。但是不管它真假，就是这样的遗迹，丁点儿大的一个人类可能活动过的一个地方，它就算是古文明的印记了。有文字吗？没有。文字有记载吗？没有记载。印证了什么先进科学技术吗？没有。这些东西统统都没有——古典文明遗迹、重要的古人类文明遗址，它都是这个样子的。它对中国以外的地区不要求你有文字，不要求你有先进性。但是对中国这里的所有的考古的东西，不管你这里的城邦已经建设得多么完善——生活区、祭祀区，包括王权管理的地区，所有的东西都已经分门别类，显然内部分工已经非常细化了——这样的城池，他说你发现文字了吗？文字上面写了是那个字吗？不是啊，那就不是。所以你一定会意识到：所谓的什么西方文化缘起、所谓这些人类文明的遗址，它不一定是建设在欧洲的，只要并非中国，它的标准就一样。只要这个地儿不在中国，它就可以是伟大的人类文明遗址。

好，欢迎大家来到直播间。所以有自己的话语权太重要了。是的。所以为什么现在我们在很多领域上面有问题？这个问题出自于哪里？很多的时候这个问题出自于话语权。就像我们今天开篇的时候，就有人冲上来说“你不要在那里谈中医”，为什么？很明显的，他是他不认为中医是一个可谈及的对象，那是一个莫可说的东西、莫须有的东西，你不能够说他是医学，他必须得是什么什么样什么样的。他在心里有标准的刻板印象，你夸他一毛钱——哪怕不从医学角度——你夸他一毛钱，他就心里头不爽。为什么这样子？因为定义权不在中国人手里头啊。就是因为定义权不在中国人手里头。
