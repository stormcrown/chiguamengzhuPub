# 第61章  儒学自身的短板，善变 #儒家 #儒学 #华夷之辩 #华夏

当你只是要求定制了一套“仁者爱人”的目标，具体的操作手法可以根据时代的不同而改变，而且之前也确实改变过很多次。那请问，到了元朝的时候来了一波新的、全新的管理的东西——就是“不管”——请问儒生接不接纳？  
之前的儒学要求的是“华夷之辨”，要区分诸夏与蛮夷，要有界限，区分你我，按照远近亲疏的方式来界定彼此的关系。这是原本的儒学的规矩。但是，因为它的目标本身是“仁者爱人”，而规则可以细则性地调整，所以到了元朝的时候，这些儒生们就开始转向了。他们为自己的理论打上了补丁，什么“华夷之变”，它就变成了“夷夏之辨”啊——这句话不是这个时候出来的，但是这个时候出现了这个苗头，就是“华入夷则夷，夷入夏则夏”。这种可以任意变换身份、想去哪就去哪、谁来谁是中国人、来了中国这里你就是中国的主子——这种思路就在元朝这样的高压治理之下自然而然地萌生了。  
这件事情，儒学有没有责任？有责任，有责任啊。为什么有责任？如果它是一个内部有刚性条款的一项秩序，别人就改不了这些东西。  
嗯，呃，讲一讲儒家的缺点嘛。专门针对性地找缺点是吗？啊，专门要针对性地来找缺点的话，那儒家就是过于开放式。儒家的这套学说呢，过于姿态柔软，过于开放式代码，使得它事实上确实过于容易吸纳其他系统的知识——这个是他身上一个比较典型的一个特征啊。  
啊，啥？儒家武力值不行？不行的话，你用你自身现行的体力去跟两千年前的孔夫子掐一架，然后你再说他的体力不行啊。儒家不存在武力值的问题。真正的问题就在于他的这一套……我们说一个学说啊，任何一个学说，学说是什么呢？你把它打比方成一个人的话，一个人是有骨架的，骨架外头附着着肌肉，肌肉由韧带关联在骨架上面，然后外面再用皮肤来包裹——这个就是一个人体，就是一套学说。你要让它完整，就得是有骨架、有肌肉、有皮肤。那儒学在这个事上面完整吗？它完整。它既有骨架。骨架是什么？就是对这个社会制度、社会秩序的极限追求，所谓的人是什么？仁者爱人。你怎么去爱一个人？“我爱你啊”“我爱你啊”就爱了吗？不是啊。儒学不承认口上花花算是爱。  
当它的骨架是这样的东西的时候，那么它如何去填充它的肉——它的肌肉？就是在不断的一件又一件的事情当中，去说这个事情“仁或者不仁”，那件事情“仁或者不仁”，“有利或者无力”。所以你看啊，吕秀才动不动就说“子曾经曰过”，这是什么呢？这就是指案例。通过不同的事例来去论证什么是仁，什么是不仁，我们应该怎样做，不应该怎样做——这个东西就叫做肉。  
而皮是什么呢？啊，皮呢，在后世被称之为“道貌岸然”啊，就是那些风度，包括了像“君子不欺暗室”“君子慎独”这些外在的表现形式，也就是每一个儒学的爱好者、儒学的追随者、儒生身上所表现出来的这些公正、仁爱、宽厚的姿态，这些东西就是儒学的皮肤。  
所以作为一套学说，它是完整的，有骨、有肉、有皮。但是你要说它的缺陷，它的缺陷就是它的那个骨骼不够固化啊。它的缺陷就是骨骼不够固化，因为它要求的东西是“仁”。仁是一种秩序，而秩序是什么？什么样的秩序才是秩序？在不同的时代里面，秩序必然体现为不同的姿态。这个就是经济基础决定上层建筑。什么样的秩序能够让当时的人活得好？这取决于当时的生产力发展水平和平均教育水平。  
你在先秦时期，在商周时期，推行强人身依附关系的这种分包制的管理，也就是分封建制的方式，这个是合理的，是对的，是正确的，是有扩张力的，是有保护力的，它就叫做秩序，也叫做“礼”。那商代的礼和周代的礼就不一样，因为商礼更加侧重于祖先的祭祀，也就是它神权的滋味更加浓郁，所以在商礼当中，关于祭祀的甚至血腥祭祀的东西占比要比周礼要更高——这个是不同的时代，它有不同的需求。就像到孔夫子说他要复周礼，他为什么不复商礼呢？要知道孔夫子其实是商人的贵族后裔啊，但是他推崇周礼。为什么？因为周礼对于当时的经济生产的发展状况来说，较于商礼更加进步，也更加有序，更加文明，所以孔子才会推崇周礼。  
孔夫子的孔儒这一套东西，解决的并不是什么所谓的奴隶制的问题，而是秩序的问题。他在揣摩：是什么样的秩序才是“仁”？“仁”就是能够对其他人是更好的一种选项啊。那这个呢，就是他在“仁”上面的诉求，对于仁爱的诉求。  
所以我们刚才说了，他在秩序的建设、在自己的理论框架上面是完整的，可是它的缺陷也在这儿了——因为他订立的是一种目标取向，而不是一个细则规范啊。这两样东西之间有区别啊。我说“我要求的这个社会往好里走，有规矩”——句号——这个是我的目标。那这个规矩是不是应该因时而变？所以呢，儒学身上的好处就是它可以因时而变。  
你看孔夫子本人所推崇的实际是什么？是周礼的管理方式。周礼里面必然包含了一部分奴隶管理方式的分封制管理方法。但是到了真正“独尊儒术”的时候是什么？治理方式已经进入到了郡县制管理方法了。郡县制当中的奴隶制占比会更低。在奴隶制占比极低的郡县制管理方式之下开始独尊儒术，为什么这样？因为儒学本身是解决“仁爱”的问题，不是规定你必须要使用周礼，它没有这种细则上面的强调。包括说孔夫子所强调的“要遵周礼”“我要恢复周礼”，他强调的是把那个周礼搬出来大家用吗？不是。他给他的所有弟子、给他的三千弟子所传授的信息，是为什么要制定这套东西。他在解读周礼的时候，讲的是周朝为什么要制定这套周礼，它有什么用处。所以他实际上要达成的是“仁爱”的目标，而周礼是一种手段。  
于是随着时代的变化，儒学呢，就非常容易接纳新的理论内容进来。到了郡县制的时候，它就与郡县制迅速兼容起来了啊。这种呢，就是儒学身上的它的一个好处，它的优点——因为它是一个开放式的源代码，是以目标来决定手段的一种操作的方式，所以它可以改变，它身段可以柔软，它可以因时因势而异啊。  
一件事情，当它的优势出现的时候，它的劣势通常也会在这个地方出现。就是因为它没有非常明晰的、内部的、逐条的、甚至于教条的准则。因为没有这个东西啊，它没有教条式的准则，只有一个美好的目标的追求，这就造成身段柔软之后，过于柔软会变成什么？它就非常容易被外来的其他思想侵蚀。  
儒学身上的短板就在于它的身段过于柔软，以至于呢，在遇到了力量的极大冲击的时候，里面的一部分儒生在转向的时候没什么心理压力。因为当你只是要求定制了一套“仁者爱人”的目标，具体的操作手法可以根据时代的不同而改变，而且之前也确实改变过很多次。那请问，到元朝的时候来了一波新的、全新的管理的东西——就是“不管”——请问儒生接不接纳？

儒学有没有责任？有责任，有责任啊。为什么有责任？如果它是一个内部有刚性条款的一项秩序，别人就改不了这些东西。儒学内部没有刚性条款，它自身就是允许改变的，所以才会呈现出随时随地它会变的这样的一个趋向。所以当大局面崩坏了的时候，儒学就随之而崩坏了。  
所以儒学有没有缺陷？有缺陷。它的缺陷就是它非常容易沦为力量者的附庸。所以当它成为一个附庸者之后啊——附庸是什么？附庸在相对较好的时代是辅佐者，但是在恶劣的时代就会变成鹰犬甚至奴才。所以为什么到清朝的时候，儒学已经到了什么地步呢？苟儒啊。清代的儒学已经呈现出了苟儒的倾向。为什么？因为脊梁骨进一步被打断了啊。

嗯，刚性条款会出问题，参考不能出错的圣经。嗯，Bible这个东西它什么时候定稿的？这个事儿其实不好说啊。我目前认为呢，Bible这本书定稿的时间，和它编造的故事之间不匹配。但是这个事儿呢，也不是很重要，因为我们现在聊的是儒学的问题，和你这个是一致的——就是当它的特征是开放式的源代码、较为柔软、有弹性的时候，它产生的缺点就会出现在这个位置上面。因为有弹性，所以当发生了力量的颠倒、是非的颠倒，于是它就变成了附庸，从宰辅变成了奴婢，变成了苟儒啊，从君子变成了苟儒——这个都是因为身段的柔软而带来的。那么如果它的内部的条款非常刚硬呢？那钢印自然就有钢印本身的问题。钢印的条款它不容易被人家更改，但是也不容易活下来。
