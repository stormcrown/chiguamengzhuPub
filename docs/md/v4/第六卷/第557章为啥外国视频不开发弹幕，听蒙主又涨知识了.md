# 第557章 为啥外国视频不开发弹幕，听蒙主又涨知识了

**如何把话讲清楚**  
如果你想跟别人沟通、说事情或讲故事，怎么才能把事情讲得更清楚呢？跟我学唠嗑儿吧。呃，有些东西属于天分，但我确实想分享一下。我来琢磨琢磨这个规律。说话要让对方听得懂，很多时候你需要考虑对方到底能听懂什么，以及自己这句话说明白了没有。对我来说，很多事是自然的，直接就能操作，但我也乐于给大家分享其中重要的关键点。我觉得最重要的、能让对方听明白的点是：你得先知道对方到底能听懂什么，然后你使用的所有措辞和逻辑推理方法，都不能超出对方自身的语境。也就是说，你想跟对方说话时，不能自说自话，不能只想“我怎么把它说清楚”，而要先想“对方怎么才能听明白”。这是重要的目的性。

**关于文字系统的优劣**  
哎，好，欢迎来到直播间。有时候那些很好的书，我真的是强忍着看下去的，因为它们的语序有问题，语法有问题，写东西时那种拧巴的劲儿特别讨厌。为什么会这样呢？这不怪作者，责任不在他们身上，所以我这话也不是在喷这些作者本人。主要原因是他们的表音文字系统——这不是学者们的错。表音文字系统最后就会变成这个样子。为什么如此？哎，在座的小伙伴们，你们在各处看视频时是不是会看弹幕？你们看视频时是不是很喜欢刷弹幕，自己也会看弹幕？会的扣个“1”吧。弹幕这个东西非常流行，但弹幕甚至包括电影字幕这种东西——凡是文字在图像上这样飘走，老外都特别讨厌，他们恶心死了，不喜欢，不爱看。为什么会这样？因为它是表音符号，你需要在脑子里转成一段声音，自己翻成声音并捋顺了，才知道它说什么。你没办法一边看视频、听人聊天，一边突然为了那个字幕在脑子里给自己配音——换你琢磨这个场景，还看什么看？都不想看了。但中文的处理方式完全不一样：我们不需要在脑子里读出声，扫一眼就知道它说什么，完事了。所以我们很喜欢在视频中叠加弹幕，看影片时不用翻译腔（那种“哎呀，朋友啊”讨厌死了），我们喜欢直接看字幕，原声配字幕对中国人来说完全无障碍。阅读为什么能这样？因为我们是表意文字，每个字、每个词都有它的含义，语序不影响阅读，因为含义自现。含义明晃晃摆在那儿，字错了、颠倒了也不影响阅读——这就是表意文字的优越性。

**表音文字的缺点**  
表音文字的特点呢，就是字母加在一起，一共20多个字母，有些语言复杂些，30几个、40几个字母顶天了。字母数量非常有限，就那么少，反复组合。所以你的眼睛去看时，就像一群乱马在那里乱滚，噼里啪啦地滚，你必须把它读出来变成声音，再转化成含义才能明白。因此，表音文字不能直接表达含义，必须用各种组合来表示，以致于它必须有精确的语法，因为语序稍微一变，含义就完全不同了。每个单词不能完整、确切地表达某种含义，所以需要复杂的语法来约束含义的表现。语法和语序要求特别精妙——这也是前几十年公知们吹嘘的，说洋文精确、中文不精确（中文可以瞎排、洋文必须准）。为什么？因为表音文字的表达能力有限，必须极为精准地摆放在特定位置，否则就全乱了。当你了解表音文字这种特定短板时，就知道洋人写的文章为什么那么乱——这不是翻译人员的问题，而在于你只要尊重对方的表达方式去翻译（而不是进行直接二创，改造成纯中文），除非你把它改得面目全非、和原本叙事不同，否则按对方方式翻译就一定特别拧。因为它本来是表音符号，必须扭来扭去、扭曲再扭来，才能印证一条逻辑，不兜这个圈子逻辑就梳理不清楚。所以为什么我读洋人的文章读到脑壳疼？每次看到都觉得他们脑子转了筋——主要责任在于表音文字信息载量不足、表达能力有缺陷，必须用复杂叙事手法才能让表达趋于完整和准确。

**文字普及与阅读障碍**  
嗯，欢迎大家来到直播间。有人说“文字不可能是那种很难普及的图画”，什么意思？我没看懂。但我只能说一件事：如果你说图画不容易懂，那你会觉得字母容易懂吗？人类最容易理解的东西就是图画——我画一个圈，你至少知道它是圆的；我画一条弧线，你知道它是弯的。真正让人类能理解且肉眼可见的就是画，而不是很抽象的笔画——比如一个简约的三角代表A，一个蝌蚪代表8，这做不到对吧？没有强硬的字典规律、规定、册子、教育、普及、推广，字母是不可能被理解和应用的。你觉得字母文字（表音文字）普及难度低，那是因为有现代化教育。亲，在现代化教育下，象形文字或表意文字的学习成本比表音文字高，因为笔画多、字多。但哪个好理解呢？我举一个正常的例子：你知道今天的阿美（美国）文盲率很高吗？虽然具体数字我有点忘了，但好像有20%几还是30%几，你们可以自己去查证。为什么他们的文盲率那么高？一方面是因为基础教育特别烂（不好好推教育），但注意：学习表音文字难度不高，所以尽管教育烂，也不应该文盲率高到这种可悲的程度。它还有另一个重要诱因——英文特别难学。有人可能说我自打脸：前脚才说好学。但对于思维正常的人，在有正常教育的情况下表音文字很好学；但要知道人类中有固定比例的一群人有学习障碍——天生的。有一群人有天生的识别障碍：他们认这种纯粹需要记下来的、抽象的逻辑型符号，完全没有理解能力，看了就是乱码。正经说，世界上确实有一定比例的人（比例因地区而异，但到处都有），他们大脑的链路不接纳这种纯粹抽象的符号。这种阅读障碍者在欧美学习字母表音文字的地方就会变成文盲，因为他们学不了。如果家庭条件好想挽救，怎么救？让他去学习绘画，把字母、单词理解成一幅画，通过绘画的关联联想方式就能学会。所以这种思维异常人群在中国有没有？有，但为什么这里不明显？因为我们的文字本身就是由图画构成的，天然就有一种疗愈作用。尽管也有阅读障碍人士，但病症程度、范围和比例比阿美强到180条街。所以你说“人类不能理解复杂的图画”？不，人类很难理解纯粹抽象、没有含义的字母，但非常容易理解清晰的、有逻辑关系的、符合自然条件的图画。

**关于西方历史造假**  
嗯，那个叫张海滨的，你知道我们直播间的规矩吧？你只是单纯胡说八道我无所谓，但你刷屏是严重违规。他在刷一个东西，我给大家念一下：“讲你的西方伪史啊”。我什么时候讲过西方历史大面积造假？连西方人自己都承认——最开始发现西方历史大面积造假不是中国人，因为中国人当时很慕强，拿西方人当圣祖娘娘看，特别崇拜虔诚，尤其是当时识字的那帮文人骨头特别软，一想到洋爹有伟大的历史简直浪到飞起。所以去第一波质疑西方历史造假的是西方人自己，没办法。因为他们那个故事编的……就像我刚才说的，他们并不是一个精准、有目标、成体系、有计划、有列表的造假，而是不约而同地按照同一本参考书目（围绕《圣经》）进行的、各自为政的造假。这种造假方式会导致左边摊儿和右边摊儿互相碰车。这种事一旦发生，他们自家的学者就受不了了。所以最初指出历史信源有问题的是欧洲学者自己，甚至最初怀疑到底是谁开辟大航海、拿的海图是哪来的——都是欧洲人自己。是因为他们品格高尚吗？不是，纯粹是因为做学问做不下去了。

**推荐一本书**  
来，给大家带一本书。这本书没什么好的写的东西，啰里啰嗦的——洋人写的书，措辞都啰里啰嗦、七拐八弯的，说话调调我特别不喜欢，尤其不喜欢各种译作。这个心态就是我老觉得他们说话不好听，但架不住内容好。这个人是个外国人，我就喜欢他是外国人这一点。给大家谈这本书叫《偷窃历史》，作者叫杰克，这名字经典，我特别喜欢——特别典型的欧罗巴洋人的名字。他写了这本《偷窃历史》，我推荐大家去看一看。我老爱这个东西了，不是因为他写的书好，主要就是因为他是洋人呐。按什么七大八大的，不知道重点是什么。我可烦老外写的书了，包括马克思在内，所有老外写的书都喜欢弯弯绕，他们自己的语法有问题。

**关于英文教育**  
好多文献是英文的，没办法。又讲到英文教育的事：我们现在的英文教育之所以始终坚持并且分值那么高，主要原因是尾大不掉——既得利益群体太多，构成了“百万曹公衣食所系”。因此，这时你说把它分值降下来，降成副科，那老多人会觉得自己天要塌了，会出问题。为什么不解决这个问题？就出在这儿：利益群体体量过于庞大，大到难以承受的地步时就甩不脱了。这跟“看英文文献”没有任何关系。注意了，跟什么“我们毕竟还是要看英文文献的”没有半毛钱关系，跟看文献一点儿关系都没有。为啥？你真上到大学就知道了，不管你学什么专业，最后会出一门课叫“专业英语”。除了以前在初高中没触及到的复杂语法需要学之外，其他项目是什么？背单词啊，亲！背单词、背缩写。增加的语法数量非常少，绝大多数输出量就来自单词和缩写。那为什么要背字典呢？因为表音文字的创作能力、改造能力、实用能力都非常差，所以出一个新东西就要有一个新词。你学习任何专业，都要单独背一本字典。语法很复杂吗？不重要。语法在很多专业文献中的重要性快约等于无了。当然了，在我之前学的专业里语法重要性还有点强，因为经常需要强词夺理（指逻辑推理），那不一样。但绝大多数专业领域，专业英语实际上是干什么？背新单词就是背新单词，把单词全背下来——字典背下来了，语法不会怕什么。我们很多老学者根本不懂语法，只是把单词和字典背下来了，看文献杠杠的，完全不影响。所以跟什么母语教育、母语环境、小学教学、早早开始英文教学、多考试没有任何关系，毫无瓜葛。
