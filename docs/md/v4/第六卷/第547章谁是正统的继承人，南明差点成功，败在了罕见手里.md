# 第547章 谁是正统的继承人，南明差点成功，败在了罕见手里

最近这两天，老是有人说"那就是正统，那就是中国，你凭什么要分裂国家"。这种东西最近谈得特别多。然后呢，大量的中国网民——或者说现在占多数的中国网民——已经突然之间都变成了1450了。在网上高强度冲浪的小伙伴，这两天应该都挺上火，因为动不动就有大V跳出来说"你说这话你就是1450，你就是搞分裂"。这种事大家今天最近几天见多了吧？呵呵呵。见多了的话，看到这种的，扣个一，呵呵。莫名其妙的都变成了网络水军。就好好的说着话，突然之间就戴上了一顶网军的帽子，还是被顶流大V、专业学者给扣上的，心里头既气愤又委屈啊。受到了这种气的，来扣个一。哼哼哼哼，好好好啊。好的，我看到了大家都受委屈了啊，呵呵，小可怜儿们。我们啊，啥，别忘了带货。我这这么多人刷一，我就看见你那个要我带货的了，真是个好人啊，真是个好人，棒棒的。

然后那个什么，这里头有一件事，我觉得这个小故事大家应该知道一下，就是南明的永历帝，南明的永历皇帝，他是被吴三桂杀害的。哎，谢谢观自在。他可乐喝多了，进脑袋了，近期的行为。他以前自己喷了多少次，哎呀，我那个啥，咱们不聊这事啊，咱们不能老是聊别人。那个永历皇帝当年呢，算不得是一个什么全中国上下都知道呀——不行，因为他登基的那个时候，南明朝廷飘摇不定，所以他的影响力其实还颇为有限。但就是这样的一位君主啊，他为什么会被害？这件事情很重要，就是他为什么会被害。很多人会以为说他是皇帝，然后造反的人把他杀了不是很正常吗？注意啊，造反的人杀皇帝并不是一个非常正常的行为。这话说的是不是有点像暴论？但是我会来帮你分析这个事情。因为大家要知道皇帝是什么人。那个什么，就是为什么皇帝不一定会被杀？这里头会牵扯到皇帝到底是什么。很多人在直觉上面老是把皇帝当成一个人，注意了啊，皇帝不是一个人。当他作为皇帝存在的时候，他的身份并不是一个活体的人类，它是一个象征物，是一个国家、国本的象征品。所以皇帝在或者不在，皇帝是不是受辱，他是死了还是被俘了，还是当了别人的亡国奴，这个中间的差别就会非常巨大。因为他不是作为个人，它是作为一个国家的象征物。一个国家的象征物可以碎掉，这叫做"宁为玉碎，不为瓦全"。它可以碎了，它可以破了，它可以没有了、消失了、离散了，但是如果他沦落敌手、沦为玩物，那么整个国家就同时沦为玩物，它的象征的意义就是集体成为亡国奴。所以我们的历史上面啊，末代君王在沦陷入到真正的仇寇之首的时候，他会选择自尽。他会选择自尽，就像是南宋的末代小皇帝，尽管他自己可能根本没有想明白这件事情，但是陆秀夫背着他也要跳下去。为什么？实际上陆秀夫是不必殉国的，他作为一个臣子，他降了其实也就降了，但是君王不能降。为了让这个小皇帝能够完成他君王的最后使命，于是陆秀夫背负着小皇帝跳海而亡。所以他才是重要的民族英雄啊。不是跳个海就英雄了，而是他背负着国之重器，跳海这件事情是挽救了整个王朝国本的一种行为，所以他是民族英雄。我们这个地方讲的就是皇帝，他的个人身份和他是一个人之间是有隔离关系的。所以皇帝他最后做出的最终选择，就显得非常的重要。例如说到了崇祯的时候，他选择自尽而亡，这也是一种要保留国本象征意义的一种行为模式。说到这儿，如果这个东西都理解了的话，我们就转进到下一个步骤：为什么说杀害一个君王并不是敌军的最优选？因为你前面的明白了，说死掉是他的最优选的时候，你就知道了，对于他的对手来说，他最好不要死，最好被俘，最好呢，能够卑躬屈膝。一旦你让他卑躬屈膝了，就意味着你征服了一整块的一整个的国家，你把整个国家的原本的旧有的权力系统和尊严系统完全掌握在了自己的手掌中心。所以真正说一个敌军冲上来了，还说"我会把他的皇帝拔掉"，不好意思啊，不会的。那个啥，不要拿刘禅——应该叫刘禅还是刘禅不重要啊，我小时候念他刘禅——不要拿它当做样本，因为那个时代和后续的那个时代是不一样的。我们那个当初那叫做王朝内部的纷争，这和后续的宋和明是完全不同的形式。我们过去把这个事分得很明白，今天很多人糊涂，但是呢，我们在看历史的时候，你要按照当时的观感去看啊，所以不要拿刘禅出来做指标，这个是不一样的。

然后呢，还有一件事情就是，当对方把俘虏一个君王作为最优解的时候，崇祯之死是不是就合理了？因为他可以死，但是他不可以被俘。这几天我听到了一个关于这个的阴谋论，说崇祯不是自己死的，是被害的。不会的，任何一个恨他的人都不会杀他，因为不杀他才是最好的羞辱的方式，让他殉国反而并不是一个最优解。让他殉国是对大明王朝的一个最优解，所以即便他不是自己动手的，那也是自己人动的手。这个就是这里面的基本原理。所以为什么在这的时候谈到永历啊，谈到永历皇帝。永历皇帝当年被吴三桂杀害了这件事情，其实是很诡异的、不合逻辑的。那这个发生了什么事呢？要知道，如果吴三桂能够俘获永历帝，那是一个了不起的大功勋。他都已经当汉奸了，你当汉奸当彻底一点呗，立下了这样的功勋，结果呢，他把人杀了。为啥呀？因为永历帝啊，在历史的记载当中，永历帝个人风采卓绝，十分具有个人魅力，言谈举止异常高雅，胸怀宽广。这种个人的魅力与气场，以及他拥有远见卓识的这样的态度，当时吸引了一批八旗兵。那个时候要知道八旗兵是什么人，他们当初啊，大量的八旗的骑兵们自己就是主子麾下的奴才。大家看过电视剧的也知道，他们内部一直都是主子奴才的，他们内部始终都是各种各样的主子奴才。而这些八旗的骑兵，当时原本他们的身份就是主子麾下的奴才，但是他们见到了永历的时候，突然之间意识到自己是一个人，突然之间见到了什么样才是天子、什么才是君王。他看到了一种超乎于他原有理解的统治者的范式，于是这一批旗人部队、这一批八旗兵决定造反，决定反清复明。啊，就是这样的时候啊，一批八旗兵决定反清复明，而且呢，他们在造反的时候数量级并不算少，并且身体力行把自己的辫子剪了。然后当时吴三桂发现了之后，嗯，这事儿也不好，这个事要是让他办成了的话，那吴三桂——你要知道，归顺了的八旗兵可以算作英雄，但是这个开门的吴三桂，到什么时候都是大汉奸，所以他是不能够容许这种情形存在的。嗯，我的乖乖呀，这只猫拿它的毛茸茸的屁股就把我放水杯的地方给占了。敲你的小脑壳，敲你的小脑壳呀。

好，来我们来带个货吧，带本书啊，带这个顾诚老师的《南明史》。当当当当当当当——哎，谢谢大明珠。虾仁堂而皇之，驱逐中华，恢复大陆。呵呵嘿嘿。嗯，《南明史》这个书呢不好看，坦然承认。为啥呢？因为南明的那帮子人吧，就能够从每一次有机会里面发掘出最没有机会的那条路，从所有可能会有正确推动力的选项当中挑出最错误的那一个，也算是很奇葩的一群人啊。完全可以去了解一下。至于说你看了它之后难受，难受是正常的，但是呢，我觉得所有的中国人都应该依然应该收藏一本。难受嘛，你就慢慢看，什么时候觉得需要上上火气的时候，你就去看两下；需要提振一下心气血爆棚度的时候，你就去看两下。其他的时候呢，就围观一下它的封面，我觉得也是好的。为啥？因为它需要你放在心里头，这是我们历史当中非常重要的一段教训。然后呢，经过这几十年，或者说经过这几年的互联网的博弈和梳理，在座的小伙伴们应该已经都知道了，有"岁月史书"这种东西。即使一套书早就已经写出来了，知道南明什么样子，知道当时的局面如何，也知道到底当时的中国人遭遇了什么，但是依然架不住岁月史书。短短的数十年间，我们解放初期课本里面的那个时代的形象，已经和今天所鼓吹的那个时代的形象变得完全不同了。"无复难民救世"变成了"南明是分裂势力"，所以这本书大家就更要收藏一下了。你要知道，当有一天整个南明都变成了负隅顽抗的反动势力的时候，你可以参照的坐标系是什么。

嗯，可以谈琉球吗？琉球族有没有可能成为第58个民族？我觉得琉球大可不必将它称为某族，因为琉球上面的人就是当时我们的人出去的，谢谢。嗯，啥？琉球——琉球36姓，琉球本来就是从我们这里过去的人，有一部分是海商，有一部分是渔民，还有一些是沿海地区的大家族的子弟。嗯，所以呢，琉球的先民原本就是我们自己的亲生的手足同胞，不是拜把子那种，不是认亲那种，不是表亲，所以不可能说还要给它单独安放一个什么身份，不存在这个事。嗯，新垣结衣就是琉球的，新垣这个姓氏在先秦的时候我们这里都还有。嗯，最后一个姓新垣的……我这个人不记人名，你们是知道的，你们可以自己去搜啊。新垣这个姓不是外国姓，是中国姓，当然应该叫"士"啊。他在我们这里的时候，活跃的那个期间，他应该叫做"士"。诶，好，欢迎来到直播间。
