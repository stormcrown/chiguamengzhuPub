# 第553章 为什么台湾青鸟讨厌义和团 540P

咱就是极右啊。从政治学的角度来说，义和团也好，太平天国也好，就是极右啊——盲目排外、极端民族主义。老娘我一个极左，干的就是你们这一伙义和团。论诈尸的鬼，义和团代表了整个中国这片土地上原住人民，在最低最低下限情况下的反抗精神。而这种反抗精神并不是在任何地方都天然存在的。例如我们隔壁的土地上，就有神棍站起来说“我们不要暴力”，于是无数老百姓就坐在那里，陪着他一起等待死亡的降临。对于侵略者来说，你愿意等，我自然愿意陪你等——怎么，我吃着饭，还等不过你饿肚子的吗？

万万没想到，建国70年了，大家都有9年义务教育可读，从小学到初中的课本都完完整整地贴在我们脸上、摆在我们面前了，还是会有人在今时今日跳出来指责义和团。而且在我反复在直播里说义和团的好话之后，依然有人特意贴着我的脸跟我说义和团的缺陷：他是何等的愚昧，他是何等的不正规，他是怎么样的没有前程、必然失败。啊，我们希望的应该是有更多优秀的将领，带领着优秀的部队来解放我们的国土，而不是让义和团这种乌合之众上马。啊，很好啊，你们都可会说话了，语言特别流畅，言辞酣畅淋漓，说得不知道多赞——我给你们点360个赞好不好？但是为什么已经过去了这么长的时间，依然有人如此痛恨、如此恐惧义和团呢？对我说的就是这个事。为什么但凡出一点点苗头，立刻就有人跳起脚来：“你义和团怎么回事？”这是源自于什么？人类的任何一次激烈反弹，碰见关键词立刻就炸毛，这种反应只能证明一件事：他从内心深处对义和团充满了发自灵魂深处的颤抖和恐惧。什么人会害怕义和团？义和团为什么会震慑他们？为什么他们想起义和团都发颤？为什么他逮着机会就要给义和团泼脏水？为什么坚决不肯放过这些拿着镐头锄头的农民？为什么一定要把他们钉到耻辱柱上去？这些深层的恐惧来自何方？

今天我们就来聊聊义和团，它到底可怕在哪里。前两天的时候，有人跟我探讨说：“哎，那义和团稀松平常，他们什么都不会，这也不懂那也不行，都是一群打乱仗的。”呃，你说的倒是没有错，他们确实是在打乱仗，也确实愚昧又无知，都是一群莽夫而已。然后呢，他同时给我举了例子，说那些优秀的将领带领着什么什么样的军队，可以怎样怎样打杀敌人，那才是我们的文明之光嘛。你说的也没错，如果在那个时期能够出来一个岳飞，能够出来一个霍去病，能够出来一个辛弃疾，建立什么戚家军——哎，我的天哪，那简直就是天降神兵嘛！但是为什么侵略者、那些列强、那些八国联军会那么恐惧义和团的出现，以至于义和团出现之后，他们就放弃了瓜分中国的愿望呢？问题就出在义和团的土，出在义和团的愚昧，出在义和团的无知上面。

我们来对照一下，岳飞够厉害了吧？能干的不要不要的，我简直不知道应该怎么赞美他，允文允武，要兵法有兵法，要治军有治军，甚至写词都写得一把好手，而且呢，人家还能筹来军费，还不是光是大撒钱。你说这样的全能分子怎么可能不崇拜他，怎么可能不爱戴他？他一个人人品又好、才能又好，简直是无可挑剔。但是他身上有一个最巨大的短板：他孤身一人，想要解决他，只需要12道金牌以及一个风波亭——结束了，句号，game over。对于敌人来说，岳飞是一个了不起的对手，但是仅此而已。

可是义和团不一样。义和团里面的人是什么样的人？他们愚昧无知，他们长得普普通通，他们贫穷得不得了，连一身完整的衣裳都买不起。但是你会发现，就是这么一个让你瞧不起的、犹如草芥的、犹如蝼蚁一样的屁民，在听到了外面的风声说“洋人吃孩子了”，我的天哪，“我们去弄死洋人”，他拎起家里的柴刀就冲出去了。他连衣服都穿不齐全，他就敢攻城略地，他就敢往上莽。而且像他这样的人，每一亩田里面都能钻出三五个来。你明白这个意思吗？你能够理解这种场景吗？这个时候你必须要站到你的敌人的那一边去，站在侵略者的那一边去，你才能够体会到这种恐惧感、这种压迫感。因为对于敌人来说，一个岳飞无非就是我需要花费10年的光阴去培养一个秦桧以及一个完颜九妹；但是面对着乌泱泱一片，没有名、没有姓、没有任何形象，随随便便就冲上来了，而且不死不休的那一个一个的王阿大、王阿二，你根本就无从扫荡得起，你不知道该怎么预防，也不知道该怎么防备。你早上还买糖果的那个人，晚上的时候就已经站到你的家门口，与你不死不休。对于敌人来说，这才是真正的反抗。

义和团，代表了整个中国这片土地上的原住人民，在最低最低之后，在他下线情况下的反抗精神。而这种反抗精神并不是在任何地方都天然存在的。例如我们隔壁的土地上面，就有神棍站起来说“我们不要暴力”，于是无数老百姓就坐在那里，陪着他一起等待死亡的降临。对于侵略者来说，你愿意等，我自然愿意陪你等——怎么我吃着饭呢，还等不过你饿肚子的吗？但是在中国土地上不是这个样子的。来围剿我们的列强，在清末的时候发现了一个非常悲哀的现实：在这片土地上面，最可怕的根本不是领导者，甚至是最好打破、最好击碎的、最容易跪在地上舔他鞋的，就是那些衣冠楚楚、衣着华丽的满清贵胄。相反的，那些懦弱的、看上去不堪的、看上去无知的普通百姓们，反而拥有着无知又无畏、勇往无前的力量，谁都打不垮他们——倒了一个又来一个，倒了一群又来一群，你永远不知道他们藏在哪个地方，犹如草芥又灿若星河。

就是这样的一群人，真正对侵略者产生了威慑力。以至于义和团运动之后，八国联军彻底放弃了瓜分这片土地的愿望，转而与满清官方合作。因为他们真的发现，这片土地上最容易打倒的就是这些满清贵族，最容易谈判的就是这些满清贵族，最容易敲诈的就是这些满清贵族，在他们身上真的能拿到好处；而他们真的想要跟人民百姓作对的时候，就会发现人民战争的力量犹如汪洋大海，没有人能够战胜它。这就是为什么，明明全世界都面临着侵略与殖民的战争，而我们的国家依然延续至今。如果当初没有义和团，其实就没有后来的革命军；如果没有革命军，就没有今天的共产党人。我们所有的革命历程都是一步一步走过来的，中间有犯过错误，有曾经失败过，但是不能因为今天成功了，所以过去失败的尝试就全部变成了劣迹。那些也是我们开始革命的最重要、最重要的基础——就是中国土地上的百姓，每一个人都不肯低头。这个就是最强大的威慑。

而为什么今天的互联网上，依然有很多的人对于义和团无比的恐惧，谈到他的时候，都觉得必须要把它摁到泥地里面，让他不能抬头？请问他是谁？谁会对这一群普普通通但又十分坚决的反抗者感觉到恐惧？他们是哪类？如果放在晚清时期，他们就是列强以及列强的走狗，还有境内的买办。那么放到今天，他们是谁呢？我也很好奇。不然你出来自报一下家门吧，阁下哪位？
