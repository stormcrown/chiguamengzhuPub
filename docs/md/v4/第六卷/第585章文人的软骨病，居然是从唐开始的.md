# 第585章 文人的软骨病，居然是从唐开始的

我们刚才还有一个付费的问题，被我提了一嘴后放过去了。那个问题叫什么来着？就是从80年代之后，突然冒出了那么多买办文人，这是怎么回事呢？明明之前几十年接受的都是自立自强的教育，为什么到那个时候，咔嚓一下就跪下了呢？你要知道，之前几十年的自立自强教育，并不是所有人都感知到了。我不知道谁能立刻理解到这件事——我们前30年的自立自强教育，是帮助谁自立自强起来了？是帮助谁自信起来了？是帮助谁察觉到了自己是主人家？是工农阶级。嗨，小伙伴们，能想起来了吗？我们前30年，整个国家昂扬向上的建设者、被赞美的建设者是谁？是工农阶级。而那个时候，文人艺人这些文艺工作者的社会地位，其实相较于旧社会是下降了的——能get到吗？前30年，我们实际上对文艺工作者群体在进行规训，是由工农阶级对他们的腐朽文化、欠缺国家本位精神的文化、充满奴婢精神和买办意识的文化进行批判和再教育的时代。所以那几十年，很多文人是很委屈、很憋屈的。他们经常在自己的书里说：“哎呀，不得了了，他虐待我！怎么虐待呢？他让我像工人一样干活。”一个有18小时工作制、有所有福利、所有保障、能够吃饱穿暖的工人，对于一个工人来说是了不起的光明时代；但对于一个以前只需要巴结富家老爷就可以当二老爷、对所有人指手画脚的文人买办群体来说，让他们去当工人、当农民，那是贬低，是虐待。所以为什么大门一开，哗啦一下这帮人就变成了软骨头，膝盖骨咔嚓一下就消失了？因为他们以前也没长上。

嗯，从明以后文人的骨头就没有硬过？那倒也不是——因为从唐以后文人的骨头就不行了是吧？不对，这代文人的问题出在什么时代？出在皇朝这个时代啊。不是皇朝，不是说黄巢做的事不对。我之前是讲过的，黄巢做的事情本身是对的。黄巢打破了原有的氏族垄断的藩篱，当初是按照户口本，把氏族门阀给咔掉了。所以在唐代的时候，原本的所谓士人群体是被过去的氏族门阀垄断的。这些人有没有骨气？这些人很傲慢，但他们不是好人——不是说善恶的问题，而是在世家门阀没有倒台之前，士人群体是国家的股东。这个事我是聊过的，相当于国家的股东。这些股东群体有主人翁意识，或者叫主人意识也可以，就是将国家视为自己的私产，不许别人碰。所以他们在傲慢、在阶层固化、在种种缺陷的同时，它是有骨头的。这就是早期的贵族。今天有很多人吹所谓的贵族精神，那贵族精神是什么呢？就是视这片地为自己的私产，拼命去捍卫——这种精神就叫贵族精神。所以这些人有没有贡献？有。在黄巢踏进公卿谷之前的这些统治阶层，由于他们事实上属于能拿到帝国分红的阶层，所以和整个王权之间有深厚而紧密的关联，他们就会有一定的归属感和责任感。这个时代他们是不太软骨头的。

嗯，什么？但是问题给特权可以吗？没说可不可以，我一直说不要二极管——我直播间里二极管待不下去，你再二极管就把人踢出去了。说了100遍，不是说他们是好人，也不是说黄巢做的事不对，而是说人身上会产生两面性。一群人由于将天下视为私产，所以社会阶层固化，但同时也造成了他们有强烈的主人意识——将天下视为私产，就不允许别人冒犯，就产生了这种主人意识。黄巢之后呢，就发生了新的问题。因为原有的氏族阶层被直接肉体消灭了，消失了，一个大的阶层完全没了。没有了这个阶层，国家又不能崩，事情还要照做，于是士人阶层就置换成了考试上位的新的士人阶层。由于采用了这种考学、应聘的方式，原有的那些可以分红的大股东、小股东士人，就转变成了后面所谓的职业经理人士人。从宋到明这个中间，所有的这些文人绝大多数实际上是职业经理人。职业经理人的特征就是给谁打工都是打工，他们身上这种味儿特别强。这个事是从哪来的？从黄巢之后开始逐渐形成的。然后你说他直接咔嚓人没了，立刻不能换成新的股东吗？那个中间，就不得不说唐留下来了一个非常黑暗的、或者可以说在那之前除了清之外最黑暗的乱世——五代十国。这个黑暗的乱世留下了很多隐患，其中最重要的一部分就是中国的北方地区长时间处在胡人政权的统治之下。尽管今天的教材会告诉你他们汉化了如何如何，但当时的百姓非常清楚统治阶层是什么人，甚至当时的统治阶层还在各种层面上推行过剃发、易服，以及将普通农民归为奴隶、奴籍这些事情。辽做过，金也做过。这种明显的奴役行为、排除异己行为，实际上会造成他统治内部是分层的。辽也好，金也好，其统治阶层内部是有分化、有分别的，包括之前那个西夏也是这样的。内部分化什么呢？他会将自己女真的朝臣和汉臣能担任什么岗位、肩负什么责任、能拿到多少福利都分开安排。大家看上去像是有科举，包括说辽的那个皇帝耶律大石，我们经常说他当年当过翰林，所以还经常尊称他为大石林牙——大石林牙就是因为他当年考学、通过科举上去的，当过翰林，这是他非常重要的人生成就，所以后来尊称他为大石林牙。但是他们的这种晋升、这种考学也是有分流的，按不同的人分给不同的岗位，考试的方式、录取的方式都还有差别。

当内部管理有分层时，会出现一种新的现象：这些文化水平更高、文化底蕴更强、甚至政治手腕更高明的汉臣，在这些地方属于附庸阶层。而附庸阶层会为自己自圆其说，解释自己为什么在这里当附庸。于是从这个时候开始，他们开始尝试去解释自己这个打工人的身份。在之前，文人集团、士人集团、门阀是当时国家的小股东，是有特殊权利的人群。但从盛唐崩溃之后，这一处又一处被割据出去的，以及后来逐渐迭代出来的新胡人政权，以及后来长期统治北方的胡人政权，他们内部由于这种分化管理的方式，使得当地的汉人士人、读书人、文人阶层变成了一种他者的附庸。既然成了附庸，就要解释自己为什么可以理直气壮地当别人的附庸。于是，成为打工人、不再是股东之一的心态开始慢慢沉淀下来。而这件事到什么时候开始落地的呢？到宋的时候。由于辽金的统治相当稳固，军事上的强盛使得他们阶段性地达成了统治的稳固性，所以内部的文人集团就在长期的安稳过程中不断积累理论基础。这些东西在后世，在元的时候达到了一个彻底的高峰，完成了一次理论上的重新建设。大家今天在互联网上经常能听到的“夷入夏则夏，夏入夷则夷”，还有什么“夷而尽忠原则”、“夷狄而近中国则中国之”等等这些话，都是在元的时候才诞生的。在那之前这种理论根本不存在，都是到了元的时候才出现在我们的理论系统当中。这背后带来的结果是什么？就是文人这个群体不再是国家主人的一环，这些社会的中层管理阶层不再需要为国家负最后的生死存亡的责任，他们的责任变成了口头责任。同时他们变身成了这个国家的打工仔——高级打工仔。当一群人掌握极大的权力，甚至很多时候可以决定一个国家的方向，但他只是一个打工仔的时候，他就会软骨头啊。
