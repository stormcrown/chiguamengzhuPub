# 第506章 什么是人民史观？怎会有人觉得劳动人民不是青年才俊？

昨天的时候，我跟人谈及过有关于“肉食者鄙”的问题。然后谈着谈着，最后就扯到了圣人的问题。圣人自然就跟儒家有关系，因为儒家的创始人就是一位公认的圣人。那圣人是什么概念？为什么有一些人可以成为圣人，有一些人也做了很了不起的事情，却不算是圣人？我们的这个标准到底是什么啊？

为什么这个话茬儿是从“肉食者鄙”里面弹出来的？其实一开始的提问是这样的：说某一些身居高位的人，他很聪明，毫无疑问是精英，视野非常广博，知识量非常大，但是他却会做出一些对于自己的国家、自己的民族不利的、对于长远没有好处的决策。这是咋回事呢？他为什么会这样做？所以就牵扯到了“肉食者鄙”，因为古人也看到了这种现象——有很多的统治阶级，他并不是愚笨的，在他自己有信息来源、有知识来源、又聪慧的情况之下，会做出不利根本的一种选项。

然后呢，我就牵扯到了一个事儿。我们有一句话叫做“人民万岁”，大家记得吧？如果记得的话，可以在公屏上打出来：“人民万岁”。那么为什么人民万岁？是因为人民特别的聪明吗？是因为人民特别的善良吗？当你去看他的个体的时候，一定会发现他里面有很多的人知识不行，会发现他会一哭二闹三上吊，会发现他不排队——这些都是人民队伍当中的人，普通的、平凡的，有时候精明，有时候又有点愚笨的这些普通的人。但是我们最后为什么会说“人民万岁”？因为我们作为一整个国家、一整个民族的整体，我们的根本利益就是人民的根本利益。是因为根本利益上面的一致。

所以，什么是国家和民族的正确的方向？正确的选项就是符合人民的根本利益。不管他是愚昧也好，还是小肚鸡肠也好，它的根本利益就是和我们的族群的根本利益是高度绑定的。因此我们才能说“人民万岁”，因为他只要整体性地为了自身的利益所考量，所做出来的根本选项，就是最正确的那一个选项。因此，人民万岁。

那么为什么会“肉食者鄙”？肉食者怎么就不能够万岁？他明明更加聪明，更加有远见卓识。因为肉食者所在的那个位置，就决定了他的根本利益和下面的、普通的、海量的这些如同蝼蚁一般的人民之间产生了沟壑。他想要满足他自己的这种个人的利益最大化的时候，就会违背国家与民族的根本利益，所以叫做“肉食者鄙”。他是很聪明的，他是很能够为自己搞到最多的好处的，他是很能够为自己规划长远的未来的，但是他不肯与那些劳苦大众站在一起。

所以说到这儿的时候，是不是就扯到圣人的问题了？什么是圣人？他是精英，他聪明，他果决，他信息来源广，他身居高位，但是他愿意和人民站在一起。两样事情同时满足，他才有可能成为一个圣人。

那么在先秦时期的孔夫子身上有没有这种特征？他是什么人？很聪明，有才华，学习了当时最好的文化，向老子问礼，手里面拥有最好的典籍，他知道如何构建一个国家的社会秩序，所以他是一个杰出型的人才，对吧？那么作为一个杰出型的人才，他做了什么事？有教无类。他让那些没有机会进入到统治阶层的人群，也可以拥有统治阶层的技术和文化水平。在那个时代——在先秦那个时代，他就属于站在了知识的金字塔尖儿上面，但是却愿意让下面的那些普普通通的人也可以拥有攀登金字塔的机会。有教无类这一件事情，你说哎呀，他还是收束脩的，他收了那么长条的腊肉，吃不饱饭的人是没有办法念书的。他确实设了门槛，但是在那个时代，你吃饱了饭，你也没有资格去学习这些东西。而他打开了这个通道。所以他为什么是文圣人？我一直都说他是一个伟大的教育家，是老师的开山鼻祖，就是这个缘故。

精英，然后同时愿意打通与大众之间的通道，这是一个非常非常艰难的抉择。在世界上的绝大多数地方，其实根本就做不到。绝大多数的人，绝大多数的精英根本就做不到这一点。所以什么是圣人？今天我们离我们很近很近的这个时代里面，也有这样的人。只是今天我们不再把这些人称为圣人了。但是这个名字，这个词汇依然存在。我们得知道什么叫做“圣”，什么叫做伟大的贡献？不是你做得吊炸天了，你就是伟大的贡献。你得在自己能够吊炸天的情况下，还愿意与那些犹如蝼蚁一样的百姓站在一起，考虑人民的根本方向与根本利益，你才能够成为那个“圣”。

所以你说儒家的贡献，儒家的贡献是什么？儒家是一个学问的载体，它就是一套学说。这套学说呢，它也有可能在某一个时间段无效，某一个时间段有效，某一个时间段他甚至可能拖后腿。这些情况都会出现的。但是儒家的这种思想，确确实实地给我们后来的数千年都开创了一个根本性的道路：就是“王侯将相，宁有种乎”？有教无类，所有人都必须要有上升通道。你如果不留这条通道，你就是一个恶劣的统治者。他让统治者也要有道德的约束，每个人各安其位。他要求权与责相对应。这个在先秦时期是一个相当伟大的开创。要知道那个时候还是贵族的天下，没有人给贵族提要求的——贵族天生贵人，草民天生蝼蚁啊。但是儒学的出现打破了这套藩篱。所以他做出了这些根本性的贡献。至于你说他后来如何如何了，他们家的衍生公如何如何了，这是两件事情。根本性的贡献是不可以否定的。因为这件事情已经过去了，它是历史已经画上了句点。他在2000多年前的那种贵族至上的时代，打通了通道。我们后来为什么会有很多人觉得，虽然我穷，但是我就是应该可以更好？就是当年孔夫子打开通道的这一瞬间，给我们留下来的烙印。不然的话，我们也会像我们的隔壁一样，王侯将相宁有种乎？有！我们之所以觉得无，之所以觉得每一个人都应该凭实力可以跨越，之所以尊重小镇做题家，都是因为当年有了这样的一个伟大的创举。

嗯，啥？这样带货的？第一次看到？嗯。你经常来我的直播间，你就可以每天看到了啊。想看这还不容易吗？想要看到带货可以如何的敷衍，这很简单，因为我每天都这样啊。

弹幕有人说：“留下的都是苟且的”。苟且的——你知道能活下来有多难吗？能活下来有多难呢？今天有很多人说现在工作有多难，赚钱有多难，生活有多难。但是放大到我们数千年、上万年的文明史，我想说的是：人呐，活下来得有多难呢？是真的难，不开玩笑的那一种。生产力水平差的时候，疾病能带走你，饥饿能够带走你，甚至在路上被石头划了一下脚，破伤风都能带走你。条件好一点了之后，地主的压迫可以带走你，贵族的奴役也可以带走你，部落之间的征伐、国家之间的不和都可以带走你。我们整个的这个历史的进程当中，有多少次面临着灭顶之灾，千里无人？你要是说最后活下来的人都是苟下来的，可以。你以为苟不是一种能力吗？在最困难的时候，低头让自己熬过去，挺过去，你以为不是一种能力吗？这是很了不起的能力。尤其是当一群人曾经昂首挺胸，曾经威名赫赫，曾经在任何地方都体面都妥帖的时候，在遇到了最艰巨最艰巨的环境，他还肯低下头来埋伏下来，等着蛰伏下来，等着下一次的崛起——这比永远都挺直了还要难。更关键的是，大部分的群体这样子苟下来了之后，他就再也直不起来了。可是我们不光能够在艰难的时候低头，在重新可以站起来的时候，我们还可以重新支棱起来。这在世界上是绝无仅有的。绝大多数的族群为什么是倏忽而兴、倏忽而灭？就是当他强势的时候，耀武扬威，然后一旦衰败了之后就四散而逃，就没了，所以就消亡了。你以为是人都死了吗？不是，是群体性的消亡——跑路了，不承认了，不承认自己是这一伙的了，就没了。但是我们没有。我们经历过太多太多的苦难：五胡乱华的时候，北方千里无人呐；剃发易服的时候，很多的人都不记得祖先衣冠长什么样子，从认字变成了文盲。就经过了这么多，这么多，最后我们还是能够再支棱起来。遇到阳光雨露的时候，能够再重新的爬起来。能屈能伸，每一种都是能力。在最直的时候能屈，在屈服了之后能够生，是很了不起的事情，绝无仅有的伟大啊。

嗯，橱窗的《南明史》能介绍一下吗？嗯，可以。主要的是你得先对这个《南明史》感兴趣啊。《南明史》这本书呢，它是很难得的一部以阶级史观和人民史观的视角来记述历史的书。然后呢，它讲的就是南明的这一段风雨飘摇的岁月。南明朝廷是从崇祯皇帝1644年吊死煤山开始算起的——就是正规的那个明王朝，在这个时候就画上了句号。但是之后呢，还有举起大明旗帜的这个南明朝廷啊，就是扶持新帝设立小朝廷，来指挥天下兵马的这样的一群人。但是这个南明朝廷吧，就是永远都在犯错，不是在犯错，就是在犯错的路上。所以这本书呢，就是拿来给大家提神醒脑，拿来给大家吸取教训的。可以看一看我们的“肉食者鄙”当中的这些肉食者们是怎样的——不能远谋是怎样的，没有全局观是怎样的，分不清敌我是怎样的，拿自己人当炮灰、敞开胸怀拥抱敌人的。如果没有这些骚操作的话，南明不至于垮成那个样子，清军入关也不会那样势如破竹。实际上他也没有办法势如破竹，一统天下的进程花了大几十年。而他的阻隔的力量是有的，但是它内部之所以分崩离析，就是因为它的内部的问题非常的严重，自我认知严重不足。不知道自己的基本盘是谁，也不知道自己的敌人几斤几两——既不知道自己是谁，也不知道对手是谁，最后就是每一次做选择题都是错的，每一次做决策都是错的啊。这个里面呢，大家会看到一些可能和你过往印象当中不太一样的历史人物啊，它充分地记述了某一些人在某些时候做出的错误的选择，以及正确的选择。所以这套书呢，属于是一旦你对这段历史感兴趣，你就应该拿这套书作为样本来学习。如果你想了解清代历史，也应该从这本书开始学习啊。你要了解清史，首先要学《南明史》。如果对《南明史》不了解的话，那你对清史的认知就会是错位的。因为在清初的数十年间，南明朝廷一直都存在。而且南明朝廷的旗帜立起来了之后——哎，前两天我，我是在哪看到了，我的评论区里面刷过的一个评论啊——不是我的评论区吧，忘了，不知道搁哪看的，反正好像是说我的啊，就是说我什么只会吹明，但是对于大明的灭亡，对于什么农民军都反抗大明这件事情完全说不清楚。农民军是反抗大明，但是满清一入关，农民军咔嚓一下子就站到了大明麾下来了。所以你说谁烂？人人都知道大明已经没有国运了，所有人都知道他民心尽丧。就在这种情况下，全靠同行衬托，他突然之间又富有了正统性。所以你要了解清史，也要从《南明史》来着手啊。
