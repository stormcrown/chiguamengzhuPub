# 第570章 再谈何谓大一统？统一思想还是统一发型算作大一统

如果内部存在两套语言系统、两套文字系统、两套官僚系统和两套法律系统，所有人之间拥有截然不同的权利和义务——住不到一起、吃不到一起、不通婚、不往来——然后你说这就是“大一统”，因为它地盘大？这符合基本的大一统原则吗？书同文了吗？行同轮了吗？车同轨倒是不知道，但那也不是很重要。当人和人的法律规范、文字系统、晋升机制都不一样的时候，你说这玩意儿叫大一统，还要以它为范本、拿它当榜样？那干脆咱们就搞种姓制好了！榜样是这个样子的，那咱们是要践行种姓制吗？我们几千年没搞过种姓制那种东西，我们讲的是“天子犯法与庶民同罪”。能不能做到是一回事，原则有没有立下是另一回事。

简单来说，就是有些人不要脸。说好了不骂人的——没有骂妈啊，你们自己说吧，我是不是没有骂人？我今天很文明的，压根没骂人，对不对啊？没有没有，完全没有，完全没有骂人。我锁好着呢，对没有骂人。来，整起啊！没有没有没有。哎呀，吓死我了，我还以为我骂人了呢。确实没有啊！只要对方不是人，我就没有骂人。

前两天我看了一篇文章，觉得不错。我很支持他的讲法：中国人在社会治理方面非常有一套，但我们有自己的短板，要坦然承认短板，并努力规避它暴露出来成为我们的弱点。这个短板是什么呢？就是我们欠缺身份治理的能力。注意，这是卢诗翰写的——你们谁给他打个名字吧，应该有人认得。另外还有一个作者，我前两天刚看见的，觉得这部分不错，所以拿出来分享一下。我说的和他说的不一定会一样，但内在逻辑是相似的：中国人社会治理能力非常强，但我们的社会治理是基于大一统观的。

这就谈到另一件事了。我们的所有社会治理是建立在大一统观下面的，是社会整体调度——把有需要的岗位安排上相应的人，协调调度资源，集中力量办大事。这是中国的社会治理方案，也是我们的特长，在全世界独树一帜。只有我们能做到这一点，全世界其他任何地方都无法长期、稳定地做到集中力量办大事。这就是大一统给我们留下的宝贵财富。但与此同时，它也有短板：当你一直用大一统的方式进行管理时，对于那些没有大一统符号的加盟商、个体或群体，就很难有效治理和管理。也就是说，我们的大一统管理法，仅限于那些有大一统观、能够在大一统系统内安稳工作的人——它的管理效率特别好、非常完善，而且极为有利。但对于那些基于丛林法则而生的群体，它不行。大一统治理方案是分人的，但我们要求一切规则都是一整套——大一统就是所有规则完整一套，不能分门别类，说你用你的法、他用他的法。而这种方式是什么呢？高种姓用一套规则，低种姓用另一套规则，这叫做身份治理。中国人自古以来就欠缺身份治理能力，不会在社会管理内部划分成不同片区，对不同的人用不同方式、打不同标签、给不同权利、做出不同约束。这件事在大一统的治理框架下无法实现。但现代社会，我们经常会遇到使用身份标签进行个人小群体权力争夺的战斗。所以，我们的治理方案中确实存在短板：对于不符合大一统观的人群，欠缺治理能力。自古以来就是这样。为什么常常觉得“怎么进来这么久了还养不熟”？中国人常感叹这件事，但这就是我们没有身份治理的能力。我们总用大一统方式进行全盘调度，但有些人自己给自己贴了标签，不同标签的人对自己评判的态度是不一样的。

到底什么是大一统？我觉得一件事提出来了、觉得它很重要，就必须对它下定义。现在经常有人篡改大一统的概念。我不知道有多少人注意到这个事。现在很多人篡改大一统的概念，嘴上说的都是“大一统”，但到底什么才是大一统？今天有些人篡改的大一统，是谁能囊括今日共和国所有土地面积，谁就是大一统。这是什么？这是土地大一统。但我反复强调一件事：自古以来，土地就是我们的财产。土地是人的财产，所以我们从来不会把土地作为标准，因为国家不是由财产决定的——国家是由人决定的，不是由财产决定的。你财产多，所以你就大一统了；你财产少，你就不是大一统了——这严重违背了中国人的传统道德观。咱们一向是“儿不嫌母丑，狗不嫌家贫”，富有富的活法，穷有穷的过法。我们不会那么在乎贫富，因为财富是自己创造的，重要的是一家人都能在一起。所以我们在乎的从来都是人，无论是从小家庭还是到大国家，我们在意的永远都是人的一统，而不是土地的一统。不是说你的地必须巨大才叫一统，地小一点就不叫一统了——那就相当于说你穷就不配称之为家，只有豪富才是家，这种逻辑是错误的。

所以什么叫大一统？如果按土地一统观来说，秦始皇妥妥没有实现大一统——秦国的土地不可能覆盖今日共和国的土地，做不到，因为那时的探索能力没到达那个位置。他达不到，于是就没有大一统？那你要不要把我们的所有史书一把火烧了？把经、史、子、集全烧了算了？真的，这种词义的篡改，焚书坑儒都没这么狠。

那么，什么是大一统？大一统是所有的人都践行同样一套规则，有着一致的行为方式。所以最标准的大一统基本范式，就是始皇帝订下的“书同文、车同轨、行同轮”。这个东西叫大一统——所有人都践行一套道德准则，所有人都运用一套文字体系，所有人都运用同一套法律体系。你根据这套法律体系对号入座，看看你在当中是什么人，对照办理。这个才叫大一统。所以一个王朝实现了大一统，但它的战争能力没那么强、疆域萎缩了，它也是大一统。但如果它内部有两套语言、两套文字、两套官僚、两套法律，所有人之间权利和义务截然不同，住不到一起、吃不到一起、不通婚不往来，然后你说这就是大一统，因为他地大——这符合基本的大一统原则吗？书同文了吗？行同轮了吗？车同轨不知道，但那也不重要。人和人的法律规范、文字系统、晋升机制都不一样的时候，你说这叫大一统，还要以它为范本、拿它当榜样？那干脆咱们搞种姓制好了！榜样是这样的话，咱们是要践行种姓制吗？我们几千年没搞过种姓制，我们讲的是“天子犯法与庶民同罪”。能不能做到是一回事，原则立没立下是另一回事。

简单来说，就是有些人不要脸。说好了不骂人——我没有骂妈啊，你们自己说吧，我是不是没有骂人？我今天很文明的，压根没骂人，对不对啊？没有没有，完全没有，完全没有骂人。我锁好着呢，对没有骂人。来，整起啊！没有没有没有。哎呀，吓死我了，我还以为我骂人了呢。确实没有啊！只要对方不是人，我就没有骂人。

好的，来，大家欢迎来到直播间。我们现在带一套书——弹出来了没有？弹出来了。《南明史》里头有没有这个？我看一看。小破站没有，小破站就凑合吧。另外两个平台有《南明史》，也是一本特别难看的书。我今天专门看难看的东西，呵呵。《南明史》很难看，你买不买？特别难看，难看到什么程度？看了之后牙都痒痒，睡不着觉，头顶上长包，吃不下饭。《南明史》就是这种玩意儿，老气人了。但这本书很重要。实在不行的话，就放家里压箱底存着，时不时地如果哪一天觉得火力不足，就翻出来看一眼。《南明史》这个东西还是有价值的，有价值。

崇祯不应该上吊的。崇祯殉国这件事，作为明王朝来说是有价值的，对于中华文明来说也有重大的历史价值，所以我不会否定他殉国的行为。我明确说，我不会否定崇祯殉国的行为。为什么？因为天子殉国代表一个王朝以最体面、最干净的方式画上句号，维系住了明王朝的正统属性。我一直强调，正统属性其实就是民心——有没有在你这里？你这旗帜一拿出来，大家突然想到“对，这是我家的旗”。有这样的效应，那就是正统的权利。正统权利其实是一种号令权，不是说我打赢、有千军万马、有兵马实力、有多少军火。很多时候军阀反而拥有这一切，但都没有正统权。为什么？因为他们把山大王的旗帜一举起来，老百姓望风而逃。那崇祯殉国这件事，实际上维系住了明王朝的正统权。因为天子死于国难，这件事本身就意味着天子与国同休。他身上的所有过去——不管把他定为无能还是昏庸——一切都不重要了。当一个天子殉了国、与国同休的时候，他与这个国号就是一体化的，于是这一面旗帜就干净了。一个干净的旗帜在举起来的那一刻就有号召力。所以南明小朝廷内部那么乱，但当他把那个旗帜取出来的时候，整个大半个中国——各种各样的流寇、农民军全部都向他看齐。这种感召力与崇祯殉国密不可分。甚至后面两百多年间，中国人此起彼伏的革命，每每举起的旗帜，都是当年崇祯殉国之后所遗留下来的历史财富。

会物理消灭吗？我看悬啊，我个人看悬。为什么我们能够保持主体民族优势，别人就算有几百年的科技领先也达不到？真想知道吗？真想知道——人多啊。我也很想把这个东西说得更加高大上一点，是吧？说因为我们聪慧，因为我们有文化，因为我们胸怀宽广——我可以把它说得更高大上、更好听。但不好意思，本质上是什么？人多。这是祖先给我们留下的血条。我们的祖先留下的血条足够厚，我们的人足够多，所以即便遭到最严酷的压榨，还留有一线生机。为什么我们能够延续？确实是人多血条厚。刚才已经开始反向殖民了，这不就是优势吗？我跟你说，人多是最重要的事。我们历史上所有的盛世，都会同时伴随一件事——人口大爆发。你们不要跟我说康雍乾盛世，那个时候的所谓“人口大爆发”本身存在数据方面的严重问题。我不打算在这个时候聊这个问题，原因不是我说不清楚，而是现在不给说。但康雍乾盛世的所谓人口大爆发，本身存在大量数据造假的问题。两种造假：明王朝人口数量的极大低估是公认的。学术界对明王朝的实际人口一直有各种口径的估计，相差范围天差地远，为什么？因为明王朝自己没有统计——除了明初为了算账统了一次之后，再也没有统计过，所以数据有问题。明王朝到底有多少人，这是存疑的，都是侧面的证据不断推论。而清晚期那笔数据增加的流程也有问题，因为那个时候也没办法进行那么精细的统计。同时它也有粮食保有率等数据的问题。也就是在这几百年间，我们对人口数量的估量一直都有数据瑕疵，而且是公开的、明确的，始终都有，中间的误差值非常之大。所以我不是在谈那个。我是在谈我们之前正常的盛世来临的时候，必然伴生的是人口大爆发，并且它是重要的评点指标。如果一个时期没有这个，皇帝就不会认为自己创造了盛世——这是古典时期中国人对盛世的基本点评方式。

好，欢迎来到直播间。什么奥斯曼就因为人不多，崩了之后只剩下火鸡了？很多今天的中国人特别喜欢说人多不好，人多如何如何。历史会证明一切的。去哪旅游了？广东。广东最近是不是变活跃了？对，我之前出去旅行之前就说过这件事：有一大笔国际资金来了，专门资助这个项目。我唯一没有料到的就是，这帮人居然把集火的项目怼到我身上来了。这笔资金我知道，但会花在我身上这个事有点意外。我没有想到自己的咖位这么高。意外了，意外了。惊不惊喜？有点惊喜。意不意外？颇有点意外。好，欢迎来到直播间。今天咋了这么多人？我那个公屏的滚动速度已经导致我有点看不清楚了。嗯，明朝水平那么高，为什么明末农民起义那么多？你都知道是“末”了，你都知道是末了，国家的管理形式都快崩盘了，管都管不动了，然后你问我为什么会有农民起义？这很奇怪吗？一个王朝的末年，各种矛盾已经开始积累到如同山一般高。各种财富的两极分化，走了两百多年的时候，就算是通过最合法的手段，都已经形成了极致的财富两极分化。普通百姓的日子——你不需要有昏君的，内部的、自身的这种累积就会导致普通百姓日子很难过。所以这个时候出现什么局面？当然就是民变。然后你说为什么明末起义特别多？明末起义再多，比得上清的太平盛世吗？比不上。就这么回事。明的起义很多，但依然比不上清的康雍乾盛世来的多——太平盛世的时候，起义的数量远远多过明末乱世之中的起义。

说到土鸡，他们现在要认匈奴人当祖宗，这个很正常。真的，很正常，很正常。全世界都这样。我们才是那个例外。不能因为我们例外了，就老觉得大家都会记得自己亲爹是谁——绝大多数人群做不到的。这世界上所有人当中，能做到这一点的凤毛麟角。我们生活在这里觉得理所应当，人怎么能够拜错祖坟呢？是不是？所以看着土鸡跟笑话似的——人怎么能拜错祖坟呢？是吧？我们家那个埋祖坟的地方，山上天天都长草，长得那么高了，我每年清明的时候还要过去，一定要烧一炷香，送一盘子白切鸡过去。我也不知道啊，广东人拜祖坟的时候是不是送白切鸡？哈哈。但在世界上的其他领域当中，很难做到这一点。为什么？它需要你持续不断地记忆，需要你有稳定的生活，需要你每一代都有子孙——你以为这容易？这是最难的。要送三牲不止是鸡啊，好的好的，你们更认真。

然后，动不动就说“明亡于小冰河”，这个事儿我不是很认同。很多人不停地跟我说要强调明亡于小冰河，我不认同。为什么？小冰河又不是气象武器，它专门只针对明王朝，对周边一切都那么温柔、那么美好？它还能针对性地圈定这一块地方，其他地方都春暖花开？你逗闷子呢。大家都在小冰河，你硬说是小冰河把它弄没的？这种归因我无法认可。你不能说小冰河对它毫无影响，那肯定有影响。但咱们注意一件事：就算在小冰河环境下，只有我们受灾吗？不可能。中国南北距离多少公里来着？我不知道，数不明白。南北差距这么大，我们的南方地区，尽管那时候也遭遇了雪灾、冰灾，但原本两熟的南方地区在遭遇雪灾冰灾的情况下，也可以实现一熟啊，对不对？所以尽管说有没有极寒天气？有。但中国这么巨大的面积，不会所有地方都受灾，最后是能够拥有一定粮食的。只要在大一统王朝的正常调度之下，完全可以把这个问题基本弥合——我不敢说它好，但基本弥合、维持一个存活没有什么问题。这本来是可以做到的事情。但为什么没有呢？因为这个救命的粮食到不了大明百姓的手上，就被卖到草原上面去了。尽管国家三令五申不准出售粮食，但照样有人走私、高价把粮食贩运出去。这是气候问题吗？这是气候问题吗？
