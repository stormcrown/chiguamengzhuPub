# 第543章 论香港电影的突然崩塌，以前还以为是我们的锅

冯喜娜：  
你给自己开了个头，写了一个“冯”，哈哈，然后“喜娜”。我觉得像个德国贵族后裔似的，不是真的吧？嗯，你这个称谓我有点尴尬啊。怎么看啊？好啊，你就说香港这个事是吧。有人看过当年香港的电影没有？有看过那边电影的人，就会有一个认知。为什么要强调这个事儿？应该是90年代，香港电影做得最好的一个阶段，非常非常繁盛。那么那个阶段的电影，你去好好看一下，你会发现，绝大多数当时的电影人，那些主创，绝大多数都很爱国，知道吧？你去看那个时代——它很繁盛的时代，高峰期出现在90年代，但是它的酝酿期、成长期是从60年代就开始了的。整个中间的这30年，是香港那边的电影行业逐渐扎根、逐渐生长繁盛的过程。在这个过程当中，那个时代的主创人员，绝大多数都非常爱国，非常有民族意识。  

然后后来就变了，后来就变了。实际上从90年代开始，就开始出现了不好的风气——就是它最繁盛、获得的资本量也最大的那个时候，不再是一个本土的电影行业，而变成了所谓的国际化的电影产业。然后资金体量也非常大，内部的人员也变得非常复杂，盈利的路子也变得越来越多了。从那个时候开始，里面就开始鱼龙混杂了。但是早期的香港那边的电影人，普遍性地会比较爱国。那么为什么像刚才这个付费提问里说的，是不是因为他们没有接受良好的我们的爱国主义教育？很遗憾，返回来去看这件事情，实际上在上世纪的那一拨人，他们的成长期当中，根本就没有机会接受我们这种正常的爱国主义教育。他们当时是在港英的统治之下的那个时代，是在港英统治之下的，但是绝大多数人都非常爱国，非常有民族意识。  

之后，港英看上去撤了，港英看上去撤了，但是在那之后成长起来的人就变得每况愈下。为什么？因为教育不只有课堂，还有现实的生活。教育不全都是来自于课堂的老师的教育，它还来自于生活的方方面面，来自于你的体验感。在港英治下的时候，所有的港人确确实实感知得到自己不是正常的普通公民，他每一天都感知得到来自于殖民者的残酷的蔑视和压榨。所以他会萌生自我意识，他会非常清楚“我是谁”，他有机会的时候就会出来讲“我是谁”，“我要做什么”，“我有什么期待”。所以那个时代，他的教育不见得好到哪里去——真的，港英治下，你以为他不审核教材的吗？所以那个时代他们的教育也并不会好到哪里去。但是人普遍性地会有爱国的精神，就是因为他知道敌人是谁，知道敌人长什么样子，也知道在街上面对他吆五喝六的那些人和他不一样。这种感知就是用社会氛围来教育了一代人起来。  

但是为什么到90年代之后，他反而不好了呢？那就是港英把自己的后手都留下来了。他侵扰了你的司法界，侵扰了你的影视圈，然后侵扰了务工、务工产业。注意，这个事儿不是瞎说的，务工劳务就是港英在撤离之前所埋下的巨雷。当中还包括开放了一大批的新移民，引入数量上面应该有数十万之多——数十万从东南亚地区引入的新移民。这些人有自己的自我认知，并且跟港人之间格格不入，在文化上、在历史渊源上完全不同，但是大家变成了在同一口锅里面抢饭吃的人。所以就出现了所谓的内部的对峙，就进入到了这样的一个阶段。同时呢，当然还有教育上面的种种东西在埋雷——教育的、传媒的、司法的，各式各样的行业里面，他都给你埋下了这些根子。但是他走的时候又表示了非常的慷慨大度，于是就相当于之前的仇怨结算了。家里面不会再说当年的港督干了什么狗娘养的事情，但是，现实生活当中，你再也看不见白人过来拿着个警棍来敲你的头这种场景了。没有社会氛围的教育，于是他的教育就来自于传媒和课堂，还有身边的朋友们。那么请问，他给你引进来的人啊，课堂教的是他们的课本，传媒是他们的老板投的资。当然也不全都是，不全都都是港英相关的这些人，还有一部分就是当年走到这个地方来的小众群体，但是很有钱。在那之前他们不敢做这些事情，而在这个时候他们就敢了。为什么呢？我们国内在那个时候也是这个死相的啊。所以那个时期，他确确实实导致整个的教育、传媒、司法等等这些关键的行业当中，香港那里都出了问题。  

为什么后来就变得情况越来越糟糕？就跟我刚才所说的那几个要点都是可以直接相关的。其中，有一个很重要的关键点，我不知道有多少人注意到了这个事儿——就是前些年他们整活儿的时候，站到队列最前面、和境外势力关联最紧密的那些小团伙、小头目们，大部分都是当年港英从东南亚那里招募过来的劳工，而不是之前真正本土生长的、一代又一代的港人。注意一件事，就是那个小英，我们都知道小英不是个好东西，这个事可以公认吧？啊，小英不是好东西，这个事完全可以公认。但是呢，小英的这个技术很好，能理解吧？啊哈哈，小英的技术是非常好的。你要能够理解他的技术，也要知道他到底干了什么狗屁倒灶的事情，你才知道应该如何应对，而不是以不变应万变。你不能理解对方为什么这样操作的时候，很多时候你的应对就会失措。  

为什么小英到每一个地方去都会搅事？如果一个地方原本的人就有多族裔，他就会给其中的一个较少的族裔以特权，让他们骑在较多人的脖梗子上面拉屎。然后等他撤的时候，基本盘一定会复仇，必然会复仇，然后这个地方就烂掉了——它不烂是不可能的，而且不复仇也是不可能的。谢谢杰哥礼物啊。而且不复仇也是不可能的，因为当年那帮子人当小英走狗的时候确实也没干过好事。啊，能不能不动手吗？所以必须要动。但是只要你一动，你就山河破碎。这个就是小英搅屎棍啊，那不是瞎开玩笑的。  

然后呢，如果说这个地方没有多族裔，那怎么办呢？两招啊。比较简单的一招是：你这个地方没有多族裔是吧？但是你总跟你家隔壁的什么小群体曾经有过仇怨。毕竟嘛，大家住得这么近，难免祖上有仇，很正常的事情。他就跑到那些跟你有点仇怨的部落里面，牵一帮子人过来，骑你脖颈子上面拉屎。他在那个小免（缅甸？）附近啊，小免附近啊，他就这么整起来的啊。他搞一堆人，挪腾乾坤大挪移，然后挪过去之后，你俩本来就有仇，他又专门搞了一堆人过来给你当人上人。然后等到小英咔嚓一走，你不复仇吗？啊，换成谁你都必须要做这件事情啊，但是你只要复仇了，一地稀碎，一地稀碎啊。就这么点事。  

然后如果这还是你周边有仇家是吧，你看他又可以利用了。那还有其他的情况：你说我这个地方又没有复杂的族裔关系，然后呢，旁边部落也没有这么深仇大怨的啊，你也找不出这样的一个角色过来迁徙。这个时候怎么办呢？你既然内部没有族裔之分，我就帮你创造一个族裔之分啊。他就按照某一种给你安排一个特点——安排的啥特点我也忘了，总之呢，就给你安排某一种特点。然后你喜欢这个的呢，是一拨人；喜欢那个的呢，是另一拨人。直接就给你分开。分开了之后，他就给其中的一群人某一种权利，给另一群人另一种权利。然后你等着吧，那不掐起来才怪的呢？能不掐吗？必须掐呀，知道吧？  

所以那个小英啊，小英他这个手法我们要了解。了解了之后，就要知道它的运作的原理。运作的原理就是身份的标签，以及身份标签之后配套的社会权益。它上面如果有不平衡，这叫什么？这叫做“物不平则鸣”。很多时候，很多人觉得就是因为分化就不好。今天中国国内互联网上就有人啊，分化了就是不好打了——那个啥说大家不一样就是不好，你必须得说是什么什么的，必须得说是团结的，必须得说是兄弟。不能说我们以前有仇怨，这些东西都是错的啊。这个就是归因的错误。注意了，你对一件事情的归因如果是错误的，那么解决方式就不可能是正确的。  

因为小英的这套操作的手法，你看上去他是把人划分了不同族裔，或者强调了你是什么族裔、他是什么族裔，你就以为他是在做分化。NO啊！很多的地方原本就有很多族裔，人家咋没事儿呢？嗯？是不是人家本来就有很多族裔也没事啊？刚才说那个小英的这个技术，他的技术好，同时你要了解他才能够对抗它。那么对抗它的过程当中，你就得分析他到底干了些什么。所以呢，不是像今天很多在我们的传媒平台上面不断说的啊——你说我们有多少多少个族，所以你就要如何如何了啊；你说谁谁谁祖上曾经干过什么糟心事，你就要如何如何分裂了。这种就是一个偷换概念或者是错误归因。有的呢，他是比较笨，所以叫做错误归因；有的是比较坏，他是故意的错误归因。  

真实的原因是什么？小英的这种贴标签的伎俩，他到底生效在什么地方？不是生效于把人拉进来，不是生效于在这个地方出现了另一群人。他真正让他生效的东西，是给其中有罪的少数的、然后一些有问题的人群以特权。注意哈，他是给这些有问题的少数的、和当地人格格不入的、甚至有点反动的群体以特权，实现了一个效应：就是付出越少、收入越大。这就是权责不对应，这是他的根本问题啊。这种手法它的根本触动点在于发放特权啊。注意了啊，所以这种手法你得知道他为什么生效。当不知道为什么生效的时候，其他的所有的猜想都没用。你说不管来了什么人，你必须得说他就是中国人，你必须得承认他就是爱中国，不然的话，那个你是不是要像小英当年干的事情一样当搅屎棍？别闹了。谁来谁就是中国人，那才是小英干的事情。你设一个门槛，让自己的人啊，越是忠诚于自己的，越是自己的基本盘，获得的待遇越好——这是什么？这是权责相对应。谁都说不出来什么，然后内部就会形成一个良性的循环机制：越是效忠啊，越是对这个国家爱得真挚，越是对这个国家付出的成本高，他获得的待遇就越好。那么这个时候这个国家就可以团结一致。但是如果付出的越少，获得的越多，越是新来的获得的好处越大，那就是当年小英整的这个活儿啊。  

哎，好，欢迎来到直播间啊。来来来，我们带这个吧。啊，我努力的先把这个国史大纲先推了啊。你们刚才谁说国史大纲不能买的？我再给你们瞅一眼哈。来来来，国史大纲。嗯啊，这本书呢，我建议每个人都收藏一套啊。现在呢，是新版新发行，然后整体来说，它就是印的东西都还不错啊。我搁哪儿来着？是在我架子上呢？是在呀？完了我又摸不着它了。算了算了，摸不着就不摸了。那，呵呵呵。按国史大纲的这个书呢，确确实实值得收藏一套，因为它是一个中华简史啊。了解中华简史啊，对于一个人来说呢，是非常重要的。就是它是什么？它是一群人的集体记忆。一个人要有记忆，知道自己从哪来，才知道今天要做什么。对于一群人来说也是这样的。一群人的共同的记忆、共同的认知，缔造了我们共同的思想，也缔造了最后我们组织起来的方式，最终呢，会影响到我们去选择的道路。知来处才知去处。而国史大纲就正好是这样的一本书。那么写这样类型通史的人不少，为什么要格外的去推送国史大纲？是因为它是在一个国家最危险的时刻，为了救亡图存而写出来的一本书，所以他的立场难能可贵。
