# 第575章 团结壬持续发力，对日还有中间派？

我认同“我们不反对日本人，反对日本军国主义”这句话。但这句话真正的问题在于，当你认同它之后，常常会误以为大部分日本人都是普通人，只有极少数是军国主义者。请注意，在我的眼里，大部分日本人都是军国主义者，极少数才是真正的日本人民。我同意这一点，我只觉得日本人民占比太低，军国主义者占比太高了。

大家好，欢迎来到直播间。这句话经常被人拿来给日本军国主义洗地，对吧？这是一句好话，但也会被坏人利用。被利用之后，就变成了洗地的工具，导致大家对这句话产生成见。但实际上，这句话本身存在定义权的问题。也就是说，当你说日本人民和日本军国主义不一样时，你需要定义什么叫做“人民”，什么叫做“军国主义者”。如果这个定义权不在我们手上，而在日本人手上，那你想，日本人说他们所有人都是人民，你就没法反对他们。我们必须把人民与军国主义者的区别定义权拿在自己手上，这句话对我们才有价值。

我可以接受这句话，但什么是人民？必须明确、坚定地反对军国主义行为，为此感到羞耻，并愿意赎罪的人，才叫日本人民。其他那些默认的、占了便宜不吐的、悄悄支持的、进入兵工厂的、为侵略行为洗地的所有人，都叫军国主义者。当定义权掌握在我们手上时，你会发现人民和军国主义者之间是有区别的：那些反对侵华战争、自证清白、表示与军国主义势不两立的人，属于日本人民；而其他那些觉得“不是我、与我无关”的人，就是军国主义者。这和某些人的定义标准怎么那么像？因为我可以这么说。

我盛情推荐一个景点，虽然我也觉得花钱去看这种地方有点糟心，但我依然盛情推荐：吉林省长春市的伪满皇宫。门票好像是60还是多少，我忘了。有兴趣的小伙伴可以去看一看。伪满皇宫这个建筑群还不错，虽然有点小家子气，但建筑质量挺好。参观时，你要看它里面的一部分文献展出，一定要上心。在文献展示区里，你能看到非常明确的宣言——日满一体。这不是别人对他们的形容词，而是他们自己发出的宣誓。所以他们什么行为模式、分类模式、自我认知模式、洗地模式都高度一致，和官方文件一模一样。人家早在上百年就已经喊过口号了：日满一体。那就让他们一起去，我们也一体计算。哎，太麻烦了，一锅烩算了。老师想省力吗？谁不想省力呢？

好，欢迎来到直播间。那具体怎么操作呢？不能把中间派都丢掉啊。我们不承认中间派，就那种反动的群体，承认什么中间派？没有中间派。你主动站出来和军国主义者割席，主动认罪伏法，承认以前的罪过，那就叫人民；不站出来的就叫军国主义者。区分方式非常简单：你要是自己不主动出来，就是想要含糊其辞，就是想要洗地，就是军国主义者。这话有什么可说的呢？不用聊这个事儿。还识别中立的？没有中立的，没有中间派，不存在这种事。你上哪找中间派去？还骑个墙？谁输了我就跟谁割席，谁赢了我就站哪一边？没这事儿。不存在中间派，人世间没有中间派这个东西。甚至于大家记得吧，前几年是不是老有人说我们要理智、要中立、要客观？现在已经简称为“理中客”了。理不理智、客不客观这事儿确实不好说，但中不中立这事儿保证不中立。人世间的立场问题是没有中立的，知道吗？立场问题没有中立，立场问题没有中间派。中间派是什么？中间派就是叛徒。中间派只有双料叛徒。无论站在哪一边看，中间派都是叛徒。如果这个人本质上是我们这一边的，他又是中间派，他就是我们这边的叛徒，他投降了，知道吧？啊，那个，日军发动侵华战争，一个中国人站出来说我中立，他什么意思？他要叛国呗，那能什么意思呀？对方要侵华，你说你中立，你就是叛国者，你就是站到了对面。这个事儿之前他们用那个什么机器猫打比方时，我觉得还是蛮合适的。就用那个《哆啦A梦》打比方：当胖虎暴揍大雄时，你说我中立，你就是站在胖虎那边。好的好的好的。

猪八戒照镜子——里外不是人。我记得是什么时候来着？嗯，就是倭寇侵华期间，中国有大量的汉奸群体。实际上当时的日本学者、日本文人也在对中国人进行分类分析，因为他要了解中国的情况，好推行他的侵略政策。所以这个期间也有一些很值得我们去了解的研究，当年他们的一些学术研究，其中就包括了他们对汉奸群体和伪军群体的分析。日本人在这个领域中的分析其实非常精到，因为他的侵华战争要大用特用的就是这一批人，所以他迫切地需要了解这些人在干什么、在想什么。因此，日本人在这些人身上动的脑筋也确确实实很多。那么，他们是如何分析这些汉奸、走狗、伪军帮凶的呢？道理上其实不复杂，非常简单：所谓的汉奸，就是一群没有自我、也没有立场的人。在他们看来，任何事情发生了，只要能获利，哪里让他获利他就到哪个地方去，其他事情一概不理会，也理解不了，甚至没有办法意识到自己其实是中国人。这就是汉奸的一种根本属性。

所以，尽管今天的中国军事非常强盛，大家不太容易看到那种目测的狗汉奸——就是非常醒目的、要为太君引路的汉奸，数量非常少。张牙舞爪跳起脚来就要为太君引路那种玩意儿，早十年还有，现在的话他们都不敢吭声了。包括当年非常明确、动不动就说他家太君怎么怎么样的那些人，现在都装得老爱国了。所以现在汉奸已经不是明面上的了。这时候你怎么识别它呢？识别的方式就是：如果你发现他有那种没有本我立场的慕强心态，他当汉奸的概率就要大得多。他爱国不是因为自古以来就是中国人，而是因为中国很强大所以爱国，因为今日之富饶所以爱国，因为今日之强大所以爱国，因为我们必将星辰大海所以爱国。这些人一旦哪天你不够强大了，他就会变身为汉奸。
