# 第516章 如何评价二流小报爱当爹

这些日子熬这些时候，我要实现的目标就是想下班的时候就下班，想休息的时候就休息，不然的话我不是白干活了吗？嗯，能聊一下系统性崇洋的风气吗？在新一代自信的人上来前，我们只能看他们胡来。咳，这话太有针对性了啊。你针对谁了？前面的那个什么系统性崇洋媚外的这种风气啊。我就觉得你这个提问不好，有点针对我们现在主流思想的意思，你知道吧，能听懂吧。啊，要知道，某二流日报专门批判了中国人的这种思想，觉得老外不行，这个东西叫做媚外。你按这种标准来算的话，那怎么才能够不媚外呢？啊，那就是得觉得老外特别棒啊，刚刚好，那才叫做自信嘛。这事儿能做到不？反正我做不到。哈哈哈哈，你们谁能做到谁去做啊，反正我是做不到这个事的。呵呵。

现在我们的很多这种崇洋的姿态，打哪儿来的呢？啊，它其实从清的时候开始培养出来的苗头啊。注意，这个玩意呢，是从清开始出现的。为什么？因为媚外的这件事情是很奇怪的。我之前做直播的时候聊过这件事情：媚外啊，就是崇拜他者，觉得他者特别了不起，他者特别漂亮、特别伟大的，这件事情本身是违背了人类的生存本能的。它和什么谁强谁如何如何，哎，不是啊，它是违背了人类的生存本能的。人类的生存本能就是因为我们有思考的能力，所以经常容易怀疑自己生存的意义，容易产生虚无感，所以人类的生存本能当中包含了一个技术补丁，就是找理由甩锅，最终目标是证明我很好，我活着很有价值，我做的一切事情都是棒棒的呀。这个是人类的基因底层代码，所有的人都是这样的。你去观察中国以外的人的时候，就会发现有普遍的这样一种心态：无论他的国家是穷是富，社会排行到底能排到第几，甚至内部的秩序是不是崩坏，文化的水平是高是低，他都不在乎，反正他就觉得他自己杠杠好啊。这个呢，才是人类自然而然的心态。这种心态你可以喜欢，也可以不喜欢，但是它就是人类的本能，它是一种自然生态。所以呢，你就知道了，崇洋媚外的这种思路本身，它根本不是人类社会会自然诞生的。它要出现，第一件事情就是要进行自我贬低，也就是自我奴才化、自我矮化。

那么中国人是从什么时候开始自我矮化的呢？从清啊。因为八旗老爷骑在所有中国人的脖颈子上面拉屎，然后呢，还不准你反抗。当时的八旗，如果在巡视过程当中看到了某一个绿营兵，就可以直接问对方：“你吃的谁的粮食？”那么绿营兵必须要第一时间回答：“我吃的是陛下的粮食。”“你忠于谁？”他必须立刻回答：“我忠于陛下。”如果答的速度不够快，答得不够干脆、不够利落，直接嘎掉了，是不用负担责任、不用负担后果的。这个是什么？这个就是一个训狗实验，长时间的条件反射式的训练。所以你想一想，连给他们相当于当府兵的人，都是这样的态度，你说普通的老百姓呢？那是日常情况下就受到各种各样明确的打击。像今天很多人说中国人的那个陋俗，动不动见人就磕头，奴性十足，什么时候来的？清来的。这是一个长时间的强力的贬低，通过仪式化的方式进行驯化，从你的着装、外表、行为方式、交税的方式、服从的对象开始进行的驯化。这种长时间漫长岁月当中的自我矮化，转变到了清末的时候，变成了另外的一种姿态：就是他从服从八旗老爷，转变成了服从洋老爷。虽然这不是所有的中国人，但确确实实有一部分中国人是这样的，而这一部分人又刚刚好是这个社会上面相对来说文化水平较为良好的一批，是受训程度最深，也同样的是受洋人威压最甚的一帮子人。于是你就会发现，在中国的土地上面出现了一幅奇景：老百姓是不怕洋人的，敢冲上去拎着镐头干的，但是官人们怕；洋人们是怕百姓的，但是呢，他不怕官僚。所以这些服从，这些被摁着脑袋磕头的行为，本身就是自我矮化的一个开端。之后发生了什么事，大家就心知肚明了啊。

那么媚外的这种情绪是由谁来带领的呢？放心的一点就是，普通的百姓并没有媚外。实际上一直到清，一直到民国，普通的百姓看到洋人的时候，依然觉得对方长得很丑，是个鬼子。包括说现在啊，现在在南方的一些省份当中，依然会将其称为某某鬼或者是某某夷啊，这些词汇依然在使用，鬼佬。哎，对对对。那这种其实呢，你说啊，这涉及到歧视、盲目排外，不要扯淡了。然后说什么啊，觉得别人很丑这件事情是自己没眼光，也不要扯淡了。这也不是什么自我安慰，这个东西是一种正常的生物本能啊。任何一个生物群体优先的都是觉得自己好看，而别人丑，所有的群体都是这样的，天然地认为自己的这个群体比较漂亮，对方的群体长得和我不一样的，很难看。中国人自古以来都是这样操作。所以那些经常吹嘘的啊，古典时期的那个大秦，后来呢，被贴金贴到了古罗马身上，实际上没有提过古罗马，就是那个大秦国。那么我们古典时期是怎么称赞大秦国的？尽管我们没有人见过他，但是呢，我们听说那个地方很好，于是对他的描述叫做“当地的人长得高大平正，有类中华”。就是说我觉得对方长得挺不错啊，挺体面。什么意思呢？就是他长得很像我啊。像我的就是好，不像我的就是难看。包括说到了唐的时候，得道高僧玄奘法师，嗯，私逃出国啊，他可不是公派的，他私逃出国，跑到了印度地区，搞了一波私自留学，民间留学啊。那他呢，回程的时候被迫写了一个，被迫口述，出现了一本《大唐西域记》，讲解自己路上所看到的风土人情啊。谢谢 win 啊，long bowing in。然后那个他被迫口述去写了这个风土人情类的书籍，在这个里面，他也是对一些人描述的好看的时候，就叫做“有类中华”，跟我差不多啊，跟我一样好看。如果说看到别人啊，长得不好看，就直接嫌弃鄙陋。所以这是啥？这是正常的人类，他在看到他者的时候，常态就是对方长得不好看。而这种心态在中国的民间其实是普遍存在的，反而是在精英阶层当中会非常的媚外，觉得洋装很好看，洋美人非常的漂亮。这种东西在民间不受接纳，但是在社会的准精英阶层非常受到接纳。

那么什么样的东西更加能够引发人的崇洋的心态呢？你就说嘛，后来我们建国了。其实我们建国之后，崇洋的心态就已经大打折扣啊，还是有这种人，但是已经变得非常少了，已经非常的非主流了。那个时候谁要是崇洋，那是被所有的人瞧不起的：怎么着，你自己建设国家不会建设吗？你自己干活不会干吗？直起腰杆子对你来说那么难吗？非得要出去磕头拜老爷吗？当时的中国人非常有革命意识，所以非常排斥这种崇洋的心态。如果觉得对方好，你学就是了呀，想方设法的追上就是了呀，为什么要崇洋媚外呢？那个时代的中国人就是这样的一种思路。实际上我们做到了没有？也做到了。就是在当年的这些机械类的企业当中，我们有很多的装备，有很多的设备自己做不出来，真的是洋人做的特别好。那个时候我们的很多工人、工程师，这些技术的老师傅们，看着别人的设备，看着别人的装备，要流口水、掉眼泪的。但是解决的方式是崇洋媚外吗？不是。解决的方法是我们斥重金去买一台回来，然后把它拆了研究做复刻。如果做了之后还是不行，那就再买一台回来再研究，最终一直做到我们也可以拥有。所以那个时候的人非常清楚，外国人有他先进的东西，也非常明确的看得到外国人的先进之处，可是一点儿都不崇洋，我们只有昂扬的追求之心。

那么今天的崇洋的心态是从什么时候开始的？是从我们觉得我们就不行啊，时代变了。后来啊，后来就开始那个啥了，哎呀，从上世纪80年代之后啊，我们国内的很多的这些工厂开始认为呢，我们自己做的这个设备呢，质量不行，当然确实当时还没有追上洋人的技术的水平嘛，然后就说我们这个东西做的不行，然后造呢，就不如买，买呢还不如租。那既然这样的话，我们就永远都做不出来。既然是一个永远都做不出来的东西，既然别人永远都摁在我们上面，既然别人永远都比我们先进，那你能不崇洋吗？然后同一时间也是这个时期啊，包括电影、电视剧、杂志、报刊这些地方，开始吹嘘混血儿的聪明又漂亮，开始刊登这些人的照片、履历，开始赞美他们的颜值啊，甚至呢，把一些典型的有混血脸的人，请到央视的舞台上面啊，去宣传混血的优胜感啊。所以打那之后，很多的人去夸耀别人家的孩子，长得好看的时候就说：“哎呀，这小孩儿长得像个混血似的。”爹妈都是中国人，他夸孩子长得像个混血，但是呢，他是真诚的，因为那个时候的通用的话术就是：你长得像混血，你长得不像中国人，你就很漂亮。其实这话的潜台词就是说中国人很丑，只要不像中国人，你就漂亮了。当然了，放到今天的语境当中，你就能够get到，其实这种赞美里面是包含了浓重的贬低的意味的。可是当时的人并不这样想，也压根想不起来。一个人长得像洋人，她就是漂亮的。那这个是什么？灌输进来呢，是一种流行语境给我们打下的思想烙印。而这种流行的语境是通过无数的电影、电视剧、报刊、杂志给我们传输的。这种载体放到今天就叫做营销号啊。今天大家的很多的东西，就是由各种各样的营销号来灌输的，很多的想法呢，也是从这里面窜出来的。嗯，好，欢迎大家来到直播间。混血也有一般的，这是什么呢？这个，这个概念啊，这个概念，哎，谢谢。这里头就是什么？我之前讲过混血的问题。混血就是远缘杂交啊。这种情形呢，在自然界普遍的存在，所以呢，在我们的整个的杂交育种界，也是普遍的存在。它是有基本的科学规律的。这种科学规律，即使我们是人，万物之灵长，你也跳不过去。因为这玩意写在DNA里面，这是属于自然科学领域，而自然科学领域不以人的意志为转移。你生下来是什么样，就是什么样，基因里面怎么写的，那就是怎么写的。所以不是因为你说了混血聪明又漂亮，于是混血就会聪明又漂亮，他不以你的许愿为主，他又不是王八啊，许愿他不听。所以他会怎么表达？远缘杂交的特点，就是偶尔会出现极优胜的个体，这个叫做杂交优势。我们之所以要进行育种，就是因为有极个别的优势个体，我们想要，这就叫做杂交优势。你在之前的正常的繁育过程当中，不会出现这种占优势，所以它是有优势的。但是杂交优势第一出现的概率非常非常非常低，要撞大运。第二个呢，就是杂交优势产生的个体非常非常非常的不稳定，一旦到了第二代、第三代的时候一定会流散，而且越散越狠，这个这个这个叫做性状分离现象。所以远缘杂交有没有偶尔出现好的概率？有啊。但是大部分是什么呢？大部分就是奇奇怪怪的歪瓜裂枣啊。

为什么会这样？其实哪怕他长得平平无奇的占大多数，但是这个平平无奇和普通人的平平无奇不一样。我们这种一般人的这种平平无奇的脸，你扔到街上之后你找不着，但是他那种平平无奇的脸放街上之后，你会感觉到很别扭，你会感觉到很奇怪哈，感觉到很违和。为什么？他是把不同的东西拼起来的，他有一种拼接感。你如果有兴趣的话，可以关注这些关键词，然后到那个各个视频平台上去刷。近几年呢，有一些混血儿啊，在中国国内长大了，然后他们会不停地去拍他们自己的视频，然后说我是混血，我怎么怎么地。你有兴趣你就去刷啊，多刷几个你就刷到概率了。你就会发现，哪怕是大部分的那些长得并不出彩，但是也并不很丑的，那些平平无奇的普通孩子啊，有着强烈的违和感，就就觉得哪个地方拼的不对劲啊，有些东西拼不到一起去，就是这种状态。所以大部分情况下都是这种充满了违和感的平平无奇，然后呢，同时还有大量的就是歪瓜裂枣。而比较惨淡的那种啊，就是拼的实在太糟糕了，理解不下来。然后呢，就是极个别的很优秀的，确实把好处都长在一起了。但是这种好处稳定吗？不稳定啊，随时随地他都会削减，因为会出现性状分离，到下一代的时候，会非常严重的性状分离。这个是自然科学，不以人的意志为转移。你不管你爱不爱、喜不喜欢他，都是这个样子的。

所以为什么在人类群体当中不适用于育种计划？因为我们在自然界当中去寻找优势个体，去寻找远缘杂交的杂交优势的时候，是要淘汰掉杂交劣势和平平无奇的东西的。但是在人类的幼崽当中，你不能这样干。我们是有人性的人，无论生下来的是什么样的孩子，那都是我们的孩子，我们都得留着他。于是就会造成种种种种的问题。至于说这个问题，哭了的话会变成什么？倒也有样本啊。就是到处都是这样的情形的时候，会发生什么局面？请看拉美啊。不要看拉美的那个选美冠军啊，我说了，他们会有极个别的优势个体。你去拉美去看他的普通人，拉美的普通人，就是那种大面积混血之后产生的局面，就那样啊。
