# 第501章 世纪大诈骗，摊丁入亩是德政？

嗯，最近去旅游了吗？  
对，去旅游了，刚刚回来。  

怎么评价“摊丁入亩”和“官绅一体纳粮”这个事？  
我是谈过的。摊丁入亩、官绅一体纳粮，都是谈过的。  

摊丁入亩是什么？丁是什么？丁是丁瑶银。丁瑶银是什么呢？是一种徭役税。徭役是什么呢？徭役是一种劳动税收。  

嗯，我们为什么会收像劳役这样的税？是因为压迫吗？不是，是因为我们是一个生产型的国家。生产型的国家，就是劳动多、产品多、货币少。所以在征税的时候，会进行实物征税和劳役征税。这些呢，都是自古以来的税收。包括像人头税这种，也约等于是劳役收税。约等于他为什么要按人头来收税，这个事咱们要能弄清楚它的基本原理。不然的话，你光觉得他是不是在控制人，他是不是在盘剥勒索。  

注意一件事情：那个生产型的国家，它有一些很基础的要求。什么样的要求？大家现在都生活在新中国，所有人从出生了之后会登记户口。登记完了户口之后呢，该到了幼儿园的时候，是不是有人通知你，哎，我们这儿有什么公办幼儿园，你得上啊。啊，然后到了上小学的时候，那就是你不上小学的话，会有人登门的。你必须要上小学。这是什么？这是义务教育。那为什么要有义务教育？因为我们需要很多的生产者和劳动者。你接受了基础的和标准的一体化的教育之后呢，就能够更好地参与到一体化的劳动当中去，能够让整个国家进行整体调度。对于我们这样大一统的生产型国家来说，这一种系统安排非常的重要。  

所以呢，古人所收的这一套所谓的人头税——你刚生下来的娃娃，你凭什么要收这个税？这是干什么？这实际上就是我们自古以来就是有这种社会安置，有基础教育的。尽管说很多人肯定说怎么可能的呢，不好意思啊，确实有。实际上到汉代的时候，就已经形成了强制教育了，是属于你必须去。这个教育水平不高，不是为了把你养成之乎者也，也是为了让你成为一个劳动者，合格的劳动者能够看懂——就是我给你发什么通知，要求你做什么事的时候，你能看得明白。就这样的一种教育水准。所以呢，它确实是存在社会公共福利的。尽管这个福利跟我们今天比起来看上去很惨淡，但确实存在社会公共福利。甚至包括了啊，公共道路的建设，也是自古以来就有。今天我们也有基础建设啊。生产型的国家，你是必须要有基础建设配套的。我们的古人在这个方面也有。  

那有了之后，你既然上头花了，那下面要怎么计算这笔账呢？所以他就会进行这样的税务征收。在征收上面默认的每一个人都享受到了国家的福利，所以他就每一个人按人来收一笔钱。  

刚才在讲到我们的这个纳税的问题啊，我讲的是他的收税的征税的基本原理。我们自古以来所征的税呢，基本上都是实物税。就是因为我们是一个生产型的国家。生产型的国家，每一个家庭是最小的生产单位，所以呢，它是没有真正的货币生产的。它产生出来的东西呢，要么是丝绸，要么是布料，要么呢，就是包括柴草。注意了，草、柴这些东西都是可以拿来缴税的。在早期，为什么这些都是国家公共建设、公共生产和军队供养、军马等等都需要的啊。我们是实物征税、实物消费，因为这些东西都能用得上。包括说家里面那个什么皮料啊，哪个东西皮革，那也是要拿来征税的。这些东西全部都是可以当做税收的。粮食那是更基准的税收啊。所以我们自古以来都有这些税收。  

同时呢，还有一项税收就是劳役啊，就是你的劳动。这种公共劳动，例如说挖水渠、修城墙啊，这些呢，属于公共劳动，它也是一种税收。我们之所以今天不太容易理解这一切，是因为我们现在已经货币化了啊。货币化了之后，我们所有的东西都被数字给隐形起来了。它被数字给埋起来了之后，你就常常觉得他为什么要让人去干活呢？你不干活你要交钱啊，他交不起啊。我们是一个生产单位的时候，你让我去干点活，这个事儿我是干得起的，但是你让我交钱，我没有这个东西，我怎么给你交呢？是不是。所以呢，我们自古以来征税，绝大多数时候都是实物征税。到什么时候发生了一个非常彻底的改变呢？基本上是在明代的时候啊，这个改变才进行的非常的彻底。就是全部都——几乎全部都转为了这个货币，或者说主体转为了货币。这不全都是好事，因为当它转向了货币的时候，就意味着你产生出来的东西需要一次货币交易。在这个货币交易的过程当中，就有可能遭遇到一次盘剥，然后再计算你银两的成色和纯度的时候，又会被盘剥一次。所以呢，它是在税收变革的过程当中，你不能说它很完美，但是它确实在逐渐的转型。  

然后讲到这儿的时候，就说，为什么会出现丁瑶银啊？你我没有忘记主题。为什么会出现丁瑶银？因为啊，丁徭是什么？就是每一个壮丁需要去什么修桥、铺路呀，然后给府衙修瓦片、钉门口呢，这些东西都属于公共劳动，这些玩意儿就属于徭役。但是到了这种市民化商品经济的时代，然后又货币化征税的时候，他就会置换成一种税收，变成了一个银子。你说，哎，我把钱交上去，我就不用过去了。朝那个官府呢，再去拿这个笔银子，再去雇别人去干你的这些活儿，它就变成了一个间接性的操作。这个东西就叫做丁瑶银。  

然后这个所谓的“摊丁入亩”，指的就是这个徭役的这个钱，我不收了，把这个钱呢，核算进田税里面去。核到田税里面去收，田多的多收钱，少的少收。相当于它变相的变成了一个收财产税的方式。听起来是一个蛮符合民生需求的——有钱的人有地吗？那你就多收。但是大家注意一件事，我就问你：有钱人自己种地吗？大地主自己种地吗？不种啊。大地主自己是不种田的，他的田都是要租给佃户的。然后他租给佃户的时候，因为他这一亩田上面叠加了一笔税，我就问你佃户要不要交？他要是不交的话，那就说明那个地主是个大善人呗。所以实际上他减了这笔钱吗？真的是让穷人捡的这笔钱吗？没有啊。这是第一件事。  

然后另外的一件事就是这个田税。作为这个田税，无论是粮食也好，财产税也好，总之田税这个玩意是什么呢？是国税，他直接收到中央国库里面去的。但是丁瑶这个东西是干什么的呢？他是进行社会公共劳动的啊，地方性的截留比例非常之高，这笔钱地方留存率非常的高，是拿来给地方上面来使用的，约等于进入了地税。但是你把他直接归到田税之后，一股脑的抽到国库里去了。我就问你，地税里头没有钱了之后他怎么办？关老爷，因为这个他就会自己去修门槛、修瓦片了吗？不会，他还是要雇人去做这个事的。但是呢，这个税又没收到他手上，怎么办？诶，他就得想方设法的再收第二茬，再收第三茬。至于找什么借口，每一个地方找的借口都不一样啊。  

啥？这不是算得明白税吗？锤神，你更新了吗？你不要以为你戳了我的心窝子，我就戳不着你的啊。锤神，你更新了吗？有没有码字儿？真的是跑直播间里面一趟，竟然啊，冲上来就说：你这不是会算税吗？来吧，互相伤害啊。呵呵呵呵。发生了什么？哈哈哈，什么都没发生啊，什么都没发生。  

嗯啥？嗯，我们的公屏上面，就我们的公屏上面，就不要老谈别的up了吧。我对别的up是啥样的不是很有兴趣。倒不是说不知道，而是你说我出来说一下啊，我觉得那个谁谁谁如何如何，我说了真话他难受，我说了假话我难受。我不点评。
