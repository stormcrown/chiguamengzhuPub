# 第554章 为什么狼来了才知道我是谁

您这段内容我不能全念完。来，那个啥，我们除了去合理地反应，应该抱什么心态去看待呢？最近心情很不好。来，我给你提供一点心情好的东西啊。我给你提供点心情好的东西：就是中国人呢，这几十年来由于经济的快速增长哈——经济快速增长总归是一件好事啊——但是呢，经济快速增长的同时，我们事实上却在经受着西方世界、其实包括了全球所有非中国的世界啊，也就是所谓的外国，他们的思想灌输。注意了啊，我们这几十年来一直在经历着外国对我们的思想灌输。我们内部的思想体系变得非常非常的脆弱，是一个严重欠缺自我意识的群体，常常误以为赚赚小钱钱就很好了，吃吃小饭饭就很好了，睡睡小觉觉就很舒服了。中国人这几十年在经济快速增长这个过程当中，产生了这种安逸思想。这种安逸思想其实危险就埋藏在其中。当你觉得小日子过得爽得飞起，这个样子过下去就很好，一切都不如这些小日子来得好的时候，危险就已经出现了。

那么怎么能够让这些人意识到他吃饱穿暖、和平万代的这件事情本身是一种危机呢？你直接跟他说是一点用都没有的啊。直接跟他说，那叫做一点用都没有。你告诉他说这个样子很危险，你老琢磨着人只是一个个体，很危险，你必须跟周边的人如何如何，你必须识别谁是我的同类、谁是我的敌人——他翻你180个白眼，他觉得你闲得蛋疼。亲，你有没有觉得在那之前，你这种忧心忡忡的人去跟身边的人说话的时候，他们都像看傻子一样看你啊？啊，是这样的吧。你根本就提醒不了任何人，即便你提醒的东西是对的，一点用都没有。你说的是真的，每一句话都是真的，将来必然会应验的，谁会理你？谁都不会理你。但是这件事情发生了之后，连那些平时傻吃捏睡的人都会突然之间一醒神啊——那不得了了，要出大事了！喂，他突然知道，原来他不是一个孤立的个体，原来他美好的日子是建立在全中国所有的人都在勤恳工作的情况之下，缔造的一个共同的家园。为什么他突然之间意识到这不是他一个人的事，而是所有人共同缔造的环境了呢？因为有外人要来了呀。亲，这就是没有狼来了的时候，你都不晓得家里面需要养狗啊。这就，这就完全不得了了啊。就，就好好不好？好啊。没有这些东西，没有这样的免疫刺激，你根本不会产生机体的免疫力。而没有机体的免疫力，那才是在未来几十年当中最可怕的局面。谢谢君啊。嗯，好，嗯。来啥？嗯，那个叫什么大保健的啊——啊，说我说我说话听不懂没意思啊，要说点听得懂的。出去找你能听得懂的那些。这互联网上那么多的主播，恨不得有上千万的主播，那里头得有一大半儿说的话都是你能听得懂的——来了，老弟！嗯，你去找呗，我拦着你了吗？是吧，直播间又不锁门，啥时候进来啥时候出去，来去自由的。这真的是还得教训一下我跟我说啊——“你说点听得懂的。”恕不从命啊。那啥，这两句他肯定听得懂的：欠儿欠儿的呗。我能有什么办法？

怎样才是华夏正统？很多人说辽、金、元、清也是正统，如如何反驳吧？啊，打错字了哈。看你的态度。你要是想喷回去，你就往死里头喷就完事了，知道吧。为什么要这样说？因为这些年这些东西呢，借着所谓团结的名头，已经渗透到方方面面了，属于是。你心里头知道他不对啊，人人都听起来知道膈应，但是呢，你但凡说一句反驳的话，你就开始要在互联网上销声匿迹。因为对方的声量很大的同时，他们还特别的龌龊，特别喜欢联系律师团啊，开始整起各种各样的司法控诉的勾当啊。他们就是通过这样的方式，防民之口甚于防川。就是道理明晃晃地摆在那个地方，史料也明晃晃地摆在那个地方，但是呢，他们指鹿为马，颠倒黑白，同时还不许说真话的人开口啊。这个就是典型的指鹿为马的操作。谁要说那个玩意儿其实是个鹿，他呢，就把人家物理性地从互联网上消除啊。这种事情呢，做得多了啊。还有物理消除，对，还有物理消除。线下也有这种物理消除的操作方法啊。所以呢，风险其实一直都是巨大的。既然有这样的客观的局面存在，这说明什么？革命尚未成功，同志仍需努力啊。

我们是如何形成我们自己的认知的，包括说这些——你刚才说的这一群到底属不属于正统呢？建议大家都去看一下上世纪50年代到60年代的历史教科书，和那个时期的历史点评文献啊，包括说时评、政论等等的。凡是出现在上世纪70年代晚期之前这些文章、这些左翼文章，甚至包括了民国时期的左翼学者的文章里面，那么言辞都是高度一致的啊。那么那些是不是正统的？那些不是啊。包括民国时期的左翼学者，也包括新中国30年的学者文章，众口一词的标准，就是所有的中国人默认的那个东西——不是。那中国人是这样想的，外国人怎么想的呢？啊，外国人从我们的革命导师马克思、恩格斯开始，所有这些对中国的近现代历史稍有了解的西方学者，也都不认为他们是中华正统啊。他们是什么时候成为中华正统的呢？之前我看过这个提问啊，说他们到底是啥时候成为中华正统的呢？来，我给大家一个时间节点啊：从上世纪80年代开始，他们逐渐开始成为了中华正统。在那之前从来不是啊。连他们当朝的时候，都没有人认为他们是。那个时候只有少量的文人学者在逐渐地为他们找补，但是那些学者并不受到这种公众的认可啊。就是这套学术只是拿来说一说，只是拿来记在史料里面，但是并不受到任何真正根本性的认同。但是呢，什么时候开始了？从上世纪80年代开始啊。从上世纪80年代开始，他们开始成为了正统。他们活着的时候不是，死了的时候也不是，新中国成立的时候他们也不是。什么时候成了正统的呢？啊，从80年之后，诶。好，欢迎来到直播间啊。我们如果太极端的话，怕不怕我们自己出现军国主义？你觉得什么样子叫极端？来，你觉得什么样子叫做极端？中国历史上最极端的时候是啥样啊？来，我给大家分享一下：中国历史上面最极端的样子是什么模样的？汉武帝。最极端的时候是汉武帝啊。哎，想。嗯。哎，商朝啊。商朝其实都不怎么极端，知道吧？虽然我也喊了口号“爱富豪、爱炖肉”，但是商朝其实并不极端。因为商朝的时候我们也是招纳方国的啊。谢谢太保。那个商朝的时候我们是招纳方国的，就是“方伯”。你在他的文献当中能够看到一个词叫做“方伯”。方伯和西周时候的诸侯之间有一定的相似性啊，它都是一种加盟的形式。所以即便是在商朝这种动辄就要剁两个人的状态下面，你老觉得他又野蛮啦，又血腥啦——那极不极端呢？不极端。因为如果真极端的话，他就不会招募方伯。但是即便是商朝啊，我们动不动自己都说自己极端的时代，他是招募方伯的。包括说汉武帝这样天天说他穷兵黩武的存在，他也没有灭人国民，他止亡人之祭祀，并不会灭绝种类。然后你非得生出来说，哎呦我的天呐，我们好极端！我们从来没有极端过。什么叫做极端？什么样子算作极端？脚盆子的那种变态的行为，我们会做吗？我们不会做。所以轮不上啊。自个儿是个什么状态，自己心里有点数啊。

那啥，历史有偶然性吗？如果三桂没有放清入关，会不会更好啊？会更好。但是没有什么偶然性啊，三桂肯定会放人入关的，知道吧。是吴三桂是一定会放人入关的。这并不是一个偶然行为，它是一个必然现象。因为整个的晚明时期啊，明代的中后期之后，我们的国内的文人士大夫、士绅群体、资本群体、企业家群体，其实都已经进入到卖国产业链上面了。要知道，一日卖国，终身卖国。这是什么？那就是你上了贼船是下不来的，能够理解吧？上了贼船的人，你想下来是非常困难的一件事情，根本就下不来。所以呢，当时的晚明时期的中国国内，是形成了一条卖国产业链的。当年关外的建州部——很多人说明王朝的死是因为小冰河——你以为东北那样的土地上面，内蒙那样的土地上面，外蒙那样的土地上面，小冰河对他们没影响吗？只会影响得更加惨重好吗！结果啊，明王朝这种还有大面积南方耕作土地的地方，因为小冰河粮食都快值了，粮价都开始飞涨了啊（主要是指北方地区），北方粮价飞涨，民不聊生。结果呢，建州部的粮价居然平抑下来了。他那个地方受灾只会更重，加倍加加倍好吗，指数级的提高好吗！为什么它的粮价能够平抑下来呢？因为中国国内有一大把的商人，连同市商人，甚至包括庙堂之上的高官，也并包含边军里面的守将，他们都进入到了卖国产业链条当中去。这个产业的链条是怎么进行的啊？就是他们将我们国内所生产的粮食、布匹、火器、金属啊，这些重要的国之利器，这些战略型的资源走私贩运到建州部去啊。当然也不只是他们啊，整个这一条线都卖出去。那卖出去之后，对方是一个穷山窠里头的，拿什么付账呢？对方就用他的骑兵进关劫掠，抢你们的金银细软。这些金银细软不能吃、不能穿，干什么呢？就拿来交给这些商人，变现变成什么？变成粮食，变成火器，变成他的战斗实力。然后再一次进关劫掠。当他已经成为了卖国产业链条的时候，即使吴三桂不开关，也会有别人想方设法的开关。为什么？因为关外那个是拿捏着他们叛国罪证把柄的金主爸爸。这件事情是一个必然发生的事情。问题不在建州部，问题在于明王朝的内部——君权虚空、卖国的情况已经变成了蛛网一般的产业链条。
