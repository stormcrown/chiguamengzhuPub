# 第534章 清的思想还是儒家的思想，要分清

哎呦，我的天呐！好，你来了啊。他很胖哈，你们看着了。嗯，讲一讲“君要臣死”这个话题——是不是儒家思想？不是啊。“君要臣死，臣不得不死”这句话不是儒家思想啊。它是清的思想，是清的八旗思想，而不是儒家思想。它只是借着儒家的儒学、儒生之口说出来。这句话摆明了它与儒学的基本原理是相违背的。

儒学的基本原理是啥啊？注意，儒学从开创的时候要干的事儿就叫做“学而优则仕”。这个买卖呢，就是说它要去培养社会中层管理人才。然后受过中层管理培训的人就是儒生。接受他们培训的这些人，就会成为士人，进入到社会管理阶层。这是在帮你突破阶层壁垒的一种教育机构。这就是儒学、儒生的系统。它既然是这样的系统，它所配套的就绝对不是什么“哎呀，那个什么君要臣死，臣不得不死”。你这个口号喊出来，谁还愿意去当那个臣呢？懵了是不是？所以儒学啊，它是培养社会中层管理人员的，它的纲要当中就包括了“择木而栖”。意思呢，就是你得挑个明主，不然你的主张推行不下去。这是就业的双向选择。

所以在春秋时期，孔夫子带领他的弟子周游列国，这是干啥呢？到每一个国家他都是去投他的简历，然后呢，去给弟子，也给自己介绍工作、安排岗位。这就是他作为一个中层管理人员培训班，学的差不多了之后，不得安排一下就业，搞一下就业对口推荐和签约吗？孔夫子当年干的就是这个活。所以这样的人他不可能让自己的什么学生践行“君要臣死，臣不得不死”。那到时候他学生没了呀！开玩笑的呢，这不是砸人家锅呢吗？所以做不到哈。

他实际上做的是啥？实际上做的呢，就是在儒学体系内，你就别说“君要臣死”这种事儿了。就是遇到了什么杀掉纣王这种事情，你说还叛乱？那天哪不得了了！然后在儒学的内部，这个东西叫什么？不闻弑君，而闻诛一夫纣。他说这玩意儿他已经是那样的一个暴君了，那就是杀掉了一个匹夫，名为纣，而不是弑君。所以他可以直接说这件事情。所以在儒学的理论当中，是存在着对于暴君的抗争、革命和推翻，这变成了它的一个合理性的符号。所以完全没有我刚才所说的什么“君要臣死，臣不得不死，父要子亡，子不得不亡”，没有这个东西啊！从来没有，中国的历史上就没有这个玩意儿，中国人就不信这一套。

包括后面父子的关系也是这样的。在儒学体系内，虽然非常强调孝道，但是呢，这个东西权利和责任都是相互的，它们都是相互的。然后父慈子孝它们是成双成对出现的，所以叫做“小杖则受，大杖则走”。就是如果他只是教训教训你，忍忍就算了，毕竟是亲爹。但是如果他真的要伤你的性命，你赶紧撒丫子跑路。他都不是慈父了，你还当什么孝子？这个东西呢，就是儒学的一些基本伦理概念。

那么你刚才说的这种东西啥时候出现的呢？是到了清朝。清朝是一个典型的奴隶制管理时期。奴隶制要求的就是绝不能背叛啊，什么理由都不可以背叛。你必须抛下你自己的一切，但是你的主子就是你的一切。如果说啊，你说的这个道理不对，你的主子是个坏人，那你就要为你的主子自圆其说。所以你会发现，儒学到了清的时候就变成了狗儒。为啥？因为它的重心就不再是儒了。儒是研究人世间的学问、人和人的关系的。但是进入了奴隶社会完全的奴役阶段的时候，它要适应这种奴役，就开始改变自己的说辞。最终它变成了皇权的依附者，纯粹的鹰犬，它变成了狗。所以狗儒的重点不是儒，儒只是它的外皮，是自己的画皮，装扮的扮相是它的行头。它的内里是什么？它的内里是条狗啊！啊！好，欢迎来到直播间。
