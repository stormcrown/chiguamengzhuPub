# 第594章 移民大势所趋？金刀之谶的大预言术，蒙主的预言更准确

嗯，想问一下，我们这样发展下去以后，是不是和欧美、日韩一样接纳移民？特指从非洲、印度和中东来的移民。我不喜欢这句话。你知道吗？今天也有人聊过这件事情，但是我不喜欢说这句话。为什么不喜欢这句话？因为这句话呢，它是一种预言术啊。“移民是大势所趋，接受移民是大势所趋，全球混杂是大势所趋，每一个地方都是多元多样的，是大势所趋”——这种话是什么？这叫做预言术啊。什么叫做预言术？我之前说过，预言术是一种社会管理方法。虽然它很原始，但它确确实实是一种社会管理方法。它相当于我发布一个计划书，尽管这个计划书很荒唐，但是当我以预言的方式发布了这个计划书之后，就会有人不由自主地按照他的配套方案来进行工作。唉，“接纳移民是大势所趋”——于是你开了一个专门的移民俱乐部，因为这边说接纳移民是大势所趋；于是你开办了一个专供移民办理手续的公司。当这样的产业、这样的配套产业、这样的住宿等等的服务，等等的融入，等等的活动等等的，各式各样的产业都配套起来了之后，你不接纳，最后也会接纳。这是什么？这叫做大预言术的社会管理法。不是因为我们需要移民，也不是因为移民对我们有好处，而是因为有人通知你“这个东西是大势所趋”，于是你不由自主地为了赚钱，提供了配套产业和配套的心理行为，最后就引入了这些实际上会破坏你未来所有生活的移民。

所以我说我特别特别讨厌这种话，是为什么？这种话是在洗脑。不是它真的是大势所趋，而是他希望这个东西是大势所趋。为什么？因为他对某一些群体有好处。但是请问你是谁？你是本土的土著吗？你是原住民的时候，这玩意儿对你有什么好处？半毛钱好处都没有。结果你来个“大势所趋”？大势所趋缺啥呀？一个人的出身，一个人的成长，全部都要为自己的千秋万代而着想。这是什么？这是大自然的本能啊，亲。结果别人跟你说啊，“移民是大势所趋”——就是告诉你说，你失去你祖先的故土是大势所趋呗。你居然会同意？你居然会同意？你居然还出来问我有什么可问的。我亲，别人通过预言术希望破坏你的生活，然后你问我是咋看的？我能怎么看？谁给你发这个大预言术，你给他喷回去。这个东西它就没有任何的必需性，没有任何的必要性，没有任何的好处，知道吧。他就是为了通过预言术给你洗脑，让你相信这个事“大势所趋，不可避免”。哪有什么大势所趋？你把门一关起，你看他谁敢进来？全都是胡编乱造的结果。你就信，你就信，你就出来问，你还到处刷，有什么好刷的？

简单来讲啊，那就是你在看到这句话的时候，你就在后边给他跟一句：“所有移民都住你家里。”我跟你说，我这边也是有大预言术的，而且我这个预言术很明确的啊——它不光是预言，而且它同时也是客观现象的描述，还是历史经验的总结。那就是：移民国家没有好下场。知道吗？你直接复刻出去，你直接给他对顶回去。再有人跟你说“移民是大势所趋”，你说“移民国家死无葬身之地”。怎么了？哪个东西不对吗？历史上面所有遭遇这样事情的地方，全都死无葬身之地了；现在世界上面遭遇这种情况的国家，也都死无葬身之地了。咋了？我这是在陈述历史，也是在陈述现实，和那些大预言术比起来，我这个客观真实唯物的多。怎么了？得啊，你还上来给我复制粘贴，问你怎么看？有什么怎么看的？怼回去啊。怎么看？什么玩意儿？又有谁说啊？

那为什么发达国家都有移民吃大户？你不懂吗？他没门没有锁紧，于是就被人家吃了大户。你听不懂吗？为什么发达国家都有移民？是因为移民所以发达吗？不是，是因为发达了，所以有移民。因为先发达了，所以有人想吃大户，就千方百计地钻进去了。就这样啊，不然呢？你以为呢？我跟你说，就阿美天天说他是移民国家所以发达——扯淡。他上世纪六七十年代之前，包揽移民的标准是90%几，还是91还是92？人家是有标准的你知道吗？人家是有内部文件规定的。上世纪六七十年代以前，他们招纳移民的标准91%还是以上啊，必须得是欧洲白人。你觉得这是移民国家还是殖民国家？这标准的殖民国家的操作好吗？剩下的那个不到10%的标准，是给有色人种，给他们招募过去当劳工的。那个时候他们的奴隶制条款都没有解除，不同的人是有完全不同的社会责任，不同的社会权利的分配方式截然不同的。就是在那种前提条件下，他招揽的移民90%几要求必须是欧洲白人，那个时候他是标准的殖民国家，那才是他的上升期。后来是干什么？他倒霉催地接了那个二战的盘，阿美接了个战争盘，然后呢，就从原来的那个准工业国变成了一个大工业国啊。但是呢，同时他也接纳了资本，然后这些大资本家的主张就和这个他们的本土的人不一样了啊。大资本家的主张就是：最好这里变成一个移民国家，什么人都有，不然的话，到时候这帮子白人看不上我们这些资本家，怎么办呢？是不是啊？于是呢，他们开始陆陆续续地变化了他们的招募移民的条款啊。从上世纪大概70年代还是80年代前后，逐渐逐渐地改变了他招募移民的标准，最后就是变成了招募有色人种甚至多过于招募欧洲白人。他从此从殖民国家变身成为了移民国家。

有个问题：1882年阿美的排华，是不是华工一路向西修建铁路的过程当中与印第安人产生了关联，而非工会的排挤？这个已经算作近代史了啊，就是排华法案这件事情已经算作近代史了。里面按说不会有这么阴谋的东西，虽然不能够100%排除，但是目前能够看到的证据应该可以证实，当时的这种排华就是因为利润分配的问题。嗯，要知道我一直强调，那个时候是阿美的上升期，我也一直强调，阿美的上升期并不是一个移民国家——尽管它看上去是有移民的啊。它的上升期是一个典型的殖民国家，他招揽过去的所谓的移民是分层次的，不同类型的人权利分配的权利啊，完全不同。所以他当时应该算作一个殖民国家，是一个白人为主体的殖民国家，把其他类型的人都不当人，都当牲口使的那种环境。所以在那个环境之下啊，华人呢会产生一种格外的竞争力，这个竞争力很有可能超出了当时白人的想象。因为那个时代的白人有非常强烈的种族主义的倾向，因为那个时候他们刚刚开始进行了全球大爆发，进而产生了一种心理的诉求，就是觉得自己天生的高高在上，有自己的种群优越性。然后呢，之前他一直在使用黑人劳工、农奴啊、黑人奴隶，然后由于黑人奴隶呢，他确实本身没有什么文化，也没有什么组织度，又很多的时候非常的服从管理，所以最后就造成了一种刻板的印象，发现说那个啊，白人好像确实高高在上啊，很适合当奴隶主，他自己的这种自我认同就被强化起来了。但是呢，当他从中国这里——其实当时是买的啊，最开始的一批是买的，他是像那个三角贸易一样的，跑到中国来购买华工当奴隶去使的，和黑人一个待遇——但是呢，这种购买过去的华人的工人，给了他一个当头棒喝。他发现，诶，这个华人的竞争力是完全不一样的。在根本没有什么文化根基，然后漂洋过海被人家贩运过来当奴隶的这种情况下，华人的工人竟然能够在这一条狭窄无比、被人层层盘剥的赛道上面卷到飞起，卷到其他的工人都快要吃不上饭了，而且居然能够形成自发的组织，而且能够迅速地达成阶层的跃升啊。这个东西是给了白人当时一记当头棒喝的，他突然之间发现不一样，劳工和劳工是不同的。所以当时的这个排华法非常的有针对性。为什么？因为那个时候的阿美是一个殖民国家，殖民国家是非常强调主体性的，就是谁是殖民者，谁是被殖民者。要是搞不清这个，他岂不是变相了吗？他当时是一个标准的殖民国家，而他的殖民者的主体就是欧洲白人，虽然他们内部也分层，例如说爱尔兰工人呢就常常被压在下面，但是他们内部啊，他们自己自成一个整体，而且非常非常的有主体意识，就是过来这里圈地圈人啊，搞奴工、搞压榨，他就是来干这个买卖，是来搞劫掠的——圈地劫掠，这是他的殖民生意。结果呢，明明看上去是被他掳掠过来、购买过来的劳工，但是华工竟然就能够在这么狭窄的方式上面去跟他的这个所谓的主体殖民者产生了明确的竞争的关系，而且确确实实有着竞争的优越性和先进性。这件事情会造成一件事，就是导致他的这个殖民主体变得不确定，这个对于一个殖民国家来说是很可怕的——就是到底谁是殖民者，最后说不清楚了，你知道吗？相当于是公司里面竟然有个二老板，嗯，你招来的门卫变成了你们家的二老板，这个可以把原来的大老板吓死了，知道吧。所以当初的这个排华法案，他的证据应该还是比较完整，不见得是因为我们跟什么印第安人有牵连所以一定要排华。当然了，你一定要说这里头有一些什么关系，不能够完全排除吧，毕竟也没有什么证据啊。
