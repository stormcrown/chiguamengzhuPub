# 第549章 那帮人何时讲过张纯如

学三问在遇到他者的时候是最容易萌发的，而在张纯如老师身上就爆发出来了。明明在战争当中，中国人受到了巨大的伤害，但是无人在意。哪怕你叫出来了，哪怕你提出来了，他也说：“这有什么关系呢？与我何干？”对，与他人何干呢？与他人何干，那就必然与我有关。所以张纯如老师才会在这个时候去创作这样一部著作，她想要找到一个问题的答案：我是谁？我的祖先经历了什么？而所有的这些经历，到底与我之间有什么样的关系？

张纯如老师确实非常不容易，呃，不是一般的不容易。最重要的一点前提呢，就是她其实甚至不是生活在中国的，她甚至没有生活在中国。所以张纯如老师才会在这个时候去创作这样一部著作，她想要找到一个问题的答案：我是谁？我的祖先经历了什么？而所有的这些经历，到底与我之间有什么样的关系？我是一个中国人，我的祖先是一个受害者，而加害方还好好的活着，甚至想要让我忘记这些伤害。所以这套书不仅仅是一个历史方面的著作，它也是一个人类行为心态方面的著作。

嗯，好，欢迎来到直播间。这里是一个带货直播间啊，我把书给大家弹出来了啊。张纯如老师的作品，早点看。为什么要看这个东西？我们现在的实力，我们还怕倭寇打过来吗？倭寇现在拿着刀、拿着枪打不过来，但是他们拿着护照可以直接过来呀，比以前难度系数低多了呀。所以你得看一看啊，好好地看一看什么叫做“其心必异”，好好看一看你所想象的“谁来谁是中国人”会变成什么。不看看现实的场景，老是会做一些和平时期的春秋大梦。这就是和平久了之后，人会得“圣母娘娘病”啊。中国人的痛也是痛，中国人也是人呐。

正是因为她身在海外，正是因为她身边都是外人，所以她才更清楚地意识到了这样的一种痛感，于是敦促着她去寻找背后的印记、背后的道路，从而积累了一些素材，形成了这样一本书。这个呢，也是一个饱含血泪的书。我觉得每一个中国的家庭里面，都应该有一本——干什么？铭记历史！而且是用具体的方式来铭记历史。不是必须背下来名字啊，不是必须要背下来名字，而是你得知道：当真正的敌人出现在你的国土上的时候，他会做什么？必须知道这件事情，这叫做居安思危。生在太平盛世之下的人，最容易失去的就是这样敏锐的感知力，因为他感觉不到外患的存在，他没有办法意识到这件事情。因为太阳太明亮了，日子太美好了，因为过路的都是游客，如此的相亲相爱，他会觉得“谁来谁是中国人”。但是我们必须要在这样安安稳稳的时代，意识到引狼入室的后果是什么，真正的敌人踏上你的家门口的时候会做什么。这个时候你才知道什么是战争，什么是和平。

就是把日本那边派过来的开拓团的人，然后呢，用祭祀、纪念的格局啊——不是谴责哦，是纪念，是怀念，是缅怀的这样的一个视角——立了一个碑，还放到了一个园子里面。之后呢，被普通百姓发现了啊。然后就有几个来自天南海北的网友齐聚一堂，他们互不相识，然后就为了这一个共同的目标，就走到了一起，来干了一件事：就是要去砸碑啊。他们从各自的地方、各自的岗位上面聚集到了一起，然后到了黑龙江去砸碑，越过了重重守卫，千方百计地进去了。进去了之后发现，哎呦，当地做的这个碑，那个质量嘎嘎好，砸不动。费老大劲了，好几个爷们啊，费了老大劲了，砸不动，连皮都没破。于是呢，就拿漆又给泼了一下啊。

之前我就带过很多次，但是还有没买的，一定要收藏。我倒不是说你非得把它逐字逐句地念啊，注意这个书是真的非常非常需要你收藏一本的。看的时候会很痛苦啊，可以不看。什么时候聚起了勇气了，你看一点；甚至说我就看了之后，翻拍我就头疼，也可以不看。但是依然还是需要你有一本。为什么这个样子？承平日久啊，承平日久。承平日久是要出问题的。承平日久，你的思维方式会陷入到一种太平盛世之下的文明重症之中。连这个东西你都理解不了的时候，会很难很难过。你不容易理解什么叫做丛林社会，你不容易去理解什么叫做敌我关系。我有的时候说，有一些矛盾是内部矛盾，有一些矛盾是敌我矛盾，有一些毛病是叫做敌我矛盾的。

张纯如还有很多的作品呢——也没有很多的作品呢。我光说书名的时候，你就应该知道了呀。虽然待的时间太长了一点，但是我实在是忍不住，我老觉得可能还有很多人没买。中国人都得有一本，哪怕你不看，哪怕没有勇气，你都应该有一本。这个书还是得再带一遍。这是帮人区分敌我的，帮人区分敌我什么样子的——是敌无差别地摧毁你生命的存在的，这种是彻头彻尾的敌人。敌无差别地摧毁你生命的存在与延续的，是彻头彻尾的敌人。所以这个事上没有什么矛盾可以转移的。什么说“你看白起还不是那么狠辣”？白起是杀俘，那是战争行为，他再狠那也是战争行为，他可不是无差别的，他要是让对方从此消亡，他要的是对方臣服——区别非常非常之大。我非常不喜欢这种转移矛盾重心的操作方式，动不动就说“你不要老说人家怎么大突突了，你看白起多狠”，只是战争当中性质一样吗？我们几千年来遭遇的最狠辣的就两拨，一波他是，一波他是——脚盆两拨还是手拉手，那是真的。来那个什么——就是张纯如这个书讲的是南京这件事情，我为什么不叫全称？有的平台不让叫，是哪个平台我就不跟大家唠了，反正有的平台不让叫全称。就张纯如所讲的这个事件，大家都知道是脚盆操作的，知道里面还有一支部队怎么定性？那里头还有一支部队是伪满，那里头还有一支部队是伪满。我认识他是谁？连连连——我需要别人给我热度吗？你知道连线的啥人？连线的不都是各种各样的人，互相抬妆互相拉热度，通过PK的方法吸引双方的注意力，实现自己的破圈。我犯不上。有你们这些恨我的人在外面到处骂街，就已经足够让我破圈了，我用不着跟别人什么连线连麦，说话又说不明白是吧。
