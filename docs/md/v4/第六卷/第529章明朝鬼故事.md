# 第529章 明朝鬼故事 #传教士

### 明朝朝贡体系下的外国人渗透

外国人来华定居的数量多不多？按照一些笔记上的记载，数量是不少的，但是没有一个非常精确的统计。同时，我觉得最可怕的并不是外国人数量多的问题，更重要的是他们渗透的层级非常之高。而且，他们渗透到高层的这一部分，被很多信息掩埋住了、隐藏住了，似乎不让你查证，就好像他们在高层当中不存在一样。但实际上，当时很多外国的传教士已经渗透到了明王朝的社会高层、社会顶层，甚至很有可能渗透到了宫廷之中。也就是说，他们的势力很可能已经影响到了宫廷内部，但在我们的记载中，却显得他们好像无足轻重一样。这是非常异常的。

如果说他们大张旗鼓地出现在史书中，大家都记着“朝堂之中有几个外国人，这几个外国人多么奇葩，他们都干了些什么事”，那这反倒是一个正常现象。但在明王朝时期，我们从各种侧面证据中能看到，有很多外国人在中国影响力非常之强，甚至可以影响科举。当渗透的层级已经这么高的时候，它在我们的很多记录中却约等于隐身。隐身比张牙舞爪要可怕很多。

所以，明王朝时期，西方人、洋人、传教士、商团这些人到底产生了多么大的负面影响，现在是真的很不好说的一件事。因为他们的层级太高了，那么高的层级却能够在史书当中几乎隐形。不特意提及的话，你甚至不会留意到朝廷当中有很多洋人的手笔。这是很荒唐的一件事。你要知道，洋人和我们长得不一样，闻起来不一样，行为方式也不一样，甚至他们的信仰跟我们格格不入，与我们的传统文化完全不兼容。一个这么醒目的角色，放到哪儿都会嫌它扎眼睛，但就是这样醒目的角色，在我们的各种历史记载中存在感不高。
