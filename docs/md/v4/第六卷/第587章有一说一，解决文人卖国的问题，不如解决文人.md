# 第587章 有一说一，解决文人卖国的问题，不如解决文人 #劳动改造

师爷的屁股在树上吗？是歪的吗？和司马懿是一伙的吗？不是，这玩意儿……我为什么要看这个呢？啊，我为什么要看这个呢？这玩意儿能看吗？啊，我待会儿再聊一个啊，我待会聊一点细节。我是真不敢跟你说这个东西啊，你当我没看见吧，不好意思。

嗯，刚才那个啥，我们讲一个东西啊。啊，我们得兜100个圈子讲这事儿，哈哈哈。哎呀，这咋讲的呢？那就属于是有些事吧，他敢干我不敢说，说了之后我容易凉，还没到十级。但是呢，能不能讲呢？啊，不管了，总归还是有一点东西能讲的。

来，那个小伙伴们有没有听过一个东西？就是，嗯，并且呢，曾经在心里头念叨过，老委屈了的东西啊。嗯，优待战俘听过没有啊？有没有听“优待战俘”这个事？然后觉得心里老委屈了，就觉得这个事不好啊。有没有啊？有听过这个信儿，并且心里头老觉得咯噔一下子难受的啊，扣个一。我兜100个圈子跟你唠啊，唠完了之后希望你知道我在说啥啊。好的好的啊，听到了而且觉得憋屈、觉得难受是吧？嗯，那就对了啊。

我跟你说这口锅，他们一般都甩给了教员，说是教员说了我们要优待战俘。那教员说没说这话呢？说过啊。然后他是怎么变成让大家憋屈的东西呢？啊，啥？优待战俘是个好事，瓦解抵抗力。来来来，我跟大家分享一下这个事儿啊。啊，我来跟大家分享这个事儿。你就说当初教员提出说我们要优待战俘的时候，有没有合理性？有啊，我明确的说有。为什么？因为当你对战俘好的时候，对方的战斗意志就会被瓦解，因为投降有好处啊。投降了之后不难受，投降了不受委屈的情况下，他就更加的欠缺抵抗到死的这种意志。所以呢，优待战俘这个事呢，它是对的啊。

然后后面就来了啊，你千万不要觉得我只说咔嚓这一小片啊，这个东西你必须要有。然后啊，教员说的另外的一件事是什么呢？就是我们要改造它啊。优待战俘，然后我们要改造它，要让它能够懂得什么是正确的，应该站队站到哪个地方去啊。这个东西叫做改造啊。所以呢，教员当初提出的优待战俘，它的本质是什么？它的本质是不虐待，并且提供收容和教育。这个是当初我们提出来的优待战俘啊。一块呢，是教育和改造，另外的一块呢，就是不虐待。

为什么我说不虐待？因为那个时代虐待战俘是一个常见现象。为什么很多的人不肯投降？就是因为很多的军中都虐待战俘，他根本就不按什么公约办事。包括说脚盆的那些什么战俘，不就跑去挖那个冻土豆什么的吗？啊，诸如此类的啊。其实呢，就是各国当时都有各种策略虐待战俘，这已经是当时的普遍现象了。所以呢，我们提出优待战俘的意思，在本意上面是指我们不虐待，给饭吃啊，不用参加大强度的劳动，我不是为了要你的命啊。这个东西呢，就叫做优待战俘。

优待战俘同时啊，就是不虐待战俘的同时，还要对他提供改造和教育，让他懂得大是大非，让他知道应该做什么，不应该做什么。目的是为了把鬼改造成人。所以呢，这套方针当初提出来的时候，是一点问题都没有的。放到今天我给谁说，这个道理是不是都是通的？是非常完整的一套系统啊，是没有毛病的。

毛病出在哪儿？毛病出在执行端。这像什么呢？就像明王朝时候的文人集团，在面对君主的指令的时候，他们会执行200%、300%。君主说我们要给臣民们减税，增加大家的幸福感，于是他们就把什么商人的税咔嚓一减啊，让国库空虚，让富商欢欣雀跃，然后理由叫做藏富于民啊。这个东西是干什么呢？执行两倍，执行三倍，最后你这件事情就会变得破坏民心啊。所以那个时候就会出问题。

啊，执行的时候就会发现战俘的日子啊，我的天哪，那就不是说和普通士兵比了，和我们战功赫赫的各种干部们比，战俘的待遇都要更好。那这个时候它原本的作用能起到吗？就起不到了啊。能理解吧？好，说完了，诶。好，欢迎大家来到直播间啊。

嗯，什么玩意儿？我看一下啊。新中国怎么解决文人集团叛国这种问题啊？我们新中国前30年的时候，基本解决了文人叛国的问题，然后现在有点儿，现在又在努力的解决，还没有成功啊。而且呢，也不知道能不能成功。我个人觉得确实有点悬。我觉得解决文人叛国的问题不如解决文人。我个人啊，是这样看的。真的，作为一个脾气暴躁的互联网主播，我觉得解决文人叛国的问题，还不如直接解决文人，难度太大了。嗯，我就觉得我很累，懂吗？文人阶层我认得的太多了，我很累。我就觉得要改造他们，我真的没有那种耐心。我爱教员，他太了不起了，他对这些玩意儿都能耐下性子来，想教他们，想帮他们，想让他们长上膝盖，太伟大了。我做不到啊。但是我可以把他的书推给你们，然后你看你能不能做得到。反正我是做不到，这话说的有点自暴自弃。但是我觉得我说的话呢，如果是面对工农啊，我是很有自信的。可是呢，面对知识分子，我真的是没有充分的自信和耐心，想要能够改他们啊。我没有这种自信，见得太多了。所以我不觉得我能够改变他们，我只想解决他们。

嗯，但是呢，如何面对、如何思考、如何判断、如何区分敌我，这个事儿还是可以从毛选当中去学习的啊。可以从毛选当中去学习。来把这套书给大家推一下啊，由家里面书读烂了的啊，以前还没读过的小伙伴啊，来把这个书搬回家去。好好看一看他在做什么呢？就是为什么我们的从历史上面也好啊，各个阶段也好，文人集团总是那么的软骨头啊？这个认识是怎么回事呢？他们的利益诉求到底是什么？他们的根本利益到底在哪里？为什么当他结成了所谓的文人集团的时候，总是表现得非常的反动？如何分析他，以及如何根本上的去解决他这一件事情？我们应该从毛选当中去寻找斗争的艺术和解决的方案。我们要解决的是一个阶层的问题，文人集团它本质上面是某一种特定的阶级圈层，而不是某一个个体。一个个体是容易改变的，经历的变化，不断的训导利益的，幼时都会让某一个人发生改变。但是要改变一群人，就要改变他的生存土壤。

那文人集团他的依附感，主要其实来自于他没有能力独立的创造价值啊。注意了啊，文人集团，尤其是典型的文艺知识分子团体，他们是没有独立创造经济价值、独立创造物质价值的能力的。所以他们需要金主。而一个迫切的需要金主的团体，他们天然的就具有依附感。而一个有依附感的群体，却偏偏是社会上可以对他人指手画脚、影响他人思维的文艺工作者、文化知识分子。于是就造成了这一群人眼高而手低，不肯做事，也无法产生经济价值，但是却觉得自己理所应当的是统治阶层的一部分，然而他只能当统治阶层的走狗。这一种利益上面的对冲，使得这个阶层即使出现了好人，也非常难以出头。更容易出头的是甘当走狗的群体。于是就造成大家所看到的这种文化的、文艺的知识分子的圈层，普遍上都有买办卖国的倾向。因为没有这种倾向的、铁骨铮铮的文人们最难熬出来了。

所以要如何从根本上解决他们，让他这种生存的机制发生改变？我们前30年的时候，就试图通过劳动，让他们亲手创造财富，但是他们不高兴，他们觉得当狗的日子比当人舒服，所以在那之后迅速的叛逃了。那么之后我们有没有可能为这些文化的、文艺的知识的群体，创造一个让他能够自力更生的环境呢？如果有这样的平台，如果有这样的可能性，那么这些人就可以因势利导的，变成为我们而服务的群体。要知道，刀笔如剑，他们的能力本身，也是我们文化实力当中不可或缺的一环。但是当他们所依附的对象、当他们所服务的对象不是我们的时候，他们就会显得无比的龌龊。所以，如何解决根本的问题？要让他们拥有为我们服务同时获利的空间，但是同时我们也必须要让他们认清一件事情：他没有资格高高在上，他必须要做到为人民服务。

嗯，上山——对，这个就是上山下乡啊。啥都别开工资，自己出书做视频换打赏？你闹呢？你猜他自己出书，谁是他的金主？谁给他出钱，他就替谁说话。你是理解不了文人的经济运作模式。所以我说为什么觉得这个事儿忒累了，一点兴趣都没有。累呀，真的很难。这种利益分配模式，不光是我们知道，敌人也知道。你知道人家在国际上面设那些大奖，明明是个科学奖，非得设一个文学选项，去打赏各国愿意叛国的文艺工作者、作家们，人家图啥呀？人家给你评奖，人家给你那些奖金图啥呀？买的不就是这个东西吗？你以为呢？你知道收买，别人不知道收买啊。所以不要说我是什么什么什么文人啊，我不是，我是一个带货主播，带货是我的职业。嗯，带货能够让我养活自己，带货能够让我获得收入，带货和打赏是我的经济来源。在座的各位就是我的衣食父母啊。做到了这个的时候，我这不就是腰杆子就直了吗？啊，我靠着卖书，靠着卖饮料，靠着卖瓜子儿，我就能养活我自己的时候，他想要买通我，那可难了。

嗯，好，欢迎大家来到直播间啊。嗯，啥？避免金主找上门？其实我也想等金主，他也不来呀。我真的，我从那个啥，我从出道开始，老多人跟我说了：“哎呀，你好有潜力，你肯定是百大啥啥啥啥啥的。”那天天那个拍马屁拍的砰砰的响，我都快要信以为真了。你说我这么有潜力，这么有魅力，那么多人觉得我天生后面就要当百大的，那不得有金主来找我吗？必须得有吧？那不有也不行啊，是不是？难道他没有眼光吗？结果没有黄巢在世。哎，我也没有兵力呀。这里是一个待卧直播间啊，千万不要想多了。
