# 第513章 历史的温情，不理解祖先苦难的人不值得我们理解他

到现在都没搞明白，他们嘴里的“黄汉”到底干啥坏事儿了。破坏了他们想要复兴八旗思想的土壤——这是最大的坏事啊，知道吧？这是最严重的坏事，就是破坏了他们以八旗为荣的思想。你知道从什么时候开始？从上世纪80年代就开始了啊。嗯，拍的电影电视剧都有些什么东西来着，我一下子都缓不过来，太多了啊。各种各样的电影、电视剧，还有小说，大家耳熟能详的那些大作家，那个特别大的那种大作家，他创作的内容当中都包括了这些内容——英明、神武、圣天子在上，这些玩意儿一样不带少的啊。那俊男靓女之间勾勾缠缠的那个眼神儿，那拉丝儿不迷人吗？迷人呐。演得不好吗？演得好，演了多少年？几代人。你想一想，几代人。从上世纪80年代开始出现的这些影视作品，让人心里面卸下心房之后，逐渐相信真的有圣天子在上。这养起来的一代人会影响他的子女啊，又是一代人。所以当这些东西铺天盖地都出来了之后，人们的思想会开始发生转变。

要知道，在建国初期的时候，国内外但凡拍一个任何一种有关于不文明时期的影视作品，如果他没有进行严厉的批判，那么在那个时代就叫做反动影片啊。大家可以去看一下，我们建国初期的这种在精神、在文化、在叙事权领域当中的高地争夺战——凡是涉及到不文明时代的任何正向描述，哪怕是看上去像正史的、客观中立的描述都不行，除非你大加批判，否则不准拍。思想高地的争夺战就是这样打的。但是后来不一样了啊，就客观、中立、真诚地对待，于是呢，你这个思想就软化下来了。反正他也是人嘛，也能做好事嘛，是不是啊？圣天子在上嘛，他还可以到街上遛弯儿嘛，是吧。这个臣子呢，有忠臣、有奸臣、有聪明的、有笨的嘛。当你的形象变成这个样子的时候，人就会开始心态上面发生转变，而这种心态上的转变进而影响了整体的舆论氛围。那么所有的这些东西都是从这个时代开始就逐渐逐渐地发生了改变。当风向开始发生变化的时候，人们心里头的认知就会开始发生变化。

所以说啊，话语权简单来说，就是我们的文艺工作者有权利为谁说话、没权利为谁说话。这件事情现在不掌握在我们手里头，这就是时势。痛了不可怕，可怕的是不知道痛，并且乐于其中。嗯，所以现在并不是什么特别坏的局面。为什么呢？你看看这两天有多少的大咖破防，呵呵呵呵呵呵呵呵。哎呀，不能这样笑，万一被人家抓包说我笑谁了。我明明没有点名是吧，明明没有点名，结果别人自己过来顶缸，自己出来认账，说你凭什么笑话我？哎呀，我一个名儿都没有点，万一有人说他笑话我，他笑话我，哈哈哈哈哈，这不就尴尬了吗。嗯，所以不能说啊。但是你看这个局面，最近有多少的大咖破防得一塌糊涂，呵呵。啥？嗯，又要被扣帽子了，我们又要变初中生了。嗯，我头顶的帽子比较多啊，可能就是因为帽子压得太多了，所以头发油得特别快，没洗头就油了。所以我还戴上了个发箍来遮蔽我自己的油头啊，棒棒的。他又解释了一遍。

嗯，我们前些天的时候带的那个书叫什么呢？什么来着？唉，带这个吧。啊，来来来啊，我们整一波国史大纲。好来好，来，给大家弹出来了啊，整一波国史大纲。国史大纲就是标准的以史为鉴，它算是一本通史，但是它和别的不一样的东西是，它出自绝望之际，是为了给整个国家、整个民族留下一个记忆的压缩包。所以呢，它从立身之本上面去记述了我们整个的历史源流。因此呢，我觉得它是一个非常好的入门级的书籍啊，大家应该可以收藏一套慢慢读。这些东西呢，都是不着急的，也不见得写得非常优美顺畅，所以呢，都是可以慢慢来的东西。但是收在家里有一套却非常的重要，因为它是那个时代的学者们能够总结得出来的、有关于我们这个民族漫长文明史当中最为重要的核心内容压缩包。要知道，哪怕你只看了它的开篇介绍，也是会产生一定的共鸣感。它在开篇的时候就强调了，这个书最重要的东西是什么？就是一群人生活在一起的时候，我们要有一种觉得我们是自己人的感觉。那如何觉得我们是自己人？就是我们要有共同的记忆。这个共同的记忆就是历史，就是我们共同经历的过去的时代。所以我们得知道一点历史。但是光知道够吗？光知道不够，你还要对这一段历史抱有温情的态度，就是你要视它为自我的来路，而不是在围观。这也是我一直强调的——历史是有视角问题的。当你站在遥远的地方去看待它的时候，那是把它视为一种客体，完全可以将它看成是别人家的事。当你把它看成别人家的事儿的时候，此事就变得与你无关了，那它就不是共同的历史记忆。所以共同的历史记忆要求你知道历史之外还要有正确的主语叙事、正确的观察视角，然后你才知道在这一段历史当中，你应该共情谁，你应该同情谁，你应该赞赏谁，你应该和谁站在一起。有没有发现这件事情很重要？这几天大家格外地发现这件事情的重要性了吧。

同样是在讲述历史，站在不同的角度上面去看的时候，它是截然不同的。为什么最近很多人都特别上火？为什么很多人在这个时候居高临下地指指点点？因为大家站在了不同的地方去看问题。我们普通人所看到的是：我的祖先是谁？他们遭遇了什么？他们在哭泣？他们面临着生死危难？于是我的心都缩紧了。但是另外的一些人高高在上，站在火星上面看这个事情：反正那不是凑一起了吗？很好啊，谁反对凑一起拖下去？这个东西就叫做历史的温情感。诶，谢谢000啊。我们现在讲，嗯，叫民族，是因为另一种共识没有了。嗯哼，咱们就不讲那戏了啊。亲，我现在在带的是那个国史大纲啊，非常重要的一本中国通史，所有的中国人都应该知道一点点的东西。我们应该用温情的视角看待祖先的来路。当然最重要的是不要认错爹啊，知道谁是祖先。然后呢，看祖先的来路，看他的经历，我们与他同在，我们记得他们，我们和他感同身受，我们跟他精神共鸣、情感一致，那么他们就永远都存续着，也是我们这个群体永远都活着。共同的记忆是不可以消磨的，我们有共识，有共同的记忆，有共同的情感，所以我们才是一家人啊。哎，好，欢迎来到直播间，这里是一个带货直播间啊，欢迎大家的光临。小黄车，对对对啊，我们现在正在带国史大纲的。来，这一点有限的时间，好些感觉就剩能带一本的感觉呢，是不是？那不行，作为一个专业的带货主播，是不是应该整得更开阔一点才行呢？我想一想啊。

对，大城市病、个人原子化这事怎么看？有办法缓解吗？只要我们要经济，我们的生存模式需要城市化，那这个事就不能够直接解决啊。你要说缓解还是有通道的，就是我们现在的传媒，知道吧？你直接解决是真的解决不了，因为大家的经济形势已经不再像过去一样地抱团了。当你的经济、当你的利益变得分散了的时候，人们就开始趋向于原子化。就像明代的中晚期，人们的市民阶层开始原子化的主要原因，最主要的原因就是它的经济形势变了。那个时代啊，它的商品经济变得非常活跃，普通人在城市当中可以获得一份收入，女子也可以单独立户。当你一个人就可以好好地生活的时候，她逐渐地就会开始出现原子化的倾向。这是经济所带过来的这个部分，没有办法，你只要寻求商品经济，那么它就会出现。但是另外一个维度上面，我认为可以补强，就是在传媒领域当中。但是现在我们的传媒领域没有完全掌控在中国人或者站中国这一边的人手里头，他们在叙事上面这几十年是很有一些问题的。所以在传媒领域当中，我们这些年应该进行真正的深耕和高度的管理啊——是不需要进行高度的管理的？管理到什么程度？大洗牌也不为过的那种程度啊。我们的传媒领域当中，是需要这种深度的管理的。它可以通过传媒的方式来达成共识的传播。要知道在过去几十年间，我们有一些共识慢慢地消失了，就是传媒之功。那么既然是他做的，现在要弥合，也需要用这样的工具。工具本身并无善恶，区别在于谁在使用它。

嗯，好，欢迎来到直播间啊。什么思想殖民？对呀，什么是思想殖民？改变你想问题的方式啊。当然它会改变你思考的内容，但是最重要、最根本的是改变你的思想方式。有没有记得上个世纪末开始啊，我们的大量的公知给大家传播的信息当中，有一些非常反复重复的符号特别强化，强调的是什么呢？要求你独立思考啊。上世纪末开始啊，大家记得吧？但凡有稍许经历过那个时代，或者曾经投坟考古过那个时代的，应该都见过。当时的这些所谓的学者啊，其实就是公知们不停地强调：中国人要有什么能力？独立思考能力。特别强调你一定要有这个什么独立思考能力。但是这个事是在干什么？这本身啊，是在消解你原有的思想，它在消解你原有的共识。本来那个时代中国人还是有很多的共识的，就是我们对经济怎么看、对文化怎么看，然后包括“帝国主义亡我之心不死”这些内容，在当时的中国人心中是有符号、有共识的。你不要觉得这些符号很土、很过时、low爆了，只要你有共识，我们之间啊就有强烈的共鸣，就有强烈的共同点。在那个时候这些东西都还在。那么这些公知要求你说你要有独立思考的能力，他在干什么？他在消解你的共识。他说你不要认那些玩意儿，你把那个共识都扔一边去，你要听我的。所以那个时代的所谓的“知识分子”，他在强调独立思考这件事情的时候，其实是要求你放弃掉共识，要求你强调个体化。今天我们就知道，当一个人孤立原子、完全个体化，与其他的、与过往啊、与家人、与朋友、与其他的所有的群体全部都隔离开了的时候，你会脆弱的——别人说什么你信什么。这是人类作为群居型的生物天然存在的，你要抱团儿。如果现实生活当中不抱团、不取暖，那么你就会在互联网上去寻找到虚拟的抱团儿。所以，打从独立思考之后，你有没有发现，各种各样的亚文化群体就开始此起彼伏地诞生了？亚文化群体是干什么的？就是把你原来的那些从原有的社会共识当中抽离出来的人，给你安排进一个小圈子里面，让你有新的共识。但是这个新的共识与社会的公共共识和我们应该要有的东西不兼容，这就是将人打散了。

嗯，所以这是干什么？这东西就叫做精神殖民，这本身就是精神殖民。他在转变的最重要的东西，就是你的思想方式。要知道在那个时代的很多年轻人，以有共识为耻——有共识啊，和纲领保持一致，和毛选保持一致，在那个时代的很多年轻人的心里面，那叫做没有个性。但是通过这样的方式，他将社会割裂了，割裂成了很小很小的一处处板块，进而方便他传播下一波的洗脑信息啊。就是这样的操作法啊。如果说大家经历过那个时代，你就知道我说的是不是真的。这样的两个步骤，就是将人打散的一个步骤。在今天它依然产生着庞大的影响力。
