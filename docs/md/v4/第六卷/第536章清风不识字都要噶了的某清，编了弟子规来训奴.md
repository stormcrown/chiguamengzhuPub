# 第536章 清风不识字都要噶了的某清，编了弟子规来训奴 #奴隶制

来，我们带一下这本书——这是关于国学易理的。  

为什么要了解国学？因为国学里头骗子太多了，注意啊，骗子多到了什么程度？多到我身边都有好多，就是那些专业讲国学，但纯属骗人的。  
所以，中国人非常需要了解一下自己传统文化的底层代码到底长什么样。  

当我们面对浩若烟海的经史子集，没有办法全部看完，也没有办法自己全部吸收时，我们应该如何梳理其中最重要的纲要？如何去理解它？如何找到里面最关键的点？  
哪些符号是确确实实由我们自古以来传播下来的？哪些符号又有可能是被篡改的？  
我们如何理解自己的思维方式？如何理解自己的行为方式？如何理解我们的祖先所留下的种种制度与道德规范？这些东西都属于国学范围。  

想要更正向、更明确地理解国学，绝不是报一个国学班（因为国学班大部分都是骗子，我见到的骗子实在太多，很尴尬）。但是，我们可以从这本书里面简单着眼，去了解一下我们文化底层代码之所在。从它作为入门，即便很多书没有时间通读，从这个入门之后，能够更好地理解：哪一种类型的思维方式是中国式的。  

这会带来什么影响呢？就是当你在看到一些外来信息的时候，能够有一个识别的能力——我得知道我到底长什么样，我才能知道对面的那个人跟我相似还是相反。  

现在也都在谈“骗子太多了”，人家骗子不是一般的多。我跟你说，凡是所有这种“国学霸”，第一个识别的东西就是：他让不让你背《弟子规》。当然，这两年这个识别公式不太好用了，因为越来越多的人知道什么是《弟子规》之后，国学班也会升级换代。但是，暂时它还能用一用——凡是这些国学班，尤其是什么少儿、幼儿启蒙国学班里头，他让你去背《弟子规》的，所有背《弟子规》的全是骗子，一个都不漏，没有漏网之鱼。只要他让你背《弟子规》，觉得这玩意儿是“圣人训”，那全骗子，妥妥的。他但凡对国学有任何一毛钱的研究，都不会觉得《弟子规》是一本好玩意儿。那是一个训奴才的奴才手册，就是清朝一个半文盲为了管理家奴编出来的一套奴才训练手册、奴才速成手册。所以，谁让你去背《弟子规》？哎呦我的天呐！  

注意啊，《弟子规》创于清。清是一个什么特殊的时代？一定要弄清楚，不要觉得“读书人的事儿能叫偷吗？”对吧？那为什么清的时候会变成这个样子？清的时候有很多规矩，因为清的统治者非常清楚自己是窃国大盗，而且所有人都不肯听他的，知道他是所有人民的敌对方。这点事，八旗的领军人物门儿清。他们从始至终都清楚这件事，就是因为太清楚自己的反动属性了，也太清楚这根本是敌我矛盾了。所以，他的统治阶层对于普通人的识字、交流、念书是极度约束的。约束到了什么程度？在他管理最严格的地区，父亲认字想教给儿子都是非法的，就演到了这样的地步。  

然后，为了在科举当中筛选、排除异己，让你在三代之内没有官身的人禁止参加科考，这叫“冷级制度”。也就是说，你家祖上没有谁给八旗当过狗，你就不能参加科考，学富五车也不好使。从而造成整个清朝文化水平的断崖式下降。断崖到了什么程度？明朝的时候，市井民俗类小说铺天盖地，三俗小说铺天盖地。有兴趣去看一下明代那些什么小说的，你就可以瞧见里面老多三俗的情节了。那玩意儿就不是给读书人看的，妥妥的是给一般工作人员看的——就是那些白天干体力活、晚上要消遣一下的人。只有他们会热衷于看这些三俗小说，而且异常通俗。这说明什么？说明最普通的劳动者也有阅读的能力。  

还有一本书叫什么来着？嗯，就是那个“且容我伸伸脚”，那个书叫《夜航船》。大家可以看看《夜航船》，它是一本讲文史典故的书。一开头的时候，它讲：我说我为什么要写这本书呢？就是说有很多普通的孩子上了香学、上了学校之后，学了老多典故，学了那些“之乎者也”的一堆词儿，但一细问，这也不通、那也不懂——就是光背了书，结果典故到底是啥一概不知道。他本来是用一种有点嘲讽的语气去说的，但这句话的反面是什么？就是明王朝时，普通市井小民的子弟会进入学校学习很多知识。尽管质量不好，甚至把词儿背下来了也不知道是什么意思，可他认字了，能写会算，于是就有了这样的通识教育。  

但到了清朝，这玩意儿没了。清朝写的小说越来越“高级范儿”，阅读门槛越来越高。这不是文化素养提高了，而是普通百姓已经不认字了。你没有必要去写那些下里巴人的内容，只需要写官面文章，比如大院里头、后院里头、宅门里头的事情。市井小民的事情没人看，因为市井小民不认字了。就有这么大的偏差呀，我的亲！  

所以我们要理解什么是国学，要理解为什么到清朝的时候，它编撰的那些入门启蒙读物那么烂——因为清就是一个这样烂的时代。在这种时代里面，普通人的启蒙教材就是《弟子规》那种奴才用的玩意儿，要求你无条件服从。不管有没有道理，你先听着就是了；他坏、他打你，你忍着就是了、听着就是了。为什么？因为在那样的时代，有理性的人容易没命，有志气的人会憋屈。所以它的训奴手册在进行启蒙教育时，要求你只管家听话就行了、只管乖乖的就好了，你讲什么道理？别人让你干啥你就干啥就完事了。  

那我们今天的人为什么要学这种东西？这是个驯奴手册！所有的国学班，凡是在那里跟你讲要背《弟子规》的——《弟子规》里面有没有可取之处？早上起来要洗脸、吃饭之前要洗手，确实属于可取之处；走道的时候要看着路、见到人的时候要打招呼，是不是可取之处？是，但它是废话，用得着你教吗？所以《弟子规》就是一个驯奴手册叠加了一大堆废话的玩意儿，不要拿它作为启蒙。这是第一件事情。  

第二件事情，所有的国学班，只要让你背《弟子规》的，立刻有多远逃多远，能省老多了。  

哎，谢谢！嗯，我又放开了，又得罪一个利益群体。我不是一直放得很开吗？我得罪的群体还少了吗？放心，放心了。我在互联网上没有任何加盟伙伴，但是我有赛道，知道吗？尽管我没有加盟的伙伴、没有盟友，但是我有赛道。什么叫我的赛道？就是说“吃瓜盟主是个好人”是一条赛道，说“吃瓜盟主是个烂人”是另外一条赛道。我一个人独霸两条赛道，棒棒的，好优秀！  

嗯嗯嗯，啥？觉得“亲”是一场剥削压迫者的狂欢？所以可能不只是遗老遗少怀念他啊。我一直在说一件事：这个世界上有很多人是向往奴隶制的。这个事我也不是第一次说了。因为奴隶制很安稳，不光对奴隶主安稳，对奴隶也安稳。人类是会有这种心理诉求的可能性的，所以向往奴隶制的人群在全世界范围内其实都有，而且很普遍。  

今天在中国的土地上面，有一群特别热爱“带清固伦汗国”的。有些是遗老遗少，祖上拿了赃款，如果不说他好，就会觉得自己有罪了。为了让自己看上去很体面，于是去洗地，把满清这一场人类最恐怖的迫害捧上神坛。有这样的人。  

但是还有另外一群人，就是幻想着自己可以成为剥削者的。你要知道，在中国的土地上，能够成为那么安安稳稳的（不管本多么烂都可以继续当剥削者的）固定剥削人群，是多么困难的事。因为我们自古以来都要求上下通达。有的人过几代之后，很有可能就撑不住了，后代就去编草鞋去了。你看刘皇叔都要出来编草鞋，别人是皇家子弟啊，传上几辈人就要出来编草鞋了，我的亲！  

然后，也有很多英雄直接起于草莽。所以我们一直有上下通达之路。这对于很多人固然是好的，但对于另一些人来说，这太可怕了。要知道，不管已经烂到什么样的程度，他都可以永远吃“铁杆庄稼”，这件事对很多人有诱惑力的幻想。“自己是永恒的压迫者”，这件事在人类之梦当中始终都存在。所以人类群体里面一直有人热爱奴隶制。中国人不太容易共情这一切，因为我们不喜欢奴隶制。但是这个世界范围内，有相当相当多的人认为奴隶制很好，虽然嘴上不多说，但在行为上面，他们是追随着奴隶制的脚步的。  

所以一定要相信一点：强者是寂寞的。强者是寂寞的，举世皆敌是很正常的，与所有人都格格不入是合理的。不代表你错，更代表你优秀。  

感觉资本主义只会发展成“资本神权种性质”，而不是社会主义。因为中世纪的神权剥削才是游牧劫掠文明的政治稳态。我一直在说一件事：很多中国人会默认（或者说也不是我们的错），当年的马克思主义在进行历史分段的时候，确确实实将资本主义列到了封建时代的后面，说它是一个更先进的社会制度。这个分段法确实存在，所以就会让我们在直觉上认为，资本主义尽管不好，但也是某一种必经之路，或者说它是某一种更先进的制度。  

这个事我在直播的时候聊过很多很多轮。资本主义它并不是一个真正的社会制度。我不认可马克思对于历史的分段，他的历史分段论是有问题的。他的历史唯物主义是对的，但他的历史史料不足，造成了他的总结是有问题的。他的这一套历史总结只能适用于西欧本地带，在任何其他地区都不能套着使用，没办法用啊，完全不可适用。  

所以，我们不能直接照搬他的这一套历史分段。不能认为资本主义一定会优于我们传统上面所谓的“封建”。实际上，我们的分段根本不能按照他的次序进行分段。他的分段论当中不包括中国，他对我们的历史没有充分的理解能力，所以他没有可能对我们做出一个分段。不能照搬，不能照搬啊，绝对不能照搬！我们的历史分段必须由我们自己来做。  

包括说经常被定性为“封建时代”的，那不是封建，那是中央集权君主制时代。中央集权三省六部是什么落后的制度吗？不是啊。中央集权三省六部是什么落后的制度？内阁算什么落后的制度？我勒个亲，请问今天没有与此相类似的现代化表现方式吗？没有吗？有啊。对！什么是政治制度？政治制度是政治制度；经济类型是经济类型。这两样东西要分开。资本主义是一种经济类型，它不是政治制度。它的分段、它的定性本身都是有问题的。所以资本主义不等于它比君主集权、郡县制管理更先进，完全没有这个比较的可能性，因为这是两种完全不同的东西。  

而资本主义这个玩意儿，最匹配的制度是什么？注意了啊，资本主义是资本至上、金钱至上、效益至上的一种管理方式。那么这种东西最匹配的是啥？最匹配的是奴隶制啊！跟资本主义制度最匹配的政治制度是奴隶制管理。所以它不会自然地进入到社会主义的。社会主义是一个更加良好的分配制度，而资本主义它走得越先进，就越趋向于奴隶制。所以很多时候你去看它的定义，直接照搬的时候是不能搬的。唯物主义很好，但你要具体问题具体分析、具体场景具体分析。  

你在进行分段论的时候，说什么青铜时代、铁器时代，在我们这里的论断时间段和在他们那里的时间段就完全不一样，依据的史料也完全不一样，你就不能放到一起来谈，你知道吧？
