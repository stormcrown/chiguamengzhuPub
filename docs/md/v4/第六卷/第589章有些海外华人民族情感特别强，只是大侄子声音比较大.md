# 第589章 有些海外华人民族情感特别强，只是大侄子声音比较大

新加坡人目前的心态——新加坡人的概念太广了，真的，新加坡人的概念太广了。因为他不是单纯地说某一批人，他的行为模式和其他地方还不太一样。他的创立者有自己的独特目标，于是吸纳的人就产生了一种很独特的心态：他们既想要当中国人，又排斥当中国人；既要使用中国的文化与特长，然后又要想办法去中国化，让自己显得中立。嗯，啥？新加坡这40多年给咱们下了很多绊子，它是一个复杂的形态。一方面因为它小，另一方面也确确实实，我们在那个时候没有能够、也没有可能给他提供什么归属感。这个事儿讲深了不好讲，直播间里面也不能讲，但是呢，言尽于此，希望大家能get到我在说什么，剩下的就不讲了。

他的立国者本身在这方面的立场上游移不定，这跟他的身份和历史有关系。他们家以前祖上就是特殊类型的豪商——特殊类型的豪商。这叫什么呢？商人是没有祖国的——商人是没有祖国的。它是一种非常欠缺自我意识的一个群体，又出现在各大强国的中间，当一个摇摆不定的小国。所以呢，他会非常强调自己身上的所谓“中立”。为什么要说“所谓”？因为你不可能中立，这个东西不存在啊！你只能是披着华人的外表去投靠到对方的时候，才能说“唉呀，不好意思，我是中立的”，不然的话你这件事儿做不到。所以那个中立都是所谓中立。但是，你只要有这个所谓中立的目标，你就会推进去中国化；而一旦去中国化，实际上就站在了另一边。但同时呢，他又借由他华人的某一种身份，跟我们保持了某一种微妙的亲近感。这属于既要又要，连吃带拿的。

嗯，然后看到美籍华人在阿美被欺负，既开心又痛苦，是正常的心态吗？很正常啊，很正常。但是这种心态之所以在这个时候显得正常，是因为之前中国的状态长时间的不正常啊——我们长时间的心态不自然。所以呢，我们在面对海外华人的时候，才会感觉到特别加深。为什么不正常？怎么个不正常法啊？海外的侨民，我们的海外侨民和其他国家的海外侨民有什么样的区别？这都来源于我们过去数百年间的不断的动荡，神州路程。

因为这种原因，造成了一个非常恶劣的后果：中国人出国出海变成了一种有负面标签的行为。你要知道，在清以前，中国人也是出海的，也是出国的。那个时候是去干什么呢？要么是去行商，有一些呢，是当汉使开疆拓土去了——各种类型的人都有，也有旅居的，也有在外定居的。他们的心态很自然，因为他们是为了讨生活，或者是为了传播一些什么东西、商品呀，然后出去赚钱呀，出去形成我们的威慑力呀——它总归都是有自己的根在中国的。根在中国，然后出去传播某一种信息、某一种资源。这是清以前的中国人的出海。

但是从清开始就变得不一样了。之前出海的那些海外的华人，由于清政府的特殊属性，他们是不肯承认海外华人的属性的，不认为那些人是自己人，非常地排斥，甚至十分恐惧他们的回国。因为任何海外华人的回国，都有可能带来大明王朝在海外所遗留下来的文化痕迹，这是满清朝廷最害怕的东西。为了这些事情，为了阻绝这样的交流，他甚至迁界禁海，使得沿海地区没有人生存，变成了无人区——就已经做到了这样的一种地步。所以海外华人从那个时候开始没了根，变成了一片浮萍。至于说成了漂泊的海外华人遭遇了什么，那就是数量上的极大萎缩，以至于今天的中国人都不知道中国人以前也曾经出海，甚至觉得中国人不敢出海，是一群自闭的人群。

而到了清末的时候，中国人又一次出海。这一次出海是为了逃难，有一些人更加是被强行贩卖出去的。这是一个并不自主的出行，是一次被迫的、没有办法的出行。出行了之后也获不得任何好的待遇，甚至人不如狗。所以不同的姿态之下出去的人，他的心态是完全不一样的。而后来呢，又陆陆续续地出国，当中包含了大量的叛国者，包括了满清朝廷遗老遗少的逃离者，也包括了各种被我们的新世界吓着了的资本、企业主等等这些革命意志不坚决的人。再后来甚至又出现了一批因为单纯的崇洋媚外、因为单纯的自恨心理而特意出去舔人的啊——特意出去当奴才的人。

所以你想一想，在这样不正常不健康的心态下游走出去的人，他当中啊，有好人，有开始意识得到自己应该干什么的人，有健康的人，但是同时又混杂了很多实际上非常反动的人。而反动的人声音通常又特别的大。所以今天的中国人，对于海外人士的认知，是非常错综复杂的。里面有大量的海外华人曾经为我们的革命事业抛头颅、洒热血；但是同时也有一批海外的所谓华人，实际上是彻头彻尾的叛国者。那么谁的声音大？谁的叫唤的声音最响亮？叛国者叫唤的声音最响，他们就靠声音讨饭吃、讨地位，所以他们的声音是最响的。你总是能听见海外华人的负面消息和恨国消息，因此心里头对他们会有一些负面的印象。然而事实并不完全如此。而其中又有很多实实在在是我们流落在外的手足同胞、亲兄弟，并且是在情感上面也有大量连接的人。所以我们既有感同身受，同时呢，又老觉得这个事儿，唉，这些人我懒得理。这种交错的情绪很自然，但是这种自然的情绪，来自于我们过去数百年间纷繁复杂、非常不自然的历史阶段。

诶，好，欢迎大家来到直播间啊。啥啊？蛇灵都不算小黑子了，都得破产，都在看。他是专业的。哈哈哈，蛇灵是非常专业的啊，所以他是我们这边黑子当中超级可爱的一种类型，你知道吧？嗯唉嗯，啥念稿啊？这玩意儿就不解释了。谁在说我念稿？我都默认他夸我啊。知道是啥吗？因为我是一个媒体人，我在干的事情叫做直播。直播的过程居然能够让他愤怒到“你这个人能不能不念稿了呀”，这就说明什么？这就说明我演说的能力、口播已经如同念稿，无稿已经如同有稿——这就是对我的绝对赞赏啊。对于这些赞美我都是照单全收的，感觉到非常的喜乐欢欣啊。然后你以后也开直播锻炼一下。好的，哎，完全可以，完全可以啊，我支持我支持。但是呢，嗯，一定不要丧失自信啊，要慢慢做，要慢慢做。每一个人都会有自己专长的领域，不同的人有不同的频段。我的频段在我这个地方，你也会找到你自己最闪光的频段。所以不要一对一的去对标，这个东西没有价值。要找到自己身上格外闪亮的闪光点，每一个人都有他自己突出的长才，发挥所长，能够让自己过得越来越快乐，成就也会越来越好啊。加油加油，完全可以上一波哈。
