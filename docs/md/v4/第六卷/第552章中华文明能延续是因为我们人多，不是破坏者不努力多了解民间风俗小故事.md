# 第552章 中华文明能延续是因为我们人多，不是破坏者不努力 多了解民间风俗小故事

很多人直觉中觉得我们的传统几乎都是糟粕，是因为我们经历了一个破坏文明的时代。倒行逆施摧毁文明啊！那么，文明是靠什么延续的？是靠人来延续的。当这片土地上的人还记得自己的基本原则是“认亲爹，不认野爹”的时候，我们的法祖系统就还保存着；当这片土地上的人还是觉得所有的文字都很珍贵的时候，那么我们的文字系统就得到了一定程度的保存。所以，有没有人破坏？有人破坏。但是延续了吗？延续了。谁破坏的？它破坏的。但是延续了吗？延续了。为什么？因为我们体量大，因为我们血条厚，因为我们的文化源远流长，因为文化的厚度和深度足够，以至于它那么深刻的破坏，竟然还让我们有复苏的机会。是它善良吗？不是。是它手段不够狠辣吗？也不是，单纯的是因为我们太抗揍了。

嗯，请区分一下政权和文明的区别，不要让坏人把它们混为一谈，误导观众。哎，谢谢。然后刚才说那个文明的延续性啊，完全可以讲一下。我们一直在说，我们是一个5000年延续不断的文明，是世界上唯一一个5000年延续不断的文明体。那么文明依靠的是什么东西呢？注意了啊，之前一直有人说：“如果你不爱这个不文明的时代，那你的文明不是断了吗？”大家应该听过哈。但是我们的文明，为什么今天很多人觉得含糊不清？为什么很多人不知道我们到底有什么风俗？为什么很多人根本不知道我们的传统到底是什么？为什么很多人直觉中觉得我们的传统几乎都是糟粕？因为我们经历了一个破坏文明的时代，倒行逆施摧毁文明啊！

所以文明是靠什么延续的？是靠人来延续的。当这片土地上的人还记得自己的基本原则是“认亲爹，不认野爹”的时候，我们的法祖系统就还保存着；当这片土地上的人还是觉得所有的文字都很珍贵的时候，那么我们的文字系统就得到了一定程度的保存。所以，有没有人破坏？有人破坏。但是延续了吗？延续了。谁破坏的？它破坏的。但是延续了吗？延续了。为什么？因为我们体量大，因为我们血条厚，因为我们的文化源远流长，因为文化的厚度和深度足够，以至于它那么深刻的破坏，竟然还让我们有复苏的机会。是它善良吗？不是。是它手段不够狠辣吗？也不是，单纯的是因为我们太抗揍了。

嗯，请问那之前女性可以出门求学、社交，进行社会生产吗？看水浒，我觉得是可以的。本来就可以。本来就可以啊。嗯，谢谢召唤最强战神啊。那个我之前分享过的一个小故事哈，其实不是故事，是属于民俗系统当中在笔记里所留下来的民俗。这种民俗记载的是当时的人怎么生活。单纯讲明代的时候的一种民俗啊，就是民间女性由于主要从事家庭等相关的工作，所以明代的女性会特别特别喜欢出门旅游。我不知道在座的有多少人看过现在的征婚广告——就是经常去相亲的小伙伴，前几年的时候是不是动不动吐槽相亲广告上有很多小姑娘说我喜欢什么呢？喜欢读书、喜欢旅游。后来都被嘲讽为“烧钱几件套”啊。那注意了，这个喜欢旅游这件事情不是今天的小姑娘们喜欢的，自古以来就喜欢。明代的姑娘们特别热爱出门旅行。但是在那个时代，没有高速公路、没有高铁，于是孤身的一个姑娘家出门是不是风险巨大？没有到处的天眼系统的时候，那真是吓人呀。那个时代一个孤身的女子出门旅行的可能性是非常小的。那这个时候怎么办呢？所以明代的这些热爱出门旅行的大姑娘、小媳妇们，就喜欢做的事情是拉一个全女旅行团。哎呀，这个听起来这味儿冲的，但是当时这是很正常的啊。就是一群女子，有几个年长的、得厚的、有名望的女子领队，然后集合在一起AA制出行。大家交旅行团的团费凑到一起之后，还会安排一个记账的财务人员来计算这个东西怎么算账，路上要吃多少、花多少，然后大家租用多少车辆、租几头驴子代步，这些东西都是常规项目。组成了这样的旅行团之后，他们会集中一起去到某一个景点去。例如写这篇笔记的人主要讲述的就是山东周边地区的女性喜欢的旅游项目——爬泰山。在明代的时候，山东附近的女性特别喜欢爬泰山，那是当地著名的景点。于是就会用这样旅行团的方式，沿途边走边玩，然后遇到了旅社、旅店的时候，他们会想办法包一两个院子下来，住得清静。之后就是去泰山——这个项目主要是立个名目，为了方便她们说“我们这就是出门烧个香，姐妹们搭个伙”，方便在家庭事务当中请个假，实际上就是出门旅行去了。

嗯，明代的裹小脚没那么严重，这个差距实在是太大了啊。一个是穿纤细的鞋子——有兴趣的小伙伴也可以到商场里面去逛一下（逛超市不行啊），去商场里面去逛一下那种专门卖时装款鞋子的店，你去观摩一下女鞋。你去看了就知道，那些特别漂亮、特别闪耀、极为摩登、一看就炫瞎了你的眼的那种鞋子，都又瘦又尖。你自己去瞧嘛，那种鞋头圆嘟嘟的可爱风的东西都不在这样的行列当中。那些特别时尚、特别前沿的时髦款式就是又瘦又尖的。现在不也是那个样子吗？你以为像我这样的小胖脚塞得进去吗？你去看那些时装杂志、时尚杂志，是不是那些模特的脚都细条条的？明代的姑娘们就好这口，她们穿的鞋子也是这样的。明代的时候还有高跟鞋呢，真的有高跟鞋，墓葬里面放着的，有现成的复原款。虽然他那个复原做得不是很好，但是确实是有高跟鞋的，鞋跟还不矮，好像有将近十公分吧。有高跟鞋，有尖头鞋。你就想想，穿这样的鞋子小胖脚塞得进去吗？那是不是要预备着把你的脚弄瘦一点点？所以要穿束脚袜嘛，就是起这个作用呀。所以明代的这个就是起这个作用。束脚袜现在的姑娘们也穿，性质是一样的，就是材质变了，所以看上去操作略有差别，性质上面是一回事。和这个时代所流行的那种把脚直接掰折了、出不了门、走不了路，能一样吗？开玩笑的呢。你当爬泰山不用脚的呢？啊，爬泰山不用脚的话，那我们还可以讲另外的一个项目。

来，在座的有没有去看苏超的呀？小伙伴们，有看苏超的来呀，扣个一。我现在真是表演精神入脑啊。啊，表演精神非常的强悍。来，有看苏超的吗？啊，不看苏超，那也有看村超的吗？嗯，不管你们看什么超啦，总之大家其实对于足球这个运动项目多多少少还是有点了解。就算是不了解，起码也有所耳闻；就算是听了之后觉得心里头不舒服，你不舒服的源头也应该就是因为国足不争气，对不对？我们这些年来，国足不争气，被全民开喷，已经喷了很久了，连海参都弄得大家名声不好了啊，海参都被他们拖累了，到了这样的程度。那么大家记得吗？历史课本里面说了，足球这玩意儿发源于中国。呵呵呵，你看了历史课本的话，历史课本里面说足球发源于中国，原本叫做蹴鞠。我就问你，它原本发源于我们这里，最后为什么会那么凉？为什么我们的队伍踢得那么次？当然了，这有其他的原因，可是这不重要。重要的是我们现在要讲的是蹴鞠这件事情。明明是自古以来，为什么我们却觉得是洋人玩得最好，甚至是洋人设了联赛教给我们的呢？然后我就问你这是为什么。那蹴鞠什么时候流行的？看过水浒的人知道宋的时候老喜欢蹴鞠了，是不是？看过水浒吧？水浒传一开篇就开始讲起来了，大家都喜欢蹴鞠，那大贪官就是因为蹴鞠踢得好啊，天天就讲这个事儿，是不是？那蹴鞠实际上不只是在宋代的时候流行啊，在汉代的时候——我忘了是哪个皇帝了，有兴趣的可以自己去找一下，他是在那个什么什么杂技还是什么什么的什么什么清什么什么记里面写的啊——那个汉代有一个皇帝特别喜欢蹴鞠，喜欢到那种早上上完了皇帝的班之后，还要想办法攒个局去踢球。汉代啊，汉代有一位皇帝就爱蹴鞠，爱到了这个地步。但是呢，他这一把子年纪了，于是朝臣们非常的痛苦，就天天苦劝皇帝说：你不要踢球了，我求求你，你这一把子年纪了，你是一个君王啊，你是一个皇帝呀，你哪天把自己的胳膊腿儿踢得疼了，我们这个上朝的事谁来主持？谁来办这个事儿呢？是不是？你要好好的上班，你不能够天天做这种剧烈运动，这不适合你。但是这个皇帝不同意，知道吧，因为他踢球上瘾，他蹴鞠成瘾，以至于朝臣苦劝他也不听。最后是怎么办的呢？那个臣子给弄了一个桌面蹴鞠啊。桌面蹴鞠我也不知道具体布局啥样，反正就是一个桌面游戏，改成了一个迷你版的桌面蹴鞠，然后让这个皇帝就玩桌面球，开始玩桌游，这样子他就不用剧烈运动了，朝臣呢，也不用担心这个皇帝哪天踢个球把自己腿咔一噶，然后第二天上不了朝，耽误国事。所以你就看一看我们的蹴鞠之风，实际上老早就已经这样子热血沸腾地流行于各个阶层。

然后我们再说到刚才不是有人说裹脚的这个事吗？啊，我们再说啊。宋的时候很流行蹴鞠，那到明的时候呢，照旧。而且明的时候有女子蹴鞠队，棒棒的、好玩的吧。你说都是裹小脚的，都爱裹小脚的？那些有闲工夫攒局的、有钱的、然后可以结社的、有时间可以出门游玩的这些姑娘们，起码是中产以上吧。多时髦多摩登的小姑娘，绝大多数都会裹脚的，因为那个东西才能让她穿流行款的小鞋鞋。但是他们有蹴鞠结社啊，他们有蹴鞠结社。所以你说那就是一回事，只是程度区别？程度区别，一个是健康坑人，一个是残疾人，你觉得这个程度的区别差多少？所以他们在明代的这些姑娘们只是穿了束脚袜。那么为什么到后来我们就觉得这个蹴鞠这玩意儿好陌生呢？甚至一直等到西方人给我们传来了足球，我们才再一次玩起来的呢？因为蹴鞠这个东西啊，非常不利于不文明时代的统治。注意了啊，蹴鞠这项运动特别不适用于不文明时代的统治，太不方便了。干啥呢？你踢一个球，一个队十几号人，你要不要经常训练？要不要结社啊？一个球队十几个人上场，再加加上替补，再加上训练团，然后再加上拉拉队，我就问你加在一起是不是30号人保底了？如果说都是中产以上，他们还要带上家里面的仆役，六七十号人同时参加活动，好不的？这叫什么？吓不吓人呐？哎，关键他还不是普通的结社，他们在结社的过程当中要踢球。踢球这个东西大家有兴趣的去看一下啊，我这个人是军事盲、体育盲，你都当我瞎说，我无所谓，但是我知道所有的足球是要讲布局的，不是大家瞎抢瞎抢球瞎踢一通。那像我一样上场了就拼命地找球在哪、球在哪追着它跑，那真正踢球的人不能这么干的吧？真正踢球的人要看一下谁在前头，我在哪个位置，和谁之间相呼应，这个东西是什么？兵法呀，亲！结社兵法演练！在这种时代，然后会造成什么样的现象？你懂的吧，亲。所以蹴鞠在这个时候没了啊。

嗯，听说他的皇帝和欧洲的君王当过笔友，能聊一下吗？目前是欧洲那边儿的啊，自己报出来的，欧洲那边自己爆出来的。这不是这边的有什么人造假的问题，是欧洲自己爆出来的啊。谢谢。
