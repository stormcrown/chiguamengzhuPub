# 第591章 温故而知新

我这个直播间里是禁止地图炮的。哎，什么“东北人都如何如何”——凡是这样的，我一定会拉黑，并带动大家一起举报。为什么？因为你骂东北人就是在骂我啊，那我绝对不能忍。所以不存在地图炮的问题，你这个事是一个虚假打靶。

那第二个问题：攻击全部的满族人。来，我们再重申一下啊——满族人是1951年的时候才出现的。在那之前是什么？是满洲群体。满族人是怎么出现的？满族人就是检讨、反省过去的八旗处事不端、缺乏正义、充满了反动性。因为承认这一切并且反思这一切，决定洗心革面，成为共和国的公民。然后我们给他登记造册，成为了我们的手足同胞——满族人。

所以满族人的特点是什么啊？注意了啊——满族人民的特点是什么？反清——能 get 到吗？满族人民的特点，和我们所有人的特点是一样的，就是反清。你要是做不到的话，那叫做“遗老遗少”，就不要拿满族人民当幌子啊。满族人民和我们能够成为手足同胞的原因，是因为我们共同携手，反抗了清王朝八旗的遗老遗少以及各种各样的罪恶。这是个根本的东西啊。

所以呢，你刚才那个就是虚空地画了一个靶子，然后就说你攻击了满族人——不存在啊，不存在这件事情。满族这个族群以民族名号设立，它的根本就是我们新中国设立之后，去推翻了三座大山，推翻了那些腐朽的统治者。所以那些跟我们一起来反对那个腐朽没落的时代、反对那些罪恶行径的人，就成为了满族同胞，是我们团结的对象，也是我们的手足兄弟——就是这样的一个来头。

至于说，你不能批清——你一说清不好，那我不跟你玩了。你这啥心态？你这是遗老遗少心态，你这是八旗铁蹄心态，这可不是什么好人啊。你琢磨琢磨自己的立场：符合天道伦常吗？符合人性吗？符合道义吗？符合忠义吗？所以我们认可的美好的品格，你认可吗？啊，这些东西是非常非常根本的玩意儿，是不能够排斥在外的。
