# 第509章 劝穿汉服的美少年多上上街

我这是一个专业带货的直播间，是不是？我们这里是专业带货直播间啊。嗯，会议制度是不是自古以来的？汉代就有会议制度，明代还有吗？嗯，是宗族制度吗？会议……你想说会议还是议会？会议的话应该是哪儿都有的啊。然后这种类似于议政的制度呢，是始终都有的。它从哪儿来的？它从部落时期来的。议政的制度是从部落时期开始的。部落管理的时候，通常都会采取议政的方式，尤其当它进入到初代的部落联盟之后。为什么会这样？因为部落联盟虽然推举了某一位盟主，但是每个部落都带着整部落的力量加盟，所以每一个部落的部落主都拥有客观的实力。这实力是自己带来的，不是盟主授权的，所以他们都有话语权。因此，在部落联盟体系内，大家的决议要通过各个部落主的同意才能最终执行。这就是各种议政制度的雏形。所有部落联盟在初阶状态的时候，没有达成集权，都是有议政制度的啊。

然后这是什么？现在感觉有人故意将明末的女真和现代绑定。我不知道你说的是哪一段，我不知道你说的是哪一段。但现在很明显，在互联网上的风潮中，有人在努力地、而且是境内外联动啊——注意了，是境内外联动的——要把中国人的历史形象、中国人的历史传统锚定在清这一个时间段。所以一直有人很执着地要把清定性为我们至高时刻，就是所谓的集权的高峰期、封建统治的高峰期之类的。这一整套很多内容实际上进了课本，大量的人都相信这事儿，说清就是我们封建统治的高峰、集权的高峰，巴拉巴拉的东西。他们一直努力做这个事，已经花了几十年时间向所有中国人宣传，说清是什么什么的高峰。实际上是什么呢？侵蚀我们所有东西的洼地，是我们一切东西——无论是制度、文化、审美，还是富庶的程度——全部都是洼地，是至暗时刻、至低点。但是他们说这是高峰。为什么他要将中国人的历史和这个时代高度绑定？最后就是，那小辫儿必须是中国人的形象。你会发现全世界都执着于把中国人的形象定性为清那个时代的形象，把中国人的传统定性为清这个时代的传统。尤其是这一两年汉服复兴运动如火如荼之后，这衣服里头就掺杂了大量的清的元素。无论是那种乱七八糟的镶边、刺绣啊，这是审美上面的入侵；还有那种更刻意的，就是给你服装改款，一定要让那种清初的经过了改变的、那些似是而非的、已经不太像汉服的衣服，硬生生地给掺杂进汉服的系统里做中转站。这个是一个非常刻意的行为，已经刻意到了什么程度呢？有一些传统汉服的大店已经不在自己的介绍中使用“交领”二字了。谢谢。啊，现在已经开始出现了这种局面了啊，交领不让提了！为什么？因为交领是汉服中最经典的礼服元素啊。所以他不让你说交领；做了交领的时候，他说这叫“直领”，硬给它改了个名字，就是不让你用“交领”这个概念，因为交领过于典型。

嗯，周朝的国人大会成型于秦的博士议政大会，往后又发展成汉代的集议制度。你是说那个乡校吗？啊，就是类似于乡校议政这样的东西吗？乡校议政这种类型的制度在中国这里历史就比较悠久，是因为我们这里走入了平民化时代，甩脱了原有的奴隶制管理方式。在非奴隶制的情况下，会出现一个群体叫做国人群体。国人群体其实就是属于有编制、有户口本的人——在整个国民系统当中获得了登记的、有福利待遇的这一群人，就是国人。那国人就有国人的待遇，国人就有发言权，这个就是议政的制度。到后来就变成了乡校议政，各个地方场合总之给你安排一个地方，大家可以出来公共讨论。这种议政的制度一直都是存在的哈。所以为什么会出现国人造反？因为别人也没有这种根本利益绑定的关系啊。

嗯，古代有技校之类的吗？如何平衡技术秘密和生产力规模的矛盾啊？和今天的技校不太一样，但是古人确实有相关的一些机构，是培养公共的技工的。那它主要是官办机构，在秦国的时候就存在。秦国属于一个那种管理制度吧，其实放到今天的话，就是国有制有点超标的状态。秦国为了践行它的军功授爵制，搞了一套东西：所有的土地都是国有的，然后进行分配；同时所有的工匠都是安排岗位、安排职位，你能做什么、不能做什么，石匠不许干木匠的活之类的，细则规定老多了。所以就出现了这种专职的分配。在这个专职分配里面，会有专门的国家技术主导人员去进行这些技术工匠的管理。而这个里面，由于它是有代际传承的，按照户籍的方式进行父传子；而它又是一个国有工匠，所以技术可以得到官方培训。所以它可以算作是一种古代技校的雏形，但是和后来那种完全敞开式的技校还不一样哈。

哎，好，欢迎大家来到直播间啊。嗯，老长了，扔一段滚得太快了。我捡着一个：“以后外国人说中国人吃动物的肉就可以回怼，因为中国人没有动物风月场所啊。”现在是心理阴影了。嗯，还是少想啊。想太多了之后心里难受。我们是人啊，我们是人，不要老琢磨那些禽兽的事。

哎，来啊，啥？大唐不夜城，汉服小姐姐看不过来呀，怎么办？我能怎么办呢？两只眼睛抡圆了看！哈哈哈哈哈，抡圆了看啊。那个老多地方了，特别多的漂亮小姑娘，我也喜欢。满街都是漂亮小姑娘的时候，高兴得不得了。其实我特别想多看几个那种翩翩少年郎，你们能懂吗？啊，你们知道我特别喜欢哪个汉服博主不？我要把他的那个书带一下啊——不是那个博主的书，但是那个里头有他，陈灵均。哎，你们知道是不是？我就喜欢这种翩翩少年郎的类型。你们有认识他的，没有认识他的，可以告诉他啊，我老喜欢他了。我记得翩翩少年郎，我还记得有一本书里头有他照片。我就喜欢这种类型的。但是呢，很遗憾的是，我经常跑到那些什么汉服特别多的景区的时候，我就看不着这样的。你说这种类型的人到底会出没在什么地方？我就希望他下一次再出街的时候，能够通知我一下，我能够看一下线下。哦，我悄悄地看，我不打扰的那种。我这个人特别低调，出门的时候捂得老严实了，你们都认不出来。就、就、就这么回事。你们要是有谁认识他的，可以看一下消息。我也很想看一下街上那种穿着汉服的、打扮得很周正的翩翩少年郎。但是在景区里头穿汉服的男孩子吧，说老实话，大部分那个妆造都不怎么样，完全掩盖了他的颜值。我就希望很多的少年能够越来越多地做一些尝试，到时候我出门的时候，就能够看到更多更多的美男子，不能够永远都可着陈灵均一个薅，是不是？我到现在就记住啊，陈灵均不错，翻来覆去就可着一个薅，这能行吗？明明有那么多的美少年啊，你们多上上街让我看一看啊，我老喜欢这东西了啊。还喜欢夏油宝啊！好的好的啊，夏油宝，就那个动不动戴眼镜摘眼镜了呗。他戴眼镜好看啊，他戴眼镜比摘了眼镜好看啊。好，欢迎大家来到直播间啊。啊，你属于虎背熊腰穿甲胄的那种，你穿那可以啊。你以为我欣赏不了膀大腰圆的壮汉吗？那完全在我的审美点上！我不止喜欢翩翩少年郎，我还喜欢膀大腰圆的壮汉，所以配铠甲没毛病啊，配铠甲没毛病。我每一种都支持，呵呵，每一种都支持啊。

我觉得现在有很多穿汉服的人，包括我在内，就是因为日常情况下应用场景不是很丰富，穿着的频次不够高，于是就造成了穿的时候经常有点缩手缩脚的，身姿不够舒展。这个呢，是需要你经常穿，时间久了之后，你适应了它的结构，然后身姿就会舒展开了，它的状态就会变得更好啊。所以鼓励大家经常去穿着。汉服系统当中其实也包括了很多日常款的，并不全都是华丽的礼服款。日常款的服装完全可以在日常生活当中经常性地应用，它既舒适又方便，同时符合我们的传统审美，并且能够塑造我们自身的仪态和自信心。当我们和我们的祖先在外观形式上面更加相似的时候，我们的精神上也会更加有共通感。这就好像我们在看待秦始皇陵兵马俑的时候，突然之间觉得，哎呀，他长得好像我二舅啊！这个时候你和秦始皇之间就突然之间达成了某一个维度的共鸣，你也会突然意识得到，我们的人数千年来没有改变啊。有没有发现我这个人在念广告的时候杠杠的厉害？太棒了！虽然并没有念，但是呢，怎么一一说到广告的时候，整个人都流利起来了呢？
