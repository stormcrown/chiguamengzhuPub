# 第530章 是谁把杀害宋朝子民的歪果仁，供奉进保护宋朝子民的妈祖庙

我们来讲一下这个聊斋。它是一个绘图版本，注意了，是绘画版本，所以既好看又好念。但这不是全本，而是一个选集。聊斋选集里面有两册：一册是白话文，对照着绘图，画得也很好看；另一册是原文。我觉得原文也非常值得好好细读。我们的文言系统其实非常雅致，当它有编译的部分做对照时，理解起来更容易，所以完全应该好好看一下原文。

然后不得不说到，为什么我今天突然卖这个聊斋？聊斋讲鬼故事吗？主要是它的作者叫蒲松龄。我不是前两天刚喷过蒲氏家族——就是宋代的那个蒲氏家族吗？结果有人说：“那你是不是怎么怎么地蒲松龄了？”谁跟你说的这事儿啊？蒲松龄是什么人，蒲氏家族是什么人，能尬到一起去吗？蒲氏家族是外国人，他们是音译之后参照了中国既有姓氏，蹭了一下姓蒲。等到他族灭了之后，残部直接噗一下子就给自己改姓蒲，有啥不能改的？然后再噗一下子又给自己改姓陈，有什么不能改的？他们一个音译名，一群外国人的后裔，姓啥不能姓啊？这玩意儿干蒲松龄什么事？硬是要扣到他脑袋上，那就是不准说“菩萨家母猪”嘛。我以后隔三差五就得说——现在蒲氏家族的名号竟然摆进了妈祖娘娘庙里面，这简直就是数典忘祖的一个典型表现。妈祖娘娘是来保护我们大宋子民的，没毛病吧？妈祖娘娘本来就是保护大宋子民的，没毛病吧？蒲氏家族出卖的就是大宋子民，没毛病吧？结果让蒲氏家族供进了妈祖娘娘庙里面，这真的是……你骂谁呢这是哪里的事？妈祖这也同意吗？哼，哪儿哪儿的事儿！真有啊，真有，不忽悠你。你们就搜我说的这几个关键词，自己去找，就这两年的事，很新鲜，新鲜热辣的一个新闻啊！蒲氏家族那几个最混账的被摆进了妈祖娘娘庙里面，还塑了像，你们自个儿去看去，老气人了！啊，一共摆了俩，一个不锈钢，还有一个是谁来着？嗯，好，好，欢迎来到直播间。现在还在庙里吗？现在还在啊。你们谁注意到这件事情，可以到当地去反映，看看行不行。

祭拜郑和要隆重地摆一个大猪头。郑和家里面现在没有正经的后代，知道吧？而且呢，郑和家里面的旁支后代当中，好像确实登记了其他民族的人，所以郑和的这笔账确实不好查啊。但是郑和当初突然变成回回人的事情，本身是谁操作的呢？是一个历史学家白寿彝。注意了，是历史学家白寿彝。白寿彝这个学者呢，编撰了一套书，他在他的书里面安插了很多人的族裔身份，但他都没有一口咬死，而是用“据传闻某人是某族”这样的语境，编了很多人，尤以明代为最，包括了郑和，然后明初的包括什么蓝玉等等的人，都被他安排了各色各样的族裔身份，就是从白寿彝这个地方开始的。但是人家老爷子嘴很紧啊，没有说“据考证”啊，都叫做“据传闻”“听说”“据信”，基本上都是用这样的方式、这样的措辞去说的。所以你找到他原文也没用，人家就是自己猜的嘛，凭什么不许猜呢？可以猜呀。但是架不住这个老爷子徒子徒孙数量特别多，尽管说师尊大人只是说“听说他是什么什么”，但是到了他的徒子徒孙那里就变成了“某人是什么什么”。为什么这样？因为那已经是他的师尊大人说出来的话了，子弟就直接把这个话一引用，然后他就改主意就成功了。郑和就是被这样改过去的。嗯，好，欢迎大家来到直播间。疑问句是真好用，职业的啊！这位老头儿，他这一辈子天天都干这个事儿，就没有歇过，你知道吗？日常如此。我想到他的时候，我脑子都是疼的。

来来来，大家对国史大纲这么有兴趣哈，来，国史大纲走起啊！这本书确实非常不错，它的装订、包装什么的也都特别好，完全可以收藏一套。它确确实实是一个很重要很重要的历史著作，它不是单纯地写一本通史，它是在一个中国人觉得中国已经要沦丧、自己要成为亡国奴的时候，刻意保存的一个历史记忆包。它是一个记忆的压缩包，是一个在那种生死存亡之际，站在中国人历史绵延的角度上面去书写的一段历史。我一直强调，历史的叙事感、叙事角度是最重要最重要的东西。而由于那个时间中国人所经历的亡族灭种之危，使得这一段的历史叙事呈现出了格外特殊的氛围，所以它非常值得大家典藏一套哈。

哦，我今天看到他发的东西了，说有人私信给他，说什么呢？说你跟盟主联系联系，那不能够再批那个什么蒲氏家族。我们这姓蒲的都在跟那个满夷对着干呢，结果你在那里说蒲氏怎么怎么不好，我们都晕了头了。我们姓蒲怎么了？哈哈哈哈！哎呀，笑死我了。我什么地方说那个什么蒲松龄是有罪的？我说这话了吗？我说了姓蒲的人有罪吗？我啥时候说了？蒲氏家族当年在宋的时候干尽了丧尽天良的事情，然后被太祖皇帝直接一秃噜给秃噜了之后，他家的分支都跑路了。这个事请问是真的还是假的？是真的。能不能说啊？能不能说啊？我还特意说了，今天姓蒲的呢，他们不以姓氏为准则，那玩意儿是个音译群体，所以他们的那个姓到底是姓蒲、姓张还是姓王，对他来说毫无意义。结果突然跳出来说：“哎呀，你不要再说了，再说我怎么跟那个什么满夷去斗争的呢？”我也不知道你到底信不信蒲啊。再说了，宋的时候蒲家干了丧天良的事，后来改姓了，请问跟你今天姓蒲的人有啥关系？怎么那么喜欢接锅呢？所以实际上这个行为本身是啥意思啊？捂嘴不让人说。我倒是也知道，我上一次说蒲氏家族的时候，莫名其妙就把我给抬了。我知道你们的能量大，能量大又怎么样？今天是共和国的天下啊，我们现在还是一个人民的政权，你就堵不了我的嘴哈。

历史得出结论：历次悲剧的根本原因是蛮族的汉化不够彻底，导致思想和能力都野蛮。哎呀，你这个总结吧，我就表达一下感谢就好了，剩下的事我就不唠了。嗯，啥罪人之后要么缩首缩尾，要么勇立功德，想堵人嘴不可能。关键是实际上没有啊，因为他们家的后代大部分都改姓了，知道吧？大部分都改姓了。为啥？太祖当年做的那手还是比较彻底的，虽然没有那么彻底啊。
