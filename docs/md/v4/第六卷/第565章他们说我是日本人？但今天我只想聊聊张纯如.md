# 第565章 他们说我是日本人？但今天我只想聊聊张纯如

只怪我们之中出现了败类，自己压制自己。更可笑的是，他们把凭空捏造、断章取义、咬文嚼字玩得炉火纯青，还自诩文人，真是让人叹为观止，连连称奇。  
brighter than the moon。  
我恍惚间听说，嗯，今天是日本投降80周年。对啊，我说了，今天是一个胜利的日子。那个，呃，我恍惚着听说啊，我恍惚间听说最近老有人说原子弹的事。哎呀，这事儿这种破事真的是啊，我，我不想聊。关于说原子弹到底炸没炸，或者当时是不是演了一场戏啊，这件事情呢，需要很多很多的证据，没有证据的时候不能瞎说。但是有一件事情，我觉得可以说，就是原子弹的这件事情啊——原子弹爆炸这件事情对中国人来说，纯纯是一个负面事件。我不知道有多少人能够瞬间get到这件事啊，我先把书弹出来，这个张纯如老师的作品啊，我先把书给你们弹出来啊。  

原子弹爆炸这件事情，对中国人来说纯属一个负面事件。因为日本之败，他到底败给谁了？他到底败给谁了？他是败给中国人的抗日战争。日本败北，败给了中国人民的抗日战争，败给了我们的14年抗战。但是因为在两颗原子弹之后，他“垮”一下投降了，于是最后就变成了归功于美帝的原子弹啊。所以实际上是我们的战斗的成果、我们的勇敢与牺牲被别人截了胡了。而且事实上，如果说是我们持续性的抗战，一直打到了最后一刻的话，日本要付出的代价，绝对不仅仅只是那两个爆炸而已。这就直接导致了我们的复仇没有完成。  

今天的中国人为什么人人心口憋着一股火？因为日本他就不承认自己败给了中国人，他硬说他自己是败在了老美的手上、败在了高科技的手上。所以他才天天地跳，一天到晚地蹦跶，一会儿拜这个，一会儿拜那个，不停地恶心我们。然后搞那些文化倒灌，文化的大水漫灌——什么“撒库拉”（樱花）是中国的呀，然后改造我们的寺院，改造我们的服装的纹理，改造我们的服装的款式，把旗袍马褂说成是中华风啊。他能够这么大张旗鼓地做这些混账事情，谁给他撑的腰？为什么他有胆色在真正的战胜国面前做这种事情？因为我们的战斗的成果被那两颗原子弹截了胡。是我们靠着14年的浴血奋战把他们驱逐出去的，但是在他们的技术当中，在全世界的记忆当中，是两颗原子弹把这件事情了结了的。突然之间，那个美帝国主义变成了我们的救命恩人。呸！如果没有美帝给他销售各种各样的武器和物资、石油，日本早就废了，他们根本就是合伙人啊。  

所以认真地说起来，就是到现在为止，中国人的心里头都还窝着一股火，因为全剧中的这个句点不是我们画上的。最重要的战果，我们没有拿到手上啊。甚至于伤害我们的人民、制造了大规模惨案的731部队，里面的当事人都在美帝的包庇之下啊，善终了！善终了！你知道吗？那帮子，那帮子混账东西善终了。有不信的自己去哈尔滨，那里有731罪证陈列馆，你们自己去看。上面有一整面墙，就是那里面的那些混账玩意儿，那些反人类的东西，他们善终了，他们寿终正寝，甚至有些东西他著作等身。我们没有拿到成果，没有亲手复仇，这件事情不算完。这种破事，日本精神分裂，一边恶心人不承认，一边心里又害怕。我们报复？丛林社会啊。唉，谢谢大华夏文明。丛林社会是这个样子的，能理解吗？中国人很难感受这些东西。为什么？因为我们文明的太久了，我们文化开明的太早太早了。我们的文明史上下1万年——官方说的，我们的文明史上下1万年的文明史。我们文明的太久了，除了最近的那几百年的悲哀岁月之外，其他的时间我们去看别人，那就是看蛮子。因为文明的太久，先进的太久，以至于中国人很欠缺一个东西，就是野兽一样的感情。  

我们很难理解什么叫做丛林法则，我们很难理解什么叫做优胜劣汰，所以我们常常是很难直接理解对面为什么要做这些事情。你不能拿他们当人来理解。你拿他当人看，他确实长得像个人，表现得也像个人，说的也像是人话，但是他的行为模式是丛林模式。你不能真的拿它当做一个人来看待，这样子你不能够理解他的行为方式。它的模式其实和野兽是一样的。而野兽的模式是什么样的模式？建立就向前冲啊，快要败了就迅速地跑，被擒获了就立刻伏地，而一旦发现主人家不备，立刻就跳起来朝着后背一咬。这个东西就是野兽法则。中国人不适应这种野兽法则，所以看到对方的操作方式的时候，就觉得匪夷所思。但他们就是运行的是野兽法则。不是他学会了写字，也不是他学会了穿衣、学会了背文章，他就不野兽了的。野兽法则不是看你经史子集背得怎么样、作文写得如何，野兽法则的意思是，你心里面有没有天道、正义、朴素的逻辑。这些东西在中国这里最普通的农民身上都有，小商小贩身上也有，但是在他们那儿，知识分子的身上也没有。  

历史狠狠教训了我们：没有尖牙利爪的文明。我之前曾经分析过，就是一个人他怎么能够想起来一个问题，叫做“我是谁”？一个人什么时候会提这个问题？他什么时候能够想起来——“诶，我是谁呢？”我只是举个例子，呵呵。有客人在场的时候，你就知道我是老张家的人；然后呢，唉，如果有其他省份的人在场的时候，你想起来了，我是河南人是吧；那是有外国人在场的时候，你会想起来我是中国人；有其他人种在场的时候，你会想起来我是东亚人种。这是什么？这就是我的自我意识，取决于我有没有察觉到别人的存在。我意识到他者的存在，就会开始提问：我是谁？我和他有什么区别？我和他有什么共同点？所以像张纯如老师这样的，他没有生活在中国，他的身边会充斥着各种各样与他不同的人，于是他反而更加容易意识到一件事情：我是谁呢？这是海外华人群体当中非常普遍的一个现象。他越是离开，越是意识得到一件事儿：我不太一样。我是谁？我从哪儿来？我要往哪去？人生的哲学三问，在遇到他者的时候是最容易萌发的。而在张纯如老师身上就爆发出来了。他突然想起来我是谁，然后想我之前经历了什么，我为什么在这里，我身边的人有没有跟我一样的情感。最终他发现我是一个中国人，而别人甚至不知道中国人受过什么苦，他们也不在乎。这样的一种情感的冲击，使得他内心十分的愤闷。他想要找到真相——明明在战争当中，中国人受到了巨大的伤害，但是无人在意。哪怕你叫出来了，哪怕你提出来，他也说这有什么关系呢，与我何干？对，与他人何干呢？与他人何干，那就必然与我有关。所以张纯如老师才会在这个时候去创作这样的一部著作。他想要找到一个问题的答案：我是谁？我的祖先经历了什么？而所有的这些经历，到底与我之间有什么样的关系？这就是他为什么会写这样的一部书。  

而当他把这部书写出来了之后，遭遇到的种种的困境与舆论上的围剿，让他越发的清晰地意识得到：我是一个中国人，我的祖先是一个受害者，而加害方还好好的活着，甚至想要让我忘记这些伤害。所以这套书不仅仅是一个历史方面的著作，它也是一个人类行为心态上面的著作。人在什么时候会意识到我与他的不同？什么时候会意识到如何印证自我的存在？什么时候会知道族群的存在和我之间有必然的关联？很多的时候他需要外来的力量。张纯如因为生活在海外，所以他意识到了这一点；而今天的中国人因为国际化的大潮流、因为互联网的扁平化而意识到了这一点。这些思想都是共通的。所以今天的中国人也可以看这部书，在看到他的时候，应该会有一些很独特的情感上面的共鸣。至少说当我们看这些书的时候，会知道侵略者来的时候我应该怎样想，我应该怎样看；侵略者会怎样表现；完全与我不同的人会如何看待我，又会如何对待我；一个人如何成为一个人——作为一个个体，你将永远是一个人吗？还是你必须依从于某一个庞大的群体，你才能够有资格称之为一个人？很多的时候，确实需要有一些人处在特殊的处境之下，他才会察觉到这个问题。而张纯如恰恰好他呆在了那个位置上面，于是他发现了问题之所在，找到了这样的一个时间的节点，让我们来回看一下南京的惨痛，看到那个时间与那个时代的祖先、与那个时代的同胞共鸣。然后我们知道：我是谁，他是谁，我应该干什么，她又在做什么。不要为眼前的迷障，不要为眼前的繁华感觉到迷惑，很多的事情是有根本利益之所在的。  

所以其实我一直都在盛情推荐这套书。所有的中国人的家里面都应该有一套《南京》。你得知道，真正的侵略者来的时候，真正的马蹄踏过来的时候，什么道理都是无力的，没有人来听到。那个时候你才会意识到什么是自己人，什么是别人。不明吗？嗯，不是全称啊，主要是我不能够把全称直接念出来，很容易把直播间都没有啊。你就说多荒唐是吧？你就说多荒唐吧。我可以说作者的名字，也可以说书的简称，但是书的全称我直接说的话，直播间容易没有。嗯，好的。嗯，就是确实有很多人，因为呢，从小就看动漫，看各种各样的漫画书，然后看电影、电视剧，可能呢，还很喜欢那边的一些女明星什么的。这些事儿呢，我从小到大看的都不少。我知道有很多人很迷恋那些方面的细枝末节的文化，于是呢，就老是觉得事情过去了就过去了，甚至呢，就直接一个敷衍，说一切都是阶级矛盾啊，就直接敷衍过去了。说这个事呢，反正就这样，啥时候都有。但是不一样就是不一样啊。怎么能够了解？怎么能够直观地知道这件事情不一样？你得看到当时到底发生了什么，你得知道对方到底在干什么，你得知道当对方是一个压迫者的时候会做什么，以及对方是一个敌人的时候会做什么。否则的话，你觉得他们都是一样的，其实不同，完全不同。  

南京之后，岛国上面的各路媒体不断地宣告说，我们南京如何如何取得胜利，然后整个岛国万人空巷出来举国同欢。庆贺什么？庆贺南京。成功参与庆贺的人当中，什么人都有。不是什么上层阶级，没有阶级的问题，也没有性别的问题。不分男女老幼，不分职业贵贱，不分阶层的高低，他们举国同欢。为什么？因为在那个时候，他们将自己视为一体，而将整个中国的上下视为他者，视为别人。只要抢赢了，他就赚了。所以那些人当时理直气壮地在自己的街头张灯结彩。这个时候你不能够再去欺骗自己，说一切都是阶级矛盾，没有的。在那个时候，日本岛上面的乞丐都会庆贺这件事情，但是中国这里即使是最不像样子的军阀也会为之而悲泣。所以将一切事情都刻板化、扁平化的时候，你会没有办法正确地理解历史事件的发生原因。当我们没有办法正确理解原因的时候，你就没有办法以史为鉴。历史的进程确实走过去了，但是记忆还在，该做什么、该怎么看待问题，这些东西还存在着。不能够眼睛一闭就说这事儿已经过去了，一切都是阶级矛盾嘛。总之呢，我们要反抗一切阶级压迫，口号喊得很好听，但问题是它是真相吗？当你已经不唯物的时候，你喊那么多政治正确的口号有意义吗？
