# 第542章 衰弱的西方，能像古典中华文明一样重生吗？

西方经历衰退之后，以其文明基础还能像中国一样重生吗？我觉得这有一个问题，就是大家心里面必须得有一个概念：到底什么是文明的重生？也就是你得知道什么是文明体。哼，来啊，什么是文明体？这件事情你必须要界定明白。

例如说，我们为什么能够在国破之后，甚至在亡国许久之后可以再次复兴，可以卷土重来？它的根基到底在哪？有些人会跟你说，来自于什么文明的韧性啊，如何如何地说我们的文明高级不简单了啊，说我们的文明有同化能力，谁来谁会变成我们。不是的，这都是谎言。注意了，这都是谎言。谁跟你说这话都是忽悠你。我们的文明没有什么同化能力啊。我以前也曾经在年少的时候，误以为我们的文明有同化的能力。抱歉，我们这样的文明没有什么同化的能力。学好是特别难的，学坏才容易。我们没有什么同化的能力，因为学成我们太艰难了。要有延迟满足感，要有牺牲奉献感啊，要让自己如同大海里的一滴水，这是极为困难的事。这个东西违背了丛林法则，所以我们的同化能力是特别差的，不是特别好啊，是特别差。我们在同化这个问题上面，甚至比不上很多的野蛮群体，因为野性本身是极有同化能力的。这叫做什么啊？学坏一出溜。我们是特别特别难以同化他人的，千万不要心存幻想啊。所以不是什么文化的韧性特别强啊，学起来特别困难的东西，那就不存在什么韧性特别强这个问题。

我们之所以能够卷土重来的根本原因，是因为血脉没断。我们的人还在，父母还在，教给孩子自己从哪儿来。我们的记忆仍然存在。记忆在，人在。父母子女的关系在，亲族的关系在。只能说是这样啊，谢谢啊。这种东西在我们才会有真卷土重来的本钱。所以文明的本体不是里面的文字内容，那些东西是上面的附加品，那是一种财产。财产是不能重生的，能够重生的只有人。我们能够卷土重来的原因，是因为我们的人没有断绝，我们的人还在，传统还在，家庭的结构还在，父母子女的亲密关系还在。这个才是我们重来的本钱。

嗯，南北朝时期，北朝对南朝，哪边对中华原生文明的正向作用更大？北边的军事实力更强，南边的呢，文化的传世能力更强。北边儿在军事实力占优之后啊，形成了政权之后，会推行所谓的汉化。是不是很多人觉得汉化理所应当？不是的啊。之所以北边的政权在军事崛起、掌控了军事核心和生产要地之后，能够推行全面的汉化，其实是因为之前有南方一直保存着文化之火，他一直拥有着文化的向心力，一直拥有着文化的样板间。我一直强调样板间这个东西很重要。你去买房子的时候要不要看样板间呢？要不要看户型呢？要不要看它长什么样子的？因为南朝它保存了文化的外观和形态，所以别人呢，能够看到：哎呀，这个样子好看，这个样子漂亮，这个样子精致。所以他在军事技术生产力占到了一定优势之后，会模仿对方的样子推进汉化。所以谁的贡献？他们来自于不同方向上的贡献啊。
