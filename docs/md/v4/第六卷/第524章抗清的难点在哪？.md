# 第524章 抗清的难点在哪？

抗清的难点主要在哪？好，我待会儿讲这个事啊。虽然之前聊过，但是没关系——你付费了吗？呵呵，为什么这个样子？因为清比元更反动啊。注意了啊，清借鉴了元？不是啊，清把元当做反面样本，它借由元进行了大规模的反思。所以啊，注意了：清之所以统治那么久，是因为它特别坏，是因为它特别反动，是因为它特别有自知之明啊。

来，注意那个啥啊：那个清之所以能够维持那么久，它的原因就是因为它坏，而且足够的坏，彻头彻尾的坏，而且对自己的坏和恶毒心知肚明。这个东西就是清之所以能够延续那么漫长的时间的根本原因。因为它知道自己的反动，因为知道所有的人民都痛恨它，因为知道自己是错的，所以它所做的一切努力，都是为了消解对方的抵抗能力。

你要知道，当一个人觉得自己“唉，我坐在皇帝的位置上是理所当然的”，然后认为“天下的子民都爱朕”，当他有这种错觉，当他觉得自己有正义感的时候，他的手就会很松，他就会放权。他会说：“天下都是朕的地方，天下都是朕的子民，唉，放手，你们去好好学习、天天向上，你们去发展高科技，努力努力地去上进。”但是注意了，只要发生了这种情况，只要民间是热火朝天的一片干劲儿，那么当他发现说“这个皇帝不行，我要造反”的时候，他的力量就会非常之强，中央王权就会制不住，知道吧。

那么清为什么那么难以反抗成功？反清那么困难？为什么？因为清太知道它多坏了。所以呢，它视天下所有的臣民为仇寇，所以它从一开始，所有的统治伎俩全部都放在“如何让他们造反不成功”这事上。它花了200多年的时间就琢磨这一件事情，别的事它根本就不管。你自己去看那个——嗯，清王朝的时候，北京城的照片，你会发现那个房子啊，不是什么穷的问题，它的房子不是穷的问题，而是年久失修。摆明了它的200多年没有搞任何的生产建设，它根本就不考虑生产，不考虑建设，不考虑发展。一切正常国家要有的东西它都不想，它只想一件事情：怎么能够让老百姓造反不成功。

嗯，谢谢“牛头人的传说”。参考阿美一天16个小时，打三份工还要输血，钱都发不起，但是没人抵抗啊。阿美的话还不太一样啊，阿美那个人心它根本就拧不起来。阿美的人和人之间没有任何的共识啊，它把自己做成一个移民国家之后呢，就没有形成凝聚力的可能性了。彼此之间没有共识的时候，怎么团起来？不团起来怎么对抗你？不可能的啊。阿美做不到这个事，它没有任何可以革命的基本盘，它的基本盘太小了，它隔离不了。

但是清不是啊。清很清楚，它必须倚重于汉人，和汉人周边的那些与汉人很接近的群体。它必须倚重这些人的生产能力，不然的话，它跑这儿打劫，劫谁去啊？大家都在那浪荡，都是劫匪的时候，你抢谁去？没得抢。它必须倚重于汉人，和与汉人高度关联的这一群人，必须倚重他们才会生产。但是同时这些人也是最有组织力、最有造反精神的，怎么办？所以它所有的脑筋都用在了“如何让对方只劳作，不能造反”这个想法上面。

所以第一，它保存了它所有的核心战力，就是八旗子弟。它从开始到结尾一直都在供养八旗子弟，尤其是满八旗；汉八旗它是会踢出去的，因为它信不着。它知道汉人无论变成了什么样子，最终都有反抗的意识，所以它有机会的时候就把汉八旗踢出去，经常性地安排出旗，尤以乾隆朝出旗的数量最多。它们是会把汉八旗那些当年投降了的汉奸们踢出旗去，不拿福利、开除编制。但是满八旗它一直都是在养着的。它一直在养着满八旗干什么？因为你拿了谁的军饷就要为谁服务。它同一时间禁止满洲八旗参与到社会劳动生产环节，就是不准经商，不准工作，不准有其他的收入来源。你只能接受朝廷的供养当军人，所以满洲八旗一直都是军事化管理的。尽管这个战斗力每况愈下，但是啊，战斗力下降了，战斗意志并没有下降。因为他只拿朝廷的钱，只拿皇帝的钱，所以他只效忠于皇帝，皇帝让他去干什么，他就去干什么。因此满洲八旗一直到最后都在为他们的皇帝而战。

它手里有军事力量。军事力量可以干什么？它很清楚自己的反动。而养这样一拨军事力量和他们家属又耗费巨甚。养不起，怎么办呢？别人会缩编，它不能缩编，它太反动了。所以它就会用这些八旗的兵马去敲骨吸髓啊，去勒索全国，去让所有的人集中起来供养那八旗。这就是经济上面的深度盘剥。它通过军事的力量保证了他的盘剥能力，然后就供养它的八旗，形成了经济上面的循环。

然后同一时间，为了能够发挥最有效的战斗力，它采用的方式呢——为了实现它的这种镇压，它设立了满城，也设立了各种各样的低级军事据点啊。这个叫什么？这个叫做网格化管理啊。注意了，这个叫做网格化管理。它就是在整个中国的各种道路上面，打上各种各样的关卡，然后在关卡上面就给你设立了一个极少量的满洲八旗兵，然后呢，监管一批绿营兵，形成一个小军队。一旦发现周边哪个地方有造反的迹象，它立刻就调兵扑灭啊。所以呢，它的这种军事力量一直都是安插在全国各地进行内防的，所以它没有边防啊。为什么打外战的时候打得都跟坨屎一样的？就是它吹得特别特别牛的什么啊，准噶尔，吹得简直开疆拓土不得了了——打了多少年？打了快100年了吧？为什么打那么久？为什么打那么惨？打了一波又一波地打一个小部落，而且当时的准噶尔实力很差，早不是全盛时期了，内部崩坏不说，而且瘟疫横行，而且腹背受敌，那一边准噶尔还面对着大俄呢。就这种局面下，它打了快100年。为什么战斗能力这么垃圾？因为它不敢大规模调兵啊，它根本不敢大规模调兵，它的兵马必须留在内地去镇压农民造反。

嗯，诶，好，欢迎大家来到直播间啊。所以它为什么能够统治那么多年？为什么反抗它特别困难？因为它太坏了。当坏人对自己的恶毒心知肚明的时候，它会把所有的心眼儿用在“反对你造反”这件事情上面。所以呢，清说不上它有什么统治的方法，它就是镇压造反的一个魔法啊。这句话谁说的来着？我忘了。诶，好，欢迎大家来到直播间啊。
