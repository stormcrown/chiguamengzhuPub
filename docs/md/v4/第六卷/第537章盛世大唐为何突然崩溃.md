# 第537章 盛世大唐为何突然崩溃

“两脚羊”这个词，大家听过吧？啊，两脚羊，大家应该听过。来调查一下：两脚羊这个词出现在什么时代？大家扣在公屏上好吗？啊，两脚羊这个词出现在什么时代？来，你回忆一下，不要去问浏览器，不要去问AI，自己直接打字放到公屏上面。出现在什么时候？嗯。

好，主流的想法当中，包含了两个主要的时间段。一个是五胡乱华时间段——如果你在说南北朝的话，那也相去不远，大概是在一段时间里面，我们就算作同一类。然后另外的一个类别，就是出现在从晚唐到五代十国的这一段时间里面。当然，也有人说出现在宋，哼哼，行吧，这个时间段相距也不是很远。好的，那么我们就大体分成了两个时间段：一个是在五胡时期，一个是在五代时期。那么两脚羊到底是哪一段的呢？五代时期。两脚羊这个词明确的出现是在五代时期。五胡乱华的时候，有没有把人炖到锅里面的？有啊，我明确的说，五胡乱华的时候是有吃人的行为的，也是会炖到锅里面去的。但是我们过去的历史上最黑暗的一段时间是五代十国。为什么这个样子？根源就长在唐身上啊，根在唐那里。为什么五代十国的混乱要比五胡乱华还要惨烈？这个事儿就跟刚才我说的，李唐王朝将名义拱手让人，这件事情有极大的关联。

来，在座的小伙伴们开始精神起来了，提提神，提提神，快点啊，开始进行调研了。觉得“盛唐万邦来朝”特别美好、令人心生崇拜的扣1，嗯，反之扣2。哦，看到大家的反馈了啊。那现在呢，给大家反馈一个信息：如果是放到20年前，任何地方去做调研的时候，说你爱大唐或者是不爱大唐，管你怎么想的呢，反正你必须得爱大唐。你不爱大唐，那就属于是你不爱中国——你这个人连中国最最伟大的时刻你都不爱，那你肯定不爱中国吗？那今天就来聊一聊啊，聊一聊关于这个盛唐的这件事情。要知道这二三十年来对于李唐王朝的赞美，他到底赞的是什么啊。有很多人可能到现在啊——我知道可能有很多的这个，嗯，特别热爱李唐王朝的人会说：“赤冈盟主说了唐王朝的坏话啊，说了李世民的坏话，所以我要跟他决裂啊。”有没有这样的人？有啊。但是呢，你决裂就决裂呗，呵呵啊呵呵，绝就绝啊，我这个人没认得几个人，所以无所谓了。但是为什么今天要着重从唐王朝身上开始？就是因为对他的这种赞美本身是有问题的。不是说李唐王朝有什么重大的缺陷，而是对他的赞美的点的选择是有问题的，他是选择性的赞美。赞美李唐，赞美的是什么呢？赞美的是李世民的天可汗的身份，而不是天策上将。这两个身份之间有什么样的区别？天策上将的身份是开疆拓土，同时配套的就是杀戮——无数马蹄踏过的地方都是我的势力范围，这个东西叫做天策上将。而天可汗的这种赞美是指什么呢？他不只是中国人的王，也是胡人们的王。所以这些年以来对于唐的盛赞，其实他赞美的并不是真实存在的那个盛唐，他所赞美的是想象之中的一个盛唐。这个想象中的唐长的什么样子呢？就是这个故事哈——看起来发生在中国的土地上面，但是呢，上面的一切故事的运行逻辑和行径，都跟上世纪90年代时期的阿美是一样一样的。什么样的国家？这几十年来所吹嘘的那一个盛唐，和实际的盛唐是不一样的。实际上的盛唐，他最伟大的东西在于什么？在于金戈铁马啊。我不知道有多少人能够直接地get到这一点。盛唐最伟大的业绩来自于他的金戈铁马，来自于他的大杀四方，这是他最了不起的东西。而他之所以强盛来自于哪呢？来自于他的大杀四方——这个是李唐王朝强盛之根源。

那么请问他大杀四方这件事情靠的是什么？靠的是他的基本盘啊。要知道当时能够崛起的各方势力，包括陇西李氏这样的各方的势力，这些世家门阀们，他们实际上都是割据一方的军阀，军政一体的军阀啊。他们当年在发迹之前都是割据一方的军阀。所以严格来讲，他的这种扩张啊，它的这种持续不断的扩张，是一种军事的扩张，是一种用自己的军事的基本盘扩张出去的一种场景。那么这种扩张是他实力的表现，但是福祸相依——就是当你扩张到一定程度的时候，军队不够用了，于是呢，他开始收纳雇佣兵。唐王朝是有很多雇佣兵的，有很多归附部族的佣兵，所以他才开始出现了后世所吹嘘的所谓的开放又包容。所以李唐王朝他的强盛来自于哪里？来自于它基本盘的南征北战。而他后来的衰败，则来自于他在南征北战的同时，不断地招募与自己并不兼容、并不匹配的各种佣兵，最终形成了一种非常复杂的内部结构。而这种极度复杂的内部结构，让他在短暂的盛极之后，迅速就崩塌了。任何一个国家在盛极的时候都有可能会面临到走下坡路，但是李唐的崩溃，崩得特别的彻底，特别的粉碎。而且他的这种崩溃的方式造成的一个现象，就是李唐崩溃了之后，遍地都是小唐啊——种种跟他完全不兼容，甚至背道而驰的，都开始自称是李唐王朝的继任者。为什么？因为他当初真的发放了准入许可证。这就是李唐王朝。他一直被人家吹嘘，吹的是什么呢？从上世纪90年代开始就一直在吹唐王朝，他吹的是什么呢？他吹的是那个想象之中、在千年以前出现在中国土地上面的阿美。所以本质上面他在吹阿美，吹的是什么呢？万邦来朝，多元、开放、包容、宽厚一体、大量的移民。这个就是从上世纪开始至今，我们对于李唐王朝在舆论上面各种各样的赞美，最集中的领域就在这里了。

有人说：听说燕云十六州的丢失，导致了蛮夷部落的跨越式的升级，后患无穷。这只是其中的一个因素啊，不光是燕云十六州。燕云十六州呢，是使得我们失去了咽喉要地的同时，失去了非常好的马场，这固然是一个方面。那另外的一个方面啊——注意啊，我，我不是说燕云十六州不重要，而是说它和另外的内容同样重要——就是李唐王朝当初在这种快速的扩张的过程当中，消化吸收的能力严重不足；然后由于在军事上面过于倚重外来的军队，所以就造成了一种现象：他必须接纳这些部落的内附，必须要安置对方。而一旦安置下来，你能够什么都不给吗？能够盘剥吗？不能够。一个这样强盛的中原王朝，是不可能把对方当成一个奴隶一样的盘剥和压榨的。于是呢，当他将对方视为准国人的时候，问题就出现了：你的文化要不要交给他？你的技术要不要交给他？你的工匠要不要在他的地方工作？所以在唐王朝崩溃了之后，以前一直在我们手里面所掌握的核心的技术，开始出现了大规模的流散啊；那么优质的管理技能和管理思想，也开始大规模的流散。而且最重要的一点就是，这些东西呢，我们凭自己的本事还是可以逐渐的升级换代的，可以重新超越他者啊——毕竟我们是真正的原创方。但是呢，更重要的就是“唯器与名，不可以假人”。但是李唐王朝将“名”这个东西送给了别人啊。“名”是什么？“名”是名义正统权啊，“名”是指谁可以掌握国之重器，这叫做“名”。李唐王朝的最大最大的问题就在于，它实际上——不管他是不是发自本心啊，事实上他不是发自本心的——知道事实上李唐王朝不是发自本心的，他们并没有想要做这件事情。但是过于快速的扩张和消化力的不足，客观上造成了这种后果：就是他将所谓的正统权、所谓的名誉权这种东西扩散出去了。而名义一旦被扩散出去，你就会发现其他的地方，他就可以说“哎，我就是小中华，我就是小唐，我就是唐的继任者，别人都是啥啥啥啥啥”，他就可以编出无数的故事出来。这就是你的名义权在李唐王朝身上开始失去。

所以你才会注意到一件事儿：唐崩溃了之后带来的这个乱世，其实是超越了之前的任何时代的。为什么要这样说？要知道当年的五胡乱华是什么？其实就是东汉的时候呢，招募劳工啊——很多人说他是后来那个我们弱了他们就进来了，不是的啊，注意了，当年的五胡，他们是东汉时期开始陆陆续续引进的新移民，他们的身份是合法身份啊，是被招募进来的合法身份。有一些是雇佣农，有一些是雇佣兵，当然雇佣兵的占比更高啊，雇佣兵的占比更高。自东汉开始陆陆续续的，一直持续到西晋，都有招募五胡的雇佣兵和一些雇佣工匠啊——工匠谈不上啊，但是有雇佣农这些，就是在那个时候引进来的合法移民和劳工。后来五胡乱华的事件是怎么发生的？因为本土开始变得脆弱了啊，没有强而有力的威压的时候，然后就轰一下子，自腹心之地开始出现巨大的变数啊。这个呢，就是五胡乱华啊。但是为什么他当时的这种破坏力，显得没有五代十国那么狠？因为无论是两汉还是魏晋，正统权这件事情都是一点争议都没有的啊，这个时期我们在名义权上面是一点问题都没有的。所以五胡也非常清楚自己在干什么。最后就会造成一个现象：尽管说他们趁机搞起了零元购，而且呢搞得千里无鸡鸣，但是最终他会意识到一件事情——哎，我只有用同样的一种方式啊，我必须用原有的那种管理方式，才能够让自己也让自己治下的人过上好日子。所以五胡走到晚期，大家打得头破血流，内部分崩离析，没剩下多少口人。然后彼此之间的实力再次出现逆转的时候，这些五胡呢，又开始回过头来啊，开始努力的汉化。这个过程是经过了血腥残酷的斗争才达成的，不是主动的，但是他最终还是达成了。主要的原因就是在那之前，我们没有失去任何名义权。所以呢，谁是先进，谁是文明，谁是主体，在那个时代非常清晰明了。所以这个乱局才可以通过这样不断的斗争开始发生改变，局势逆转，最终出现了隋和唐啊，为之前的乱世画上了一个句点。

但是到了五代的时候，局面就不同了。如果有谁去参观过一些博物馆的话啊，在西部地区——我上一次在山西哪个哪个博物馆里面，我一下子想不起来了啊——在这种博物馆里面，你会发现他会去盛赞说“我们辽朝，我们辽代”啊，然后呢，在某一些博物馆里面会说“我们金朝，我们金代的文物”什么什么什么什么东西啊。这种是什么？这是自我认知，这是什么？这就是名誉权啊，这就是名义权。但是当你真的去细看它的时候，就会发现这个名义权上面是有问题的。为什么要这样说？要知道当时辽所留下来的很多的建筑是很有唐风的。他之所以宣称自己是李唐王朝的继任者，宣称自己是正统王朝，其实呢也不是毫无依据啊，它的里面呢确实保留了很多从唐获得的文化、技术、工匠，包括在他的建筑的格局当中，他所修建的寺院的那些建筑当中，你都能够看到唐代的风范。但是问题就出在这儿了：当你把辽和宋一做对照的时候，就会发现宋啊，它在技术、在审美方向上面是在迭代变化的，也就是他是活生生的改变了、调整了。但是辽像是冰箱一样停滞下来了。有兴趣的小伙伴可以去观摩一下辽所残留的一些寺院的遗迹遗址啊，你去这些地方找导游，去给你介绍的时候，他也会告诉你它完整地保留了唐代的记忆。听起来像是一种冰箱论，但是实际上它代表的是什么？就是他没有进步。唐王朝崩溃的时候，它长什么样子，它就长什么样子。什么局面才会导致一个地方，明明之前的王朝已经崩塌了很久了，但是呢，它依然和原来的模样一模一样的在搞复刻？因为这个东西不是他原创的，他是搬过来的，他不知道原理是什么，只知道可以这样做，这样做就是对的，所以他不改，所以他不变，所以他没有进步。这是什么？这个就是你的名义权授权让出去了。所以呢，大家抢的时候都会觉得自己是对的，别人是错的。那么这一种就不是简单的排位赛，打完了之后就可以继续了，而是你死我活的。所以为什么在五代的时候，情形要比之前更加的惨烈？这件事情最重要的根由就是：唐的快速扩张和消化不良，最终导致了名义权的外泄。
