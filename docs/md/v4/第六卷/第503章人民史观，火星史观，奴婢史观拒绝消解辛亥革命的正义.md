# 第503章 人民史观，火星史观，奴婢史观 拒绝消解辛亥革命的正义

为什么说辛亥革命不彻底？辛亥革命没有能够完全成功，它成功了一部分，但没有完全成功。有人可能觉得这是黑子的说法，但我倒觉得可以聊一下，客观看待历史，别陷入个人情绪。要记住，看历史不能没有立场。注意，我是拿这话当反面教材来看的。看历史一定要先找准立场。在座的小伙伴们，除了黑子之外（黑子不用管），所有正常的人一定要记得：看待历史，务必要先找准历史的站位，务必要先搞清楚一件事——我是谁。历史叙事是一个记叙文，记叙文是要有主语的，包括时间、地点、人物、事件。你必须要有一个视角，是谁在看待这件事情。如果搞不清楚这一点，那么世间的一切都好像如同蝼蚁一般的人类在演戏，那是遥远的火星视角。还有另外一种视角，就是赢家视角——谁赢了我就站在谁那边来看书。这很好笑，是一种奴婢情结。正常的人，我们这种正常的人应该怎么办？我是谁，我就站在哪一边来看。如果两边都不是我自己的人，那我就站在我自己的道德正义观这边来看。总之，你必须找准“我是谁”。一个人没有自我认知的锚点时，看东西都是混乱的。

然后，他的后半段我也觉得可以拿来当反面教材聊一下。这个小黑子品质还蛮高的。他说“不然你会学孔老二‘人分三六九等’，三跪九叩”。显然，他对孔夫子毫无了解，但他所描述的这些人的形象，就是被清朝篡改后形成的、假装为儒学的“狗儒”们。孔夫子当年所推行的是：任何一个人通过学习了之后，都有机会成为士人，都有机会参与到社会的公共管理，都有机会实现社会阶层跃升。这就是孔夫子当时设立的学习班起到的作用，所谓“有教无类”。他是一个老师，教的是政治经济学，要求从他这里结业的学生就可以成为士人，士人是社会中层管理。所以，他所说的“三跪九叩”这个东西有没有？有，但只在祭天帝行大礼的时候才有。周礼这套礼制中，为什么有各种各样不同的礼节？这种礼节象征着你的远近亲疏，象征着你在社会上承担着什么样的责任，同时也匹配着什么样的权利。例如，哪一个诸侯、什么级别的诸侯，在吃饭时可以看几个人跳舞。听起来是多余的事情，但这相当于是你是什么职称，就有多少工资，跳舞属于福利待遇。你是什么岗位，就可以开什么车、穿什么衣服、用什么方式出行，代表了你的身份，既是权利也是责任。这就是周礼里面的社会管理结构。所以，所谓“三跪九叩”存在，但仅在特殊的行大礼时。其他时候，中国人自古以来日常的行礼都非常简约，因为我们不会去折辱他人，我们不是奴隶制，不会让人平白无故地跪在地上。那么，什么时候中国的跪礼特别猖獗？两个时代：一个是元，另一个是清。中国人的跪礼特别猖獗，见到一个上官就要跪地磕头，离我们最近的时代就是清。为什么？因为它践行奴隶制。

谢谢“秦皇岛”的留言：“如果您想看风景，那就好好看风景，只是这风景的悲欢离合、喜怒哀乐与你无关。”我们直播间从来没有这么文艺过，你竟然给我写这么文艺的东西，不担心我看不懂吗？都说好了，这个主播的文学水平不行。大家晚上好，刚才说的是什么来着？刚才我看到有人想聊一个东西。我给大家推荐一本书，稍等，我拿一下——这本《国学易理》我推荐过几次，我觉得这本书非常好。为什么我在讲到这段时去说《国学易理》呢？因为刚才我专门拿来当反面教材的那个小黑，我解析了他所说的“你不能，你要客观”。现在互联网网友已经明白了：当一个人说你要理智、中立、客观的时候，基本上就是要求你背叛自己的群体。现在大家意识到了这一点。我总结性的公式就是：任何一个人在面对这个世界、看待历史、看待他人的时候，都需要自己的站位，需要自己的坐标原点，都需要先问自己“我是谁”，这是第一要务。如果前面的几方都与我没有直接瓜葛，你依然要意识到“我是谁”这件事，我要用我的价值标准来看待，这就是以我为圆心去看待世界，这叫做立场问题。为什么要强调这种叙事？因为强调这种主语叙事，我们的历史就与你有关系了。当你去看岳飞殉难的故事时，主语是“岳飞是我们的前辈，他是为了保护我们的祖先，为了保护我们的土地，为了保护我们的国民”。这个主语存在时，这种视角本身存在时，你的代入感就是他是大英雄。但如果你把这个抛开，再用地图视角看“岳飞直捣黄龙，他是想打我的”，这时你看岳飞就是敌对势力。所以，我们如何看待历史？先问一件事情：我是谁。

那么，“我是谁”如何订立一个基础？就是要了解我们的祖先。为什么我特意强调要先了解祖先才能做到？正常人确实不需要，但我们真的需要。因为这些年来，大家有没有看过一本叫《中国人的劣根性》的书？或者至少知道“中国人劣根性”这个说法？知道的扣个1。与此同类的还有“你不可能是汉人”、 “秦始皇是说蒙古语的女真人”、还有《狼图腾》——它的影响力非常大，我觉得很好，这本书呱呱地摔了耳光子之后，让很多人稍微清醒了，知道那种叙事有问题。这些类型的书一以贯之传递的信息就是：我们身上有劣根性，这个劣根性是自古以来的，我们以前就很糟糕，留下来的传统都很糟糕。就像刚才的反面教材说“你就是学孔老二，人分三六九等”，既不考虑远近亲疏，也不考虑时代背景，直接来一句“孔老二，人分三六九等”，句号，然后就反动了。这种刻板印象，这种向几千年历史泼莫名其妙的现代化脏水的操作手法，完全欠缺历史唯物主义，以今天高高在上的态度去要求古人，只要古人没有达到今天的水平，没有让我用上iPhone，那就是落后的、愚昧的，甚至是愚蠢的。这种叙事主要打击的是我们对自己的自我认知。这些年很多营销号也致力于做这个事情，就是所谓的污名化，包括之前攻击我的时候说“黄汉都是反贼”。“黄汉”是我们自古以来的一个词汇，从来都是干干净净的，完全正面的词汇，它给你玩污名化。还有像“同志”、像“菊花”、还有“小姐”，都是纯粹的正面词汇，被污名化之后，你就觉得我们以前的古人好像很龌龊。当你真的相信“菊花”全部都是你想象中的那种意象，再去看陶渊明，会不会觉得心里打鼓犯恶心？这时候你就会觉得两千年前的人好烂，进而产生一种“我们的过去不堪回首”的心态。这本质上也是一种对过去的混淆，同时造成一种现象：对于我们自己的过去、自己的文化，我们并不了解，甚至很多时候了解得可能还没有老外多。因为有一部分汉学家虽然不安好心，但研究真的很深入，他们怀着敌视的态度去研究，为了破坏我们而去研究，知道我们以前有多好，只是不跟我们说，或者只说一半。可是很多中国人真的不了解。

那么，我们要如何知道我们的过去？就得知道到底什么是国学。为什么要专门挑这本《国学易理》？因为国学这玩意儿，骗子太多了。从上世纪大概90年代开始，对面那个岛上面兴起了一帮子国学大师讲课，搞了很多国学班，风靡一时。里面有没有好的？多多少少有一点点含金量，但骗子忒多了。当你很想知道、很想了解，结果进去一个是骗子，进去两个又是骗子，你就会觉得以前真是垃圾。而一个人觉得过去都是垃圾时，你不会觉得自己的现在很好，也不会觉得未来很有希望，很容易出现依附于他者的奴婢心理，觉得别人很强、很美、很对、很文明。这种心态对中国人的破坏非常严重。所以我推荐这本《国学易理》，它是讲解国学的本源、国学的基本逻辑。什么是国学？国学里面应该包含什么东西？你应该怎么去看它？很多之前关注的博主，为了吃饭说话，感觉虽然我们基数庞大，但缺少先锋和执牛耳者。我们缺少自我认知，很多我们自己的好东西会被别人拿去研究，然后话语权被删减后再教给我们，夹杂了一堆糟粕。这确实是一个现实，已经进行了几十年。所以我希望大家能从这种基本框架上开始梳理。因为一般人不可能把浩如烟海的典籍全部翻一遍，老学究也要搞一辈子，甚至都搞不完。比较实际的方法是，从框架、从逻辑上去分析。你要相信，我们的先辈，他们只是古老，并不愚蠢。他们在自己的时代，用他们的智慧解决了当时面对的问题。我们去考虑任何历史问题，包括文化建设，都要考虑到当时的生产力、社会环境和国际环境。如果不考虑现实条件，虚空索敌，那么到处都是烂的，什么封建王朝一样烂，所以全都是烂的，这种论调就出现了。但是当你真的相信过去都是烂的、没有好的、所有皇帝都是狗皇帝，把这个东西全部信以为真，又说“我们要复兴五千年文明，实现伟大复兴”——你先说以前都是烂的，又说要伟大复兴，复的是什么星？这个逻辑是不是颠倒了？要伟大复兴，就要知道以前的人到底在干什么，他们做的东西哪些对、哪些错，哪些好、哪些坏，为什么在那个时候做出那种选择，思路到底是怎样的。中国人的思路确实有独特之处，所以很需要了解。

哎，都几点了，我今天这么能白活吗？不得了了，我今天忘了要赞，一次也没要过，都怪臭猫，我每次都是栽赃它的。来，点赞点赞！我没有看公屏——谢谢“月光与少年”。公屏上字太多，一直在飘，我一看某句话引起注意，话题就被打断了。所以我聊天时更喜欢直接看着镜头，你们是不是觉得我盯着镜头时，专注力、压迫感都直接上来了？我跟你说，这就叫实力。哈哈。什么？又欺负猫不会说话了？对，就是欺负猫不会说话。记得点赞啊！可惜了，他们花了那么多大价钱刷屏结果我没看公屏，那不得把他们委屈死。谢谢。不认同观点可以不听，正常人是这样想的。但他们不是不认同我，他们是害怕我、恨我，是恐惧加愤怒。所以他们会一直盯在这里，希望能够把我送走，这是完全不同的心理诉求。如果说觉得没意思，自然就划走了。但有些人一听就恐惧、愤怒。至于为什么恐惧……谢谢“夜兔”，这是不是老大一个特效了？

以后如果有小孩如何从小进行国学教育？把启蒙读物打小的时候念一念、背一背，不求甚解。先说明“不求甚解”，即使初期没有遇到善于讲解的老师也不要紧。从幼年期开始先背，背什么？包括声律、三字经之类的。除了《弟子规》之外，其他传统的蒙学启蒙读物从小的时候能背，还有那些诗歌、顺口的东西，该背就背。不要听人说孩子又听不懂。我前两天才看到，小孩子的逻辑训练是从小的时候学的吗？小的时候先是形象记忆，效率特别高。你说他懂不了，懂不了怕什么？年纪大了有印象，他就懂了。要趁最早的形象发展期，让他把形象的东西都记住。相信我，我们的文字不光内部有逻辑，而且本身是形象的，所以不会干扰孩子的思维发展。从一开始让他背这些东西，有思维形成的时候再逐渐帮助他理解。什么？三字经不要背？教材说的是《弟子规》是奴才教材，《三字经》是好好的书，不要看《弟子规》。你们再刷屏说不让背《三字经》的那种，被改了好多东西，夸清？那你就挑别的版本，那是《三字经》的问题吗？那是篡改的问题。《弟子规》千万不要背，小孩不差那一分两分，高考又不考《弟子规》，《弟子规》就是奴才训练手册。猫猫干什么呢？打盹儿呢。
