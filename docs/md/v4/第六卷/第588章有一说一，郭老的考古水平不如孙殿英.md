# 第588章 有一说一，郭老的考古水平不如孙殿英

啊，然后那个刚才说郭沫若啊。郭沫若是一个标准的左派文人。你说他一生有没有瑕疵？我觉得他最大的瑕疵就是他想要主导考古。郭沫若最大的毛病啊，就是他老想要主导考古，这是他最烦人的一件事情。非得要整什么考古——考古他也有水平，考，考啥考啊，啊，老烦人了！他就不应该干这个事儿。但是其他的领域啊，在个人的立场上面，在革命的意志上面，在坚定性上面，郭老都是非常优秀的。嗯，这一点啊，这一点就是郭沫若在革命立场上面，意志都非常的坚定。你要说他这个人有什么毛病，他就不该沾考古的边儿。尽管说不能说他安了坏心啊——那个时候的中国确实需要梳理古典历史，但是考古这事他是有技术门槛的，轮不上他。老烦人了！他凭什么去挖坟啊？啊，讨厌死了！他就不应该去干这个事情。他可以干考据，就好好的去考据嘛，他非得要挖坟。他有多大的技术，他去挖这个坟啊，最后把人家都给挖废了。所以这个老烦人了啊！说到这个事的时候，恨不得要破口大骂的这种程度，真是恨死了。他把人家给毁得一个接一个的。

但是呢，他的革命立场上面非常的干净。所以为什么上世纪特别多黑他的人？就因为他的立场太坚定了啊。那些技术瑕疵你可以恨得牙痒痒，毕竟造成了什么负面的影响确实存在啊，但是那个不是根本性的问题。一个战士身上最根本的就是你到底站哪一边，你有没有站人民那一边。而郭老在这个问题上面从来没有含糊过，一点问题都没有。所以为什么那么多人恨他？因为他就没有站歪过啊。从上个世纪开始，备受抨击的人都是立场特别端正的人。公式啊，走起，公式啊！来来来啊，公式这玩意就是公式啊。我们这个直播间里面公式嘎嘎多。从上世纪90年代到本世纪初，受到全面的、各色各样的刨坟——一样的刨祖宗本一样的这种系列批判的，基本都是大好人，立场特别坚定的革命志士。

嗯，郭沫若的《甲申三百年祭》怎么样？很好。我刚才就是在说郭沫若立场上面一点问题都没有。一定要说他有什么瑕疵，我也特别特别讨厌他去搞考古这事儿啊。他只要不搞考古，别的东西都特别好。嗯，昨天那本哥伦布什么的书叫啥？叫做《他们认为中国人比哥伦布先到》。嗯，好，欢迎来到直播间啊。嗯，只是不知道考古队的内部的坏人很多啊，我们当时的技术本身就不行啊。有关于我们当时解放初的时候啊，实际上呢，面临到很多的困难。其中有一段儿的困难其实来自于历史困境。就是有一些的历史，当时的人就知道有问题，但是不知道问题出在哪里。所以呢，我们建国初的时候，很多的左翼的历史工作者做出了极大的努力，进行一些历史事件的梳理。但是当时能够拿得到的史料确实非常的不足。今天的很多的学者可以去海外找史料，但是建国初的时候，中国的历史学家做不到。而问题出在哪？我们的很多的档案全被洋人抢走了。唉，亲，我们那个时代大把的这种档案，然后一些重要的文物都被洋人抢走了。然后没抢走的那些被运到对岸去了。我们这里的很多的学者，巧妇难为无米之炊。但事实上他们都知道，那个时候的很多的历史的记录是有问题的。大家都知道有问题，都想整出来，但是不知道咋办。所以为什么啊？所以为什么郭老当时被人家一推，他就去同意刨坟啊？一推他就同意刨坟？为什么他一推他就同意了呢？我们当时太缺这个了。嗯，好，欢迎来到直播间啊。

然后那个，包括说刚才我就聊过一件事情：说我们在互联网上，其实有一种类型的人是非常奇特的。他前脚反清，然后呢，后脚就反对对西方历史的质疑，然后紧跟着呢，又跟赞美西史辨伪、同时爱清的人亲亲热热的打成一片。所以呢，这一群人背后影射出来的实际的、真实的立场问题，到底体现在哪儿？那就是西方历史的造假的行为和中国历史上面的史料短缺的问题，全部都锚定到了一个特定时代的特定人员身上，那就是清的八旗子弟在这个事情上造成的破坏。正说古今。所以对于西方历史，尤其是西方科技史上面的质疑，与清是有高度关联的。而我们的史料和技术档案的流失，与清也是高度关联的。它就是我们从古典时期走入到近现代中间，一个重要的暗房式的转折点。它是转折点，但同时是一个暗室。因为他是偷偷摸摸的，是因为有无数的人为他涂脂抹粉的，甚至明确的知道他有问题的人都找不着问题出在哪，就已经祸害到了这样的一个地步。所以西方历史的造假和中国历史上面的档案的断档短缺，都跟清有关啊。都绕不开，全部、全部、全部的线索都锚定在了他身上。所以这几个问题是一体化的，是一回事。
