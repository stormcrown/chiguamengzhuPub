# 第1046章 视死如视生_中国人的独特祭祀观念

**整理后的文本：**

视死如事生：中国人的独特祭祀观念！

菊花本是秋季的芬芳之物，并非丧葬祭祀的专属。近世风气变迁，菊花才被引入殡仪场合，也不过二十来年。华夏的鬼神观念崇尚“视死如事生”，祭品应当遵循逝者的喜好与时代的适宜。如果单独将某一种物品标记为丧葬用具，并禁止活人使用，这实际上违背了礼制与伦理。如今观察祭器的演变，从实物到仿制品，从奢华到俭朴，也可见世风的变迁。

对于中国的古人而言，在进行丧葬和祭祀时，需要考虑的是：我们的供奉之物是给往生者使用的。这就叫做“视死如事生”。过去，我们并不使用菊花来进行丧葬服务。菊花是每年秋天——秋高气爽之时——非常重要的观赏花卉。我记得每年秋天都会举办菊花展，里面的花样特别多，五颜六色，大大小小，有卷曲的，也有直瓣的，有重瓣的，还有单瓣的，各色各样的菊花非常漂亮，争奇斗艳。这才是过去比较标准的对菊花的观感——我们是去欣赏它的美。但将它直接应用到丧葬行业里，这个历史其实很短暂，我个人认为大约只有十几到二十年左右。

关于丧葬中使用什么物品，我计划要做一个视频但还没录制。其中有关于祭祀和丧葬领域里使用的物品，其实有一个基本原理：中国人对于往生这件事，有着自己独特的鬼神观。鬼神观是一个民族非常底色的东西，从数千年前的各种故事里一点点流传下来，刻印在每个人的习惯和思想习惯中。实际上，我们的鬼神观中关于“鬼”的问题，认为鬼是人的另一种形态，因此它会有一个去处，那个去处也是一个世界。所以，对于中国的古人而言，我们在进行丧葬和祭祀时，要考虑的是：我们的东西是给往生的那一个人用的。这就叫做“视死如事生”。因此，给先人进行祭祀时，所提供的物品并不是现在规定的那样——祭祀要用菊花，要用白的、黄的——没有这个规矩。古人搞祭祀时，强调的东西是这个时期或者这个祖先喜欢什么，或者说我喜欢什么、认为什么是最好的。这和面对人世间的亲属的态度是一样的：你给自己的尊长安排吃的、用的，首先考虑的第一条是他爱什么——他爱什么你就给他什么；第二条是现在最流行什么、我最爱什么、什么最好最昂贵，你就给他什么。这两个选项都是正常的祭祀思路，而不是听说菊花用于丧葬，就盲目使用。你的祖先不认为菊花有什么特别，你也不是最爱菊花，为什么要拿菊花给他？他明明最喜欢吃红烧肉，你就应该用红烧肉给他；你认为今年最流行某款小汽车，你就应该给一辆车给他。这个才是正常的祭祀逻辑，叫做“事死如事生”。

把某一种东西单独划归为殡葬用品，然后活人都不用了，人人都避之唯恐不及，再把这个东西单独供奉给别人，这是非常失礼的行为。你这是挑拣了活人不用的东西去给祖先。你挑了一个你嫌弃的、认为晦气的、不合时宜的、拿出来现眼的东西去给祖先搞祭祀，这岂不是恶心人吗？严重违背了我们祭祀的伦理和底层原则。所以，要说现代人应该用什么？如果你喜欢玩卡牌，就给祖先整一副卡牌；你喜欢打游戏，就给祖先整一套游戏的东西——这就可以，拉着老祖宗陪你打牌，这个逻辑是对的。再想一想，你听说他年轻时喜欢什么酒，就给他整一瓶。原理就是这个原理。所以，单独划出某一种东西作为殡葬专用、活人不用，这个思路本身是错的。

确实有做纸人、纸楼、纸车的。其实这些东西已经是一个简配版了，用的是纸做的，这说明社会走向了平民化，大家已经不能搞真玩意儿了。最早期的西周时期、先秦时期的老祖宗，能够搞祭祀的全部是贵族，普通人没有机会也没有权利搞祭祀，也没有权利记自己家的账本。那时的贵族非常看重祭祀，因为祭祀就是权力本身。那个时代叫做“国之大事，在祀与戎”，不可以假人——军事和祭祀权属于核心权力。在那个时代，祭祀搞的都是真玩意儿：请吃的时候就是真的猪、牛、羊，整头的就可以往上上，而且是新鲜现宰的；需要珠宝时，就整箱整箱灌进去。所以才会有那么多陪葬品。早期这种模式都是真玩意儿。后来需要奴婢、美人伺候时，也是真人——即人祭。但后来慢慢发现这种方式不行：第一是奢靡浪费，第二是当贵族数量扩大、祭祀之风平民化之后，大家也玩不起了，总量上做不来了。到这一阶段，就开始走一些仪式化的方向，纸扎之类开始慢慢出现。纸扎当然也是纸业成本降低后才开始普及的，但替代品逐渐累积进去。最早的替代品是以陶俑来代替活人，这是第一波替代。之后就是各色各样的替代，包括唐三彩这种烧制的物品——我觉得它已经可以算瓷器了。所以到后来，越来越多的是工艺品、手工品陪葬进去，而极端珍贵的东西只有顶流贵族才会用。再后来，由于造纸成本越来越低，摆件成本也越来越低，丧葬中就开始形成这种替代品的高峰——纸人、纸马、纸车等开始出现。发展到今天，出现了高仿真的纸电视、纸电脑、纸小汽车，甚至纸别墅等等。这些东西其实是符合我们祭祀逻辑的——不管它的污染性如何，那是另一回事，但逻辑是相符合的。比如有些景区看武侯祠时，有人会在里面放一些三国卡牌搞供奉，这也是符合逻辑的：你得关心那位王者他自己要什么，他关心的东西才是你应该给他的。

祭祀只有两种选项：要么把你拿得出来的最好的、最贵的、最用心的心意送给他；要么把他生前最爱的东西送给他。只有这两种选项，没有第三种。其他东西全部都是别人传过来的、胡编乱造的歪门邪道。

（注：唐三彩是有釉的陶器，我知道。所以我说觉得它已经开始接近瓷器了——接近，我说的就是它已经有点像瓷器了。）
