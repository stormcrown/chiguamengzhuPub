# 第1008章 从弟子规看扭曲的清代儿童启蒙教育

从《弟子规》看扭曲的清代儿童启蒙教育！

《弟子规》本是一部童蒙读物，却充满了奴隶规训的思想，强调服从而非启智。这本书原本是给儿童启蒙用的，但所传递的，基本上是当时那些腐朽、如同走狗一般、膝盖底下生根的软弱文人身上的奴役化气质的思想。其中所教的内容，大部分都是一些规矩——教规矩这件事本身就有点奇怪。因为在过去，包括《三字经》在内的各种蒙学读物，给孩子教的更多是道理，而非规矩。比如：为什么会这样？历史上发生了什么样的故事？你为什么要学习？学习的价值是什么？什么是真善美？什么是假恶丑？什么是天地人？这些蒙学教材讲的是知识与道理。而到了《弟子规》，从名字到内容都在讲规矩：你得怎么做，出门时怎么说话，进门时怎么说话，抬脚时怎么走路。实际上，这是一种非常僵化的教育方式，拿它来教小孩子是不合适的。它更像是用来训练基层员工的，在清代也非常适用于训练奴仆。它是一个讲规矩的东西，而不是在启蒙教育，不是在讲知识点，不是在讲我们的文化要素。你把这个记住了，将来读书也没什么用。它讲的就是这些规矩。

那么，能不能讲讲《弟子规》算不算儒家文化？《弟子规》开篇就是“弟子规，圣人训”，可以说它与儒家之间是有关系的。你要说一点关系都没有，那肯定不合适。但这个关系比较微妙，因为《弟子规》出现的时候，儒家已经走到了比较僵化、甚至犬儒的阶段。它借用了儒家的元素，里面每句话未必全错，也确实借用了一些儒家的道理和思路。但所传递的，基本还是那些腐朽文人带有奴役化气质的思想。

《弟子规》的问题在于它产生的时代。整个儒家思想到清代已经僵化到了相当地步。早期的儒家思想，比如被视作开山祖师的孔夫子，一辈子并没有专门为他的学术写一本书。我们今天看到的《论语》是他弟子的笔记，因此他过世后还出现了很多版本——每个弟子记的笔记都不一样，大家还要对照、对质，彼此之间有争议，这才区分出各个流派。流传到后世，孔子“述而不作”的习惯使他的学说变得非常开放，每个人都能在上面找到自己想要说的东西。尤其是汉代“独尊儒术”之后，儒学实际上被当作一个骨架，其他各种学说都可以嫁接上去。因此，儒学在两千年的历史中一直不断变化：心学怎么样，理学怎么样，都是不同阶段、不同学者借由儒学的方式和思路开创自己的哲学道路。大家开创这些道路的目的，都是为了解决社会问题；至于能不能解决，那就不一定了。要知道，搞社会研究的大部分情况都是马后炮，提前预言某种方式肯定能行，成功率非常低。比如马克思开创马克思主义道路，他自己的个人实践也是颇为失败的。真正让马克思主义思想生根发芽的，是在中国这个地方——只有中国做得最好，能够延续下来。这是为什么呢？是因为马克思主义本身好吗？如果它好，之前用过它的地方都应该好，结果只有中国能好。那是因为它与中国传统观念和基本伦理观中的很多东西是融洽的、有契合的，所以才能用得好。

由此可见，一种思想是随着时间不断变化的。当它变化到清代时，实际上已经僵化到快要崩溃的地步。为什么会这样？因为明代时，儒学的学者们已经看到了世界巨大的变革——由生产力改变带来的变革。明末的思想家有反君权的、有要求自由贸易的，各种类型都有，自由主义的萌芽也在那时出现。但到了清代，这些思想一概被打压，因为清代是绝对皇权集中的时代，比之前任何一个时代都要集权。然而这个时代压根儿不应该搞皇权集中，这完全是开历史倒车。因此，清代的儒学学者们无法再向前走，向前走的尝试也没有地方试验，连块试验田都没有。所以他们研究的东西变得非常僵化，搞什么考据：这个人说过这句话没有？这个字有几种写法？这个笔画应该怎么来？天天就研究这些东西。因此，这个时候的学者搞少儿启蒙，概念就会非常有趣。

到了清代，有些人搞儿童启蒙，《弟子规》本来也是给儿童启蒙用的，但教的大部分是规矩。教规矩这件事很奇怪——过去包括《三字经》在内的蒙学，给孩子教的大部分是道理，而不是规矩。比如：为什么会这样？历史上发生了什么样的故事？你为什么要学习？学习的价值是什么？什么是真善美？什么是假恶丑？什么是天地人？这些是知识与道理。而《弟子规》从名字到内容都在讲规矩：你得怎么做，出门时怎么说话，进门时怎么说话，抬脚时怎么走路。这是一种非常僵化的教育方式，拿它来教小孩子不合适。它是用来训练基层员工的，在清代也非常适用于训练奴仆。它是一个讲规矩的东西，不是启蒙教育，不讲知识点，不讲我们的文化要素。你把这个知道了之后，将来读书都没有帮助。它讲的就是这些规矩。所以它不应该被当作启蒙教材，它属于“入职培训”——你进到这个岗位，就应该做什么：脚必须先迈左脚，喝水只能喝半口，等等。《弟子规》这本书能不能存在？反正它也存在了，我能怎么样呢？但它不应该给孩子看。孩子应该拥有知识和道理，应该学习：天上的日月星辰真美，它们有什么故事？这个星星叫什么名字？那个星星叫什么名字？我们的山川河流是什么样子的？孩子要学这些，而不是学应该用多大声音跟爹妈说话。
