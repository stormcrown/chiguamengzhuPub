# 第1076章 最极端的满清批判_移动的粪坑理论

最极端的满清批判：移动的粪坑理论 #历史暴论 ！！！

---

粪坑蝇蛆岂堪留，  
浊史腥风四百秋。  
蠹简空谈以史鉴，  
腐儒妄作沐猴讴。  
应知秽土生妖孽，  
更见遗毒泛九州。  
莫道掏粪非雅事，  
除残须效介推忧。

---

啥玩意儿啊？你一直批评满清，为什么？嗯，为了什么？你到底想批评什么啊？听不懂的话，出门左转啊，好走不送啊。为什么能为什么呀？

我昨天打了一个很好的比方，估计很多人还没有听到。我现在重复一下，免得你们没听见。我觉得这个创想还是很不错的，这个创想非常好啊。我重新给大家重温一下昨天刚刚讲过的一个比方，我今天要重来一遍啊。

为什么我们要坚持不懈地批清？哼哼，为啥满清是一个粪坑？批清的这种行为就是掏粪。掏粪会让人恶心难受，身上沾一身味儿，很辛苦，甚至有时候很心烦。但是为什么非得要去掏这个粪不可呢？因为这个粪它不光是粪。粪只是拉完了之后在那里 old 的肥，但是啊，它里面咕涌咕涌的全是蛆。掏粪的目的是把蛆掏出来。为什么要掏这些蛆？难道不是可爱的小生命吗？不行。因为蛆会变成苍蝇，苍蝇又会生蛆，会严重地破坏我们的生存环境，会使得我们沦落为之前那种恶臭的时代。所以必须持续不断地掏粪，一直掏到里头没有蛆。然后这个粪坑就可以放在那个地方，静静的发臭了。这个就是为什么要批清啊。

那么粪坑只有这一个吗？不是，我们历史上大大小小的粪坑有很多。但是其他的你为什么不批呢？因为其他的都在那里静静的发臭，而这个坑里面到处都是蛆，爬得满屏幕都是。我现在任何一个屏幕上面，都能够看到一群白花花的蛆在那里咕涌。有啥原因呢？要是你们这些蛆不在那里挽救那个罪恶的时代的话，我就不用掏粪了。你当掏粪没味儿吗？我很嫌恶心，我很嫌臭。我这个人有点轻微洁癖，虽然不勤劳，但是被迫熏得我受不了了。看着那个白花花的玩意儿在那里咕涌，我恶心，我就得掏啊。

喝酒听节目？别这样，喝不下了啊。对，对不起啊，耽误你吃饭了。哎，我觉得，我这个比方是不是打得挺好？我昨天想起来这事儿之后，我就觉得我这个想法，我这个类比是不是做得非常精妙啊？实在不行的话，以后每一期每一天我都把它重复一遍。我一定要让大家都知道，这个段子是我想出来的，这个类比法是我琢磨出来的。我觉得非常好啊。

正常人类世界里面到处都有粪坑啊。一般的粪坑起的是什么作用？那叫做以史为鉴。正常粪坑的作用就是以史为鉴，路过一个人就指着说：“那地是粪坑，你别碰啊，碰了之后粘脚，千万别掉里头去。”这是正常的粪坑。而清的这个粪坑不一样，它会爬动，所以就得掏啊。咱们还得持续不断地，一直掏，一直掏到没有蛆为止。
