# 第1005章 五胡乱华的定义是客观正义的_入华叙事纯属胡说八道

五胡乱华的定义是客观正义的！“入华”叙事等论调纯属胡说八道！ #五胡乱华

他们作为雇佣兵、雇农而存在，长期以部落形式在边陲地带生活，是被招揽进来的人群，并非没有往来的人群。因此，他们不是“入华”，也不是这个时候才来的。他们早已在此地，但此时已反客为主，制造了天下动荡、生民涂炭。

这个定义是很准确的。古人在面对历史问题这方面比今天的人要客观，因为他们身上没有那些政治正确的枷锁，不会因为十年前才出现的伙伴去说一千年前的祖先是假的。古人比我们今天的人诚恳坦白，所以五胡乱华这个定义没有任何问题。因为他们当时不是“入华”也不是“归华”，他们早就已经内附生活了。只是在这个时候，由于整个西晋的动荡，他们自己把自己的核心群体损耗得差不多了，人数、驻军能力和管理能力都一团稀糟，脆弱得不堪一击。于是这些之前就已经内附的部落开始造反，企图反客为主，并采用了非常残暴、野蛮的手段。所以这个阶段，不管是谁下的定义，五胡乱华是毫无问题的。那种“五胡入华”是中性的、含糊其辞的说法，而且很不客观——因为这些人不是这个时候来的，他们早在东汉时就来了。他们作为雇佣兵、雇农而存在，长期以部落形式在边陲地带生活，是被招揽进来的人群，并非没有往来的人群。所以他们不是“入华”，也不是这个时候才来的。他们早就在此，但此时以客逆主，制造了天下动荡、生民涂炭。
