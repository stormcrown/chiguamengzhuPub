# 签条约速度创吉尼斯纪录的朝代？解密清廷投降动力学！！！

## 不平等条约的创纪录速度

防汉锢鼎，银倾海疆。宋瓷映月，沉舟照殇。赏梅角畔，输文宗主；签非陆约，丧鼎彝章。八旗星布，内防胜虏；满城昼警，外战先降。准部鏖兵，疫疠犹抗；天朝宣诏，翌日输偿。万顷膏血，夷算分账；千年积弱，谁剖肝肠？横向观史，汗青有鉴：弱宋寰尊，强清寰戕。

有人会说不要过度贬低近代清，北京都不敢这样说。不行你去告我。你要是实在忍不下去了，你就去告我。别说什么不要贬低近代清——清就不是近代。我们现在定义的近代史从1840年开始算的，所以你清不算近代。第二，你要是不贬低清，你就得贬低整个中国。你选一个：要么是中国整体自古就烂，要么就是清是烂的。你挑一个。你是让我们烂5000年、1万年好，还是烂满清260年好？你自个儿选一个。你觉得我国会认可哪一种说法？简单的选项就这么简单的一个选项。

要自信，必须讲清楚我们的来路。世间所有人都这样：一个人想要自信，你就得说我打哪来的，我以前咋样。这玩意儿叫做基础。我的基础好，我才是自信的。所以我们国家现在这些年搞文化自信，看搞的什么文化自信：我们在不停的考古，不停的发掘我们的古典遗迹、上古文明，说我们的文明史已经推进到了1万年了，而且还发现里头有自古以来——这个就是中国人现在骄傲和自信的来源。

你说既然我们以前这么牛，我们近代为什么垃圾成那个样子？为什么连刚果都能骑在我们脖颈子上拉屎？解释一下。我们那么厉害，我们的文明辉煌了上万年，请问为什么在近代史上面，连刚果都能骑我们脖梗子上面拉屎，签不平等条约？解释一下吧。要么呢，1万年都是屎坑；要么你清200多年是屎坑。你挑一个。你觉得我国会挑哪一个？你要是觉得我这种说法就叛国了，那我肯定是叛大清的，毫无疑问。结果呢，结果那个啥——你清是敌国，我就反你清。那啥很轻，是跟刚果分账。我管这事儿，反正你敢签我就敢喷，是不是？你敢签我就敢喷呐。他连不平等条约都签到非洲大陆上去了，还能不能喷他两句？至于他到底是为了卷款潜逃，还是真的就怂了，那关我什么事儿？实际上他跟清签的那些条约，我都怀疑他在分账。但是他卷款潜逃这件事情，卷的款不就是我们家祖祖辈辈攒了几千年的家当吗？是你落中国土地上也不一定进我的荷包，但是肉烂了在锅里头，知道吧。就算是土财主截了胡，他置换成了万顷良田，那也是在我的国土上面。哪一天我家翻了身的时候，那就是我的一份结果。为什么我们建国的时候一穷二白，甚至开始出现了积贫积弱？我们攒了上万年的家当去哪儿了？他是不是跟刚果去分账？是不是卷款潜逃？我管他无论是哪种方式，该喷的就喷。

为啥谁觉得满清好的，要说出理由和历史上的其他朝代比较着说？你这个选择方向错了。为啥我跟你说你这个选择方向错了？因为要说满清好和坏，他没有资格和其他的朝代做比较。没资格。任何的时期，这个国家强与不强，你要进行横向比较。不然呢，拿清朝的生产力和秦作比较，你会发现满清的生产力竟然强过于大秦。你看到了还不是我清最强吗？他的牛就敢这样吹。他们从来都是穿越时光式的对比。他为什么敢在那里嚣张的说“我们家的那个什么比大明强”？因为他更改记录就行了。咱们不带这么跨时空、跨时代的去进行追索的。任何国家，你强与不强，你看你在当时世界上的地位，你看你在当时世界上的人口占比，你看你在当时世界上面别人眼中的形象。这就是好赖，优与败就来自于这个。

我们其他的时代，包括说大家心目当中最垃圾的弱宋——咱们都不说北宋了，咱们就说南宋。很多人心里头南宋垃圾的不得了了，偏安一隅的地方。它的科学：周边的人都要跟我们学文字，周边人要跟我们学服装，震慑寰宇，那叫一个漂亮。服装鞋帽，各种各样的装饰品、瓷器都是当世顶流。文化影响力从来没有下降过。日本的赏梅文化，这一套赏花系统就是宋的时候学过去的。我们的文化宗主国地位完全没有流失。包括说南宋为什么喜欢买和平？因为太富有了，所以财力上面还非常的优胜。为什么他老喜欢跟其他的地方勾连，喜欢交岁贡？因为交税更便宜。前脚交完岁贡，后脚通过走私贸易，他能赚的盆满钵满的。他不想打仗，因为打仗要付硬成本，而且耽误他走私贸易。所以在经济上面、技术上面、文化上面、艺术上面，宋都是当时的顶流。甚至于我们现在南海底下还沉着很多的沉船，就是宋代的海贸沉船，上面还有大量的铁锅走私贸易。所以南宋这种垃圾的时代都有环球航海贸易——有没有转整个地球不知道，但是一定是走了很远的，有海上贸易通道，海上丝绸之路。这还是在弱宋这个时代，而且挑的是更弱的南宋。就那个时候，我们也没有说所有的国家都能来揍过来。无论是辽、金还是后来的元，我们跟哪一波打的时候，就南宋那个废物的样子，还打的是有来有回的。蒙古的大汗也是死在了宋的土地上。蒙古的军队也在宋的土地上面，一回又一回的被拦下来了。甚至有很多蒙古底层的缺口直接就归顺了。这是什么？这就是国际地位。我就挑最弱鸡的一个国际地位，上面是不是强于大清？天天吹自己天朝上国的大清是啥地位？任何一个列强来了之后，我表示一下我宣战，然后打两枪，我投降啊。不都是这样的吗？我跟你们所有宣战，所有洋人都对不起我，我向列国宣战，开两枪，我投降。不都是这样干的吗？

为什么要这样做？满清根本不敢打外战。从清初到清末，他都不敢打外战。包括在那里天天说“我们平准噶尔，没有我清谁能够收复西域啊”——牛皮是这样吹的。打准噶尔多少年？打了好几十年，屡战屡败。真的是因为准噶尔特别强吗？没有。当时准噶尔部已经弱到了内部分化、两面受敌的程度了，直接能够拖出来来战的兵马也就是个35万。就这个程度，腹背受敌、内部分化、瘟疫横生的准噶尔，能够跟清之间打拉锯战，打了几十年。为啥？为啥不是说八旗兵特别威武雄壮吗？为什么？因为他不敢调动。真正的八旗兵是收着的，永远都是拢着的，一次只能调动一点点的部队。他不敢大规模的调动军队。为什么不敢？因为他的八旗兵要镇守内地防造反。他敢大规模的调兵遣将，内地就瞬间反清复明。

其他的时代老是嘲笑说“你看你修长城，你就那么大点儿的土地”。第一，长城就不是边界线，长城是哨卡。第二，其他的游牧民族，他的人少，而且不定居，它是没有边界线的，所以地图学本身就有问题。但是到了清的时候，清还说不修长城——他不在草原上修长城了，他在内地修长城。为什么哨卡长城起的就是一个哨卡、调兵以及发信号的功能，他就是干这个的。清为什么要在内地修哨卡？防造反，防联动。就这样。所以为什么他外战的时候那么烂：我宣战第二天我投降，他降的比谁都快。所有人有眼睛的都能看出来，你再拖个十天，对方自己都能不行。急着必须现在立刻马上投降，一天都不能等了。为什么？因为他只要调动军队，只要发放军饷，就有可能让汉人的这些指挥官、让汉人的这些绿营兵马获得权力，获得饷银。而只要他拥有权力，拥有财力，他就会造反。满清从来都是这种态度，从他建国一开始到最后一直都是这种态度。

所以你说那时候列强多强？厉害了，太狠了，我们都打不赢。强成啥样？武器之间真的有代差吗？有这么大一点儿？这点代差，就凭清的那些白花花的银子就足以弥补了。你以为西方的那些商人多有良知多爱国、坚决不能把武器贩卖给清？不存在这件事情。他们那里的商人缺德的底儿掉，你只要付银子啥都能买得着。为什么最后没有啊？因为他根本就不敢打仗。定是根本不敢打外战的。为什么？因为他把他自己的主要兵马都放在内地，而且是分散式的放在了内地。怎么个分散法？第一，就是各种大地区、大的关键节点上面，大城里面会有专门的满城。在各种各样的交通要道和主要市镇上面，有小股的八旗兵。满八旗兵协同汉军旗兵，设置一小股一小股的分支兵，马多的那些几百人，少的时候连几十人的都有。这样逐层的这样分下来的，到处基本上说你每一个像样的镇子上面都有满军配合汉军旗的这样一个小股兵马，驻点儿定点儿的就在那，随时随地的应招干嘛？防造反。稍有风吹草动，他立刻就调动起来了。但是这种格局你就想一想：任何一个国家，不管他怎么样的掘地三尺，他能够供养的兵马永远都是有限的。这个有限的兵马，他只要是把这些兵马放在了内地，那么他在边陲地带就一定严重的欠缺主要可以调动的兵马。他没有人给他打仗。你以为他可以随地征兵？他敢吗？他不敢随地征兵的。满清去随地征兵，征下来的就是推他的人。

为什么这样说？第一次鸦片战争的时候号称是要抗英，但是满八旗的领军带了兵马出去抗英，打仗的时候路过汉人为主的城池，先回头把这个城给屠了。为什么？他担心自己在抗英的时候，后面汉人抄他的后路。这还是第一次鸦片战争的时候，这已经是清代的晚期了，都是这种局面。你觉得他敢征兵？你觉得他可以就地征兵？你觉得他敢发动什么战争？怎么可能呢？他不敢打外战。所以经常性的就是我宣战，第二天我投降，签约，句号。为什么满清签的不平等条约可以签到非洲去？他敢动一下谁呀？非洲那里哪怕是一个欧洲殖民者的傀儡政权，他也得听人家的。
