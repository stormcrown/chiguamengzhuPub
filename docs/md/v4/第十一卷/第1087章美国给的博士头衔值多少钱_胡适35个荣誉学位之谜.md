# 第1087章 美国给的博士头衔值多少钱_胡适35个荣誉学位之谜

美国给的博士头衔值多少钱？#秘闻的35个荣誉学位之谜#胡适文柄之殇——论学术阍者与国族精神之蠹。昔韩非子《五蠹》有言：“儒以文乱法，侠以武犯禁。”今观胡适之故事，乃知文蠹之为祸，尤甚于刀兵。其执学术之牛耳，非以才学服人，实挟外力以制士林，恰如《墨子·尚贤》所讥：“不能治百人者，使处乎千人之官。”此诚当代文柄之耻也。其人以《兰花草》一诗传世，本为洒水车俚曲，竟成学术霸权之隐喻。恰似《吕氏春秋》所载……

阿弟说胡适难听的话，你不高兴了是吧？哈呵呵。啊，一件事情啊，关于胡适。第一件事情是有关于民国时期，红学直接被考据派一统江山，索引派呢，烟熄火熄不能抬头。这事谁干的呢？啊，胡适干的。那胡适这个人，他怎么就能够做到在文坛上面一统江湖呢？是他的文采特别好吗？啊，他这一辈子最优秀的一个作品，大家见过洒水车没有？见过的吧？啊，洒水车放的那个音乐“得得得得”，这个音乐大家都听过吧？听见这个音乐的时候，撒丫子就跑路啊，因为很有可能水要崩你身上了。这个歌儿叫做《兰花草》，兰花草的原作者呢，就是胡适之。这个是他一生当中最优秀的一篇文学作品，其他的东西就不用谈了啊。所以当一个人，他的文学作品、文艺作品就剩下这一个短诗可以拿出来一说，尚可，仅仅是尚可，而且我们把这个诗拿来干什么？拿来洒水。说明这个诗也是很水的，就是到这个程度。他居然能够引领民国的学术界，引领民国的文坛。你觉得“文无第一，武无第二”这话啥意思？意思是文人在文人面前从来不服软。但是一个文采、才学都没有的人，是怎么引领中国的学术圈，引领中国的文坛，带领那一群坚决不肯依从的文人的呢？因为他背后有腰子啊，腰杆子硬啊。那他的腰杆子来自于哪里呢？啊，是吧。

谈到他的腰杆子从何而来，那就不得不谈一下他为什么会被尊称为胡博士了。要知道他当年出去留洋啊，他当年出去留洋这件事情本身啊，是那个时代算是潮流吧，反正都出去留洋嘛。好，他出去学习，可以去修博士学位。OK，反正是一个文科生嘛，你修到什么程度是什么程度，写一篇论文出来，导师承认你也可以说你是博士。然而但是——可是他这辈子连一篇论文都没有做完。他在美国那边留学，所谓的博士论文开了个头就扔在那了，撒丫子回国就成为了文坛领袖。最后是他的导师发现自己的学生怎么不见了，然后发现他已经在中国国内引领学术圈了，当了扛把子了，然后就给他送了一个博士学位。这是他所有博士头衔当中最真金白银的一个——就是课也没怎么上，论文也没怎么写，写了一个开篇感言就句号了的一个东西。这个就是他一生当中最重要的学术成就，但是就因为这个啊，他还得了一个博士学位。剩下的其他的博士学位就全部都是荣誉博士、名誉博士，送的博士。所以他一生博士的头衔，但是没有念过博士的书，也没有写过博士的论文啊。所以就是这样的一个角色，他在中国国内能够引领学术圈儿，这说明他必须拥有某一种力量。我说了，文无第一，武无第二，文人墨客是不愿意服软的。你没有二两油，我不肯给你提鞋，这是天经地义的事情。但是为什么他能掌控呢？因为它有通道啊，他有通道。谁讨好他，谁就可以成为学术界的上流人士。也就是说，他是一个发当时学术门槛，发入场券的人。哎，谢谢哈老师啊，谢谢哈土鳖科维奇哈老师。

那为什么他能够开始去发放这个学术入场券呢？唉，说到入场券这个事情，现在我们国内也有一些圈层是要发入场券的。大家记得我之前聊过刀郎吧？刀郎你可以喜欢他的歌，也可以不喜欢他的歌，这都无所谓啊。但事实可以证明一点，就是他的歌儿呢，是受到很多观众的喜爱的，他的唱片是可以卖掉的，说明他在演唱上面，在音乐创作上面是有才华的。既然是商业型的音乐，那观众买账，别人哪有资格说话？然而中国国内的这个音乐圈就不承认刀郎啊，认为说他是没有艺术造诣的一个人，认为他是一个土包子，认为呢，啊，之所以他的唱片能卖出去，那是观众错了。这是什么？这就是音乐圈：说你没过来拜码头，你没有认我当老大姐，你不承认我在守国门啊，那不好意思，你不是音乐圈的人。所以呢，你就没有这个入场券啊。只要有圈层就会有看门儿的，看门的就是负责发门票的那一个。你要进门，你就要付出某一种代价，例如，嗯，你承认自己是什么什么什么人啊，这样的代价叫投名状。然后呢，你就可以进入圈层了。那么胡适之，他在民国时候担任的就是这个看门儿的职位。他是给当时中国、民国的学术界发门票、入场券的那个人啊。一个小马仔，所以他有腰杆子，所以他硬气，所以他呢，学术水平不行，文化水平不行，文艺水平不行，但是可以引领文坛，并且做出决策。决策到了什么程度呢？就是不管你说什么，反正我说红楼考据派才有理：他是曹家公子，他是江宁织造家里的小公子，那他就是。哪怕他家的户口本上没这个人，我贴也要给你贴上，给你们家就追认一个祖宗。你认也是认，不认也是认，你家以前就是有这么一个人存在了。没有权利是做不到这件事情的。

那他的权利从何而来呢？啊，谁给他发放的那么多的荣誉博士的头衔，让他成为中国国内学术界当中博士头衔最多的那一个人呢？啊，谁给他发的福利？谁给他发的身家？谁让他拥有这个看门的身份？谁就是他的腰杆子。这个推论总归是合理的嘛。那么他当年是到哪里去留学的呢？啊，美国。好，那么美国当年对我们有没有善意呢？不要做梦了嘛，美国从诞生伊始，对我们就从来没有善意这种东西存在过啊。那这种情况下，美国资助某一个国内的学者来这里，引领我们的学术界，发放学术入场门票，合不合理？有没有点道理？那么，谁想加盟美国当他的看门人都行的吗？你想要去给人家看门，你总要付出点代价，付出点成本吧，不然你凭什么当别人的看门呢？是不是？所以他付出了什么代价呢？啊，据传说，据传说他将当时国内最完整的一套《明实录》带到了美国去。所以今天我们国内研究《明实录》的人，是通过多个版本的彼此参照啊，互相印证才可以进行研究。而清代保留下来的最完整的一套，他搬走了啊，送到美国那边去了。所以你说哈，他交了这样的一套重要的文献出去，拿来当自己的投名状，从而换得了美国给他的护身符，以及呢，啊，虽然没有著作等身，但是可以学位等身的一个身份。然后拿着这个身份再出来号令文坛，在文坛上面排除异己：谁听他的谁就是文坛上的头牌，谁不听他的，谁的学术成果就要被湮灭。从而就可以引导中国的思想界，引导中国人的思想方向。

你就说这个事儿对于中国的发展重不重要啊？它呢，对于国际的形势的影响重不重要？即使晚清和民国时期的中国，在世界上的排位赛上面的水平并不高，但是排位再低，我们也是一个庞然巨物，不可忽视啊。这样的一个庞然巨物，如果在思想界被人引领了，被人掌控了，你想什么？怎么思想？你的学者要由外国来发放他的准入许可，那这个国家的发展大局、发展策略、发展方向是不是就被控制住了？所以我们看后来啊，截胡革命的光头——蒋介石，他们一家子，是不是他的夫人就是亲美的这群人啊？具体的就不说了啊。常凯申最后的这个夫人，是不是亲美的这一派的？他与这个夫人成婚之后，是不是改宗从原来的佛教徒变成了基督徒？所以呢，是吧。嗯，啥？她最后的那个夫人啊，对对对啊，我在说她最后的那个夫人哎。宋氏啊，最后的那个夫人宋氏。他前面呢，当然还有夫人啊，中间还有夫人啊，最后的那个夫人宋氏啊。
