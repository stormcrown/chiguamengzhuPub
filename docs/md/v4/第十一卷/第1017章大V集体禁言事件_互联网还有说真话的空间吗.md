# 第1017章 大V集体禁言事件_互联网还有说真话的空间吗

大V集体禁言事件：互联网还有说真话的空间吗？

舆论汹汹势未休，顶流封口使人愁。  
岂因片语伤麟趾，实有重金筑蜃楼。  
润物无声真手段，操戈显迹假权谋。  
可怜九十年间剧，犹唱天家恩泽稠。

因为国际顶流大IP的事情，被禁言是一个好事——能够让互联网上的人看见，有些人客观地批评一下也是不行的。这说明什么？嗯，这说明他有老板啊，对吧。如果说之前还认为这是一个偶然性的失误，或者是因为个人在这个方面没有研究过、没有想过，所以造成的、情有可原的话，那这一批大V的禁言实际上带来的后果就是：让脑子里不缺弦的人基本上都意识到了——这不是哪一个超级大V、超级IP能够解决的问题。连指点他一下都不行，连说他这个事情有一点问题的这种质疑都不可以说——说不得、摸不得、碰不得。这就意味着他不是他一个人。自己什么样的顶流都配不上这样的待遇的，谁可以呢？老板可以啊。

什么事儿呢？嗯，啊，就是某位国际顶流，嗯，是吧，这一次复出——时隔3年的复出——然后呢，出了这几期视频之后，因为里面C位的宣传位，导致了一些质疑的声音。然后现在呢，在某些平台上面，那些质疑他的大V的大号被禁言了。哎，好，谢谢六郎啊。

嗯，什么？他们的资本强大，为什么还会乐观？强大吗？不强大啊。那个小伙伴，你要知道一件事情：真正的咬人的狗是不叫的，真正强大的力量是润物无声的。能够让你在互联网平台上面都能够发现“这背后有势力”“这个背后有老板”“这个老板会下黑手”“这个老板不着调”——能够让你都看得见这个东西，闻得见这个味儿，并且明晃晃地瞧见中间的冲突，这意味着它力量不足。

真正力量强大的时候是什么模样呢？大家记得上世纪90年代开始在互联网上……嗯，那个时候没有互联网啊——在那个时候的电视平台、电影平台上，大家所看到的各种影视的作品，清王朝是什么样子？盛世迷人的、英俊的、浪漫的，皇帝和民间女子之间缠绵悱恻的爱情，对吧？华丽的衣衫，体面的着装，爱民如子，深入民间主持公道，是这样的吧？从上世纪90年代开始，你看到的清王朝的一切影像的资料，都是这个样子的——这才叫润物细无声。你看那些东西的时候，你怎么能够意识得到那个时候是怎样的荒唐，人们经历了怎样的苦难啊？你根本就想不起来这种事情，甚至你会不由自主地去带入到那个君王、那个臣子或者那个美丽的民女的身上去，想要享受天恩呐。那个、那个才是力量。今天的这个算什么力量啊？
