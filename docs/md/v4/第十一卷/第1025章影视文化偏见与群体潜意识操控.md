# 第1025章 影视文化偏见与群体潜意识操控

影视文化偏见与群体潜意识操控。  
嗯，啥，低配版吴彦祖那种行？你啥时候爆个照啊？  
紫色在古代难染嘛——紫色在古代全球都难染啊，紫色在全球都难染。  
嗯，什么武侠剧喜欢扮？嗯，外族的服饰特别的鲜艳明丽，故意的啊！这些文创领域里面啊，就是影视创作、文艺创作的这个群体里面，特定的人群很多。他们呢，就特别喜欢把非汉群体描述得特别的璀璨；然后如果条件落后呢，就说他们特别的直爽；如果说经济的生态不好呢，就叫做特别自由。特别喜欢给他们包装，然后呢，同时就来贬低中原人。这个是他们这个圈层当中很普遍的一种现象。就以金大侠为首吧，他算不算领头也很难说，但是呢，就这个群体，他有这种很刻板的行为：就是遇到中原人士、遇到汉家人士的时候，如果他很文雅，那么他就是伪君子；如果他学识很高，那么他就是表面文章；嗯，如果说他讲究礼节，那么就是沽名钓誉。反正呢，就是没好话。你看上去两边他都给你正反两项说了，但是重点在哪？正常人阅读的水平还读不出来吗？所以实际上他就是想强调说，非汉群体自由啊、博爱、直爽、没有小心思是吧？他所有的那些什么，呃，落后的东西，都是因为我们这边过于狡诈、过于卑劣，所以他们才跑到那个地盘去吃草的啊——就是在制造这种氛围。所以呢，在他们去拍影视剧的时候也会这样：会刻意的把那些相对其实比较贫穷、落后的地区呢，拍得特别豪奢，然后把这些真正丰饶的地区呢，拍得特别贫困。这是一个刻意的行为啊——制造群体潜意识。
