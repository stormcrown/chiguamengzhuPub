# 第1045章 西方人狡猾_一边自夸摩登一边把你钉死在原始人标签

西方人：狡猾！一边自夸摩登，一边把你钉死在“原始人”标签！  
————————  
西人饰智，冠冕自雄。标榜摩登，诋异为蒙。漠南非锢，楚囚南冠；印第非固，标本樊笼。假护文之名，行桎梏之实；托仁术之表，藏劫掠之锋。白右鸩酒，苦人等级；白左鸩酒，混世昏庸。嗟尔寰宇，同此凉热；岂容画地，永锢愚衷？  

他们喜欢固化所有看上去落后的东西啊。  
西方人就是没希望的。  
这些人是很狡猾的：他一边在给全世界传递一个信息，就是他们才是摩登的，是现代的、是发达的——他一直刻画这样的印象；然后同时呢，他还要刻画另外的一种刻板的印象：就是和他不一样的这些群体，一定要把他们钉死在落后的、愚昧的、古典时期的，甚至是原始时期的这种条件之下，永远都会把这些形象固定在那儿。  

包括说漠南非洲。在漠南非洲确实存在着一些较为原始的部落，但是难不成这些部落民并不想过好生活吗？当整个世界都打开了之后，他们也是有机会接触到外界的世界的，有了机会发现了可以进步的空间的时候，他们也是愿意学习的。无论说你觉得他们的能力如何，基础如何，但是他们都是人，人就有上进之心。他也想要改进，他也想要和别人一样穿着那些漂亮的衣裳，他也想要学习书写，也想要学习工作，也想要吃更好吃的东西——这都是天然的诉求。  

但是西方世界派了一大批搞文化的，搞什么所谓文化保护的啊、族群保护的等等的，各色各样的人呢：啊，有拍纪录片的，有学者，然后有志愿者，聚在一起搞什么事？保护部落文化。请问什么叫做保护部落文化？对世界各地的人们没有用同样的步骤向前走，大家的发展的进程参差不齐，甚至这个参差的距离可能相距千年万年。但是那又怎样呢？不能走了吗？不能改变了吗？不能上进，不能学习了吗？  

西方人整的就是这个。他要把非白人的所有人都定死在过去，要让他们表现成愚昧和落后的样子。包括在阿美自己国家的内部也是这个样子的。大家自己闭上眼睛去想一下：你觉得阿美自己国家内部的印第安人是什么样子的？他们自己国家内部的印第安人是什么样子的？是不是你只要闭上眼睛，出现的形象就是头上的鸟毛，身上的珠珠串串，脸上画的彩绘，唱着那些悠远的歌——是这种样子吧？难不成这些人他是不想上进的吗？没有啊！不存在呀！但是干嘛就是要把他圈起来，像动物一样的向全世界展示：你看他们是这么落后的，我们是这么发达的，所以我们是文明人，而他是野蛮人。他就用这种方式，把他的这种劫掠的行为，变成了一个正义的行为。  

诶，好，谢谢红色的旗手啊。嗯嗯，好，欢迎大家来到直播，直播间。嗯。  

希望人是希望他们自己白人至上的，这个愿望一直都存在啊。他们从一开始的时候，其实就构建了一套有关于这个种族差的操作方式，认为某些人呢，更高级啊——这些东西都有。其实你要是说种族之间有没有差异，那一定是有差异的。现代的这些什么所谓的主义的那些啊，人类学硬是要说所有的人一模一样，没有任何差异，这纯纯的是在胡说八道，完全背离科学知识啊。但是西方人所搞的那些就是种族差和种族歧视，和我们客观的认知种族的差异，然后呢，用不同的方式去建设不同的群体，是完全不同的思路。  

因为如果你按照今天说，哎，白种人老是对其他的人产生歧视，所以我们必须要说所有人是一模一样的，这就直接就是一个方向上的急转弯，其实是从一个错误转向了另一个错误：就是人种和人种之间确实不一样，但是大家呢，能力可能不同，特征可能不同，特性特长可能都不一样，可是我们的人格是一样的，所有的人都可以在自己能力的范围之内，取得自己的进步。这个是一个正常的思路。  

但是呢，哎，西方的所谓的白右这一派，就是白人至上，一切以他为中心，其他人都是低下的，并且要努力的把其他人摁在地下的这个位置上，不许他们出来。而白左的那一派呢，就是所有人都是一模一样的，无论怎么混都很好，越混越好。但是这个是完全背离了科学精神的，一点客观的属性都没有，而且会造成无数的祸患。很难说哪一种是好的，可以说这两种都是错的。
