# 第1068章 学术权威信口雌黄时_该不该挨巴掌

学术权威信口雌黄时，该不该挨巴掌？

《警世七律·读阎教授事有感》  
青史斑斑血未干，鹰犬犹自颂康乾。  
格格巷里迷金钏，粪坑堂皇饰玉冠。  
剃发令沉三百载，复辟心燃一夕寒。  
劳动骨节铭真史，莫使丹心付笑谈。

教授当年挨打的事情又被人家翻出来了。  
有人说打人就是不对的。  
那个人是个什么什么分子，那个人要搞什么什么分裂，给人家的帽子戴得老高老高的了。  
为啥挨打呀？对，他是个老师，他是专业人士。  
那为什么就不能挨打呢？当他满嘴信口雌黄的时候，为什么不能挨打？这不是活该吗？他就是欠揍啊。  
他做了错事又狡辩，讲道理又不听，又仗着自己有学术权威的身份打击异见者。那别人万般无奈的时候拽他一巴掌好吗？对方还成分裂分子了。  
他在上面吹满清十二帝个个是圣君的时候，咋不说他这个人历史虚无主义呢？

这个事儿也算是一个离谱的现象。就是在我们国内学习清史的人当中，有相当相当的一部分，学完了之后就变成了满清鹰犬。  
学历史要以史为鉴啊，好的东西要发扬，坏的东西要抛弃，这是应该的吧？这才叫做以史为鉴吧。  
但是学满清史的人不一样啊。相当一部分学清史的人学到的就是该怎么夸呢？应该怎么变着法儿地再夸一下呢？怎么夸能够让大家接受呢？怎么夸能够让大家觉得这个事儿没错呢？  
啊，就连一个狗皇帝活得很长，都变成了他们盛赞的东西。一个糟心的玩意儿，活得不就是祸害大的意思吗？结果他们吹得呀，哎呀，那个花啊。

这里头啊，嗯，我当年有个同学，他不是学这个的啊，不是学历史的。他就是单纯地去看了一下清史稿。大家知道清史稿那本书，实际上就是一个清初写的啊。在那里面那个是就写那个圣君来着，圣天子啊。然后他看了这个书之后，每天都在那里分享满清的盛世，心向往之。  
哦，我的天哪，真的是心里堵得慌。为啥呢？你一个女的，你向往满清？你知道女的在那个时候活得有多艰难吗？男的兴许还能遨游，你知道那女的那里活得有多艰难吗？完全出头无望啊。结果他爱得亲爱得不得了。  
这就是学清史啊，最后学成了清粉，是一种非常离谱的行为。

嗯，这个小干杯是谁给的呀？我这也看不见啊。哎，好，谢谢。嗯，这个小小干杯。哎，好嘞。嗯，啥裹小脚还向往呀？那谁知道呢？万一人家心里头想的是，嗯，过去了是个格格呢？这也不好说啊。你看我们现在互联网上有多少精神格格。  
刚才的时候不是还有人在那里反馈吗？啊，说批清啊，就是放过了那些江南士大夫卖国贼。江南士大夫卖国贼和满清是同伙啊，批清和批他不是一样的吗？唉，不行。他把这两个拆开，他说卖国的是士大夫群体，不是清士大夫群体，就是把国卖给了满清，然后满清再二遍倒手，再卖给了列强。咋了啊？为什么要把这个中间最重要的、破坏性最强的满清成建制、破坏的满清摘出去？为什么？因为他自个儿是个精神格格。  
你把清批得一塌糊涂，说清是个粪坑的时候，他岂不是变成了蛆？他不想当蛆，他想当小公举，他想当小可爱。那怎么办呢？他必须说啊，清的建筑美轮美奂，之所以你觉得那个地方不好，那都是你们汉人不好，都是你们的士大夫不好，都是你们那里的汉奸做的乱。  
汉奸跟这个粪坑之间确实有关系，汉奸缔造了这个粪坑。但是精神格格、精神阿哥就是在这里头雇佣的蛆。那啥，觉得清好的，推荐他剃个头。嘿嘿嘿，你这容易伤害到人家脆弱的小心肝儿啊。要知道别人是颜值主播，别人爱美得很呢。

啊，我是承认自己是一个中年劳动妇女的。你看到我的手的时候说，哎呦，骨节怎么那么大？因为我是劳动妇女，我干活啊。啊，我那个书上楼，我要搬，我要扛呀，我做事情的呀，是吧？我要做饭的呀，是吧？几十年的劳动妇女，我的手的关节有一点点粗，是很正常的事情。  
但是别人是娇小姐啊，别人是格格，别人是精神贵族，别人是颜值主播。那你不能说别人啊，说你去剃个头啊，那他不带干的啊。他怎么可能会去剃头的呢？  
嗯，我我还做饭啊？对啊，哈哈哈哈。我我我做饭啊。为啥吃饭呢？啊，你要吃总得做吧。就我这把年纪了，你不能想象说我还在那里让别人给我做饭吧？不能够吧。我啥时候还能够说成为被人家照顾的小公主？不可能的，不可能的啊。  
嗯，做饭骨节会粗吗？呃，简单的做一点其实也不会。但是你要是长期长期都自己做的话，嗯，做普通的家务可能就是我这样。那如果做的活儿更重一点的话，可能会更粗。我并不是那么严重，一点点而已。你干活的话，多少都要用到手的嘛。持续不断地用的时候，难免就会有一点点粗啊。  
啥？你也会做？你也做饭呢？对啊。大部分人，大部分人在家里面已经设立小家庭了之后，都会开始做一点饭，多少的问题而已。然后年纪再大一点的时候，就会觉得外面的东西真的是不好吃啊，就会开始自己做饭，多一些。  
没关系呀，爱做的做，不爱做的不做。时代在发展，技术在进步。我也支持大家能够过一点轻松的日子，我支持所有人享受生活。只是呢，咱们只要别忘记来路就行了啊。知道一个人呢，劳动是很可贵的，尊重每一种劳动就可以了。但是有条件不劳动、有条件休息的时候，我是支持的。谁还不想躺在那个地方呢？是不是多舒服呀？我也觉得很舒服啊。所以不做饭也很正常。但是呢，做也是一个正常的劳动需求啊。就就这么单纯地扯两个闲话啊。闲话闲话。

嗯嗯。什么什么什么？嗯嗯，满夷是哪些人？谁能列举一下啊？不行的话，你去查一查那个2019年的那个大案要案啊，它的名字叫做满洲协和会啊。虽然你自己肯定知道，甚至你有可能是成员之一，但是不要紧啊，人家有组织的，组织有名单的，自个儿去查。还被抓捕了，还有公告，有新闻，自个儿去找。嗯，干啥呀，还让我给你备清单呀？嗯，满地上撒泼打滚？不然呢？不然呢？他们是这个样子的啊。自个儿就是个满遗，一定要假装纯洁无辜地说，哪有满遗？哪呢？哪呢？哪呢？没有啊。嗯啊。不然呢？他一定要掩盖一下自己的罪恶吗？肯定是这样的。

嗯，2019的抓捕没有清干净吗？怎么可能完全清干净呢？他只是一个，那只是一个复国组织啊。这个是矛盾比较尖锐，行动比较剧烈的一群人。就是说匪首被抓了，但是头目被抓了，不代表底下的这些二五仔们或者是这些小马仔们还在啊。这些人依然活跃在互联网上啊。所以呢，我们的互联网上到现在依然活跃着大量的小满遗。存在是一定存在的，但是呢，他们成不了气候。为什么？因为人民群众的眼睛是雪亮的。一个东西是美还是丑，是好是坏，一种行为是对还是错，今天的中国人是有判断的能力的。我们已经不再是满清时代那种蒙昧之下、被愚弄之下的百姓了。我们今天有眼睛会看，有耳朵会听，有书会学习。时代变了。这些精神格格、精神阿哥已经没有能力愚弄大众了。所以群众必然觉醒，中华必然复兴。
