# 第1027章 揭秘热搜背后_推流发行与上榜真相

揭秘热搜背后：推流、发行与上榜真相！

本来就是上不了台面的事情，奇怪的是它能上热搜。上热搜需要的是经费，而不是台面。我们一定要理解它整个真正的运行模式。你得明白这种传媒系统当中运行的基础模式——从传统媒体开始，它的基础逻辑都是一样的：一方在进行内容的创作，同时还要兼顾发行端，这两手都要同时抓才行。

所以大家今天所看到的所有的推流也好，热搜也好，实际上意味着它有一个推流端，它就是过去的发行端，相当于过去的出版社、报社、电视台的功效，是进行内容发行和分发的机构。那么，如何能够登上某电视台的发行端呢？要么你这个事儿确实已经扎眼到不报不行的程度，否则就说明你失职；但凡这件事情没有这么扎眼，不是非投放不可，不是非看不可，错过了也无所谓——只要不是这种爆炸性的内容，其他内容想要进入大推流的浪潮之中，背后是需要有经费的。

这种经费有几种方式：一种是有专门的单位来对它进行推送，这属于大推手的批发价；另一种是零敲碎打的，大家在很多平台上能看到，比如我给它转评赞，给它打赏，给那个视频发送一些小花花，或者专门给它投流等等。这种属于散户进行的分散式推流，属于零售端。大户集体推的那种就是批发端。所以按照这种方式来算，就比较清楚了。你为什么会在那些非常核心的点位、那些C位的点位上，看到那些不咸不淡的东西？原因就在于，人家买了广告位。

热搜不是大家搜出来的吗？那好，我给你分享一下热搜的运行原理。就是一件事情——尤其是比较典型的，那些流量小花小草——你有没有注意到，有很多流量小花小草你压根儿就不认识？你认识谁？你认识哪位？你不认识他，但是你经常会在热搜榜上面看到这些流量。有时候名字我也不记得，就叫他张小花好了。说那个张小花今天早饭的时候崴了脚，粉丝心疼泪崩，上了热搜榜榜三。你看的时候懵不懵？在这个东西上热搜之前，你怎么可能知道他吃早饭的时候崴了脚呢？在他上热搜之前，他的粉丝怎么可能知道他崴了脚然后泪崩呢？所以你颠倒一下次序——真正的次序是什么？真正的次序是先把通稿写好，咖位买好，然后第二天再出现“买早餐的时候崴了脚，粉丝泪崩”，这才是正常的演出顺序。哪位也不认识，还以为有很多人喜欢呢？没有，别想了。那不是什么大众投票，你把它理解成广告位比较简单。大家知道的就是那个什么新闻联播之后，天气预报之前，那属于全国最黄金的顶流广告位，那个地方的广告位价位特别高。

其实就是不同的平台有不同的广告位。商业平台有商业平台的广告位，内容平台有内容平台的广告位。你能够上这种地方的时候，因为流量很大，所以一般都需要批发价冲进去，机构批发进。但是我们目前的这些普通的、小up主的、散装的视频，你在背后给它什么点评赞、投币、小花花、小草草这些东西，它提供的是属于一个零售端的微量推流。其实意思都是一样的。
