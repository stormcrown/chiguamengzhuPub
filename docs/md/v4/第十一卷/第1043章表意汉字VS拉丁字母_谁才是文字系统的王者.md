# 第1043章 表意汉字VS拉丁字母_谁才是文字系统的王者

表意汉字与拉丁字母，谁才是文字系统的王者？嗯，是不是忘了讲象形文字了？对啊，呵呵呵呵呵呵呵呵哈哈，忘得干干净净了。谢谢提醒，幸亏你还在，哈哈哈哈哈。嗯，其实啊，中国的文字——如果你是在说汉字的时候，我们不应该把它直接归类为象形文字。象形文字呢，可以说是汉字的前身的前身的前身，哈哈。它的草创期，有一部分是可以归类为象形文字的，就是用画画的方式来写字。但是很快我们就会发现，用这个方式没有办法描述所有的东西，所以我们需要把它进行升级。这个升级的过程呢，就是把它抽象化，然后进行拼接组合，再创作。于是到了这个阶段的时候，它就不是用画画描述一件事情了，而是直接传达一个含义。因此，汉字应该叫做表意文字。它真正和其他文字的区别在于一个表意，一个表音。这个世界我们可以把它归成这两个系统，就是表意系统和表音系统，千万不要归拢什么象形系统之类的——没有啊，象形永远都是一个草创期存在的东西。苹果，你可以画出来一个苹果，但美丽你怎么画？画不出来。所以呢，在我们的文字发展进程当中，非常早期的时候，是有一部分象形的符号，但很快，真正到了仓颉造字、鬼夜哭的这个阶段的时候，他是在传达含义本身。

那么表意文字到底是干什么的？或者说今天很多人去吹嘘的表音文字，到底有什么好处？它为什么会出现？其实我们可以这样说：表音文字并不是一个自然产生的东西。为啥在自然诞生文字的状况下，你不可能创造表音文字？因为——这个叫什么来着？相隔十里，乡音不同。这句话是咋说来着？忘了啊。反正就是只要隔了一段距离，我们的口音就会有巨大的差异。所以呢，当你创作的一套文字系统是表音的时候，就意味着你只有很近的距离才能用。但是很近的距离你用文字干嘛？喊一嗓子不就行了吗？啊，走两步道累不着，你犯得着去创造个文字吗？不值当啊。所以表音文字不可能是——十里不同音，对对对啊，你看这脑子瓢的，分分钟就能把它撂开啊。那表意文字没有办法自然诞生，因为你能够保持同口音的这个范围之内，你不需要文字，距离太近了。而你需要用文字的方式进行传递的那个地方，跟你口音不一样，你用这个文字一毛钱作用都没有。而且呢，表音文字一个巨大的问题，就是它不可传承。口音不光是距离远了就会变，迭代了也会变。你现在去到那个——这还是我们现在有各种普通话的普及教育，如果没有的话呢，你隔了几代人，大家的口音差距就会非常非常大，而且措辞、用词的方法都会发生改变。现在很多小孩子是不是就会自己创作一些他们自己内部流行的东西？如果我们没有稳定的语言文字的教育系统的话，他们到成年的时候，也会在他这一代人当中使用这些他自己草创的词，然后再迭代下去又变形了。所以语言是非常容易变化的。那么表音文字无论是距离传输上面，还是时间传输上面，它都没有作用，因此它没有办法自然诞生。它必然是已经意识到有文字系统的存在，然后才会为了便利而创作出来表音系统。

表音文字是有它的优势的，它最大的优势就是学得快、容易、简单，或者叫做 so easy。表音文字是非常好学的。那个我们隔壁的思密达他们家，他的那个符号就是圈圈杠杠的那个玩意儿，他是一直试图向全球推广的，而且做了不少的努力，派了很多的团队跑到那些文字系统不健全的落后的地区去推广它的表音文字。他为啥那么自信就出去呢？就因为思密达是宇宙中心嘛？其实不是，是因为他这一套表音符号还真挺好用的。属于你拿它当拼音的时候，有很多的文字系统都可以和它的这个拼音兼容，学起来的效率很高，非常容易书写。因此呢，它属于那种短、平、快、迅速就能上手的一种文字。表音文字的特征就是这个啊，就是学得快、成本低、普及率高，这是它的好处。那么坏处呢，坏处就是说的话很有可能出现歧义，经常性的不准，然后呢，多传了几句话之后，很可能就偏得让你理解不了了。哎，这个就是表音文字的弱点，那它的强项和弱项就放在这儿。

那表意文字呢，它就是第一，它稳定。像中国自古就是一个大一统的国家，我们的那个中央政权常常居住在离它的边陲地上千里之遥。这个距离内，经常会遇到边陲之地的人说的话就是鸟语、外语，听不懂，压根听不懂，完全理解不了，又到了这种地步。到了这种程度的时候，我们如何沟通？用文字。当这个文字是稳定系统的时候，它就可以超越于这个距离之外去进行沟通，这是距离上的。同时呢，就是时间上的，我们现在的人啊，时隔千年，只要说仔细地去端详端详，我们去看秦代的那个书简的时候，还能认识一部分的字。因为秦代的时候已经出现一部分的隶书了，所以呢，有一些字我们就可以认得了。如果你还钻研了一些各种字体的变化的话，更早期的字也能够认得一些。这个呢，就是我们表意系统里面它的文字的稳定性，所以它对于距离、对于时间都有非常非常强的优势。但是它也有劣势：学得慢、成本高，你要去进行训练、进行普及的时候，确实比较昂贵一些。

嗯，表音文字好学是错的？又不是只学音就行。我这话是针对于他们本民族而言的啊。因为这些文字，当你作为一个中国人去学 ENGLISH 的时候，你当然会觉得我还得背本字典，但是你觉得人家本岛的小孩子呢？学了基本的组合之后，就已经可以写大部分的东西了。啊，这个是它的应用场景。我们的这个教学成本其实是真的挺高的。所以呢，包括在比较重点的就是在宋、明代、宋明两朝，尤其是明代的时候，其实是有一部分的强制教育的。就是官府会专门地去推广识字，对于普通人、对于平民子弟强迫识字。这个识字不等于有文化啊，你不是认了字儿就啥都会了，不可能的。那些知识、那些经史子集的东西，没有人引导、没有人教你是不可能会的。但是呢，官府会强迫大家去认字，认了字之后，你的工作能力、远程交流的能力会提高。所以我们会看到明代的时候，出现了通俗小说、白话小说的创作井喷，就是因为普罗大众很多都认字，但是文化水平不高，所以呢，他就去读这些比较通俗易懂的，甚至很流俗的一些文章，满足他的需求。这个就是明代的时候之所以形成创作井喷的原因。他这识字率比较高，但是需要强迫。嗯，其实也意味着它成本比较高啊，那不是你背个十几个符号，你就可以把你说的话都标出来的，标不出来。你必须要认得相当几百个字嘛，起码要认得几百个常用字，才能够基本地进行一些小字条的书写，认得一点基本的物件儿怎么标志，你才能够在店里头当个伙计，基本上是这样的一种模式。但是到了清代的时候，这个门槛就瞬间提高了，识字率极大地降低了，只有那些有机会读书的人才能识字了，普通人就几乎丧失了识字的机会。所以它本身上面还有一个就是成本的问题。

因此，在革命时期、五四时期的时候，有人专门地提出来说要消灭汉字。当然我不知道他背后的动机有没有一些其他势力的怂恿，但是当时有一部分的学者是希望消灭汉字的，并且把它拉丁化，因为那样能够快速地普及教育。当然，谢天谢地这个事儿没成啊，谢天谢地也感谢那个写“施氏食狮史”绕口令的那位伙计，也谢谢他。总之呢，这个拉丁化的过程没有完成，不然的话，今天我们也认不了那些我们的先贤书写的东西了，我们也变成了一个认字儿的文盲了。呃，谢谢他们当时的这些力挽狂澜的举措，所以今天我们还认得祖先的文字，也能够直接阅读祖先写下来的笔记和历史书，我们才能够看到历史的真相。如果没有这个的话，那真的是别人怎么骗你怎么是，就像隔壁的思密达，他为什么看上去那么自我为中心？一个呢，是他的国家有这个需求，所以给他们灌输了这个；但是同时另一个方面就是只要国家教了，他就会相信，因为他自己没有识别的能力，他不认得那些字。那些书他也看得着，但是他不认识，他的历史书就是用汉字书写的，可是他不认识，于是就断了，于是他就变成今天那个模样。中国人不要变成那个样子，所以我们得保留汉字。
