# 第1031章 曾国藩_圣人还是恶魔_捧上神坛的刽子手

曾国藩：圣人还是恶魔？捧上神坛的刽子手，如何腐蚀现代价值观？  
——————————————  
乱世枭雄血洗裳，金陵城下骨凝霜。  
厚黑窃鼎郑伯术，急功焚玉商於殇。  
九仞台倾规则裂，千疮舟覆众生长。  
休将刽子手奉周鼎，留取清流待新阳。  

嗯，关于那个什么曾国藩的吹捧这件事情，确实有可能是比较重要的一件事。因为呢，在那种乱世之中，出现他这样的枭雄、刽子手、恶魔，很正常。真的，当一个世道乱成这个样子的时候，一些没有什么良知、但又很聪明的人，借势而起，就会变成他这个模样。但是呢，我们近几十年始终都有人在推他，这个事，反倒比曾剃头他自己要重要一些。就是为什么把他推起来？为什么要吹捧他？他身上有什么特点会影响到大众呢？  

我个人的认知当中，认为他和那个我们今天直播间刚刚开始的时候说的厚黑学，还是很配套的。对于他的吹捧，和对于成功学的吹捧、对于厚黑学的吹捧，都是差不多一个时期开始的。应该是这样，我个人判断的时候应该是这个样子。它是一种什么状态呢？就是非常逐利，非常个人中心主义，非常唯结果论。这种东西会带来一个什么样的好处呢？就是在个人层面上，当你抛弃了道德，抛弃了一切的约束的时候，然后呢，又脸厚、心黑、手狠的时候，很有可能你能拿得到比别人多的好处——这是好处。  

但是另外的一个层面，嗯，啥？嗯，没有。嗯嗯，诶，好，嗯。好的好的，我突然这么说话，有点不知道怎么接了。你们在公屏里面跟他说吧。嗯，我不太介意这种事啊。诶，好好好。然后呢，哎，就是他对于整个社会而言，却会形成一种非常坏的风气和破坏性。他的要求啊，就是厚黑学也好，成功学也好，要求的是一种急功近利、不择手段的方式。当你已经开始走向了急功近利、不择手段这条道路的时候，就意味着你要去钻规则的空子。你所占到的每一分好处，都是从别人身上捞出来的。那这个时候对于个体来说，它有可能从中渔利，但是呢，从整体上来说，它破坏了规则本身，就会使得这个世界上面呢，或者说这个群体的内部，失去互信的基础。我不知道你什么时候捅刀子啊，这个就是最大的问题。它坏掉了规则，这就意味着会提高后来的所有人进行经营、进行交往时候的成本，因为每个人都必须要提心吊胆了。  

我们不能够要求所有人都是圣人，说全部都是胸怀宽广，十分宽厚，一切都容让，确实也做不到。但是如果说我们明知道社会上面会出现这些蝇营狗苟、脸皮厚、心又黑、手又狠的人，我们谁都知道有这种人，但是呢，这些人如果说他是负面评价的话，他毕竟要收敛一点，而且那些普通的人不太容易往这条路上走，毕竟评价低啊，因为大家都知道他缺德。但是当你的社会评价把这种人哄抬到了一个很高的位置上面去，让他成王成圣，说他这个道路才是对的，他才是精明的、聪明的，不会这个东西的人活该倒霉。当你的舆论氛围是这个样子的时候，就意味着大部分的人都要想办法放下心中的规矩，学会用这样的方式去钻营。而当越来越多的人钻营，那这个规则千疮百孔，社会的运营就会成本高昂，是每一个人都要为这个事付出代价的。不光是我们这些傻乎乎的人啊，也包括那些机灵古怪、在前面捞到了好处的人，都一样。规矩坏了，所有人都会跟着一起吃亏。  

所以这些年来就一直在哄抬那个曾国藩，把他吹得特别特别特别厉害，特别特别优秀。然后去吹成功学呀，去吹那个厚黑学。现在好像互联网上还有人在吹厚黑学，就是你不会这些东西你就成不了，这些东西才是别人不告诉你的不传之秘啊。什么不传之秘啊？察言观色啊，做小伏低，伺机而上，能占就占，能好处就捞。这玩意儿算什么不传之秘？需要的不就是你缺德不要脸吗？但是大家要脸的目的是让整个社会都有好处，然后自己呢，能够慢慢慢慢地向前走，一点一点踏踏实实地做，也行。为的是这个。如果我就是为了眼前就能把好处拿到，那近道有的是啊，这世上有几个人瞎到一条近道都看不见的？但是人人都去找这个近道的时候，秩序崩坏，所有人都在丛林法则之下，你每天都要提心吊胆，那日子还过不过了？所以这种宣传比那个什么曾国藩本人要严重得多。  

曾国藩嘛，乱世出这种烂人很正常的。而且就这种烂人，在那种乱世之中，本来就没规矩的时代，他比较容易起来，比较容易成事，很正常的。乱世出枭雄就是这个事嘛。规矩乱了之后，这种人就比较容易异军突起。所以他能够成事儿，他能够拜相封侯，很正常，很合理。但是你吹嘘他，把他当做绝对正面人物，让大家视他为榜样去学习，这是很坏的事情。
