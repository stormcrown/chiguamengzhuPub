# 第1065章 天文学为何只能在中国诞生_需要定点观测加脱产团队加文字加数学

天文学为何只能在中国诞生？这需要定点观测、脱产团队、文字和数学的支持。正如《观星谣》所吟：苍昊悬星汉，愚氓妄测天。徙族逐寒雁，断简葬荒烟。耒耜盈廪日，观台继火年。谁言彗迹渺？七十六回圆。算经传九数，文脉续千川。莫笑前人痴，长夜独醒先。

古人很早就知道星空的重要性。但是，如果你要说他们完全理解了重要性，我觉得不太对劲。为什么呢？因为不见得是因为理解了重要性。事实上你会发现，全世界所有的人都对星空非常好奇，各个地方都有关于星星的传说。比如，有人讲古埃及史时提到，古埃及的天文学有一部分时间是靠尼罗河的汛期来界定的。但这个事肯定很荒唐，汛期哪能每年都那么准时呢？还有另一种类型，也是观测星空的故事，假定天空是平的，然后用什么车拉动着星星移动。他们那里有一套这样的传说。而我们北边的游牧群体，也有关于星空的传说，迁徙时会锚定某一颗星星的方向，跟着它走。所以，对星空的观察和好奇是全人类共有的。全人类都对星星很好奇，都想知道它的秘密。

区别在于：中国人有能力做到这一点。所以有兴趣去了解，这不奇怪。人类都有好奇心，文明能够进步，正是因为有好奇心。全世界的人类都对星星很好奇，但是中国人做到了这件事。怎么做到的呢？我之前做过一期与历史相关的视频，里面专门强调了记录这些东西的难度——要把一个东西长年累月累积下来，这个难度比很多事都重要。有一本书叫做《星空帝国》，专门讲我们的天文史。我觉得这本书值得一看，虽然我不认为里面的东西都让人满意，但整体上还不错。关于星空和星星的问题，大家都好奇。但你需要有条件去满足这个好奇心。

世界上天文学只有一种诞生通道，要成为科学，只有持续不断的记录，没有别的选项。为什么？因为星星的变化触不可及，只能看。而且看了之后，你知道它变了，但不知道怎么变的，也不知道变了多少。这需要持之以恒地记录、记载，然后观察期间的变化和移动，再去计算、总结规律，这才叫天文学。不是说我有一个工具能看见就叫天文学，不是的。天文学指的是长年累月的记载、总结和计算，这才叫天文学。

谁有条件缔造天文学？天文学的第一大前提是定点观测。你要确定星空怎么转，自己得锚定。如果你还在移动，怎么确定星星怎么动呢？所以必须定居，有稳定的定居生活，这就是天文台。没有安安稳稳、固定在特定位置的天文台，这个学科必然子虚乌有——这是第一个要素。

第二个要素：粮食生产要足够。为什么突然扯到粮食呢？因为你需要持续不断地监测，这意味着观测者是脱产的，而且不能一个人看，一个人看不动，会累死。至少需要三班倒，监测天空的变化，这需要大量脱产的工作人员。每一个脱产工作人员都需要几个农民在后方支撑粮食供给。所以粮食生产必须达到有富余。而且要知道，即使这样，这些脱产的科研工作者依然不是社会统治阶层，因为必然还存在神权贵族和军事贵族，他们也脱产。这么多脱产的人，优先满足谁？优先满足神权贵族和军事贵族的需求，之后还有富裕，才会给科研人员。因此，整个社会的农业生产必须非常稳定，有剩余粮食，才能干这件事。

第三，还需要稳定的文字系统——记载符号。因为星空的变化非常缓慢，需要观察很多很多年，几十年起步，而且还得不出任何结果。为什么？扫把星的运转周期是76年（或70多年）。你得观察它几次，才能确定这玩意儿有规律，那几百年就过去了。光是确定扫把星有规律，就得花几百年。比如，看一次A发现规律，用我的名字命名，好牛啊。但正常人类做不到。我们是正常人类，做不到怎么办？就持之以恒、持续不断地去记录。要保证前一代人的记录，后一代还能看懂，所以文字系统必须稳定，符号系统必须稳定。整个记录必须安稳、有传承。要知道，能打断传承的东西特别多：一次大规模混战、一次彻底的文明破坏、一次完全的蛮族入侵，都可以直接摧毁一切。但如果能把所有档案都留下来，就意味着活下来了、传承下来了。传承听起来就两个字，但能一直传承，说明一直活着。在人类社会里，这多难啊！死一次就回不来了，这才是常态。世间所有活着的人都有祖先，但别人都记不得自己真正的祖先是谁了。我们还记得，为什么？因为我们没有彻底失败过，这叫做延续性。这些条件都必须具备。

同时，光记下来就好使了吗？不行。光记下来只是档案，要总结出规律还得会运算，这就是数学算学。数学方面，先秦就有了九九乘法表。九九乘法表是什么？是顺口溜。一个学科已经出现顺口溜了，说明这个学科非常成熟。而所有天文记录如果不经过计算，是没有任何价值的。你不算账，记这个账干什么呢？所以天文学必须配套数学知识，两者是一体的。因此我们的数学也必须非常发达：十进制、二进制、五进制、十六进制、十二进制、六十进制，这些不同的进制法则用在不同的地方。这是什么？数学能力，系统性的数学能力。要缔造天文学，就需要这些东西，缺一不可。甚至我提出来的还不是非常完整，很多细枝末节的东西都不屑于讲。但就光是这几条——世界上有哪个群体能够实现长期、固定、稳定的居所？延续的文明文字、语言系统和技术符号？还有算学系统？以及充足的农业发展，能够供养脱产人员？这个世间有几个群体能够做到？所以天文学的诞生是非常伟大的一项学科，没有积累，就没有办法获得天文学。
