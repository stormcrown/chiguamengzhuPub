# 第1021章 宗族文化探秘_为何宗族能在现代社会中依然存在

宗族文化探秘：为何宗族能在现代社会中依然存在？

宗族其实是人类以血缘为纽带建立的一种基层社会组织。这种组织能够延续数千年存在，就是因为它确实对宗族内部的人来说很有用——对它的生存、存续很有用，所以它才能一直延续。宗族始终与大一统的中央王权之间存在一种矛盾关系：它承担了基层治理的同时，一旦宗族势力做大，就会影响中央权力的推进。因此，历朝历代都对宗族有打压的策略——这不是今天才开始的，而是从来都会打压宗族。通常，一个宗族聚的势力过于大了之后，就会被强行打散，比如强制分家或强制迁徙。这件事在汉代的时候就已经开始成系统、成规矩地推行了：那些大型宗族每隔一段时间就会被专门拆解开一部分，送去修皇陵。以至于汉代皇室的皇陵周边都会形成非常大的市镇和市集，而且生活条件非常好，城市的条件也非常好。为什么建得这么好呢？因为朝廷专挑那些大宗大户，专门迁大户去修皇陵，就是为了把他们的系统打散，否则他们的力量就会特别特别大。所以，宗族本身是在不断被打击、被拆散的过程中，数千年绵延不绝地流传下来的。

因此，我们首先要承认一点：宗族本身非常有生命力。为什么有生命力？主要原因是人需要组织，才能把力量团结起来用到最大。一旦遇到任何乱世，你会发现——平时你可能会觉得亲戚烦人，还要走亲戚、讨厌送礼、来来往往嫌弃死了——但实际上，每当遇到真正的乱世，你就会发现它叫做“打断了骨头连着筋”。因此，虽然宗族历朝历代都受到打击，但最后依然是星星之火地流传下来，而且在很多地方都有这种遗存。现在那些过去宗族已经衰败得不成样子的地方，也开始一点一点地重新出现。它出现的原因，必然是由于我们在基层上面的一些人缺少组织管理，需要自己投靠一个群体却找不到合适的，那么这种情况就会去寻求宗族的帮助——这就是人的自然集结的属性。它有这个好处，所以才聚起来。

像我们在潮汕、两广地区，宗族保护得比较好，留存率比较高。我们就会注意到这些地方有一个特点：他们的公共活动办得很好——这是第一项。这些宗族在地方组织、活动组织、节日礼节办理这些事情上做得很好、很顺畅。另一个方向，有没有发现那些做生意的人，好多都出自这些地方？为什么？你想一想，你做生意的时候，第一件事情要有第一桶启动资金。启动资金，一穷二白的时候怎么办？需要借。怎么借？你之前没有任何信用时借不到，找谁？用什么方式？那么宗族就可以提供相关资源，因为再怎么样算是同家，而且你跑得了，你爹妈跑不了。所以相当于有背书、有信用作保——以自己的血缘纽带身份作保，就有了第一桶金。然后在做生意的过程中，所有亲戚之间互相拉拔、提供信息，最后就团结起来了。因此，这些宗族保留得比较好的地方，你会发现这些人在做生意这件事情上，比其他地方的人顺畅、丝滑、容易得多，甚至年纪轻轻就把自己的买卖盘起来了。这就是宗族能够对这些人起到的作用。

它为什么能够留存？就是因为它有用。但也正是因为如此，这群人很有可能就会越聚越大。而当他大到一定体量时，就会寻求其他的诉求，所以他依然要面临一种控制——一定要有序，要控制、要约束它的成长，不能让它肆无忌惮地长起来。要知道，过去中国人讲究“家国天下”这样一个阶梯的进阶。家是什么？家就是家族。家国天下能够把家、国和天并到一起，放在一个序列上讲，就是因为家族做大就约等于小国，小国做大就是大国，大国掌控就叫做天下，我们就是这样进阶涨起来的。因此，它是我们的底层文化代码、组织代码，所以它没有办法彻底消灭，一定会存在。但是同样的，也因为它的这种成长性，我们必须对它予以约束。

基本逻辑大概就是这个样子。这件事上，你要是说它的坏处，其实大家都知道了，因为这玩意儿已经被批判很多年了，坏处我就不单独说了。但是它也有好处。好处之一，是刚才所说的，它会对其家族内部的人，尤其是年轻人提供帮扶，提供一定的兜底条款，相当于有一个社会公共信用背书。同时，还有一个好处：当你有这种基层组织可以依附时，就不容易进入到那种信教的系统。大家去看一下宗族兴盛的地方，那种传教士就很难踩进去，因为大家已经有互帮互助的机构了，就不会去相信这些地方。而什么样的地方会被传教渗透得特别深？就是那些留守老人特别多的地方——他们缺少儿女的帮助，缺少周边亲人的帮扶，缺少基层组织。而所有的人终其一生都需要生活在人群当中。如果你没有身边这些人群给你提供互助，没有家族或同宗各色各样的人提供互助，又没有共同信仰、共同理想的同志提供互助，那么最后就很有可能被共同宗教信仰的人拐进去，开始信教。这种情形在中国已经开始出现，我们要开始警惕这种事。

所以，当我们面临这种局面时，宗族在今天的中国起到的正面作用会更大，因为现在国家的掌控力是足够的，不用担心它造反。既然它最大的风险我们是可控的，那么剩下的几乎就都是好处了。

我搞不清楚你是怎么想到说“宗族思想当中有墨家思想”的。因为宗族建立的最基本原因就是“我的亲人比外人重要”，而这种观念叫做“爱有等差”——我的亲人比其他的人要紧。宗族建立的根本就是爱有等差：优先考虑自己的家人、亲人、族人，这是宗族的立身之本。而墨家最根本的东西是“兼爱”，就是视世间所有人为一样的、一致的。那么在爱有等差的系统看来，这种兼爱思想实际上就是对自己家人的一种背叛，也是一种背德的行为。因此，我不认为宗族系统和墨家有任何的关系。但是宗族盖的祠堂有可能用了墨家的一些技术。
