# 第1014章 古典时期中国人饮食状况揭秘_粮食产量与耕作条件

古典时期中国人饮食状况揭秘：粮食产量与耕作条件是关键！古之华夏，饮食俭朴，丰歉不均，明时粮产颇丰。中土善用粪肥，循环不息，故田畴永续。清季虽引新种，然汉民困于十八省，地寡人稠。观乎村舍，中土净洁，泰西秽乱，此乃农事之别也。盖农为国本，法异则效殊。

关于古典时期的中国人吃不吃得上饭——能吃。但很多时候，我们应当从一些侧面的角度去关注它。如果凭想象或按“真话”来说，古人能否吃饱饭，与今天相比肯定不能。若与今天对照，古人的标准中“吃喝嫖赌”是可以拿来败家的。其中“吃喝”二字——后面两个不说了，大家都知道能败家，什么时候都能败；但前面“吃喝”也能败家，这说明在古典时期，对于吃用方面能省则省，并没有多少富余，是“可着人头做米饭”的那种。因此，富裕程度绝对比不上今天。但任何时候评价它好或不好、吃饱或吃不饱，一定要建立在当时的生产条件和时间背景上。所以前面一节，我从一个方面切入，讲粮食产量——种子的投入与产出量之间的对比。

中国这边，明代一亩田的产量已经可以达到400斤了；而欧洲那边，撒一斤种子只能结二斤粮食。这种情况下，指望能吃饱饭会很困难。为什么欧洲产量低？一方面育种有影响，但更重要的是耕作条件。耕作条件涉及很多方面，第一个就是肥料。农家肥这种东西，欧洲中世纪城市那么脏，就是因为他们不会利用农家肥。英伦岛上甚至把河道堵塞成街道，就是因为不断倾倒垃圾和粪便。这在中国是不可想象的——那就相当于把黄金泼掉了，等于把自己最重要的生产资料扔掉了。中国绝对不允许这种程度的浪费。所以之前我曾带过一本书，不知道现在还有没有货，前阵子断了，叫《四千年农夫》。这本书是清末时，一位美国农学教授富兰克林·金来到中国考察农业专业技术。当时他对中国乃至整个东亚地区的农业感到十分震撼。虽然当时的美国现代农业已经很好了，但面临困境——绑定了化肥和机械，导致大量土地板结，很多地耕作后产量越来越差。所以这位农学家来中国考察，就是想了解我们的土地耕作了四五千年，却依然能养活那么多人，并且保持肥沃，是怎么做到的。因此，这本书留下了不少珍贵档案，记录了清末时期中国农夫从年头到年尾做了什么、如何使用耕牛、甚至牛奶质量如何等等，全都是工作笔记。这东西其实是一个地方的人能不能吃饱饭的基础中的基础。

当然，因为他考察的是清末，条件已经不太好了。清代农产有好处——从明代开始的全球种子大扩张，在清代推广了很多高产且能利用空余土地的物种，例如玉米、红薯等，这些高产物种让人半饥半饱，在清代大面积播种，这是有利的一面。但同时也有负面：清朝的政策把最会种田的汉人圈定在汉地18省，不准他们向外行走。我之前的直播讲过这事儿——所谓“只有汉地18省”，就是清代皇帝把汉人圈禁在18省范围内，不准离开。这导致土地数量不够，因为人口不断增长，正常王朝需要开疆拓土、移民实边、开垦新土地，但清代没有，反而压缩了土地，所以清代的耕地保有量和人均耕地变得非常少。富兰克林·金来考察时已是晚清，所看到的场景属于比较糟糕的环境，但他依然感到震撼：第一，我们的农村非常干净，连苍蝇都很少，和美国农村的脏乱差完全不一样。为什么会这么干净？因为农村没有什么垃圾，所有东西都必须有效利用，没有垃圾自然干净。那些粪便之类的东西——你现在去欧洲旅行，动不动找不着厕所，难受；但在以前的中国农村不存在这个问题，现在也不存在。因为每隔一小段路，换了一家田地，上面就会有一个茅厕，人家免费修的，就是为了路过的人憋不住时方便，这叫“肥水不流外人田”，是基本的原则。现在去国道上行走，也基本上每隔一公里就会出现一个私人茅厕，到现在农村都这样。因此不存在粪便堆积的情况。所有粪便每天收集起来，拿去沤肥，用发酵法高温灭菌、消灭寄生虫，然后灌溉到田地，形成了自然的循环。所以耕作技术是一种可持续的、永续耕作的状态。

这本书当时翻译的人是为了宣传永续农业，翻译者是温铁军教授的学生石嫣，她的小团队翻译这本书就是为了推广永续农业。但在这个笔记中，我们依然能看到当时中国的形象——中国人到底怎么经营：条件已经这么差的情况下，依然很震撼、很精细地把田地像绣花一样种出来。这是一种能力。
