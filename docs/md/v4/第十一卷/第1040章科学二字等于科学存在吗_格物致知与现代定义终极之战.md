# 第1040章 科学二字等于科学存在吗_格物致知与现代定义终极之战

“科学”二字等同于科学存在？格物致知 VS 现代定义：终极之战。  
什么玩意儿？一五几几年有什么现代科学吗？没有。一五几几年有古代科学。如果你认为科学必须要这两个字，并用之后才出现的话，那么确实德先生、赛先生来了才有科学。在那之前它叫做格物。  
嗯，我们研究天文就是研究天文，研究几何就是研究几何，研究数学就是数学，研究建筑就是研究建筑，没有人说这个玩意儿叫科学。研究医学的就研究医学，也没有人把这东西叫科学。但是呢，研究建筑的人不科学，他能把建筑搞起来；研究数学的人不科学，他能把数学搞起来；研究天文的人不科学，他能够画出星图来。就这个标准啊，你看你对于这个，你对于科学是怎么定义的。一定要说科学二字，德先生、赛先生来了才有。啊，在那之前是没有科学这两个字的。
