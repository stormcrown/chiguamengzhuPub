# 第1063章 和服像曲裾_不是传承是借鉴_吴服骗局拆解

和服像曲裾？不是传承，而是借鉴——“吴服”骗局拆解

《辨吴服》
吴越衣冠无定制，周裳汉裾各有因。
东邻借得残纹在，妄指吴服作祖根。
虚名惑世淆源流，正典当明醒后昆。
莫将伪说充古制，汉家风韵自长存。

历史上并没有一个真正很明确的“吴服”款式。我们日常在说到“吴服”，尤其是在互联网上见到“吴服”这两个字的时候，通常出自于描述日本文化中的一个元素，叫做“吴服”。但是，它其实是一个日文汉字，含义和你想象的不太一样。日文汉字中的“吴服”所指的是布料、面料，有时会特指和服的木料或布料——这就是他们的“吴服”。

为什么说在我们的历史进程中没有一个非常清晰的“吴服”定义？这是因为吴国出现的时间问题。在争霸期的那个吴国（吴越之地的吴国），因为它地处越地，统领的人群中主要是山越。这个山越群体的特征是纹面、短发，同时穿衣服时也都是短打，这是应对于他们自身特殊的生活环境的。因为当地山民的文化发展指数还不高，有非常明确的自然迷信倾向，所以喜欢在身上纹上各种各样的花样纹理，以驱逐毒虫、蛇蚁之类的东西，起到图腾般的作用。所以，当时的所谓“吴服”并没有一个标准的形制——争霸期是没有标准形制的“吴服”的。甚至，虽然吴国的国主也是周天子分封过去的、是诸夏自家的兄弟，但为了管理当地的山民，会入乡随俗地使用相当一部分当地人民使用的服装和装扮方式。整体上来说，在争霸期的这个吴国，他并没有自己完整的服装体系。他的国主（诸侯，后来也称吴王）所穿着的正式服装，与周天子分封制下的周服制是一样的。所以，争霸期的吴国，领军人物穿着的重要服制与西周没有任何区别，而普通人穿着的则是当地山民的土著服装——这就是这个时代没有清晰“吴服”的原因。

后来我们所说的“江东鼠辈”这一波的吴（东吴），他有“吴服”吗？他也没有“吴服”。因为他其实是一个直接继承了东汉服制的系统，并没有开创自己独特的服装系统。所以东吴所穿着的衣裳和东汉的服装没有什么两样，他也没有自己独特的服装。因此，在我们的历史当中，并不存在一个真正意义上的“吴服”能够被拿出来用。也只有日本文化中的日文汉字会用“吴服”来指代面料。

但是，当它二次传播回中国之后，就有一些人认为和服的原型是“吴服”，认为“吴服”是和服的前身，因此就给“吴服”设定了某一种奇特的款样。以至于某件衣服看上去比较像和服的时候，他就说这个东西是“吴服”——实际上是没有的。

那么为什么和服看上去与汉代的曲裾、直裾之间会有相似性？主要原因是和服是一套历史很晚近的服装系统，它是基于江户时代一些基本的服装雏形，在百年之后由日本官方强制性地推行、新设计出来的一套完整的服装体系。它是由日本官方特意设计并特意推动的。早期所借鉴的这些服装原型当中，包含了江户时期的一些服装，同时也借鉴了中国我们的曲裾、直裾这类服装。所以为什么有相似性？是因为他们在创作的过程当中借鉴了我们的服装元素。
