# 第1072章 我们模仿苏联_多民族划分背后的政治逻辑

我们模仿苏联？多民族划分背后的政治逻辑 ——#吃瓜蒙主 《族辩》

夫鸿蒙初判，族类始分。日韩标“单一”之帜，实藏千绪；华夏举“多元”之旌，暗隐孤桐。昔者扶桑强定律法，琉球吞声而泣血；韩半岛固守藩篱，阿伊努掩面而伤魂。岂知蓼虫忘辛，鸱鸮嗜鼠，竟以律令为绳墨，视认同作刍狗。

观我中土，汉裔曾占九八之数，本可独标青史。然效北熊分族之策，拆玉镜为碎琼，调朱墨以改史。遂使昆仑竹尽，难书纷杂之谱；沧海珠枯，怎载重叠之篇？

嗯，日韩是单一民族？我们是多民族？优点和缺点是什么？

实际上，日韩都不是单一民族。第一，日韩并不是单一民族；他们是通过法律规定，直接说自己是单一民族。但实际上，他们内部是有完全不同的认同支系的：血缘也不同，文化也不同，认同感也不同啊。日本（本州）岛上面是有阿伊努人的，然后包括琉球的人，也跟他们没有任何的共同认同感。彼此之间承认的组员是不一样的，文化特征是不一样的，认同是不一样的。好吧。

所以呢，他们之所以做这件事情，是因为知道——是因为明确的知道你强调这个问题会出事儿，所以强行的规定了，强行的进行了这种所谓的单一民族的规定。这是他们的一个专门的操作。

而我们国内啊，有一说一。就按我们当年建国的时候，汉民族的占比——我们第一次进行统计的时候，汉民族的占比是超过98%的（第一次实际统计占比超过了98%）。后来呢，为了表现我们是多民族的，把这个比例又调低了一点，进行了一下拆分，然后又调低了一点——好像是90%几来着，96还是95？就调了这么一个数字。实际上最初的时候，我们进行第一版的普查，汉民族的占比超过了98%，你知道吗？

这样的比例放到全世界任何一个地方，这都叫做单一民族。就是按你在全世界范围内，如果是用同一个标准来进行衡量的话，那么新中国它就是一个单一民族的国家啊。我们是根据当年苏联（大毛）的这套理论而强行划分的多民族。

其他的那些国家，你以为它是单一民族的国家，实际上它的民族构成也是多民族的，而且他们的主体的占比没有我们高。他们只是认为这样子划分会出问题，认为这样子划分不好，所以强行的规定了不许划分，所以才会形成你脑子里所谓的单一民族啊。
