# 第1032章 汉服复兴之路的艰难险阻与内外推动力量

汉服复兴之路的艰难险阻与内外推动力量。

汉服复兴历经20年艰苦努力，期间遭遇诸多阻力和泼脏水，但最终挣扎出来。

汉服复兴到现在，也就是20年的光阴啊。这20年何其艰苦！当时其实是在一种极其极其艰难的情况下，开始第一次萌芽的。要知道，那个时候已经试图拿满装代替唐装，来推行国服了，整个中华服饰体系就要彻底满化了。这个时候，汉服像石头里面的一颗种子一样，一点儿一点儿地伸向阳光。这个过程当中有无数的人泼脏水，无数的人用剪刀、用火烧，但是最后挣扎出来了。多困难！

那在这个困难的艰难的路程当中，谁起到的力量的作用最大呢？嗯，为什么？为什么交趾也要偷汉服是吧？你不要瞧不起人家，人家曾经是我们的一个郡啊，曾经以郡县的方式接受直辖管理的地方，他自身曾经长期自称小中华。但凡说中华系统的文明真的崩了啊，真的消散了，真的遭人奴役之后，忘记了祖先的方向（注：原文为“放心”，应为“方向”），他会和思密达一起来争夺中华正统权的。因为他们是真的学生，甚至是亲儿子、干儿子，这样的关系非常的紧密，学习的内容很丰富、很全面。所以呢，他就会做这些事情。你说他是去偷啊，那是偷吗？不是。可以说，他是学习，因为他们的祖上曾经穿过和我们一样的衣裳。虽然中间的几经变换啊，也经过了很多的本土化，但是他的祖先确实曾经和我们穿一样的衣裳。

嗯，啥？但是他不承认汉服是中国的，那是因为我们现在还没有拿回宗主国的话语权啊。我们拿回宗主国话语权之后，他就不敢这样狡辩（注：原文为“犟嘴”，应为“狡辩”）了。现在为什么我们周边的这些，包括思密达、包括交趾、安南这些地方啊，都敢一边穿着汉服，一边说这个玩意儿就是他本土的，跟中国没关系？他为什么敢喊这个号？是因为中国现在自己内部对于汉服的系统是含糊其辞的，是因为我们这里还有很多人认为清代的服饰是汉服。当我们这里认为清代的服饰是汉服的时候，人家拿明宋的东西去说是他的正统，和我们不一样。怎么着？你还能狡辩吗？你自己认的呀！你自己说旗袍是你的，那人家宋明和你的这个旗袍没明显的啊，不是一个审美系统，不是一套服装逻辑，剪裁的结构版式完全不同。你说了你那个玩意儿是你的正统，那还不兴别人用别人的吗？而且人家祖上真的穿过，并不是毫无实据的啊。人家祖上是接受过赐福的，是真正穿过这些衣裳的。虽然穿的人不多啊，主要的原因是穿不起，但是呢，别人是有渊源的，不是毫无瓜葛的。咱们首先是拿回自己的话语权，我们自己能不能把自己的东西说清楚？在今天还在含糊其辞，说不清楚哪些东西是自己的，不敢硬骨头的站出去说这个东西就是我的那个东西，我嫌弃我不要你。这个话说不出去，那旁边的人就会继续偷。

而且我也不在乎他偷真的。你不要那么介意他们偷啊。你要知道，汉服复兴到现在，也就是20年的光阴啊。这20年何其艰苦！当时其实是在一种极其极其艰难的情况下，开始第一次萌芽的。要知道，那个时候已经试图拿满装代替唐装，来推行国服了，整个中华服饰体系就要彻底满化了。这个时候，汉服像石头里面的一颗种子一样，一点儿一点儿地伸向阳光。这个过程当中有无数的人泼脏水，无数的人用剪刀、用火烧，但是最后挣扎出来了。多困难！那在这个困难的艰难的路程当中，谁起到的力量的作用最大呢？最核心的当然是我们自己的汉服运动的这些人啊，牺牲奉献的也是最大的。但是外来的力量当中，对我们的支持力量最大的，就是旁边这些偷我们祖传服饰的这些邻居啊！不是他们天天偷了之后，说这些华美的服饰就是他的，跟我们没关系，咱这个上面能有那些人松嘴吗？是不是？你看当初，当初我们很多的那个传统节日不放假啊，不是所有人都认为传统节日不应该放假，但是你要定传统节日为假期的时候有阻力。有一些人就打着各种各样的什么革命的幌子啊，就反对你把这些什么糟粕拿出来哦。当时不是所有人都认为传统节日不能放假的，有一些人是认为传统节日应该优先放假的，但是为什么推进不了？就是因为有阻碍，有一些人呢就专门阻碍我们传统文化的复兴的。但是谁推动了这些呢？虽然内部的人最重要啊，这是我们发自内心的，就是有内部有很多的人是真的早就希望我们的传统节日尽早放假了，这是核心的力量，这是内部的力量。但是外部也有推动力啊！要不是当时思密达大面积的去搞那些申遗，说这个节日是我的思密达，那个节日是我的思密达，那个节日还是我的思密达，这个仪式是我的思密达，那个仪式也是我的思密达，要不是他们这样子满世界的跳脚，咱的传统节日都上不了台面，你知道吗？这个外来的助力要感激他啊！你不管他是什么样的动机，也不管他做的事情恶不恶心人，反正造成的结果，最终的结果是好的。就是这些去跳反的人，在那里泼脏水的那些人，撒泼打滚的那些人，一哭二闹三上吊的那些人，客观上让中国人产生了自己的族群认知，让中国人意识到自己的传统是有价值的。如果他真的是像某些人泼脏水说的那样子毫无价值，早就断代的话，那为什么我们的邻居左一件右一件的都来拿，都来抢都来要呢？是不是？唉，就是幸亏有贼惦记着，要不然的话我们这边都要把他当成洗脚水泼出去了。

嗯嗯。对我，我反正是一直觉得就是那些什么，呃，提出各种各样的反对意见的啊，然后包括那些很刻意的造谣污蔑的人，他在客观上面带来了好的结果，就是让人发现了他的问题所在。呃，这几十年里头，有很多的事情都是这样发生的。内因最重要，但是内因有的时候需要一些出发点，而这些触发点常常是外部的，这些反对的力量，甚至是反动的力量。因为他反动，因为他恶劣，因为他丑陋，因为他卑下，所以呢，他能够带来这种刺激性的力量。
