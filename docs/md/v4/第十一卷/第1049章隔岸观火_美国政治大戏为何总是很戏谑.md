# 第1049章 隔岸观火_美国政治大戏为何总是很戏谑

隔岸观火：美国政治大戏为何总是很戏谑？

嗯，为什么我们总是以戏谑的态度旁观美国的政治景观？政治必然是严肃的吗？那个，指出啊——我们现在看到很多媒体上面的内容，关于美国那边的大乱斗时，都是很诙谐的，动不动就是嘲讽的、影射的。主要的原因是我们现在真的站得很扎实，这才是根本上的原因，别的都不重要。因为现在他那边闹出的那个浪，打不着我们了。我们的护盾啊，铸得很结实，所以他那边在扑腾扑腾闹出来的浪花呢，我们就可以觉得这个浪花哎，挺好看。就是这个原因啊。

你一定要说他那边，我们是不是应该严肃点对待？可以严肃对待，但是呢，没什么必要。因为哪怕是这两个人在之前啊，说哎，要角逐大位啊什么什么的，重要吗？不重要。因为我们已经在之前的这几年里面做好了各种各样的预备，我们知道谁上来都不可能对我们好的，只是拳头用哪边出、用什么手法的问题。它影响到他目前的这些大乱斗的行为，影响到的只是他本土对其他地方的影响，已经非常有限了啊。所以我们才能够这样子，像是在隔岸观火一样地凑热闹——就是因为它确实也就剩下一个热闹了。

但是你要是说我个人的观感是什么？真的，我还是比较喜欢川皇的。不为了别的原因，就因为他是个人。他，他不是那种群魔乱舞里面的妖魔，你知道吗？他，你不管他对中国好不好——他不可能对中国好啊，他对中国好了，那我会觉得有点什么毛病。他不可能对中国好。所以这件事情我就是在看待：他就是一个人，他在说正常的人类的语言，他在为自己的国家着想，他在为自己的国民着想。就这么一件事，我就希望能上来是个正常人。我在看他的时候，能够不感觉到脑子里面在抽筋就行了。你看我已经不做其他的指望了是吧？那你要是弄了一个什么哈哈哈哈，上来那啥呀？我每天就得听他哈哈哈哈哈，我也不知道发生了啥，反正他就先哈一顿，先哈为敬。那你跟他聊得下去吗？不为了别的啊，主要的就是这个事儿。
