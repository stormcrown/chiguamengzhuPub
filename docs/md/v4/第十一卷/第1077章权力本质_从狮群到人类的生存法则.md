# 第1077章 权力本质_从狮群到人类的生存法则

**权力本质：从狮群到人类的生存法则 ——《世权鉴·上古箴》**

昔者鸿蒙初判，牝牡未分。母系遗踪，徒存推演之迹；父系肇基，始立文明之纲。观狮群之治，雌狩雄卫，岂因性别而定尊卑？察边陲之民，女耕男戍，何以劳逸论功过？夫草原坏男孩逞凶，几绝狮种；南陲柴刀客待旦，常悬死生。故曰：权非天授，力强者掌乾坤；位非虚设，能者守存亡。今人妄议上古，饰母系为桃源，殊不知：石斧开天时，弱肉终为强食；血火争地时，白骨方筑王庭。

—— 

这两天看人吵嘴，就是讲母系、母系。所以女性又如何如何，从女性视角解读母系什么什么的，我觉得十足没有必要。为什么我觉得这个争论十足没有必要？因为实际上，在我们现代对于这一时期的认知——这已经不是古典时期，这是上古时期——对上古时期社会关系演变的研究，都包含了大量的逻辑推论。为什么？因为母系时代就没有成文史留下来。人类进入我们能够理解的真正文明时期的那一刻起，就已经是父系社会了。我们从父系社会开始走入了真正的文明期。母系社会存在吗？存在，客观存在。但那个时期我们还没有真正进入大家能直接理解的文明期。只能说在早期父系时代中，留有一部分母系时代的残余，因为这两个时期不是突然切换的，而是逐渐转变的。但我们走入文明时，就已经是父系时代了。因此，对母系时代的一切研读，都包含了推论与推理。

那么哪一种推理是更富有合理性的？一个世界、一个社会、一个群体，永远都是强权主义的。最强的那一个站在最高处指点江山，什么时候都这样，现代社会也这样。只是现代社会，我们更加看重智力和情绪的稳定性。所以现代社会的强者，是智力上的顶尖和情绪上的顶尖稳定。但在上古时期，相对原始蛮荒的时期，什么才是最重要的核心生产力？体能。所以认为母系社会女性或母亲的权益一定最大的这个猜想不合理。有没有一部分女性拥有极好的权利？有。她倚仗于什么？倚仗于她的父亲、兄弟、孩子。你要拥有某一种力量，才会获得对应的话语权。

那么这种事情在自然界当中有没有表现？要知道，人类的上古时期与自然界之间其实就是一个转身的距离。自然界当中有没有这种社会结构？有的，典型的母系社会——狮群。狮群是母系社会，铁打的母狮家庭和流水的雄狮。但这个狮群当中谁做主呢？雄狮。母狮是日常捕猎的，你看它又要干活，又要生孩子，又要养孩子，贡献老大，但为什么雄狮做主？因为在自然的生活当中，一个群落最重要、最核心的价值其实来自于防卫力，就是安全底线。雄狮保障的是狮群的生存底线、生命底线。至于捕猎，雌狮子也能捕猎，它捕猎的能力弱于雄狮，但它也能捕。同样，在人类社会当中，女性一样能干活儿，但体能上弱于男性，这种表现是高度相似的。

那么流水的雄狮，是通过母狮去招揽的吗？不是。没有哪个雄狮是由母狮选聘进来的。雄狮是怎么来的？是不同的雄狮进行了决斗之后，找出了最雄壮的那一个，然后自己竞争上岗的。这个雄狮日常的工作项目是什么？就在那里睡觉。他睡觉的这个行为，其实是在防卫、守卫、放哨。所以一旦遇到鬣狗群的时候，就是雄狮出征的时候；遇到大型动物需要捕猎的时候，就是雄狮必须上手的时候。他防卫的是这个群落的安全底线。

有人说“母狮捕猎能力更强”？哼，你最好去问一下母狮。非洲草原上曾经发生过这样的事：有一伙雄狮不愿意照顾母狮了，五个还是六个一起，组成了“坏男孩”雄狮联盟、雄狮团队，一个母狮子都不要。你知道他们强大到什么程度吗？大杀四方，任何狮群都争不过他们，没有哪一个狮群可以与他们抗衡，差不多就要造成狮群大灭绝了，把整个非洲草原杀穿了。最后，这个坏男孩狮群是被保护组织消灭的，因为要是不把这几只雄狮干掉，非洲的这些狮群都要消亡了。你说母狮的捕猎能力更强，只是因为日常情况下，面临中小型猎物时是母狮出手，因为雄狮要在旁边待着，等待什么？等待战斗，维护安全。他是一个维护安全底线的角色。所以即使这个狮群里面的主要劳力是雌狮，而且是母系的社会，但狮群的走向、狮群的核心是那个雄狮。都是这样的。

包括实际上到现在，这种社会结构在一些相对湿热的地区依然存在。大家有没有听说过？在我们靠南边的一部分地区——当然中国人勤快一些——更南边的几个邻国里面，这种情况更典型一些。中国国内也有，就是靠在南方，尤其是西南角的这个部分。有些地方女子承担了大部分的劳动，农田里的活儿也干，家里的活儿也干；男人就是甩手掌柜，啥也不干，敞开肚皮晒太阳。这样的地方到现在都还存在着。你说贡献是不是贼大？但是这种地方都有一个特点：部落群体很小，防卫的边界线非常非常靠近家庭。所以这些男人为什么在家里面晒肚皮？因为他随时要积累能量，随时随地准备好，遇见了争端的时候，他要拎着柴刀冲出去搏命。这些地方的群体都是这个样子的。

你觉得为什么这样奇特的生态会存在？为什么在这些地区，很多女人在那里积极地干活，看上去就是克勤克俭，什么事都是她干的，养孩子也她干，背着孩子下地；而家里的男人就在那里闲着没事干，抽烟袋？好多这样的地方。为什么这样？他们的传统为什么会积累下这种习惯？因为他们有巨大的防卫风险的需求。而男性在家里面待在这个岗位，就是随时随地做好准备要去响应战斗。
