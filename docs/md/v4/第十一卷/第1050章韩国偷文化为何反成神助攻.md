# 第1050章 韩国偷文化为何反成神助攻

韩国偷文化为何反成神助攻？中山国、韩国、匈奴集体尚白？  
我国古代把老虎称为“山君”，就是把老虎视为山神的意思。虎军神话思密达。思密达什么玩意儿都是思密达的。你不要太在乎那个东西，不用太在乎。思密达什么玩意儿都是他的——见过的、听过的、尝过的，隔壁有的全部都是他的。不要太过于介意。我就不是很介意他们这些年偷我们的东西，偷多了。偷就偷呗，偷着最后他自己的特点一点都没有了，就被埋起来了，是吧？  

因为实际上过去很多年里头，他的大量的文化或者说习俗，确实是向我们这边学习的。哪怕是照猫画虎画得四不像，他也确实是向我们学习的。所以他在跟我们学习的过程当中，不断地就发生了改变，最后变得和我们越来越像。就像等到哪一天，我们再一次地拿回我们文化宗主国的地位的时候，他从我们这里拿走的东西就会物归原主。谁的就是谁的。思密达那点事情我觉得完全不用介意。甚至在有一些事情上面，思密达的这些不要脸的行为，对我们是大有助益的。他给我们提供了多少的帮助啊，你想一想。如果不是他在那里抢注非遗，咱中秋节能放假吗？你中秋节休的假，那都是思密达送给你的。  

包括这些年，我们从2003年、2002年左右开始啊，汉服复原运动。你想这个活动一开始的时候，受到了多么大规模的打压。从上至下，从有文化的到没文化的，各行各业，从中到外，全部都在打压。一会儿说你封建复辟，一会儿说你封建迷信，一会儿说你保守落后，一会儿说你盲目排外，什么东西都有。一会儿说你那个什么“妹上了”，一会儿说你又“may下了”，反正你就没有好词儿。啥时候他开始能够上得了台面、算一盘菜了呢？思密达说这玩意儿是他的。没有思密达，我们的道路还会更曲折。不要太过于在意这个玩意儿。它友军——我，我从来都认为思密达是友军。我不是很介意他们整这些活儿，哈哈哈。  

你像他跟我们学的东西其实是很多的。例如说什么逢年过节的时候给人塞红包这个行为，他是跟我们学的，但是他塞的是白的。为啥是白的？他们以前也跟我们学衣裳，为什么他们却是白衣民族？因为没染料。他那片土地条件比较差，气候也不是很好，山区的比较多，平原比较少。整体上它的物产不丰，造成的一个结果就是相对的贫穷。这种情况就使得所有的东西传过去之后，很容易就变成简配版。所以我们这儿的红包、对联什么的，一传过去之后都变成白的了。他说叫做尚白，以白为尊。谢谢堂皇大道。对，好，谢谢堂皇大道。他自称的叫做以白为尊，但是以白为尊的这个尚白的行为，不光是他一家。当年我们周边有很多向我们学习文化的、这些经济不太好的地区，都尚白，是这些群体很普遍的一个特征。所以根本的原因应该就是因为染料用不起。织染这项工艺其实是比较复杂的，你能够把所有东西都染得五颜六色，这件事情并不是那么容易，没有像大家今天想象的那么容易——我想要什么颜色就什么颜色。干净的反而还好一些。实际上在古时候，你能够把一个东西染得五颜六色、非常明艳是很困难的事儿。  

所以我们当时周边，例如说什么中山国不是子姓，中山狼的中山国，当年他们也是说他们那个叫白狄，为啥叫白狄？因为他们尚白。他们不是白人——有些人编排说他们是白种人，不是的，他们和我们同种但是不同族。这些人他们在和春秋时期中原的诸侯互相往来、深入学习之后，也算是一种学习华夏礼仪。但是我们不承认他是诸夏，因为诸夏的前提是你得是咱家亲戚，得是长期通婚之后、确实有血缘关系了才行，你光是学文化咱不认得。所以他当时就跟我们学习文化，很多东西都在向我们学习。但是这个中山国，他当时的特征就叫做尚白。不管你学什么，反正到他那儿就变成了简配版。尚白其实也是因为条件不好，他没有机会去进行这些织染。像包括后来跟汉朝一直在对峙的匈奴，匈奴也喜欢白色，以白色为尊，什么东西都喜欢弄白色的，有的皮毛啊也喜欢白色的。那以白为尊的这个行为，大概率都是因为条件不够好。所以后来什么“失我胭脂山，使我妇人无颜色”——他们也想染，也想让东西变得更明艳一点，但是大部分情况下做不到。所以这些群体就会出现尚白。思密达也是这样的，他们自古的尚白。包括狮子——狮子绘画在我们这里是很明艳的一套东西，传过去之后扁平了就算了，它还变成白的了。啥玩意儿？到他那都变成简配版、扁平版、白色版，就是掉色版。这个其实是他们以前的这种条件不好造成的。  

那现在条件也起来了，是吧？人家也算是一个发达国家了，那自然他想追上来就让人追上来。你瞧他们现在把他们自己的传统服饰是不是也改造得非常的明艳？他拍的古装剧是不是也是五颜六色的，颜色璀璨得不得了？反而是我们这边的一些什么古偶仙侠，还喜欢摆拍那些染不起布的那个样。其实挺颠倒的。因为你凡是说你的经济条件好，你的生产力水平高，那当然就是有什么好的用什么好的。那古时候他缺染料，染料就是贵，所以古人他在颜色上面会有很多的讲究，越难染的颜色它的等级就越高，甚至在有些时候它禁止你使用相应的颜色，就是因为那个染料忒贵了，你知道吗？他就是通过这种方式拉开阶级的层次的。
