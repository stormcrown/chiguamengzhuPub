# 第1085章 破除中产迷思_无产阶级与生产资料的当代辨析

破除中产迷思：无产阶级与生产资料的当代辨析  
—— 「水调歌头·阶级辨 」  

金谷宴初散，玉楼笙歌休。  
谁言朱门绣户，便是百年谋。  
不见中产浮云，转瞬风卷残楼，错认稻粱谋。  
薪火非恒产，镜花水月舟。  

辨阶级，明生产，破迷眸。  
寒梅傲雪，从来清气满神州。  
莫道脱贫改质，当思本源守拙，青史鉴沉浮。  
千载兴衰事，尽在掌中筹。  

我们全面脱贫以后还有无产吗？  
啊，无产指的不是穷人啊。  
我们要搞清楚，我们要搞清楚：无产不等于是穷人。  
当大家脑子里面把无产阶级与穷人直接绑定的时候，你可能就会陷入到一个困境之中去——觉得自己现在薪水涨高了，然后就觉得自己是有产者了，然后呢，就陷入到了一种精神内耗之中去。这就是现在的这个中产困局啊。  

记得前阵子经济条件呢，有一点持平，有很多的行业呢，出现了裁员。那个时候发现了没有？就是那个时期有一批中产破产潮。为什么无产者啊——明确自己是无产者的人没出事儿，有产者也没出事儿，就是这些中产者出事了呢？是因为他们错误的定位了自己的身份啊。  

什么是无产，什么是有产？这个无产与有产的“产”指的不是 money，不是薪水，它指的是生产资料。意思是你有没有可以掌控在自己手里面的生产资料？有没有哪一条销售线路是完全属于你的？有没有哪一种生产工具是完全属于你的？有没有哪一套生产的系统是完全属于你的？这个东西它叫做生产资料。有了生产资料，你才叫做有产者啊。  

有产者当中呢，当然也有小有产者，例如有一部分的个体户，他拥有自己的生产资料，但是呢，抗风险能力不强。这些呢，是小有产者。还有大的资产、资产阶级、还有大的资本家。那么这些呢，就是掌握了特定的生产资料的人。他们即使是生病了，即使是休息了，他依然有收入直接滚入到他的账户当中来，因为这是由生产资料而带来的收入。  

但是呢，有一部分的高薪打工仔，就误以为自己的薪资是自己的产业，于是他就产生了一个问题：他错误的定位了自己的身家，从而在消费上面陷入了一种困局。所谓的中产破产三件套——他真的破产了吗？其实没有。他只是从原来的年薪百万，变成了一个年薪十几万而已。在普通人那里的话，他的这个收入并不算差。但是他怎么就破产了呢？是因为他之前按照年薪百万的方式进行消费，以至于到了年薪十万的时候，他过不下去了。他错误地衡量了自己的生产资料，也错误地衡量了自己的社会阶级，于是就造成了后续的很多错位。  

那个电影《西虹市首富》大家看过是吧？主角王多鱼。为什么王多鱼他想要把钱花空的这个愿望没有能够成真？因为他的钞票到了一定量级的时候，这个资产累积到一定量级的时候，他变成了一个资本家。于是无论他做或者不做什么，无论他做的是正确的事情或者错误的事情，只要你不是天翻地覆似的把自己给卖没有了，那么你的资产就会不断的增值。所以他就会出现越花越有的局面。这种情况下，那个中产破产三件套的东西对他来说是无所谓的——消费是收入的一部分。  

但是对于这些错误认知自己的所谓的中产阶层而言的话，他把薪水定义为资产，他把房产定义为资产，最后就出现了问题啊。  

所以那个无产与是否脱贫没有任何的关系。无产阶级不代表他已经进入到了贫困。而我们的这个脱贫攻坚是干什么？是让我们的无产者拥有正常生活、幸福生活的基本权利。这是一个社会再分配的表现啊。不代表说他脱离了贫困，于是呢，他就不再是无产阶级了——不存在这个概念啊。
