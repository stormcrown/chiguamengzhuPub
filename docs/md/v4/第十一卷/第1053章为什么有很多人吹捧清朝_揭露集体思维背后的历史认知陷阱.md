# 第1053章 为什么有很多人吹捧清朝_揭露集体思维背后的历史认知陷阱

为什么有很多人吹捧清朝？揭露集体思维背后的历史认知陷阱！#文化复兴 #文化自信

《满江红·警世书》
暮霭沉沉，凭栏处、残阳泣血。
翻青史、粉脂污卷，蠹鱼啮阙。
三百载衣冠易犬马，五千秋礼乐成灰屑。
最堪悲、粪土饰金銮，歌无绝。
精神格，犹喧聒；迷途客，争趋谒。
叹汉唐英魄，竟蒙尘亵。
粪坑妄称尧舜世，蛆虫敢拟麒麟骨。
待从头、涤净腐腥风，朝天阙！

作为一个群体生物时，我们进行集体思维，常常会陷入某种误区——这是一个非常常见的现象。其中确实有一些人是无辜的，但刚才他用了“吹清”一词，我觉得略微有偏差。因为那些无辜的人基本上不会去“吹清”，而是“认可清”，这两者之间有点差异。

那些精神阿哥、精神格格们会去“吹清”，即为满清创造一些它并不拥有的优点来赞美；而另一群人之所以成为这个势力中的一份子，是因为他们觉得“哎呀，有没有那么好？算了吧，反正你也不能说它那么差”。这是一种随同认可的表现。这有几种原因：

第一，对历史的认知比较欠缺。这挺正常，不赖任何个体。甚至有人读了很多书，但历史认知有偏差——这个责任不在于普通人，而在于满清时期本身对历史进行了大规模篡改和掩盖；同时，80年代之后的标准课本及科普读物也刻意把满清的一些黑暗与罪恶行为进行了粉饰。它不是完全不说，而是轻轻绕过去，很容易让读者忽略。

这是知识方面的问题。另一方面是自我认同的问题。自我认同很麻烦，哲学最重要的三个问题：我是谁？我从哪里来？我往哪里去？自我认同就是“我是谁”。今天在座的很多人，你们身边大量的人在自我认同上有问题。你让他们拿身份证，会看到上面有“汉”字，但你问他“汉人是什么？”，甚至有人会突然说“你是不是不团结？”这是常见现象。为什么？因为汉人作为主体民族，体量过于庞大，在日常中根本意识不到自己是汉人。当你把这个问题单独提出来时，他怎么会在乎？甚至觉得你有问题。这实际上是自我身份认同欠缺的表现——他不知道他是谁。所以在看待满清历史时，他不知道该站在哪一边，于是站到了声音大的那一边。

谁的声音大？精神格格、精神阿哥们声音最大。为什么？因为他们知道他们所说的是谎言，而谎言必须重复百遍千遍才能变成真理，所以他们声音特别大。既没有自我认知，也没有清醒理性的理论依据，就非常容易被舆论场上各色声音裹挟。如果这些声音还捆绑了一些被断章取义、看似正确实则有问题的口号和旗帜，那就更难越雷池半步，生生把人拴在了那里。

因此，在普通人群体中，常存在这种态度：“虽然觉得他干的挺糟糕，但啥时候不糟糕呢？过都过去了。”这种含糊不清的人很多。在明确的问题中，选择含糊不清，就已经站在了罪恶那一端，但他意识不到，因为他想不起来自己是谁，也记不得该站在哪里。这是一个非常常见的心态。

那什么时候能好起来一点？文化复兴会逐渐让大家转向。这两年我们国家国力越来越强，大家有没有发现？当国力越来越强，我们就需要树立心理上的自信。所有人都这样：当你发现自己所处的地方经济繁荣，你一定会为自己找一个心理依托、自信的来源。所有人的自信都来源于过去——我从哪里来？必然如此。所以文化自信变成了一种刚需。这不是中央推广的要求，而是普通百姓内心深处最需要的东西，国家只是代表我们讲出来了。

那你怎么拥有文化自信？只要你认为满清是一个不错的时代、是封建时代的巅峰，你就不可能拥有自信。我们最巅峰的时期是那么一个垃圾、那么一个粪坑，还自信什么？所以为什么这几年互联网上很多人开始提出问题？哪怕没有坚决去思考、没有坚决去演，他依然提出问题，依然感觉心口发堵——觉得哪里不对。哪里不对？不对之处就在于：你只要赞美满清，就意味着否定你自己；你只要赞美满清，你就无法拥有任何程度的自信。如果满清王朝是封建统治的巅峰，那我们的巅峰就是一坨屎，汉唐盛世就只能是连屎都不如的东西。都这样了，你还能有什么自信？你相信自己从前都是蛆，怎么可能自信？而当我们面临自己的国家已经如此繁荣昌盛时，你又说它从来都是一坨屎，这两者不兼容，会让你极度痛苦。

所以这几年，不管怎样，不管他的学识如何、站队如何、有没有想过这个问题，哪怕只是凭着心里头单纯的那一点点难受，他也终于愿意听一听像我这样的人说话了。这是历史大势，滔滔洪流不可改变。因此，这几年这些精神格格们的吹嘘变得越来越敷衍、越来越艰难，每一次吹嘘都像急赤白脸的狡辩——就是因为清它就是一个罪恶和黑暗的时代，就是一个大粪坑。客观事实摆在那里，你只要认为它是你伟大的老祖宗，你就会不断内耗、不断痛苦。这种行为、思考、烙印，会和你现实的生活完全割裂，你无法达成一个完整的自我内核。
