# 第1082章 清朝女装为什么感觉阴森？ 

感觉清朝女装的衣服花纹都挺好看的，但为什么呈现的感觉会有点阴森呢？嗯，我看见了，看见了。这个就是清代的服装发生了演变。沈从文是中国第一波开始研究古典服装史的，当然由于种种原因，他后来做纹样史的部分没有完成，只有一个出版的草稿。但前面那部分——整个服装款样的变化——他是做完了的。在他的这套研究里，其实对清代服装的演变有相当一部分论述，主要讲清可分为前期和后期，基本上以乾隆作为中间节点。乾隆之前，清的女装相对比较典雅，或者说这个时候的女装，除了襟口上面的剪裁发生了一些变化之外，其他地方，无论是版型还是配色，都跟之前的没有很本质的差异，依然是我们传统审美系统里的东西。但是到了乾隆年间之后，朝廷对整套服饰体系进行了专门规范，并且进行了强力的监管和打击。之后呢，这种叫做“厂字襟”的结构，就被强行推动了下去。

这个事就跟你的纹样有关系了。大家会注意到，所有厂字襟的衣服，都会在围绕着领口和襟口的边儿上，出现很繁复的大花边，也就是这种滚镶边。这在所有满人体系中都有，它出现在哪一个群体当中，哪一个群体就会用当时当地流行的纹样对它进行装饰。为什么呢？因为这种长尾排扣（也被人称为蜈蚣扣，因为它有点像长角），加上厂字襟的搭配放在一起，无论如何都会特别有撕裂感。你可以自己在纸上画一下这个东西：画一条接线，然后在上面画上长尾排扣的线，就会特别有割裂感、撕裂感。而且由于它是厂字襟，不对称的，整个服装剪裁结构、受力结构也不对称——它是歪的，是一个斜襟儿，彻底整个结构都是歪的，在这边缝合。使用了长尾排扣之后呢，就会有一种撕裂感，而且整个位置是偏向的。这种时候，你在视觉上看待它，会有一种不对称、不和谐的感觉。因此，无论是落到了谁手上，人们都会在这个面上进行一圈一圈的滚镶边，就是为了遮蔽这种撕裂感。这些繁复宽大的滚镶边，能够把那个长尾排扣遮蔽、掩盖在重重纹样中间，显得不那么醒目、不那么打眼了。主要是这个原因。所以不光是汉民女会这样修饰服装，其他那些经历了剃发、易服、管制的地区，所有的部落民女或少数民族女子，全部都使用了这种宽边滚镶边。在那之前的话是不需要用这种装饰的，因为之前的服装结构在这个地方，你加那个东西很多余、很拖累、很复杂。但是到了清代，因为它的领口本身的原因，就出现了这些大面积的滚镶边，变成了你不把它拼上去就会变得难看的状况。

这种情况就属于清代服装的一个特点：你去看它的细节的时候就会很好看，因为清代的手工艺水平其实是进步了的，人们的手艺、技术，尤其是手工业方面的技术，是有进步的，不是真的止步不前或者倒退，针线活的手艺越来越好了，很精细。但是这些美丽的纹样、美丽的花、精美的纹绣，放在这样的衣服上之后呢，就变得很堆叠——相当于你把各种各样的东西都串在了一起，然后都铺满了，它就变成了一个在审美上没有中心、没有核心点的状态。我们知道所有设计或审美，需要有一个核心的诉求——比如花的主题、万寿的主题，或者子孙满堂的主题，要有主题。但是在清代这个地方出了问题：他太需要装饰，不装饰就不好看，必须要装饰。于是就出现了累进式的，什么东西都堆到一起。东西堆多了、繁复了、没有重心、没有重点了，没办法好看，整个审美就偏了。

之前有位伙计，他去看了一个朋友发给他的女装复原品，是手工专门进行的复原。他看完后，作为一个直男，直接发图片过来说：“你看这衣服还是很好看的，我们老祖宗这个厉害啊，这个衣服多漂亮。”我说：“这个衣服好不好看咱们撂一边不谈，我认为这件衣服是清代的。不行的话，你去问一下你的朋友，他复原的是不是清代的衣服。”他说：“这你都能知道？”我说：“我虽然不是研究服装史的，但我一眼看去就知道它是清代的。至于原因，你可以问清楚之后我再告诉你。”他就去问了，问完后说确实是复原的清代的服装。于是我就告诉他为什么我一眼就知道那是清代的衣服——因为堆叠。他把各种各样的东西都堆在了一起，美不美呢？每一个东西都是美的，每一个手工都是精细的，每一个材料都是昂贵的，但是他把所有的东西都放在了一起，最后就变成了那个样子。你为什么觉得清代的衣裳有点奇怪？怪就怪在这了。
