# 第1007章 什么是精神包衣？ 

嗯，现在哪有什么包衣？只不过是客观理性的人被扣上帽子罢了。现在确实没有包衣，但有“精神包衣”——就是那种幻想着要“抬旗”，却找不到王爷家的门槛，满地瞎磕头的那种。这就是现在的精神包衣奴婢。编制上的包衣肯定不存在了，社会主义国家怎么会有包衣奴婢呢？嗯，但精神包衣咱拦不住啊，毕竟人家颅内高潮这个事我管不了。他心里头就相信那个什么满清十二帝，个个是明君。那你说这种算什么啊？就就就万岁呗，啊，万岁万岁万万岁。你爱上谁家去磕，上谁家去磕，屁股呆在草垛子上，脸呢，就已经贴到王座前边的板凳上了。我也是纳了闷了，你是怎么贴的啊？

随意随意，我无所谓的啊。我从来都尊重他人的选择自由。这个选择自由呢，就是你可以选择生而为人的自由，也可以选择生而为奴婢的自由。但是别人笑话奴婢的时候，你不要委屈你自己。愿意当奴婢的，又不是谁逼你的。哎，我们社会主义国家不主张任何人当奴才啊，我们认为我们所有的人都是有主心骨的人，都是坚强的，都是自立自强的人。所以呢，每一个人都可以在回看历史的时候，去进行系统性的批判以及针对性的批判。对的就是对的，错的就是错的。因为我们已经走出了那个黑暗的时刻，可以客观地去看待它了。但是有一些人认为，你客观地看待它就是泼了脏水，你必须三叩首、九叩首，长身伏地，高呼万岁，圣天子驾临，才叫做客观公正。那我只能说奴婢心入了脑，没有办法，我也不想救。我对这件事情毫无意见啊，毫无意见。你想去哪磕就去哪磕，你觉得哪位圣天子特别光荣，你就去研究圣天子，然后自己发视频说：大清圣天子何其的伟岸，如果没有他的话，哎，今天就没有我呀。我的天老爷，诶，都可以啊，随便，悉听尊便。但是呢，不要跑到我这儿来跟我说啊，你不客观，你不承认人家的伟大贡献。对，我不承认他的伟大贡献，因为我认为他拖的那些伟大的后腿太狠了。嗯，明朝还有后腿啊？啊，那还是有的啊。为啥会有呢？就明朝那么烂的皇帝都已经吊死自己了，结果呢，就只是举了一下明朝的旗帜，大半个江山上的人就直接聚过去了。连推翻明朝的那些所有的流寇也好，革命军也好，全部都站起来，站到了南明小朝廷的身边。虽然我们明知道啊，南明朝廷是个烂人，是一帮子混账混在了一起，但就只靠了大明王朝的那么一杆子破旗，风雨飘摇，都能够聚起半壁江山的力量。哼哼哼，你还说他没后腿？他后腿粗得很。什么后腿呢？哎，就是普通人的后腿。要是没有对比的话，大家可能也就算了，一有对比就发现，算了，还是明王朝有腿。
