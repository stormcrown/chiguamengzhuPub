# 第1010章 伪宋说的背后_藏着什么不可告人的目的

“伪宋”这个说法背后，到底藏着什么不可告人的目的？  
什么玩意儿啊？就“伪宋”——人家是正经的中原王朝，凭什么叫“伪宋”？  

在正统观念系统里，一直到30年前，都不存在其他王朝的正统属性。在那个时代，唯一的中原正统就是宋，无论是北宋还是南宋，哪怕是南宋偏安一隅的小朝廷，一边被人喷一边说它是正统，这都是天经地义的。  

也就是从几十年前开始，有一群人跳出来嚷嚷，说起来了：不止是宋，其他的也都是正统，人人皆是正统，所有来中国的都是中国人——就是这一套呗。这一套历史观总共蹦跶了也就这几十年。在那之前，哪怕是满清那样的时代，他们都会把宋列为唯一的正统。满清时期，连乾隆皇帝那种政治机器人，他都说宋是唯一的正统。那些编造出来的野史啊，现场编的那些历史观啊……  

现在更邪门了，来了一个“伪宋”。什么玩意儿？南宋没出息、软骨头，这没错啊，该喷的不是正常喷？但它是中原王朝、中原帝国，有什么问题吗？没有问题。
