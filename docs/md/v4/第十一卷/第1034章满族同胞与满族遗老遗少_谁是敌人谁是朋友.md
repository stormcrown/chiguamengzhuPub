# 第1034章 满族同胞与满族遗老遗少_谁是敌人谁是朋友

满族同胞？满族遗老遗少？谁是我们真正的朋友，谁又是我们必须划清的敌人？  
就说今天，就像昨天动不动冒出来的那些黑子，一上来就说：“你侮辱了满族同胞！”  

我说，满族同胞不也应该和我一起去批判满清对中华世界的奴役与蹂躏吗？满清给这个国家带来了巨大的创伤，对绝大多数人造成了根本性的伤害，这是客观存在的罪孽。满族同胞也应该站起来一起批判，这不是天经地义的事吗？它带来了整个国家的倒退，那么满族同胞就该站在一起，客观看待历史，批判那些错误的事情，然后我们才能携手走向明天。  

什么？满清余孽？你不要闹了。站在一起共同批判满清时代所带来的巨大创伤的，才叫满族同胞。而那些指着我们鼻子说“不能说满清一句坏话，否则就不团结”的，才是满清余孽。咱们得分清楚。因为人做事要凭良心，要有良知，要有天道轮回，要有道德观。你要知道谁是谁非。满清做了那么多罪孽，你站出来一起批判，你就是手足同胞；而你站出来说“受害者不可以说自己当年受害，你必须说大清盛名，否则我就不跟你当同胞”，那你这人确实不是同胞，你这样的人，就叫满清余孽、遗老遗少。  

你，必须要把这个事情弄清楚啊。客观看待历史，该批评的地方就要批评它做错的事情。你非得要给它洗地，不然就伤害你感情了？怎么着，那就是吧，感情那么脆弱的吗？嗯？  

什么？辛亥时期的同胞是汉人互相称呼？那得看不同的时间。因为解放之后，接受了改造、愿意正式加入的满族同胞，就成了我们的手足。虽然加盟的时间有点晚，但也加入了。可是呢，他是这个时间点加入的，所以之前犯下的错，该批判的正常批判。哪怕是今天的满族同胞，也应该站在中国的立场上，去批判那些给中国带来破坏和倒退的统治者——这是天经地义的事。  

这件事情你要是做不到，然后说什么团结？团结，那我要团结什么？我必须要说“大清盛名”才叫团结？那做不到啊，臣妾做不到呀。
