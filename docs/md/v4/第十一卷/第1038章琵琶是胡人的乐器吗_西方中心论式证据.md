# 第1038章 琵琶是胡人的乐器吗_西方中心论式证据

琵琶是胡人的乐器吗？西方中心论式证据：琵琶的命名方式是拟声？

这个问题比较有意思。关于琵琶这件乐器，主要争议在于：一直以来有人说它是胡人的乐器。那么，他们的理由是什么呢？既然提出了这种说法，总得给人家一个依据——为什么要说琵琶是胡人的乐器呢？可你又找不到源头：哪个胡人弹过？其实也没有找到任何明确的源头，没有说明琵琶发源于哪里、通过谁带过来，这些东西都没有。这些证据是不存在的。

然而，虽然没有证据，却由某些官方学者信誓旦旦地公开发表，并且在传媒系统上公开接受采访，声称琵琶是外来的。可笑的证据是：琵琶的命名方式是拟声，而汉语不用拟声的方式给乐器命名。这个证据你知道吗？他的理由是拟声——因为琵琶的命名方式是拟声，所以这件乐器不是中原的。这怎么行呢？这就好比说：为什么说你不是中国人？因为你的名字叫“哈里”——“哈里”这个名字一听就不是中国人，所以你就不是中国人。琵琶这事儿就是通过这种逻辑推论出来的。

这是前期那种西方中心论之下，我们这里一切都是西来的——就是在这种基础上推出来的玩意儿。“两个字的都是外来乐器”，理由多了去了。我每次看他们编出来的理由时，都为他们感到汗颜——他是怎么有脸说出来的？要说某种东西是外来的，也不是不行，咱们又不是与外界隔绝，也不是不向他人学习。但你总得有个证据：哪个地方先发源的？别人在哪个地方演奏了什么乐曲？是通过什么方式、什么样的商路传过来的？你总得有个路径吧。什么都没有，直接说命名方式不对、名字起得不对，所以不是中国的折扇？是日本的吗？不是日本——学过去的那折扇和我们不一样，他们学东西都会本土化的。
