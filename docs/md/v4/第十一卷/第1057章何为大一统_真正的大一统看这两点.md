# 第1057章 何为大一统_真正的大一统看这两点

何为大一统？——真正的大一统看这两点。《华夷鉴》有言：“秦关铸鼎，周室分封。昔者禹甸同规，尧封共贯。岂惟弓矢绥万邦，实乃文章被八表。辨统绪于毫芒，照肝胆于史乘。”然而，清室称制，诡称混一：汉垣委于胥吏，朔漠托以盟书；雪域假活佛之号，朝鲜冒中华之名；朝堂悬五色冕旒，边塞筑千重藩篱。遂使衣冠异制，法令殊途。岂有金瓯无缺，翻作马桩罗列？观其理民之术，塞雁各有其道：草原汗号盟血，卫藏佛衣惑众；汉地但行包税，满城独守满城。

凡是初高中需要考试的同学，请出去别听，不是为了耽误你的考试。当我们说一个王朝是不是大一统的时候，先得弄清楚什么是大一统。中国人是从什么时候开始界定起我们要大一统的这种心愿的？目前而言，大家心目中的第一个大一统王朝，通常认为是秦。一般的人会觉得第一次大一统是秦，但实际上不能这样说。因为秦可以说是第一个郡县制的大一统王朝，但那之前难道就不是大一统吗？只是先秦时期最清晰的是周。周王朝有没有一统？它下面的诸侯分封国有没有认周天子？行的是不是中国之法？很明显，无论是商还是周，无论是方伯还是诸侯，实际上都是按照同一套管理规范、同一套道德规范逐层划分出去的管理形式。所以，从我们设立文明国家的那一刻开始，我们进行的就是一种一统观的建设。

一统是什么？一统应该指我们用同样的一套守则、同样的一种方式、同样的一种认同的方式把人组织起来，这才能够叫做一统。就是大家都来听我的话，大家都按照我的守则做事；我们用的是同一套密码本，用的是同一套工作笔记。这个东西就叫做一统。实际上，先秦时期我们也是一个大一统国家，只是管理的方式不一样——是分封制的、是分封管理的一统式国家。而到了秦，我们实现了深层的管理，就是郡县制之下的大一统。大家觉得秦算是一个典范，是因为郡县制确实比分封制管理的程度更加深化。所以，如果你要非常严格、一定要管理得很深才能够叫大一统，你也可以把大一统界定为从秦才开始。弄清楚什么叫大一统之后，才能去点评满清的统治算不算大一统：它用了同样的管理方式没有？它用了同样的秩序规范没有？它用了同样的文化体系没有？它是不是在同一套纲领之下管理了庞大的国家？这才是判定大一统最核心的东西。

而清是怎么样进行管理的？满清皇帝自己承认：第一，在汉地18省内部，实行的是用汉官进行羁縻管理的管理方法，即间接承包管理法。这是在汉地18省——其实控最深入的地方，满清皇帝自己承认推行的是用汉臣、士绅、地主进行的承包管理法——羁縻。所以，这个地方算不算一统？它在汉地运用的是汉官汉臣的间接羁縻管理方式。然后，在蒙古草原上面，用的是拜把子歃血为盟的部落联盟管理方式。所以满清的皇帝在草原上时，会有专门的名字叫某某某汗。大家可以问任何一个AI：满清历代皇帝都有一个专门的封号，专门用于草原地区，叫某某某汗。然后他去管理其他地方时，例如藏区，也有自己专门的名字，叫什么什么文书之类的，本质就是说他是活佛。虽然用了皇帝这个词，但叫作文殊皇帝——文殊是一个菩萨名，“文殊皇帝”是特定菩萨的名字，他给自己安排了一个专门的活佛岗位。也就是说，他在藏地进行宗教头人管理，在草原上进行部落联盟管理（交给了蒙古王公），在汉地进行由汉官和汉臣进行的羁縻统治。

到了这儿，就不得不提到老外设置的一套“新清史”系统。这个系统有一个特点：心眼特别坏，但逻辑很完整。简单来说，就是视角是坏的、心眼是坏的、屁股是歪的，但把道理讲圆了。为什么它能讲圆？因为它就是站在满清朝廷的立场上去讲这段历史，所以虽然坏，但事儿讲得很圆。我们国内的清史学者跟新清史学者辩论时老是输，原因就是我们这边逻辑链条讲不清楚，自己打自己嘴巴的情况太多了，所以老是吵不赢。新清史系统会专门强调这一点：满清皇帝在面对不同对象时有不同的自称。面对汉地时，他称自己为中国皇帝；面对北边草原时，他自称为好兄弟、好安达，称自己为某汗；面临西域及其他外国（如大毛等）时，他自称为“代清固伦汗国”——一个专门的满语名字，既没有用汉字，也没有使用任何中国元素，就是给自己起了一个单独的满语名字。对于所有外藩进行自称，在国际上它的名字叫“代清固伦汗国”。只有一种情况下他会自称“中国”，那就是面对中国传统的内藩藩属国时，例如朝鲜、交趾。因为曾经有相当一段时间，这些地方甚至曾经被列为郡县，所以后来它们是不是都自称小中华？就是因为当年深入的程度非常之深，和我们往来的深度极强，绑定的关系极紧，所以在文化系统、服装系统、行政系统上都对我们有大量模仿和学习借鉴。这些群体汉化程度比较高，你要去指使它们，不自称为“中国”，它们理都不理你。所以当时满清皇帝对于中国传统中国的范围、汉人的范围，以及传统汉化地区（如交趾、朝鲜这些深度汉化地区），会自称为中国皇帝，而对其他地方都有各自的身份。

于是推演到下一件事：它有用同样的制度进行管理吗？要知道，在过去的传统中国概念当中，我们是进行分层管理的：内部的边户齐民区有我们的一套管理秩序，然后越往外面越宽松。这种宽松——羁縻、朝贡——这种逐层分发的管理系统，其实是由中原自己制定的规则。你看上去它不是太一样，但它都是中原皇帝、中原皇室制定的标准规则，只是离得越远标准越宽松，是逐层强力递减的方式推送出去的，所以它是同一套管理方略、同一套制式。但是到了清代，它在每一个地方都运用当地的管理秩序和文化秩序进行间接管理，所以对它治下的地方都是一种名义上和间接上的管理形式。里面所推行的管理秩序也不是满洲秩序——满洲秩序推行到了各大满城，他们自己内部的满洲规则是推行到了各大满城的。超出满城范围之外的所有地方，其实运用的都是当地原有的秩序框架。当然，在它管理深度比较深的汉地，它把原有框架的框架留下来了，肉全改了，但这又是另外一回事。因为这个地方八旗驻军比较多，它想干啥就干啥，确实管理深度比较强。但是在这个问题上，我们看出来一件事情：它的管理方式非常“入乡随俗”——基本是我到了哪里、征服了哪里、武力镇压了哪里，就借用当地的一些力量来进行间接管理。那么这种方式，能不能说它在制度上面达成了一统？如果说制度达成了一统，为什么中原那么拥挤了，它居然都不移民实边？所以整体上看，满清的这种管理方式就是一种种姓隔离制的管理方法。无论是在地区上还是在层级上，都是一种隔离制、分化制的管理方式。这种严格的界限、避免大家产生交流、不断进行细分和区隔的方式，你一定要说它是大一统，那么可能我们就需要颠倒一下，“一统”到底是什么才行了。

所以，你说清是一个我们最后的大一统王朝——靠地图的话是挺大一统的，但在管理上、文化上、甚至称号称谓上，它都没有一统。
