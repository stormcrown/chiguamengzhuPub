# 第1001章 为什么会出现汉地十八省？ 

所以，这个所谓的“汉地18省”，本质上就是一个圈禁的概念，是一种刻意的约束。它不是说“我们只有这里”，而是在满清的统治下，汉民族只能被限制在这一片地方，其他地方都不允许你去。等到满清放开管制之后，闯关东的人就出来了，同时走西口的也出来了。这样我们才保住了北方相当大一片原本属于我们、但被清朝抛弃了的土地。

如果清朝没有这样做，那我们就只剩下汉地18省了。呵呵呵……哎呀，我们为什么会出现汉地18省这个概念？如果没有清朝，世间绝对不会有汉地18省这个概念。这一点倒是真的，因为从来没有哪一个中原王朝会限制汉人向边缘地区迁徙，甚至还会强行移民实边。因为没有汉人的地方，就不能确保这个地方千秋万代。所以，任何一个君王在开疆拓土之后，想做的事情就是把人口稠密地区的人迁徙到边境地区来开放，让他们垦田、驻守。当这个地方出现了大量的村镇集市之后，这块地就妥妥当当地属于中央王朝的管理了，是那种跑不掉的地方。如果是没办法的时候，采取了饥民，那其实就是一个半管制地区，万一出点篓子是有可能离散的。

所以呢，唉，正规的中原王朝都会采取移民实边的方式。因此，汉人始终都处于一个扩张态势下。只要这个地方他们可以去，朝廷立刻就会安置大量的人，或者是用利诱的方式，或者是用强迫的方式，反正就是让他们在那里种田。所以不存在所谓的“汉地18省”，因为汉人的脚步是没有人约束的，甚至朝廷会专门鼓动、强迫性地让他们向外扩张，当然也就没有所谓约束的18省这个概念了。

汉地18省是清代才出现的概念。为什么会有这个概念？因为满清的皇室严格禁止汉人涉足边境。无论是在哪个方向，都特意圈定了范围，汉人不可踏出边疆一步。因为其他地方哪怕沦为无人区，变成浮动而难以掌控的地方，也不允许汉人到那里去。因此才出现了所谓汉地18省。这不是什么爱，就是因为有清朝，我们才有了这个地儿；不然就只有……嗯，就是因为清朝，汉人才会被圈禁在18省的范围之内。不然以当时那么广大的实力、那么稠密的人口、那么多的人和那么好的耕作技术，他们老早就跑到外边去开荒去了。所以柳条边失控之后，很快大批量的人就开始闯关东，去开发东三省的土地。如果没有闯关东，没有大量农民在东北耕作、开垦、经营农庄，这片地我们守不住。因为没有百姓在上面居住，只是派了几个部落在那里巡守，你怎么可能把这个地方变成自己的固有土地呢？

所以，这个所谓的汉地18省，本身就是一个圈禁的概念，是一种刻意的约束。嗯，不是我们只有这里，而是在满清的统治下，汉民族只能在这一片地方，其他地方不允许你去。在放开之后，闯关东的人出来了，同时走西口的也出来了，这样我们才保住了北方相当大一片原本固有、但被清朝抛弃了的土地。没有人住的地，你光宣称说这是自己的，哪天别人宣称一下，你有什么办法吗？你没有办法。

嗯，东北在康乾时期也有汉人。在康乾时期，发动了几次关于派驻人到东北去种田的项目，而且还安排说，凡是拉人头到那边去种田的，就给你升职，升职加薪、迎娶白富美的机会。所以确确实实吸引了一部分人出去，但数量非常少，数量是非常非常少的。为什么不去？因为实际上到那边的人，理论上你给他的身份叫平民，但实际上压根不是。因为在东北，被清朝视为龙兴之地，留下了很多自己的老家人，在那里称王称霸，当地的这些领头领主都安排好了。理论上的平民去了那儿之后，背离自己的乡亲，然后在别人的领主的治下，问题是很大的。所以呢，当时有没有呢？有啊，但是很少，数量非常少。真正在东北布开广大农民的，依然是柳条边撤离之后，那些闯关东的人自发行进，举家举族迁徙，他们彼此之间有牵连、有纠葛，然后才能够和睦地集成村落，开始开垦土地。啊。

嗯，外东北被物理消灭的汉人也有，不光外东北，东北也有啊。你以为大家为什么不敢去？就是当年那个满清入关之后，第一波的逃人法，就是所谓的抓逃人——逃跑的奴婢嘛。为什么会有逃人法？为什么会有那么多逃人？首先奴婢逃跑，才会有抓逃人的事情。那为什么会逃呢？其实就是这个所谓的奴婢，原本就是从关内抓过去的劳役兵丁，那些人就是在历次掠边的时候被抓走的。因此他们在关内有亲戚、有家园、有土地，在关外则举目无亲，没有办法就给人家当包衣奴婢了。但是一旦满清入关，入关之后，相当于这些奴婢就路过自己的家乡了，所以这个时候出现了大规模的逃窜，包衣奴婢大规模逃窜返乡。所以这个逃人法最开始的时候，就是专门针对这一拨人的。但是事实上在操作的时候，它只是给了一票名额：你原来声称你丢了十个，你就可以去抓十个；至于这十个是不是你原来的那十个，没有人管。于是，就出现了很多普通的民家的人被凑数，甚至有些权贵连数都不考虑了，直接见人就抓。关外八旗、蒙八旗、汉八旗会抓百姓带到关外为奴，那是他们的长期项目。因为那个地方很重要的几个项目：第一，缺乏劳动力，严重的缺乏劳动力；第二，缺乏工匠，工匠也是严重缺乏的。所以实际上，他们所谓的“进关掳掠”，重点对象就是工匠、劳丁，还有缺乏的粮食。也有一些贵族喜欢抢金银细软，但都靠后。最核心的就是抢人。每一次入关劫掠都有非常重要的项目，就是抢人。抢回去之后当奴隶、当奴婢，帮他种田、帮他干活。因为他们那套管理方式本身有很强的人身依附属性，你只要到了他手下，根本就跑不了。所以他非常放心就让自己的奴婢替自己干所有的活，是很热衷于到关内掳掠奴婢的。因此当时关外有相当多的包衣奴婢，就是在一次又一次的掳掠过程中被劫出去的人。满清进关之后，这些人就跑路了。

至于之前我们曾经聊过的，在东北那里有驻军、有屯垦、有农民，也有军户、农户都在那里。那他怎么还需要天天出来劫人？他就那么缺人？对呀。因为他们当地原本的那些军户、农户没有了呀。为什么没有了呢？嗯，因为西无谷人嘛，闹饥荒嘛，就把那些吃粮食的人全部都消灭了嘛。这事儿谁干的？努尔哈赤干的——杀无辜之人。嗯。
