# 第1060章 华夷经纬_三千年族群治理启示录

### 华夷经纬：三千年族群治理启示录  

**【华夷鉴】**  
青铜铸史，殉坑现异域之颅；竹简承训，华夷辨亲疏之序。商鼎烹异族而固疆土，周礼定远近以安社稷。色目东来，元廷分四等之制；明律禁婚，朱皇融万族之血。  

昔大唐盛景，胡商云集长安市；至宋室倾危，蒲氏倒戈闽海隅。盛时慕唐风而称臣，衰日认祖源而裂土。三千载兴亡轮转，九万里风雨沧桑。  

今观寰宇，当持古鉴：非固步自封而排外，非盲目包容而失序。差异既存，当施差别之策；认同有别，须立远近之纲。此非沙文之霸道，实乃共生之智光。  

---

种族区别不是西方人来了之后我们才知道的。全世界所有人其实都知道种族的区别。我们可以去看一下商代墓葬的殉葬坑里的考古报告，会发现殉葬坑里的人那叫一个多元，甚至包含了大洋洲土著的头骨。也就是说，我们这片土地并不是只有我们的祖先看上了这片风水宝地。在历史上，曾经有许许多多的族群、部落、群体，曾经看上过，曾经在这里经营过，或者至少试图来这里生活过。只有他来了，才有可能进入商人的殉葬坑里当祭品。所以，种族之间的差异这种客观事实，古人在几千年前就已经很清楚地见证过了。  

那么，当年的我们的祖先做出来的选项是什么呢？放进锅里去啊。这是在商及以前我们所做出来的策略。我不说这个事是好还是坏，但是今日中国所有人都获益于此。我们之所以长这个样子，之所以有这样的身体特征、智力特征，有这样的组织能力，有这样肥沃的土地，有这样广袤的生活空间，就是因为我们的祖先在几千年前，在看见异种的那一刻，决定把它们放进锅里面，而不是放进房里面。所以好或者不好，至少说我们之前曾经经历过，保底3000年的论证。我们今日之中国，获益于我们祖先的选择，我们拿到了好处。我这个人一向这样：拿到了好处，我就不会端起碗吃饭、放下碗骂娘。我获益于商人当年的选择，所以我就不会说商人太残暴了，所有的人都是一模一样的，我们应该相亲相爱，生活在一个大屋子里面。既然他当年没有做这样的选择，而我今天过下来了，活下来了，过得还不错，那就证明了他们3000年前的选择是对的。  

到了周王朝的时候，中国人界定了一套新的理论纲领，叫做“华夷之辩”。这个“华夷之辩”在今天有一些人非常敏感，或者说他的受害者心理爆棚，总觉得“华夷之辩”一翻出来就要割他的脑袋。但实际上，“华夷之辩”不是干这个事儿的。“华夷之辩”是做什么？是界定远近亲疏：谁和我更亲近，谁离我更遥远。越远的那一个，我就要运用其他的方式与他往来；近的用什么样的方式往来，远的又用什么样的方式往来，这个叫做“华夷之辩”。我们不能够把所有人都说“兼爱”，一毛一样的。当你选择所有人都一模一样的时候，其实你就已经背叛了和你最亲近的人了。因为人类的社会应该优先爱自己同类，爱自己身边的人之后再推广出去，这个叫做“达则兼济天下”。什么时候达啊？达到什么程度？是不是什么所谓的“登顶”？我不太喜欢“登顶”这个词，尽管说看上去我们现在越来越强，将来可能又会实现我们的伟大的复兴，成为这个世界的真正的、甚至是唯一的翘楚。但是，“登顶”这个词我依然不太爱，因为任何一个群体一旦登顶，就意味着它往哪走都是下坡路。我老是觉得这个词儿不吉利，所以我不喜欢这个词儿，仅此而已。那么假设说我们又成为了文明的代言人、文明的话事人，后世我们还要不要按照“华夷之辩”的思路，按照远近亲疏的思路来进行管理呢？就看你如何看待“华夷之辩”。你能够觉得自己的家人与过路的那一个乞丐是一模一样的吗？如果做不到的话，我们还是要坚定一下远近亲疏。谢谢白金海洋心啊。这个呢，是我们从西周时期开始进行的一套规范。  

那么这种方式在后世的时候又发生了什么呢？我们都知道，在元的时候，中国的国内由于蒙古西征，从中亚的地区带进了不少的游牧的群体，于是出现了“四等人制”。四等人当中的第二等叫做“色目人”。这些人是蒙古王公从西边带进来的一些包税人、代理人、管账的、收账的、放贷的。这些人作为社会的中层管理，作为白手套而存在。这些角色在元亡了之后，实际上也有相当一部分就留在了中国的土地上面。所以呢，明王朝界定的规则叫做“归我者，永安于中华”。这些归我者当中就包含了色目人。那么明为什么要将它称之为“色目人”？色目人是同一类人群吗？其实不是的。色目人指的是与中原人群有明显外貌不同的人群。所以这些人很有可能压根儿彼此之间就不是同类，不是同族，但是因为与我不同，所以统称为“色目”。那么明王朝是如何管理这些人的呢？明王朝对于色目人的规定叫做：禁止他们进行族内婚，凡是婚娶，必须与汉人婚配；只要你婚配、只要你生育，必须与汉人婚配；不可以集中居住，必须与汉人杂居。通过这样的方式，在数十年之后，这个群体在中国就没有了巨大的影响力，没有能够成为一种颠覆性的力量。那么，明王朝为什么要做这样的选择？是因为朱元璋预感到了什么吗？不是的，是因为他看到了什么。因为在过去的无论是唐还是宋，实际上都曾经招揽过不少域外的胡人。在他强盛的时候，无论是商贩也好，甚至是佣兵也好，其实都曾招揽过一个复数。平安的、繁盛的、顶流的国家，是一定会吸引到各方移民的。所以唐宋至夏都有外人，都有胡人，而且生活条件不错，允许他们举家聚居，甚至宋的王室还特意给一个色目人的家庭提供了皇商的待遇，让他们进行海贸，这个就是蒲氏家族。后来这群人做了什么呢？在南宋败北之后，宋国的王室试图到蒲氏家族里面寻求一定的庇护，然后蒲氏家族直接背刺一刀，将它们卖给了蒙古人，一口不留。所以这种事情发生之后，因为朱元璋知道这件事情，所以当他喊出了“色目人也可以永安于中华”之后，禁止他们合族聚居，要求他们在婚配的时候必须与汉人通婚。目的是什么？目的是为了消解他们的认知，消解它们的外貌差异，消解它们的独特属性。只要你的认同感与他者不同，在一个国家的管理有疏漏、秩序有崩塌的时候就会出乱子。所以你看明王朝的崩溃其实也非常的可悲，但是他在崩溃的那一个时候，没有出现唐末与宋末的时候那种大家都说自己是中华正统，但是长得不一样、文化也不一样、认同也不一样的奇特崩溃场面。明王朝死得很干净、很利落。  

所以你说人种应该如何管理，应该如何识别，应该如何面对？中国人是全世界最宽厚的一群人，没有之一。但即便是最宽厚的这一群人，也一定不能够把自己的宽厚当作自己最荣耀的勋章，当作自己的束缚，说我必须宽厚，必须要视所有人一视同仁。这世上没有人可以视他者为一视同仁，不一样就是不一样，一定要有区别的待遇。而且一个国家、一个群体聚居在什么地方，形成了什么风气，大家长什么样子，有什么认同感，这个东西在盛世的时候可能没那么要紧，大家都觉得大唐很美、大唐很帅，我都承认自己是唐人嘛。但是一旦大唐走向衰败呢？在衰败的那一分钟，你就突然开始认自己的老祖宗了。人类的群体在我们的历史过程当中，反复不断上演同样的戏码。所以如果今天在我们还没有完成复兴的时候，就已经开始想象着说所有人都是一模一样的，天下尽是朕的子民，那么唐末的那场悲哀的场景，那种到处都是“小唐”的场景会不会重现？我不希望历史像一个不断重复的车轮，我也不希望在人类的世界当中，永远都不能在历史当中吸取教训。人总应该在自己的记忆当中获得一点什么有用的东西吧。当不同的群体他的外貌特征客观上就不一样的时候，为什么不能够起名字呢？但是当我们有了世界的定义权、有了话语权的时候，为什么就不能够去影响他人，即使不同也可以和而不同呢？不一样就是不一样，不同的人群就是应该进行差别的待遇，但是这不意味着虐待，这也不意味着我必须压榨他。中国人的规则没有那么僵化，也没有那么非此即彼，不是说我区隔了我就一定要如何如何蹂躏它。有一些识别根本不应该做，我们做了；那么有一些识别，无论你做不做它都客观存在，却有人说我们为什么要做这个识别呢？这是很荒唐的一种想法。
