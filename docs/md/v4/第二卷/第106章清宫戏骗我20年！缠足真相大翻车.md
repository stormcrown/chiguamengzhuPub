# 第106章  清宫戏骗我20年！缠足真相大翻车

嗯，我们这边来了一个应该是小青粉啊，不然的话不会咬死了明代缠脚什么的这个事啊。这种撒谎的行径只有小青粉能干。但是他给自己起的ID叫高欢，我觉得可能有遗传性的精神不稳定的迹象啊，祖传的嘛。觉得封口一下会比较爽，是这样的吗？啊，好的好的啊。

实际上就是次次都一定要跳出来讲缠脚、裹脚这个事情。为什么这个样子？我为什么也要车轱辘话不停地在讲这件事情？因为呢，这是一个小青粉、小满夷非常重要的一个战略高地啊。我们知道一件事情，那个北齐那个疯子，对，不然呢，还能是谁？高欢啊，祖传的精神系数不稳定啊，祖传的，治不了的那种。家里面就剩一个好人，还没有好下场，就已经到了这种地步的那种家庭啊。我也不知道他为什么要这样爱他。

那个缠脚这件事情，为什么会成为现在小青粉儿们非常咬死的一个战略高地？因为什么呢？你要在一个和平的国家，在一个正常的国家里面去栽赃污蔑一群人，你需要做的事情不光是空口白牙的造谣就行了，你还需要占据一个道德制高点。必须有啊，你必须显得自身像一个白莲花似的，才能够去说别人啊。这个很容易理解吧？就是你想要去攻击他人的时候，总要假装自己是个好人嘛。所以这些小青粉、小蛮夷呢，他在造谣生事的过程当中，就要假装自己是一个好人。那怎么假装自己是一个好人呢？啊，他直接去讲清代真实的历史，那没有办法啊。他一旦说我认可清代的什么什么什么事情，那就完犊子了啊，那就不可能当一个好人了。因为清就没干什么好事，他一认可就完蛋了。

所以呢，之前前一波的这些小满夷，他是怎么占那个高地的呢？啊，他就是把今天的满族啊，和当年的遗老遗少绑定在一堆啊。你只要说清不好，你就是破坏团结了。因为啥？满族同胞会不高兴。满族同胞就是因为骂满清才会成为满族同胞的，他这个事儿就不理是吧。所以呢，唉，这是他们的一个战略高地：你凡是说清不好，那你就是不跟我团结了，那我要跟你翻脸啊。这个就是他的一个战略高地。但这个高地站得不稳当，因为这个高地很显然不符合逻辑啊，而且特别的歧视主体民族。这个小高地，他是一个带有破坏性的小高地，所以呢，他没有什么很大的价值。有没有点力量呢？有一点，但是不大。于是呢，他必须有一个真正的让自己看起来特别清白、特别高尚、特别善良、特别纯和这样的一个战略高地，然后才能来侮辱你。

所以这个时候他用了什么方式呢？就是用裹脚啊。因为什么呢？因为清在关外的时候啊，清在关外的时候发布过两条禁止裹脚令，禁止的是谁呢？禁止八旗女子效仿汉家女子缠脚。但是呢，他断章取义嘛，他就说你看我清一直禁止裹脚，是你不同意。然后呢，他就可以去栽赃汉人的几百年的传统啊，说汉人呢，从来都残酷地压榨女性，我大清都是女人当家做主，女人在家里面姑奶奶地位老高了。就特别喜欢整这个事儿。这是什么呢？这就属于一个战略高地。尽管每一句话都是谎言，每一个字都是现编的，但是呢，他把这个东西编了反复重复了之后会干什么？它会产生一个效果，就是他是一个正义方。他正义方站直了腰杆子，再说你的不好的时候，他就显得这个事儿顺理成章啊，居高临下。这个就是造谣的时候他必须要做的事：一定要塑造自己的一身白莲花的景象。

但是呢，因为这里面全是谎话，所以是能够戳穿的。可是这个戳穿的过程却会非常艰难。为什么？因为人家走量，虽是谎言，但是量大。而且这个数量级上面呢，还包含了很多实际上很科学、很官方的叙述啊。为什么？因为缠脚这个事儿确实存在，甚至很多今天活着的人都已经见过啊，这种小脚的老太太一生过得多么的苦难。这些事儿呢，中国人是亲眼所见、亲耳所闻，是明明白白知道的这个事。所以他只需要造一个假就行了：他只需要把清代出现的缠脚、小脚老太太挪用到宋明，他把清代强迫式的缠脚扩大化成为宋明的所谓传统，然后再把清代针对满人女子的禁止缠足令扩大成针对所有汉家女子的禁止缠足令，最后就把自己编造成了一个正义方。然后呢，把汉人的这个被迫的行为，说成是一个很反动的传统。一旦这种趋势形成，我们就开始理亏，他就开始变成了一个正义之神了。

所以为什么他铺天盖地地咬死了要在这个事上扑腾？因为一旦你把他这个谎言戳破了，他身上真的再没剩一点干净的地方。这个虽然是假干净，但至少现在装得还挺干净的样子。所以他特别痛恨我在直播间里面、在视频里面反复地去讲，缠脚的风俗是清代才出现的这个客观的现实。他生怕中国人知道了这个客观的真实现象。一旦你知道了缠脚是在清代才出现的风俗，这种客观的历史事实，你就会知道他们造假这件事情深入骨髓。而一旦你知道了这种客观的历史真相，你就会知道一件事：缠脚不是风俗，缠脚是残酷的奴役与压榨啊。

为什么他非得要在这个事上纠结啊？他就那么在乎女人吗？就那么在乎小脚吗？不是，他是为了缔造自己的白莲花形象，以及呢，对主体民族的传统进行栽赃与污蔑。所以他刻意地用断章取义的方式，制造了一段有证据的假历史，然后铺天盖地地用网军数量级的方式进行全覆盖。当覆盖的量级足够大，那就叫做“谎言重复千遍就变成了真理”啊。

嗯，啥？取名叫高欢的能说什么好东西？那个都愿意给自己起名叫高欢了，那说明他对心理不稳定、精神不稳定这件事情还是颇为向往的啊。虽然我个人不理解，那不理解就不理解吧，我不理解的事老多了。我对于人呢，向往办秃瓢这事儿也不理解啊。

嗯，啥啊？宋代的缠脚发生在少数的爱美……嗯，而且啊，宋明两代的缠脚，第一呢，它属于是一种流行风尚；第二呢，它是这种摩登女郎才会用的啊，普通人不会。而且呢，它是跟强迫一毛钱关系都没有。因为宋明两朝的裹脚就约等于尖头恨天高。你在墓葬当中也能够看到，他就是把脚呢，缠得瘦瘦窄窄的，弯弯翘翘的啊，就是向上翘。那你把脚趾头翘起来，这个姿势其实就是穿尖头恨天高的那个姿势啊。你穿尖头恨天高的时候就就这样啊，你自己去看鞋子。你们到网上去搜鞋子，我没有这种东西，我自个儿都没有这种高跟鞋，嗯，没办法展示啊。你们自己去看，就是脚塞在尖头恨天高里的时候，他的姿态就和宋明两代的女尸上面的裹脚是一样一样的啊。

老师不写怎么防雨水天啊？我们的古人在雨天的时候啊，是不是单穿鞋子的啊？指导那个老式的布鞋底下要上浆子，然后要一层一层地压起来去纳鞋底儿，所以呢，那种鞋子确实没有办法在雨天穿哈。所以我们要用木屐来。哎，下雨天的时候要干什么？下雨天的时候要穿木屐。一穿木屐说，哎呀，你是那个什么啊，你是倭寇，你是倭奴啊。然而啊，我们的古人是穿木屐的，所以大家是很，古人是很聪明的啊。

东北特色乌拉草，咸啊？乌拉草是拿来保暖用的啊。在那早年间的话，早年间的时候，他的老话的传闻，东北是有三宝：人参、貂皮、乌拉草啊。后来呢，他的三宝就变成了人参、貂皮、鹿茸啊。在早年间他的传闻当中呢，叫做人参、貂皮、乌拉草，他和鹿茸没什么关系。为什么？前面几个啊，人参是干什么的？在古人的概念当中，它是救命的东西啊，所以呢，是珍宝。而且呢，它是当年关外部落非常重要的核心收入来源。同时呢，这个貂皮在今天看起来有什么？违反了动物保护啊，什么奢靡之风啊，但是在当时的苦寒关外之地，貂皮这个东西拿来保命的啊，在苦寒之地是可以保命的东西，所以它确实是极致的珍宝。然后乌拉草是什么？乌拉草这个东西，它是你塞进鞋子里面，就相当于起了绒一样的，可以起到保暖作用。所以它不是一个真正的草鞋，它是它是一个专门的保暖层，也是可以救命的东西哈。所以呢，在过去的这个传说当中，它叫做三宝啊，人参、貂皮、乌拉草。

嗯嗯，什么？徐霞客好像还专门发明了登山木屐啊？登山木屐应该是谢公履啊。登山用的木屐就是一个活齿的木屐，它下面的那个齿是可以动的啊，根据你的角度不同它会动。这个呢，是登山使用的木屐。我们知道古人在穿鞋子的时候，强调的其实是它的功能属性啊。穿鞋子最重要的是它的功能属性，什么样的功能才值得你去穿着它。这个木屐呢，就是我们在劳动的时候、在下雨天行走的时候、在遇到泥泞的环境的时候、在地势复杂的情况下去穿着的一个保护型的鞋子啊。
