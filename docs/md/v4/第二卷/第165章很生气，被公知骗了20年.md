# 第165章  很生气，被公知骗了20年 #抗日战争 #大会战 #游击战 #百团大战

关于抗战期间大会战与游击战的作用大小，如果不去拉踩（即不直接评比），这个话题是可以聊的。但若非要分出哪个作用大、哪个作用小，就不好谈了，因为很难量化。大会战与游击战的贡献多寡，如果放到20年前，某些人会告诉你——正面战场上他们如何如何，而你们都是躲在后面如何如何。前20年时，这类论调是为了否定我党的贡献，推动中国人走自由主义西化道路，从而刻意编造谎言。但由于前期大会战和后期游击战的作用方向有所不同，我们确实不能直接用量化指标来判断谁贡献大、谁贡献小。

那么，大会战有没有必要？当然有必要。当敌人整建制集结过来时，这不是游击战能直接在后方一点点戳动的。必须迎头跟上，否则敌方大扫荡直接冲过来，怎么办？所以正面会战非常重要，客观上能为我们守土。例如，我们第一次的重要大捷，就是国共两党合作后，通过正面会战在局部战场取得的。因此，我们也参与了正面大会战。

同时，游击战的重要性也需明白，这涉及《论持久战》的内容。看上去日寇势不可挡，因为他们是一支现代化军队，战斗力强，虽然个子矮但装备好且狠辣，组织能力也强。我们在正面战场上常常节节败退，这是客观存在的，因为装备、机动性等方面当时有很多先天缺陷。但对方也有缺点：日本是小国寡民，没有充分的基本盘；而中国幅员辽阔，只要敌人深入进来，我们就能用本土力量分散和截断它。游击战就是在己方占据的敌占区下，像钉子一样使敌人首尾不能相顾。所以游击战也重要。两方面必须同时存在，我们的抗战才有可能成功。不能只靠某一种手法操作，每一种都重要。关键在于：正面抗战时要血战到底，敌后游击时要保持革命意志。做到这些，我们才能革命成功。

之前批判国军方面水平不行，是因为他们在调度上常存在军阀作风——喜欢保留自己的力量、抗争不彻底。比如“友军有难，不动如山”，他们为保存有生力量，不能全力抗战，因此说他们具有反动倾向。这并非否定正面战场的重要性，而是他们没打好。如果他们打得更好，战局就会不同。

这里顺带推荐张纯如老师的著作。她付出心血甚至生命代价撰写此书。我认为每一个中国家庭都应该拥有一套。至于读不读，看个人心理建设水平——当你准备好时应该看一看。为什么我们需要了解真正的侵略者是什么样子？这很重要。一个国家承平日久，普通人会变成“日子人”，这是天经地义。日子舒心时会萌生爱意，觉得万物平等、一切皆有可能。但人类社会的竞争并非永远春日暖阳，一旦出现敌我之争，博弈会极为残酷，你我皆可能身处其中。就在可查证的年代，我们的先辈经历了惨痛历史。我们需在过于温暖的时代提醒自己：倒霉时是什么样子？要见证历史的惨痛。忘记历史就等于背叛。所以所有家庭都应该有一本。成年人看压力较大，小孩子看更容易，但我尊重个人选择。我认为小学五年级就可以看。每个中国人有生之年都应读这本书，以深切理解面对侵略、面对敌我关系时那种你死我活的本质。千万不要被今天的美好生活蒙蔽感知。生活太好时，感知会钝化，我们需要磨砺自己，用历史教训警醒自己。要意识到历史真相和真正敌我关系的模样，用最具体、最形象的方式呈现出来，我们才能有概念，从而知道自己为什么而活、为什么读书、为什么工作。一个群体不是天然或理所当然存在的。我们能够活到今天、蒸蒸日上、享受承平后的钝化，所有历史都包含其中，包括失败的历史。所以必须了解这一切。
