# 第151章  何不食肉糜被嘲讽千年，康熙一天3顿浪费粮食。千古明君

啊，我们众所周知，他是一个智力有障碍的君王。最显现出他智力障碍的一句话千古流传，叫做“何不食肉糜”——你都饿肚子了，你干嘛不吃肉粥呢？就是这种糟心烂肺的角色。作为一个君王的时候，他会去思考百姓怎么吃不上饭；吃不上饭，那就吃肉呀，你总不能饿肚子的呀。他依然是有好心的。

到了清的时候呢，天下饥荒，他想的不是赈灾，想的是：“你这帮农民一天到晚竟然吃三顿饭加宵夜，你不饥荒谁饥荒？从此以后给朕改吃两顿饭。”同样是谈论饥荒，同样是谈论吃饭。一个是著名的傻子皇帝，一个是千年一见的圣天子在上啊。哪个好哪个坏，你自己琢磨琢磨，这是一回事吗？嗯，啥玩意儿？

哎，我我我我这边的直播间里面，今天还出现了“蒋吹”呢。啊，你们观摩一下呗。真的，我这边一般招惹来的都是“轻吹”，今天居然来了一个“蒋吹”。哎哟，我来，诶诶诶诶诶诶！真的，我们今天来了一“蒋吹”啊，忒厉害了，竟然冲上来给我整了一波“蒋攻”啊，呵呵呵呵呵呵呵呵呵呵啊。他的那个，那个名字啊——他的名字我就不念了，念他的名字我容易没。我也不知道为啥，可能确实这个世界上的“蒋吹”还不少。所以呢，讲到他的时候，我一般都叫他“常凯申”。没有办法啊，说他“常凯申”的时候我比较平和，一说他“蒋介石”啥啥啥的时候，我容易没有啊。人家后台老硬了，我都不懂啊，他咋就后台那么硬呢？

嗯，讲有功有过吧。那他年轻的时候啊，他年轻的时候还是有点出息的，知道吧？啊，常凯申在青年时代的时候，还是表现出了一些志向的，率领一个小队啊，突袭满城，还是做出了革命贡献的。你不能说他这个人一辈子啥也没干，那也不至于啊。在那种动荡的时代，能够把那些一地散沙的、各怀小心思的军阀们整合到一起，虽然说不是真的整合啊，但是呢，好歹凑齐了一个形式框架，这个事倒也也算是挺重要吧。啊，他的贡献大体出现在这个地方啊。但是其他的事情上面就乏善可陈了，甚至呢，糟心烂肺的事办的有点多。

当然，我个人认为呢，他主要的缺陷来自于个人的能力不足。常凯申呢，是一个比较典型的一个——在刚刚好的时间，用刚刚好的时机，占据了一个刚刚好的位置，非常的重要。但是呢，因为个人的能力不足所调动不了。

昨天看到博主说：“中国历史不单只有清罪孽深重，其他朝代也有。”总觉得不对，但是又无法反驳。嗯，他这就是一个典型的拉别人下水的一个套路啊。那是什么呢？就是你看啊，别人家也不咋地是吧，所以呢，这边的烂就是天经地义的了。都烂嘛，都烂，他这个“烂黄瓜”就显得不出彩了嘛，就这么个意思啊。但实际上它一样吗？哦，我跟你说，它是不一样的。

大家在高中政治里面应该学过一句话，叫做“量变引发质变”。学过这句话的来举个手啊，扣个一。学过量变引发质变的这句话的小伙伴，来在公屏上面扣一个一。嗯，你初中生你知道，嗯，好的哈。我看你看起来这是初中课本是吧，好的，行。这是在我们的标准的教育课程当中，给每一个中国的孩子都教了的东西啊，叫做“量变引发质变”。所以你不能说啊，因为有个贼老板，所以老板阶层都是不好的。因为你要看他的程度，程度依然是重要的事情。程度是什么？以什么目标去做这件事情。

就好比说，嗯，大家都知道西晋是个粪坑啊。我说西晋是个粪坑，这件事情应该没有人反对吧？啊，觉得西晋辉煌很璀璨的，嗯，出门左转啊。现在喜欢在互联网上扯的那个犊子，叫什么东西来着？叫做“魏晋南北朝，荒唐又美好”，是不是这么个段子？网上是不是有这么个段子，叫做“魏晋南北朝，荒唐又美好”？有这样的吧？应该有啊，有有这样的人啊。你要是真的觉得荒唐又美好的，那个可以出门左转了。西晋就是个屎坑啊，毫无疑问，西晋就是一个屎坑。

然后西晋大转折，西晋的大转折点出在哪儿呢？出在他们安排了一个傻子当皇帝。这个事儿大家也都知道。因为有了一个傻子皇帝，所以他的内部变得更加的混乱了，出现了重要的转折点，出现了乱政，然后开始此起彼伏的造反啊，逐渐逐渐的一波又一波，一波又一波，最后凑齐了八王之乱，然后造成了历史的重大转折点啊。所以呢，你说出了一个傻子皇帝，这事够悲哀了吧？西晋啊，本身就是一个生于不义的糟心烂肺的东西，他还出了一个傻子皇帝，看起来就更糟了，更屎坑了吧。

那这个傻子皇帝最重要的证明他是傻子皇帝的言论是什么？叫做“何不食肉糜”啊。我们众所周知，他是一个智力有障碍的君王，最显现出他智力障碍的一句话千古流传，就叫做“何不食肉糜”——你都饿肚子了，你干嘛不吃肉粥呢？够没用了吧。但是他的本意是希望百姓能吃得上东西啊，就是这种糟心烂肺的角色。作为一个君王的时候，他会去思考百姓怎么吃不上饭；吃不上饭，那就吃肉呀，你总要不能饿肚子的呀。他依然是有好心的。我们嘲讽这句话，嘲讽了千年，但是毫无疑问，他是有好心的。

到了清的时候呢，天下饥荒，他想的不是赈灾，想的是：“你这帮农民一天到晚竟然吃三顿饭加宵夜，你不饥荒谁饥荒？从此以后给朕改吃两顿饭。”同样是谈论饥荒，同样是谈论吃饭。一个是著名的傻子皇帝，一个是千年一见的圣天子在上啊。哪个好哪个坏，你自己琢磨琢磨。这是一回事儿吗？

嗯，他有好心，但是没脑子。他是先天的发育不良啊。那个，嗯，晋惠帝他是先天的发育不良。然后呢，因为他被立太子的时候，还没有显示出他的发育不良——因为他是年幼的时候就被立了太子，那个时候她看上去还是一个正常的孩子。而这个时候呢，嗯，立太子这件事情，对于一个国家来说，它叫做国本。我们自古以来是一个叫做那个嫡长子继承制啊。我们的嫡长子继承制，它的本来的目的，就是维持国家内部传序的稳定性。所以为什么君王即位之后，只要他是成年的，他就赶紧的要安排立太子的事情？因为立下了太子，就意味着整个朝堂上面可以安定下来，大家不用去思考下一波是谁，不用去选择站队，不用去撕裂，不用去考虑博弈的问题，而是今天是今天的皇帝，明天就是太子当皇帝。大家可以安稳的、持续不断的把整个朝堂运转下去。它是有历史为证的，是一种有效的传续机制。

所以呢，晋惠帝他在年幼的时候，看上去还是一个挺不错的小孩儿的时候，被选中成为了太子，这个是稳定国本的一种操作。而为了更加好的稳固他的太子的地位，于是他的父亲为他选了一个妻子，这就是选择妻族啊。记得选妻这件事情，不是选一个配偶，他选的是他的岳家，选的是他的靠山，是他的盟友。而这个时候，因为已经选定了他的盟友，选定了他的未婚妻，于是呢，这个太子的身上啊，就拥有了特定的政治身份。这个时候他的太子地位就会非常的稳固。而在这之后，他慢慢长大，被人家发现——完了，这孩子发育上面有点毛病，发育有问题。他长大了一点之后，就发现了他发育有障碍，看上去不太聪明，有点毛病。按照常规来讲啊，如果这个时候没选太子，那一定选不上他；如果这个时候他只是孤零零的一个太子，然后身上又有毛病，那很有可能就要改立。然而因为他的政治盟友已经选定了，而对方家大业大，恨不得要只手遮天，所以这个时候他的岳家把他保下来了。最终啊，最终就出现了这么一个先天发育障碍的君王啊。他是这样子上位的。
