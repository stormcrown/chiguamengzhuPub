# 第155章  清朝最怕的刁民，确是人民心中的英雄

在互联网上，凡是提到义和团、提到太平天国的，永远都要把他们身上的瑕疵拿出来，而不是去赞美他们身上根本的革命属性。为什么呢？因为他们身上的属性就是典型的“总有刁民想害朕”。那么这个视角是谁的视角？这是清朝君王的视角。当他站在清朝君王的视角上时，义和团和太平天国就是“总有刁民想害朕”。但是当我们站在普通人民、普通百姓、站在我们千秋万代的历史角度上时，义和团和太平天国即使有瑕疵，也是伟大的革命者。那么到底是谁在恨他们，谁又在爱他们？决定这一切的是你的立场、你的角度问题。

嗯，啥？清末？你说的这“老登”，指的应该是清末朝堂上的那些诸公们吧。好，清末的那些人知道自己落后于世界了吗？有什么行动？他们知道——我们不需要谈到清末，不需要谈到清末。他们从清初的时候就知道西方什么样子。很多人在学习过程中听说过一个故事：乾隆皇帝接见境外使团时，因为对方不给他下跪，很愤怒，于是朝臣给他解释说，那个洋人缺个膝盖骨，所以跪不下去，腿是绷直的。然后就好像乾隆皇帝一辈子没见过洋人一样，孤陋寡闻、闭关锁国，这就是衰落的原因。这个故事过去多年一直流传，但它是子虚乌有、现编的，一个类似《意林》类的故事。不管是什么类型的故事，总之它是完全虚构的。

事实上，从清初开始，清王朝内部就一直有西方人供职。最典型的例子是，清初他们推动了一次专门的大案要案稽查，在这次查案过程中，除了牵连一个西方传教士（所谓的天文学家）之外，其余被牵连进去的都是汉人的天文学工作者。从这之后，天文台——也就是钦天监——就被洋人掌控了。所以，当一个国家最重要的、核心的、掌控时间、监管天象、监管气候的国家天文台都已经由洋人执掌时，你说皇帝认不认得洋人？知不知道洋人有膝盖骨？他们从来没有和这个世界的今天所谓主流（或者说事实上就是当时的欧洲）断绝联系。这件事情子虚乌有。满清的统治者一直和西方人关联紧密——紧密到什么程度？康熙皇帝的老师是洋人，钦天监的管理者是洋人，乾隆的笔友也是洋人。而且乾隆非常清楚自身与西方之间贸易往来的内容，知道买卖了什么东西，甚至知道那些不要脸的商人夹带走私了什么东西。所以乾隆年间，他还专门批评过走私的商贩将铁锅走私给洋人。所以清的统治者不存在孤陋寡闻、不知道世界发生了什么事情、突然一头雾水被人家敲开国门的情况。他们一直都知道西方的存在，这是第一件事。

到了你说乾隆年间，他都还很清楚西方做了什么。你想一想，1840年被人叩开国门的时刻，乾隆才死了几十年。就这么中间短短三四十年功夫，至于遗忘得干干净净、一无所知吗？不可能的。中国境内，在有清一代长时间有洋人出入，一部分是所谓的天文工作者，一部分是内廷的教师、画师，还有一部分是在民间到处流窜的传教士。这些传教士要获得传教的许可、修建教堂的许可，甚至很多时候教堂内部如何管理、教民之间的关系如何协调，还需要通过官府来进行工作。你说上层统治者知不知道洋人？知不知道域外发生了什么？他是不可能错过这一切的。不止晚清的时候知道，清初的时候也知道。他们始终都知道西方什么样子，也知道别人进步的速度有多快。但是，为什么看到别人进步很快，却完全不想追赶？因为对于清的统治者来说，最重要最重要的任务并不是强大自身的国力，而是避免百姓造反。因为有清的二百六十多年，从来没有一天停止过民间的造反，从来没有停止过反清复明的革命。虽然说当时的百姓早就已经不记得大明是个什么样子了，然而他们知道：如何反清？要打起大明的旗帜才能反清。即使他们不爱那个大明，也会用这个旗帜，因为这个旗帜有用。所以整个清的二百六十多年，就是反清复明革命运动的二百六十多年，直到1911年革命成功为止。

所以，在这个过程当中，统治者是希望这个国家经济、技术、科学、军火越来越发达、越来越先进，普通百姓身上都能有一个火器来傍身好呢？还是希望所有百姓都蠢得不可救药，一个字都不认识，手里只有个镐头来得安全呢？所以即使是你，也可以轻易地在这样的力量角逐中分析出来：什么样的形式对当时的统治者最有利，什么样的形式对他最危急。他真正需要直面的对象，并不是那些努力在科学、技术、经济等等方面不断发展追赶的西方人；他所敌视的对象，始终都是治下拿着镐头也要闹革命的普通百姓。

所以，这几十年来大家有没有发现一件事情？在互联网上，凡是提到义和团、提到太平天国的，永远都要把他们身上的瑕疵拿出来，而不是去赞美他们身上根本的革命属性。为什么呢？因为他们身上的属性就是典型的“总有刁民想害朕”。这个视角是谁的视角？这是清朝君王的视角。当他站在清朝君王的视角上时，义和团和太平天国就是“总有刁民想害朕”。但是当我们站在普通人民、普通百姓、站在我们千秋万代的历史角度上时，义和团和太平天国即使有瑕疵，也是伟大的革命者。那么到底是谁在恨他们，谁又在爱他们？决定这一切的是你的立场、你的角度的问题。

所以我昨天的直播一直在说：文科为什么在中国的这几十年总是备受抨击？真的是文科这个学科不好吗？不是。是因为我们的文科有太多人没有为中国而战斗，他没有拿着自己的知识的武器、拿着自己学到的本领为中国而服务。他假装自己中立而客观，事实上却在为围剿我们的列强服务、去供职，因此才引发了所谓“文科无用”的刻板印象。不是这个学科无用，而是参与其中的很多人没有发挥正面作用。
