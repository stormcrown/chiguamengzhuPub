# 第117章  古希腊靠啥殖民？咸鱼倾销还是山地马战

在晚清之前，中国没有人知道“罗马”二字。直到民国时期，才有大师考证说，我们古籍当中的“大秦”就是罗马。你能理解这件事吧？甘英曾遇到一片海，那片海因为变幻莫测而让他不敢出海。那个变幻莫测的海，大概率就是地中海，因为在地中海地区，它是最变幻莫测的一个海。但这事儿就很麻烦了：因为有人记得给古罗马画的地图是什么样的——地图上环地中海全是古罗马的势力范围。古罗马的繁盛正是环地中海的。如果你看到了那片海，你铁定会看到古罗马。他们画的地图显示，彼时的古罗马把地中海包圆了，但我们去了却没看着。这不就有意思了吗？

现在有人打了几个补丁：一个补丁是说，古罗马可以殖民周边地区。但这就纳了闷了——它附近号称被它殖民的地区，却是适合养马的。到底谁打谁？这很难说，你知道吧。要评价罗马，比如希腊的玻璃器皿，汉朝是否有进口？第一，你要找到原文，把原文贴出来的同时，告诉我原文里它的名字叫什么。你不能只听那些营销号写的小故事，按小故事阅读很容易，但不能当真。

为什么这么说？因为在民国之前——或者说晚清之前——中国没有人知道“罗马”二字。民国时期的大师考证说，我们古籍里的“大秦”就是罗马。能理解吗？民国时期那些言必称希腊、罗马的大师们，考证出我们两千年前的文字里的“大秦”是罗马。但在我们的历史文献里，没有“罗马”这两个字，没有！民国大师说，以前的“大秦”书里提过，但我们过去对大秦的描述是什么？我们没见过那个地方。第一，我们想去来着，走了老远，但没找着；第二，我们所知道的所有大秦的信息，全是二手消息。我们历史上记载的所有大秦，全是二手消息。什么叫二手消息？就是商团跟我说的，使团跟我说的，听说的——这叫做二手消息。我们史籍当中有没有记载？有记载，但内容都是“听说在某地有一个大秦”。第一，是听说的，不是我们亲自考证的；第二，对当地人的描述是，我们曾接见过从那儿来的使者，那使者“颇类中华”——就是长得和我们很像，体格、外貌、衣着都像。所以这玩意儿是不是Cosplay，你也不知道。

能理解吗？在我们的历史典籍里，从来没有中国人亲自到那个地方考察过。我们不是没去，而是去了没找着。我们所知道的所有古典大秦的消息，全是二传手的消息。所以，所谓“大秦等于罗马”这件事，是民国大师在言必称罗马之后，才把古籍中那个“颇类中华”的大秦考证为罗马的。它是不是真的？咱们以观后效吧，不如等一等再看，谁知道呢？

甘英没去吗？不，他去了。他特意西行，执意西行，却没找着。嗯，好，欢迎大家来到直播间。可惜当时他要是能去对面看看就好了。这里就不得不谈到他到底去了哪儿——对于甘英当时西行的目的地，实际上是有争议的。有说他到了红海，有说到了地中海，还有说到了其他地方。这些都是后来的考证，不是当事人。你也不知道他当时到底到了什么地方。只知道他因为一道海拦住了路，当地人说这个海变幻莫测，不能直接驾船出去，出去十去九不回，必须等多久多久，等风来了才能去，而且去了也不保证能回来，风险极大。于是他就掉头回来了。只有这样的描述。按照这样的描述，其实不能断言它到底是哪一个海。有人判定是红海，有人说是地中海。这里头确实有问题——我们不知道甘英当时到的是哪一个海。但如果说他确实到了地中海，因为按当时描述，和地中海很像——虽然有人说他到了红海，但描述中那种风的变幻莫测、航行难度极大、十去九不回等，和地中海非常像。所以不能排除他当时到了地中海沿岸。一旦他真的到了地中海沿岸，那这事儿就很麻烦了：因为你们记得他们给古罗马画的地图吗？环地中海全是古罗马的势力范围，古罗马的繁盛是环地中海的。你但凡看到了海，铁定看到了古罗马。他们画的地图，彼时的古罗马把地中海包圆了，然后我们去了却没看着。这不就有意思了吗？

所以，不知道。简单来讲，虽然我们的古籍里提到了大秦，虽然民国时期有人说大秦就是罗马，但它们之间是否一致？不知道。无法保证这件事，高度怀疑，而且认为可能性不大。不过，有没有可能存在罗马这个地方？还是有可能的。欧洲人以前也不是凭空出现的。他们过去虽然是神权时代、蒙昧时代的神权统治，但神权也是一种管理方法，奴隶制也是一种社会管理制度。生在新中国的人，常常觉得奴隶制、神权制似乎不算文明，但我这里很明确地说：人类进入文明期之后，都会经历奴隶制或神权阶段，因为这个阶段最高效，能把人从野蛮野放的状态收拢成一个组织。而进入文明最重要的就是构建组织力。神权管理也好，奴隶管理也好，都可以形成组织力。所以即使当时欧洲人普遍在神权笼罩下，并践行奴隶制、农奴制，它依然是一个文明体。欧洲人不是突然从树上下来的，包括我们也不是从树上下来的，我们几万年前就住洞穴了，上哪棵树上下来？动不动编那些谎话。第一，它不是突然蹦出来的，它之前只是相对落后、文明指数不够高、神权统治较为黑暗，并不意味着这边的人不存在，也不意味着它不是人类文明体。所以完全有可能存在罗马这个地方。但这个地方有多大？文化水平达到什么程度？有多少脱产者？是否真的能拥有那么多贵族，一天到晚看戏玩耍？这取决于它的生产力水平，而它的生产力水平比较存疑。所以它所说的那些伟大而辉煌的古罗马，其伟大与辉煌是高度存疑的。但存不存在一个古罗马地方？有可能，万一呢，起个名嘛，大差不差的，地上总是住人的，没什么问题。

如果他们都是光膀子的，甘英应该会说明。但甘英没碰见他们，没见着。甘英见着了一片海，说那片海变幻莫测——这种大风、季风、海浪的描述高度疑似地中海，因为地中海就是这个样子。但因为没过去，所以不清楚。理论上被他们划出来的环地中海的这片势力，可能没那么大，万一海对面有一个小城或一个小村叫罗马，也有可能。嗯，他说他到的是波斯湾？对，甘英到的地方有几种推测，认为波斯湾的也有。但具体是不是，都是推理，包括地中海和波斯湾。哪一种更合理？按当时描述，波斯湾的气候条件远远没有地中海那么恶劣。而甘英之所以没渡海，就是因为海上气候极为恶劣。按他所描述的海洋状况，地中海概率更大。

刚才讲到地中海气候的时候更好玩了。我今天还看到了，他们坦然承认古希腊的粮食不够。谁换谁说也一样——古希腊核心地区粮食肯定不够。所以必须打补丁。现在有几种补丁：一种补丁是，古希腊不是那么一小块地方，还包括周边各种城邦，是一个用殖民统治方式统治了相当大领域的帝国。第一个补丁说，所谓古希腊不是希腊那块地方，而是涉及周边丰饶的土地，都是它的殖民地。这事儿就很好玩、很幽默。我就纳了闷了：人家丰饶的土地能种粮食、能养人，人口多，技术发展速度肯定比山旮旯里要快，它凭什么把人家当殖民地？这个事儿不好解释。比如中国这里，因为农耕水平高，生产力进阶速度快，人口也多，有了这些基础，文明指数就一步一个台阶叠上去了。我们有没有被他人劫掠成功过？有，比如内部组织力崩坏时，周边的游牧群体就会静观劫掠。但劫掠的群体普遍是马帮、游牧，他们地势较高、平坦，能放牧牛羊、驯养马匹，有较好的机动性，相当于人均拥有好几辆小汽车。因此他们能在组织力不全时劫掠成功。但希腊那个地方不适合养马，可以理解吧？它不适合养马，又不适合种田，哪怕周边确实有丰饶地区，它拿什么抢？凭什么劫掠成功去殖民他人？有人说它适合造船，离海近。但造船需要木头，也需要技术。除非是刨独木舟——独木舟是部落民就能做的。但海上航行、海战用的船，技术要求就高了，必须有足够粮食才能拥有脱产者研究技术、伐木、生产海船，然后才能出海殖民。能不能殖民先不谈，前面半截儿它吃什么呢？所以，在这套补丁中，它的基本物质条件不足，就不得不为粮食寻求来源。

现在它打了几个补丁：一个是它可以殖民周边地区。那就纳了闷了——它周边地区第一粮食产量不是很好（当然比它强一点），第二那地方很适合养马。路上打马战，你真的很难打过人家。它附近号称被它殖民的地区是适合养马的，你到底谁打谁？这很难说。另一个补丁是所谓地中海贸易：虽然本地不产粮食，但可以通过地中海贸易。刚才我们不是说甘英遇到一片海，因为变幻莫测而不敢出海？那个变幻莫测的海大概率就在那个地区，最变幻莫测的就是地中海。大家可以自己看地中海气候，那里的海洋条件、水文条件非常恶劣。近海捕鱼问题不大，但海上航行风险极大。在这种条件下搞跨海贸易？我们这些种田的人确实理解不了——人家在粮食都不够吃的情况下，怎么还有条件做海洋贸易？就好像海洋贸易是吹口气就能成的样子。听过一个说法，说一个木桶就能渡海？独木舟是可以渡海的，但风险非常大。通常人类早期文明部落期，会用独木舟渡海迁徙，这种迁徙是客观存在的，很多海岛部落就是这样的。但第一，岛和岛之间距离不能太远，不然支撑不住；第二中间风险巨大，折损量大。但因为是部落民，可能不算计得那么清楚，有成功的概率，也确实存在。不过，第一它不可能带大量货物，能带上一个人自己的食用和水就不错了，容量很有限；第二风险巨大。这种情况下，如果部落人太多、不出去就养不活，冒着风险出去找另一个小岛迁徙，是正常的。但你说“我们这儿要做买卖，跟人搞贸易，然后我用这个船”——那不行，这种船干不了买卖。
