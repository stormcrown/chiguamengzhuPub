# 第156章  科学尽头是神学？那些歪屁股通史可以看吗？ #冷知识  #历史真相  #人物故事  #科普辟谣

科学的尽头真的是神学吗？少看点营销号，少看一点西方人物传记。这两样东西你回避了，就不容易看到这句话。他们特别喜欢说“科学的尽头是神学”。是谁在推这些东西？一个是国内的各种营销号，专门抓人眼球，用截选的方式找爆点——断章取义、移花接木、瞎编乱造，制造心理冲突，让你的小心肝扑腾扑腾跳。另一种场合是西方人物传记。不管那个人多有名，什么科学史上的扛把子，你要是不了解他，就好像错过了半个科技史。这种西方人物传记，有一个算一个，别看就对了。如果你非要看，就得挑拣，要能识别他这一长篇到底在干什么。做到了这个，你可以敞开来看，无所谓的。

为什么西方人物传记容易让你想到“科学的尽头是神学”？他们一般论证的方式就是“你看那些大科学家晚年都信了神”。为什么很多西方古典时期的顶级科学家晚年开始信神？因为他们的青年时期、求学的时期，学的就是神学。他们原本就是神棍，原本就是信徒，只是中间短暂当了20年的科学家，其他时间都是信神的。你看那些科学家，他们出生的阶段都是在神学氛围中成长的。那些号称什么大学奠基人、顶级校友的大学，在进入科学时代之前，全都是神学院编神话故事的。所以在这种环境下成长的人，当科学领域没什么可研究时，或者研究成果不知怎么研究出来时，晚年信神，这叫做回归老本行。

看书时，看人物传记一定要记得：尽管它表现得很像客观记录了一个人的成长过程，甚至写了很多心路历程，像从日记本直接转译过来的，但不保证真实。如果是你给自己写传记，你会不会把丢脸的事情写上去？除非最后取得了大成就，让丢脸变成笑谈，否则你不会写。所以人物传记、笔记、日记不能完全信以为真，里面掺水很常见。连史官记录的档案都不排除注水和篡改，所以才需要交叉印证。

有人看我视频不舒服，这很正常。互联网上恨我的人乌泱乌泱有如过江之鲫，很正常合理。我完全支持你们，看我烦人没有毛病，看我讨厌很合理。所以拉黑不就完事了吗？你不会吗？我也不会，我特别不擅长。你可以问别人，或者问AI，它会告诉你如何拉黑一个不顺眼的主播。这就是付费啊，不好意思害你破费了。

有朋友问：现在发行的通史美化的不少，该不该继续看？怎么看？通史毫无疑问可以看。我家有好几本中国通史，不同学者的论述，可以看。怎么看？第一，你是不是学生？如果是学生，老师让你看哪段就看哪段，否则容易干扰考试。考试不讲究学术自由，因为有标准答案。如果你不是学生，课本里的内容就没有绝对意义，因为学术是自由的，允许讨论。历史本身就是存在争议的，谁都没有监控录像，写通史的史学家也没有。他也是在浩如烟海的史籍中筛选出他认为合宜的内容发表。所以任何一个作者的著作都有自身立场和看法。有没有美化？有，不光有美化，还有丑化，只是分不同角色，取决于著书人的立场和抉择进行的定向描述。但我可以保证，所有写通史的作者，里面写的内容基本上都是真的，含金量很高。之所以不同通史有差异，有的让你不舒服，原因是作者的立场和筛选角度有问题，而不是具体哪句话有问题。为什么他非得在这个地方筛选这个史料作为实锤，而不是用另一条？就像我们昨天，明明在我们的史料中，郑和是一个佛教徒的证据占到99%，但他在写书时一定要用那1%受质疑的证据，说他是一个回教徒。这就是选择的问题。所以你说他说的这个事有没有道理、有没有源流？有源流，但他是有选择的。今天有些书让你不舒服，原因也在于此——事是不是那个事，有，但他说的视角、角度、分析方法和节选史料的段落，有他自己的主观目的性。如何解决？第一可以多看几本，参照交叉印证。如果没有这个精力只能泛泛而看，不舒服的地方你优先怀疑他，不要怀疑自己。

直播不能说钱谦益的事吗？能，咋不能？他早年爱国，后来降清，再后来又暗地里资助各种反清运动。这就是他跌宕起伏的一生。这个人属于典型的那种有一点骨头但不太多的类型——知道应该怎样做才对，但同时也知道这样做要付出代价。他明知道要做正确的事，但正确的事要付出代价，他对代价有掂量：让他出钱，他乐意；让他出命，他不肯。所以在遇到需要殉国时，他水太凉；站定安稳之后，保证了自己活命的前提之下，就开始资助反清运动。这件事于史有据。他晚年一直资助反清，包括朱成功（国姓爷）那边也拿到他的赞助，这也是毫无疑问的。他青年时期也确实写过书哀叹江山易主。所以这个人身上有多重属性：好事干过，坏事也干过。如果你觉得因为他降过清，所以晚年资助反清就叫做反清不彻底，相当于彻底不反清，那标准有点过高了。文人的臭毛病就是有点软弱，但又有一点是非观，在那个时代把自己玩得比较夹生。

吃东西好开心啊。有朋友要买除螨仪找不着链接。除螨仪我放得很靠前的。给你们弹出来：除螨仪，顾名思义，除螨的。有望的除螨仪品质很不错很过硬。经常拿它吸一吸床铺、枕头、沙发、布面椅子，能去除上面的粉尘、碎末和螨虫。整体卫生条件会提高。螨虫这种小寄生虫既无能又怕光，但它的分泌物、代谢物会对人造成危害，所以还是需要处置一下。能待在干净的环境里是更好的。
