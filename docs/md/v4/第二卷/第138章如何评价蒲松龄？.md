# 第138章  如何评价蒲松龄？

什么？如何看待蒲松龄？  
我能如何看呢？  
他能够在清（朝）这样的时代去书写这样一套书出来，是担了莫大的风险的。你以为他在写志怪故事，但是大家注意到没有？你读《水浒》的时候是不是还蛮顺畅的啊？我们在座的小伙伴们，很多都读过这个什么《水浒》《三国》《西游》啊，这些呢，其实都是明代作品，包括《红楼》也是明代作品——它只是成书啊，成书有一部分在清初，但是人全部都是明代的人。然后你会注意到这个现象：它们基本上是白话文作品，你读起来很简单、很流利啊，非常流利。但是你去读读《聊斋》的时候就会有点吃力，去读《聊斋》的时候你一定有点吃力，为什么呢？它基本上是文言。为什么这个样子？要知道在那之前，明代的时候，小说已经基本上都已经是白话文了，非常非常普及啊。那个明代的小说基本都是白话文，实际上小说这种体裁自古就有啊。先秦时期的时候，还有专门的小说家，就是说书的。那个时候的小说呢，是拿来讲段子的。后来，到了唐代的时候，也有一个小说的兴盛期，那个时代呢，就基本上都是文言的小故事，各种的志怪故事啊，在那个唐代的时候是很流行的，这也属于是小说的一种品类。到了明代的时候，这个小说的类型啊，就变得非常非常丰富，和我们今天的网文的爆炸程度是差不多的——什么讲情情爱爱的啊，讲那些阴谋诡诈的啊，讲修仙的、讲经商的全有，这和你们今天跑去看什么男频、女频是一回事儿。这种爆炸式的小说的井喷，它伴随的是什么呢？就是基本上的全民识字。明代啊，明代是有强迫教育的，它是规定了强行规定你必须去念书，必须要认字啊。那个他这个认字的普及程度，就是你能能写会算，但是呢，之乎者也的，真正的经史典籍是没有人给你教的啊，不会给你好好地教什么经史，但是呢，能写会算的基本功能是存在的。于是人类拥有了这个能力之后，就相当于说9年义务毕业了，但是高中没考上。那这个时候你的日常的消遣，就完全可以靠网文来刷时间了。对，明代的人就是这个水平，所以他才会出现了通俗小说、白话小说的一次井喷啊。同时呢，明代由于物产较为丰富，出现了大规模的这种出版的制版行业，图书的出版发行行业非常的发达了之后，使得每一本书的单本的售价降低了，这样子呢，一般人也能够负担得起，甚至能出现图书租赁行业。大家有没有记得那个二三十年前的时候啊，大约是20几年前的时候，很多的学校的附近都有专门租书的摊子，明代也有啊！明代还有租服装的店，专业租赁服装的店啊，这些店都是有的。这个呢，都是一般的市井小民做生活的时候很重要的，包括了消遣，也包括了仪式感，你所有美好的生活其实都在这个系统当中了。那在这里面我们就能看到一个侧面：那个时代，他的这些作品是给普通百姓去看的，因此呢，它数量级非常的大，用语非常的朴实、接地气。同时啊，他就不会再使用那么文言什么的，用点什么啊，他尽量的就会避开文言这一套啊。但是呢，到了清的时候急转直下啊。蒲松龄已经是非常有志向的一个人了，他致力于要写一套有警示功能的书，但是他为什么最后写了一个志怪故事？因为这个时候你讲人风险是很大的啊，你讲事儿风险也是很大的，那能怎么办呢？讲鬼、讲传说、讲妖精。所以他的书为什么写成了一个志怪，是因为他不写志怪的话，他很容易人就没了啊。于是呢，后面啊，后面所出现的场景，就是你会发现他用文言来写，为什么要用文言来写？其实文言文它本身相对来说呢，更加的精准，也更加的浓缩，它会造成一个效果啊，就是它在制版上面成本要低廉很多。一本书本来呢，它用白话文需要写这么厚，但是用文言一根指头那么厚度就够了，就是能够浓缩到这样的程度，所以它会降低成本，降低传播成本。然后同时啊，同时还有另外的一个问题，就是你在明代之所以写白话，是为了让所有人能听懂，但是到了清代的时候，普通人家里面根本就不敢收藏有文字的东西，因为会被查抄。所以这个时候他要怎么办？所有的这些书只有可能在文化人的内部周转的时候，写白话就是降低对方的阅读体感。真正的文化阶层啊，他受到的文言的教育是非常多的，让他去读白话的话，他会觉得浪费了很多时间，因为字太多了、太长了，太啰嗦了。他的书籍书写的对象也发生了改变，然后呢，印刷的成本也发生了改变，同时呢，它面临到的风险又指数级的提高。所以他为什么要写志怪故事，就在这个地方。所以呢，你就会发现在清代的这些小说啊，清代的小说当中呢，就开始属于是倒行逆施式的，开始出现了文言化的倾向。嗯，原来《聊斋》不是YY小说。当然不是啊，它里面是包含了很多隐喻的啊，它里面是包含了很多的隐喻的。你一定要注意到一件事情，就是在《聊斋》里面啊，它有的时候会讲人的伦理道德，《聊斋》里面有这样的故事，有的时候呢，是讲那个什么母子之间的这样的故事啊，强调里面的人心，强调里面的道德。我们今天看起来的时候会去想，唉呀，那个时候的人如何如何，你只会这样揣摩。但是对于清代的人来说，他的感知是完全不一样的，因为清的那个时候，伦理的规则、秩序其实已经开始发生转向了。那就是在明代和以前的时代里面，中国人其实是一个很自由、很洒脱的群体，非常自由和洒脱，并不是什么僵化、教条型的啊，就是所谓的什么啊，封建的、保守的。没有啊，明代及以前的中国人是一个非常潇洒的群体，洒脱自在。谁要得到最高的这种气质上的点评啊？气质上面的至高点评叫什么？叫做“风流”啊，这个词大家总是听过的，风流啊。风流讲的是一个人的姿态、气度和气质，什么样的气质呢？如风一般的、如水一般的会流淌，这是什么？这是潇洒啊。这种潇洒的感觉不是浪荡啊，是潇洒自在，能够脱开樊篱，就如同一个自由自在的鸟儿一样的在飞行，可以御空；就如同一条鱼在水里面噗溜噗溜的就钻走了，可以控水，这种姿态叫做风流。这是对一个人气质和他的行为风范上面非常高的一个点评。所以中国人追求什么？中国人追求的是这种自在洒脱感。但是到清的时候变成什么了啊？大家在清代的时候至高点评是什么？讲规矩啊。清对一个人高级的点评叫做“讲规矩”。所以现在，现在还有很多人在那里吹嘘什么啊，现代人还有吹嘘说清代的那些什么贵族礼仪，讲老理儿，嗯是吧，讲老理儿有规则有规矩干嘛呢，啊，就这么回事啊。所以中国人喜欢什么呢？中国人并不是好这一口的。有的时候有一些人行礼不到位，说话的时候说得不体贴，但是呢，他意态潇洒，就会被人家点评为有风流态度，大家对这样的人是非常的宽容的。但是到清的时候不行，为什么？因为清是奴隶制，奴隶制一定要谨守主奴分野，但凡你一个地方不周到，你就是有可能要叛逃的意思。所以奴隶制的社会内部，是不允许一个人追求这种自在洒脱的。所以今日啊，大家所想象的中国的传统，什么中国的规矩，中国的老理儿，大部分是清代留下的刻板印象。其他的时候有没有规则？有规则，但是很少有那么严苛，甚至说有一些规则你逾越了的时候，如果有理由，旁边人还要给你鼓掌，这个才是真正的中国人的态度啊。
