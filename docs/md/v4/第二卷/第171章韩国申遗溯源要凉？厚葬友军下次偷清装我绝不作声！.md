# 第171章  韩国申遗溯源要凉？厚葬友军下次偷清装我绝不作声！ #韩国 #非遗 #溯源 #联合国

现在是申遗的主席国，要求思密达拿出证据，证明他所申遗的端午祭、榫卯和泡菜起源等五个项目。最终的大结局还没有，最终的文件还没出来，对吧？只是进行了最后一次举证，对吧？最终的还没出，应该是这样吧？我看的应该是这样啊。

反正呢，思密达这样的角色，其实我一直都是觉得他无所谓的。甚至有的时候呢，我觉得他们在那里蹦蹦哒哒，蹦蹦哒哒说“这也是我的，那也是我的”，这件事对中国还是很有好处的。这是认真的说话啊，这事一点都不带含糊的，真的有这事啊。思密达当年不停地说“这也是我的，那也是我的”，这件事对我们是大有好处的。

嗯，我们很多传统上面非常重要的一些节日、手工艺等等这些东西，在之前呢，很多时候没有获得足够的尊重和保护，甚至有的时候受到商业化的侵蚀，非常严重。而且呢，在一些地方上，还不排除有一些传媒势力的群体，在对我们的传统形式进行各种各样的口诛笔伐，把它视之为落后的糟粕，什么封建迷信的东西，各种各样的词儿都往上砸。但是就是因为有思密达持之以恒、坚持不懈地在不停地强调“这个是我的思密达，那个是我的思密达”，以至于啊，我们国内都已经开始进行了深度的保护了。在这个方面上，我觉得思密达方面对我们是颇有一些贡献的。而且呢，它对我们的伤害值其实非常低啊，无论它去侵占了什么玩意儿，对我们的伤害值都非常低，因为它实际上确确实实自古以来就是我们的藩属国。它的这些习俗也好，风尚也好，衣着也好，建筑也好，玩儿的东西也好，都是极大地受到了我们的影响的。

所以呢，当它不断地强调“这个是我的，那个是我的”，然后呢，它就会在自己本国内部不断地推广。而它在推广这些东西的过程当中，就会发现一个大结局的效果：它越推广，就越像我们啊，最后它连自己的本土特色都快消失了啊。这个也是他们一些思密达那边的老年人、中老年人比较忧虑的事情，就是现在呢，它已经越来越失去自己独特的、传统的那些微妙的东西了，因为它学得越来越多，以至于啊，变得跟中国越来越像啊，非常非常非常主动地自主汉化啊，咔嚓咔嚓地就变了样。

它的电视剧对，你看嘛，嗯，要知道从2003年的时候啊，2003年的时候，王乐天走上街头，然后呢，汉服复兴运动开始出现。从那一刻开始，斗争就开始了，而且呢，从一开始斗争就是白热化的。有经历过那个时代的人应该是有所体会啊，什么叫做白热化？就是所有的媒体众口一词，这个东西叫做封建复辟啊，然后说它丑，说它这个东西是一个死掉的，什么刨了坟啊什么的。差评如潮啊，千夫所指啊，认认真真所面临的局面就是这样的。然后去混淆汉服与和服，然后挑动人跑到街上去攻击穿汉服的人，这些事情都是当年在10年前后的时候频繁发生的事件，这种事始终都有。你说背后没有推手，那才是奇了怪了呢。一件事情出现了，你正常地去推广，正常地进行新闻推送，绝对不会出现这样的局面，但是那个时代就是这个样子的。我们要恢复自己的传统衣冠，步履维艰，每一步都是障碍，每一步都是坎儿，处处都是阻碍，随时随地会遇见敌人，甚至它会伪装成友军，然后在里面跳反，这都变成了常态的现象了。

但是后来这个屏障是怎么突破的？因为思密达说“这玩意儿是我的思密达”。自从思密达喊号说“啊，这东西是我的思密达啊”，然后呢，就越来越多的人开始伸直起了腰杆子，开始强调说这个东西是汉服啊。你以为没有助推的力量吗？有的啊。所以我从来认为思密达，不管它是真心还是假意，它事实上在我们的传统文化复兴领域当中，它就是一个友军啊，你放到哪它都是一个友军。包括说那个什么，嗯，之前马面裙出圈这件事情，思密达也是有功的。迪奥当年所声称的是借鉴了思密达的版型，你知道吗？你以为思密达在里面没有做手脚吗？有的呀。它想借由这样的国际大牌推行，说这玩意儿就是它的思密达，结果造成了马面裙的出圈。嗯，这样一说都不好意思骂它了。该骂骂，不要耽误啊，骂它也是它做贡献的一部分。不然呢，它每天跳吧跳吧的，然后大家一骂，这个事儿就办成了。所以我说它是友军，但是你要是不骂它，这个事儿呢，就没有达成友军的效果啊。所以呢，正常的该喷的喷，该骂的骂啊，该跳脚的跳脚，该急的急。
