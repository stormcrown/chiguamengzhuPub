# 第185章  付费提问之谈谈“黑人混血” #黑人 #混血

谈一谈广州的混血影响。几件事情啊。嗯，你认为自己对混血儿的感觉是什么？你觉得他们好看吗？嗯，觉得混血儿很好看的扣个一，真是什么数都有。哈哈哈哈哈哈，你们都是什么人呀？哈哈哈。

啊，几件事情就是你要相信一件事：大部分的中国人是有自己的要求的。而且呢，有没有听说过一个词叫“东亚怪物房”？这个事儿大家应该都听说过，叫“东亚怪物房”。指的是什么呢？就是东亚三国手牵手要卷的时候，往死里头卷，卷死了。有的时候你都痛恨这种卷，是吧？啊，大家都不喜欢卷，卷了之后太难受了，太累了，累死我了，晚上都不敢睡觉了。但是注意一件事儿，东亚怪物房里混血是很难活下来的。嗯，在东亚怪物房这样的地方，混血的生计十分堪忧，很难长时间存续下去。为什么？因为竞争太激烈了啊。我们这种往死里头卷的地方，对于混血其实是很不友好的。

认真一点地说，这几年或者这几十年，这个地方能够有混血儿生活条件还不错，其实是因为逆向民族主义思维。这是一个很本质的现象啊。就是所谓的什么“你看他长得那个样子，所以它好看”，这个是逆向民族主义思维。这是这些年以来很多的人，包括媒体，也包括教育，包括各色各样的传闻、小道消息，共同制造了一个庞大的信息茧房，以至于让中国人幻想着混血又聪明又漂亮，这个谎言啊。这个很明显是一个违背科学基本知识的谎言。这是一种传媒信息流在掌控大家的想法，以至于呢，你看见了混血的时候，有人会专门给他提供一个工作的机会，因为就会有人认这一套。包括说很多的那个什么外教岗位，甚至是专门的非法从业的外教岗位，只要有个外国人脸，都不管人家母语是啥的，家长就把孩子送过去念英文去了。这是什么？这是逆向民族主义思维。而这种思维随着国力的不断增强，它会慢慢减弱啊。当这种额外的，或者说这种异常的思想、完全违背人类正常思维逻辑的思想消失的时候，当中国人变成正常的中国人，而不是自我否定的中国人的时候，咱们就认认真真地卷起来了啊。这种卷里头各凭本事。

谢谢吴伟平。如果说混血在这种竞争当中真的有优势，那就活该他能够好好地繁衍生息下去。但是大概率是做不到的。为什么？因为我们历史可以见证这一点。我们的历史进程当中，你说有没有出现过混血？一定有，而且数量不会很少的，真的数量不会很少的。我们在不停地扩张的过程当中，就一定会出现大量的混血。但是为什么今日之中国的后裔，基因的这种纯度会特别高？不需要欺负你，就正常的竞争就行了，你只要正常的竞争，这件事情就过去了，知道吧。

那啥气味问题是吧？我们以前啊，有一些地方有一种传统，分享一下。有一些地方有一种传统，就是在双方谈婚论嫁的时候，会让姑娘到婆母家里面和婆婆住一晚上。有一些地方在双方相亲谈婚论嫁的时候，会邀请这个女孩子到婆婆那里跟婆婆共寝住一天。为什么要这样做啊？是去找那个有没有狐臭啊。啊，有一些地方是这个样子的，就是找狐臭。你用什么东西隐藏都藏不住，因为婆母跟你睡一晚上，就保证知道有没有狐臭啊。啥，第一次听说啊？回家里面打听打听啊，往早两辈人之前去打听打听啊。现在肯定没这事了，以前你往早两辈人之前去打听打听，这个不是非常非常局部地区的那种风俗啊，还有不少地方就好这口呢。嗯，就是大家在婚嫁的过程当中，狐臭绝对是一个减分项啊。在所有的地方都这样，中国这里狐臭是一个绝对的减分项，这就会导致他的后代数量减少，这是毫无疑问的嘛。所以为什么最后竞争着、竞争着，然后你发现，哎，不对呀，怎么那种样子的没了呢？哎，就是这样的缘故啊。

啊，我看看啊，对，到点了啊，再见啊。到点了，到点了。我的光标又去哪了？
