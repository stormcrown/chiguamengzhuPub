# 第119章  “痴”瓜蒙主

我反反复复地要跟大家说一件事情：不要弄得好像我这个地方风萧萧兮易水寒，你们时刻都要送壮士兮不复还啊。没到这个份儿上，为什么要这样说？如今现在此刻，是中华自1644年以降最好的时刻，没有之一啊。啊？啥？你们，哎，你们就天天跟我哈哈哈哈哈哈……不然呢？啊？啥？听说我又被抓了啊？对对对，啊，我应该一年往少了说，被抓个百八十回吧。满纸荒唐言，一把辛酸泪啊。随缘做着痴，嗯，谁还没点痴性呢？说真的，在我们的传统文化当中，“痴”就是痴傻的那个“痴”。我们这个“痴”这个字儿并不是一个贬义词，很多时候“痴”是正面的意义。这个事儿，这个事儿确实可以聊一聊啊。啥？吃瓜？哈哈哈哈哈哈哈，你们又给我改了是吧？好的好的啊。啊，实际上就是这样：我们，我们的古人的概念当中，“痴”是什么“痴”呢？就是这个人有执念，有在乎的东西，有在意的东西，并且很专一的、很执着地在意这件事，很执着。

其实啊，我，我们在不同的、不同的思想体系当中，有不同的判断啊。就是对执着这件事，在佛学当中要求大家要放下贪嗔痴啊，会很强调放下痴、执着、执念。但是呢，在世俗文化当中却会奖励“痴”这个行为。我们会认为友情痴的那种情痴，尽管他可能最后吃亏了，但是他却是令人赞叹、令人叹惋的一种行径。舞痴也许日子过得不好，但是也是值得赞誉的一个行为。为什么？因为这个视角不同。我们的这个视角，当它放在一个非常世俗成就的视角上面的时候，放在一个较为唯物的视角上面的时候，你就知道一件事：“痴”不是坏事。你有执念，有要的东西，有在乎的东西，并且拼命地去追逐它，就有更大的概率取得成就。这个是一个世俗追求者很需要的要素。

其实认真地说起来，这几十年来的中国难道不痴吗？我们何其的痴！近几十年来的中国，是非常非常有“痴”这个念头的。当年全世界都在告诉我们：中国的文化不行，中国的人不行。中国人不肯认这个账，我们不认输，我们非得就觉得我得争一争，我得做一做。凭什么要说我不行呢？就算不行，我也得先试一试。看我什么都没有学，我仿着你来，我就是装，我也要装那个样子。这个行为是什么？这就是痴。我们用这样的一个执念挣扎了几十年，最终缔造了这个全世界最了不起的全产业链的工业国。我们拥有了所有的产线，我们想要做什么就能有什么。我们想到了什么，我就把它弄出来。你画PPT，我给你做成真的；你做动画，我给你搞成现实世界的。这个东西就是痴念。我们这些年的成就靠的不是放下痴念，我们靠的就是执着、专一的执着。我们就是想成就，这个成就就是很世俗，就是不蒙骗自己。不会说我成了就我成了，而是我得看见我成了。我定义一个指标，我要成为什么样子，哪怕这个指标是别人骗我的，我就信了，我就去做。这就和你们在武侠小说当中看到的那种大侠是一样的：别人都说这个秘籍是假的，练不成，但是他不信，他就觉得是真的，一定可以，最后他练出来了。这个，这个东西就叫做痴。中国人身上是很有痴这个劲头的，所以它不是一个贬义词。也可能最后挂了，对呀，那又怎样呢？我们选了自己的目标，执着于自己的目标，最后做到了自己的目标。至于最后看到目标之后发现它也没有那么好，认了，我们接受这个现实。

小蛮夷那些反动分子们，他们只做一件事情就行了：他们把所有的力量、所有的财力、所有的精神都放到了为满清洗地翻账这件事情上面。而我们的普通人只能够被动防御，因为我们要干的事情太多了：这个国家的生产建设离不开，你们孩子们要成长离不开家长的陪伴，整个社会的进步离不开在座的每一个人。大家都是很忙的，都是有自己的事业，都是有自己的梦想和目标的，每一个人都在为我们的建设添砖加瓦的。但是小蚂蚁、小青粉那些人，他们的目标就很单一：他只要拆掉我们的认知的根基，去把他们家的那个主子、奴才的那一套灌输进来就行了。就去好了。事情是客观存在，可是对于我们这些人来说，我们只能见招拆招，因为还没有到我们反攻回去的时间。什么时候？是等我们掌握了更多的话语权的时候，它就会变得非常重要了。这就是主要矛盾与次要矛盾的转化。现阶段他已经暴露出来了啊，让很多人产生痛苦了，所以开始运动，开始改变格局，但是还没有到他成为主要的矛盾，必须立刻一波冲倒的这样的程度。所以呢，就是徐徐图之：他做了什么，我们就拆什么啊。因为我们的主要关注点不在这儿，所以他也没有办法很快地、迅速地、普遍性地把它推翻。但是呢，我们有一份力出一份力，有一点火就发一点光，努力地让这个局面不至于非常的恶劣。等到我们终于开始吹响集结号的时候，我们就可以团起来，等到下一次冲锋的时候，我们就可以跟着队伍一起冲锋了。念念不忘，必有回响。说得好啊！

嗯，哎，好，欢迎大家来到直播间啊。嗯，什么？这里是一个，这里是一个带货直播间啊，不要闹岔了啊，不要闹岔了。嗯，啥？你还真带货呀？还以为是文化主播？这不是误会了吗？啊，这误会了啊。误会解释两下啊：我这里是一个正规的带货直播间，字正腔圆吧？啊，我这里是一个正规的带货直播间。我这里甚至可以大声地说：我卖的是酒，而不是像其他直播间跟你说要8+1。为什么呢？因为我考了卖酒许可证，所以我是持证卖酒，亲。我不光是一个正经的带货直播间，而且是一个持证带酒直播间，棒棒哒呀亲！所以那个啥不要误解啊，我这里是一个，我这里是一个正规的、专业的带货直播间。嗯嗯，啥啥啥啊？所以其他的那些什么聊天的项目那叫什么？那叫做赠品啊！我这个直播间就主打一个赠品多，赠品特别多的那种啊。嗯呵呵呵呵呵呵呵，嗯嗯，好的好的好的啊。

什么？应该用红楼史官修清史？清史的话最好暂时修不出来啊。我反复强调这件事情：清史最好近些年别修出来。清史什么时候应该修出来呢？清史，嗯，应该是在我们真正回到历史的巅峰啊，真正回到历史的巅峰，并且拿回了所有的学术和文化方面的话语权之后，我们再去修清史。清史不是一个随随便便拿出来修的。为什么这个样子？因为立场问题啊，因为立场问题。你站在哪个角度上面去看待这段历史，太重要了。这些年来有没有民间的这些清史相关的著作？有的呀，量很大呀。之前被扇巴掌的那个，那不就是清史的专业研究学者吗？姓严的那个，被人家当众地呼了巴掌，还把人送老李的那个啊，就是那位学者。他就是研究清史的，就是给满清的皇帝拍马屁的，认为满清十二帝个个是明君的那一个啊。这种学者乌泱乌泱的老多了，雇佣的到处都是，那小翅膀扇的嗡嗡的。这么大体量的群体，作为专业学者甚至是头牌学者的时候，你觉得修编这套历史他们会不参与吗？只要他们参与了，你猜会修成什么？那他为什么会不停地拍马屁？为什么他写的这种历史书，会让别人忍不住上去呼他一巴掌呢？就是立场的问题。你说事儿是不是那个事儿？大差不差，总是那个事儿。研究历史的专业学者，总不至于在事件上面给你描述得差太远，当然也会有一些创作啊，但是不会差太远。问题就在于视角。而对于满清这个整个的历史段落而言，你眼睛放在哪一边，屁股放在哪一边，你按照什么方式来描述它？你认为康雍乾是不是盛世，是不是明君，这完全取决于你是谁。而当中国人在写历史书的时候，还不能够决定“我必须是谁”的时候，就书咋写？没法写啊。什么啊？读教材为什么用了那么多年？因为有一些人呢，确确实实在这个事情上面下了大力气。而我们自己呢，我们自身的自己人，其实量是最大的啊。大家一定要相信，我之前说了：今日的中国是自1644年以来最好的局面，没有之一。这句话本身是诚恳的。但是啊，咱们得知道，我们这些正经人，这些平凡的人，这些普通的人，我们要干活的，要吃饭的，要做很多正经事的，我们要忙的事情很多很多很多的。但是那些小青粉、小蛮夷，那些反动分子们，他们只做一件事情就行了，这个东西也叫做痴：他们把所有的力量、所有的财力、所有的精神都放到了为满清洗地翻账这件事情上面。而我们的普通人只能够被动防御，因为我们要干的事情太多了：这个国家的生产建设离不开，你们孩子们要成长离不开家长的陪伴，整个社会的进步离不开在座的每一个人。大家都是很忙的，都是有自己的事业，都是有自己的梦想和目标的，每一个人都在为我们的建设添砖加瓦的。但是小蚂蚁、小青粉那些人，他的目标就很单一：他只要拆掉我们的认知的根基，去把他们家的那个主子、奴才的那一套灌输进来就行了。就句号了。所以他会花全部的精力去做这些反动的事情，所以你才会觉得他声音特别大。但实际上，在真正的体量与实力上面，我们的力量才是大的，他们的力量是弱的。可是呢，他们表现出来的影响力却会很强，是因为他们的压强特别强，他们的所有力量都用在了一个点上面。

就像今天看的那个图片啊，司马光砸缸，大家看了吧？司马光砸缸的雕像啊，长什么样子呢？猴儿。司马光砸缸是一个典型的正面故事，正面寓言故事，但是里面的角色设计成了一个猴儿。你觉得他是不是故意的？必然是故意的。相信我，艺术不是拿来闹着玩儿的，艺术是可以拿来直接影响人的感知，影响人的心灵，影响人的情绪的。它是有作用的，它是可以当做武器使用的东西。所以你说它只是一个艺术品？不，艺术品的美与丑，会影响到大众的心理认知。当我们的艺术品上面，我们的先贤（例如前些年爆出来的老子像啊）、我们寓言故事当中正面的角色（例如司马光），都被塑造成了丑陋的、愚笨的、甚至非人类的形象，那么我们对于自己的先民的认知就会发生错乱。这不是什么“小孩子不懂，只是可可爱爱”不存在这个问题啊。所以呢，事情是客观存在。可是对于我们这些人来说，我们只能见招拆招，因为还没有到我们反攻回去的时间。什么时候？是等我们掌握了更多的话语权的时候，它就会变得非常重要了。这就是主要矛盾与次要矛盾的转化。现阶段他已经暴露出来了啊，让很多人产生痛苦了，所以开始运动，开始改变格局，但是还没有到他成为主要的矛盾，必须立刻一波冲倒的这样的程度，还没有到这个时间段啊。所以呢，就是徐徐图之：他做了什么，我们就拆什么啊。
