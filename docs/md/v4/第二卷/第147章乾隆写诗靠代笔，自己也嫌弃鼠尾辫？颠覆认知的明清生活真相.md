# 第147章  乾隆写诗靠代笔，自己也嫌弃鼠尾辫？ 颠覆认知的明清生活真相

有个同事对你挺照顾，但他自称祖上是旗人，应该保持距离吗？嗯，祖上是旗人的人肯定有很多。那么请问他在讲自己祖上是旗人这件事的时候，是很炫耀、很自豪地说，还是希望你认为“100年前得给他磕头”？这两种表达方式区别可大了。如果客观事实是他祖上是旗人，但他今天认为自己只是个普通公民，那些往事都随风而去了，那这就是一个可以正常往来的朋友。但如果他说“我家祖上是旗人，搁100年前你得给我磕头，您得称我为格格，我也妥妥的是个阿哥，你要听我们家的老理儿，要如何如何，我头上是有通天纹儿的，你是个臭要饭的”——那样的人，不管他对你关不关照，你是不是都得有点防备之心？这已经不是朋友的问题了，这是立场的问题。

乾隆是个别字先生，别字都不是什么大事。乾隆少年时期做皇子时留下了很多手稿，你们知道的，他喜欢作诗，除了作诗还喜欢作画，是个非常附庸风雅的人。作为一个君王，在艺术上果然没什么造诣，我们只能说他附庸风雅，尤其是当他的审美非常糟心的时候，就更要这么说。但他在附庸风雅这件事上却一骑绝尘：留下了大量手稿、底稿，包括他作为皇子在书房读书时留下的文字版作业，现在也有一部分手稿留存。里面经常能看到老师的点评，就是“这字写得真没意思”。简单来说，就是字写得不行。他那首“一片两片三四片”的诗，后面两句是朝臣帮他续的，不是他自己写的。在他上万首诗当中，“一片两片三四片”已经算拿得出手的了。怎么说呢，这个打油诗系列里，好多可能都比他写得好。比如张打油当年的打油诗：“天上一笼统，井上黑窟窿。黄狗身上白，白狗身上肿。”就拿这首著名的打油诗来说，张打油的写诗水平还不如这个呢。“一片两片”那首已经是他作品里相当不错的一首了，能上得了台面，还颇有点趣味。但然而可是，这首诗后两句是朝臣帮他续的，不是他自己写的。嗯，什么玩意儿，跟钱钱有得一拼。这还不能直接比，把张总和浅浅放到一起，一下子把我整尴尬了。我该说谁好、说谁赖呢？不能评，这俩人都不好意思评，不能讲不能讲。嗯，好的好的。

亲皇帝的画像上能看到大块的鬓角，你说剃发了没？嗯哼，清的那个剃发，这玩意儿放到其他人身上就是“留发不留头”，非常残酷。但放到皇帝自己身上时，他们是知道美丑的。大家应该看过，无论是雍正还是乾隆，都很有喜欢自拍的倾向——放到今天就叫喜欢自拍、热爱美颜自拍；放到那个时代，就是特别喜欢让宫廷画师不停地给他们画像。在他们的画像中，有很多汉服cosplay，尤其是乾隆，特别喜欢玩这个。而且他还在自己的光脑壳顶上画上一个小头冠。如果真的一根头发都没有，剃成标准的金钱鼠尾，那头冠肯定戴不上——谁听说头冠能直接粘在光头上面呢？但乾隆有没有可能确实戴过头冠？这是高度可能的。为什么？因为清的皇帝自己也知道，把头发剃成那样不好看。他们的眼睛是正常的，眼光也是正常的。虽然乾隆的审美档次可能不高，有点土、有点暴发户的气质，但他的眼睛是正常的，有头发和没头发，哪个美哪个丑，他很清楚。所以实际上，清的帝王并没有电视剧上看到的那么喜欢给自己剃头发。很多时候，他们都给自己留成了板寸，大部分时候都是板寸。所以在画像中能看到鬓角，才是正常、合理的现象。这就是他们给自己剃头时，从来都不剃干净。画像都是清晰可见地表明剃得干净——对，为什么他在画像里就剃得特别干净，甚至出现头上没头发、没法戴头冠，却还要硬画一个头冠上去？那画师在画的时候，应该是他头顶确实戴了头冠。要知道，即使是板寸，只要有头发，用一些小技巧、小卡子夹子，还是有可能戴到头上去的。但如果真是金钱鼠尾，那无论如何都戴不上去。所以很有可能，他在后宫玩cosplay时是戴了头冠的，因为他头发剃得并不多。然而放到画像上就不一样——画像是要给别人看的，也要留给子孙后代打样，昭告天下，告诉你“你爹什么样”“你家君王长什么样”，画像有这样的教化宣教功能。而他对于全国上下的管制方式是“留发不留头”，要求所有人的头发必须标准化。在这种情况下，他在画像里就必须把自己剃得非常干净、非常标准才行。而实际生活中，他并不会剃得那么干净。

明朝百姓生活如何？首先，你要区分时间段和地理位置。完全不做区分就直接说好或坏，显然是不公道的。但当我们笼统地说生活好与坏的时候，应该怎么做一个基本判断？第一件事：民以食为天。大家应该没什么争议吧？别的事总有争议，但民以食为天总没问题。如果连这个都怀疑，那就别聊了。如果你支持这句没错，那么明的生活条件应该比清好大约一倍。因为明的人均口粮保有量是600多斤，而清是300斤左右。所以如果按能不能吃上饭的标准评判，明百姓的生活指数相较于清，大约是乘以二的关系。今天很多人感慨，我们的年轻人一代比一代高。大家到各个学校走走，小学、初中，不去看大孩子，就去小学和初中的校园，会看到很多1米8的高大个子，脸上还带着婴儿肥，时不时哭唧唧喊妈妈的小男生，拧巴拧巴地跑过来。为什么呢？因为人家才十几岁，还要看儿科，但个子已经比你高一个头了。这就是身高的差距。短短几十年间，这几代人的平均身高不断提升。到今天，19岁男性的平均身高在山东省已经超过了1米8。这个水平是什么概念？大约达到了先秦时期贵族人群的男性身高。我们先秦时期的华夏先民，贵族阶层就是这个平均身高。到清的时候，虽然统计口径有差异，但毫无疑问的一件事是：明代的男性平均身高在1米7多，不同口径下在1米73到1米76之间。然而到了清，这个数字下降了——平均男性身高下降了5到10公分不等，有些地区下降得更严重。平均身高下降，男性变矮了，这难道是什么好指标吗？就像之前看过一个女主播说，让你选男朋友，选胖的还是矮的？琢磨半天太痛苦了，选胖的吧，胖能减肥，但矮的长不高。所以只要没有重大缺陷，最后你会选高的那个。生活指数水平怎么样，就是用这些很刚性的指标来衡量：身高、平均寿命、粮食保有量。这些是数据方面的指标。同时还有一些更具体但可能更难量化的指标，比如你能吃得上什么肉、什么物价指数。按照不同方式统计，从明末到清初，物价基本以倍数飙升，范围在不同商品中有所不同，但即使是基本口粮这类刚需项目，物价也有飙升，大约提高了三倍到八倍不等。什么1米7黄金身高？有人说1米7黄金身高吗？哈哈，我们先秦时期贵族的平均身高超过了1米8，平民男子的平均身高超过了1米7。这事上，经常打架的小伙伴都知道“一力降十会”，没毛病吧？有打架经验的小伙伴一定会体验得到，在所有的斗争与博弈中，你自身的体格大小、体重大小直接影响战斗力的基本盘和起征点。其他所有技巧都要建立在自身力道足够强大的基础上。如果个子又矮、身材又瘦，技巧学得再好，也很有可能破不了对方的防。
