# 第115章  赢了叫勤王，输了喊东归。土尔扈特：这波我在大气层

对，拿着授权书回来。这件事情就不得不说到那个土尔扈特部了。土尔扈特部东归干啥来的？拿着大明的授权书回来的。他要是打得赢，他就叫做进京勤王、驱逐胡虏；那打不赢的时候呢，他就叫做东归故土、认祖归宗。非常灵巧。所以在这种局面上面，他既怕外头的来，然后又怕里面的作反。这种时候八旗没有办法飘，他飘不起来啊。嗯，啥？拿的是明朝的印件？对呀，土尔扈特东归拿的就是明朝的印件，他拿了明朝授予他授权的文书的，你知道吗？所以这种时候，你说土尔扈特当年东归的时候，他要是进来咔咔一顿打，就把八旗给干废了，你说他会说什么？你猜他会说什么？他手里头有大明的授权书，然后呢，他又把这边的那个什么八旗给干废了。他要是做到了啊——假设，假设他当年做到了——你猜他干什么？这玩意儿妥妥的秦王多正经啊！啊，他那个什么这理论一套一套的大儒辩经，现场给你写八本书出来。这也就是他打输了，他要是打赢了，历史都不是这样写啊。哎，你这话说的很溜嘛。啊，赢了万里勤王，输了东归故土。呵呵呵呵呵。

啊，刚才还有一个什么复位提问是吗？诶，我都没看见。嗯，啥？为啥有些黄汉要黑唐朝？说谁呢？没见着啊。说我呢吗？这不说我呢啊。我经常说李二凤文治武功，但是留下了一摊祸害啊。没有李二凤，后世的小宋不至于那么宋。这话是我说的啊，这话是我说的那个啥是说我的吧。啊，一个人呢，他确确实实强大，确确实实文治武功的同时，他也可以确确实实地留下隐患。能够理解这件事情吧？人无完人，金无足赤。这件事情它客观地发生了，在后世也确实产生了影响，这就是一个现实。至于你说这件事情是不是说啊，这是他诚心的——那不是诚心的，这不是李二凤成心的啊，这件事情不是他成心的。中原王朝的老毛病，仗着家大业大容易飘啊。这个事儿呢，你不能说是中原王朝的老毛病，这个是人类社会全都有的毛病啊，哈哈哈。人类社会都有这臭毛病啊。之所以你会觉得是中原王朝的毛病，是因为中原王朝经常强大到有点离谱啊，就是因为这个原因。为什么其他的人群好像没有这臭毛病呢？他基本上没有来得及强大到这么离谱的程度就嘎了啊。所以呢，你常常会觉得这个毛病是什么？中原王朝身上特有的。其实不是啊。你去看一看今日之欧洲啊，你去看一看今日的这个欧罗巴，你就说他有没有这毛病？瞧一瞧今日的欧罗巴有没有这毛病？有吧。现在的欧洲和以前的欧洲长一样吗？不一样。但是你去问他这样子长相都变了，对不对呢？他说对。然后难不难受呢？她说难受。虽然难受，但是对。这是什么？这个就是阔气了之后觉得理所应当，然后呢，就进入到飘了的状态，然后就觉得它可以帮助全世界，然后呢，就觉得谁来还不是一样的呢，无所谓的嘛。反正啊，就是这么强大，就是这么傲，就是拥有先天的优越性啊。所以呢，因为它发展到了这样的一个阶段，甚至已经开始傲视群雄，看天下的其他人啊，都是开始悲悯了，知道吧？开始我怜悯你——出现了这样的心态的时候，这叫什么？这就叫飘了。他看旁人都存着怜悯之心，这就是飘了。飘了之后他就怜悯，觉得对方无足轻重的时候，他就会收到家里面来打工嘛，干脏活、干累活嘛，天经地义的嘛。我就跟你说，他的这种操作的方式，跟咱们当年汉操作的方式、唐操作的方式一模一样，在心态上面高度相似。只要你强大到了这样的地步的时候，就非常容易陷入到这种迷梦之中去，觉得谁还不是一样呢？天下皆是朕的子民，那朕独爱之如一呀。所以当年的那个什么欧罗巴，当年啊，说还应该让他们进来，人应该让他们来，多可怜呢，可怜见儿的——就他们的这种心态和当年李世民的心态，本质上面是一回事儿啊。

但是最后的结局呢？最后的大结局呢？啊，李世民当年的泛爱之如一，虽然他后来后悔了，虽然他确实本人后悔了，但是他造成的结果已经没有办法挽回了。等到后面啊，再遇到了李隆基晚年的时候，那种失控晚年了，变成了老糊涂蛋了——之前还警醒着要控制，到这个时候变成糊涂蛋了啊，然后控制也没有了，于是之前所埋下的雷就这样爆了，直接把整个的大唐轰一下子炸到腰斩。什么？啊，你大清就没有这种问题啊？对对对，啊，你大清确实没有这种问题，因为从头至尾他就没有飘过。为什么呢？从头至尾他都是小弱鸡。嗯，就是你们最高呼巴图鲁天下无敌的那个时代，他们自己的八旗统治者自己都很清楚，自己是个小弱鸡啊。所以他没有机会飘，确实从从头到尾就没飘过。你看他在那里天天耀武扬威的“我要如何如何”，每天心惊胆战，担心那些贰臣又叛了。为什么？因为力量是偷来的、是借来的、是骗来的，不是自己的。他下面最能战的部队不完全属于自己，是偷来的、骗来的，通过离间之术哄来的啊，是抓了人家把柄之后窃来的。用这样的方式获得的力量，它飘不起来。他分分钟都知道啊，下面的那些贰臣极有可能跳反，那些看上去已经从了的地主极有可能跳反，那些看上去没有二两肉的农民天天都在跳反。他明知道这一切的时候，他怎么飘？他没有机会飘。然后啊，外面的那些外国，那些外国满清的高层统治者非常清楚，那些国家当年都是拿了大明印信授权书的，但凡是哪一下子人家野心起来了，要驱逐鞑虏，别人可以直接拿着自己家的授权书就回来的。他要是打得赢，他就叫做进京勤王、驱逐胡虏；那打不赢的时候呢，他就叫做东归故土。他那个什么这理论一套一套的大儒辩经，现场给你写八本书出来。这也就是他打输了，他要是打赢了，历史都不是这样写啊。
