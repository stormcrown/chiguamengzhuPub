# 第192章  表音文字为啥总崩盘？汉字：怪我太强咯！ #汉字 #语言学习 #文字密码 #女书

长江、黄河流域为什么没有出现过使用拼音文字的部族？实际上，这些流域范围内并非完全没有表音文字。今天所说的部分少数民族的文字，其实是表音文字，只是它们看上去和汉字长得很像，容易被认为是汉字的变体或象形文字。首先需要明确：汉字的雏形是象形文字，但汉字本身是表意文字，已经不是象形文字了，而是表达含义。

关于发源地的问题：黄河流域没问题，但长江流域的南部地区，确实有一些部族在开始使用文字时，会借鉴中原的文字，然后用于表音。这体现在一些地方还有残余，例如女书。这些文字属于改造后产生的残余。北方有很多游牧民族建立过政权，今天一般称为汉化政权，他们也会创作自己的文字，主要存在于唐代以后，比如辽、西夏都曾创造文字。这些文字在创作过程中使用了部分表音性质，不完全表意，其中囊括了表音的内容。为什么这样？实际上，尽管汉字是表意系统，但借鉴汉字而创作的文字就不一定完全表意。例如日本借用汉字进行创作，制造自己的文字时，就不是完全表意的，而是一部分表意、一部分表音。

关于女书：女书这个文字目前还有一些争议，主要来自编撰女书字典的人。研究女书的人试图将文字系统扩大化，吹嘘成一套了不起的文字体系，希望让它更完整。但女书系统原本并不完整，它是一种相对简单的、在女性内部传承的符号，类似于暗号，借用了当地方言的发音方式和汉字系统进行二创。它的应用范围不广，内容也不复杂，很多用于服装纹理。但发现女书的人希望它成为大的学术体系，就编了一套复杂的字典，把原本简约的符号体系复杂化，导致出现各种密码传说。实际上，女书本是一套简单符号，被人掺了水。随着教育普及，女书这种作用不大的符号系统会自然退出历史舞台，因为它的表达能力有限，扩张能力也有限。它曾经在某些地方起过作用，但今天已基本无用。把它作为历史印记、刺绣纹理或历史记忆碎片没问题，但有人想把它当成学说，就会制造问题。前几年女书问题上还打过嘴仗，有人说把女书说得档次低就是攻击女性，这是刻意搞社会撕裂的行为。过去一些山区，受教育水平和经济条件差，女性受教育条件更恶劣，她们内部用旧文字符号体系创作一套内部识别代码，是合理且常见的操作。女书系统原本没那么复杂，一旦被人复杂化利用，再纠正时就会有人说不承认其伟大贡献就是瞧不起女性，然后在互联网上打嘴仗。之前一些科普人士都曾被冲击过。

所以，有没有表音的符号系统？有的。我们周边有很多表音符号系统，但那些借鉴汉字体系创作的表音符号系统，后来基本没起很大作用，因为表音不持久。无论用什么创作的表音符号系统都不持久，因为表音太容易改变。日文是半表音，现在表音占比越来越高，所以日文也越来越差了。过去日本学习汉字时，上层贵族直接使用汉字，但同时创作了一部分表音符号，因为日语和汉语不兼容。为了让读写更兼容，他们掺杂表音符号。社会上层顶流贵族直接使用汉字，文化水平下降的中层则开始使用表音符号。表音符号用得越多，意味着汉字的水平越差。在日文系统较完善阶段，汉字占比、表意占比很高。但日文汉字通常经过改造，要么缺笔画，要么改变笔画方式，要么改变含义，不是直接挪用。这套表意夹杂表音的日文系统，在全盘西化后表音占比越来越高，因为日本开始崇拜西方，而西方使用表音符号系统，所以他们对表音越来越放纵，表音词汇占比越来越高。

有一个传闻：中文现代词汇大部分来自日文。这个谣言传播了很多年。实际上，日本在晚清到民国时期翻译了大量西方文献，他们运用汉字的表意系统进行翻译，因为熟悉传统文献。中国的学者直接拿来使用，却被营销号造谣说没有日本学者我们就不会说话。其实日本是用汉文系统翻译的，只是当时他们对文字系统还有自信，使用了表意性翻译。但今天日文开始用表音翻译法，比如“电脑”不叫电脑，而叫“computer”（具体发音会日文的人知道）。现在日本对外文翻译直接音译，以至于后来的人不知道是什么意思。这就是表音文字不可持续的原因：表音会不断迭代各种音节，越来越复杂，无法直接表达含义，最后要记忆无数音节。现在英文词典每年增加约一本，就是因为新造词汇和专业词汇导致隔行如隔山——不是不懂专业，而是看了像天书，一个字都看不明白。这就是表音文字的天然缺陷。日本当年借用汉字创作时考虑了表意，但现在深度西化后走表音路线，文字系统变成半死亡状态。

不能否认表音文字（我用拼音输入法），我觉得拼音很好。韩国的精英没有放弃汉字，是因为他们没办法。日本是文化上的干儿子，但韩国和我们关系非常紧密，汉代时有大片土地属于郡县制管理，是亲儿子系列。由于文化往来极其深刻，韩国语言发展受到汉字系统影响。过去长时间直接使用汉字，因为汉字表意不会表错情，所以对同音字不在意，中国人也不太在意同音字。韩国原本也不在意同音字，但转成表音符号后，同音就变成了灾难。
