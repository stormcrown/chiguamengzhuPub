# 第122章  殉葬黑锅谁来背，元朝习俗还是建文政争（嫡长子继承与幼子守灶）

殉葬这个事儿，跟朱元璋半毛钱关系都没有。真正搞了殉葬的是谁？是朱允炆——好皇孙啊，好皇孙搞的殉葬。他真的是为了风俗搞殉葬吗？不是，他是为了政争。嗯，啥？能不能讲建文帝？请问我现在讲的是谁啊？我，我已经在说朱允炆是好皇孙了，然后你说“能不能讲一下建文帝”，你觉得他俩是俩人，是吧？嗯嗯，好的好的啊。哦，我已经唠他唠这半天了，然后说能不能讲讲建文帝，我能咋说呢？啊，这咔嚓一下就给我怼没了啊。

啊，啥说他恢复殉葬啊？朱元璋没有恢复殉葬啊。朱元璋一生，包括他的遗诏，所有的东西当中都没有恢复殉葬这件事情。没有这事，就不是朱元璋干的，跟朱元璋一毛钱关系都没有。而且殉葬这个风气是谁带来的？为什么中原在当时会有殉葬之风？是蒙古王公带来的，这个是蒙古王公的习俗。尽管他们原则上不设坟墓，原则上啊，尽管原则上草原群体不设陵寝、不设坟墓，但是他们有严格的殉葬制度。在他们的蒙哥汗这里表现得非常清楚：蒙哥汗死在了钓鱼城之后，他需要运尸回到草原上面去安葬，行葬礼时，沿途丧葬团队——沿途所遇到的所有人，全部都要如此。这是什么？这就是殉葬制度。不要觉得他不设陵寝就没有殉葬啊。这个事不光蒙古王公有，满洲群体也有，他们都是有殉葬的传统习俗的。尽管说他们喜欢把这个锅扣给汉人，但实际上我们自打周之后就不太搞殉葬了。从周代开始，殉葬尤其是生人殉葬这件事情，就已经开始走向下坡路了，我们就不搞这一套了。

然后呢，朱元璋这个时候是没有所谓的什么恢复殉葬这件事情的。事实上，恢复殉葬、带入殉葬之风的是蒙古王公。而朱元璋他一生的所有诏书当中，都没有提到要搞什么殉葬。他活着的时候，他的太子去世没有殉葬，他的爱妃去世没有殉葬，这些都没有殉葬的行为。所以这个事儿——殉葬这个事儿，跟朱元璋半毛钱关系都没有。真正搞了殉葬的是谁？是朱允炆——好皇孙啊，好皇孙搞的殉葬。他真的是为了风俗搞殉葬吗？不是，他是为了政争。为什么要政争？因为朱允炆严格来讲，他不符合继承规则。尽管他是太子之子，但是他不是嫡长。朱允炆不是嫡长啊，他尽管是太子之子，但是他不是嫡长，他得位是有问题的。他得位有问题，立身不正的时候，他就必须要使用一些阴暗的政争手段来维持自己的地位。所以他才会找殉葬这种借口排除异己。莫要以为这玩意儿是什么风俗，这跟风俗半毛钱关系都没有。

嗯，啥？能不能讲建文帝？请问我现在讲的是谁啊？我，我已经在说朱允炆是好皇孙了，然后你说“能不能讲一下建文帝”，你觉得他俩是俩人，是吧？嗯嗯，好的好的啊。哦，我已经唠他唠这半天了，然后说能不能讲讲建文帝，我能咋说呢？啊，这咔嚓一下就给我怼没了啊。

嗯，嫡长这个也是嫡庶天然不同吗？啊，不是啊。咱们弄清楚一件事情：就是嫡长这个事情是干嘛的？嫡长指的是祭祀的继承权。很多时候我们现在受电视剧的影响，你会觉得嫡出啊怎么滴，庶出啊怎么地，就嫡庶神教这一套，还觉得有什么不同。不是啊，注意了：嫡长这个东西是拿来继承什么的？继承香火。今天的人老是觉得继承香火这个事是一个贬义词，我理解大家的心情啊，因为之前重男轻女的时候，老拿男丁当做什么继承香火。这个所谓的继承香火，实际上意思指的是：家里面谁掌握了最后掌家之权，谁来继承家里的名号，谁来继承家里面点香的权利、祭祀祖宗的权利，谁来当家族的领队人物，这个叫做继承香火。嫡长解决的是继承香火的问题，它不是财产分配。财产分配，庶子都有权利，甚至说有一部分来自于他母亲的私财，那是直接交给庶子继承的都有。庶子是有财产继承权的，也不会在任免官爵上面去苛待他。但是嫡长是干什么？嫡长的这个继承香火讲的是宗族长，讲的是家庭祭祀权。注意了啊，这种权利——掌家立户的权利，你只能挑一个人，这个玩意儿是不能分的。财产、财物、钞票、房产、地都可以分，这些东西都是可以分割的。但是家里的香火不能分。你姓赵的就要记姓赵的祖宗，这件事情你一辈只能有一个宗族长，只能挑一个。包括皇权的继承，你也只能挑一个皇帝继承，你不能说搞好几个太子。所以你看有的时候，那些号称自己继承了汉制的一些胡人政权，动不动大太子、二太子，这个就很荒唐啊，这个命名本身就有点荒唐。在一些胡人政权当中，他使用这样的称呼——大太子、二太子，但是太子是什么？太子就是所有的儿子当中掌握了香火祭祀继承权的那一个，是唯一的，它必须是唯一的。因为你家长不能够有好几个，家是不能拆的。财产可以拆，但是你家的这个核心的名号不能拆，就是我们这个国不能拆，国是不能有好几个皇帝的，所以太子只能有一个。这是一个继承权的问题。

为什么要强调嫡长？因为没有争议。嫡长这件事情会造成你的继承权、继承香火的这个事情毫无争议，不用为这事打破头。所以它不存在说在这事上歧视庶子——你看吧，你这还不是嫡出的，庶出的——不是这个问题，而是你总得立一个规矩。你没有规矩的话，大家就只有只好靠打破头去抢这个香火了。你不想抢，那怎么办？你必须要立一个规则。什么规则？最简单、最简单的规则就是嫡长。立嫡长的这种方式是最简单、最有共识、也最稳定的一种继承方法。香火分了就是另立山头了，对，财产可以分科，香火不能分。所以那个继承香火要由嫡长来继承。

那个包括草原上面，草原上面会搞什么幼子守灶，是吧？它是不同的生产方式带来的继承法则。为什么草原上面的人会搞幼子守灶，而我们农耕的群体要搞嫡长继承？因为我们这里是一个稳定的定居生态，所以你需要的这个领军人物要有一个特征是什么？他要有权威性，要有充分的智力，要有充分的理性来掌控这些东西。所以呢，长子相对来说年纪更大，经历更多，受父亲的培养的时间更长，权威性就更好，他就有更长的时间进行权力的交接，因此他可以继承这个天下，去掌管你这一大摊子的事业。但是在草原上面，它就会出现幼子继承制、幼子守灶。为什么在草原上这样？因为草原的生态模式、生产模式没有办法在一个部落里面维持很多个人，因为土地的承载力、草场的承载力非常有限。你今天也可以去搜一下，就我们今天的草场上，一个草场上养多少羊，有规则的不让养多，养多了你这块草就废了。所以呢，草原的群体，一个家、一个家庭、一个部落，它的人数是受限制的，人数绝对不能够多起来，人数一多了就乱了，草就不够吃了。所以它会生下来一个儿子，长到一定的年岁能够独立放羊、骑马的时候，就让他带着一部分的羊、一部分的牛羊、带着几匹马——他的财产，就让他走。你不让他走，你这个家就废了。所以他才会一拨一拨地让长子先走啊，大儿子、二儿子都走了，最后留下谁？最后留下最小的一个继承父亲的衣钵。这是它的生产方式来决定的继承方案。而我们由于是定居，而且是积累型的大产业，所以我们就需要嫡长继承。嫡长子继承制，继承的是一个核心名号，继承的是整个的事业框架。而庶子庶女获得的是什么？获得的是财产。
