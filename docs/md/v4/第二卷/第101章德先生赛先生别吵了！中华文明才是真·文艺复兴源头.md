# 第101章  德先生赛先生别吵了！中华文明才是真·文艺复兴源头

哎，大家好，大家好啊！你们真不错，都跟上了。那啥，你的意思就是文艺复兴复兴的是中国文化。嗯，这件事啊，这件事不是我原创的啊——文艺复兴复兴的是个什么玩意儿？这件事情不是我原创的，完全可以再聊一聊。

嗯，就是“文艺复兴”这个词儿，它从一开始被创作出来的时候，其实包含的内容是什么呢？就是讲我们从前是很牛的，中间经历了一个黑暗的中世纪，然后现在呢，我们又醒过神来了，又牛了。所以它才能够叫做“文艺复兴”。你要知道，如果说一个地方以前从来都是遍地荒芜，它是没有资格说“复兴”的，最多只能叫做“崛起”，是不是？但是呢，他们给自己命名的时候，认为自己是复兴重来了一回。那他怎么重来一回的呢？你既然是重来的，重开的盛世天，那你就得告诉我，你的前一个盛世是什么时候，那才叫做复兴嘛！不管你中间熄火熄了多长时间，你总得有一个祖师爷摆在前头，然后你才能够叫做“再兴师门”，对吧？你必须得有这个东西。不然的话，那叫做“开宗立派”嘛！那你怎么能够证明以前的事呢？

所以呢，咱们记得一件事啊——民国的时候，中国这里是不是开始引入了一个东西，叫做什么来着？啊，德先生与赛先生！是吧？民国时期我们这里非常流行地引入的一个概念，就是德先生与赛先生。这个呢，是当时非常重要的、有革命意义的东西。为什么呢？为什么是它有革命意义呢？因为它认为说啊，我们的这个传统当中是没有民主思想的，然后也没有科学思想的，于是呢，它才要引入西方人的德先生与赛先生。那么既然对方是负心的（对方以前从来都很辉煌，甚至是文化源头），那你得告诉我文化源头是什么。所以，民国时期特别流行的一个东西，就叫做“言必称希腊、罗马”——只要说话，只要开口，就要言必称希腊罗马，就是引经据典。啥叫引经据典？放到今天互联网上，就是要玩梗啊！古人的引经据典，放到今天就是心里有梗，什么梗都接得住，什么梗都抛得出来，这个就叫做引经据典啊。那么那个时代的引经据典，引的是什么经，据的是什么典呢？就是一个很普遍的现象：就是“亚里士多德曾经说过”“柏拉图曾经说过”啊。这个东西就是“言必称希腊罗马”。唉，这个臭毛病呢，到现在其实互联网上都有很多的大咖是这个样子的。至于说的是谁啊，大家心照不宣，你们说了我也当没看见。总之呢，就是包括现在中国还有很多非常有名望的人、非常知名的人，包括在互联网上行走的知名UP主，都是要开口闭口西哲先贤的。这个东西就叫做“言必称希腊罗马”。

而这件事情在今天啊，有没有影响力？那不光是我们普通人这里有影响力，在学术界也大有影响力。影响到了什么程度呢？就是“万物西来说”。我们昨天的时候才批了一个那个脚盆（日本）那边安排过来的一个学者啊，他讲的就是“青铜西来说”。那既然中国这里有青铜，草原上面又有一些贵族的墓葬里面陪葬了青铜，那么青铜就应该是草原人教给我们的。那草原人也没有这玩意，那就一定是西边传过来，有草原的这些游牧的群体，骑着马、牵着羊从西边给我们搬过来，教会了我们如何锻造青铜啊。那这里头就出了这种逻辑性的问题：为什么我们这里什么都有，但是我们不是原创的？为什么别人那里什么都没有（就一挖耳勺），但是他们是原创的呢？啊，不为什么，因为“言必称希腊、罗马”这个东西根深蒂固啊！从清末开始，世界的文化的源头、科技的源头就板上钉钉地放到了西方那里去。

包括说我们在刚才聊天的时候，我们直播间的公屏弹幕上面，就有人说：“那你说怎么可能是中国人弄的？你这个人是九楼鱼（九漏鱼），你就在那胡编乱造吧！你一点证据都没有啊，人家那板上钉钉的，就是别人弄出来的。”这个是什么？这个也是“万物西来说”的一个产物。就是说，如果西方人说了这东西是他创作的，那一定是他的；如果说我们这里有，西方人据说也有，那一定是他创作的；如果说西方人有裤子，那我的裤子就是跟他学的；西方人戴了帽子，我们的帽子就是跟他学的；西方人做了泥塑——据说西方人的那个雕塑，古希腊的雕塑特别逼真，然后我们的秦始皇陵兵马俑里面的俑也特别的逼真，长得跟真人脸似的，那么我们就必然受到了古希腊的影响。这些学说大家一定是听过的。不光是民间的传闻啊，这些学术界的顶流的学者也是这样想的。他们在我们的古典墓葬当中发现了轮子：“哎呀，我们跟西方人这么早就学习了轮子！”他们在我们的西域地区发现了大澡堂（有地热、地暖的大澡堂、公共浴室）：“哎呀，我们跟古罗马之间有这样深度的往来，连大澡堂都跟人家学了！”这个就是我们的学者啊，我们的学者就是这样整的。所以这是什么呢？“言必称希腊、罗马”它不是一个历史名词，它到现在都对中国人影响深远，这是一个极为普遍的现象啊——从上到下，到处都有这样思想的人。互联网上传播信息的人是这样想的，搞学术研究的人也是这样想的，大家都一样。

所以，当我们整体的思维的底色里面，始终如一坚持了近200年的这种“言必称希腊罗马”，那我们现在应该干什么？去魅（祛魅）。怎么去魅呢？这个倒是不困难啊，这个去魅（祛魅）不困难。为啥？因为对方说了他是伟大的文艺复兴。既然是复兴，那你就得说以前的兴，以前怎么兴的呢？他就得给自己找一个老祖宗，必须找你。没有老祖宗怎么叫复兴呢？古希腊、古罗马都有具体的地方。既然有具体的地方，我们就可以分析：你这个地方能够出产多少的金属？有多少的田地可以耕作？有什么样的水系？用了什么样的农具？投入产出比是多少？按照这样的投入产出比，你能够养活几口人？你的气候条件是什么样的？水文条件是什么样子的？地层条件是什么样子的？耕作的水平是什么样子的？使用的农具到底是石头的、青铜的还是铁的？这个东西直接影响到你的产出。而最后咱们就可以直接地证明一件事情：在工业革命之前，欧洲人的粮食的投入产出比可怕到有相当一部分地区是一比二。我们的古诗当中叫做“春种一粒粟，秋收万颗子”，虽然这个比例有点夸张，但是明确的一点就是你绝对不会种一粒种子收获两粒。但是欧洲有相当一部分的地区，农耕的投入产出比是一比二。逗我呢！这能养活人吗？啊，这玩意儿能养活人吗？你要有多少的农夫才能够养一个脱产者（仰天观望月亮）？你要有多少的农夫在这样的投入产出比下，能够安排一个脱产的人出来，进行艺术的创作和数字的编撰，去搞这些科学、搞这些计算、搞这些观测？说我们远古就可天文学了——我们天文学、原创天文学，经年累月的观察和计算，做档案，这些东西是脱产全职工作。你的粮食生产效率那么差，耕地质量那么差，耕地范围那么小，气候条件不能够达成完全的雨热同期，我就问你：你得有多少的农民能够养得出来一个学者做这件事情？而如果在你的土地上面，根本就没有足够量的农夫可以让你产生足够量的脱产者的时候，你那科学是不靠人、靠天降下来的吗？那一比二的粮食，比剩下来的那些就都产成了人脑子是吗？不然呢？他编撰的那些伟大的科学家栩栩如生，写下来的煌煌巨著动辄万言，安排出来的这些学术的成就硕果累累，唯一的问题就在于，你也不知道那是不是活人啊！

所以今天这是什么啊？叫“言不必称希腊”。这个就是一个去媚（祛魅）的行为。他说他的老祖宗是古希腊、古罗马，但是他给他的老祖宗安排的这些辉煌伟大的历史，没饭吃。所以我们这里要问的东西也不多啊，比较的简单，比较的朴素：吃什么？穿什么？住哪里啊？

这套书呢，就是何晴老师编撰的《以图证史》，上下两册的一套图书，是质疑西方历史的造假问题。它一共是两册：一册呢，是《言不必称希腊》，是从一次希腊之旅做切入点去描写的。实际上他不是这个时候才想的这个问题啊，他是一个正经的学者，是搞美术史研究的官科学者，所以呢，他最开始的时候是从绘画上面发现了西方历史造假的问题的。美术史也是历史，美术素材也会反映人文和历史的进程的。他在这个过程当中发现了问题，研究了很多年之后，在一次希腊之旅之后啊，用这个旅行作为引子，去对古希腊地区的文物和历史造假的问题进行了一番揭露。然后呢，第二册就是《光从中华来》。既然说你那个是假的，你那个源头是假的，那你总得有点源头吧？啊，既然是复兴，那你复的是谁的心呢？你总得给他找一个切实的源头。于是就回到我们今天开篇所讲的那样东西：你要发明一个东西，你总得给它起名字。十进制和中国人的语言是高度匹配的，和欧洲人的语言、中东人的语言都不匹配，所以十进制是谁发明的？十进制是这里发明的。而现代科学的运算都拿十进制作为基底，那这些运算到底是谁开创的？这些运算的、基础的、前置的、学科类的东西，在中国的土地上面都找得到各种各样的溯源的版本，而在其他的地方没有。所以到底谁才是前面的那一个伟大的兴盛？这个复兴到底复的是谁的心？这个呢，就是这一套书啊，还是推荐大家可以来读一下的。你说他的论证已经非常完美了吗？啊，我没有认为他的论证已经非常完美，但是它很重要。

真要造史金字塔，干嘛不造欧洲去？你知道古埃及是他们的文化源头吗？不知道的话去念一下Bible啊，用不了念几张，哎，往前面稍稍地翻一点点，你就能看到了。他给自己设计的源头就在古埃及，所以动辄按照地理决定论来说，这个地儿又不是西方。嗯？你以为什么是西方？你以为什么是西方？在文化史上面，在西方中心的概念当中，除了东亚文化圈，其他的全部都是西方。你一定会注意到这个现象，我也反复地强调过这个现象。西方人在推崇这些伟大的古典文明的时候，他只质疑一个地方，就是中国。只有中国这里发现的遗址，他会问：真的吗？假的吗？什么时候的？假的吧？证据不全吧？只有在中国这里发掘遗址的时候，他会这样问。他在任何其他的地方发掘出来的遗址，哪怕那个遗址非常的荒谬，看了之后都觉得荒唐，像个笑话，他也说：“这是一个伟大的人类古文明，而且历史非常的悠久，而且一定要比中国的历史悠久。”所以你以为什么叫做“西方中心论”？不是西方才是中心，而是“非中国”的就可以是中心。这个东西就叫做“西方中心论”啊。西方中心论这件事情本身就是一个将全世界的科学与文化、历史去中国化的过程。所谓的西方，就是除了中国以外的其他的地方。他们的整个这种西方中心论的分析学术当中，只排斥一个地方的历史，就是中国的历史。除了中国的古典遗迹，他对其他的地方全都是无条件的赞赏的，无条件的恭维，无条件的夸耀啊。但是他为啥那么恨我们呢？啊，恨我们就对了，不然呢？因为我们这里都是真古董，我们这里都是真文物。我们刨坑啥都能刨出来，我们刨个几千年的坑啊，十进制也有了，12进制也有了，七音阶也有了，九九乘法表也有了。西安经常挖出古墓啊，咱们要聊西安的那个什么造地铁的历史吗？呵呵呵呵呵，你们都看了那个西安造地铁的那个东西没有？我每次啊，看他们的新闻都想笑啊。挖一个地铁，哎呀我的天啦！喂，挖的是个啥呀？啊，挖着挖着，嘿嘿，就挖出个坟啊！挖着挖着就挖出个坟！哈哈哈哈！我说我要挖一个地铁，挖一站地挖几年，呵呵呵呵。啊，所以呢，咱们得看一件事情啊：你看我们的西安是一个古城，是一个古都，洛阳也是的，河南这边古都特别多。你看啊，我们这些古城的遗址，这些地方一旦修建地铁，有的时候不光地铁啊，修那个公路的时候也会打地基，建筑地基的时候也会碰到啊。动不动啊，一个钻头打进去，哎呀我的天老爷，诶，底下有一个什么什么古墓呀！呵呵，是这样的吧？基本上就是你刨一个坑，一发现，哎呀，这底下这个东西怎么有点不对味啊？完了，停工不知道停多长时间啊！

但是你有没有发现一件事情啊？西方人这几年，他们在自己的本土发掘了不少几千年前的文物啊，包括那个什么青铜剑呐，然后还有一些什么马赛克啊——我也不知道为什么要在田地底下发掘出来马赛克啊。他挖掘出来的这一堆一堆的文物什么的东西，最近这几年量不少。你会发现一件事啊：他们去发掘这些埋在地底下的文物的时候，刨的那个坑都可浅了。你们自己去网上去看那个他们的新闻图片，你自己去看啊，这个玩意儿不需要什么门槛。哎呀，不行，我不懂考古——这不需要你懂啊，你自个儿去看啊！就他们在大马路上面、闹市区里面，就贴着别人家的房屋的地基，噌噌地往外头刨开了那么几米见方的一个坑，就在里面精准地刨出来几千年前的文物。那我就纳了闷儿了：之前你修那个马路的时候，底下一寸不刨那个土的吗？啊，你以前修公路的时候，一寸地皮都不用掀的吗？你以前盖房子的时候，那地基就完全没碰着你这个文物的区域的吗？啊，你有点逗闷子知道吗？你的这个工程的施工的地点很荒谬，知道吗？我们这里随便的一个地方修一条公路，得刨出多少个文物的坑坑来啊？啊，那些基建标准比较高一点的公路，刨得深了啊，那个就要打爆文物局的电话，就到了这样的地步。结果你那个公路已经号称用了好几百年了——古城啊古城，你那个马路都用了好几百年，说那个房子、那城堡已经住了好几百年了，结果啊，你搁旁边40厘米深（真的，小孩做个游戏、挖个小迷宫都有可能刨着的地方），就能等着你刨出3000年前的文物来？你以前到底是干啥的呀？你以前你家那个工程都是怎么干的呀？从来没有人在街上面去搞生产搞建设吗？一镐头都没锄过吗？都是千年工程？那马路就在平面上直接一铺，铺完了之后就永远不动弹了嘛！我也没发现他的基建水平那么高啊。啊，啥？人家小孩素质高啊？我们的素质不高啊？我们这里的小孩还会撒尿和泥呢，动不动在地上刨土，这跟人家确实不一样。老老实实的，离地皮还不到半米深的地方就放着文物，结果呢，放了几千年没有人发现？
