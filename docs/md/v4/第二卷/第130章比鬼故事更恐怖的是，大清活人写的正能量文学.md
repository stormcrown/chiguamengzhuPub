# 第130章  比鬼故事更恐怖的是，大清活人写的正能量文学

我们相对能够理解那个时代的文人在创作时身上的那种扭曲感。因为他需要入世才有可能改变民生，才有可能影响天下的行走方向，所以他必须出世。要出世，就必须让八旗认为你是狗，所以他必须当狗。他想让自己流芳百世，就必须拥有君子的正德；但他想要让朝廷认可他的身份，让他的书籍能够传播下去，而不是把他们家的九族玩一波“消消乐”，那他就必须对这个反动的朝廷歌功颂德。每一对行为之间都是尖锐对立的矛盾，甚至非此即彼。所以这个时期的文人就会表现出扭曲感：他既要写——这是他的责任，必须写；写的时候又必须歌功颂德；但他真正要倾诉的东西，在道义上、在规则上、在现实层面上应该倾诉的，却是底下血淋淋的真相。因此，这是什么？这叫做风月宝鉴。正面是公子红妆，背面是白骨如山。

我们今天要聊的是谁？在哪里？有人跟我说一声吗？好的，我找着了。hello，hello，又见又见那位盟主。诶，你好你好你好，诶，你好。我今天主要是想谈一下那个《红楼梦》同时期的那些作者的文人的精神状态。然后就是癸酉本《红楼梦》，我是在你直播间时首次听到的。我以前对《红楼梦》背后故事的了解仅限于曹家故文学，然后听到你强调“白骨如山”，姓氏改的时候，就觉得这个味道很对，因为它跟我小时候看过的与《红楼梦》同时期的一些很拧巴的文人的文学作品状态很对味。

这些人的话呢，就是一些有名气的人：一方面在满清做官，要服务于他们以获得一些社会地位和利益；另一方面，又保留了自己的一点民族正义感，有自己的风格。所以写的东西会显得很拧巴，但这个味道又非常好看——越扭曲越好看的那种感觉。其中有一个叫钮琇，他应该是在满清朝廷做过县官的，写过一部传奇志怪小说集叫《觚剩》。之前获得过茅盾文学奖的写《少年天子》三部曲的作家，抄袭过——不是，借鉴过他好多篇《觚剩》里的文章，是大段落的借鉴。在这本书的开头，他花了不小的篇幅来写张汤水。而且他的书里有很多很多的人物，出现了众生百态：有僧侣、有豪侠、有侠女、有道士、有文人，还有才子佳人，写得非常有趣。书里也会反复出现“鼎革”“兵燹”“甲申”这些字眼，反复出现甲申前后芸芸众生的生活状态对比。另外他还写过嘉兴屠城，写得非常隐晦，带了一点鬼怪志异的色彩，但明眼人一看就知道他在写屠杀。他用词比如“天下初定，王师南下”，这个“王师”就是表示满清朝廷；“所致归命，何俊以改服发”，这是讲剃发易服。他称满清军队为王师，似乎是给戴了高帽子，但又反复描写屠城的惨状——笔笔在撇清，但又笔笔在清清楚楚地记录这个血债。

还有关于广州屠城，像“顺治某年某月，耿继茂、尚可喜兵入广州，屠戮什么什么的”。在钮琇那个年代，其实才赶上三藩被平掉，但在顺治年代的时候，它代表的是满清。还有一个人叫姚文燮，他写过一篇文章叫《圈占记》，讲的是圈地的事。他在文章开头强调圈地从周朝就开始了，让大家不要见怪，然后开始详细记录当时的八旗是如何圈地的，写得非常具体：两匹马一前一后拿根绳子，一圈圈个几百晌，然后就开始分赃，好几拨人一起干这个事儿。旁边有笔帖式记录，有各种满官的分工，分门别类地进行文书交接，把圈地这个活儿做成了一种产业化，甚至是制度化。尽管他强调圈地自古以来就有，但他详细记录的笔触中，我能感觉到他心里有恨。

这个姚文燮家庭关系比较复杂，家里很多亲属成员包括他本人都服务过满清朝廷，但至少在《圈占记》里，我是觉得他心里有恨。我也见过考据派怎么把《红楼梦》跟曹寅还有康熙扯上关系的，做得认真，一遍一遍读脸谱，包括起居注和两人的诗词——就是应制诗和御制诗写得很烂，但研究得很细；还有生平经历，围绕在这两个人之间的亲戚关系、人际关系、活动范围，甚至是每一次文坛诗社的酬唱招引，都研究得很细。但如果《红楼梦》真的是讲曹家的家底被康熙下江南挥霍一空，主子奴才之间的贪污腐败对上穷奢极欲，一个愿打一个愿挨的事时，感觉它配不上这么大的格局——一个家族的衰落，配不上这么深的情绪，配不上埋这么大的伏笔，更配不上这么大的恨。

我觉得那个时候的文人，这些人的作品有一种被摧残的心态：他要活下去，就必须进行一次精神的自我阉割；但同时他又保留了正义感，导致他们的文章会有很多影射，甚至有一种较强的社会责任感——他要给后人留下一些东西。当时的时代，他不能说是谁干的，但会把事情具体地记录下来。比如钮琇，通过写各种才子佳人志怪故事，反复强调甲申前后，让你做对比。还有姚文燮，把八旗圈地的行为具体写下来。这些人，你看他们的文章，就是觉得他们想要把见到的不公通过影射的、有一些春秋笔法的方式记录下来，希望总有一天有人能够读懂。这样的状态与《红楼梦》的作者也很契合。同时代的这些文人，能力不一样：有人能写《觚剩》，有人可能只能写一两篇，但有人能写《红楼梦》。他们的精神状态很一致：在文里强调“我不是这个意思”时，潜台词就是“我就是这个意思”，而且希望后人能记住这个事儿。

钮琇这个人很有意思，他开了一篇崇祯末年进士榜的名单，称赞是“绝有明一代之举”，科书多盛世讲那个进士榜——崇祯最后一年的进士榜。他还列举了那年的进士，比如陈名夏、陈克、还有梁清标、张玄希，称赞他们是新朝的肱骨之臣。但后来这些人进了乾隆的《贰臣传》。所以，蛮有意思的——你说他是不是故意的？我觉得他可能是故意的。我可能今天有点啰嗦啊？没事，因为你要讲的具体的东西很多。对，就讲完了，就到这儿是吧？好的好的，谢谢你的分享，谢谢你的分享。

我们今天这个小伙伴跟我们分享了一位清代的作者叫钮琇，他写的书里记述了一些志怪故事，写了一些有情节的文章，但专门琢磨了一件清代人写文章为什么那么扭曲。这确确实实与清的特殊历史背景有关系。为什么在清代时，文人进行书写会格外隐蔽？之前我们聊过，清的这种生态是极其不健康的，因为统治者不光施行了暴力和野蛮的操作方式，而且他自己非常清楚他干的就是坏事。这和之前很多暴力的军阀不一样：历史上也有一些很疯的统治者、很暴力的人，但那些人通常会觉得自己做的是对的。清代的统治者不同，他明确知道做的事情丧尽天良，但他一定要做，并且不准任何人指摘他做错了。这是他的特殊心态。

所以在这种特殊的历史环境下，出现了这样一群独特的人群：他们有文化，也有是非观。事实上中国文化中有相当一部分就是教你什么对、什么错——是非观。所以这些有文化的人普遍对是非观非常清晰。同时，我们从来对君子都有道德尺度上的要求，但到了清时，这件事完全颠倒了：愿意为朝廷鹰犬、愿意背叛自己的民族、愿意背叛百姓的人，才能成为朝廷中的一份子。这时，我们这些人面临到的选择，在清初有各色各样的文人做出了自己的选项：有一些拒不出仕，坚决不当官，且规定家中子弟不能当官；有一些开始蓄发避世，逃到山里去；还有一些人在家里院子修地窖，从此不见天日——这些属于避世类的选项。但要知道，儒学是一个非常世俗、现实主义的学问，它要求大家要入世，因为入世才能解决问题。这种想法也留在了文人的思想系统中。俗语叫“学得文武艺，货与帝王家”，本质就是儒学的入世精神：你学习的东西不透过秩序性的方式深入到人世、影响大众，那你学这个东西是干嘛使的？这是一个很底色的思维方法。

有了这些基础，我们相对就能理解那个时代的文人在创作时身上的那种扭曲感。因为他需要入世才有可能改变民生，才有可能影响天下的行走方向，所以他要出世；要出世，就必须让八旗认为你是狗，所以他必须当狗。他想让自己流芳百世，就必须拥有君子的正德；但他想要让朝廷认可他的身份，让他的书籍能传播下去，而不是把他们家的九族玩一波“消消乐”，那他就必须对这个反动的朝廷歌功颂德。每一对行为之间都是尖锐对立的矛盾，甚至非此即彼。所以这个时期的文人就表现出扭曲感：他既要写——这是他的责任，必须写；写的时候又必须歌功颂德；但他真正要倾诉的东西，在道义上、在规则上、在现实层面上应该倾诉的，却是底下血淋淋的真相。因此，这是什么？这叫做风月宝鉴。正面公子红妆，背面白骨如山。《红楼梦》是这些类型作品当中的一个典范，因为它遮掩得非常好，表述系统非常完善，所以能产生最大的传播力。一个书你没有传播力，说得再对也没价值，因为它产生不了影响。《红楼梦》好就好在它藏得好，表面文章做得好，让人觉得有谜语但又觉得故事完整，于是大家都去纷纷传抄。这种操作形式使它能够扩散出去。

这是一个底色。但只讲风花雪月多简单？容易，但也毫无价值，完全背离了一个人的初衷、信仰和追求，甚至背离了做人的基本底线。这时怎么办？他就要把真相藏在所有花团锦簇底下：在所有的花儿下面埋了一个东西——在《红楼梦》花丛底下埋了什么？埋了薛宝钗的冷香丸。冷香丸是什么？是一年四季所有花的尸体。它在花团锦簇之下，埋藏了已经败亡的真相。所以它所有的隐喻、册写，都在强调：你要来看呀！但看什么呢？如果你只愿意看那些美丽的、可爱的东西，也行，它写得很可爱；但在背后又有什么呢？就像我前几天聊过，为什么林黛玉在《葬花吟》里要那么悲苦？必须弄清楚。你难道没有困惑吗？《葬花吟》这个歌也很好听，但你在听的时候不会想哭吗？你有没有觉得那个词太惨了？问题在于林黛玉哪有那么惨？不管你觉得她是不是寄人篱下，她待在一个有传承有底色的亲戚家——而且是至亲家里——哪怕是寄人篱下，为了面子也不可能对她很坏。这样的情况下，为什么她的诗那样悲惨？这种悲惨呼嚎只是在闺中因为扯头花的事受点委屈就可以这样哭的吗？这是一个明显不合理的段落。所以你必须得问她到底在干什么。谜底在哪里？谜底就在癸酉本《红楼梦》里去啥啥啥。

在整个清代，文人的创作、艺术创作的能力其实已经跌到了地下十八层了。这是正经的。为什么会这样？没办法创造。诗是什么？诗言志。什么是诗言志？讲的就是我的情感、书法，我要把我的志向、情感、胸怀写进诗里，这才叫诗言志。但在清代，这些文人的志向与胸怀能写吗？一个真正有志向的文人，他的志向应该是什么？国泰民安。怎么着你也得是这样的志向吧？作为一个文化人，你给自己列理想时是当条狗，合适吗？书念得下去吗？如果说从一开始学之乎者也的那一刻，你立下的人生至高目标就是当狗，这个人念书是念不出息的。一个人首先要有梦想、有目标、有追求，这个事儿才能干好。但如果是清代的一个文人，他想要国泰民安，他能干什么？干不了什么。因为国泰民安意味着什么？民怎么安？很简单的道理：民吃得上饭、穿得上衣、能够出门行走游玩，放假时能表示自己开心快乐，民就可以安。但清的管理方式是种姓隔离的管理方法，整个八旗是完全的吸血型群体，所有百姓的身份就是纯粹的供养者，所有行为全部停留在一件事上——压榨。而任何朝廷鹰犬，只要供职，你要干的事儿就是帮助朝廷践行这种压榨。所以这时你说你要国泰民安，上陈词表时可以这样吹牛，但骗不了自己的内心。只要你践行现有的税率的执行方法、管理方式、驻军管理方式，就绝无可能国泰民安。但又得说自己要国泰民安，这时扭曲不？一定是扭曲。诗言志，你的想法和行为之间全部对立，这个志往哪儿输出？没处说没处讲。所以他们的文章不扭曲就怪了。

清时小说像志怪一样的平行时空。对呀，清的时候，志怪故事、鬼神故事还是有一些流传的。在这些故事中，你会注意到一件事：鬼特别多，而且这些鬼特别像人。为什么？其实他要说的就是人都变成了鬼。人什么时候会变成鬼？人死了变成鬼。所以你说他在讲鬼怪故事、志怪故事、山林里的幺蛾子，但实际上在讲的是：以前有人，而现在没有了。好，欢迎大家来到直播间。好的好的。清的时候，汉人的精神状态不正常，换成谁谁也正常不了。就那种生存生活的方式，你怎么可能正常？没有人能在那种环境下保持正常，这事做不到。
