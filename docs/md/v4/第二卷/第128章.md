# 第128章  #双标现场  #贵族精神 #基因真相  #行为艺术

啥？好像国内只有猫狗等才讲血统。  
哼，这句话很好玩啊——啊，国内只有狗才会讲血统这个东西。  
这句话本身很有意思。  
为什么说这句话本身很有意思？因为它的来历很好玩。  
我们可以讲讲这件事：什么叫“只有狗才讲血统”？这句话是打哪来的？什么时候出现的？非常微妙。  

要知道，从大概上世纪90年代开始，我们的各种书刊、杂志，到后来出现了网络化，之后就开始有了网络信息。所有这些从传媒的、从知识界的各种各样的信息网当中，向我们灌输了一个观点。这个观点今天还有很多人信以为真。什么观点呢？就是中国的主体民族——汉民族是一个混血的人群，只是一个文化认同体，没有任何共同的血缘来历，都是“串子”，都是“杂的”。而其他的人群呢，是有着高贵的血缘来历的，叫做高贵的血缘民族。如果是外国人，那就越发的强调高贵的血缘来历，信了什么东西？就是某某血缘的后代啊，某某真神的后代啊，什么什么高卢的，什么什么玩意儿啊，各色各样的这种东西。  

哎，这种看上去像是知识、但实际上是传媒信息的东西，铺天盖地。自上世纪90年代至今为止，始终都在大力的传播，很多很多人信以为真。只要你去跟身边的人打问一圈，问十个人里头往少了说，有六个人抵死会相信这句话——一定会出现这样的局面，这分你的圈子。  

然而，后来这个东西就发生了一些很微妙的变化。这个变化来自于我们国家进行了一个科学活动。上世纪大概是70年代的时候，国际上面设了一个组织。这个组织是干什么的呢？它最开始设立的目标，是为了印证西方人的血缘的纯净性——要科学的组织，对，科学的组织。但是那个时候的科学主流话语权掌握在西方人的手上。而西方人在过去的上百年的时间里面，就跟全世界宣传他们高贵的血缘来历。这个事儿大家都知道，至今他们还在不停的宣传他们的血缘贵族精神，动不动就是谁家谁谁谁的后代、谁家谁谁谁的后代这种破事啊，特别喜欢讲究这个东西。  

所以上世纪70年代的时候，在这种利益目标的驱动之下，国际上面搞了一个科学机构、科学联合体，打算运用基因溯源——当时刚刚把这个技术打通了——所以打算运用基因溯源这样的手段，去论证各个族群的血缘来历。这个玩意是西方白人主导的，目的就是找到他们高贵的父系祖先。然后当时中国也参与了。当时中国也参与了哈，参与来了之后呢，但是尽管参与加盟了，我们却没有开始调研。为啥？因为没钱、没科学家、没技术，只是报了名，但是没做这个事儿。  

后来，到了上世纪应该是80年代的时候才开始进行采样，就开始去进行基因调查、采样、搜集信息。然后组了一个小团队，去把这个东西进行基因分析。到了上世纪应该是90年代初的时候啊，应该是94年还是九几年的时候，我们这里就汇总出了第一批的数据。而这个数据能够公告天下，大约是在90年代末——我们把这个数据弄到全民皆知，已经是90年代末的事了。但是在90年代的早期，这个数据就已经出来了。  

好，什么数据？什么数据啊？这很重要。这就是我们——呵呵，我们顶着一个筐，我们顶着一个框——就是所谓的“中国，这里汉民族的血缘是最杂的、最串的，五颜六色的跟调色盘似的”。顶着这口筐去进行了基因采样调查，然后把这个报告一输出，发现：汉民族不光是在国内，在全世界范围内都是独树一帜的——父系基因、血缘来历最干净的一个群体，没有之一。上世纪90年代初的时候，我们的这个初步的调研报告就出来了。后来本世纪的大概是10年前后的时候，我们又有专项的调研，成果是差不多的，都是一样的，只是进行了细化，把里面的一些具体的元素细化归类了，但是结论上面，大方向上面是没有区别的。  

总之就是这么一件事：就是我们从大概94年、95年的时候得出了第一批的科研报告，然后获知了一个石破天惊的消息——汉民族的父系血缘来历居然干净到首屈一指，独霸全球啊，我的天哪！之前不是说好了我们最串儿，别人都是高贵的纯洁的血缘，只有我们是杂的？结果完全颠倒了，完全颠倒了。  

当然了，我们是很唯物的群体，测出来什么就是什么，我们总是能够解释它的嘛。这个事本来看上去是无所谓的，接受科学的现实不就完事了吗？承认实际上它不是那样的，不就完事了吗？我们完全可以用我们的历史来解释这个问题，这本来不是一个问题。但是啊，因为之前所传递的所有的信息，它实际上是一个传媒信息——就是你所相信的什么“你们都是杂种”这个事情，看上去像科学、像知识信息，实际上是传媒信息覆盖。那传媒信息这个东西，就不是说我出了个科研报告就解决了，解决不了。你必须要有一个对冲。  

而这种科研报告一出来，冲击到了原来的这套谣言系统之后，怎么办呢？对方啊，这个制造之前那种舆论信息的人群，就会开始反攻。要知道对方不是科学人群，之前他是编造的谣言故事。那当你科学打击到他的谣言的时候，他要谣言升级换代。这个升级换代是干的什么呢？啊，既然你的科研报告说了你们的民族父系血缘是最单纯的，那么下一波的谣言就进阶了，那叫做“只有狗才看血统呢，人看什么血统啊？”所以你刚才在说的这句话很好玩儿，你说“在中国只有狗才会看血统”。这句话恰恰好是在发现了中国的主体人群父系血缘极为单纯了之后，专门创作出来解构父系血缘单纯的优越论的。而在我们证明了自己的父系血缘单纯之前，在全世界所流行的就是父系血缘单纯是最优越的，而汉民族因为很混杂所以不行。等到最后发现原来是反的，他就说“只有狗才看血统呢”。  

所以这事儿你以为是科学，实际上是舆论战。

2. “初二” → 修正为“杂种”（原文“你们都是初二”是明显笔误或口语讹误，根据上下文“杂种”最贴合意思）。  
3. “筐” → 保留，但根据语境解释为“框子”（指被扣上的帽子或偏见），未改动。
