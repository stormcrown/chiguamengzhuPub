# 第166章  苏麻喇姑为啥狂煊洗澡水？丑破天的金钱鼠尾发展史 #洗澡水 #金钱鼠尾 #东亚 #服装

刚才有一个付费的用户提到“金钱鼠尾辫”，哦，我想起来了，不用你们再重复了，棒棒的。那个金钱鼠尾的眼镜，还挺……呵呵呵呵呵。关于金钱鼠尾辫这个发型，我们需要知道一件事：在北亚大草原上，有人有世界地图吗？手上有吗？你可以看一下世界地图。从我们的北方开始算，一直往西那么一溜，有一大长条，是一条草原通道。这个地方有一条草原通路。在整个草原通道上，一直有人生活着。这些人在历史进程中始终在不停地迁徙，因为这里降水较少，主要是草原地貌，适合畜牧业。而畜牧业非常容易伤害草皮，所以需要游牧，不断迁徙。在漫长的岁月中，持续不断的迁徙使得这片地带的人群处于不断游走的状态，非常不稳定。同时，也产生了另一个效果：他们之间有一定的联系。

这里还有一件事：在整个北亚草原通道上，哪个地方的游牧群体战斗力最强呢？这就不得不谈到“东亚怪物房”。靠东亚这一边的战斗力特别强。为什么呢？因为靠这一边的人想吃饱饭、想享受生活时，眼前有一个明晃晃的样板——中国古典文明。中国花了几千年搞生产、搞建设，非常富庶。于是产生的特效是：中国看起来像肥羊，但本质上组织能力、生产建设能力非常强，所以战斗力一点也不拉胯，反而很强。在这种不断的战争与博弈中，我们靠北边的这一部分草原，在整个草原大通道中，靠我们北部的部族战斗力最强。当我们的鼎盛王朝开始驱逐他们时，他们会向西游走，跑到西边去。一到西边就大杀四方。然后，人员的迁徙出现了，文化的流传也出现了，各种物资的流转也开始出现。由东向西，存在这种迁徙行为，已经发生了很多次。我们驱逐匈奴，他们往那边跑；驱逐突厥，他们往那边跑。反正一波又一波，都走这条通路跑掉了。

为什么讲头发会扯到他们身上？因为得注意一件事：这一整条特殊的地理条件、气候条件、生产物质条件，使得他们的生存环境非常恶劣。客观存在的事实是：缺少燃料、缺少水，而畜牧业非常容易沾染寄生虫。所以整体的生存环境很恶劣。在这样的恶劣环境下，这些人想要保存健康，有几项共性的生活基本准则。第一，他们洗澡的频次都比较低。整条通道上，受限于气候条件和由气候决定的生产条件，最终的生活习惯就是：很少洗澡，洗头风险极大，头皮上寄生虫很多。所以他们普遍有剃发的习惯。是不是突然就转过来了？一下就转过来了，咔咔咔的。从“东亚怪物房”转进到剃头，哈哈哈。所以，讲金钱鼠尾要从这里讲起，就是因为这个原因。

整个北亚通道上的群体都有剃头的习惯，不同群体用不同的剃法。例如契丹，比较早期曾向中国人推行过剃发易服，建立了政权，但不太彻底。他们留头发的方式是：这儿留两绺，后头剃了；有的在这底下又留两绺。他们剃头发的方式比较复杂，有很多种剃法，大体上就是这块留一块、那块留一块，其他的就没有了。后续紧跟而来的其他群体也各有各的剃头方式，有的剃掉这块、留着那块。大家可以去网上搜图片，剃得稀奇古怪。总归有各种各样的留头发和剃头发的方式。怎么区分呢？基本上就是一种分类方式——他们都剃，同时每个不同群体剃的部分和留的部分不太一样，基本是这个逻辑。

然后说到建州女真部，就是大家说的金钱鼠尾这部分。建州女真部自己选定了这样一种剃头方式。实际上女真系在剃头上之前也有一些偏差，剃的方位不太一致。但建州女真部专门剃了这样一个地方：就在这个地方留一小块，留很少很少的一小撮头发，这就是金钱鼠尾。之后，他们推行剃发易服，将金钱鼠尾辫推进到剃发易服区，也就是我们祖上生活的地方。我们以前讲究秀发浓密，结果咔嚓咔嚓就剃了，留发不留头，留头不留发，极为残酷。

那么，它的标准之所以叫“金钱鼠尾”，并不是他们以前就叫这个名字，主要原因是要求辫子够细——怎么个细法呢？就是剃完之后，这根辫子能够塞进铜钱的孔里去才算合格，否则就算不合格，要砍脑袋的。所以才会称为“金钱鼠尾辫”。这不是一个正面的称呼，包括经常用到的看似客观的名词，实际是带有轻蔑感和耻辱感的称呼，不是正面称谓。

后来金钱鼠尾变成了电视剧里面的那个半秃瓢，这有个演进的流程。金钱鼠尾推行了多久？事实上，有清一代260多年始终如一地坚持推行剃发易服。怎么推行？每个地方官的行政KPI考核中都包含了剃头方面的考核项目。剃头进入了政绩考核，管不了百姓剃头问题，官就不用当了。同时，八旗子弟提笼架鸟这件事大家都知道，常被拿来抨击说他们是小废物。但他们提着笼子遛鸟逛街的行为本身带着任务——监控。这些人是行走的监控器，他们的铁杆庄稼不是白吃的，要向其主做汇报，汇报看到谁家小谁干了什么事。这些人叼着烟袋、提笼架鸟、把玩小物件，看上去像混混，实际上是朝廷的监控录像、耳目。他们拿铁杆庄稼的收成做耳目之用，这是他们的工作项目。所以天天压马路、露露脸、逛街、溜来溜去，你以为不累吗？这是工作之一，要汇报，得知道乡里乡亲都干了什么，有没有谁说小话，都要报告。

剃发易服的金钱鼠尾实际上坚持的时间是：至少到乾隆晚年时还非常严苛。乾隆最在乎这个东西，不光在剃发上管得特别严，还制定了标准的服制。今天看到的那些清装、满装、轻民装就是他设计的，之所以不好看，因为他设计的东西都不好看。从乾隆晚年往前一直都是标准的金钱鼠尾。那么之后松了吗？其实理论上并没有松，注意，理论上没有松弛。但因为整个朝堂的管理能力下降，有一部分地区可能略有松弛，这是实操层面，而非制度层面。所以你所看到的只剃了一点儿的半秃瓢是什么时候出现的？是晚清时期中国南方出现的。实际上看晚清时期的照片，在北方地区的照片上，留的依然只有这么一小块。大家可以自己去晚清照片集里找，八旗群体以及各大城市里头发依然只留一小块。只有那些鞭长莫及、管制相对松懈的地区，留的头发可能多一些，但不是不想管，也不是制度改变。所以电视剧里那些只剃了脑门的一点点半秃瓢，是一个标准的美化行为，尤其当演的是八旗子弟时。实际上八旗子弟一直到最后也只留这么一小块地方。八旗子弟一直到清末也只留这么一小块，这是他们的标志物，他们不以为耻，反以为荣，认为这是身份象征——有独特外貌特征、着装特征、着装特权的人，在社会上拥有特权和特殊话语权。所以一直到清末，八旗子弟剃的依然是这一小块。因此，最近那个什么鬼电视剧（我喷过的那部，挺恶心的），但里头的发型还不错，还蛮还原的。就是八旗子弟到晚清时依然剃的是这个样的，所以缠起来的头发像一坨屎，就是因为这个缘故。电视剧《人生若如初见》啊，对对，就那个电视剧，被我喷过、怼过，但它里头的发型还原度不错，夸一下。让我瞅一眼啥？哎呀，我加班了，下班下班下班，再见啊。
