# 第127章  满语变成了东北话？谢邀，你不够格

您觉得汉语常常表达与行动不一致，词不达意。可您想过没有，“说一套做一套”这句话本身就是在批评一个人的品行？那叫做言行不一，即言辞和身体力行不一致。以您的文化水平，我得说，我这个直播间确实有点文化门槛——起码得超过小学五年级的阅读理解水平。您达不到这个水平，在我这儿就会很痛苦，因为永远接不上话茬；就算接上了，也是驴唇不对马嘴。

您还提到东北话是满语。如果东北话是满语的话，那你们怎么能听懂我说话呢？呵呵，这纯粹是胡说八道。满语这个语言没有能力影响汉语——不是它不想，而是没有那个能力。它压根儿做不到。在一部分俗语的角落里，可能有个别词汇与满语有关，但在结构和语法上，它无力影响汉语。为什么？因为汉语的传承远比它丰富，结构远比它稳定，词汇量也远超它。因此，一个高维的语言不可能被低维的语言从骨骼上影响。只是有些可能需要用到的地方，可能会借用部分词汇——这种可能性是存在的。

至于说满语“先天发育不良”？这是因为，当年这个语言从蛮荒走入文明的速度过于快了。要知道，语言的发展与人群正常的、不断的进阶相关。我们自己的语言也是在上万年间逐渐变化，才成为今天的模样。我们也在调整：有了新东西要想办法起名，有了新叙事方法要不断调整，甚至因为迁都、国家重心偏移，核心的官话口音也会受影响。大家主流认为是雅音、官话的口音也会发生变化。所以语言会不断变化、成长。但满洲部落的语言，它的成长并不是一个正常的过程，所以它没能力影响别人。因为他们在创造这套语言（包括配套文字）时，文字也是借来的。当他们着手打造这套语言文字系统的那一刻，他们刚刚甩脱部落民的身份——而且，他们甩脱部落民身份的方式不是靠自身成长，而是依靠暴力进行劫掠，硬生生地从别人那里学了一堆东西，一下子就升级换代了。这种快速升级，在几十年间走完别人数千年道路，直接导致它的语言进阶跟不上。别人积累的词汇，它压根儿没有；别人说的概念，它想都没想过、听都没听过。这时候它怎么影响？只能借鉴。所以只会是汉语影响满语，满语没有能力影响汉语。这不是想不想的问题，也不是有没有马、有没有刀的问题，而是凭实力做不到——做不到就是做不到。

再回到“言行不一”这个话题。这东西是个人品、行为方式的问题。要求言行一致，恰恰是汉人这个群体因为文化绵延时间长、对自身道德要求比较高。事实上，前些年公知批判我们时，就包括“道德文章太多、规则要求太多、不洒脱”。他们批评我们道德标准太高、细则太多、不洒脱。但道德标准高，意味着什么样的标准要求言行一致。在汉人这里，言行不一是一种背德行为。但有没有呢？全世界所有人群都会出现言行不一，因为言行不一能带来私利：我答应你的事不做，实际上是在占你便宜。所以言行不一是一个占便宜的行为。只要是人，在没有道德伦理秩序、没有约束、没有熟人社会指指点点的环境下，都会普遍存在言行不一。

那为什么您总觉得汉人特别容易言行不一，而其他人都特别诚实？因为您武侠电视剧看多了。不为了别的，就是因为您武侠电视剧看多了。这些作者和编剧非常执着地让大家相信汉人都是虚伪、表里不一的，而其他来自蛮荒之地、苦寒之地的人都是特别朴实、真诚的。这是一个刻意打造的形象，跟现实毫无关系，也一点都不唯物，纯粹是舆论传播上的刻板印象。

所谓“仓廪足而知礼节”。对呀，我们是有这种基本总结的。谁会更诚实？吃得上饭的人会更诚实，穿得上衣的人会更诚实；而饥饿随时会夺走生命的人，会不惜撒谎去保命——这是天经地义的。包括中国古代的人，都不会对贫寒交加的人提出道德要求。有些人说孔子动不动用道德来打击被统治阶层、说他是阶级敌人——之前很多人这样说。但实际上孔夫子那个时代规定的所有道德，针对的是贵族；压根儿没有针对平民、黔首。这就是为什么“礼不下庶人”？因为做不到——就是做不到。穷困潦倒、需要持续工作维持生计的情况下，不能提出至高的要求。

包括在新中国，我们一直在说劳苦大众是品行最端方的人群。这建立在什么基础上？建立在为所有工农人群奠定了政治地位的基础上，也建立在工农立国、进行平等分配的前提上。有了这些，才有了尊严；有了尊严、体面，才有了知礼节的前置条件。有了这些东西，普通人朴素的正义感才会彰显光芒。所以什么时候、什么情况下，道德标准应该列到什么档次，是有限定条件的。因此，认为一个地方越穷困道德就越高的想法肯定不对；但说穷人缺德、富人有德就更不对——因为道德去要求有钱人，恰恰也意味着他们特别容易贪婪、特别容易放肆，所以才专门列规矩去要求他们。
