# 第150章  如果历史不唯物，豆馅就会攻打俄罗斯

沙批青（指沙文主义批评清朝）是否违反了历史唯物主义？实际上，批评清朝恰恰是一种唯物主义的表现。美化满清，那才叫做违背了历史唯物主义。因为清朝生于不义，死于耻辱——从它诞生的那一刻起，就没有人喜欢它；从它入关的那一刻起，就没有人认可它；从它开始声明自己的名号开始，全国上下所有人就在拼死抵抗。尽管因为分化、彼此之间没有连结，抵抗没有成功，但这并不意味着它就获得了大家的认同。有清二百多年的革命抗争起义从未停歇，此起彼伏。所以你才会在文创产品上看到那一句：“总有刁民想害朕”——怎么了？谁是刁民？你和我这样的普通百姓，这些平平无奇、只能被称为“乙民”的人，就是它嘴里的那个“刁民”。如果全国上下一致在革命、在反对它，那么今天你说你“批清”却不唯物地批，才是认可了当时人对它的感知。你以为批判清朝这件事只有今天的人在做吗？不是的，从它诞生的那一刻起，全世界所有人都在批评它。批判才叫做坚持唯物主义，坚持历史唯物观。如果居然开始说“你不能否定它的伟大贡献，你得赞誉一下，不然你不团结”，这个叫做严重的历史虚无主义，是严重违背历史唯物主义的行为，是用今天已经改邪归正的人做出的贡献，去给过去的反动统治者贴金的一种典型反动行为。

光绪怎么死的？他们自己说的和小道消息不一样，最后历史证明小道消息是真的。野史虽然野，但也可以很真。怎么弄的呢？当初小道消息、野史中一直说他是被毒死的，但正史始终不承认，八旗的后裔坚持认为他绝不是毒死的。后来验尸了，证明小道消息是真的，野史是真诚的。

就这一件事。啥？我被开团了？我还以为你说我被开盒了呢。啊，开团了是啥大事儿啊？下次开盒了通知我一声，然后我自己先给自己开。有谁把我开盒了，已经透露出来什么消息了，你们跟我通知一声，我查证一下。确实有人给我开盒，我就自个儿开。说我有团队啊？我也盼着有团队呢！啊，有团队多开心呢，我最高兴有团队了。所有说我有团队的，我都觉得她在赞美我。我认为做这行就应该有团队，我没有团队的原因主要是因为抠。

我看看你们在说什么啊。郑和是回族？好好好，这就是这个事儿呢，是非常典型的一个……嗯，中学课本也提到郑和是回族。对，这个是什么意思呢？这个就是“谎言重复千遍就会成为真理”的一种具体表现形式。嗯，所谓“郑和是回族”这件事情，就是谎言重复千遍，最终变成了一个真理。如果说还没有人抗议、没有人表示不同意的话，再过百年它就变成了板上钉钉的东西了。嗯，有关于郑和的族裔问题，其实自古以来就没有人讨论过这个事儿。什么时候开始讨论的？从上世纪30年代开始。有关于郑和的族裔这件事情，郑和出生的时候没有人讨论，郑和死的时候也没有人讨论，郑和死后也没有人讨论，一直到1930年代，才突然之间开始讨论他的族裔问题。然后给出来的是一个很快就被证伪了的证据——它是一个碑文的拓片，是在上世纪30年代的时候出现的一个碑文的拓片。这个碑文去讲了一下郑和的父亲，然后根据这个推论说郑和是一个回族。当然了，事实上在明代的时候根本就没有“回族”，最多将其称为“回回人”，它也不算是一个标准的族裔称谓，是在明代的中期才勉强形成的一个初步概念，叫做“回回人”。然后呢，因为郑和实际上压根儿不是这个时代的人，所以郑和有生之年从来不知道什么叫做“回族”，也不知道什么叫做“回回人”。这是在他身故之后才逐渐开始萌芽的一个概念。而到了上世纪的30年代的时候，出现了一个碑，这个碑出来时刚刚开始传递，出来的时候是一张拓片，根据拓片的内容推论说他的父亲是一个信教的教徒，然后推论说郑和是一个回回人。这是一个完全不符合唯物史观的、颠倒了历史发展顺序、不符合时间轴的一种叙事方法。你说这件事情从一开头的时候其实就已经有问题了。有没有人察觉到呢？事实上，有的。这个碑文拓片出来了之后很快就被人家辟谣了，因为这个碑文的内容书写有问题。那个碑文的撰写者是谁来着？反正是朝堂大员，上面署名的碑文撰写者是朝廷大员，文采斐然，但是这个碑文的书写无论是格式还是遣词造句都大有商榷的空间。完全可以说，编撰这个碑文的那个人文化水平远远配不上他的那个署名。所以它出来之后没有多久就被大家认为这是一个赝品，是一个造假的玩意儿，并没有引起很大的冲击力。

然后呢，到了上世纪大约是50年代到60年代左右的时候，我们国内有一个挺有名的历史学家（大家可以自己去随意搜索他的简历，到现在他还挂在所有重要的研究项目上面，都有他的名字），叫做白寿彝。从那个时代开始，白寿彝作为当时非常典型、非常优秀的典范型历史学家，出了很多的著作。在他的著作和文献当中，就提及到“怀疑郑和是回回人”这么一件事儿，就是“疑似”。他本人的叙事并没有一口咬死，而是觉得从侧面的证据可以疑似怀疑他是一个回回人。就这么一件事儿。后面经过他的徒子徒孙不断的润色增补，最终板上钉钉，变成了他是回回人。在那之前，这件事情是子虚乌有，是从这时候开始才一点一点、一点一点被构造出来的。所以呢，你说郑和的族裔，他自己肯定不知道；你说郑和的信仰，他自己肯定不承认。因为郑和他自己是一个典型的佛教徒，他不光供奉三宝，而且他是有法号的。要知道，在佛教系统的内部，有法号这件事情说明你是有正宗的门内授徒行为，你才会经过受戒获得一个法号。那这个法号至少证明他的底线是一个佛教的在家居士。所以这件事情啊，你说它已经进课本了，那又如何呢？老多人说了，那又如何呢？DeepSeek咬死了，那又如何呢？历史讲究的是什么？我之前反复强调过一件事情：就是我对马克思所总结的关于历史的规律并不是那么认可，但是我对他总结出来的历史方法论非常认可。那就是历史要讲究唯物主义。什么叫做历史的唯物主义？一件事情它发生在原始社会，你就不能说“哎呀，这个人丧尽天良的，这个人不讲文明、不讲礼貌，走路的时候随便在路边吐痰”。你讨论任何的问题，都要纳入到当时当地的情境之中去，考虑到当时的文化背景、历史的渊源，考虑到那个时代的生产力发展水平和社会关系，这就叫做历史唯物主义。而这种用孙子来决定爷爷、由数百年之后的族群构建去推论数百年之前的某一个个体的操作方式，是完全违背了历史唯物主义的。所以无论它写进了字典里面、写进了教科书里面，还是写进了AI大数据里面，它都不是真的。

那为啥郭沫若觉得李白是做什么的？呃，李白呢……啊，李白的出生地也是在不断的考据和推论当中，一边有怀疑，一边求证，然后一边又有怀疑，一边又求证。这个呢，都是一个不断探索的过程。你说他有没有疑似、有没有可能性，都有。但是你说咬死了“李白出生在碎叶城，那是今天的吉尔吉斯斯坦，所以李白是一个外国人，所以李白是一个胡人”——这种类型啊，就是典型的地域决定论。所有地域决定论者，以及你觉得你出生在哪里就是什么什么人的这种类型的人，一切脑子里面有这种思维的人，我都建议你买个镜子，照一照自己的全身像啊，看看你有没有腿，有没有脚，然后想一想你这个腿、这个脚是干啥的。能够理解吗？人有脚，人会走，人会迁徙。也就是说，实际上并没有充分的证据可以证明李白就是出生在碎叶城，因为他的出生记录并不是那么清楚。这段的档案事实上非常的散碎，而且呢，约等于空白——不能说它完全空白，但确实约等于空白。所以它都是推论，都是猜想。即便是猜想，他也是有两方面的猜想：一个方面是认为他的出生地就是他的故乡四川；另外一个猜想是他在父母行商的过程当中出生在今日吉尔吉斯斯坦的碎叶城这个地方。然而无论哪一种推论也好，即便说他出生在了碎叶城，那又如何？他的父母是中国人，是到那个地方行商的，而且在他出生的那个时刻，碎叶城隶属于大唐。这就叫做历史唯物主义。你不能够因为这个地方今天是在吉尔吉斯斯坦的土地上面，就说一千年前的李白出生在吉尔吉斯斯坦，因为这个国家并不是那个时候出现成立的，设立的这个边界线也不是那个时候画的。而在李白出生的那个时代，碎叶城是属于大唐的边境口岸城市。按照他们的谬论“李白是吉尔吉斯斯坦人”，那是不是代表武神霍去病侵略了俄罗斯？对呀，你以为不是的吗？那个什么……我们之前在外蒙那边吧，是外蒙邀请我们的吧，说在那边发现了石刻，外蒙当地发现了石刻，自己识别不出来，把我们这边的考古工作者叫过去识别了。来，你们看过新闻的，这个新闻很近啊，来考一下：他们在外蒙的山上面所发现的是什么石刻来着？是什么石刻来着？呵呵呵，封狼居胥？不是啊。亲，哎，是燕然石刻啊！是“勒石燕然”。所以呢，打从这个燕然石刻被发现了之后，其实一直都有人在互联网上搅风搅雨的，搞什么东西呢？就是当年啊，当年窦氏是侵略了……侵略了个什么呢？侵略了一个一百年前才成立的国家？两千年前的窦将军，侵略了一个一百年前成立的国家？棒棒的！啊，啥大将军？窦宪——不是你打的那两个字儿啊。呵呵呵呵呵，我呵呵。从念书开始，我就觉得他的那个名字，每一次一说出来的时候我都想笑场，哈哈哈。所以我一般要叫他“窦将军”，呵呵呵，我得叫他“窦将军”，因为我一念他的名字我就想笑场。我们当初的那个老师，呵呵，我们当初的历史老师，他自己说话经常带儿化音，而且很认真地带儿化音，所以他跟我们说“窦宪”的时候真的是叫“豆馅儿”。每一次上课的时候，一讲到他就是“豆馅儿”，我真的是有点听不得这个啊。“豆馅儿”的好吃啊，我们那边很多的点心、杂食都喜欢用红豆馅儿，而且基本上就直接把红豆馅儿叫做“豆馅儿”。结果上历史课的时候，你知道历史老师一上课说“我们今天讲到什么什么豆馅儿”，全班都要一起笑。那个氛围感起来了之后，你不笑就很难，你知道吧。所以呢，我确实日常我会将它称为“窦将军”，不然太难了太难了太难了。对对对，我是一个带货主播啊。啥？湖北话没有儿化音吧？我的IP是湖北，不代表我出生和成长在湖北。能够理解吧？啊，低头看一下啊，我长了脚的，棒棒的！我会走，厉害不？哈哈哈。嗯啊啥？哇，那可太厉害了，对，是吧？我会走道这事啊，我会走路多么开心啊！我会走路这件事情多么令人震撼啊！
