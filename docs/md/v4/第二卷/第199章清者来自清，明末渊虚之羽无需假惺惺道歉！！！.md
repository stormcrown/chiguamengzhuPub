# 第199章  清者来自清，明末渊虚之羽无需假惺惺道歉！！！ #明末 #渊虚之羽 #大明 #崇祯

哪怕是到了全民文盲的清末，大家的心里面对于反清复明的这杆旗子都还是认的。为什么能够认？因为崇祯皇帝用他的死洗干净了这个旗子。如果前面的一个王朝都跟溥仪一样的，说认野爹就认野爹，说改姓就改姓，随时随地跟仇寇之间绑在一起，说日满一体。如果末代的皇帝是这个样子的话，后续哪里有人能够举起他的旗子？他们举的是什么啊？举努尔哈赤的旗子，举皇太极的旗子，举康熙的、雍正的旗子，有的时候呢，更不要脸一点的，会举乾隆的旗子。但是没有人举溥仪的旗子，因为举不动。因为溥仪自己就认栽了，自己承认自己是一个卖国求荣的大汉奸，所以这杆旗子没法儿用了。你会看到它们铺天盖地的在那里说：“哎呀，我是那个什么第一巴图鲁。”但是呢，他不敢说他是溥仪在世啊，啥？因为溥仪也不举。嗯，啥？满一退了是吗？啊，那我是不是应该继续战斗？是因为我这会儿有点平和吗？我是不是应该继续战斗？我再战斗，战斗，是不是他们就回来了？嗯，好的。嗯。

崇祯皇帝吊死在煤山上几百年，也没唤醒中国汉奸的人性。都当汉奸了，哪里来的人性？但是啊，但是崇祯皇帝的这个决策很重要。事实上呢，当一个王朝的君主愿意死于社稷的时候，就意味着这个王朝他最后是干净体面的离世的。而这种选项带来的影响就是在后世，它的名号始终可以拿来用。不只是名有这样的作用，事实上连宋都有。因为宋代的末帝蹈海而亡，所以呢，朱元璋他的这个革命行动，实际上就是一个反元复宋的行动。所以他后来会专门的写诗：“日月重开大宋天啊。”所以朱元璋啊，朱元璋实际上呢，在做的事情是什么？是反元复宋啊。不是说你反元复宋，最后立的国号就得是宋。前几十年常常说：“哎呀，生个儿子继承香火。”这玩意叫重男轻女呀，所以这个东西呢，他就被含混过去了。动不动就说“我家里面有没有皇位要你继承啊”什么的，就开始嘲讽。继承香火和这个东西完全没有关系啊。继承香火的香火是什么？香火是祭祀权。祭祀权是什么？祭祀的意思是我可以祭祀这个天，这是天子的权利啊。天子的权利即天，当我是天子可以祭祀天的时候，凡是在这个天道以下的所有人，都是我可以组织、我可以调度的。我这个旗子一举起来，你们都要来看我啊。这个呢，就叫做组织权。

所以祭祀当你是一家一户的时候，当你是一家一户的时候，就意味着什么？你家里面姓陈，你家的宗族长啊，拥有尘世的这个供头香的权利，这就叫祭祀权。这个祭祀权，你说是谁还不能点个香呢？哎，他去点这个香的意思不是点香，他是什么？是所有姓陈的子弟，我在点香的时候，你都得来看我。这是什么？你名义上叫做香火继承，叫做祭祀权传承，实际上是组织权传承。那组织权传承这件事情它可以分散吗？不能啊。组织权是不能够分散传承的。组织权一旦分散，这个地方就单门立户就分了，你的每一个小个体的力量就分散了，你就约等于无啊。组织权一旦分散，你这个地方就约等于零了，就散兵游勇，什么东西都抗争不了，连自己的利益都保护不了了。所以我们为什么要强调所谓的香火传承？他讲的并不是财产，说我们家好穷的，我不用管。而是你想要保有你这个家族名义的存在，就需要在乎这件事情。因为他要的就是这个组织框架和组织权。所以香火传递、祭祀权传递啊，这个国家正统权的传递，它其实都是组织权传递，也就是我从谁的手里面接过了这杆旗帜的意思。

所以你看共和国立国的时候，我们是从中山先生手里面接过了革命的旗帜，这个就是一个香火传递，这是什么组织权的传递。我因为拿到了正义，拿到了人民的感召，拿到了正义的属性，所以这一杆旗子是干净的。我这个旗子举起来，全国所有人都要向我看齐啊。这个时候这个就是一个组织权的传递。所以你才会看到说，我们共和国立国的时候，为什么特别的尊重中山先生？为什么叫另一派叫做反动派？为什么要把他和中山先生区别开来？明明他们都是国党，因为我们决定要从中山先生手上接过这一杆干干净净的革命旗帜，然后传续到今天。这玩意儿就叫做国统传承，这个才是传递的一个原理。所以古人那里叫什么？叫做香火，叫做祭祀。啊，今天就叫做旗帜。这个东西是保持了一致性的。啊，这是一个组织力的传承法。

所以崇祯皇帝为国而死于社稷这件事情，他在干什么？他在洗涤旗帜上的污垢。因为他作为一个君王，作为一个天子死于社稷，于是呢，他的整个的这个号召力就没有变。即使他在这个具体的事情上面办糟了，但是他这个旗帜这个象征意义是好的。当你的象征意义是好的时候，只要有一个真正有力量的人，把这个旗子举起来，他就有效啊。这个呢，就是一个传承力量。所以大家会在那个——应该是阿Q的那本书吧？是啊，革命军白盔白甲为皇帝戴孝啊，是有这个说法吧？我，我记得应该是这个事啊。那放到这里的时候，为什么像阿Q这样的一个完全离文明老远老远了，没有任何文化啊，没有任何文化，然后心思甚至都有点飘忽的一个小民，但是呢，他知道这么一件事情：就是白盔白甲的革命军是在为皇帝戴孝。为谁？哪个皇帝啊？为崇祯皇帝戴孝。也就是说，哪怕是到了全民文盲的清末，大家的心里面对于反清复明的这杆旗子都还是认的。为什么能够认？因为崇祯皇帝用他的死洗干净了这个旗子。于是呢，它的象征意义就保留下来了。即使那个王朝早就消失了，他的继承人也早就被人家屠戮殆尽了，但是即便是这个样子，它在人们的心中还有一个种子。甚至到了清末，历经近300年的这样的奴役，这样的文盲化的操作，人们心里头依然还有这么一个影子。当你为崇祯皇帝披麻戴孝的那一刻，就决定了你的立场所在。你站在了这个干净的旗子下面，你这个旗帜就有用啊。这个东西就是作用。所以崇祯的这个殉国的行为，它本身是有历史价值的，不光是在精神层面有历史价值，实际上在物质的层面上面，它依然有这个价值。如果不是因为他采用了这样的干干净净的方式，那么后面的200多年间，此起彼伏的每一年都存在的农民起义与革命，举什么旗号呀？找什么借口啊？

如果前面的一个王朝都跟溥仪一样的，说认野爹就认野爹，说改姓就改姓，随时随地跟仇寇之间绑在一起，说日满一体。如果末代的皇帝是这个样子的话，后续哪里有人能够举起他的旗子？我们公屏上雇佣的这一支又一支的小蛆们，他们都不举溥仪的旗子。他们举的是什么啊？举努尔哈赤的旗子，举皇太极的旗子，举康熙的、雍正的旗子，有的时候呢，更不要脸一点的，会举乾隆的旗子。但是没有人举溥仪的旗子，因为举不动。因为溥仪自己就认栽了，自己就出去拜了码头认了爹，自己承认自己是一个卖国求荣的大汉奸。所以这杆旗子没法儿用了，哪怕是这些不停雇佣的这些蛆虫们都不敢沾啊。所以你会看到它们铺天盖地的在那里说：“哎呀，我是那个什么第一巴图鲁。”嗯，尽管巴图鲁也不是什么好词儿，但是呢，他不敢说他是溥仪在世啊。啊啥？因为溥仪也不举。又开车吗？我这里是一个幼儿园，不带这样的啊。呵呵呵呵呵呵呵呵呵呵。
