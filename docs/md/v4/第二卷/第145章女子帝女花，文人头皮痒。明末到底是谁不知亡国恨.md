# 第145章  女子帝女花，文人头皮痒。明末到底是谁不知亡国恨

我们的个体越脆弱，与这个国家的兴亡关联就越紧密。那么，当你去看到风尘女子的时候，你说她们是坚强还是脆弱啊？大家在公屏上扣一下吧。风尘女子每天可能得到的打赏，都是以万金而计的。这样的风尘女子，我问你，在整个社会大局动荡的天下面前，她是强还是弱？

刚才说到那个，呃，秦淮八艳。要知道，秦淮八艳在我们今天——包括说你在刚才提的时候一想呢——就是什么李香君、柳如是。但实际上这一群人，这个八艳的角色是一直都有的，只是不停地换人。为什么到最后有名的是这最后的这几个呢？是因为她们做出了符合时代的潮流，因为体现出了她们自己的独特的高风亮节，所以被我们格外地记录下来。但实际上，以秦淮八艳这种身份而言，她们本身是不固定的人啊。干什么的呢？她们就叫做交际花，放到民国时期就是交际花啊。她们是明代的时候的一种交际花的类型。那她们在干什么？哦，你就得知道那个秦淮八艳的相处方式，她和一般的这种风尘女子是不太一样的。因为呢，她的社会往来的角色当中，一直包含着相当多的一对一的关系。你会在看她的故事的时候发现，哎，这些人好，怎么好像看上去像在正常的谈恋爱啊？虽然谈的是一个有点浪漫、有点独特的恋爱，但是她看上去更像是谈恋爱，而不是像是一个青楼女子在出来倚门卖笑。这就是她身上独特的符号特征，也是她独特的价值所在。这个呢，就是属于说，嗯，它达到了一定的层级。无论是她的才艺水平、文化水平或者颜值水平，已经突出于、超越过了普通的青楼女子的时候，她的价值就和其他的女子不一样了。她就不再只是普通的——还有颜值更高一点啊、要价更高一点就不同了——量变引发了质变。她的身份在这个时候变得非常的微妙，因为呢，她已经有能力进行大规模的宣发了啊。注意一件事啊，就是大规模的宣发干什么？就是不断地通过传唱的方式，通过说书的方式，通过讲段子、写话本子的方式，把她们的才情，把她们的奇闻异事传递出去。然后，她会成为她们那个时代的流量明星。当她成为人们心目中的流量明星的时候——来，在座的各位啊——去看流量明星的圈儿的时候有没有发现啊？看他们节目的人都会在下面做一件事情，叫做组CP呀。组CP，她和谁和谁一定怎样怎样，这个CP也是她的故事环节的一部分。当你组了一个足够浪漫、足够咖位的CP的时候，你的传播力又更上层楼。当你的这个圈子里有这样的角色的时候，意味着你这个圈子的档次都格外的高。于是呢，你去到这个地方，就似乎是进了剧组里面去探班了啊。你想一想，是不是和今天的这个圈子——这个什么粉圈啊、什么街机啊——这一套东西老相似了？运行的模式高度地相似。这就是在青楼的风尘产业线上面增加附加值。当然，这个部分呢，是指秦淮八艳的固定岗位，她的人是不断地轮替的。但是呢，这种操作方式进行到了晚明时期的时候，局面就开始发生了改变。

任何一个人，包括你，也包括我，我们所有的普通人，都是和我们的国家休戚与共的。我之前做过一期节目叫做《讲的就是月薪3000和这个国家的大事之间有什么关系啊》——有这么一期节目，很早以前的了。那么这里头讲到的就是，我们像我们这样的普通人，与国家的兴衰之间的绑定关系是非常非常深刻的。于是我们就知道一件事：一个人他的个体越脆弱，与这个国家之间就越紧密啊。我们的个体越脆弱，与这个国家的兴亡关联就越紧密。那么，当你去看到风尘女子的时候，你说她们是坚强还是脆弱啊？大家在公屏上扣一下吧。风尘女子每天可能得到的打赏都是以万金而计的。这样的风尘女子，我问你，在整个社会大局动荡的天下面前，她是强还是弱？对的，她们很有可能很富有，很有可能每一个人都不止一个百宝箱，但是她们无比的脆弱。脆弱意味着什么？脆弱意味着什么？意味着她非常非常需要那个宏大叙事当中的、了不起的王朝来给她撑腰。一个社会内部的秩序越文明、越有规则、越能够保护弱者，她在这个地方才能够保有一定的尊严和一定的可能性。所以在这个时候你会发现一种迹象：晚明时期你会看到这些风尘女子，这些以前的交际花儿们，在这个时候居然表现出了格外的气节，甚至比跟她组CP的那些才子们还要有气节。为什么？因为那些勋贵也好、才子也好，他们在社会动荡的变革期的时候，事实上他们是有了某一种封侯拜相的幻想啊——能够理解这件事情吧？就是原本他可能只是一个富家子啊，但是呢，他有才名，可是想要争出头又不太容易。如果这个时候改朝换代了，我又押对了宝，这叫什么？这叫做从龙之功。所以在这个时间的节点上面，在这种社会转换期的时候，有一些有野心的人、觉得自己还有一点点才华的人，实际上会做一种选择，就是押宝在他们身上。你能够看到明确的这种选择和犹疑的迹象。干什么？这是自我认知不坚定的一种表现，那同时也是他自身的实力相对较强、较为稳定的一种表现。他觉得无论投靠哪边，他都是能够有价值的，所以他会做这种选项。

但是这个时期的风尘女子，也包括了很多大家族当中的女子，却表现出了一种非常非常坚韧的品格。为啥呢？想一想，在社会的混乱期，女性的处境是比男性要惨淡的。社会动荡期的时候，女性的处境是远远比男性惨淡、悲惨、可悲的。而当时明清易代交界的这个动荡期，它不是普通的改朝换代，因为当时其实所有人都能够看到，关外来客是一群什么样的人——是一群过于落后、制度性的落后，甚至是过于野的一群人。而一个社会的体系，当它也落后的时候，它就代表着女性的处境会格外地堪忧。而明朝是一个女性的社会地位已经走到了相当不错的一个阶段——这个时候的女子会非常敏锐地察觉到，风雨要来了。所以她们为什么在这种品格上面，会显得格外格外的坚韧？甚至会出现在明清易代的交界前后出现的文学作品当中，都开始出现了尊女的情形。你们一定会察觉到啊，明清交界的这个附近，各种各样的文学文艺创作当中，会非常地赞美女性的品格。这是一种社会的评价的风气。为什么这个样子？因为这个时候确实有很多的女性，尽管处境卑微，甚至社会的等级非常的低下，但是她们保留了自己的情操。那为什么这个样子？因为她们不是可以投机的人，她们没有条件投机，她们没有条件选择，她们只能硬扛。这是根本利益诉求上面的不同。

嗯，我们谈到根本利益诉求的时候，就一定要带这个了啊。来来来，整波毛选啊，整波毛选。嗯，怎么分析人群？怎么理解一群人的集体选择？我们呢，就要通过毛选的分析方法来进行解析。你看个体的时候，每一个人有他自己的成长履历，有他独特的身份、独特的视角，甚至偶尔会有人超越于自己的身份。但是当我们作为群体去考察一群人的时候，他会有一个相似的行为模式。那这个区分的方式，就来自于他的根本利益的不同。还是那句话：有背叛阶级的个人，但是没有背叛阶级的阶级。所以呢，在这个事情上面，我们一定要学习毛选里面的分析方法和思考方法，去判断在不同的时代背景之下，不同的利益群体会做出的截然不同的选择。什么时候、什么人和我们是有着共同利益的，这个时候我们用什么样的方式和他达成一致，最终达成目标。这个就是毛选里面能够教给我们的应对事件的分析方法和解决方法。所以所有的中国人家里面都应该有一套，而且呢，看旧了之后直接换，再重新翻，因为常看常新。什么时期发生什么样的事件，我们都能够在里面找出相应的方法来，去思考、去应对。注意一件事情：我们不要本本主义。这也是毛选里面不断地教给大家的东西，而且专门有这个篇章告诉你，千万不要本本主义。这个不要本本主义是包含了毛选本身的——所有的书都不要本本主义，不要刻舟求剑，不要去把它里面的哪一个金句背下来然后就说这就是真理啊。啊，没有哪一个句子可以拿出来单独地当做指导思想。但是呢，整套的方法确确实实在可以预见的、长远的未来都能够成为我们的指导思想。就是你能够在里面学到一些什么：怎么去看待问题，怎么去分析问题，怎么去解决问题，怎么去面对面前的这个人所说的话到底是符合了哪一个群体的根本的利益？他站在了哪一个群体的内部？他站在了哪一边？这时候会有理中客出来说“你这不够客观，你在选边站，你在站队”。很遗憾的是，这世上所有的人都是有立场的啊。你是农民的孩子，我也是。嗯，我们绝大多数的人都是工农的子弟。中国绝大多数的人都是工农的子弟，只是有一些人忘记了啊。有一些人啊，这几十年来，因为日子过得太舒服了，以至于他忘记了自己从哪里来，忘记了自己是谁。于是呢，你就会发现他怎么那么夹生啊，一会儿要东，一会儿要西，既要又要。那这些人呢，就是很明显的一种忘本的行为。他忘记了自己的来路，不记得自己是谁，不记得自己的根本利益是和谁放在一起的，或者说他忘记了根儿在哪儿。我们，我们有多少人是不是工农的子弟？中国绝大多数的人都是工农的后代。我们知道工人是怎么谋生的，知道农民是怎么种田的吗？记得这些东西，记得自己是从田里来的，是从工厂的一线的每一颗螺丝下面走出来的，你才知道自己要站在哪个地方，你才知道自己要站在哪一边。知道自己是谁的时候，你就不会说我们要客观、我们要中立。两边一强一弱掐起来的时候，你说要中立，你已经选边站了。
