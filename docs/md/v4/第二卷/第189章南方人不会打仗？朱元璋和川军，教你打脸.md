# 第189章  南方人不会打仗？朱元璋和川军，教你打脸 #历史真相 #文化冷知识 #互联网热梗

有很多反贼喜欢到处气人、呵斥别人。  
第一件事啊，我就是天天被他们骂成反贼的那一个。嗯，如果你每一天都被人家骂作反贼的话，你可能也会心气比较平和——这是第一件事情。  

第二呢，就是我不太喜欢把谁谁就定性为反贼。为啥呢？因为这个词儿的含义，它是一个不确定的含义。如果你硬要说“反贼”这个词要让它确定化的话，它应该是指犯了叛国罪的才应该算是反贼吧。不然的话，因为一言一行与自己不是同一队列，他就是反贼了？那这种定义的方式是在什么呢？他在掌控互联网执法权。为什么要这样说啊？因为啥玩意就叫反贼啊？你咋知道反贼呢？因为他跟你不合，于是他就是反贼，你就是爱国者？这东西是啥？这是在试图掌握互联网定义权。有人有资格掌握互联网定义权吗？有人有资格掌握汉字的定义权吗？有哪一个在互联网上，随随便便的一个小网红，就开始舔着脸说“我这样子就叫做爱国，你那个样子就叫做反贼”？它配吗？谁都不配。因为汉字它是一个表意文字，含义是固定的。反贼是什么？反贼指的是一个人有叛国的行为。所以尽管说痛恨我的人很多啊，乌烟瘴气的玩意儿老多了，但是尽管他们劈头盖脸的骂我是反贼，但是我从来都不说他们是反贼。因为我是没有执法权的，我也没有裁判权。不管他干了什么，我都不能说他叛国。哪怕我知道他们说的东西在胡说八道、在造谣诽谤，但是只能说他诽谤了，说他造谣了，你不能说他是反贼。叛国罪要求的证据是很多的。但是这个互联网上就是有一些人开口闭口的把与自己不和的人称之为反贼啊。那这种行为是什么呢？这就是拉队伍、站队列，要求你站队。  

嗯，好，欢迎来到直播间啊。问一下为什么北方民族能横扫天下，南方那些怎么不行呢？你觉得朱元璋算南方算北方啊？嗯，那个你挂狗头也没有用啊。来，你说一下，你觉得朱元璋算南方算北方？从南往北打的这种算不算南方呢？嗯，朱元璋算什么呀？扯一个吧，好不好啊？我不喜欢这种地域决定论啊，我是超不喜欢这个东西的。真的是冲出来就什么南方怎么地，北方怎么地，什么意思？是不是要搞地域撕裂啊？包括你在说，如果你硬要说为什么草原民族那么那么嚣张呢？因为他打仗成本低呀，能理解吗？这个事儿是聊过的啊。为什么我们会被生产力水平较低的、接近于部落联盟阶段的群体所攻陷？是因为对方的成本低。他在往丰饶的定居人群当中进攻的过程当中，他甚至可以不用带粮草，随便的烧杀掳掠就可以进行补给。战争成本不一样。而且实际上他们的战胜率非常之低啊，他们的战胜率很低，因为生产力和组织能力都不行，战胜率非常的低。但是他们只要赢一次就会摧毁一大片。所以你老是记得他就是这么个原因啊，他们日常情况下都并没有战胜。这个是客观实力来决定的，你放到什么战场上都是这个样子啊。但是呢，等到他真的遇到了某一个雄主，哗啦一下把人都集结起来的时候，然后又刚好逢上这边的内乱，两种情况同时存在，就会呼啦啦一下子啊，因为他的战争成本实在是比我们要低太多了。所以这个玩意儿啊，它不是一个战争的能力的问题。  

实际上呢，你看我们在抗战期间，你你们这边说了是吧？嗯，我们在抗战期间的时候，特别能打的几支队伍来自于什么地方？川军。我就问你，川军算南方算北方？广西兵算南方算北方？论战斗的能力，南方兵差吗？一点都不差。著名的小个子，但是打起仗来的时候一点都不差，非常的优秀。所以压根儿不是什么南边行不行、北边行不行的问题，而是定居群体的征战的成本远远高于非定居群体。嗯，啥？戚家军就是在沿海地区征召的部队，其实也是南方人啊，战斗能力也很强啊。所以啊，这不是南北的问题。  

嗯，啥？战马它提供的是机动性。注意了啊，战马提供的是机动性、机动属性。它最重要的是降低他对方——降低他自己的损耗。就是遇到攻而不克的时候，他撒丫子就跑，你追不上他。这是他最重要的首要的功能。第二功能才是攻城略地。实际上进入到真正的这种农垦区的时候，战马能够起到的作用——在攻城略地上起到的作用就比较有限了，因为这边的地形复杂，不适合这种非常快速的奔袭，他的那个战马的这种能力发挥不到最大。但是呢，不管怎么说，有机动属性，至少说跑得快啊。  

嗯，也和地方民情有关。戚家军的精基础精锐是义乌的矿工，这些人抱团儿打得都狠啊。是这样啊，就是之前聊过，就是关于征兵标准的问题。古典时期已经总结出了很多征兵的东西。说那个矿工的事情就是我们的征兵标准：什么样的兵源是质量最好的？边缘啊，我们从先秦时期就做了各种各样的总结。我们先秦时期啊，有春秋，有战国，大家不停的打仗，在打仗的过程当中，就已经汇总出来了很多的总结的经验。什么样的人群适合于战争呢？你看秦国最后给自己搞的什么功啊？耕战立国，找谁？找农民啊。为什么？因为农民是老实头，家里面有地，他有恒产这个概念。放到后世的时候变成了什么呢？啊，放到后世他叫做良家子。到汉的时候，他征兵的标准就是良家子，就是家里面有田产、有稳定的生活，是个老老实实的妥当人啊。我们的这个征兵的标准就从原来的那个农民啊直接转进到了良家子，他们之间是一条线索的类型，相差无几，就是找这种老实人。这些人是什么？特别斗狠吗？没有啊。不找武林高手，也不找那些该溜子，也不找那些剑客，不找那些浪人，这些人都是不找的。为什么？因为我们当年在先秦时期做过实验的。那个叫什么来着？是哪儿的？叫——技击。先秦时期诸侯征战的过程当中，曾经组织过用游侠、剑客这些武林高手组织起来的部队，叫做技击。全是高人啊，打一场输一场，足足快要把国家都快打废了的那种，哈。啊，游侠，对啊，就是找的这些人。然后这些人你说他个体能力行不行？那行，整一条街上他最狠，他最厉害啊，吆五喝六的就都是这种人。集成部队之后，有一撮败一撮，只要打仗必输无疑。后来就发现了，打仗和你的个人的逞凶完全不是一个概念。所以要什么？战争需要守秩序、有规矩，要令行禁止。我让你走三步，你不许走两步；我让你往后退的时候，你不许往前面冲。这个东西才是战争当中能够用得上的兵源。  

所以为什么要找良家子？因为良家子可以听话，可以受训。他不会因为桀骜不驯就说“听我的啊，我就是我来我打架的时候最厉害，我小爷如何”，他不能找这个啊。一定要找老实人。所以为什么矿工可以？因为矿工和军队的管理之间有异曲同工之妙，都是要服从组织管理。你在矿上你不听组织管理，我让你凿两个镐头，你非得要凿三下，你能把大家都埋进去，这就是生产安全事故。所以为什么后来我们说革命的时候，要发动工人阶级的力量？工人阶级有什么特征？一个他是无产者，他愿意往前面冲啊，这是一方面；另一个方面，工人是有组织、有纪律的群体。所以你征兵的时候条件是什么？这个人有可能变成有组织、有纪律的人，这个是好兵员。最重要的要求。什么个体的强壮啊，是不是凶悍呢，这都不重要啊。最重要的是他愿不愿意服从管理，你能不能够让他快速的开始服从管理，他会不会挑事儿，会不会作妖？这才是兵源最重要的事。所以，宁可去找一个老老实实的农夫，你都不要去找那个身手不凡的该溜子。这个事儿我们验证了2000年的，好使。所以任何的时候啊，那些什么刺客、游侠、高手，军队里都不欢迎，不要他们，用不上的啊。  

啊，游侠不利于兵法？那肯定的呀。因为战争是干什么？战争是两个团队之间的斗争，他不是个体的斗争，它是两个团体的斗争。他要讲究目的性的是吧？你往前不停的去冲，你看着你冲得很猛，但是你影响了整个的大局。我们要求的是停在这个地方保住这条防线，结果你不停不停的往前冲，你就把自己的防线葬送了。所以他是要服从调度。哪个群体有组织、有纪律、有服从性，就挑哪个群体啊。这个是军队筛选最重要的要求啊。  

啥啥啥啥玩意儿啊？到点下班是吧？到点儿了吗？啊，好的，到点了啊，再见再见。
