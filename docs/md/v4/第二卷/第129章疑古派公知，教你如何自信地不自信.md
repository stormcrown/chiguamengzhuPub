# 第129章  疑古派公知，教你如何自信地不自信

不管你在这里多喊什么“你爱我”“我爱你”，都没用啊。我说把你家房本写成我的名吧，啊，你立刻说：“那个我不行，我不要，我不可以。”对吧？所以他说他爱中国，他说喜爱中国文化，行不行？行啊，可以啊。你来旅行呗。中国文化那么了不起，那么辉煌又灿烂，你喜欢是天经地义的事情，怎么还需要我谢谢你吗？这里地方那么美，这里的资源那么丰富，你爱这里是天经地义的事情。怎么着？你过来占我的便宜，我还要谢谢你吗？你来了，我邀请你，我请你吃饭，我请你观光，然后到了时间之后礼送出门，这是与人往来的正常的方式。而不是“我爱中国呀，我得是中国人呐，不然的话你就不好客了呀，不然的话你就盲目排外了呀”。凡是喊这个话的人，我默认他从小到大没朋友、没邻居，没见过陌生人。

这个事儿也很好玩儿。按说的话，应该是棋逢敌手的时候，才会视你为敌，对吧？所以我们很多的普通人可能会误以为：我们今天强了，所以对方与我们为敌很合理啊。确实啊，当你强起来了，有威胁的时候，视你为仇敌，这是一件比较有合理性的行为。但是事实上并不是这样。事实上，我们从民国开始就抬眼望去，举世皆敌。众所周知的，我们那个时候很弱呀，既穷且弱，但是所有的列强都视我们为仇敌。怎么个视我们为仇敌法呢？他不光是冲上来揍一顿的问题。冲上来揍一顿这件事情很有可能是以大欺小，很有可能只是为了抢一波。但是视为仇敌这个性质和抢一波、想打一顿不一样。他是特别恨你，特别想要毁掉你。无论你有多弱，他都想要拆掉你，他都会想要撕裂你，他都想要让你消失。哎，这个才叫做视为仇敌的心态。我们尽管当年从清末也好，民国也好，我们都是很弱的国家，但是一直以来被列强视为眼中钉、肉中刺。怎么证明这一点？你会发现啊，在那个时期，卯起来奔着我们抢，哎，抢也就算了啊，他抢所有的弱者。但是呢，他在那个时期会对我们进行很多文化和舆论上的攻击。这种文化上面的知识建构上面的进攻，是需要下大力气的。这和直接说我扇你一巴掌、抢你一个钱袋子不一样。要知道，我揍一顿、抢个钱袋子，只需要我体格好就可以了。但是我要把我说的话塞进你的脑子里面，然后让你自觉自愿地给我当跟班儿，是需要付出极大的耐心和努力的。而当年的列强对中国做的事情就是这个。他们在想方设法地向中国这里投入教育资源。这也是之前很多的公知在拼命去西方列强洗地的时候，强调说：“他们在我这里开设了很多的教育机构啊，教育机构啊。他们在其他地方可不这样干，但是在我们这开设的教育机构很多，而且不光是基础教育，还包括高等级教育。”干嘛呢？因为教育可以改变一群人的思想。费大力气，花大投入，花漫长的时间做长期的规划，就是为了改变一片土地上一群人的思想和认知。这样大的功夫是只有对待真正的仇敌才会下的力气。这不是简单的“落后就要挨打”。那为什么要这样做？是啊，那么弱，你怎么能够成为别人的仇敌的呢？想怎么抢就怎么抢的一个肥羊，你有什么资格当别人的仇敌的呢？所以很明显啊，在对方的眼里，我们并不是那样一个单纯的小弱鸡。所以很有可能在全世界相对强大的国家当中，最不了解中国历史性质的人是中国人自己。嗯，敌人比自己更了解自己。周边的列强都把我们的祖上视为极其强大、极其有扩张力的群体，所以他们始终将我们视为仇敌，想方设法地从根本上面打击我们的自信，打击我们的认同感，摧毁我们的认同感。

甚至前几十年的时候，有很多的公知纷纷地向我们宣传一项东西，就是我们应该抛开历史，轻装前进。背着历史的包袱，就像个蜗牛一样动不了。但是欲灭其国，先灭其史啊。所以谁会下这个力气？谁这么怨恨中国的历史？谁呢？啊，谁操作了，谁就害怕我们。他为什么要这样恐惧？因为他心里很清楚，中国人的衰败是阶段性的，而我们以前曾经是绝顶的强盛，这个强盛的程度是可以摁着他打的。只有在这种情况下，他才会觉得你今天尽管弱，但是我依然恐惧，我还要从根儿上去拆解你。所以一定要让你觉得你的血缘不好，一定要让你觉得“你爹是你爹”这件事情是一个错误，一定要你觉得有历史这件事情很悲哀。这都是过去几十年公知孜孜不倦地向我们传递的信息。你回看过去一些年的杂志，里面充斥着这样的论调。还想方设法地让你相信，只要你去混血，你的后代就会比你好看。实际上就种下了一个种子：认为你又丑又笨。这是什么？这是对于自己基因分类的一种不自信。而这种不自信和什么“认为自己很穷所以不自信”完全不是一个档次的。穷困是可以发达的，但是如果基因上面有短板，那是不可改变的天生的东西。所以他们想方设法地就在基因的层面、遗传的层面上面拆解。我们拼死地要告诉我们：只要串了就会好。这句话后面的潜台词就是：你太烂了，不管怎么串都会变好的。而且他想方设法地希望我们的后代和我们长得不一样，想方设法地告诉我们说：“你只要文化认同，我爱中国就可以了。”请问真的可以吗？历史和世间的所有的现实都告诉正常的人类：喊一声“我爱你”，就说我是你的家人，这件事情在哪里都不能够成立。历史的教训乌泱乌泱的，现实的教训也乌泱乌泱的。但是他还是会反复不断地用各种各样的营销号给你传递一个信息：我爱中国就是中国人，来了中国就是中国人。你们今天所看到的很多的爱国营销号也是这样说的。你真的相信？嗯哼，别人就赢麻了啊。

至于说“别人都已经爱中国了，你还要怎么样呢？视别人为仇敌吗？”谁说他爱了中国，就得是中国人才算当朋友？没朋友的吗？我喜欢你家的房子，我就得登到你家户口本上的吗？不可以是做做客吗？我喜欢你家的饭，不可以是你请我吃一顿饭，然后我就回家吗？为什么我喜欢吃你家的饭，所以我就得是你们家板上钉钉的亲生的手足同胞了呢？谁规定的这种事？凡是说这种话的人，我都默认他从小到大没有朋友，没有邻居。认识他的就是他的亲人，不认识他的就是他的仇人。他的世界当中只有亲人和仇人两个类别，非亲即仇。而像我们的正常人，这里除了亲人和仇人之外，还有朋友，还有邻居，还有陌生人啊。啥，那不行，你不能分我的财产，你也知道对吧？

嗯，不管你在这里多喊什么“你爱我”“我爱你”，都没用啊。我说把你家房本写成我的名吧，啊，你立刻说：“那个我不行，我不要，我不可以。”对吧？就是这个道理啊。所以他说他爱中国，他说喜爱中国文化，行不行？行啊，可以啊。你来旅行呗。怎么了？中国文化那么了不起，那么辉煌又灿烂，你喜欢是天经地义的事情，怎么还需要我谢谢你吗？你爱这个地方那么好，这里地方那么美，这里的资源那么丰富，你爱这里是天经地义的事情。怎么着？你过来占我的便宜，我还要谢谢你吗？不存在这件事情。你喜欢过来做客，可以，我很慷慨，我是一个很好的主人家。你来了，我邀请你，我请你吃饭，我请你观光，然后到了时间之后礼送出门，这是与人往来的正常的方式。而不是“我爱中国呀，我得是中国人呐，不然的话你就不好客了呀，不然的话你就盲目排外了呀”。凡是喊这个话的人，我默认他从小到大没朋友、没邻居，没见过陌生人。

真有自信的人都是这样的啊。包括说那个大家经常去嘲笑阿三对吧，但是也都知道阿三是真的有自信的人。不管他干得有多烂，但是他就是有自信。所以你说他任何的不好，它都会自动地转化成：你觉得我很好啊。你请我吃饭，那肯定是你欠我的；那你对我笑，那肯定是你等级没我高。这是什么？这就是一种盲目的自信的心态。因为他认可自己，对自己的认可度极高，就觉得自己老好了。虽然这个事很不唯物，很不客观，咱们不能那样干，但是咱们得知道什么样子的人叫做自信啊。真正自信的人不会因为说“你挺帅的呀”，于是“哎，你真是啊，大善人一个”。只有你觉得自己不帅的时候，别人说你帅，你才会感念得如同见到了恩人。如果你真的觉得自己活脱脱就是一个在世吴彦祖，别人说“啊，你还挺帅的呀”，你会说“对，你真是慧眼识珠”。这玩意儿才叫自信，你知道吗？嗯。所以啊，别人冲上来说：“哎，那个什么啊，我好喜欢中国呀。”你说“太好了，终于有人喜欢我了”，这是自卑心理。很多人喜欢说：“哎呀，我们现在自信了，我们麻了，我们赢了，我们太厉害了。”别人说爱我，别人说爱你，你就觉得你赢了。这件事情是一个极致的自卑。真正的自信是什么？真正的自信是：他终于说“啊，我这人挺好的”，可算是长眼睛了，瞎了那么多年，咋治好的呀？不容易啊，抽空过来看一看。嗯，开开眼界吧。这玩意儿这才叫做自信。你对自己的认同感足够高的时候，你是不会感激别人对你的赞美的，只会认为别人赞美了这个事儿，他终于做对了点儿事儿。
