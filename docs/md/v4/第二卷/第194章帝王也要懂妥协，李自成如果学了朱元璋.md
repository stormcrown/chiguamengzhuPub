# 第194章  帝王也要懂妥协，李自成如果学了朱元璋 #朱元璋 #帝王心术 #历史智慧 #权谋

啊，朱元璋是吧？朱元璋打宋旗之后，又承认了元的正统。嗯，就是朱元璋的这一段。我之前呢，嗯，应该是什么时候？那老早了。我刚刚开始做直播的时候啊，曾经讲过朱元璋的这一段东西啊，包括说朱元璋一开始的时候，他跟的一个政权也叫做宋啊，就是那个龙凤政权，因为年号是龙凤，被称之为龙凤政权。后来呢，反正啊，这个政权也没了啊。为啥没了呢？因为他们家的主家韩林儿，被朱元璋麾下给咔嚓了，然后这口锅他必须得背下来。最后呢，就自己举了旗帜啊。那不管他举的是什么旗帜也好，他一开始的这个操作的方式，很明显的是一种反元复宋的操作方式。在那个时代，他呢，和后世的反清复明一样，是那个时代可以借的一种旗。我们知道所有的革命起义也好，战争也好，它都需要你有一定的理论的依据，要有一定的正义性。不管是谁都要有一定的正义性，那就是说你就算找借口，你也得给我找个借口出来啊，意思是你必须要编一个借口出来。所以呢，他为什么要使用这样的方式？因为宋的这个旗帜，由于宋的王室蹈海而亡，从而洗干净了旗帜上面的污渍，所以呢，反元复宋的一个旗可以用。因此他的革命当中才曾经使用过这个反元复宋的口号啊，这个理由曾经是用过的。包括后世呢，他其实也是承认继承了宋的国祚，这件事情都有，它是有延续性的。

但是中间曾经有一个短暂的时间，朱元璋呢，很含混啊，他对于元的定性变得很含混。什么时间呢？就是他从立国开始到大约洪武20年左右为止啊，就是这中间的20年，大大约是这中间的20年，朱元璋呢，对于元的性质变得很含糊。朱元璋在他立国安稳了之后，曾经呢，有约等于20年的光阴啊，对于元的态度非常的模糊。他不再像之前那样子劈头盖脸的就开喷，而是用了一些相当柔和的手法去应对元这个政权。甚至呢，在他的这个即位的诏书当中啊，怎么说，勉勉强强也算是夸了元两句吗？啊，反正呢，就是没有再骂了啊。总而言之，就是他从即位开始啊，他就有20年的时间在国内基本没有骂过。但是呢，从20年之后开始，局面就变得不一样了啊。从洪武20年之后，他的国内的诏书当中就开始写上了“元非我类”，就不一样了啊。

那为什么？实际上呢，就是因为啊，一个国家可以马上打天下，没有办法马上治天下。那治天下靠的是谁？治天下你要靠文人、士大夫。换谁来都一样，你必须要用他们。因为这样的一个庞然巨物，像我们这样的国家啊，你没有办法靠一些兵头把这个国家管好，他还能活下来呢，做不到就是做不到。你要让这个国家维持生产、维持秩序，你就需要无数的文人去践行旧有的这一套完整的社会秩序。但是在明初的时候，巨大的缺陷有一个巨大的毛病，就是朱元璋的团队虽然英才辈出，有一些人也号称文武全才，但是呢，拿来进行整个国家的治理，那是远远远远不够的。这就意味着他必须要妥协，必须要吸纳之前所留下来的官僚体系、士绅体系、地主体系，他必须要跟这些人妥协。而这些人是什么人？这些人在元之时啊，那可都是降了的。有一个算一个，这些能够就职的大小官僚系统里面的从业人员，在元之时全都是降了的。所以真的是要算起来的话，他们全部都是汉奸。按照朱元璋原本起义的标准，他们全部都是汉奸，人人得而诛之。但是能这么干吗？不能啊。一个这样子兵马立国的国家，刚刚开始安稳下来，需要休养生息的时候，你把之前那些能干活的吏员们一个一个的都嘎了，说你当了汉奸的理由倒是很正当，问题在于你真嘎了之后，你这个摊子就散了，你懂吧？所以这个阶段他呢，措辞相当的含糊啊。

哎，我们建国初是不是也这样？我们建国初也有这味儿啊。那个时候我们为什么要不断的……我们立国之初的时候也有这方面的问题，就是我们国内的文盲率太高了啊。能够做活、能够干活、能够认字儿、能够好好做事的人，里面有文化的那帮子，要么是遗老遗少，要么是反动派麾下啊。纯粹革命队伍里面长出来的，然后还有文化的人，数量远远不够治理一个国家。所以你看我们建国初的时候，是不是搞什么文艺座谈会呀，然后去约束知识分子啊，不停的对知识分子的思想导向进行批判呀？干什么？就是要让这些思想上面很有可能有问题，在既往的历程当中其实有黑点、有污点的人，学会如何照我们的规矩办事。明初的时候也面临这种问题，因为手上能用的人太少了，所以他没有办法，他必须做出一定的妥协。他不能够破口大骂，他真的是搞破口大骂的话，国内的很多的士绅阶层那简直战战兢兢，没有办法做事，甚至很有可能在底下勾连起来，你就不知道他们要干什么了。所以为什么朱元璋在立国之后，有20年的时间非常含糊不清，甚至会让很多人认为说他承认了自己继承蒙元的国祚等等是吧？这些事情你能产生误解，就是因为他含糊，确实在对于元的点评上面，是一个趋于正面的点评，所以你才会造成了这样的印象啊。这个呢，是朱元璋在那个时代所做出的一种政治上的妥协。

但是啊，老朱可不是什么好脾气的人，能理解吧？他可不是什么好脾气的人。所以呢，你看他在当时啊，明朝国内一直是很含糊的，甚至有的时候说两句好话。可是呢，他在给周边的藩属国去发放国书的时候，那可就从来没客气过了。说那个什么元、什么兴山、什么非我类什么的，乱七八糟的，跟骂街一样的话，都在那个给朝鲜那边，给朝鲜那边的半岛，还有给中南半岛这边儿去写国书的时候，那里面都是这些词儿啊。嗯，谢谢。他当时在对外写国书的时候，恨不得破口大骂，但是呢，在国内的宣发上面含糊不清。而这些破口大骂的东西，到他洪武20年，在天下已经治理安安稳稳，有了自己的秩序了之后啊，就在国内也开始有了。所以朱元璋什么态度啊？他为什么似乎曾经还说过元的几句好话呢？哎，原因就在于这个啊。

啊，然后那个刚才说到那个什么摩尼教啊，呃，他呢，曾经有过很多种的名字，但是其实在宋代的时候，应该就已经开始在国内流行了。它一般在什么情况下流行啊？摩尼教呢，在宋代的时候，在中国国内的一些地区流行，是由于经济问题。因为呢，这些我之前说过，宗教也是一种社会组织形式。而宋代的时候，由于他的一些经济治理方面的问题啊，使得呢，他的一些基层的农户，在经济上面趋向于需要找到一些社会联合，甚至社会保障。这个大家作为平民子弟都能够理解哈，我们打工仔月薪3000的那种，都盼着说我得有个社保什么的，希望有一个组织能够分配一定量的什么低息的贷款，然后希望能够提供一些基本的保障啊，希望有这个东西。但是呢，宋王朝在很多的乡村地区，这个事儿他是办得很不好的，尤其是他征税的效率不错，事实上也意味着他在末梢神经的管理上面，有点儿搜刮过剩啊。再加上他那个没有制约文人士大夫的土地集中，宋是没有制约文人士大夫群体的土地集中，土地兼并的啊。他不制约土地兼并，就会使得农民的生活变得不好，他就希望获得某一种支持保障。那摩尼教在这个时候干什么？就是提供这个啊。他作为一个教门组织，他在从中协调低息贷款啊，然后呢，他在从中协调借贷关系当中，偶尔的返利行为，这相当于一种变相的扶贫。其实呢，放到田氏代齐里头，这种方式叫做市恩，就是我卖恩情给百姓，从而获得名声的行为。所以之前他一直是受到打击的，这种组织形式是受到打击的啊。为什么呢？因为民间需要，所以这个组织呢，就借此牟利，借此营造自己的组织。但是很明显的，他又违背了我们大一统管理的方略和方向和所有的细则，所以他也受到一定的打压。就是在这种夹缝之中啊，摩尼教在这样基层的领域当中一点点成长起来。所以你说元末的起义当中，摩尼教有没有参与的可能性？极大的可能性。为什么？因为生活无着的底层的农民，很多的时候都会被这样的管理方式裹挟进去，它是一种经济分配方式啊。但是思想控制能控制得到吗？其实思想控制的东西很有限。这个点上面我们要有一点自信，对我们的祖先啊，在思想领域当中，我们对自己的祖先要有一点自信，就是这些教我们所讲的故事，文化水平都不够啊。这个东西他文化水平不够，这个事儿他骗的人有限，不是因为被骗。相信我，那个时代的我们的祖宗曾经参与过这些群体啊，他们大概率的不是因为被那些什么思想欺骗了，而是因为他们确实需要一个有一定保护机能的基层的组织帮助他。

呃，感觉这 up 是历史科普主播，怎么评论区这么乱？有啥瓜吃吗？没有瓜吃，我这里是一个带货直播间啊。这啥玩意儿啊，这我怎么突然之间变了身份呢？算了，不重要啊，不重要。反正那个啥看见了吧，我产品全都给你弹出来了啊。带货直播间不许弄差了知道不？这玩意儿吓得我一哆嗦，突然之间给我转向了，说我是什么知识主播，这不行啊，这不能够哦。

哇，我国主要民族汉奸的比例高，比其他大国主要民族出叛徒的比例高，是错觉吗？汉奸不是什么违禁词吧？你这搞的这个我都看不明白了啊。啊，有关于汉奸这个事儿，我是聊过的这个事啊。这个事肯定是错觉。你一定要说汉奸比例特别高的时间呢，那可能在清的时候啊，清到民国这个时期，我们的汉奸比例确实与其他的任何时代都更高一些啊。但是呢，你直接觉得跟其他国家、其他族类比起来，我们都特别高，这事绝对是错觉。为什么？因为我们在乎汉奸啊，我们很在乎背叛者。谁背叛了，我们要给他记住。谢谢纯洁啊，谁背叛了我要记在小本本上呢。从中行说开始，我们就记叛徒，一直都记叛徒。嗯，然后呢，嗯，但是我们对于别的群体叛逃到我们这来的人叫什么？叫弃暗投明。知道吗？啊，别的群体叛逃到我们这来的，那叫弃暗投明。至于对方有没有说，哎呀，他是叛徒，我记一个小本本，诶，啊，他没记下来。我们拥有全球最完整的信史系统档案本啊，日记本在我们这记着呢。我们特别记仇，谁背叛了我们一直拿笔记着。然后隔壁其他的那些人他都不记，然后他们那里叛逃到我这里，那种不算叛贼，算弃暗投明。所以你就会觉得我们这里的叛贼的比例特别高啊。真正说叛贼比例特别高的时期，就是清和民国时期，这个时期是清朝的200年奴役而造成的，出现了越来越多的人对于自己的认知出现了含糊不清的表现，所以呢，开始放纵了啊。他不知道自己是谁，也不知道什么叫做尊严和体面了，这个时候他觉得有奶便是娘，谁强谁是爹啊，就是这个样子了。
