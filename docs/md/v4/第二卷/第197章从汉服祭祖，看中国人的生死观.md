# 第197章  从汉服祭祖，看中国人的生死观 #汉服科普 #冷知识救命了 #文化传承 #历史那些事儿

我岂不是红了？我这肯定是红了啊！不然呢？嗯，讲你的历史吧。讲什么衣服？我是一个带货主播，我爱讲啥啊！而且谁说衣服就没有历史了？衣服是谁设计出来的，不是历史吗？衣服是什么场景下谁穿的，这不是历史吗？它不光是历史，还是现实，还是自我认知，还是文化的表现形式。怎么着了？我一个带货主播，我要聊什么天儿，居然还有人约束起来了！说你好好聊你的历史，你不要聊衣服。所以谁可以聊衣服设计？旗袍的设计师？我是学服装设计的，我很会做旗袍了，我可以聊衣服。呸！嗯，啥服装也得随时代发展。不符合时代要求的事物，该进博物馆就进。所以汉服是民间自主复兴的，结果你们都拦着他说“该死了，该死了”，让他去死。人民自己要他活着，人民百姓的要求干你何事？老百姓喜欢你，不喜欢你算老几？老百姓说这个衣服我要复兴，你说让他去博物馆，让他埋起来。你是哪位啊？汉服运动是起自民间，起自草莽，是一砖一瓦由人民百姓构造起来的殿堂。你们这些人要是说文化、衣服这事儿不要紧，由他去吧，让他死吧。他是自己活过来的，是中国的百姓一针一线地让他活过来的。既然你说由他去吧，不要管吧，你咋那么多话？他自己都活过来了，轮得到你说他要不要进博物馆吗？阁下是哪位啊？

嗯，好，欢迎来到直播间啊。啊，抖音上好多小黑子。何止现在，连小破站都有好多小黑子啊！我这边条件越来越好了，知道吧？我现在条件越来越好了，以至于连小破站上面都有黑子了啊。之前小破站这边还是非常清静的，以至于有的时候觉得在小破站上面呢，经常难以找到可以怼的对象。结果现在小破站上面的水军都开始涌起来了。刚才是不是有人说那个什么汉服是丧服？刚才是有人说这个事了吧？刚才是不是有人说汉服是丧服、给死人穿的？有没有人说这个事儿啊？我晃眼看见了。哇啊，有是吧？好啊，这事！好，我们就来聊一下汉服是不是丧服。它是丧服会如何？不是丧服就会如何啊？说到这事儿的时候，你必须得弄明白什么叫做丧服。能够理解什么是丧服吗？能够理解什么是给死人穿的衣裳吗？能理解吗？凡是说这话的人都理解不了这件事情。什么叫做给死人穿的衣裳？请问死人入殓应该穿什么？你们觉得死人入殓应该穿什么？嗯，丧服就是平时的服装。这不就对了吗？中国人的思维叫什么？事死如事生。我们为什么在很多的古典的墓葬当中都能挖出来丝织品的陪葬？我们是不是挖掘了很多丝织品的陪葬？为什么要把这个玩意儿陪葬下去？事死如事生，要让过世的人如同活着一样的生活。所以我们要把一年四季的衣裳给他配齐了。这是家庭条件不好的，就把她最喜欢的一件搭下去就行了，其他的舍不得。但是呢，不管怎么样，总而言之，它的原理就是：你活着的时候穿的衣裳是面对着世人的，而过世的时候身上的衣裳是面对祖先的。因为在中国人的传统的思想当中，鬼是什么？鬼就是归啊。鬼鬼啊，就是归去的人。归？归到哪里去？归到来处去。所有人都归到来处，那个地方是哪里呢？那里就是我们祖先聚居之地。所以你会发现中国的先秦的人士特别的仗义轻生死，动辄就把人头借给你。为什么？因为在中国人的概念当中，你只要完成了人世间的任务，你的去向是很清楚的，就是和祖先走到了一起去。我走得干干净净，体体面面啊，为大义而死，那么我到祖先面前的时候就是个体面人。所以为什么能够做到那么的轻生死，都违背本能了？随便的一下子，咔嚓一下，就我把脑袋借给你了，你去用吧。我们先秦也有很多这样的人的，包括秦汉时期这样的人都很多。为什么？因为我们的传统的逻辑当中，死亡就叫做归去。那么当它的概念叫做归去的时候，你想：我要回老家了，我应该穿什么啊？过世的意思就是我要回老家了，回到祖先聚居之地了。那回老家你应该穿什么？穿你最漂亮的衣服呀！亲，你啥时候回老家不是挑一件最漂亮的衣裳？所以什么叫做原理？我跟你讲的都是基本原理啊。我这里讲的都是基本原理，就是我们的先民、我们的古人去定义这件事情的时候是怎么操作的，他是怎么思考的。在它的基本的伦理的概念当中，人过世了，身上应该穿着的衣裳是什么？是要能够面见祖宗的，这是非常根本的事儿啊。你穿的这件衣裳的目的是为了见祖先，所以为什么要挑你最好的一件衣服去？这和你回乡去见爹妈的时候应该穿最漂亮的一件衣裳是一样的。就是这个原理啊！视死如生。你考虑先人和考虑自己，用共情的方式来考虑，在中国这里就完全可以直接达成了。有很多的细节不需要那么讲究，但是基本的原理你要符合，就是你见他就如同见一个活着的祖先。因此在有清一代的200多年间啊，有清一代的200多年间，中国人活着和死了穿不同的衣裳啊。来，能够理解，能get到吗？其他的任何时间，中国人活着穿什么，死了就穿什么。只是特别严守礼义的会把身上的那一件改成左衽啊，这是按照礼记的规范啊——就是活着用右衽，死了用左衽。只有一部分非常讲究这个礼记规范的人，会在下葬的时候穿一件专门的改成了左衽的衣裳。其他的人活着穿什么，死了穿什么，因为他就是去见祖先去了。但是到了清的时候，局面变了。为什么？因为剃发易服。活着的人穿的东西，是他的祖先耻于穿在身上的东西。他活着的时候，他的头发、他的衣裳，他觉得他见祖宗是个耻辱，他不敢这样现眼，他怕丢人。能够理解吗？这就相当于是你出门说“我去打工了，我要飞黄腾达”，老爹老娘也在家里呢，等我。然后出门了之后变成了一个乞丐，破衣烂衫的。这个时候你被迫必须要回乡的时候，你会干什么？你会找人去借一件光鲜亮丽的衣裳，然后再去回乡，而不希望你的老爹老娘看得见你在城里是做乞丐的。所以在清代的时候啊，人们活着的时候剃发易服，头发剩一小绺儿，身上穿着别人安排的衣裳。所以他在过世的时候，觉得自己这样子无颜面见祖先，所以会把那个小辫儿打散，尽管没剩多少了，还是要把它盘成发髻的样子，然后身上呢，会去穿上汉服。因此在有清一代的时候，汉服是专门用于丧葬的，是因为他不允许你活人穿，不是因为汉服就是丧服，而是因为他不允许你活人穿着它，所以没有办法，他才会在死人身上使用这样的服饰。
