# 第125章  为啥阿三必须相信“脚皮转世”才能活下去

好的，我已按照您的要求，将原文整理为更易读的段落，并修正了其中的错误词汇，未删减任何内容。整理后的文本如下：

---

呵呵，那个阿三那边……阿三那边的那个阿Q精神，对，是的，是啊。阿三那边的那种精神状态呢，我们是分析过的啊，前阵子还做了一个视频，是的。

第一呢，咱们得能理解对方为什么这样啊，这个也确实很重要。就是为什么会这个样子？因为呢，人类的存在是有一些存在的基础的。人类的生存呢，和其他的生物不是很一样，或者说呢，智慧型的生物都有这样的缺陷：就是他开始思考之后，就会开始去想下一个问题——“我是谁？我从哪儿来？我往哪儿去？”这个问题其实就是在思考生命的价值、生命的意义。因为能思考，因为有智慧，所以在乎意义这件事情。你必须得有个意义，不然的话，我为了什么呀？我图什么呢？所有的人心里头都有这个声音，一切踏入了文明体的人，心里头都有这个声音。

于是就产生了下一个问题：既然一定会提问，但是大部分的人在大部分的时间里面，过的日子都很苦——这件事情是一定要承认的。就是大部分的人在大部分的时间里面都很苦难，仅仅为身而已。那既然只是喘气儿，既然只是想想方设法的也只能吃到一点点，图什么呢？是不是这种问题就一定会出现？那因为这种“我到底图什么呢”的疑问会导致一个人有自毁倾向，这是智慧型生物自身天然存在的短板，他一定会出现自毁型的倾向。

于是我们给自己打了一个补丁——有缺陷就有补丁。我们的补丁就是找借口、自圆其说、编故事，让自己能够达成逻辑自洽和平衡。所有的人啊，所有的人身上都有这种特征，包括你，包括我，包括整个地球上面一切拥有自主思维能力的人类，都是这样的。他一定会想方设法的给自己找到某一种真理，自圆其说，让自己感觉很不错。这个东西是人类活着的刚需。

所以任何一个地方，如果他是奴隶制，奴隶主一定会给奴隶讲故事，告诉他：你为什么是奴隶？你为什么要过苦日子？你为什么吃的东西和奴隶主不一样？为什么住的房子和奴隶主不一样？为什么生生死死都不受自己的控制？你必须有一个故事给他，不然的话活不下去。

于是你去看啊，世界范围内都有各种各样的传说。而在这些传说当中，绝大多数都有血统论的迹象。而这个血统论啊，是我们现在较为好理解的西式的血统论，而不是中国人自身的那种内部的血缘传承论。这种血统论的意思就是：谁生的孩子啊，海王的儿子就是海王，血统继承、贵族继承啊。这种所谓的纯血就是这个玩意儿啊。这种传说、这种故事的体系，在世界的绝大多数地方都很盛行，包括那个什么雷神那一套故事啊，那是北欧传说啊。北欧传说的故事当中也非常的强调神明的孩子拥有什么样的力量。包括说不知道什么时候创作的希腊的神话当中，它也存在说神明生下来的孩子是什么样的，半神拥有什么样的伟力。这些都是血统论的雏形。

而这种故事、这种传说会打造一个什么样的印象呢？就是今天之所以受苦，是因为我原本就活该待在这个位置啊。那么到阿三那个地方，阿三那里的所有的故事都是拿来解释他为什么过苦日子的——那就是你只是我们那个大神的脚皮儿化成的，所以呢，你天然的就低。但是呢，哎，如何解决这个问题？就是把你该吃的苦往死里头吃，吃越多越好，然后再来下一辈子的时候，你可以往上升级了啊。所以呢，他的这一套的故事驯化，就是为了满足这样人类的思维自洽的需求。

在神话故事体系里面完整了之后，那下一个问题就是：他为什么要让这些人的生活无法改善？如果生活都可以改善一点，大家不是更加能够用唯物的方式自洽，你也不用一天到晚的讲故事了吗？这个呢，是中国采取的逻辑啊。中国人在先秦时期也是贵族时代，在更加古老的时期，很有可能我们也经历过保有大量奴隶的时代，这种情况是不可排斥的，很有可能拥有过的一个阶段。但是呢，中国人这里，因为我们的发展方向的问题啊，我们产生的最后的决策的方式是：我们想办法让所有的东西都一起升级——一起啊，集体升级。我们呢，选择的方式就是逐渐逐渐的世俗化。

你要知道，一个国家走向世俗化是很困难的，因为世俗化最大的难处还不是那些神权群体的利益的需求。最重要的事是，一个国家如果它的生产力水平不行，它就进入到世俗化，大家呢会因为日子过得太苦而不停的去造反，不停的去抢掠。而这种不停造反、不停抢掠的方式就会让这个国家陷入混乱，秩序就直接崩塌了。所以，如果你不能够先把生产力抬上去，直接去改变生产关系，那是没有用的。这就是生产力与生产关系之间一定是相辅相成的一个关系，相互作用，相互的影响啊。

这是中国人的选择啊：生产越来越好，然后我们的秩序呢，就跟上了，改了；唉，然后生产又更好了，然后我们的秩序又跟上了，又改了——逐渐的升级换代。这是中国人，所以我们可以走向一个世俗化的时代。

那隔壁的阿三为什么呢？到现在都还神神鬼鬼的，甚至他们的老板都一天到晚的被称为老仙儿是吧？为什么会是这样的一个姿态？是因为他们呢，不是一家人啊，这是一个很根本性的问题。虽然今天很多人喜欢弱化这件事情，但是这就是他们的核心。他们的根本就是他那里的人不是一家人，而且这种“不是一家人”不是说“哎，我设一个规则以后，我们都是一家人了”就可以画上句点的。有一些地方可以这样操作啊，例如我们隔壁的脚盆。隔壁的脚盆，他们那里的人其实也不是一家人，但是呢，他们可以用法律规定的方式，如规定了“我们就是一家人”，句号行不行？行，在岛国那里可以。但是在阿三不行。为什么不行？因为岛国那个地方的人，尽管说他们来历上面差异有点大，但是呢，长时间漫长的共同生活，使得它们的外观差异啊比较有限，外观差异比较有限。所以当他强行的规定了大家就一起了的时候呢，这个事勉勉强强的就过去了。

但是阿三那里不行。阿三那里的漫长岁月当中的种姓制，是有着种姓隔离和婚姻隔离的，甚至可以说它其实就是一种生殖隔离。而生殖隔离带来的后果就是不同的种姓，它长得不一样，外观上面有非常典型的差异。当你的脸都不一样，隔老远就能看出来咱俩不是同类的时候，那就不是说我规定了咱们就是一家人——你这个规定不好使，因为他不唯物。而阿三那里很大的一个毛病就在于，他实实在在的就不是一家人，而且在那之前是真真切切有巨大仇怨的各色各样的人等。为什么？之前讲过，他那个地方的地形很特殊。地形很特殊啊，生活条件并不好，但是呢，又还不差，于是他就维持住了一种时时刻刻有新的殖民者进来，但是新进来的很快就会丧失充分的战斗力，然后又被新来的殖民者所征服。于是他像压地层一样的，一层一层的蛋糕就摞起来了：最开始的本身的原住民就压在了底层，变成了不可接触人群，然后一层一层一层地摞上去。

而这样的不同的人群，他们祖源不同，信念不同，甚至很多的时候语言也不太一致。当它们的差异性已经大到肉眼可以直接识别的时候，那我就请问你，他的社会治理上面，能够直接一竿子规定了然后就画上句号了吗？不能。事实上在他们现行的规定当中是没有种姓制的，所以你今天去问任何一个阿三，他会告诉你说“我们法律规定了没有种姓制”，但事实上它依然存在，因为这个东西已经成为一个历史的鸿沟，明晃晃的摆在所有人的面前了。

于是放到这个问题上的时候，你说阿三为什么那么的有阿Q精神？因为不这样活不下去。为什么这样不这样活不下去？因为你想要不阿Q又能活下去，就需要全民共同富裕。而要实现全民共同富裕，哪怕是从苦难当中一起吃苦，再一起富裕，慢慢的爬起来，这个过程需要大家成为一家人。而阿三由于他的历史的演进的方式的问题，造成他根本就没有办法直接说“是一家人”就句号一家人了——他做不到，谁都做不到这件事情。他是一个拼盘被硬拼起来的。

所以就回到了最初的问题：他为什么要阿Q？因为他只能阿Q。他只能想方设法的自信和自圆其说，不这样他就活不了。呵呵。

你说中国人古时候有没有贵族？当然有贵族。但是啊，但是在我们这个地方，贵族是什么？贵族是统治阶层，然而他不是种姓。为什么呢？因为“王侯将相宁有种乎”——这个“种”才是种姓。意思是不管你是什么血缘来历，只要在竞争当中获得优胜，你就可以向上行走，这个叫做上下运行的通道啊。因为我们这里叫做有贵族，然而这个贵族是可以推翻的。所以性质上面与绝对说“一日为主、终身为主、一日为奴、世代为奴”是完全背道而驰的路径。
