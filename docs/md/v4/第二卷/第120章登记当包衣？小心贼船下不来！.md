# 第120章  登记当包衣？小心贼船下不来！ #曝光陷阱 #网络安全  #社会真相 #举报有奖

想我们普通汉人如何看待清朝？到底要不要看作中华历史的一部分？当然是历史的一部分了呀，不然呢？咋的，倒霉的时候不是历史吗？跌到底的时候不是历史吗？丢人现眼的时候不是历史吗？人生所有的履历都是你的历史进程啊。对，它毫无疑问是我们历史进程当中的一部分——拉胯的那部分，丢人现眼的一部分，倒霉催的一部分，跌到地下18层的那一部分。怎么了？这话有什么可问的？这奇怪了，是不是啊？能问“是不是历史”的人，其实是反串的啊。我知道现在都是些什么人，他们现在让我头疼得很。我动不动就什么喝茶了、喝咖啡了，还有什么啊，马上要被抓了，诸如此类。反正我自个儿全不知情，怎么办呢？

另外，大家要适应一件事情：你们在我这里看什么直播、看什么视频，然后老觉得大部分时候和和气气的，有时候怎么说话那么刺儿呢？是不是个刺儿头啊，是不是个杠精啊？很容易产生这样的心态。大家要适应一下，我日常情况下是如此的。为什么如此？因为实际上互联网上爱我的人是有的，也不少，但是恨我的人也是乌泱乌泱的。而且呢，爱我的人都是以个体的身份在爱我，恨我的人呢，是成组织、成建制的恨我。所以我这里经受的攻击，比你们所看到的要严峻一些。所以有的时候呢，我的攻击力会略高一点，通常情况下不会非常暴躁，因为不太需要。

那个，嗯，清粉儿啊，你们会看到一件事情：清粉的文化水平是不错的啊。为什么这样说呢？是因为他真的文化水平不错吗？不是。为什么？你会发现清粉儿他的文化水平会表现得比较好，主要的原因呢，是他们成了组织，成了系统。所以即便是里面的九漏鱼，他也知道什么问题要怎么回答，什么样的情况要怎么进攻。因为什么？因为他们上面有专门的大脑组织，帮他们把这些玩意儿都拼起来了。这东西不是他们自己做的。就是清粉这个群体，他们这些东西不需要自己去思考，不需要自己学，也不需要自己练习，因为他们是成组织、成建制，有团队，然后呢，有稿件。他们特别喜欢攻击我说我有稿件什么的，但是他们是真有。我不是说在有稿件这事不对，而是呢，他们那里是专门的有一伙人制造假历史，成为所谓的弹药库，然后呢，由各种各样的、分散式的、看起来像个人的小号向外抛洒。而他们这个大脑组织在干的事情是什么呢？就是把历史当中存在的段落掐出来，看上去对他有利的东西拼合在一起，形成一个似是而非的假历史，形成一个知识库。然后这个知识库呢，他们都整理成了一条一条的、各种各样的细则、条款，以供下面出击的这些个小拳手们、小蛆们来使用。所以他们因为是一个有规则、有内部秩序，而且呢，有利润分成的群体，你就会发现它里面出现的人常常会表现得比较引经据典，尽管他引的是错的，据的也是错的，但是呢，他引经据典，看上去信誓旦旦，仿若真相，就是因为它实际上确确实实是一个有组织的操作。而这种组织呢，不光是在理论上存在，实际上也存在，他们甚至在国外是有专门的机构的，有物理驻点的，有线下机构的。他们还在互联网上有网站——那个网站叫啥玩意来着——你只要去他们那个网站上登记，你就可以当他们满洲国复国主义者的成员了。而他们自己还在外头设了这种网站，你去登记造册，拿了证，就变成他家的走狗和包衣奴婢了。下次呢，他分口粮的时候就带你一口。为了这个铁杆庄稼，确实就有人参与进去啊。我也不知道今天中国的经济这个水平，大家日子过得也不至于这么苦。我倒是也没有觉得说讨生活这件事情多耻辱，关键是我觉得在国内呢，只要肯干活、讨生活，保持一个基本的生活底线，这件事情是做得到的。有个别的人可能遇到了困难，做不到的时候，国家是有帮扶政策的。所以我不是很理解，为了那些其实十分微薄的仨瓜俩枣，去出卖自己的良知，出卖自己的脊梁，甘为鹰犬，甘为奴婢，这些人到底是怎么想的？如果是活不下去也就算了，结果在一个普遍都能活下去的时代，在一个国力蒸蒸日上、大家努力地给自己装脊梁骨的时代，在年轻人已经开始傲视群雄的时代，有人还在想要去给满清的这些僵尸老爷们当奴婢，想方设法地去寻找一个门槛去磕大头。我理解不了这种操作方式。关键他也拿不着什么啊。拿了钱能不能不干事儿？你要知道人家那边呢，就属于上了贼船啊。你只要是做了这个登记，就是上了贼船。有人干过这个事，但是主要是你刚登记的时候，他给你那仨瓜俩枣的不值。当你做一点什么——之前有人整了个大活儿，我忘了是谁，反正呢，他就是打入了，迅速地打入了内部，然后逮到了某一些材料，然后转头就举报给国安了，拿了一笔奖励金，就坦然承认我在他那拿了什么、登记了什么，然后回来我举报他们，他们有这有那啊。就相当于自己潜伏进去了，这边拿了一笔奖励金是吧，就给自己，嗯，洗白回来了嘛。这种的话呢，就尚可啊，这种还尚可。但是一般的，当你没有真正合适的切入口，你没有一个真正有牌面的介绍人的时候，光是登个记给你那仨瓜俩枣儿的，你去举报，你都不知道报什么结果呢。你还在人家的名单上面了，我们这里监控去打扫战场的时候，扫的就是你。这就很冤枉，你知道吧。所以不要琢磨着为那一点儿事儿。

他们控制人手段更多的是把柄。对呀，你只要你拿他一分一毛，那玩意儿都叫做把柄。你只要有了这个把柄在，你想下贼船很难的，知道吧。你有这个把柄在人家的手里头的时候，想下贼船你只能自首。自首也是记档案的。很多人就没有这个胆色去自首，除非他抓到了对方的把柄。你有本事抓着的时候，你可以说这个事；没本事抓着的时候，你就只是被人家抓了个把柄。我的亲啊，你就只能去自首了啊。如果是真干了这个活儿的，建议尽快自首啊。

嗯啊，点点点赞，我凑个数了我就下播啊。到点了，嗯嗯好，谢谢大家的支持啊。我这是到点了，再见。
