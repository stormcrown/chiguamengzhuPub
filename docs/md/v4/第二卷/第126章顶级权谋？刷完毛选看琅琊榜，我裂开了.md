# 第126章  顶级权谋？刷完毛选看琅琊榜，我裂开了

什么玩意儿啊？怎么看《琅琊榜》？我不看啊，我不看。我看那玩意儿干啥呀？——要是说他长得帅，我去看两眼可以；但是看情节的话，我一般不怎么看这样的影片。我觉得目前而言，我们能够演得出来的权谋剧都不怎么样。实实在在讲，就是我们的权谋剧都不怎么样啊。为啥？因为编剧不懂权谋，编剧没有权谋，编剧他自己的阅读理解能力达不到权谋这个程度。写爽文的作者是写不了权谋的，因为权谋是什么？权谋在高等级的权谋斗争当中，绝对不是我晚上偷摸给你下了一杯什么茶，或者是头天晚上悄悄地讲了什么故事，第二天你就突然觉得我应该往哪里去走。权谋这个东西，它应该是从高处往低处看。你的视野足够高，胸怀足够大，看到的信息足够广，拥有正确的判断的时候，你的权谋才会生效。不然的话，那都是小伎俩。嗯嗯，诶，什么？不是一般人玩的。嗯，当然不是了。真权谋不以谋财害命为手段。权谋怎么会是以谋财害命为手段呢？那是宵小之争啊。谋财害命这种事纯属于宵小之辈，是阴沟里的耗子干的事儿，是偷偷摸摸不能见光的事情。而权谋这件事情，什么是权？可以掌控、可以判断的东西叫做权。谋是什么？谋略、识别力、判断力、决策力，这个东西叫做谋。你能够拥有庞大的掌控力，你的触角足够深远，同时拥有敏锐的判断、识别能力和良好的决策力，这个玩意儿就叫做权谋。我知道我如何提高自身的实力，能够与对方角逐，这个玩意儿叫做权谋；知道在我最弱的时候如何蛰伏下来积蓄能量，这个东西叫做权谋。不然的话，你说那些电视剧里面能够演出来的这些鸡零狗碎的冲突，那就跟我们商战里头去拔掉隔壁家的发财树是一回事啊。所以什么叫做权谋？这个东西在我们的近代史上其实是发生过很多的。什么叫做权谋？《论持久战》啊。你们看一看《毛选》，你得理解一下什么叫做视野，什么叫做判断。一个人他站在什么样的高度上，看到多大面积的场景，然后做出什么样的判断，这个东西才叫做权谋啊。哪怕说你的一生当中可能都是在打工，也一样值得一读。为什么？因为之前我曾经做过一个视频，就是在讲说「国之大事与月薪3000到底有何关系」。这件事情我们得理解，就是我们国家整体的发展态势，包括国际格局，甚至中国人这一类型的人群在世界范围内拥有什么样的形象，这些东西，它都对于我们最普通的人是有影响的啊。因为我们普通，因为我们拿工薪啊，因为月薪3000、月薪5000，所以这些局面对我们的影响至关重要。对于那些真正豪奢的人啊，真正的有产者而言，稍有动荡，他说不定还能够发发财；但是对于像我们这样普通的上班的人、打工的人而言，社会的安稳、秩序的有效，就是我们生计的来源。所以这个东西叫什么啊？这个才叫做「匹夫有责」。你必须必须能够意识到这个事儿。所以普通人也要站到更高的维度，去观察世间事情的大势走向啊。所以呢，它不是纯粹地在吹水啊。可是你要观察这些走向，光是听人家去讲故事行吗？不行。包括来听我讲行吗？不行。我这里会反复地告诉你，不要相信任何一个人跟你去讲这些东西。你应该知道怎么形成自己的判断，你应该知道一件事情发生了什么是主、什么是次，应该知道社会事件的运转规律，应该知道如何识别敌我，应该知道如何判断利益、根本利益，应该判断清楚，立场分明。这些东西拥有了，你才能够有一个明亮的眼睛，去看待你需要看到的东西。而这些东西在哪里学？在《毛选》里面学。所以它是我们普通人最重要的精神食粮。而当你把它翻开的时候，就会意识到：教员当年写这本书的时候，目的就是给最普通、最普通的人看的。他没有对人提出任何高精尖的学术的要求，他没有任何技术的门槛，只要你认字就能够读得懂。因为他掰开了、揉碎了，就要把一点最简单、最朴素的道理传递出去。而这些东西就是世间最本源的玩意儿啊。我小的时候经常这样说：这世间所谓的真理，都是些废话。这个就是这个道理啊。大道至简，返璞归真。真正有效的东西，它是非常非常明晰，甚至就在我们的生活的身边啊。你日常都能够见到的这些玩意儿，它反而是真实的、是本质的。这件事儿我们在前些年的时候，嗯，就是口罩那几年。口罩那几年，老外不是闹得飞起吗？有好多外国的记者在中国这里采访啊，就偷偷摸摸地采访啊：「你对这个事怎么看？你觉得你们这样这样如何如何？」你有没有发现，当时这些外来的记者进行偷摸的地下采访的时候，他采访到的很多的艺术家、知识分子等等的人，常常是忧心忡忡，老觉得我们这也不好、那也不好，完了，快快散了呀，这也错了、那也错了。但是这些记者跑到菜市场，跑到各种各样的街头、店铺、大排档里面去采访那些一看就知道勉强念完了九年义务教育的这些小老板的时候，画风突变。怎么回事？这些最朴素的劳动人民，这些生活在一线的菜市场里面的、每天恨不得要跟人家扯着嗓子骂街的人，他有着最朴素的逻辑认知，没有被任何外来的信息所干扰。他甚至你说信息茧房是啥？他什么茧都没有。他不看，他休息的时候就打打牌。所以他所知道的就是最朴实的啊：你拿到了好处，那就是拿到了好处；我吃了亏就是我吃了亏；停下来有好处，那就停下来；跑动起来更有好处，那就应该跑。这些最朴素的真理，在他们那里是天经地义的。这就是朴素价值观、朴素逻辑啊。但是呢，那些信息接收量比较大的、非常前卫的、非常赶时髦的知识型的群体，反而在那个时候好像眼前有一股迷障，硬生生就觉得这不行了呀，你看人家咋咋咋。为什么他会丧失判断力啊？为什么今天能够干脆利落地说他们丧失了判断力？因为事实证明，菜市场的大哥大姐们说的是对的，而这些小布尔乔亚们所吐槽的都是错的。这是历史证明的真相。那么我们回看那个时代的时候，你说为什么那些菜市场里面没有经历过高精尖教育的人——这些很可能只会打打小牌的人——他能够对时事的识别做得更好？因为他的价值分析、价值判断的方式是正确的。而这种方式在别的地方需要到大学以上，甚至要有专科的学习才会得到。而中国人很幸运，我们在初中的时候其实就学了相关的知识。我们在初中的时候就有相关的知识学下来，而只要你拥有小学五年级以上的阅读理解能力，就可以读得懂《毛选》。而对于世界的看待的方式、视角分析的方法就写在里面。所以今天把它称之为「屠龙之术」，因为它就是哲学的。哲学是什么？哲学是学问之上的学问，它是规律的总结，它是方法的归真。而中国人的哲学，用最朴素、最朴素的、唯恐你听不懂的语言陈述出来。所以这个书的宝贵之处在这里啊。
