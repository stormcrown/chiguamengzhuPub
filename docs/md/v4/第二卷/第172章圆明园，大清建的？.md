# 第172章  圆明园，大清建的？ #历史冷知识 #圆明园 #清朝那些事儿 #文化揭秘

好，晚上好啊。  
啥？  
昨天是被抓进去了吗？  
嗯，好，我——我凡是旷工的日子都被抓进去了。  
嗯，啥？  
圆明园是不是清建的？  
他的皇家园林为什么会取个大明的名字啊？  
这样子啊……圆明园这个事儿，其实里面有一部分的内容是存疑的。清肯定是建了一部分，这个事儿是稳的啊——清一定建设了其中的一部分。但是不是全都是清建设的？里面有没有明的成分？占比多少？因为目前呢，它的图纸啊，是吧，也大面积的遗散，然后呢，整个的建筑群落也基本上都被毁了，所以不能够说我们有证据说明他就是如何如何。所以按目前的主流的观点来说，依然你得说这玩意儿是清建的啊。但是有没有一种可能性：实际上它的主体的内容，实际上是明时期所留下来的园林，然后清进行了改造，或者说清在上面扩建了一些什么？这件事情的可能性本身是完全存在的哈。  

而且那个圆明园里面的话，有一部分的建筑做了一些复原图。然后会注意一件事情：就是当时的这个建筑的复原当中，有一部分的建筑的形式、里面的布局的形式，跟后来的西式的宫廷颇有一点相似之处啊，是颇有一点相似之处的。那按照之前的一些学者的分析，会认为说是这个时代呢，属于是西方的学者向中国传播了西式的建筑学，然后圆明园呢，就是西式建筑的一个集大成者——这是其中一部分学者的看法。但是按照今天啊，按照今天我们已经开始质疑西方的相关历史记载的时间点，是否真、是否假——按照这些方面的梳理来说的话，那么既然你能够怀疑是西方的学者向中国传播了他的建筑学并且构造了一个园林，那么同时能否逆向的怀疑一下：它本身呢，是明王朝时期所构建的一个新式的建筑范式的园林，那做试点的地方，然后被别人学走了呢？这种可能性是否存在呢？啊，哪怕是猜一猜，琢磨琢磨啊，也不是不能行啊。谢谢沈某某。  

所以呢，这里头依然牵扯到这几天在互联网上面不停的扯头花、打得头破血流的西史辨伪的问题。因为传播这件事情，你只能说很多时候我们发现这边的某个东西和那边的某个东西挺像啊，啊，然后呢，就会说他中间存在传播的关系。然后这个传播是怎么传播的？传播路径清晰吗？如果传播路径、时间、记载都很清晰，我们可以说它是从哪里向哪里传播的；如果中间的记载路线都不清晰的情况下，那它是从何而向何传播？这件事情里面就存在着推理。而目前的话，以中国官科的习惯性操作方法啊，通常情况下，如果一个东西我们这里有，西方也有，那么我们就是向西方学习的——这个是他们的一个研究的惯性。目前一直如此的时候，我觉得不能够说就保证这个事说了就是准的。  

嗯，西史前段时间确实打得头破血流，现在不是还正在——现在还正在打的呢。呵呵，现在还是在打的呢。打呗，我觉得打才好啊。这叫什么呢？叫做真金不怕火炼。就是这个标准：真金不怕火炼。打嘴仗怕什么的呢？只管争去啊，只管去争。争完了之后最后出什么结果就是什么结果，只要你愿意愿赌服输就可以啊。至于这里头谁不肯愿赌服输，这事儿不好说。嗯嗯。  

啥？搞不懂的是，清——忌讳名字为什么叫圆明园啊？哼哼哼。清没有像你想象的那么忌讳名字啊。清忌讳名字记在什么地方？我们公告一下啊，这个事是可以保真的——刚才的东西里面一定是包含了推理啊，但是现在这段呢，是保真的什么呢？就是清忌讳“明”这个字，它是由场合和对象限制的。如果说你是一个平头老百姓，你说“我大明万岁”，那他绝对忌讳名字啊，他一定忌讳。那如果说你是一个散帮子的文人，你写书的时候在那里什么给你玩“日月当空照”，我跟你讲啊，这玩意儿活脱脱的就是和“清风不识字”可以并列一起做文字狱冤案的啊。这就是你的身份、你的定位，在什么场合下出现这个名字，确确实实——你只要在这样的地方出现了“明”啊，那分分钟的头颅都要系上去了。但是并不是所有的地方他都会忌讳“明”这个字，例如说什么“正大光明”这个“明”字——明白，因为这是把它作为一个好字拿来吹捧自己的，这个比较简单。  

但是同时呢，还包括一个项目：我之前直播的时候聊过的，有一种真的文物啊，真的文物叫做“大明康熙年制”，知道吧？记得啊，真的真的正规的文物——大明康熙年制。有人说这是工匠不知道，别闹了。我跟你讲，这是官窑出品，官窑出品啊，官窑出品，大明康熙年制。你以为是干啥？因为清王朝啊，清王朝他立身不正这件事情环球皆知——不是举国皆知，举国皆知，你好理解。实际上他得国不正这件事情环球皆知，所以1840年的时候，小英发的战斗檄文说要反清复明啊——啊，小英的战斗檄文都说清的皇帝得国不正，他要去光复正义，他要反清复明，他要以明王朝授权的海外地……啥啥啥那个身份啊。也就是说在明王朝的时候，小英还是拿了授权书的那种，他家很有可能有档案，人家心里门儿清：他当初所知道的中国是什么样的一个国，他们清楚得很。他们当年就派了所谓的传教士——实际上是探子过来，而且在明王朝内部都已经驻扎了百年之久，他能不了解这些事情吗？他们心里清楚得很。  

因为这个缘故啊，所以呢，清王朝在立国之初的时候立身不稳、立身不正，就造成的一个问题是，你作为一个国家啊，你是不能够完全不搞国际关系的——不能说我跟其他的地方完全隔绝了，咔嚓一下子闭关锁国——这个事你谁哪个王朝都做不到。虽然我们过去曾经在课本里面老说“我们衰落就是因为闭关锁国呀”“我们明清闭关锁国”，这个这个东西都是当年那个在一些地方，有些人把它定义为各政治正确啊，实际上这是一个事实上的错误——明清两朝全都没有闭关锁国。但是呢，啊，明王朝的开放程度匪夷所思。实际上在明王朝内部自己的档案当中，对于他自身的属国范围啊，也就是藩属、授权、朝贡的这种范围，界定的范围是非常之广的。在某一些西人的口径之下啊，就是洋人的口径之下，甚至可以达到3000国之多。当然，这个数字呢，很有可能是以城池或者部落为准的啊，如果是以这个数字来说呢，3000国也不算太离谱——全世界有3000个部落啊，很正常啊，挺合理的一个数。那么在一些我们内部的档案当中呢，大约是180个朝贡国这样的一种口径啊。无论哪一种口径，可见明王朝是有着广大的国际关系网的。  

那你想内部出现了这样崩塌式的改朝换代啊——这不是正常的改朝换代，而是崩溃式的改朝换代——最容易发生的局面是什么？就是说假设啊，假设你是一个藩王：嗯，你是一个外藩的藩王啊，离得有点远，但是不太远；家里面这会儿呢，正好挺阔气；然后你听说宗主国那个地方被人窃国了，然后你这里正好年富力强，哎呀，热血沸腾——这个时候你很容易做出一种选择啊，其实历史上也经常出现这种选项：那就是我要进京勤王，我要打倒反贼。然后至于说进京勤王了之后谁当王啊，那就是下一步的事了，反正这个理由现成的放在那好好的。其实历史上面大家都是干这件事，包括清入关的时候找的借口也是要勤王啊——大家都会找这个借口，不止你清会找这个借口。那你觉得其他的藩属国、其他的朝贡国会不会找这种借口去进行讨伐、去占便宜、去撕一口肉，或者是来分一杯羹？会不会这样做？所以为什么在清初的时候，会出现“大明”这样的年号？它用在哪呢？它就用在对外的外宣上面啊，它会用在外宣上面。他用外宣上面的“大明”年号，就可以回避一个问题：就可以使当年的远藩和朝贡国，至少说在明面儿上依然认为你是大明，你就没有借口说你这个地方得国不正、我要来反清复明——他就找不出这个借口。你看欧洲人是1840年才找的这个借口，而清是什么时候全方位的以国际的身份宣告大清的呢？乾隆年间啊，从乾隆年间他才在国际上面大规模地去实锤的认证清的这个身份。在那之前，这件事情都是含含糊糊的。甚至他们还保留了一个印啊，是对外签署一些文件的时候的一个印章，叫什么来着啊——啊，忘记那个印章的全称了啊，但是呢，我们在对外签署这些文件的时候，会有一个专门的印、用印啊。那么清一直到康熙年间的时候，他对外用印依然保留了一个印，就是“大明某某某制”这个印。
