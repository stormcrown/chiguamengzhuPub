# 第121章  贩卖自家牧民的王公贵族，在让被卖的人引以为傲

很有趣的啊。在元朝这个时代，真正的奴隶阶层不是四等人，不是男人。男人是一个受盘剥过甚的平民阶层，但是如果你生产力足够好的时候，扛得住的时候，日子过得还可以。但是啊，在男人的下面，其实还有另外的一个群体，就是真正标准化的奴隶阶层。而这个奴隶是谁呢？蒙古草原上的牧民驱口。而蒙古的牧民、普通牧民这些驱口是什么？连公民都不是，他们从开始到结尾始终都是奴隶。那个时候的蒙古王公好大一票买卖，就是拿来贩卖出口的。他不光在国内贩卖，他还搞国际贩卖呢，过什么好日子呀。所以你说今天那个莫名其妙站起来说：“你记得你当年是四等人吗？”对，我可以记得我当年是四等人。你记得你当年是啥人吗？一点数都没有，你知道吧。

啊对，为什么要说元是一个普世型的国家？为什么要说元是一个普世型的国家？因为他根本就不关照自己的族人。就像我们互联网上活跃的几个啊，只能叫代号的人啊，例如某个导演啊，以五为代号的导演，他希望让别人想起来：“哎，我们当年曾经当四等人，我是如何如何。”很遗憾的是，他没有明文规定四等人啊。老有人在这事上嚼舌头。这个蒙古帝国治下第一等的蒙古人，是不包括驱口的啊。第一等的蒙古人不包括驱口，你心里得有数啊。这个蒙古人指的是蒙古王公及其亲卫，是王公和他的禁卫官僚这一派，他自己亲近的管理层人士，这才叫做蒙古人，一等蒙古人。而蒙古的牧民、普通牧民这些驱口是什么？连公民都不是，他们从开始到结尾始终都是奴隶，被贩运。而且呢，贩运的主要贩奴主就是蒙古王公，主要贩奴主就是蒙古王公啊。

所以为什么今天中国的土地上面，一直都有很多蒙古人的后代留下来？是真的有一部分的蒙古人的后代留下来了。什么人？就是当年大明驱逐胡虏之后，留下来的一部分。什么人？驱口。当年造反推翻蒙元统治的革命军当中，有一伙特别愿意下死力的什么人？驱口都是蒙古人。为什么他们被盘剥得比四等男人还要惨？是正经还要惨。为什么会这个样子？因为蒙古王公的统治逻辑非常的粗放，也非常的简单。没有明文的四等人制，但是在他的实际操作当中，不同的人所有的权利和义务都是不一样的，这玩意儿它就叫做四等人制啊，只是他没有明文写这个名字而已。他之所以歧视第四等的男人，是因为这些人当年抵抗最久，所以彼此之间有仇怨。但是同时呢，他们的生产力水平也最高，所以纳税的水平高，可以让他获利。因此，四等人在这个时候是一个受到打击、受到深刻盘剥的平民阶层。可是驱口不一样，驱口是财产啊，驱口是财产啊。所以从那个时代开始，逐渐流下来的这些蒙古后裔，什么人都是驱口的后代，能怎么办呢？那个时候的蒙古王公好大一票买卖，就是拿来贩卖出口的。他不光在国内贩卖，他还搞国际贩卖呢，过什么好日子呀。所以你说今天那个莫名其妙站起来说：“你记得你当年是四等人吗？”对，我可以记得我当年是四等人。你记得你当年是啥人吗？一点数都没有，你知道吧。

所以在这个范围之内，咱们就理解那个蒙古王公他自身的统治的这种基本的逻辑，就是跑马圈地。我圈下来的这个范围，我想怎么勒索，怎么勒索；我想怎么敲就怎么敲。我想方设法的用最节约的、最节省的方式获得最大的利益，利益最大化，这个就是他进行管理的基本逻辑。而在这套逻辑当中，剃发易服并不是一个非常好的策略。因为你推行了这个，一旦强制推行，就会引发反抗。引发反抗，你就要付出成本去镇压。你的管理成本提高了，但是收上来的东西不见得提高，甚至会破坏生产，对他来说这是不划算的买卖。而且呢，他推行了短暂的剃发易服之后，很快地发现了一件事，就是那些什么斥候啊，那些儒生啊、那些官僚啊，这些群体非常主动地就头皮痒了。你们知道的，明末也有这样的人啊，非常主动地就头皮痒了，迅速地就把衣服啊、头发就全都弄好了。弄好了之后，你知道他们干什么？他们就打进了蒙古王宫的内部。你说：“看，咱们是一样的吧。我还可以起一个蒙古名。”人家起名起得可溜了啊，一嘟噜一串就起好了。起完了名之后，这一个托托那一个哗哗的就出来了。来了之后干啥？跟你玩心眼儿，搞官僚竞争、内卷啊，就开始往死里跟那些蒙古王公内卷。你就说这些草原上面，前几天还在那里骑着马、架着刀在那哐哐砍头的人，在朝堂之上，你要跟这些世儒内卷脑筋，你卷得过吗？你卷得过谁呀？你玩得过哪一个呀？所以很快地啊，这个蒙古王公就意识到了这个事儿不对。这帮子人都来了之后，论玩儿心眼儿，论当朝臣，你还真玩不过他啊。你啥时候都竞争不过他，怎么办呢？禁止。所以他短暂地推行了这个剃发易服，而且很松散地短暂地推行了剃发易服之后，很快就把这个东西取消了，收拢了就不管了。而且还禁止了这些人冒充蒙古人。

所以在这一套逻辑下面，我们就得知道啊，那个元朝的统治的基层底层逻辑是靠暴力镇压的方式啊，是通过真正的军事威慑的方式。他用军事威慑，而且自信满满地认为自己的军事威慑完全可以管理这一切，所以他只需要分包去收税就行了。他是很粗放的管理法。但是到了清的时候就不一样了。清的统治逻辑和元不同。因为呢，元虽然你说按照中国人传统的方式来说，不可能说他得国正啊，因为他甚至没有资格去谈得国正与不正的问题。但是呢，在元身上，他至少还有一个特征，就是他是凭实力拿下的东西啊，这也算是他得正的一个方向。就是我是凭实力拿来的，所以呢，你就算不喜欢我，你也要尊重我的力量。元身上是有这样的自信的。但是清身上没有。清不是凭实力拿到的，他是通过各种各样的技巧的运用，它是利用了大量的汉奸的联动，利用了商人的无组织、无纪律，利用了各种去策反啊、挑拨离间，用这样的方式去拉拢积累自己的实力。他积累的实力都是从明王朝这里偷窃过去的，他自己心里头也很清楚。而即使是这样子，当面锣对面鼓的对撞也是撞不过的。他是趁着明王朝江河日下、山河破碎的时候，用自己所积累的、从明王朝那里借过来的力量，开始慢慢地侵蚀下来的这个进程。为什么要说慢慢地？因为抵抗的力量极强，只是因为分散，所以没有达成最终的成果。所以清号称一统江山的过程，他花了好几十年的时间。为什么？因为到哪里哪里就抵抗，没有人同意啊，没有人认可，他一家都没有。所以到哪里就抵抗，他就只能突突突突突突突，然后用不断的这种策反汉奸，然后又想逼迫等等的方式，运用了各种各样的技巧，趁着南明朝廷统治无力、内部撕裂、内部内讧的机会，然后一统河山。他是用这样的方式非常艰难地占据了这片土地。

所以他心里头知道一件事情：第一，所有的百姓都恨他，世仇，往死里逼迫的一种世仇，很清楚这一点。第二就是他自己的核心实力不足。什么是他的核心实力？就是他自认为的他自己的人。那些降将、那些降军、那些后来招揽的那些汉奸部落、汉奸群体，他信任吗？他不信任。你能叛一次就能叛第二次吗？他很清楚这一点。所以这些外围的当年出了大力的反贼群体，他并不信任。他自己真正的核心群体是从关外带来的八旗组织，而且这里头啊，他对汉军旗也不是很信任。所以到后来说那个养不起八旗了，要出旗的时候先出汉军旗，这就是一个信任和远近亲疏的关系。所以他自己很清楚，他的核心力量群体很少，力量严重不足，同时呢，权威约等于没有。那这个时候，他的管理方式和元的那种粗放的经营方式就会完全不一样。他知道自己没有能力管的时候，他就会想方设法地去搞一件事情，叫驯化啊，就一定要驯化啊。那怎么驯化呢？剃发易服就是一个驯化的方式。啥？嗯，刘必烈也不相信，汉军旗的人谁能相信啊？嗯，不相信。但是呢，要用啊。因为他们自信，所以呢，还肯用一用。而到了清的时候，他就很害怕了，所以他始终都是在搞满汉分治啊，所有的官僚系统、军事系统全都是用的满汉分支。为什么？因为他不相信，他也没有办法相信。他明知道对面都是一些叛贼的时候，上哪里信得起来去换你，你也不信呀。
