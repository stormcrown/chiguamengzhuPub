# 第133章  明朝就有立领？但汉服真不是靠领子人的

最近有人问：“嗯，最近出现的所谓立领汉服是怎么回事？”其实并不是最近才出现的呀，立领这个东西到晚明时期就已经开始出现了。立领的衣裳在晚明时期就已经出现了，立领不是什么了不起的东西。咱们再重复一遍吧：立领晚明就出现了，它不是什么了不起的东西。

领子是什么呢？领子是服装上面的功能性配件，也可以是装饰性配件，也就是说它根本就不是服装的主体结构部分。为什么你会执着于说，它到底应该是立领还是翻领之类的东西呢？你为什么执着于这个呢？一个是老有人提醒这件事情，另外一个就是因为它比较醒目，你容易注意到它。再加上各路试图混淆是非、颠倒黑白的人，会借由说领子的相似性，说这两套衣服是一样的。最典型的、最容易被利用的，那就是立领、立领呀、斜襟儿啊这样的结构，都容易被后来的满清服饰所利用，因为它们在表现上面会有一点点的相似性。因为后来的清也有立领，所以他就跟你说：“你看吧，这个立领我们是跟那个明学的，所以我们这就是汉服。”

这咱们得弄清楚一件事：就是领子是服装的功能性配件、装饰性配件，它是一个配件，它不是主体，一定要弄清楚它不是主体啊。什么是主体呢？主体是它的剪裁结构、襟口的受力结构，这个东西是它的主体属性。而明制的汉服和清的满服之间的差异，就来自于受力结构的不同。汉服自古以来的受力结构都是对称式受力结构，无论是对襟、圆领、交领、交襟啊，它全部在内部结构上面通通都是一样的，都是对称受力结构。你的右边的襟口是栓在左边的腰上面的，然后左边的襟口是栓在右边的腰上面的，这个东西呢，就叫做对称受力。这样的衣服用两到三个节点上面打一个结就可以固定住了，不需要很多的扣子，它的结构就是稳定的，因为它对称，因为它平衡啊。它符合我们这种温带的气候啊。温带季风条件的气候是什么样子？冷的时候冷，热的时候热啊，有时候一会儿冷，一会儿热，一会儿热，一会儿冷，这样的气候经常出现。而这样的环境同时呢，又没有超级极寒，也没有超级极热。那这样的温带气候，就需要我们的服装兼具通风与保暖的双重功效。那这种交襟的对称受力的半开敞式的结构，就非常有利于我们的气候条件和生活方式，所以我们才会长时间地使用交领交襟的对称结构啊。这个呢，是服装剪裁的根本原则。

至于你说：“哎，那上面立领天冷了，在交襟的上面搭一个立领”，这不是什么石破天惊的事情，很正常啊，很正常。相当于把你的围脖缝在了衣服上面。所以它本质上啊，它和清的那种完全不对称的结构是不一样的啊。清的那个结构是什么呢？一片襟口很小，另一片襟口很大，它们呢，没有办法对称。在不对称的情况下，他就需要用很多的扣子把它们拼起来。而这样的结构来自于哪里呢？来自于西伯利亚的苦寒之地。由于寒冷，它需要防风的同时又要节省布料，因为资源匮乏。两种东西都同时存在，使得呢，它这个人群不那么在意通风，它更加在意保暖。所以像这样子的一种密合式的结构，就会在这种极寒的地方诞生。一开始它出现的时候，里面的那个襟没有那么小，后来变得特别小，越来越小，为什么？节约啊，因为可以节约，可以节省它，可以省更多的料子。所以你就知道啊，它本质上面是截然相反的设计原理和截然不同的拼接方式、剪裁方式造成的两个服装体系的格格不入，而不是什么领口的问题。它并不是什么左右衽或者是立领的问题、翻领的问题，跟这个领子一点关系都没有啊。就好比说你去穿西服的时候，现在西装是改动量最大的一款衣裳。这个西服你去看那些时装款的西服当中，有翻领的西服，有立领的西服，有无领的西服，但是他们都是西服。领子是什么？领子是服装的功能性和装饰性的配件啊。永远不要把领子当做主体来看待。

嗯，有人问：“什么？粗布、麻衣才是普通人的衣着？”你知道粗布和麻它指的是面料吗？而我刚才讲的是什么？我刚才讲的是剪裁。面料跟剪裁是截然不同的两件事情，你是怎么拼在一起的啊？我们的结构是一致的。你穷你只能用化纤来做，他富他就用丝绸来做，但是衣服的款样一毛一样，这个就叫做同一体系下面的不同层次。所以普通人穿粗布、穿麻衣，别说麻衣了，就算他穿的是拼布，把各种的碎布头拼在一起做的百衲型的衣裳，有什么不行的呢？完全可以。就算他破衣烂衫的，它的结构是什么结构就是什么结构。所以你说这个衣裳属于谁啊？什么阶层啊？说它是给富贵人家用的？扯淡。同一套设计原理、同一套剪裁方式，下面富贵人家用丝绸、用织锦，而穷人家里面用破布头拼起来没有问题，他们都是同款的衣裳。现在西装里面的高定的西装，几万的、几十万一套的都有，但是最便宜的西装呢？69块钱包邮一身啊。所以你觉得有什么，他们两个就突然之间不是同一类衣裳？你看这个才69，那个都是69万了，那能一样吗？是吧。有69的是吗？越来越划算。哈哈哈哈，我真看着了，人家模特穿的还挺好看的呢，估计我拿到手的话不可能好看啊。我没有买，但是我看着了人家推的那个69一套的西装，整得挺好看的，很漂亮的女士西装什么的。

嗯，不要到我的直播间里面去说某一些特别痛恨我的主播，好吧。倒不是说我怕他什么，关键是这几个特别痛恨我的主播呢，他们都是有组织的啊，他们都是有组织的。那些特别痛恨我的人，他们是有团队有资本的啊。所以呢，这些人常年付费请各色人等在我的各种出没的地方监控，然后呢，只要出现说我提了他的名字，或者是我的公屏上面、我的评论区等等的地方提了他的名字，那就是我“区区”他了。所以我为什么跟你们说，别唠他们没意思，就是因为这个缘故啊。
