# 第636章 皇上，您还记得大明湖畔半秃瓢瘸着腿的夏雨荷吗？

尤其有一些人幻想自己是一个汉家女子，然后去跟满洲八旗谈恋爱。知不知道满汉不婚，是不能够结婚的？你一定要谈的话，那就是在他们后院连个小妾都不是，那就是被包进去的奴婢，谁过来都能删的那一种。诶，他就想，他就畅想着她要谈恋爱，还亏他想得出来。乾隆年间的汉家女子，还生活在济南府这样的地方，那他一定小时候是被剃了半秃瓢儿的。被强制剃了半秃瓢儿，你就想一个，哎呀，小美人儿美美哒，然后剃个半秃瓢，啥样造型，你自个儿琢磨。

而且从清代开始，女子的这个裹脚，就从原来的那种修饰脚型的裹脚，变成了断足式的裹脚。半秃瓢在脑袋顶上，脚底下是个残疾的，走路的时候一摇一摆。闹的第一件事情就是，哎呀，陛下，我好爱你啊——这是啥玩意儿啊？这。

新中国不搞那一套大杀四方的方式。我们要解决历史问题，采取的方式是进行彻底的批判和解放。包括我们新中国的时候是如何解放满族同胞的，就是对这些过去的罪恶的、黑暗的时代和犯罪的人群进行彻底的批判，对他们的罪行进行彻底的总结，然后我们才达成了最终的成果。这是什么？我们不是在消灭肉体，我们是在消灭这个特定的阶级，我们是在消灭满洲八旗这个阶级，而不是在物理上消灭某一些人。所以新中国是怎么能够达成今日之成就的，就在于我们坚守了文明的底线。要解决问题的时候是以直报怨。谁还不知道满清是一个垃圾呢？但是我们如何处置这个东西？最简单的方式就是像古人一样的大家屠戮，但是新中国选择了我们自己的道路，而这个道路依然是光明的。什么道路？彻底的批判，要求对方彻底的改正，彻底地站到人民的这一边来。这不是在消灭肉体，而是在消灭特定的阶级。实际上，在前30年，这件事情已经取得了阶段性的成功，但是今日又有人卷土重来。所以我们应该如何做？坚持新中国的理念，重复批判，重启批判，深入批判，继续批判，一直到他们改邪归正为止。彻底的消灭这些满洲复辟主义者的阶级。他的肉体在不在，那取决于他是否有违法犯罪的行为，但是这个群体、这个阶层、这种类型不可以存在，因为他们是错的。

嗯，满清还想复辟？那不是一般的响啊，不是一般的响。嗯，我们这个之前也专门聊过，2019年的时候，我们还在境内抓了他们的组织。已经有线下组织了啊。这些群体在哪里集结的？阿美那里有一部分，脚盆那里有一部分，思密达那里有一部分啊。所以他们是有组织的行动，在境外有势力的、有资金支持的。他们出书，他们搞联合，他们搞舆论渗透，他们写书、拍电影、拍电视剧，给别人洗脑、灌输，裹挟着很多人跟他们一起干啊。2019年的时候，他们把这个组织伸手伸到了大陆来，然后被我们连锅端了。就是因为端了这个组织，所以从2019年开始，我们设了一套新规则，就是这个广电审核影片的这种报批的时候，每年清宫戏不得超过一部。所以打那之后，实际上最后累计审出来的就一部片子过审，其他的通通都没过。大家记得以前清宫戏、辫子戏有多么的铺天盖地吧？不管是什么故事，他都要给你套一出清宫戏的皮。明明是在讲办公室政治，但是他一定要给你串到清宫戏的皮里面去。包括甄嬛传，他的原著小说是一个架空小说，跟满清朝廷、满清制度一点关系都没有，甚至里面的很多规则压根儿就不一致，但是他硬给你嫁接到满清上面去。为什么？因为这样子操作之后，你会对这个时代形成一种滤镜。他选俊男靓女去给你演戏的时候，你对这个时代会有一种美好的滤镜。我们今时今日长大的很多的人，小的时候都看过那些很帅的、很美的演员去演清宫戏哦。第一呢，就是这些东西一定会产生精神的影响。因为你喜欢某一个特别好看的演员，因为你喜欢某一个特别风趣的、有意思的故事，于是你对这个时代就产生了美好的幻想。甚至有一些人啊，甚至在看完电视剧之后，就畅想着跑到清朝去谈恋爱。谈个得儿啊，谈个得儿啊，那是个什么谈恋爱的好地方？

尤其有一些人幻想自己是一个汉家女子，然后去跟满洲八旗谈恋爱。知不知道满汉不婚是不能够结婚的？你一定要谈的话，那就是在他们后院连个小妾都不是，那就是被包进去的奴婢，谁过来都能删的那一种。诶，他就想，他就畅想着她要谈恋爱啊，亏他想得出来。呃，啥？夏雨荷说什么了啊？好夏雨荷啊，啊，你还记得大明湖畔的夏雨荷吗？乾隆年间的汉家女子，还生活在济南府这样的地方，那他一定小时候是被剃了半秃瓢的啊。来来，我们大家想象一下，夏雨荷到底长啥样？他脸啥样咱们就不说了啊。第一，他从小要被剃半秃瓢，对女孩子也是要剃头的。满洲制度下的剃头的样子，女的剃半秃瓢儿，骑着鬓角剃，一直到清末都剃。所以乾隆年间正是抓这玩意儿的高峰期，一定是剃的120%的要剃，何况他还在大城里面被强制剃头。你就想一个，哎呀，小美人儿美美哒，然后剃个半秃瓢，啥样造型，你自个儿琢磨。

而且从清代开始，女子的这个裹脚，就从原来的那种修饰脚型的裹脚，变成了断足式的裹脚。也就是你后世所见到的这种残疾式的半截脚。所以夏雨荷半秃瓢在脑袋顶上，脚底下是个残疾的，走路的时候一摇一摆。至于说啊，裹脚布拆下来的时候是啥样，那咱们就另论。我也不想伤害任何小脚老太太的心情，因为他们是受害者啊，他们是被迫的，他又不是因为爱那个，他又不是因为喜欢受罪。但是那个时期如果有一个这样的夏雨荷，他受到了这样残酷的欺压，穿着那样丑陋的衣裳，剃了半秃瓢儿，脚底下还是个残疾，然后他面见到造成他身体伤害和审美伤害的罪魁祸首的时候，想到的第一件事情就是：哎呀，陛下，我好爱你啊。那我觉得他受什么罪都活该。受害者遇到加害者的头，木讷地第一个反应是想谈恋爱，这是啥玩意儿啊？这啊，啥也别讲了。正啃鸡爪呢，我还呵呵呵呵呵呵。

嗯，啥？好像汉族的地主家、官员家的女子才要断足？不是的啊。那个其他时代的那个裹脚，这个事我谈过太多次了啊。裹脚呢，是从五代南唐开始的，但是那个时候的这个裹脚是一个正常的缠足，它是为了让脚变得修长、精巧，甚至提供一个辅助作用，方便你跳舞，就像芭蕾舞演员一样啊。那么什么时候开始出现了这种断足？清代的时候大面积的断足。这种覆盖的范围，并不是按照清楚的阶级来划分的。要知道以前的那种正常的修饰型的缠足，跟阶级有关系，家里面有闲钱的、有闲工夫的才会缠；干活儿的人，嗯，一般是不会把脚缠成那样的，他犯不上啊，也完全没有这个必要。但是呢，到清代的时候，局面完全不一样。虽然说确实上层地主家庭是普遍性的缠足，但是普通平民家庭里面也有很多是缠脚的，而且这个缠足的比例并不取决于你的阶层，而取决于你距离满城的远近啊。就是缠脚的这个比例的高低，是取决于你受满洲八旗的实控能力到底有多远的距离。离八旗的满城越近的地方，缠脚率就越高，越远的地方缠脚率就越低。这才是它的一个规律啊。这是根据墓葬来进行的统计，坟里头的东西才是真话。很多人就在那里说，哎呀，明朝就缠脚，清朝没救下来。明朝墓葬当中的女尸的脚骨都是正常的骨头，但是清代都是打折了的。嘴会说谎，尸体不会说谎。大家总归是看过法医系列的电视剧的吧，死人不会说谎。因为他不会说话，他肢体上面的任何一个反应都是真实的。清代墓葬当中的女尸普遍性的是断足，而明代墓葬当中的女尸只有一例，在晚明时期出现了断足，而这一例，她身上的穿着，身上盖的被子，丧葬的下葬的仪式格式啊，是满洲制度。所以你听他说：哎呀，我们满洲人不缠脚，你们汉人缠脚，我们没救下来。全中国土地上发现的最早的一例断脚式的缠足，采用的是满洲墓葬法。到底是谁的习俗？到底是谁干的？到底是谁推广的？到底是谁阻拦的？对，我们的历史当中有很多的迷雾，也不保证每一个段落都保真，但是至少说墓葬里的尸体不会说谎。骨头断了就是断了，哪怕你接起来也能看得到。明代的墓葬，宋代的墓葬，女子的脚都是完整的，而清代的时候咔咔都是折的。所以什么狡辩，在这个历史的事实面前都一点用都没有。
