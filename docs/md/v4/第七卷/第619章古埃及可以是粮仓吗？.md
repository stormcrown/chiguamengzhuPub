# 第619章 古埃及可以是粮仓吗？

上面最开始的时候，讲到他们源头的时候，就是从埃及那里走出来的。有一个地方叫做埃及，出埃及记。所以他们的文化源头当中，脱不开这片地方。这就是他为什么那么看重这个位置，为什么要在这个地方下力，去开发他伟大的古文明。那也就是说，古埃及文明的辉煌与伟大，和西方中心论、白人至上主义之间是高度绑定、高度关联的。这代表了什么？这代表了只要世界的话语权还在西方人的手上，古埃及就必然是一个辉煌伟大的文明；而只要话语权在他们的手上，所有的学者都必须要按照这个话语权来说话，才有可能在期刊上面发表文章，才有可能成为专家学者。

所以你说，专家学者为什么在这个事情上面不与质疑呢？不是完全的不质疑，而是过去的这20年间，凡是质疑的专家学者都是跨专业的，本专业的学者就没有质疑的。为什么？因为他就吃这口饭。他但凡质疑了你的那个报告，不要想着上期刊了，本科结业之后就可以说再见了。这事儿真是笑死我了。前几年的时候，我忘了是哪一年，时间也不久，在那个某博上面有一个美女考古学家，他他自己主打的商标，叫“古埃及美女考古学家”。为什么要强调美女？因为他在他的考古现场搞时装秀、搞cosplay。就是这个美女考古学家，日常在他的考古现场，就是那个埃及的那个小沙漠、小戈壁滩，这种地方，这种荒漠背景的场景之下，他就搞古埃及风cosplay。那画全妆，那大眼线就跟那个什么埃及艳后电影似的，贼漂亮啊。那个小发型修得那叫一个整齐，我觉得修那种发型，那个理发师都老贵了。总之呢，他就像走时装秀一样地做考古。

前几年的时候，他说他考出来了一个特别厉害的鼓啥呢？5000年前的一组巨大的象形文字，真大啊！那个高度好像有1米5还是1米2，巨大的象形文字，就是鸟啊什么的玩意儿那种。然后他在哪儿发现的呢？户外裸露的岩石啊。你想想，沙漠里边裸露在外的一些孤零零的岩石，上头刻了老大老大的纤细的所谓的象形文字，它就这么考古出来的。我也不知道他是为啥就这么考的啊。然后呢，就像在他的这个考古学家这里，他就去介绍他对古埃及文非常了解，于是他就专门去讲解了他这几个字。然后这一个啊是个什么读音，那一个是什么读音。这是早些年的时候他发出来的。他发出来的时候弄得我一头雾水：你画一幅画，然后代表了一个什么字母，是个什么读音，还是开口啊，还是闭口。哎呦，你给我讲了这么一大溜，你画都画出来了，你直接把它表述成一个含义不好吗？你为什么要把它表述成一个元音或者是辅音？所以这就是人类很难去想象他从未接触过的东西。人类非常难以理解他从未接触过的东西。所以西方的这些文字的研究学者，实实在在很难理解什么叫做象形文字，什么叫做表意文字。你要知道，一个谎要用千个谎来圆谎，这才是常态啊。当你去怀疑一件事情的时候，顺带着其他的很多的东西就会被推翻，这就是逻辑链条起到的一个作用啊。

于是呢，产生的一个后续的结果是什么？当他们编撰自己有一个伟大的祖先是古希腊文明的时候，大家发现古希腊那里种不了田，养不了人口，一定是没粮吃的。于是呢，他就给自己找了一个说是可以给他供给粮食的土地，说是从古埃及进行的海上的粮食贸易。就不论技术的问题了，以前讨论过技术。那今天就来单独讨论一下，既然他打算把古埃及当做古希腊的粮仓，那古埃及能不能做粮仓？两个彼此之间没有什么本质渊源的国家之间，去进行粮食这种战略资源的贸易，本身就是很离谱的。要知道春秋战国期间，我们的诸侯国之间那真的是亲戚关系、血亲关系，彼此之间的粮食贸易日常都是要使绊子的，各种各样的阴招都在里头埋着雷呢，随时随地准备着，靠粮食战把你们家给收拾得干干净净的。结果两个之间关系并不大的，跨海的两个国家之间搞大规模的粮食贸易，就为了养活你好，营造一个伟大的古文明——我是理解不了这种脑回路啊。

但是呢，即使把这一切都抛开，单论古埃及这一片地方，他有能力生产出那么多的粮食去支持它的国际的跨海的粮食贸易吗？要知道有余粮才能卖，有余粮吗？怎么产生余粮？于是他就不得不去给古埃及安排上肥沃的土地。那这个肥沃的土地哪来的呢？嗯，是吧？这就离谱了。你这肥沃的土地哪来呢？人人皆知这个地儿大面积的沙漠。于是他仅仅能够找得到的所谓的肥沃的土地，就是按照河流的泛滥地区，用所谓的冲击的原理，去给他安排一条沿岸的耕作地带。但这个时候这个概念啊，中国人特别容易接受，特别容易相信是真的，因为中国最好的耕地区就在黄泛区的附近，它就是黄河带下来的泥沙积累下来的这些广大的农田，然后就成为了我们最重要的农垦区。所以中国人默认地会直觉性地相信，在这种大河的周边，由于大河的季节性的泛滥，有可能出现肥沃的土地。但是这件事在中国成立，不代表在非洲也成立，因为它那条河吧，尼罗河那条河吧，它是个清水河，它里头冲的是一些颗粒比较大、重量也比较大的沙子，泥土的含量非常少。泥土的含量少，你怎么往旁边冲呢？多多少少也有一点泥，但是数量太少了，以至于它没有办法像我们的黄泛区一样冲出那么大片的农田、可耕地。所以一直到今天，在高科技的加持之下，埃及地区的可耕作地或者人员聚集、可生活的地区依然非常的狭窄，就留在河两岸很窄的一片地区。窄就意味着面积小，面积小就意味着它产不出巨大的粮食来。这个上面甚至不去考虑说泛滥这件事情有可能会造成粮食的减产，就算你的耕作水平极好，但是你总量少了呀。总量少了，你拿什么玩意儿出去出口呢？

所以粮食的产量有问题的时候，你就没有办法养活极多的人；而没有办法养活极多的人，又会产生下一个步骤的怀疑——就是你人的数量不够的时候，脱产的数量就会更少。没有办法直接脱产，脱离开觅食这个专业的时候，你有几口人可以让你去搞那些大工程、大建设，去做那些宏伟的金字塔？中国的古典时期为什么有那些大工程？因为我们拥有极广阔的良田，然后开发了足够好的农具，提高了生产效率的同时，又滋生了大量的人口，然后逐渐逐渐地扩张，造成了在大量的人口当中出现了越来越多的脱产人员。脱产人员才能够去研究技术的进步和文化的创作，之后才这样子一点儿一点儿地向上积累与滚动。我们的发展的路径脉络是非常清晰的。而在这些天然的资源不足的地区，你这个脉络不管怎么理他都有问题。
