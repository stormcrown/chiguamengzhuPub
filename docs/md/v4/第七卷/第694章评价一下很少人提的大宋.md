# 第694章 评价一下很少人提的大宋

### 对宋朝的看法

**关于“东北是鲜卑人的”这一说法**  
东北不是鲜卑人的。谁给你定的？东北就属于鲜卑人了？一个游荡的群体，怎么就成为某一个地方的主人了呢？一片土地属于谁，得看谁长期在上面生活、生产、建设，谁能够守护这里，谁能够维持这个地方的系统存在，你才能说这个地儿属于谁。一个游牧的渔猎部落，今天在，明天跑了，你说这个地儿是他的祖地，他是怎么拥有的？就因为他路过了吗？这就是马粪圈地法啊——凡是他家的马拉过粪的地方，就全都是他的领土。这个事不公道。

为什么？因为汉民族是一个农耕群体，我们只有自己种上田、盖上了房子的才算我们的。但是那些四下游走的流浪群体，直接是喘过气儿的，就是他的了。这个标准太不一致了：对我们太严苛，对对方太宽松。最后你就会发现，他人很少，技术很落后，但领土极大。他怎么拥有那些领土的？没有建设城池，没有种田，没有垦荒，他怎么拥有的？他在那喘过气儿。但是哪个地儿是属于汉人的呢？你必须深挖洞、广积粮，踏踏实实在这个地方建设、生产、维护1000年，然后他说这个地儿就是你的了，“我们不管啦，我们不占了，这确实是你的了”——这什么成本啊？但对于那些游荡的群体呢？只要看见了，只要呼吸过，在她的传说当中，“哎，我的阿妈跟我说过，当年我们怎样怎样”，然后那块地就自古以来属于他了。这是不是太容易了一点？

你按照这种标准的话，全世界都是我们的。为什么我们有《山海经》？我们不光是传说，我们还写下来了呢；不光写，我们还画下来了呢。他的这种马粪圈地法和《山海经》的这种有什么区别呀？区别就在于它是口口相传，还没有《山海经》的可信度高。你真要用这样的方式圈地的话，我们就用《山海经》来说话。势力范围、国土范围，你要怎么圈？大家标准必须一致：我们是扎扎实实地住在上面算国土，那你也得扎扎实实地住在上面才算国土。你就一个丁点大的小部落，然后给自己圈了一大片的无人区进去，你怎么占的那片无人区？你在那能生活吗？你在那盖了什么房子？种了什么田？没有啊。就因为望见了，就因为传说，就因为听过，然后就变成了他的地儿了？不公道啊，不公道啊。

东北周朝就是我们的了？嗯，应该是指那个啊，那个时候呢，是燕国占了辽东——燕国在扩展的时候占据了辽东。我们真正深入管理东北的话，还是要到汉代的时候，汉代设置了郡县才算实行了深度管理。那个燕国的时候呢，占据的范围比较小一点啊。

**对宋朝的看法**  
宋是一个很了不起的时代啊！宋是一个很了不起的时代！想要断章取义的，可以直接啊，在这个地方给你留了气口了啊，直接一截选：“你看这是个宋吹，他喜欢弱宋，她喜欢垃圾宋啊，赶紧的出去，可以去整活了。”——但宋是一个了不起的时代。

他身上唯一的短板，其实也无非就是他的统治者非常软弱无能。但是仅限于高层的统治者的软弱无能，他的国民、他的军队、他的将领、他的名臣都很优秀。科学技术非常发达，市场经济非常繁荣，海外贸易非常昌盛，市民经济非常兴盛。所以宋是一个仅仅在军事布局上面有短板的国家。当然，你说这个短板重不重要？确实，这个短板太重要了。所以大家呢，哀其不幸，怒其不争。

可是这里头有一个很重要的前提：宋所面临到的对手，和以前的任何时代都不一样。以前为什么在汉的时候，我们可以高呼着“一汉当五胡”，早期的时候甚至“一汉当五胡”？因为我们有技术代差，因为我们的科学技术远远高过于对方的部落级水平，我们是有技术上的降维打击的。但是到了宋的时候，由于唐的光速扩张，而且实现了技术的输出，以至于在唐走到末期分崩离析的时候，崛起的这些异族政权，它在技术上面跟我们已经没有明确的代差了——至少在军事技术、军事装备上面已经没有清晰的代差，甚至在社会管理的方式上面，也跟我们没有了清晰的代差。所以宋所面临的对手和其他任何时期都不一样，他升级打怪的难度比之前的任何时代都大。

然后偏偏他又很富有，富有带来的是统治阶层的软弱。这几样东西绑定在了一起，你就会发现他在军事的战略目标上面稀烂，但是他实际上在具体的军事战术上面并不差，因为他还是有很好的装备技术，也有很好的组织能力。但是第一，对方已经被培养起来了，比过去的时代要强；第二呢，这边的统治者确实太软太肉了。但他为什么那么软？为什么那么肉？因为他富，因为他太富有了。富有到了什么程度？富有到他宁可去纳税供，宁可去花钱换和平，因为这个方式对他来说很便宜。他前脚去交了岁贡金买了和平，后脚就通过走私贸易，从对手手上把这玩意儿赚回来了，甚至十倍百倍地赚回来。他把这个战争和平的这件事情做成了一项生意，而且赚得盆满钵满，得了好处，所以他就会变得愈发的软弱。这就是什么？这就是福祸相依，这就是福祸相依啊！就是因为经济上的极大繁盛，使得他变得格外的软弱可欺。他根本就不愿意打起来，哪怕下面的士兵将领都想打，将士用命也不好使，因为统治者还是觉得花钱买和平是最划算的，而且翻脸我就能十倍的赚回来，那他干嘛要打呢？打仗一定会折损民生，打仗会影响走私贸易，打仗会让他的利益严重受损；而花钱买和平这件事情可以让他赚得盆满钵满。所以为什么他的高层统治者总是表现得特别的软弱、特别的无能？因为好处太多了。

嗯，啥？出卖了吊死自己的绳子？对呀对呀，就是买卖干得太好了。北边不论是宋——不，无论是辽还是金啊，还包括蒙古部落都一样——都有大量的走私贸易，直接就是赚得盆满钵满，爽死了，不要不要的。他怎么可能想打仗的呢？

嗯，啥？下班了？到点了是吧？啊，好，谢谢提醒，谢谢提醒。到点了，下班。
