# 第673章 如何解决当年大毛给我们埋下的雷？

在早期，大毛（指苏联）进行理论假设时，过度强调纯净的理想。因此，他专门强调除了理想之外，我们之间“这不同、那不同”，天天讲述差异，同时又强调共同的理想。结果到了晚期，由于经济和文化等多方面原因，大家开始对理想产生动摇。理想一旦动摇，再一看，发现“我们这也不一样，那也不一样”，于是就有了分家的意愿。所以大毛在晚期崩溃得如此之快，之前其实许多加盟共和国确实获得了不少好处，取得了长足进步。如果在其进步期，我们强调大家的共同点，后续可能大家还会尽力挽救。例如那一大片斯坦国，如果不是大毛当年的努力，他们无法进入工业化。是前苏联强制将他们拉进工业化时代，极大改善了他们的生活和科学技术。但为什么到最后分家时跑得极快，一下子就跳反，而且跳得很彻底，疯狂驱逐境内的俄罗斯族人？现在我们附近的前苏联治下的几个斯坦国，都在努力排除前苏联带来的影响，包括人员、文化、技术等方方面面。原因就在于当年设立这套规则时，他强调你的不同点；他假借团结之名，实际上根本不想统一，只想强调自己的差异，时刻要求“我和你不一样，你要格外照顾我”。一旦你强调“我们就是一样的”，他就慌了，说你挑拨离间。怎么挑拨离间了？我说我们是一模一样的，我们的基因是一模一样的，连长得不一样的人，我们的父系基因都是一模一样的。我在强调我们是亲生兄弟，他却说：“你这样的说法是不团结，你欺负人。”一个户口本上的人是亲兄弟，这件事很难理解吗？我讲的是求同，他们要求存异。他们天天强调不同，却声称那是团结，而求同反而叫不团结。真是离谱，属于小学语文没学好。

日韩是单一民族吗？实际上日韩都不是单一民族。他们是靠法律规定直接说自己是单一民族，但内部有完全不同的认同支系，血缘、文化、认同感都不同。日本岛上有阿伊努人，琉球人也与他们没有共同认同感。他们之所以这样做，是因为知道强调这个问题会出事，所以强行规定为单一民族，这是他们的特殊操作。而我们国内，新中国成立时，汉族占比在第一次统计中超过98%。后来为了表现多民族，把比例调低，进行了拆分。最初普查时汉族占比超过98%，这样的比例放到全世界任何地方都算单一民族。我们根据前苏联的理论强行划分为多民族。其他那些你以为单一民族的国家，实际也是多民族，而且主体民族占比还没我们高，只是他们觉得划分会出问题，所以强行规定不许划分，才形成了你脑海中的“单一民族”。

很多人觉得大毛是故意来祸害我们，但实际不是。这事上我们得为大毛证明，他真不是故意的。他自己就是这样玩自己的，把自己玩死了。他自己身先士卒，没觉得这事不好或错误，他在本土也这么做，而且更离谱。我们当时是受其规范，被强行划分。当年的领导人意识到风险，在他们的专家指导我们要分成几百个民族时，被拦了下来，最后汇总成56个，已是折中方案。如果真按大毛的指导，按自主申报方式，我们得有400个民族。所以第一不是他在祸害我们，他自己也是这样做的；第二我们国内不是没人意识到风险，实际上知道，所以进行了调整。

人类社会的所有管理方式都是试错行为。因为人类行为难以标准化，人类会思考，不同时间有不同情感和想法，一句话就可能影响态度。所以人类社会制度性变迁都是试错，试过不好，能扛住就赶紧改。这事不能说大毛心怀歹意——如果他只害我们而自己不这么做，那他是坏；但他自己也这么操作，说明他当时觉得这事对，他想做这种尝试。这是当年苏维埃时期提出理论时的一种社会治理假设，与苏联体制有关：加盟共和国形式，把各色国拼在一起，说“为了共同理想而集结”。基本逻辑是：我们是用某种理想拼起来的，要证明这一点，就得强调我们血缘不同、族群不同、认同感不同、信仰不同等一切不同，但因为有共同理想走到一起。在这一假说基础上，苏联制定了理论，最初是假想，包括共产主义在马克思时代也是假说。通过社会实践不断补强和调试，让其真正应用。苏联提出了“团结型假说”，因为自身是加盟共和国，所以强调各加盟国达成一致和团结。为了论证能靠共产主义理想实现团结，他刻意强调大家身份上的不同。因为如果身份相同，团结就不一定是共产主义带来的了。所以他在自己领地上划分出好几百个民族，其实很多群体还处于部落期，根本没有民族意识。谁发号施令能控制保护他们，他们就听谁的。这时提供标准教育和经济一体化帮扶，事情就过去了。但大毛要强调“根据某种理想”，于是专门识别、发掘对方特点，强调特别属性，然后告诉说“虽然不同，但因为有共同理想”。这种纯洁的主张最终造成大问题：当理想出现瑕疵，大家不再认同时，剩下的就是“我和你不同”。所以强调“这也不同那也不同，但因共同理想走到一起”的那一刻起，理想就绝不能有任何瑕疵。瑕疵一旦暴露，认同感不足，分家意愿就来了。

因此，在早期假设时，太强调纯净理想，专门强调不同，又讲共同理想。到晚期，理想动摇后，大家一看“我们不同”，就散了。如果进步期强调共同点，后续可能还能挽救。例如斯坦国，若不是前苏联强制工业化，他们无法进步。但最后分家跑得极快，跳反彻底，出于同样原因。

现在全世界都知道该怎么处理：前苏联已经给全世界打了样，搞多元强调不同，一定会出乱子。各国都有所警醒。我们这几年共同体意识的建设，与前几年强调“这特色那特色”完全不同了。我公屏上每天有人跳脚说“你不团结”“你挑拨离间”，但实际上谁在挑拨离间？就是这些人。他们为什么着急？因为国家风向变了，现在讲究求同，强调我们有共同来历、共同祖先血缘、DNA高度相似、文化一体。这些人开始肝颤，特别反感汉民族，因为一旦强调一致性，一致性以谁为主？人多为主。人最多、文化最繁荣、历史最绵长的群体不可能当点缀。所以这些人慌张，不停贬低汉民族。但求同时一定要着重强调主体特点，因为求同求的是最大公约数，主体身上核心底色就是最大公约数。只要强调最大公约数，就能实现真正一统观。那些假借团结之名、实际不想统一的人，只想强调自己的差异，时刻要求“我和你不一样，你要格外照顾我”。一旦你强调“我们就是一样的”，他们就慌了，说你挑拨离间。我讲的是求同，他们要求存异。他们天天强调不同，却说那是团结，求同反而不团结，真是离谱，小学语文没学好。
