# 第610章 从知道西医吃木乃伊开始，西医悄悄改名现代医学。

嗯，西方古代真的吃木乃伊吗？啊，算不上，我觉得对于中国人来说的话，算不上古代啊，但是他们确实吃。这么说吧，就是现在很多的人觉得，西方人从近代开始对我们形成了某一种技术上的碾压之后呢，他就觉得西方人必然是在所有地方上都是对的。因此呢，对别人的历史故事都非常地推崇，甚至有很多人啊，言必称希腊嘛。这个风气呢，从清末的时候就已经开始了。现在呢，也有很多看上去很专业的人啊，有这种味道。就是凡是说到点什么事的时候，就是什么啊，柏拉图说过啊，亚里士多德说过等等等等的啊，这些口号就出来了。

那么在这些内容当中啊，它就产生了一种什么呢？你完全可以说它是一种迷信。因为啥？因为迷信就是指你没有理性地去相信他人，没有判断地就直接进行了这种相信的选择。这种呢，就可以叫做迷信。那么对于西方文化的盲目崇拜，其实也是迷信的一种。在这种框架之下，就会有人觉得西方人做的东西呢，都有合理性，都是对的。当你发现他们以前做过很多很荒唐的事情的时候，就会有一种违和感。包括说吃木乃伊，就是这回事啊。

实际上，他这个吃木乃伊的行为是拿来治病的。为什么这样？因为他们以前没有真正的医学。所谓的医学，医生在早期的时候是拿来干什么？放个血啊。放血疗法一直到近近代的时候都还在持续不断地用。那这个放血疗法谁操作的呢？相当多的放血疗法的操作者是剃头匠啊，就是理发师。所以你看得到，他其实并没有真正很专业的医生这个群体，他也没有真正意义上面的这种医学作为一种学问。你需要经过不断的实践。就像很多人去攻击中医的时候，说中医不科学——那怎么不科学的呢？因为他没有科学的理论。虽然中医有理论，但是他说这个理论是不科学的，于是中医不科学。好，但是你不管说它科学不科学，可是它在原理上面，哪怕你再不理解它，你也知道它是符合大数据法则的。因为它是通过什么方式来进行尝试的呢？叫做神农尝百草。这是什么？这是积累数据，积累经验。一个东西从无到有的时候，你要怎么操作它？尝试，不断地尝试，这个错了就试下一个。这个就叫做神农尝百草。你可以瞧不起这种落后的方式，你觉得这个法子啊已经不对了，不科学了，行。但是在你什么都没有、一头雾水、压根儿什么都不懂的情况下，在从0到1的这个过程当中，除了用大数据的方式进行堆砌，你还有什么别的方法吗？而中国人的医学就是用这样的方法积累起来的。我们积累了数千年，甚至有可能上万年的时间，积累出来的大数据的结果。你说它不科学，OK，可以。但是你就不能够去说西方人的医学科学，为啥？因为他没有。西方人的古典时期是没有这些东西的，他压根儿这方面的积累都没有。他是靠猜的，靠猜啊。他们在大瘟疫期间，因为会造成身体的一些局部的脓肿，所以他的治疗的方式就是拿刀子把东西放出来。这是他仅剩的治疗方案。你觉得这样的程度可以算作医学吗？

而在这种框架之下，他们在其实非常晚近啊，已经是所谓的中世纪的晚期的时候，才真正地做到了这些技术型的尝试，开始尝百草。而这种尝百草呢，他们那里比较倾向于使用各种矿物质，其中就包括了木乃伊啊。各种各样的这种，他的从0到1的医学创举当中，包括了品尝木乃伊。而这种方式造成了什么结果呢？就是木乃伊断供了啊，真的断供了没有了。然后形成了一堆腊肉贩子去盗墓掘尸，然后自己做成木乃伊，再贩卖给那些贵族。所以早期的那些呢，是跑到别人的那个什么墓葬里面去找的木乃伊；到后期去吃到木乃伊的那些，都不知道他吃的是啥。嗯，光听就想吐。

嗯，其实这种思路本身啊，并不离奇。因为人类的社会当中其实一直都存在有这种，就是在原始时期、非常非常早期的人类群体当中，一直都存在着食用同类尸体的这种事情。就是所谓的非洲食人族，其实就是这样的。他的这个食人族很多故事里面啊，动不动就好像他把人活割了似的。其实食人族是干嘛？他们是有一种那个就是丧葬的这种典仪。因为在他的逻辑当中，我要如何与自己的亲人永远在一起呢？就是在他死后，我把他吃下去。而这种食用尸体的这种所谓的，你说它是传统也行吧，文化也可以吧。这种东西在人类文明的早期是普遍存在的，并不离奇啊。别说是欧洲，也别说是现在非洲，其实还有一些地方有这种习俗。就是中国文明的早期，我们也是有这种的。中国文明的早期也有。所以在那个商代的祭祀当中，他会使用人牲来进行祭祀。所有的祭祀品，它的本质就是请神明和祖先吃饭。而这些顶流的祭祀呢，就会使用人牲。所以在我们的文明早期阶段，也会存在有食用同类尸体的这种问题啊。所有人类文明的早期都有这种特征。这是在文化的草创期，对自然的理解比较蒙昧的阶段，他会采用的一种思考的方式。有的是为了和自己的同类永远在一起，有的呢，是认为这是至高的贡献与牺牲。不管他是怎么想的，总之，这种现象在人类的早期文明当中普遍地存在，没有什么例外的啊。

个人的推论认为，这个东西有可能诞生于人类对于食物的节约。这是一种可能性啊。由于食物的匮乏，来源的匮乏，获得食物的不容易，所以呢，有可能会有这种出于节约的目的而食用同类尸体的这种行为。然后呢，去找了一个理由来自圆其说。这种可能性是存在的。我们只能陈述一种现象，就是在所有人类文明的早期阶段，都有这种食用同类尸体的现象。只不过说有一些开化得较晚的群体，他把这种现象保留到了比较晚近的时代，让我们看见了，仅此而已啊。这是文明发展进程不平衡的一个结果。因为大家不可能是齐头并进的。我们这里到了什么阶段，他就必须到什么阶段？没有这种刚需啊，也没有这种必然发生。所有的文明的进阶都是要靠人的努力的。有一些人群，无论是先天不足或者是后天没有机会，他的文明进阶的过程就是比较缓慢，就是比较滞后。这是一种客观现象。而欧洲当年所面临的就是这种客观的现象。他们不排斥食用同类的尸体，甚至会把它想象成某一种药品，因此才出现了大规模贩卖木乃伊的这种行为。这是他们在医学的草创期开始尝百草的一种尝试。
