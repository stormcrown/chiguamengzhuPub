# 第672章 如何看人有无奴隶心态，让他评价南宋

这个世界上没有什么奴隶拥有自我，奴隶没有权利说“我要什么，我是谁”。奴隶只有一种身份：他是奴隶主的财产。所以，当一个人或一群人的心里头有奴婢心态的时候，他自然找不准自身的定位。他想不起来自己究竟是谁，非常容易和那些声势浩大的、耀武扬威的、权力极大的人群站到一起。因为奴隶的最佳选择，就是跟对一个强势的奴隶主。只要你的心态当中是丛林法则至上的，有奴婢心态的，你一定会去追随那个看起来声量最大的奴隶主，这就是你的最优选。

为什么今天的新中国，你会觉得奴婢心态显得不那么醒目呢？是因为今天的我们的国家强大了。于是，那些心里头都是奴婢心思、都是慕强心理的这些精神奴婢们，就开始把共和国当做慕强的那个对象了。所以你就会觉得这个问题好像不那么显眼，因为他被混淆起来了。所以，如何判断一个人的是非观是不是正确的？如何判断一个人他的远近亲疏能不能分清楚？光看今天的太平盛世不行，你得看他的历史观。一个人的历史观当中，对宋是什么态度非常的重要。南宋软弱，偏安一隅，但是他是正统。为什么？因为他和我是同类，这个就是爱有等差。

喂，盟主，呃，我，我现在我是想跟您交流一个问题，就是关于这个认识自己和应该是和谁共情的这样的一个问题。嗯，我觉得我们每个人要想清楚自己是谁，在这个过程中会受到很多干扰，有些时候会找错共情对象。在这样的情况下，就会对自己是谁这个问题产生比较大的干扰和混淆。比如说，嗯，这里举一个例子：前一阵子我们就会探讨一个问题，为什么从小到大感觉清宫戏特别多。呃，还真的一直思考过这个问题。以前有一个逻辑，就是觉得清朝离我们特别近，所以我们对清朝的这个情况掌握会比较全，于是我们拍出来的作品会比较真实。所以我们秉持着一种对文艺作品的敬畏心，对真相的一种敬畏心，所以我们拍清朝会比较有信心。所以呢，我们的作品都是清朝，明朝呢就离得有点远了，唐朝就更远了，我们拍容易拍不准，所以不会轻易地拍。一直就有这样一种善良的、朴素的心理去解释这个想法，于是这种想法就给自己带来了一种干扰，带来了一种负面的逻辑闭环，导致看到清朝剧的时候反而会更加相信这应该是有比较多的考证的，这就形成了一个特别负面的恶性循环。所以这里就暴露出一个问题：我们在和谁共情的这个过程中，远和近我们可能会跟近的共情，谁和谁熟和谁不熟，我们会和那个相对熟悉的去共情，但其实这种就是典型的找错了共情对象的一种表现。嗯，这是一个情况吧。

还有一种情况就是强和弱，我们和谁共情的问题。一直以来都说一种慕强的心理，呃，这种心理其实是非常普遍的，也是人很难以避免的一种本能的情形。为什么会慕强呢？我觉得有两个心理原因：一个是我们希望自己也是强的，所以会对自己有一种寄托，有一种渴望，于是看到强的时候就非常的羡慕，非常认同，就希望自己也这样，这是一种心理。还有一种心理呢，觉得强的他比较有力量，希望他能对我好，希望他不要伤害我，于是心里就会对他有一种比较积极的心态，这是一种取悦的心理。那在这种情况下，就是慕强，这是人的一种弱点，或者说是人的一种天然的心理结构，其实是比较难超越的。但是呢，人他又是一个非常精密的仪器，他的脑子里就算再慕强，他也会有一个非常朴素的、非常清醒的那么一个角落，提醒他：你还是有些时候得和弱的共鸣一下情，因为什么呢？万一自己是那个弱的怎么办？所以就是说你得懂这个弱的是一个什么样的状态，以至于自己将来某一天成为了弱的情况下，你能够有应对的方案。所以就是说在这种时候，强和弱到底该和谁共情，就会有一种特别混乱的局面。

嗯，我这里再举一个例子，就是关于崇祯皇帝的一个问题。我本科的时候上过一个课，那个明清史的课，那个老师呢，其实是一个挺厉害的老师，也非常有名气。他讲的很多东西我已经不记得了，但是他当时讲的一个梗，他讲到崇祯皇帝的时候，他说，嗯，我先说啊，就是他对崇祯皇帝的描述，当时让我听着很难受，这个难受和内耗让我持续了很多年，最近这个问题才在我心中得到了一个翻转。他当时说什么呢？他说崇祯皇帝吧，你要说史书上也好，坊间的流言也好，都说他不是一个沉迷酒色的，不是一个特别昏庸的皇帝。他甚至呢，非常勤政，非常有励精图治的这样的一个精神状态。但是呢，他这个人天分不高，智商不高，非常的蠢，非常的笨。我当时就比较震惊。然后他紧接着下一个观点就是说，嗯，一个人如果他只是蠢和笨的话，他悄悄地呆在那也就还好，不会有什么不好的影响。但是呢，如果一个人特别蠢、特别笨，他还特别勤奋的话，对这个社会的毁灭、毁伤效果是非常高的，何况他还是个皇帝。所以呢，我们这个国家当时从明朝最后变成那个样子，是因为崇祯皇帝他特别蠢，还特别的勤奋，两个事情同时发生的时候，这就是一个民族的灾难。然后当时课堂上大家听着的时候，好多人都跟听段子一样，哈哈哈就在那笑，包括我，我当时也听着觉得挺乐呵，因为我觉得他这个话术、他这个叙事非常的逗，而且当时也没有意识到这个东西有什么不妥。但是这个东西就让自己心里很难受，为什么难受呢？就是因为我不觉得自己是一个特别聪明的人，但是我的情况又要求我必须得很勤奋，那这种情况，那我该怎么办呢？我作为一个不那么聪明的人，我勤奋是一件不配的事情，是一个羞耻的事情，这个就弄得我自己就很内耗。但是我的常识告诉我，我只有勤奋这一条路可走，所以他把我搞得很凌乱。

但是再说另一个问题，他当时讲崇祯皇帝的时候，我们居然在跟着笑，为什么会产生这个情况呢？我觉得当时就是我们找错了共情对象，就是我们把共情对象放在了胜利了的清朝，放在了努尔哈赤、皇太极这些人身上。我觉得有这样一种潜意识的共情，我们当时站到了那一边。但其实了解那个历史，了解当时的情况，我们不应该单纯地粗暴地慕强，不应该单纯地跟那个赢了的鞠躬致敬，而应该想清楚当时的我们穿越回去的话是谁。我们是汉人，我们是明朝的人，我们是崇祯皇帝这一边的人啊。我的意思不是说我们跟崇祯皇帝就完全的利益共同体，不是这个意思，但是相较于满清而言，我们的共情对象绝对应该是明朝，而不应该是所谓最后取得了胜利的那一方。所以我说这个是什么意思呢？我感觉好多潜在的这种共情的倾向，有些时候不那么理智，有些时候特别容易被带跑偏，潜意识里的一些想法特别经不起事后的回望和推敲，就会发现自己那个时候怎么会是这样一种精神状态。包括当下我们社会上好多人一说起西方来，就觉得西方特别牛、特别强，好像就是不容置疑。嗯嗯，盟主和一些学者们在做西史辩伪这一块的时候，大家好像听起来就感觉甚至有些时候有些人会觉得很有偷感，说我们怎么能说人家西方是不对的，这是不是说明我们不够自信呢？我们自信的话是应该去相信别人。我觉得这个逻辑也特别的神奇：你如果真的自信的话，你是应该非常公允地去质疑一切，大胆假设、小心求证的，而不是说西方那边他就是一个正确到英明神武到不容挑战、不容质疑的一个地步。这本质上就是一种文化自信的缺失，是一种对西方的一种仰望的态势吧。而这种局面的背后，其实就是共情对象错了好些人，他共情西方，也是一种慕强的心理在作祟。所以刚才我也说了好多这个慕强是因为什么，以及慕强这种心理本身是非常常见的，但是我们应该去除掉这个慕强的本能，应该回归到一个理性的态度，去找准自己共情的对象，这个是非常稀缺和非常需要被提醒和唤醒的一种精神状态。

我今天讲这个，我其实就是想输出一个我的观点，但是我可能没有理得特别清楚，所以整个说得也比较绕。最后简单总结一下的话，就是我们自己是谁，嗯，需要找准相应的共情对象，才能够对自己的认识更加立体、更加深刻、更加准确，从而才能让我们以后走的路更加正确。就是这样啊。

这个小伙伴啊，在聊的就是其实这个是确确实实存在的一种现象，就是中国人其实在相当长的时间里面，我们的这个心理不是很健康。这件事呢，就首先坦然承认：绝大多数的中国人，在历史上面有相当长的一段时间，心理非常的不健康。这个事儿为什么这个样子？他说的是我们的共情对象的问题。所谓的共情对象，其实指的就是你屁股到底坐在哪里，你到底把屁股放在了哪个板凳上面。我说过，所有跟文史相关的一切的内容，首先要决定的第一件事情就是你的板凳：你是谁？你和谁站在一边？包括刚才我们今天去讲服装的内容，实际上讲的也是你穿着什么服装，意味着你站在哪个文化系统内部，意味着你是哪个团队的人，这个都是最核心的东西。

那么为什么我们发生了漫长的心理异常？为什么有相当长的时间，中国人的自我认知出了问题？这是因为自唐之后，自唐代之后，中国人有相当长的时间陷入到了奴隶的阶段，这是一个很严重的身份转换，这是一个很严重的重大变化。不要以为说我们以前有什么奴隶制社会，这件事情不确切，这种历史分段论本身不确切。我为什么说唐？因为唐之后出现了这种号称自己是中华正统，但实际上却是异族政权的这些政权，他假借唐王之后、就是小唐的这样的名头设立国家，但是他却是一个异族的政权，他直接颠覆了过去我们的国家传承体系。于是在这种情况之下，北方接受异族统治的这些汉人的群体，就自然而然地在这套管理体系当中成为了奴隶。如果是短期的，可能问题还不严重，但问题在于这个时间非常的漫长。北方地区的这些汉人们，自从晚唐之后，开始成为了各族铁蹄之下治下的奴隶，一直到明王朝卷土重来，你想一想，中间跨越了多长时间，跨越了多少的世代？它会直接影响一个人的心理。奴隶持续不断地当奴隶，你又要活下来，你又要在这种好几等人制度的规则之下当最下等的人，在战场上面去无条件的当炮灰，永远都是当别人的下位者。这样的角色、这样的身份，你要想活下来，必须对他产生认同感，这就是奴隶心态、奴婢心态。奴婢心态他其实是什么？我之前做视频的时候讲过，所谓的奴婢心态就是谁强谁有理，这个就是奴婢心态啊。他是完全依从于丛林法则的，你必须得承认，奴隶主奴役你、蹂躏你的那个是正确的，不然的话你逻辑不自洽，你活不下去。但是反抗了又反抗不成功，最后活下来的、形成了稳定奴隶心态的人，他就会建立这样一种“有奶便是娘，谁强谁是爹”的丛林法则心态，这个就是他刚才说的慕强。而在这种心态之下，你想找出来你是谁是非常困难的，因为奴隶是没有自我的。这个世界上没有什么奴隶拥有自我，奴隶没有权利说“我要什么，我是谁”，奴隶只有一种身份，他是奴隶主的财产。所以当一个人或者是一群人的心里头有奴婢心态的时候，他自然地找不准自身的定位，他想不起来自己究竟是谁的时候，他非常容易和那些声势浩大的、耀武扬威的、权力极大的人群站到一起，因为奴隶的最佳选择就是跟对一个强势的奴隶主。只要你的心态当中是丛林法则至上的、有奴婢心态的，你一定会去追随那个看起来声量最大的奴隶主，这就是你的最优选。所以奴婢心态当中，其实有的时候奴隶会嫌弃自己的奴隶主不够强大，奴婢会因为奴隶主的强悍而与有荣焉，这是典型的奴婢心态。鲁迅先生当年也曾经批判过晚清时期的中国人身上就是有这样的心态，这是怎么来的？这是清代的200多年，杀得人头滚滚之后驯化出来的。而奴隶心态这种东西，会让你不知道自己是谁，不知道应该和谁站在一起，也搞不清楚自我的认知，只会接受谁强谁有理，谁声音大、谁对谁铺天盖地，谁就是正确的，这个通通都属于奴婢心态。

而当你想要当一个真正的、客观的、找得准定位的人的时候，很遗憾你们必须要重新翻出孔夫子的老文章，就是我今天开篇的时候就讲了：儒学讲究爱有等差。什么是爱有等差？所有的权利、义务、责任甚至复仇都有等差观，以本人为圆心，向外逐层地画圈，越靠外距离我越远，越靠内距离我越近，距离我越近的需要我为他负的责任越重，这个就是中国人的正常伦理观。所以为什么会有人共情满清？因为他的伦理观错了，他把奴隶主看得比谁都重。而事实上一个人应该拥有的是什么？他首先爱自己，然后爱家人，然后爱同家族，然后爱同乡，然后爱同民族，然后爱同国，之后再爱天下，逐层推进。把这些东西理顺了，你就知道你对谁应该负更重的责任，你对谁应该更加的关爱。你要把这些远近亲疏的关系理清楚，而不是听着谁的声音大，然后一肝儿颤腿一软，对，他是对的——这个是奴婢心态。为什么今天的新中国，你会觉得奴婢心态显得不那么醒目呢？是因为今天的我们的国家强大了。于是那些心里头都是奴婢心思、都是慕强心理的这些精神奴婢们，就开始把共和国当做慕强的那个对象了。所以你就会觉得这个问题好像不那么显眼，因为他被混淆起来了。所以如何判断一个人的是非观是不是正确的？如何判断一个人的远近亲疏能不能分清楚？光看今天的太平盛世不行，你得看他的历史观。一个人的历史观当中，对宋是什么态度非常的重要。南宋软弱，偏安一隅，但是他是正统。为什么？因为他和我是同类，这个就是爱有等差。
