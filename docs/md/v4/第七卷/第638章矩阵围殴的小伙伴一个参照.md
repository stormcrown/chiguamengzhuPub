# 第638章 矩阵围殴的小伙伴一个参照

嗯，今天直播聊到玻尿酸的事。其实直播间不卖玻尿酸也没问题，之前有玻尿酸公司找过我，真的找过。但我从技术角度吹这个产品理解不了，所以没接。至于沙沙声，不是下雨，是静音烧水壶在烧水。老观众应该听过，我买的时候宣传是静音，但从来没安静过，挺无奈的。

今天聊什么话题？我开播前想了的，结果一看到直播间有人就忘了。秒忘——你们觉得要说什么都行。关于历史，你说去年啊？不就是被人断章取义了嘛。但我也承认，断章取义是我的错，毕竟我没学会别人那种：虽然胡说八道、造谣，但每三句话就说一句“我爱中国”。我没这个习惯，因为不做贼心虚，所以不像他们那样隔三差五喊口号。正常人谁整天喊口号？可有一拨人心虚，他们不管胡说八道什么反动内容，都每三句话打水印似的加一句“我爱中国”。我这儿没这项目，直播时没这习惯，就被别人掐了段视频，看着完整其实不完整。

那天直播讲的是清末历史，有人提到联合满蒙。我说清末时满蒙王公是典型的分离主义势力，然后就有人喊我痛恨满蒙同胞，直接篡改概念，把“清末”移花接木到共和国。就这么一档子事，全网开始批斗。各路“爱国主播”——那些歪门邪道的爱国主播纷纷下场，因为不下场就被打成我的同党。我理解他们，毕竟要吃饭，不下场就会被扣上“力挺女皇汉”“不团结”“恨国”“用1450”等帽子。别人有大号在，把声誉搭进去不划算，所以都跟着一起来喷我。那段时间，任何人检索我的ID，关联的联想就是“进监狱了”“被抓了”“喝茶了”“喝咖啡了”“分裂势力”“谁豢养的”等等，天天都是这些。

不过没事，正常现象。当年拼命举报我的人，已经被禁言的、被封号的，已经好多了。当时局面就是这样。所以还能说什么事呢？就像刚才直播间一个老伙计说“我们直播间不卖玻尿酸”，这也是那时候流出的梗。因为无数人造谣诽谤、人身攻击，而我做的是正常文化输出。我不能冲上去跟他们泼妇骂街掐架，那样就变得和他们一样了。但事要说清楚，只能慢慢来。这种推动过程艰难，对方也咬死不撒嘴。于是我跟粉丝说，不要冲出去掐架，那让互联网乌烟瘴气，没价值。我禁止粉丝谈论其他主播的ID，避免制造冲突。因为制造冲突就像被流氓拽到粪坑里打滚，跟蛆打滚自然会沾上屎。所以不能互殴，就习惯用代称来称呼那些栽赃陷害的主播。

其中有个主播很好玩，他执着于自己是颜值主播。跟别人连麦时被讽刺，说“打多少针玻尿酸都救不了你的脑子”，他直接就炸毛骂街。从那以后，就有人用“玻尿酸”来称呼他。这个称呼本身跟我没关系，互联网掐架互殴我从来没参与，只是单方面被围攻。解决方式就是继续慢慢做视频、做直播，持续输出内容。过程大概如此。攻击我是很正常的事。
