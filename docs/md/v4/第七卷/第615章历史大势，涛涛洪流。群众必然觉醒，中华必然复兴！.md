# 第615章 历史大势，涛涛洪流。群众必然觉醒，中华必然复兴！

觉得事情哪个地方、哪个角落里面有不对的？不对，哪里不对？不对的地方就是：你只要赞美满清，就意味着否定你自己；你只要赞美满清，你就没有办法拥有任何程度的自信。如果满清王朝是封建统治的巅峰，那么我们的巅峰就是一坨屎，我们的汉唐盛世就只能是不如屎的东西。都这玩意儿了，你还能有什么自信呢？你相信自己从前都是蛆，你怎么可能有自信的呢？而当我们面临到自己的国家已经这样繁荣昌盛的时候，然后你又说从来都是一坨屎，这个东西之间不兼容，会让你感觉到极度的痛苦。

所以这几年啊，不管怎样，也不管他的学识如何、站队如何想，没想过这个问题，哪怕就是凭着心里头单纯的那一点点难受，他也终于愿意听一听像我这样的人说话了。这是历史大事，滔滔洪流不可改变。

所以这几年这些啊，精神格格们的吹嘘就变得越来越敷衍，变得越来越艰难。他们的每一次吹嘘，看上去都像是急赤白脸的狡辩，就是这种原因。因为清它就是一个罪恶以及黑暗的时刻，它就是一个大粪坑，客观事实放在那个地方。你只要认为它是你伟大的老祖宗，你就会不断不断地内耗，不断不断地痛苦。这种行为、这种思考、这种烙印，它会和你现实的生活完全地割裂，你没有办法达成一个完整的自我内核。

嗯，好，欢迎大家来到直播间啊。嗯，那个啥，我今天是有个连麦对吧？嗯，谁来着？来来来，嗯——喂，啊啊，连上了啊。感谢您给连麦的机会。呃，之前是想聊一下关于这个清吹是怎样的一种心理结构这样一个问题。嗯，我做了一些小的调研，想跟您呃分享，并且请教一下您的观点。

呃，我先说一个——呃，先说一个——呃，起点的一个观点吧。就是说小时候呢，课本上都说近代的中国饱受西方列强欺凌，于是呢，我们本能地代入的就是清政府的角色，因为相对列强来说，总觉得他是自己人，就会有这样一种先天然的一种心态。于是呢，就恨不得穿越回去，拧着他的脖子告诉他：你不要腐败，不要无能，你硬气一点啊，把这个家门守住。但是呢，直到有一次看了一个电影，就是《火烧圆明园》里面，慈禧太后在跟大臣们说的时候说：“我们面对这个太平军，面对列强，我们宁与洋人，不予家奴。”就是说这个江山即便断送，也是要给到洋人手里，绝对不能便宜了汉人。当时一整个就震惊了，但是我整个也就通了。怎么就通了呢？我突然意识到，我本能地带入回去，我觉得我是代表清政府，是站在我们这一边，需要跟列强打擂台的。但是后来发现，从清政府当局的角度来看，他们真正的对手、他们真正的对立面，其实放在第一顺位的一直是汉人老百姓。为什么会这样？你就总觉得特别的奇怪。俗话说肥水不流外人田，他们怎么是这个考虑啊？然后呢，后来就发现他毕竟是一个少数人在统治多数人，所以他整个统治的过程中，不可避免的就是血腥残暴的啊，甚至是非常呃恶劣的一个过程。所以就造就了他始终是一个满汉大防的这样的一个局面。

嗯，所以我觉得在有了这些常识的基础上，就不会再去吹清了，就不会再去做一个清吹了。但是问题就是，我现在想说的就是，我觉得清吹分为两类人。一类就是包藏祸心的，就是您在直播间里批评的那一类人，今天我们不讨论这些，因为他们没什么可说的，也没什么可讨论的。我想说的是另一类人。就是这一类人，他清吹，或者说他吹这个清朝，他是充满了一种投鼠忌器的心理。我想说的是，这一部分吹清朝的人，其实他们是有朴素的爱国立场的，只是呢，因为掌握的历史知识不够，所以在这个过程中，本能地还是把清朝跟我们看作了一体，所以在这个过程中充满了投鼠忌器。

主要有三个方面吧。第一个方面就是大家默认文明肯定是向前的嘛。我如果把清朝说得特别烂，把他说的特别差，那意味着明朝——那在他前面——那肯定只能比他更差，所以就会投鼠忌器。为了怕伤害明朝，所以也得把清朝裱糊得强一点，粉饰得强一点，这样的话才对得起祖宗。这就是一种心理结构，就是总觉得文明是向前的。但殊不知文明真的有时候是会倒退的，从清朝做的这些事，包括对这个历史的呃篡改，对这个科学技术的这个抹煞和这个销毁，以及对审美的这种系统性的破坏，它文明居然是在倒退的。很多人对这种倒退是没有概念的，所以总觉得是向前的，于是就会对那个呃明朝就会有投鼠忌器的伤害的这样的一个顾虑。这是第一种心理结构。

第二种心理结构呢，就是一直有一个说法，说世界四大文明古国，中国是唯一延续的古文明。于是呢，这个时候又有一种投鼠忌器的心理，就是说一旦否定了清朝以后，我们的文明就断了，从明朝直接就跳到中华民国了，我们作为唯一的延续的文明，这个殊荣就不复存在了。所以出于这样一种顾虑，大家有些时候就觉得这个清这一段，尽管知道他是外族入侵，我就装傻充愣地把他忍了吧，好在我们中华民族是继续延续的。这也是一种心理。当然，这种心理他有很大的逻辑漏洞，但是很多老百姓在想的过程中，他想不得那么周全，于是大家就本能地就会有这样的一个谬误。

然后第三种心理结构就来源于一句话，就是“落后就要挨打”。因为我们近代被挨打的太多了，这句话一直在教育我们要奋发图强。但是“落后就要挨打”这句话说着说着就会有一种感觉，就是落后你就该挨打。我是实名反对这种观点的。就是说落后就算我弱，你作为一个文明，你是不应该打我的，你打我你也是强盗。但是呢，作为我自己，我确实该奋发图强，我落后了我被打了，我确实我也无话可说。所以说我们确实要警惕落后就要挨打的这样一种被动的局面，但是呢，并不代表我落后了，你打我你就有合法性。所以呢，这个时候我们的老百姓就会有——呃，我老说我们的老百姓，我不是想代表大家，我就说我们有些时候会有一种情况，就是有些时候好像我们的落后就成了列强打我们的合法性依据。于是呢，大家在说起清朝来的时候，就不想把他说的特别弱，其实就是想说：其实清朝时候我们也还是挺强的，你列强打我是因为你不讲武德，是因为你不好，是因为你在当强盗，而不是因为我弱了你就该打我。所以这个时候大家就还是不愿意把清朝说的那么弱，到一个列强来打我们就很合法这样一个地步。我说的有点绕啊，反正就是：文明是向前的、中国是唯一延续的文明、还有“落后就要挨打”呃不应该作为列强打我们的合法性——这三点心理结构造成了很多其实我们是爱国的、对我们的国家、我们的中华文明有着朴素的感情的人，他在这个时候投鼠忌器，不得不去替清朝吹一把，去啊挽尊一把，是有这样的心理结构的。

我觉得这些人，他们呃处在一种迷雾之中，需要知识的科普，也需要呃去——嗯，怎么说呢，就是需要去把这些东西明辨清楚，需要论一论。所以我觉得这是一个非常任重而道远的事情。所以就是说大家的感情本身是珍贵的，所以即便是吹清，但是他的出发点是好的。但是呢，这个是严格区别于那些出于包藏祸心、出于篡改历史、出于呃黑白颠倒的目的的那些清吹的人，这是完全不一样的。所以就是因为在盟主这里听到很多关于清朝的这些知识，呃，我也是希望就是说这些真正的知识能够抵达该抵达的人民群众这里。啊，谢谢盟主，我提的问题。我、我就是想希望您能围绕我刚才说的这些给一些批评啊。谢谢啊。啊，好好好。嗯，哦，我说完了。诶，好的好的好的。

嗯，这是一个算作心理和行为的分析啊。就是人呢，作为一个群体性生物的时候，我们在进行一些集体思维的情况下，其实呢，常常会陷入到某一种误区，这是一个很常见的现象，非常的常见。那么对于清，那实际上呢，这里头确实呢，也是有一些人是无辜的。但是无辜的这些人啊，刚才呢，他用了一个词就是“清吹”，嗯，我觉得这个词上面就略微有一点点偏差。因为这里面的那些无辜的人，基本上呢，不会去吹清，而是认可清啊。这两、这两个东西中间是有点差异的。呃，就是那些精神阿哥、精神格格们会去吹清，就是为这个满清创造一些它不拥有的优点，然后来赞美。而另外的这一群人，之所以会变成他这个势力当中的一份子，是因为——嗯，它有没有那么好？算了吧，反正呢，你也不能说人家那么差吧——这是一个随同认可的一种表现啊。这个呢，当然也是有几种原因。

第一个呢，就是对历史的认知比较的欠缺，这个事儿挺正常，也不赖任何的个体。甚至有一些人呢，读了很多的书，但是历史的认知有偏差，这个责任不在于任何的普通人，而在于满清时期本身就对历史进行了大规模的篡改和掩盖。同时呢，在80年代之后的标准的课本以及科普的读物当中，也刻意地把满清的一些黑暗与罪恶的行为进行了粉饰。它不是完全不说，而是呢把它轻轻地绕过去了，很容易让一些读者就把这个事给忽略掉了，它非常非常容易忽略掉。

然后另外的一个方面的问题，就是自我认同的问题。这个自我认同也是一个很麻烦的事。自我认同是什么啊？哲学最重要的三个问题：我是谁，我从哪里来，我往哪里去。自我认同就是“我是谁”的问题。今天包括在座的很多人，其实你们身边所遇到的大量的人，是在自我认同上面有问题的。你让他们拿身份证出来的时候，你会看到它上面有“汉”这个字，但是你问他啊，汉人是什么？甚至有一些人会突然之间——你是不是不团结？这个是常见的事情，这是常见的事情。为什么？因为汉人这个群体作为主体民族，体量过于庞大，以至于他在日常生活当中根本意识不到自己是汉人。于是当你把这个东西单独提出来的时候，你怎么会在乎这件事情呢？你是不是有问题？这实际上是一种自我身份认同欠缺的表现。他不知道他自己是谁。所以在看待满清的历史的时候，他不知道自己应该站在哪一边。于是呢，就会站在声音大的那一边。谁的声音大呢？精神格格、精神阿哥们声音最大。为什么？因为他们知道他们所说的事情是谎言，谎言必须要重复百遍千遍才能够变成真理，所以他们的声音特别的大。没有自我的认知，也没有清晰的理论依据的情况下，它是非常容易被舆论场上面各色的声音所裹挟的。如果这些声音还捆绑了一些被断章取义、看上去正确、实际上有问题的段落，有这些口号，有这些旗子插在那个地方的时候，那就更加难越雷池半步，就生生地把一个人拴在了这个地方啊。

因此呢，在我们普通人的群体当中，常常存在有这种——虽然啊，看了说对他那个干的挺糟糕的，但啥时候不糟糕呢，是吧？你过的过都过去了——这样含糊不清的人是很多的。在这种明确的问题中间选择含糊不清，就已经站在了罪恶的那一端，但是他意识不到，因为他想不起来自己是谁，也根本记不得自己应该站在哪里。这是一个非常常见的心态啊。

那什么时候能够好起来一点点呢？文化复兴会逐渐地让大家开始转向。这两年我们国家国力越来越强，大家有没有发现一件事情啊？当我们的国力越来越强，我们就需要树立心理上面的自信。每一个人都这样啊，全世界所有的人都这样。就是当你发现自己所处的这个地方其实经济蛮繁荣的时候，你一定要为自己找到一个自己的心理依托——自信的来源。所有人的自信的来源都来源于过去，就是“我从哪里来”，必然如此。你一定要去找到你的过去。所以这个时候文化自信变成了一种刚需。这不是什么中央推广的要求，不是这样的，而是这就是普通的百姓内心深处最需要的东西，国家只是代表我们讲出来了。

那你怎么拥有文化自信呢？只要你认为满清是一个不错的时代，是封建时代的巅峰，你就不可能拥有自信。我们最巅峰的时期是那么一个垃圾，那么一个粪坑，还自信啥呀？所以，所以为什么这几年你会发现，互联网上的很多人开始提出了问题，哪怕说他没有坚决地去思考，但是呢，没有坚决地去研究，但是呢，他依然提出了问题，他依然感觉心口发堵。他觉得事情哪个地方、哪个角落里面有不对的。不对，哪里不对？不对的地方就是：你只要赞美满清，就意味着否定你自己；你只要赞美满清，你就没有办法拥有任何程度的自信。如果满清王朝是封建统治的巅峰，那么我们的巅峰就是一坨屎，我们的汉唐盛世就只能是不如屎的东西。都这玩意儿了，你还能有什么自信呢？你相信自己从前都是蛆，你怎么可能有自信的呢？而当我们面临到自己的国家已经这样繁荣昌盛的时候，然后你又说从来都是一坨屎，这个东西之间不兼容，会让你感觉到极度的痛苦。

所以这几年啊，不管怎样，也不管他的学识如何、站队如何想，没想过这个问题，哪怕就是凭着心里头单纯的那一点点难受，他也终于愿意听一听像我这样的人说话了。这是历史大事，滔滔洪流不可改变。
