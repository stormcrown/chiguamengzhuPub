# 第639章 种子快快长大呀，中华文化必将傲立世界民族之林

一直被黑啊，从来没有红过，始终都是黑的。最近肯定被黑得比较严重，因为有一些境内外势力联动，发动了一大笔资金，就在专注地整我。我也是没想到啊，一语成谶嘛！呵呵呵呵呵呵。还是我预告过这波事儿，结果我变成了当事人，哈哈哈哈。这可怎么整？呵呵呵呵呵呵呵。本来是想围观吃个瓜的，结果吃瓜吃到自己头上了。

好，欢迎来到直播间啊！网络围攻啊，啊，会不会被封？你希望背还是不背？我先看看你们的期待。而且实际上那很重要吗？我只是一颗种子，最后的结果是谁？是在座的你们，亲！不要小看了自己的力量，真正有力量的是你们，brighter than the moon。大家来到直播间啊，怎么看——红本本、小破站上面大规模批判索引，说这是盗名、是歪屁股？没有啊！

他们现在不是在搅浑水吗？现在的这种操作方法叫做“万物皆盗名”。我家门口的大柳树是盗名，对吧？嗯，我在看的语文课本是盗名啊，三体是盗名，还有什么电影是盗名，对吧？哎，这个东西是什么呢？一哭二闹三上吊啊！这个操作方法非常常见，是一种注水掺沙子的撒泼打滚式的操作方法。

实际上呢，之前大家听说的什么“万物皆来自于永乐大典”，就是同样的操作方式。那是前些年的时候，实际上是从本世纪初开始，对于西方历史的质疑就已经出现在中国的舆论场上面了。从出现开始就引起一片哗然。之前那个小伙伴还专门分享过这个概念，所以我觉得他说得也很有道理。就是本世纪初，开始有几件事情几乎在同一时间起步，而且都是从民间自发开始的，逐渐生根发芽，星星之火进而燎原。什么事呢？一个是汉服复兴运动，一个是红楼索隐派复兴，一个是西方历史造假的大揭秘。这三个事情全部都是自下而上的，从民间自发而生的，从草根开始崛起的星星之火。它们的开端全部都在本世纪初。

那么这几个行当当中都出现过你刚才所说的那种撒泼打滚的行为。什么样的现象呢？就是当年，本世纪初，西方伪史辨伪出现的那一刻，也就是所谓的西方伪史论那个时候，叫做唯史论。那时候唯史论者刚刚开始出现，名声非常坏。为什么呢？因为掌握学术话语权的全部都是科班出身的，全都是官科，掌握了学术话语权，他们说话才算数。于是呢，当民科站起来问“他到底吃的什么”的时候，官科就开始疯了一样，引经据典——什么什么书上面说得明明白白的，人家有什么样的民主制度；什么书上面说得明明白白的，人家有什么样的雕塑艺术；什么书上面明明白白的，他们有什么样的交易、什么样的密码，如何如何特别特别多的论述。但是别人问的是啥？问的是“你吃了吗？”他们从来回答不上来这个问题。就这样一点一点左一个补丁、右一个补丁，越补越离谱。最后由于互联网上不断的完全不对称式的厮杀，引得旁观的群众意识到：诶，这个提问的人好像比回答问题的官科还靠谱一点。那些官科怎么从来不正面回答别人的问题呢？是不是答不上来啊？于是就开始出现了攻守易势。

一开始的时候是唯史论者主攻，攻城之仗特别困难，因为对方是有壁垒的——学术壁垒、学术堡垒，而且自身是一个非常成建制的部队：各种各样的学生、老师全都在里面，官科体系庞杂。但是经过不断不断的辩论，引得围观群众意识到这些官科每一次科普都绕开了核心点，每一次批判对方的时候都没有放在对方提出来的问题上面。于是越来越多的观众开始转向：不对劲吧？到底谁说的对呢？哦，我觉得可以听听这边在说什么。当这种局面开始风向转变，叫做攻守易势出现的时候，局面就变成了一个新奇怪的现象。

于是你就会发现，这个时代开始出现了一种“万物来自永乐大典”——说永乐大典里面有核武，之所以中国现在发展不起来就是因为永乐大典不见了，核武器被人家偷走了；永乐大典里面有太阳能发电，中国人电力系统不行就是因为太阳能发电的东西被人家偷走了。万物都是永乐大典，意味着旁观者一听：怎么可能呢？开什么玩笑逗闷子吗？所以就会觉得某一边特别荒唐。这是什么？这是那些之前西史的信徒们，从原来在高高的壁垒上面防守，很轻松自在地居高临下俯视他人，变成了养攻啊。他已经开始需要用这样造谣生事的方式去养攻，他变成了攻方。这叫做攻守易势。

你看着对方龇牙咧嘴的，特别牙尖嘴利，似乎营造了一个极为可怕的舆论氛围，但事实上它恰恰反映了此时此刻这种西史信徒的群体内心里非常虚无，以至于他需要用造谣生事的方式去对对方发起进攻。要知道，以前他不这样的。他随便一个防护盾比别人的脑壳都硬的时候，他就不用做这件事情，现在他要做了。

那么现在，包括红楼的这个索隐派，道理上面是一样的。从本世纪初开始，红楼的索隐派卷土重来，它不是刚刚兴起。注意了，前面汉服也是复兴，不是新建；而这个索隐派也是复兴，卷土重来。因为实际上在清朝的200年间，红楼的解析从来都是索隐派的天下。一个小说，一个文字狱王朝治下的小说呀，它里面用猜谜、用埋密码的方式，天经地义，这就是最合理的方式。根本不是什么“你在那玩文字游戏”。小说不玩文字游戏，你让什么东西玩文字游戏？他说小说必须得记载真事儿。谁跟你说的？规定了小说里面的A必须是现实里的B，小说里面的B必须是现实里的C？有这种规定吗？没有。小说这个东西，它就是在那里搞指桑骂槐、移花接木、文字游戏、猜谜语的，这就是小说呀。所以实际上索隐派在清朝的200年间就是绝对主流，大家都是在那里猜谜语，而且绝大多数的谜语大家都已经猜得差不多了。这是清朝的迹象啊。

到民国的时候，在那个百忙之中，在国破山河在的那个时刻，这些百忙之中的归华文人纠结起来说：“不行，你不能够猜谜语。猜谜语那叫小道文字游戏，你必须要走大道。你必须把这个旷世奇书写成某一家里面的家长里短、鸡零狗碎，这才叫做伟大的恋爱史诗啊。你必须得是某一个家里面的后院里面的狗屁倒灶的小事儿，它才叫做恋爱史诗。如果你写的是国破山河，那你那玩意儿就不对啊，你这就错了。那叫小道，那叫做文字消遣，那叫做娱乐。”这就是民国时期的宗师们给我们定下来的调调。

所以一直到上世纪末，这些曹学的红学家一直都是趾高气昂、著作等身，一本一本地输出，到处去演讲，生怕大家不知道曹家的小公子是一个包衣奴才的出身，他家里面谈了一场自由的恋爱——到底多大个事儿啊，值得是千古名著？你开玩笑的吗？所以呢，本世纪初，索隐派的卷土重来顺理成章。你不重来就难受了。大家看那些东西都要恶心吐了好吗？天天都在那里牵强附会地硬生生跑到清朝几百年间，去寻找一个似乎跟这个小说里面的某些情节能够对得上号的某一个人。文字狱啊！清文字狱！在文字狱的时代里面去写家里面的东西，然后还吐槽当时的皇帝，你是家里头活够了吗？这种完全不合理的操作方式，是一定会激发逆反的。

于是在本世纪初的时候，索隐派卷土重来，而且一来了之后一发不可收拾。凡是今天依然觉得必须得是考据、只有考据才是真实的——没关系，你可以慢慢等，我无所谓。你是不是在乎考据这件事情，我也不主张你必须得相信，所以我没有这个传教的欲望。我只说一件事情：一个人，不管怎么样虔诚地相信考据派，无论多么坚定地相信曹家有那么一个子虚乌有的曹公子，只要你有一天意识到索隐派有道理，一定会一去不回。目前而言，我观察到的所有人、所有读者、所有观察者们，有从考据进入索隐的，但是没有一个从索隐进入考据的。这是一个单向法。所以我从来不担忧这件事情，也从来不规劝，也不需要拽着你、拉着你必须过来看，没有这个必要。你只管去看你的风花雪月，你只管去看那些鸡毛蒜皮，无所谓的，你想看什么看什么。但是当你哪一天意识到里面有谜语的那一刻，你一定会进入索隐派，而且从此以后再也不会看考据派一个字。

这就是现阶段的一个局面。这种此消彼长会导致索隐派声势日隆，而考据派江河日下。局面出现了之后，你说对方会不会急？要知道这个背后的利益链条极长，长到什么地步？大家可以去搜一下：前一阵子我不是才讲过那个谁——绿发会，知道不？唉，搞演讲的那个蒋小姐，她背后的介绍人不是绿发会的吗？绿发会的理事啊，是理事还是理事长还是会长什么的，反正他们家的小头头，和某地的曹学会——就是曹雪芹研究协会，那个小头头是同一个人。兼职兼职知道吗？他搞兼职的。我就问你这个绿发会是不是境外势力？你自个儿琢磨琢磨啊。所以这个背后是有一个非常非常密的、历史也非常悠久的、绵密的利益链条在背后。卖书的、卖课的、卖教程的、搞研究学会的，甚至还包括今天在互联网上不断吹“满清贡献论”的。你们今天所有在互联网上游走的人，一定会看到的东西就是满清贡献论，没有人看不见吧？就是你不管怎么着，他毕竟是有那些了不起的贡献，有这样的人吧？啊，不管怎么着，那清代穿的衣服肯定也依然是中华服饰，肯定也依然是汉服，看到这样的人吧？到处都有啊，乌泱乌泱一片。明明理亏、明明词穷，为什么一定要梗着脖子在互联网上不停不停地发声呢？背后必然要有利益链条的推手啊，不然的话，谁丢得起这个人、掀得起这个眼？

包括现在在那里那个什么“屯宝小娘娘”推广服饰，包括说专门去帮那个明代的皇后设计大礼服、设计场子、金大礼服——闲的吗？那么多的复原的东西，那么多的依据，明晃晃地贴在你的脸上，摆上了，你一定要逆潮流而动。没有利益的推手，谁干得了这个事情？

所以呢，就是这个考据派和他背后的这一整套的经济利益的庞大链条，他一定会反攻。那反攻怎么个反攻法？要知道他们那边实际上除了强词夺理和胡说八道之外，也没剩下什么东西了。没有武器的时候怎么办？要么找一个歪理，但是现在连歪理都找不着的时候怎么办？叫做把水搅浑。哎，大家有没有看到这种战术？叫做当我什么办法都没有的时候，要如何把水搅浑？怎么搅浑呢？三体是一个盗名的小说啊，三体是盗名的小说，发现了没有？这个叫做把水搅浑。旁边的路人一看：什么玩意都盗名？啥玩意都盗名？我家门口大柳树也盗名，左边一颗枣、右边一颗枣，它也盗名？你开玩笑的吗？于是盗名说就变得很荒唐了。这个操作的方式叫做把水搅浑。一点道理都讲不出来的时候，他就会把水搅浑。所以放到西方历史造假问题的时候，他就说一切都是从永乐大典里出来的；放到红楼索隐派的时候，他就说一切都是盗名的。白噪音战术。

嗯，好的，欢迎大家来到直播间。嗯，那个啥都到这个点了，点赞、点赞、点赞啊！开始动手啊，开始动手，开始动手啊！点赞，点赞点赞！你看我现在提要求已经提成这样了啊，点赞！戳脸，梆梆梆，砰砰砰咚咚咚哒哒哒啊！动起来啊，动起来！动起来！我们要开始进入到今天的正题了。不是前面白活了一小时，白误了一节课，然后跟我说“现在终于进入到正题了”？谁家老师这样啊？哎呀，谁家老师能这样啊？
