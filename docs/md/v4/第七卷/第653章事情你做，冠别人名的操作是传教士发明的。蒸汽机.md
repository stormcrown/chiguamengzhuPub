# 第653章 事情你做，冠别人名的操作是传教士发明的。蒸汽机

他去追溯一些当年西方传教士传给我们的科学典籍时，就会发现里面有一种通行的问题：你看着书上的署名是某个传教士，开篇也会说哪个传教士什么时候把这本书传给了我们。但一旦看到书中的细节和描述，你就会发现里面所有的措辞和原理都是中国典籍里的固有词汇和固有算法。这意味着什么？就是你里面做题、解题的所有内容，全都出自中国典籍，但署名却是个洋人。

这个蒸汽机到底是谁发明的？不管你觉得蒸汽机是谁发明的，蒸汽机本质上就是一个烧水的装置加上动力传送装置。那么，烧水的东西是谁发明的？中国人发明的啊。那传动机制呢？我们有水排、风排，这些全部都是动力传送机制。也就是说，传送机制是中国人发明的，烧水的装备也是中国人发明的。然后你说，唉，就是西方人如何如何——就算他们发明了蒸汽机，没有这两样东西，蒸汽机也只是个梦。

蚕丝也是如此。他们在书里说他们也有蚕丝。如果真把这项技术学会了，并在那个地方又种又养，搞了300年，出现了丝织产业，那我们今天还能证明这玩意儿是我们的原产、是我们的原创吗？所以，你说去对历史进行修编、对历史较真儿有没有用？去质疑西方历史里那些词不达意、逻辑漏洞的东西有没有价值？要我说，这世上除了极少数的专业科研人员需要保有谨慎的自信之外，世间绝大多数普通人需要的就是盲目的自信。不然呢？你天天自我怀疑，又学不会如何找证据，脑子里的逻辑不足以支撑你去质疑和批判，也不足以直接研究清楚事实如何。绝大多数人就是这样的普通人，如你如我一样平凡又普通。结果你说必须要有所有证据才能说话，否则就是盲目自信？大家一起闭嘴啊。你不能只让中国人闭嘴，也不能只让中国人不自信，你必须让所有人都这样。如果你能做到这点，我就支持。

你会听到很多很多信息：除了水稻是我们培育的，小米是我们培育的，其他东西都是外头传来的。我们发现了什么都没有用啊？我们的祖先天天饿肚子，因为这也没有那也没有。前些年甚至有人说，猪都不是我们驯化的了。这也就是因为我们驯化猪的证据实在太多，后来这套说法才少了。但实际他们也曾尝试过说猪不是我们驯化的。嗯，水稻是实在的打脸，要不然还是国外的。实际上，在世界范围内有相当一部分群体至今依然认为水稻是隔壁阿三、隔壁巴拉特他们家培育的。到现在都有这种言论。之前我们的水稻被挪用到阿三那边、巴拉特那边，世界范围内绝大多数人都承认那种。后来是因为发掘出的考古遗迹实在太多，我们的考古太扎实了，所以水稻才没被刨掉。不然的话，我们的水稻也是从外头移栽进来的。

在这个事儿上再分享一个：明代的时候，不是有很多传教士来中国吗？后来都说我们的地图、科学、数学、几何、天文都是跟这些传教士学的，都是欧洲人教给我们的。这是后来的说法。其实在当时，这些传教士在国内干了什么事呢？他们当年来的时候号称要传教，但传教的成果非常糟糕。基本上他那套神话故事在中国忽悠不着人，简单来说是这么个道理。所以他在传教领域没什么成效，基本上没有任何成效。但他们身上有一样东西——白银。今天我们知道，很多地方搞传教时发鸡蛋，鸡蛋没吃完的时候你还要信一信，一直等到下一板鸡蛋来。鸡蛋一没有，信仰就直接归零了。中国人基本的态度就是这样。非常虔诚、非常魔怔的人不多。但在明代，他们身上有很多银子，而中国当时最重要的货币就是白银，且中国严重欠缺银矿。你觉得会发生什么？也就是说，他不发鸡蛋，改发银子。这种情况下会产生什么影响？他想让谁听他的话，拿银子砸都能砸出来。于是你会发现，这些传教士在明王朝占据了相当不错的岗位，甚至其中一部分人的工作可以切入到图书的编撰中。而编撰图书这件事，对中国人的影响非常大，因为我们对于文化非常在意。所以，凡是板上钉钉写在文献里的东西，我们总觉得是真的。但那个时候的图书编撰中，出现了很多洋人的参与。

所以这几年如果你关心互联网，就会看到有一些专门研究西方伪史、西方历史造假问题的人——主要是民科，也有少量官科。这些人去追溯当年西方传教士传给我们的科学典籍时，就会发现里面有通行的问题：你看着书上的署名是某个传教士，开篇也会说哪个传教士什么时候把这本书传给了我们。但一看到书中的细节和描述，就会发现里面所有的措辞和原理都是中国典籍里的固有词汇和固有算法。这意味着什么？就是你里面做题、解题的所有内容，全都是中国典籍里的东西，但署名是个洋人。你有没有觉得这事儿有问题？换任何人都会觉得有问题。而这就和明代时这些传教士手握白银，已经切入到图书编撰管理工程中造成的结果有关。

其中有一个比较极致的例子：当年的这些传教士曾编撰过一个小册子，介绍欧洲的风情和科学技术发展。其中有个段落说欧洲人会种桑养蚕，会种植水稻，还讲了一些种植和养殖的基本技巧，跟我们高度相似。你说他独立开发出这种可能性有没有？万一呢，说不定也行啊。但这里头有太多存疑的东西了吧。如果说他真的自古以来就能养蚕了，那我们古时候2000年的丝绸之路上的奢侈品到底卖给谁了？买家突然消失了，你知道吗？这个买家家里啥都有，我们到底卖给谁了呢？就不知道了，然后直接就归零了。我们2000年的丝绸之路就给你归零了啊。咱们就不说他说不通的东西了。重点是什么？重点是这东西在明代被放到了书里，被传教士书写下来，塞进我们的书里，说他们自古以来就有水稻种植和种桑养蚕。但最后很明显的，他那儿没有。然后今天我们才会依然公认水稻、桑蚕都是中国原产、中国原创。那你想，如果在明王朝时他们在书里塞进他们自古就种水稻，然后他把我们这里的水稻种植的东西和种子完全复刻到欧洲去，欧洲也真的大面积推广了水稻，种了300年。然后今天我们再一翻这些典籍，发现里面真的就写了他们种水稻，你还能证明这个水稻是我们这里原生的吗？是不是难度系数就更大了？包括蚕丝也是如此。他们在书里说他们也有蚕丝。如果说他们真的就把这项技术学会了，并在那个地方又种又养，搞了300年，出现了丝织产业，那我们今天还能证明这玩意儿是我们的原产、是我们的原创吗？

所以，你说去对历史进行修编、对历史较真儿有没有用？去质疑西方历史里那些词不达意、逻辑漏洞的东西有没有价值？嗯，为什么明自己不在海外挖银子？挖了呀，刨了呀，明王朝也出去找银矿了呀。但是这不是赶巧了，他们找到的银矿比我们的大。呃，啥？一切现代文明都源自我东大老祖先的四大发明是吧？比如芯片就是用四大发明之一的造纸术是吧？嗯，你这个就是低级红、高级黑是吧？看得出来啊。你说那个什么永乐大典里有原子弹，然后大家就觉得了，原来永乐大典是骗子啊。我能理解你这种思路，但是你的这种嘲讽毫无价值。为什么所谓的四大发明，是在清末的时候才被总结出来的中国文明的贡献。而清末的时候，中国人已经自卑到了什么地步？整个清代，我们的技术不断倒退，文化水平不断倒退，人的外观形象不断下降。清末的时候，中国人的自信感、对自身文化的认同感已经差到了一定的地步。为了给自己提个气儿，所以总结了一下所谓的四大发明。我们对世界的贡献真的是四大发明吗？在我们最自卑、最瞧不起自己、甚至最自恨的时候所总结出来的四大发明，怎么可能是我们真正最伟大的贡献呢？我们在这个世界上真正伟大的贡献来自于什么？天文、地理、数学，也包括大量的工业。就是你所以为的这个动力装置，是所有时代迭代最重要的东西，对吧？那动力装置是什么？无论你觉得蒸汽机到底是谁发明的，蒸汽机就是一个烧水的装置加上动力传送装置。烧水的东西是谁发明的？中国人发明的啊。西方人一直到乾隆年间才实现了高炉冶铁，在那之前他连个锅都做不出来。所以烧水的这种装置本身就是中国人发明出来的。那传动机制呢？我们有水排、风排，这些全部都是动力传送机制，也就是说传送机制是中国人发明的，烧水的装备也是中国人发明的。然后你说，唉，就是西方人如何如何，就算他们发明了蒸汽机，没有这两样东西，蒸汽机就是个梦。

水利工程建设，你觉得这不重要吗？全世界有几个群体能在古典时期真正驯服大江大河？中国人做到了。中国人在良渚文明中就已经有了大规模的水利基建。为什么西方人的传说中大洪水需要上帝的指导去造一座方舟，然后拯救极个别的人？因为他没有驯服水的能力。为什么中国人是大禹治水？因为我们在学习如何驯服自然，而最终我们做到了。你觉得这是不是贡献？我们驯服了水，驯服了植物，驯服了动物，是不是贡献？我们各种各样的手工艺品，是不是贡献？我们的农具，是不是贡献？水车、风车，是不是贡献？我们天文学几千年的档案，是不是贡献？这是你一做梦就能出现的东西吗？我们这里有二进制、五进制、十进制、十二进制、十六进制、六十进制，各种各样的计算方法、计算法则，在现代社会的所有基础研究中都继续应用，这是不是贡献？结果你说，哎呀，那都是西方人整出来的。一个在语言上使用十二进制的国家，为什么开发出来的所有技术运用的都是十进制？它的语言当中没有十进制，但它的所有东西都用十二进制、都用十进制？好不好玩啊？罗马数字没有办法直接拿来进行计算，这个数字编出来是干嘛用的？你根本就解释不清楚这些问题的时候，就说中国人盲目自信。我们的祖先每一个脚步都是扎扎实实地踩在地上的算数。我们是从零到一，一个一个数出来的；天文，我们是一个星星一个星星描画出来的；水利，我们是一条河沟一条河沟梳理出来的；种子，我们是一个品种一个品种驯化出来的。我们知道我们的来路是什么。他们说他们也有，可以，但是你的来路呢？为什么没有？为什么在某一天的时候，你突然之间就有了？之前呢？打哪来的？能解释一下吗？解释不清楚，然后就指着中国人的鼻子说：你盲目自信。

要我说，这世上除了极少数的专业科研人员需要保有谨慎的自信之外，世间绝大多数普通人需要的就是盲目的自信。不然呢？你天天自我怀疑，又学不会如何找证据，脑子里的逻辑不足以支撑你去质疑和批判，也不足以直接研究清楚事实如何。绝大多数人就是这样的普通人，如你如我一样平凡又普通。结果你说必须要有所有证据才能说话，否则就是盲目自信？这世上所有的普通人都不是这样做的，为什么只有我们是这样做的？如果你说全世界所有的人都必须拥有百分百的证据才能说话，相信我，这世上99.99%的人都会立刻变成哑巴。因为在各行各业中，真正能够实现这样高精尖研究的人在人口中的占比也就是0.01%，其他人都不是做这些事情的。你说你就必须有什么什么才能自信，然后大家一起闭嘴啊。你不能只让中国人闭嘴，也不能只让中国人不自信，你必须要让所有人都这样。如果你能做到这点，我就支持。
