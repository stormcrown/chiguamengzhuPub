# 第605章 中华文明天然强大的原因？

本文考古到八九千年前的作物，是因为它和中国接壤。大家关注三海平原的考古学，最近很多人都在讨论这个，但是里面的很多东西并没有公布，所以我觉得可以先等一等消息，先等一等公布。现在他公布的信息不全，那么，能不能从地理历史学角度解释一下中华文明天然强大的原因？

其实我之前所讲的也差不多是这个领域，就是关于我们的文化发源，之所以能够灿烂成这个样子，是基于什么样的物质原因。这件事情还是挺重要的。因为近代以来，很多人以为中国的地缘政治条件不好，这几十年其实一直都有人认为中国的地缘条件不好。但实际上就文明发源来说，我们的地理条件是特别好的，特别好，不是一般的好。

之前西方的中心主义者，在全世界安排各种文化源头的时候，其实都在考虑一件事：什么地方能够诞生文化，什么地方能够诞生文明？所以你会注意到，他们给自己创作这些文化源头的时候，选择的常常是一些大河流域。你一定会发现他们会选择大河流域。那么，大河流域为啥非得这样选？你想一想那个两河流域，两河那个地方，问题在于气候条件非常不好，很不适合农耕。但是因为那个地方有两条大河，所以最后就说这个地方是农业发源地。这就是他们在选择文化源头时都要选择大河流域的缘由。他参照的样本是什么呢？他参照的样本就是中国。暗示中国就是因为有两河所以才牛起来了吗？不是，我们的地理条件是非常得天独厚的。它还真不是因为只有两条大河。你想驯服这两条大河，那个年代相当久远。驯服这两个大家伙，相当久远。所以你注意，我们的文化发端时，是出在大河的支流支系上的，不是在主干道上。

那我们先说一下，我们这个地儿到底好在什么地方。一个就是我们的青藏高原。注意了，青藏高原是一个重大的地理优势。一方面，它是我们各种水系的发源地，就是水塔、高地；另一方面，它又影响了季风的吹拂，影响了从海上带来的各种湿气等空气。整个西部的各色高原、山脉，最后总结下了这一部分湿润的空气，造成了我们这片土地上非常独特的现象：一个是有两条主流的大河，以及它们的各种支流、支系的小河汇合；同一时期还有一个最重要的东西，很多地方都没有，就叫做雨热同期。雨热同期这个玩意儿，绝大多数地方都没有。你看他们在那里吹得牛哄哄的，什么古希腊伟大的文明，他们那雨热不同期：下雨的时候冷，天热的时候干旱。这种条件不利于农耕，很多作物长不好，只有耐旱型作物才能长得好，但产量不够高，所以能够养育的人口数量就会少。人少，精英的数量就会少。要知道所有那些划时代的想法，那些特别优胜、一出现就开创了新时代的人，那都是要撞大运的，叫做基因彩票。你要碰运气，没有足够大的量级，你上哪碰运气去？这玩意儿都有命中率的。所以实际上，雨热同期这一件事情就决定了这里的农作物丰富程度和生长效率都远远高于其他地方。这个东西叫做得天独厚。

然后有了这些物质上的前提条件。另外一边是大海，东边是海，西边是群山，只有北部会面临草原。北部面临草原，草原上降水比较差，土壤层比较薄，所以游牧群体会迁徙经过，它们在上面不停迁徙移动，你方唱罢我登场，时不时来一波，时不时走一波。这上面有一个草原通道，产生了一个什么样的影响？第一，草原通道负面的影响就是经常出劫匪；正面的影响就是你知道外部世界，还有一个世界是流通的。所以它有一个流通的通道，同时也确确实实形成了一种现象：中国人始终都知道有外人的存在，始终都知道有敌人的存在，也始终都知道遇到军事威慑时会出现巨大问题。所以我们这个地方才不是只出农民，除了农民还产生战士。就是因为无敌则无我，一定要注意这点。如果真的是全封闭地形，我们未必能发展到今天这样。你都不知道外面世界什么样，不知道抢劫的人长什么样，自身的战斗力就完蛋了。

所以你看隔壁的印度，它在某一些角度上看，它的封闭性比我们强。它只有一个山口是打开的，你但凡把那个山口守起来，整个南亚次大陆就是非常单独存在的一个地方。但这就造成了问题：它的敌人很难路过，它不经常看到敌人；但一旦有敌人发现这个地方并进来，那就一马平川地被推下来了。所以那个地方就变成了殖民地，一波又一波，始终不停地沦陷为各种各样的殖民地。而我们这个地方，上面有一条通道，既可以提供一定量的贸易交流，同时也让我们清楚地意识到有外人的存在。所以我们是耕战传家的国家，这是前提条件。

然后，讲我们内部的农耕生态的复杂性。它是由我们的主干河流——长江与黄河两条主要河流，以及它们复杂的支系共同构成的密布水网。这些东西是我们拥有农耕基础最重要的前提条件。我们去驯服大江大河的历史其实很短暂，难度系数特别高。但是我们通过了解、学习这些密布的水网，能够理解河流的运转方式，这叫做堵不如疏。如果没有那么多河流支系给我们提供示范，我们那么容易就学会应该如何治水吗？我们就是从小学、小河到大河到汇合，这个过程当中学会治水的。我们从自然中学习，再应用到自然中去。所以当我们学会了治理水资源，就可以让耕地的效率提高到一个新平台。我们的耕作能力越来越好，人就越来越多。人多，一方面是战斗能力强，另一方面是生产能力强，还有最重要的一方面就是会产生大量的精英。
