# 第651章 万世降表的孔家，代表儒家吗？

儒学诞生的那个时代，中国人站起身来，直面蛮荒。除了我们自身闪闪发光、作为一个文明体之外，其他群体还在茹毛饮血、奉行至尊强权，遵循丛林法则。而你却在嘲笑我们的嫡长子继承制过于迂腐，你可曾站在那个时代考虑过它的先进性和跨越性？没有，这就叫做本本主义。《毛选》常读常新，能给你换换脑子，让你想清楚：我们如何看待现实、如何看待历史、如何分清敌我。嗯，怎么了？聊儒家？哼哼，嗯，又有人又有人开始痛恨儒家了是吧。嗯，儒家多技术啊？儒家说吧，儒家说技术是奇技淫巧——这是满清皇帝说的。清代那些犬儒、狗奴才们所做的一切事情，都要算在2000年前的儒家身上，是吧？2000年前的慷慨义士是儒生，但满清200年的走狗不代表儒生；所有侠客、志士、抛头颅洒热血的那些人，他们不是儒生，只有孔家那几个万世降表的狗东西才是儒生，对吧？当你想挑一个群体里的反动分子作为个例，然后说这就是普遍性、所以儒学是坏的，那就意味着你的屁股已经歪了。这件事有什么可谈的呢？你想说儒学坏，至少要把《论语》里的东西列出来，说哪句话坏了。哪怕你说一句“唯女子与小人难养也”不符合男女平等，我都感激你——至少你还有点文化。嗯，什么？嗯，刚才还吹大英美利坚？呵呵。嗯，我们现在有一批人，很执着于否定儒家。如果你是在说万世降表的孔家有什么不好，那他肯定不好——他做了坏事，确实不好，毫无疑问。但这些人一定要说儒家不好，你以为他是在批评某一种学问，实际上不是的。他说这句话，本质上是在批评中国人的文化是落后的、没希望的。在座的所有中国人，不要觉得人家说儒学不好跟你没关系。为什么这样说？儒学是什么时候奠定的？单论“儒”这个字，它的创作是从我们开创《易经》、八卦时就已经开始出现雏形了——它是人对思想的理解，这是最早的儒。我们今天所理解的儒学，源自于先秦时期恢复周礼的孔圣人。你觉得他恢复周礼这件事落后，但周礼是什么？周礼是社会秩序，是一位学者思考如何构建和平、安稳且可持续发展的社会秩序的一份考量。然后你说它落后——什么意思？是指我们的社会秩序从古至今都是落后的？而中国人今天身上所拥有的所有优势，包括我们父系基因的纯粹、积累下来的所有文化典籍、所有技术思维、包括我们的九九乘法表，所有这些，都是广义儒学中的一部分。我为什么说身上的血缘都跟儒学有关系？儒家最讲究的就是华夷之辨，最讲究的就是父父子子，最讲究的就是“微管仲，吾其披发左衽矣”。我们今天和我们的祖先还能长得这么像——今天的陕西人去看秦始皇陵兵马俑时，还能看见一个跟他大舅长得特别像的俑——就是因为我们有儒学这样一种思想，所以我们才能和祖先保持如此高的相似度，文化的稳定性才会那么好。自先秦以来一直到明末，我们都是当世第一流的国家。只有在战乱时我们会受一点苦，但全世界所有人都会频繁地经历战乱，而我们这里的战乱远远低于世界平均值。这是什么功劳？这是社会秩序构建的功劳。我们今天所有人活下来，成为今天的样子，长成今天的模样，都受益于先秦儒生的努力。虽然不只是他们，但跟他们息息相关——它是我们文化的重要来源之一。政治是我们身上根骨中非常重要的东西。大家心里想的“有仇必报”，复仇主义来自于儒学；“复仇不共戴天”来自于儒学；“国仇虽百世可复”也来自于儒学。所以那些人跟你说“儒学耽误了我们的发展”，你以为他在说什么？他在说中国从古至今就是个错误。你愿意承认自己从古至今都是个错误的话，买根绳子也不贵，很简单。你那么痛恨儒学，认为儒学影响到了你的所有身家性命，让你感到耻辱——买根绳子不贵，下辈子去巴拉特啊。那里没有儒学，那里又多元、物产又丰富、又浪漫。想去哪儿我都祝福你。好，欢迎大家来到直播间。我这里是一个带货直播间，现在带的这个东西……凡是家里没有的，赶紧入手一本啊。《毛选》是必须要读的。你以为我前脚说儒学，后脚就推荐“打倒孔家店”的《毛选》，是对抗的行为吗？不是的。我推荐大家一定要好好看一篇文章——《反对本本主义》。啥意思呢？时代在发展，矛盾在变化，不要刻舟求剑。要找到主要矛盾与次要矛盾，找到事件发生的真正要害，并解决问题。而不是把今天所受的某一个痛苦——比如我今晚只赚了十块钱——这笔账算在谁的头上？哎呀，这就是2000多年前孔夫子的罪孽？就因为孔夫子说“君君臣臣”，所以我今晚只赚了十块钱？这叫刻舟求剑。因为你听说儒学不好，于是你受的所有的罪都是儒学造成的，这叫做本本主义。我们唯物主义的第一要求、第一思考，就是直面现实的现实，也直面历史的现实。面对任何时代的历史，要站在当时当地的环境下进行考量。儒学诞生的那个时代，中国人站起身来，遍野是蛮荒。除了我们闪闪发光是一个文明体，其他群体茹毛饮血、至尊强权，他们在丛林法则。而你在嘲笑我们的嫡长子继承制过于庸俗，你有站在那个时代考虑过它的先进性、它的跨越性吗？没有，这叫做本本主义。《毛选》常读常新，可以给你换脑子，让你考虑清楚我们如何看待现实、如何看待历史、如何分清敌我。我们现在，在互联网平台上，你所能看到的各种稀奇古怪的对中国历史上人物的点评，以及对中华文化缺陷的点评——例如“我们之所以落后，是因为商鞅缔造的秦制”。这时候他就不考虑商鞅缔造的这套制度能让秦王扫六合，把这个事直接放掉了，甚至论证“秦王扫六合就是我们落后的根源”。他在干什么？他在反对大一统，他希望让中国沦为像欧盟那样的散装小弱鸡。就这种动机。所以你说，他是在否定什么什么，他认为那个东西不好，他能真正证明那个东西不好吗？人类社会是什么？人类社会所选用的一切治理方案都是社会实验。人类社会的一切社会治理方案都是社会实验。有人提出畅想，然后我们组织实践兑现它，再核对这个东西对或不对，有好处或弊端，有弊端我们下次调整，有好处我们就照办——这个东西就叫做社会实验。这种行为是什么？这是科学，这是科学啊！亲，提出理论、付诸实践、考察利弊、不断调整，这玩意儿就叫做科学，放到哪里都叫做科学。我们从古至今的所有制度都是这样建立起来的。所以你说“哎呀，万世降表是错的”——我们实践了2000年，行之有效，一直有效，我们才用。古人——谁傻？反正不是古人。什么事都怨？祖宗当年要是不搞统一文字，我们今天就可以怎么怎么样了？哼，你要是觉得什么东西特别不好，你自个儿改。今天还有人说“哎呀，华夷之辨太排外了，不好”——华夷之辨让我们活下来的，我就说它好。你要是觉得它不好，说我们不应该排外，应该大混血，你自个儿出去混去。不要什么锅都背给祖先，你家的先人不为这件事情负责。嗯，实践之前是不是要先小范围试验？不都是这样的吗？谁会突然一下子就实践出来了呀？郡县制最开始是在各个诸侯国里小范围实验的，后来发现还行，才一点点推广的。商鞅变法那种深入的变革，也是在秦国的一隅之地去做的，后来发现确实能富国强兵、横扫六合，才在全国推行。不就是这样的吗？谁有本事说立一个规矩，咔嚓全国照办？他有那个能力吗？他们本质上做不到。嗯，嫡长子继承制。嗯嗯，你脑子不好，不喜欢嫡长子继承制？嗯，看出来了。你猜我看出来啥了？嫡长子继承制是干啥的？一个国家要维持稳定和安定，重要的并不是那个君王是不是横空出世的天降奇才，而是这个君王能不能按时上班、按时下班当他的皇帝。所以嫡长子继承制干什么？它可以避免打架、避免竞争、避免偷鸡摸狗，可以让我们的政权传递安安稳稳、十分有效，让底下的人知道该听谁的、该拥护谁的，有明确的指标，不需要分裂站队。为什么我们的先民说“夷狄之有君，不如诸夏之亡”——也就是我们华夏先民就算亡国了，都比那些有君主的蛮夷过得好？为什么？因为我们是有秩序、有体系的，知道跟谁走，知道里里外外，所以它才好。那为什么蛮夷的秩序有君也不好呢？因为他的君王是怎么得来的？是通过力量的角逐而得来的。力量是怎么角逐的？你随时随地都要打一架，那才叫做力量的角逐。这是一种自然属性。大自然里的猴群就是这样的：老猴子年老了之后，其他猴子就把它揍一顿，揍趴下了就变成新猴王。这就是那种相对原始、相对落后的群体用这样的方式进行部落主的推选。你想它乱不乱？每次更新迭代都要打破头。这还只是在猴群里面，只需要武力战胜就可以了。在人类社会当中就要比这复杂得多，因为人类不可以单对单打架——那没有意义，那叫匹夫之勇。人类的斗争常常是一个队伍和另一个队伍的斗争。斗起来了之后，什么玩意儿都没了——你家里搞一场大规模的械斗，你觉得你房子还在吗？牛羊还有吗？所以你只要是搞这样的更新换代，用这样的方式更新换代，一定会乱。为什么要嫡长子继承制？可预期，我们可预测、不站队，安安稳稳地传递下来，然后这个国家才是一个有秩序的、可以生产的、可以维持文明体系、可以攒下家当的国家。嗯，美国的选举就是猴儿打架。对呀，4年选一次。你觉得他们的总统在他们岗位上面到底能干多少活？选完了到就任，就任那一刻开始就要安排下一次选举的所有预备项。他中间安安稳稳坐在凳子上的时间就两年。一个国家的政策推广，很可能你下一条政令要10年之后才能看到效果，但当你只有两年的有效期限时，你能干这样的活吗？做不了。所以永远都是短线操作，每隔几年就打一架——打得是挺文雅的，只剩下嘴上打架，但依然劳民伤财，依然不可持续。所以为什么要嫡长子继承制？因为我们的秩序非常完整，即使上面那个皇帝不太精明，只要他是个正常人，问题就不大。嗯，好，我到点了啊，到点了到点了。嗯，再见。
