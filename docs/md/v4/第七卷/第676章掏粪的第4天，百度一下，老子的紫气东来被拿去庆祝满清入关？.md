# 第676章 掏粪的第4天，百度一下，老子的紫气东来被拿去庆祝满清入关？

嗯，欢迎来到直播间。什么？哦，欢迎啊。什么？紫气东来？真的，那个“紫气”这俩字儿，我觉得以前曾经是个好词儿。但是打从他们搞了那个“紫气东来”的活动之后，我觉得“紫气”这俩字儿都变得很恶臭了。已经这么臭了，是不是应该刨粪？感觉应该是这样啊，感觉应该如此啊。我们这儿那个什么……我们这来了一个……嗯，对，紫气，紫气。为啥说这个呢？因为之前有一波满洲复国主义势力，满洲复国主义势力整了一套文化活动。当然了，因为被抓的那群人，跟他们之间不是一对一的关系，所以现在可能还会有人继续办那个什么“紫气东来”的活动。因为这个玩意儿可以归类到民俗里面去，尽管这个民俗的诞生史大约是从90年前后开始的啊。为啥我不说具体的年份？因为那年份不让说，你就知道是哪一年了。就是从那个时候开始，他们开始纪念满清入关，开始纪念满清之崛起，并且把这个行为、把这个日子、把这个气象万千啊，称之为“紫气东来”。那如果说满清入关叫做“紫气东来”的话，那这个“紫气”就是我们中国人的巨大灾难，灭顶之灾。所以从这一刻开始，从他们开始吹嘘“紫气东来”的伟大纪念日开始，“紫气”这两个字儿和粪气就没有区别了啊。就是粪坑又开始冒臭气了。为什么呢？因为蛆又活了啊。这个就是我这两天天天都在给大家推广的粪坑论啊。为什么要说清的烂？因为满清就是我们历史进程当中的一个硕大的粪坑啊。为什么要刨屎啊？主要的原因就是：这个粪坑里面有大量的蛆，它不肯安静地在那里发酵、发臭。正常的粪坑应该会发臭，然后成为养料，我们以史为鉴，吸取经验与教训。但是呢，满清不一样，满清是一个活着的粪坑，里面拥有无数的蛆虫。这些蛆虫呢，开始吹嘘“紫气东来”，开始吹嘘剃发易服是改善了大家的卫生与健康，然后开始吹嘘“女真不满万，满万无人敌”呀，去吹嘘他们仁义之师啊，说扬州十日、嘉定三屠都是造假，都是一些伪造的故事啊。实际上呢，他们是一些满汉如一体、爱天下皆如子民的圣明天子呀。自打他们玩起了“紫气东来”，这些谎言与谣言，基本上就属于眼睛都不需要眨一下、就可以撒谎撒满篇的程度了。因此这个静静的、沉默的、恶臭的粪坑，又因为蛆虫的鼓勇，于是就活跃起来了。它只要活跃了，会产生什么样的景象呢？在粪坑里面的这些蛆，就会把屎鼓涌到岸上去，脏我们的鞋底。同时呢，蛆会长成苍蝇，苍蝇又会下蛆，子子孙孙无穷匮也，也逐渐影响到我们的生活环境。我们吃一口饭能够想到蛆，然后吃食物，能够看到苍蝇屎，日子就会变得越来越难过。睡个觉的时候，身边都能听到苍蝇的嗡嗡声，脑子里面时不时的就会回想着“我满清第一巴图鲁”这些苍蝇的嗡鸣。所以如何解决这个满天苍蝇飞舞的罪恶景象呢？要除四害，要从根儿上刨起。如何除四害？一定要掏粪，持续不断地掏粪，把里面所有的蛆虫都掏干净，断绝它的繁殖链条，然后让这个粪坑变成一个静静的粪坑，它才能够成为我们以史为鉴的历史养料。怎么以史为鉴？那就叫做勿复南明旧事。所以这会儿弹一波南明史。哎呀，好丝滑！太丝滑了！感动感动感动！这是我第一次这么丝滑啊。快点走起走起走起，赶紧给我鼓个掌啊。我之前老是想不起来应该带什么。你看，我们讲完了掏粪之后啊，我们如何吸取历史教训？我们怎么能够在这件事情上达成共识？第一件事情就是要学习南明史。要了解满清入关到底罪恶到了什么样的地步、反人类到了什么样的地步、我们的损失又惨淡到了什么样的地步，我们在那个时期到底犯下了多少的错误，为什么在局面可以扭转的时候居然急转直下，那些拥有技术的、拥有知识的、拥有眼界的、顶流的精英们，到底怎么样从一个错误走向另一个错误的？这个东西就叫做以史为鉴。所以这一套南明史完全值得大家去典藏，因为顾辰老师所写的这个南明史，是这几十年以来非常难能可贵的一套以阶级史观立案的历史书。我们这40年来出版的各色的历史读物，基本上都没有运用阶级史观进行分析。你说阶级史观就完美吗？我没有觉得它完美。我刚才在一开场的时候就说了啊，所有的文史类，只要你转变了视角，就会出现新的正确的答案。那么阶级史观就是视角的一种，它站在广大的被压迫者的角度上面去看待历史，因为站在了多数人的这一边，所以绝大多数的情况下它都大有价值。还有一种历史观叫做人民史观，它跟阶级史观之间有大量的重叠，它也是站在绝大多数的、并未犯错的、正常的、正义的、有朴素的正确价值观的人这一边去看待历史，那这叫做人民史观，这也很可贵。这几年啊，用这种视角来叙事的专业历史书也不多了。而现在所流行的那种啊，什么地域史观——也就是看地理啊，地在哪人在哪啊，你在哪你就说哪的人——这种地域史观；还有那种特别流行的团结史观——无论如何先团结了再说，至于那个你要团结的对象是谁，它才不在乎，你必须先团结了再讲——于是呢，它的历史叙事就颠三倒四，因为它没有一个特定的人群的视角。没有历史观的时候，你的历史叙事是逻辑不能自洽的。因此能够在今天还找到像这样子用阶级史观和人民史观的角度来看待历史的书，已经非常不容易了，所以这套书很值得珍藏。一旦你想了解满清的历史，最重要的就是南明史这一段。它是从哪里开始？嗯，从崇祯皇帝吊死煤山开始，明王朝的正统到这个时候画上了句号之后，出现了一个南明小朝廷，这一段的历史进程就是南明史啊。整个的这个过程就属于那种，嗯，不高兴、不开心，想一想就觉得有点上头啊。就是这一段，这段历史看起来了之后呢，不那么舒心，不那么快乐，但是它非常重要啊。我们在那个，嗯，抗战期间的话，抗战期间国共联合怎么联合的？双方之间那真的是乌眼鸡似的，谁都看不上谁，恨死了咬牙切齿的，为什么能够国共联合？勿复南明旧事啊。就是因为无论如何，咱们总能够看到在历史的进程当中，这一段历史放在那儿，真正的主要矛盾摆在那儿的时候，还内讧，开什么玩笑呢？所以如果没有南明史在前面打样儿，我们的抗战可能还要更不顺利一些。

嗯，啥玩意儿？矮大紧说明朝三无朝代？嗯，这个我们在互联网上，尤其是公知界啊，恨明朝的人特别多。那恨明朝的人可不止矮大紧一个啊，无数的人都痛恨明朝。为什么痛恨明朝？因为如果明朝是一个强大的时代、是一个富庶的时代，那么清就更加的地狱了。所有把明朝贬低得一无是处的人，他要干的事情就是哄抬清的历史地位，有的时候是哄抬元清的历史地位。因为明就是夹在这俩中间的。但是明是一个什么样的时代？它的历史进程当中有一部分的内容扑朔迷离，还有迷雾遮掩。为什么会有迷雾？因为历史上面被篡改的东西太多了，湮灭的档案有点太多了呀。啊，清代的时候是系统性地对明代的原始档案进行了销毁的。所以今天能够流传下来的，有一些呢，是各地的啊，手抄本。但是这个过程当中有没有造假的行为？你就别说在那种特定的时代了啊，有没有特意安排出来的、专门造假的手抄本？流行不清楚。同时即使什么都没有啊，即使这些障碍都不存在，这世间所有的手抄本都和原版不一样。中国的历史传承当中，所有的手抄本都和原版不一样，这是一定的。因为抄写的人有他自己的主观意志，你就别说这些历史档案的东西了，讲的都是几哪年、哪月、哪号什么事啊，就这种东西，那随便改两笔，你都意识不到，你都想不起来。就连孔夫子的书、老子的书，都有人在进行传抄的过程当中不停地改。孔子身故之后，他的门徒很快就分成了好多个帮派。为什么会分派系？为啥分派系？因为他们每个人手上都有自己的读书笔记。老师不在了之后，最权威的就剩下这些近身门徒的读书笔记了。于是每一个自认为读书笔记做得最好的学生，就会成为一个新的流派。但是每一个学生学的时候，他都有侧重点，都有自己的想法，所以每一个人去记述的读书笔记，跟原始版本一定是会有误差的。所以为什么我说传抄这件事情，只要传了一定会有变形。所以你看我们今天进行这些资料汇总的时候，要在各种各样的传抄本当中不断不断地对照啊，各种各样的对照。那这个对照进行参详的人，有没有自己的主观意识？还是有的。所以我们去进行明史复原，这件事情做了好几十年，但是还原度到底做了多少？研究的人说做得已经很好了，但是既然明代在中国人的直觉印象当中还是一个黑暗、狭小、封闭保守的时代，那么这段历史就还没有实现真正的还原。什么时候明的客观的地位提升到了一个辉煌而伟大的盛世，什么时候这个明史才有值得信赖的一丝可能性。为什么？为什么？因为刚好，在明代开始逐渐走下坡路的时候，西方人就突然之间拥有了我们之前的各种科学、艺术、文化。他咋有的呀？咋崛起的呀？从哪儿来的呢？为什么他那里这也有、那也有，而中国啥也没有呢？为什么时间上就那么刚刚好呢？为什么我们能造大船，它造不出来，但是它的几何和工业学都比我们强呢？为什么我们的高炉和传送机制、水动力、风动力的机制传递得这么好，但是别人啊，发明了蒸汽机呢？所以整个的这个历史进程当中，它就是有客观性的各种各样的障碍，在阻碍我们对那个时代形成真实的认知。而能够把横跨欧亚大陆的蒙古势力彻底赶出中国的土地，能够圈定那么大片的疆域，设立自己的卫所系统，能够实控马六甲海峡，能够环球航行，拥有航母级的战列编队，然后你说这是一个弱鸡的、闭塞的、保守的、落后的时代？只要你还说他一句在当时落后，我都认为这个明史有问题。什么时候你复原它成为一个伟大的时代，什么时候再说明史恢复了。为什么这样？我们的敌人比我们了解我们自己。今天的阿美、今天的加拿大，一次又一次地说，今天中国的成长就是试图恢复明王朝的国际地位。他怎么不说我们要恢复清王朝的国际地位呢？哦，要是恢复了的话，他们就可以赶超我们了，对吧。嗯，咋回事呢？怎么那么巧呢？嗯嗯，仅仅是郑和七下西洋就可以给大明封神了，但是七下西洋后来被点评为劳民伤财、没有成就？怎么可能的呢？你觉得哪一个王朝花费那样的巨资出去下西洋航行那么远，然后亏得底儿掉、裤衩都没有了，然后还去下一次？一百个豆不嫌腥吗？吃多少亏都不嫌腻吗？怎么可能的呢？他一波又一波地下，他必然是赚的，必然取得了卓有成效的成就。但是为什么最后档案被销毁了呢？就算他的成就不好，就算他有灾难，就算他有负面的作用，你记下来，以史为鉴不好吗？你说他劳民伤财，你说他是错误，错误的答案正应该留下来呀，为什么要销毁呢？所以明代的历史扑朔迷离。什么时候明代的形象变成了光辉的、灿烂的、伟大的、拥有话语权的、世界性的帝国，什么时候明史算是恢复了。其他的一切选项我全都不支持。为什么？因为我有自己的客观的立场。我永远站在五千年、一万年的中国人的这一边，而不是按国籍说话。按国籍说话啊，非洲过来的球员，他也和我一个国籍的，但他跟我不是同胞。我不按国籍说话，我按我们的传承来说话。

嗯嗯，啥？十世之仇犹能报？嗯，虽百世可也。你要是按照这个公羊派的学说的话，儒家公羊派的大复仇理论当中，对于国仇虽百世可也，就是我们不计较时间，不在意时间，只在意会不会去复仇。时间这件事情，对于我们这样一个拥有记忆能力的群体来说，不重要。我们一定会等到那一天的。因为什么？子子孙孙无穷匮也。嗯，啥？百世得二千多年？那多正常点事啊，两千年怕啥的呢？您记得那个什么，嗯，朱棣，朱棣北伐的时候找的借口是啥？要报汉高祖的白登之围，要为汉高祖的白登之围复仇。就这个借口啊，就这个理由啊，就是为了汉高祖而复仇啊。中间隔了多长时间？这咋不行了呢？当然了，我知道他是借口，为啥？因为那个匈奴都不在了是吧，啊，匈奴都没了。没了也接着可以来啊。所以百世可也咋了？没问题啊。
