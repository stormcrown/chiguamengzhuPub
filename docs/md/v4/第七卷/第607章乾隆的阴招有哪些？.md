# 第607章 乾隆的阴招有哪些？

对于望门寡都疯狂地发放牌坊，明明是一个很小的小姑娘，只是因为婚约的原因，就不让人家嫁人，还鼓励她在家守节。为什么？因为这样一可以降低你的生育率，二可以降低结婚率，三降低生育率，四降低你的人口占比。之所以大家觉得这个事儿不明显，是因为汉人的底子太厚了，血条太饱满了，以至于用了各种各样的手法之后，我们依然成为当时第一个人口大国。

关于今天要讲的女性地位，在这个领域当中，其实清代的女性地位是非常非常堪忧的。除了我答应要讲、实际上最后还是没讲的缠脚这件事之外，还有很多其他方面的约束。今天大家所认为的很多腐朽的所谓习俗传统，其实大部分都是从清代的时候留下来的，之前没有那么严苛，直到清代的时候才变本加厉。就包括最典型的这个贞洁牌坊。很多人说宋代的时候女性地位下降，因为“饿死事小、失节事大”。实际上人家在讨论“饿死事小、失节事大”的寡妇再嫁问题时，谈的不是寡妇再嫁，而是义理之争。这是文人说话时矫情，喜欢借由生活场景去聊义理之争。当然了，这个事儿不重要，你先把它放开。无论怎么说，在宋明两朝，寡妇再嫁都是很正常的事情。这个事儿要从民俗上去理解。什么样的民俗呢？就是那些剧本、话本、民间流传的小故事。在这些内容当中，你会发现民间女性的生态并不是那种绝对的大门不出、二门不迈，也不是不准与别人说话。同样呢，里面也有抛头露面做生意的人，甚至也有生意做得不错的女性。同样的，宋明两代的女性是拥有继承权的——当然，只是在室女，在室女就是没有出嫁的。那为什么要排除掉出嫁的女性呢？因为出嫁的女性已经把嫁妆拿走了，嫁妆就是女性拿走的她那部分应该拥有的财产。那么到该继承的时候，还没有拿到继承权、还没有出嫁、没有拿到嫁妆的人，就会拥有一份单独的继承。这个呢，就是在室女继承权。所以女性的地位并不是在那个时候就咔咔咔地掉下来的。

包括刚才讲的贞洁牌坊这个东西，你说明代的时候其实就有，对吧？但是明代发了多少呢？数字好像是明代200多年一共发了，我印象当中应该是3万个左右，针对女性守贞的内容一共是3万个左右。那么清代发放了多少呢？嗯，乾隆年间，好像某一年就发了3万个。这性质完全不同。所以就是名字看上去一样，但实质上却根本不同的事情出现在明清两朝中间。你看上去它叫“承袭明制”，它承的是什么呢？它承的是这个制度的名字与框架，但是在具体的执行层面上面，却是完全背道而驰的。

嗯，压抑、打击、压抑女性有什么好处呢？还是有好处的。至少说清代那种贞洁牌坊式的压抑与打击，会造成一个情况，就是生育率下降。很多人觉得清代的时候出现了人口大爆炸，于是忽略了这个问题。为什么之前的历朝历代都非常鼓励寡妇再嫁？汉代的时候，甚至皇后都是寡妇再嫁的人。为什么之前的时代会鼓励寡妇再嫁？寡妇身上有什么特征？青年的寡妇，通常情况下她是生育了一两个子女的，这意味着她有良好的生育能力，而且生二胎、三胎、四胎以后的孩子会比较容易降低生育的风险。鼓励寡妇再嫁，其实是在提高社会生育率的一种操作手法。你以为那些官老爷们闲得没事干，就出来干涉别人家里面寡妇嫁不嫁人吗？这些官老爷特别喜欢推动寡妇再嫁，甚至有很多的地方官积极鼓励当地的寡妇在家，是为了促进人口增长，是这样一个根本的原因啊。

所以清代的时候，为什么那么疯狂地发放贞洁牌坊？为什么对于望门寡都疯狂地发放牌坊？明明是一个很小的小姑娘，只是因为婚约的原因，就不让人家嫁人，还鼓励别人在家里面。为什么？因为这个样子可以降低你的生育率，可以降低结婚率，降低生育率，降低你的人口占比。之所以大家觉得这个事儿不明显，是因为汉人的底子太厚了，血条太饱满了，以至于用了各种各样的手法之后，我们依然成为当时第一个人口大国。你是被成果遮蔽了眼睛，但是这个成果不是满清造成的，这个成果是我们数千上万年的祖先给我们留下来的宝贵财富。这不是满清的成就，他们是千方百计地打击、分化、压抑之后没摁住。而且呢，他是真的没摁住吗？也不好说。因为当时由于物种的大爆发，全世界范围内受到影响的地区都有人口爆炸。别人的地区哪怕农耕水平不如我们，勤俭节约不如我们，人口爆炸的程度也高达7到10倍。但是中国人人口增值速度只有四倍，区区四倍。你看到的那个庞然巨物，是建立在我们祖先留了1万年的老本儿的基础之上，所以在绝对数量上面很大，但是在相对的成长率上面，我们当时在清代的200多年里头，人口增长的效率已经是世界垫底级的水平了，比很多刚刚开始建设国家的城邦制的地区都还要落后。所以你说它有没有用呢？他为什么要禁止寡妇再嫁？为什么要疯狂地发放贞洁牌坊？你发一个望门寡的贞洁牌坊，就会让一个女性一辈子少生五六个，甚至可能七八个孩子。满清的皇帝是不愿意汉人变多的，你以为他只把你当成韭菜来割，韭菜越多越好？实际上满清的皇帝最恐惧的就是汉人多起来，因为多了就一定会造反。他们自己都天天提心吊胆的时候，你替他操这个心？
