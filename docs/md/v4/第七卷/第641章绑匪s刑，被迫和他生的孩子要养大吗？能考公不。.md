# 第641章 绑匪s刑，被迫和他生的孩子要养大吗？能考公不。

啊，我们的中华法系，其实在过去呢，一直是受到非常系统、非常坚定的批判的。为什么？因为里面有连坐制度，有诛九族、夷三族，还有各种酷刑和严刑峻法。它通常情况下是受到严重批判的。但是古人为什么设置这些东西？就是为了蹂躏他者吗？其实，它遵循的是我们自己内生的原则：当你因为错误的行为而受益的时候，你就已经成为共犯了。所以，为什么会有一些重罪、大罪会牵连家人？为什么要牵连家人？因为你犯这件事情的时候，让你的家人得利了。你获得的所有这些非法收入，都去养活你的家人了，他们得利了。不能在得利的时候，他说：“哈哈哈，我爹好能干！”等到需要付出代价的时候，他就突然变成无辜的那一个了。这对受害者、对整个社会都不公道，而且会形成一个恶劣的、后续的、长远的影响——就是“舍得一身剐，愿意付出最后的代价，让全家得利”。

嗯，前几天我在讲儒家治国方针的时候，讲到孔儒这个阶段——孔子之儒、先秦之儒。他们制定的治国方向，上面专门强调了一个东西：人。其实，所有的儒家追求的都是“人”。“人”是什么？“人”是你如何爱这个世间，如何让这个世间变得美好的一种方向，一种努力的方向。这个东西就叫做“人”。那么先秦时期，先秦之儒所推崇的“人”，讲究的是这个人，他的后果是什么？他是以目标、以结果为导向的一种思维方式，直接影响到中国2000年所有的治国方针和行为方式——就是不光看你眼前做的这件事情是什么，还要看这件事情在后续的时间当中引发什么样的结果，造成什么样的社会后果和社会影响力。

所以呢，在这个事上面，我突然之间想牵扯到一件事情。呃，其实这几年一直都有人提出这样的问题。例如说，每年都会有人专门着力去报道受拐骗的妇女，对吧？然后，这种情况下，很多妇女被解救回来的时候，已经有了孩子。然后在互联网上就会讨论说：这个孩子怎么办？一般情况下，它会告诉你说：“孩子是无辜的。”而在现代的这一套系统当中，它会告诉你孩子是无辜的。而这种总结——就是在这种恶性事件之后，当有人跟你说“孩子是无辜的”的时候，你有没有觉得心口有点发堵？反正我是一定会心口发堵的。我不知道在座的各位反应是什么样子。所以，如果你和我一样听到了这种话，听到了这种恶性事件产生了一个孩子，然后有人告诉你说“哎，这个孩子是无辜的”，然后你觉得不舒服，你觉得哪个地方不对劲儿——你来在公屏上面扣个“1”。这个就是我的感知。就是它客观上会造成人心里头感觉到不舒服，这是精神层面的影响。那么在物质世界上面，它长远的影响是什么？就是哪怕说我被逮着了，哪怕说我被发现了，做了这样的错事、这样的坏事，伤害了一个无辜的女子，哪怕最后你把我逮进去了，我起码还落下一孩子呢！而且这孩子是无辜的，拥有所有的人世间应该拥有的所有权益。甚至他可以去找那个女子要抚养费。你觉得这是一个什么样的社会导向呢？你觉得这在漫长的后效当中，体现出了一种什么样的效应？它是正向的吗？嗯，说什么“讨论小概率事件没意义”——这不是小概率事件的问题，这是规则的问题。不是事情的多少的问题，而是规则的问题。什么样的行为是合理的，什么样的判断是正确的，这是价值观的问题。是不是无论如何，那个孩子都是无辜的？在现社会当中，它默认的一种运行规则叫做“无论如何，孩子都是无辜的”。它是诞生于什么样的理论？那么我们的古典时期又是如何看待这种问题的？你对它的看法、你对它的观点是很重要的事情，因为这是社会风气，这是社会导向，这是价值观评判。你能说这个玩意儿对一个社会、对一个国家不要紧吗？一定要紧。

那么，当一个人用了错误的方式，繁育了一个不被祈求、不被期待的后代，然后他付出了一部分代价之后，那个后代成为了一个无辜的、纯洁的、纯白的人物，还可以向那个受害者去索要抚养费的时候，你觉得它在社会导向上面是正义的吗？所以我们古典时期是怎么操作这件事情的？有很多小说话本里面都有这样的故事：女子被山贼掳掠走了，掳走了之后被迫生养了孩子，然后她的家人终于得到了机会，从山寨当中把她解救回来。这种故事在我们的古典话本、传说当中都是有的，而且也不止一个。这种情况下，她会做出什么样的选择？很多时候，这个女子会把这些孩子如同匪徒一般同等对待、同等处置，女子孤身离开。如果有人问说：“难道你不爱你的孩子吗？”她会告诉你：“这孩子不是我想要的，是被迫的。这孩子是我受害的证明。”所以你看，我们在进行所有逻辑梳理的时候，我们今日定下来的这套规则，它的原理是什么？今日这套规则的原理是“所有的孩子天生都是纯洁无辜的”。那么这种原理，它基于一个什么样的精神内核而成立？大家有没有听过这样的一句话：“你的孩子不是你的孩子”——嗯，这是哪个作者的诗来着？你的孩子不是你的孩子。这种思维来自于哪儿呢？它来自于神学系统。在西方一神教的神学系统当中，每一个人都是神明赐予人间的，他是神明安排过来的一个人，而父母只是他来到人世间的一个通道。所以没有人能够拒绝这个孩子，而且这个孩子无论他的父母做了什么，也无论他诞生的原因是什么，总之他一定是无辜的，因为他是神明的孩子，是天地的孩子，是自然的孩子，唯独不是父母的孩子。于是，父母哪怕是被迫，甚至是受害、受欺骗而生下来的这个孩子，孩子依然是无辜的。它把孩子与父母进行了一次剥离：父母无论是怎样的，都与这个孩子没有绝对的关系。实际上它来源于神权的统治。神权为什么要这样统治？因为神权要求你只信奉那个神明，成为神明的羔羊。而你如果说非常依赖于你自己的家人、这种客观的血缘牵绊的话，就意味着你有可能会脱离神明的掌控。当你面临在神明与父母之间二选一的绝境时，你有可能选择父母，这是神权社会不能够容许的背叛。所以为什么在它的这套系统当中——你听着——“每一个孩子都是无辜的”，听起来这句话没错啊，听起来似乎没错，毕竟那个孩子什么都不知道。但是如果说没有那个之前的、错误的、邪恶的、犯罪的行为的话，这个孩子根本就不会存在，他原本就是犯罪的受益者。所以，在我们的传统思想当中，任何人不可以因为错误的行为而得利。一旦你因为罪恶的行为而得利，那么你要赎罪，要达成一个社会秩序上的公平。嗯，为什么有人觉得出生是得利？不然呢？啊，活着不是得利，死了是得利的话，你现在可以去买根绳，又不贵，何必呢？是不是？

就像我们古时候，之前几十年一直都有很多学者在批判我们古典时期的法制系统。我们的中华法系其实在过去呢，一直是受到非常系统、非常坚定的批判的。为什么？因为里面有连坐制度，有诛九族、有夷三族，然后呢，有各种酷刑、严刑峻法。它通常情况下是受到严重批判的。但是古人为什么设置这个东西？就是为了蹂躏他者吗？其实它遵循的是我们自己内生的原则：就是当你因为错误的行为而受益的时候，你就已经成为共犯了。所以为什么会有一些重罪、大罪会牵连家人？为什么要牵连家人？因为你犯这件事情的时候，让你的家人得利了。你获得的所有这些非法收入，都去养活你的家人了，他们得利了。不能在得利的时候，他说：“哈哈哈，我爹好能干！”等到需要付出代价的时候，他就突然变成无辜的那一个了。这对受害者、对整个社会都不公道，而且会形成一个恶劣的、后续的、长远的影响——就是“舍得一身剐，我愿意付出最后的代价，让全家得利”。这种情形非常非常容易产生，在现代社会也有。所以当年我们在云南那里扫毒的时候，就做了这件事情。在云南那边，整村子、整村子的就搞这个毒品，然后最后怎么治都治不根，治不断根，它们互相包庇。最后怎么解决的？就是用的连坐牵连法：只要逮着一个，你们家里头盖的那些房子，直接推倒。你不是靠得利去养活你全家吗？我让你所有的利润一概推平，一毛不剩之后，这个东西就断裂了。所以整体的治理原则——在治理原则上面，咱们还得向我们的祖先去考虑。我不是说要直接复刻以前的规则，规则要因时而变、因势而变，规则是一定要变的。但是规则背后的原理、制定这个规则的原因，你肯定要考虑到。因为中国的古人治理我们这样一个庞大的社会，已经治理了两两千多年——这是记载非常清晰的2000多年、3000年，之前还有一些记载含糊的。那么，无论你按照多么苛刻的标准，我们至少治理庞大的群体治理了3000年。那么在这种情况下，我们是有自己独特的思维方式，有自己独特的总结归纳，对于它的规律有自己的汇总、有自己的思路。所以这些思路、这些经验、这些教训，对于我们是非常有用的。

而现在呢，应该说，从清末开始，我们传统的这一套管理的秩序，就已经由于西方人的强力渗透而开始转向了。于是你就会发现，我们在所谓的走向现代化的过程当中，自觉与不自觉、主动与被迫，都进行了不同程度的西化。现代化是不能够等同于西化的，但是因为这一波的现代化是西方人所引导的，于是我们在进行现代化的过程当中，就不自觉的——或者说不管怎样——它一定会有一定的西化内容。而这个西化的内容就是我们需要去考虑：它对还是不对？它有没有合理性？要知道，不是说人家当年有坚船利炮，于是他就什么都对。他们当年发达的第一桶金是靠着抢劫呀！所以，他能够总结出来的经验和教训，对于我们来说不一定适用。我们整个的这个管理系统和它是完全不一致的思想方式。为什么在很多事情上面，我们在处置起来的时候感觉到有一定的违和感？就是因为我们现在使用的一些管理的方式，甚至思想的方式，实际上是西化的表现，但是我们误以为它是现代化，所以它是对的。西化和现代化完全不一样。西方人呢，他可能在某一个阶段上，在科学技术上面是走在前面的——我也不敢说是全部走在前面，但是在某一个阶段上，他科学技术上面是走在前面的。好，他走在前面的那个部分叫做现代化。但是，它传递出来的那些无论是宗教也好、文化也好、婚姻结构也好、社会制度也好，不保证是先进的，完全不保证。因为人类的行为和机械的行为是根本不一样的。人类社会制度应该如何建设这件事情，就没有办法做实验。它和你研究任何学科都不同：其他学科你可以做实验，一遍一遍地做实验，一遍一遍地排除里面的错误选项。但是在人类社会里面，你做不到。只要去操作，那就直接操作了。如果错了，那付出的代价永远都是巨大的，有一些错误甚至要有灭国之灾的。所以人类社会的这种大型实践，很多时候都是一次性的，必须一条过。这根本就不像你演戏的时候可以“咔咔咔”再来啊——这是第几幕？嗯，不管，在人类行为上面，它是一过性的，就是一个一次性消费。那么这个时候，谁最有话语权去说这套东西好呢？只有中国人有这个权利。因为只有中国人有上下5000年的文明史——指的是上5000年加下5000年，一共1万年的文明史。这个漫长的拥有记忆的文明史是什么？是我们这个群体在历史上面不断地进行人类社会行为试错的大型社会实践。而这种实践在所有的人那里都是一样的，很公平——你只能做一次。而我们不光做了，而且做了上万年，到今天都还记得。今天我们的人没变，文化没变，文字没变，语言也没变。这意味着什么？只能意味着一件事情：在社会管理方面，我们才是对的。那个，所以你去学什么？别人发达了，我要去学他们的先进经验——你怎么知道是先进经验呢？好，欢迎大家来到直播间。

嗯，啥？完全不认同古人不会科技的观点。科技当然不可能是凭空诞生的，神说有科技于是就有了科技——这种事情不存在啊。所有的科学技术都是在人类的实践当中一点一点积攒下来的。所以一个群体，如果他没有历史、没有记忆、没有清楚的严格，你说他拥有先进的文化技术？不可能。
