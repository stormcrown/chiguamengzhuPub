# 第665章 啥是日耳蛮赢学？赢赢赢，赢麻了！

好啦，好啦，好啦。大家晚上好啊，我们又开始上班啦。今天是……今天是礼拜五啊，大家记得了吗？今天是礼拜五，明天是礼拜六啊。礼拜六是什么日子？礼拜六是我的常规休息日啊。嗯，啥玩意儿啊？太平天……太平天国你都打不全是吧？嗯，太平天国破坏大，义和团主要是打洋人。你是看每段历史都只看一半是吗？就是太平天国你也只看一半，然后义和团也只看一半：义和团看前半段儿，太平天国看后半段儿。漂亮啊！那个叫“摸摸头”？

我也不知道你这种看法，是不是格外的会让你感到快乐啊？你这种搭配的方式非常的风趣。嗯，义和团主要是打洋人，集中在什么时候呢？啊，集中在他的前半段。后来呢，就被满清朝廷给忽悠着要开始“扶清灭洋”了啊，然后就被人家当成炮灰推出去了：羊也没灭成，氢也没伏成，反正就直接被人家给摁下去打了啊。所以你说义和团是在打洋人，那就是在说前半截。那太平天国呢？啊，太平天国的所谓的“破坏性”，来自于满清以及列强在他崛起之后的大规模围剿，也就是曾剃头的大杀四方。所以你说太平天国的破坏性来自于哪？来自于他崛起了之后，引发了满清朝廷与列强的勾结和打压。也就是在他后半段的时候，出现了来自朝廷鹰犬的大杀四方，这才形成了所谓的破坏性啊。这个破坏是谁做的呢？满清朝廷治下的走狗鹰犬。所以你看你，你所有的历史段落都只看半截儿的时候，对于他的评价就一定会跑偏啊。

嗯，啥？哎呦，来了个律师是吧？哦，还是看上去是实名律师。我瞅瞅，说什么啊……啊，有主播讲“没有主体民族”，然后说啊，说我是吧？啊，也没什么可怼的。我知道凡是说这话的人，我一听都知道是什么人，我都能想出来小表情。嘿嘿嘿。虽然我不看他的视频，也不看他的直播，但是呢，经常有人给我发他的截图啊，整的我对她老熟悉了。你说不看都能那么熟？为什么？因为她比我妈还关心我。我直播刚一下播，我这一天说了什么东西，人家就已经把小笔记弄出来了啊，就开始就开始喷了啊。我视频刚出去还没放几条呢，人家的那个开喷的东西就出来了。真的盯我盯得这么紧呢？除了我的几个铁粉之外啊，也就这样的那个仇家死对头，恨我恨得牙痒痒，时时刻刻的看着我。我有时候都怀疑他是不是爱我，爱的深沉——不然的话怎么忍受得了呢？像我就不爱他，所以我明知道这个人的存在，但是压根不去看他的视频，也不会去看他的直播，因为扛不住——一点开我就脑壳疼。我这个人性情中人啊，忍不下去这种犯恶心的事。但是人家就不一样，素质非常好啊，专业度非常强，日复一日、年复一年的一年盯着我检索关键词啊，一个字一个字的抠。你说我一个好端端的带货主播，他抠我的那个演讲稿，抠我的这个直播切片啊，抠到什么程度呢？恨不得拿我当新闻联播来进行这个赛博政审——就直接整到了这种地步啊。这真的是我铁粉都没有的爱——他爱我真爱啊！这是真爱啊！那啥，消消气，我也没生气啊，这事儿有什么好生气的啊，不值不值当生气，不值当生气。呵呵呵呵呵。就是属于那种什么赛博法庭啊，在线审判啊，我已经被审判了不知道多少回了，逐字审判啊，我自个儿都不知道有这些项目。你说直播的时候，直播的时候谁还念逐字稿啊？我视频都没有逐字稿啊，人家直播能够一个字一个字的给你抠。这种深情款款让我十分的感动啊，感动的我能怎么办呢？无以为报，想起来的时候，就把他最肉疼的东西再翻腾出来喷一遍，我就开心了啊，我应该就可以满足他的需求了。怎么喷呢？基本上我只要喷清，它就老难受了。我只要说清的那个卑劣啊、黑暗、腐朽，我只要说清的罪恶，他第二天早上头发都能竖起来。哎，我一想，哎，真情——这就是真情。

嗯嗯，什么？嗯嗯，能讲一下赢学吗？呵呵呵呵呵呵呵，嗯。赢学？赢学？好好好好好。嗯。为什么要赢？大家觉得为什么要赢啊？有没有人有感觉？一个国家为什么要天天赢？赢赢赢麻了？为什么非得赢不可？哎，对，你看我们小伙伴的，嗯，我们小破站上的伙伴啊，非常的聪慧啊。之所以有一些国家呢，必须发动这种赢学，主要的原因是这个国家不是一个正常国家——这是一个根儿上的问题啊。就是这个国家从立国的那一刻，它就不正常。前几十年的时候，很多的公知其实一直在吹嘘的东西，就是说，嗯，阿美那边呢，是一个人类设计的巅峰啊。我记得早些年的时候，还有一些专门研究法律的一些专业人士，就分析人家的那一套规则——底层规则建设分析完了之后，总结出来的东西，就是说阿美的那一套设计太完美了。为啥高屋建瓴？那就是直接拉通了所有基于人性的这个上层建筑。前几年的时候，很多的公知就这样狂吹不已，他们特别热爱那一套秩序的建设。但是阿美今天只能通过赢学来搞他自己的国内建设，也是因为这个原因——就是因为它不是一个正常国家。

正常国家是什么样子？很多人觉得国家啊，就是从现代这个战后的这套体系出来了之后，从雅尔塔体系出来了之后啊，人们对于国家的理解常常是局限于某一种疆域、某一种国界线的。现代社会的普通的人对于国家的理解，常常指的是某一个边界线圈定的这块地方。这种直觉会影响大家的本能判断啊，因为国家不是这个意思。嗯，就像前几天的时候，是谁问的那个什么来着？说……嗯，推恩令啊，说推恩令说是千古第一阳谋，但是他说那不对啊：你说推恩令就把下面的地块分的越来越散了，这是不是收税管理就越来越麻烦了？不应该是大地块的才会好管吗？是吧？你对国家管理的时候，你把东西拆的那么散，你怎么管的呢？所以实际上，这就是对于地理理解上面的一个误区啊。当你把一个国家理解为某一条边界线里面的这块地的时候，你就容易产生这种想法。因为国家管的不是这个国家，管的不是地——国家管的是人，这是最根本的啊。就是什么是正常的国家？正常的国家是一群人的组织体系，重点在于人。所以我们中国会喊“人民万岁”，因为人，尤其是占比最大的普罗大众，才是这个国家的根基，是这个国家的根本属性。所以这一群人的属性——这一群人是谁？这一群人要什么？这一群人想什么？就是最核心的东西。正常的国家是什么？正常的国家就是一群人的集合体。但是阿美不是一个正常国家，他从一开始的时候就是一群打家劫舍的匪帮联合在一起做的一个山寨。他一开始的时候攒起来的一个山寨，后来呢，又通过高屋建瓴的这种制度建设——伟大的制度，制度创想——做成了一个什么玩意儿呢？做成了一个伟大的企业。这个事早30年的时候是被狂吹的，认为这个事是正确的，是人类发展的方向啊。就是它把它构建成了一个帮会型的企业。它作为一个帮会型的企业，就会产生一个根本性的问题：你一个企业里面的这个人，是为了什么而来的呢？为了赚工资而来的，为了股权预期而来的——你只能因为这个原因去上班吗？所以阿美那个地方从一开始的时候就是这样集结起来的，包括他的原始股东，最开始加盟的人也是为了结伙分赃。当大家都是出于这样的一个逐利的梦想啊，为了淘金逐利的这种梦想而集结在一起的时候，你所有的操作方式都会走向这种逐利、要好处。没好处的话我就跟你翻脸，没有好处的话我就跑路了——就是这样的一种集结方式。

当他是靠好处而集结起来的时候，那么这群人就只能拥有上升期，不可以拥有下降期。因为上升期的时候，我幻想着后面我还有股权可分、可以预期啊——有预期、有收入。但是当这个幻想被打破了的时候，发现他居然不够赢、不够上升，好像快完了——树倒猢狲散。这本质上面是因为这个国家它就不正常，它不是一个真正的国家，他不是同一群人的组合体，这是他非常非常本源的问题。而且这个毛病改不了，没得改。他中间曾经试图改变过——就是他曾经有一段时期啊，试图把这个国家建设成一个白人国家。那个时候你把这个东西叫做西方中心主义和白人至上主义和种族主义，对吧？他曾经有一段时间非常排挤有色人种，甚至驱逐有色人种——干什么呢？啊，我们把它叫做种族主义——但是他那个时期在做的这件事情是做什么？他在提纯他自己的公民，他试图让自己的公民归为同类，因为同类的话就更好组织起来。所以那个阶段反而是他试图建设正常国家的一个步骤。但是显然失败了，因为啥？因为参与的人他都是为了干买卖才来的，你跟我说提纯你就提纯？我凭什么跟你提纯？我跟你又不是一家人。尽管大家长得都是白的，但是我是德裔，你是英裔，他是法裔，那个还是荷裔——咱们这几个啊，祖上有仇，习惯还不太一样啊。所以最后就是硬是没拼起来。然后当他拼不起来一个正常国家的时候怎么办？他就得把这个公司做大做强，而公司做大做强就要不断的去招募新员工，然后它里面的成分就变得越来越复杂。成分越来越复杂，它就只能靠着这种分红预期、上升预期来让大家做贡献。一旦失去了这种预期，大家立刻就散了啊。所以今天他为什么要讲究赢学？时时刻刻的告诉你：我们很好，我们很赢，我们又要伟大了，我们又要崛起了——他必须喊这个的东西啊，就是因为这个一旦不喊他就散了。

而正常的国家是什么？你像中国啊，中国有没有跌底的时候？我们从1644年之后，每况愈下，不断跌底啊，触底不知道触到啥时候才能够反弹的那种持续性的下跌。但是为什么不散？因为他本来是一家人，叫“500年前是一家”，你知道吗？我们这里的这个是土著人民的紧密连结，以至于呢，唉，在遇到大灾大难之时，我们本能的还是会聚在一起抱团取暖。就是哪怕按照自然分类，你摒弃掉一切文化的元素，就像野生动物一样的，它还是会聚在一起。这个东西那就是密不可分的一个结构。当它密不可分的时候，这样的群体他是一群人的组合，而不是某一条边界线。作为一群人的组合，那么在遇到跌底的时候怎么办？熬——摇苟一苟、熬过去，争出头，是吧？那么这个时候如何宣发呢？我们知道我们古时候的战争领域当中，还有这么一句话叫做“哀兵必胜”。就是一个国家走在下坡路的时候，走在下行区间的时候，对于那些真正牢固的、真正密不可分的、真正的国家，他会有一种宣发的口径：就是我们现在很苦，我们要共克时艰。这个就是缔造全民性的哀兵必胜——大家都觉得这个时候必须要努力，必须要克勤克俭，必须要付出一切代价，让我们这个整体重回巅峰啊——这就是正常国家的一种反应。不光是中国有，其他的国家只要他是正常的，它都有啊。包括原始部落也有这种能力——就是哪怕他没有形成国家，只要他确确实实知道谁和谁是自己人，在遇到外来侵犯的时候，他就会有一种向死而生的这种反抗的欲望。这个东西就是正常国家才会拥有的东西。那么它就不需要100%的赢啊，他在输的时候可以用“我们现在败了，我们得赢”、“我们现在不行了，我们得撑住”这种理由来号召大家团结起来。但是像那种企业，像那种公司，你只有赢才能够团结起来。

嗯，然后那个就像……嗯，很多人这几天啊，天天看那个阿美的赢学之后，老觉得川总的脑子可能不太好，但实际上的话，他的选择——你说他很精明倒不至于——但是他的选择是正常的啊。第一呢，他现在确实需要赢赢赢，这个对他是很根本性的问题。第二呢，你看他当年竞选的时候，他的基本盘是谁？他的基本盘是红脖子啊。你觉得在阿美那个地方，他为什么要选红脖子当基本盘？红脖子现在的占比也没有多高，而且众所周知的，他们文化教育水平不高，脾气也不好，素质也不怎么样啊。而且呢，有很多人受到快乐教育的那种奴役之后啊，确实呢，嗯，水平上面颇有一点不济。红脖子有这样的一种特征之后，为什么川总当年还选择了他们去当自己的基本盘？因为在一个这样的公司型、企业型的国家走下坡路的时候，他没有办法，他必须选择一个能够跟他密不可分、能够在这个国家崩溃的时候还坚定地站在这片土地上的人，这些才能够成为基本盘。而阿美那块地方也就剩下红脖子可以担当这样的岗位了。复杂到什么程度呢？那么这个复杂的这个人群当中，哪些人能够跟着一起苟到最后呢？啊，你就说那个土著能不能？他对这个土地一定是有依赖感的，但是对于当地的这个土著部落而言的话，所有的其他的人都是外来的殖民者，他是没有办法跟这个政权站到一起去的，因为他想平等地创死这一切。所以土著既人少又不可拉拢啊——你把他们弄起来了之后，他就平等的创死你们所有，因为人家才是真土著啊。然后呢，第二梯队就是他所选定的这个红脖子的群体。红脖子呢，因为来的时间比较久，已经形成了一定的乡土观，他已经觉得自己跟这片土地呢有比较紧密的连接了——相对来说啊，他对自己的祖先之地遗忘的比较多，产生了一定的归属感。而后面的其他的群体基本上都还记得家乡在哪。哪怕是说老早就被贩卖过去的黑奴的后代，时代、时间虽然久远，但是他一照镜子立刻就知道他的祖籍在非洲。这就是之前那个被警察给跪地上的那个叫什么来着啊？什么……那那那人啊，跪杀的那位——当年他搞起的这个“黑命贵”运动，是不是迅速的，非洲那边是响应了，光速响应。就当年这点啊，就那弗洛伊德，就他那件事情出来了之后，你说跟非洲有毛线关系啊？一点关系都没有，这完全是阿美内政，根本性的就是一个阿美内政，跟非洲一毛钱关系都没有。但是整个非洲一起联动起来了。所以这就意味着，阿美土地上面的黑人，无论他在这里生活了多久，他一照镜子就知道自己的祖先生活在哪，而且呢，非洲的人也承认“你的祖先肯定在咱这儿”，他们之间是会形成某一种精神上的连接的。所以这种情况啊，有点啥事儿的时候，牵一发动全身，甚至牵一发动两周，性质就会不一样。那么其他的人群，无论是拉美过去的还是亚洲过去的啊，或者是包括现在量特别大的印度过去的人，这些那纯粹是过去打工的。打工的人公司倒了之后，直接就撒丫子跑路，那跑的一定一个比一个快。所以你看，就这么族群类比一比较起来之后，你说川总的选择是错的吗？不对，他的选择是正确的。他想要选择一个与他同生共死的群体，这个群体在阿美的土地上面，就剩下红脖子这一拨人了。所以不管他们的素质啥样，也不管他们有没有那个水平，也不管他们的心智是不是够坚定——没得选，他就得选他。所以他们在搞这个MAGA的时候啊，“伟大”啊——要搞伟大的时候他选谁？他选红脖子跟他一起干。然后面临到阿美的这种下降趋势的时候，拉拢着这一票人一起喊赢赢赢、又赢了、赢麻了。所以你说他，他的选择没有那么好，也没有那么精妙，但是是有合理性的啊。
