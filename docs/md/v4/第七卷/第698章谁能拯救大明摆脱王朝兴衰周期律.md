# 第698章 谁能拯救大明摆脱王朝兴衰周期律 #经济危机

当然，以下是根据你的口述内容整理后的易读段落，同时修正了部分词汇错误，未删减内容：

---

**什么大顺能救明王朝吗？**  
救不了。他自身有巨大的缺陷，救不了。  
**那谁能救大明？**  
没有谁能救大明。也许朱元璋重来一次，可以？但没有人能直接冲出来救大明，因为那个时候大明确确实实千疮百孔。哪怕是那些非常有远见卓识的人，也一样觉得大势已去。大家都觉得大势已去。其实没有人为明王朝而惋惜。这个王朝走到风雨飘摇的时刻，是由于它自己的原因，所以也不需要为它惋惜。真正有问题的，是最后获利的那个人——是最恶劣的那个选项。没有谁需要为明王朝而惋惜。需要惋惜的是：为什么明王朝灭亡之后，来的却是最糟糕的选项。

其实，认真说起来，我们在历史进程当中一直都注意到一件事情：我们有王朝周期论。之前叫“历史周期律”，就是王朝周期。这个周期大约多长时间呢？大约在300年左右。那么是从什么时候开始的？基本上从周就开始了。虽然我们常说大周800年，但真正由周天子控制的时间段，也就是300年上下——就是西周那前面一截儿。前面的300年到了后面其实就已经崩盘了，开始洗牌了。所以实际上从周开始，我们的王朝能够撑得最久的，也无非就是300年，就是这么个长度。

最后我们说这是一个周期律。那么这个周期律取决于什么？是不是你多出几个明君、多出几个好的天子就可以解决？实际上并不是这样。它不是出昏君的问题——虽然一般到那时候你会发现有昏君，但即使出现的是明君，也没有什么用。因为这个历史周期造成问题的，是一种财富的两极分化，财富的完全不公正的分配，直接造成了这个问题。它是一种经济系统已经完全拧巴、已经完全固结之后的产物。如果置换成今天更时髦的语境，可以叫它“经济危机”。放到今天，放到我们很时髦的语境当中，它就是经济危机。但由于我们有很好的管理能力，而且古典时期整体的上升速度、发展速度比较缓慢，所有人的进程都比较缓慢——就是“车很慢，马很慢，爱一个人一辈子，但可以纳很多妾”这个梗是吧？古典时期我们确实速度要慢一些，所有东西的发展进程都慢一些，所以它的周期也会拉得比较长。但实际上都是一种经济分配的问题：当你的生产受到市场之手的掌控，然后生产越来越多，最后越来越集中，财富越来越集中，在终端的分配上面就会越来越出问题，一直出现到这种极度两极分化，底层贫者无立锥之地。这其实和经济危机的表现形式是高度相似的，原理上也大差不差。它都是经济系统崩溃带来的一个后果。

那么如何摆脱这个？其实历史上也有先例，比如汉——两汉中间是有一次变革，这个变革看上去更像是一次全面的改朝换代。为什么这个样子能够达成问题的解决，能够给它续命？因为在这种大规模的、彻底的变动过程当中，就对旧有的经济系统进行了一次推倒重来，我们重新分配了一次——原本的那些贵族被揍下去了，他们身上掉的金币被后来的人捡了之后重新分了一道。所以大家一定会注意到，我们立国之初的时候是进行了土地的国有化。我也一直强调：土地的完全国有化，是我们在试图跳出历史周期律当中非常重要的、根本性的一局棋。但是你回看我们的历史进程就会发现，在每一个王朝的初期，其实皇帝、君主都曾经试图用这样的方式进行土地的分配。所有的王朝一开始的时候都曾经尝试过土地的再分配——也就是说，所有人其实都默认清楚这一点：我们每一次的更新换代、改朝换代，就是一次财产分配行为。所以当你已经积累到了两三百年的这个节点上之后，面临到的局面就是那种极度的两极分化。要想改变它，就需要你亲自出来造一个反。只有造反能够解决这个问题，因为它需要的是对旧有的利益阶层的彻底推倒。除了造反，你能用什么理由去推翻利益既得群体？你用什么方式能够把他们革除掉？割不掉嘛。所以就会出现这种王朝周期看上去像循环的东西。

但是呢，西方人就找了一个借口，说“你这个都是循环，所以你没有历史，你没有进步”。这个是胡说八道的。因为你看，说西方人他们没有历史周期律吗？不知道。他们记载的历史、他们可信的历史太短了，不知道有没有。但是他们有经济危机，而且这个经济危机的周期很短暂，几十年就可以来一波——这个是清楚的。但是在那之前，他们是不是也有这种周期律？不知道，不清楚。因为他们的历史不清晰，他们的记载不清晰。而且他们是一个领主制的，没有非常完善的国家体系。没有国家体系，你说什么历史周期律呢？你就没有资格去说你活多久啊。总不是这片地属于这个领主，然后他今天结婚了就归到这边，明天离婚了又归到那边——这种像跟着主子跑的财产一样的东西，你不能说它是一个国家。虽然也把它叫做“country”，但是不能够真的算它是一个国。这种私有财产式的、领主制的管理方式，没有这种周期律，很合理。因为家里面就这一个主子，你有什么好分配的吗？

但是当我们作为一个庞大的帝国的时候，就一定存在这种经济系统的问题。所以我们每隔几百年就会推倒重来一次。这个推倒重来并不是一个文化的死循环，而是经济的再分配。我们的文化是一直在发展的，科技也是一直在进阶的，所有的东西都在不断地进步、不断地发展——除了清，突然之间就给打断了，是不是？除了清，其他的所有时间，我们都是一直持续不断地进步的。有的时候快一些，有的时候慢一些，有的时候遇到了战乱有点停滞，但是无论如何都是一直在向前走的。所有的这些王朝的兴兴败败，它都是财富的分配、再分配的问题。这种分配，无论怎么分配，肉烂了还在锅里头啊。所以真正对我们造成伤筋动骨式破坏的，就是一个清。因为啥？我不知道他把那些自己抢下来的脏银挪到哪去了。因为别人家嘛，砸烂了，烂肉烂在锅里；它这个砸烂了之后，肉都烂在全世界其他地方了，跟我们没关系。它烂完了之后，我们一穷二白，啥也没有。

嗯，西方还是有点实力的，不然丝绸之路和谁贸易的？我们知道，我们的产品是卖给他们的。我们不光卖给西方人，我们的丝绸之路上的这些瓷器也好、茶叶也好、丝绸铁器，不光卖给欧洲人，我们还卖给了非洲人。你是不是也觉得非洲人其实很厉害的？不然的话，我们海上丝绸之路卖给谁呢？有一说一，正经话，我们的海外贸易当中是包括了非洲的——非洲的土著部落，我们曾经跟他们做生意。他们那里现在还有我们的瓷器呢，现在都残存着。我们明代贸易过去的瓷器，是不是老发达了？跟你的欧洲老家一样发达，对吧？没错啊，凡是跟我们打过贸易关系的地方，全都老发达了，很厉害。有实力，你可以去试一试，体验一下。嗯，我们要的贵金属和发不发达没关系。本来就是这样的呀。因为当年在这个国际贸易当中，又没有那个什么绿票子，没有dollar需要你去争取，说你必须得有外汇的国家你才能贸易，不然你那个dollar从哪转是吧？但是那个时代不存在这个问题。我们有我们要的东西：要原材料，要宝石、要香料、要珍稀动植物，然后也要一些珍贵的矿藏。这些东西对于当地的土著来说没什么价值，但是我们的这些精细的工业产品，对他们来说是一个极具生产力的、非常先进的东西，他们根本弄不出来。所以我们的这种贸易，为什么不需要对方很发达、很先进？因为我们跟他交换他自己的原始资料就可以了。我们跟土著部落也可以做买卖，没有任何问题。我们拿我们家的瓷盘子去换他们家的长颈鹿，这种生意是可以做的。我们拿我们家的丝绸去跟他换树胶，这种买卖是可以干的。这个不需要对方有多么发达的生产力和工业水平，他可以什么都没有，他可以是部落民。只要他有想要的东西，我们就可以满足他。所以我们的那个海贸所收获的东西，和你所想象的对方也有先进的生产力，完全不同。

7. “啥啥啥” → 删除多余语气词，合并为连贯句子。
