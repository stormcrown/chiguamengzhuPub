# 第679章 撒哭拉拍了那么多反战片，为啥我们觉得不诚恳

以下是对原文的整理版本，修正了错误词汇，并调整为易读段落，未删减任何内容：

他这个反战思想是咋反战的呢？就是选他们侵略者队伍当中的女子，开始说：“哎呀，我思念家乡了，家乡多么美好呀，我想念家里面的什么什么啊，想念那个鱼啊，我想念那个花，我想念那个房子，我想念那个老太太，不让我摘枇杷是吧？”他们就喜欢讲这些细细碎碎的故事，然后说这个叫做反战片、反战。然后你说你出门侵略这事儿太辛苦了，这玩意儿叫反战吗？反战应该是什么？反战应该让你意识到你做的事情禽兽不如。但是他所有的影片里面传递这个信息了吗？没有，它传递的就是打仗太辛苦了呀，侵略也是那么累的呀，抢劫其实也没有那么快乐吧。这玩意儿叫反战？他跟受害者共情了吗？他想过当年干了什么丧尽天良的事情了吗？这些东西他统统都没有反思。他就反思了一件事情：原来出门抢劫也还是蛮疼的嘞。他就把这个东西定义为反战。

现在很多为他们的侵华洗白的，对呀，不光洗白，一边洗白，一边把他们的习俗、他们的文化灌溉到我们这里来啊，直接一个大浪潮就冲过来。甚至呢，给这些真正爱国的人扣上反华的帽子，你干嘛呢？因为，因为你抗日了啊，你这好好的过日子，你是不是要破坏我们市场经济是吧？你是不是要逼得日资撤退？人家的给我们拿了什么什么好处？那个这几十年来，好多人用这样的方式洗地。一开始的时候是夸赞日本的人啊，多么的彬彬有礼，多么的认真学习，积极向上，甚至夸他们包容。我好些年前看过这么一个公知文章，我给大家分享一下：它里面赞美日本人民的包容是咋赞美的？这个是当年公知的一篇文章。他说啊，中国的男人特别的小心眼儿，他非得让自己的孩子是自己亲生的，人家日本就不这样，日本的男人非常的开放、包容，对养子和对亲生孩子没有区别。这就是当年他们吹过的牛啊。这个时代，这些公知们用这些细碎的生活当中的碎片来洗白日本身上的罪孽，让你觉得他呢，是一个素质很高的一个群体，说他的人呢，也很善良，非常的温柔，是不是？之前也有很多的中国男性，曾经畅想过娶一个日本媳妇儿。为啥？还不是那个时候吹的，说那个日本的媳妇儿是最好的啊，回家跪地服务，给你摆鞋子，有没有啊？照顾孩子，照顾老人，照顾家，把那个饭做的精致一顿啊，一顿怎么安排，下一顿又怎么安排，明明白白的，一回来的时候还要说一下，您辛苦啦，听过吧？都听过这种梗吧。这是干什么？这些都是对于这些群体的洗白。当你经常看他个体上面的这些内容的时候，你总是看他个体上面的这些细枝末节的表现的时候，会让你带来一种心灵上的共情，然后这种共情呢，他就循序渐进地推进到了下一个步骤，就开始传播他们的反战思想啊。

在座的有多少看过他们的反战电影的？当年他们的导演啊，他们的制片人非常专业，他们的影视专业啊，相当相当的厉害。厉害到什么程度？无论是漫画还是相关的影视产品里面，都一定会塞上他们自己的反战思想啊。他这个反战思想是咋反战的呢？就是选他们侵略者队伍当中的女子啊，开始说：“哎呀，我思念家乡了，家乡多么美好呀，我想念家里面的什么什么啊，想念那个鱼啊，我想念那个花儿，我想念那个房子，我想念那个老太太，不让我摘枇杷是吧？”他们就喜欢讲这些细细碎碎的故事，然后他说这个叫做反战片，反战。然后你说你出门侵略这事儿太辛苦了，这玩意儿叫反战吗？反战应该是什么？反战应该让你意识到你做的事情禽兽不如。但是他所有的影片里面传递这个信息了吗？没有，它传递的就是打仗太辛苦了呀，侵略也是那么累的呀，抢劫其实也没有那么快乐吧。这玩意儿叫反战？他跟受害者共情了吗？他想过当年干了什么丧尽天良的事情了吗？这些东西他统统都没有反思。他就反思了一件事情：原来出门抢劫也还是蛮疼的嘞。他就把这个东西定义为反战。但是因为影片做的确实好，演员也确实漂亮，影视的节奏也确实把握的好，于是有越来越多的人在这些作品当中沉迷，然后呢，就跟他共情了，甚至觉得这些人也挺辛苦的。同时呢，他们会找出一个理由说，我们的革命者也说过，他们的军国主义者和他们的人民不是一回事，对侵略者和他们的人民不是一回事。但是他家有几个人民？这事咱们得理清楚吧，咱们得分清楚到底谁是军国主义者，谁是人民。你不区别清楚，你看见他啊，看见一个个体，你永远都会说这个个体他就是一个普通人，这是自然而然的嘛，是不是？但是这个个体真的就是一个普通人吗？因为她是一个女子，因为她是一个老人，因为她是个孩子，她就是普通人吗？没有这种必然属性的。要知道当年啊，当年当年的这个日本岛上面的宣发当中，发了一个公告，说他们侵略南京成功，然后万人空巷，举国同欢，就庆贺这件事情，男女老少都出来啊。请问他们是人民吗？他们有朴素的正义观吗？他们跟受害者之间有任何的共情吗？都没有。所以你冲上来就说谁谁谁是人民，这件事情对受害者不公道啊。
