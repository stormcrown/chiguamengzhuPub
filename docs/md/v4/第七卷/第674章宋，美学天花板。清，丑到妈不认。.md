# 第674章 宋，美学天花板。清，丑到妈不认。 #殖民

满清的这200多年间，所有的民间被规定下来必须遵照的审美，全部都变得一塌糊涂、丑陋不堪。服装、发型、鞋子、建筑里面的雕花、各种各样的家具、瓷器，没有不难看的。这个就是丑，直接带给人卑弱感。但是这种操作方式不是自然产生的，而是在清代刻意塑造出来的。我们都看过乾隆皇帝的画像，他的画像当中有很多都是汉服cosplay。他知道什么好看，也知道怎么打扮漂亮，但是他不，他要求所有人都扮成很丑的样子。甚至于满清皇帝他自己，你说剃头啊，他自己都不剃，他的那个头发比其他的人都多，他是不秃的。满清的皇帝们不秃啊，他们留寸头，确实也有辫儿，但是留寸头不是一片光溜。

所以，当一个国家的整体审美变得非常丑陋的时候，你没有办法站在别人面前的那一刻还感到自己有自信。你没有办法觉得自己美的时候，就不可能觉得自己比别人高级。好，欢迎大家来到直播间啊。我觉得清代对我们还有一个非常重要的破坏，就是他破坏了审美这个东西，我觉得影响力其实非常非常大。清代对于审美系统的破坏是极为严重的，严重到了什么程度？说到这儿的时候，咱们很需要做一件事情，就是审美对于一个国家、对于一个民族来讲，到底有什么价值。审美看起来是一个自然而然的事情，有的时候觉得他毕竟是依附在硬实力之外的东西，是吧？他是一个依附者，他并不是真正的硬实力，你不能靠漂亮战败你的敌军。所以很多时候，尤其是受到工业党——就是万事功业最大的这些人的引导的人——常常会忽略审美的价值。其实美本身是有力量的，你自己就去揣摩一下你自己的人生。假设说，年轻时候正当盛年的那个林青霞站到了你们的面前，说要跟你借1万块钱，你借不借啊？就是林青霞年轻版的啊，年轻版的林青霞刚刚出道的那个，站到你面前跟你笑，求你借他1万块，啥借？但是你那时候没有美丽，美丽是有力量的。我真正的体验到这个东西啊。你知道的，我年轻的时候颜值也是有那么一丢丢的，如果站在一些普普通通的人群当中，甚至我有点盲目自信，会时不时的看周围的反光的东西，想看看我自己。就是这样的一个人啊。我曾经在一个会场里面见到了一个非常漂亮的姑娘，漂亮到什么程度？就是他仅仅只是在那个地方站了一会儿，笑了一下，我就觉得吧，我为什么要跟他待在一个屋子里面，我都不敢大声说话。他一声都没吭，而且我不觉得她非常的聪明，但是她太漂亮了。就这样啊。美丽是一种力量，甚至在很多时候可以形成威慑力。所以当一个群体拥有了生产力之后，很快要去做的一件事情，就像你荷包里面有了仨瓜俩枣之后，很快要做的一件事情就是“人靠衣装马靠鞍”，你就要更新自己的服装，干嘛？提升自己的外观价值。群体上面也是会这样的，生产力提升了之后，我们就会做一件事情——改变自己的外观形象，这个东西就是审美的意义。

所以中国人事实上是花了好几千年的时间去研究什么样是美的，哪一种美最大气、最优雅、最典范，最能够影响他人、最有魅力，同时也最能够震慑他人。这样子的美是我们过去数千年来孜孜以求的，我们历朝历代的所有的先贤都执着于在这个地方上面去努力。他不是只吃饱了撑的、闲的没事干，而是这个东西它有价值。之前很多年大家去嘲讽宋，说宋软弱可欺、无能丢失半壁江山，偏安一隅没出息。但是就是在宋的那个时期，宋所缔造出来的所有的艺术品、所有的美的这种定义，是可以辐射到整个东亚文化圈，包括他的敌国的。他为什么会去交岁币、花钱买和平？他是怎么赚到的？一大部分是靠海贸，是靠生产力的这种代差式的贸易；还有一大部分就是靠他的敌人。你看他前脚交了一大笔岁币出去了，后脚他的对家就要背地里朝他私下购买那些奢侈品。他交10万的岁币出去，能够赚1000万回来，你说他会不会花钱买和平？美丽是一种力量，是会产生强大影响力的东西。但是清代破坏了这套系统。

这几年呢，一直都存在着大量的“以倭代华”的纹样，就是把脚盆那边的传统纹样直接替换进来，作为我们的纹样。我们很多的商用纹样库里面啊，都会直接把脚盆的元素标称成中国元素，然后就有很多的设计师直接就那样选用了。因此我们现在在商业元素当中，确实有大量的“倭风”就是这样被灌输进来的。但是这两者之间其实是有着明确的分野的，是有非常清晰的审美代差的。它是源自于不同的人群，在不同的生活维度之下，产生的不同的审美的情趣。脚盆由于他生活在岛国，而且历史上一直非常的贫弱，资源非常的匮乏，人的体格也非常的弱小，所以整体上就存在有一种向死的倾向。他会欣赏这种灰暗的、悲催的、没有希望的东西，并且把它视为瞬间之美。同时呢，他在进行纹样设计的时候，也会非常强调那种固定的、刻板的、可重复的、不断不断重复的符号和意象，这是他在审美的连贯性上面非常在乎的东西。所以脚盆纹样上面就会出现很多非常僵化的纹样。但是中国人的审美系统当中是非常回避这个东西的，我们认为人世间的所有的东西都是活动的，水是活动的，风也是活动的，树是在生长的，鸟是在不停的盘旋和飞行的。所以我们的所有的纹样都要求他拥有活跃的生命力，有流淌的线条感。这些审美也是我们本身的文化内核，甚至是精神价值当中不可或缺的东西。

所以为什么我说清代对于我们的审美系统产生了极为巨大的破坏？它在剃发易服之后，尤其是在乾隆皇帝很刻意地制造了满清的满洲式样之后，事实上它破坏了原有对于服装格局的审视方法，并且他专门设计的东西就是一个对抗式的产品。他这种完全不对称的、使用蜈蚣扣进行割裂式缝补的这种服装拼接的方式，其实直接就让一个东西120%的变丑，他的目标就是让你变得难看。包括说在之前的任何时代，我们会注意到服装上面有一个配饰，就是一个腰带。腰带是干什么？腰带是我们服装上面非常重要的一条分割线。分割线放在了黄金比例上的时候，你的身材的比例就会变好，整个的视觉的线条就会发生改变。这个东西就是美学，这个美是符合视数学和几何的相对的原理的，这是科学之美，放到哪里都是这个样子，什么时候他都可以用。但是到了清代，你去看晚清时候的照片，他的那个袍子从上通到下，是一个圆锥体的造型，中间没有腰带，没有分割线。所以不管你的身材是什么样子，穿上去那就是一个墩儿，就是一个桶，不可能存在任何想象的空间。这是什么？这是一种很刻意的对于审美的破坏。所以当一个国家的整体审美都变得非常丑陋的时候，你没有办法站在别人面前的那一刻还感到自己有自信；你没有办法觉得自己美的时候，就不可能觉得自己比别人高级。

所以你就会注意到过去的几十年，西方人的很多的电影、电视剧里面，他一定会让有中国面孔或者有东亚面孔的这个角色看起来不好看。即便他想描述这个人是个万人迷，那么他也会用比较色情的方式来描述它，例如性感、例如浪荡、例如东方风情。他用这种方式来进行魅力的塑造，但是这些东西跟美并没有直接的关系。也就是在他的影视作品当中，会刻意的回避东亚人种身上的美丽，这是一种故意的行为。但是呢，他们会哄抬其他人种身上的美，即便长得并不是很好看，但是他通过灯光、通过妆造、通过角色语言、所有环境的烘托，会告诉你“她真美，她太美了”。于是这些年里头，大家就会一发现啊，哪怕是你自己，你去看到某一些人的时候，哎，你就会觉得，哎呀，这个白人姑娘好漂亮，这个东欧帅哥太帅了，果然如此。你怎么会产生这种审美的意识的？你说它本来就好看？不是的。人类在跨种族的时候，很难觉得对方非常漂亮，除非她已经突破了平均审美的临界点，变成了大美人。任何一个族群当中，这种顶流的绝色美人都是凤毛麟角，无一例外。没有哪个群体普遍性的好看，只有绝大多数的普通人，甚至有相当一部分的丑人，还有凤毛麟角的漂亮的人——确确实实所有人都会觉得她好看，就是这样的。但是呢，在今天的中国人，很多中国人的眼里，白种人大部分好看，为啥？因为被驯化了。在跨种族的情况下，除非是绝色，否则你根本就不可能觉得对方好看，因为我们之间有审美屏障。这是自然界的，你不经过外界的教育和驯化，你没有办法突破这种自然界的屏障。但是他通过不断的一代又一代的各种作品的营造，就让你的审美发生了扭转。你看到中国脸的时候非常的挑剔，很严苛，你必须达到什么小花小草以上的水平，他才说你好看；但是对于洋人，他非常的宽容，只要不太丑，他都说你好看。这就是驯化的成果。所以我们看啊，这种操作的方式——这种“你很丑，而我很漂亮”的操作方式——其实在任何时候、任何地方它都是通用的。
