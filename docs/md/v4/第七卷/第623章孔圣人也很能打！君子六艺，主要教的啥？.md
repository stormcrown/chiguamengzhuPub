# 第623章 孔圣人也很能打！君子六艺，主要教的啥？

为什么中国的学生每天晚上做作业要做到十点钟？唉，就是孔夫子的国。孔夫子不好！为什么中国的员工要996啊？孔夫子不好！都是儒家的锅啊！你们这些人老会了，只要一说到中国，这中国，那你也就不管那个事是啥时候发生的，也不管这是什么事儿，最后一个大总结都是儒家的错啊！就因为我们用了儒家的东西，所以就这样。嗯，对，因为，因为他们哪里知道儒家是干什么的呀？他也不在乎儒家是干什么的。反正只要有点什么东西啊，但凡是那些事情，他觉得呢，似乎不是很好。哎，好哎，儒家的锅！嗯嗯，啥？

孔子不是2米多高，能举城门吗？举城门的是他爹啊！举城门的那个是他爹，他爹2米多。孔子比他矮，孔子没那么大个儿啊。孔子呢，据测算应该是1米9冒头，没够2米，1米9冒头。然后孔子的这个事迹呢，主要是他特别擅长于射箭。孔子出城射箭的时候，城里头万人空巷都跑出来看他射箭啊。所以孔夫子呢，是箭术超群的那种啊。啥，有点矮啊？对对对，孔夫子有点矮啊，1米9。嗯，据说孔夫子也很能打的是吧？嗯，孔子那个时代啊，孔子那个时代的君子六艺，君子六艺啊，里面呢，就是包括了这个射箭的。他其实只带了武艺。孔子里面设定的君子六艺里面是包含了射箭，包含了驾车。听起来好像挺敷衍的啊，但是呢，在那个时代这种技艺，它不是普通的那种啊，他和你一般的打仗的性质还不太一样。我们讲一下这个君子六艺和其他的那种武力值的差异啊。就是君子六艺，咱们得先知道君子是什么。在先秦时期，君子和今天我们所说的君子之风的这种君子不一样，或者说是不完全一致啊。君子在最开始的时候，它指的是一种特殊的贵族的地位，君子是贵族的意思。他是一个血缘身份，代表了他拥有一个治国的继承权。因此呢，君子这种身份，它就意味着他将来成长了之后，要去治理国家，治理天下。所以孔子为什么设定君子六艺？他是在讲你要去管理一个国家，需要拥有什么样的才能之后，你才能做到这件事情。因此，他不是在进行道德上的训导，而是在进行社会顶层管理阶层的高管培训。

那他需要的君子六艺是什么呢？啊，这里头的那个，呃，驾车啊。驾车这个项目听起来简单，但实际上并不简单。因为什么？因为先秦时期的那个车呀，是战争当中最主要的一种交通工具，你可以把它类比成今天的坦克啊。你不要把它想象成马车，因为呢，你很可能就难以理解它的历史地位。它在当时的战场上面，他的地位约等于坦克。所以他是要求这个管理人员是拥有驾驶战车的职能的啊。他要会驾战车。那这种驾战车为什么难度很大？因为那个时候他的那个就是车的缘啊——这个东西应该算什么呢？啊，就是你控马用的这一套配件，装备没有后世那么优秀。所以就会导致一个问题：一个战车不能只用一匹马，那拉不动。因为战车上要站好几个人呢。坦克里面是不是有炮手啊，还有那个探看的，还得有开炮的好几个人啊。那个时候战车也这样，是好几个人配伍的。所以他驾车的那个人，在他的配件没有后来那么好的情况下，实际上他是要分别控制好几匹马。你就想他的这种微操作啊——我，我也，我就觉得给我的话肯定不行啊。就我这个水平，你又给我一个方向盘，我觉得我都控制不了它。相当于一次性给我三四个方向盘就这么玩儿啊。每一匹马有一个单独的方向盘进行控制，有的是两匹马，有的是三匹马。你想一想，那马还是活的，马还有自己的思考能力啊，他还不是纯车呢。你让他左就是左，那马不一定，马有自己的判断能力。所以你想那个时代的驾车，真不是一个普通的活儿，相当于你能够独立操作主战坦克啊。这个就是那个时代对于君子，对于这个统治阶层的一个技术培训。

所以这里头还，它还包括什么？包括射啊。他为什么不搞其他的那个项目？因为作为管理者啊，他在这个战车上面的地位不是在旁边扯着那个长刀、长枪在那里阻拦的，他是站在中间这个位置要发动指挥的。而这个位置上面最重要的一个技能是要能够射箭。所以他对君子的要求是要能够射。所以你看他去进行培训的时候，是有针对性的啊。所以呢，然后他这个里面还有一个项目，就是礼。礼是什么？礼是社会制度管理啊。礼不是礼貌，它包含了一些礼仪的规范，就是什么职级该怎么做的这些规范。但是这些是什么？这些是社会制度管理方略。它相当于今天的什么？嗯，组织法啊，啊，同时还包含了刑法、民法啊。他这是一整套的社会治理纲要，包含了法律规范，也包含了这种职责划分、岗位区分，每一个岗位有什么样的责任，有什么样的权益，他可以干什么，不可以干什么，这个东西叫做礼。你看一个君王，一个君子，一个社会的高管是不是得会这个？所以他的礼也不是给你怎么样行礼这点事情，而是包含了所有的社会管理的内容的。里面是不是还包括了算？算是数学啊。你要学会数学，数学是干啥？你得能看账本儿，你得能够弄清楚你们这一个国一年的营收是多少，支出是多少，亏损是多少。你不能让下头的人把你忽悠的一头雾水，他说今年赚了，结果你看不明白，实际上亏了，那你怎么办？所以君子，君子的六艺，他是一个非常实操性的东西，属于岗前技术培训，而且档次非常之高。

因此当你说到这儿的时候，你就得弄明白：儒和文弱之间一点关系都没有。儒是干嘛？儒是进行社会治理的。而社会治理当中最重要的是什么？唯祀与戎不可以假人。戎是什么？戎是军事。你咋能冲上来就说儒就是柔弱的、就是文弱的呢？他是要管军事的呀。因此儒学从一开始介入到整体的制度建设的时候，就没有任何文弱可言。所以今天你在我们身上找出来的任何的问题，你要具体问题具体分析，而不是看见了哪个地方不好啊，都是儒学的错，都是孔子的错。孔子自己身高1米9，擅长驾车，擅长射箭，而且武力值爆表，是可以直接靠暴力收学生的。
