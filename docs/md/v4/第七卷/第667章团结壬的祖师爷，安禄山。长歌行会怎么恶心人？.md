# 第667章 团结壬的祖师爷，安禄山。长歌行会怎么恶心人？

当年招那个谁啊，招李治进宫的时候，这个消息被泄露了。然后他自己招募的胡人卫队就趁机造反，差点把他弄了。幸亏那天李治迟到了，放了鸽子，所以中间出现了时间差，他的汉人卫队把他给救下来了。所以李世民在这之后和在那之前，是两种完全不同的态度。

就像今天我们在任何一个我的直播间里面，在我任何时候开直播的时候，都会出现那样的人。哎，我们只要一谈主体民族，只要一谈满清的罪，唉，立刻跳出来了：“56个民族是一家啊！”安禄山当年干的就是这个活儿。他明晃晃地把他麾下原本的汉臣替换成了胡臣，换成了胡将。然后满朝文武都说他这个事干得不对，他错了，他有问题，他要搞分离主义。然后呢，安禄山说：“大家都是一家人，我换什么都是对的啊。你说我换成了胡将就错了，你难不成觉得我不是一家人吗？”哎，就是这种原理啊。他当年的这种操作方式和这些人的狡辩，是同样的一种话术，同样一种原理。然后干的是什么呢？啊，最后他干了什么，你们自己心里有数啊。

呃，那个帕斯，你的那个什么《长歌行》，嗯，来，帕斯，你的那个刚才说的那个《长歌行》，那个是个电视剧吗？我看着这个名字有点像电视剧啊。我没有看过这本小说，这个小说我没有看过。嗯，没事儿。那个我就是说这个是不是一个电视剧？如果是一个电视剧的话，我好像瞧见过这个电视剧当中的一个小的片段啊，其他的我没有看过啊。是一个动画是吧？啊，好的好的，嗯。唐代背景的是吧？嗯，好的好的啊。唐代的背景我都能猜得出来他们会怎么恶心我们。就是唐代的那种制作背景的话，他想要恶心我们，他的操作的手法很好玩：他会盛赞这个盛唐啊，他和其他的时代不一样啊。唐代的时候，唐代的那些故事，他在进行拍摄和编撰的时候，他会做什么呢？他通常情况下会盛赞唐代是一个开放包容的时代，然后呢，给里面的那些主角、配角安排胡人的配偶，然后给那些生下来的那些面孔比较杂糅的、混血的这种特征的人啊，给一种什么“美丽的、聪明的、人人都爱的人”人设，通常会有这样的一种倾向啊。是，是这种设定吗？啊，如果是这种设定的话，我就猜对了。他们通常在拍摄那些跟唐代背景有关系的艺术故事，在创作的时候就会用这样的操作手法啊。嗯，就是你看上去他在夸，而且是盛赞，夸得非常非常好，简直就是在赞美这个大唐。他把长安会吹成一个国际化顶流的大都市啊，“万邦来朝”。当然也确实是万邦来朝，但是性质和他描述的完全不一样。可是他描述的那个场景，就让你觉得这个地方和乐融融，全世界所有的人在这个地方都有一席之地啊。这是一个很刻意的行为。而且呢，在这个过程当中，他会努力去缔造一个唐王室是一种充满了混血感的，而且非常追求混血感的这种人设。一定会这样做。这是这么多年以来，历史虚无主义在唐身上所下的一记重注。他们这么多年来就一直要打造“唐是一个混血王朝”，他非常执着于打造一个印象，就是说唐呢，是一个混血王朝，唐的统治者是一群混子啊，他们呢，特别喜欢外国人啊，拿外国人当亲儿子。特别喜欢打造这种印象。他在干什么？他试图让今天的中国人愿意接纳外来的人口，愿意相信混血很优秀，因为李唐很优秀。李唐优秀是因为什么？是因为他们愿意接纳混血，愿意接纳胡人。这是他们这么多年来试图污名化、试图扭转的一种历史观，而他完全颠倒了历史进程的时间轴，这是严重违背历史唯物主义的一种操作方式。

为什么这样说？因为李唐王朝是怎么强起来的？他是靠让所有的胡人都在他的首都跳舞所以才强起来的吗？不是啊。他是靠着兵马强盛大杀四方，他是靠着在敌军当中七进七出，他是靠着直取敌军首级——他是用这种方式建立的强大的帝国。因为他很强，所以那些胡人才会到这里来朝圣，才会在这个地方载歌载舞。如果不是唐军极强、磨刀极狠、砍头极利索，他们进来那就是过来劫掠的。他们又不是没来过，为啥进来跳草裙舞？不然的话唐刀就要剁下去了呀。

因此你就会看到这种类型的作品当中，总是会给你夹杂这个玩意儿。它是完全违背唐代的那种制度管理方式的。唐代是什么制度？唐代呢，其实呢，它因为扩张的速度很快，但是在初期的时候啊，因为李世民吃了两回亏啊，所以他很快地就反应过来了：就是你去吃下这片土地并且招揽上面的人行不行？行。但是呢，这个过程会面临着极大的风险，因为对方有自己独特的认知，而这个独特的认知一定会在你某一个时刻不精神的时候反刺、背刺进来。这个事儿他亲身经历啊。他在当年招李治进宫的时候，这个消息被泄露了，然后他自己招募的胡人卫队就趁机造反，差点把他弄了。幸亏那天李治迟到了，放了鸽子，所以中间出现了时间差，他的汉人卫队把他给救下来了。所以李世民在这之后和在那之前，是两种完全不同的态度。在那之后他的态度就正常了。

什么样的操作方式？在唐王朝的时候，胡汉不婚，而是有这种规定的。尤其严禁汉女嫁给胡人，而且胡人严禁穿汉装。胡人的市场是专门划定的，许可证是单独分派的，税收是单独收的，管理人员是针对性单独管理的，完全不一样啊。连在司法的那种方向上都是不一样的，权利、个人权利都是不同的。只有其中一部分专门被招揽进来的胡人高层头目们，这些呢，是为了招安来使用的，他们的地位会略高。但是呢，胡官的地位低于汉官。这些所谓号称的这些胡人的贵族，你瞧着已经进入朝堂了，但是他们的地位是高于汉人的平民，他的地位仅仅高于汉人的平民啊。就是这样的一种分层管理。这种管理有一个什么好处？就是因为你对后来新加入的队伍提出了较低的福利待遇，于是呢，他会努力地投效，努力地打积分，努力地加盟进来，想方设法地印证“我就是你家人，你不许把我踢出去”，而不会明目张胆地搞分离主义。所以唐初的时候，他就是用这样的方式进行的管理。

但是这件事情到了李隆基的中后期就搞坏了，因为他过于地信任胡人，哪怕说已经出现了反叛的迹象。而且这种反叛啊，就是安禄山当时造反之前，首先做的事情就是胡将换汉将——他的治下，他的麾下原本是有很多的汉将汉臣的，但是他打算造反之前，就陆陆续续地把这些汉人替换成了胡人的部将、胡官。于是满朝都认为他要造反了。这是一个很典型的分离主义操作，放到今天也是很容易理解的一个行为。你天天这样做，你是不是要闹妖？就像今天我们在任何一个我的直播间里面，在我任何时候开直播的时候，都会出现那样的人。哎，我们只要一谈主体民族，只要一谈满清的罪，唉，立刻跳出来了：“56个民族是一家啊！”安禄山当年干的就是这个活。他明晃晃地把他麾下原本的汉臣替换成了胡臣，换成了胡将。然后满朝文武都说他这个事干得不对，他错了，他有问题，他要搞分离主义。然后呢，安禄山说：“大家都是一家人，我换什么都是对的啊。你说我换成了胡将就错了，你难不成觉得我不是一家人吗？”哎，就是这种原理啊。他当年的这种操作方式和这些人的狡辩，是同样的一种话术，同样一种原理。然后干的是什么呢？啊，最后他干了什么，你们自己心里有数啊。后续的咱们就不说了。嗯，太阳底下没有新鲜事，人类的操作方式总是很相似的。

嗯，太片面了。安禄山地方派？你想要按地名来区分是吧？那他倒是别换胡将啊。他在搞胡将换汉将的时候，他咋不按地名分呢？他那些悍将可都是本地人，胡将是外地人啊。你呀，最后按地名分是吧？自己心里知道自己做了什么，但是嘴一定是要硬的啊。
