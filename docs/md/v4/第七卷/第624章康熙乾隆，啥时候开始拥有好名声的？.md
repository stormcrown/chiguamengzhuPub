# 第624章 康熙乾隆，啥时候开始拥有好名声的？

嗯，小满又来了是吧？哎呀，清粉儿嘛，清粉儿的日子不好过啊。我现在已经开始心疼清粉了，你们懂吧？虽然我讨厌他们，看了之后就犯膈应，很恶心，但是呢，我现在都开始心疼他们了。为啥心疼他们？因为他们动不动一说起来，你不能说满清的坏话，不然你是不是痛恨东北人民？你是不是要跟东北割裂了啊？如何如何？你是不是在东北怎么怎么地了？他为什么要搞这种地域决定论？他自己都知道，满清身上一坨一坨的都是屎。所以呢，他必须蹭在好人的身上啊，扒着别人的裤腿儿，你不能把我扔开，你把我扔开了，我留下的这个脏手印子可就在你身上了，你就看见了。他们特别喜欢蹭别人的关系，这种蹭关系的行为，寻求庇护的行为，去找个好人，说你看你对满清不好，就是对这个好人不好啊。呵呵。

为什么这个样子？因为这几年来，他们之前20年所营造的那种假名声越来越失效了。这个假名声是哪来的啊？既然今天他的名声越来越坏，咱们就得先弄清楚一件事：他为什么之前名声居然是好的？跟你讲啊，今天的这个名声坏，他甚至还没有进入到历史的常态，只是稍微扒了一点点皮而已，他们就开始慌张得不得了了，就想把所有人都拉下水啊。只要批判满清，你就不团结；只要批判满清，你就要分裂；只要批判满清，你就不要国土了——毕竟国土都是满清打下来的。真是一点脸都不要啊！他为什么喜欢蹭别人？因为他危机四伏，他感觉到他自己的名声岌岌可危，他感觉到了大家都嫌他恶心。但是就现在这个局面，已经不是他的名声的常态了，因为他的名声的常态比现在要恶劣得多。所以第一个问题是，他的名声咋好起来的？名声坏不奇怪啊，为啥呢？因为满清王朝从1644年开始，就从来没干过一件好事。这样的群体，他要是有什么好名声，那说明这个地方的人都活该当奴隶。既然我们站起来了，我们不是奴隶，那他就不可能有好名声。因此，他什么时候才开始有了洗白的可能性？什么时候居然出现了好名声？上世纪80年代。

上世纪80年代。为什么呢？为什么这个时候开始了呢？因为这个时候我们在效率与公平上面，倾向于优先选择效率。这是一个导向啊。从上世纪80年代开始，我们着重于追求社会的经济效益。于是大家会注意到，当我们努力地追求经济效益的时候，他必然会引入很多的资本、老板、企业家、商业运营——这个就是一个客观现象啊。你只要导向于这种效率优先，那就肯定会有无数的这样的企业老板、资金大头就开始冲进来了，要在这个地方渔利。那吸引进来的这些老板当中包括了谁呢？不可避免的，在这些大规模的资产的流动当中，包含了当年的满洲遗老、遗少。不可避免。为啥我之前直播说过很多次啊？就是我们清代的时候有大量的资产外流，没有查账，不知道去哪儿了。这个资产外流的量有多大？不知道，但一定是天价。为什么是天价？中国人能够绵延万年的文明史，一直到今天都对自己的祖先记忆清晰。这意味着我们在历史的长河当中，一直都处在胜利这个段位上面。偶发的失败是短暂的失败，长久的环境当中是胜利。我们才能够有今天的这样的一个场景存在，我们才能够记得自己的祖宗是谁。凡是曾经真正失败过的人，都一定会认野爹。我们没有，是因为我们一直在胜利。当我们上万年的时间都取得了长远的、长远方向上的胜利的时候，你觉得我们会不会攒下家当？不要说什么盘剥、剥削这个问题，这不存在。因为剥削也是内部的剥削，肉烂掉了也是在锅里。作为整体的族群而言的话，我们这一万年的生产建设与积累，还包括了大规模的国际贸易，攒下了天量的财富。为什么到新中国的时候一穷二白？被谁卷走了？光是那些不平等条约远远凑不够这个数，远远不够。所以有没有人卷款潜逃？一定有多少不知道。去了哪儿？能追到吗？不晓得。那么当我们敞开国门，说我们现在效率优先，希望引入资金，敦促我们的社会经济高速的发展，尽快地追上国际贸易的这条大赛道的时候，你猜这些大体量的资金会不会回来？而他们一旦回来了之后，他们会干什么？都当老板了，腰缠万贯了，衣锦还乡啊。需要在家里面修一个宗祠，说我们家的老祖上是这个地方立了大功德的角色，不对吗？有什么不合理的吗？所以从上世纪80年代开始，大家就会注意到一件事情：书籍、报刊、杂志上面所有文字化的板块当中，就不断地强调满清的这些贡献。这个贡献当中呢，直接说的还不多啊，主要的是讲一些清代的逸闻趣事，讲一些皇室的小秘密啊，讲一些听起来就很可爱的小故事，让你觉得那个时代有可能——其实我是那个可爱的小格格啊，享受着美美哒的生活的呢。这个时候你的共情的心理，你的站位就开始逐渐发生转变。然后那个时期是不是开始逐渐逐渐地就越来越多的戏说啊？戏说这、戏说那的一些清宫戏。然后这些戏是不是都请的当时最顶流的明星来演？谁漂亮谁帅就请谁。为什么呢？因为你喜欢那个演员，那个演员好好看，气质太好了。然后等到他去演了清代的那些皇帝贵人，你就自然地觉得：一想到那个皇帝，你就想到了那个大帅哥；你一想到那个贵人，你就想到那个大美人。于是你在直觉上面就会觉得那个时代阳光又灿烂。就是这种美化的方式，从上世纪80年代开始，通过各种各样的宣发，通过各种各样的驯化，就让中国人在某一些直觉上面相信了清代其实挺好，甚至觉得清代特别的好。甚至很多的公知在各种各样的采访当中啊，说满清从来没有出过昏君，满清12帝个个是明君啊。其他的时代这个昏君、那个昏君，这个毛病、那个毛病啊，清代从来都没有。你信赖学者，也相信他们身上的岗位职称，也相信他们背后的高校的背书。然后你觉得他说的是对的，这也会形成社会的群体潜意识。最终就打造了这样的一个时代的印象，它变成了阳光的、漂亮的、贵气的、璀璨的。至于说那个时代，你真实的祖上到底是在哪个地方讨饭？腿断了没有？那个晚上缺了多少的口子？你看不见、摸不着、记不住，你就不知道它，你就不在乎它。于是你就把这些祖先蒙受的真实的苦难忽略过去了，去跟那些蹂躏百姓的、蹂躏江山的、让这个国家跌破了底线、沦为18流国家的这些统治者们深度共情。

当年在演《还珠格格》的时候，你猜有多少的小女孩在那里幻想：我要是个格格就好了？今天的互联网上，有多少的网红在那里影影绰绰地映射自己是某某旗之下的格格？格格变成了个好词儿。格格是什么？格格实际上原本只是满语里面的小姑娘的意思，她没有其他的含义，就是满语当中的小姑娘、小女子、小娘子，就是这样的身份。后来入关了之后呢，这个格格就开始被收进了贵族系统啊，被称为贵女的这种专用名词。但是除了这些贵女之外，还有谁在用呢？还有亲王的小妾也在用。所以这个名词本身就是一个含糊不清的、界限不明的名词。但是通过各种各样的电视剧，尤其是像《还珠格格》这样的电视剧，还有那个什么《新月格格》这样的电视剧，它就变成了又高贵的、又雅致的、又有文化的一个象征，就产生了大量的小女孩幻想着自己去当格格有啥好的？包括说电视剧里面去那里跟你说，哎呀，满洲姑奶奶的地位老高了！弄得好像这些渔猎和游牧的民族特别尊重女性一样的。生产力越低下，女性的社会地位就越低下，这你没有得选。因为生产力低下，技术水平不足，就意味着大家更加倚重于体能。只要你倚重于体力，只要你倚重于体能，男性的地位一定非常高。你越倚重于体力，男性的地位就越高，女性的地位就越低。这是在全世界都通行的社会法则。所以在渔猎和游牧的群体当中，女性常常是作为财产、作为家庭的附庸而存在的。为什么她们存在收继婚？收继婚是什么？是因为这个父亲的小老婆们不是你的小妈，不存在伦理关系，他们是父亲的私产。你要继承父亲的财产，就要把这些女子也作为财产继承下来，所以才会有收继婚。一个女子只是一个财产的地方，你说满洲姑奶奶那个地位老高了——她相对于谁高？相对于奴才确实是高。谁家的主人家不比奴婢的地位高呢？当然了，清代的时候啊，雍正皇帝曾经吐槽过一件事情，说那个汉人有一种陋习，主奴不分。你自己觉得这个东西是不是陋习啊？这个是雍正皇帝的吐槽啊。他认为汉人不如满洲人，满洲人主奴界限非常清楚，主子就是主子，奴才就是奴才，主子永远是主子，奴才永远是奴才，界限分明，泾渭分明，中间不可逾越。但是啊，汉人就老是主奴不分，主不主奴不奴的，互相之间关系居然那么亲厚，奴才居然能够赎身出去当自由民，这真是没有规矩。这个是清代对于汉人的怒斥。之所以他们觉得汉人野蛮，就是因为汉人不搞奴隶制。所以有趣吧？就是这样的。

所以我们这个地方啊，在这个互联网角落上面，各个地方囤积的这些小青粉们，这些格格入脑的人，你想当格格，你去当去。你觉得那种给别人裤腰带上面拴配件的生活，是一个美好的高贵的生活？你去，那些精神包衣、精神奴婢，你就觉得人家的那个生活贼好了啦，你去。我不喜欢那些主奴界限分明的时代。他们的主子瞧不上汉人，因为汉人主仆不分。但是我认为这是先进，这是文明，这是伟大。我们之间有立场之别，所以我没有兴趣，没有兴趣挽救你的思想。但是呢，我挺希望你们别来恶心我的。唉，谢谢中正评测啊。
