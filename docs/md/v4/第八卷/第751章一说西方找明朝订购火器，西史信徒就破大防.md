# 第751章 一说西方找明朝订购火器，西史信徒就破大防

嗯，他造谣西洋人在晚明时期订购火器？说我造谣晚明时期西洋人找我们订火器？这事是你洋爹说的，不是我说的。你们欧洲的洋爹自个儿写书说的，不是我说的。OK，是你洋爹说的，不是我说的是哪一牙来着？好像是葡萄牙。你洋爹说的，不是我说的。嗯，葡萄牙和西班牙都定了，他们肯定都定了。这主要是那本书，我记不清到底是葡萄牙还是西班牙，应该是葡萄牙。他们专门写了书，自己的书里面去写了他们找明朝订购火器，而且特别强调了在明朝生产的火器质量非常优越。为什么会这样？因为火器直接跟金属铸造有关，而金属铸造需要温度，对温度要求很高。你的炉温控温能力更强，铸造基础更好，做火炮这些设备时就要有更先进的生产工艺。所以明朝一直有火器出口项目，还有国际火器订购项目。这事儿在欧洲人自己写的书里都有，叫什么什么史来着。他们自己也写近代历史书，几本从五六百年前开始写的东西，当时的信息还是有点可靠的。更早的他们好多都编，但从五六百年前开始记述往后的这段，就有很多东西比较真实了。因为从这个时候，他们真的开始记档案了。开始记档案了，你写的东西就多少保留了一部分真实性。然后，他们所记述的关于明朝的内容非常非常多元，因为当时很多欧洲人到访过，有商人、传教士，传教士占比很高。他们来到东亚、东南亚地区，基本上最后都会到明朝的土地上溜一圈。这些传教士会记录当地风土人情，也绘制当地人的面貌、服装。所以现在欧洲出版的书籍，他们的古早派书籍中，有明代各族人群的画像，生活在不同地区的人都穿什么衣服、戴什么帽子、做什么发型，男的什么样、女的什么样、生产工具什么样，洋人的传教士在那个时期的绘画作品里都有。同时，关于火器的定制贸易，也是他们自己记录的专门档案。

那么关于铸造炉温这个东西，西方人自己，包括美国现在的高中历史教科书里也明确一点：高炉冶炼这个玩意儿，他们一直到乾隆年间才掌握。高炉冶炼这种高温冶炼工艺，他们是在乾隆年间才开始掌握的。在明朝的时候，他们还没掌握高炉冶炼，所以那些非常精致、依赖高温铸造的元器件，他们做不了，做不到，没有这个生产工艺。所以他们才会到中国来定制火器，并且会记录下来这里的火器质量非常好。

嗯，高炉冶炼是乾隆年间的？对，高炉冶炼他们在乾隆年间才开始掌握。就是我们在清代时有很多工艺外流。清代时发生了很多工艺外流。很多人觉得——我回忆一下，是什么类型的杂志里讲的，想不起来了——反正就是那种可能跟《意林》有关系的杂志里面，曾经有一段时间特别着重于批判中国人不注重知识产权保护。我不知道还有人和我看过同样的杂志内容没有，几十年前的内容了，老内容我也记不住杂志是什么了。讲的大致是关于造纸、景泰蓝工艺的流失，说我们的工坊没有保密机制，没有保密意识，所以别人来看呢，就让别人看，然后别人就都记下来了，最后别人就把这技术开发出来了，出现了技术外流。技术外流之后，我们赚的少了嘛。当时这杂志专门批判说中国人没有保护意识，随便就把技术流出去了。你看别人设计技术壁垒。就这么一篇文章，我印象非常深，当时是拿来批判中国人没有自我保护意识的。但是如果真的没有的话，我们的国际顶奢品贸易怎么可能干2000年呢？啊，当然好像也不够2000年，大概是一千五六百年。咳，千年的时间里，我们一直拥有全世界最顶流的顶奢产品贸易掌控权。全世界的丝绸、茶叶、铁器（铁器是我们的重要贸易产品）、陶瓷，这些都是需要高温的，高温的这些设备别人那里没有。这些呢，都是当时的顶流科技产品、工业产品。我们是这些工业品的输出国，而且它们一直在国际上可以当硬通货使，这种高科技工业产品卖了1000多年。这1000多年，那些人从来没有试图学习我们的技术吗？为什么一直到清代时，技术开始出现大规模流失？结果他们戴的帽子叫“中国人不在乎这之前1500年怎么啥事都没有，把技术弄得干干净净”，甚至最荒谬的是什么：他一面说，你看洋人一来了你就让他们学会了东西，你没有自我保护意识，不知道怎样保护知识产权，你活该被人家偷走；一说到中国内部我们自己的工艺品断代时好遗憾，他就说你看吧，你天天守着你那手艺也不知道向外头去教育、去传播，还有什么教男不教女、什么什么一堆词儿，说你之所以造成秘方断代就是因为你太狭隘了，生怕教会了徒弟饿死师傅，天天藏着掖着。不是我们一边藏着掖着，一边敞开门让别人抄技术，请问到底哪边是真的呀？这些杂志都是我几十年前反复看过的，都是这样的文章：一边说我们的东西自我保护到了断代，一边说我们不懂自我保护让外人都学去了。两头堵都不带这么玩的。

实际上是什么？实际上是我们之前的1500年间，我们的技术垄断做得非常好。你任何一个企业老板都知道，当我有自己的秘密且可以形成垄断市场时，是我赚得最多的时候。然后我说我把工厂敞开来让你随便学——脑子这样的人开不起工厂来，脑子这样的人也学不会技术。所以这是不可能存在的现象。所以实际上，我们的技术外流有没有？有的。怎么外流的？什么时代外流的？景泰蓝——清代外流的；陶瓷技术——清代外流的；冶铁技术——清代外流的。我们的核心产业的核心技术有很多一部分是清代时实现技术外流的。然后你说就是中国人不在乎，中国人之前都在乎，怎么到你清的时候就不在乎了？到底是你清的问题还是中国人的问题？这事儿能拆明白不？所以怎么就那么巧都到了这个时候技术就开始外流了呢？之前洋人不想学吗？他被我们天天用垄断技术赚赚钱，羊毛薅得都快秃了，他不知道要学习，一直到清的时候突然开窍了知道要学习了？咋就那么巧，到了清的时候这些技术左一样右一样地开始全球扩散了、洋人都学会了呢？别的时候他怎么就学不会呢？

然后那个技术上的什么传男不传女——啊，这个传男不传女根本不是性别问题。之前我也聊过技术流，它的传男不传女不是性别问题，是家庭归属问题：它教谁？不是不教女的，是不教会外嫁的女儿，而不是不教女性。它会交给儿媳妇。因此它不是性别问题，而是家庭归属问题。这是在进行技术保密。如果确认了这个女儿会外嫁，她就会造成技术外流、财产外流，所以就不教这个会外嫁的女儿；如果打算教女儿的话，就会要求女儿在家招赘。这是自己技术秘密保全的一种办法，根本不是性别问题。所以什么传男不传女、你欺负女的——她会教媳妇的，会教给儿媳妇。

嗯，历朝历代的更替都没有技术外流？嗯，对啊。因为不同的时代，其他的所有时代，只要是一个正常的时代，不需要其他标准，也不管是明君还是昏君，也不管这个时代质量到底高还是低，只要它是一个正常的时代，想要维持正常的经济运转，任何一个国家都会保护它的科技。所以我们以前受到北边人群的侵扰，袭扰一直都有，但他们一直不断地掳掠我们这里的工匠过去干嘛？即使他们没有能力自己去创作这样的科学与技术，即使他们没有真正的定居生产、积累不下这样的知识文化，但他也知道这个东西是好的，就是抢也想过抢过来。这才是一个正常的人群会做出的选择。所以历朝历代，无论昏君还是明君，都会重视技术，重视这些生产科学技术的人才。而且拥有技术的家庭、家族都会受到认可与保护。为啥？因为他们掌握着这个时代非常高精尖的东西，相当于你家里面的高科技工厂。正常的国家都会保护自己家里面的高科技工厂，因为这是我们国力的基础中重要的一部分。这是正常的时代。但是清朝不是一个正常的时代。清朝不是一个正常的时代，他是八旗用很小很小的一批人，以纯粹军事暴力的方式镇压了比他先进很多的人群，奠定的一种统治机制。在这种统治机制下，下面的普通民人们如果技术先进、文化优秀、认字会读书，就意味着他会很迅速、很有力地推翻上面的统治者。因此，整个清代的200多年就是一个刻意的让技术倒退的200多年——不是停滞，是倒退。因为技术越倒退，水平越差，文盲越多，他的统治就越稳固。所以明朝时是有民间的强制教育的，所有普通人都需要认字，不需要有多高文化，但得认字。但到清代时，普遍性的文盲，文盲率超过99%——你不要说是啊，这是清朝自己的官方统计，敢情自己的官方统计他自个儿说的，不是我给他说的。99%的文盲率，超过99%的文盲率。因为他制定的计划是7年之内（还是5年？），7年之内让文盲率——原话是啥来着忘了——反正大概是7年之内让文盲率降低，降到什么程度呢？能够让识字率达到1%。7年让识字率达到1%，这是他的一个计划目标。所以他自己官方统计就是在清末时识字率低于1%，具体是零点几不知道，低于1%，就是这样的水平。所以清朝不是让技术停滞、不是让文化停滞，而是直线倒退。这就造成了一个现象：他非常不尊重、非常不重视、非常蔑视、非常奴役，甚至刻意消灭那些有技术、有生产能力、有创作能力的家族。典型的是康熙年间，明末的移民戴梓，他是一个军事器械专家，发明了连珠铳，献给了皇帝。因为军火这个东西，你不贡献给统治者是毫无价值的。他把东西献出去了，然后举家流放关外。这就是清对技术工作者的要求。然后连珠铳干啥了呢？锁起来了，藏仓库里面了。同时，他还把钦天监（国家官方天文台）交给洋人管理，不让汉人的天文学家进入天文台工作。我们有没有传统上的天文学世家呢？有的，叫什么名来着忘了，反正是让人家回家务农去了。别人专门写了文章献给皇帝，说对天文对于历法的计算有新的想法——这是干什么？这就是一个国家天文台的求职信。然后皇帝的看法是：写得挺好，回家种地吧。就这样。所以它不是技术的停滞，是技术的倒退。而这个倒退是刻意造成的，是存心的。他对于技术人员、对于科学家的迫害是空前的，他就是要销毁你的科学技术。他在编撰四库全书时，焚毁的书籍当中有大量是技术类的书籍。这还是在留了目录但书找不着了的情况下。那有没有书连目录都没有的呢？那就不知道了，对不对？有一些账已经对不上了。就是留了目录但找不着书的，有数千册之多，还是科技类的，都还不说其他类目。还包括了其他历史类的、哲学类的书籍，其实都有。因为这些内容中会包含动摇统治根基的一些思想性著作，这些东西也会被有的删、有的改、有的销毁。
