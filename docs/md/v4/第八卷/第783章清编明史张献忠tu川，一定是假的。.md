# 第783章 清编明史张献忠tu川，一定是假的。

建州部当年被思密达那边打得抱头鼠窜，流亡到了辽东土地上。是明王朝接纳了他们，作为一个流亡部落，给他们发了“暂住证”，甚至安排了专门的工作项目，给他们安排了个地方。所以，明王朝有大恩于建州部。但是，入关之后，他们大屠特屠；入关之前也大屠特屠。那些接纳他们生活、与他们共同生活在辽东土地上的汉人群体，在入关之前就被屠戮殆尽。请问这个行为和商代的人祭之间，可以拿来做任何程度的对照吗？这玩意儿叫恩将仇报。

嗯，好，欢迎大家来到直播间。刚才我瞥见了一句话，我不确定，我给大家复述一下。他说：“满清再残酷，还能有商代的人祭来得残酷吗？”大概是这么说的。来，满清再残酷，能有商代的人祭来得残酷吗？这两个能是一回事吗？我讲祭祀的那条视频，大家看过没有？那条视频里讲到了祭祀的本质问题——到底什么是祭祀。商代的祭祀是干什么？商代为什么有人祭这件事情？商代的人祭主要分成两种类型：一种类型是把敌军的头颅供奉给祖先。这个呢，你不能拿清去做对比，因为拿清的时候，我们这直播间里面大量的小清粉、小满遗，你就难说清楚他心里到底站位在哪了。我们举一个更清楚的例子：商代的人祭中拿敌军首级供奉祖先的行为，类比到今天好理解的场景，实际上是“马踏东京筑京观”。这个才能直接对照商代的人祭。你觉得“马踏东京筑京观”这件事情，是属于残酷无情，还是复仇成功？见仁见智。但是如何对照这件事儿？要这样子对照，这两样东西才能放在一起比。

所以，商在做什么？商做的事情，是将敌军的首级供到了祖先的坟头，告诉祖先：“唉，这事我解决了，仇家我帮你报了，仇帮你报了，了结了。”这个是商代的人祭中最重要的一个项目。另外一个项目，就是把自己送过去，谈大事。这是一种。还有大面积的那种，其实是供敌军的首级。这是商代的人祭。那么，请问满清的各种各样的大屠特屠，目的是什么？要知道，明王朝接纳他们在东北这里暂住，还给他发了“暂住证”，这件事实际上是有大恩于建州部的。建州部当年被思密达那边打得抱头鼠窜，流亡到了辽东土地上，是明王朝接纳他们，作为一个流亡部落，给了他们“暂住证”，甚至安排了专门的工作项目，给他们安排了个地方。所以明王朝有大恩于建州部。但是，入关之后，大屠特屠；入关之前也大屠特屠。那些接纳他们生活、与他们共同生活在辽东土地上的汉人群体，在他们入关之前就被屠戮殆尽。请问这个行为和商代的人祭之间，可以拿来做任何程度的对照吗？这玩意儿叫恩将仇报。

嗯，啥？说一下张献忠和屠川。四川是谁屠的？四川是清屠的，板上钉钉。四川绝对不是张献忠屠的，这妥妥的是一口黑锅。四川的这件事情，妥妥的是张献忠身上的一个黑锅。你说张献忠一定脾气很好？那不保证；人品优良？不保证。一个军阀割据头子，也难说他什么样的性质。很有可能他确实很暴力，很有可能确实很暴躁，也很有可能确实没什么品德要求，但他绝无可能屠戮自己的基本盘。他作为一个割据军阀，最重要的生计来源是啥？蜀中百姓啊。他要活下来，他要拥有势力，他靠谁？靠蜀中百姓。一个能当割据军阀头目的人，他可能不好，他可能坏，他可能暴躁，但他不可能蠢啊。这个事情绝对不是张献忠干的。

而且，在清修的《明史》当中，还留下了这样的一个bug：他说张献忠屠川，然后给了一个数，叫“六万万”。大家自己去算算数，六万万是个啥数？川中有没有可能有六万万给他屠？或者张献忠得牛到什么程度才能屠六万万？所以那个数字上面也是有毛病的。是故意留下来的一个引子还是怎么回事，咱们就另论。但是无论如何，川不是张献忠屠的，但锅归他背了。而且实际上，到后面确实有“湖广填四川”这件事是真的——也就是说，川中确实已经无人了，战至最后一兵一卒，川中无人，然后湖广填四川。湖广填四川是真实的，但当年张献忠屠川必然是假的。

嗯，那时候不是总共才四亿吗？你说的这个四亿已经是清末了。清末是四万万。关于人口问题，咱们一定要弄清楚一件事：所有关于人口的问题，你必须弄明白——古时候的人口统计都是一坨屎。咱们复述一遍：古时候全世界所有的人口统计全是一坨屎，因为没有人能算得清楚，没有统计能力，做不到。根本原因是做不到。有一部分是依据当时的官方统计，因为官方为了收税，会搞那个“黄册”来统计户数，然后按照户数乘五，约等于每户五个人，这是一种计算方法。但这个计算方法非常粗陋，尤其是当有人头税、丁役等问题的时候，会有逃人——就是有一些隐户，他活着呢，正常生活、正常工作，但把户口挂别人身上，于是户口上没他，收税不带他。就会出现这样的现象，而且并不罕见。所以你说他能统计清楚吗？统计不清楚。所有的这些数都统计不清楚。到了清的时候，确实已经把丁税归到了田税，因此隐户变少了。那你说到清的时候统计就好了吗？是的，隐户变少了，但虚报变多了呀。之前为了避税，隐户偷摸挂别人户口；但到了清代，不需要隐户了，官员要报政绩——人丁滋长说明我的政绩好，又不会因为人丁滋长就多交税，没有这个负担的时候敞开了吹。所以清代有虚报，别的时候有隐户，清代有虚报，所以全是一坨屎，就没有谁能统计得明白的。民国时候的那个数也是一个概数、一个估计，都不是准的。

所以在这个统计上面，还有几种统计方法：一种是用盐来统计。为什么盐要分官盐、私盐？为什么官盐要发放盐引？为什么贩盐的私盐贩子要打击得那么惨？因为盐是每个人必须吃的，所以用盐提高售价，变相收税，盐实际上是一种收税工具。所以在计算人丁的时候，会用盐的销售额来倒推一个数据，这也是一种人口统计方法。但因为私盐的存在，这个数准吗？不准，就是个大概。全部都是大概。还有一种计算方法是依据田亩的产量，进行倒推反运算，去计算当时到底有多少人。这些都是现代进行古典时期人口计算时，会运用的几种分析方法。所以你说准吗？就没有准的，没有一个是准的，放心啊。

所以在这个性质下面，咱们就得知道：您说“哎呀，他那个时候有多少人”，不知道为啥——尤其是明代的人口统计，那是一坨大的。为啥这样说？明初为了收税，确确实实统计了一下有多少人、多少户，还做了档案，全部收入了档案库。但是之后再也不查了，再也没有普查过。明初多少人，明末就多少人；明初几亩田，明末就几亩田，动都没动过一下。你说它有没有变化？两百多年怎么可能没有变化呢？最后一口没变。所以这些数据实际上是今天的学者通过侧面的数据来印证、推算，算出来的账。那按照这种算法，明末的时候有多少人呢？有几种不同的口径：较低的一种是1.2亿，特别高的一种是在2.2亿或2.4亿左右。它的计算因为算法和口径的不同，偏差会比较大。目前相对来说被认可度比较高的说法，大概是在2亿上下。在2亿上下浮动的这个区间，这种算法一般受到的认可会比较多。因此我们现在说，明末的人口大约是在2亿左右，是这个范围、这个推理范围。

那么清初的时候有多少？清初进行了一次大普查，清初是有一次大普查的，清出多少？八千多万。八千多万来。中间的这个落差，中间消失的人、不见的人，上哪儿去了？嗯，这个还只是一个概述。我们知道明末的人口统计实际上是各路学者施展自己的本事进行的运算，并不是一个非常精准的数字，所以中间有较大的浮动。我们按照其中的一个公约数，被认可得比较好的一个公约数是2亿。然后清初是官方进行了统计的，是八千万。这个八千万准不准？有没有毛边？不确定。但是清初有这么一个数据。所以你看一下差额。这个时候就会有一个——之前《红楼梦》里的“白骨如山忘姓氏”，你觉得一个江宁织造家里面被抄家夺财的事件，配得上“白骨如山”吗？明清易代的过亿人口损失，算不算“白骨如山”呢？

嗯，啥？清末的——刚才说了，那个四万万是指清末，四万万是清末。嗯，然后整个清王朝持续的期间，这个还要再说一下。整个清王朝两百多年的存续期间，你可能会以为他是从八千万涨到了四万万，就是涨到了四亿，是有一个大的膨胀期。但实际上，在同一个时期，同样的这个区间，我们周边的其他国家，包括东南亚国家等等，这些受到了物种大扩张、农作物大扩张影响的地方，它的人口平均暴增是七倍到十倍。也就是说，如果我们当时跟上了世界人口膨胀的比例的话，清末我们应该拥有的人口是八千万乘以七或十。我们就按最少的一个七来算，多少？5.6个亿。按照最少的这种计算方法。要知道中国人是最会种田、最会种田过日子、最会积攒的人群，还动不动喜欢说“中国人就喜欢生孩子”，繁殖癌是吧？好，那么最喜欢生孩子、最喜欢种田、最喜欢积累财富的人群，在全世界人口大暴增的这种环境下面，我们增长率应该是多少？别人在条件并不良好、技术并不良好的环境下面，能够底线增长七倍，多的会增长十倍。请问正常的中原王朝应该增长成什么样子？底线5.6亿，上限八亿。那么清末的人丁多少？四个亿。最少缺了一个亿。那一个亿去哪了？所以当很多人说“我大清给你带来了那个什么人口大爆炸，这就是功德”，你知道怎么比吗？你得知道怎么比较。不是说你跟清初比较，哎呀，人长得好多呀——是这样比的吗？不是这样比的。是任何地方在这个时代都出现了七到十倍的大扩张，而中原的人群在那么擅长耕作和积累的情况下，从八千万到了四个亿，增长率远远落后于全球。这意味着什么，是吧？

这个例子呢，跟今天的阿美（美国）还有点像。今天的阿美是一个移民国家，而且这么多年来一直是一个移民国家，每年都吸纳大量的移民进去。它的人口增长率是大有问题的。阿美这些年的人口增长是极有问题的。你们自己去偷摸算一算账啊：他2亿几千万的时候，离现在不过是二十几年，现在才多少？破四亿了吗？没有吧。这么漫长的时间，作为一个移民国家，而且它的生育率也没有暴跌到那种让人崩溃的地步，新移民很多时候还很爱生，它里面的新移民在过去几十年一直说新移民的生育率很高，牛都是他吹的。新引进来的移民也放在那儿了，新移民的生育率又不低，为什么这么多年了，它的人口增长数量那么少？去哪了？
