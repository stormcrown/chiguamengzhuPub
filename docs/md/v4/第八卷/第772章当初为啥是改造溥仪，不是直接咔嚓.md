# 第772章 当初为啥是改造溥仪，不是直接咔嚓

来来来，嗯，有人说教员对康熙、雍正的评价很高啊，出处是在哪里？那个，老舍啊，老舍。来，你先搜第一个关键词，老舍啊。哎，谢谢秦恒。然后再搜第二个关键词，儿子。连在一起呢，就是老舍的儿子。你搜他。他呢，为他的父亲写了一个传记型的书啊，有这么一本书，就记录他的父亲。就这个书啊，在这个书里面呢，他说教员跟他父亲说话的时候说啊，“清的贡献吧啦吧啦吧啦吧啦，嗯，康熙的伟大，巴拉巴拉巴拉啊。”就是打字来的啊，就打这来的啊。就是老舍儿子，然后你去找他的东西啊，就在那里头，就在那里头啊。因为这个原因，于是呢，大家就说啊，原来教员还夸过康熙呢。在哪夸的？在老舍的儿子为他爹写的传记体的文章当中说，教员曾经跟他爹说过这个话。教员说过吗？反正教员自己的记录当中没有啊。最早的出处就在这。至于说他到底是咋知道的，是通过梦游，通过托梦，还是通过小孩在旁边吃糖豆、听闲嗑牙，那不清楚啊。这个事不知道。

哎，啥？秦始皇说要传位给你，那我可以信啊。这这我我我我觉得我可以信你行吗？我可以信，我可以信。哈哈哈哈。啊，始皇帝还送你兵马俑手办了，我也可以相信。要不然你送我一个？可以分享吗？你可以给我分享一个啊，我有兴趣啊。

水太凉是钱谦益啊。咋的？哦哦哦哦哦，因为老舍的事是吧。那个钱谦益这个人呢，一生也算是起起伏伏啊。直接就从头到尾的嘲讽，我个人觉得好像也不太对味儿，因为他确实起起又伏伏。在一开头的时候，他其实还是颇有志向的啊。钱谦益呢，写了一首诗。今天一直有人说什么，嗯，“崖山之后无中华”是日本人说的，所以你不能信。但是这句话呢，就是钱谦益写的啊。钱谦益写了诗：“海角崖山一线斜，而今也不属中华。”这个很明显的是一个爱国诗篇啊。所以呢，钱谦益在他的青年时代是爱国的，并不是一个汉奸、背叛者。但是呢，到后面啊，他的软骨头就暴露出来了啊，然后就降了嘛。他降了的这个梗，大家应该都知道，就是所谓的“水太凉”啊。应该殉国的时候他觉得水太凉，于是呢，就没殉啊。这个就是“水太凉”的梗。但是在那之后，他的晚年生涯一直都在资助反清复明。所以我说他这个人的一生呢，一波三折，起起又伏伏，就是因为钱谦益的晚年实际上一直都在资助各种各样的反清复明运动。

嗯啊，为什么会这个样子？咱们得知道一件事啊，就是你像明王朝的晚期的时候，其实确实不太争气啊，大家心里头应该有数。任何一个王朝的末期，他一定会显现出乱世之相。那当一个王朝走向末路的时候，积重难返，内部的经济两极分化哈，再加上各种各样的天灾，所以明代的晚期实实在在的讲，日子过得不好啊。既然日子过得不好，就会出现民变、兵变嘛。所以呢，明代的晚期一直都有流寇，流寇作乱，农民起义军，然后包括边军造反。唉，谢谢RICHARD啊。这些事情在晚明时期呢，都是频繁发生的。实际上整个晚明的朝廷，都把这些流寇作乱当做心腹大患，事实上也是最后他确实是被这些人推翻的啊。啊，他选定的这个实力最强的对手确实没选错：流寇啊，边军作乱啊，农民起义军，那确确实实是拥有战斗力的啊。但是呢，你最后就会发现一件事：当崇祯皇帝吊死煤山之后，正常来讲，我们就可以开始进行下一轮的喜迎新军了。然而呢，满清入关啊。从满清入关之后，你就发现了一件事情：之前致力于推翻大明王朝的这些革命军，这些流寇，甚至其中的匪帮啊，所有的这些人在满清入关之后，直接调转枪头，开始一致对外。为啥呢？啊，是的啊，明王朝不怎么样了，快咽气了，确确实实大家都看不上他了，想换了。但是架不住入关的八旗跌破了底线呀。所以实际上很多投降了满清的文臣武将，在后来都陆陆续续地造反啊，到后来也开始陆陆续续地造反啊，包括像钱谦益这样的。你一开始觉得他降得太顺畅了，就是一个软骨头狗汉奸。但是呢，当他真正地在满清朝廷内部待下了之后，他就开始资助反清复明的事业了，因为他看到了更糟糕的场面啊。

钱谦益也配提？这跟配不配没有关系啊，这跟配不配没有关系。他早年的时候当过软骨头是毫无疑问的，但是晚年的时候资助过这个反清复明，也是真的啊。

哎，好，嗯。他们自己挖的坑，自己跳进去又想爬出来。其实在那个晚明时期，确实有一部分的人觉得，谁来当主子都无所谓，谁做江山都无所谓。这样类型的人在晚明时期是存在的。但是啊，他们之所以会这样想，也是会认为关外马帮群体的统治者入主中原之后，不会跌破某一种他想象中的下限。但是啊，不幸的是跌破了。所以当他看到真相之后呢，就后悔了啊。这样的人其实不少。没想到这些人没有底线，对呀，嗯，是啊。他一开始的时候，他觉得反正也就是换个皇帝的问题，当时很多人都会这样想的，很多人都会这样想的：无非就是换个皇帝嘛，多大个事儿啊。结果发现此皇帝非彼皇帝啊。

诶，好，嗯，啥，嗯嗯，现实和幻想存在区别。嗯，很多的时候呢，人们容易犯下这样的错误。所以你要是说点评，那点评的话，他们都是祸害，毫无疑问。但是你要是说，诶，那个必须从头到尾地往死里头喷，他们也确实后悔了。把他们后悔的这件事情翻出来，我觉得也是很有教育意义的啊。

嗯，实际上呢，就像那个像老舍这样的人啊，他在解放初期的时候是获得了很多优待的。那为什么这个样子？因为呢，共和国需要表彰一部分作为表率的、作为带头人的满族同胞。因为那个时候我们专门设立了这样一个群体之后，就希望他们能够做出榜样来，带领之前的一些八旗子弟能够改变自己过去的陋习。同时呢，嗯，谢谢哥伦布啊。同时呢，为了实现这个目标，他还要破除其中的一部分的障碍。什么障碍呢？就是在民间自发性的，当时会特别讨厌满人。为什么这样？因为他们之前几百年祸害太严重了，就不说国家层面上的，甚至有很多的人跟他们之间的私仇都非常的深。记忆还在的时候，在解放初确实存在着这样的一种歧视的倾向啊。就是结婚的时候不愿意嫁给满人啊，结婚找不着对象的，谁家要找对象找他们的话，家里面都要受歧视的那种。找工作也要受人家歧视的，各种各样的地方他都走不开。所以呢，共和国需要立一个榜样，说他们已经改邪归正了，要告诉公众啊，他们已经改邪归正了，给他们机会了，要让他们好好学习天天向上了，要让他们为我们的生活做添砖加瓦的贡献了。所以这个阶段呢，需要提出来几个比较典型的，嗯，然后呢，愿意为共和国做贡献的人，作为一个表率。例如说溥仪就是表率之一嘛，溥仪的检讨书是很重要的东西，代表了改邪归正的意志。至于他个人其他心里头想什么，咱不在乎啊，反正你只要写了这个公告天下，那就意味着你要带队承认错误，改邪归正了啊。那老舍也是这样的。那个时期呢，其实我们大大方方地提拔了很多过去的这些八旗的后裔。为什么啊？为了表示我们确实可以接纳他们的改正啊，确实可以接纳他们成为共和国的公民，成为添砖加瓦的建设者啊。这个就是我们坦坦荡荡之心啊。但是呢，这个过程当中，有没有就有一些心怀歹念的人就埋伏下来了，这个事儿也不好说啊。按今天的局面看的话，确实当年也有一些人假装投诚，实际上是埋伏下来了啊。

啊，那么暴力哈，这种暴力手法并不好啊。直接的这种暴力的手法并不好。为什么？因为暴力可以解决肉体问题，却解决不了精神问题。一旦用纯粹的暴力手法去解决它的话，那就意味着我们在之后要不断地面对很多的人，说他们的好话，他们就变成了殉国者。真的，当你用纯粹暴力的手段去解决这个问题的时候，他就会变成殉国者。因为哪怕是八旗后裔，所有人都是恶贯满盈吗？里面总能挑出几个好人来的，一定能，你知道吗？即使是八旗群体，你也能挑出好人来的。然后一定会有人揪着这几个好人做文章，就把你这件事情彻底地毁掉，然后让他们变成历史上面板上钉钉的英雄。必然会这样操作的哈。所以为什么不能够直接进行物理性的摧毁？因为物理性的摧毁，最后就会把他们拱上神坛。这件事情在当初那个革命军杀入满城的时候，我们就践行过一次啊。辛亥革命的革命军当年就做过这件事情啊。当时因为辛亥革命如火如荼，所以有相当一部分的满城做出了这种给所有的女人和孩子发放武器、匕首，以实现全民皆兵，万有不带的时候啊，他就要干什么？要自裁殉国。当时他们是做了这样安排的。所以有很多的满城实际上殊死抵抗的同时，开始安排了这种自尽的手法。后来革命军是怎么面对这件事情的？把他们救下来，把这些所有试图自尽的人全部都救下来。干什么？明正典刑，公开枪毙。因为公开被惩戒，和他们在那里自己拉了，性质完全不一样。甚至有一些明正典刑之后，连枪毙的条件都不够的，关一关、罚一罚就给你轰出去当自由民。为什么？因为你程序正当、理由合理的时候，对方那就叫做贼子啊。这个定性是非常重要的啊。

嗯嗯，所以纯粹的暴力会把中间派赶到对面。是的，那个完全纯粹的这种暴力手段是不可取的。我们所有的这一些运用武力、运用暴力的时候，一定要有正当的名义，一定要有正当的目的。你绝对不能把它当成一个泄愤的工具来用。因为你只要这样子做了，就会让自己陷于不义。陷于不义这件事情你看上去不重要，但实际上它会直接影响我们自己内部的价值判断，最后影响我们内部的凝聚力。
