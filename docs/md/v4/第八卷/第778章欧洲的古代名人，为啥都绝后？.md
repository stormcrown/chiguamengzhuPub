# 第778章 欧洲的古代名人，为啥都绝后？

近年来，归国的技术工作者越来越多。这种局面出现后，你有没有发现，从这个时候开始，他们的科学技术研发进度开始出现各种停滞？但是，他们的PPT开发进度一点都没有停滞。那些创想、创意类的东西确实还在不断推陈出新，但实打实的成果——真正把东西做出来的东西——越来越少了。所以这咋回事呢？你难道没有一点关联性的联想吗？来整点关联性的联想吧：为什么排挤了华人华裔的科学研究者之后，他们的实验室就做不出真实的成绩，只剩下PPT了？

嗯，啊，没有牛顿，你还在挨饿？牛顿是种地的呀？你确定牛顿是种地的吗？你心里有数吗？挨饿这件事难道不是靠农民解决的吗？来，走起。请问你吃的到底是砸过牛顿的苹果，还是地里长出来的粮食？你总得选一样。我们靠的是农民同志解决的温饱问题，是种田啊。我们在明代的时候，亩产已经超过了400斤。明朝的亩产超400斤，而欧洲在工业革命前夕，它的亩产是200斤。你如果说我靠牛顿解决的吃饭问题，那就做个人吧，当个人吧。

西方为什么在中国办学？他们知道中国人聪明。明朝的科技用完了，科技就发展不了了，必须依赖中国人。嗯，现在确实有这个猜想——有这方面的猜想。为什么有这方面的猜想呢？多多少少还是有一点依据的。但你说实锤了吗？那确实没办法，实锤不了。这里头涉及几个问题：第一个，这些年以来，实际上已经有越来越多的人怀疑西方的科技史的来源了。为什么会这样怀疑？当然了，现在依然有很多人很急，一听说有人说这个，就说你民科，你思密达了，你宇宙中心了，你因为自卑所以骄傲了，不能够客观认知世界了。你们的词儿我都知道。我在这儿讲的是正常的人类行为的逻辑本身。那为什么今天的人会开始质疑西方的科技发展史？是因为当我们越来越了解西方的科技发展史，甚至西方人越来越多的科技发展小故事被铺天盖地传播到中国之后，中国人意识到了一件事情：这些了不起的伟大的科学基础成就，都是无根之木、无源之水。他的学者没有可证实的老师，他的伟大人物没有子嗣、后裔、传承。每一个绝顶的英才都如流星一样突然出现又突然消亡。出现的那一刻就璀璨万分，直接留下了无数的珍贵财富，但消失之后无人承接，一直到百年、几百年之后，甚至千年之后，才开始有人意识到他伟大的思想。从前我们在清代，技术极端落后，以至于中国人觉得只要别人有科技，就是了不起的，就是别人的。但今天的中国人不一样了。我们身上的那个枷锁已经开始松动了——满清数百年给我们捆上的科技枷锁开始松动了。中国人突然意识到一件事：西方人所吹嘘的这些由他们原创的伟大的科学技术，在他那里找不到根，在我们这里却能够找得到。因为意识到了这一点，就会有人不断去尝试挖掘。而在这个挖掘的过程当中，越来越察觉到不对，一定会察觉到不对，因为事儿它真不对。

这里头有几个很根本性的问题：目前而言，全世界最重要的这些科学技术的基本知识，整个系统、这些产物，基本都是建立在十进制数学的基础之上的。所以，你没有十进制，就不能说这个东西完全是你原创的。十进制实际上是非常非常重要的东西。那十进制打哪儿来？当你去找十进制的时候，就会发现一个东西：我们大家现在通用的阿拉伯数字，就是标准的十进制。那它打哪儿来？之前说阿三发明，阿拉伯人传播向全世界，于是它叫做阿拉伯数字。然而出现的问题是：在阿三那里，并没有一个长期延续的国家实体，也没有一个长期延续的、中间没有断裂的文明体系。同时，因为它内部种姓制度的横向切割，以及邦国制的纵向切割，所以它是一个非常粉碎的地区，从来没有形成一个完整的群体。在这些复杂的语言和文字系统当中，它就没有十进制。一个地方的语言当中没有十进制的时候，你说它发明了十进制并向全球推广，你哄鬼呢？你发明出来了一个这么好的数字系统，不应该给它起一个合适的名字，然后自己家先用起来吗？起一个合适的名字并且自己家先用起来，这就是进入到语言系统应用当中啊。中国人就会发现一件事：我们自己在念十进制的所有数字时特别便利。为啥呢？因为我们的语言、我们的文字就是围绕着十进制而生的。我们的九九乘法表在先秦时期就已经完成了，这就是算学基础已经开始进行普及化教育的一种表现——不是发明的表现，而是进入普及教育阶段的表现。先秦时期的时候，入门级的普及教育当中已经包含了九九乘法表这样的算术口诀系统了。

在这种时候，自然而然地就会产生一个疑问：阿拉伯数字哪来的？阿拉伯人不用十进制，欧洲人也不用十进制，阿三也不用十进制。这个世界上用十进制最娴熟的是谁呢？是中国人，是东亚文化圈。所以，当十进制是来自于中国的时候，出现的第二个问题就是：现代的各种各样的科学，绝大多数都建立在十进制的基础之上。如果说它的源头是来自于一个它的语言当中数字是五进制、十二进制的地区，难不成五进制不能开发科学吗？可以的，没有问题。五进制也能运算，五进制也可以开发科学，十二进制也可以。那些号称自己是文明源头的地区，它推行的数字、语言系统、符号系统是五进制和十二进制的，甚至还有一些更复杂的进制，咱们就先略过不提。当它自身的语言和文化系统是五进制、十二进制的时候，它开发出来的科学、技术、文化的算法，却居然都是遥远的异乡那个陌生的十进制？你开玩笑的吗？逗谁呢？你自己原生开发的，当然用你自己的算法，没问题的呀。五进制能算，八进制能算，十二进制能算，十六进制也能算，六十进制都能算，不是算不了。结果你开发科技的时候、推行算法的时候都是十进制，自己的语言、自己的文化传承、符号传承是五进制的、十二进制的，这是糊弄哪位呢？

嗯，啊，没有牛顿，你还在挨饿？牛顿是种地的呀？你确定吗？你心里头有这个数吗？挨饿这件事情不是靠农民来解决的吗？来，走起。请问你吃的到底是砸过牛顿的苹果，还是地里长出来的粮食？你总得选一样。我们靠的是农民同志解决的温饱问题，种田啊。我们在明代的时候亩产已经超过了400斤了，明朝的时候我们的亩产超过了400斤。欧洲工业革命前夕，咱这都不是同一个时间做的对比啊。欧洲在工业革命前夕，它的亩产200。你说我靠牛顿解决的吃饭问题，做个人吧，啊，当个人吧。吃饭这个事儿，他竟然跟我说，哎呀，没有牛顿——牛顿的力学系统我当年上学也是学的这个呀，谁上学学的不是这套力学呢？谁还没有听过苹果砸脑袋的小故事呢？人人都听过，我也知道。所以你说牛顿他的这套符号系统、他的这套公式有没有用？是不是真的？是不是现在我们所有的汽车、所有的盖房子都在用？没问题，可以，可以。继续再用这套符号，这一套的定理、这一套的守则都棒棒的，到现在都在用。可以，它是科学。但是这和这个科学的开发者是不是牛顿有什么关系吗？如果这一套科学系统不是牛顿开发的，是不是苹果就不会掉下来了？现在我们在讨论的是什么？现在讨论的是牛顿这个人，他学习的这个知识到底从何而来。他站在巨人的肩膀上，看世界的那个巨人到底是谁？他的老师是谁？他师承哪里？他之前学习的那些玩意儿是打哪儿来的？讨论的是这个问题的时候，结果你跟我说没有那个玩意儿，你吃得上饭吗？牛顿力学的存在，不代表牛顿的力学是牛顿发明的，这个事儿能理解吧？如果理解不了，那就没办法了。进修一下小学五年级阅读理解，我基本上是不会超纲的。我的表述水平就一般，保证小学五年级以上能够听懂。
