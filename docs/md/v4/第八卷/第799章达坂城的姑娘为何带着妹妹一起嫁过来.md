# 第799章 达坂城的姑娘为何带着妹妹一起嫁过来

喂，兄弟，剩下的才是你好不好，心里有点数啊。别动不动就说“哎呀，老祖宗的优秀传统没有了”——这套传统跟你没什么关系啊。“礼不下庶人”的庶人才是你呢。

什么加入天主教结婚不要彩礼，是真事儿吗？呃，别说是加入天主教结婚不要彩礼了，你要是入籍巴勒特的话，女方必须给嫁妆。女方不给嫁妆的话，你就可以不结，而且对方会哭着喊着给嫁妆。肿不肿？你就说肿不肿吧。

因为世界各地都有自己的婚俗，全球各个不同的地方有自己独有的婚俗。而且每一种婚俗都有它当地、当时、本土、特定环境以及特定生产力水平作为基石。一定是有经济基础，然后才形成的上层建筑啊，这个东西是客观存在的。所以你说信什么叫于是就不给彩礼啊，那其实是因为这一套规则在诞生的那一刻的时候，在它诞生的那个地方，女性处于纯粹的附庸地位。所以呢，是不需要给彩礼的，甚至对方需要给嫁妆。那么在这种女性纯粹的属于附庸地位的地方，因为两性之间的地位非常的不平等，男性的地位极高，而女性的地位极低，以至于造成一种最后的社会点评，就是所谓的“结婚是在养一个女人”。于是呢，这个女子如果想要谋求婚姻当中相对合理的地位，他就需要附赠一套嫁妆来证明自己并非是一个纯粹的附庸。这个是一个基本的原理啊。

所以呢，你会发现在有一些地方，女子需要附带巨额的嫁妆，而男子是不给彩礼的。这是产生于它自身特定的经济形态和文化形态的，不是说你信了什么或者不信什么，而是这套规则它诞生于什么样的环境之下啊。咱们一定要弄清楚这件事：就是一个规则它诞生是有它自己的经济基础在那里的。为什么这个世界上很多的地方都是女子要出嫁妆？这是因为在绝大多数的地区，女性的社会公共生产贡献较低，已经低到了一种必须需要嫁妆来进行补强，否则的话他活该不受尊敬的程度。到了这样的地步的时候，女子想要出嫁，想要获得一个夫家的奉养，想要获得正当的妻子的权益，就需要贴补嫁妆啊，这是一个补足条款。所以这个世界上有很多很多的地方，女子出嫁是必带嫁妆，但是没有彩礼的。

那么中国这里为什么古时候就有“聘仪”这样的东西存在啊？为什么会有聘仪？它其实代表着两个家族之间的联合。聘仪也好，后世常常被俗称为彩礼也好，性质上面不大一样，但是呢，它其实是一种变种。那么从一开始的聘礼上面，我们就可以看出来一个端倪：它其实是因为婚姻和两性之好，它是两个家族之间在相对平等的条件之下去谈合约的情况下，所产生的一种契约。在这种情况之下，它诞生了一种合约的定制条款：就是某一方（主要是合约的受益方，男方）他呢，需要提供一个诚意。他提供了诚意，然后对方呢，带上了流动资产（就是所谓的嫁妆）。然后两方，一方以固定资产入股，另一方以流动资产入股，合成一个小的生产单元——这是我们早期的婚姻的缔结的基本原理。在这种基本原理之下，你就能够理解为什么男方需要出聘仪：这是一个合作的要约，意思是我有诚意来缔结这项婚约；而对方呢，要带上他的流动资产（也就是嫁妆）然后来缔结这个婚约。这是因为双方都有缔结盟约的诚意，双方是相对来说平等的关系，才会出现这样的、特定的条款在里面。

而那种真正的男女两性之间完全不平等的条件，就会产生什么呢？啊，它叫做“带上你的嫁妆，带上你的妹妹啊，坐着马车来”。相对较为不平等的地区就是这样的：女子出嫁必须要带嫁妆，否则的话她会完全没有办法获得正常的婚约的地位。所以才会出现说，在现代社会也有相当多的地区是女子出嫁妆，而男子并不出彩礼的。彩礼这个东西啊，聘仪这个东西在我们诞生之初的时候，是一个男方的家庭的一种诚意保证金。

嗯，行，嗯，呵呵，得等一个月。诶，好的，嗯。

小姨子算什么地位？为什么小姨子这样的角色会显得比较特殊？其实在我们的先秦时期的婚礼当中也有这样的角色，它叫做“媵”。来，那个有知道这个字的小伙伴可以在公屏上面把这个字打出来啊，就是“媵”切的“媵”啊。大家可以在打字的时候用这种方式来进行拼音拼写啊，你可能比较容易找得着这个字儿：媵，妾的媵啊。她呢，这个“媵”原本就可以是她的姐妹——应是妻的姐妹，通常是他的庶出的妹妹。这就是那个“带着你的妹妹驾着马车来”，我们的先秦时期是有这样的风俗的。那为什么会带着媵来一起出嫁？媵是什么角色？媵是“副妻”啊，妻的副手。为什么要带上这样的一个角色？因为双方缔结婚姻之后啊，他是双方各自出资缔结了这样一场盟约，但是这个盟约当中包含了一个非常核心的条款：就是要产生继承人。但是这个继承人呢，说老实话，不保证一定会出现，尤其是当你要求说必须是男性继承人的时候，那真的不保证必然会出现这样的一个角色，它有运气的成分的。于是呢，它有一个保证的兜底的条款，就是“媵”。媵是什么？媵是一个妻的副手，干嘛呢？它相当于是说：如果这个妻没有诞下子嗣，那么这个媵就可以诞下子嗣；如果这个妻早亡，那么这个媵呢，就可以弥补中间我们缔结婚盟的这样的缺陷。所以在这种婚姻的合盟的过程当中，她不是只出一个女子，而是同时出几个。尤其是在西周的这种分封制的条件下，这种媵的角色有的时候不只是妻的妹妹啊，不只是妻的姊妹，她还会包括其他的诸侯国也会搭送一部分的贵女、侯女、侯爵等等的，这些贵族的女子过来担任。这个媵是干嘛？是来保证合约的完成啊。

“啥老祖宗好传统都没了，哭你呵呵”，你想多了啊，你想多了。我在说先秦的时候，我在说先秦风俗的时候，跟咱们大概大概率没什么关系。先秦我们能够找到的礼制相关的所有的规则，叫做什么？“礼不下庶人”。记得这句话啊，有知道怎么打字的，来给我们打出来：“刑不上大夫，礼不下庶人。”这句话你总是会打的吧。来，“礼不下庶人”的这个“礼”是什么呢？社会规则。这个礼当中就包括了婚——婚姻关系就是礼制当中的一部分。所以“礼不下庶人”，也就是先秦时期的婚姻，跟庶人、跟黔首没毛线关系。这是人家贵族家庭内部事件啊，贵族家庭内部的事由与黔首无关啊。不要在那里幻想说老祖宗的优秀传统没有了。你自己琢磨琢磨：如果说我们两家缔结一下婚盟，然后我们家所有的姑娘都归你一个人了，请问这个社会上会出现多少单身汉？你一听就知道这是在扯犊子好吗。虽然说在出生率上面，男性确实比女性略高，但是也就是高那么一个百分点而已啊，就高那么一丢丢而已啊。而且男性的死亡率还更高一些，实际上是一个约等于持平的状态。一个约等于持平的状态，然后一个男人要娶好多个女人，你觉得剩下的是谁？喂，兄弟，剩下的才是你好不好，心里有点数啊。别动不动就说哎呀老祖宗的优秀传统没有了，这套传统跟你没什么关系啊，“礼不下庶人”的庶人才是你呢。

“啥这句话没有被曲解过吗？”哪句话？“礼不下庶人”吗？那“礼不下庶人”肯定是被曲解过的啊，而且历朝历代都被曲解过。就是常常会觉得“礼不下庶人”是指不用对平民讲礼貌啊，有不少人会这样幻想。实际上当然不是的啊。礼是一种制度。礼这种制度呢，是诞生于贵族制度的，它是诞生于贵族制度的这种基础的。所以呢，“礼不下庶人”是指贵族的这一套规则不要运用于平民。而“刑”是什么？刑是干涉于平民的这些犯罪行为的，违法犯罪行为的。而这些行为呢，就不要上到大（注：应为“大”）夫级别——就是“士”这个级别，这些人呢，他就不用参照适用于平民的这一套法律。所以这是古时候在贵族政治前提之下所产生的一种制度性的分化。相当于是说，有一部分上层人士借由礼这样的制度来进行管理，而一部分的基层的人士呢，用刑这样的方式来进行管理。这是两套纲领啊。嗯，礼太繁琐，平民学不来啊，本来就是学不来的。我之前曾经做过关于“礼不下庶人”的相关视频，里面呢，也专门讲过这件事情。礼为什么能够、为什么要“不下庶人”？因为礼制当中大量的内容，庶人根本践行不了：你走到什么地方必须要用什么步法；你去到什么地方、有多少里程的远足，你必须要用什么样的车；然后去参与什么样的宴席，必须要穿什么样的衣裳，喝什么样的酒，你必须要配什么样的舞蹈——这些东西都是庶人根本做不到的事。而这些都是礼制当中的一部分。它在那个时代的贵族秩序当中非常的重要，它是来标称你的社会身份和社会分工的。但是这些东西在庶民阶层能用吗？不能用呀，亲。所以“礼不下庶人”，那确实是因为在当时的社会管理条件下面，它必须是双轨运行。
