# 第781章 汉服复兴会失败吗？如何锤炼思维言谈，如何进行中国文化的扩张

我们历史上是曾经敞开胸怀让他人学习的。但是学的是什么？很多人觉得我们什么都让人学，实际上我们过去啊，包括说日本那里派遣唐使等等的，这些各种各样的时代，我们去进行文化传播的时候，传递的都是一些“道”。什么是“道”呢？“道”呢，就是希望化蛮夷为文明，希望让他们知道丛林法则不是世间唯一的真理，希望他们知道如何懂得礼义廉耻。我们古典时期的最大目标其实就是这个：希望大家都能够遵守文明的规则，这样子呢，生活起来会更舒服一些。我们这种生产型的文明呢，也可以活得更稳妥一些。但是成功了吗？阶段性的成功啊，阶段性的成功。因为祸福相依，当你传递的信息出去的越多，培养出来的就有可能存在像表面服从而内心不从的人群。但是即便有这种风险，那做不做呢？依然还是要做。为什么？因为这是为了我们自己。每一次将真正的“道”传递出去，就更有可能让整个世界进入到有序，让世界进入到有序，为的是让我们这种生产型的文明更加能够成长。

（对话部分）

“喂，听见了吗？”
“啊，能听见。蒙祖老师。”
“诶，好的，你好，可以讲啦。”
“嗯，好。本来准备了，因为连听了这段时间的视频，本来准备了很多很长一段赞美之词，我看时间不早了，就总结一句话吧：就是我感觉，嗯，您说出了很多正义人士、有正气的、有文化的人想说而没有能说出来的话，压抑在心里的话都听到了。呃，我有三个问题。第一个问题就是经常听您说那个服装问题，衣冠文物嘛，肯定是文化的首要问题。我对这个问题呢，就是有感受，赞同您的所有观点，嗯，学习了很多。我想请问的是，现在咱们处于这个讨论和学习阶段，如果将来这个社会安定下来了，咱们国家这个大局稳定下来了，社会环境允许解决这个问题了，在盟主看，怎么样才能够逐步切实地、真正地解决这个问题？从设计和制度的这个层面，怎么样能透过服装的设计，实现华夏审美的普遍实现？”

“嗯，我第一个啊，第一个我觉得我们现在是安定的。第一件事情就是我们现在是安定的，然后也不存在说我们还需要怎么安定下来的问题。包括现在你认为说我们现在舆论场上面有问题，有没有呢？有。那为什么会有这个问题？我们目前在服装领域上面所存在的问题，是我们自身的古典的审美、我们传统的服装体系在受到某一种系统性的蚕食。而这个呢，并不是国家层面的问题，它目前而言是特定的群体在刻意地从中制造障碍。而什么时候能够解决这个问题，或者说如何解决这个问题？第一呢，就是在现代社会，我们说我们要设立自身的传统审美。在一般的国家里面啊，当它的国家内部的构成比较简单的时候，它会设立一整套的纲领啊，大多数的国家都是这样做的，就是它自己设计一套新的、传统的风尚。我们隔壁的，无论是岛国（如日本）还是半岛（如韩国）啊，都有这样的设计化的操作。而中国呢，体量庞大，在我们这里啊，只要能够做到一件事情，我们的华夏审美系统就能够回来——就是把我们的历史审美的视角转换成5000年视角。只要解决这个问题，其他的东西全部都迎刃而解。因为实际上现在对于我们的审美系统能够构成破坏的只有一个东西，就是满清审美。其他的啊，虽然说这些小蛮夷们特别喜欢裹挟各种各样的少数民族，但实际上，除了满清刻意引入的这一套西伯利亚和东欧审美之外，其他的所有少数民族，在过去的漫长数千年间，和我们之间已经达成了审美美学上面的高度一致。在满清之前，我们周边的包括各个部落也好、部族也好，他们所使用的服装与审美，都是向中华体系靠拢的，虽然有自己的某一些地方性的纹饰特点，但是整体的框架上面是完全向中华系统靠拢。不靠拢的只有满清，是倒行逆施。所以目前我们真正需要解决的，就这一个时代的问题。而这一个时代的问题好解决吗？其实很好解决，只要解决我们的历史断代法就可以了——到底我们是从什么时候开始衰落的？只要弄清楚了我们的历史断代法，从清开始衰落，后续的事情迎刃而解啊。他不需要说什么等到什么什么安定啊，我们已经安定了。之所以还有问题，是因为我们现阶段有某一些群体啊，裹挟着某一些力量，在试图反攻倒算，为满清的这个罪恶黑暗的时代洗白。解决掉这部分的历史的冗余，后续我们顺理成章地就会把我们的传统走向现代化哦。”

“嗯，第二个问题。因为本身我也在学习嘛，在网上就看这些新时代的、同类的爱国的历史学者。在这个同类的学者中，我感觉您是有格局、有立场，观点清晰，表述准确。虽然说起来简单，但实际达到——因为我也是一名基层教育工作者——我觉得是非常不容易的，能这么清晰准确地表达出来，起到这个对社会的正面作用，这是非常不容易的。我想请教您的是，您是怎么样达到这种造诣的？我觉得如果能分享出来，对于我们这些正在学习中的学人和后学，应该是有指引和启发作用的。”

“呃，有关于如何锤炼自己思维方法啊，我也讲了有几天了。那重点呢，就在我现在弹出的这套书里面。我们的史料的累积，你说我得通读二十四史啊，绝大多数人都做不到啊，事实上我也没有通读二十四史。谁吹这个牛呢，我都觉得他得是老学究才有资格吹这个牛。所以实际上说，我们吸纳的这些所谓的史料也好、庞杂的知识也好、浩如烟海的信息也好，这些呢，都只是素材。就像你炒菜的时候，它们是菜，但是最重要的是什么？最重要的是那个厨子。厨子是什么？是方法。而什么是方法？《毛选》里面就给大家提供了方法的问题——如何思考，如何看待世间的事物，如何选定自己的立场与站位，如何区分朋友与敌人，如何判断矛盾的转换变化，如何分辨不同的时间应该怎么做。在这套书当中其实都有。当然了，如果你只把它看作眼前这一篇文字的话，那又陷入了《毛选》最反对的本本主义，就是教条。我们看书不是为了把书背下来，而是为了学会书里面教给你的东西。而《毛选》这套书就是非常非常朴实，非常非常朴素，也非常非常简单，但是却披露了人世间的真理与真相：逻辑是怎样推演的，人类的社会是如何运行的，人和人之间的关系为什么那么莫测，都在这个书里面。所以如何锤炼自己的思维啊？先把这套书理清楚，甚至说你不需要全本都理清楚，一篇一篇地看，一篇一篇地想。当你哪一天想出来了，想明白了，你自己的逻辑思维能够形成一个闭环了，再去看其他的素材的时候，你才能够豁然开朗。看见白菜你知道要如何炒，这个是什么？这个就是方法论啊。要解决人世间的问题，要知道自己如何思考问题，如何看书，如何看电视，如何看新闻——不是看的多了就行，而是要学习如何看。这是什么？这是方法论，这是厨子在学习如何炒菜，这是在学习菜谱。而《毛选》就是这样一套书啊。如果说嫌这套书费劲的话，就看高中思想政治。呃，我能说吗？”
“嗯，请讲啊。毛选，就您说毛选，我倒是挺喜欢毛选的，嗯，这个中国出的、日本出的，买了好几种，经常看。因为之前是在军队服役的，所以经常看。但是看了，肯定是感觉好，但是平时也读历史书，但还是达不到您这样的造诣，这是不是跟天赋有关系了？”

“嗯，我不是专门进行这种教学的，所以我不能够确知每一个人的问题到底出在什么地方。我只知道，当一个人能够站在他自己的岗位上面好好工作的时候，已经完成了自己人生当中最重要的责任当中的绝大部分。剩下的东西有一些呢，你可以去教给更加专业、更加灵巧的人做。如果一定要训练自身的话，那就是反复地去想。很多的时候，教员呢，也会面临到难题，他也会反复地去揣摩。甚至于呢，你可以说他特别理解人类社会的运行，可是他并没有理解当年印度为什么要打过来。所以也不是所有的事情都能够立刻、现在就想明白的。可是我们总有一天会融会贯通，因为我们始终在开拓这条道路。我们的思想你要让它形成闭环，它是一个逐渐开通的过程。什么时候首尾相顾了，什么时候就通了。所以不是看的越多越好，而是想多了才好。你要把现实世界看到的东西，印证到理论之中去进行思考啊。就像你刷题的时候，不是说你做的题越多越好，而是你总得知道这个题应该用到哪个公式才行吧。”

“太好了。就同样的话，听您说就好像特别有感觉似的。嗯，再认真读一读毛选。嗯，我第三个问题是，为什么我们会这么关注这个历史主播的视频？然后我一推荐就很多人也都觉得好。我觉得这是一种这个时代的人心和社会的整体需求。所以我感觉现在这个文化处于复兴和筹备的阶段，而且在国际上来说，也处于国际竞争正激烈的时候。我的问题是，我感觉我们国家这样健康地发展下去，将来一定能取得世界的先进地位。就是有一天，如果我们的国家，我们中国人真的扬眉吐气了，在世界上取得了绝对的话语权，我们在安全性、国际地位、组织能力上都没有任何担忧了，到那个时候肯定有其他国家的民族自然会来跟我们请教，就像今天很多人到美国留学一样。如果他们来请教的时候，咱们储备了这么多，包括您在内今天都在做这么重要的工作、储备工作。如果到那个时候，人家真正谦虚地说‘我要跟你们中国人来学习了’，到那个时候我们应该怎么样？是不是能提前预计一下，到时候应该以什么样的方式和步骤来落实这种文化的对外传播，它才是有效的、对人类来说能起到真正的长治久安作用的？唉，我的问题不知道说清了没。”

“一个人呢，首先要立正自身，一定要先立足自身，然后才有向外辐射的问题。我们现阶段啊，对于自身的这个文化的体系，在建设这个过程还没有完成的时候，已经开始想我们将来要如何如何进行文化的扩张啊，以及说别人过来拜师的时候要如何如何。第一呢，我觉得想得有点早。第二呢，我们历史上是曾经敞开胸怀让他人学习的。但是学的是什么？很多人觉得我们什么都让人学，实际上我们过去啊，包括说日本那里派遣唐使等等这些各种各样的时代，我们去进行文化传播的时候，传递的都是一些‘道’。什么是‘道’呢？‘道’呢，就是希望化蛮夷为文明，希望让他们知道丛林法则不是世间唯一的真理，希望他们知道如何懂得礼义廉耻。我们古典时期的最大目标其实就是这个：希望大家都能够遵守文明的规则，这样子呢，生活起来会更舒服一些。我们这种生产型的文明呢，也可以活得更稳妥一些。但是成功了吗？阶段性的成功啊，阶段性的成功。因为祸福相依，当你传递的信息出去的越多，培养出来的就有可能存在像表面服从而内心不从的人群。但是即便有这种风险，那做不做呢？依然还是要做。为什么？因为这是为了我们自己。每一次将真正的‘道’传递出去，就更有可能让整个世界进入到有序，让世界进入到有序，为的是让我们这种生产型的文明更加能够成长。就不是为了其他的人，而是当整体上进入到有序循环的时候，我们就不会再像之前那样天天提心吊胆地防贼。当然，最终的效果啊，最终的效果在过去来讲，阶段性地取得过很多的成效，同时呢，也咽下过不少的苦果。将来等我们再一次地构建清楚自身的文化体系之后，才会存在说再一次地设立规则这件事情。而目前而言，我们不认为说中国已经开始设立国际秩序了，我们只是希望整个世界能够是有序的。实际上，无论是哪一种规则，中国人总是能够适应，总是能够运用，这个也是我们的天赋异禀。”
