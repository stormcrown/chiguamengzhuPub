# 第798章 跪族如何诞生，停滞的思想追不上发展的速度

唉，人民群众要求自尊自爱，和有一些人试图跪在地上之间的矛盾就会出现。当这个矛盾出现的时候，我们要解决矛盾。怎么解决呢？那就是实现正常的、公平的、有效的管理方法，而不是和稀泥或者偏帮一些实际上根本就不值得帮的对象。因为你要出现这个问题，你才会去解决这个问题。今天社会上会有这种乱象，我也承认，但是这不是中国弱，而是中国的强的速度——这个过程当中有一些人的思路没跟上，他还误以为应该讨好全世界所有的友邦，但是中国已经远远迈出了这个阶段。

鬼子的内部是有问题的啊。他在这个战争领域上面，为什么特别喜欢赌国运一把梭哈？为什么他老是喜欢用这样的方式？因为他没有办法持续性地作战。为什么没有办法持续性地作战？因为他不义啊。不义之战是没有办法持续下去的，坚持不下去。这种一把你可以用威逼利诱的方式——他们当年侵华这件事情，就是动用的威逼加利诱，出去抢一波，出去干一票。他就是用这样的方式裹挟着人出来的。但是当你的这个战争的本质是不义之战的时候，你就没有办法延续下去。你要知道，抢劫这件事情，只有在得手的那一刻，大家是团结的；抢劫这种事情只有在得手的那一刻，大家是团结的，其他的时候都会乱。所以我们为什么啊？所以我们为什么要那个啊，持久战？哎，啥？嗯嗯嗯，我们讲仁慈也没见得多团结，还是挨打。你不觉得我们这70几年已经走到了国际巅峰吗？你觉得历朝历代谁最仁慈？历朝历代共和国最仁慈。我们花了70几年的时间已经走到了巅峰，而我们的起点是什么？我们的起点是什么？是不是那个时候说我们积贫积弱，是不是说我们一无所有、一穷二白？那啥，嗯，还不算巅峰，很多中国人脑子还糊涂着呢，需要时间。人的思想的改变需要时间。虽然说我们技术上面起来了啊，技术上面起来了，生产上面起来了啊，包括在国际上面。现在他现在公屏上面还有人在那里喊，说我们现在还不是在挨打。谁敢打我们？现在国际上根本就没有人敢打我们。但是他想象当中，只要对方放一个狠话，那中国就是怂了，就是输了，就是不行了。为啥呢？因为他的思想没有跟得上中国进步的脚步。那为什么没有跟上呢？因为思想界总归是要比社会的现实滞后一点点的，这才是一个常态啊。嗯，啥内部是不是在挨打啊？我说了，思想领域会比客观的现实更加滞后。所以你为什么会看到那些奇怪的现象——那些从稀烂的地方来的二流子，在我们这里可以拽得二五八万的？因为思想界还滞后于现实的生活。我们的真实国际地位，和很多人在社会生活当中、参与社会生产的人的思想，甚至包括一些规则和潜规则，它现在不配套。因为我们进步的速度太快了，而很多的这些制定的规则也好、潜规则也好、习惯的行为方法也好、思想的方法也好，还停留在那种需要讨好所有老外的这个环境里面，它有滞后性。所以这个时候怎么办？不满意的你就说出来。当大家都不满意的时候，新的矛盾就出现了。这个什么矛盾呢？唉，人民群众要求自尊自爱，和有一些人试图跪在地上之间的矛盾就会出现。当这个矛盾出现的时候，我们要解决矛盾。怎么解决呢？那就是实现正常的、公平的、有效的管理方法，而不是和稀泥或者偏帮一些实际上根本就不值得帮的对象。因为你要出现这个问题，你才会去解决这个问题。今天社会上会有这种乱象，我也承认，但是这不是中国弱，而是中国的强的速度——这个过程当中有一些人的思路没跟上，他还误以为应该讨好全世界所有的友邦，但是中国已经远远迈出了这个阶段。

嗯，哎，好，欢迎大家来到直播间啊。啊，来来来，我们走一波毛选啊。我们在面临到现实生活当中的问题的时候，如何看？如何分析？如何思考它什么时候会解决？应该怎么解决？你就应该在这个书里面找答案。我们如何看待现象？而你需要对它进行利益方面的、层级方面的深刻的分析，需要对他的矛盾进行拆解。你知道矛盾的主要矛盾与次要矛盾，知道矛盾的主要方面与次要方面，知道哪两样东西之间可以构成一个矛盾。诶，你知道了这一些东西，就不会因为世间上出现了某些事件，于是呢，哎，你就说完了，天塌了，又跪了，起不来了。很多的时候啊，有些东西出现到了台面上，反而是事件快要解决的一个征兆。

呃，就这么说吧，前些年的时候我忘了哪一年，至少有10年了，曾经有一个案子是在某一个场合下面，一个老外啊，一个外国街溜子调戏一个中国女孩子，然后呢，有一个过路的中国男性，伸张正义，见义勇为啊，结果呢，那个白男啊，非常的暴力，把这个男人呢就……然后这个案子当然是按照杀人来处理的，但是呢，他蹲了一年牢房之后，就给遣送回国了，说是回国接着蹲啊。那回国了之后多半也蹲不了了。那这样的一起案子在处理这件事情上面啊，不存在一个程序上的根本问题。就是杀人虽然没有偿命啊，但是呢，确实判的是终身啊，这个刑罚呢，不算离谱。那这个东西的问题在哪？问题在于这个事件就是这个人被移送回国的时候，有一个新闻报道，而这个新闻报道上面说：哎呀，这个高大英俊的男子，看上去气质如何如何啊，他很高兴他获得了什么什么什么东西。就是这样的一篇报道，而且是官方报道啊，很正式的官方报道。这是10年前的光景。我就问你，今天的这些官媒也好，准官媒也好，敢不敢写这样的文章？大家都觉得今天烂透了的时候，你再去和这个对照一下，今天的官媒与准官媒敢不敢写这样的文章？在面对这样的穷凶极恶的匪徒的时候说：高大的英俊的外国男士，优雅绅士地拽着手铐上了飞机？今天的不敢了吧。为什么？因为大家都不同意。因为时代变了，风向变了。虽然那些脑子有点问题的人，依然待在岗位上面，影影绰绰地想要恶心大家，但是他已经不敢像10年前一样地明目张胆地把一个杀人犯叫做高大英俊了。好，欢迎大家来到直播间啊。

所以事情我们的目的是解决它的时候，当我们的目的是解决它的时候，你可以提意见。提的是什么？提的是意见事。这件事情你办的不对，这件事情办的不公，这件事情办错了，而不是在那里说天塌了，跪着吧，啊，现在还是那样到处挨打。提的是正面的解决意见吗？想要解决问题吗？还是想要围观的人认为今日之中国依然软弱无能？说话的时候讲究自己的目的，自己的动机。我要实现什么？我到底要解决什么矛盾？是不是？而不是跑到我的直播间来，唉，说其实我们现在还是是如何如何，还是到处挨揍啊，别人都瞧不起我们。你的目的就是为了让别人相信大家都瞧不起咱吗？不是吧，目的不是解决问题吗？解决问题应该说什么？办这件事情的人有问题他办错了，正确的应该如何处理？该喷的喷，该怼的怼是吧，这叫做正面出击。解决问题的核心，而不是丧里丧气地说：哎呀，完犊子了，挨揍了。丧气给谁看呢？不是要解决问题的吗？人民群众当家做主，不就是让你做主来解决问题、提出问题的吗？哎，好，欢迎大家来到直播间啊。
