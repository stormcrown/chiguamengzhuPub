# 第791章 萝莉岛的证人自杀，想起了张纯如老师

人生的哲学三问，在遇到他者的时候是最容易萌发的。而在张纯如老师身上，就爆发出来了。她突然想起来“我是谁”，然后想“我之前经历了什么”，“我为什么在这里”，“我身边的人有没有跟我一样的情感”。最终她发现：我是一个中国人，而别人甚至不知道中国人受过什么苦，他们也不在乎。这样的一种情感的冲击，使得她内心十分的愤懑。她想要找到真相：明明在战争当中，中国人受到了巨大的伤害，但是无人在意。哪怕你叫出来了，哪怕你提出来，他也说：“这有什么关系呢？与我何干。”对，与他人何干呢？与他人何干，那就必然与我有关。所以张纯如老师才会在这个时候，去创作这样的一部著作。她想要找到一个问题的答案：我是谁？我的祖先经历了什么？而所有的这些经历，到底与我之间有什么样的关系？

嗯，南京是——嗯，张纯如老师的南京啊。哎，我的这个南京呢，是张纯如老师的那个版本。嗯，一下缅怀张纯如老师啊，对对对，啊，是的。嗯，张纯如老师确实非常的不容易啊，呃，不是一般的不容易。最重要的一点的前提呢，就是她其实甚至不是生活在中国的，她甚至没有生活在中国。但是说老实话，在我个人的这种对于人的行为分析上面来说，恰恰好就是因为她生活在一个周边都是他者、周边都是外人的环境下，于是她特别的有自我意识。这个东西呢，我之前曾经分析过：就是一个人，他怎么能够想起来一个问题叫做“我是谁”？一个人什么时候会提这个问题？他什么时候能够想起来，诶，我是谁呢？我是什么样的特点呢？我应该干什么呢？我应该穿什么呢？我应该说什么呢？我有什么特色呢？我和别人有什么不一样呢？这种问题是什么？这种问题就是自我意识，这就叫做自我意识，自我认知。而这种自我认知在什么时候会出现？在有别人在场的时候会出现。这在全世界范围内都是通用的啊，这是人类的行为，这不是中国人的行为。人类的行为——就是有外人在场的时候，你知道你是谁。有父母在场的时候，你会意识到自己是孩子；有客人在场的时候，你知道自己是张家人啊——当然，随便你姓什么啊，我只是举个例子——呵呵，有客人在场的时候，你就知道“我是老张家的人”。然后呢，唉，如果有其他省份的人在场的时候，你想起来了，“我是河南人”是吧？那是有外国人在场的时候，你会想起来“我是中国人”；有其他人种在场的时候，你会想起来“我是东亚人种”。这是什么？这就是我的自我意识，取决于我有没有察觉到别人的存在。我意识到他者的存在，就会开始提问：我是谁？我和他有什么区别？我和他有什么共同点？

所以像张纯如老师这样的，她没有生活在中国，她的身边会充斥着各种各样与她不同的人，于是她反而更加容易意识到一件事情：我是谁呢？这是海外华人群体当中非常普遍的一个现象。她越是离开，越是意识得到一件事：我不太一样。我是谁？我从哪儿来？我要往哪去？人生的哲学三问，在遇到他者的时候是最容易萌发的。而在张纯如老师身上，就爆发出来了。她突然想起来“我是谁”，然后想“我之前经历了什么”，“我为什么在这里”，“我身边的人有没有跟我一样的情感”。最终她发现我是一个中国人，而别人甚至不知道中国人受过什么苦，他们也不在乎。这样的一种情感的冲击，使得她内心十分的愤懑。她想要找到真相：明明在战争当中，中国人受到了巨大的伤害，但是无人在意。哪怕你叫出来了，哪怕你提出来，他也说：“这有什么关系呢？与我何干。”对，与他人何干呢？与他人何干，那就必然与我有关。所以张纯如老师才会在这个时候，去创作这样的一部著作。她想要找到一个问题的答案：我是谁？我的祖先经历了什么？而所有的这些经历，到底与我之间有什么样的关系？这就是她为什么会写这样的一部书。而当她把这部书写出来了之后，遭遇到的种种的困境与舆论上的围剿，让她越发的清晰地意识得到：我是一个中国人，我的祖先是一个受害者，而加害方还好好的活着，甚至想要让我忘记这些伤害。

所以这套书不仅仅是一个历史方面的著作，它也是一个人类行为心态上面的著作。人在什么时候会意识到我与他的不同？什么时候会意识到如何印证自我的存在？什么时候会知道族群的存在和我之间有必然的关联？很多的时候，他需要外来的力量。张纯如因为生活在海外，所以她意识到了这一点。而今天的中国人，因为国际化的大潮流，因为互联网的扁平化而意识到了这一点。所以越来越多的年轻人开始做出这种提问：我是谁？我从哪里来？你为什么这样对待我？我遭遇过什么？我应该获得怎样的待遇？而为什么今天没有？这些思想都是共通的。所以今天的中国人也可以看这部书，在看到它的时候，应该会有一些很独特的情感上面的共鸣。至少说，当我们看这些书的时候，会知道：侵略者来的时候，我应该怎样想，我应该怎样看；侵略者会怎样表现；完全与我不同的人会如何看待我，又会如何对待我。一个人如何成为一个人？作为一个个体，你将永远是一个人吗？还是你必须依从于某一个庞大的群体，你才能够有资格称之为一个人？很多的时候，确实需要有一些人处在特殊的处境之下，他才会察觉到这个问题。而张纯如恰恰好她呆在了那个位置上面，于是她发现了问题之所在，找到了这样的一个时间的节点，让我们来回看一下南京的惨痛，看到那个时间与那个时代的祖先，与那个时代的同胞共鸣。然后我们知道：我是谁？他是谁？我应该干什么？她又在做什么？不要为眼前的迷障，不要为眼前的繁华感觉到迷惑，很多的事情是有根本利益之所在的。

所以，其实我一直都在盛情推荐这套书。所有的中国人的家里面都应该有一套《南京》。你得知道真正的侵略者来的时候，真正的马蹄踏过来的时候，什么道理都是无力的，没有人来听到。那个时候你才会意识到什么是自己人，什么是别人，明白吗？嗯，不是全称啊，主要是我不能够把全称直接念出来，很容易把直播间弄没有啊。你就说多荒唐是吧？你就说多荒唐吧。我可以说作者的名字，也可以说书的简称，但是书的全称我直接说的话，直播间容易没有。嗯，好的。嗯，就是确实有很多人，因为呢，从小就看动漫，看各种各样的漫画书，然后看电影、电视剧，可能呢，还很喜欢那边的一些女明星什么的。这些事儿呢，我从小到大看的都不少，我知道有很多人很迷恋那些方面的细枝末节的文化，于是呢，就老是觉得事情过去了就过去了，甚至呢，就直接一个敷衍，说一切都是阶级矛盾啊，就直接敷衍过去了。说这个事儿呢，反正就这样，啥时候都有。但是不一样就是不一样啊。怎么能够了解？怎么能够直观地知道这件事情不一样？你得看到当时到底发生了什么，你得知道对方到底在干什么，你得知道当对方是一个压迫者的时候会做什么，以及对方是一个敌人的时候会做什么。否则的话，你觉得他们都是一样的，其实不同，完全不同。

南京之后，岛国上面的各路媒体不断地宣告说，我们南京如何如何取得胜利，然后整个岛国万人空巷，出来举国同欢，庆贺什么？庆贺侵蚀南京成功。参与庆贺的人当中，什么人都有，不是什么上层阶级，没有阶级的问题，也没有性别的问题。不分男女老幼，不分职业贵贱，不分阶层的高低，他们举国同欢。为什么？因为在那个时候，他们将自己视为一体，而将整个中国的上下视为他者，视为别人。只要抢赢了，他就赚了。所以那些人当时理直气壮地在自己的街头张灯结彩。这个时候，你不能够再去欺骗自己，说一切都是阶级矛盾——没有的。在那个时候，脚盆岛上面的乞丐都会庆贺这件事情，但是中国这里即使是最不像样子的军阀也会为之而悲泣。所以将一切事情都刻板化、扁平化的时候，你会没有办法正确地理解历史事件的发生原因。当我们没有办法正确理解原因的时候，你就没有办法以史为鉴。历史的进程确实走过去了，但是记忆还在，该做什么，该怎么看待问题，这些东西还存在着。不能够眼睛一闭，就说这事已经过去了，一切都是阶级矛盾嘛。总之呢，我们要反抗一切阶级压迫。口号喊的很好听，但问题是它是真相吗？当你已经不唯物的时候，你喊那么多政治正确的口号有意义吗？
