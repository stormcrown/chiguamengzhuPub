# 第759章 从古代婚姻制度看中西方契约精神的差异

嗯，欢迎大家来到直播间。今天我们来谈谈“七出”。首先，“七出”是古代婚姻制度中的条款，经常被拿来讨论是否歧视女性或反映女性社会地位低下。但“七出”应该和“三不去”以及“义绝”放在一起讨论，它们是一个套系，而不是单独的片段。只讲片段是不够的，必须把七出、三不去和义绝放在一起来谈，这些都是古人离婚的几个条款。

“七出”就是有七种情况可以休妻。这七种情况包括无子、嫉妒等，我一下子背不全了，你们可以帮我背一遍。这七种休妻的条款，还有补充条款，就是“三不去”——三种情况无论怎样都不能休妻。所以我要把它们放到一起来讲。

比如经常被人诟病的嫉妒、无子，请问为什么会这样立规矩？我在上一期视频里也说过，古人的婚姻和今天所谓的“你爱我、我爱你”不一样。古人的结婚叫做“合两姓之好”，意思是两家各自出一笔股本，设立一个子公司，这是一个产业。家庭就是一个产业，一门生意。在这种时候，如果妻子没有生孩子（无子），这是常见的事情，可能是身体不好或其他原因。但“无子”不是指妻子没生，而是这个家庭没孩子。同时，妻子没有协助纳妾。很多人对纳妾有意见，但家有产业的情况下，纳妾有严格规定，比如年龄限制——有些时期规定40岁无子方可纳妾，还有职级规定，一定职级可以纳几个。不是随便就能讨18房姨太太，那种通常违法乱纪（民国时期例外）。正常纳妾是为了让家庭后继有人。

“无子”就是让家庭后继无人。我们重复一下：古人设立家庭是为了设定一个生产单元，要求产业欣欣向荣且后继有人，这是产业目标。结果妻子自己没生，还禁止纳妾，不让别人生孩子，导致产业无法延续。这是婚姻合同的违约行为。不要误以为是因为不孕不育就把人踢走，不是这个道理。“无子”不是妻子没生孩子，而是妻子没让这个家里有孩子。人家结婚的目的就是要家里后继有人，结果违约了。古人的婚姻是契约婚，是两个家庭谈生意、立产业，有立项标准、KPI考评，包括家庭和睦、子孙兴旺等。一旦涉及无子，就是违约行为，所以可以休妻——这是解除合同。这样就能理解为什么会有这种规定，它不是性别问题，而是为了维持家庭和社会契约的运转。

如果纳了妾还是没孩子，别人会说是男人不行。古人很聪明，知道可能男人有问题。如果纳妾解决不了，就会过继收养，优先过继同宗共祖的孩子。所以“无子”范围很宽，包括本人生育、纳妾生育、过继子女，核心是让家庭合约兑现。

关于嫉妒：嫉妒代表家里出现口舌是非、家庭不和睦。古人讲“合两姓之好”，就是双方好好合作。契约婚的前提是两家和睦共处，这也是合约条款。如果因嫉妒生出口舌是非，导致两家不睦，就是违约行为，可以解除合同。

真正让我觉得不公的一个条款是“恶疾”。当然，恶疾限定很多，比如严重的不治之症，尤其有传染性的。本质是妻子无法履行职责，需要换人。但因为生病就休妻确实没良心，所以有“三不去”补充条款。

“三不去”有三种情况不能休妻：第一，“有所娶无所归”——娶她时有娘家，但之后娘家没了（父母兄弟不在了），休妻则无家可归，这种情况不能休。第二，结婚时贫贱，后来飞黄腾达，休妻再娶不行，这叫“糟糠之妻不下堂”。第三，“与更三年丧”——妻子为公婆守孝三年，已尽到当家主母的重要贡献，已经奉养老人，不能休妻。所以“三不去”是限制条款。

无论是七出还是三不去，都是婚姻中的限制条款。任何时候休妻必须有清楚理由，理由不清晰不能休。这是在保护妻方的权益。即使符合七出，但如果同时满足三不去，就不能休。这是限制休妻，而不是可以任意休妻。否则强势方随意休妻，婚姻稳固性下降。古人强调契约婚，要保证契约履行，附条件才能解除条款。这就是七出与三不去。

离婚还有一种方式叫“合离”，即两家协商解除契约，比较和睦。另一种是“义绝”，当有重大人身伤害行为，两家结仇、成了仇家，这时恩断义绝，可以判离婚。

这一套规则是为了维护婚姻这样的社会结构。设规矩的目的是维护婚姻存在，因为古代家庭是社会基层生产单元，稳定生产单元才能保证社会稳定。所以严密的约束是为了防止今天结、明天离，生产单元不稳定，国家运行也不稳定。规则越清晰，动手脚余地越小。

现在我们的婚姻保持了大致外观相似，但内核不同了。现在是自由婚恋的“看对眼”，核心是感情存在与否。司法判例强调感情有无破裂。而古人不问感情，只看当初婚姻目标是否实现，双方是否负责任，像合同执行。感情参照有，但权重不高，不会因感情散了就离婚，因为古人认为感情容易散。

婚姻不是男性欠女性的。婚姻诞生前，女性也要自己养家糊口。哪怕到几十年前，仍有女性借各种理由要求男人负责。比如“你得为我负责”——牵手、发生关系、有娃等，大家应该听过。这不是欠不欠的问题，而是当女性要建立家庭养育子女时，有男人搭把手成本完全不同，双方可以达成合作。现在这种情况少了，是因为经济形态和思维改变。但往前倒20年，这种话语铺天盖地。现在的变化是经济基础与上层建筑不匹配的表现，是阶段动荡。新规则还未制定出来，所以出现混乱。动荡期过去后，形成新规范就能稳定。不存在男人欠女人或女人欠男人，本质是合作关系。而且相当一部分男性是被强行裹挟进来的，因为直接跑路不负责任太容易。过去的道德规范和社会评价都是在约束男性要为家庭负责，否则他真跑得很快。
