# 第757章 五胡乱华和满清入关的区别有哪些？ #五胡乱华

五胡之所以能在中原闹出大乱子，根本原因在于他们早就已经住在这里了。所以，这不叫“五胡入华”，也不叫“来华”——你换什么词都不好使，因为他们早就来了。他们是怎么来的呢？是作为外来务工者来的，包括雇佣兵，也包括过来帮忙种田的，各行各业都有。五胡乱华最终选择了汉化，并把汉化视为唯一的现代化升级换代方式。但唐朝灭亡之后，那些自认为已经学会了本领的群体开始构建自己的文明管理系统，这实际上是一种去汉化的表现。他们甚至大量运用汉人、地主、官僚提供的策略，表面上显得非常汉化，动不动就讲“以孝治天下”“以儒学治天下”。可是，你说他们尊崇孔夫子，却在画像上给孔夫子剃发易服——大家见过孔夫子头顶小辫子的图片吗？清朝时，连孔夫子都要留“金钱鼠尾”发型。

那啥，你们要改名“吃菜”也行，盟主们加油吧。我现在已经干涉不了了，每天看你们的ID都紧张，不知道该怎么念——左一个右一个改出来的ID，我念完怕直播间都没了。辛苦你们了，都不容易。

嗯，说到五胡乱华和满清入关对中国影响的对比，这两个时代有本质上的不同。这个本质上的不同要放到唐朝来说。大家喜欢盛唐吗？有没有人很喜欢盛唐？喜欢的扣个一，我这儿滚滚屏——不然平台老觉得我不认真当主播。喜欢大唐的扣一，不喜欢的扣二。来，我们瞅一瞅。

五胡乱华之所以和后来的胡人政权不一样，最重要的转折点就在唐朝，这是没办法的事。唐朝以前，五胡乱华的表现为何与后来不同？要知道，五胡是什么时候进入中原的？现在有些学者爱说“五胡入华”“少数民族南迁”这些所谓的中性词，其实一点都不准，不符合历史条件。当时五胡是以什么身份、在什么时间到达中原的？他们最早是在汉代给汉朝打工的外来务工人员。五胡之所以能在中原闹事，根本原因就是他们早就在这里了，所以不是“来华”“入华”——你写什么词都不好使。他们是怎么来的？就是外来务工者，包括雇佣兵、帮忙种田的，各种职业都有。整体上，他们就是外来的新移民、外来打工人员。这就决定了一件事：作为新移民，他们对汉人有一种天然的仰望视角。因为他们来打工时，汉王朝极度鼎盛，哪怕在灭亡时都极为强盛，甚至三分天下时，每一家都还很强大，还能扩张。这种强大的威压造成了什么样的心理变化？就好比现在有些人走线到美国，把美国当成天朝上国，幻想融入当地。当年五胡面对汉朝时，就是这种心态。这种心态很重要——他们自认落后，视汉为先进。这种崇拜慕强的心理，最终决定了五胡乱华发生时，他们只是趁势找机会打抢一番，相当于一次大规模的集体零元购。本质上，这是一次带有极度破坏性的零元购行为。为什么与后世不同？因为搞零元购时，他们仍然自认落后、视对方先进，心理落差没变。于是，即便北方汉人（无论豪强还是平民）在五胡乱华早期处于战败方，军事武力上弱势，但心理上依然强势：我才是先进的，你们不过是打杂的。这种心理落差导致他们破坏了生产和土地，但没有破坏文化的本体。文化的本体除了用什么字、说什么话、写什么文章，更核心的是：我相信自己的话、自己的字、自己的衣服特别好。这种自信和自我认知是文化最核心的要素。五胡乱华期间，中国人没有失去这个。尽管那时很乱、遍地无人、很惨淡，甚至有人被炖了，但心理优越感和文化碾压意识还在。所有人——不论是汉人还是胡人——都公认谁先进、谁落后。在这种情形下，对方只是趁势劫掠，他们自己清楚，我们也清楚。文化上、先进性上没有落败，败的只是军事。

但唐朝之后，这个局面变了。唐轰然倒塌后，崛起了一群军事割据群体，他们有很强的自我认知，同时又借盛唐的声势，自称盛唐传人。唐亡后北方建立的几个胡人政权，心态与当年的五胡完全不一样。当年五胡走到末期，一个接一个主动汉化，因为他们认为汉化是先进、现代化的选项。但唐朝之后，心态变了。唐末崛起的胡人部落有极强的心理优越感，觉得自己学会了“秘术”“屠龙术”。怎么学的？从盛唐学来的。盛唐急速扩张时，在新开发的土地上直接授权管理，让对方学到了部分文化、管理学甚至军事技巧。他们真的学到了一些有用的东西，并建立了自己的心理基本盘。于是你会发现，唐之后崛起的胡人政权会干一件事：创造自己的文字。尽管原来没有，但他们借鉴汉字，拆胳膊去腿地创制独有的文字系统。这是干什么？这是去汉化的表现，与当年的五胡乱华完全不同。五胡乱华最终选择汉化，视为唯一的现代化升级换代方式；但唐灭亡后，这些自认为“也行”的群体开始构建自己的文明管理系统，形成了强烈自我意识。后来，契丹、女真、西夏等群体设立政权后，都在自己管辖的地区推行了剃发易服——剃发易服就是从唐灭亡后才开始的。因为唐朝的扩张带来了一个衍生品：周边群体拥有了强悍的自我意识，要形成自己的文化体系。当他们知道汉人非常看重衣冠、而衣冠对统治很重要时，就设立自己的服装形貌标准版本，要求别人践行，这就是剃发易服。辽金都推行过，只是时间不够长、深度不够深，但确实推动了。这意味着形成了鲜明自我认知，并且很骄傲。这种心态在唐以后才诞生，一旦产生就难以消弭，影响力深远。在此之前，胡人都比较自卑；但唐以后，他们开始骄傲了。这种骄傲带来更强的进攻力，设立政权后更能保有自我特征、自我意识，甚至刻意去除汉文化影响。唐以后的胡人政权都有这个特点，只是程度深浅不同。

李二凤（李世民）的锅确实与他有关，实实在在有关。后来他后悔了，以他的能力也能解决问题，但坑已经挖下了。我之前直播聊过：李世民在大规模开疆拓土后产生野心，觉得自己可以做天下的天子，幻想“天下尽是朕的子民，朕独爱之如一”。这种心态下，他做出了一个非常疑惑的万年选项：带动胡人内迁，整部落整部落地向内部迁徙，分地种、安排岗位。但他让他们聚落生存，没有直接要求他们怎么生活、怎么说话，只是整聚落内迁。这是李世民早期的政策。后来他遭遇叛变，后悔了。但后悔有用吗？能把迁进来的人轰走吗？不能。除非大杀四方、下黑手，否则赶不走——进来容易出去难。所以后世的乱象，李世民确实要负一部分责任。虽然他在位时没有爆发，他的文治武功足以形成威慑，但这些成规模、成建制的内迁埋下了导火索。一旦皇朝内部管理出问题，有自我意识、随时想取而代之的群体就近在咫尺，甚至待在咽喉要地。这和仇家在关外不一样，等于有野心的人睡在床榻旁边，一旦动手，回天乏力。所以李世民在这件事上确实有责任。

嗯，好的好的，欢迎大家来到直播间。刚才的议题讲到了辽金的剃发易服。唐朝这种特殊的、不合时宜的政策埋下了漫长的雷导火索，到唐末轰然倒塌后爆发出来。这时再出现的胡人政权有非常强烈的自我认知和自我意识，觉得自己了不起，于是构造自己的文化系统、文字系统、制度系统。尽管参照了一部分中原文明，但参照的方法与晚清“师夷长技以制夷”是同一个道理——不是汉化，不是融入，不是融合，而是学了你的方子想把你弄掉。从唐以后诞生的胡人政权与之前完全不是一个类型，学习是为了学习技巧和术，回来反杀。这种行为走到满清入关时，就变成了一个结果。满清是之前所有这类政权的集大成者。我从未蔑视过他们，从未说过他们愚蠢——他们非常聪明，非常狡诈。他们运用了之前各种胡人政权用过的所有手段，从中挑选出所有对自己有益的方法归拢使用，所以是集大成者。你看上去，现在很多学者爱说满清汉化最深——其实根本没有。整个清王朝第一官方文字是满文，第一官方语言是满语。所有官员满为正、汉为负，岗位满人为核心、汉人为边缘，这种逐层分野、种姓隔离式的管理方法，之前任何政权都没做到这样极致。即使元的四等人制，也没有隔离得这么厉害。清朝是历朝历代所有胡人政权“师汉长技以制汉”手法的极致政治。他们运用大量汉人、地主、官僚提供的策略，表现得非常汉化，动不动以孝治天下、以儒学治天下。但你说他们尊崇孔夫子，却在画像上给孔夫子剃发易服——孔夫子头顶小辫子的图片大家看过吗？清代连孔夫子都要“金钱鼠尾”。这种管理方式是穿越时空式的极致，借由所有能学到和想到的方式来进行隔离管理。
