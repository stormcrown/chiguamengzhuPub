# 第446章 青粉为了洗倭寇，找的角度清奇的让人想笑

今天我看了看《倭寇图卷》才明白，所谓“倭寇里面中国人居多”其实是个谎言。那个事儿是在说什么呢？你知道吗？就是关于倭寇的数量以及倭寇的人员占比问题。实际上，它讲的是给皇帝的报告——为什么要强调报告中的差别？因为他在向皇帝报告有多少倭寇的时候，那些倭寇不一定是真倭寇，里面有相当一部分是监守自盗的。为啥要这样说？这就不得不扯到最早的时候：第一笔买卖就是郑和相关的这一组团队做的早期出海跨海贸易。这笔贸易在后世一直被说成是单纯的耀武扬威、亏本赚吆喝。注意了，这笔贸易特别赚钱！特别赚钱！当年的郑和团队最初搞对外海贸时特别赚钱，赚到什么程度呢？不能说它是五本，但起码是一本万利——可以做到这种程度。

为什么能这样？首先一个大前提：今天的很多人不相信或不认可，但迟早你一定会开始认可——明王朝是当时世界上唯一的工业国。你爱承认不承认，爱相信不相信，迟早有一天你会看到相关的东西铺天盖地地朝你砸过来。你现在觉得这是暴论也好，我无所谓。总之当时的明王朝是当时世界上唯一一个工业国，唯一拥有工业产品的国家。有工业产品代表它有生产力的代差：我们生产的这些东西别处没有，而且是工业品，不是什么矿藏、农业产品或你看一眼也能学的东西——不是手工业，是标准的工业化产品。比如丝绸——注意了，丝绸是工业品！它里面的每一个流程都是工业流程。此外，我们所有对外海贸中都有非常核心的产出，比如铁锅、铁钉、针这些东西，都是标准的工业品。没有工业化的加工流程、没有高炉冶炼，你是拿不到的，做不出来。但这玩意儿特别有用。所以明王朝早期的对外贸易卖的是瓷盘子、铁锅等对方无论如何都做不出来的工业品。那结算用什么呢？第一个项目是我们有工业产品；第二个项目就是结算货币也很重要，还有定价权的问题。今天（美国）已经不是这样了，但之前几十年，美元作为国际通行货币，只要进入国际贸易市场，美国就能赚得盆满钵满，因为定价权在他手上——这个东西值多少美元，他自己说了算。印刷机开足，爱印多少印多少，只要找出理由就能印，负债也可以是一种理由。所以美元印得满世界都是，这是它获利非常重要的渠道。

明初的时候，大明宝钞就是一个国家主权信用货币，它也是一个印刷产品，和你想象的白银这种通货不一样。白银要考虑矿藏，中国的白银矿藏不够，但我们的印刷业发达，有防伪标签。所以当时使用大明宝钞进行货币结算，又是赚第二道。你说赚了没有？铁铁的赚，而且赚得特别大。既然这么赚，为什么后世都说它劳民伤财呢？因为赚到哪去了？转到皇帝的私库里面去了！这是最重要的。要知道，这钱要是赚进了国库，那些文臣武将们高低的还能摸上一把——只要我在中间跑流程沾上边了，我都能赚到一点。结果它转到皇帝的私库里面去了，然后皇帝拿这个干什么了？充了军费！注意，皇帝拿这个钱充了军费、军饷，培养皇帝自己的力量，皇帝的话语权就会特别强。枪杆子里出政权，没有这句话，但朴素的逻辑认知就是这样。所以明初的时候，皇帝的权重非常强，权重从哪来的？因为足兵足饷啊，军费是他花的。不然你以为朱棣动不动就征漠北，“打他、打他、打他”，兵马未动粮草先行，他是为了几张羊皮打出去的吗？他要自掏腰包才能打出去。不管有什么战略价值，没有军费、军资，你都出不去。朱棣能够不断征漠北，根本原因就是郑和团队赚得很大，他才能干这个事儿。

注意，真倭寇一直都存在，不要以为有假倭寇就把真倭寇说得无罪。倭寇是一直都有的。但是我们现在讲前期的郑和团队——不是郑和一个人，我一直强调郑和团队，不是在说郑和本人，他啥时候出生、啥时候出门不重要，而是相关的整个团队在做这门买卖。实际上他也不光下西洋，还有下东洋的，干买卖嘛，哪边有生意就往哪边去。所以他的团队分了批次、分了方向、看了航道，人家是出门做生意赚外快，赚完了回来贴补皇帝的内库。皇帝的内库对哪儿开放？对军事用途开放。这，就是明代前期的局面。

但后来发生了改变，一个转折点就是土木堡之变，从此之后君王基本上失去了对军队的直接掌控权。然后紧跟着就能看到，郑和变成了一个劳民伤财的角色。之后寻到了当时“三杨”这些老臣、重臣、国之重器们，表示之前郑和劳民伤财，国家撑不起，所以不许干了，就把这个项目取缔了。项目取缔了，外贸取缔了吗？能够赚这么大发的外贸，你觉得谁舍得这块肥肉？只是他们看不下去皇帝手握这块肥肉，而想要放到自己手上来摸一摸油水。于是取缔了皇家外贸之后，变成了民间各种外贸。海贸没有停，一直没停，包括半官方海贸和私营海贸，一直没有停过。但在这个过程中，因为有各色人等的参与，我就请问你：既然你把官窑的东西拿出去卖换外汇，怎么赚过手钱？你按照规矩登了表格，出去多少回来多少，你能赚，但赚不了那么多。怎么赚得多呢？报损吗？海上风浪无情地报损，有海盗报损。所以有没有监守自盗的？肯定有。怎么监守自盗呢？就是养海寇。这波海寇里面有一部分是这些朝臣们自己养的海寇。但同时注意，因为我们原本就存在海上贸易，只要有海上贸易，本来就会诞生海寇。所以并不是说这些朝臣们有一些自己白手套式的海寇，就意味着倭岛上面没海寇了——不存在。原本倭寇犯边这件事是存在的，是发生了的。所以这些朝臣们每一次说有海寇劫掠时，才会把黑锅往倭寇头上扣。所以倭寇身上有没有冤情？有，但不全是冤情。

（好，欢迎大家来到直播间。）嗯，来来来，欢迎来到直播间。刚才说到齐实名的世界贸易非常发达，不但是海贸，陆贸也非常发达。陆上贸易这些年被掩埋得非常严重，为什么很难查证到现在？陆上贸易的部分特别难以溯源，原因就在于满清王朝的两百多年一直在掩盖明王朝的成就、夸大明王朝的缺陷，努力在做这件事。海上的印记消弭起来比较容易，因为证据端的消弭不是满清一家人干的活，它是联合着各种列强协同作战，一起来消弭证据。尤其是满清朝廷，海上力量非常弱，海军实力恶劣，没有能力去全面清剿当年海上贸易留下的痕迹。但是陆上贸易的痕迹有条件清理，尤其在西域。大家记得新疆这个名字是谁改的吧？来，小伙伴们，新疆这个名字是谁起的？记得吧？很好，你们都记得。新疆这个名字是乾隆起的。他起了这个名字之后，就号称这个地方是他开疆拓土的，变成了他十全老人的丰功伟绩。这事儿大家应该知道。

然后问题就出来了，这玩意儿开始好玩起来了。整起来！乾隆确实文化水平不高，写诗词一塌糊涂，但他不是文盲，能够理解吧？乾隆的文化水平并不好，审美水平跟坨屎一样，但他不是文盲，政治敏锐度非常之高。对于哪个措辞可能产生负面影响、哪个能产生正面联想，他太清楚了。所以我就问（谢谢中华花龟英雄）：乾隆特意给我们自汉以来就有话语权的西域起个名字叫“新疆”，你以为他不知道这个名字会产生什么样的直接联想吗？你以为他不知道什么叫做“唯器与名”吗？乾隆很清楚什么叫“唯器与名”，很清楚命名会产生什么样的关联。因为起了“新疆”这个名字，无论他给自己打什么补丁，说什么“故土新归”，都没有用。当名字叫“新疆”的时候，无论从命名、直观、直觉上，都会让人觉得是他开疆拓土的成果。但这真的是吗？不是，就不是真事。因为明王朝的时候在西域就有管理权。你说那是“机密”？“机米”怎么了？你以为清王朝在那个地方拜把子、歃血为盟，就比“机密”强吗？还不如“机密”呢！“机密”是要管的，机密是要进行管理的。（谢谢哈德森。）结果他给这个地方起名叫“新疆”。我跟你说，当一个对你的文化非常清楚、政治敏锐度极强的君王，给一个华夏故土起名叫“新疆”时，他编多少补丁都不好使，他就是在掩埋历史。而事实上他做了没有？他做了。他把哈密卫几乎就掩盖下去了。是这几年，陆陆续续有哈密卫相关的考古出现，才重新引发了大家的重视。在那之前，哈密卫简直就是被沙子埋了，从来没有这回事——明王朝在西域没有经营。有卫所就意味着有屯兵，有屯兵就必须配套兵户、军户、农垦，相当于建设兵团。他假装这玩意儿没有。

同时，乾隆还整了一个活：他把西域及其周边包括一部分衔接中亚地区的卫所名，还包括了蒙古地区的一些卫所名，直接搞了一个谐音梗，摁到了辽东。（谢谢核动力洗碗机。）有兴趣的小伙伴自己去看，不用找我，我是一个带货主播。自己去查，是不是明王朝在辽东的卫所数量特别大、密度特别高？可以自己去看。为什么要强调这件事？注意，辽东是一个什么地方？辽东是熟地。什么叫熟地？就是那个地方我们自古以来——你别说建州卫，自古以来别闹了。建州部是流亡部落，明王朝给他圈了一片地方在建州地区居住。辽东地界原本就是明王朝实控的领土，而且这个地儿是自古以来——不是自明以来，而是自先秦的燕国以来，辽东就是我们的属田。汉代时设的是郡县，是直辖的地方。所以不管怎么说，辽东这个地方我们的原住民数量占比非常高，一直到明末也是这个样子的。后来怎么变少了？你问努尔哈赤去。我们的驻军、屯垦、长期在那个地方种田的民户最后怎么没了的？你去问努尔哈赤，别问我，不是我干的。所以，在这样一块长期统治（无论是明还是以前都是实控土地）的熟田上面，你建那么多卫所，是封赏吗？卫所是一个什么机构？卫所是建立在管理深度不够的地方、去搞屯垦开发的机构。它相当于兼有一部分军事作用，同时兼有农户作用，双重叠加之下既有刀剑又有农垦，起到边防作用。这才叫卫所。然后辽东那块熟地，我们经营了2000年不止的地方，虽然中间有些浮动，但统治时间非常之长。明王朝的两百多年间，你不敢说整个东三省都经营得很深很透，但辽东这片熟地经营得非常深，居住的百姓可以数十万、上百万计。然后它上面设了一大堆卫所，你逗闷子呢？（谢谢李玄。）所以辽东上面那些密密麻麻的卫所哪来的？乾隆用地图开疆的方式，从蒙古、外蒙古、西域乃至衔接中亚地区相关地界的过去明王朝经营管理的卫所——因为他要说明王朝对这些事情完全管不了，根本就没有，都是他开疆拓土的。为了证明他的开疆拓土，他把在蒙古地区的、整个西域疆域内的这些卫所名称，用谐音梗乾坤大挪移放到了辽东的熟田上面。
