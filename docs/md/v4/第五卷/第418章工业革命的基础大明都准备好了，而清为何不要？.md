# 第418章 工业革命的基础大明都准备好了，而清为何不要？

嗯，啥？嗯，人家强，你不会打吗？怎么打不过呢？不是很先进吗？就明末你们心中的那个悲催的、已经废物一样的、一点用都没有的、毫无调度能力的明末，一个普通的边军的队伍，就可以直接把你心目当中伟大的海上的海盗群体——海上马车夫，这个牙和那个牙——一巴掌拍海里面去，爬不出来。所以我们是打不过洋人吗？不是，我们是打不过内鬼。探春的那句话啊，又可以上大分了：“像我们这样的大家子，从外面打进来是杀不死的，必然要内部自杀自灭起来才会嘎掉。”所以为什么没有啊？为什么还倒了呢？为什么破了呢？因为像你这样的人多了，知道吧。因为像你这样的角色太多了，太猖獗了。为什么像你这样的人多了呢？因为吃饱了撑的。简单来讲，就是好日子过多了，觉得皇帝有什么重要的，我最牛；觉得管理者有什么重要的，我最牛；民族有什么重要的，我有钱，我是最重要的。这是什么？这就是思维上的原子化。为什么会产生思维上的原子化？因为太富裕了，惯出来的毛病，温室里的花朵儿，不知道什么叫做房子。就这样，最后培养出了一个又一个和你一样的、那一种原子化的、不知道自己是谁的、也不知道自己归属于哪个群体的这些二五仔，然后自杀自灭起来了。这个东西是什么？这个叫做文明病。所以文明为什么会败于野蛮？是因为野蛮很强吗？不是，是因为野蛮很野。

嗯，那啥，问一问：吴三桂最后一家子被噶得整整齐齐的，爽不爽？他在那之前觉得他投诚到谁的麾下都行，他觉得他把他把国家卖给谁都可以，谁来当皇帝，他还不是拽得二五八万的啊——哪个当皇帝能不惯着他呢？啊，不好意思，野蛮人不惯着你啊。

嗯，工业生产和农业生产很像。觉得只有一个农耕民族，才有资格升成为一个真正的工业民族。嗯，谢谢。那个什么白起啊哈，白起。哎，好的好的。那个什么，刚才你在说那个是关于农业生产，能不能直接衔接到工业化的问题啊？注意了，那个所谓的工业革命，为什么没有发生在中国呀？为什么？因为中国被人抢了呀。因为中国当时的，嗯，主体政权噶了之后，被大家分尸吃了呀，大家都在吃它的好处啊。这个时候你怎么发展工业化？不可能的呀。后来的满清王朝，那是最害怕工业化的，一工业化第一个死的就是他自己，所以他压根儿不会同意工业化的呀。所以我们为什么错过工业革命？因为我们被一群最恐惧工业化，或者说最恐惧中国人拥有工业化能力的人统治和管理，所以我们才倒退下去。实际上清代啊，不光是人均粮食的保有量降低了，实际上亩产也降低了。实际上啊，清代的时候连亩产都降低了。为什么？因为技术倒退了。两方面的技术：一个是天文技术啊——天文是直接拿来指导农业生产的——天文技术倒退了。整个清王朝，他把钦天监这种天文台机构交给了洋人去管理。洋人对你安的什么好心？那个时候你可以幻想洋人对你安好心啊，现在你总不至于存这种幻想了吧？他从一开始就没有安过好心。所以呢，从清代的时候，钦天监受到了洋人的掌控之后啊，我们在天文学上面开始倒退。天文学是指导农业生产的，这是一个方面。还有一个方面就是技术上面的倒退，包括说以前在《齐民要术》等相关的科普类读物当中原本就存在的这些生产的技术，很多的时候是为了让农民能够更上一个台阶，知道如何育种、如何培育、如何使用工具，目的都是为了提高生产效率啊——这些都属于科普读物。但是呢，到了清的时候，科技类的科普读物大面积的禁止发行，甚至被销毁。所以技术上面的倒退、全民文盲的普及，以及呢，钦天监的丧失，共同造就了一个结果，就是清代的亩产啊，平均亩产是低于之前的时代的。所以我们为什么会错过所谓的工业革命？因为当时的统治者不允许中国人有任何的技术进步，甚至技术越倒退越好啊——到了这样的一个地步，所以才会没有所谓的工业革命。

那么问题又来了：嗯，什么样的地方可以诞生工业的革命？按照现在继承的历史，你会觉得那肯定是像那些海盗文明的地方，他们有野心、有欲望，他才会去搞工业革命嘛。来啊，注意了，物质基础才是一切的根基。当我们是一群唯物主义者的时候，我们得知道：你要有榔头，你才能够去敲打；你要有钉子，你才能够钉；你要有一个耙犁，你才能够耙地。这是什么？这就是物质的基础。那么请问什么是工业的物质基础？第一，你需要漫长的定居生活，积累财富与知识，能够理解吧？这个事儿不跳脱吧？一点都不跳脱啊。你必须有能力积累漫长的岁月当中累积下来的知识，不然的话你怎么推行技术的进步？如果一个东西你今天记下来，明天忘了，或者100年前写下来的，100年后你不认识了，我就问你这玩意儿你最后能够累进出科学技术的发展？骗鬼呢！欧洲人玩的什么表音文字？400年之后他们莎士比亚的文章看不懂啊！他们的档次，他们的记录的水平，就到这样的程度的时候，你说他们积累了特别多的科学？400年前连屎怎么倒都不知道，什么叫做城市污染都不晓得，爆发了大规模的瘟疫都不知道该怎么处理，然后突然之间就积累下来了大量的知识？连国王都不认字的一个时代，没过几十年，咔嚓一下子，天降知识，天降技术？你逗我呢！这是什么？这就是物质基础。什么样的地方能够拥有这些物质基础？长期的稳定的定居，稳定的文字和符号系统，才能够让你积累你的知识与技术。然后啊，所有的这些工业化的技术的生产，他需要那些有秩序的、能够埋头做事的普通的工人，这很好理解吧？那什么样的人是最好的工人？全世界最好的工人在哪儿？用我说不啊？全世界最好的工人在什么地方？就连清末那种全民文盲的时代，中国人都是全世界最顶流的工人，没有第二家。嗯，清末的中国人文化水平很差的，什么基础都没有，对吧？最好的工人在这个地方能够积累记忆，能够积累知识的载体，在这个地方长期稳定漫长的定居生活，而且呢，绵延万年的文化文明史在这里积累了多少的财富啊？工业革命最重要的第一桶金吧。什么叫做第一桶金？为什么他们说一定要海盗文明才行呢？他们说因为他们想要抢劫，他们有抢劫的欲望，他们贪婪，所以他们去搞技术革命。我就问你，外面没有财主，他抢什么？他从哪里抢的？他跑到原始部落里面去抢别人的皮毛，然后就能开发出工业革命来吗？嗯，啥抢地？抢地，抢地不要人种的呀？所以什么人可能创作出来工业化的进步？谁一直在积累技术？谁一直在搞大规模的建设？谁一直在定居和配套的生产？谁家的整个的朝堂管理部门当中有一个专管技术的部门叫做工部？谁啊？你没有这些自古以来的基础，你说我突然之间一做梦，咔嚓一下，科学技术全在我脑子里了？我可相信你啊！

嗯，啥？嗯，人家强，你不会打吗？怎么打不过呢？不是很先进吗？认得崇祯吧？我知道你们是什么人，但是认得崇祯吧？崇祯是个小废物吧？虽然我不觉得崇祯是个小废物，但是在你们的嘴里，崇祯是个小废物吧。就这样的一个废物的崇祯啊，在大明王朝快要咽气的时候，明朝的海防的边军一巴掌，就能够把所谓的海上马车夫的这个牙和那个牙拍到海里面去啊，知道吗？不需要调动主体部队。一个字不听，装聋啊？没有关系的。我不是说给他听的，我是说给所有的过路人听的这种。他们自己的号背后是人是鬼又不知道啊，是中是外你也不知道，到底是人还是青蛙你也不知道，到底梳的是满脑袋的头发、掉一根都觉得发愁，还是只有金钱鼠尾那一小绺你也不知道，知道吧。所以呢，在网上不需要跟这些人生气。为啥你都不知道他是不是人啊？所以不需要，不需要生气。遇见这种话头呢，我怼的也不是他啊，说出来的话呢，也不是说给他听的。这种玩意儿你也不知道他是干嘛的，你说给他听，他也听不见，他就是听见了心里头疼，他也得犟嘴。为什么？他是接了任务的，他就必须要制造乌烟瘴气的效果。我们所有的言论都是说给这些智力正常、心胸开朗的普通的路过的人的，知道吧。我们说话是说给过路客听的，不是说给这些叛贼们听的。
