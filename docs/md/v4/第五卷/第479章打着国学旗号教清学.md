# 第479章 打着国学旗号教清学

《弟子规》明显是糟粕。这些国学班基本上都是让孩子先从背《弟子规》开始。注意，《弟子规》是什么玩意儿？《弟子规》是一个——嗯，那是个秀才还是落第的秀才？不管了，反正就是一个考学都考不明白的人，为了教育他的家奴——注意，他是为了教育他的家奴和家奴子弟，编撰的一套教材。简单来讲，《弟子规》就是一个奴才入门版本。所以，你要是给孩子学《弟子规》、背《弟子规》，拿《弟子规》当做自己的教条、学习对象的话，那教出来的孩子会充满奴才思想和服从思想，就是遇到事的时候，他不知道这个道理是什么，反正你先把它记住，照章办事，知道吗？你就不能学这个东西，知道吗？

哎，我们的小朋友应该学什么？小朋友学以前有过什么事儿啊？他通过故事来学习。以前有过什么事，有过什么人。甚至呢，你去给他讲七仙女很漂亮，这个都是可以的。结果呢，你跟他讲：“我没背下来啊！什么父母、什么玩意儿必须顺承？什么玩意儿就须顺承？”在我们的传统孝道系统当中，父母对于孩子必须是慈爱的，必须是引导的。如果父母不慈，子女就有权利不孝。父母殴打子女，“小杖则受，大杖则走”。所有的爱和尊重都是相互的：你今天爱我，我明天孝顺你。这是权利义务相适应的一种表现，这是我们传统的孝道系统，是有一个你来我往的。结果这倒好，变成了一个什么鬼啊？管他怎么骂的，反正你就得听着，听着就对了。所以《弟子规》这玩意儿千万不要学啊！国学里面有很多经典的东西，但是呢，哪怕是神话故事里头都有很不错的、很精妙的。国学的情况下，《弟子规》里面几乎都是糟粕。

之前有人跟我解释说，《弟子规》里头不全是糟粕：“早上的时候要洗脸，你看这哪是糟粕呢？”但是这是不是糟粕啊？早上的时候要好好洗脸——这是废话呀！对，这话确实不是糟粕。走路的时候要看着道啊，要小心着；早上的时候要洗脸，洗手的时候要挽袖子。对，他不是糟粕，他是废话。你放到启蒙读物里面干啥呀？旁边都有个大人，那你教他他不会吗？幼儿园阿姨就会教的东西，需要你背下来吗？我勒个天，老爷喂！我们搞认字，我们在认字，在学书，然后在学道理的过程当中，你学“早上起来要洗脸”“走道儿的时候要小心”。整个这本书，除了废话就是糟粕。爱学不学啊，这玩意儿我劝不了。

好，欢迎大家来到直播间。嗯，那啥，听君一席话，如听一席话。一席话不就是那玩意儿吗？所以呢，你说你唠这玩意儿干什么？《三字经》呢？《三字经》可以学啊，《三字经》是好书。《三字经》虽然也是一个入门的蒙学书籍，但是呢，你看它里面是在讲道理啊，是在讲典故，是在讲以前发生了什么事，然后这些事呢，对我们有启迪的作用。是吧？哎，它里面跟你讲说孟母三迁啊，为什么呢？因为孟母要让孟子学好，人会受环境的影响，所以呢，我们要结交好朋友，是不是？你看老师就可以顺带着就给你讲这个事了吗？孟母三迁的故事天然的就可以跟你如何交友、如何游玩关联起来，既是生活，又可以启迪孩童，同时又学习了历史知识，还可以映照背后的道理。这就标准的启蒙读物啊！所以《三字经》是标准的启蒙读物。《弟子规》？不是啊，《弟子规》是训奴才的。

什么国学义理？有人在求讲解啊，这边还有求讲解的呢。啊，还有这呢。好，行，可以带一下啊。来，这个国学义理啊——来来来，我把这个书给你们弹一下。嗯，前几年的时候特别流行国学班，大家应该知道吧？包括互联网上有很多什么国学大师给大家讲课，然后还有很多类似于像幼教机构搞什么国学班。这些地方呢，我去看过啊，然后身边也有些人开办这个，我去参观过、观摩过。你家里边有小孩的话，你不要让他去，诚心劝谏啊，就是什么国学班这个东西能不去就不去，国学大师这种东西能不信就不信。为什么？这玩意儿基本上都有问题，我看过的都有问题，就没有没问题的。就是特点呢，就是特别的教条，特别的僵化。你以为他学的是国学？学的都是清学啊！怎么说呢，就是他讲解的那些东西，都是被清朝阉割和篡改之后的玩意儿。那这本书就是《国学义理》啊。它的好处是什么呢？就是我们的文化内核是什么，我们的思维底色是什么，我们去了解自己的文化源流的时候应该看什么，就在这本书里头。之所以要梳理这些玩意儿，就是因为有清一代破坏得太严重了，知道吧？从清开始，我们的所谓的国学变成了奴才学。你要是不把这个玩意儿里面的基本逻辑理出来的话，你去看那些经史子集，会被他的著书给坑里头，会被那些国学大师的解释给绕进去。就已经到了这样的地步。我们的儒生在有清的200多年间，把自己的膝盖骨都已经主动地摘除了。你不能说他们没文化，但是他们的心思都用在了那个歪解上面了。所以清啊，清这个时期确确实实对文化的这种破坏力，他比直接把书撕了还狠啊。所以那个完全可以去看一下这套啊，就是国学的基本原理啊，基本原理是什么。
