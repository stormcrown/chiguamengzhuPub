# 第438章 英格利希，百万漕工衣食所系

英语主科会一直在吗？直到我国不再需要文化的对外宣传。这事儿其实跟对外宣传没有什么关系。我怀疑有人能不能理解这事。我也相信一点：我们的英语主科不会那么快取消，但原因不是对外宣传的问题。原因在于，我们国内英语的既得利益群体非常庞大。一旦取消了英语的主科地位，有大量人群会失业，会失去收入来源。既得利益群体会保护英语的主科地位和它的分值占比。

但是，你要是说讲外宣，外宣这件事情跟你用什么样的语言文字没有任何关系。为啥？因为只要你想要宣传，用中文宣传，你以为对方真的看不懂吗？我跟你说，我们外宣的屏障从来不是语言，不是文字。我不知道多少人能直接理解这事儿。外宣的屏障从来不是语言和文字。

小黑子学学用渣作家的神仙姐姐称呼盟主。因为没有天赋，还不会学吗？嗯，好。有没有人看过上世纪七八十年代的时候，就是阿美那边正是强势崛起期的那几年，他们的好莱坞电影、电视剧这些作品？有看过的吗？有看过的请举手。你以为他们用什么手法？注意了，那个时期他们的宣传，你以为他们靠的是语言吗？不是。语言这个东西，几个搞翻译的人就可以翻译过来了。今天很多人还知道那个什么译制片，译制片还是一个特殊的腔调，对吧？我们那个时候会英文吗？绝大多数老百姓不会英文。但是我们有没有看别人的电影、电视？有啊。所以语言是一个障碍吗？语言根本就不是一个障碍，文字也不是一个障碍。

当时他们的影片能够宣发过来，能够让中国人觉得“哎呀，太好了，他那个生活太美好了”，是因为他拍得好，是因为他找的演员好看，是因为他会宣传，会运用光影色彩，会运用镜头语言叙事，知道吗？所以他的这种宣传，是因为他的英语特别有优势吗？不是。我们看的时候没有人听英语的，那个时候普遍用的是译制腔：“唉，亲爱的，你终于来了。”是这个样子的啊。中国人说话也不是这个腔调，是因为这个腔调好听吗？不是的呀。所以那个时候他的宣传是通过什么？是通过美好的叙事、美丽的镜头、美丽的人。

所以外宣靠的是什么？外宣靠的从来不是你会不会英语。你英语特别好，但是你讲的东西都是别人已经知道的，这叫外宣吗？不是。什么叫做外宣啊？你们知道的，我对李子柒不是特别有好感，因为他后来整了一些活。但是李子柒去拍摄那些乡村小故事，一句话都不说，形成了外宣。这个外宣形成的印象是什么？那是另外一回事，但是它是外宣，知道吗？这就是外宣。他会英文吗？大概率不会。他可以一句英文都没有，一句话都不用说，是不是就实现了？所以外宣跟你会不会英语毛线关系都没有。

我们之所以英语到现在和未来的一段时间都会是主科，并且地位很高，是因为有大量的百万漕工衣食所系。这就不是我们的需求，这就不是我们的需求啊。嗯嗯，其实好莱坞电影最好少看。好莱坞本来就有问题，好莱坞就是阿美的外宣呐。啊，你不是以为我觉得好莱坞特别好，需要给大家推广，我才说好莱坞的吧？我是说他们特别强，我没有说他们好啊。他强在哪个地方？人家是认真地做外宣啊，知道吧？好莱坞是阿美的外宣，他认真地做外宣，做得非常优秀。所以今天还有很多人觉得阿美特别好，这就是人家的外宣的实力。这跟他好不好没有关系，但是他做得非常优胜，非常强大。咱们得了解啊，能够搞清楚他在干什么。当然现在已经没用了啊，现在已经没用了。现在他这个外宣做得跟坨屎一样。那啥，广告拍得不错。对啊，你这个词用得是很好的。为什么这么说？就是好莱坞当初拍的这些电影、电视剧，实际上就是阿美的宣传片。那阿美为什么要拍这种宣传片？因为它是一个伪装成国家的企业啊。作为一个企业，它需要宣传自己的产品，需要宣传自己的厂区，然后呢，去吸引其他地方的人才来为他工作。这个呢，是他当时所谓的“美国梦”，就是这一套东西。那这个梦拿什么来缔造呢？就是通过各种各样的电影、电视剧来缔造他的梦，然后用这种梦去招揽别人家的成熟型人才来为他所用。那这种操作的方式在我们这里能不能用？我跟你说，这种招叫做“饮鸩止渴”。能不能用？谁想用谁用。但是饮鸩止渴，解完渴之后，下一步就是奔向必死之路了啊。
