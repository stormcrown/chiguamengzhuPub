# 第464章 古人把喜欢引用名人名言，结果常识都没有的本本主义，叫两脚书橱

有人看那本《夜航船》吗？有人看过《夜航船》吗？看过的扣个一啊。我为什么突然讲到《夜航船》这本书呢？因为呢，很多人即使没有看过，也会知道这本书。它在开篇讲了一个很风趣的故事：夜里行走的船只、客船上，一个和尚因为见到面前的书生侃侃而谈、博古通今，于是觉得对方太厉害了，就把脚缩起来给别人腾位置。一直到后来，他鼓起勇气去问这个文人，问他“澹台灭明”到底是一个人还是两个人——他问了这些更常识性的问题。结果呢，对方觉得“澹台灭明”是一个人，“灭明”是两个人——就这么个事儿。于是呢，最后就是“且容小僧伸伸脚”。那么，这个风趣的故事放在前面，是为了引导大家去读这本书。

但是我觉得，这本书当中有一个非常重要的开篇需要关注到。就是在讲解他去写这本书的初衷时，他强调了一件事情：他说乡里的少年们呐，自幼都是念书的，也学了大量的典故，背了许多的东西，但是一到细问的时候，却常常牛头不对马嘴。对于自己所学到的东西，其实都是生吞活咽下去的，压根儿不知道这些典故到底代表了什么，所以常常在这些对话当中引经据典的同时，却又丢人现眼。所以呢，他才会去写了《夜航船》这一本——实际上主要是在罗列经典内部这些所谓的“点”，就是今天大家在网络上所用的“梗”。它就是一个集梗的东西。那他为什么要做这个？就是因为他开篇的时候讲了这么一段。我觉得这一段话就很很重要，这一段话就非常的重要。为什么？因为他在强调说：这些乡间少年们文化水平并不好，但是被家里面人都塞进了学堂里面去念书，什么东西都念，什么东西都背，背的东西还挺多，词汇一点都不少，但是说话的时候却会四六不着、胡乱隐喻。

这是什么呢？一方面，他是在讲读书读不通的现象；另一方面，它也代表了在这个时代，普通的百姓家、没有什么才能的寻常子弟，在他的少年期也是读了许多的书，背了许多的典故的。这些是什么？就像是我们今天接受了九年义务教育的人，在老师的敦促之下，记下了许多的名人名言，但是不知道这些来自何方，也不知道在人生当中到底有什么用。除了它明面上面所展现出来的东西之外，它同时也印证了另一件事情：就是那个时代大家都念书，甚至有义务教育。

啥？等等我，我把他抬了啊——不是不是那个啥，我把他抬了。又被人那个群体搞到什么监控？他搞到什么群体监控？监控的话我就歇一会儿再开啊。嗯，他反对什么？估计是反对——“施琅是个大汉奸吗？大地主头目大汉奸。嗯，一百遍的话他也是那样的。嗯，这边干净多了，倒也没有啊，这也谈不上，因为其实差不多的。哼哼哼，局面差不多，只不过说某一边的人稍微多一点呢，就显得他似乎杂乱一些啊。嗯，哦，什么什么什么什么东西。嗯，犯什么天条了吗？哎呀，说了现在那个啥，恨我的人多集中啊，这你们还不知道的吗？呵呵呵呵呵呵。你们天天在互联网上沉浮，难道没有看到恨我的人吗？不可能的吧。恨我的声音比爱我的声音还要响亮呀。奇怪吗？一点都不奇怪啊。嗯，别的不好说，但是你们先把赞冲起来吧。我这个人实在是啊，我这个人实在是有一点很强迫症啊，那个数据跟不上的时候，我觉得很不爽啊。什么？这两天骂的可多了？那就对了啊。嗯，知乎上基本上都是看到支持的，那上面有反对的呀，而且还有很多事前反对的呢，看不出来吗？撰稿的水平非常非常的专业，是专业写稿的人写的稿子呀，很多的啊。啥？国姓爷是谁？不行的话你先百度一下吧，这我怎么给你讲呢？啊，差不多行了啊。呵呵。啥玩意儿？我的小迷弟啊？我是信不过任何冲上来说是我什么迷弟迷妹的人啊，我信不过这样的角色。所以不用说什么谁谁谁是我的小迷弟啊，完全不需要做这件事情啊。东西呢，听得过就听，听不过就不听。然后如果听了之后特别的厌憎，你就可以直接开喷。几种情况都允许存在。但是别冲出来说“哎，我是小迷弟、我是小迷妹”，不行啊。我这里又不是搞蛊惑人心的。我讲的每一件事情，有前因、有后果、有过程，中间有因果联系。一件事情是在什么样的背景下，是怎样发生的，它产生了什么样的后果，以及什么样的长远影响，这些东西是我们任何一个人都应该知道的事儿。

这是什么？这是基本的逻辑认知。我们看待历史事件的时候，一定要保持一种基础的逻辑认知。不然的话，你背的书越多，人就会越迷惑，因为你不知道事情为什么发生，也不知道这些事情最终会对你产生什么样的影响。当没有这些的时候，你看的历史都是白看的。有很多科班出身的人，真的历史都是白看的。他在给你讲故事的时候，你会发现，哎呀，他知道的历史故事好多，历史事件好多，一开口就是什么什么时间谁谁谁、时任某某某职位、就职于某某某地、在某某某衙门，甚至呢，他在某一个衙门的时候，又写了什么稿件。说真的，每一次当他用这样的叙事，把这一整长段的东西都先讲完了之后，背后的逻辑问题就已经被掩盖掉了，因为你的关注点就偏了。

但是我们去读历史的目的是什么？是以史为鉴啊，是为了对照、参照，是要吸取经验与教训，是要知道过去到底发生的这件事情对今日有什么样的影响。什么样的情形就造就了什么样的结果。所以今天我呢，应该按照什么榜样去推行政策，或者是需要避开某一种错误的选项。这个才是去学习历史的根本要义。这些都需要你有因果推论关系，你要有逻辑性，而不是记得清清楚楚“这个衙门其实门是朝北开的啊——你以为衙门都朝南开，但是他那个衙门确实是朝北开的”，结果你的注意力全部都跑偏了。

刚才谁说我文学学得不好，就是因为毛选读多了？那可能是吧。文学那个东西呢，学得好不好的，对于人生的影响不是很大。但是毛选能不能看、有没有看过、看了多少、记没记住，却对人生影响很大。所以呢，我依然会在这种情况下啊，推荐大家去读一读毛选。毛选里面就没有什么需要你引经据典的东西，尽管有些人特别喜欢引用其中的一部分的只言片语，也拿来当典故用。但是呢，啊，简单来讲就是毛选本身就反对你使用这些只言片语当作典故来用，它叫做反对本本主义。毛选是反对本本主义的，是反对你把它背下来当作典故来使用的。它是干什么的？它是一个实操手册，它是一个工具书，它是来帮助你思考的，是帮助你梳理逻辑的，是让你能够知道，诶，遇到事情的时候，我应该从哪几个方向去想问题。毛选是起到这样的作用的。所以呢，毛选是必读书目，一定要看，慢慢看啊。每一篇都不长，阅读的难度一点都不大，实际上小学五年级以上的阅读水平完全可以读。毛选阅读的难度不大，篇幅也并不长，而且完全可以形成一个一个的独立篇章啊，一篇一篇地看，一点儿一点儿地看，一点儿一点儿地想，已经足够了。

重要的是什么？重要的是你在看每一篇的时候，都一定要想一件事：我得琢磨他到底是要干什么，为什么要这样做，他是怎么想的。这才是你最重要、最需要思考的问题。然后同时呢，印证着我们现实生活当中所见到社会现象，比照看待，去进行深入的思考。这才是毛选能够起到的引领作用。我永远都不推荐大家背书。当然了，这可能也确实跟我这么多年来不好好背书有关系，但是我确实不推荐大家去背。更多的应该是去想啊——他是怎么回事，他应该怎么办啊，我的生活当中应该如何参考。这才是它最大的价值所在。一定要注意不要本本主义啊。任何一个让你说“你必须说，就你当年说了这一句话，是什么什么什么东西，你必须如何如何”——凡是用这种教条的方式的，通常都有些小心思啊。不然的话，教员最反对教条主义。他让你严格地按照他的某一句话去操办，这就是典型的教条主义。所以呢，啊，说他说了什么什么事，所以你必须如何如何的这种，绝大多数没安好心啊。

啊，不背怎么考试？我这里是什么考试培训班吗？不是啊，亲啊。我这里不是什么考试培训班。你问我考试怎么办？我不考试，我也不抽查考试的事儿。你听你们考试培训班的老师去。你问我，你问我问不着啊。
