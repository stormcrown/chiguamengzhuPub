# 第458章 别卷英语了！让孩子读古文的家长，早就赢在思维大气层

大学才撤了不少文科专业。最近我刚看完《思想殖民》报告。那还有必要让孩子花精力学英文吗？

我不知道你孩子多大啊——那个叫小雨爸的——孩子的年龄是非常重要的一件事。

第一，如果孩子过几年就要考试了，那么肉眼可见的，近几年是不可能取消英语考试的，所以还是得花精力去学。没办法，你必须要通关。分数拿不到的话，升学进不去，后续的事情也办不了。你要把它当成一个工具来学。所以我从来不是反对英文教育，我是反对英文的母语化教育和英文权重过高。不是不能学，而是你不能拿它当主子来崇拜——这个是非常根本的一点。

不要为了拿英语高分，给孩子提供什么所谓的英语母语环境。母语是非常可贵的东西，而中文、汉语本身是一套极有逻辑性、非常训练思维、能够避免一个人陷入二极管阶段的语言和文字系统。所以你的母语、你的初始文字系统必须是汉字汉语。这能让你的孩子将来的路走得更长远，甚至对他智力发挥达到上限都有帮助。这是一个部分。

另一个部分是，在学习英文的时候要摆正态度：我学习它是为了使用它，而不是为了成为它。这个区别要弄明白。如果孩子容易混淆，家长就要在后面有所敦促。例如，在孩子学习英文补习培训的同时，也让他在古籍文献、古文传统文化等领域有所涉猎、学习和研究。这样就会形成文化上的对冲。

注意，研究传统文化不要进国学班。国学班我到目前为止没看到正经的。要去读那些载于史册的像样文章，逐字逐句地念一念，了解它到底在说什么。一旦孩子把古文读明白了，你可以放心：我们在古文经典文章中出现的那些丰满的思想、精确的言辞，足以碾压你能够接触到的大学（包括大学本科）整个前面这个阶段的英文教育。英文文献和文章里的优美、精妙、明确以及思想深度，不可能在你大学毕业之前碰到比我们的文言文更高级的东西。所以，如果你的孩子有这些东西做对冲，你就不用怕他。

至于孩子将来要不要继续好好学英文这件事，回答清楚了：如果你把它作为一个工具，到什么时候都可以学，没关系。哪怕英文已经消亡、世间没人愿意用了，也没关系。研究古文也是门买卖。但是重点在于，你不是为了崇拜它、不是为了成为它、不是为了全心全意地爱它而去学习，而是为了用它。这件事区分清楚了，学就是很正常、很合理的。

啥？加三天班了？法定节假日要有三倍工资。我倒盼着呢，我又没看着。

嗯，来啊，我给大家欣赏一个二极管。来来来，那个啥……所以应该推崇近亲结婚吗？那我就说了：所有在座的家长一定要听劝。孩子从小务必要让他生活在中文、汉语的环境中，这样可以让孩子的思维逻辑更好、更多样化，更有辩证思维的能力，而不至于陷入英文母语环境中的二极管——也就是非此即彼。一说“远源杂交会产生大量劣质后代”，他立刻就说“那难道要近亲吗”？这世间又不是只有远源和近亲两个选项，但是不行，因为他的脑子里面只有两个接口：A或者B，是或者否，除此之外一片混沌，什么都没有。大脑里面是这种接口的情况下，你只要说出“杂交会产生劣质后代”这种科学结论，他立刻就会跳转到“难道要近亲繁殖吗”这种思维链路。这是一个非常西化的思维。所以在座的父母一定要小心，千万不要让孩子变成这个模样。这就是一个典型的样板，样板间摆在你面前了。

如果你让孩子从小以英文这样的表音文字系统作为他思维建设用的母语，那他将来长成这个样子的概率就要比汉语母语的孩子大很多。因为人从小就形成的思维模型，很有可能长大了之后就是什么样，除非他经历非常严苛的逻辑学训练。但这个东西很难啊，把这一门学科学通不容易。既然我们从出生开始就有一个优越的先决条件，让我们可以从幼儿时期就好好的训练逻辑，你何苦等到他上了大学之后再去上那个特别难的逻辑学课呢？而且大概率还学不好。
