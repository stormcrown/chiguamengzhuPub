# 第442章 西方命名思路，撒泡尿圈地才是第一生产力

嗯，来啊。  
为什么对新发现或发明的命名，外国人多以人名命名，而中国很少？我记得很早之前聊过这事儿，就是关于命名的。这是一个典型的野兽撒尿圈地的行为——到哪个地方撒一泡尿，就占了这块领地，这是有强烈领地意识的人会去做的圈地行为。你反应两秒钟，自己琢磨琢磨，是不是这么回事？  
当某个新事物被命名为“吃瓜盟主方法”时，这种方法就和某一个人绑定在了一起，这就是圈地，是典型的撒尿圈地行为。但中国人为什么不这样做？中国人讲道理。什么是道理？就是你得描述这个东西是什么，而不能说“我写了这首诗，谁写得最好呢？李白写得最好，所以这个体就叫李白体”，不行。为什么？因为这个体它不叫李白体，它的特征是七个字一行，所以我们叫它七言。它的本质是七个字一行，所以叫做七言。至于说李白第一个写或李白写得最好（当然这是瞎编的），也不能叫做李白体。  
但西方人就不一样了，他们对自己也是这么干的，动不动就撒泡尿占个圈。十四行诗就十四行诗嘛，居然还弄出了一个专门的人名来命名，十四行诗还得归个类。这种人名命名的行为，相当于签字、署名、圈地的一种行为。这和他的各种知识产权、西方中心论、原创论等东西绑在一起操作。名儿都已经起了，定理都已经叫牛顿定理了，那他还能不是牛顿发明的吗？至于说后来找出证据不是牛顿发明的，啊？天哪，你说它不是牛顿发明的，你难道说那个定理是错的吗？你是不是不懂物理？这不就来了吗。  
所以他为什么要这样做？因为一旦圈定，人们就会把这两样东西之间达成一个思维关联，它就不跟事物本身有关系了，它跟人建立了联系。就像那个扫把星叫哈雷彗星，哈雷有能力在他有生之年发现哈雷彗星的运转周期是70多年吗？没有啊。他们当地的天文记载有可能这么全面吗？做不到啊。按他的生产力、按他的组织结构，他做不到。结果哈雷说哈雷彗星的运转周期是70多年，然后给它起了个名叫哈雷彗星。从此之后，哈雷彗星必须得是哈雷观测和发现的，必须得是他算出来有多少年的周期，别人登记了多少年不好使。这是什么？小狗尿尿。
