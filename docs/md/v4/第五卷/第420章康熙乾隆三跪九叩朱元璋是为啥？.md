# 第420章 康熙乾隆三跪九叩朱元璋是为啥？

如何看待朱元璋的鞋拔子脸画像？这是一种刻意的丑化还是美化？你会觉得鞋拔子脸是美化吗？亲，你觉得鞋拔子脸能算美化吗？呵呵，不行啊，那妥妥是丑化。

关于朱元璋鞋拔子脸画像到底是不是明代的，明确告诉大家：你今天能看到的所有的鞋拔子脸画像，没有一幅是明代的画作。如果你觉得是，请自己去找证据，不要撒泼打滚，硬说“就是的就是的”，非说它是明代的，说明代就喜欢他那样，觉得他天生异象有帝王威严。我这里不接受撒泼。你要是咬定是明王朝画的，那就把纸、墨、衣服、妆造都弄明白，画出画师是谁，为什么画完九族还在，这些都得解释清楚。这些证据拿不出来，就别撒泼打滚说必然就是明代的，说什么“明太祖的怎么可能不是明代的”。

所以第一件事：你今天能看到的所有的鞋拔子脸画像，全部是清代画的，没有明代的。

第二件事：所谓的“异象”“龙像”之说。有人说画成这样只是你觉得丑，明代人觉得这叫天生异象、长得帅。你要是觉得鞋拔子脸帅，自己去找个那样的男人或媳妇儿，我支持你，毕竟世间总有人审美有毛病。谢谢同人于野。所谓异象与龙像，注意，我们的祖先会在书里写某人天生异象，比如“奇男子”“伟男子”，这些词汇在历史典籍中确实存在。但它是文字，意思是面貌奇伟，不代表鞋拔子脸。能理解吗？你说面貌奇伟就是鞋拔子脸，谁定义的？你把证据找出来。什么“奇伟等于鞋拔子脸”？是因为你长鞋拔子脸，所以你觉得鞋拔子脸就是奇伟吗？这是第二件事。

第三件事：所谓龙像。龙像这东西存在吗？存在。宋代有皇帝画过龙像图，是哪位你们自己找。宋代皇帝给自己画龙像，有具体形象：一个人本来长得珠圆玉润，画龙像时就给自己画小小一点犄角，像“我头上有犄角”那样，鼓两个小包，脸上点几个麻子，胡须也画成特殊的形状。这东西叫龙像。宋代的皇帝画过龙像，长这样。所以龙像不代表丑，而且画了龙像必须能认出是本人。如果画完和本人风马牛不相及——不是说古人有“九族严选”吗？结果羞辱天下君王时没九族严选的问题了？画的东西跟你家皇帝长得没一毛钱相似，九族还在，你不觉得铡刀生锈了吗？所以不存在什么天生异象，也不存在所谓的龙像图，没有这回事。

关于朱元璋本来的面貌，他有自己的官方画像，现在台北故宫博物院有馆藏，国内很多博物馆也开始展出相应的画像展品。他长什么样？大家自己去博物馆看，或上网查。现在明孝陵里的绘画图就采用了台北故宫博物院里的官方画像。你去看了就知道，他和他儿子长得一样。爹和儿子长得很像，你觉得有没有合理性？反正某些人认为爹和儿子应该长得背道而驰。但不好意思，我们这个民族非常讲究亲爹亲儿子，很狭隘，要求爹是亲爹、妈是亲妈、儿子是亲儿子、姑娘是亲姑娘。随便捡来的我们不认。所以你看明太祖朱元璋的画像，他和儿子面貌、骨骼结构、甚至毛发都非常相似，一看就是一家人。哪个准？你自己说哪个准。他长什么样子？很典型的一个伟男子的面貌，我很难描述，但属于我特别会欣赏的那种——五官周正的棒老爷们儿。在座经常看我直播的应该知道，我就好那一口，喜欢五官端正的棒老爷们儿。朱元璋就长那样，典型五官端正的棒老爷们儿，是我最爱的那种类型。简直就是在说你？哈哈哈哈哈。朱元璋就是这种形象。他的官方画像有中年、老年，一脉相承，还有子孙画像，也能看出血脉关联，面貌相似程度一脉相承。所以哪个是真的？肉眼可见。他长得五官周正、相貌堂堂，然后你说他丑得不行，那个鞋拔子脸歪得一塌糊涂的东西叫了不起？这画像怎么回事？为什么非得画得跟他不像？

这就不得不讲清王朝。大家知道有个瓷器叫“大明顺治年制”“大明康熙年制”，这瓷器是真货，很多博物馆和海外都有真品。为什么会有“大明康熙年制”？因为大明是一个国际化的王朝，推行大量国际对外贸易和紧密的对外国际关系，有朝贡体系。在座可能有人不承认，无所谓，反正你美爹、加爹、澳爹都承认这个事，他们认为大明当年设立了环绕全球的朝贡体系。当大明有明确的国际体系时，清突然出现说“我取代了大明，我最牛，快来跟我做买卖”，他面临的最大风险就是正统性不足。当然，今天大家说必须承认他是正统，不然我们怎么怎么了。你看，没人讨论说必须承认大明是正统，不然怎么怎么了；也没人讨论必须承认大唐是正统，不然怎么怎么了。但到清就有不同，各种人在网上撒泼打滚说“你必须承认他是正统，不承认我上哪认爹去？去哪找主子？磕头时怎么找门槛？去哪抱格格大腿？”所以清王朝所有统治者很清楚自己得位不正，想要对外贸易、对外关系时就借用明王朝留下的痕迹，比如年号、名号和印件。他们还刻了关于明王朝的印章，在涉外文件上盖章，造了很多看上去有意思的假古董（真古董但看起来像假的），比如“大明康熙年制”。说到这你就知道，当他一直借用明王朝的牌面、名头时，就不能说明王朝是反动而将其踩到地上。他怎么做？他年年祭祀朱元璋，因为祭祀朱元璋代表他继承了朱元璋的国祚，是继任者，是明码标价的正统。这是他的正统权来历。所以每年都祭祀朱元璋。

但我要问：这个马帮群体是否迷信？你们自己心里清楚。他们后宫不敢请大夫，跳大神，这事你们知道。他们特别迷信。特别迷信的人，把朱元璋画像供在堂上，每天早晚三炷香，搞国际活动，还说“我就是你亲孙子，赶紧认我”，天天说这话，你觉得他心里不慌吗？他必须拜，不拜显得没有正统权；但拜了又心里肝颤，怕太祖复活把他给嘎了。这时他该怎么办？他拜太祖，但把画像换了——就像给你寄一封信，但名字写错了，你收不着。
