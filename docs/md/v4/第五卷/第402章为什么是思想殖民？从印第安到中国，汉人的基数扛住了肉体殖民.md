# 第402章 为什么是思想殖民？从印第安到中国，汉人的基数扛住了肉体殖民

那么，精神殖民是干什么呢？我们知道殖民是做什么的。啊，之前我在那个报告里其实没有讲得那么细，因为我讲得比较简短。那今天讲一下到底什么是殖民。到底什么是殖民。殖民这个事儿，你要是论字儿，那肯定不是西方人发明的。但是我们现在所理解的这个殖民概念，是西方人开始开创起来的。这个词不完全属于他们，但是这套定义确确实实是他们整出来的，因为他们开启了一个全球大殖民时代。在那套系统当中，殖民干的是什么事呢？就是第一，我先把你那片土地上的东西清空，然后我搬我自己的人住上去。这种行为在那个时代就叫做殖民。

至于说它清空了对方土地上的什么，那就取决于对方的组织能力强弱。如果说对方土地上面的原住民有较好的文化水平和较先进的组织方式，他就直接给你荡平了，一口不留。那就是像北美的保留地，北美的保留地就是按照这样的方式来操作的。为什么？因为在他们的故事当中，极端落后的印第安人群体实际上是有相当程度的文化水平和社会组织能力的，也有相当多的文化传承档次的。因为有，所以他必须要摧毁对方。因为这样的人没有办法拿来当做奴隶使，你不能拿他当牲口使。一个有自我、有认知、有组织能力、有自己的追求和信仰、有自己的目标的族群，是没有办法当别人的奴隶的。当他不甘为奴的时候，于是当年的殖民者所做的事情就是肉体毁灭。

那么在其他的地方，他有其他的殖民方式。例如在非洲，它的殖民方式就不一样。他会去震慑当地的土著部落的头人们，有的时候是收买。注意了，它还包含了收买的行为，又通过收买头人，然后由头人去捕猎奴隶，开启了三角贸易。这是一个方面。另外的一个方面，就是镇压头人们，有的时候也产生合作。最后要做的事是什么呢？总之就是把人当作奴隶来使。这是干什么？这是因为非洲大陆上面大量的部落，在那个时候还没有形成一个非常优质的自身的组织能力。当他自己没有这些东西的时候，他被人驱使成为奴隶，他不知道该怎么反抗，因为他没有更加先进的组织方式，也没有更加先进的信念，非常容易被人家忽悠。所以你就会发现，他们的殖民方式来到了非洲大陆的时候，就变成了贩奴，以及驱使奴隶圈地占田。这也是他的一种殖民方式。

接着，这个殖民的操作手法，我们就来到了亚洲了。同样的这一拨人，他走到亚洲的时候，他又有了新的殖民方式。为什么要这个样子？为什么？为什么要用这样的方式进行殖民？这其实取决于他的自身能力的问题。你想一下，印第安那个地方那些人，属于什么孤悬海外。北美大陆上的印第安人孤悬海外，当他采用了循序渐进的方式，逐渐围剿对方的时候，消灭了就是消灭了，旁边的人可能都不知道的情况下，他就实现了他的灭绝式的操作，这个事儿他就能办到。而非洲那片地方上呢，那是因为他们原本的组织形式就是一个刚刚进入奴隶制游戏、大部分地方连奴隶制都还没有进入的部落时期，所以他可以进入到一个奴隶制阶段。

但是到了中国的时候，这个局面就会完全不一样。尽管说当时的中国人经过了200多年的残酷奴役和压榨，大部分人都是文盲，但是有些东西，这不是你把我变成文盲我就会忘记的。我的祖先吃几顿饭，穿的衣裳和今天是否一样，我的祖先对于外界的人是怎么看的，我的情感怎么传递，我觉得什么是神仙，我觉得什么是鬼怪——这些基层的代码，不是你把我摁成了文盲他就会消失的。这是什么？这是文化。所以中国这片土地上，不光是人多。这个人多啊，很多人常常误以为是个负担。注意了，人多是什么？人多就是这一群人的血条，人多就是一个本钱，这是我们祖先留下来的本钱。你以为其他地方的人是不想多吗？是因为优生、优育、优选吗？不是，他是因为生不下来，因为养不活，因为人无法长大，所以才会人那么少。我们能够当时有那么多的人，在重重的压迫、盘剥之下，在极端狠辣的制衡之下，还能够有那么多人，是我们的老祖宗在之前上万年的时间里面积累下来的深厚的血条和生存的能力。我们有这样大的一个庞大的人口库，这就造成了对方说“我想把他圈起来，然后屠戮殆尽，我占了这片地方”——做不到啊，他做不到。包括说，你以为当年的殖民者、侵略者们没有动过灭绝的心思吗？他们一直都是这样想的。你要知道，八旗入关的时候，整个这个地方的人口损失量达到了什么量级？根据不同的口径和计算方法，损失量、折损量从8000万到1.2亿不等。就这种量级，你放到任何一片大陆上面去，都是一片白地了。为什么我们能够扛下来？血条厚。这是一个非常重要的物理先决条件。就像这几天很多人在那里讲的731，他为什么一进入侵略状态，立刻就要开始设731这种秘密部队？为什么要设？因为他没有办法消灭所有人，他就想要用生物的武器来摧毁所有人，大规模杀伤。这是什么？这是因为我们血条厚啊。换点其他的地方，这种手段他不用上，他都可以把事办完了。所以呢，他才会去设立这样的反人类的机构。这是第一要素，就是血条厚。

第二个要素，就是我刚才说的，我们有文化的基底。一群有文化的、有组织力的、有共同认知的、彼此之间有认同感的庞大人口，对他来说是很恐怖的一件事。如果你不能够直接理解，请看一下我之前做的视频《谁在恐惧义和团》。义和团当初就是给了他们这样的重重一拳，他就能够看到，即使这些人已经跌底，跌到所有人都瞧不起了，但是他的天然的组织力、天然的本能、天然的实力，依然可以抗拒侵略者，而且侵略者在这个地方无能为力。所以他为什么在中国这里的殖民方式和其他那些地方不一样？是他不想吗？是他不能，他做不到。那既然他既做不到完全消灭，又做不到让我们当奴隶，这个时候他怎么办呢？他又要殖民，他又要掌控，于是他就借由这片土地上面天然存在的一些非常软弱的所谓的精英群体，培养他们当买办走狗，然后做了一件事情：把他的思想灌输到中国人身上，然后让中国人奉他们为主上，认为他们很了不起。那这个过程如果他没有通过暴力的手法直接达成这种可能性的话，他可以用什么方式？那就是精神的方式。要让所有人觉得他们就是高高在上的，听他们的就会先进，听他们的就会进步，向他们学习就会发达。实际上，在今天的很多中国人身上，还留着这样的痕迹。依然有很多人觉得“你不要排斥向西方人学习，你向人家学习才会国际化，才会先进，才会走入文明时代”。就这样的一种想法本身，其实它背后的潜台词、背后的认知就是西方代表了文明，我们代表了落后，我们需要去追逐对方。这就是他背后的思维模型。

所以为什么那个时期他对我们进行的殖民当中，武力只占了其中的一部分，而不是全面武力对冲？因为做不到全面武力对冲，它是必败无疑的。他早就意识到这件事情了，所以他选择了另外的道路，就是改变这一群人。他用思想的、用经济的、用文化的方式来改变这一群人。当某一群人——当中国人觉得和服是我的传统服装，日语是我的母语，当你觉得是这个样子的时候，你不是日本人，也是日本人了。所以他为什么采取的殖民方式那么的奇怪？为什么要有精神殖民？因为他要进行精神上的置换。当你觉得你穿三件套的西装就高级了的时候，你的所有的智力活动、体力活动，你所有产生的价值，就会贡献给那个设计三件式西装的老大，从此你就会变成他某一个维度上的奴隶。所以为什么强调这个事儿？它是一种殖民的形式。以及对方是不是因为仁慈所以才这样做，是不是为了教化众生？不是，他是因为劫掠我们而不成，所以转而进行了精神上的驯化。那这种方式行不行呢？在今天看来，依然有影响。只能说不是所有的中国人都受到了他的影响，而这些没有受到影响的人，就是这个国家得以存续、得以发展、得以突破的英雄。
