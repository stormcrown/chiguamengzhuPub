# 第427章 有人利用连麦，让蒙主审论文（浅析人类社会三种运行模式）

嗯，建奴能留下来，不就是靠朱棣仁慈吗？所以啊，滥什么好心呢？那为啥主播有连麦啊？有连麦是吗？好的，我瞅一眼啊，我不知道能不能连得上，你们知道的吧。啊，连麦这种事情我不知道能不能成功啊。啊喂，盟主晚上好，晚上好。我居然听见你了，我的天呐！好的好的好的，听见了听见了。来，继续啊。好啊，盟主晚上好。今天我连麦的主题是：人类社会的三种运行逻辑与历史分段的想法，大约需要9分钟。

呃，盟主有个视频讲了金权、王权、神权三种权力，受此启发，我发现人类社会有三种运行逻辑，我暂且将它们称为利益本位、人民本位、神农本位。当然名称不重要，叫别的也行。利益本位就是逐利和剥削，与金权相对应，发展到最后是极端自由之下的丛林法则，不愿受到规则的约束，代表制度是资本主义制度。神农本位就是媚上和欺下，与神权相对应，发展到最后是绝对权威之下的迷信盲从，不允许质疑崇拜的神明，代表制度是宗教种姓制度。人民本位则是以人为本，以民为本，争取利益但有原则，追求积极自由但有底线，为大多数人着想，推崇英雄但不神话，尊重权威但讲道理，让人可以成为人。中国自古以来大部分时候都是这样。

以元明清为例，蒙元时期是利益本位的包税制，蒙元本身也是从丛林法则里杀出来的，只管剥削，直到最后把自己玩崩了。明朝重新建立了人民本位，明朝中后期人民本位被利益本位侵蚀，有人想恢复蒙元的利益本位，引清军入关。结果满清实施的是神农本位，把满清统治者都神化为圣主明君，推行奴化和愚民政策，禁锢思想。时至今日，依旧有人看了清史稿等神话满清统治者的所谓史料后，就一屁股坐到满清统治者那边，我认为也是神农本位思想在作祟。这些人看到的满清统治者不是人，而是被他们神化之后的他们的奴才，神话之后的神明被洗脑成了他们的信徒，所以才会美化他们的一切行为。洋奴也是同样的道理，只是他们和亲粉崇拜的神明不同。甚至某些人打着自由私有化的旗号，看似是利益本位，但他们并不维护自己的利益，而是崇拜自由主义和西方价值，本质上还是迷信盲从的神农本位。真正的利益本位奉行的是强者为尊，有奶便是娘，哪怕表现得非常崇拜某个神明，甘愿为奴为婢，也是因为有利可图。一旦对方失势，他们就会立刻转变立场，甚至崇拜另一个得势的神明，身段十分柔软。例如某些曾经将西方视为天堂、现在又表现得异常爱国的行为表演艺术家们。

由此可见，不同的运行逻辑在同一个社会中也是可以同时运行的。上层运行的是利益本位，为了主义剥削下层，藐视权威；下层运行的却是神奴本位，把某些上层视为神明，哪怕损害自身利益，也甘愿为奴。这三种运行逻辑，我认为只有人民本位占主导，是可以长久运行下去的。利益本位占主导，运行到最后一定会崩溃，陷入混乱无序的状态；而神农本位占主导，则会让社会发展停滞甚至倒退，最终被外力打破。总之，不论人类社会的生产力如何发展，这三种人类社会的运行逻辑从古至今都是共存的，并且随时在互相博弈，此消彼长。

基于以上的三种人类社会的运行逻辑，我对历史分段也有一些新想法，想请盟主点评一下。呃，第一次和盟主连麦时，我说过一个观点：我认为人类社会已经完成了六次生产力变革，我们现在正在进行的是第七次生产力变革。在三次农业革命之后、工业革命之前，还有三次农业革命；而我们现在正在经历的第七次生产力变革，不仅作用在工业上，就像第一次工业革命也不仅作用在农业上一样。未来的人类在回顾现在所谓的第四次工业革命时，可能会将其称为第一次智能革命之类。所以截至目前，人类社会按照生产力分段，可以分为：原始社会的旧石器时代，农业社会的新石器时代、青铜器时代、铁器时代，工业社会的蒸汽时代、电气时代、信息时代。当然也不排除一些地方跳过其中的某些阶段，直接从新石器时代到铁器时代，或者从铁器时代到电气时代。

生产力决定生产关系，社会存在决定社会意识，经济基础决定上层建筑。生产力变革会产生新的生产关系和利益群体，衍生新思想，推动社会制度变革，使之适配新的生产力与生产关系。同时，新的利益群体与旧的利益群体之间、新思想与旧观念之间也会发生矛盾冲突。社会制度相较于生产力和生产关系的滞后性，也会造成思想混乱、制度崩坏、组织力下降，直到新的社会制度最终确立。而生产力变革也不一定是一定成功的，也可能会被旧的利益群体强力破坏，打断后续变革。例如明清易代，就是代表落后生产力的清廷利用中国生产力变革时期、旧利益群体发生矛盾的契机，强行征服中国后，破坏了先进生产力，把生产力拉低到与他们的旧思想、旧观念、旧制度一样落后的程度，反向适配。

以前我认为奴隶制是落后的，资本主义是人类社会发展的一个阶段，在古代封建社会出现资本主义萌芽是一种走向先进的表现。但是在发现人类社会的三种运行逻辑后，我觉得并不是这样了。因为在任何生产力阶段之下，都有可能出现奴隶制，也都有可能出现资本化。现在我认为在生产力变革时期，新的利益群体想要打破旧的规则秩序，利益本位就会抬头，出现重商主义或者资本主义萌芽，例如春秋战国时期、明朝中后期、欧洲近代。而当生产力陷入停滞，旧的利益群体过于强大，被外敌、殖民统治等，统治者不想让生产力进步，想要固化社会制度时，神农本位就会抬头，推行奴化和愚民，禁锢思想，比如满清、印度、欧洲中世纪。

不过我的想法中也有一些地方解释不通。一是欧洲近代的思想变革——文艺复兴和启蒙运动之前，发生了什么样的生产力变革不清楚。因为按照现在的公认的历史，欧洲早就完成了第三次农业革命，进入铁器时代，而第一次工业革命还在文艺复兴和启蒙运动之后。思想变革发生在生产力变革之前，看起来倒像是上层建筑决定经济基础，社会意识决定社会存在的。二是明朝中后期，新的生产关系、利益群体和思想变革出现之前，发生了什么样的生产力变革也不清楚。毕竟现在公认的历史，第一次工业革命开始于1765年的西方，哈格里夫斯发明珍妮纺纱机，那都是明朝灭亡的100多年以后了，而中国早在春秋战国时期就完成了第三次农业革命，进入了铁器时代。要把这两个问题解释通，只有两种可能：要么第三次农业革命和第一次工业革命之间还存在其他生产力变革，要么就是现在公认的历史有问题。这就是我今天想要分享的内容。

好的好的，啊，这是又想了很久啊。我们这个小伙伴呢，每一次都会把他自己总结出来的想法分享给大家听，为了让他的分享能够更加流畅，所以他将自己想到的东西书写成一个完整的文章，然后拿出来直接阅读。如果你觉得这个样子让你感觉不适配，那就是你听不懂人家念书呗，听不懂的话可以出去啊。他在分析自己思考得来的关于人类的历史分段进程和我们的历史发展进程的一些分类的方法啊。那这种分类的思考，我觉得还是颇有一些可取之处的。就是人类的社会治理方案当中，它有哪一些治理方向？上面的思路到底是以民为本还是以利为本？或者是以权威为本？当它是以利为本的时候，考虑的就是盘剥与贸易；以人为本的时候，需要考虑的就是这些人如何能够发展得更好，如何能够多多益善、绵延万代，如何能够产生扩张；如果它是考虑的威权治理，那么就会非常相似于神权统治，设立一个绝对的权威，不允许他者背离。这是不同的一种社会治理的思考方案啊。

那这种思考的方案有可能是由于生产力决定的，但是也有可能是一种特殊的选择。我为什么这样说？因为我现在察觉到了一件事情：论生产力的发展，实际上欧洲从元蒙古扩张时期就开始意识得到遥远的东方有着繁荣的文化，那个时候他们就看见了这些场景。我相信他们也并不愚蠢，一定会开始学习。后续的发展，科技史相关的内容虽然不能够完全一对一地对照，谢谢，忙不忙，但是也同时从侧面上可以看出一点，就是从那个时代之后的欧洲其实悄悄地发生了一些转变。这就是他妈的技术在努力地进步，可是他的制度进步了吗？没有啊，没有。为什么呢？因为他这种神权的、威权的、威慑型的统治方式对他更有利。甚至在那之后漫长的时间里面，他们并没有试图推广知识，也没有试图扩张识字率。这些事情，中原王朝其实在历史上面一直都在做啊，除了清，其他的时代人们的识字率都是在想方设法地提高的。但是呢，你会发现啊，在这种威权压榨之下，实际上他的统治者会希望自己的底层的民众是愚昧的、是无知的，才会在他的指挥之下指哪打哪，所以他会刻意地控制自己的制度发展进程。但是同时呢，他又非常贪婪地希望获得技术的进阶，两种东西会同时存在啊。啥？快乐教育？对，现在的西方人也会做这个事。例如说阿美，阿美呢，是一个以利益为导向的国家，大家都知道谁钱多谁说了算。但是同时呢，他也在缔造这样的一种愚弄百姓的管理方式，希望所有的普通人越愚蠢越好，越无能越好。所以他身上又同时具有这种威权，也就是他刚才所说的这种神奴的性质，就是半奴隶制。

所以你瞧，资本其实是支持奴隶制的。奴隶和资本之间有兼容属性，他们不是背道而驰的啊。神权的统治和奴隶制，那兼容得不知道要多好；而神权统治和资本能不能衔接？能衔接，一点问题都没有。所以不同的治理方案完全有可能是交错进行。那么他们是绝对的由生产力直接导向的吗？因为生产力和生产关系是有一个相互作用的，所以不排除有一些人群，他在生产力进阶之后还很刻意地让自己的制度维持着落后的阶段啊。那为什么会这样？我们在亲身上可以窥见一斑：那就是清，它事实上在入关之后，技术上面、科学水平上面、生产力水平上面实现了划时代的跃升。它直接让自己从原有的那种十分草莽的环境进阶到了一个在当时最顶流的科技环境、技术环境，但是他要考虑的第一件事情是维护自身团体的存在，自身的根本利益，这就是谁是我们的敌人，谁是我们的朋友。因为它的根本利益就是搜刮，所以它需要有稳定的可以搜刮的制度，什么样的制度能够帮助它搜刮，它就会运用哪一种制度；什么制度能够让它稳定地当大家的主子爷，它就会推行哪一种制度。所以生产力较之于它以前是进阶了的，而且进阶了很多，甚至于制度的发展也是有可以参照的对象的，但是它都不学习。它会努力地将这个国家拖入到制度性的落后，因为这一种落后对它才是最有利的。所以不排除有一点：当某一群人没有退出历史舞台的时候，那么这一群人会强行地推行他自己认为对他有利的那一种制度，而不是顺应着民心，不是顺应着生产力的发展而直接进阶。欧洲当年的黑暗中世纪是如此，满清的邪恶统治也是如此。

哎，好，欢迎来到直播间啊。嗯，以民为本，民可以是士，可以是农，可以是工，可以是商，也可以是君。所以有了民就有了一切。不同的环境、不同的语境下面，民的含义确实是不一样的啊。民是什么呢？民它的本质就是治理的基层，治理权能够深入到的最下面的一层就叫做民。所以呢，在古典时期，很多时候说民是不代表普通的佃户啊、这些流民啊，这些都不包含在内啊，很多时候都不包含。为什么？因为他管不到啊。权力的触角能够深入的最基层的那个阶层，就会被称之为民，就是我可以管理的那一个阶层。所以今天我们的民为什么会变成所有的全体人民呢？因为我们今天的管理能力可以切入到任何一个位置上去，我们的管理深度最深，所以他才能够说啊，民这个字在今天会没有争议。在其他的时代，民确实是有争议的啊，那你这是没办法的，确实以前有争议。例如说早期的时候，周代这个国人暴动，那个时代的民最下到哪呢？到国人城外的这些管不了那么多的野人啊，就是非在编人士就不在民的这个范畴里面了，就不管了啊。所以不同的时期，民出现在文献当中的含义会有一些偏差，它基本上可以指的是政权的触角能够管理管辖到的最底下的一个层级，称之为民。

嗯，能说一下古代神灵精怪录吗？诶，能能能能能。哎呀，我的天老爷，诶，我竟然可以带货了呗？我看一下啊，我先，我先把所有的那个商品都弹出来。有人主动地跟我说要带货的时候，我突然之间，哎呀，好感动啊，你们终于记得我是一个带货直播间了吗？高兴得一塌糊涂，庆贺一下啊！为了这个，你们也得点个赞，快点啊！这个神灵精怪录啊，它是哪本书来着？应该是这一本书啊，对对对，是这一本。它是一本画册啊，注意了，这本书是一个画册，它画得还是蛮好的。我给大家展示一下啊，然后呢，它绘制了这样的一个……这个应该属于精怪啊。它呢，做了分类，然后把我们过去民俗传说当中的，还有一些典籍的志怪故事当中的一些神灵精怪类的角色进行了绘图，绘完了之后呢，它会有一段文字去对它进行介绍，并且呢，有一个基本的分类啊。所以呢，它可以作为一个良好的亲子阅读的读本，你可以把它作为自己看，我觉得也是挺好的科普嘛。啊，也有一些段落呢，就是文字介绍较多，就是它的历史背景上面较多的段落呢，它就会多一些文字来进行介绍啊。这个是这样的一本书。这个书它本身，嗯，编撰的成本其实不算低，知道吧？它的编撰成本真的不算低，因为你要从各种典籍当中去挑选这样的门类，然后进行艺术的创作，然后再进行专门的采编。整体来说呢，嗯，性价比还说得过去，但是书本身我个人觉得还是不错的。如果家里有小朋友，那么他的一些民俗启蒙里面可以有这样的一本；如果你自己对于中国传统的神灵精怪有兴趣的话，自己也可以拿来读。
