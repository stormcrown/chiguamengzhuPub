# 第467章 国有私有盐铁论，中国“集中力量办大事”是祖传的

嗯，能说一下盐铁会议吗？嗯，我一下子没反应过来，盐铁会议指的是什么。要不然我现场百度一下盐铁会议指的是哪一茬儿，指的是哪一件事情？万一我只是忘了词儿，实际上那个事儿我知道。哦，原来是这一件事情啊。嗯，不是讲《盐铁论》的那一茬子事情吗？他一说盐铁会议，让我蒙了一下。啊，对对对，就是那啥事儿。倒来倒去，捣出一头驴，最终还是《盐铁论》那一趟。

西汉王朝呢，它是有着强烈的扩张性的。那么西汉的扩张性到底是从何而来呢？其实，就跟它的盐铁专营是关联非常紧密的。完全可以说，这是一种努力制造国有资产的操作方式。制造国有资产，其实这个事情从西周时候就已经有君主在努力尝试了。

我之前曾经跟一个朋友讨论过一件事情，就是资本主义到底是什么时候诞生的？注意啊，我们讨论过这些问题，现在我觉得他说的有一部分东西可能是对的：有关于资产阶级的萌发、资产阶级的崛起、资本主义的诞生，在中国这片土地上的历史恐怕比能够想到的段落都更加古老。古老到什么程度呢？就是我们在西周的时候——西周，不是春秋与战国那个东周——其实已经开始出现国有资产与私人资产之间的博弈。那么这种博弈，这种分歧，本身是否就是资产阶级崛起的一种表征？就是私人资产要求自己必须要拥有多少多少的利益。实际上在西周的时候，就已经开始——忘了是哪个王了，你们自己找一下——他试图要开始推行“观山海”，就是要让所有的山林都归于国有，不允许民间采伐、采探所有的矿产，甚至伐木都进行了相应的管理。当然，当时整出这一招之后，国家就乱了，迅速就崩了。

嗯，啥？不是吧？那个时候不是奴隶主制吗？不是啊。我们西周的时候，你不能够真的就把它说成是一个奴隶制。因为我们按照历史的这种现行的分段论，对于我们的历史进程描述是不精确的。这件事情我们曾经探讨过，但现在还没有得出一个非常有共识的结论。可是我确实觉得，中间有一部分地方是可以值得细细研判的，有一些位置非常的有争议——就是西周的时候算不算奴隶制。我现在也认为西周的时候不是奴隶制，不然的话，如果是奴隶制，不会有“国人暴动”。注意了，当时的“国人暴动”里面的“国人”是什么人？国人就是有社会编制的、编户齐民的、有公民身份的人，他们是自由民。然后与国人相对应的“野人”呢？野人实际上是没有编制的、没有福利的，但他们也是自由民。他们都不是奴隶。西周就不是一个真正的奴隶制。

那么西周是标准的封建制度吗？对，西周是一个封建制度，它是分封建国制度。它将治下之人分成了国人与不受严密约束的野人，然后叠加了一部分的“隶”、“臣妾”这些奴隶群体。奴隶是被圈禁的人群。所以有没有奴隶？有奴隶。但它是奴隶制吗？不是标准的奴隶制。它没有一个说所有人建立严密的人身依附关系这种事情。你像到了清朝的时候，它的八旗制度是一个标准的奴隶制，所有人群彼此之间都有互相制约的奴隶身份，不同层级的奴隶。但西周的时候不是。西周的时候，无论是国人也好，野人也好，他们都不是奴隶。而在这个时候，西周已经不是一个标准的奴隶制时期了，虽然它有奴隶。

那这种情况下，它已经开始产生了很多的民营资产。要知道，有大量的自由民的时候，就会有很多的私有资产。所以在西周的时候，其实已经出现了君王试图要将山野之利归于王室、收归国有的这种努力。然后呢，国人表示不同意。国人为什么会不同意？为什么会反对这些资产的国有化？那必然是因为他自己有私有的资产、私有的力量在其中。所以这个时候是不是已经出现了资产阶级的雏形？否则的话，他搞这个暴动干啥啊？目前是有这个怀疑的。

当我们的时间节点一下子提前到了西周，这个时间段就已经开始尝试资产国有。那这个时候我们就可以想到一件事：君主努力地将所有的矿藏、资产、林业、土地都收归国有，这件事情本身是试图让全国形成一盘棋调度的一种管理方式。这个方式到了西汉的时候，就呈现为《盐铁论》的这种管理方式——专营，盐铁专营。它也是经济命脉、矿藏的一种专营，是国有资产的方法，也是一种国资应对民间私有资产的方式。所以你说，我们的这种资产阶级崛起到底是从什么时候开始萌芽的？然后呢，国与私之间的这种不断的博弈又是在什么时候开始进行的？我认为这个时间节点完全可以向前提前，一直提到西周。

然后到了春秋与战国的时候，民间的金融战、贸易战已经非常普遍了。各个诸侯国之间耍心眼儿的时候，采用了大量的金融手段、贸易手段。如果说他没有充分的资产化，没有充分的商业经济和活力，这种贸易战你是打不起来的。如果他真的是那些所谓的大地主，你怎么打贸易战？一定是中间有着大量的资产阶级、商人群体异常活跃，他才有可能打起贸易战来。春秋战国期间，贸易战是非常非常频繁的，所以这个时候资产阶级已经极大地爆发了。

那么到了西汉的时候，为什么会有盐铁会议？因为西汉是一个强力扩张的时期。一个王朝强力扩张，他首先要解决的一件事情就是资产国有。因为资产是国有的，它在扩张的时候，才能够直接把所有东西撒出去。因为他会以全局观去看待这个国家，他会认为这个国土应该出去。那么我需要粮草先行的时候就粮草先行，我需要修路的时候就要修路。这个大基建我必须要做，尽管说要花很多的钱，但是这样花钱，民间的私有资产不会同意。这相当于什么呢？大家今天去看那些欧美国家的地区，号称的老牌资本主义的地区，说他们是发达国家，但他们的基建通常非常老旧，想要更新基建的时候非常困难。而中国更新基础建设的效率，全民万众一心的意志就要比他们强很多。为什么？因为我们整体的核心的资产是国有的。核心资产和基本的土地、矿藏是国有的时候，我们需要进行大规模的基础建设，我们就可以做。但是当他这也私有、那也私有的时候，这些事情你都推不进去呀。他这个东西就推不动。

所以西汉的时候为什么会推行盐铁专营？这个就是标准的要将所有能够收拢上来的财富收拢到中央财权当中。有了中央财权，才能够进行整体调动，这些大规模的基础建设、国家的扩张、边境地区的安防，你不可能指望任何一个私人的资产。即使这件事情对他来说也涉及到身家性命，但是你要动他的钱袋子，那是万万不能够的。所以中央王权与地方之间，始终都存在着经济的博弈。而《盐铁论》这件事情，就是在那个时期所提出来的一种非常精妙的、能够把资产收拢为国有的一种操作方式——集中力量办大事。

那么为什么他要选择盐和铁？因为这个东西是当时所有的阶层都离不开的经济命脉。所有的人必须吃盐，不吃盐人活不下去。他甚至很多时候用盐代替了税，因为你收税的时候，人可能哗啦一下跑山里了，你一跑，你管得不严的时候，税就收不上来。或者他把家里的孩子一藏，这个人头税就算不清楚账，这种事情都有可能出现。但是盐它是要吃的。所以为什么要把盐作为重要的纳税手段？很多东西它可以藏匿，但他必须要吃盐，不吃盐人动都动不得。所以管制住了盐，就可以管制住主要的一条经济命脉。因此呢，这个盐铁专营本身就是一个将核心资产收归国有的操作方法。

其实这样的一种整体调度，一直到今天的中国依然起着极大的作用。我们一直到现在，我们的国有资产都是国之利器。我们最重要的基础建设，最重要的核心项目依然是国企在操办。你要让民企去做这个事，有的能够参与，但参与大量的是不行的。

啊？什么？盐吃得少或夜盲症吧？你搞岔了啊。那个夜盲症是维生素A，亲啊。得夜盲症的是维生素A，维生素啊。然后那个缺盐的话，人会动弹不得，就是缺乏力量。嗯，好，欢迎来到直播间啊。嗯，啥？记反了是吧？嗯，对，看不清楚的那个是维生素A，用胡萝卜炖肉可以解决这个问题。呵呵呵呵呵。

嗯。所以现在的共和国土地都是国有化，就是为了防止土地兼并。一代人有一代人的事情要做。那个土地国有化这件事情，我们也是自古以来就有先例啊。只是说呢，大家很少能够做到像今天的国家我们执行得这么深入。在历史上面，土地国有是一个大家都知道有利于国家的一种管理方式，但是你推行下去，难度系数那个高，就是属于你推不动。

事实上，在当年最强盛时期的鼎盛时期的秦国，所推行的制度就是土地国有制。秦王朝当初的军功授爵所绑定的就是土地国有。他通过土地的国有实行土地的分配，然后每一次有人军功授爵了之后，他要给对方分配土地。因为一个军功授爵的军事人员，他必须要有一定的土地供养，没有饭吃的话，他怎么当兵？所以呢，每一个获得军功职衔的人，他都要有相匹配的耕地分配给他。那如果说土地是私有的，你拿什么分给他？去哪里分？你抢别人的地分吗？所以实际上，鼎盛时期的秦，他就是推行的土地国有化。但是当时的生产力和管理能力，让他没有办法非常深入地践行这一项制度。尤其当他征服了六国之后，在六国故地的土地上面，这件事情办得就非常非常困难。因为六国的旧贵族当中，还包括了各种各样大大小小的封建土地主，就是所谓的“封君”啊。他们通过分封建国的方式，拥有了自己的一片固有的土地，某一座城和它周围的地都是归他世代所有的。这样的大小封君数量非常之多。所以秦王朝在征服六国之后，面临到一个极大的困境：他家的士兵打了这么大的胜仗回来，但是他没有办法非常直接就把别人小封君的地一次性撸干净，然后分给自己的军事人员。因为你要让这个战争快速结束的时候，你就不能说“抢了抢了抢了全抢了啊”，没有办法全部都抢。一部分是抢了的，但是很多地方他抢不了。于是就残留了一些原有的封君、原有的地主，造成了核心地区的土地没有办法完全地再次分配，保留了一些原有的地主阶层。最后呢，造反的人群当中，主要的就是这些地主阶级。为什么？因为秦制是非常不利于这些地主阶级的。
