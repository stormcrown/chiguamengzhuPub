# 第409章 元朝和清朝到底有啥区别？

欢迎来到直播间。我们先聊一下满清和蒙元统治上的区别。为什么今天的中国人对蒙元没有那么厌恶？第一，明王朝把这个仇了结了，账已经算过了。我们这群人不是特别记仇，仇过了就完事。另一方面，当年蒙古王公的统治虽然是一种落后的统治，带有压制甚至种姓隔离制的色彩，但他们的本意是为了逐利。蒙古王公主要目的就是贪财，在逐利的过程中，只要有钱就行，其他事情不管——当然他们也管不动，没有管理能力和技巧。所以他们只求利益，又不想付出成本深入管理，于是借用代理人制度、包税制度、分包承包制度。结果造成两个现象：第一，统治非常不稳固；第二，中间的代理人日子过得太舒服，层层加码收税，底层百姓民不聊生，中间实力阶层却爽到飞起。

中间实力阶层在其他时代也存在，比如帮忙收税、管治安、管文教的群体，始终都存在。但其他正常王朝的统治者很清楚这些中层会干些什么狗屁倒灶的事，所以有多方位监管和制约。可到了蒙元时期，蒙古王公根本不管事，最后中层管理人员个个变成脑满肠肥的土皇帝，盘剥极盛，对下面十分残酷，对上面糊弄了事，对所有人都不负责任，变成极端自私自利的群体。这样的局面导致整个元朝的统治模型直接让底层人民活不下去，所以短短几十年后就开始不停地爆发起义，而且是摧枯拉朽式的起义。起义最重要的第一件事是有多少人感受同样的痛苦，第二件事是这些痛苦的人能否连接起来。

这就不得不说到清和元之间的重要差异。当然，我们今天对清的痛感首先来自于这个事没有了结，甚至很多遗老遗少还在吹嘘自己有通天纹，觉得自己是贵族血脉高高在上，这让人格外恶心。但这只是情绪感知的一部分。清的统治模型和纯粹的逐利模型不一样，它不是单纯要钱——当然它也要，而且要得特别狠——但同时它也希望能减少盘剥中的中间商。注意，当年的蒙古王公不在乎给中间商分利，只要自己手上有就行；但后来的满洲八旗不想给任何中间商分利，所以变成了所有八旗骑在其他所有人脖颈子上拉屎的局面。当然，你也可以说八旗当中没有直接掌握权力的人成了新的中间实力阶层——我也这么说过——但它跟其他实力阶层不一样，因为八旗的实力不是自己拿到手上的，而是由中央财权分配下来的权力，所以理论上是被中央王权豢养的鹰犬和走狗。因此，这种治理方式让内部结构非常稳定，因为他们持续分赃；同时，又把这些人网格式地派往全国各个要害地段进行镇压，所以也有工作。此外，为了避免重蹈覆辙，他们努力让中国人遗忘自己的过去，编造历史故事，切断中国人跟祖先的连结，缔造大量文盲，让中国人没有能力造反和抵抗，迫使整个社会生产力倒退。要知道，到清的时候连平均亩产都下降了——我们是全世界最擅长种田的人，平均亩产都下降了，倒退到这种程度。

所以元和清有什么区别？目的不一样，操作的精细程度差很远，社会管理方式不一样，造成的破坏更不一样。最不一样的是：一个结了账，另一个到今天还在餐桌上恶心我们，还希望我们认它、爱它。好，欢迎来到直播间。这里是带货直播间，我们带一个采访集，叫做《他们相信中国人比哥伦布先到》。到底是谁进行了所谓的地理大发现？地理大发现这个名词本身就非常西方中心主义，因为这个名词很烂——那片地一直在那儿，地上还住着人，他们有自己的生产、团队、文化、族群和特征，结果你硬说“发现”，就好像原来没有一样。我不喜欢这个词，但至今还在用。

这本书是一位作家访谈了全球多位学者，都是有身份的有识之士。他们分享对于到底谁才是最早进行地理大发现的人的探讨。最可贵的是，采访的人当中绝大多数都是老外，而且是正经学者，有学识、有靠山；也包括一些华人。不少学者认为中国人才是地理大发现的开创者。想了解这些学者的看法，可以入手这本书。今天在直播间分享的内容是：要推进地理大发现，就需要大航海；要大航海，就需要船和航海技术。哥伦布时代的所谓大航海，他的船稀烂，船员也不知道如何航海——他们连坏血病都解决不了，不知道海上该吃什么、喝什么。这只能说明他们没有先例，没有前人探路，没有长期海上生活积累的经验，是一群没有航海经验的人闯出去就出发。没有好船、没有生存技能、没有航海技术，凭什么搞地理大发现？一个人什么都没有，就跨越大洋去寻找更远的地方，这不科学、不唯物、不现实，也不符合人类正常的生存动机。好比今天花3500买了一个小电驴，明天就跑撒哈拉大沙漠去探险——哎呀我的妈！能这样吗？不能。这不是有没有勇气和野心的问题，而是当你只有小电驴时，应该先骑一条街道、半座城市，探讨最远行程和该带多少东西，循序渐进找到上限，才知道能开到哪里；而不是刚买小电驴就冲进沙漠说“我要远航了”。然而他们连街道都没走出去，那就没办法了。

刚看到一个视频，说梵蒂冈有明代史料，肯定有呀。不然他们当初派那么多人到中国来，你真当他们是来传教的？就他们那种神话故事有什么值得信？有什么好听的？你没到倒霉催的地步，会去相信那些狗屁倒灶的玩意儿吗？一个神话故事都讲不明白的团伙，跑中国来传教还能产生多大影响？换成谁都不信。中国的哪个神话传说不比它完善？中国的道德文章哪一个不比利索？轮得上它吗？传教根本轮不上。所以那时的所谓传教士来中国到底干啥？给我们传播知识与文化——传播了很多他们自己那里没有、而我们自古以来就有的知识和文化。所以这个传播到底是拿走了还是拿进来的？你说得清楚吗？行了，到点了，你们该下单下单，下播再见。
