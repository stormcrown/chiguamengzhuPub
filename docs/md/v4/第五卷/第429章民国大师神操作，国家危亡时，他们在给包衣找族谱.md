# 第429章 民国大师神操作，国家危亡时，他们在给包衣找族谱

小说它就是个文字游戏，一个文字游戏。你不允许猜谜语，要当历史书来看：你必须在历史事件当中找出来一个除了名字、其他的东西都一模一样的事，否则的话那就不对劲。这种思路是谁灌输的？民国的大师不是说它的作者必须得姓朱，或者必须得是吴梅村，而是它的作者绝对不可以是民国那些奇奇怪怪的大师们所总结出来的“包衣公子”——这才是核心。

红楼的作者可以是任何人啊，哪怕是现在说他就是朱由检写的啊，那万一呢？是吧。兴许啊，树上没钓着啊，冒领了，这事儿不好说啊。咱们小说故事桥段当中不是有好多——你就是猜崇祯复生诈尸了、诈尸了，都比猜曹家小公子来得合理。为什么？因为红楼这部书，哪怕是他暴露在人世间公开的这80回里面，也到处都是在影射式的谩骂满清。嗯，红楼梦是吴梅村写的？还是那个可能性更大。不行的话你就说朱由检写的吧，行吗？还真有人猜啊，真有人猜说朱由检最后好好活着呢，红楼是朱由检写的都有啊。

但是有一条是毋庸置疑的：红楼它是一个集体作品啊。啥啊？红楼呢，反正很明确的一样啊，红楼是一个集体作品，它不是单一某一个人的作品。但是呢，呃，尽管它是集体作品，但是大家不会把自己的真名拿出来的。所以呢，我不是很建议说非得要去找出它的作者是谁。今天对于作者的所有的讨论，我也只集中在一个点位上面：就是它的作者绝对不是江宁织造家里面的小公子，他所写的也绝对不是江宁织造这个包衣奴婢家庭的家庭内部纠纷。

所以你会注意到一件事情：民国时期的这些大师们，为什么在一个整个中国都风雨飘摇、千疮百孔的这样的一个时代，专门纠结起一群人，非常非常不合时宜地去追究一个小说的作者，并且一定要咬死了它的作者必须是某一个人啊？这件事情是不合理的，知道吧？就是一个小说它可不可以没有作者啊？一个小说可不可以没有作者？完全可以啊。一个小说是完全可以没有作者的，它可以没有真实的名字，它可以是好几个人合伙，它可以刻意地不写自己的名字，可以用一个笔名让人找不到它的根由，它可以就是一个无名氏，很正常，知道吧？啊，一个小说它没有作者、作者不肯署名是很合理的现象。所以非得说追究啊，这个作者必须得是谁、这个作者到底是谁呢？猜谜就行了，但是不需要咬死它。作者必须得是谁，反而真正逆潮流而动、完全不合逻辑的一种行为：就是民国的那些大师，在整个国家十分动荡、特别需要文化人去进行建设的一个时期，特意地揪起了一本几百年前的小说，并且一定要为它找出来一个清代的历史背景和清代的作者。这件事情是完全不合时宜，也是一个完全不合逻辑的事儿。能够在这种紧张的时刻做这样的一个闲篇闲笔，说明这就不是一手闲棋。所以真正要讨论的是，他为什么非得要求红楼作者是包衣奴才家里的小公子？这才是最糟糕的一件事情啊。

嗯，清的这个文字狱是很可怕的。文字狱就是一个这样的小说里头，他没有作者是合理的事情，因为清的文字狱是极其恐怖的程度：只要他封文有事，就可以把你家的九族，连同朋友家的九族，连同朋友的朋友家里面的九族都给你拉干净——就已经到了这样的地步。所以在清代里面包含了影射、包含了指桑骂槐、包含了直接骂胡虏的这些内容的小说，在当时人们是看得懂的。今天很多人在那里揣摩，说你不能够猜谜语。不好意思，小说它就是个文字游戏，一个文字游戏。你不允许猜谜语，要当历史书来看，你必须在历史事件当中找出来一个除了名字、其他的东西都一模一样的事，否则的话那就不对劲。这种思路是谁灌输的？民国的大师。民国的大师在一个国家十分动荡的时代，专门花巨资、花巨大的力量，要求红楼这个小说必须有一个作者，同时要求它里面的故事不可以是虚构的，必须是一个真实的事件。你知道这有多异常吗？啊，你知道这有多异常吗？

所以现在去解决作者的问题，不是说它的作者必须得姓朱，或者必须得是吴梅村，而是它的作者绝对不可以是民国那些奇奇怪怪的大师们所总结出来的包衣公子——这才是核心。红楼的作者可以是任何人啊，哪怕是现在说他就是朱由检写的啊，那万一呢？是吧。兴许啊，树上没钓着啊，冒领了，这事儿不好说啊。咱们小说故事桥段当中不是有好多诈尸的、假死的啊，万一呢？猜呗，只能靠这么猜是吧。但是即便是这么猜，它的合理性也远远高过于曹家小公子。你就是猜崇祯复生诈尸了、诈尸了，都比猜曹家小公子来得合理。为什么？因为红楼这部书，哪怕是他暴露在人世间公开的这80回里面，也到处都是在影射式的谩骂满清啊。它整体上面就是在那里不停地骂街呀，啊，红楼一直在骂街，从头到尾都在骂街的一部书。然后你说它的作者是曹家公子，然后曹家还活着，还有后人，还留下了族谱，族谱上刚好没有这个辈儿——这辈儿就没有这样的一个曹家公子，你说有意思不？

嗯，听说红楼说的是明朝灭亡时的事，真的假的？嗯，红楼啊，对于红楼的解析呢，一贯是分成两个流派的。红楼的解析分成了两个流派：一个是索引派，一个是考据派。这两个流派都客观地存在，在清代的时候也存在啊，在清代的时候也是存在的。然后呢，在清代的时候，考据派就已经开始出现，有人认为是曹家小公子的作品了。哎，就是考据，这群人在清代本身就有。但是清代的时候更加盛行、更加普遍的是什么人呢？是索引派。这两派有什么区别？考据派的特征就是说，咱们得找出来作者姓甚名谁，是谁家的小谁，然后呢，要考据出来这个故事当中的这些情节是谁家的故事啊，是讲的是哪天晚上有哪个大臣所发动的某一场晚宴——这个东西就叫做考据派，就是拿它直接地当一个真实事件的记录名册来看。这个玩意儿就叫做考据派啊。

然后呢，还有另外的一个，实际上在清代的时候真正占据主流的是索引派。索引派是什么？索引就是猜谜语啊，索引派就是猜谜语。那猜谜语是干嘛？就是，哎，我这个地儿说了一句什么什么话啊，例如说让薛家人住到了东北角上面的梨香院啊，有这么一段，是不是？哎，薛家人过来借住，借住了之后呢，赖着不走了。啥原因啊？不管，反正他就是不走了，就赖住了，赖到哪了呢？赖到了东北角上面的梨香院去住。然后后来啊，后来这个院子呢又给谁住来着啊，给一群小戏子来住了。好，然后就在这样的一个过程当中，你有没有发现啊，在整个的园子专门挑了一个东北角，安排了一个过来借住的、犯了事儿的、流亡的人家，来了之后他就不走了，而且家里面还专门编造了一个金玉良缘的传说，来惦记家里面最嫡出、最重要、最珍宝的一个公子宝玉。你就说这个过程到底有没有在骂街？明晃晃的骂街嘛。放到今天可能有很多人啊不会那么敏感，但是放到了清代的时候，就光东北角这仨字儿，你知道会引发多少人联想吗？就清代所有人都在那个地方，不敢说真话的一个时代啊，你在这里面去直接讲啊，哪个地儿的东北角赖着住了一家人不跑了，还觊觎家中的长公子，我就问你，这件事情本身不够引人遐想的吗？在一个文字狱横生的时代，大家都会直接进行联想的。这个是什么呢？这个就是猜谜啊，根据里面的只言片语去猜测作者的本意是什么，这个东西就叫做索引派。

因此你任何的时候啊，任何的时候就需要考虑一个问题：一个小说的解读，一个小说的分析，你到底应该用猜谜语的方式，还是应该用翻别人家的族谱的方式啊？尤其是当你明确地知道清代的文字狱十分猖狂的时候，在清代说一句真话都可能举家没头颅的这个时代里面，你觉得到底是考据可靠还是索引可靠？到底是一个书里面作者隐去真名可靠，还是一个作者想方设法地让你能够想得到他到底户口本登记在几巷几号来的可靠？所以很明确的一点，即使是从小说的基本逻辑分析来看的话，也同样是这个道理：就是索引派，你不管它里面猜谜猜得对不对，但是小说里面有谜语、小说里面有影射才是正常现象，因为小说它就是一个文字游戏。但是前些年红学盛行的时候，谁要是拿索引派的话出来说事的话，那些红学家就会指着鼻子说：“你就是在搞文字游戏！我们这么郑重其事的，你怎么能玩文字游戏呢？”是吧。但是问题在于小说，小说它不是文字游戏是什么？小说它本身，小说的本质就是一个文字游戏。
