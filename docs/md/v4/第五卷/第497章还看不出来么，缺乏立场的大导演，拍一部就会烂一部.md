# 第497章 还看不出来么，缺乏立场的大导演，拍一部就会烂一部

### 整理后的易读段落

好，我们继续。刚才有人说那个什么方阵啊——其实是黑龙江省方正县，这个地方闹妖好几回了。早几年间的那个砸碑事件就是方正县的。大家知道砸碑事件是啥事吧？哈哈。

那个黑龙江有一个县叫做方正，就是方方正正的那个“方正”。来给大家分享一下：那个县之前设了一个园子，在园子里立了一个碑。那个碑上都是日本当年开拓团的名单——墓碑啊，纪念碑。上面的成员全都是日军当年的开拓团，知道是干啥的吧？明白吧？就是把日本那边派过来的开拓团的人，用祭祀礼纪念的格局——不是谴责哦，是纪念、是怀念、是缅怀的这样一个视角，立了一个碑，还放到了一个园子里。之后呢，被普通百姓发现了，然后就引发了很多人的不满，全中国都不满意。当地采取的方式你知道是啥吗？他把那个园子锁起来了。嗯，媒体已经注意到这件事情，并且当时刚刚开始出现互联网，有一些网友也发现了，大家都表示了极大的不满。于是当地就把这个园子锁起来了，把这个小山包锁起来、圈起来了，还派了人防守，看门防守，就给锁起来保护起来了。

然后就有几个来自天南海北的网友齐聚一堂，他们互不相识，就为了这一个共同的目标走到了一起，干了一件事——就是要去砸碑。他们从各自的地方、各自的岗位上聚集到一起，到了黑龙江去砸碑，越过了重重守卫，千方百计地进去了。进去之后发现，哎呦，当地做的那个碑，那个石头质量嘎嘎好，砸不动。费了老大劲了好几个爷们儿，费了老大劲了，砸不动，连皮儿都没破。于是就拿漆又给泼了一下，最后就只做到了这样的程度。后来又过了好几年，这件事情闹得特别大，当地呢，就表示说我们把它拆除了。诶，欢迎来到直播间。总之人没事，就是那几个壮士。那几个壮士当时呢，号称要把他拘起来，理由也都是现成的——你破坏了这破坏了那，那随时随地都可以给你揪起来嘛。但是最后呢，都免于处罚，都给放回去了。

嗯，什么玩意儿？前几天群里有人喷731的电影，然后紧接着就推荐《辛德勒的名单》，都无语了。你要是论电影的制作水平来说的话，那《辛德勒的名单》还真的是水平很高，承认吧。尽管说他讲的故事本身是颠倒黑白的垃圾事，但是呢，他这个叙事很好，影片也特别好。所以认真地说起来，我真的是觉得，我们的国内就缺少这种有立场、有主张的导演、制片、编剧。这种类型的影片就只能这样子的人出来拍，而不是那种游戏人生的下三路的导演出来拍。那些导演他能拍得出这样子的民族伤痛记忆吗？他拍不出来。他拍一个烂一个，拍完了就跟那个什么《饥饿游戏》似的，还是叫什么游戏？《鱿鱼游戏》。反正他就拍不明白。为什么？因为他自己心里头没有这根弦的人，他怎么可能拍得明白这种影片呢？

那啥，现在都是破圈的导演。我可以分析一下，为什么现在很多优秀的导演都是破圈出来的——哈哈，为什么非科班的、破圈的、冷门的导演会拍出好电影来？为什么会这个样子？为什么那些热门的、科班出身的、在圈子里面的非常有资历的导演拍一个烂一个？主要的原因就是：电影这玩意儿其实是一个巨大的利益分配链条，能get出来吗？嗯，有一个叫蒋同学的，他说你说话声音很像日本人，你是不是日本人？呵呵呵呵呵呵，哎呀，老厉害了，老厉害了！嘿，那个啥，日本人说话啥啥动静？你们能跟我分享一下不？哈哈哈。哎，我特别关心这件事，我特别有兴趣，我很想知道他们都是什么动静，哈哈哈哈。

我们刚才讲的是那个电影啊。我们这些年呢，一定会注意到一件事：我们的很多的冷门的、跨圈的、突然之间横空出世的导演，他的作品的质量会超出你想象的好。这说明什么？这说明中国的电影市场不缺技术——这是第一件事。一个跨圈的导演一来就能出一个像样的电影，说明我们不缺生产端、不缺技术、不缺服务人员，甚至也不缺money。你要知道，一个新生的导演，尤其是跨圈的导演，在以前没有多少样板间展示的时候，他想要拉到money本身也不是很容易，想要拉到团队本身也不是很容易。但是他都能做出大的影片、好的影片来，这说明我们的内部的生产端上面，实际上是不缺money、不缺技术、也不缺人才的，这些东西我们通通都有。要生产它需要的一切东西，在我们这里都是齐全的。所以我们缺的是什么？我们的电影产品不容易出现攻心之作，我们的短板到底在哪儿？我目前看啊，短板在于它有内部的利益分配链条，而这个利益分配链条掌握在了一些跟中国人毫无共情关系的人身上。

啥环境？要知道这个利益分配这件事情，你可以去稍微算一算账，也不需要很细，毕竟像我这样不会算数的人都能够盘得明白，所以你不需要算得很细。那什么样的账呢？那就是在电影这样的一种庞大的体系当中，你每一部电影都要有一个巨额的投资，然后这个巨额的投资进来之后，你要让它能够变现。而在变现的这个流程当中呢，需要什么？需要广告商，然后需要院线的支持，还需要各种媒体平台的支持，还需要一些自媒体账号的支持——就是所谓的流量支持。这些东西都打通了，那就意味着你这个电影至少亏不了，他有变现的渠道，他可以回来，他可以出去的钱有回头钱，这是最重要的一条利益分配链条。那在这个利益分配的过程当中，最需要的是什么？媒体、流量。过去是传统媒体，现在是互联网媒体。无论怎样，它其实都是耳朵、眼睛被谁掌握，这个叫做流量池。这个东西通通都可以算作流量，某一个维度的流量。所以一个电影你要最终能够变现成功，你必须掌控某一种流量的发行通道，这是必须要有的东西。那么谁手上有这个流量的分发通道呢？谁手上有？以前是传统媒体，传统媒体出来说哪个电影是最好的，大家就去看，这个东西叫流量。然后现在呢，就是各大的流量平台。这个东西就是我们的流量分发值。所以为什么很多你看到的那些专业的电影人，他会去找几个小鲜肉、小花小草过来拍，尽管这些人肉眼可见的演戏的水平一塌糊涂，但是他一定会找他过来。为什么？又贵又不好用的演员，你用它干什么？他有流量池，这个人脸只要出现在这儿了，他的流量就为你打开了，这个东西呢，就是他天然的变现通道。而只要电影这个东西是通过这样方式变现的，那么掌握流量通道的人就可以决定这个电影的叙事方向。而我们现在掌握流量通道的都是什么人呢？你只要看一看，我的直播间是在什么样的话题上面会被抬，像我的人体上都是什么死相的，你就知道了吗——什么人在把控互联网的流量方向。所以为什么你看到的那些电影大制作更精良，导演也很不错，演员也很专业，但是最终做出来的东西总是不对味儿，看了之后心里头难受，甚至会看了膈应。为啥呢？因为他所期待的变现流量池跟你没有关系，他要的那个变现的流量池跟你没有任何的关系，它推动的是他的那一块的流量池，而那块流量池正好就是来恶心你的。

这里是一个带货直播间，我们来带一下这个张纯如的作品。来带张纯如的作品啊，之前我就带过很多次，但是还有没买的一定要收藏。连这个书家里面都没有，那你到时候想看的时候，你去哪儿找呢？这个书啊，是值得典藏、值得珍藏的。我倒不是说你非得把它逐字逐句地念，注意，这个书是真的非常非常需要你收藏一本的。看的时候会很痛苦啊，可以不看。什么时候聚起了勇气了，你就看一点，这些都属于可以慢慢看的那种类型。它就不是一个说你必须此时此刻一定要看，没有这种规矩，你可以慢慢来。甚至说我就看了之后我翻开我就头疼，也可以不看，但是依然还是需要你有一本。为什么这个样子？承平日久啊，承平日久是要出问题的。承平日久，你的思维方式会陷入到一种太平盛世之下的文明重症之中。连这个东西你都理解不了的时候，会很难很难过。你不容易理解什么叫做丛林社会，你不容易去理解什么叫做敌我关系。我有的时候说有一些矛盾是内部矛盾，有一些矛盾是敌我矛盾，有一些毛病是叫做敌我矛盾的。嗯，什么书名？我不是弹出来的吗？这边有啊。张纯如还有很多的作品吗？也没有很多的作品呢。我光说书名的时候，你就应该知道了呀。嗯啊，啥？抖又被抬了？我不知道，我不知道。他刚要审查我，我就把他给抬了，所以他到底抬没抬我，我不清楚。我这招是不是怪厉害的啊？呵呵，明明是他要抬我，结果我操作的方式是我把他抬了。呵呵呵呵，哎呀，这娘们好狠啊。
