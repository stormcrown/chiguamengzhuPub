# 第440章 蒙主网络安全课

我看一看啊，又有人教育我了。我瞅见了，有人教育我：“希望主播永远坚守有良知的底线。”听起来像是一句好话，但是通常情况下，说这话的人啊，一般就代表着说“我觉得你要戳破底线了”。好，嗯。爱国赛道没问题，但是一味利用爱国情绪歪曲事实不可取。你看吧，啊，我就说了那个开头的一句话，乔治很中肯的一句话，意思就是我要教训你了。那后头教训的东西，麻烦你拿出点实证出来，拿出点具体的内容出来啊，不用拿那种训斥啊、听起来好像有道理的一句话来训斥。你说每一句话、每一个字对不对呢？啊，每一个字听起来都对啊，不要裹挟情绪。你说对不对？对。但是你冲出来说这句话的前提得是我裹挟了某些人的情绪，不然你跟我说这干啥啊？你得先出来说啊，你哪一句话是裹挟了别人的情绪？不然的话你就别说这种东西，知道吧？谁是玩情绪的，你自个儿心里头有数啊。要是实在连这都没有数的话，我建议小学的补习班去上一下。嗯啊，来来来来来啊。啥设置前提？我说句公道话，意思就是这个啊。我来说句公道话吧，意思是啥？他要拉偏架了，知道吧？他要是不拉偏架的话，他干嘛说我来说句公道话呢？知道吧，正常人不需要说“我来说句公道话吧”。嗯嗯，正常人直接你有话就说话就行了。有人说“我觉得你这个事儿说的不对”“我觉得他说的对”“我觉得那个谁说的这个是对的”，这是正常人啊。正常人点评就是“我觉得某某某人说的更有道理”“我不是很支持你这个看法”，这些都属于正常人，无论支持还是反对都是正常的，都是OK的啊。但是冲出来说“我来说句公道话啊，大家不能够那个什么拉偏架”，后续那就叫拉偏架了。他要是不拉偏架，他就不带说这句话的。

嗯，啥？为什么要喊真名？不知道网络环境啊？我知道，又是什么奇悦呀、齐悦呀，你失去了网名了呀！你这个齐悦呀，来，齐悦你接着说。来，我给大家分享一件事，这件事情是咋回事啊？啊，他们之所以说我叫齐悦，是因为这一次西史辩伪的论坛上面的那个名牌标签上是这么写的，然后主持人呢，也专门念了什么“吃瓜盟主齐悦女士”，知道吧。但是啊，我不叫齐悦。至于说为什么那个上面会这样写，这个也有缘由啊。就是呢，我曾经进了一个学术论坛——就一个群啊，一个学术的学者群。然后呢，那个学者群里面，他说我们这里都是可靠的人，必须要那个写实名啊，不写实名的话不行。其实我当时犹豫了一下，你实在不行，我就不进来不就完事了嘛。我就跟那个中间拉我进群的介绍人说了，我天天在网上挨网暴，我是不可能跟任何人说我的实名的啊。他说没关系，那你就编一个。所以我就编了一个，知道吧？我就编了一个就放在那了。然后呢，这一次是西史编委论坛啊，他也没有人问我是要用网名还是要用真名，知道吧？实际上在这个过程当中，没有人问过我这件事。然后就直接可能就是从那个群里面把这个什么名头就给复制粘贴，直接贴到了这边来，就贴到了这个公告栏里面。他们在做什么标签啊、做什么公告的那个解说词啊，就直接把这个玩意儿粘贴过来了。于是呢，他就开始念啊，“那个什么吃瓜盟主齐悦女士”，弄得好像是我授权说我就叫齐悦女士似的，我也不叫这个呀，知道吧？所以为啥说那个什么“我预判了你的预判”呢？我就知道不管那个群有多正经，也不管组织群的这些科学家们有多正经，也不管去组织活动的组织人员有多正经——谢谢叶啊——总而言之，这互联网上的马仔数量庞大，这些学者们、专业的搞研究的学者们啊，他们根本就识别不了马仔好吗？他们也防不住马仔的好吗？所以他甚至不知道什么叫做马仔，也不知道马仔在什么地方。我一个在互联网上面一天到晚地被人家围攻、被人家整活、被人家动辄要威胁开盒的人，你不知道什么叫马仔，我知道呀！所以我是不可能把自己的头颅咔嚓一下就送到你的祭坛上面去的，知道吧？所以为什么会出现这样的一个名号呢？他们为什么要在互联网上天天嚷嚷这个事呢？就是因为他以为那个是真名，可以拿真名来威胁我啊。所以他在公屏上不停地滚动，到处瞎嚷嚷干啥？不好意思，我早就知道这个互联网上到处都是暗桩，到处都是马仔，到处都是二五仔。虽然我不知道是谁，但是没关系，我也可以让你不知道我是谁啊。啊，嗯嗯，我看一下，我看一下都在倒腾些什么东西啊。你不叫厉飞宇，你是谁？嗯，好啊，我反正大概就是这个样子的，等着他们，等着他们继续扒啊。谢谢修无山的你只能看出来一件事情，知道吧？只能看到一件事情，就是他们埋伏的人可多了呢。无论你觉得这个地方多么的干净，无论你觉得什么这个群体里面每个人我都是认识的呀，如何如何的，没什么区别啊，一样一样的。
