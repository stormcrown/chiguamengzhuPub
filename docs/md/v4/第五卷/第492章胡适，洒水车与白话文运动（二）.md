# 第492章 胡适，洒水车与白话文运动（二）

胡适是有贡献的，这点应该承认啊。有人知道胡适有什么贡献吗？除了培养了很多很混账的“孝子贤孙”之外，除了破坏了我们整个中国文人的脊梁骨之外，他还有什么贡献啊？我想起来了，我想起来了——千万不要这样说，胡适之是有贡献的。我想起来了，每天我在街上面被洒水车追着跑的时候，都是胡适的歌——《兰花草》啊，来来来，《兰花草》！贡献啊！贡献贡献！贡献了所有洒水车的配套音乐。我一听见这个音乐，我就知道我要找小巷子钻了啊。《兰花草》走！嗯，有贡献，有贡献。我，我想起来了啊。好的好的，你不能说他没有贡献，棒棒的。嗯，很好很好。

嗯，洒水车和《兰花草》有什么渊源吗？你不知道是吧？那我讲一下啊。就是那个胡适之，当年呢，推崇过一个白话文运动啊。一般来说呢，白话文运动被说成是胡适之的重要贡献，但是这个事跟他其实没有什么关系，他就是当初捡了现成的，然后呢，就是举了这么个旗子啊。但是咱们先来弄明白一下：为什么在民国的时候要搞什么白话文运动？然后呢，我们就得先说到底什么是白话文。白话文其实就是口语转变成文章，这个东西就叫做白话。白话是和文言相对应的。文言是怎么诞生的？文言就是一种书面语啊，文言就是单纯的书面语。为什么要有单独的文言？为什么不能把说的直接写下来？

我们是一个自然诞生的文字载体，自然诞生的文字，在早期的时候，文字的书写能力非常的差。甚至我们很长的时间，有可能是出现在一些石头啊、泥土等等上面的刻画，这种临时的书写的东西有很多，不排除这一点。所以我们的书写载体，当它能够形成稳定的材料的时候，你就会发现，哎呀，这么珍贵！如果说我在上面啰里吧嗦地说一堆，你那块木头板子很快的就给塞满了。这么贵的东西怎么办呢？所以就要精简啊。所以文言呢，一方面是有标准文法组合方式，另外的一个方面呢，就是它要求简约，因为你不能够长篇累牍地就塞进去，不然的话完全超标，又贵又沉重，携带传递都不便，你就没有办法达成它的要求。所以文言系统诞生在这样的前提条件下，形成了一个完整的发展脉络。而口语呢，则是随着时间的流变，人和人之间的交流在不断的发生演进。所以白话这种东西一直都有，文言这种东西也一直都有。

那么白话文是什么时候形成的呢？实际上在宋代的时候，就已经开始有不少的白话文了，然后到明代的时候，白话文形成了一个完全的高峰期。为什么会出现白话文？是因为这个时期，我们的印刷产业获得了飞速发展，于是就出现了一种状态：印刷变得非常的便宜，人们就可以用文字的载体产生大量的商业产品。宋代开始就出现了印刷产业的大爆炸，然后呢，就出现了报纸产业这种流行的小报，那个产业链条嘎嘎的好啊。就要求说普通百姓你得认字，得天天看吧，看小短文吧。普通百姓认字，看小短文，你让他去“之乎者也”，难度非常大。这个时候白话直接转成文章啊，就开始成型了。所以到了明代的这个小说大爆发的井喷期的时候，你会发现绝大多数的明代小说都是白话文小说，他就是用口语和半口语的方式来进行书写的小说文章，这些都是典型的白话文的运行方式啊。它之所以诞生，是因为普遍性的识字和大规模的印刷产品的发行。

然后呢，到了清代的时候，这个玩意儿就完犊子了，因为他不准你搞这个东西。你想一想，就满清朝廷的那种管理方法，他说你可以印小报，那你猜小报上印的是啥？反清复明，驱逐鞑虏——保底是这样的呀。清风不识字，这好文雅的嘛，保底如此。所以他根本就不让你搞这种民间的这样的什么小报的、什么乱七八糟的产业。于是呢，当印刷产业被严密地监控起来，数量变得非常稀少，然后民间呢，又由于考试的等级制度，所以导致了很多学习上面的监管，然后又对很多的学社书舍进行了打击，最后造成了大面积的文盲。大家不认字，不能够上学堂，也没有什么书刊杂志可以看，小报更是早早的就断绝了。那么这种情况下就变成了文人，人与文人之间有文字交流的需要，平民之间没有文字交流的需要，也没有文字阅读的需要，于是白话文没有应用的场景。这个时期的小说甚至都开始出现了文言小说。文言小说这个东西在明代的时候非常少，但是到了清代的时候，文人写小说又开始拿起了文言的方式来进行书写啊。这就是清代的时候，文言文的使用开始下降，普通的百姓呢，这个时候就没有阅读理解那个普通的白话文的能力了啊。谢谢生身父亲，你这 ID 起的啊。

所以到了这个民国时期，为了快速地推广教育啊，你想要快速地推广教育，让大家认字之后，然后再去学文言，这流程，天哪，教都教不起。所以呢，又开始重新推行明代其实就已经有了的白话文教育。所以这才是为什么在那个时候会有白话文运动。而白话文运动的时候，胡适之非常的支持，所以呢，就算作他的功劳之一。其实在那个时候是必须会有白话文运动的，也一定会出现，无论他举不举旗子。你想要普及教育，就不可能全面地都推行文言，一定会有白话文进入其中。这件事情是历史在早几百年前就已经印证好了的客观现实。

然后啊，注意了，那么这个时候为什么说这个胡适他重大的贡献是《兰花草》呢？那就是他在当初号称说，我要推广那个什么白话文，然后还是有一些老学究说，为什么要用白话文？那玩意又不精确，那玩意儿又不优美啊，是吧？然后这个时候，胡适之为了表示他对白话文的支持，就开始写白话文的文章，但是他写文章的水平相当之次啊。有兴趣的小伙伴可以去看一看，他写的白话文的那个档次，真的是照鲁迅先生差180条街的那种水平。但是在他的这些各种各样其实水平并不高的白话文的文章当中，有一首白话文的诗看上去还不错，能够看，就叫做《兰花草》啊。然后他叫什么？“我从山中来，带来兰花草”啊，大概是这么个诗，你们可以自己去搜啊。然后呢，就是这么一首诗之后呢，被改编成了一个歌曲，叫做“哒哒哒哒哒哒哒哒哒哒”，就这么个歌啊。然后呢，这首歌在新中国的时候，被纳入到了洒水车的提示音乐里面。所以我刚才说，我一听见《兰花草》的时候，我就知道要往小巷子里面跑了，因为街头洒水车要出现，他们要洗街道了。棒棒的啊！那不光是这个洒水车要用《兰花草》这个歌儿啊，实际上呢，我们的海警船上面要进行高压水枪喷射的时候，也会使用《兰花草》这个歌儿。也就是说它和水是搭配的，就是这首诗很水的意思。大概如此啊。

诶，好，欢迎来到直播间。哎，我为了讲一个洒水车，讲了这么长一段。我这里是一个带货直播间，带货带的都带得不认真，净讲洒水车来着。家里面有人洗锅的没有啊？有没有勤劳的小伙伴？家里面的锅底黑了的啊，我给大家介绍一个锅底清洁产品啊——锅底小面膜，老好用了啊！尤其是铁锅啊，尤其是铁锅，底下一黑起来还厚，最后呢，就会造成整个的那个锅呀，看起来就不像口锅，看着就头疼啊，就那个玩意儿。然后怎么弄掉它呢？以前我是用小铲子，那玩意太累了，我就不想铲。后来呢，我就啊，我终于发现了一个好产品啊。这个好产品确确实实处理锅底灰的利器，解决的方法特别简单啊。只要说你那个锅不是铝锅都能用啊。它像是一个面膜袋一样的，一边是液体，一边是面膜纸，你把它们摁通，然后呢，在里面让它浸泡透了啊，然后把那个东西扯出来，敷在你的锅底上面，静置一夜。第二天的时候，如果锅底灰不厚，冲一冲就掉了；如果很厚，再用钢丝球稍微擦一擦，带一带就干净了啊。它的清洁效率非常好，而且不伤锅，对于各种焦化物、碳化的这些污垢效果非常之好。除了对铝制品不太友善，其他的东西一概都可以使用啊。效果嘎嘎的，加油！
