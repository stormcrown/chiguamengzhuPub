# 第487章 百越成了民族？又是谁在编这种两眼一黑的瞎话

嗯，说一下百乐吗？有带节奏说姑苏、余杭等地名不是汉语命名，是百越的壮侗语族，语言就不能是汉语啊。我知道啊，他们特别喜欢说百越，然后弄得好像百越是一个什么族群一样。而且呢，特别喜欢拿当年的吴越争霸说事儿。其实这个事情没有那么麻烦。我说了，所有的历史事件，我的要求都不高，就一件事情：不要双标。我没有额外的要求，就一件：不要双标。

所以，当你说“那他这个地方算不算是汉人”的时候，凡是提出这种问题的人，让他弄清楚界定民族的标准到底是什么。我无所谓他的标准是什么，但是他必须提出来一个标准，并且普遍适用。我的底线就是不准双标。其他的事情我一概不要求标准，甚至都可以对方来提，我都无所谓。这些人在提出这件事情的时候，通常说：“我们那里是什么什么的血脉。”注意了，凡是提“我们这里是吴越后代、百越人”什么的，提的都是血缘标准。如果说他嘴里面的民族是按血缘标准来界定的，那简单——我们现在有分子生物学，中国90%以上的人有着共同的五个父系祖先。如果再往前面溯源，溯源到七个超级祖父的时候，能够囊括中国97%还是96%以上的人口。如果你按血缘来定，那不好意思，所有的这些父系血缘高度一致，来源高度同源的，全部都是同一个民族——华夏后裔，那就是今天的汉民族。当他按照这个来算的时候，你说“我这里是什么百越血脉”，你既然按血缘算，我们就按血缘算，但是你不能双标。当你按照血缘来计算的时候，全中国96%、97%以上的人都是一个民族，那你就全按这一个来画，该改户口的改户口，行不行？支不支持？然后他又不同意了，对吧？他肯定不会同意的。我知道再怎么想他都不会同意，无所谓。我说了，无所谓的，你什么标准我都不介意，但是不能双标。

既然血缘的标准你搞伪血缘论，搞种族歧视，行，那咱们不搞血缘标准，就搞文化标准，好吧？文化标准是什么样子的？使用汉字的算不算是同一套文化体系？自古以来就使用汉字的，是不是同一套文化体系？使用汉字的同时还拜祖先的，是不是同一套文化体系？拜祖先还拜多神，还搞自然崇拜，同时容纳多种神明信仰的，算不算同一个文化体系？同时兼备这几点的共同的文化形式，是不是一个文化族群？按照这个标准，中国百分之九十九的人都是同一个民族，全部都是华夏后裔，都是汉人。改不改民族？你按不按这个标准来做？你按这个标准来做，我们就来改，我支持。改什么名字都无所谓，大家改成一样的就行。我不是非得要求用“汉”，无所谓。你说要重用“中华”就用中华，用“华夏”就用华夏，用“炎黄”就用炎黄，随便你用哪一组字。自古以来就使用这一套文化体系的人，全部都是同一个族群，有没有问题？我要求不要双标。这个时候他突然说：“那我们是百越，百越文字呢？”为什么文字、服装、习俗都是一模一样的，你硬说跟我不一样？你什么准则、什么标准？你血缘血缘不敢验，文化文化不敢讨论，结果硬撑出来说“我就是百越后代”。怎么百越？你知道百越是谁吗？你连百越是哪个都不知道，就只知道“百越”两个字。啥？百越成了一个民族了吗？在他们的嘴里面，百越就是一个民族。百越实际上是什么？百越，越是一个地区，百越是生活在这个地区的各个部落。百越他就不是一个族群，他是生活在特定地区的各个部落的一个汇总型的称呼。他们内部的分歧啊，比跟我们之间的分歧大多了，有的时候彼此之间还有世仇。就是因为居住地相对较近，就这么个事儿，结果硬生生弄得好像他们高度一致性，彼此之间如何如何亲如兄弟，不存在的。百越是在一个开化程度不太高、深山老林特别多的地区，散居的各种各样部落给出来的一个总称。为什么要给总称？因为我们认不明白他们太多太多的名字了。每一个地方都有自己的小名字，每一个地方都有自己信奉的一个小图腾：有的信青蛙，有的信什么？还有信蛇的，还有信鱼的，还有信鸟的，什么样的类型都有。他们是拼起来的，各种各样的散装的部落，彼此之间没有什么高度一致性。偶尔有几个部落产生会盟，但是呢，同时也会有几个部落动辄翻脸。这种情形下形成的群体，我们将它称之为百越。甚至这不是他们的自称，知道吧？甚至这不是他们的自称。这是当年中原人士对于这个地方觉得有点分不清楚：怎么这里一个，那里一个，这里一撮，那里一撮？于是把他们统称叫做百越。什么看中国地形图的时候，突然大概明白（现在不太流行用“明白”这个词啊，因为有点呵呵）。嗯，始皇帝是能把能种地的地方都打下来了。嗯，对啊，我们的古典时期的扩张很明显是有一种地理的边界线的，我们是有一个重要的地理边界线的。哎，我们在进行扩张的时候，其实很多时候在意的就是这片地能不能耕种。那为什么要在意是否耕种这件事情？我们会注意到，我们的土地的那种边界线的迁移，就是实控边界线的迁移，因为我们古代的时候算不得有非常标准的边界线。那么这种实控区的迁移，很多时候是和降水线相匹配的。降水线指的就是在降水线以南的地方，它是一个可耕作土地、长时间的轮作，可以长期耕作的地方；而在降水线以北，它的土层很薄，它的降水不足以支撑你的农业生产，于是这个地方的人就只能进行游牧经营。所以你会注意一件事情：我们的实控区，常常就是随着这种降水线的迁移而变迁。我们所有能够征服的土地，基本上说，实控区在哪里？诶，就实控在这个农耕的地方。为什么这样？因为在那个时候的生产力标准下面，只有能够耕种的土地，可以确保这里产生固定的经济价值和固定的基础建设。这是最重要的。就是能够产生基础建设，能够让人长时间居住在上面的地方，对我们来说是最有价值的。那么其他的，你说你不要游牧区的话，那就会造成什么问题？在于我一直强调一件事：就是我们在画古典历史地图的一些人，很多地图学家，现代的地图专家很双标。我跟你们强调一件事情：现代的地图专家非常双标。这种双标的离谱程度就在于，它对于我们的所谓的实控区要求得很严格。我们能够兴建水利、兴建城池、有农田、能够编户齐民，他说：“对对对，你这个样子才算实控区嘛。”我完全接受这样的方式。但是当你按照这种方式去看的时候，北边的游牧地区是没有政权的，也没有标准国界线的，也是没有国土的。为什么他那个地方就建不了这些东西？他们是游走迁徙的，是一直都在游走迁徙的，连里面的人都经常更换的那种——隔个几年抢个草场，原有的部落就消失了，就替换成另外的部落。他们只是都在同一片地区不停地游荡而已。结果他把这一整片随机游荡的地区都划成了他的固有领土，这种绘图的方式是不公道的。如果你按照这种完全的稳定的定居的基础建设的方法来算的话，他们没有实控领土，你就不能给他圈这些地。那么中间的这些草场的地方是什么呢？缓冲区啊，亲！这些草场是什么？这些草场都是缓冲区，战略缓冲区。我们是没办法在上头种田，但是我们也会骑马，时不时的出去遛一弯儿是吧？对方也在这个地方时不时的骑马遛一弯儿。这种地方是典型的战略缓冲区。但是他在画国界的时候，把我们实控区以外的地方全算别人的地方了，于是你就会发现游牧群体的疆域特别广大。这是研究地图学的人在绘制地图的时候采用了双重标准造成的一个问题。所以为什么我们古时候在进行土地方面的征服的时候，主要都放在了降水线这一个附近？主要的原因就是这一片它是可以直接种田的，是可以进行固定的基础建设的设施的。他把这些东西都铺排过去了之后，这个地方叫做100%实控。但是不代表说我们的马没有出去过啊。长城也不是什么所谓的边界线，它是一个保护保护最后的临界点的一个装置，而不是标准边界线。因为对方是一个游牧群体，他是不停地迁移的，他不会住在那个地方，他只是时不时的遛一弯。而我们的兵马也会出城去，时不时的遛一弯，维护安全嘛。都是时不时遛一弯的，这个地方叫什么？缓冲区啊，亲。所以是画地图的人有毛病啊。嗯，啥？长城是前线的军事基地，对，它有很多重功能。它有很多重功能：它可以作为一个防备的军事装置，也可以作为一个瞭望所，然后也可以作为出兵运兵的，然后呢，还可以作为烽火台（它里面包含了烽火台），同时呢，它也是海关，它同时也是海关收卡。它有多重的功能。但是今天很多人在画地图的时候把它当边界线来使。它有很多功能，它唯一不是的就是边界线。它不是边界线，长城根本不是边界线。它是我们更好掌控的一个军事基地。那它外面的地方我们会不会走？我们也会走啊，时不时的带马出去溜一弯儿，扫一扫场子，监控监控。我们是这样走的，对方也是这样走的。结果你说那里是对方的实控土地？它怎么实控的？它实控到我们时不时的可以到上面去乱晃一圈的程度，这玩意儿叫实控吗？但是呢，画地图的人把这个玩意儿叫实控——只要他家的马在这拉过粪，就算实控了。我们呢，我们得修建长城才算实控。我一直很厌恶现在的古地图学家们的双标行为。
