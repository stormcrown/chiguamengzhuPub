# 第433章 皇汉偏右？拜托你先搞清楚左右

皇汉都偏右吧？为啥主播之前说右坏？你们这些啊，就属于理解不了什么叫做左、什么叫做右，知道吧。

嗯，今天被人推了一个大女主电视剧，差点让看过一点明史的道心破碎。你看那个什么大女主，大女主——你看你不管有什么倾向，你都会很难受的啊。你只要脑子正常看大女主的，就没有不难受的啊。

注意了一件事情，所谓的左右这个事，我之前是做过视频的啊。什么叫做左右？左右是方向——听起来像废话似的，哈哈呵呵。左右是方向啊。左是什么？左是激进、快速地冲向前面去。右是什么？右是停下来看一看，甚至往后退啊。所以呢，这两种是方向选择问题，能理解吗？啊，先弄明白这个啊——就是它是方向选择问题。所以你必须先界定一件事，就是方向就是目标啊。先有目标，然后才能选择左右。没有目标的时候是谈不到左右的问题的。

而民族主义或者说皇汉是什么？皇汉是一个民族的自我认知。这个东西有目标吗？没有，能够理解到吗？皇汉他不是一个组织，他是一个人的自我认知，是以自己的民族为荣的、这样的一个自尊自爱的自我认知。自我认知是没有组织的，没有组织他就没有目标，没有目标就没有左右之分。能领会这事吧？啊，当我们是不可能有左右之分的时候，完全就可以存在一个人：他在社会的目标发展上面是趋向于进步的，甚至激进的，但是同时他拥有民族主义，这个中间一点矛盾都没有。如果有人跟你说他们中间有根本性的矛盾，那是因为他刻意地篡改了其中的一部分的概念，为的是将某一些人裹挟到某一种落后的、保守的、愚昧的战车上面去，仅此而已啊。

一个人的自我认知是不需要组织的。我知道我是谁，需要什么组织呢？不需要，只需要知道我从哪儿来就行了，保有自己的记忆就可以了啊。你清晰自己的记忆，清晰自己的好，清晰自己的祖上曾经干过什么，并且充分地认可，这个时候你已经是一个皇汉了。但是你需要什么组织？需要什么方向？都没有，完全不需要这个东西啊。这是两个压根儿不是同一概念的东西。

所以在互联网上说啊，“凡是走这个的都是什么”——我跟你讲，他是在故意地将某一群人、某一群根本没有这种诉求的人，绑定到某一个战车上面去，然后说他们是有组织的，目的是为了摧毁这一群人。就酱，嗯嗯，谢谢汪公社足球，嗯。

郑和下西洋之后，没有再听过大规模的对外海事活动。发达的造船技术为什么雪藏呢？造船技术没有雪藏啊，技术没有雪藏。为什么郑和下西洋之后，就显得这个事烟消火息了？因为之前郑和下西洋的时候，他赚回来的钱不入国库，入的是皇家私库。它是一个皇室专有的国资的涉外贸易项目啊。进来的那些钱呢，不能够被朝臣直接掌控，是被皇室掌控的，是被中央王权掌控的。当他们发现啊，这个东西居然让他们这些当大官的竟然在上面占不到便宜的时候，你猜他干什么啊？所以郑和下西洋所谓的说他劳民伤财不能干了——不是他劳民伤财，而是他劳了这些达官显贵，但是获利他们身上拿不到。好，所以呢，他们要干的事儿就是把这些技术收归己有、私有化，然后呢，变成这些人麾下的产业。所以后续并不是说我们没有大规模的海贸了，而是这个海贸变成了一个走私海贸，叠加了各种各样的海盗的一门生意。啊，啊，啥？下班了啊，对对对啊，这会儿确实是要下班了，再见啊，嗯。
