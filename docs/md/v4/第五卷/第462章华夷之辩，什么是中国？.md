# 第462章 华夷之辩，什么是中国？

还有一个付费的，我看一下啊。嗯，谢谢。

跨雪原穿林海，才意识到我们从小听的演义、评书、刘罗锅之类的，也在宣传满清价值观。大家要适应一件事情啊：就是我们实际上呢，从清开始，从满清开国开始，他在相应的这些思想文化领域上面，一直都是浓墨重彩的。很多人以为对方的文化水平不高，所以啊，最后呢，他就会汉化啊。现在互联网上天天在嚷嚷的人当中，也包括这种类型，说那个从入关之后就汉化了，后来所有的毛病都是因为汉化造成的。他要是不汉化，他继续彪悍，那就要牛哄哄、牛哄哄到全世界去了——小清议一直就是这样吹嘘的啊。

注意了：他们从入关开始就是在努力地拒绝汉化。他们一直在拒绝汉化，一直在去汉化，一直在反汉化。从剃发、易服到修四库全书，所有的东西都在进行这样的逆操作，逆流而动，把整个先进的国家搞到奴隶制，把有科技、有发展的国家拖拽到一个没有创造力的、遍地文盲的国家。他一直致力于去做这样的经营。那在这个过程当中，当他明知道思想文化的领域会给人以启迪，会让人心里面产生一种傲然之气的时候，你说他会怎么做？所以实际上从他入关开始，我们的各路的文化的、文艺的作品，都受到了他的影响啊，全部啊，都受到了他的影响。里面讲述的故事，里面的侠义的精神啊，里面的角色的着装面貌，实际上都受到了影响。

啥服中国当位先汉中央帝国起义，兼汉汉家之土，并报寰宇之心，今日之中国，不过此一朝耳。以为如何呢？这个呢，我专门做了视频。嗯，你要是觉得那个视频说的还不够透的话，那么不好意思，另外的更透的东西就不能说了啊。我能够拿到的最深的分寸就只有到视频里面。其实在直播间里头能够说的东西就更少。

所谓的“何为中国”这件事情，我们最早看到“中国”两个字呢，叫“宅兹中国”，它出自于何尊啊，那么是大约在3000年之前的一个青铜器上面的铭文。那什么意思呢？那就是我们就选定这个地方，作为我们驻扎的地方，然后此地就是中国了。那这个东西是什么啊？这个东西是什么？当我们要去说什么是中国的时候，首先就是什么是国啊。国是什么？国是一群人建设的有防卫力的城池。它最初原本的意义就是如此——有防卫的，有守卫的、有边界的，例如有篱笆，例如有城墙啊，有这样界限，有这样守卫的，一群人所驻扎的城池地域，就可以称之为国。所以我们看国这个字的某一些字体当中，其实里面是一个“域”啊，就是那个“或”实际上它是“域”——疆域的域，就是特定范围被圈起来的。特定范围被称之为国。

那么国是干什么的啊？人为什么要修建城池？其实呢，人们不只生活在城池里面。我们有了一个城池之后，是要辐射到旁边去的，所以会有国人与野人的区别。野人就是后来我们说的“在野之徒”啊——他没有住到城里面，而是在外面驻扎。他接受的福利较小，同时呢，他的责任也较轻。那这种呢，就叫做在野。而国内啊，就是城池之内居住的就是国人，他们的政治权利也更高，福利待遇也更好，同样的背负的责任也更重啊。这个呢，就是国人与野人之分。

在这种范围之后之后啊，你能够知道什么叫做国。国呢，就是一群人他聚落所生存的这一个地方，他最终的目的本身是辐射周边，掌握咽喉，然后呢，把自己的影响力辐射发散出去——这是国要起到的作用。所以呢，不同的国就待在不同的位置，辐射周边。而中国是什么？中国就是所有的国最中间的那一个，他是来调动诸国的啊。这种呢，才叫做中国。所以中国在最初的概念当中，就是天子世居之地啊。因为什么？因为这是号令权。天子能够号令天下，能够发号施令，那么他的居所就成为中国。所以“中”这件事情，它本身就包含着中央发号权拥有者、命名权拥有者，所以叫做“唯器与名，不可以假人”。身为君主，你要能够命令他者啊，然后你才能够起到作用，这个时候这个地方才叫做中国。

所以在今天呢，由于我们的国号和这个简称，跟“中国”呢，直接就重叠了，它就特别容易引发误解，以至于呢，中国被固定化，成为了960万平方公里的土地。啊，注意了，土地是什么？土地是一个国所拥有的财产，它不是国的本体。国的本体永远是人，是国民啊。只有永远追随你的国民，才是这个国的本体。你要去号令天下，你的号令是土地听不到的，他号令的也永远都是人。所以你的人能够辐射到哪里，你的号令能够传递到哪里？能够传递到整个世界吗？如果说全世界都会聆听你的声音，那么发号施令的这一国就叫做中国。所以今天为什么我们实际上还没有达成这一点，知道吧。我们在努力的达成这一点，但实际上还并没有达成这一点。而在这个过程当中，如果有人很刻意地去把土地啊、土地的面积和中国绑定在一起，它就会产生一个误区，说啊，你看那个国多小，那个时候的中国多小，那只有今天才是好的啊。这个土地，注意了，土地是财产，财产多固然是好的，但重要的在于什么？重要的在于财产在不在你的手里啊。

嗯，我看一下，我看一下。嗯，希望不要被稀释了，我们的团结不会被腐蚀掉。哎呀，我的天哪，当我们说我们是要号令，号令发出去的时候啊，他很容易被人利用，成为另外的一种操作方式。就是如果谁来谁都是中国人，全世界都可以住到中国的土地上面，那他就全都听中国的话了吗？现在的很多那个什么高举大旗的人，就是整的这一手——谁来谁是中国人，反正呢，就是只要在这片地上走过的、路过的、打劫过的都算中国人啊，历史上曾经在这个地方穿过地、跑过马、打过劫、杀过人的，也都算中国人。按这种标准来计算的话，到底谁是发令的人，你可就弄不清楚了啊。就像现在谁来谁是中国人，那到底谁是中国人就含糊了。这是什么？这些都叫做唯地域论，反复地强调一件：土地、地域、疆域是人所拥有的财产，我们不能让财产凌驾于人之上啊。
