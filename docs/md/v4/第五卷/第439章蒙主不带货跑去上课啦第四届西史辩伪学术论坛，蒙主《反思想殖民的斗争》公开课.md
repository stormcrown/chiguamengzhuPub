# 第439章 蒙主不带货跑去上课啦 第四届西史辩伪学术论坛，蒙主《反思想殖民的斗争》公开课 #思想殖民

今天讲到这个题目的时候，其实目的是抛砖引玉。因为在座的各位老师，在很多领域当中，都有比我更深远的研究，所以今天我们就从更加外围的传媒以及受众影响的方向，来解析反殖民的反思和反殖民的斗争。

首先看四个字：中华文化博大精深。我不知道大家看到这句话时有什么感想。这是20多年前我读书的时候，我有一位老师，他在讲中华法制史，每次上课开篇，每次讲到一个段落，就会说“中华文化博大精深”。那时我们有什么反应呢？教室里坐满了人，大家哈哈大笑，就像孔乙己走进酒馆时，到处都洋溢着欢乐的笑声，我也是其中之一。

后来局面是怎么发生变化的？我开始做这位老师的课题，起初只是一个泛泛而谈的主题，结果我发现了一个问题。在古今法制对比的过程中，我发现现代社会所需要、所感知到的这些规则，在我们先民的规范中全部存在。当然，他们没有股票市场，没有手机，所以这些规范是没有的，但所有与人类社会行为、与人类日常生活相关的规范，我们自古都有。

然而那时，为什么我们的教室里洋溢着欢快的笑声呢？因为我们其他所有课堂上的老师，都在给我们传递另外的信息，叫做“汉谟拉比崇拜”。当时我们所有老师传递的信息是：我们今天的现行规则从哪里开始？从哪里延续？谁给我们开启了现代秩序文明？现代社会的基础又从何处构建？人类的理性又从何而来？——从汉谟拉比而来。所以当时我在做这篇论文时，感受到了一种前所未有的困扰。你说汉谟拉比最优秀，没有他就没有现代社会的一切，但我分明在我们的古籍中找到了所有的东西，找到了everything。那么它什么时候消失的？我们的老师会告诉我：因为我们落后了，因为那个东西没有用了，所以被淘汰了。那是真的吗？

后来多年之后，我看了一本书，相信很多人也看过。我们日常生活中，通常说一个人有“狼子野心”，是妥妥的负面词汇，可是看过那本书后，开始吹嘘“狼性文化”。你愿意去争夺，你去撕咬，你列队去狩猎，这是什么？这就是最先进的野性，这是人类的本源，这是文明的高级形式。然而狼会种田吗？民以食为天。不会种田的是高级的，种田的是低级的，面朝黄土背朝天的是没意思的，有狼子野心的是开启现代文明的。这个东西是什么？

所以我们现在在谈思想殖民。那什么是思想殖民？思想殖民最终会表现为什么形式？它一定直通一样东西，就是种姓制。我们两千年前的革命者说：“王侯将相宁有种乎？”这个“种”就是种姓制的“种”。中国人最反对的就是种姓制，天生贵种——你活下来就比我高贵，你会喘气就比我高级。但是什么是种姓制？为什么殖民一定会推向种姓制？因为殖民这件事情就是要证明一件事：我比你高级。怎么证明？就是种姓制。

我特别想把它展开来讲，因为我觉得这个制度太了不起了。我不知道在座的怎么看，但对我来说，我意识到一件事：种姓制本身是禽兽制度中的顶流，它太符合人类心理上的需要了。我们需要什么东西？我们要满足精神上高高在上的优越感，几乎所有人都需要这个东西。包括现在互联网上，看各位的各种文章时，也会有人说：“你不就是想满足你的心理优越感吗？你不就想说你的东西比别人的更好吗？”他为什么这样说？因为人类普遍有这种需求。那么种姓制就在满足这种需求，它是顺势而为，顺流而下，和人类成长的逆流而上完全不同。它太顺了，所以太容易推广，力量极为强大、极为顽固。只要有殖民，就会产生种姓制。种姓制就是我比你高，你比他高，它比另外那个更高，就是整个世界的排序。而现在我们世界上所谓的殖民心态、现代社会的思想殖民，它的表现形式就是种族优劣论，所谓的西方中心主义。它最终表现出来的姿态就是：西方人是最高级的，然后游猎、渔猎、游牧的群体是第二阶段高级的，所以第一列的就是种田的，面朝黄土背朝天的这些“土坷垃”们，从而将中国人排在了整个世界的种姓等级的最下面。

所以当我们要说反殖民的时候，反殖民应该怎么走？它一定要调整一件事：叙事主体，谁来说话。我发现一件事，很多在座的老师们都在追求客观、准确、科学地描述我们所知的一切，这是绝对科学的态度，也是唯物主义的态度，但它非常容易陷入一种局面——你会失去主语。我们说一句话时要有主语：“我去做什么事情”、“他做了什么”，这就是需要主体。而我们非常容易在极微妙的地方失去这个主语。一旦失去主语，事儿还是那个事儿，但它为谁发声、对谁产生影响，就会开始发生转变。所以这些年以来，精神上、世界上的思想殖民，他们不全是在说谎，他们也经常客观、科学地讲述一个事件，但因为他们的讲述主体和中国人的主体不同，所以他们讲故事，既科学又理性，却把中国人又放到了种姓制的最低端。所以反殖民就必须改变我们的叙事主语。中国人必须在世界上有发言权，必须能够说话。而历史叙事这件事情，本身就是我们站在自己的角度来说我们的故事，它会决定一个群体的群体记忆，就是：我是谁，我从哪里来。

田老师是搞哲学的，一定知道所有哲学必须首先解决“我是谁”的问题，否则其他所有讨论就变成了空中楼阁，无法落地，无法应用。怎么去用我们的知识？怎么去用我们得出的结论？在座所有老师考据出的这一切科学结果，放到哪个战场上，能不能起到反殖民战争的作用？就看我们能不能通过叙事的方式来调整我们的群体记忆。而我为什么要强调群体记忆？因为我们的群体记忆被改变了。中国人这几百年来受到的肉体折磨固然是一个部分，但还有更重要的部分：我们的精神折磨来自于我们的叙事被人家改了。从清开始一直到现在，始终都有人在改变我们想问题的角度和方向，以至于哪怕是清代那些逃过了种种约束、有了学术知识、有了文化水平的人，他也不知道自己要为谁说话。他会开始在纸上写文章：“圣天子在上啊！”那么你的哲学思想优秀吗？黄宗羲的哲学思想是优秀的，他的哲学内容哪个地方不好？很好，很有用，放到今天可圈可点、很有价值。但是当他说出“圣天子在上啊”，不好意思，你的主语变了，你所有的东西在为谁服务，就开始发生转变。

那这个时候，谁的记忆才会关系到我们民族的存亡？我们要影响的到底是谁？他们在怎样想问题？所以有时候我觉得，现在的自媒体、现代科技带来的广泛讨论空间非常重要。它会让很多优秀学者研究出的内容，不仅仅只在书斋之中。这些优秀的学术成果，能够最终进入群体叙事、进入群体记忆，让所有人都知道“我是谁”，这是什么？这就叫做民族存亡。所以在民国时期，钱老师为什么要写《国史大纲》？因为他觉得中国可能要亡了，我们可能要当亡国奴了。那么当一个中国人要成为亡国奴时，我们怎么让自己最终还有机会活过来？就是记忆。《国史大纲》就是我们的群体记忆。群体记忆决定了群体的存亡。所以西史辨伪不仅仅是一个单纯的学术之争，除了学术之外，它还是叙事权的竞争。我们研究出来我们的冶铁需要多少温度，所以就能说：“你们英国人不可能做出这个东西”；因为我们知道炼一块透明玻璃需要多少温度，所以我知道：“你们不可能做出那个透镜”。但是谁说了算？谁讲的故事有人听？谁说的话算权威？这就是在争夺叙事权。

而今天的中国人迫切地需要去争夺叙事权。叙事权则必须争夺人民群众的关注度，要让大众都在意，所有人都知道：这些数学的、几何的、天文的、冶炼的，看上去高高在上、与我的一日三餐毫无瓜葛的东西，是我的立身之本，是我在这个世界上存在的本源之一。那么这个时候我们就拥有了力量。就像这一次，何新老师与唐继庚老师的学术争端，看上去是学术争端，但最终甚至不需要很多学者下场，评论区就可以把唐老师的很多讨论直接推平了。为什么？因为有越来越多的普通人都意识到一件事：你在替谁说话？为什么你说的每一个字我都听得懂，但我就是不高兴？这个东西它就是人民的觉醒。

所以我们今天在座的所有老师，都一定用得到的武器，就是唯物主义。就像刚才那篇论文，什么是唯物？一切事情的发生都是有源头的，一切事件的发展都是有背景的，这就是唯物主义。我活着，所以我才能吃饭，这就是唯物主义。因为我有纺织机，所以我能够穿上衣服，这就是唯物主义。因为我有炉子，所以我才有金属，这也是唯物主义。因为有金属，所以才能有蒸汽机。当你只讲“我发明了蒸汽机”，那请问前面的呢？为什么前面什么都没有？为什么会出现无根之木、无源之水？有越来越多的学者在解决这个问题，于是通过唯物主义的叙事，在帮助所有中国人挣脱国际种姓制的枷锁。从今年的九三阅兵之后，越来越多的人发现：“我的天哪！我们中国人花了才几十年的时间，追上了你们几百年说的遥遥领先的东西，然后你说我们几千年都是落后的、不可能进步，你在骗谁？”当现实的、唯物的重拳捶在所有人面前时，叙事权就开始逐渐转移阵地了。

所以为什么要强调“农村包围城市”？因为它是我们力量的来源。我们从哪儿来？包括所有学者做的所有努力，最终为谁而服务？当被服务的对象得利，并且意识到它很重要、这些武器对他极为有用时，我们就实现了“农村包围城市”。当然了，我在最初提出这种想法时，是学习了西方人的。西方人当初来搞颠倒叙事时，做的就是这项工作。不知道大家注意到了没有？几十年前，中国的平均受教育水平并不高，绝大多数人没有高级教育，受到一个职业教育就已经是社会精英阶层了，哪里知道什么这个历史那个历史，亚里士多德念完了舌头要打卷的。后来怎么那么多的人越来越相信西方历史？辉煌的地摊文学大家都看过，也曾经瞧不起过、蔑视过，但这就是西方人当年进行舆论围剿战时，所使用的“农村包围城市”。只是那时我们是被包围的对象，而今天我们希望用这样的方式反制回去，寻找到我们自己的叙事道路，然后走向真正的历史正本清源。

我亲自关注这个领域20年的时间，当然这不完全是西史辨伪发展的20年。但一开始，我最初听到的就是何新老师的质疑。当时他那一声质疑，突然解答了我在学校读书时遇到的困境：凭什么中华法系不好？我们的系统很完善，而且跟我们的社会道德、公共伦理是配套的，这种完全一致、一点都不割裂的东西竟然是不好的；而西方人传进来的那种完全孤立的制度，要求你必须抽离出理性才能够去理解的东西，竟然是好的。当时何新老师提出的疑问一下子让我说：“对！不然呢？还能有什么其他的解释？”

但是最终能够让这些思想落地生根的到底是什么？我意识到一件事：在西史辨伪产生的早期，很多时候是在论坛上、在一些很草根的地方讨论它。而这种讨论的过程中，我觉得最难能可贵的是，质疑西方历史的所有从业者、参与者、讨论者，甚至旁观者，都非常冷静，愿意好好说话，愿意慢慢说。我觉得这就是那时最可贵的东西。因为西史的信徒们特别着急，他们特别喜欢给人戴帽子：因为你不相信，所以你是九漏鱼；因为你不相信，所以你是文盲；你都不相信西方哲学的伟大，那你肯定烂透了。他们在不停地进行攻击，而我们在不停地进行解释。这种憋屈的、时不时就要受委屈的局面，使得我们在整个斗争过程中获得了群众的支持，因为围观的人看得清楚谁心虚、谁胆怯，谁可以讨论，谁是一言堂。它的可贵之处就在于它是接地气的，是愿意讲给最普通的人听的。

而我们今天所延续的就应该是这样一条路：你打你的，我打我的。我们只管讲自己的东西，把我们的内容用最细致的、最接地气的、最符合普通人认知的、最符合朴素唯物主义的方式传播出去，让所有人都听见。至于那些污蔑的、泼脏水的，说你是九漏鱼、你这个臭文盲、你就是在割韭菜——欣然悦纳之后，局面的转变其实来得很快。就像今天，我们已经开始攻守易形。继续这样走下去，直到最终胜利的来临。而谁会胜利？有来源的会胜利，有人民基础的会胜利。好，谢谢各位老师。
