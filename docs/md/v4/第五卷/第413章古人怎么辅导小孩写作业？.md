# 第413章 古人怎么辅导小孩写作业？

嗯，啥？  
嗯，谢谢中华花龟英雄啊。  
辅导孩子写作业太累心了，把烦恼忘了，哈哈哈哈哈。  
哎呀，带孩子写作业是这样的啊。实在不行的话，想办法外包。嗯，不然的话愁。  

嗯嗯，啥？古人是怎么辅导孩子写作业的？知道孟母三迁吧。啊，知道孟母三迁对不对？哎，来，请问孟母为啥要三迁？为啥非得要搬家不可？不就是因为孩子不学好吗？所以呢，他最后搬到一个让孩子能够念书的地方。这件事情本身就是一个辅导作业的行为。就连孟母啊，要教导孟子这样的天生读书的材料、读书的种子，他都得搬家。你自己说难度大不大啊？连孟母都那么困难，换谁谁不难呢？  

我们从上世纪大概80年代中期开始，逐渐逐渐地从一些一线大城市的、发达地区的先进人员那里啊，开始传播的信息就是：教育应该是快乐的，要快乐地学习，要快乐地教育，要因材施教啊。你说他对不对呢？你也不能说一点儿都不对啊，不能说毫无道理。但是大部分人用不上。你不能说这玩意儿毫无道理。你说因材施教对不对？那肯定对啊。那有一些人就是：你在上头给他敲快板，他底下什么都能背下来。那你就把经史子集都用快板给他啷当出来不就完事了？他咔咔地背，还全都明白了。问题在于，这就说明你要专门为这一个学生安排一个会打快板、又会经史子集的老师。我就问你，你的成本放在什么地方？  

教育可不可以因材施教？我也觉得挺好。你要是觉得每一个孩子都根据他天然的才能、天然的智力、天然的情绪控制能力，去安排一个和他完全匹配的教学方案，会不会让他实现更好的自我？我相信可以。但问题在于这事儿你做得倒吗？做不到。而能够做到这种——十个老师教这一个孩子定制方案的——不会出来问这个问题，对吧？  

有没有可以因材施教、不管家里面养出来的孩子天生是一个什么样的娃，最后都能够教得差不太多？你也不能说很优秀啊，毕竟有些孩子真教不出来。那你都能教得差不多，都能够顺势而为，做得好的这种家庭存不存在？存在呀。反正我的家庭不是，绝大多数人的家庭都不是，都做不到这一点。所以你怎么算这个账？  

教育是什么？教育是能够让更多数的人达成思想、行为模式上面的进步，进而构建一个更加良好的社会运转模式。这个是教育的根本目的。所以在这种条件下，你说：“哎，我们应该快乐。”不好意思啊，做不到就是做不到啊。因为你要投入可控的成本去教育出尽量多的人，所以你必须要实现一个最大公约数。  

那为什么唐朝的那个、时隔千年作业被人家扒出来的小孩哥啊——这老师咋还不放假呢，是不是？哎，那他的作业也是很苦的。他也觉得自己日子过得不舒心，他也觉得天天抄抄写写。为什么呀？为什么要让我抄抄写写？今天的小朋友们也是一样的。老实说，你把这个字抄十遍，下一个字再抄十遍。为什么？为什么要让我写错那么多遍？大家不是一样的吗？  

教育啊，正常的这种教育成长，它就不是一件快乐的事情。为什么？因为教育要求你突破你自己啊。教育要求你突破你自身的现有的维度，要你走到更高处去，这个东西才叫教育。你总不是说教你一下吃喝享乐吧？你觉得这玩意儿叫快乐吗？嗯，这个玩意儿叫教育吗？啊，我教你如何吃得更爽，我教你怎么吃得更开心。我跟你说，就是在美食品鉴这个栏目当中，你要登堂入室，成为专业的食品品鉴师，那都是要吃苦的，对吧？哪怕是吃东西这么快乐的事情，你要进阶到食品的品鉴师行业的时候，都要吃苦。  

孩子刚一岁，有些教育方面的焦虑，有什么教育经验可以分享一下吗？算不得什么教育经验啊。但是我觉得刚一岁的时候，最重要的一件事就是家长千万别焦虑。为什么要强调这个事儿？因为家长的情绪会直接影响孩子的成长和发育。一岁大的孩子，最重要的不是他将来学习怎么样、就业怎么样，最重要的是您让他的身体发育、精神发育、所有的神经链接发育走向健全。人类六岁之前是在长脑子的过程当中啊，他是在一个生长发育的快速道上面。在这个阶段当中，最重要的不是你已经规划好了他必须上哪个小学、必须上哪个中学、必须上哪个大学啊，必须学什么东西，必须拜哪个师门。这个阶段麻烦你让他长得完整一点啊。家长情绪的焦虑、安排的任务的繁重，很有可能会干扰到他的物质层面、身体基本层面的生长。  

我觉得现在有很多的小孩啊，到了小学、初中阶段的时候，明明长得那么好，那么高高大大、漂漂亮亮的，但是在精神上面非常容易紧张。不能够排除一点，就是他在这个精神的神经、大脑发育期的时候，家里面过于焦躁。我觉得现在有很多的小孩啊，到了小学、初中阶段的时候，明明长得那么好，那么高高大大、漂漂亮亮的，但是在精神上面非常容易紧张。不能够排除一点，就是他在这个精神的神经、大脑发育期的时候，家里面过于焦躁。  

嗯，好，嗯，欢迎来到直播间。嗯，然后我刚刚弹出来那个商品，是刚才进直播间的时候，就有一个小伙伴说：“那个，那个《名学》是哪本书？给我弹一下啊。”哼，所以我就弹了一下啊。当然了，《名学》这个书呢，本身也很好。就是它在最后的编辑呀，编辑的环节做得非常好啊。  

首先就先来讲一下什么是名学。名学呢，是有关于命名、分析和逻辑，基本上约等于学术方法论的这样一套学问啊。名学这一群人呢，在先秦时期的诸子百家当中也算是独树一帜的。为什么叫独树一帜？就是他们辩论赛老赢。大家知道的那个先秦时期的诸子百家，经常之间打嘴仗，甚至呢，他们会在稷下学宫这样的地方开辩论会啊，互相之间动辄这个瞧不起那个、那个瞧不起这个，然后荀子瞧不起 everyone，经常是这样的操作方式。那么名家是什么呢？名家就是论辩论：在座的都是乐色。但是呢，因为他老是这样干，所以呢，大家也都不喜欢名家的这一群人，因为他们特别擅长辩论，特别擅长狡辩。以至于名家最后呢，就被拐到了这种什么诡辩之学上面去了。哈，名家在很多的名声上面变成了诡辩之学。典型的东西呢，就是大家都知道的白马非马。但是这是诡辩吗？不是，它其实讲的是概念、定义、逻辑推理的方法啊，它讲的是是非、属于和不属于这种关系，它是一种逻辑推理公式。  

所以名学是干什么？名学就是定义权、命名权、逻辑分析方法这样一套学问。所以名学不只是名家在应用，实际上诸子百家都会应用到名学。所以这一套书呢，它的可贵之处就在于，它不是简单地搬运了名学的学者的论述，然后给你一趟翻译，而是通过它内在的架构，通过名学本身的逻辑，将与名学运作相关联的各个古文段落都给你拆出来，然后去分析里面与名学相关联的哪个部分，是属于名学探讨的内容。  

那么这本书，或者说名学这门学问，在今天重不重要呢？很重要。因为在西方哲学概念当中，中国人的哲学是没有逻辑学的啊。在之前的所谓的西方哲学的定义当中，认为中国的先民们是没有逻辑学的。然而呢，怎么可能？开什么玩笑啊？啊，我们的所有的学科都是互通的，怎么可能会没有逻辑？只是我们不跟它叫做逻辑而已啊。因为它起了名字，所以呢，我们就没有了。这是什么？这是一个被颠倒黑白了的名学。西式的名学，它不叫这个名字了，然后它起了一个名字叫做逻辑，然后说：“你看，中国人没有逻辑。”但不好意思，中国人有名学。名学讲的就是命名、定义、推论啊。而且这也是这么多年一直被诋毁的一个方面。这个事儿是啥呢？这就是西方中心主义啊。这个东西就是典型的一个精神殖民，西方中心主义叙事的一个表现。
