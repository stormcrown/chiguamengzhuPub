# 第401章 东林党与改土归流，小青粉与美国新清史

谁篡改了明史？或者说是谁编造了明史？是清朝，花了100年的时间，编造了一部秽史，成为了现在的定论。而我们要把这个东西推翻，第一个要推翻的，就是今天在全世界范围内都很猖獗的满洲复国主义群体，还有清史、新清史群体。注意了，我为什么把这些都绑在一起？因为他们是一伙的。不要以为新清史很反动，所以国内的清史就很正义，不存在啊，他们是一伙的。我们国内的很多清史学者，一直都拿不出一个像样的、最终的这种方向性的定稿出来，就是因为你看着他表面上反对新清史，但是在叙事上面，他和新清史如出一辙，内在逻辑高度相似，所以最后的东西越做越夹生，越做越难看。所以清史这个编撰为啥一直都是难产的状态？他确实做得不好吗？他能不难产吗？

嗯，可以说一下东林书院吗？除了民族立场，不也要谈阶级立场吗？谈呢！我们这个这么毛派的直播间，怎么可能不谈阶级立场呢？所以我们从来都是要谈阶级立场的。那么东林学社、东林党人，他们一开始的时候都是坏的吗？不是啊，不可能。一个完全错误的东西，你是没有办法感召那么多的人的。所以这不是正确与错误的问题，甚至说你去翻他们所有的文献，你会发现这些人活脱脱的伟光正，你知道吧？东林学社的这一群人，你看他们所有的文章、所有的言谈，那就叫做一个伟光正，属于是你但凡与他的思想背道而驰，你都会觉得自己缺了大德的那种人。但是问题在于什么呢？他们没有立足于国家的根本利益。而为什么他们没有立足于国家的根本利益？因为他的根本利益和国家普通百姓的根本利益高度不一致，是根本利益的阶级问题。所以这本身就是一个阶级问题啊。但是注意了，当你涉及到满清入关的问题的时候，这就不是阶级问题了。那为什么东林党人后来就变成了一个祸害？没办法的呀，因为他们的根本利益是觉得他们在哪儿都能发财，在哪儿都能当社会精英。他们有了精英意识，觉得自己高于其他人，觉得整个群体的利益与他无关。所以他们当中出现了这些卖国求荣的角色，不是所有人都卖国，但是他们当中的一部分关键人物，出于自身的私利和出于自身小团体的需求而卖了国。只要你在关键的岗位上面做一件坏事，对于整个国家来说，那就是重大破坏，这就跟你破坏了顶梁柱是一样的效果啊。所以呢，不是所有的东林党人都烂，里面也有好人；也不是他们的理论多么的烂，都不是。而是当他的根本利益的站位和大家不一样了的时候，只要最关键的岗位上的人做了烂事儿，整体就糟了。

嗯，好，欢迎大家来到直播间啊。嗯，讲一讲大清朝改土归流取代土司制度的利弊。怎么着？讲到改土归流的时候，突然变成他大清了呢？改土归流这件事情是他大清一朝之功吗？啊，这就是遇到正经事的时候，突然之间就变成他大清的事了是吧？之前都是明清一体的好不啦！烂账的时候都叫做明清一体，哎，进入到改土归流、这个看上去还有点用的东西的时候，突然之间变成他大清的了啊。来来来，注意了啊，改土归流这件事情从明代的时候就在做，而且明朝做得更加的温柔、更加的合理、更加的正义，也更加的彻底。如果不是明王朝做了大规模的改土归流，我们今天的云贵地区就不会这么安稳。这些地方就是明王朝通过改土归流和深入的融合、深入的同化，把他完全接管下来的。在之前这个地方飘摇不定，为什么？因为当地的地势条件决定了当地的土司们太容易割据了。地势是一个半隔绝，地势割据了之后，你鞭长莫及的时候就没招儿了。所以到明王朝的时候，通过移民迁徙、通过文化的同化、通过制度性的梳理、通过深度的管理，让整个云贵地区大家在认知上面和文化的上面保持了一致性，最终实现了根本的改土归流。所以一说到改土归流有用的时候啊，你别归到他大清身上去。然后改土归流这件事情是需要循序渐进的，因为你需要顺应着对方的发展阶段和当时的时代背景去逐步操作，而不是突然之间一棒槌就揍过去了。谢谢。法无常啊。那么清在这个事上面办了没有？办了。但是办得好吗？办得不好啊。清朝做了改土归流，但是他做得不好。为什么他做得不好？因为他没有文化上的感召力，也没有制度上的向心力，最后他只能用暴力的方式去搞各种各样的镇压，然后大加屠戮，制造仇怨和撕裂。现在有很多的族群说我们跟汉人不和，他们的传说当中经常这样说，但实际上这个传说是假的。为什么？当年真的暴力镇压、让这些地方民不聊生、不得不卑躬屈膝的势力是谁？是满清朝廷，是八旗呀！结果这口黑锅被主体民族背下来了，说以前就是你们把我们轰过来的。就不是这么回事儿好吗？谁干的这种事情啊？到了他大清身上的时候，这个东西歌功颂德。歌功颂德就是什么啊？大脚金川之战，然后还有什么准噶尔收复。准噶尔原本就是羁縻管理的地方，而且是逐渐深入管理、慢慢改土归流的地区，到他这个地方管不住了，收不了了，开始大屠刀哐哐一顿，把大家的仇怨结下来了。他的锅一甩，那就是中央王朝自古以来的改土归流结下了什么什么东西。事全是他干的，锅就要大家一起背。
