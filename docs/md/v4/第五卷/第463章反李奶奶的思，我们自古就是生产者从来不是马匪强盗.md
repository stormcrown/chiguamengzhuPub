# 第463章 反李奶奶的思，我们自古就是生产者从来不是马匪强盗 #强盗

但是我们不能等军国主义出现了再去反对啊，这个东西惯性太大了。我们早早应该警醒才对，不是吗？你是半句也听不懂是吧？真的，你蹲这么长时间干啥呀？中国不可能出现军国主义，中国人又不是死变态。你知道中国人为什么不能够出现军国主义，也不可能出现军国主义吗？来，我们今天来聊一聊中国和外国完全不一样的战争的思维逻辑、思维模型。

很多的中国人被PUA狠了啊。明明我们是过去数百年的被侵略国、被殖民者、被压迫者、被奴役者，是一个妥妥的完美受害者。这句话需要我重复吗？在过去数百年间，全世界范围内，只有中国人是最完美的那一个纯粹的受害者。另外的那一拨儿，犹太人号称自己是受害者，实际上他们就是始作俑者。他受的所有的害都是他自己作的。只有中国人是最完美的那一个受害者，我们什么坏事儿都没干，但是所有的倒霉事儿都没落下。结果今天的中国人不停地反思：那万一我要是强大了之后打别人怎么办？那万一我要是厉害了、我欺负别人怎么办？那万一要是我支棱起来了、我压迫别人怎么办？你想啥呢啊？一个奴隶连翻身的精神头都还没有呢，就开始反思了：我当了奴隶主，压迫别人了，可怎么整啊？那我还是在这当奴隶吧，当奴隶安全，当奴隶不会压迫别人。你是不是有点欠儿啊？啊，你琢磨什么东西呢？

我们这几百年是一个纯纯的受害者，结果心里头那随时随地警惕着自己，千万千万不能强大起来欺负别人。你琢磨什么事呢？什么时候你干过这个事情？你反思什么啊？那些以前恶心过人的、那些以前压迫过人的、那些出现在各种各样农奴制度当中的奴隶主、农奴主、奴隶头人们、那些土司们、那些酋长们，现在左一个右一个的高喊着自己有贵族精神。结果我们这帮子被压迫被奴役的人，直腰都还没直起来呢，就开始思考：说完了我要当坏人了，哈气死我了！我怎么能够当坏人呢？我要好好的反思一下，我要当坏人了，可咋整啊？你哭，你接着哭。你实在不行找根绳勒自己脖子上，然后把那一头交给奴隶主，那你就永远都不存在压迫别人的可能性了，因为你就是一个纯纯的被压迫者嘛。所以当我们的局面是这样的时候，你动辄反思这玩意儿、警醒这玩意儿，说我们万一有军国主义了怎么办呢？到时候来不及了。我们几千年的历史从来没有过军国主义，你在反思什么？因为日本有军国主义，并且侵略了我们，所以我们就会有军国主义？所以我们千万不能支棱起来，我们只能被人家侵略？这是不是有点欠儿？你知道我要说什么词儿，之所以不说那个词要替换成欠儿啊，主要的原因是我怕平台说我骂人。能理解吧？先get到我前面说的这句话啊。然后后面一段我来分析一下中国为什么自古以来都不存在军国主义这种行为。

我从唯物的角度上面来给你分析，为什么我们数千年、上万年的文明史当中，就从来没有军国主义思想的存在。因为我们不存在这种思想的土壤。请问什么样的群体会诞生军国主义？那种说我非得把别人家都占了，我一定要把他们家抢得透透的，我把他们家的房子都归我，他们家的那个奴仆也都是归我的，他们家的作物全是我的，我的、我的、我的还是我的——这种人才会产生军国主义，能够理解吗？那么请问什么样的人会变成这样的人呢？没有生产能力的人。没有生产能力、靠着自己一天要饿三顿的人，他会看着别人家的房子、看着别人家的地眼馋。这样的人会成为军国主义者。但是中国人为什么数千年、上万年来，哪怕是最强势、最扩张的时候，也从来不搞那种虐民之事，从来不做军国主义者？连汉武帝这样穷兵黩武的人，都是在招纳对方的流民，去训练他们成为工匠和农民的呢。因为我们是一个生产建设型的民族。我们从一开始就是有什么想要什么，我们自己做出来。我们想象到了什么，我们就去实现到什么。我们想要开山就去开山，想要饮水就去饮水。我们觉得大河泛滥了不好，我们就去修水利工程。我们想吃饱饭，我们就去种田。我们想穿暖，我们就去织布。我们是一群生产者啊亲。为什么中国自古以来没有军国主义这种东西的存在？是因为我们是生产者，是建设者。生产者与建设者是没有侵略的意图的，也没有抢劫的意图的，因为别人家没有什么东西可以让我们抢的。而战争会严重的破坏生产系统的稳定性，所以在我们的军事哲学当中，上兵伐谋，不打仗是最好的。获得资源的同时不打仗，不战而屈人之兵，在我们这里是上策，最下策才是攻城略地。一个哲学一定是诞生在它的物质基础之上的，我们的哲学为什么是这个样子的？我们的军事哲学居然是爱好和平的，请问这是什么？因为和平对于一个生产者来说至关重要。而对于那些非生产的群体来说，和平那就意味着他要饿肚子。所以他们为什么一旦稍有强势的可能性，就会变成劫掠者，就会变成军国主义者呢？因为它内部的生产能力严重不足，他就迫切的需要去劫掠、去抢劫、去侵占他人的东西。因为这一种欲望的驱使，所以他拥有强烈的成为军国主义者的可能性。

所以身为一个中国人，身为一个农耕民族的继承人，请你不要动辄地替中国人反思，说万一我们成为军国主义者可怎么办。我们没有那一种土壤。我们作为全世界唯一的全产业链的工业国，作为庞大的拥有完善农耕体系的农业国，我们不会做这样的事情，因为做这种事情对我们来说是彻头彻尾的不划算。中国人绝无可能成为军国主义者，所以不需要谁按着头颅要求中国人反思，说你千万不能直起腰来，直起腰来你就变成军国主义者，你要来欺负人了。注意，摁你头的人才是欺负你的人。如果连这个东西都理解不了的话，那最好在自己的脖子上拴根绳，然后把这个绳头交给奴隶主的手上。这个人呐，太喜欢当奴隶了，什么毛病？我真的是理解不了，知道吗？不要动辄反思自己，我们身上没有那些臭毛病的时候、没有这些历史的时候，反思啥呀？反思？罪人都没有反思，结果犯罪的人按着我们的脑子反思，你就真的来反思了诶？好，欢迎大家来到直播间啊，天呐，我这个点赞也忘了要快点抓紧。我这个点赞的数量看起来也寒碜，成交的数量看起来也寒碜。你们该刷的刷，给点力好不好？
