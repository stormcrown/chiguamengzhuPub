# 第268章  满字烫嘴吗？ #创作底线 #汉服 #唐装 #屯堡服

嗯，啥玩意儿？穿不穿汉服直播？那你觉得我穿的是啥呀？你之所以不知道穿不穿汉服直播，就是因为你不认识，对吧？嗯嗯，多去关注几个汉服UP主啊，然后呢，看一看汉服长啥样。别的都已经怼到脸上来了，然后不认识，说“这世上哪有人穿汉服的呀”，多吓人呐，是吧？结果呢，人家大家都已经站你面前了。

这个是什么？这个是唐圆领啊，是一个唐代的哦。那啥，你也有，唐圆领是个绿色的，我也有一件绿色的。嗯，呵呵呵呵啊。今天的这一件呢，是唐圆领。当然了，我现在穿的这个呢，算是一个简化版啊，因为属于夏装，只做了半截儿，是一个短款的唐圆领。

唐圆领的话，它的穿法当中包括工工整整的扣上，以及呢，解开一颗扣子，甚至还包括嗯，坦半壁。所以呢，啊，想咋穿咋穿啊。唐圆领的话呢，它通常情况下是内外双色的。然后呢，哎，全部都扣上的时候，就是一件这样子的衣裳啊，完全是一个圆领。然后呢，如果说其实厚一点的款式，你会感觉到它很像那种风衣的折领、翻领啊，就是你当你解开一颗扣子的时候，它会出现一个撞色的翻领。然后如果说天气很热，或者说你想要动手啊，那怎么办呢？就坦半壁啊，既方便干架又很凉爽啊。这个就是唐圆领的穿着方式。干架服？可以可以，可以是拿来穿的啊。

所以呢，我们古人其实对于衣服穿着，一直都是很自由的。不光穿着的方式上面自由，实际上穿着的款式也自由啊。现在抖音的公屏上面就已经开始在那里说啊，你应该穿那个什么清汉女这种子虚乌有的衣服。清汉女不存在，只有清民女，是一种被迫穿上的、如同囚服一样的屈辱型的衣服。那为什么那个衣裳就要被排挤在汉服圈以外呢？啊，我们说了，我们宽容，我们连胡服都能穿，为什么不能穿它呢？因为名未定啊，因为名实不符。为什么不能穿清民装？宁可穿胡服都不能穿清民装，宁可穿西装打领带都不能穿清民装。为什么？因为名实不符。西装就叫做西装，胡服就叫胡服，藏袍就叫藏袍，蒙古袍就叫蒙古袍。我们到南方旅游的时候，进了苗寨啊，也会有苗族的传统服饰，虽然那个款式也属于新设计的，但是人家的刺绣是老刺绣，都可以穿。但是为什么到了清民装的时候，大家就一定要拒绝？因为名实不符。它实际上是清代的时候特意设计出来，进行压迫、摧毁他人服装传统系统的一个攻击型的、羞辱型的、监禁型的服装。如果坦然承认这个客观的事实，把它的名字也标得清清楚楚，把它的历史介绍得明明白白，那么谁愿意穿谁就穿，无所谓。但是它名实不符，它非得给它编一个子虚乌有的“清汉女装”的名字，但它不是；它非得说那是我们的传统服饰，然而它也不是。这就是它的名字、它的名义、它宣传的方式和它的本质之间背道而驰，所以这个衣裳不能穿。

嗯，满字儿烫嘴是吗？我就觉得这些人“满”字儿烫嘴怎么了？他们自己的传统服饰非得蹭到别人身上去，他们设计出来的衣裳非得说是汉族的传统服装，这是逗谁的呢？

苗族的年轻一辈改成改穿了那个交领。对，现在已经开始出现了啊，我们南方的、南部的很多少数民族的同胞呢，已经开始逐渐地调整他们的传统服饰，在里面运用了更多他们原本真正的传统啊。要知道，他们之前几十年在很多景区里面去穿着的所谓的传统服装，大量的都是上世纪60年代到80年代期间进行新设计的。这种新设计出来的传统服装上面的元素、配色、款样，其实呢，都充满了刻板的印象，和人家的传统之间常常是背道而驰的。所以这几年啊，在南方的一些少数民族同胞的努力之下，他们已经越来越多地使用了他们真正的传统元素，也就是交领右衽的传统服饰啊，这件事情本身是可喜可贺的啊。

然而呢，里面还是有一些奇奇怪怪的东西。就是很多的少数民族的小伙伴已经开始将他们真正的传统服装拿出来展示了，并且呢，非常努力地向大家宣传这件事情。然而在我们的福建地区，开始正式地推广所谓的三把刀。如果你说就是那几把簪子没有问题啊，在一些局部的地区，曾经有使用过多把刀簪的形式进行自卫的情况，这种事情是存在的。你想宣传这种自我防卫力，你想宣传巾帼英雄都没有问题。但是三把刀的这种簪子是为了什么设计的？是为了抗击倭寇设计的，是明代的一种抗击倭寇而设计出来的簪子，后来的时候他们也拿来作为反清使用的武器。然而今天去宣传这些东西的，身上穿的都是清代被迫穿上的民装。这就是人家带这个三把簪，目的是为了反倭寇，是为了反清，结果它硬生生地给人家摁上了一个清民装。

然后唐圆领的话呢，按照比较通俗的说法，其实是可以称之为简称为唐装的。但是呢，可惜了的是，唐装这个词儿啊，这个好词儿已经被占用了，所以没有办法，我们只好就叫它的全称唐圆领啊。无奈啊，属于无奈。

唐人对于自己的审美的情趣上面，非常的开放与包容啊。我说了这个词儿呢，现在已经被用得挺恶心了，但事实就是，唐代的人在穿着上面是非常开放的。但是这个开放是怎么个开放法？意思是唐人在出门的时候觉得起码可以穿胡服会更帅，他就会穿胡服；他觉得打马球的时候穿胡服会更好看，他就穿胡服。然后呢，如果说他看到哪个胡人身上的某一个印花或者是刺绣，感觉到很别致，他就会把这个花样子用在自己的身上。他会不断地给自己调整自己设计的元素。而这些东西呢，就是他所拥有的开放与包容。但是这并不意味着这个服装系统是混杂的，也不意味着这个衣服就是胡服，这是两种完全不同的概念。这相当于是你在西装上面绣上了一朵祥云，但是不能说它就是汉服了。所以唐代的衣装啊，一以贯之的是华夏服饰系统当中的一员。它有着对称结构啊，对称受力结构，普遍性地使用了交领右衽的结构，依然保持着原有的系统。只是说呢，它会在这个里面运用更多的外来的花样子，给它进行装点。这种装饰的纹样，在唐代的时候会出现比较风靡的情形。其实历朝历代都有这样的风气，干嘛呢？这个玩意儿就是赶时髦，什么时候的人都会赶时髦。
