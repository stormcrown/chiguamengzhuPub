# 第204章  当我们再说“多元”时，到底在消什么

多元这个词本身就有问题，因为它是一个贬义词。虽然我们现在也用，并且长期把它当作正面词汇来使用，但事实上，“多元”这个命名本身就存在缺陷。为什么这么说呢？因为“元”是什么？“元”是初始，也是根本，还是本源。当“元”被理解为初始、根本、本源时，“多元”又是什么意思呢？“多元”就是说，同一个地方有很多个根本，有很多个主体，有很多个唯一。“元”就是这个意思，然后“多元”就变成了这样。请问，一个地方既要追求大一统，又要说“我们自古以来保持一致”，还要强调“我们是真正的亲生手足同胞”，同时又称“我们是多元的，是完全不同的源头、不同的主体”，这不开玩笑吗？一个群体的内部怎么可能存在多个主体呢？这是做不到的事情。所以，这个词从被设计出来的那一刻起，就恶意满满，心怀歹念，不是一个好词。但是呢，大家已经习惯了，它就这样开始出现了。

有人说：“如果不能多元，我们哪能发展到今天？”但事实上，如果真的能够多元的话，我们早就凉透了。在这个世界上，当一个国家要绵延不绝地发展时，它一定有自己的本体特征——你自己的本体特征。你说“中国是什么”，中国长什么样？中国由什么样的人构成？它以什么样的系统生存？它的核心特点是什么？对于这些问题，你必须要有标准答案，一定要存在，而不能说“中国是多元的”。如果中国是多元的，那么任何人只要站在某一块地方上，他就是中国人；任何一种文化在这个地方，它就是中国文化。这样下去，最后就没有中国了。你知道吗？当全世界所有的人都是中国人，所有的文化都是中国文化的时候，中国就不存在了，它变成了一个符号，被虚无化了。所以，如果“多元”是指全世界是多元的，那没有问题——整个世界是多元的，这没问题。就好比我们这里的味道是隔壁家的味道，北边有北边的味道，南边有南边的味道，这种叫多元。每一个群体都有自身的特色、特性，有自身独有的文化特征和人群独有的标志，这种叫多元，没有问题。所以，在世界视野中，多元是可以的。但你说一个国家内部，甚至一个群体内部是多元的，那就出问题了。

多元和多面不是一回事，偷换概念吗？对，他们就是故意的。使用“多元”这个词，本身就是被故意设计出来的。因为“元”是根本、根基，一个群体只能有一个根本，就好像一个人只能有一个大脑。如果说一个人是多元的，那就是LGBTQ之类的东西——你早上是男的，中午是女的，下午是小孩，晚上是个老太太，睡着的时候是个老头儿，第二天起床的时候是个垃圾袋，吃早饭的时候是个电饭煲，到中午变成了公交车，下午变成了跑车，晚上又变成了溜溜球。这是什么啊？一个人是多元的，意思就是他的根本有很多个，一会儿男、一会儿女，一会儿老、一会儿少，一会儿美、一会儿丑，一会儿聪明、一会儿愚笨，这就叫多元。所以，一个个体或一个群体，玩什么“多元”呢？怎么可能有一个群体，它的内部来源截然不同、本源截然不同、根本诉求截然不同，然后还说“我们就是亲兄弟”？

有人反驳：“多元是必须的，盲目排外只会灭亡。”这就是什么？混血是最好的，近亲就会导致智障。这种分类方法就叫二极管思维，非此即彼、非黑即白。谁说混血的另一边就是近亲？不存在，这是一个伪概念。同样，多元的反面不叫盲目排外，多元和盲目排外之间没有任何关系。一个群体有自己的特征、本源，有极强的主体性，这跟它是否友善有关系吗？没有任何关系。它既是一个友善的人，开放宽容，同时又有清晰的自我认知，这是一元，而不是多元。什么叫“禁止盲目排外”？你要求的是敞开怀抱，但我一会儿是这样，一会儿是那样。多元是什么？多元是我有无数个根本。你对“元”这个字都理解不清楚的时候，就会说什么“不喜欢多元就是要盲目排外”“不喜欢混血就是要近亲繁殖”，这就是二极管思维。

有人问：“有人说李唐是混血，说中国没有真正属于自己的封建帝国盛世，你怎么怼他？”怼他干什么？不用怼，直接开喷就行了。他不是不知道，这些东西现在都是可以公开查证的，并不稀奇，没有什么保密信息，也没有什么找不着的信息，通通都有。那么，凡是现在还在说“你这个东西是混血，李唐因为有胡人的血所以才高贵”的人，根本就不是知识的问题，而是立场的问题。立场问题是什么意思？就是彼此之间是敌我关系。对于敌我关系的事，你讲道理、怼，没什么意思，直接开喷，用你学过的所有开喷的词就喷回去，完事了。凡是坚持不懈地说“李世民是鲜卑人”的，我建议你亲自到李世民面前去，跟他说：“高贵的皇帝陛下，幸亏你是一个胡人，要不然的话，我们怎么能够拥有大唐的盛世？”然后你看看你家的九族还能不能留下来。你自己亲自去说，好吧？所以动辄冲上来喊着“李世民是胡人”的，你就直接去跟李世民说，我看你有没有这个胆儿。反正唐代的人不敢。除非你掌握了极高的话语权，那也是在流放的途中悄没声地就没了——这还是当时顶流的名嘴，在那个时候不能杀、不能碰，碰了容易当昏君。说他是胡人，下场就是流放的过程中悄没声地就不见了。

有人问：“中华核心特征是什么？”我做过视频，叫《什么是中国》，你可以去看一下。有些内容视频可以做，直播的时候不方便讲。第一件事，在唐代的时候，这个话题很容易把九族送进去，所以当事人是绝对不会认可你的说法的。那么第二件事，请问标准是什么？凡是在那里说李唐皇室是胡人的，好，你来界定一下标准。你说他是胡人血脉，好，你来界定标准是什么。简单来说，我这个人非常包容，愿意接受任何一种标准，任何标准都可以。请记住，我承诺一件事情：在我这里什么标准都行，但是不能双标。你界定一个标准，什么样子叫胡人？是不是家里面但凡有一点点胡人的母系血脉参与，他们全家就变成了胡人？如果是的话，那么你想象当中的那些胡人，家中都有大量的汉族父系以及母系——可不光是母系，因为汉人这边只接纳胡人的母系，我们的认同感是父系认同、认爹的。然而你所认为的那些胡人，你将他们称为胡人，可他们的血脉当中既包括了大量的汉族母系，又包括了大量的汉人父系。我就问你：你对胡人、汉人的界定标准到底是什么？如果按照“一滴血原则”来算，哪怕你周边那些被你称为纯粹胡人的，全部都是汉人，一个都没有胡人——好吧，这也算是天下同一吗？按照“沾上了什么就是什么”的标准，这片土地上面行走的全部都是汉人。那么，但凡有汉族的Y染色体基因又怎么说？中国这里有99%以上的人是汉人。你只管去统计，到任何一个专门进行采样的研究所去统计数据，都是这个样子。你按照Y染色体上面的点位来核算，99%——中国土地上面99%的人，包括你认为和我们长得不一样、肯定不算的那些人，都一样，99%的人都有汉族父系。所以这个事情很简单：当你说李唐王朝是胡人，好，李唐王朝为什么是胡人？因为他们家曾经娶过汉化鲜卑的媳妇。行，既然娶一个汉化鲜卑的媳妇，全家就要变成鲜卑人，那么为什么会有“汉化鲜卑”这个群体呢？因为鲜卑人娶了大量的汉人媳妇，所以没有鲜卑，只有汉人。就这样。你要按照同一个标准来界定这件事情，有没有同一个标准？结果到你那儿没有。

李唐王朝的父系血统非常纯正。实际上，整个中国的父系血统来源都非常单一。中国90%的人拥有着五个（还是六个？）超级祖父。父系血缘的来历非常单纯，上溯到一万年前的标准时，我们就那么五六个老祖宗——父系祖先、老祖父就那几个，全中国90%以上的人都直接囊括进来了。五个是吧？好的。很多人说“三皇五帝”是传说。我们目前的推论是这样：可见我们三皇五帝的传说很科学。但这里得说一下，我们把它叫做传说。你所知道的女娲、三皇五帝、共工怒触不周山、祝融共工打仗、夸父追日、精卫填海这些东西，叫做传说。但是，就这些玩意儿，你放到中国以外的地方，它叫做“史诗”。这也就是双标的问题。我说了，所有的东西你不要双标。只要你不双标，我什么标准都能接受。如果说你必须要有时间码清楚的档案、文字记载和汇总，所有的文献都要有可考证的源头，所有的抄本都要有上一个版本，它中间的流变在哪个地方发生了什么样的变化，遇见了什么东西，都应该可以交叉印证并且必须交叉印证，如果做到了这个才叫做历史，那么全世界只有中国人有历史，别的人就不要出来谈历史。如果说所有在这片土地上面发生过的、听说过的、记得曾经有过的事情全部都可以叫做历史，连荷马史诗都能够叫做历史的话，那么女娲造人就是历史，盘古开天就是历史。我们的这些东西全部都属于史诗。很多人老说“中国人为什么没有史诗”，谁跟你说没有史诗？我们有，只是因为我们有历史，所以我们的史诗叫做传说、故事、神话，它们就不再叫做史诗了。
