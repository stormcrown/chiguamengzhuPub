# 第217章  武当道长的少林梦 #武当 #少林 #武功 #主持

等下，历史上有没有少林寺武功的记载啊？是这个样子啊，我们得知道一件事情：就是你说有没有武僧这群人啊？它是有的啊。武僧本身是存在的，包括我们的各大寺院当中呢，会培训一部分的僧人做武僧拿来护院，这种传统也都是有的。但是呢，说少林武功特别高强、雄霸武林的，那是小说和电视剧、电影给大家进行的创作啊，给大家进行的创作，不是武功高强。

那少林为什么会有武功高强的传说呢？包括说为什么小说会挑选它拿来创作呢，而不是其他的更加有名的千年古刹呢？呃，这个呢，可能跟少林寺当年的一些经历有关系。因为少林寺曾经啊，有一段时间就非常的善于经营。今天大家应该也看得到，它还蛮善于经营的啊。那个参佛参得好不好？不清楚，没聊过。但是呢，善于经营这件事情是板上钉钉的，这个买卖确实做得很大。然后它历史上面其实也有一段时间非常善于经营。我昨天直播的时候聊过我们传统寺院啊，历史上面的佛教的寺院，很多时候它的这个经营项目当中，一个方向就是圈地啊，一个方向就是放高利贷啊。这两个东西呢，是在历史上面寺院长期经营的项目。而这两种项目呢，相辅相成：放贷放到一定程度，哎，你还不起，你的地就归我了，于是呢，就圈进来了啊。圈地、圈人以及放贷，是历史上面很多寺院的发财之道。

但是呢，这种买卖凭什么要由你来做？是不是？所以除了这个之外呢，它还会勾结一些势力团伙，找一些靠山。那有靠山，它这个地方又有金融通道，就会产生什么呢？就会产生洗钱啊。这个呢，都是属于寺院系统在历史上面曾经有的各种各样的发财的道路。这些事儿呢，也一直都存在啊。甚至说它的高利贷利到了什么程度啊？它曾经有被限制的时候，规定它的利率不得高于100%啊。那就是属于啊，这个就是属于它的一个生财的东西啊。那你想一想，用这样的发财的方式会不会结怨啊？你考虑一下这种发财的方式会不会结怨？既然会结怨，而且呢，它又一定会圈地，涉及到地权之争的时候，地是农民的命根子。那它如何保证自己能够把这个玩意儿都圈拢在自己的手上、没有问题呢？它就需要有一个武力来保护自己。所以它确确实实有动力来培养武僧，但这不是因为它是武林高手，有什么秘籍——跟秘籍没有任何的关系，也跟它是不是武林高手没有任何的关系。而是这样的，寺院对于培养武僧啊，是有刚需的啊。这是这一块儿。

然后呢，啊，这是它啊，历史上的经历。然后在新中国是不容许这样的事情的——我们的所有的土地都是收归国有的，不允许圈田，不允许占地，不允许欺男霸女啊。所以新中国的时候，对这个少林寺是进行了一次大整顿的。

但是我好像是不是跳级了啊？应该再往前面倒一段啊。大家把刚才那轱辘掐了，呵呵。往前面再倒一段：就是它为什么显得武力很强盛？就是在战乱期间的时候，它之前本身就有这个保安团队，然后在战乱期间的时候呢，有各种的军阀的争端，军阀的博弈和斗争就会在这个地方上演。而这些僧人呢，啊，依靠了其中的一个军阀，于是获得了一些军火，参与到了这种火并之中啊。之后呢，就被清剿了啊，寺院呢，也被烧了啊——说的就是少林寺。它呢，因为参与了战乱年代的军阀纷争，集聚了军火之外啊，还因为战败的缘故，遭到了殿堂遭到了焚毁。就是这样的一个经历。

然后到了新中国的时候，尽管说它的形势已经大不如前，但依然显得自己很硬气，因为它还留了一部分的武器和武僧在手上。所以呢，啊，它就显得还极有威慑力。所以为什么就有一种民间的传说，说少林寺盛产这些习武之人啊、盛产这些武僧，说他们都是特别强干的武林高手？其实是因为他们当年曾经参战过啊，是这么一个原因。是因为他们曾经参战，所以确实有一定的武力基础，就有了这样的一种民间传闻。后来呢，它经过了新中国的改造，把那些欺行霸市、欺男霸女、做过坏事的僧侣该处置的处置，该查办的查办，寺产呢，都经过了清算和清缴，变成了一个正规的、受国家管控的寺院。这个时候它内部的僧团其实也没剩几个人了啊——这个玩意儿到建国的时候，它的僧团就没剩几口人了，因为那些之前干过坏事的都被清理出去了，好像就剩下十几口啊。这个呢，就是建国之后的经过了改造的正规的少林寺，就是这样的局面。

但是呢，你要知道，过去的民间有它的那个，嗯，啥——善于武争的这样的一个名声，名声在外就有故事。而这些传言与故事呢，就被小说家收集起来，变成了小说的创作素材。之后大家就知道了：你所看到的说少林寺武功雄冠天下啊，如何的这些东西哪来的呢？就是这些小说对于民间传闻的再创作。然后这些小说有名了之后，又改编成了电视剧和电影，然后由于取景的原因，由于对于那些故事的各种各样的畅想，尤其大家经常说，哎呀，哪儿哪儿有扫地僧，是不是啊？这些引发大家想象力的东西，才是让它变得红红又火火的一个啊，基本的前置条件哈。所以它的那个传闻其实呢，一个是民间传说，另外的一个呢，就是来自于小说和电视剧。

嗯，啥，批清的人祖先就是顺民——怎么批不批自己的祖先这种言论？我的祖先当过顺民。第一，承认这一点，我的祖先当过顺民。第二，我的祖先革命成功了，洗刷了自己的耻辱。所以我今天要继承他的遗志，而不能曲解他的忍辱负重。不能因为他曾经卧薪尝胆，我们就只说他吃屎的事情，而要去讲他是如何折服、如何博弈、如何抗争、如何成长啊。就是少林寺这边它的所有的这个创作的基础啊，基本盘刚才梳理了一下，基本来自于此。

然后呢，就是我私人的小消息分享给大家的：就是我之前跟武当山这边的一些武馆啊——啊，我参访过一些武当山的武馆，然后里面呢，有这种武馆的道长，聊过他的一些生平啊。其中有一位道长是这个样子的：他在少年阶段呢，就受到了少林寺相关影视剧作品的吸引和感召，就特别想要去少林习武啊。然后这个道长呢，他就在自己的少年期出于这样的梦想，他就离家出走，自己跑到了少林去，想要去当一个武僧，想要习武。他就跑出去了啊。然后去了之后，过了一段时间逃跑了啊，他还特意描述过他逃跑的场景。为什么？因为那边的习武特别的苦啊，异乎寻常的苦，只有吃苦，他不给你讲道理，你不知道自己为什么吃这个苦，也不知道这样吃苦了之后有什么好处，什么都不知道，只管吃苦就是了。所以呢，过了一段时间之后，他就扛不住了啊，就天天都是在那里吃苦，别的东西一点拿不着。所以呢，他就扛不住了，就逃跑。他说呢，就是他那边训武僧的这个地方啊，就是培训的那个地方啊，纯粹培训的这个地方，一旦有人要跑的话，他们的武僧会在外面去围追堵截，把人抓回来。然后他说他自己的身手还不错，很灵巧——当然了，他到现在都很瘦啊，那可能确实身形格外的灵巧一些，反正就寻机跑掉了。但是呢，他热爱武学的这个思想没有变，既然投少林不成啊，于是呢，他就改投了武当啊。现在他是武当山的一个武馆的道长。

嗯，好，欢迎来到直播间啊。什么？嗯，央视转发汉服身份证呢？哎，好的啊。啊，你们谁需要换证了？现在就可以开始置办衣裳了啊。需要换证的可以开始置办衣裳，捯饬自己的造型了，去研究一下这个做造型啊、选服装啊，都有些什么规范。然后呢，可以去拍新版的身份证了。嗯，啦啦啦啦啦。嗯，啥，国防大学也扛不住了？可以退学了吧？哎，姓不一样啊，那能一样吗？这不是扛不扛得住的问题啊，这就不是扛不扛得住的问题。嗯，什么说啥呢？有一种断网的感觉。说少林寺呢，啊，说天下武功出少林啊，说少林武当之争大概就是这么样啊。嗯，注意了，那个啥逃兵啊，当逃兵和从一个没有良好的教学、只是单纯让学生吃苦受罪的一个武校里面逃跑，性质是完全不一样的啊。所以千万不要把这个东西混合到一起去。嗯，啥身份证？全国所有民族吗？啊，专门强调汉服的原因，并不是其他的民族服装不让用啊，而是之前其他的民族服装就可以用，只是汉服不让用。现在呢，是汉服也可以用了。棒棒的。我非常支持民族平等，支持的就是所有的人都平等啊，大家都是一模一样的共和国公民，所有人的权益都应该是一样的，权利与责任之间应该是对应的。就这么一个要求。
