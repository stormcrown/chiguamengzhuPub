# 第213章  古代版“现代化”是西化还是汉化

好的，以下是整理后的易读段落，已修正错误词汇，内容未删减：

嗯，好，我们刚才说的是汉化和现代化是吧？无所谓。我知道这个事情其实一直都有人说，但有一件事大家不用太过放在心上。要知道，在古典时期的时候，我们发生过很多次大规模的汉化。最典型的一次就是鲜卑人的从上至下的强行的汉化——改汉姓、用汉俗、着汉衣。这些很明显就是一个交融的行为，是一个主动的交融的行为。实际上，在他们的晚期，始终都存在这样的一种努力，尝试融入的情形。打从他们开始设立政权开始，就发现了问题。

为什么会这样？因为五胡乱华在最开始的时候，其实他们只是想打砸一番，起初的动机没那么好，就是想趁乱打砸一番而已。但是后来事情做得越来越大，最后出现一个问题：把自己的阵仗搞太大了之后，他发现你想要在这个地方扎根生活下去，就不能像游牧、渔猎那样随随便便地四处游走、放牧——这种方式不可持续。于是，他们后期的领导者们开始一个又一个地建立政权。当他建立政权的时候，就发现了问题：他没有办法按照之前那种比较野放式的、部族式的规则去设立政权。设立出来的东西没有办法搞好生产建设，没有办法维持良好的生活，也没有办法维持良好的社会秩序。因为只要人群的密度足够大，你的社会管理秩序就和那种地广人稀的地方不断流浪完全不一样——这是量变引发了质变。所以当他进展到这种设立政权的阶段时，他就猛然意识到，按照之前的路子，这事儿办不成。所以，普遍性地，他们开始招揽汉臣、招揽世家大族的加盟，进而改变了自己的行为模式和文化操作方式。这就是五胡的后期，从设立政权开始，他们一个又一个地开始做自己的汉化。在那个时候，它是明晃晃地叫做汉化的。那他为什么要汉化？因为只有这一人群的文化和社会管理方略，是为着高密度的农业夹杂工业式的生产方式而设立的组织方法。而其他的五胡都不是他们自身的传统，自身的文化不支持这样的先进生产方式和先进的生活方式。他做不到用它原有的规则，会对这些生产产生系统性的破坏；破坏了之后，他自己都享受不到好生活了。所以变成了他本来是想出来享受一番，结果最后发现如果继续这样做，会虐待到自己头上去，而且会越来越乱。所以我们古人为什么将其称之为“乱”？就是因为这不是一个有秩序的行为。我们的古人将其称之为“乱”，这个命名是有缘故的，因为他们不是一个有秩序的行为。实际上，它本质上是发现了自身的秩序不足以进入更加良好的生活态势，所以才会去主动改变、主动汉化。为什么那个时代明晃晃地叫汉化？因为它没有第二条路可选。注意了，那个时期他们是没有别的道路可选的：他们要走向文明、走向先进，只有一条路，只有一个样板打在前面——就是汉化。在当时的五胡时期，只有这一条路，没有别的先进的样板放在前面。谢谢“淘气的哥布林”啊！重要的是榜样、是模范的问题——他不知道前面还可以走什么路。就是说“摸着石头过河”，石头呢？那个时候的石头就剩下这一家，他只能这样做，这就叫做汉化。所以古时候的汉化就等于当时的现代化。当时只有一拨人是逆行的，就是高欢的高氏家族，他们是倒行逆施的。其他的人群都在努力地、积极地汉化。这个流程可以说，在那个时代，汉化就是当时的现代化，毫无疑问，因为只有一条道路。

那么今天为什么有些人特别纠缠“到底是汉化还是现代化”呢？啊，这就是刚才那个提问了，哈哈！我终于转进到你这儿了。为什么今天会有人特别喜欢纠缠“我们才不是汉化呢，我们只是在现代化”？为什么他会纠缠这两个东西？因为在今天，至少说在绝大多数今天的人的视角当中，现代化不是只有中国一条路线，也不是只有汉化一条路线，它还包含了西方路线——能够理解吧？因为今天的现代化，连中国以前都是以西方人为样板的，以欧美为样板的，因为他们优先地步入了现代化，他们走得比我们快，他们看上去就比我们发达先进。这是我们过去很长的时间去向他们学习，所以说“摸着鹰酱过河”——大家现在都听过这个哈，“摸着鹰酱过河”。所以，过去的很多年间，其实从清末就开始了，我们所有的这些改革也好、尝试也好、试点也好，都是在推向着西化的。那个时代说的西化就是现代化——清末和民国时期的西化就是现代化的意思，因为那个时候所打出来的样板房是什么呢？是欧美的这一套西化的路径。包括当时中国国内的很多先进的革命者、优秀的文人志士也是这样想的。大家前两天是不是还聊过关于汉字的罗马化问题、拉丁化的问题？就是这么个事儿。就是那个时候的这些文人志士，他热爱自己的国家与民族，但是他认为走向现代化的路径是什么呢？哎，你前面有打了样的是什么呢？他们都是表音文字，都是脱胎于拉丁字母的表音文字，所以他们学习容易、学习快，所以他们可以如何？所以他们可以发明什么什么东西，这些符号可以怎样怎样。因为那个时代，欧美的这些西方人走在了现代化的前头，他们是第一波走过去的，于是他们成了样板间。所以后面跟过去的人就觉得现代化和西化之间就划了等号。因此，从清末开始，我们有相当长的时间，一直到民国，有这几十年的光阴，一直在做的一件事情就是西化——文化上如此，技术上如此，人们的行为方式上也如此。大家在很多电影、电视剧当中去看到的——谢谢“牛峰峰”啊！——大家在电影、电视剧里面所看到的所谓的“民国的传统”，包括什么那些“冠夫姓”的，大家看过的吧？啊，什么“蒋宋”啥啥之灵的那个啊！这个就是民国时候冠夫姓的这种命名方法——女人结了婚之后，在前面加上丈夫的姓。现在其实在HONGKONG那边还有一些资深的女性还是用这样的方式来进行称谓的，大家知道哈，我就不全念了。总之，为什么会有这样的一种风俗？这不是中国风俗啊！很多人以为说古时候它就是冠夫姓的？没有啊！中国人自古以来女子不冠夫姓！我们自来不冠夫姓哈。但是到民国的时候，冠夫姓了——哪来的？西方人冠夫姓，至今都冠夫姓。西化了的群体都有冠夫姓的这种操作方式。这个呢，就是当时以为现代化等于西化、以为西化等于先进化的一种普遍性的社会风尚。所以到了民国的时候，这种风气就开始出现了。很多的民国留下来的所谓的传统，都是西化的传统，都是这个样子的。这是一个相对普及的状态。

那么什么时候发生了改变的呢？啊，新中国立国后的前面几十年是发生了改变的。那个时候为什么这样？唉，因为人家不让我们抄，不让我们“摸它过河”，不让我们学，它把我们封锁了。建国初的时候，因为被封锁了，所以我们必须走自己的革命道路。所以感谢你的敌人——有的时候因为敌人的摧残，会使得我们萌生自我保护的认知。我们立国之初的时候，我们的苦日子是敌人造成的，但是我们的自主意识也是敌人造成的，这两样东西相辅相成，共同存在。然后呢，啊，但是后来我们希望说和他们再重新接洽起来之后，这个现代化的进程、现代化的学习对象又变成了西方的时候，西化和现代化就又开始混杂起来了啊。注意，这是一个混杂，不是我们真的认为西化等于现代化，而是因为这个时候我们学习的先进方向是在向欧美的发达国家学习，以至于在很多人的认知当中就觉得西化就等于现代化。我要是English说得贼溜，English说得跟母语似的，那我就比别的那些只会汉语的人高级？嗯，怎么看？像游戏，游戏领域的渗透——一说游戏不好，玩家就会说“打游戏的管那些干嘛”。嗯，这些水军你理他干嘛呢？是不是能理解吧？啊，不要理会水军啊，水军能安什么好心眼儿呢？没有任何的文化作品可以脱离它的意识形态，没有任何的文化作品能够脱离它的思想内核。文以载道，“道”是最重要的，而不是它的“文”。“文”可以粗鄙，也可以精致，但是重要的是“道”。这叫做什么？这叫做“话糙理不糙”，但是你是不是太糙了？一点太糙，理也没糙——重要的是“理”啊，而不是你那个“话”糙不糙。所以游戏不在于游戏好不好玩，而看这个游戏它到底在干什么。谢谢“帅瓜群众”啊！

总之，在我们的这个路径当中，很明确的一点就是：事实上有很多人就是把现代化等同于西化的。一直到今天，很多人的潜意识当中依然把现代化等同于西化。而今天，我们的新中国用自己的路线超越了西方。所以我感激所有在物质层面、思想层面为我们的国家的建设添砖加瓦的一切工作人员，尤其是理工类的工作人员——他们用自身的智慧、自身的努力建设了这个在技术上面、在产品上面、在所有的物质层面上面客观性地超越了美西方、独树一帜的中国。而后我们今天可以理直气壮地说：现代化不等于西化，现代化有多种路线，中国人有自己的现代化的路线，而且我们的路线走得更加长远，我们的路线走得更加坚决。所以今天为什么有人还在说“我们只是现代化，我们不是汉化”？来，中国的路径是什么？是中国人缔造的道路。你可以说你不是在汉化，那它是什么呢？那它可以是“中华民族化”可以吧？能不能做到今后出门，别人问你什么族的时候，你说“我是中华民族”？我可以，你行不行？嗯，啥？嗯，我那天刚锐评完，第二天就差评如潮了啊。啊，我这个人别的不好说，在那个什么立场领域上面搞预言，我还是很靠得住的啊。这玩意儿，我的预言有时候需要穿越一年，但是这一次穿越的时间比较短，一两天的时间就发现我的预言成真啊，呵呵。啊，我这个人的预言的能力还是有一点的，尤其涉及到立场问题的时候，我棒棒的啊。我来夸一下我自己，我都夸自己了，你们给我点个赞呗。
