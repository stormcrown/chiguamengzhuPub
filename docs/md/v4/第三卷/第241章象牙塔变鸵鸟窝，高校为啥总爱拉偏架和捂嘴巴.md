# 第241章  象牙塔变鸵鸟窝，高校为啥总爱拉偏架和捂嘴巴

以下是整理后的易读段落，仅修正了明显错误的词汇和不通顺的表达，未删减任何内容：

摁一下，嗯。  
阿三在复旦打国内的学生，复旦掩护了阿三跑回国啊。第一件事情哈，第一件事情——那个咱们能够理解一件事吗？就是我们过去几十年，一直将高校称之为“象牙塔”。这个称谓大家应该看过吧，这个称谓大家总是看过的哈。就是什么叫做象牙塔呢？意思是这个地方相对封闭。过去呢，是把它当做相对纯洁啊，相对纯洁。但是按照今天我们所看到的场景，不是的——这跟纯洁一点关系都没有。只能说它是一个相对封闭的环境。所以呢，如果说有什么信息进去了，那他就会在这个地方被提纯啊。  

那么高校里面呢，就是有这样的一种特征：他的社会化程度不够，是一个社会化前置的一个前置的环境啊，他或者说是一个预备役。但是呢，他的实践程度呢，就远远地不足。所以社会化程度不够的状态下，他的这种相对封闭的小圈子、小群体，就导致了他一种奇怪的行为模式啊——就是出了任何问题的时候，他生怕社会上面的大众看见，他总是像鸵鸟一样地把脑袋往沙子里面一插，就觉得这事儿已经过去了。这就好，为啥？因为他把大门一关，真的就是一个象牙塔——你进不去，打不着、摸不着。你再多说两句的话，我们这都是未成年人，你要怎么样呢，知道吧？他就可以撒泼打滚。因为这样的一种啊，社会化程度不足、相对封闭的环境，再加上里面的人绝大多数都是知识分子和知识分子的预备役，这就造成了他的信息密度不足，与社会的理解深度不够的情况之下，他还自视甚高，并且认为自己才是那个可以“以智而喻仁”的人。这种情况就使得他们可以理直气壮地去做一些大众很反感的行为，甚至他自己的这种思辨的能力呢，都会受到这些事情的影响。这就是我建议他们都去接受一下贫下中农的再教育，在大夏天的太阳底下去除一亩地的草，然后他就知道什么叫做冷热，什么叫做寒暑温凉啊。不然的话，他们还会像今天一样的操作方法——就是遇到的事情，觉得这个事很有可能被外面的人谈论之后，我会丢脸，他转瞬就把那个门窗一关，这个事儿就给你画上句号了。所以他们经常性地生怕事情闹大，就特别喜欢使用一些手段，捂着嘴说：“你别说了，别说了，不说了，这事儿就没了。”这是一种习惯性的操作方法啊。  

那么这个方法对不对呢？要知道啊，人世间最大的永远都不是高校，也不是知识分子，人世间最大的是道理——什么人都不能够脱开道理而单独地存在。所以对于啊，地位更高的、格调更高的人，你可以说他是否符合道；那么对于没有什么文化的普通人呢，那就是你这个人讲不讲道理。中国人最在乎的就是道理，它是什么？它是规则，它是规律，它是铁律，它是应该应分的东西，它是默认的准则。而这个东西是不能被跳脱开的。所以有些时候，我觉得这些所谓的象牙塔，也需要经受一下舆论的洗礼，让他们多多地经历一下社会化的捶打。当然了，我依然还是建议他们多多地参与真正的劳动生产，而不是叉着腰在那个地方指手画脚的。因为这样的环境，会使得里面的人特别感觉自我高高在上，然后呢，就听那些五迷三道的声音给他传教了之后，他立刻就觉得这就是真理啊，我要拿这个东西去实操啊。就像这一拨儿的姓杨的那个小姑娘，她干的就是这件事情。她在做什么？她在拿自己的生活，她在拿自己的造谣、诽谤、诬陷、构陷的行为，当做一个社会实践来做。这是一个一点道理都没有，也没有任何的人品底线可言的行为。但是呢，因为他处在一个这样独特的、封闭的环境里面啊，受到了一种格外的保护，以至于他觉得自己身上盾牌无数，才会有这么猖狂的表现啊。  

说真的，以前获得了这种资助也好啊，这些乱七八糟的教育就是——嗯，境外渗透过来的，包括女权主义，包括LGBT，嗯，诸如此类的吧。之前很多年还包括了什么性解放等等的这种教育。在象牙塔内部，这些思潮的传播的速度，要远远地高于社会的公众层面。为什么？因为那个地方就是一个专门传播思想的人，而且那个地方的人欠缺实践基础。社会公众层面的人，他能够看得见实践，看得见社会是什么样——谢谢任跃亭——他知道世界是什么模样，所以知道传过来的那个东西似乎有点不对劲，哪怕没文化，他也知道不对劲。但是圈子里面的人不是这样的，他是说洗就给你洗干净了。所以像是这样的传播，你会发现他都会花大力气推进到高校里面。这个事情按照目前的、目前的证据看起来的话，他的对方的推进的流程，大约是在20年或者是20多一点点——这样的年份的长度啊——就是从21世纪初的时候，西方开始着手在中国的高校里面搞系统性的这种思想攻势。那么有没有成果呢？啊，有成果啊，就是这样的啊。  

当然了，刚才我们在聊的，实际上是关于复旦处理那些事情啊。这里头呢，谈到第一件事，就是所有的高校都有这种臭毛病——就是有事儿的时候，他喜欢把脑袋埋在沙子里面，把事情掩埋下去，就好像事情没有发生。然后另外的一个方面，也包括了另外的一个流程，就是我们从建国初的时候留下来的一个伏笔啊。我们从建国初的时候留下了一个伏笔，就是我们基于一个理论，叫做“涉外的没有小事”——啊，那个原话叫什么来着？“外交无小事”是吧？啊，还是叫“涉外无小事”？不重要。反正呢，就是基于这句话，它不是单纯的一句话啊，它不是单纯的一句话，而是基于这句话，它实际上构建了一整套的管理的流程。我不知道有多少的小伙伴，在这个相应的部门里面工作过，你们应该会相对地更有体验一些。实际上，任何这种案子、这种案件一旦涉及到涉外的时候，你的流程、你要填的报表、你要通告的单位的对象、你要对接的人就全变了，你的流程变得极为的复杂。所以为什么你会发现，哎，怎么遇到事儿的时候，遇到老外这事儿就变得特别容易解决，特别容易抹平啊？为什么呢？因为让对方按照正常的流程慢慢跑的话，成本极为高昂。诶，这个是我们从建国初的时候就埋下来的一个伏笔。所以在今天，哎，谢谢毕龙啊——所以在今天他还留下了一点痕迹，那就是我们的、我们这个社会管理的流程手续上面，有很多的东西还没有能够做到一视同仁，还没有能够做到一个完全顺畅、丝滑的管理流程。所以这属于是我们管理技术上面出现了滞后反应，有些人趁机占了便宜。那么有事儿的时候，我们就一次一次地讨伐这个事件的本身，最终的目的是敦促整个流程的改变，必须要让基层去解决这些事情的时候没有负担，他才能够按照正常的执法流程把这个事办好啊。所以我们在聊的是关于这两套啊，两个原因，造成了中国国内的很多境外的不法分子啊，再涉及到一些不是影响极为恶劣的案件的时候，他常常会被人家轻拿轻放，甚至放纵，就是因为这样的缘故啊。  

诶诶，谢谢观自在啊。那，呵呵呵呵呵呵，我就觉得啊，我就觉得在我的各种小伙伴当中，观自在是最聪慧的一个，哈哈哈哈哈哈。关自在能够关自在，能够看到我什么时候瞧的屏幕，能看见特效，哈哈哈哈。  
“历史上的女性大多数没有名字，只有姓氏是因为女性被压迫”——这个怎么怼呀？让他数一数，能够把名字列出来的女性都有什么样的特点，让她去数一数啊。能够写名字出来的女性有什么特点？他们共同的特点就是他们和那些男性一样，在社会上面、公正意义上面做出了贡献，或者做出了破坏，嗯。所以中国人所有的人都在讲名流千古，什么样的情况才能够名流千古？要么然是流芳，要么然是遗臭啊。无论是哪一种，你总归都能写上名字啊。男性想留下名字也非常难呐。对呀，就这样。你要产生影响力啊，你要产生影响力，然后才能说这啥事儿。  

哦，好，我看了，嗯，啥。看了最新的视频，我能做些啥？应该做些啥？想为文明做一点事啊。嗯，好，我们能做什么？该做什么啊？我们普通的人啊，注意一件事情：普通的人好好地工作，成为社会运转流程上面必不可少的一环，就是我们对集体做出的最大的贡献。你千万不要听那些互联网在那里撒泼打滚的人，就在那里鬼叫着说什么“啊，那你有什么贡献呀，你凭什么怎么怎么样啊”。我跟你说，我们所有的社会活动的参与者，只要你是全心全意地做出正面的贡献，一定在未来的哪一天获得应有的回报。所以不用说我特意设计了，我为了这些事儿必须得干什么？我们在自己的岗位上面兢兢业业，这就是最大的努力。  

西方伪史视频捅了老窝了，一堆捶我的，而且视频好像没了啊。没听懂到底发生了什么事啊。但是不重要啊，嗯，不重要。捶我的人，那不是满大街都是吗？到处都捶我啊，左边捶我，中间捶我，右边的也捶我，男的也捶我，女的也捶我，嗯，爱西史的捶我，爱明史的也捶我，爱清史的更是往死了捶我，喜欢外国人的捶我，觉得中国人呢，就是要团结、团结、再团结的人啊——不一定是觉得，也许是嘴上的依然在捶我。所以很奇怪吗？不奇怪啊，我已经很适应这种强度了。
