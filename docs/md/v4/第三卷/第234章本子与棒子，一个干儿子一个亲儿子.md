# 第234章  本子与棒子，一个干儿子一个亲儿子 #文化 #华夏文化 #日本 #韩国

啥？嗯，为什么日本越学习华夏文化，对我们就越仇恨、越恐惧呢？他不是因为学习华夏文化所以仇恨和恐惧的，而是因为自己的蒙昧和单薄。那个地方呢，土地非常狭窄，资源匮乏造成的问题，就是他们那里的人自身的心态也很狭隘。他那里的人心态是非常狭隘的。为什么？因为他的一生危机四伏，各种各样的风暴、地震停不下来。每一个人从出生到死亡，哪怕你什么都不教他，他都能悟出来的一个道理就是“人生无常”。你啥都不要教他，他都知道人生无常。因为他们那儿想活下来可太难了，耕地数量少啊。虽然渔业资源丰富，但是人类在渔业上面不当宝。这个我之前讲营养的东西时讲过，你全靠渔业能把你饿死。而人类的营养吸收当中，是不能只靠渔业生存的，做不到啊。我全靠吃鱼还能活？挺好？做不到的啊，这个事儿不可能。所以他需要耕地，但是耕地面积很少，技术又不足，所以自身的条件非常恶劣。然后自然上的各种各样的台风、地震，又没有办法放过他，因为他就待在那个地方。他天生的有这样的思想缺陷，没有宏大的心胸，也没有安稳的远见。因为在那个地方，这玩意儿不存在。能不能活着、能不能喘气，瞬息万变。在这种心态之下，他的国民属性就变了，就永远都是那种调调，那种丧里丧气的，然后又有点发疯的调调。这个是他的精神的本质。

至于说他学华夏文化之后怎么还没改呢？我做个视频吧，去专门讲这个，他为什么不改，为什么学了之后他没学好，为什么他学了之后学不明白啊。所以他不是学了华夏文化然后恨我们，而是他原本就因为自身的狭隘与匮乏，所以会萌生这样的贪婪之心，以及这种左右博弈的狭隘态度。这本身是客观存在的。在它的历史上面啊，明王朝的时候，他有一茬也觉得自己行了啊。有记得的吗？明王朝的时候，他曾经打到了朝鲜半岛，是明王朝派军过去援朝抗日啊。明王朝的时候哈，就是这样的。所以他们一直有这种贪婪的野心，干嘛呢？因为他那儿太难了。我们瞧不上思密达的那块地，那块地不行，不好，资源不好，但是那块地相对于那个岛来说，安稳太多了。所以他连思密达都羡慕嫉妒恨，你想想他在看到我们的土地的时候会怎样。因此他只要有机会觉得自己又能了，有了枪炮，我的枪炮好像比你行，你那好像很弱鸡，他瞬间就起来了。这跟他学习我们的文化没有任何关系啊。

哎，啥？嗯，如何评价高丽民族？为什么总喜欢偷咱文化？好，来，刚才说的那个高丽民族啊，这是付费的，我们要优先答啊。呵呵，说他老是偷我们的文化什么的。嗯，有的时候你委屈一下，生气一下，喷一下，我觉得我都是支持你的啊。你想喷就喷，想怼就怼，非常自由啊，绝对支持。这是第一个啊，我绝对支持你去喷隔壁的那个高丽，他们那一波啊，绝对支持。然后第二呢，就说他这个缘故啊，那就有意思了。其实我对他这个行为是非常宽厚的，我真的觉得无伤大雅啊。认真的说起来就是我觉得此事无伤大雅啊。因为隔壁的思密达，他们那个半岛上啊，他确实自古以来与我们之间的关系极为的亲密。在汉代的时候，曾经列为郡县啊，就是当过亲儿子，他们是当过亲儿子的。后来作为藩属国，也长期作为干儿子。所以很多的人说什么“汉唐在脚盆”啊，那是多年之后都有人在那里重复的一句谎言，叫做“汉唐在脚盆”，其实是扯淡。真正一直跟我们文化往来极度的深入、向我们学习的东西学得更多的，其实是朝鲜半岛。那这个事儿呢，是历史上面一直持续不停发生的，他们始终都是跟我们保持着极近、极近的关系。所以呢，在文化上面，它有大量从我们这里直接拷贝过去的东西。而且有一点它比脚盆好的东西，是它的这个学习呀，它的这种学习就是半岛上面的学习，它比岛上的学习要深入。我们对岛上的脚盆人的评价，可以叫做“有小礼而无大义”。他学我们的术，但是不学我们的道；他学习表面文章，他学习礼仪规范，但是他不学这些东西背后的东西啊，这是脚盆上面的特点。但是思密达那边不一样啊，这边的这个半岛上他是真学。尽管说他学的也有点夹生，但是这个夹生，嗯，我为什么很多的时候对他们比较宽容？因为这个夹生可以理解。他们的生存环境比较逼仄，在这样特定的生存环境下面，自古以来苦日子过多了，所以呢，他们就有一种自卑兼杂着自大的心态。尽管很多时候跟我们学了，但是学完了之后就特别喜欢老大声说：“这个东西就是我的，就是我的，就是我的，我做得老好了！”他们就特别喜欢这个样子啊。但是这个呢，就属于是，嗯，你说行不行呢？挺膈应。但是吧，我们之间的渊源很深，他们也确实自古以来跟我们是同步发展的，所以这个事儿上面我保持一定的宽容度啊。实际上我们自古以来也都没有阻止过他们的学习，从来没有阻止过他们的学习啊。我们对于半岛的这方面的学习，始终都是支持的。包括今天，其实全世界最反对半岛上面的人学习我们的是什么呢？是上一波对中国有歧视心态的老思密达们，他们对这事儿确实是耿耿于怀。为啥年轻人越学越偷啊？偷的东西越多，思密达就越像中国，都找不到他自己的影儿了啊。这是这几年那些老派思密达身上最担心的一个点，就是因为他们copy得多了之后，也找不着自己的根儿在哪儿了。就像现在他们的电视剧的那个古装剧，那服化道简直就是从我们国内的剧组里面直接拿去的，怎么看怎么像啊。很有可能我们这边剧组刚一歇业，他那边就把东西都搬走了，就这种感觉。当他所有的文化的符号都这样高度雷同的时候，哎，你搭理他干什么呢？让他玩儿去吧。暗杀，还有那啥，不搭理他。有一个前提就是足够强，一旦弱了，这个反噬是很厉害的。不是很厉害啊，不是很厉害。他们那里有一个奇怪的毛病，当然了，但他们自己觉得那是一个优点啊，就是他们自古以来的一个传统叫做“事大主义”。也就是它的周边呢，都是一些大国、强国，所以呢，他就是谁强了跟着谁，谁强跟谁混，好能够获得头一口汤。他的行为逻辑就遵照着这个事大主义去操作的。所以你说他那个弱了之后，它会如何反噬？没见到另一个强国的时候，他不会反噬的啊。有没有反噬背刺的时候？有什么时候呢？有另一个强国给他当主子，他换了，换了主人的时候啊，换了主人的时候才会这样。所以呢，你说担心吗？不用那么担心啊。一个谁强跟谁、主动给别人当追随者的——哎，我现在是计划着我这一个礼拜顶破天了啊，你就当我发个公告吧，我一个礼拜顶破天了，连一回麦连麦。这玩意儿太伤号了，这个平台有计算的规则的，嗯，一连麦咔嚓一下，人垂直性地消失了好几百人，然后平台就默认你这个地方有问题之后就不给你开流，知道吧？这个东西它影响的是真的很大啊。
