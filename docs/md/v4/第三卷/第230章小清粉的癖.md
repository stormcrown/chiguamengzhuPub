# 第230章  小清粉的癖

嗯，又讲大脚，又讲裹脚啊？裹脚的视频没看是吗？嗯，裹脚的直播也没看。天天都在琢磨裹脚，动不动就“皇后马大脚”什么的。皇后马大脚，嗯，是个电视剧。这是第一件事。然后说朱元璋的皇后是个大脚，是个天族，呵呵呵，笑死了。这个言辞从哪冒起来的？从清朝开始冒起来的。所以它不意味着明朝的女子都是掰断了的脚，它意味着清朝的女子都是掰断了的脚——因为是清朝的时候说的这个话啊。

来，下一段。明初的时候，朱元璋和马皇后在后宫里面非常喜欢任用女官。注意，女官和宫女完全不是一个概念。女官是皇后的助理，是要做很多公务公文相关工作的，有上传下达的责任，甚至里面有一部分要记载宫内历史的史官。这些岗位都是客观存在的。明初的时候，后宫里面需要选聘大量的女官，这个时候女官的第一要求标准是什么？识字啊！民间筛选要求识字。所以这里头反映出来一件事情：哪怕是明初，女性的识字率也不差，所以她能够在民间说“我们要招聘女官了，认字儿的过来考”，乌泱乌泱的就来了几千号人，然后一次就聘好几百。明初的时候，朱元璋如此，朱棣朝也是如此，聘用了大量的女官。这是一个关于文化水平的问题。

然后第二个就是，所有的女官在进宫担任女官岗位之后，第一个要办的事情就是放脚。注意了啊，明代的女官在成为女官入宫就业的那一刻，第一件要做的事情就是放脚。你能理解什么事吗？这就意味着她没掰折。大家见过清末的那种断小脚吗？见过的吧，大部分人都见过。就那种脚，你让他把裹脚布拆了，放脚他不能走路啊——就那种，你把他的裹脚布一拆，他就完全走不了路了。但是明代的女官进宫的第一件事情就是要放脚，因为他们之前会把脚缠瘦，好穿细高跟鞋。但是进宫之后你要经常为君王奔走，所以就不能穿高跟鞋，“duang duang duang”地，必须要穿平底，必须要把脚放开。能够放开就意味着一件事情：她之前不是残疾。嗯嗯，去查查史料呗。有图的两种裹脚方式，你敢不敢把墓葬都翻出来？明代的墓葬脚骨都是完整的。不行的，你去找法医鉴证啊，不要看那个图片说“你那几百年了之后，那脚趾头耷拉了”——对，有。你想没想过，几百年之后，你的皮肤、韧带、肌肉都没了，骨骼就啷当在这儿碰着了，一下耷拉了，碰到两下又耷拉了。所以他们的脚趾头不分男女，都有前搭了后搭了，这种类型不等于它裹脚，不等于他断足。骨骼有没有断，这玩意儿你要通过技术分析做进去。明代的墓葬当中女子的脚是没断的，清代的缠足脚骨则严重受损，损到什么程度？是这样的程度。明代的受损程度是什么呢？拇指外翻。明代的缠脚的受损，骨骼受损是什么？拇指外翻。这种东西你们看过的吧？什么叫做拇指外翻？你们在网上自己去搜图：常年穿细细尖头高跟鞋的那种，都会出现拇指外翻。所以你说啊，那个凡是裹脚都不行——请你从现代高跟鞋开始喷起，从法国王室开始喷起，从时尚圈的名流开始喷起。你如果是从他们开始喷起的话，我还可以跟你聊一聊；不然的话，你放着所有的时尚达人把自己的脚掰成了拇外翻，哎呀，起漂亮，人家是爱美；一放到了明代，哎呀，残酷压迫女性。双标！

嗯，好，欢迎来到直播间。嗯，高跟鞋是给男的穿的，男的女的都穿啊。法国最开始流行的一个人是男的，他家国王，但是很快地就风靡开了啊。好，欢迎来到直播间啊。嗯，啥？高跟鞋是躲屎的？那他要躲的东西可多了啊，脚底下有屎，头顶上有尿的。

啥？西方不认的？夏朝到底存不存在？别说夏了，嗯，我们现在认为我们的文明史一共是一万年。我们现在在正规的博物馆、展览馆里面的解说说明里面都是这样认可的，这属于官方定调——我们的文明史是一万年。所以不光夏是真的，前面的虞朝也是真的。只是说呢，文献什么的没那么多，只剩下了一点痕迹。但是我们说啥就是啥啊，我们说啥就是啥。为什么这个样子？因为实际上我们挖出来的文物足以证明那个时候有一个王朝，而且建设了极大的王城，王城的内部组织结构非常分明，它有不同人群的聚居区，聚居区的建设方式、格局都是非常明确的——这是一个典型的形成了阶级社会、形成了组织结构的一种现象。这种状态你必须说它已经进入到了一种王权管理时代了。唯一的、唯一的争议无非就是这个时代叫不叫夏，因为没找着这个字。但是这重要吗？这重要吗？不重要。中国人说了它叫夏，它就叫道夏，我就给它起一个名字，哪怕是我今天起的名字，它也是存在的。除了它的遗址当中没有找到“夏”这个字之外，它的文物的量级是非常庞大的啊。

这个事情啊，所谓的西方不承认，是因为西方给我们定了一个异乎寻常的标准：我们这里必须要文献和文物对应，并且字和字都要对上才能够承认；其他的地方呢，要是字对不上的话，我就再编一个就可以了——只要有东西在那、有人类的痕迹，那就是伟大的文明体。它就没有用一致的标准来做点评的时候，我们非得要听它的吗？当然了，我们这里的学者还是有很多人非得要听它的啊，不然怎么吃饭？
