# 第207章  镇宅要买什么五帝钱，姜文电影翻车怎么看

我红了，居然是吗？我才开播诶，没有几个人呢，结果竟然有黑子。你看见了吧？我嘞个天呐，人家居然在这里埋伏上了。哎呦我去，高级了喂，高级了喂！黑子比粉丝努力，那肯定的呀。我就说正常情况下，他们都是我开播十几分钟之后才出现，那就比较明显——你下单之后人再来嘛。结果您瞅瞅，这才开播，真的，我这边才开始，人就已经来了。厉害了啊，高级了，高级了。

那啥，是不是在看守所里待了90多个点？嗯，那看守所条件挺好。嗯呵呵，嗯，住的酒店不错。嗯，感谢各种领导的安排啊。那个住的酒店确实不错，主要是位置好，门口那么多商场，随便我逛。但是吧，我这个人太容易走丢了，没敢逛啊。唯一吃多了一顿，然后跑出去逛商场遛弯、消消食，结果差点没出来——门太多，找不着门了。其实我完全可以体会到的一件事情就是，哎，给我安排住的地儿呢，一个是离会场要近，免得我走丢了；另外的一个呢，就是，哎呀，什么偏僻的地方都无所谓，因为逛商场我也是不敢逛的，逛着逛着人就丢了啊，找不着门。是的，我去逛个商场，把自己逛没有了啊。坐个火车也差点坐错了啊。坦荡地承认，我这个人出门的时候自理能力比较差，走路的时候容易丢，逛商场的时候容易丢，连坐火车都容易丢。我是很认真的——坐火车的时候坐错了。嗯哼，我能怎么办呢？我不是故意的。两辆火车放到同一个站台上面去，一左一右地登车，结果相差的时间不过10分钟左右，我上错了车。幸好我的座位号上面，对面的那个车上面有人，幸好那个位置有人。我愣了一下，于是向对方咨询了一下这是什么车次，发现不是他错了，是我错了。上错车补票没有？不存在这个事儿哈，我踩着人家发车之前的点下去了，换到了正常的位置上面，知道吧。很多人很关心啊，对的对的啊，我踩着点儿冲下去了，找到了自己正确的位置。

嗯，说到那个买了大词典赠的五帝钱，是不是夹带私货呀？啊，好啊，我们来聊这个啥的。那刚才这个小伙伴呢，提到的就是他买了这个书，然后觉得里面送的五帝钱是不是有点问题。对啊，有问题。而且我每次卖的时候都跟大家提啊，说这个里头送的那个五帝钱，书到手就可以扔了，晦气啊。嗯，这个书本身很好，但是啊，如果说你想要一个五帝钱啊，那这里头的这个赠品你别要啊。他的那个五帝钱收到手就可以顺手扔了，因为呢，他送的这个五帝钱，是清武帝，是清的五个君主啊，这种呢，也叫做小五帝钱。那五帝钱呢，它有很多种的类型，然后里面还包括一种更加古老的那种，五帝钱的系统呢，有点过于昂贵啊，甚至还有什么刀币啊什么的，这一串就有点贵了。然后还有另外的一种也比较贵啊，那就是秦、汉、唐、宋、明啊，各取一枚，这种呢，是大五帝钱。这种五帝钱呢，在市场上面还有，但是价格上面就比较高。最便宜的呢，也最普及的，因为便宜所以普及嘛，因为便宜走低价路线嘛，大家满大街都是嘛，所以那个五帝钱呢，常常都是清武帝。但是这种呢，就很晦气，因为用五帝钱的人，你的目的是啥是吧，你又不是过来搞集邮来了，你要是搞集邮那倒是无所谓。但是呢，用五帝钱的人啊，多多少少是图着吉利啊、镇宅等等的这种理由是吧。但是当你图这个的时候，你就得知道多晦气呀。清，200多年，一天好日子，人也变丑了，也变成文盲了，穷得底儿掉，尊严一点不剩了。在国际上面是个列强，就可以把我摁在地上摩擦，啥好日子呀？那个清的几个君主，不管他什么盛不盛世的，就算他是盛世，也不关咱们事儿是吧，他的盛世就是从我们这里搜刮粮食，喂饱他自己，那叫做他的盛世。所以呢，不管是什么时代，总而言之，只要是清的那个五帝钱都是晦气啊。他就对咱们没有安任何的好心眼儿，这几个君王就对中国的平民、中国的普通人、中国的原住民没有安过一分钱的好心眼儿。所以你指望他看门，指望他图吉利，那是不可能的。简而言之就是到手了你就撇了。这个书很好，可以看，没有毛病，但是它里面赠的那一小串钱儿，趁早扔啊，晦气。

好，欢迎来到直播间啊。啥，你说的正好相反？清武帝是收藏的热点，你们只管不要脸啊，我无所谓的。我无所谓的，你们只管不要脸，我无所谓的。为什么价格可以说明一切？嗯，价格可以说明一切啊。谁便宜谁贵，大家心里是门儿清的，知道吧。

嗯，我看看这个有一个付费的是什么啊——看过南京照相馆了吗？还没有啊，我还没有去。我前面专门去看了那个什么，看了那个姜文的。为啥要看姜文的呢？我欠他很多张电影票啊。我先承认一点，我前两天去看了那个姜文的那个电影，我不想点评。我已经不在乎他在里面又塞了什么东西了啊，反正呢，这个电影作为电影本身拍得不好啊，那它里头又塞了什么，我就不在乎了。因为电影拍得不好，故事讲得不好，其他的东西就不用聊，你知道吧。这电影我就不聊了。但是我为什么要去看呢？我欠他老多电影票了啊。之前，他之前他拍的那个电影我都没付钱看啊。我坦然承认，他之前拍的电影啊，我都没出钱。所以我就想着，他既然不是有他吗？我就出来付个费吧。就为了这个啊，我专门去看了一下那个，你行你上。那我能怎么办呢？就去看了呗。那你一个电影，你连故事都讲不利索，其他的事情还有什么可以说的啊？能把事情讲明白吗？票房不理想太正常了，那可不是嘛。那个故事讲得稀烂啊，你说里头有没有一点东西好的？有，有东西好的啊，不是什么都不好。但他故事讲稀烂，这件事情聊啥呀？嗯，一个电影的基本，你得把那个什么啊，你得把那个电影的基本盘盘住。电影的基本盘是你讲一个像样的、完整的、由起承转合的故事啊，他这个东西它没有，这玩意儿怎么聊啊？那没法聊，知道吧。

我看看啊，啥啊，姐姐，你这一带货多了一些铜臭味，思想深刻又犀利的文艺妹子形象打折扣了。不要给我瞎下定义啊，我是一个带货主播，不许说带货主播？带货的这件事情，你是想让我失业吗？先弄明白主体啊，一定要弄明白主体，我是一个带货主播啊。OK，你要我聊的、你想听我聊的任何的东西都属于我卖货的赠品。我在所有的带货主播当中，赠品给的那叫一个杠杠的，然后还不满意，说你不应该带货，啥意思啊？让我失业啊啊？嗯，啥加钱主播？对啊，要提额外的要求得加钱啊。嗯好，来来来，那个啥，嗯啥，带货最专业了。你看还是柠檬树下嘴甜啊，不愧是每天前三秒的榜一大哥嗯。
