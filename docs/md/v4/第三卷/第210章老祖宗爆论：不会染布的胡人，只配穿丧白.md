# 第210章  老祖宗爆论：不会染布的胡人，只配穿丧白

请问为什么在博物馆里看到许多西南少数民族的服装，会显得特别艳丽，很突兀的感觉啊？你注意一件事情：那个什么颜色这个东西，它其实取决于生产力啊，能理解吧？颜色取决于生产力。好，我们来整一波。那个我们古时候呢，说一些我们周边的胡人部落，说他们尚白，当年的匈奴也尚白。然后那个呢，啊，半岛上那边儿那一群人，你去问他，他自古以来喜欢什么颜色？尚白是干啥？简单，那就是没染料啊，那是技术问题，那都是技术问题。所以呢，但凡说你还有条件，人们都是希望自己身上的东西变得艳丽的。为什么？这是实力的展现啊，尤其是在古典时期，你的衣裳的艳丽程度，和你的经济发展水平、技术发展水平是几乎一比一的关系，知道吧？几乎是一比一的关系。所以为什么传统的服装你会发现很多好艳丽？为啥呢？我们老祖宗也曾经特别喜欢艳丽的东西，认真的说就是这样啊。因为你用这种很艳丽、很华丽的色彩的时候，你和旁边那些寡淡一片的一比起来，就看得出来自己的档次，这玩意儿叫做耀武扬威。所以客观上面来讲，事实上啊，凡是跟我们往来的比较深刻的，不光是西南少数民族，所有的地方只要跟我们往来深刻的，它是社会的上层，都是五颜六色的啊，老绚烂了，特别喜欢用那些极其艳丽的颜色。为啥漂亮啊？不然呢，一直很喜欢艳丽的。你放心。

嗯，那大家有没有去那个什么，就是一些建筑类的地方去参观？我今年不是去了趟山西嘛，也是网信办的活动啊。去那个山西的时候参访它的古建筑里面，就有专门的宋代的建筑的一些复原。那宋代的建筑它就有一个特点——艳丽啊。今天大家知道宋代很艳丽吗？但是宋代非常艳丽啊，非常艳丽。它在那个雕梁画栋，拿上边那个彩绘，狂狂的色彩就往上堆。宋代的审美是极为艳丽的，但是它艳丽却雅致，因为它把这种艳丽搭配得非常协调，主次非常分明，所以它虽艳但不俗。然后到了清的时候也艳啊，就变得庸俗了，你就理解不了了。你今天看见那些怎么艳丽的、很丑的，那就是清的，放心。

嗯，刚刚的中国现代化和西方现代化有什么差别吗？嗯，其实我觉得呢，这个事确实不敢一口咬死，但是呢，完全可以说一点，就是西式的现代化和我们的现代化之间，它有一个是不是有内生动力的问题啊，是不是原发性的，是不是有充分的自我成长的动力，是不是有足够的远见。在这个方面上，我们的差异很大啊。西方人的现代化，它的这个动机有毛病。之前我一直在谈一件事情，就是有关于说中国人的道德标准太高了，是不是约束了我们的发展，这个事我谈过很多次。我说道德这件事情是对我们大有助益的啊，对我们非常的有好处，所以我们才坚持了几千年。很多人不是特别能够理解这事，但是这里就有关系。那西方人他的现代化的崛起是为了什么啊？是为了发财，能理解吗？西方人的现代化的崛起是为了发财。无论是发战争的财、发殖民的财，他都是为了发财，包括他在本国推行工业化啊，也是为了发财。所以你会看到他的工业化的第一桶金，血淋淋到什么程度？大家会去看一些纪录片或者是一些传记的吧，嗯，就是当年第一波工业化的第一桶金的时候，他们对于自己的国民的压榨，压榨到了几岁大的孩子就要出来做童工，孩子夭折的原因是因为做工，这根本是一种反人类的操作。所以他的现代化的动机是什么？是为了发财，而且是为了少数的统治者发财。所有的人，包括他自己的臣民、自己的手足，都是他盘剥压榨的对象。我们后来所看到的那些，那个他们的社会福利啊，啊什么自由啊什么的，这些乱七八糟的东西，那是他们已经开始劫掠全世界之后，给自己脸上贴的金。这不是他的动力本身，这是他劫掠全世界获利无数之后，给自己贴的金，然后打了广告。用这种广告，用“我们这个地方老美好了”这个广告，去购买其他国家培养出来的各种各样的建设者与生产者。所以他的现代化从一开始，一直到发展到现在都一样，第一动机不纯，第二呢，非常的善于盘剥压榨，而且非常爱买人才、抢人才，用别人的人，用别人的地，用别人的资源啊。他不是独立自主发展起来的，也不是为了自立自强而发展起来的，是为了抢劫而发展起来的。所以他为什么会进入到瓶颈期？当你内部的东西不够吸引人的时候，人才就不往你那儿去了，他的自身的发展的能力不足，他的内生动力不足。所以不同的现代化本身，它在根本的动机、在它建设的方法上面都是不一样的，只是在某一些阶段上面，你看到它的成果很好看啊，但是阶段性的成果好看不等于它必然优秀。摁一下，呃，压榨到任何时代都有啊。你知道量变等于量变导致质变啊，量变到质变它是啥啊？注意了啊，一定是这样子的，量变最后会引发质变的。不是什么所有的盘剥都一样啊，十税一、三十税一不一样。如果进而到你压根儿不是人，你的东西都是我的，都是盘剥，能一样吗？不一样啊。

当代如果选汉服做婚礼仪式，男女服饰配颜色有什么讲究？除红以外有更多选择来历吗？啊，其实我们在那个婚服的选择上面相当自由啊，注意我们的婚服的选择，在历史上面相当自由。包括说大家认为可能我们的祖先不会喜欢的白，其实也曾经出现在婚服的系统当中啊。但是呢，注意了，白——什么样的白能够进去？那种漂得干干净净的、极为透亮的那种白啊，是能够进入到婚服系统当中的。然后我们的婚服系统当中呢，还会用青色，青也是非常高贵的颜色啊，青色。然后红色也会有啊。那个，嗯，很多的时候你说的那个绿呢，它的那个绿呢，通常指的是青啊，不是真的那种很绿很绿的颜色啊，通常是指青色，它是泛着蓝的啊，是这种颜色。
