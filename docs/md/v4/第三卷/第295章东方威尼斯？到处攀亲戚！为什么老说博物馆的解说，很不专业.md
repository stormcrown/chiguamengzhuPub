# 第295章  东方威尼斯？到处攀亲戚！为什么老说博物馆的解说，很不专业

嗯，之前是谁说的来着？你质疑西方历史造假，那人家质疑中国的怎么办呢？很多的官科都这样说：别人质疑了西方，然后他就开始回过头来质疑中国的历史。中国的文物合不合理呢？合理，有什么不合理的呢？咱又不能要求“二鬼子”也是刀口向外的，是不是很合理啊？他们以前也是这样干的呀。中国的文人在西方指导下，质疑中国的古代史，把中国的古代史全部都叫做“为史的时代”——就是中国“唯史论”的时代吗？“疑古派”横行怎么了？我们没有从那个时代走出来吗？走出来了呀。所以怕什么？质疑不怕啊，无所谓的。

事实上，只要有人活动的地方，它的历史当中都不可能是100%真实的，你没有办法进行一比一的复刻。这件事情放到哪里都做不到，除非哪一天你搞出了时空穿梭机——这个玩意儿也不太符合科学的逻辑。反正就是他没有办法一比一复刻，没有办法完全还原的时候，我们都承认它里面会有一些存疑的内容。只是有一些地方它存疑的东西太多，多到了让人觉得完全是空中楼阁啊，就是到了这样的一个地步。

所以呢，中国的文物当中有没有假的？不排除我们的文物当中有造假的呀。事实上，我们的文物造假这个行业很专业的。我们这些造假的专业人士，有的时候能把专家给忽悠得一愣一愣的，甚至专家都未必玩得过这些造假的专业人士。所以有没有造假的可能性？本身就存在。那能不能质疑？可以质疑。你可以翻来覆去地质疑，直到找出真相为止。为什么？因为中国人是唯物的，我们在乎真相。只管去质疑，最后这一来一去，我们不断不断地求证，不断不断地求真，大浪淘沙，最终得见真金啊。我们只想看最后的大结局。

嗯，啥？“昭武九姓”是什么定位角色？刚才的三个问题都是带孩子旅游，官方讲解觉得奇怪的地方，不是故意挖坑。是这样啊，就是这个昭武九姓呢，其实也是所谓的西部贵胄嘛。你觉得他感觉到有点……有点让你觉得不尴不尬的地方啊。我倒是大概知道是怎么回事。就是因为呢，目前而言，一直都有一群人，把当初的这个昭武九姓这个范围——本来呢，他是一个西部贵胄的群体，然后呢，把它归拢为“昭武九姓”——然后这一群人呢，已经出现了一部分的西迁。这件事情也存在。然后现在出现的状况是，他会把一些中亚、西亚地区的群体啊，称之为昭武九姓的后裔。这种攀亲戚的这件事，就容易造成后续的一些问题啊。在我们的各种各样的博物馆的宣传当中，包括了官方的宣传当中，现在特别喜欢去跟境外攀这种亲戚。你说这个亲戚攀得对不对？有没有渊源？那可能也是有那么一点渊源啊。但是呢，特别热衷于攀亲戚这件事，会造成一些问题啊。

啥？历史上的突厥活动范围到达了今天的中亚？我就是啊，我，我就是说，我就是说他的这种特别热衷于攀亲戚，这件事情会出衍生的问题。像我今年的时候不是去了一趟山西吗？啊，我去山西的时候，他的这个博物馆里面就特别热衷于说讲讲辽国故事啊。注意了啊，你们要是去了的话，你也会发现他的博物馆里面特别热衷于讲辽国故事，而且是以什么样的视角去讲的？就是以“他是我自己家的人”、“他是我自己”这种态度啊去讲的。那很坑！你们能够理解吧？我不能够讲得特别特别明确啊，我没有办法讲得那么细。但是呢，我可以清楚地说一件事情：就是这种特别热衷于用这样的名号贴金，特别热衷于用这样的方式攀亲戚的方法，它会形成自我认知的紊乱啊。他一定会形成自我认知的紊乱。因为他在讲故事的时候，故事的主角、故事的主体是另外的一拨人，而且跟你之间并没有直接的关系的另外的一拨人，甚至曾经是敌对关系的另外的一拨人。然后他讲得无比地亲切，就好像在讲亲爹妈一样的啊。最近还喜欢说西夏。对，他们很喜欢讲辽国故事、西夏故事啊，那简直就是停不下来，知道吧？你说事有没有发生？有发生。人存不存在？存在。但是当你用讲亲爹妈的语气去讲一个外人的时候，问题就出现了。

那么刚才你说的昭武九姓，为什么你会感觉到有点奇特啊？为什么有点奇特？就是因为这个现在确实有这种倾向。你说昭武九姓，特别喜欢在这事上攀亲戚啊。我剩下的就不说了，再说的话又刺激着人了。

好，为什么西方油画里有那么多不着衣的女性，但故事听着又很正经？哪个地儿正经啊？亲啊，他的故事你听着正经？呃，他的故事我听着也不怎么正经啊。那个就是西方的绘画，它是分阶段的。西方的绘画发展是分阶段的。有一些阶段呢，它确实里面老是不穿衣服。后来啊，他就开始普遍性地穿衣服了。那穿衣服这件事情，衣服会影响什么？衣服会影响你对这个人的判断啊。衣服会影响对一个人的判断，因为人靠衣装马靠鞍。啥意思呢？我要是穿上小旗袍啊，你就会猜测我是一个什么样的人；我穿西装打领带，你又会猜测我是一个什么样的人是吧；我穿汉服，你又会猜测我是什么样的人。所以，服装是一个人身份的某一种标志物。为什么他早期的绘画当中，有很多的人都不穿衣服呢？尤其是神明绘画和古典绘画，描述他们祖先的生活的时候总不穿衣服。你要知道，一穿衣服啊，只要一穿衣服，你这个绘画的时间段你就可以框定了。什么样的款式，什么样的面料，什么样的色彩，什么样的剪裁，什么样的配饰，在什么时代才有可能出现。这些东西都是时间码啊，这些东西都属于时间代码，不是随便往上加的。加了之后，你就会意识得到他是什么时候出现的。它相当于什么呢？相当于之前很多年我们一出什么古装剧，立刻有人说：“哎呀，他吃苹果了，那个时代错了。”那明代之前是没有苹果的。虽然这个科普是个假科普，但它确实是一种我们进行古典时代断代的一个识别的方式啊。就是某一个时代是不能够有某一种物品的，这个是最常规的一种识别方法。而在服装上面，它是特别容易出现这些细碎的印记。所以为什么早期的绘画中，绘画中他的衣服一般情况下是一块布啊？为什么是一块布呢？是吧？一块布这种事，你总也说不清楚它到底是什么时代的事啊。

有苹果，但是叫“紫柰”啊。“紫柰”是其中的一个品种啊。那个啥，苹果在古时候呢，也叫过“林禽”，有叫过“柰”啊，它有很多个名字。然后不同的品种还会有专门的名字啊。中国人是有苹果的。后续我们今天经常吃的苹果确实是外来的啊，那是因为品种的问题。不是我们没有，而是我们今天的很多的品种呢，是外来品种之后进行的改良和本土化啊。所以呢，它会表现为外来。但如果你按这种标准的话，那猪也是外来的。我们今天吃的绝大多数的猪，都是欧洲人培养的猪。我们传统的土猪剩下的量不多，之前有一段时间甚至于濒临灭绝。如果说那个时候我们没挽救，就让我们的土猪灭绝了，我们的土地上面全都是欧洲人培养的猪了。之后是不是就可以根据基因溯源，就说猪是欧洲人驯养的，跟中国人没关系？不能吧。

我们来带一个《博物馆全书》。这套书性价比可高了啊。快，家里面有没有小朋友的啊？有小朋友的需要亲子阅读的也可以这样啊。然后自己喜欢看，了解那个博物馆的基础知识的也可以看。这套书呢，它是全彩图印刷的，大开本礼盒装，一箱子是实测啊。还送了一个应该是随书送的一些赠品，赠品的质量也不错。咱们单独地去说里面的书，它是什么呢？它是中国的十个核心的大型博物馆里面的核心典藏品。把这些典藏品进行了精细的高清拍摄，也有局部图的放大，并且进行了详细的介绍。你看了这个书之后，就相当于逛了一个博物馆，足不出户逛十家博物馆。而且呢，你逛博物馆的时候，很多时候逛不了那么仔细看，也看不了那么清晰。但是这个书呢，把它进行了非常清晰的拍摄啊。这一套书我觉得真的是很不错。如果家里面有小朋友的话，可以做亲子阅读来使用。而且呢，可以让孩子们在一开始的时候，就形成一个审美上面的基础。因为文物上面是能够看到一个群体、一个国家、一个文明体系，它最初、最初的一种思维的方式，他的技术水平、审美情操。这些玩意儿其实需要一些直觉上的灌输。我嘴上跟你说“我们的审美老好了”，这个影响力很有限。但是你在博物馆里面看那些质朴的文物的时候，你自己的内心深处就会对这东西产生共鸣。如果从小就看到了这些，你就不容易被那些很花哨的、近现代的、其实十分浮华的东西所吸引。心里头有真正的美感、有真正的历史感的人，就不容易被现在五颜六色、光怪陆离的东西所欺骗啊。这个呢，才是我们去读书很重要的价值。这一套书性价比很不错，质量又好，又是一个礼盒装，拿去送人也是很有面儿的啊。
