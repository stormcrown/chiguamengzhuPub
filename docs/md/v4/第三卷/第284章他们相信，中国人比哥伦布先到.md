# 第284章  他们相信，中国人比哥伦布先到 #李兆良 #坤舆万国全图 #利玛窦 #西方历史

立马窦（利玛窦）来华是干什么的？  
嗯，好啊，好，可以聊一下啊。利玛窦呢，是明代来华的一个传教士啊。啊，说是传教士，传的是什么教呢？啊，他过来的时候，并没有一个成文的经书可以拿过来传教，有意思吧。他来的时候还没有一个真正成文的、成系统的经书可以拿来传教。在他来之后啊，就有了——就是这么个来历。  

这样的一个传教士啊，第一件事情是他的身份；然后第二件事情就是他到底是过来干什么的。按照我们内部的描述，包括说你去学校里面学习也是这样的答案啊：就是他来了之后呢，给我们传授了天文学、数学、嗯，几何、地图学等等的相关知识。通过他的口述和中国学者的翻译，给我们引入了大量的先进的科学技术知识。这个呢，是利玛窦他生平当中非常璀璨的一页啊，就是对他的身份的一个描述。  

之后啊，这里头呢，有一个标明是他参与创作、并且是他主动创作的作品，就是地图学上面的一个作品。现在呢，我们把它命名为《坤舆万国全图》。在国内的绝大多数书刊杂志上面，它的作者第一作者依然是利玛窦啊。但是呢，目前教廷方面已经把这个作品的作者栏当中的利玛窦删除掉了。为什么呢？因为在除了中国以外的其他的地方，大家都知道了：这个地图不是他画的，是明代的中国人自己画的，而欧洲人在那个时候没有画这个地图的能力啊。目前呢，已经中国以外的人都知道了，但是中国以内的话，标准答案还是利玛窦绘制了《坤舆万国全图》啊。  

当然了，因为在这个命名当中将它称为“万国全图”，所以有些人认为说，那个时代已经出现了什么啊，“列国平等”才会这样“国”那样“国”如何如何的定义啊——很多联想。但是咱得弄明白：《坤舆万国全图》这个名字是后来起的啊，这个名字是后来起的啊，这个肯定不是它的原名。包括说这个图本身，实际上呢，也是分了很多很多的切片儿，然后背后拼起来的。  

而现在去论证这个图的来历是谁在做的呢？我刚才说了一个说，现在很多的官科在揭露西方历史造假的问题，基本上都是跨学科的。其中呢，有一个类型是研究生物化学的啊，对吧？我刚才提到了生物化学——对，就是这个搞生物化学的学者，他的名字叫做李兆良。他去研究《坤舆万国全图》，通过图上面的文字、地方上的命名方法、书写的字体等等的这些细节信息，去论证了一件事情：这个图绝对不是欧洲人做的，而是明代的中国人自己做的啊。就是这么一件事啊。  

所以，在这件事情上面呢，我们就可以看到啊：利玛窦来到中国之后，在明面儿上面，在纸面上面，在纸面的说明上面，他给我们传播了地图学，但是他给我们传播的地图学，是欧洲人自己都没有的地图学。在纸面上，他给我们传播了数学、几何，但是这个数学和几何，他们在欧洲找到的源头——所谓的古希腊的那些伟大的学者——目前证明找不着他们。这些学者搞研究的那个地方，在那个时代是海洋，甚至不是陆地啊。于是产生出来的问题就是：大概率的，利玛窦在明代的时候，向中国传播了欧洲并没有的学说。一个传教士，一个在家乡讲神话故事的人，来到中国之后，向中国传播了他的家乡并不存在的学科。  

所以你说他来中国干了什么呢？啊，他向中国人传播了天文学的知识，然而在他死后很多年，欧洲才出现了历史上的第一个天文台。没有天文台的地方是怎么诞生天文学的知识，并且向世界去进行传播的？尤其是他在没有天文台的情况下，是怎么向一个原本就有天文学、并且是有系统性的官修天文学的地方传播天文学的？  

总之呢，就是他向我们传播了很多的信息，然而这些信息在他的故乡欧洲不存在。而且呢，在他的这个《坤舆万国全图》上面，有一个很荒唐的地方啊——就是这个图在东亚地区绘制得非常的精细，包括中国、包括脚盆（日本），我们周边的东亚文化圈里面的这些内容，绘制得非常详细，甚至有一些地方涉及到了当地的典型人物事件啊，已经精细到了这样的程度，对某一些地方的命名甚至做出了解释，非常精致，非常精美和准确。但是在利玛窦的欧洲老家，却画得极为粗糙。这张图画的欧洲很粗糙，大概的形状是有的，但是很粗糙，就是一个框架，有些地方并不准确。尤其是在这个地图上面，甚至没有教廷。你能想象得到一个传教士传播给我们的地图当中，没有教廷吗？  

嗯，是不是群都散了啊？抖抖上面的粉丝群散了。为什么还那里面？我天哪，请不起管理员。为啥啊？管理员说不能盯着里头天天干架。那么多的群，那么多稀奇古怪的人，干架干的方式五花八门的，理解都理解不了。然后管理员说干不了这活，干不了啊，得加钱。就我这么抠门的人，我怎么可能的呢？既然干不了，干不了就不干啊，这群直接解散了。这个工作项目就取消了是吧？这个事儿就简单，简单又直接：抠门的人，就是要用这样的方式来解决问题。  

啥可以全员禁言的？全员禁言和解散有啥区别呀？我要禁言那一定是24小时、365天一直禁，那请问和解散有什么区别吗？一、一模一样的啊，没有任何区别。所以直接解散。答案让他们吵呗？那不能够啊。你们知道那个管理规则吗？管理规矩就是群有问题，然后他们举报，最后把我这个up弄没了，我冤不冤呢？我都没有工夫看那玩意儿，结果把我弄没了，那凭什么我不能背这个锅啊？我没有时间管这个事情，然后一定要说管的时候呢，有人说啊，“那加钱可以管”，那还是算了，我喜欢免费的啊。就这么简单啊，我喜欢免费的。免费的是什么呢？免费的就是取消，就是免费的啊，所以我就取消了。  

嗯，是不是要重修清史？清史？哎呀，我管不上那些事情。那破玩意儿最好修不出来啊。问我1000遍、1万遍也是这个答案：那东西最好修不出来，修出来就是个祸害。啥时候能修出来清史？什么时候中国重回世界之巅，掌握了学术定义权，掌握了舆论话语权，我们啥时候再修清史。在那之前修，一坨屎一坨屎，修出来必然是一坨雕，多少花都是一坨，最多也就变成了雕花的那一坨。为什么？这不得不谈到偷粪这件事情了呀。我是很久没掏粪了对吧？我是很久没掏粪了。  

就是为什么现在此时此刻不适合修清史？其实早就修了，修了十多年了。修出来了没有呢？修出来了，修出来好几个版本，一经过审核，T（踢）回去了重来，然后再回来再修，再出来再踢回去重来，不让你出来。为什么？因为这破玩意儿吧，凡是在今天能够有资格参与修清史的，都得是学清史的，没问题吧？你都没学，你让我去修，那不是开玩笑的吗？我又不是搞内行的，怎么可能让我去嘛。所以呢，必须得是学清史的去修清史。  

然而学清史的人当中，能够学完了之后正确的批判满清的人有，但是呢，在舆论场上面不占主流。那些你们能够耳熟能详的、著名的研究清史的学者，能够站到讲堂之上，公开的对大众演讲的，能够写畅销书的，那些清史的研究学者，有一个算一个，全是轻吹（清吹）。所以你说他们都这么有名望了，影响力这么强了，你就说在修清史这个事上，邀不邀请他？让不让他参与？他只要参与，他是不是说满清十二帝个个是圣君？他要不要说康雍乾盛世？他要不要说封建社会的集大成者？而这些东西，这些颠倒黑白、指鹿为马的东西一旦出现，我就问你那个满清两百多年恶臭无比、巨大无比的粪坑，是不是又活跃起来，又开始雇了，又开始翻腾了？能忍吗？不能忍。  

所以修了踢回去，修了又踢回去啊。如果说他朝向人民史观的方向去修，那那些觉得满清十二帝个个是圣君的，就不高兴；如果你按照去批判他们进行去修啊，说他们是一群反动的奴隶主啊，那就不说是其他的学者和审核人员了，光我们评幕（评论区）上面雇的区，就得有多少人反对？所以为什么出不来？因为有两种声音在不停地博弈。那这个过程当中最好谁能赢呢？啊，这个事儿放到我这里，放到绝大多数的中国人这里，最好对满清进行深入批判、揭露真相的人会赢。但是你能保证这件事情吗？就目前而言，这个事儿保证不了。所以最好的成果就是他出不来，最好，最好就是出不来。  

那什么时候能出来呢？中国真正的完成了复兴伟业，站到了榜一的位置上面，然后掌握了舆论的学术的话语权——咱说啥就是啥，咱说哪个对，哪个就是对的，咱说哪个丑，哪个就是丑的——到了这个时候，你再去修这个，这个时候我们就可以定义哪一种方向是对的了。不然的话，现在那些千人憎万人厌的、恶心人的清史研究学者，老百姓都拿鞋底子呼他们了，但是他们依然是顶流的学者，可以出书，可以上电视。为啥呀？他背后有金主爸爸呀，不就是这种区别吗？所以最好别出来啊。啥时候能出来？啥时候咱成了啥时候再出来啊。  

哎哎，你们说的很好啊，你们说的很好。到点儿了到点儿了，再见。
