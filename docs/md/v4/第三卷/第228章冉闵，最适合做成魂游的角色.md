# 第228章  冉闵，最适合做成魂游的角色 #明末 #渊虚之羽 #冉闵 #魂游

有人说冉闵是英雄，也有人说冉闵是小人。能讲讲这个人吗？为什么说他是小人呢？你宁肯说他是坏人，也不能说他是小人。嗯，我之前聊过他。冉闵是啥呢？啊，冉闵就是那种行为方式上非常粗暴，看上去没什么素质、品性，道德上也没什么底线的人。所以按照常规来讲，他不是个好人——就跟我差不多，属于“这娘们不是个好人”。冉闵也属于不是个好人。他的第一个特点，就是不是个好人；第二个呢，他伟大的贡献恰恰来自于他不是一个好人。他最伟大的贡献就来自于他不是一个好人。你能理解这件事吗？咱们得知道那个时代的特殊性。五胡时代是一个没有道德的丛林时代。咱们先得把时代背景弄清楚——那个时代是一个没有道德制约的丛林时代，谁强谁有理，有奶就是娘。所以这个时候，坚持道德伦理但手上兵不够强的人，那就和拴个绳吊在树上没什么区别。但冉闵不一样，他没有这种约束。他用了当时五胡那样的运行方式，里面的什么道德底线、行为操守，和五胡是一样一样的。但同时，他有自己身份上的认可，于是圈起了自己的一票人。这就属于在冉闵这里，他终于能够在这个乱世里进行真正的公平竞争了。所以他为我们这个群体带来了喘息之机。因此，我们今天活着的所有的人，都要感激他的存在。如果没有他在那个时代当坏人，如果没有他在那个时代放弃了仁义，我们今天就没了。端起碗吃饭之后，放下筷子是不能骂娘的。所以冉闵是英雄。

至于说冉闵这个人，他的名声或者说他的作用如何，大家可以去反向印证一件事。冉闵是有一个纪念园区的。以前呢，他的纪念园区里面对他的定性，标着大字叫“民族英雄”。然后他的园区里面会有他所处时代的背景浮雕。那种乱世之中，只有枭雄才能取得这样的成就，所以他才能够被定性为一个英雄，而不是说他完美无缺。好，那么冉闵的这个纪念园，遭到了某些知名人士反复的、不停的骚扰以及投诉举报。来，特别喜欢投诉举报的是什么人？大家知道吧。凡是你要对我们自己的民族有什么热爱之情，他就恨你恨得牙痒痒，然后不停地举报，告到所有的单位去，然后在网上再耀武扬威，说“已经处理了，保证把他处理的底儿掉”什么的。“谁敢穿汉服，我就弄死他”什么的。哎，就这个人啊，他们也会去举报这些园区。然后园区不堪其扰，将园区内相关的文字说明给糊掉了，将浮雕给糊掉了。所以能够被这样的狗东西当成眼中钉、肉中刺，冉闵必然是好样的。敌人痛恨的，我们不说热爱，起码也要支持。

嗯，啥？昨天刷到有女的说我搞民族分裂？才昨天才刷到？嗯，你多看看历史视频，大概人家能给你刷个一年。嗯嗯，为什么冉闵的知名度不如岳飞？因为冉闵没有岳飞那么完美啊。冉闵的人生履历远没有岳飞那么完美。他身上呢，你想一想，他相当于是被胡人养大的，所以他的一些基本的行为逻辑等等，确确实实在那个时代属于——你都变成人家约等于养子了，你什么身份呀？这些事都有争议的。而且呢，你想他提出我们要干什么之后，他先发了杀胡令，之后呢，他又招募了胡人的军队，这叫什么？这叫出尔反尔，反复无常。所以呢，你说他最后取得的成就，切切实实地影响了我们的安全与发展，所以我们应该支持他。但是当你拿他去教化众生的时候，就不太好用。因为我说了，他这个人行为模式上不是一个很有道德感的人，也不是一个很有底线的人，也不是一个非常有文化的人。也就是说，他的行为带来的是好的结果，但他个体身上有太多的缺陷，他没有办法拿来去教育别人。谁照他的样儿学，那你很可能就把自己学成了一个鬼。你不能照他直接学习景仰。所以呢，有宣传也有认可，但力度不可能大。

但是岳飞不一样。岳飞的一生几乎是没有瑕疵的。他呢，一直都在向诸葛武侯学习，鞠躬尽瘁，死而后已。从生到死，他一直都践行着所有一个武人、一个优秀的武人应该拥有的品质，包括战力、包括服从性、包括品格的高尚。所以为什么后世纪念岳飞会更多一些？尽管说功败垂成，实际上岳飞并没有真正挽回局面，但他确实是一个值得效法的角色。要知道，我们去纪念一些人的时候，除了纪念感激之外，还要用他为榜样教化他人。冉闵呢，他确实挽狂澜于既倒，带来了巨大的成果，但是你不能说我们向他学习，没法学。他起不到这个作用，但岳飞可以。啥？岳飞长了个大小眼，怎么能没有瑕疵？外貌也可以拿来说事儿的吗？嗯嗯，好，欢迎来到直播间。嗯，那啥，岳飞现在有后代吗？有，有啊，岳飞有后人，当然了。这咋说的呢？不说了啊。今天有一些岳飞的后代又重新捡起了岳家的精神。但是啊，该说不说的，清代的时候也确实出了几个太孬的。可是呢，今天还是有很多的岳飞后人竖起了岳飞的精神，向他学习，这个就是非常可贵的啊。嗯，啥？孔家不也剃发了？没啥好说的。所以没啥好说的呀。确实存在啊，但也确实没啥好说的。没错啊，你说的可以。当然了，对于孔家的这个“万世师表”，我一向强调还是要分一下南北孔家哈，还是要分一下南北孔的。北孔本身就有点问题，他从元的时候就有问题了，专门负责写降表的。
