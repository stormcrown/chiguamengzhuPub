# 第274章  他们把我们的典籍和音乐，当人质 #典籍里的中国  #音乐 #古医书 #考古

中国什么时候能拿回那些被抢的海外典籍？这个问题很难回答。我们现在已经努力了一部分，已经找到了一部分。主要是来自于欧洲，一些欧洲收藏家手里的文献，我们弄回来了。这些可以凭实力和货币说话的东西，已经拿回来了。不能说全拿回来了，但已经拿回来了一些。但是要知道，有一些被他们弄走的文献，那不是普通的钞票和实力可以拿出来的。为什么？一定要是我们已经产生了碾压级的威慑力，然后对方基于恐惧在跟我们谈判。如果说你不赦免我的话，我这东西不给你了。这种情况下我们才会拿回来。因为把这些东西收在他们的库房里，不露头。这些人不是普通的侵占了我们的财产，他们大概率还篡改了我们的历史，大概率还吸了我们历史上大量的财富，犯下的罪孽可不是平平无奇、表面上那一点点问题啊。所以呢，即使我们已经做到实力上足够威慑他了，肯定也是要经过谈判才能把这些拿回来。

这些书现在在他们手上的价值是什么？就和当年嬴政在赵国一样，他是一个质子，是人质。一些我们的古典书籍，这些他们连放出来都不敢放出来的书籍、图册，现在在他们的手上就是人质。我们砸老鼠的时候，就怕碰着这个瓷瓶子，就得斟酌再三。等到我们的威慑力足够足够的时候，他们会找出来谈判的啊。谈判到一定的程度的时候，东西就能搬出来。你说他会不会把它销毁了？不会的。这个东西能留他一条狗命，他不会销毁的啊。

能否说说话语权、解释权、权威壁垒？嗯。我，我，我不是视频里面常常在讲那个话语权的问题吗？啊，话语权这个事儿哈，在中国人这个地方，我们会觉得格外的难受，知道吧？中国人为什么老是觉得话语权这个事儿让我们特别的难受？其实我个人觉得，我们在潜意识里面都还保存着一个东西，就是中国是应该拥有话语权、拥有定义权的。为什么这个样子？这个直觉，我们真的是在这个世界上面掌握定义权掌握的太久了。我们之前几千年一直都有文明的定义权，我们说了才算。

最典型的一个例证，就是我们的公屏上面也时不时的都会跳出来人说什么啊，“清王朝不管怎么样，还是对我们的疆域有贡献的啊。”这是互联网上非常普及的一段言论。这个呢，就是我们话语权和定义权沦丧的一个表现。为什么呢？什么是疆域？什么是疆域？这就是一个定义。在今天，我们的疆域就是我们的国界线啊。两个国家有的时候协商，有的时候是屈从于战争等等的原因啊。不管你是用什么原因，总之两个国家是平等的关系，然后中间画一条线，这个线有可能是自然线，有可能是历史线，也有可能是像殖民者画的，拿直尺笔直咔一下子一条线。这个是国界线。这个东西就是一个定义权。什么叫做疆域？那么今天的这个国界线的这种定义，谁下的？西方人下的啊。注意了，西方人下的。为什么今天老有些人说，这个东西就是清的时候奠定下来的，没有清，你哪有这个东西？问题在于，这一个国界线就是中国丧失了定义权的一个表现。因为中国丧失了话语权，丧失了定义权，于是我们的势力范围、疆域要如何划定，国与国之间的关系是什么样子，中国人说了不算，洋人说了算。于是才出现了所谓的啊，什么秋海棠叶那么大的一个圈圈。那是什么？那是西方人要划定势力范围的时候，他来界定世界的，他来在世界上面打格子、画边界线的时候，因为中国周边有一大片很荒芜、很难控制的地区，他们把这些地方视为战略缓冲区。因为他们想占领进来很困难，地方又大，条件呢又不是很好，人呢又特别的少，这样的地方他想直接掌控非常的难，他就把它作为一个无人区和半无人区来看待，作为战略缓冲区，然后就把边界线就划在那个地方了。他压根儿不意味着清在那个地方有管理的能力。但是为什么这个线可以那么画呢？因为这个线是洋人画的啊。你觉得这个线老大了，多多多体面呢，但是这个线中国人说了不算啊，这个线他就不是中国人说了算的东西，这个定义不是中国人下的。这是什么？我们看上去好像占了便宜似的东西，实际上是我们地位沦丧和话语权沦丧的一个表现。我们失去了对疆域、对势力范围、对国际关系的定义权。

而在那之前是啥样的？中国是实行圈层管理的啊，叫做朝贡体系，也叫做华夷之辨啊。它是干什么的呢？它是区分我们的亲近度的。用亲近度来画圈圈，不同的圈圈用不同的方式来管理。约束的程度不一样，税赋的程度不一样，管理的形式不一样。这叫做圈层，分远近亲疏。这个玩意就叫华夷之辨。最亲近的人用什么样的方式来管理？这个叫做啊，编户齐民区。然后呢，更外面一个圈层，亲近但是没有那么亲近，干什么呢？羁縻区。再往外面一个圈层，鞭长莫及，但是呢，它也象征性地听我的。这是什么呢？这个叫做朝贡区，知道吗？所以我们的普天之下莫非王土，是用这样的方式圈层圈层的堆叠出去的。这是什么？这是定义权。疆域的定义权在中国人手上。所以那个时候我们没有边界线。所以今天你去看古地图的时候，老说：“你看我们那时候边界线老小了！”我们那时候没有边界线。你觉得边界线小，是因为今天的地图学的学者，按照今天西方人画边界线的这种定义的方式，去画了古人的边界线。古人没有这条边界线。不存在。

哎，谢谢“跟着刘哥混”啊。我们是唯一的文明，干嘛不掌握话语权啊？这话语权也不是我们一开口就直接能够掌握的啊。所以我们现在是在逐渐的进步之中，达到了新的发展阶段的时候，我们应该就可以掌握更多的定义权了啊。我们现在呢，已经开始逐渐的掌握一部分的定义权。例如说，表意文字才是最高效、最准确、最有前途的文字，这件事情放到10年前，包括中国人都不相信表意文字的优越性。在10年前连中国人都不相信，但是今天它已经成为普世的真理了。这就是定义权的转移。

嗯，诶，谢谢“跟着刘哥混”什么？嗯，以前是做武器的。相信蛮夷都懂东风的力量啊。他们是真懂啊。这个叫什么呢？这个要叫做用对方能够听得懂的语言与他说话。用对方听得懂的语言与对方说话。咱们很多的时候啊，老是用我们自己的语言与对方说话，但是对方听不明白，理解不了。为什么？因为大家思维体系完全不同。一个在丛林法则之中成长起来的群体，他没有办法理解我们的这种什么谦让啊、谦逊呀、克制啊这些东西。他理解不了什么长远的利益啊，抛弃短线的收入去寻求长线的预期呀这些东西。丛林竞争之下，你不能惦记这玩意儿。所以丛林法则之下成长起来的人，他能够听得懂的是什么呢？错了你就应该扇一巴掌，对了就给一颗糖。他能够听得明白，其他任何婉转的方式对他来说，你只要对他好，你就欠了他的。因为丛林之下不存在这件事情啊。这就是天命，身为中央之国的天命。签条约，就是失去天命的表现。嗯，虽然这个词儿我觉得有点那个啥啊，但是道理是这个道理啊。

诶，这是什么啊？我看一下。“现在女权那么跳，怎么没有人管她呀？前两天启迪女权的关键内容还被删了啊。”我倒是没有看删不删的事。嗯，“那么跳怎么没有人管？”我们国家有的时候面对这种大事的时候啊，小事我们处理的还是挺快的，大事处理有滞后性。就是真真有滞后性，我也承认。有的时候我也等得毛焦火辣的，对吧？我也等得着急。但是确实有滞后性。可是呢，我就很相信一点，一定会处理。为什么？因为这帮子玩意儿，他背后真的有金主，他们真的有组织，真的是在搞颠覆啊。这帮子真的不是好东西，所以一定会处理。而且呢，他们已经搞了很多的事情出来了，也必然被处理，也已经被盯上很久了。但是这真是慢呐。你以为我不着急吗？啊，是真慢。我们的那个什么，我们的那个速度，有的时候我也等的毛焦火辣啊。关键是这玩意儿，你也不是这一天两天的催得成的。不然的话，你说那个叫宋什么的，那个家伙一天到晚的在网上面跳啊，光是在那里威胁我，威胁了多长时间了？威胁了大半年了吧？是不是他同时威胁多少人？一天到晚的这举报、那举报，这整活，那整活，还牵连了不少那个什么内线，听了风声，知道哪个地方什么人要调职了，立刻就搞一个大活，开始公告说我又举报了谁谁谁，哪个地方犯了多大的政治错误，然后等到人家被调职的时候，你一看就是我弄的啊，就是他们这些人。你以为看不见他们吗？看得见。但是真慢呢，我也是很烦的。但是这就是慢啊。诶，好，欢迎来到直播间啊。

虽然但是，日本（脚盆）的音乐是真厉害。你注意一下，这两年你有没有觉得他的音乐不怎么样了？发现了没有啊？大家发现了没有？这一两年他的音乐开始变得不怎么样了，发现了吗？发现了的小伙伴扣个一啊。他的音乐开始缺乏创造力了啊，开始走下坡路了，发现了吗？大家注意一件事儿啊，日本那里，想当年从我们这里是抢了大量的典籍去了。他抢了什么东西呢？可以保证的是，大量的医书和音乐类的书籍。他们抢走的不光是有钱没钱的问题，医书、音乐类的书籍，他们是成堆成堆地抢走的。中国的雅乐系统什么档案都没剩下来啊。我不是说民间音乐没有传承啊，我是说中国的雅乐系统基本上凉了。那些能够产生什么所谓国际影响力的，那些宏大的东方音乐、中式音乐哪来的？那个就是中国的传统雅乐系统。我们今天的雅乐系统，是我们音乐人一点一点地复原出来的，其实里头还有点七像八不像的东西。但是日本那里有原始档案，而且人家明确地告诉了中国人：来，要不给中国人看？不让看。但是啊，尽管他们抢了那么多的医书，抢注了大量的汉方，但是他们现在的这个相关的这个类别啊，汉医药这个行业越来越不济了啊，越来越烂，越来越不济了，越来越没用了。为什么？他不会。嗯，你们知道吧，他们不会啊。他们中间搞了“废医存药”，搞了“废医存药”，然后他们把这玩意搞废了。他们根本就不会辩证，不会看病，然后就只会卖，卖成方。我之前看的那个啊，我之前看的医生，我老是跟他说，我说我就不爱熬那个汤药子，费死劲的呀，累死我了呀，能不能开个成方给我直接买现成的呀？他说我可以给你开成方，但是你有没有本事按照成方生病嘞？我就觉得这话啊，特别的讲道理。为什么？你病的情况不一样啊，所以就是要有加减，要有调整的呀。日本人没有这个东西啊，他不会看，不会调节，不会治啊。日本鬼子这个事吧。

沈老师说的对。嗯，那个啥，嗯，干日本（脚盆），中国人不会有心理负担的，没有负担。这有啥负担的呀？我们已经准备了很多年了，压根没有负担这回事啊。不存在，不存在。《明实录》在“阿美”（美国）那也不给看。哎，对，明石路那边有一个，有一套啊，有一套，有一套完整的。然后我们呢，就是看其他的版本，还有各种版本参照啊什么的，东拼西拼的互相印证啊。嗯，支持。我认识的同学认为中国音乐只有五声大调体系，认知很明显是缺失的。对啊，我们今天的音乐系统是很有一点问题的。因为啥呢？因为我们传统音乐的很多典籍确实遗失了。除了音乐典籍的遗失之外，还有清的破坏。注意了啊，清也造成了破坏的。因为雅乐这个玩意儿以前呢是国乐大礼，还有宫廷内部使用。清的朝廷呢，他在这方面的审美能力就很次啊，他本身的审美能力就很次。所以清就已经对我们的雅乐系统造成了破坏了。然后呢，到了晚清时候，然后这些典籍又被日本搜刮一空。
