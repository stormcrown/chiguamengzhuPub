# 第270章  西方伪史注的水，想要淹没谁的复兴路？ #历史冷知识 #文化自信 #考古那些事儿 #西方伪史

好，欢迎大家来到直播间。嗯，关于西方说古埃及、古巴比伦比中国伟大这件事，我已经强调过了，大家一定要注意：在“西方中心论”的框架下，并不是说只有西方地界上的才算西方文明，不是这样的。关于西方中心论是如何看待历史和古代史的，咱们一定要弄明白：只要是非东亚文化圈，那就全部算作西方流。为什么要这样说？为什么只要不是明确的东亚文化圈，就都可以算作西方历史？因为国际历史学的话语权掌握在西方人手上，而他们对于非东亚文化圈——严格来讲是除中国文化之外的地方——所缔造出来的任何古代、远古文明，都特别尊重、特别维护、特别吹捧。你一定会注意到这个现象，你自己去汇总，是不是有这样的特征？你自己去回想、分析，是不是这种特征？他们对中国的考古特别严格。我们的考古，历史书上有写，文献有记载，墓葬里有出土，同时两边还得对得上。对到什么程度呢？就是书上写的字，你坟里头也得有那个字能对上，时间码也得对齐，什么碳14、光谱，一堆乱七八糟的鉴定都得做出来。这就是对中国考古的要求：所有的印证都得对得上，文献和考古对得上，考古和时间码对得上，字对得上，图样对得上，然后这东西才被认定为真的。对中国是这样的要求，但对除了中国以外的地方的考古，就不是这个标准了。在中国以外的地方，只要不在中国，他都可以吹。我们隔壁的巴拉特就有伟大的古文明，那个地方特别小，而且明显有毛病，还没有文字。你记得吗？对于中国的古典文明，他要求必须是有文字才算文明，没有文字的不算。但中国以外的地方，他就不要求了。记得那个伟大的玛雅文明吗？伟大的玛雅文明，因为它的继承者不在了，没人跟他叫板了，他就开始吹嘘伟大的玛雅文明，用各种杂志、传说、小道消息，给你讲那里了不起的寓言，最后缔造出的印象就是：那个地方确实很了不起，有一个遗憾消失了的伟大玛雅文明。这是真的尊重人家的文明吗？真尊重的话，你怎么把人家那个地方搞到没人了？他尊重别人的文明，却把别人的人给搞没了。之所以最后后继无人，就是因为被他弄没的——他灭掉了别人，然后说那是一个伟大的古文明。所以，只要是非中国的，他们都会说这是伟大的古文明，包括两河流域。两河流域有人类最古老的文明，这是西方人说的，根本就不是阿拉伯人说的，不是当地人吹的牛，是西方人吹的牛，是欧洲人吹的牛，好吗？所以你会发现一件事：只要不在中国，不管这个古迹多么不合理，他都说这是真的。所以我才说，所谓的西方中心论、西方文明史里面的这些古典遗迹，你不能只在欧洲找，而是所有非中国的都要纳入到这个圈层里。因为非中国的古典文明遗迹是用西方标准来界定的，而中国的古典遗迹是用专门针对中国的标准来制定的。

好，欢迎大家来到直播间。所以说到这里，就把这个以图证史——何晴老师的《以图证史》再给你们推一下。这本书挺好的，因为它是官科著作。我很珍惜这件事，很重视这件事。因为黄河清老师是一个官科，而且是史学方面的官科，亲自下场去撰写一本质疑西方历史的书，这极为难得。实际上，何晴老师这些年一直受到大量的围剿和抨击，非常困难。当然，这个书的论述一定很完美吗？嗯，不能够，做不到。为什么？因为西方历史造假的东西太琐碎、太复杂，而且有很多真实的、真相的东西不让我们参与进去。能够查证到的东西其实都很浅表，而且证据不足。但对于西方历史的质疑，你并不需要把西方历史全部整理出来。很多人喜欢在这个地方抨击，说你都弄得西方没有历史了，那西方难道没有什么什么东西吗？我不在乎西方历史什么样子，我可以反复一遍一遍地跟你们重复：我不在乎西方历史什么样子，他爱什么样什么样，想住树上住树上，想住坑里住坑里，我不在乎。我在意的是，西方伟大辉煌历史当中的这些注水行为，缔造了一个西方中心论，而这个西方中心论在影响着中国人的复兴，所以我必须戳破他的泡泡。撒谎了就是撒谎了，不对劲就是不对劲，编了故事就是编了故事。至于戳破之后，人家还有历史呢，难道说人家没有历史？他爱有没有，爱啥样啥样。他的历史到底应该是什么样子，应该如何打补丁？质疑你吃了没之后，他去哪里找粮食？那是他的事，跟我们没有关系，让他们自己去找。不要在这个地方上说“那人家那个地方如何如何”，你得让人家有什么？跟我没关系。但是他的西方中心论影响了中国复兴，那就不行，我就要戳泡泡。戳完之后他的历史千疮百孔了，最后想要自圆其说，怎么圆？他圆不归我管，这烂摊子是他自己缔造的，跟我没关系，我不参与，他爱怎么编怎么编。只要他编的东西不影响中国复兴，我管他吹什么呢？实际上，这个世界编造历史的人群多的是，不光是西方人编，非洲人也编。你们看了没有？前几年非洲政客演说，说白人殖民者在我们这里抢走了一切，抢走了我们的文明，抢走了我们的文字，抢走了我们的建筑——我也不知道咋抢走的，总之西方白人的一切科学技术文化都是建立在劫掠非洲的基础之上的。你说有些东西确实能抢，有些东西真抢不走呀。然而，当他用这样的方式去宣讲时，他在干什么？他在提振自己国家的士气，给自己编造一个了不起的古文明，让他的人民有更强的归属感。他编没编历史？编了，肯定编了，里面有没有造假的成分、吹牛的成分？肯定有，120%都有。但是不重要，没关系，你只管讲，我甚至可以给你鼓掌，我可以帮你传播他讲的什么故事，没关系的，这不要紧。为什么？因为人家没有用这个玩意儿伤害到中国人。

那西方有历史吗？肯定有，肯定有。但是，一个是长度从什么时间开始算有历史，另一个是含金量——里面哪个地方吹了牛，哪个地方是真实的；第三个是哪个历史是他的，哪个历史不是他的。要知道他的很多历史故事中，不排除有一部分是他从别处拿过去的传说，完全有这种可能性。所以你说西方人有没有历史？有，并且一直延续下来了。尤其是事实上，整个欧亚大陆上的人没有我们所想象的那么隔绝，我们在过去的时代里很有可能——甚至极大极大的概率——我们之间是一直有往来的。有往来的可能性，但人的往来不代表彼此就有了解，信息上有交互也不代表我们之间真的就很清楚对方是什么样子。所以我们不要为他们做保，他们真实的情形是啥样，那是要他们自己解决的问题。至于有或者没有，无非是长是短的问题，这世上谁还没个历史呢？

哎，好，欢迎大家来到直播间。啊，什么什么什么东西？嗯，古罗马冬天穿短裙？呵呵，你要是按照服装来看，你会发现他们的历史进程有点荒唐。以他们当年给自己缔造的这些伟大的祖先，他们给自己认的这些伟大的祖先，时刻石雕说祖先的形象，了不起的艺术，什么古希腊呀、雅典呀什么的。这些文明啊，缺衣裳，他们都缺衣裳，这事儿真的是特别缺衣裳，就是太缺衣裳了，让人很头疼。
