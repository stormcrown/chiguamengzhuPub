# 第293章  清廷的秘密武器是光速投降

经常在头条上看见一帮人否定农民的贡献，否定农民种粮的重要性。我一直想不通，这是一帮什么群体在投入经费，对农民有这么大的恶意。实际上呢，这种所谓的对农民的恶意，从清末就开始了。它不单纯是在侮辱农民，在传媒领域当中，更多的是针对中国人的文化。因为中国人的所有文明体系，包括我们的文化创作，都是从土而生的。我们的一切东西都是从土而生的，连造人的传说都是抟土造人。我们的生命、财富、文明都来自于土壤，对于土壤的信念极为强大。甚至在大禹治水的故事中，他的父亲鲧最初的治水方式，就是借了一块息壤——无限增殖的土壤、生生不息的土壤——去堵塞洪水、覆盖洪水，但最后发现堵不如疏。无论这个故事是什么时候创作的，有一点毫无疑问：中国人的思维方式和文化缔造中，有一个绕不过去的东西，就是泥土。

所以从清末开始，所谓“农耕文明落后、海洋文明先进”的说法就开始在文化宣传领域出现了。驾船的是勇武精神，种地的就是没用的废物。很多时候，你以为他瞧不起普通的劳动者，但它不是针对所有劳动者，而是针对农民。一定要意识到：他是在针对中国人都吃不饱怎么开船的问题。人家不在乎吃饱饭的事情，觉得有劫掠之心就可以拥有一切财富，所有人种田都要去供养劫匪。我不知道哪个地方的农民这么懦弱，我只知道中国的农民，谁想抢他一点地，他就要跟人家搏命。

甲午清廷故意勾结日本打击李鸿章，后来南方地主东南互保还以颜色。那不一定是打击李鸿章，但确实是在打击汉臣势力。李鸿章这个人，他的背景存疑。注意了，作为个体来说，李鸿章背景存疑。但是甲午战争这种光速投降，实际上还可以扛但他坚决不肯扛，以及本来可以买军火却一定要在里面偷工减料，这件事本身是为了打击汉臣的势力。为什么？因为清廷200多年一直持续不断的一件事，就叫做防汉抑汉。你就算翻遍所有清廷的典籍，最终都一定会得出这个结论。除非你觉得自己是八旗，否则一定会承认这个现实：清廷200多年的统治，干的事情就是防汉抑汉。既然宗旨如此，当走到清末这种独特的时间节点，八旗实力已经大打折扣，无法威震八方时，谁会起来？那些汉家的军头们就会起来。那如何控制这些军头们的实力呢？要么像过去很多年间一样，以八旗治汉军——200年他们一直在干这个事。但到了清末，八旗实力不行了，治不住了。这时应该怎么办呢？对他来讲非常简单：把这些部队当作炮灰，送到对外的战场上去，由敌军来消灭他。

八旗进关十几年就没有战斗力了？不是，不是，注意了。很多人喜欢说八旗进来之后就没有战斗力了，不是这样的。谁家战斗力沦丧都没有这么快。他老是跟你说八旗进关后很快就没了战斗力，这是让你掉以轻心。注意了，八旗的战斗力并没有下降。之所以表现不出来，不是因为战斗力下降，而是因为实力分散了。为什么分散？因为他在掌控了广大但所有人都在反对他的汉地之后，必须要在所有道路的要害要冲分布他的八旗兵马。注意，所有时代都有边防。今天的国家有没有边防？也有边防。所有时代、所有国家都有自己的军事防务要地。一般情况下，我们的要害之地在北方，在降雨线附近。为什么？因为这是两种不同的经营业态，中间会出现战争、劫掠，所以所有防线都设在这条降雨带附近。然后有时候，因为来犯的群体从海上过来，就在沿海设哨卡，这也是海防，属于边防。

但是有清200年，从他进关之后，他所有的防护策略、所有的边防军，注意了，整个200年他所有的防卫力量都放在了内地，也就是所谓的汉地18省。这个汉地18省的定义是清的统治者界定下来的，不是我们只有这块，而是他圈了这块地方，只允许你待在这一块——这是一个圈禁的边界线。然后他在这个范围内打上了密密麻麻的格子，在所有关键路口、要害地方设立军营，方便他随时调动、镇压暴动——就是干这件事的。为什么会有这些？因为大家都恨他啊，“总有刁民想害朕”。整个200年，中国所有百姓没有不恨朝廷的，因为200多年一直都痛恨他，年年都造反，年年琢磨着怎么联动造反。所以他就必须要在所有点上安插他的八旗兵马。不是八旗兵马战斗力下降了，而是他把所有兵马都分散在这些小节点上进行监控。所以打仗的时候就不行，没人可用，不敢调动。假设他有10万可靠的八旗兵马，但10万里头得有9万放在内地分散打格子用。于是你会发现他每次说“我出去打仗”，比如打准噶尔的时候，那个废物样子为什么要打那么久？为什么最后都打炸毛了？为什么打得那么惨？因为他不敢调用兵马，能用的力量非常之少。当时的准噶尔已经很弱了，腹背受敌，内乱横生，已经不强大了，是一个很脆弱的准噶尔，已经走到末路了。就这时候，清廷号称自己要开疆拓土，打仗打得跟废物似的，打了好几十年。为什么啊？他会告诉你，是因为八旗不行了。别闹了，不是八旗不行了，而是八旗没有人可以用。他能够用的那些可靠的有生力量，必须放到内地去监控所有百姓、所有农民。

你看农民就是这样的：谁耽误他过生活，他拿镐头锤谁。嗯，啥？什么时候当过情感主播？进错直播间了。咋了？有人问我情感问题吗？啊，真的吗？啊，我这个人很会讲情感问题的啊，知道吧？我讲情感问题的解决方式就是：还能离咋滴？嗯，来来来，我们走起。啥？怎么找女朋友？别问我啊，多出去溜达溜达，凑合过呗，还能离咋的是吧？哎，有道理啊。呵呵呵呵呵呵呵呵。嗯，啥？准噶尔若明也打不过中二？你知道准噶尔是受了明朝册封的。不，不知道是吧？嗯，准噶尔部是接受了明朝册封的，有印绶的。打个得儿啊？谁打自己的藩属啊？谁打自己的羁縻部落呀？他有什么大病吗？像爷们儿一样的娘们儿讲什么感情？这话说得怎么了？爷们儿没感情吗？我就觉得中国的爷们儿很重感情。这话说得有哪个地方不对？来，你们给我反馈一下吧。啊，他说我像爷们儿，所以不能够聊感情。就算我像个爷们儿怎么了？中国的爷们儿没感情吗？中国的爷们儿多重感情，哪个地方不重视感情了？结婚的时候不养家糊口，不照顾孩子吗？没有时刻想着保护老婆孩子吗？这不叫重感情叫什么？当然了，按照今天的女权主义的说法，只有提上裤子就跑路的才叫做浪漫的重视爱情。这种什么动不动就想着老婆孩子热炕头、要负责任、要养家糊口、要百子千孙的想法，就是物化女性了。女权是女权的思维方式，和正常人不一样，你不能拿她当个人来看，也不能拿她当人的思想方式来思考。在他们的心目中，负责任就是废物，必须是先骗后跑才叫做浪漫的爱情，才是一个有感情的人。这帮玩意儿，他就是想坑女人。简单来讲，女权主义这个东西，在设立理论的那一刻开始，就在坑女人。
