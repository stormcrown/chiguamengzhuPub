# 第261章  小时候的国际主义，还是崇高精神的代名词 #国际主义 #援建 #付出

嗯，各国都在逐步保守化。国际主义是否在未来不会再出现了？我每次讨论问题的时候，都喜欢先说一个定义的问题啊——就是什么叫做国际主义？大家心目当中、想象当中、印象当中的国际主义到底是什么东西？它是什么模样的？什么样子才算作国际主义？它在不同的时候，在不同的人操作的时候，是一种完全不同的操作方式。

那什么才能够叫做国际主义？这只要说我们的船只、我们的飞机、我们的人拿着证照可以到处行走、做生意、旅行、学习，这就叫做国际主义了？不，这算不算？你想去哪个地方，只要合规都可以去，请问这算不算国际主义？如果说啊，那这样子肯定不能算嘛。不管是什么情况下，我们都有可能去别人那里串串门啊，旅个行啊，对方也要赚一下旅游资金的嘛。那你要觉得这样子不能算的话，什么样子算作国际主义呢？

要知道，有一个阶段啊，有一个阶段，中国人所想象当中的国际主义，是所有的阶级兄弟是一家。有一个阶段就是这样的啊。我们想象当中的所谓的国际主义，中国人自己想的，在某一个阶段上面，它叫做所有的第三世界是一家呀，什么无产阶级是一家呀——用这样的方式构建的一套国际主义的理论。但是这个东西在实践当中没起到作用。尽管说我们今天就说啊，那个时候讲它的时候都是正面的讲，但事实上就是没有起到什么作用。因为对方拿好处的时候都说“兄弟，兄弟”，转脸都不认人的。所以这种行为可以叫做国际主义吗？当时我们视之为国际主义——援助啊、帮助啊，视为兄弟啊，非常非常和善的、友好的往来啊，这些事情我们都做过。这是一个极为友善的国际主义的主张，但是在实践当中处处碰壁。接受过我们帮助、援助的这些，逮着任何的机会都要背刺一把。所以为什么到后来大家都不怎么提这件事情了？因为国际主义到底应该什么样子，到底怎么操作，其实它不是那么好直接拿出来宣传、做定义的。

嗯，和平共处、互惠互利啊。所以呢，之前的这套国际主义的东西，其实已经被一部分人玩得让你没有办法玩了。你用了帮助的手段，对方接的时候都接了，但是转脸就给你翻脸。那么，我们能够认可的共识，那个国际主义的行为是什么样子的？到现在啊，我们认可的国际主义的，其实呢，就是类似于白求恩医生那样的行为。他呢，是自己支付了成本，自己负担了一切的风险，为了自己的理想而去为另外的人群做出牺牲与贡献。我们认为他是拥有国际主义精神的。但是你想一想，他的祖国当时跟我们什么关系？有国际主义的精神吗？没有。这是一个个体的行为，这是一个个人的理想主义的行为。这些行为始终都存在人类的社会当中，就是有这样璀璨的星辰，但是他没有能够连成片。因为当他作为一个实体，作为一个群体的时候，他要考虑他们的群体的利益。而国际主义的操作的方式当中，当你界定的不够明确的时候，常常会面临到一个局面：你付出的都是白付出的，甚至会变成回旋镖打到你自己的脑门子上来。所以为什么这件事情到后来就慢慢慢慢的不容易提及，大家都开始要考虑自身的利益呢？因为这确确实实就是一个国际丛林社会当中客观存在的现实。

嗯，谢谢。直面楚歌啊。然后，然后啊，这只是其中的一个阶段。然后这个阶段发展到后来，就出现了另外的一种偷换概念的群体，这也是我们互联网上现在特别多的一个群体啊——说那个什么啊：“你说我老外不好，你是不是不顾及世界人民大团结啊？”大家见过这样的人吗？大家见过这样的人吧。就是那个在互联网上就说：“你反对移民啊，你瞧你不让外国人来中国人待着，你不支持别人爱中国就是中国人，你就不支持世界人民大团结。”有这样的人吧。这是什么？这是慷他人之慨。他举着国际主义的大旗，想要牺牲本土土著的根本利益，去讨好各色的那些占便宜、吃大户的人。就是这个路子呀。他自己出成本吗？任何一个外来的人，都会让整个的社会付出成本与代价——包括认同感的代价，包括价值观差异的代价，包括文化差异的代价。都不谈那些经济贡献上的代价了。这些持续存在、很难弥补的代价，是你必须承受的东西。而这个成本谁来出？都是土著来出的。你来一波，土著出一波成本。但是这些慷他人之慨的呢，就指着大旗子说：“你必须要接受。你不接受的话，你是不是不支持大团结？你是不是没有国际主义精神？”于是国际主义就变成了这些人的一个幌子啊。好词儿是好词儿。然后啊，那就是你必须要这样做，不然的话你就如何如何了。你让他付成本的时候，他撒丫子跑得不知道多快。这就属于是和欧洲的那些接受采访的人是一样的：应不应该接受难民？应该，我们应该有这样的爱心。那让难民住你家去呗？不好意思，我们家没房。就是这样的人呢，慷他人之慨。这样的人老多了。

然后呢，你再用这个概念继续下去，那就是被他们玩死。所以这些年为什么所谓的全世界都向右转？没办法，因为中间想办法占别人便宜的越来越多，已经多到令人发指，多到各个国家无法承受了。所以原本的那一套——什么那一套系统就很难运作下去。尤其当我们的整体的经济啊，事实上经济的发展的速度没有那么飞快的时候，大家就越发难以承受这样的成本。接受外来人口的成本是非常非常高昂的，远不是说我给他安排一个居住，给他点房，给他点吃吃喝喝，远非如此。他的这种社会公共成本的支付，高昂到你想象不到，而且是十分绵长，甚至是数代十数代的代价。如果数量超标了的话，那就是以举国举足的代价去贡献这个东西。这样高昂的成本没有谁家付得起。但是呢，就有人喜欢贴着这个大旗啊，你就说：“唉，就是应该这样做，这样做是对的。你不这样做你就反人类，你就是反这些字。”哎，就喜欢举这个旗子的人特别多，在中国的互联网上也一堆一堆的。那也就是说，你再去整什么国际主义这个词啊，已经被这些人玩坏了。

国际主义曾经是一个理想，理想主义的行为是高尚的情操，但是早就被这些不像样子的狗东西玩坏了，玩成了吃大户啊。有的时候是吃大户，甚至有的时候他们就是在吃绝户，丧尽天良。所以为什么转不动了？就是这样。一个是大家都付不起成本了，再一个大家被背刺的时间可太长了，都受不了，形成了群体直觉。所以我们现在的应对方向是什么？我们现在啊，不说国际主义，我们现在说的是构建人类命运共同体，是不是？什么叫做人类命运共同体？是说我们都在一个球上面：好，大家都能好；坏，大家都倒霉。所以无论如何，你总得做点像样的事情，不然的话啊，我可不一定要怎么制约你。为啥？邻里邻居的，你家失火，我们家也烫啊。所以我们必须要共同前进啊，我必须要促使大家朝向一个能让集体走向良好方向的道路上去行走，去矫正中间的错误行为。这个东西叫做人类命运共同体。
