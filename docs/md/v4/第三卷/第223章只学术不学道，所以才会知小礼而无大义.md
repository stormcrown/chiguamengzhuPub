# 第223章  只学术不学道，所以才会知小礼而无大义

本子（日本）的切腹文化和武士道精神，是不是从咱们这儿传过去的？尤其是那种“不干那么恶心的事儿”的文化——真的，那个切腹的文化是很变态的。我不是说自尽这件事本身，而是说他们切腹的仪式很变态。你意识到这件事了吗？

先秦时期的楚国，也就是我居住的这个地方，当时就非常有这种“就义”的风俗，就是很容易为了道义而死。比如义军战败后，将领直接现场一拉，就再见了。甚至他们的就义过程还搞出了仪式感，要“倾尽而归”，特别讲究这个。但这属于什么？属于告祭祖先：我败了，我得来跟你汇报一下，我这丢人现眼的，干得不行啊，咔嚓一下就结束了。这确实是当年楚国曾经有过的一种风尚，但这跟切腹之间有任何关系吗？毛线关系都没有。为什么？因为你看切腹的流程，它要求的是这样，然后又这样，人根本做不到，所以才要安排一个什么鬼“介错人”卡在后面，给你来一下……这种方式纯粹是在玩变态。别刷屏了，诽谤就诽谤嘛，非得刷屏。

朱舜水（原文本“朱顺水”为笔误）到日本之后，对儒学后来的影响——哎呀，我倒是很希望能给他们点好影响，但是显然没有。日本这个国家特殊也不特殊，也算不得非常独特。朱舜水是一个明末的移民，在发现抗清无望之后，就流落日本，客居异乡。他有一本《朱舜水文集》，里面有他自己的心得，也有他对日本的一些观察，还有一部分各色各样的文章，比较杂，什么都有。我之前在做视频的时候提到过他，因为他在日本居住多年之后，突然发现日本的樱花（SAKURA）和中国人喜欢的樱桃花不一样，不是一回事儿，压根搭不上界。这件事情最早就是朱舜水发现的，所以“樱花”这个东西不是今天才突然开始搞渗透、搞置换的，他们在明王朝的时候就做过这件事。他们利用了我们都是使用汉字符号的这种共通性，给他们的“樱花”起了那样一个名字，使得中国人误以为这两边是一样的。朱舜水也有这个想法，但他在旅居日本多年之后，终于意识到完全不一样，压根搭不上界。

朱舜水自身的文化造诣非常高——能出文集的人，在古典时期文化造诣都很高。那么他有没有对日本的儒学产生影响呢？我个人认为多多少少有点影响，但影响到了什么程度？我觉得挺可悲的一点是：我们这里对他的所有文化影响，日本都只是“学术而不学道”。我不知道大家有没有意识到这个问题——日本是一直都在跟我们学习的，不只是朱舜水向他传播的内容，他们一直在跟我们学，但是只学术不学道。我们文化向外传播的量级很大，但他们只学术不学道，只学那些形象的、表层的东西。所以你会发现，在后来日军侵华期间，最先派出来的一批人里面有很多很有文化的人。这些人作为前哨，有一部分是探子，跟中国人的关系非常紧密。看过那部电影《罗曼蒂克消亡史》吗？里面是不是有一个日本女子，面上看起来跟中国人一样，甚至方言说得贼溜、牌打得贼好？是这样的吧？但是当她洗澡的时候，你发现她穿的是兜裆布——这就是表里不一。这是一个非常典型的特征。他们最早一批派到中国来的探子，文化水平非常好，能够让中国的文人们都觉得很有共鸣、确实不错。这就是一波又一波的文化传播，包括朱舜水这些知识分子带过去的知识。他们学习了里面的东西，这些词他会，这些句子他也会，这些书他也会背，甚至还能写不错的文章、不错的诗。但是做人要有良知、要符合天道伦常，这件事他们做不到。所以他们的学习也好，文化特征也好，都变得越来越“夹生”。包括今天很多人担心他壮大什么的，很难。日本这个国家非常难以壮大，不光是地域狭窄造成的问题，还有这种“缺德”造成的问题。我之前专门聊过“缺德”这件事。德这个玩意儿看上去像负担，因为道德水准高了就容易当老实人吃亏。但是在社会的整体层面上，你有道有德，能够激发更大的创造力，有更好的组织力，能够让后面的人缅怀过去——对自己的来路非常赞同，因为自家的祖上不是什么缺德带冒烟的狗东西。这种朴素的感情可以让这群人在苦难的时候撑下去。所以我们为什么有机会一次又一次在衰败之后重新崛起？这跟我们的道与德是息息相关的。德是什么？德就是德，就是我们人去试图匹配天道的一种方式。所以两个东西之间确实有关联，有这个，你自身的创造能力就会更好，内核的稳定性就会更高。但是日本不学这个东西，所以它的内部非常拧巴：它想干的是缺德事儿，但嘴上背的是道德文章。这种矛盾与冲突会使得它自身处在一个矛盾的聚合体状态下，以至于它的脚步就被这个东西困住了。

血缘上，确实是有点关系——当然我不想拉这个亲戚，日本那边我一点亲戚都不想拉，但是确实排除不了。一方面，这边有一部分人群东渡，历朝历代都有，东渡的人群本身也会留有血种；另一方面，日本那边的人过去一直都很矮，先天禀赋不足非常严重，他们自己也在意这件事，所以一直都在做“度种”（原文本“杜种”为笔误）的事——就是借种。高峰期就在宋朝，他们经常跑这儿来偷男人。所以有没有血缘关系？真排除不了，是真有。虽然不喜欢，但是真有。不过在文化上，他们学了我们的皮毛，最核心的东西学不到，也不想学。所以你会发现，他们在彬彬有礼的表象之下是一颗冷厉的心，而中国人不管面貌看起来是什么样子，心里是热的——这是一个截然相反的路子。中国人并不太讲究“术”，一个人可以文采飞扬，但如果没有也无所谓，有正气才是最重要的。有了正气，再去谈别的都是加分项。但在日本那边不这样，他们看表演。近代他们也借了外国人的种——对呀，阿美大兵驻扎之后，他们官方专门安排过把青年女子送过去，这又是新世代的“接种”。这个国家的基因来历非常混杂，为什么混杂？就是因为他们一直在干这个事儿，老是觉得自己的基因先天有问题，所以一直在尝试自我改变。

如何看待“物哀”文化？丧气玩意儿。这个总结是不是有点过于干脆了？那丧气玩意儿看了之后烦死了。理论上有一定的合理性吗？有，理论上有一定的合理性——就是什么“短暂的美好”“彻底的破坏”，反正咔嚓一下没了，总之就欣赏这个东西。我认为他的这种审美倾向的来历，应该来自于他们比较苦难的过去。他们的日子过得确实一直都不好，除了短暂的时间里面他吃到了战争红利、占了一点便宜、当了一段时间的发达国家之外，其他时候日子苦啊，普遍过得都不好。一个长年累月都过苦日子的人，如果从来没有见过美好，其实也就无所谓了；一直过苦日子但没看过好日子，其实也就那样能过，没啥大毛病——这就是所谓的“原生态”，会有些自我萌发的东西，千奇百怪的啥都有。但是架不住这个地方离咱们往来的历史非常长，这就造成了一个问题：他自身的禀赋很差，长期过苦日子，却见到了极其恢弘的盛世，而且见到了很多次。于是他就开始去追求这种美感——因为看到了，所以他格外追求美感、格调，追求理论上的高屋建瓴这些东西。但是他做不到。那这个时候，什么样的东西既能够满足这种对格调、高尚甚至所谓思想性的追求，又能够非常平价，还能够应和他们自己苦难的人生呢？那就是觉得一切都极短暂，所有美好的东西都必然消亡，消亡才是永恒的。你把这个东西拼到一起去，这个玩意儿是什么？就是“哀”。这种情感就叫做“哀”——他看见美丽的东西，想到的是它要没了，这就叫做哀。在中国人这里，就是“有花堪折直须折”。中国人为什么能想到“有花堪折直须折”？因为折完这一片还有下一片，拆完这座山还有下座山，所以才有“有花堪折直须折”呀。

日本人也喜欢吃饭团，这事儿大家应该都知道。你会发现，在大概几十年前他们的影视专业特别好的时期，拍摄一些较早时代的影片时，会让里面的人去吃一个饭团，通常情况下你会注意到他们啃白米饭团时表现出吃到绝世美味的表情。今天如果有人去推广什么日式便当，也会发现很多时候就是一盒子白米饭，上面放了一个蛋，或放了一颗梅子，有的是一颗梅子一个蛋，有的是放了一片鱼——看上去之后你会觉得这玩意儿是给谁吃的？但是他会觉得特别好吃。为什么？日本不是今天米贵，今天是特定的环境和经济操作手法造成的米贵，但过去他们真的是一直都米贵，异常地米贵。为什么？因为能种的地太少了，产量就不大，哪怕再努力，效果也有限。所以能够吃得上一个饭团，哪怕里面啥也没有，他都觉得太好吃了。就是这样的。历史上日本始终都米贵，近期的特殊暴涨肯定有特殊背景，这是另外的事。他吃了一盒米饭像要爽死过去一样——嗯，有这个基本的原理。我们经常吃米饭、主食吃得比较多的人体会不到。包括之前很多搞营养学的人会说精致碳水不好，你看人家吃鱼吃肉的才健康均衡。我跟你说，人类的营养配比中，碳水的重要性极为核心，因为它是最高效的一种能量，直接就是能量。相当于你是车的话它就是汽油，碳水就是直接的能量，吃了之后脑子就有能量可以转了，因为大脑特别烧能量，是一个高能耗的东西。所以有碳水就有幸福感，有碳水就有饱腹感，这是当中的核心。

再举另一个例子：闹饥荒的时候，也就是缺粮的时候，大家日子一定过得很苦。但大海里的鱼不会消失，鱼还是那样啊——地上确实闹饥荒了，但海里的鱼还在游。这个时候海边的渔民应该日子过得比较好，因为他们可以正常打鱼、吃鱼。然而事实并非如此，因为鱼类的蛋白没有办法良好地供给身体能量。甚至说如果你只吃鱼，你会越来越瘦，因为你要花费能量去消化这种蛋白质。所以在饥荒的时候，渔民过得并不会更好，他打上来了鱼之后必须要去跟农民交换粮食油，尤其喜欢猪油，因为动物类的油脂和精致的碳水就是救命的东西。说完了这个前提条件，你就能够理解一件事：你再想一想日本那边什么环境——被海包住了，所以它的食谱中海产占比非常高，靠海吃海天经地义。但是海产这个东西在没有好的碳水化合物、好的动物性油脂做补充的时候，它是一个耗能型的东西，越吃身上越难受，越吃抗力越不行。所以对于这样的人群来说，优质的碳水化合物就是救命玩意儿。他为什么吃一口米就享受得不行了？就是因为他真的缺那个，对他来说是至关重要的。哈哈，米饭仙人啊，对对对，还有各种仙人，什么仙人都有。那啥，赞还没要够，先给我投数，快点动手点赞！我这么多人呢，总不会每一个都是小黑子拒绝点赞吧。好的，到点了，再见！
