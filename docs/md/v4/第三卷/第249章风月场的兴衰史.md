# 第249章  风月场的兴衰史 #新中国故事 #社会变迁 #女性力量 #冷门历史

风月场在古代有时允许，有时禁止。如何看待它对社会的利与弊？我们彻底禁止那个啥的，好像也没几天吧。啊，我们彻底禁止这事儿了，真的是没有几天哈。因为大部分的时期，他是没有能力禁止的——风月这个行当，据说是人类最古老的职业之一。古老到什么程度？古老到可能就是正常打猎的那个时代，就有这种交易了啊。啊，啥？古代有禁止吗？啊，曾经有一些时候，是在军旅内部有禁止。就是有阶段性的，在特定的区域范围之内，有一定的限制条款和禁止条款。例如在战国时期，秦国在军事系统内部就是严禁这个风月的。诶，是的是的。有管理，有管理。为什么要管？为什么要制约？因为它会严重动摇士气，会让人趋向于精神萎靡。这个事很好理解啊。就是啊，风月类型的这样的生意，会使人趋向于精神萎靡。所以呢，通常情况下，一个国家处在高速的扩张期的时候，它的内部会对这样的场所进行约束，进行制约。为什么不直接消灭？因为消灭不了啊。这种原始型的职业，几乎存在于人类生活的方方面面，里面都塞进了它的信号。所以过去的执行力是没有能力消灭它的，不太可能。它属于食色性也，是一个标准的食色性也，里面和吃什么并列的色。这个玩意儿它确确实实是因为管不了，所以呢，只能通过什么税收啊、许可证啊，然后行业准许啊等等的这种方式进行约束。同时呢，在历朝历代，通常都对于官员这些群体加以格外的约束，不允许他们去。因为这里头，常常存在各种各样的交易，那就变得更复杂了。那啥，食色性也是这样解释的么？啊，差不太多啊。嗯，本能怎么管？唉，对，因为它是我们生活当中原本必不可少的一个环节，是人类最早的职业之一。所以确确实实不是不想管，不是不知道它会有什么影响，而是没有办法管啊。所以只有约束而没有彻底的制止。真正彻底的制止，就是我们建国初的那几十年。这个字，大家知道哈，就只有建国初的那几十年，我们把这个真的制止了啊。因为它的成本其实是非常高昂的。你去管理这个行当，那真的是成本高昂。有，嗯，对，就是那个电影，我刚想说他就是那个电影，叫做《姐姐妹妹站起来》。我们建国初的时候拍的电影，讲的是旧时代的娼妓的生活。有兴趣的小伙伴可以到网上去搜一下。这个电影里面的这个老鸨，就是你在那里面看见的那个凶巴巴的恶婆婆啊。那个老鸨她就是一个受害者，她就是在解放前被迫长期工作，身体遭受重创，精神也遭受重创的一个娼妓。这样的女子，到后来呢，在我们解放初的时候，我们也解放了她们的身体和思想，并且让她们成为了自食其力的劳动者。后来呢，为了能够更好地宣传与教育，产生社会的影响力，也为了留下这样的印记，所以呢，特意拍了这部影片。这个影片非常的真实。这和你们后来在那个各种古装的剧里面去看的那个什么卖艺不卖身、青楼头牌妖妖娇娇的，还可以指手画脚、挑三拣四的，完全不一样。你得真的去看一下：当一个人失去了尊严，失去了人身自由权的时候，他到底面临到的是一个什么样的局面。所以为什么解放初的时候，我们格外格外地在这个事情上面着力？第一呢，这个行业确实对于一个如同朝阳新生的劳动型的国家，是有巨大的冲击力的，因为它其实会把人趋向于堕落，趋向于本能。而我们当时的这种建设，其实就是逆人类的本能而动——你想懒着，但是不行，你必须得干活；你想不学习，你想在街上遛，不行，你必须要学习；你想随心所欲，不行，你必须要学习先进的思想，必须要上进。你不上进不可以，我们拽着你上进啊。这个东西呢，是我们在解放初的时候投入了大量的精力去做的一件事情。其实那个时候特别的难，特别的难。最大的难点在于当时的娼妓不愿意从良。那个时候，我们解放初的时候遇到的最大的障碍就是：大量的娼妓不愿意从良。她们从来没有好好干过活儿，没有干过辛苦活儿，没有吃过这种苦。很多当时的这些女子，她不愿意啊，她觉得自己就用自己的身体去获取收入，挺好的。已经堕落下去了之后，她压根就不想支棱起来，当一个体面的工人。她甚至觉得当工人很可笑。所以在那个时候，我们投入的精力，一个方面是提供教育、提供改造，然后还有治病啊。这里头有病的人真的是很多，而且当时的医疗条件，这个治疗的难度其实也很大。堕落更容易对啥？他们还看不起劳动人民。对，那个时代，那个时代的很多的旧社会走出来的娼妓，确实存在这样的精神上面的堕落。所以我们是花了大力气把他们解救出来的。不光要解救肉体，还要解救精神，最终让她们走入到劳动之中去，然后走入到婚姻之中去，成为我们普通的社会一份子。这件事情在全世界都是独有的，没有哪个地方真的能够做到这么彻底啊，只有我们做到了。这个就是新中国的执行力。嗯，啥？这是对历史周期律的改变的尝试。我们一直在尝试着改变这一切，包括到今天，包括到今天，我们依然在尝试着改变周期律。那啥又回来了？第一，现在依然是一个非法的啊；然后第二呢，只要抓着了，依然是非法的对吧？只是说因为现在的经济模式，管起来的难度更大了啊，所以我们还是定期定点的在那里管。在世界范围内横向比较而言的话，
