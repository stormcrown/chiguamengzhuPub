# 第247章  西史信徒多去种种地，再回来编历史

还记得河殇时代吗？啊，大家记得河殇时代吧。河殇时代呢，将我们称为闭塞的大河文明，然后将西方人的那种海盗行为称为开放的海洋文明。这个事儿其实是很离谱的，因为它在里面解释西方人为什么会成为海洋文明，是因为它的陆上产出不足，所以它非常想要出海去探索未来。然而问题就在于，在你陆上的产出严重不足、生产发展不起来的情况下，你是没有能力出海的。这玩意儿就好像是有人拖着一辆三轮车说：“我这个日子啊，跑货运的日子，就这么一辆破三轮车跑货运，实在是过不下去了。所以应该怎么办呢？我应该开进撒哈拉大沙漠里面去。撒哈拉大沙漠里面，我一定是唯一的那一辆三轮车，那绝对是垄断型生意。”这个就是它的一个分析的逻辑。但问题是，当你的陆上的竞争能力、陆上的生产能力都不行的时候，请问你到底是怎样航海的？你疯了吗？

嗯，我一直强调的一件事，大家要多多的参与劳动生产。能理解这件事吧？咱们要多多的参与劳动生产。为什么要这样说？因为劳动生产这个东西它太重要了。你干过活，你的逻辑要好很多。你知道干活需要出力气，你才知道人类是怎么发展成长起来的。这个玩意儿真是所有人类的刚需啊。我的天，多干活啊，少在书斋里面。哎呀，我的天哪，他们那么厉害啊，啥三轮车？那得登月，撒哈拉算什么也可以。

嗯哼，好的，那个啥，我们这里是一个带货直播间哈。我们来看一下这个《星空帝国：古代星宿揭秘》。星空对《星空帝国》这本书呢，确确实实不错。有兴趣了解一些我们天文学方面知识的小伙伴呢，完全可以入手一本啊。这一套书呢，它里面主要解析的是我们的星宿。为什么在这个时候谈到这本书？因为刚才呢，我们去讲过，说西方人他给自己设定的这些历史先贤设定的，他们才是发现全世界的这样的前哨、先行者，这是他们在近代一直向我们宣传的所谓的科学史、科技发展史。但是问题在于什么？我们要去航海解决的第一件事情就是造船，而建造一个足够航行、足够探险并且能够为人提供安全感的足够大的船只需要什么呢？需要微积分啊。注意了啊，是造一个能够为人提供充足庇护的大船，它需要微积分的理论，没有微积分，只能造中小型的船只，大船做不出来啊。这就是科技可以决定后续的一切。然后同时，怎么能够让人在苍茫的大海之上、在无法辨别方位底下洋流乱窜的情况之下，还保证自己的航行安全，知道自己要去哪儿呢？我们要有天文学的知识，要会观星。观星是所有海上航行初级阶段必须要具备的基础能力。那么，天文学从何而来？天文学不是从你的定理而来，不是从符号而来，不是从装备而来，这些东西都是天文学出现了之后才会有的工具。天文学它必须产生于日积月累的观测和记录以及运算，这是天文学诞生的唯一道路啊。所以明代已经有微积分计算的技术了，嗯，必须如此啊，必须如此。一个船啊，一个船，你的体量已经超过了80米，没有微积分你是做不了的啊，做不出来哈。这个事儿啊，就是微积分影响造船业这件事，是中国科学院的高级工程师分享的资料。你们可以说我就是一个可怜的文科生，我不懂微积分没问题啊，一点毛病没有，但是科学家懂，工程师懂，造船的人懂，没有问题吧。所以大型的船只必须有微积分，没有微积分，大型船只做不出来，做了之后下水就翻，就这样啊。所以你想一想，我们百米的船老早就做出来了啊。我们百米的船，宋代就已经做出来了。明初的时候，我们航海的巨舰，巨舰啊，已经超过了100米，已经超过了150米啊。我们目前因为找到了它的舵杆，它的很多东西都被销毁了，它的档案被销毁的一字不剩，但是我们找到了造船厂，找到了船厂里面遗留的部件，倒推回来，这只船的尺寸是超过了150米的。这样的庞然巨物，在没有微积分的情况下，你根本就做不出来那啥破船，还有3000钉。欧洲长期铁钉造不出来铁锅，19世纪之后才有能力造的。嗯，西方神人说他蹲家就懂了。所以呢，我们的航海第一件事情是你要有一个能够提供庇护的足够大的船只，但是西方人的航海的船只常常只有二三十米，他就说这么小的一个船，他就敢于乘风破浪跑出去，他这叫勇者之心。这玩意儿啥呀？这不就是骑着三轮进撒哈拉吗？那为什么在这种情况下，他就特别有勇者之心呢？因为他真知道前面有什么，他是拿着海图出去的，他知道前头有什么，所以他才敢出去。不然的话，茫茫的大海上，根本就不知道那边是什么的情况下，你就驾着小舢板，你就出去了，疯了吗？西方人啊，他的智力是没有欠缺的，他的智力是正常的，一群智力正常的人，不会驾着小舢板就开往未知之地，那不叫做野性，那叫做疯狂啊，不能那么干的。然后航行的时候，你必须要运用到天文学的知识啊。天文学一定诞生于对于天象的持续不断的观测与计算，这是唯一的道路，没有第二条。这世上绝对不会出现说：我先弄出来天文学定理，然后再诞生天文学。这件事情绝无可能。你没有长期的观测，你做那个定理，为啥呀？你图啥？又是神赐？没那啥，圣经都有人信，史记为什么不行？所以我就说史记是可以信的，尽管他文学性过强，立场过于鲜明，以至于他的史料上面、真实性上面多多少少有点存疑，但是即便是这样，他也是在世界范围内非常可靠的一套信史了。要知道，这世上没有哪个史书能够100%地还原世间真相，做不到。你就算有监控录像，你都做不到100%还原真相，因为它依然有它的拍摄角度的问题，有它的镜面的宽广度的问题。真相的还原是极为困难的，需要多方面的印证。而在唐代以前，第一，我们的史官的历史地位比较高，相对的独立性较强，专业指数呢也比较好，所以呢，先秦时期的史书，那简直了就是字字珠玑。而唐代以前的史书呢，基本可以说它可以大概齐地保真，它可以做到大概齐地保真。所以我不是特别推荐说，你把唐代以前的历史掰开来揉碎了，然后不停地质疑里面的每一个碎片。它对今天的现实世界的影响比较有限。我们今天受到影响最严重的是在唐以后的历史。唐代以后的历史啊，对于我们今天的现实生活影响严重，而且里面的失真的情况也比较严重。史记再怎么有罪，也不至于跟圣经一档啊。对呀，史记只是文学性大于史学性，但是它依然是一本史书嘛。嗯，真相就是把所有的当事人全都找出来，让他回忆也还原不了真相。我们没有办法100%地还原真相的，只能还原一个整体现象啊，只能还原整体的现象。所以我们在整体上面，唐以后我们的历史当中的脉络也是没问题的，这个，这个大家没有什么争议吧？哪怕是在唐以后的历史记载当中，它的注水量大了一些，但是它的整体的脉络没有问题啊，它整体的脉络依然没问题。而西方的那个东西就完全不一样了。啊啊，啥？全是李世民的错？没有说全是他的错呀，跟他有关系，但不全是他的错。实际上在那之后，就开始陆陆续续地出各种各样的状况啊，这不是他一个人的责任。

嗯，啥？圣经本来就是译名啊。你说Bible是吧？我一般情况下呢，就跟它叫作Bible。我为什么要这样叫？主要的原因是他的这种翻译的方式，是一个标准的对它的美化。因为在中国这里，圣是一个很了不起的词汇。我之前专门解释过什么叫做圣，圣是指一个人的能力极为顶尖儿，但是他能够顾及到众生的权益。而在Bible里面，他们的这个造物主的神灵，也许实力确实是顶尖的，都造物了，可是他在良知上面看不出来好，更不要说什么对大众有什么样的伟大的贡献了，哈。所以你把它称之为圣经，过度美化了他们心目中想象的先贤。
