# 第206章 点评【明末】，提了个名字都就给我乱扣帽子

我们再次重申一下：满族人是1951年的时候才出现的。在那之前是什么呢？是满洲群体。满族人是怎么出现的？满族人就是检讨反省过去的八旗处事不端，缺乏正义，充满了反动性。他们承认这一切，并且反思这一切，决定洗心革面，成为共和国的公民。然后我们给他们登记造册，成为了我们的手足同胞——满族人。所以，满族人的特点是什么啊？注意了，**满族人民**的特点是什么？反清啊。

明末的主题是什么？明末的主题是反抗。所以，当你给自己的名字取了一个开头，说自己在讲明末的时候，你只要说是“明末”，你就是在蹭革命者的热度，就是在蹭反抗者的热度。因为明末是属于革命者，是属于反抗者，是属于农民军，是属于流亡朝廷，是属于朱成功，是属于所有这些人的，也是属于李定国的。这个才叫做明末。当他给自己的标题起了一个名字叫做“明末”的时候，它本身就是有含义的。明末是有含义的。

清立国，它定立国号是哪一年？你们自个儿去搜，然后给我弹到**公屏**上面。清立国是哪一年？对，1636。不要动不动1644，1644是崇祯殉国之年啊。立国号“清”这个国号确立的时候，1636。所以你要是说，明末代表了什么？代表了你必须要站在明这一边。而明的国号延续到什么时候？明的这些相关称谓，包括年号的纪年法，在朝鲜半岛那个地方一直延续到了清末。知道吧？在朝鲜半岛上一直延续到了清末。崇祯皇帝的祭祀也一直延续到了清末。咱们就只谈我们自己国境线之内的东西。那么，在全部的国境线之内，明末，如果以最后一个名义上的君王也消亡为止，那就是168几年。那不管你用什么样的标准去界定它，总之，当你要去谈到明末的时候，时间的节点、时间的**标尺**一定是要放到明的这个立场上来的。因为明末的这一段时间，实际上清已经建立了。也就是说，如果你的**立场**坐在清那里，你的这个名字就不要叫“明末”，要叫做“清初”。

我们刚才在聊什么？就谈他的名字。所以，在这个事情上，你只要用了“明末”二字，就意味着你在蹭明末的热度。而明末的热度是什么？明末的热度是反抗入关，是民族独立解放运动。这种革命一直坚持到了什么时候？我们从1644一直坚持到了1911。中国人在这一个抗争的历程中，从1644年开始，一直坚持到了1911年。所以，当你给自己的名字取了一个开头，说自己在讲明末的时候，你只要说是“明末”，你就是在蹭革命者的热度，就是在蹭反抗者的热度。因为明末是属于革命者，是属于反抗者，是属于农民军，是属于流亡朝廷，是属于朱成功，是属于所有这些人的，也是属于李定国的。这个才叫做明末。明末的主题是什么？明末的主题是反抗。

但是如果是同一个时间节点，你说我这个主题就是要把这些人都一锅端了，行不行呢？可以，**立场**挪了吗？你要把这些反抗者一锅端了，在那个时代有没有？当然有，而且他们还颇为强大。但是你就得把自己的名字改成正确的，因为那不叫做“明末”，那叫做“清初”。你叫“清初”，做这些事情，我一点意见都没有。至于会不会举报，那就是另一回事，你也不用问我。但是，做了“清初”，你的内部逻辑就自洽了。可是他为什么不做呢？是不想吗？不是，他是不敢啊。当他做了“清初”的时候，那就是反人类的、反人民的、反革命的，所以他不敢动这个手，他怕他自己的游戏凉了。而且，他想蹭这个革命者的热度，想蹭绝大多数人和**利益**的热度，想让大家因为觉得他身在明末、想要革命，所以给他点个赞，就图着这样的一种推流，图着大家的这种心向往之，然后给自己炒作起来。但是炒完了之后呢，他的选项是他把李定国给灭了，把早就**被**厌弃的崇祯皇帝拉出来重新来一遍。这个时候不要说：“唉，这只是一个游戏而已，好玩就可以了。”游戏只是一个载体，好玩只是让这个载体更加普及的一种具体操作方法。而任何一个载体，它都有承载的内容。这叫啥？这叫文以载道。这个就是标准的文以载道。我不管写了什么东西，我写一个记叙文，讲的就是小明去上学这么一点事情的一个记叙文，讲讲风景，讲讲道路，讲一讲见了人的时候怎么打招呼，就这么稀松平常的东西。你用什么样的语气去讲？你用什么样的形容词去描写**它的**风光？什么是承载的内容？那就叫做道。如果你不能载道，你起码要载德，如果不能载德，你就去**载E**。总而言之，文以载的，总不能是那些缺德带冒烟儿的反人类分子吧。这是一个很根本的问题，这是价值观的问题，也是立场的问题，这是选择谁敌谁我的问题。

嗯，对于现在包括我的直播间在内，很多人地图炮攻击全部满族人，怎么看啊？我很关心，我很欢迎你们都付费提问啊，我高兴得很。第二，我这个直播间里面是禁止地图炮的，什么东北人都如何如何，凡是这样的，我一定会拉黑，带动大家一起举报。为什么？你骂东北人就是在骂我，那我绝对不能忍。所以不存在地图炮的问题，你这个事是一个虚假打靶。那第二个问题，攻击全部的满族人。来，我们再次重申一下：满族人是1951年的时候才出现的。在那之前是什么呢？是满洲群体。满族人是怎么出现的？满族人就是检讨反省过去的八旗处事不端，缺乏正义，充满了反动性。因为承认这一切，并且反思这一切，决定洗心革面，成为共和国的公民。然后我们给他们登记造册，成为了我们的手足同胞——满族人。所以，满族人的特点是什么啊？注意了，**满族人民**的特点是什么？反清，能get到吗？**满族人民**的特点，和我们所有的人的特点是一样的，就是反清。你要是做不到的话，那叫做遗老遗少，就不要拿满族人民当幌子。满族人民和我们能够成为手足同胞的原因，是因为我们共同携手反抗了清王朝、八旗的遗老遗少、各种各样的罪恶。这是个根本的东西。所以，你刚才那个就是虚空地画了一个靶子，然后就说你攻击了满族人，不存在，不存在这件事情。**满族**这个族群以民族的名号设立，它的根本是新中国设立之后，为了推翻三座大山，推翻了那些腐朽的统治者，所以跟我们一起来反对那个腐朽没落的时代、反对那些罪恶行径的人，就成为了满族同胞，是我们团结的对象，也是我们的手足兄弟。就是这样的一个来头。

至于说，你不能**批清**，你一说**清**不好，那我不跟你玩了。你这啥心态？你这是遗老遗少心态，你这是八旗铁蹄心态，这可不是什么好人。你琢磨琢磨自己的立场，符合天道伦常吗？符合人性吗？符合道义吗？符合忠义吗？所以我们认可的美好的品格，你认可吗？这些东西是非常非常根本的玩意儿，这个东西是不能够排斥在外的。

嗯，怎么看电影的那个啥？彭某海战打弯弯的幌子催……呵呵呵呵。文以载道啊，这和我们刚才聊的东西是一样的。就像是你一个游戏，你说我设定一个场景，它叫做“明末”。明末的场景要干什么？明末的场景要抵制入关，明末的场景就是反入关、反八旗。如果说你做了“明末”，然后不反入关，不反**八旗**，你自己就是八旗，那这个东西就不叫做“明末”，叫做“清初”。包括刚才你说的这个电影，这个什么什么海战啊，我就不专门念了，省得人家电影的团队过来找我。包括这个电影，这个电影讲的是什么呀？我的心中在过去相当长的时间里，施某人都是大汉奸，朱成功是大英雄。这是什么？这是视角问题，这是立场问题。朱成功有没有待在弯弯岛上面？有啊。他做了什么呢？他打跑了荷兰殖民者。朱成功打跑了荷兰殖民者是吧？朱成功打跑了荷兰殖民者。然后当时的康熙做了什么呢？康熙做了什么？康熙和荷兰殖民者签了合同，**勾结**一起消灭朱成功。所以你选择哪一个立场？所以你选择哪一个立场？郑成功，嗯，这200多年一直叫他郑成功，其实**他**应该叫做朱成功。因为他姓郑的那会儿他不叫成功啊，姓郑的时候他叫郑森，他有自己的名字。后来被赐姓，姓了朱之后，大家称他为国姓爷嘛，就是因为他被赐姓赐为朱。当他获得这个新的姓的时候，他也获得了一个新的名字叫成功。所以他姓朱的时候就叫做朱成功啊。
