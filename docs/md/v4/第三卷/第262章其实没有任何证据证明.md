# 第262章  其实没有任何证据证明

所谓的“欧美好福利”，其实是建立在压榨亚非拉人民的基础上的。他们本质上就是奴隶主。话说回来，之前是不是有很多人一直在赞赏资本主义？包括我们中国——新中国是一个追求共产主义的社会主义国家，这是我们的身份。即使在我们这里，“资本主义”不是一个非常好的词，但我们也承认它是人类发展中一个较为先进的阶段，以前一直是这样说的。比如当你拿它跟古典帝国比较时，它就会被默认比古典帝国先进，认为资本主义是较为先进的一个——只是我们更先进，社会主义与共产主义比资本主义更先进了。但是注意了，资本主义是什么？资本主义是以货币流通为标准、以货币流通为管理方式的一种管理方法。没有证据证明它真的很优越，也没有证据证明它可以使一个国家发达起来。其实根本就没有这个证据，它只是在理论上被称为更加先进。没有证据证明资本主义能够让大家走向发达。尽管今天的发达国家都说他们是资本主义国家，但他们走向发达国家的那条道路，不是资本主义道路，而是有资本家参与的帝国主义道路。

来来来，今日之列强，他们的发家史——他们常常自称是靠资本主义的先进性而发达起来的，但事实上他们的发展史不是这样的。他们发达的那条道路，是有资本家参与的帝国主义。这是干嘛？也就是说，它基本上属于君主集权——君主集权强力扩张的帝国主义，只是说参与的人群中包含了资本家和资本群体。所以，他们发达的过程跟资本主义没有本质的关系。但是资本主义这种以货币计量的管理方式，非常容易构造一个奴隶制度。我不知道有多少人有这种感知——资本、资本主义和奴隶制度之间的相似性，是非常非常高的。

北征哈皮，嗯，从来只有千百姓开荒发财，没有引流民入内致富的。嗯，对啊，谁能治谁有理。这件事情，我们还是得说古典时期一直想讲的“存人失地，存地失人”的问题——谁最重要？人最重要。古典时期所有的军阀，都讲我治下有多少民，有多少民人，这是他的实力象征。土地是什么？土地是民所能圈拢的财富，土地是财产。它的根本属性是人、是国、是人的组织。我们的古人对这个事看得非常明白。我觉得今天的很多人特别糊涂，特别喜欢按地来算人，这是一个极为糊涂的操作方式。一个国家，它是人的组织，不是土地的拼图。资本主义是非常容易沦陷入奴隶制的，为什么？因为资本讲的是货币利益最大化。而你一旦进入到要求货币利益最大化的状态，它实际上必然面临极度的两极分化——财富的极端两极分化。而极端分化就会造成一部分人贫无立锥之地，变成依附者。在中国的古典时期，一旦出现贫者无立锥之地，就意味着这个王朝在经济上要崩溃了。为什么？因为失去土地的这些农民，失去自己家资产的小工业、小手工业者，他不是死了，他不是消失了，而是变成了大老板们的依附者，变成了雇佣农，变成了帮工。他失去了人身自由，但他没有消失，他还在社会当中存在，只是他的身份变了，变成了他者的附庸——这是一种变相的奴隶。为什么我们历朝历代都要解决这种分配的问题？甚至等到分配问题无法解决时，整个王朝都要被推倒重来？因为陷入这种情况时，其实就进入了奴隶制阶段。人们会失去自由，会变成大地主的附庸，这就是一个变相的奴隶制。

而你想一想，资本社会货币至上，以货币利益至上的标准来进行衡量，它更加容易陷入财富的两极分化。而只要进入了财富的两极分化，那些失产者——我不管你心里向往什么东西，自由主义啊，必须学会什么——没有用。当你变成一个彻底的失产者时，你就已经成为他人的附庸了。你啥也没有的时候，你敢辞工吗？那就是一个强制打工仔。强制打工仔就是奴隶。所以，资本主义和奴隶制之间其实是相通的，并没有背道而驰。所以动不动就说资本主义是多么多么的先进，这件事情老是让我们产生误解。在它的环境下，人们可以如何、不能如何——你彻底失产了之后，不就是帮工吗？就是变成了帮佣啊。

嗯，说说说，我太能说了，嘴不能停是吧？嗯，好好好，不然的话我怎么吃这口饭啊。连续两个小时的直播，就我一个人独角戏，你要怎么办？嗯，哎呀，漏了一个付费，我又漏了吗？啊，好，上播以后，嗯。

土地自由买卖是不是就已经是农业资本主义了，而不是所谓封建主义？商鞅以后土地自由买卖？这真是让我摸不着头脑。商鞅啥时候土地自由买卖了呀？他肯定不是封建制。我给你表示，那肯定不是封建制。但是呢，商鞅啥时候土地自由买卖了呀？我亲，商鞅进行的改制，商鞅开创的这套秦制，最重要的一件事情就是搞这个军功授爵，而且是田爵制。你也在想是吧？那把我搞得真的是很一头懵啊。商鞅改制、改秦制搞的是什么呢？你要知道当年有一个故事：商鞅入秦去找工作，一开始讲的是王道，讲怎么当天子，然后人家打瞌睡——孝公是吧？孝公。啊，反正呢，一开始的时候听得人家打瞌睡，后来就开始讲什么霸道之术，怎么尽快地富裕起来，尽快地组织起来，尽快地强国，尽快地强兵。这个是商鞅当年改制的一个根本目标。所以呢，因为他搞的就是一个短平快的操作，要求尽快地富国强兵，所以他整个的路径上面，就完全不存在土地自由买卖这事。事实上，他把土地自由买卖全否了。为啥？因为他为了能够超速地富国强兵，就做了一件事情：搞这个军功授爵和军功授田两样东西都有。一个是给你升级，比如我让你升了大夫——这是一个爵位，我给你升了一个大夫的爵位。那申请了这个爵位之后，这个爵位要配套一套产业，什么产业呢？就是受田，田土多少亩的地就给你了。我就问你，如果是一个自由买卖的状态，你拿啥赏给人家？能get到这一点吗？就是如果说他是土地自由买卖，他根本就做不了他的军功授田啊，这件事情做不了。所以实际上商鞅为什么后来那么惨？因为他彻底得罪了秦国的所有勋贵——除了唯一的统治者秦孝公本人之外，其他所有的勋贵都被他得罪得底儿掉。别人的财产直接一竿子收公了，田土归公，他才能够把东西发放出去。不然你说军功授爵，一个人投几亩地，你怎么发？你从哪里搞？田土发出来？你一定要从这些有钱的大户手上把田先抢下来，然后发下去。夺人财路如同杀人父母啊。所以为什么商鞅到后来跑也跑不了？这个事跟他自己定的制度有关系。但不管怎么说吧，总之他开罪的这个群体，充分地印证了一件事：他事实上是搞了土地的强制国有，而不是自由交易。秦是一个根本没有土地自由交易的国家。而且他不是一般的不能自由交易，实际上普通的人自由行走的权利都没有。秦的话啊，秦所有的道路沿途的时候，你不得居住一下是吧？你得吃个饭，你得喝个水——客舍青青柳色新，你总得有个客舍吧？秦这玩意都是国有的，知道吗？秦这个东西它是国有的马路道边，所有的小旅馆、国资。所以为什么商鞅到后来开罪了那么多贵族，大家想要收拾他、想跑路的时候跑不掉呢？他定的规矩：全国资，你所有的道路上面，每一个点、每一个酒店要看什么？看介绍信。有没有人在中国经历过介绍信的时代呢？你说晚上要睡个觉，没有介绍信不让你进酒店。他那个时代就是这样的。所以你说，哎呀，那个秦搞什么土地自由买卖，别闹了啊。秦的那个国有制进行得非常深入，但是这个国有制不是公有制。为什么只强调国有而不强调公有？因为他是一个君王天下的状态，他所有的规则都是围绕着君主而运转的。

诶，谢谢，真哈皮。嗯，啥？商鞅变法就是秦分堂口，争周天下社团话事人，谁敢打谁钱多啊？性质差不多啊。呃，我一直说那个什么，就是春秋与战国，其实呢，就是诸侯之间在打一个排位赛，他们在争自己的头名，争夺掌控权。春秋的时候还没有那么大的野心，所以他们喜欢当什么盟主啊——不是我这口啊。所以那个春秋的时候，五霸、霸王啊，霸主其实呢，就是一群人结了盟，然后认我当老大，这种呢，就是属于盟主的性质。春秋的时候，他只是想要当一个结盟军的老大。但是到了战国的时候，大家就已经打破头了，不再是什么结盟的老大——认可了就完事，就可以继续像原来一样你是你的国、我是我的国？那不能够。到了战国的时候，就正经地打出了狗脑子了，是纯纯粹粹的一个彻底的排位赛，赢者通吃。
