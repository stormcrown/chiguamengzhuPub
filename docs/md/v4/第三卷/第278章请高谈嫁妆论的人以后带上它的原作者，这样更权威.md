# 第278章  请高谈嫁妆论的人以后带上它的原作者，这样更权威

欢迎来到直播间。  
啥？我剽窃别人作品还在这读呢？小云，做人啊！我剽窃搁哪呀？能告诉我原作者是谁吗？不要空口讲啊！你要不然把那个什么原作者弄出来。嗯？什么？没有什么满洲东北嫁妆说啊？这个嫁妆论啊，嫁妆论还蛮有名的。但是呢，给你讲嫁妆论的人，只会讲你团结你或者不团结，你还要不要D了之类的啊？你是不是不支持大一统？就喜欢扣帽子。但是呢，就是嫁妆论这个东西的原创是有作者的啊。刚才谁在那里喊着什么那个原创啊？原创啊！对，嫁妆论这玩意是有原创作者的啊。  
啥啊？意大利说我抄袭念稿啊？你说他是原作者？不会吧，他一天到晚都在骂街，我抄不了啊！骂街的东西我抄不了。你要想说谁，你就把原创作者叫出来，你不要跟我搞什么那个转译啊，转译不了啊。就这个嫁妆论啊，嫁妆论这个东西它是有原创作者的，这个事是真存在的啊。就是你们所有在那里讲嫁妆论的人，尊重一下原创作者的身份好不好？这个人的身份是什么呢？啊，是一个我不能念名字的人啊。他姓汪，字照明。嗯，年轻的时候当过革命者，后来当了大汉奸啊。就是这个汪某人，他的夫人说的，明白了吧？啊，能get到我在说谁了吧？我不能提他的名字，我把他的名字全篇念一遍之后，我的直播间就卡没了啊。就这个人，汪某人的夫人提出了嫁妆论这个学说。  
所以呢，你们下一次啊，再提出说啊，那个啥是啥啥啥的嫁妆，东三省是啥啥啥的嫁妆，一喊这话的时候，你一定要把人家的原作者说出来啊。汪某人的夫人说了：东三省是某某人的嫁妆。你下次把这个话完完整整地说，好吧！在全网络到处公告，你只管去说，是不？还有名人背书呢，多刚啊，杠杠的，老硬气，老硬气了啊！去啊，“引刀成一块，不负少年头”的那个，对，青年的时候当过革命者。他要是那个时候嘎了，我们今天课文里面妥妥地就得背这首诗。这首诗写在那时代来讲还不错啊。所以呢，如果他当年真的嘎在牢里头了，这个诗今天就是大家必考必背的内容。但是现在呢，在互联网上你念一下，背后都要再接着蛐蛐两句，嘲讽两句。为什么啊？因为，嗯，是吧啊，就因为啥呢？啊，因为后来就咋地了啊。就这么件事儿。  
简单来讲，就是汪某人在结婚之前还是一个有志青年，结婚之后呢，就变成了一个反革命分子。然后呢，就是他结婚的这个对象，这位大小姐啊，她说的东三省是嫁妆。下次呢，你们一定要把这个东西完完整整地说出来啊。啊，不要动不动冲上来说：那啥啥啥是嫁妆啊！一定要说啊，这玩意儿，这话名人名言有背书的。史书上面说的明白的是什么什么人啊？他说的东三省是谁谁的嫁妆。你们只管说，你把全称都弄上去之后，我看还有谁有脸说东三省是什么什么嫁妆啊！  
那啥，任你再怎么诋毁，乾隆能力就是强。我啥时候说他能力不强了呀？我啥时候说过乾隆能力不强了呀？没有啊，没有这事儿啊。我从来没有说过他能力不强，我只说他写诗水平不行。这事儿公认的吧？写字水平不行，公认的吧？画画水平不行，公认的吧？我可没有说他不强啊。你要知道，当坏人是要有本事的，当顶级的坏人是要很强的。他只是坏，我从来没有说过他笨啊。他怎么会不强呢？他很强啊，不强怎么能做那么多糟心烂肺的事呢？  
嗯？什么？为什么网上风评说“雍正是清唯一正常的皇帝”？怎么评价呢？啊，他本来就是清唯一按照皇帝的工作方式去工作的一个人。不是说他好，而是说他在整个清朝的二百多年当中，他是唯一一个用皇帝的思维方式去处理天下事件的人。是这样的一种评价指标。因为从顺治入关开始啊，顺治时间比较短，不谈，时间比较长的就是康熙。那么康熙啊，他呢，其实不是一个君王的思路。君王的思路是要考虑这个国家怎么能够持续安稳，考虑国本是哪里，然后考虑我们的政策导向是什么方向，这是君王要做的事情。但是康熙没有。康熙是以一副征服者的姿态，在这个地方玩儿耀武扬威，他一辈子干的事都是这个。当然了，确实因为他在登基之后啊，清王朝还没有立稳根基，所以相当长的时间都在处理各种各样的内乱、叛乱等等的事。所以呢，打仗的时间，他在位的时候啊，打仗的时间很长。他是一个立国的过程，整体来说他就是一种征服者心态，而没有做任何关于君王需要搞清楚“我的国是什么样的一个国、我应该如何治理、应该如何构建秩序”这么一件事啊。在康熙的时候基本没做。  
做这件事情的人是雍正。雍正做了这件事情，做得好吗？不好。为什么？因为他最擅长的东西是敛财，是收税，是制定规则。看上去好看，但实际上非常残暴的收税方式，这些规矩都是他定的。但是呢，尽管这个规矩定得很烂，但是他是在立规矩。尽管说他的《大义觉迷录》是一个狡辩，但是他确实在试图构造一个法理。雍正在努力地缔造一个他存在很合理的法理系统，这是历代的君王都必须要做的事，这是君王的职责啊。所以为什么说雍正算是一个君王呢？因为他确实做了这个事。做得好吗？他要是做得好的话，他儿子就不会把这本书销毁。因为清从立国开始，雍正的那套自圆其说的东西就不好使，骗不了人，没有办法自圆其说，还不如不说。所以到他儿子那儿，把他爹说的话全部都咽回去了，又重新收回来销毁，大批量地销毁。就好像他爹以前这段儿，这本书就没写过一样。
