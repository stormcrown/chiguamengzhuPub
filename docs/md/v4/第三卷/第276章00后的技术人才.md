# 第276章  00后的技术人才

诶，好，欢迎来到直播间啊。为什么他们早期吸纳移民能强大？你说阿美是吧？阿美为什么早期吸纳移民能够强大？这个原因呢，主要源自于阿美本身的本体太弱鸡了啊。很多人可能不太相信这个啊，阿美自身的本体太弱鸡了，所以它吸纳了移民之后会变强。大家可以自己去看的啊，没关系，你可以尽管自己去查证啊。他们早期的时候，科学技术的发展团队当中，大量海量的核心的创作人、工作人员、科学技术工作者都是华人、华裔，从中国这里弄过去的啊。早期的时候大量的华人、华裔，现在这些技术类的岗位当中，大量的印裔是吧？毫无疑问，一直都是这样的呀。为什么会是这个样子？他自己的本体太弱鸡了。你要知道，要想让一个国家的内部的人才不断涌现，你是要投入教育成本的。你要投入教育的成本，从你的最开始的基础教育开始，就在不断地抓紧。然后你才能够在这些经受了扎实的基础教育的人群当中，去筛选出足够的产业工人。产业工人是需要有一定的技术教育和基本的文化水平的，你必须要有，没有的话做不好活儿，你必须要有这个东西。那么这些基础的教育培养产业工人，然后呢，在这些基础教育当中的优等生，接受到更高级的教育，就可以成为工程师。然后在这些接受了高等级教育的人群当中，出现那凤毛麟角的、极其优秀的、超出于其他人存在的，这些就会成为科学家。而一切的这种筛选的过程，都需要你在一开始投入的时候，投入到海量的、普及的、优质的基础教育。而这个东西很贵啊，这个东西是很贵的啊。

但是阿美那边，他的立国就在逐利。他一开始去的人就是过来打劫的人。最开始那艘船上，他们说自己无欲无求，过来开拓新大陆，实际上是干啥？抢劫。他们从一开始去的人就是抢劫，就是逐利。整个国家在立国的时候，包括他贩奴也是为了什么？为了投入更少，获利更多，都是为了逐利。当他就是为了这种目的的时候，只需要扒拉算盘算账的时候，只需要计算投入产出比的时候，投入基础教育不划算得很，是吗？对于阿美的这种经营方式来说，他投入基础教育是不划算的。那什么对他来说最划算呢？买呀，买买买对他来说是最划算的。所以他当年缺乏劳动力的时候用了什么？买奴隶、买奴工。然后他缺乏技术人员，缺乏科学家，缺乏大脑的时候，他买别人家的高精尖的、优秀的青年。这是什么？这就是搞移民。为什么搞移民的时候他会发展？为什么他搞移民会快速地进步？因为他省下了这些正常的、甚至作为刚需的投入，但是却直接去截胡了别人家的成熟的、受教育的高等级人才。

但是为什么到现在就不行了呢？要知道，当你是用买买买的方式的时候，请问你付出什么成本，能够买到别人家的高精尖人才？你自己算一算账。当你是高精尖人才的时候，对方给什么价你肯去？第一呢，是收入，毫无疑问的嘛。第一件事情是收入。但是收入这个事儿，他给你的是钞票，而这个钞票最终你需要考虑的是他的购买力啊。也就是说，它本质上必须要满足你某一种程度的生活，你才会受到一些吸引。那现在问题来了，中国才是世界上最大的工业国，甚至说我们才是唯一的全产业链的工业国，按照严格定义当中的最大的、了不起的、完整的工业国，可以自己独立生产的工业国，全世界只有中国这一个。这意味着什么？这意味着我们这里的生活质量、购买能力，会在这种情况下，只要我们的金融不被他人掐喉咙，我们就会水涨船高，不断提升。大家现在有没有拿到国补？拿到国补了？没有拿到国补的人自己去看，现在各个平台上面电器的价格震撼啊，你去看这些东西，震撼。这是什么？这就是购买力的提高。你的薪水还是那些钱，但是物价降下来了，工业产品的价格打下来了，这个时候就是购买力提高。但是呢，你去阿美那里看一看啊，薪资还是那个薪资，然而物价起来了。为什么？因为很多东西他生产不出来，生产不出来他就只好抬价啊，购买力下降。他的钞票对你的诱惑力就会下降啊。这是一块，这还是很小的一块。为什么？因为真正的高级人才不是很受这个东西的诱惑啊。这些呢，是应对一般劳动者的。对于真正的顶级人才，这些生存层面的、购物层面的东西不是很诱惑他。甚至对他来说，这个事不是很重要，对方必然会保证这一切的。那这个时候什么东西能够诱惑他？高大上的理想魅力、吸引力。注意到了吗？阿美靠着从外部引进人才、提升自身竞争力的那些年，做的顺风顺水的那些年，也是他的好莱坞飞黄腾达的那几年。好莱坞是什么？广告片啊。亲，好莱坞就是他的广告片。他不断地用这些广告片向全世界宣传他的价值观，宣传他的生活方式，宣传他的理想与梦。动不动就战斗，外星人动不动就保卫全球，动不动就为全人类而做出什么什么什么东西。尽管阿美这些事情一概没做，但是她在电影电视里面演得老清楚了，很多人都信以为真。他还开放这种大学，发放奖学金，然后让那些人真的相信有这么一茬事儿啊。当你相信自己可以在这个地方践行这样拯救人类、拯救未来的伟大理想的时候，那些最绝顶顶尖儿的人才就会受到他的吸引和感召。

嗯，登月就是在加拿大拍的。呵呵，这个说法本身是存在的啊，也提出了一些证据。但是目前来说呢，我们在整个世界范围内还没有说这个事已经实锤了。哎，没怎么聊这个事啊，所以就先这么混着吧，也不是很要紧的一件事情，随他去吧。反正呢，他那里的这个登月还是问题挺大的啊，他那里的登月的问题挺大。但是啥时候真正完全地在公开情况下就揭底，还是到时候再说的事儿，不着急啊。哎呀，我就说了，你们这些人不要刷屏，当黑子造谣，绅士有没有点素质？我这里对黑子的要求是很高的。
