# 第277章  关于输赢这件事，有清的理解好像有点微妙

以下是将原文整理后的易读段落。根据您的要求，我没有删减任何内容，仅修正了错误的词汇和表达，同时调整了部分语序以提升可读性。修改之处在末尾备注中说明。

“你看人家的劫匪，直接说‘我要求抢了！你那么肥，活该被我抢！你那么肥又弱，活该被我抢！’——这个是什么呢？这个就是社达啊！这个就是丛林法则：弱肉强食，赢者通吃。那么为什么我们会秉承那样一种看上去‘你好虚伪，你非得要打一个好名头，非得要是正义的，否则就不如何如何’的这样的态度呢？唉，为什么非得要这样呢？因为，当你践行了内部有道德的社会秩序和分配方式，当你遵守了道德、天道、伦常，从善的方式去构建一整套社会秩序时，它可以做到一件事情——保护弱者。而社会达尔文主义是一个保护强者的秩序。对，它能够用竞争的方式选出最强者，然后赢者通吃。你看上去这才对嘛，最后出来的是强的。但是，如你如我一样的普通人，要么为奴隶，要么活不下来。为什么？因为你不是全方位最强的那一个。一旦弱肉强食，先没的就是咱们这样的。

嗯，啥？中俄、中法、清都有输赢啊？嗯，清贵也不少救国的。你对外战的归纳是否片面了啊？你是想说《尼布楚条约》是吗？啊，中俄来啊，你自己说啊，你自己说打赢了是吧？啊，打赢了这个条约叫做《尼布楚条约》。《尼布楚条约》签订之前，尼布楚这个地方是中国的土地。然后这个公平的、而且是战胜了之后所签订的公平的条约——之后，尼布楚不再属于中国了。棒棒的啊！你清的、公平的、战胜的条约就是这个样子！然后那个中法战胜啊，来来来，你再说一下：请问胜了之后，你从法国人手上拿了什么好处吗？还是把我们对于越南的声明权拱手相让的同时，割让了什么样的利益呢？有兴趣的啊，在这里头有了啊。棒棒的啊，在这里头有了啊！来来来，你说啊，赢了我赢了。来，我就问你：你赢了之后，到底是从人家那个地方把别人的土地拿过来了，把别人的殖民地圈过来了，把别人势力范围圈过来了，还是把自家原有的势力范围（至少说是宣称范围）拱手相让了啊？这就叫做赢了？对，赢了这件事情你理解得特别棒啊！对，赢了这件事情的理解你从来都是特别棒的，我很赞赏。尤其是喜欢他是付费提问，棒棒的。嗯，我特别喜欢。要是所有的小清粉都给我付费提问多好啊，我得开心成啥样？呵呵呵呵呵呵。

哦，还有那个什么啊，清的贵族的救国啊——你清贵族自己总结的：“保中国则不保大清，保大清则不保中国”。你清贵族自己总结的。所以说清有没有救国呢？肯定有。救的到底是啥呢？是吧，你自己算，你自己算你们家自己的贵族定下来的规矩，是吧？这话又不是我编的，这话不是我定的啊，是你清的贵族自己定的。所以他到底啊，他到底有没有救呢？救了，救的是谁呢？这玩意儿老有意思了。

嗯，啥？你讨论的是军备啊？好的好的好的啊。中俄指的是伊犁啊？这样啊，你指的是这个呀！你莫不是指的汉臣顶着压力——顶着满清反对的压力，硬生生扛住了压力，然后自费跑出去打仗的那回吧？啊，你对那个啥啊……对对，胜败这件事情理解得很微妙，然后对打仗的主体这件事情理解得也很微妙啊！哎呀，厉害厉害厉害厉害。嗯，那个啥，左宗棠是自己自费自筹军费，他还借了债的。人家军费都筹齐了，朝廷上面说‘行吧，去吧，打吧’，同意了。在那之前天天都反对。然后你说啊，天哪，这个是啥？伟大的贡献啊！伟大的贡献啊！对对对，这是伟大的贡献，他们一直都说这个事儿呢，是他的伟大贡献。嗯，然后我下血本了，你这个是吧，这个是吧？哎呀，对对对啊，下血本了，我从来没有那么大方过，真的，我就没这么大方过啊。我还专门付费自己买的。呵呵呵。那啥，今天没开小黄车，完了我一个带货主播。谢谢你们提醒啊，我今天都忘了带货了。我一个带货主播，然后不带货是不是过分了？

嗯，刚才那个是谁？有一个什么什么打赏来着，这是好啊。清代，清代所有的朝臣里面呢，但凡是出一两个好人都会很拧巴。你一定在他的身上能够看到这样的矛盾点。那个所有的清代的像样的朝臣、像样的好人，一定会在他身上看见矛盾点，而且非常的矛盾，是那种会让你很痛苦的矛盾。为什么？因为清的统治者——清的最高统治者是极为反动的，他所制定的方略也是极为反动的，从头至尾生于不义、死于耻辱。你问我一千遍也是这个结论啊。然后这里面为这个朝廷供职的人就会变得非常的加深。因为这里头有一个问题啊，因为这里头有一个问题哈。出现的状况是什么呢？简单来讲：这个国家，虽然说他在统治阶层上面极为反动，但是国家是什么构成的？国家是人民构成的。你放到哪里说破大天去，国家也是人民构成的。而这个国家当时的人民是什么人？他们不反动，他们一直在革命，他们一直在生产，也一直在抗争。但是呢，他们一直在被压迫，也在被愚弄，是这个样子的吧。那么当广大的人民面临到这样的境况的时候，革命又始终不成功的时候，就一定会有人去思考一种解决方案。而这种解决方案实际上呢，在我们的历史进程当中出现了很多次。是什么呢？就是打从李唐王朝衰败之后，中国的北方就始终都沦陷于各种各样的胡人政权统治之下。那个时代呢，是你方唱罢我登场。从那个时期开始，中国社会的——你说他是学者阶层，或者是知识阶层，或者是儒生阶层都可以——就是这些士人们，他们呢就开始去不断地扭转自己的思维。他本身没有办法推翻这样的一种管制阶层、统治阶层，他推翻不了。那么如何能够在有限的框架之内实现自己要去造福乡里等等的这样的想法呢？甚至他们在书里面也会去学到说要如何的兴天下，他学到了，但是看着眼前的场景又不知道该怎么办的时候，于是他扭转了想法。最后总结出来的东西就是：他还是要出事，他还是要当官，然后呢，努力地当一个好官，能够努力地做一件好事。这是其中的那些真正有志向的人。当然了，还有一批呢，其实就是谁来给谁当狗啊，这种比较好理解，奴才嘛，哪都有。但是这里面一定会有一部分实际上是内心非常曲折，但是呢，啊，有自己目标、有自己理想的人。这些人历朝历代都存在。而这种解决的方案，从唐灭亡之后就一直都有。尤其是元经过了将近百年的统治之后啊，士人群体在胡人统治者的治下去进行社会管理，已经变成了一个默认可以存在、默认很正当的行为。所以到清的时候，他们在这个事上面没有非常明确的一个心理压力。但是呢，麻烦在于什么呢？麻烦在于清的统治者太反动了，以至于呢，他只要当官，一定会参与到一些反动的行为当中去。所以呢，你要想在这个地方维持住自己的位置，坏事一点不干不可能，多多少少都要奉送一点良心出去。但是同时如果他还保有自己的内部的、内心的一些要求、一些东西呢，他就会努力地在自己所遇见、有机会的时候去做点好事情。所以你才会发现啊，有清一朝有没有名臣？有没有正臣？有没有像样的人？有没有做出贡献的个体？有的。但他们身上都会表现出非常非常拧巴、非常非常曲折的这种心理动向。因为你要做事就不能够在社会上面没有身份，你要去造福乡里就不能够去说——我就单纯地去当一个反贼头目是吧？这样子呢，如果成了固然好，改天换地；如果不成，那就是带着乡里一起啊，就沦丧了。而这些事情，有清的两百多年，一直有人持续不断地在做啊，一直一直都有人在做，也一直都有人在失败。所以一定会有相当多的人在努力地践行自己的理想的同时，又去当他人鹰犬。这些行为普遍性地同时存在啊。这不是某一个个体的问题，甚至很多的时候，我觉得不是他个人品性上面的问题，而是清这个王朝——他立身不正，他的反动属性所带来的必然的结果。一个极端反动的团体的内部，想要做一点好事，甚至在逮到机会的时候想要做点大好事，最后表现的形式就会像是左宗棠这样子啊。

我们来带一波南明史。嗯嗯，南明史呢，就是从崇祯帝殉国之后啊开始，一直到南明小朝廷彻底败亡为止啊。这一段呢，叫做南明史。他们指的呢，还是明王朝的旗帜。但是呢，很明显的一件事情：人人都知道他是节节败退，最后没了啊，几十年的功夫就消散了。但是这一段历史呢，是中国历史上面非常重要的段落。这个段落就是我们到底应该是什么样子？我们到底为何而成功？又为何而失败？尤其是在为何而失败这件事情上面，南明史是最值得借鉴的一件事。看这个书啊，注意了，看这个书心情会不好，头上的筋会砰砰跳啊，这个是很常规的感觉。如果你觉得一看之后眼睛都发红啊、头有点疼啊，请放心，这是正常的症状啊。那个，这个书就是这样的。但是呢，它还是很重要，因为它是能够让我们在抗战当中最终取得成功很重要的一个前车之鉴。如果没有他在前面打样的话，当时我们国内的抗战很有可能没有办法联合那么多的群体。这个就是所谓的“勿复南明旧事”啊。因为他是一个很典型的分不清主次矛盾、分不清敌我关系、然后不断地内部的纠缠、不断地内斗、不断地撕扯，最终呢，导致了败北啊。他是一个难分敌我而造成的一个问题啊。

可以理解为以社达的眼光看历史就会认同清，是这样吗？唐皇现在提问的水平越来越高了啊。简单来讲，如果说你看待历史的角度是“赢了的那个就是最棒的”，那就是所谓的社达。社达是什么？社达其实就是丛林主义。我把它用更难听一点的话描述出来的时候呢，你就会更好地理解它，否则的话你还会觉得它还蛮时髦的。因为我们在很多很多的文章当中都会看到社达。社达呢，其实就是谁赢谁有理啊，赢者通吃的一个丛林竞争的法则。或者可以说社达主义就是禽兽主义。也就是说，当你以禽兽主义的态度去看待清的时候，你会很赞赏它的。它以小而博大，以弱而胜强，以落后而至先进。这在人类的历史上面是难得一见的盛景。而且呢，它用分化隔离的方式，让这样的反动的统治持续了两百多年，才在多方力量的交错作用之下陷于崩溃。这是非常非常难以做到的。所以我从来不说清的统治者们蠢，我只说他坏啊。因此呢，如果说一个人觉得啊：“他毕竟在那个事上面统治了多少多少年，那他毕竟疆域如何如何如何，所以他就是正义的。”那么这种思维的方式呢，就是一个典型的社会达尔文主义的竞争方式，也是丛林法则的竞争方式，也是禽兽的竞争方式。所以他就是一个禽兽世界观。

嗯，好，欢迎来到直播间啊。嗯，好，然后，然后，然后，嗯，我看一下，嗯。所以那个我们刚才在带货带的那个《南明史》啊，就显得尤为可贵了。你得知道，当你遵从了一套丛林法则、遵从了一套禽兽世界观的时候，你会遭遇什么。这件事情很重要。不是说“谁赢了谁算”嘛？赢了的那个就是最狠的。这种禽兽世界观，你一定要秉持的话，可不可以？你——你自己的选择，我很尊重别人的选择的。你一定要用这种世界观去看待事情，我不反对，我甚至一点儿都不介意。但是呢，你得先回答一个问题：你是谁？当你接受禽兽世界观的时候，你先要琢磨的一件事情是：你是谁？你是禽兽里面赢了的那个还是输了的那个？你是那个不计代价也要赢，还是那个代价？想清楚这个问题啊，想清楚这个问题之后，你再来选择到底是要天道伦常，还是要禽兽世界观。

嗯，啥？社达是指社会达尔文主义啊，就是弱肉强食的那一套。我其实之前好像是聊过这个话题的啊，就是有关于到底这种弱肉强食的竞争——弱肉强食的竞争一定会让这个社会产生真正的强者，但是呢，中国这里自古以来的更迭会特别地讲究师出有名、道义为先啊。你一定要有一个好看的名目，不管你之后做了什么，反正你的名目必须是清楚的。甚至山匪打劫都要打个旗子叫做“替天行道”。看上去中国人特别地虚伪。而你看人家的劫匪，直接说“我要求抢了！你那么肥，活该被我抢！你那么肥又弱，活该被我抢！”这个是什么呢？这个就是这个就是社达啊！这个就是丛林法则：弱肉强食、赢者通吃。那么为什么我们会秉承那样一种看上去“你好虚伪，你非得要打一个好名头，非得要是正义的，否则就不如何如何”的这样的态度呢？唉，为什么非得要这样？为什么非得如此啊？因为呢，当你践行了内部有道德的社会秩序和分配的方式，当你遵守了道德、遵守了天道、遵守了伦常，从善的方式去构建一整套社会秩序的时候，它可以做到一件事情——保护弱者。而社会达尔文主义是一个保护强者的秩序啊。嗯，还是那句话：你不能只在赢的时候去爱你的民族。为了赢而赢，必然会认贼作父，嗯，认贼作父，冠履倒置。对啊，就我们在讲说为什么道德、伦常、天道这些言辞很重要，因为用这样的方式去构建的社会体系，它是可以保护弱者的。而弱肉强食的体系是拿来保护强者的。对，它能够用竞争的方式选出最强者，然后贏者通吃。你看上去这才对嘛，最后出来的是强的。但是，一个群体作为群体能够长时间地扩大化地始终存在，是要保护弱者的。为什么？因为所有的群体都是金字塔形状的，下面大上面小。如果你只保护那个尖尖，就意味着放弃了基石。嗯嗯，好，欢迎大家来到直播间啊。那如果说你采用社达的管理方式，有没有人能活下来？一定有人能活下来，毫无疑问的。但是如你如我一样的普通人，要么为奴隶，要么活不下来。为什么？因为你不是全方位最强的那一个。一旦弱肉强食，先没的就是咱们这样的。而当一个社会当中啊——而当一个社会当中普普通通如你如我这样的角色都没有了的情况下，或者我们变成了一个行尸走肉一样的奴隶——如果变成了这个样子，这个群体就死掉了，这个群体就会失去创造力、失去生长力、失去传播力。这个东西是必然会发生的啊。啊，因为放弃了基石啊。这个是什么？嗯，“清朝笼络名贵，维护贵；无根无源，下层奴役；所以清贵族以下人人皆是地狱；故步明清，比沙俄崩”——我虽然看不明白你在说什么啊，我没有看明白你在说什么，因为你省掉了太多的词。但是呢，呃，我不建议把明清放到一起。这是两个背道而驰的时代，它们所有的风貌都是不同的，只是时间上面是连续的。它们之间只有时间上面是连续的，而所有的风貌全都迥然相反。所以在这种情况下，我不建议把这个两个字放到一起，然后去跟别人去做对比。比不了啊，没有办法比。
