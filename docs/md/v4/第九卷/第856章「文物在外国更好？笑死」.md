# 第856章 「文物在外国更好？笑死」 

昨天去了一个西吹，有人说我们的……嗯，谢谢磊二哥。我们的文物放在别人家的博物馆里更好，怎么看这个牛吧？打从上世纪末的时候，从上世纪末的时候，有很多有官科背景的公知学者，就已经反复强调过了：说我们的文物幸亏被老外拿走了，不然的话在我们这儿都啥也没剩。

文物是什么价值？文物是干什么的？这件事儿先要理清楚。文物是历史记忆的物证——谁会在乎历史记忆，谁才会在乎文物，这是很明确的事情。而这些欧洲的各个博物馆，它们的来历是什么？为什么这些博物馆那么热衷于收集各种各样的文物呢？这些地方啊，这些博物馆的来历就是各种各样的官方文物贩子——这些人最早的来源就是文物贩子。他们和各地的所谓探险家、所谓的考古工作者勾连起来（指的是他们当年派出来的那些西式的考古工作者）。虽然他们自己并没有文物，也没有古墓，但是呢，他们特别热衷于到中国这里来考古，并且给我们带来了考古学。这些西式的考古学家，连同文物贩子，他们勾连在一起，缔造了西方的各大博物馆。

这些地方是干什么的？它们的作用就是拿来给文物增加附加值的——能够被大型博物馆收藏，说明这个文物值钱、有价值，然后呢，在各种拍卖上面就可以不停地转手。那你就会发现，西方有很多所谓的文物不是他自己本地的，但是却始终都收藏着；还有很多的民间藏家，干什么呢？他们是把这个玩意儿当做市场经营来操作——谁手上的文物多，是可以拿来直接变现的。既然有这样的作用，所以呢，他们就会不断地哄抬文物的价值，进行炒作。一旦发现哪些地方的文物似乎价值降低，它就要开始编故事进行宣传。

大家有没有看过一个电影？好像是电影吧，叫什么《达芬奇密码》，看过没有？看过的来扣个一啊。有一个电影，干什么呢？这种电影里面就在那儿讲一个什么圣杯的故事是吧？圣杯，你管它圣杯是干啥，反正呢，就是先说有这么一个圣杯，然后说有个学者研究这个圣杯，然后来溯源它的故事，咔咔咔地开始搞一个探秘。这种电影是干什么？这种电影就是来哄抬文物价值用的广告片。包括什么那个跟盗墓相关的那些电影作品啊，也是这样的作用啊；还包括什么法老归来之类的——除了收割这种电影票的钞票之外，它还同时存在着文物宣发广告的这样的作用。你看了之后就觉得那个地方神秘又辽远，然后呢，他那里的文物就有一定的价值，就开始逐渐地炒作——就是让那个地方的文物有路人缘，人人都知道那个文物有价值，然后就有人接盘，这个事情就可以循环转动，成为一门生意。

所以，国际的文物市场上面，有各种各样的贩子在不停不停地交易。这种交易的过程，它最怕的就是文物贬值。因此，为什么西方的历史当中有那么多的假文物？你一戳破它的话，一堆人都跑出来，乌泱乌泱地开始撕咬、开始咬人。有一部分中国国内的人，是戳破了他的信仰；但是在西方世界里面啊，你戳破了他也要铺天盖地地咬人呢。为什么要咬？因为当你戳破了，说某一个东西它不是真文物、是造假的，你又找出了证据，这就意味着这买卖干不成了——直接就让它贬值，就会让这个生意做不成。断人财路，如同杀人父母啊。所以呢，这个过程，它就是一个不断编造故事的过程。

那么，西方的这些文物的收藏，包括博物馆这样的地方，当它的目的都是经济利益、都是为了保值、升值的时候，对它来讲，这个文物是没有历史意义，也没有情感意义，没有这些连接的——只有一件事情跟它有关系：就是这种类型的文物，如果在这个地方有，那我的身价就会提高，我的后台就会看起来比较硬，我做生意的时候就会比较硬气。这个是它的核心价值。所以这些博物馆普遍性地都非常不尊重中国文物。大家去过博物馆的都会发现，去博物馆待着老难受了——为什么呢？稍微正经一点的博物馆，稍微像点样子的文物，全部都在玻璃箱子里面闷着呢；除了玻璃箱子外面，有的时候还有个警戒带，你得隔老远看啊，一个一个地把脖子抻得老长。而且一会儿禁止拍摄，一会儿进这个灯，一会儿进那个灯，规矩一大堆。但是你去到那些国外的博物馆里的时候，啥事儿没有，敞开来的。为什么人家不在乎啊？人家不在乎这个事情。

所以，中国的这个文物被劫掠出去之后，其实在海外的部分已经大面积地毁损了。之所以很多的公知依然能够各种各样的吹嘘，是因为他们当年抢的、偷的、盗的东西太多，以至于毁损了一批，又换一批新的上来展示——实际上已经出现了大面积的毁损。包括最著名的《女史箴图》，就是毁在他们手上的。文物上面出现一些破损时间的痕迹很正常，但是呢，修复是需要技术的。他们偷了中国人的文物，但是甚至在修复的时候，都不惜得找中国人来修复，找了一个脚盆人去修复，直接把那个画就给裁切了——这个画变成了不可逆的损伤啊。

所以什么东西放在洋人手上更好？怎么就更好了？放到一个根本就不爱它的人贩子手上，这孩子会长得更好？你怎么琢磨这事儿的啊？说这个孩子在家里的时候会被爹妈打，所以放到人贩子手上的时候更好，人贩子会给他糖吃。所谓的“西方保管文物会更好”，就是这套逻辑。啥？人贩子真的会给糖？嗯，对啊。嗯嗯，也有一些人看多了西方的阿拉伯冰箱……嗯，对，因为故事听多了，所以就会出现各种各样的幻想。这就像是觉得某些地方很发达啊，某些群体很先进，于是认为他们有更大的善意。啊，这个叫什么来着，是叫“降临派”吗？啊，你们看看这些小说看得比我好啊。这种思路是降临派吗？就是觉得对方看上去很发达、很先进，所以他比我们文明，应该听他的——这种思路啊，引入他者的这种思路，是不是叫降临派啊？其实呢，就是有一部分的中国人，现在心里头还怀着这种幻想：看到洋人的时候，就觉得“哎，他们很先进，是文化职员（？）”，他们有各种小故事啊，自幼都听过、耳熟能详的。听那个故事的时候，觉得他们那里的人太了不起了，太厉害了啊，所以呢，就应该听他的，听他们说的就对了。当这个玩意儿形成一个习惯模式的时候啊，你想问题就是这样的时候，你麻烦就大了——因为欠缺理性，欠缺唯物，没有办法按照客观的真相进行分析啊。那个到时候的偏差，失之毫厘，谬以千里。
