# 第818章 消费降级的中产困局 月入15W的网红月入3千的包租婆，谁才是无产者

好的，我已经根据您的要求，将这段口语化的文本整理为更易读的段落，并修正了其中的错误词汇，同时确保了内容的完整性和原意不发生改变。以下是整理后的内容：

**全面脱贫以后还有无产吗？**

看看，我们全面脱贫以后，还有无产阶级吗？啊，无产阶级指的不是穷人，我们要搞清楚。无产阶级不等于是穷人。当大家脑海中，把无产阶级与穷人直接绑定的时候，你可能就会陷入到一个困境之中去，觉得自己现在薪水涨高了，然后就认为自己是有产者了，接着就陷入到了一种精神内耗之中。这就是现在的中产困局啊。

记得前阵子经济条件有些持平，很多行业出现了裁员的时候，发现了没有？就是那个时期，有一批中产破产潮。为什么无产者——明确自己是无产者的人——没出事儿，有产者也没出事儿，就是这些中产者出事儿了呢？是因为他们错误地定位了自己的身份。

什么是无产？什么是有产？这个无产与有产的“产”，指的不是money，不是薪水，它指的是生产资料。意思是你有没有可以掌控在自己手里的生产资料？有没有哪一条销售线路是完全属于你的？有没有哪一种生产工具是完全属于你的？有没有哪一套生产系统是完全属于你的？这个东西叫做生产资料。有了生产资料，你才叫做有产者。

有产者当中，当然也有小有产者，例如有一部分的个体户，他拥有自己的生产资料，但是抗风险能力不强，这些是小有产者。还有大的资产、资产阶级、大的资本家。这些就是掌握了特定生产资料的人。他们即使生病了，即使休息了，依然有收入直接滚入到他们的账户中来，因为这是由生产资料带来的收入。

但是，有一部分高薪打工仔，误以为自己的薪资是自己的产业，于是他就产生了一个问题：他错误地定位了自己的身家，从而在消费上陷入了一种困局。所谓的中产破产三件套，他真的破产了吗？其实没有。他只是从原来的年薪百万，变成了一个年薪十几万而已。在普通人那里，他的这个收入并不算差。但他怎么就破产了呢？是因为他之前按照年薪百万的方式进行消费，以至于到了年薪十万的时候，他过不下去了。他错误地衡量了自己的生产资料，也错误地衡量了自己的社会阶级，于是就造成了后续的很多错位。

《西虹市首富》那部电影大家看过是吧？王多余为什么他想要把钱花空的愿望没能成真？因为他的钞票到了一定量级的时候，资产累积到一定量级的时候，他变成了一个资本家。于是，无论他做或者不做什么，无论他做的是正确的事情还是错误的事情，只要不是天翻地覆地把自己给卖没了，那么他的资产就会不断增值。所以他就会出现越花越有的局面。这种情况下，那个中产破产三件套的东西对他来说是无所谓的，消费是收入的一部分。但是，对于那些错误认知自己的所谓中产阶层而言，他们把薪水定义为资产，把房产定义为资产，最后就出现了问题。

所以，无产阶级与是否脱贫没有任何关系。无产阶级不代表他已经进入了贫困。而我们的脱贫攻坚是干什么？是让我们的无产者拥有正常生活、幸福生活的基本权利，这是一个社会再分配的表现。不代表说脱离了贫困，于是他就不再是无产阶级了——不存在这个概念。

**个体经营者属于无产者吗？**

关于个体经营者：在自己做个体经营者是否属于无产者这个问题上，在我们从前，都是将它定义为小有产者的。但是，有一部分的个体经营，他没有掌握生产资料，他只是变相地打工，那么这种就是无产阶级。但是，有一部分的个体生产者，他拥有自己独有的生产资料、生产技术、生产渠道，这种就属于小有产者。我们要按照他有没有生产资料来划分他的阶层，而不是按他财富账户的多寡来划分，同时也并不是按他的职业来进行的划分。这跟钱多钱少没关系。就像哪怕说现在那个——我开播之前刚刚看了一个热搜，就是有一个自媒体大号，搞那个什么反诈的（原来是有单位的，后来做反诈自媒体红了之后，一头懵地辞职，专职做自媒体了）。我开播之前正好看到发的一条热搜，讲他现在的年收入下降到31万了，那他可能压力就会比较大一点。那在这上面，我看到评论区里有很多人在那里感叹：“哎呀，网红赚钱可真是容易。”那么请问，互联网的网红，他算有产者还是无产者？来，大家判断一下。大家觉得互联网网红这样的职业，他是有产者还是无产者？

嗯，对呀对呀。你想一想，什么叫做有产者？如果说我什么都不做，下个月还有这么多钱进入账户，那有可能是一个有产者。但是你想，我每一天在我的直播间开头，上面给大家标称的东西是什么？——“上班了”。我下播的时候会跟你说什么？——“下班了”。为什么要这样？因为，在这样的一个——无论你是主播也好，还是做视频也好——你所有的这些工作都是你在做的时候有收入，不做的时候就没有收入。也就是说，生产资料不在我们的手上，平台不在我们的手上，流量通道也不在我们的手上。我们所拥有的是什么？是自己的工作。一个人使用和控制了自己的工作，然后去获取收益，实际上就是一个无产者。

所以，为什么这个反诈账号看上去现在有点灰头土脸的？实际上31万的收入在一般人当中不低，实实在在并不低。但是为什么会显得有点灰头土脸的呢？因为，如果一个年收入31万的状态，和他原来那种铁饭碗的身份来比较的话，他划不来，因此就会显得灰头土脸的。但他是有产者吗？不是。他是无产者。为什么？因为他的这个之前所预期的百万年收入可以实现，但也随时会消失。这种不受他本人掌控，只要劳动的能力——甚至是风口的机会一过，收入就会消失、就会打折。这种依赖的就是劳动置换收入，所以依然是无产者。

**大网红是有产者吗？**

能让别人用数字人的方式运作的，这种就已经是企业了，这不是大网红，这是企业。企业主当然是有产者，因为他拥有一部分的通道，尽管只是一部分，但也是拥有了通道。企业主是有产者，但是网红是无产者，因为网红这个身份实际上是一种互联网打工者的职业身份。所以大家不要用钞票的厚薄去衡量一个人的阶层，尤其是在今天的这个收入很可能波动非常大的时代里面。很多时候，一个人收入不菲，但他却是一个无产者；有些人看上去收入不高，但却是一个有产者。这种情况都是同时会存在的。咱们不能用荷包的厚度来衡量人有产或无产，而要去看他的收入到底来自于生产资料还是劳动。

**主播、工人、小企业主怎么算？**

我对我是无产者。我是标准的无产者，正常的上班，正常的下班，正常的加班，正常的打工。这个直播工具算不算资产？不，不算资产。工人上班也得有把起子的，也得有个榔头的；农民种地也要有个锄头的。所以这种生产工具不算。

无产者赚到了钱，开了公司算有产吗？你的企业是不是在通过生产资料来获取利益？我们知道现在注册公司特别容易，无论你注册的是什么样子的公司，你的企业是否可以在你休息的情况下继续为你获利？实际上，我见过很多非常小的企业，严格来讲，它的老板就是一个独立打工仔。那你这种的话，你说他是一个有产者就很不合适了。但是，如果你确确实实，这家企业是可以运转起来的，它自己有一些技术、一些通道可以去做事，那你就可以属于是一个有产者了。这个性质还是有不一样的。因为我这边见过不少这种——你说他是企业主，他确实是企业主，有证照注册，收入其实也还可以，但是吧，实际上就是在打工。那能一样吗？

你做摄影工作室的就是这样。是的，我们很多做摄影的、做后期的、做特效的，很多是艺人工作室、艺人公司，他有收入，技术好的人年收入也不错，但实际上他就是做工。这种就是用劳动来换取收入。他的很多工具设备，当然了价格也不低，但这些是它的生产工具，而不是生产资料。生产资料这个东西，如果说你这个生产资料它本身就能产生价值，那才是生产资料；而如果要你来用它然后才能获取收入，它就是一个生产工具。

侦察兵提醒了那个乐视老板——资方都没有了，但是最后剩下了一部分的知识产权，然后这个知识产权就足以给剩下的员工进行分配。那这种知识产权就属于他们的生产资料。
