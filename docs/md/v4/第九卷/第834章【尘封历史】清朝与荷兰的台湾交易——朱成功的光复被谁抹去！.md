# 第834章 【尘封历史】清朝与荷兰的台湾交易 ——朱成功的光复被谁抹去！ 

他大清画过台湾的地图吗？他大清画过台湾地图的话也没有用。为什么？因为他大清从一开始立国的时候，就努力地把台湾送出去啊。虽然今天很多清粉说，要是没有郑成功，你都没有台湾。哎，注意了，台湾当初在法理上面开始走向失控，就是因为，就是因为清朝啊，就是清朝整出来的。为什么？因为当初明郑政权，明郑是标准的中国政权啊，即便他是流亡小政权，也是标准的中国政权。然后呢，当时的清朝为了能够打击明郑政权，于是做了一件事情：就是他跟当时的荷兰殖民者签订了一个合约。来啊，有证据第几页的什么的，我也没细看。反正不管他肯定是签了这个东西的，你们可以到这三大本里面去找啊。所以呢，当时注意一下，当时清王朝为了能够消灭对他形成巨大的正统攻击性的明郑政权，于是呢，勾连了荷兰的殖民者，跟荷兰人签了一个条约，声称一件事情就是请求荷兰人协助，把明郑给打了。打下来了明郑政权之后就要把这个台湾送给荷兰人啊。那么当初郑成功注意了啊，一直在说那个什么啊，一直在说郑成功，他郑成功的时候已经被赐姓为朱，所以呢，他的正规称谓应该叫做朱成功啊，他叫朱成功的时候是国姓爷，对吧，已经受封了。来，那么朱成功这个人，当初他在台湾本土上面作战的时候，一方面是对清军作战，维护明郑政权，但是另一方面，他在做的事情就是抵抗荷兰殖民者。抵抗荷兰殖民者这件事情，明郑政权成功了，但是抵抗反清的这件事情最终没有成功啊，在大汉奸的进攻之下，最后功败垂成。那这件事情就很有意思了：清要把台湾卖给荷兰人，但是明郑政权不同意，于是明郑政权从荷兰人手上打下了台湾。那么我就问你，请问这个清朝对台湾的法理，到底是打哪儿来的？为什么他送给别人的地算他的法理？为什么明郑打下来的地儿不算是明郑的啊？来，能不能带几件汉服的货？真的好好看，被种草了。自个儿找地儿找裁缝做去啊，我这个东西就是粉丝手做的。
