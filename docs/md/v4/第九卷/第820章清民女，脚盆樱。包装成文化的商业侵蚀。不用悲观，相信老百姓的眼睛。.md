# 第820章 清民女，脚盆樱。包装成文化的商业侵蚀。 不用悲观，相信老百姓的眼睛。

我们的汉服研究与清装渗透

在专业的汉服研究中，姐姐跟我讲过，现在清装的渗透范围越来越广，深度也越来越深。如何直接解决这个问题呢？如果你冲上去直接理论，往往会伤及无辜——为什么？因为有些人穿着清装时，根本没想过这么多，他们只是单纯地把一件自认为传统的衣裳穿上身。你告诉他们这个传统是假的，要给他们上一堂长课，他们不一定想听；同时他们会辩称：“我祖母就是这样穿的，那肯定是传统。”当他们心里有了这样的底牌，就不愿听你的观点。那么，我们该如何破局呢？这就不得不提我们的祖先，他们在这个问题上颇有先见之明。我们的先见之明，就体现在我们设计出来的东西是真的适合自己，也真的好看。用真实的美与强大去产生震慑感，这是我们目前能做到的最佳方式。

至于清装，尤其是清民女装这套东西，我其实很希望越来越多的人能去还原一下清民女装的使用场景：那种暗无天日的宅门，那种女性地位极端低下的时代，那种把后面的小辫梳到额头上的奇怪发型，那种被掰断的脚，那种大烟袋。把这些东西全部配套并反复呈现之后，你自然就形成了美丑的认知，就会知道中国女子在穿着清民装的那个时代，究竟过着怎样的生活。

汉服有没有可能被旗袍马褂夺舍？目前确实有人有这个目的，也有这个想法，并且正在这样做，资金体量非常庞大，还形成了一种所谓的“政治正确”。你只要说清代那种屈辱的服装不是汉服，他们就会说你破坏团结，或者破坏地方的民俗文化，找了很多借口。但事实上，清代凡是受到八旗控制的地区，全部都被变更了自己的服装体系。所以，清代的这套服装体系与我们的传统服装是完全背道而驰的。而现在，他们正试图把清装改个名字，直接称为“汉服”。那么，汉服有没有可能被取代呢？对方确实在这样做，但最终结局如何，取决于人民百姓的眼睛是否雪亮。为什么这么说？因为我们的传统华夏衣冠，其最根本的作用是显示我们的美与威仪——既美且飒，既美且强，这正是传统服饰设计的根本。如何形成威慑感，如何形成美感，这里面包含着科学原理。什么样子真的好看，什么样子只是堆砌、堆叠而并不好看，其实有客观性。美并不完全主观，它包含了大量客观因素。

所以，我想推荐一本书，但忘了书名。这本书讲的是传统服饰，不是专门讲汉服的，而是讲传统服饰与纹样演变。里面有一套专门的纹样图册。我为什么说这本书能让你产生直观印象？因为它不是专门讲汉服的书，而是讲传统服饰的书，其中也会把清代的这套东西放进去。你会发现汉服和清装被拼在同一本书里。我一直推荐这本书，为什么推荐？因为把它们放在一起大有好处：你一页页翻看历代衣裳，看到清代时，会突然意识到审美不一样了，设计原则也不同了。我在直播间里给你讲一千遍一万遍，你可能觉得我只是嘴上说说，理解不了很正常，因为这东西本身就是客观的衣裳。但在这本书里，无论是纹样叙述还是款式叙述，你都会顺着时间轴一起看到清代，这时你会发现审美的路线大致这样：有起有伏，有刚有柔，到了清代却“咔”一下急转直下。你会在直观上明白什么叫做美、威仪、宏大和影响力。这套书在这方面起了很大作用。我推荐它，就是因为它有价值——尽管它不是纯粹的汉服书籍，包含了清装，但正因为包含了清装，所以更好。它能让你在看了一整篇汉服后，突然翻到清装，瞬间觉得“我勒个去”——之前也有花样，为什么到这里突然变得这么丑？所以，虽然现在有大量势力正试图把清装换个名字渗透进汉服，但我不认为他们会成功。因为只要把真实的清装和真实的汉服放在一起，无论哪个时代，你立刻就会察觉两者格格不入，完全是两套不同的审美标准，有如天渊之别。这种直观印象一旦达成，就是所谓的“人民群众的眼睛是雪亮的”。谁好看谁难看，谁像主人家，谁像贼子，是有区别的。它会形成一种直观印象：哪种款式更小家子气，哪种款式更暴发户，你的眼睛会给你答案。

中山装是不是新中国自己的服装？你觉得中山先生是新中国的人吗？中山装是中山先生设计的，所以才叫中山装。它是民国时期设计的服装，而且是特意设计的，但不能把它作为传统服装使用。它是特定时代的革命服装、特定时代的礼服，但不是传统服装。不然，我今天穿西装，一百年后有人说“你看以前那些传媒业人士都穿西装”，那西装岂不是成了我们的传统服饰？这显然离谱。中山装在设计之初，实际上是以西装为底板的三创（不是二创）。因为西装经过日本的二创，形成了日本那边动漫、电视剧里常见的学生制服，这是西装的日本本土化。中山先生又借鉴了日本那套制服，再次进行本土化改造，成了中山装。所以它不是新中国的衣裳，而是民国时期的衣裳。

有人说“中国有土生樱花”，该怎么反驳？中国确实有土生的樱桃花。我们得先弄明白汉字的命名原理。大部分物种不会起非常复杂的名字，比如“樱花”在古文中也会出现，但不要望文生义——看到“樱花”二字就以为和日本的SAKURA是一样的。用了同一个字，不代表是同一种东西。为什么呢？因为日本所学习的中文本来就是一个变种；同时，“樱”这个字是为果实命名的——它来自“璎珞”，即小宝石。“樱桃”这种果实很漂亮，一串串像小宝石，于是人们把这种果实叫做“樱”，生出这种果实的就叫“樱花”，长果实的树叫“樱树”。所以，中国有没有樱花？有，但指的是樱桃花。我们本土包括云南地区的一些土生的所谓樱花品种，其官方学名叫“高盆樱桃”。樱桃最重要的区分在于以果实为目的——我们不是在欣赏花，而是在期待果实。但日本的观赏樱（SAKURA）行为逻辑完全不同，它是为了赏花而特意培育的，用的是他们本土的樱进行培育，目的是让这些花非常璀璨同时又非常丧。这不是巧合，而是目标——他们要让花呼啦一下就消失，这是“死亡之美”。日本就喜欢这种很丧的美感，在培育SAKURA时已经融入了自己的文化与精神。尽管全世界最早的樱的原始植株可能出现在喜马拉雅山东麓，而喜马拉雅山东麓又确实在今天共和国的土地上，但两者之间没有关系。因为最初版的樱植株出现并扩散全球时还没有中国，它作为自然品种原本就分布在欧亚大陆各个角落。日本人用他们本土的樱培育出了完全符合自己审美、没有可食用果实、纯观赏型的SAKURA。所以，它和我国本土的樱无论是血缘、文化、审美还是果实上，都毫无瓜葛。

日本人培育的樱花（SAKURA）有没有果实？大部分品种没有可食用果实，只有少数品种有但不能吃。实际上，SAKURA应该有樱桃类的果实，但日本培育的是纯观赏性花朵，所以多数品种根本不结果。而我国本土的樱花（樱桃花）是有果子的，果子就是樱桃。我国本土欣赏的是樱桃花，但樱桃花其实不如桃花、梨花、杏花好看，它花瓣薄、皱，并不美观。然而樱桃的果实又漂亮又好吃，所以古人喜欢樱桃花是因为爱它的果实——没有果实就和我们没关系。樱桃的花本来就不如桃花、杏花、梨花好看，但胜在果子漂亮，所以大家热衷于种它。苹果花都比它好看，海棠花更不用说。我国本土的小樱桃非常好吃但皮太薄，没法运到外地。

关于武汉的樱花（SAKURA），是日本安排过来的吗？武汉最早一波樱花是当年日军种植的，后来大部分被砍了。现在武汉很重要的赏樱地点——武大里面的樱花，其中现存的老树是中日友好蜜月期时日本赠送的礼品，明确是从日本本岛运来的小树苗，在那边种植，名义上代表友好。当然实质是什么目的，今天再看友不友好？反正当时的名义是为了两国交好。最早一波是殖民时代种植的，充满殖民主义目标；后来邦交蜜月期又送进一批，代表文化交互；后来关系紧张时，他们又开始宣传说这东西不是日本的，是中国自古就有，你若排斥SAKURA就是排斥中国。为什么？因为如果把真话说出来，让所有人都知道这些树是日本来的，那树就保不住了，他们还怎么搞商业渗透？没有了这些树，他们怎么卖SAKURA的笔、冰淇淋、橡皮、笔记本、包包等等？这一整套商业文化侵蚀，会让你和他们的丧文化建立某种精神上微弱的联系。这种联系平时察觉不到，也不干扰生活，但某一天遇到一个契机，它就“嘭”地爆发出来——这就是文化渗透对人产生的影响。
