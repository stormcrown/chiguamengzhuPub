# 第853章 99_的人误解了明朝的疆域——没有边界线≠不管辖？

明代东北领土的边界在哪里？明代没有边界线。明代没有边界线，这个一定要反复强调。清以前的任何时代都没有边界线。所以你不能说明代在东北的边界在哪里，但他掌控的区域和声明的区域到哪里呢？今日的东三省以及外东北，直到库页岛，全部都在领土疆域范围之内，而且是有驻军的那种。

奴儿干都司是一个临时机构，它负责管理人员和依附部落，是一个招募机构，所以是临时的，很快后面就取消了。取消的原因是因为这个地方已经管好了，在当地已经设立了卫所，安排了屯垦的军民。到了这个程度，就没有奴儿干都司了。那边有屯堡的，我们在东北和外东北都是有军屯的。今天大家去吹的那个什么屯堡女装在贵州，对吧？去年崔护说我们在外东北有这种军屯。在外东北的土地上，有军屯，有驻军，有屯垦，他们是双重功能，既是军事驻点，同时又是农民住家。用这样的方式才算是这个地儿我们控制了的。同时，会在这个地方上面，对依附的部落予以授权，就是发放印绶。

当年努尔哈赤造反的时候，最开始他喊起七大恨之后，第一波冲的是谁呢？冲的就是东三省上面的各个部落。而这些部落是什么呢？这些部落早就已经领了大明王朝的印绶，原本都属于明王朝治下的土司，他们是领了印章、拿了授权的土司。所以努尔哈赤起兵的时候，是要先把这些人揍一遍的，挨着个儿，用极其残暴、极其血腥的方式吞并他们的部落，最后将他们的印绶以及相应的文件全都付之一炬。这就像事业编一样，差不多就是个事业编的意思。所以今天有一些人，既在那里幻想自己是叶赫部，又特别崇拜爱新觉罗氏，这就理解不了了。当年叶赫部被冲成了啥样？都快冲凉了好吗？结果今天崇拜起来了，列为同道了，厉害。

他是有编制的。关于这种类型的编制，就是这种授权。例如努尔哈赤拿的是龙虎将军这样的授权。这种羁縻授官的范围是很广的，广到了什么程度呢？广到了那个岛国上面。岛国上面我们也授了权，那边您特别讨厌的那个岛，那岛国上面我们也是授了权的，也是发了印章的，有身份的。都有我们的卫所相应的这些遗址，包括专属的各种印章的这些凭据。这些文物都在哪儿发掘出来的呢？一个是在今天大毛的领土上面，就是由我们外东北地区的相关卫所留下来的各种各样的印章、授权书等等这些东西——在大毛的土地上有。然后往西，到那边斯坦国那边，就是哈密卫。哈密卫的印章、文物、证据在隔壁斯坦国那边。

所以实际上你说明代的疆域，哎，你说，你们不能吹牛。历史上面对疆域吹牛吹得最大的是谁呢？清。清的那个你们认为很对的、很帅的那张秋叶海棠图，吹牛最大的就是那张秋叶海棠图，因为那张图跟清就没有什么关系。为什么这样说？因为那张图严格来讲，是在洋人规定的国际秩序框架下面，别人决定这个地方成为战略缓冲区，然后他就在这个地儿画了一条线，跟他实控不实控毛线关系都没有。他这个号称实控的地方都有别人的驻军。你就看那个什么清，号称实控藏区。号称实控的标准是啥？实控说那个驻军是吧？除了早期的时候有几百个人上去晃荡了一阵子，然后就回来了，除此之外再没驻军。而在那个上面有谁呢？隔壁啊，隔壁的那个国家啊。这阵子B站上有个博主，标题叫做“大清远胜大明”。我知道啊，我这边儿的黑粉头目当中就有他的榜一。对，就是你说的这个up主，他已经安排他的榜一在我这个地方当黑粉潜伏很久了，连粉丝灯牌都很高的那种。藏区，隔着一个山那边那个泥，那家那个国家，他是在藏区驻军了的，收税了的，管理程度要比清还要高。所以你说，你看我这个地方都失控了，我画了图了，他就靠一张图了。小尼，那不止正经话，正经话那个隔壁的那些你今天看起来小得有点过分的，不光是能够驻军到他，大清宣称是自己实控的领土上面还可以签不平等条约呢。他们还签不平等条约呢。千万不要想的他们怎么还有这个优势、那个优势，这些统统都不要想。他们还签不平等条约，你能理解吗？就跟隔壁的那个小尼，那个名字我就不全念了，为啥不全念？我也不知道为啥不全念，好多的国家名他都不让我全念啊。这不平等条约里头，你们看见了的正经有东西。隔壁的小尼有，那个跨大洲、跨大洋的那个，叫什么？刚果。有啊。所以你就说他签不平等条约，签得虽远必赔，但是他失控了，领土疆域老大了。

蒙古旗制？驻藏大臣？金瓶掣签？怎么怎么说呢？有什么怎么说的呀？从元开始对于藏区就有这种类型的控制，从元开始就有这种控制。当然，元在那个宗教方面是被藏区的一些僧侣控制的，那另外一回事。但是明王朝是可以掌握那边的转世的，明王朝已经可以掌握高原上面的转世了。事实上，当时高原上面的很多僧侣时不时的就被请下山，来到王爷的府上面去讲经，这是常规项目。我就问你，这样子随时随地派一个卫队就把那个上面的高等级僧侣给拽下来，这件事算不算失控？算不算影响力？不算是吧？好，那收税算不算？明王朝是可以收税的哟。收税算不算？也不算是吧？那顺治皇帝说的话你算不算啊？顺治在所谓开国的那个时候，给高原上面的统治者写了一封信，说：你们这儿呢，自古以来就受大明的统治，现在呢，我接盘了以后你也要听我的。这是你们家顺治说的啊，这是你家顺治说的。

嗯，啥啊？对对对。他还在那里说那个什么蒙古旗制，那不就是不管的意思吗？你以为蒙古旗制是啥？就是不管。因为事实上草原上的群体是部落联盟的后期阶段，因为他那个地方地广人稀，没有办法实行直接的这种集权式的管理，所以一直都是一种各种各样的部落分开，然后彼此之间用合盟的方式聚在一起。所以清的这种所谓的治理方式，那就是没治理，改了个名字，但事实上和人家原有的管理方式是一模一样的。所以满清在草原上的管理方式就是歃血为盟，拜个把子，然后用人家原有的方式进行管理。他真正算是管理的东西，就是用阴谋的手段在草原上面传教，然后摧毁了草原上的百姓的生计，就这个东西是他真干的，其他的东西都不是他做的，他就没管过。事实上他不但不管，他还放纵草原上面的奴隶主南下捕奴。这个捕奴捕的是谁？你们自己心里有数吧？这个不用我专门说吧？汉地十八省是满清的圈禁的线路。什么叫圈禁？这个圈禁就是，汉人不许出去，不许脱离汉地十八省的范围，你不许往外走，不许去实边。你一旦到了边疆，那个地儿真变成了你的了，他就要肝儿颤。所以他是允许汉人走出汉地十八省的范围的。但是我们不能出去，可是其他地方的奴隶主可以进入到汉地干什么？捕奴，可以抓人走，想做驱口。倒也不一定，那还真不一定。因为他们过来捕奴的时候，有相当一部分人家也不笨，你正经说起来就是他不笨。他过来捕奴的时候，也会优先的去抓一些有知识的、有技术的，他会优先的去抓那些有技术、有知识的人。对于这样的人的话，也不能够当做真正的驱口来驱使。所以倒也不至于说抓的这个奴隶就是直接当驱口了。他们要对工匠，他们对工匠的话还是蛮看重的，因为那个地方缺技术、缺资源、缺知识积累，工匠过去了还是有一点点待遇的。但是捕奴这件事情的本身，并不在于对方对他的待遇如何，而是你只要用了捕奴，就没有把这里的人视为平等的人类。

驱口是奴隶吗？驱口理论上面是奴隶，但是驱口的这种操作方式更像是面对牲口，它和我们直接认知的这种奴隶还不太一样，更加严峻一点。正经说起来就是更加严峻一点哈。蒙元的驱口有多惨？应该有科普。那个驱口相对于我们可以直接理解的这个奴隶来说，他的处境更加的艰难。中国人传统历史上面也是有奴隶制的，也是存在有奴隶的，尽管没有整体的奴隶制，但是我们确实是有奴隶的。而对奴隶是什么样的态度呢？就是第一你不能逃跑，第二要沉重的劳动的负担，第三就是给予的福利待遇很差。这就是奴隶，奴隶、奴婢。但是它同时还有专项的保护的策略，就是哪怕说你只把它当作可以买卖的商品，也要有相应的保护的措施，包括说禁止擅自殴打和杖杀奴婢，这些规则从汉代以后历代都有。所以实际上就是我们这里有没有这种奴籍？奴隶的群体存在，这样的群体是有的，但是只是在他的社会这个分工上面极度的歧视，并没有不把他当人。记得，我们本土的这个奴隶制，包括大家直觉上所认知的奴隶制，没有说不把奴隶当人。但是驱口这种，那就不是奴隶那么简单了，和我们脑子里的奴隶不一样，他们更像是人形的牲口，根本就不拿他们当人来看待，这个标准是完全不一样的。攻城的时候驱口填河。
