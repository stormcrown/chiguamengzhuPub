# 第832章 《被销毁的明朝档案》 ｜从英国自认册封看全球授权体系！ 

大英对清讨贼檄文  
哈哈哈哈哈，你们都已经把人家的战斗檄文复制粘贴拷贝过来了，是吗？我跟大家说，当年从什么时候开始，实际上我们的这种行为模式，是从夏商周的时候就开始了的。我们特别喜欢发“授权书”这件事情，是从夏商周的时候就开始开启的文明制度建设方法。注意了，所有的制度建设，都是跟你的这一群人的组织方式和一开始的社会建设方法之间是有共通性的。他为什么要用这种授权的方式来进行国家的建设和权力的分配啊？我们从夏商周开始的——从夏商周开始的。那就是我们一开头的时候是啥？要知道人类是怎么把自己组织起来的。大家有没有发现一件事：全世界其实在人类进入到早期统治阶段的时候，基本上都是一种相似的东西：一个是神权，一个是奴隶制。这两种东西绑定的同步出现，而且几乎是所有的群体进入到人类文明期的时候，都会走入的一个阶段——就是神权、奴隶制，对吧？这两样东西在全世界范围内普遍呈现出来，它是一个共性。那请问：为什么要采用奴隶制？为什么要采用神权这个东西？因为它最好用啊！我不管大家现在怎么想，今天的中国人不会对奴隶制、神权有什么好感，但是在人类文明的早期阶段，它是最好用的一个玩意儿。为什么好用？要知道在原始时期，人们的欲望是非常少的，最想要的是什么？安安静静地能够好好地活着，隔三差五打点好东西，还能吃上、能滋补上，到这个程度的时候，他就心满意足了。你说我们要牺牲我们暂时的一些享受，集中力量干大事，要把这条河先驯服了——你知道吗，在人类社会当中，这是第一道坎儿，而且这是一道大坎儿，是一般的人群都走不过去的那种坎儿。在中国，今天的人看起来这是天经地义的：你要拥有最肥沃的土地，首先就是集中力量办大事，牺牲每个人身上的一块利益，共同地去把这条河驯服了，再说我们大家就都可以吃饱饭了。但是你要做到这一步，难度系数之大无异于登天。那如何让一群人组织起来听话呢？最简单的一种方式是恐吓，不是诱惑。诱惑这件事情很难，因为人类文明的原始时期人们的欲望不多，欲望少的时候，你勾搭不了他，诱惑不上。所以最好用的手法就叫做恐吓。你就会发现，人类早期的神权管理当中，充斥着恐吓的味道：你要是不这么干，你会倒霉，倒到什么程度——全都是吓唬人的。这个是什么呢？这个就是标准的神权统治。无论是恐吓你要下地狱，恐吓你不得好报呀，还是恐吓你那个山下来的石头一定优先砸你呀，诸如此类的，总之就是吓唬人。神权统治的雏形，就是为了把一群人集合起来，让每一个人都付出一些代价，付出一些他并不想付出的代价，集中力量办大事。这就是搞组织。而人类在初始阶段搞这一组织，这一步是最困难的，而最容易达成的方式，就是用神权的方式进行恐吓，搞预言，老巫婆、巫师吓唬人、跳大神，隔三差五地再搞点小伎俩糊弄你。所有的地方都有这种招式，这个就是标准的神权统治。即便即便如此，你都吓不着人了，咱们一起去干活，你会发现一定会有人跑。为啥？因为集体干活这个事儿特别的累，特别辛苦，特别累，而且充满着延迟满足——就是你要拿到的好东西，那真的是不知道啥时候才能拿到，甚至很多的大工程，你本人一定享受不到，得两代三代人之后才能拿到，要等到后续的子子孙孙无穷匮也。你又看不着这个东西，每一次期待干脆回山洞里面去打猎去，这才是人类的本能诉求。所以这个阶段，一方面他要用神权来恐吓人，另一方面要用奴隶制来绑定人，不许跑，跑了之后拿鞭子抽。这个就是人类最早期的原始社会的共同的相似的组织方式。这种组织方式符合逻辑，并且简单易操作，而且还有路径依赖，因为你用这种方式获利的效率、回报率非常清楚，它就有路径依赖。所以你就会发现，中国以外的绝大多数的地区，一直到近代的时候，依然践行着奴隶制加神权统治的方法，包括大家认为的很发达、很先进的欧洲也是相似的，长期无法摆脱神权统治，无法摆脱农奴制的约束。这个是人类文明早期阶段的时候的一种路径依赖。

而中华文明是如何在这样的一种环境之下脱颖而出，都觉得自己是唯一的文明体，别人尽是蛮夷的呢？是怎么达成这一切的？是因为中国人找到了一种东西：我们用所谓的天道论，用人文观，用道德观，用这些更文明的、更善良的方式，来构建了一个正向反馈的文明系统。你注意到，神权式的管理方式是最便捷的，而我们的祖先找到了另一条道路，它是正反馈的——通过善意的、通过生产的方式、通过正向回馈的、通过道德分配的方式来获得的整个社会的正向前进，就是所谓的文明观、价值观、天道观、伦理观、人文观。所以我们会说，上古天真时代，我们以德治天下，以道治天下。这是啥玩意儿？这个东西就是我们找到了一个人类进行的正向的、文明的逻辑形式，而不是用强迫的、恐吓的方法进行组织。我们的组织方式，就是说我们设立了一套规则，在我们的部落和周边的部落联盟当中应用，而我这套规则特别好，好到什么程度？它可以和所有的人直接坦荡地交流，说我们为什么要驯服这条河，为的是将来要实现什么目标，而为了这个目标，我们今天要做出什么样的牺牲，让做出多少分的牺牲，做出这个牺牲之后，必然会获得什么回馈。那为什么它能够达成这个？为什么他去讲这件事情就能够实现？为什么对方不跑？因为他讲信用！因为我们的祖先踏出了最早的文明的那一步，开始讲信用，开始做好事，并且把这个东西反馈出来，让人立刻都能看到。没错，他说得好，而且他会践行这个东西，就叫做以道德而治天下。这是我们的祖先所开创的一套文明体系，而这个文明体系在自己的内部试点成功了之后，扩展出去，成为了部落联盟，拓展开了，这就是一个文明圈。那这个文明圈你要产生影响力，不是说你吹一吹，说漂亮话就可以了的，它必须要产生实际的影响，让外部的人也能够怎么看到——我这样做了之后，真的驯服了河之后，我们真的吃饱了饭，并且吃饱了饭之后，我们还穿上了衣服，而且我的衣服还特别漂亮。这叫什么？黄帝垂裳天下治。我把我所有的外观，最美丽的东西，最漂亮的房子、漂亮的衣裳、充足的饮食，最后带来的饱满的面孔，我展示给所有的人看，你看了之后就知道他的这套东西怎么样，有用。这就产生了辐射力，而这种辐射力会引来别人的向往。这个向往当中大部分的向往是贪婪，想要抢劫——这才是正常的状态，丛林法则。你家有好东西，我先抢过来再说。这个时候你要做到什么？你不能只有歌功颂德，必须要有刀枪剑戟。这就是最重要的正反馈之一。就是通过这样的良性循环，我们的生产力的水平、发展速度远远超过于其他的族群，所以这个生产力既是一种展示实力的展示，同时也是一种实力的震慑。你想要跳脚，我直接给你拍地上去，但是你想要加盟，我经过考核之后可以同意。这个东西，就是我们进行后续的天下治理方案的一种雏形。从哪儿开始的？从我们最古老的祖先走出天道观的那一步开始，我们的社会治理架构，就从这个地方开始出现曙光。所以到我们的整个国家的制度建设开始出现了之后，对于天下的治理，就变成了一个允许对方加盟，但是我对加盟有限制、有规范、有要求，有各种各样的界限，要求特别的严格，甚至有的时候你加了盟，杀血为盟的，但是有些行为不符合的时候，我还一脚给你踹走。标准特别高，但是确实是允许对方投效的。这种方式就在我们的夏商时期是普遍存在的，商代的方伯很多就是这种加盟制的部落酋长。这种管理方式有没有效果？尽管说它的管理方向也还比较严，实际上商代的时候，动不动那些跳反的方伯，就被一巴掌摁到地底下去了，这种操作方式时不时都有。它是有管理的，没有管理，那对方随随便便不都占你的便宜？不能这样干。所以实际上是有管理。

但是到周的时候，大家就发现了一个问题。周的崛起，实际上可以说是某一种维度上面的方伯崛起。于是周王朝就吸取了一个教训，他认为这种纯粹的加盟、接受管理、接受认可的这种方式不够紧密，中间会有背叛的风险。于是到了西周的时候，他订立了一套新的规则，依然是分封建制，但是他分封的都是自己家的亲兄弟、表兄弟呀、堂兄弟呀、干兄弟啊，他就只由自己授权，派自己的人出去，就不再是招募外来的加盟商，而由自己派人出去，用了这样的开疆拓土的方式进行授权。但是不管怎么说，我们延续至今，一直到明代都始终存在的家国天下的治理观当中，始终都包含了“社会治理授权书”这样的一种东西。这个就是所谓的朝贡授权，天下都是用的这样的方法。所以你才会发现我们经常说，我们给哪个哪个地方发了印绶了，甚至倭岛上面，动不动在倭岛上面，都能够找到我们当初授权的证据，有旨意有印章，叫“倭奴”或“倭奴国王印”来着，那个字反正称他为倭奴国。但是他冲上来，我给你授权——实际上是真实的一种授权吗？对方本来就住在那儿，本来就设立了政权，但是因为我们的治理方案本身是文化体系的、是道德体系的、是天道体系的，它不是一个单纯的暴力恐吓体系，所以它通过文化道义的领域，是可以通过授权的方式，认可对方的现有秩序。所以我们会发放很多的授权书出去。明王朝时期，我们的授权书到底发放到了什么范围？今天已经不能够完全考证清楚，因为有大量的史料，其实确确实实是被销毁了的，包括海外的很多的印记，也是被刻意地销毁了的。但是不能说什么都没有，现在也有很多的海外的华人华裔们，在一些西方的图书馆和典籍的档案册当中，去寻找当年明王朝授权的痕迹，而且找到了不少。这都是在陆陆续续地寻求当年的印记。最后就会发现，明王朝远非后来清修明史里面所描绘的那一个局促的地方性政权，它是一个拥有着极大的全球视野的、相当国际化的一个国家。但是他具体做到了什么样子，因为各种各样的实证被销毁得实在是太过于严重了，所以很难在短期内直接复现出来。可是有一点可以确定，就是当年小英在发起鸦片战争的时候，他自己雇人写了一篇战斗檄文，在檄文当中，他自己主动承认自己是接受了明王朝册封的国家。也就是说，当年至少小英自己承认的，明王朝的授权书，已经发放到了西欧。
