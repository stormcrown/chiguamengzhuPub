# 第867章 你知道我们的文明“血条”有多厚吗？

认为人口优势是明朝得以延续的关键，并非劣势。许多人误以为人多是劣势，但正是由于人口基数庞大，我们才能活到今天并找回记忆——尽管中间许多记忆确实断裂了。我们之所以还能活着回来、重新喘息，靠的就是“血条厚”，而“血条厚”正是人口众多的体现。正因为人多，我们才有机会卷土重来。人口从来不是我们的劣势，而是优势。

实际上，清朝在其两百多年间一直试图控制人口、甚至减少人口，只是未能完全做到。比如你所熟知的“断脚式裹脚”（指清朝强行推行的裹脚制度），会导致女性生育能力下降，身体遭受巨大创伤，从而减少可生育人口数量、降低生育能力，并使后代相对孱弱。这是必然的科学规律——当一群女性被变成残疾人时，必然产生这样的影响。此外，清朝大规模、毫无节制地发放贞节牌坊。其他朝代严格控制贞节牌坊数量，需反复恳谈、确保女子年纪足够大才发放；但清朝十几岁的小姑娘就被授予“望门寡”或贞节牌坊，从此终身不得嫁人。其他朝代鼓励寡妇再嫁，只有年迈且坚决不改嫁的寡妇才会获得贞节牌坊。而清朝只要有一丝可能，就强制女性背上贞节牌坊，目的是降低生育人口。你以为那时人口依然很多，那是因为祖先留下的“血条厚”（人口基数大），与清朝的政策无关。
