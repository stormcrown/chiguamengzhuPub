# 第859章 中西方历法之争：阴阳合历的智慧为何碾压西历？

**整理后的段落：**

@姬子恒 中西方历法之争：阴阳合历的智慧为何碾压西历？从24节气到闰月调节，解析中国历法的科学功能性。

我们用的是阴阳合历，我们没有用阴历。纯阴历对我们来说并不好用，因为你种田需要的是太阳历。24节气就是标准的太阳历——太阳历是观测太阳的规律，对太阳的运转周期打上节点，这个是太阳历。太阴历呢，就是观测月亮的规律。因为太阳和月亮的运转周期完全不一样，所以它们中间的差值会比较大。

之前有人专门出来聊，说关于西方的历法是不是一种抄袭现象，包括它莫名消失的十天。之前我曾经聊过这个话题：就是莫名其妙的消失十天，这件事印证了一件事——历法能不能调整呢？当然可以调整。历法调整了有十天的误差，正常吗？正常啊。注意，因为你有很长的时间使用一套固定的历法，最后由于岁差等等的原因，累积出来了一个十天的时间差，这件事本身并不奇怪，它很正常、很合理，一点儿都不奇怪。但是你把这时间在单独的一个时间节点里面，把这十天直接抽走，这件事说明你的历法有问题。

为什么这样说呢？西方的历法里面最明显的一个bug，一个缺陷就是它有12个月。啊，一定要注意一件事：就是西方历法现在的所谓的西历，就是大家一直在运用的这个——现在每天你抬起手机来，今天几月几号的这套历法——它最大的一个缺陷并不是准确与否的问题，最大的缺陷来自于它有12个月。因为它是一个纯粹的太阳历。我们在元的时候就已经编撰出来这套历法了，就是授时历。啊，我们在元代的时候就已经把这个纯粹的太阳历编出来了。岁差我们早算出来了，没有问题啊，就是一年是零点几几，那事情我们早算出来了，很准，小数点后的四位数我们都算出来了。不差这些啊，这些都算得出来，没有问题。

但是我们的历法之所以不是那个样子，是因为我们要考虑月亮。而西方的历法身上最大的一个bug也不是所谓的十天，不是十天这个数字，而是他集中地把这十天直接抽走的这个行为，因为这意味着它只是在凑数。当你只是在凑数的时候，问题就出现了。你干啥要12个月？中国人定12个月是因为我们要观察月亮，我们确保每个月的15月亮是圆的，这是太阴历。因为太阴历会影响人的身体节律和潮汐运转，所以对于水文作业和我们的身体调节是有用处的，所以才观测太阴历啊。我们观测月亮不是闲的没事干，是因为月亮影响潮汐，它跟我们的水上作业、跟我们的渔民的生存生死相关。我们的历法当中包括太阴，它的目的是指导生产以及保护人体健康发育，它是功能性的。

所以我们界定12个月，是因为一年当中大概有12个月圆，这才是我们定12个月的原因。但是呢，因为中间的误差就会很大，所以我们才会出现闰月。而这个闰月闰在哪一个月，也是因为我们需要调剂太阴与太阳之间的偏差，然后再来决定在哪个地方去进行闰月，塞一个月进去，好让这个15能够和月圆对起来，这才是我们搞闰月的一个基本原理。所以我们的一年是12到13个月，是这样的一种规律。

那西方人他搞一个纯粹的太阳历，你安12个月干啥？你那个12个月有价值吗？毫无价值啊。你知道吗，一年你分成十段也是没有问题的。纯粹的太阳历，你分成十段不好数一些吗？说那个什么，说十进制的时候，一个个的信誓旦旦的人人都有十个手指头，那你一年的时候分什么12个月？你又不考虑月圆月缺，你每个月的15，你月亮也不圆呢，闲的没事干嘛在那里搞12个月？所以它本身啊，有十天的误差是件很小的事，很合理。但是他一次性地把一个十天从一个随机的地方抽调走，这件事意味着这个历法不是他原生的。

嗯，好，欢迎大家来到直播间啊。啥？"once"来源于"moon"？嗯，按照词汇来说是这样来源的。也就是说他的月也是跟月亮有关系的。但是你自己去看历法，他哪个月能够跟月亮对上？压根对不上啊。多少年能够碰上一个巧儿能对上了？结果他非得要给它起个名字叫月，然后呢还一年硬生生的一定要分成12个月，然后每个月的那个时间，有的时候30天，有的时候31天，有时候28天，图什么啊图什么？你一定要弄明白，如果说真的是你自己原创的，那它一定是有原因，一定是有依据的，必须有依据啊。我给他定了这个日子，目的是什么？我订了这个月，目的是让15的时候月亮是圆的。但是西历呢，定了那个月也跟他叫月，但是他那月亮一点都不圆，随机模式，有的时候上弦，有的时候下弦，你说不准你整那晚上干啥？
