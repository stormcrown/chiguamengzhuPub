# 第849章 跪着的人生-真汉奸从不需要外国主子！

@姬子恒 跪着的人生｜百年前脚盆人早已看透：真汉奸从不需要外国主子！那么，什么样潜质的人会成为汉奸？

如果大漂亮倒了，是不是我们这里的汉奸99%会销声匿迹？大漂亮倒了之后，我们这儿应该是会损失掉一部分汉奸。但是这个比例，我确实不敢说它是大部分，真的不敢保证是大部分。因为我觉得我们这里的汉奸走狗们，他不全都是背靠大漂亮的。目前我的判断是这样。

关于汉奸，咱们得知道汉奸是一种什么特征。很多人以为汉奸就是有主子的人——有些人误以为汉奸是有洋主子的人。来，咱们纠正一下：汉奸不是有洋主子的人，他不一定有洋主子。甚至很有可能，他压根儿就拜不着洋主子的码头。我们国内的汉奸，很多的汉奸是没主子的，洋人不要他当狗。他压根儿就没有拜对过码头，根本就没有被别人收过。他纯粹的就是一条狗啊，没拜过码头的那种，很多的。

那请问，汉奸和非汉奸正常人之间，他的区别到底来自于哪儿？说到这儿，那就不得不提：脚盆在这件事情上面洞若观火，他对这些群体的观察是特别清楚的。为啥？因为他们当初侵华的时候，招募了大量的汉奸，他们跟汉奸老熟了。所以这个事你一定要参照脚盆人的说法，因为脚盆人见到的汉奸，比我们体会到的数量要多、档次要高，效果还显著。所以它们是有样本的，脚盆是很有样本的，他们见的汉奸太多了。于是呢，他对汉奸做出了规范性的整理和总结啊。他总结出来的汉奸，我觉得非常的精妙。

那怎么说呢？就是汉奸这群人吧，他就是典型的没有自我。来来来，汉奸的特征叫做没有自我，而不是说他有了洋主子，或者说他特别爱脚盆或者特别爱阿美。阿美都没有啊，全都没有，没有。他不光是对自己的祖国不忠，他对洋人也不忠。汉奸的典型特点就是：他压根儿不知道自己是谁。嗯，有奶便是娘，谁强谁是爹，谁撒狗粮它就是谁家的狗，对吧？这是当年脚盆通过了大样本的伪军观察得出来的结论，我觉得还是颇为科学的。毕竟一个中国人能见识几个汉奸呀，是吧？你搁村里面出个三五个汉奸已经不得了了，村长还要天天带着大家一起去呸他。但是脚盆天天见汉奸——他们见的汉奸那都是成组织的，一帮子一群的，乌泱乌泱的站起来比他高一大截儿，但是给他当汉奸、给他当走狗。所以呢，脚盆见识的汉奸比我们见识的数量多，质量高，组织度还挺强。但是呢，他们就是汉奸，他们是没有任何自我的群体。

所以大家可以观察一下自己身边的，谁有汉奸的潜质？那就是：“哎呀，你不要说那些话啦！阿美就是厉害！” “你不要说那些话了，欧洲人就是好看。” “你不要说那些话了，脚盆当初就是强。” “你不要说那些话了，其实就是奠定了我们的那个什么疆域。”注意了啊，这种类型的人就充满了汉奸的潜质。唉，啥？嗯，那还挺多的。我这一说你就知道了，就这样的人——谁强谁是爹，他强他有理。他是没有自我的。

谢谢阿瑟·鲍啊，没有自我。谢谢法无常。因为他没有自我，于是呢，在强敌出现的时候，他直接就去给人家当翻译官、当走狗去了。那这些人在那个时候会表现为汉奸，在平时的时候就表现为“你不要说那些话了，什么什么中国，什么什么什么什么什么什么什么东西都没有用，别人强，别人厉害，别人有钱，别人是老大爷啊”。这种类型的人就是当年的汉奸。他们会说他们是文明的。哎，对啊，这些人特别喜欢说他洋人是文明的。然后你再问他到底什么叫做文明，他说：“你看他要是不文明，他能那么强吗？”所以最后归拢起来就是：强的那个就是文明的，强的那个就是伟大的，强的那个就应该是你追随的。至于他自己是谁，他是不是爹生娘养的，对他来说不重要。

然后这种理论呢，又非常容易倒推回去，推成一个什么东西呢？叫做“生养不是恩，托举才是恩”。然后来，有没有在网上看到这群人？生育不是恩，托举才是恩。见到过这样的人吗？啊，见到过这样的人扣个一啊。你自己总结一下这个理论——这个理论：生育不是恩，托举才是恩。什么是托举？恩主老爷给钞票，给道路的那个人，提携他的那个人。这是什么？谁强谁是爹亲。
