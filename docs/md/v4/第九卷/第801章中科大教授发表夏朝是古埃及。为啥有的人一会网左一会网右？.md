# 第801章 中科大教授发表夏朝是古埃及。为啥有的人一会网左一会网右？

古埃及夏朝论者的目的：  
我们中国国内，一直有一派学者主张“中华文明西来说”。这一派学者其实从民国时期就埋下了苗头，他们的发展进程也相当漫长，因此整体体量并不小。他们的主张是什么呢？他们的基本标准是：世间人类最伟大的文明的发端都来自西方。这其实是西方人缔造的西方中心主义下的一个产物。直到现在，中国国内还有很多非常主流的学者，甚至是学者中的翘楚，在他们各种各样的研究中，都包含了大量的、大量的“西来说”内容。  

这种“万物西来说”的东西，对于中国人的自信和中国文化的完整性，其实有很明显的破坏。如果说我们所有的文化都是西方人弄出来的，那我们为什么跟他们之间有这么大的差异性？我们的很多文化内核完全不同。然后，又一定要说“这也是那儿来的，那也是那儿来的”，那中国人对于自己的历史认知和自身定位，就会出现麻烦。  

在这种情况下，出现了一个折中型的人群。这个人群也算是一个学术流派吧，他们属于想要找回一点自信，但是没找回来。在互联网上的说辞中，这应该叫做“挽尊”。这群人呢，他们认为文化确实是从西边往东边来的，但在试图挽尊的过程中，就要说：虽然这个文化是从西边冲到东边来的，但冲过来的人依然是我们的人。于是就把古埃及整成了我们的夏，甚至更加古老的时期，说埃及其实就是我们所传说的那个夏。这是干啥呢？这是想要说：“你说西来的对不对？但西边来的是什么呢？西边来的也是咱自己人。”大体的原理就是这样。这是一个试图挽尊、想要找回自信，但没找着的状态。这个流派比较尴尬：西来说的人也不是很看得上他们，中国原生派的人那就更看不上他们。他们就是一个很尴尬的人群。  

嗯，好。欢迎欢迎大家来到直播间。嗯？什么？嗯，苏美尔就是夏？我在所有历史演进的这些项目当中，凡是很明显搞谐音梗的，都不可靠。为什么这样说？因为研究历史问题时，一定得注意到一件事：语音的流变是非常非常严重的。包括表音文字系统中的一些要素，比如语素这些的流变，也就是所谓的词根的流变。很多时候，它是用某一种符号固定下来之后，再继承下去的。直接通过口音就复刻过来，这种事情其实很荒唐。所以，一听有人说某个东西声音上如何如何——如果只是几百年，这事儿好说一点，确实会有影响，音译的痕迹会留下——结果他们动不动就是好几千年，连当地的人都已经没有了，当地的文字都已经是今天的人莫名其妙翻译出来的了，他们还说什么“这个东西是怎么传承下来的，当时的语音是什么”。这种对几千年前的语音的复原，尤其是原住民都已经不在那儿的时候，靠猜想进行语音复原，太扯淡了。  

嗯？啥？有人要搞我？嗯嗯，好，谢谢你们提醒啊。嗯，什么？中庸？不偏不倚，居于正中。嗯，那个中庸啊，很多人写了一段关于中庸的事情，以为它是一个“不左也不右”的过程。当你按照“不左也不右”的方式去思考的时候，常常会觉得它是一个和稀泥的处置方式，甚至是一个不咸不淡的处置方式。但是中庸是啥？中庸是直奔目的地，是直奔目标。它指的是界定目标之后，无论你怎样偏移，最终都要挪回来，都要奔向目标而去。这个东西才叫做中庸。你一定要先强调你的目标是什么。  

它其实跟今天的“左右”是有关系的。左右是什么？今天很多人说：“我是进步的，我是左倾。”他一说自己是左，就觉得自己很高级；一说谁是不好的，就说谁是右。以至于你会直接觉得：你说谁是右，谁就是反动的；他是左，他就是进步的。其实这事儿不一定。因为左右是什么？左是快速前进，快速前进就叫左；停滞甚至往后走就叫右。这是一个倾向性的选项。但你是基于什么目标而做这事儿，就很重要。你先得设立一个目的，然后再决定是往前跑还是往后退。当你这个目的需要你往后退的时候，你往后退才是对的，就会变成对的；但如果这个目标不需要你向前奔跑，那么左就是对的。因此，中庸强调的是：你把目标找着，你是奔着目标去的。不要跟我谈路往这儿往那儿，无论你选哪条路，最终你的目标是走到你的目的地，这个东西就是中庸。  

所以为什么它要强调“不偏不倚”？不偏不倚就是：你不要执着于选择方向性的问题，而是要侧重于你的目的性——你到底要干什么？你先弄清楚，然后才存在“我到底应该左还是右”。就像抗战期间，为什么左就是进步、先进的代言？因为抗战时我们的目标是什么？实现民族独立解放，这是一个解放运动。这个时候，你到底是抗争对，还是站在那儿妥协对？很明显，抗战期间当然是殊死搏斗的这一方才是正确的，是奔向目标的正确方向，所以这时左才是正确的、先进的。但如果哪一天你设定的目标本身就是极端冒进的，甚至是一个错误的目标，那么停下来的那个才是对的。你设定了错误的目标，这个目标会伤及根本，这个时候你越向前跑，犯的错误就越大。就是这个基本原理。  

所以，什么叫做中庸？中庸就是选定正确的目标，奔向目标而去。好，欢迎大家来到直播间。
