# 第830章 《导演不破圈，电影就破防》 ——中国电影工业化的暗伤与曙光！

现在很多导演都是“破圈”出来的。我可以分析一下：为什么现在许多优秀导演都是破圈出来的？为什么非科班、破圈、冷门的导演能拍出好电影？为什么热门的、科班出身的、在圈内非常有资历的导演拍一个烂一个？主要原因就是——电影这玩意儿其实是一个巨大的利益分配链条。能理解吗？电影这套系统，就是一个巨大的利益分配链条。

这些年我们一定会注意到一件事：很多冷门的、跨圈的、突然横空出世的导演，他们的作品质量会超乎想象的好。这说明什么？说明中国的电影市场不缺技术，这是第一件事。一个跨圈的导演一来就能拍出像样的电影，说明我们不缺生产端、不缺技术、不缺服务人员，甚至也不缺钱。要知道，一个新导演，尤其是跨圈的导演，在以前没有多少过往作品展示时，想拉到钱本身就不容易，想拉到团队也不容易，但他却能做出大制作、好电影，这说明我们的生产端实际上不缺钱、不缺技术、也不缺人才——这些东西我们通通都有。生产电影需要的一切，我们这里都是齐全的。

那我们缺的是什么？为什么我们的电影产品不容易出现攻心之作？短板到底在哪？按我目前的观察，短板在于内部的利益分配链条，而这个利益分配链条掌握在一些跟中国观众毫无共情关系的人手上。什么环境呢？要知道这个利益分配的事，你可以稍微算算账，不需要很细——毕竟像我这样不会算数的人都能盘得明白。什么样的账呢？那就是在电影这样庞大的体系里，每部电影都需要巨额投资，投资进来后要让它变现。在变现流程中，需要广告商、需要院线支持、还需要各种媒体平台支持、需要自媒体账号支持——也就是所谓的流量支持。这些东西都打通了，那就意味着这个电影至少不会亏本，它有变现渠道，可以回本——钱出去了，有回头钱。这是最重要的一条利益分配链条。

在这个利益分配过程中，最需要的是什么？媒体，流量。过去是传统媒体，现在是互联网媒体。不管怎样，它其实就是耳朵和眼睛被谁掌握——这个叫流量池。这些东西通通可以算作流量，某一个维度的流量。所以一部电影要最终成功变现，必须掌控某一种流量的发行通道，这是必须的。那么谁手上拥有这些流量分发通道？以前是传统媒体，传统媒体说哪个电影好，大家就去看，这就是流量。现在呢，现在就是各大流量平台，这就是我们的流量分发池。

所以为什么很多你看到的专业电影人，会去找几个小鲜肉、小花小草来拍？尽管这些人肉眼可见演戏水平一塌糊涂，但一定找他来——为什么？又贵又不好用的演员，用他干什么？因为他有流量池。他这张脸只要出现在这儿，流量就为你打开了，这就是天然的变现通道。而只要电影是通过这种方式变现的，那么掌握流量通道的人就可以决定电影的叙事方向。现在掌握流量通道的都是什么人？你只要看一看——我的直播间是在什么样的话题下面会被封，送我上热搜的都是什么德行的——你就知道了吧？什么人在把控互联网的流量方向。

所以为什么你看到的那些电影，制作精良、导演也不错、演员也专业，但最终做出来的东西总是不对味儿？看了心里头难受，甚至看了感到膈应？为啥呢？因为他所期待的变现流量池跟你没有关系——他要的那个变现流量池，跟你没有任何关系。它推动的是它那一块的流量池，而那一块流量池正好就是来恶心你的。
