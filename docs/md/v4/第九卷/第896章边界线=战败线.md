# 第896章 边界线=战败线

我们现在的古典地图都有问题——全部历朝历代的地图都有问题。这些地图之所以有问题，是因为它们要为满清歌功颂德，从而造成了历朝历代的历史地图册都有问题。为什么这样？因为古时候的地图没有边界线。中国传统意义上的地图（堪舆图）是没有国界线的。你今天看见的所有有国界线的地图，都是现代人做的臆想型假地图，其圈定范围由地图创作者按照自己定下的指标进行筛选，所以它不是古人的地图，而是今人为古人安插上的地图。

中国自古以来没有国界线，一直到清朝才出现。为什么？因为清朝太弱鸡了，以至于遭到了全方面的列强围剿。列强侵蚀了无力的草原、戈壁滩、高原、荒漠、无人区，这些地方变成了战略缓冲区——因为已经被各路强者围在了中间，于是才形成了一条边界线。这条边界线就是列强打过来之后，成本太高、运转不下去的战略缓冲区，它靠合同来界定，并不是清朝的实控区，也不是清朝的开疆拓土。

中国人自古以来没有边界概念。为什么没有？因为我们疆域范围的界定方式就不是边界线。用这种画根线的方式来界定，在那个时代全世界都不行——画不了，画完了也守不住。你实际上拥有什么，取决于你能占据哪个要塞、哪个关卡、哪个城池。所以你看先秦时期的故事，动不动就说“我把哪个城送给他了”，没说“我们两国之间重修边界线”。春秋战国时，你们是不是都这样讲的？我跟哪个国打仗打输了，或有点什么事，就要把哪几个城池送给他。为什么要这样？因为划定边界线是没有价值的事情。你需要的是掌控某个关键节点，这个节点能辐射的地方，必然是你的实控区。

中国古人没有边界、国界线的概念，而且当时全世界都没有国界线。没有人有资格与中国签合同，说那条线那边是他的、这边是我的——没有这回事。边界线这种东西，是到近现代才开始出现的。之所以有人在这里断章取义，说“满清奠定了现代的中华版图”，听起来特别有道理，为什么呢？因为他的版图看上去跟现代的很好理解——因为有边界线了。为什么有边界线？因为他输了，因为他无能为力，因为他无法管控。边界地区、边疆地区在他两百多年的经略之下，变成了无人区以及半无人区，人口凋敝。到了这样的地步时，那个地方属于三不管地带，变成了满清与列强之间的战略缓冲区，所以才会签合同、划势力范围，然后就出现了所谓的“了不起的大地图”。

兵力到了就是势力范围，或者建造各种关卡。你看到古时候动不动就说“出某关、出函谷关到哪里哪”，然后哪个关卡如何如何，被夺了、属于谁、被谁掌控，哪座城池属于谁——这些都是来划分势力范围的。为什么？因为你只要掌握了这个节点，周边的东西你就能够真正控制，而不是画条线。古典时期你画条线有用吗？画在那个地方，然后要在这条线上隔几百米站一个人来守住边界线吗？所以这种边界线自古以来就没有。一直到清朝被列强围剿，才出现了边界线。这不是强盛的表现，这是衰落的表现。所以不是什么“奠定了现代版图”——这听起来像是真的，为啥呢？因为中国近代垃圾得一塌糊涂，所以才会被西方人画上边界线，把我们圈住了。

嗯，那也不对呀。清朝人口数量是明朝的一倍多，咋能是人口凋敝呢？你是听不完完整的句子吗？我说的是边疆地带被他整得人口凋敝。来，草原上面，我们建国初期的时候，内蒙草原上的牧民有多少人？不足50万。疫病流行……谢谢远山明，是我们在建国之后专门推行的医疗救护才把他们救回来的，不然人都没了。这不叫做半无人区吗？你知道天天有一斗那些清粉会冲出来说：“哎呀，你们汉人地盘特别小，只有汉地十八省。”汉地十八省哪来的？清代缔造的，干什么？圈禁汉人，不允许汉人擅自垦荒，不允许你出这条界线。汉地十八省是一个圈界线，不是国界线，也不是汉人的势力范围，是圈禁的监狱的栅栏。那么请问汉地十八省以外的那些地区呢？他采取的策略就是使当地穷困，使当地人丁凋敝——连他的蒙古好汉达都是这样的下场。所以经过了满清两百多年的管理，才会出现边疆地带人口凋敝。

你看上去是不是觉得，至少说中原的人那么多，汉人还占了便宜呢？汉人人丁增长，从清初的几千万涨到清末的接近四个亿，涨了3~4倍。但是同一时期，我们周边的其他地区涨了7~10倍。你要有横向对比，一定要有当时的横向对比。因为物种大扩张，所有地区都出现了人口爆发，而全世界最会种田、最会省吃俭用的人群增长了3~4倍，那些刚刚开始进入稳定生活的地区增长了7~10倍——这叫增长吗？

满清边疆凋敝中。明朝时期，新疆、西藏、内外蒙古都不在他控制之下——这句话，你说这是说明明朝啥都没有？你这句话本身就是满清编造的谎言。清朝在这些地方跟当地的头人拜了个把子，然后就回家了，他说他对这个地方实控了？明朝在内外蒙古的草原、在青藏高原、在西域都有卫所屯垦收税，但明朝对这个地方没有实控？你能把标准一致吗？要知道，明代的亲王有条件随时随地到高原上面去，派一小队兵马把当地的活佛拉下来给自己讲经。明代的亲王也有爱听经的人，隔三差五就跑到高原上面去，派一个小分队就把人家的活佛、高僧拽下来，拽到自己府里讲经。请问这个东西算不算实控？还是只有满清的皇帝给对方写封信，才叫做实控？顺治皇帝给高原上面的僧侣、统治者写过信，他说：“你们以前是被大明统治的，今天我接了大明的活，你就接受我的统治。”对方给他回信：“可以。”然后从此他就可以算实控了？你这个标准是不是有点不太一致？包括什么“当地的转世要受到朝廷的管理是清朝开始的”——明朝就这样，转世授权这件事情明朝就推行了。但明朝不实控？清朝虽然不收税，但他实控了？棒棒的。

嗯，农民伯爵说“满是实控咋办呢”，啊，他开口就是实控吗？谁知道他怎么控的呢？收税的不算实控，卫所驻军屯垦、有人种田也不算实控，一定要等到酋长过来跟当地的头人拜了把子，然后各回各家、各找各妈，这才叫做实控。你只要改变实控的定义，你可以实控全天下啊。
