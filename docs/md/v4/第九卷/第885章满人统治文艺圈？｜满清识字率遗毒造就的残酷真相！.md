# 第885章 满人统治文艺圈？｜满清识字率遗毒造就的残酷真相！

为啥演艺圈那么多满人？其实整个文艺圈里，当年的八旗后裔确实数量不少。实实在在说，就是数量不少。这有几个方向的来历。

第一，我们之前说过，清代普通人的识字率都快跌到零了，但整个中国的识字率还剩一点点。那哪个群体的受教育指数会比较高呢？自然是八旗群体。八旗群体的识字率比较高，受教育比例比较高，这就造成了一个现象：他们在文化、文艺方面比较有工作的能力。这是第一个方面。

第二，辛亥革命成功后，八旗及其后代的社会地位急转直下——从原来骑在别人脖子上拉屎，变成了受人歧视的群体；从之前的提笼遛鸟，变到后来你必须出去打工，而且受到他人排挤，成为社会当中大家都知道、都看不上的群体，社会地位极低。这时候，他们的就业渠道就非常狭窄。他们可以在社会上担任什么样的工作呢？其中有一部分是干体力活的。例如，有些八旗里保留了一部分财产、在社会上还在正常运转的人，就包下了车行，然后招募这些找不到工作的八旗子弟来当黄包车夫。所以，旗人的后裔当中，很多男性去做了苦力活，而女性则有相当一部分流落风尘。这也是互联网上有些人玩梗时说“八大胡同”的原因——辛亥革命后，一部分旗人妇女流落风尘，一部分旗人男性开始干苦力。

但是除此之外，他们还有一个就业渠道，就是走文化文艺的路线，例如说唱戏。为什么？因为我们自古以来都认为唱戏属于贱籍，归在下九流里面。所以，尽管在就业上旗人遭到了群体性的排挤，但这些行业由于它本身是贱籍行业，所以他们就可以在里面工作。同时，他们相对来说识字率较高、文化水平较高，就可以在这个行业当中崭露头角。这是一个基础。

之后到了建国的时候，你想一想，那时我们的文盲率很高。尽管经过了民国几十年的推动，实际上比清末时好很多了，但文盲率还是超过了80%，我们还要不断推行扫盲运动，普遍的受教育指数较差。但是，国家安稳了，就需要很多能够工作、能够念书、能够认字的人，参与到文化工作当中来。那谁有这个能力进来呢？就是经过了改造、表示愿意成为共和国公民、不再怀念过去那些耀武扬威的日子、承认过去错了、今天改邪归正了的这些八旗后裔。他们就有一批进入到了文艺和文化工作的岗位上来。

所以为什么今天你就会发现，文艺圈里似乎他们的人特别多？就是因为这样子一代又一代传下来，实际上会导致他们在特定的职业范围当中形成自己的传帮带系统——拉拔同类嘛，朋友、亲戚，这样的内部圈层就被固化下来了。所以才会出现“哎呀，怎么那么多呢？满人就是这个样子”的现象，就是这样的缘故。

当然，由于已经是新社会了，所以人家在正常做这些文艺工作的时候，你不能去挑剔这个。除非他在文艺工作、文化创作上确实掺沙子、注水，那咱们就去批评他掺沙子、注水的行为。有没有这种行为呢？有，坦然坦荡地承认有这种行为。所以从80年代开始，我们的文艺文化创作领域当中，受到监管的程度开始逐渐减低，逐渐开始宽松自由。创作之后，你就会发现这个群体就开始陆陆续续写各种各样的小故事、演各种各样的影片，开始为满清朝廷洗白了——把那个时代的人描述成什么有规矩、讲老理儿、有传统，就这些东西就开始陆陆续续回来了。
