# 第879章 文化警报｜你买的 现代汉服 可能是假传统！3秒识别满清元素！

为什么不推现代汉服作为民族服饰？

我们现在啊，没有精力去处理这个问题。现在这件事情能在民间推动，并且没有官方的阻力，你都要谢天谢地了，还想什么呢，亲？嗯，好，现代汉服——你但凡说真的要推现代汉服，你猜会设计出来一个什么东西？现在你自己到某宝上面去搜“现代汉服”，你看能搜出什么东西来？搜一下“国风”，搜一下“新国风”，你看能搜出什么东西？就不说怎么推动了。

你知道那是零几年，2001年还是零几年，就是那个唐装诞生的那一波。是01还是02年，忘了。那时候也是要设计一个拿得出手的传统民族服饰，因为国际大会上的传统，就是要给各国领导人赠送传统民族服饰，记得吧？那是哪一年？01还是02年？然后设计出来的是什么东西？以西装为底板的蜈蚣扣的唐装——以西装为底板，满洲元素为装饰的一套唐装。除了上面那个“唐”字跟我们的历史传统有关系，其他的东西有多少关系啊？记得吗？记得的话，你们不行的话自个儿到网上去搜图啊。有当时还是小布什呢，当时小布什过来串门的时候，他也穿了。你知道的，小布什长得那个脸有点猴像，他穿上那套衣服，非常非常的离谱。大家可以看互联网上，现在都有这个照片。

所以你就说嘛，你说我们设计一个现代汉服，你猜会设计出来什么？设计过没有？设计过。谁主导的啊？官方主导的，提出来的要求：我们要展现民族形象，要展现民族传统。然后去招聘专业设计师进行投稿，最后出来的成果就是那东西。你想再来一次吗？为什么后来03年的时候会有汉服复兴运动？就是因为被这一首给恶心着了，给刺激出来的。如果我们的传统服饰就长那样的话，都不想要那个传统服饰了，于是才有了汉服复兴的星星之火。如今这星星之火已经燎原，然后有人下山摘桃子了，说清汉服也是，汉服博弈进入到了新阶段。然后这个时候你说“我们弄一个现代汉服”，你猜会变成什么样子？现在皇后大礼服在准官方设计里面已经出现场资金了（注：原文“已经出现场资金了”疑似口误或遗漏，保留原意）。你真推啊？能推出啥来？推完了之后又得剃头啊。

嗯，啥？咋叫唐装跟唐有什么关系没？跟唐一点关系都没有啊。那个唐装之所以会起这么一个名字，我个人认为啊，他实际上是拿来忽悠领导的，忽悠上级。你看都“唐”了，那肯定是我们传统服饰，然后领导也不懂啊，然后看“唐”，哎，好，签字就同意了，就过关了。就这样一个忽悠的作用。然后他这个起名字的源流，实际上跟唐王朝一毛钱关系都没有。他靠的是什么？唐人街啊。他用的是唐人街。为什么呢？因为唐人街上面有一些人保留了清代穿马褂的这种习惯，因为他们是那个时候出去的，所以留下了当时的衣裳。唐人街里面有一部分的华人会穿着马褂，他就基于马褂，然后结合了西装的剪裁，做出了那一套衣服叫唐人装，实际上跟唐半毛钱关系都没有啊。就这么一件事儿。

嗯嗯嗯，什么嗯？寿衣就是打那之后，寿衣才出现这种款式的啊。现在可能很多人直觉上面会觉得穿寿衣的款式不吉利什么的，要挑一个比较晦气的款式。你记得一句话吗——“事死如事生”。咱们重复一下，“视死如事生”是什么意思？中国人非常看重逝者的尊严，我们要让逝者保有尊严，如同他还活着一样。所以寿衣这个东西，不是挑你觉得晦气不吉利的东西给她穿，而是挑你认为最好的一件给他，或者挑他最喜欢的一件给他，这个东西才叫做寿衣。因为我们在传统上面认为什么？认为我们死后是要见祖先的。什么叫做“列祖列宗在上，不孝子某某某要如何如何了，做不好我就去见你”，这种誓言是干啥？就是中国人认为自己归去之后的结果是要见到祖先。所以当你要去见到祖先的时候，你希望身上穿什么？你要见你们家几十年不见的长辈的时候，你希望你穿着什么，打扮成什么样子过去？穿成灰气的、不吉利的、丑哈哈的样子，还是穿成最漂亮最美的样子过去？你自己考虑一下嘛。你去见祖宗，你不应该穿好一点的吗？穿一个让你的祖先看了之后就觉得很体面的衣裳吗？所以寿衣应该什么样子？寿衣应该挑你一生当中觉得最好的、最体面的、最可以穿上之后去面见祖先的衣裳，这才应该是寿衣。今天这种所谓我挑那种最花里胡哨的、日常最穿不上的、看了之后就晦气的、看了之后就不舒服的衣服当寿衣，你穿这玩意儿有脸见祖宗？没有。所以寿衣的这个概念，本身现在就已经出问题了啊。

嗯嗯，但是寿衣是左衽吧？在那个，嗯，在周礼当中规定寿衣穿左衽啊。但实际上根据后世的墓葬来看的话，大部分人没有听这话，大部分就没听。这正经事啊，大部分压根儿没听。就是说是得左衽，实际上穿着右衽就下去了啊，不怎么管这事啊。所以我们在那个，就是在葬仪方面、丧葬典仪方面，中国人最重要的一个规则就是视死如事生。你把握了这个底线的原则，你就知道什么东西是传统，以及什么东西不是传统。什么东西是传统呢？今天很多的那种什么烧个房子下去的呀，烧个车下去的呀，弄个笔记本下去的，你看上去“哎呀，太现代、太摩登了”——这是最符合传统的啊。这个思路是特别传统的。你给那些牺牲的战士去发我们的那个六代机的照片，烧过去，这就是最符合传统的祭祀方式。所以很多的时候，你以为很摩登的东西，那是最传统的；你以为很传统的东西，那是假传统啊。

啥？弄个航母？嗯，倒也不是不行啊，就是会不会有点儿太大了啊？

胡赵武灵王胡服骑射咋说？有啥咋说的呀？改军装。胡服骑射的意思是改军装。军装是什么？军装是军事装备的一部分。军事装备怎么好用怎么来，它不算礼制服装的内容啊。所有军装的系统、铠甲的系统，你不能把它算到服装系统里面来。它不是服装系统，它是功能型的配件，它的目的是作战。能够满足作战的，穿啥样都行，就是这样的啊。所有涉及到军事的，不能够算进礼制系统当中。这是完全两条线索。军事是纯功能属性的。这相当于什么？相当于在抗美援朝战场上面，我们确实是在跟美军打仗，但是如果你捡到了美军的军火，捡到了他的罐头，肯定是要带回来的呀，肯定是要吃的呀，肯定是要用的呀。这是什么？这是军事装备，军事的配件。作用是什么？要有用，要有战斗力。哪个东西好用就用哪个，它不是服装系统啊。嗯啥？你会不要美械吗？那肯定得要啊，是不是？
