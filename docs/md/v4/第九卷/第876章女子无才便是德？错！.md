# 第876章 女子无才便是德？错！

“女子无才便是德”这句话，是用来约束帝王女人的吗？呃，不是这样，完全不是这样。这句话自明代开始，就是受到中国文人批判的一句俗语。它存在，但存在哪里呢？在文人的笔记里面，是以批评的形式存在的。从它出现的那一刻起，就是文人听见市井当中有这样的传唱与流言，然后写下笔记，对这种行为进行批判。所以，“女子无才便是德”有没有？有，在哪里？在市井流言中。这相当于有人讲段子、讲故事，有人在戏台上面插科打诨。在这种场合下，就有了“女子无才便是德”。但它是公认的吗？不是。是引导的吗？也不是。是道德规范吗？也不是。明代的文人看到这个流言之后，专门去进行了批判，并写在了自己的笔记里面。所以，它并非中国人的文化规范，而是在民间曾经流行的一种俗语，而且是不受欢迎的那种俗语。它并不是什么真正约束贵族女子的教条——不存在这样的事情。

这种俗语流行于明代，而明代恰恰非常盛行才女文化。什么叫才女文化？就是你得有才华，你才有面子。宋明两代都非常盛行才女文化，这是士大夫阶层家庭当中非常流行的教养方法。要知道，宋明两代的女子是有公开的、社会型的社交的，包括AA制旅行团、出门旅游、集体打马球等社会公共活动，有很多。她们热爱出门，热爱交际。这些士绅家中的女子、地主阶层的女子都有这样的爱好，也有自己的圈子，喜欢出门玩耍。这就像今天小红书上的小姑娘，你要是不把照片拍好看了，你都不配开一个号。所以，当明代的、宋代的女子都流行这个时，你想想看，会写诗和是一个文盲，哪一种在圈子里面能够混得开？看看李清照就知道了。虽然李清照的才华有点过于高，太高了，高于很多男性，在整个词坛的比较中，她也是绝对的翘楚。这事儿确实也不分性别——看出来了吧，这事儿跟性别没关系。但很明确的一点是，宋明两代的地主阶层家中的女子，非常热衷于用自己的诗文进行广告招贴和社交。所有那些诗词写得不好的、出口不能成章的，出门社交时会被别的小姐妹嫌弃，属于没面子，真的没面子。

什么？李清照爱喝酒、爱赌钱？其实李清照一生当中，除了留下的诗词之外，最伟大的贡献是在金石学——也就是考古考据学上面。当年，她和丈夫非常热衷于搜罗各种各样的文物，包括碑文的拓片、孤本典籍。除了收集收藏之外，她还会进行修编和整理。以现在的标准来看，这就是一位文物工作者，也是一位大收藏家。他们两人愿意出卖自己家中原本那些豪奢的装饰品，变卖各种珠玉珍宝，然后去购买各种文物拓片。后来，她所搜集的这些文物在靖康之乱的离乱中不断散逸，最后几乎没剩什么了。但是，她留下的一部分汇总型文献还在，所以我们还是能够看到，他们夫妻两人在太平之世时，在文字、文物乃至考古的工作上，作出了很巨大的贡献。然而，由于金人的铁蹄南下，这些文化的记忆大部分就飘散了。
