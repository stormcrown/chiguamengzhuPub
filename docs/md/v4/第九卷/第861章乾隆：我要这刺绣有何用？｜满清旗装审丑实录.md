# 第861章 乾隆：我要这刺绣有何用？｜满清旗装审丑实录

他提到的这套衣裳，其实是乾隆皇帝从西伯利亚群体那里借鉴过来、挪用过来的。它并不是满族人原本创造的东西，而是从西伯利亚那边借鉴的。直到现在，西伯利亚那片土地上还有原创族群在那里生活，每年俄罗斯那边搞传统民族服装展示时，他们还会出场。这套衣裳在当地至今都有人在穿着。乾隆借鉴了西伯利亚群体的传统服饰，进行了二次创作，但非常不符合中原地区的气候，是一种刻意的文化抄袭和挪用。所以满族人自己也觉得它不好看，这很对，也很正常。

因为人家西伯利亚族群居住在苦寒之地，设计出来的东西本来就不是为了追求太漂亮、太洒脱——他们不可能追求这些。人家要求的是既能节省布料，又能密封保暖。所以别人设计的衣裳是符合自己当时当地的气候条件和生产条件的，那是西伯利亚苦寒之地的渔猎群体为自己设计的衣裳。结果硬被乾隆搬过来了，搬过来之后，为什么变得好像比西伯利亚原创的还难看呢？这又跟乾隆脱不开关系。一个东西猛然变丑，大概率跟乾隆脱不开干系。

他把这东西搬过来的目的，是因为这种苦寒之地的独特版型与中原的传统服装格格不入，在设计原理上就格格不入。所以这不是说包容就能消化掉的——根本消化不了。他就是为这个目的引入的。引入之后，最后造成的问题就是：关外群体入关之后，突然成了上等人，可以享受生活了，开心快乐。开心快乐之后，下一件事情就是享受。享受中包不包括美学享受？必须包括。我都已经发达了，以前打猎时穿得破衣烂衫没问题，毕竟老在山岭子里刮树枝。但现在住在四九城里，一排奴仆跟在后面，走路叮叮当当的，那这时候是不是该漂亮了？可是从西伯利亚搬过来的衣裳，它的版型就不是为了漂亮而设计的。那怎么办呢？

于是，乾隆执导的设计方案，有人称之为“去式留纹”（原文为“去世留文”）。意思就是：我虽然用这种独特的、完全不符合正常风格的版型，但你喜欢那些漂亮的东西，可以用漂亮的面料和漂亮的刺绣。保留它的纹理，保留它的面料。所以以前汉家服饰上那些装饰性的刺绣纹样、精美的纺织品，就开始用在这种西伯利亚风的袍子上了。这就是乾隆指示的东西。大家都觉得你搬来的那套衣裳不好看，怎么办呢？人家汉家的衣裳那么好看，你非得搬一个丑的过来，我们这些人好不容易过上好日子，我们也很难呐。不同意的话，回家之后枕头风受不了，家里的小姑娘要哭的。怎么整呢？最后乾隆的指示就是：你要好看，就把人家的刺绣挪过来。

于是你会发现，他们在西伯利亚风的服装上面堆叠上了海量的汉人传统刺绣纹样，结果变得更难看了。为什么呢？什么东西都经不起堆叠。你把各种各样的花儿都塞在一起，不讲究主次，不讲究配合，最后反而比西伯利亚原创版本更难看。它华丽了，但也难看了。所以这些满族人后来也不喜欢这件衣裳，很正常，因为它完全违背了服装设计的基本理念。西伯利亚人设计这个东西，虽然在美学上稍逊，但符合气候，是实用型的设计，是一个合理的设计——设计领域必须考虑这一点。但到了乾隆及其之后推进的这套东西，第一，你用了不合适、在审美上不足以体现美感的服装；第二，又在上面堆叠上每一个细节都好看、但硬塞到一起变得乱七八糟的纹样。然后它变华丽了，但也变丑了。

所以，这套衣裳之前，每一次有人说“旗装并不难看，旗装很好看”的时候，找出来的都是同一套衣裳。就是哪个博物馆里来着？有个博物馆展示了一套清代的、很华丽的旗装，用的也是他们流行的藏青色。他们老喜欢这个藏青色了，离谱得很。这个颜色本身很老气，中老年穿穿就算了，他们给小姑娘也穿这玩意儿，真是离谱。总之他们特别喜欢的藏青色，几种青堆叠在一起，但它的刺绣主要集中在特定的几个滚边上面，不是那么堆叠，还不算太夸张。所以那套衣服尚可，不算很漂亮，但在所有清代的袍子当中算是不错的了。每一次有人说旗装很好看，就挑那一件，为什么没别的了呢？唉，好，欢迎大家来到直播间。

好看的是刺绣和大袖。那件旗装袖子还比较宽，说它大袖我觉得不够大，但相对于一般的旗装来说，它比较宽一点，而且那一套略微能看得到一点点腰线。我不知道有多少人看过清末的一些老照片，里面有一些旗人女子的摄影摄像，你会发现其中是没有腰线的，大部分都不用腰带，用腰带的非常非常少，很罕见。而且极其宽松，没有腰线，裁切是一个大桶，哗一下下去。就这种剪裁方式，啥身材都遭不住。一般人的身材都顶不住它，没办法，这个玩意儿拯救不了，这是一个结构性的问题。

什么？有视频说“慈禧是洪秀全设计和资助的汉女，却断满清气数，美人心计”？编故事编到这份上就很无语。不行的话你先把年谱弄出来吧。谢谢木火，谢谢木火。不是这个，野史？不，这已经不是野史了，这玩意儿能当野史用吗？这不是野史，这是穿越啊，属于穿越剧，不是野史。来来来，咱们跳过去吧，我这里不讲穿越剧。
