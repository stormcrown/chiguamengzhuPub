# 第860章 为什么清能统治那么久呢？因为清知道自己很反动。 #镇压百姓 

他太反动了。他太知道自己反动了。他视所有的人民为仇敌，也知道人民视他为仇敌，所以他随时随地安插军事的庄子，在每一个村镇的节点上面安插军事驻点机构，时时刻刻地盯着你。然后用铁杆庄稼供养这些军事机构，让他们能够死忠地效命于朝廷，豢养朝廷鹰犬啊。他豢养了大批量的朝廷鹰犬，用于镇压百姓。那么为什么清能够绵延那么久呢？因为清的统治者非常清楚，他很反动。清的统治者极端清楚、极端冷静地知道他很反动，世间所有的百姓都痛恨他，所有的人民都反对他，这就是“总有刁民想害朕”。因为他清醒地知道这件事情，所以他始终都费力掌控着他的神经末梢，他的手指头。他的手指头是谁？八旗子弟。啊，他一直都在努力地掌控着八旗子弟。那么怎么掌控呢？用经济的方式来掌控。什么样的经济？铁杆庄稼。所有的八旗啊，从出生的那一刻开始，他就有一份口粮，这份口粮谁都不能够剥夺，只要活着就有饭吃。要知道，在古典时期，活着就有饭吃这件事情非常了不起，他能够拥有的指挥的能力超乎你的想象。之前我在哪看的那个，好像是某乎上面有人提的那个问题，说我要是穿越到啥啥啥的时候啊，每天都能够自动给我的兵马刷新出那个什么两份青椒盖饭，我能干什么什么。后面的回复怎么跟他说的？陛下何故造反是吧？哎，谢谢李坏啊。所以当你能够做到能吃饭的时候，在古典时期这叫什么？这叫做强组织力。而清呢，从头到尾都拥有这个强组织力，其他的时代常常没有这个啊。他是逐渐失去的。他怎么逐渐地失去自己的基层的暴力统治能力的呢？这其中有相当一部分的原因是，因为其他的正常的王朝的统治者不认为人民与自己是根本敌对关系，认为自己只是一个普通的剥削关系，这个时候只要给你减减税呀，啊，抚慰一下这事不就完了吗？正常的剥削关系一定是这样想的。包括说你在家里面觉得的那个贼老板，哎呀，已经把他当牛做马了，怎么办呢？唉，下个月给他加点薪，下个月请他去团那个建啊，下个月的盒饭给他加个鸡腿。你的老板就是这样安抚你的。古时候绝大多数正常王朝的君王就是这样安抚老百姓的，偶尔下一个税诏了啊，偶尔减免点税负啦，偶尔发放一点福利啦。啊，这个东西呢，就可以相对弥合一定的这个阶级矛盾啊。这个是干啥？这个就是正常的王朝，他不觉得百姓是他的根本敌对势力，所以他会用这样的方式进行管理。这样的管理方式是不需要随时随地都有兵马镇压的，所以正常的时期他不会使用大体量的兵马和暴力机构驻扎在每一个村镇周边，正常王朝不干这种事情。干不了，他干这个事干啥？他为了养这些兵马，他得花多少钱啊！是不是？正常王朝不会做这件事情的。于是呢，到了末期的时候，一旦出现了这种如臂使指的功能失控，唉，地方上面就会出现局部的暴动，就越来越失控，就调动不了了。但是清为什么还能够践行？他太反动了，他太知道自己反动了，他视所有的人民为仇敌，也知道人民视他为仇敌，所以他随时随地安插军事的庄子，在每一个村镇的节点上面安插军事驻点机构，时时刻刻地盯着你。然后用铁杆庄稼供养这些军事机构，让他们能够死忠地效命于朝廷，豢养朝廷鹰犬啊。他豢养了大批量的朝廷鹰犬，用于镇压百姓。
