# 第828章 非物质文化传承（创作）人

咱们一定要弄清楚时间的节点：就是日本那里的这种市井文化，无论是衣着还是餐饮，包括赏物这些文化——所有日本现在流行的所谓“传统文化”“传统符号”，它跟我们的“唐”之间的瓜葛并不深。相对来说，在文化传播上面更紧密的是宋还有明。明呢，就不光是文化了，还包括大量的物质传输。

电视剧《梦华录》是吧？电视剧《梦华录》我看过吗？我不知道啊，我看过吗？嗯，梦华剧版的《梦华录》宣传和果子啊。和果子啊，那说到这个的话确实可以聊一下——就是关于和果子的问题。和果子从它的名字上面你就知道了，它典型的是日式的点心。但是为什么这两年很多人会注意到这件事情？是因为有一部分的创作者，有一部分的非遗创作者（呵呵，我，我这个名字你有没有觉得尴尬？非遗，然后是创作者啊）。但是现在是真的有很多非遗的创作者，他们典型的特征就是打着非遗的旗号，实际上是在进行自己的一些创作——不是什么非物质的遗产，不是以前留下来的，而是他自己进行的创作。其中呢，就包括了和果子转移出来的点心。

实际上我好像时间轴记得没有那么准啊。我印象当中更加醒目的一个，并且受到了官媒推荐的，是一个女孩子说她复原了“糖果子”（就是唐代的点心），但是她实际上展示出来的作品和制作的方法是什么呢？就是日式的和果子。所以和果子置换点心的这件事情，置换到宋代其实都不算离谱，是真的不算离谱。因为宋代的时候有那种生产条件，可以生产类似于和果子的点心了。但是唐代实际上是根本没有这些物质，没有这些条件让你去做那种类型的点心的。结果呢，有一个号称自己是在做非遗产品的、做非遗传承人的女孩子，还联合了官媒做推广，就介绍说自己做出来的这个玩意儿是“糖果子”。但是唐代的这些点心到底什么样子，咱们知道吗？其实唐代由于它自己的拓边，所以对西域地区影响力很大，而西域是一个非常干燥的地方，很适合保存这些物质。所以我们在新疆地区其实挖掘出了不少唐代的点心，在博物馆里面就有啊——就是唐代的这些点心。其中呢，就包括了什么麻花、今天的那个应该叫什么？反正就是一层一层摞起来的那种酥，然后麻花、小饼干啊，这些东西在新疆的这种文物系统当中都是存在的。所以唐代吃的是什么样的点心呢？其实和我们今天菜市场里面去卖的那些传统点心非常像——就是那些烤饼子、烤饼干，嗯，然后还有那些什么油炸出来的，炸出来的像麻花、小果子这些东西，这个就是唐代人去吃的各种各样的小点心。

但是呢，那个号称复原“糖果子”的女孩子做的东西是什么呢？她就是跑去了东京（日本的东京，可不是北宋的东京），去学习了当地的点心制作工艺，然后回来打了个包，就说这个玩意儿是复原的糖果子。我也不知道她是怎么就哄出来了一个非遗传承人的名号，然后就开始大肆地推广。但是为什么这个东西它完全不可靠？因为它里面涉及到一种很重要的材料，就是白糖。因为日本那个地方呢，实际上，怎么说，就是那里的生产条件不怎么样，资源禀赋很差啊。日本那里地理狭窄，资源禀赋很差，所以从自古以来他的条件就不好，资源就不足，吃吃喝喝的东西就不够——这是他的一个典型特征。所以呢，那个地方缺东少西的，能够开发出来的点心什么的，不可能非常的丰富。我们今天所知道的那些很丰富类型的日式点心，基本上都是明代之后才出现的，出现的时期非常的晚近。为什么一定要到明代之后才会出现？因为这些典型都需要用到一种东西，就是白糖。而白糖的制作工艺是到宋的时候才开始出现，到了明的时候才开始大规模的推广，并且进行了白糖的外贸。在唐代的时候，人们制作点心是没有白糖的，会使用的是红糖和麦芽糖。而红糖和麦芽糖做不出和果子那种东西来，完全不具备这种条件。所以你说那个和果子这样的类型的点心，它能不能够在宋出现？其实存疑，是不能够说他一定在或者一定不在的。因为宋代给我们留下来的很多的菜谱，它在技术水平上面是有可能的，那么在他的这种制作上面是不是有真实的这种款样的存在啊？这个就不敢保证。因为有很多的这种点心呢，它是写了他的名字，描述了一下它的外观，然后讲了一下大概的制作流程，就是这个样子啊。它里面会有一部分和今日的和果子相似的东西。为什么？因为日本那边，他的很多的点心确确实实是跟我们这边学习了，甚至很多原材料就是从我们这里引进的。但是同时呢，他也二创、三创出了他自己的本土的内容。

可是咱们一定要弄清楚时间的节点：就是日本那里的这种市井文化，然后无论是衣着还是餐饮，包括赏物的这些文化，所有的这些日本现在流行的这种所谓的传统文化、传统符号，它跟我们的“唐”之间的瓜葛并不深。相对来说，在文化传播上面更紧密的是宋还有明。明呢，就不光是文化了，还包括大量的物质的传输。因为啥？因为那个日本那边发现了银矿。日本那里有大的银矿。我们知道明代的时候我们设立了银本位的金融体系，银本位就是拿白银作为基础的一般等价物，把白银作为标准货币来使用。当你用白银作为标准货币的时候，而我们本土又严重的欠缺银矿，那么拥有银矿的地方就会拥有强购买力。当时的日本呢，因为有银矿，所以他的购买力就提高了，虽然生产力还那样，但是他购买力提高了。所以在明代的时候，实际上我们有相当一部分的海贸是运到日本去的。那么在明代的时候，日本那里确确实实会获得相当一部分明王朝的物产。物产足够多了之后，他就更加有可能去开发出一些更加精细的产品。而在唐代的时候，这件事情日本还做不到。他受宋的文化影响很深，同时受明的物质与文化影响很深。基本上说，今天日本岛上面的传统的、号称来自中华的文化，大部分是来自宋明两朝。
