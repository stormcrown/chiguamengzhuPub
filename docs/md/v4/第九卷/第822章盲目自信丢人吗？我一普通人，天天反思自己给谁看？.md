# 第822章 盲目自信丢人吗？我一普通人，天天反思自己给谁看？

### 整理后的易读段落：

所以，现在的中国人到底需要什么东西？  
我们瞧不起那些什么盲目自信，也瞧不起那些“赢学”。但是，不管是“赢学”也好，盲目自信也好，它们身上的这些闪光点，恰恰是今天的中国人所缺少的东西。我们身上的这块拼图不完整，我们需要它。所以不能说：“哎呀，它就是不好，就是不对”，然后全面批判。我们不能那样做。我们天天要反省自己，“吾日三省吾身”。但日三省吾身的人是谁？嗯，啥？我有吗？我没讲那个……嗯，你在说那个外国人对吧？啊，你在说那个外国网红，是不是？好的好的，我没有怎么看他的直播，也没有怎么看他的视频。所以你一定要说发生了什么，中间又怎么样了。我在网上不怎么看这些信息，确实也不知道具体干了什么，我不清楚。

但是，有一件事我想说：我对老外未必那么了解，但对于中国人或者互联网传媒业，我比较了解。其实，当你问“甲亢哥又怎样怎样了”、“李美月又怎样怎样了”，当你不停地提这种问题的时候，在我们的心灵潜意识中，其实都有这样一种符号——就是这些人的点评对我们很重要。他说得不好就不行。这种非常在乎他人点评的状态，在中国人身上表现得非常典型。大家不要太忽视这种心理状态。这种心理状态实际上很特殊，在人类群体的范围内是非常特殊的一种。正常的心理状态是什么？你说我不好——你凭什么？你说我不行——你哪位啊？这才是正常的人类心理状态。谁有呢？你会注意到，除了中国人，除了东亚文化圈，绝大多数地方的人都是这个样子的。

现在大家是不是天天在那里说“赢学”？那“赢学”这个东西里面，包含了一个根本的逻辑：首先，因为我是我，所以我赢了；剩下的一切，都是为了赢而找的借口和理由。当我们去嘲笑它的时候，也同时在中国人的心里埋下了一个种子——就是我们不能自视甚高，不能盲目自信。当你随时随地觉得自己又赢了的时候，你已经变成了一个笑谈。在中国人身上，这个负担很重。你说这种思想有没有帮助？有的：经常性地反省和反思，能提高自己的思维敏捷，对我们的社会发展进程大有裨益，能让我们更加客观、真实地面对世界，更加保守地评估自己的能力，避免冒进。但同时，它也有不可避免的短板——那就是大家普遍对自己比较怀疑。而这件事，会影响到方方面面。

谢谢“沧海连浮云”。这种对自己的无条件怀疑，会影响到我们生活的方方面面。就像来了一个外国网红，严格来说，他原本的主战场对于中国的绝大多数人来讲非常陌生。绝大多数的中国人，包括互联网的网民，实际上不会天天去外网上面看他的那些视频和直播。真正经常跟他有互动的人很少。但他只要来了中国，就会出现大家一起来围观：“哎呀，你非得说点什么”、“我每天都在考究你的态度”，甚至一个眼神都要揣摩一下他的心态是什么、讲不讲礼貌、心里面是不是善良。这种过度关注是一种问题。嗯，这种过度关注是一种问题——它是“赢学”的反面。但是“赢学”很可笑，不意味着它的反面就绝对正确。谢谢“惠”，嘻嘻。为什么？因为这个世界上绝大多数的人，如你如我，一般都是平平无奇的普通人。平凡到什么程度？就是走到街上的时候没有人会认得你；在单位里打工的时候，隔个一两年，老板才终于记得住你；在班里念书的时候，班主任只有在拿到花名册时，才会想起来班里还有这个人。大部分人都是这样平凡又普通的。

那么，当一个人心里永远都有这种质疑自己、反思自己、反省自己、提醒自己千万不要太自信的心态时，对于这些绝大多数的平平无奇的普通人而言，麻烦就来了。因为他本来就普通，本来就丢在人堆里找不出来，结果你还告诉他不能盲目自信——他有什么条件盲目自信？他压根就没有机会盲目自信好吗！结果你还要敲打他说“你不要太跳脱，你不要太自信，容易如何如何”——他从来就没有这个基础啊。这就造成了你死循环一样的自我怀疑。大家会注意到一件事：在中国以外的绝大多数地方，那些其实比我们条件差很多的人，拥有非常非常强烈的自信感。典型的一个大家经常瞧不起的例子——隔壁的巴拉特（印度）。巴拉特人身上有一种很好用、很好用的品质：他不管发生了什么事情，永远能够在自己身上找到优点、找到闪光点。这能提升什么？大家记得那个“矮大紧”（高晓松）当年吹嘘巴拉特的贫民窟时，他说：“这个地方虽然贫穷，但是叫什么来着……平静还是什么玩意儿？”他用的什么词我忘了，总之就是说这个地方的人非常平静地接受这个现实，并且过得很不错。我们是不是在嘲讽这件事情？对，应该嘲讽他过那么苦的生活，然后还觉得挺不错的——这件事情值得嘲讽。可是，他之所以那么苦还能够活下来，就是因为他对自己有着一个极好的个人认知。这个好的个人认知本身，对于一个人的心理健康和心理防线非常重要。而中国人在这个事情上切切实实有短板。

你要是说“我们自古以来就这样，这就是传统”、“我们能够这么强大，就靠这个”——那还真不是，真不是这样。中国人以前也是拥有极其豪横的自信的。事实上，这种极其豪横的、近乎盲目的自信，才是中国人绵延至今的底牌。我们想一下，回忆一下：是不是当年我们曾经遭遇过五胡乱华？五胡乱华的时候，中国人惨不惨？遍野无人，变成了一盘菜。到了这样的地步时，请问我们自己的传说、故事、史书，除了叫惨叫疼之外，有没有说自己很烂、不行、是个废物？没有。我们只说对方是“蛮人”。这是什么？这是在极大的失败面前，依旧保有坚定的自信：“虽然他强大，但他野蛮；虽然我现在失败了，但我才是那个文明体。”中国人自古以来就有这个。我们怎么扛过一波又一波的低谷的？靠的就是这种坚挺的自我相信的信念感。我们在蒙元统治了近百年之后，大明王朝又重新卷土重来，重塑华夏之后，说什么“胡虏无百年之国运”。而我们对于过去的这段历史的总结，叫做“虽然他强了，虽然他抢了，虽然他沉了，但是——胡虏无百年之国运”。这是什么？这叫做盲目自信。这种东西，就是那种无条件的、不找理由的、不找任何借口的自信。这种自信才能让我们熬过低谷期。

中国人最悲催、最猥琐、看上去都变得丑陋和矮小、生活衣食无着的满清时期，还保有一种比今天的人更高的底线。什么底线？我们把外国人称为“红毛夷”、“洋鬼子”。不要小瞧这种命名。这种命名是什么？意思是“虽然但是”——洋人还是丑。不管洋人干了什么，不管他穿得多好看，不管他长成什么样子——它丑啊，直觉上的丑陋。这是什么？这是我们依然保有自己的审美话语，至少在面孔上面保有自己的审美话语。这是一个族群跌落到底线之下之后，还保有的一种自我认知。请问这种自我认知，在今天的互联网上还能够畅行无阻吗？真的不能。如果今天这些自信依然畅行无阻的话，这个互联网上就没有我生存的空间，我就得改行。为什么会有我的存在？就是因为今天的很多中国人，甚至对于自己的基因都开始质疑：觉得自己长得难看，觉得自己腿短，觉得自己皮肤暗黄。有没有这样的？有没有觉得自己鼻子塌的？觉得自己眼角不够开阔的？在美容院、整形医院里一刀一刀地割。这是什么？这是丧失了自我认知的一种表现。

所以，现在的中国人到底需要什么东西？我们瞧不起那些什么盲目自信，瞧不起那些“赢学”。但是，不管是赢学也好，盲目自信也好，它身上的这些闪光点，正是今天的中国人所缺少的东西。我们身上有这一块拼图不完整，我们需要这块拼图。所以不能说：“哎呀，它就是不好，就是不对”，然后全面批判。然后呢，我们绝对不能那样。我们天天要反省自己，“吾日三省吾身”。日三省吾身的是谁？饭没吃上呢。三省吾身的第一省，就是为啥今天没有米。每个人处在不同的位置上，会有不同的心理需求和客观诉求。咱们在这个事上就不能混为一谈。那么到了这个时候，是不是刚才在那里说那个甲亢哥的事情？我知道这个人，但是我只看到过他的表情包，他的直播、他的视频我都没有看过。不是这个人有或者没有问题，而是这种过度关注、这种过度在乎——它本身就是一个问题。
