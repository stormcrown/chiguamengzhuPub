# 第899章 青铜蒸器证酒源

蒸馏酒，一样来自中国啊！对，对，对，是的。这个之前讲过，就是那个蒸馏酒。之前呢，一直默认蒸馏白酒是西来的。为什么这样默认？倒也不是完全没有依据。主要的原因呢，是我们中国流行喝白酒是在元代以后。确实，这个风俗——这种爱好、这种偏好——是蒙古人带过来的。他们喜欢，所以后来开始形成流行，慢慢地越来越多的人开始喝白酒了。所以很多人就默认地觉得（谢谢小鱼郎啊）这个蒸馏酒是西边来的——无论是哪个“西”，总归是蒙古西征之后带过来的成果。

但是呢，最后证明了一件事情：蒸馏酒是我们本土产的，只是我们不喝。但我们确实能蒸啊！这个是在汉墓当中发掘出来的青铜蒸馏器。这套蒸馏器早就发掘出来了——就是海昏侯墓嘛，早出来了。但是出来了之后呢，绝大多数的考古工作者和相应的所有工作人员都认为它不是拿来蒸酒的，因为我们一直都认为自己不喝蒸馏酒啊。所有人——当时主流确实认为中国人不喝蒸馏酒，哪怕看到了蒸馏器皿，也不觉得是拿来蒸酒的，因为咱不喝嘛，你蒸它干什么呢？所以呢，一直默认这个墓中的青铜蒸馏器是拿来蒸药材、蒸花露的。就这些用途啊，我们猜想了很多种，也确实存在这种可能性，但始终都认为它不是拿来蒸酒的。

后来呢，我们正式把它复刻了一套，做了一套，然后不断地做实验，说这个玩意儿到底拿来蒸什么，会出来什么效果。我们争了好多东西，实验人员做了各种各样的试验，最后发现原来这玩意儿是蒸酒的呀！它蒸酒效果最好，它蒸馏效果最好——你知道吗？它就是拿来蒸酒的。所以海昏侯墓、汉墓当中，我们已经有了清楚的蒸酒设备。所以那个时代，至少从汉代开始，至少从汉代开始，我们就已经有了非常清晰的蒸馏酒的技术和经验。这些东西都有，但是确实没有形成习惯去喝这个东西。所以不排除那个时代我们有蒸馏酒，但是把它用于医疗用途，而不是拿来用作饮用。

嗯，酒在以前是入药的。对，酒从一开始酿造的时候，就是拿来入药用的。酒从一——从它最初的时候，就是利用它身上这种特殊的、疏通血脉的性质拿来入药用的。所以完全不能够排除，在我们汉代的时候已经形成的这种蒸馏酒，是为了入药而使用的，是为了医疗用途而使用的，却不是拿来喝的。拿白酒拿来饮用，确实是从元代之后才风行起来的。只是以前我们的学者认为它是西征之后学过来的，而现在呢，可以有一部分学者认为说，这个是在中国这里，他知道了这个东西味儿挺冲，他正好就喜欢味儿挺冲，于是就用上了啊。
