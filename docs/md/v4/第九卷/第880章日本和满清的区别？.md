# 第880章 日本和满清的区别？

他说：“女真入侵与日本入侵没有区别，所以为什么要反日？你可以跟他说：不光反日，还可以反清。不光反日，还可以反清。中国人持续300年，持续不断地反清啊。从满清入关的那一刻开始，中国人民就一直集结起来反清，只是没有成功而已。失败的英雄依然是英雄。中国人持续反清200多年，直到1911年取得成功。既然我们连清都要一反到底，那为什么不可以反日？所有欺压了我们的、所有蹂躏了我们的，都可以反。而且不光今日的各民族要反，满人也要反清。因为你不反清，你就不是满族同胞；不反清就是敌对势力。就是这样简单的事情啊。当年辛亥革命的时候，就有八旗子弟投靠革命军。为什么他要站在正义的一边？八旗子弟当中也有个别学好了的人，也有学好了的人，投靠了革命军，他就成为了我们的同志。49年之后，大家一起接受再教育，洗心革面，重新做人。所以，当年的八旗能够转身成为满族同胞的前提，就是咱们一起反清，占了同样的革命立场。”

**备注：** 修正了以下词汇：
- “脚盆入侵”改为“日本入侵”（“脚盆”是日语“日本”的谐音，此处更正为通用说法）。
- 调整了部分口语化重复（如“啊”保留但使语句更连贯），未删减内容。
