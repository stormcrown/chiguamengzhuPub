# 第821章 满纸荒唐言，一把辛酸泪”最契合开篇诗的红楼癸酉本。

啥玩意？说一下。鬼友本红楼啊哈哈。鬼鬼有本红楼。哎，我这里还真有这个书啊。但是，但是，但是非常明确的一个——但是啊，这本书不好看。你看我在这带货啊，我在带这本书，我在要卖这本书，但是我首先先告诉你：这个书不好看。

不好看到了什么程度？你如果没有非常强烈的欲望，一定要知道某一种谜底的话，你看了整个脑袋都是疼的，整个心脏都是炸的，毛孔都能崩出来。看了之后就觉得这个写手是干啥呀，有点什么毛病吧，就这种心态啊。这个书差到什么程度？就差到这个程度啊。阅读难度极大，阅读难度极大。

为什么难度极大？是他说话特别的纠结吗？不是啊。是这个玩意儿吧，他那个节奏把握跟坨屎似的。前一句还在那里吃饭呢，后一句啊，他就开始在一个园子里面开始表演战争片儿了啊，花园里面给你演战争片，攻城略地。你能想象得到不？所以这本书你要只是想猎奇，你想看本书，别看，痛苦死的，难看的不得了，非常非常难看啊。

但是转折来了啊。但是如果你看红楼的时候，意识到这本书里面有谜语、有秘密、有隐藏的线索，然后你又想找谜底，你想找一个密码本儿，你想找一下这个书——红楼这个书到底在讲什么，你想去听一听红楼它还有什么背面的故事。你怀着这种探索之心的话，那这本书就应该读啊。这个书就是起这个作用的啊，心里有点数。这个书的作用就是当密码本儿用的，你不要拿它当小说看。当小说看的话，会把你疼的浑身上下都难受。为啥？因为这本书呢，完全没有经历过润色，甚至里面的有一些段落很可能被刻意的精简和粗俗化了。啊，因为它的书本身的来源也非常的成谜啊，宣称的这种来源很可能有问题，背后另有话题，这个事不好说。

但是呢，这个书啊，它有一个好处，就是你在前80回这个红楼的故事当中，看到的那些让你疑惑的东西，让你觉得后续发生了什么事儿的这些东西啊，让你感觉到里面有秘密的东西，在鬼友版的这一套书里面有谜底。它告诉你的答案，你能够看见答案，而且你会发现这个和它开篇的谶语之间可以达成呼应，它可以形成逻辑闭环啊。它的宝贵之处就在这了。它形成了逻辑闭环，它把整个书里面的架构和人物全部都收拢起来了，整个的故事的脉络就完整了。你就不想看那个断了尾巴的壁虎，你想找到最后的尾巴根儿到底长什么样，你就看这个。它里面有谜题的答案啊。

所以呢，这个书有没有用？有用。如果你对谜题有兴趣，应不应该看？应该看。但是它好不好看？异常的难看啊，那个故事写的跟屎一样的啊，就到了这种地步。

所以这个书应该是什么？这个书应该就是在红楼的创作的过程当中啊，红楼的作者应该是一个集体创作，不是哪一个人。有人书写了梗概，然后有人为它填肉，有人为它润色，有人为它进行掩饰，把里面那些露骨的东西遮掩过去，描述成一个正常的市井小说啊。这个呢，就是我们前80回的完整的、优秀的红楼。但是后面的鬼友版的这个续上了这个调啊，属于什么呢？没有皮，没有肉、没有毛，他就是骨头，纯纯的骷髅架子啊。看上去很鬼畜、很吓人，但是呢，骨骼走向是对的，他就是这么一本书啊。

嗯，看前80回就行了。如果你是看小说的话，看前80回就行了啊。如果你是在看小说、看故事，那前80回当然好了。但是呢，红楼它不是一个单纯的故事书，这个书里面有很多隐藏的线索，有很多隐藏的谜语，包括它的别名《风月宝鉴》。那这个镜子——风月宝鉴，这个镜子也特意特意的提醒你了：它分正面和反面，一面是红妆，一面是白骨。但是它让你看什么？它让你看白骨。什么东西的白骨呢？白骨如山忘姓氏。所以整个书里面，它本身就是包含了隐喻，包含了谜题。所以你需不需要一个密码本进行解析呢？看你个人的需求。你只想看红妆啊，公子与红妆的部分可以，没有问题。我们对人没有这么高的要求啊，你可以只看公子与红妆，你可以只看家长里短，看吃吃喝喝都行。但是当你在意这个故事还有一个背面，那么什么是“白骨如山忘姓氏”呢？他绝无可能是某一个包衣的家族史。任何一个奴婢的家族都配不上“白骨如山”，更何况曹家当年压根儿就没有到屠满门的程度。他们家族谱到现在都还在呢，后代都还在呢，哪来的屠满门呢？连屠满门都没有的时候，哪来的白骨如山呢？

所以这本书啊，红楼这个书，他就不是曹家公子的自传，它是一个集体创作，它是集体创作，不是哪一个人单独书写的。所以它的背面的故事是什么？当你看了鬼友的密码本的时候，就会意识到，他很明确的是在讲明亡清兴的故事。而在这一段故事当中，无论是千红一哭还是万艳同悲，是不是都可以解释了？如果只是家里面表哥与表妹的那点爱情故事，她配得上这些词吗？配不上的。家里面查抄了一点金银，他配得上“白茫茫大地真干净”吗？配不上的。整部书在前半段所书写的，无论是谶语还是故事的伏笔，都远远不是家长里短、市井流长。那么这个密码本就有价值，你用这个去看前80回的时候，就会发现原来里面还有其他的故事。就像林黛玉的谶语当中，“玉带林中挂”，玉带是什么？玉带是谁都能用的吗？玉带是天子的腰带。“玉带林中挂”什么形象啊？自个儿想啊，自个儿想他是一个什么样的形象啊。

所以呢，呃，所以放到鬼友本的这一套书里面啊，大家心里头就要有点数。虽然它很难看，但是呢，你可以把它当做密码本来用，因为它里面揭露了很多的谜题。你把这个谜题拿去对照到前面的时候，你就会发现很多事情豁然开朗了。所以它大有价值，可大有价值啊，大有价值。

嗯，八大山人是明的皇族吗？当然是了呀，当然是了呀。朱耷呀。啊，你看了他的翻白眼的鱼了吧，啊，他的鸟儿，他的鱼都不太正常是吧？啊，他当然是了。他很清楚的啊，这个人的身份还是非常清楚的。
