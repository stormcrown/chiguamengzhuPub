# 第844章 癫狂非浪漫丨没毒时代的艺术遗产正在毒害当代审美 东西艺术观

他们以梅毒为荣？那倒不是。你不要以为真的以梅毒为荣，而是因为这个病太普遍了，大家都有。得了这个毛病之后，还会有一个副作用：它会让人思维特别混乱。思维混乱这件事会带来什么呢？它会让人觉得特别有艺术感。

中国人的艺术是不允许你思维混乱的。注意，我们的艺术要求你极有逻辑性。你画一幅画，要有主题思想。古时候的绘画产品都要有主题思想，画画也要传递你的意志，它不允许你逻辑混乱。一朵花放在这里象征什么？花和蝴蝶为什么要放在一起？猫和蝴蝶为什么要放在一起？都是有逻辑性的。我们的艺术品是有逻辑性的。

但西方人不是。那个时代，他们追求的就是那种迸发式的、颠覆想象的、和正常人不一样的东西——越异常就越浪漫。所以，梅毒这个毛病在那个时候被认为：艺术家要是没有它，就没有想象力。于是就出了这种颠三倒四的东西。所以我就觉得，他们的艺术本身也有问题。现在艺术又被他们定义了，这点就很糟糕。我们今天被他们定义了，这是很恶劣的影响。

什么叫做艺术？艺术不代表美，不代表理性的传播，也不代表成长，不代表天道，不代表善意。艺术代表特立独行、与众不同——扎眼睛。如果把艺术定义为扎眼睛，越扎眼睛越艺术，你看不懂就是你文化水平不行，你不艺术。这个定义权一旦被人家掌控，那些颠三倒四的、乌七八糟的、影响人心灵意志的东西，就会开始不断出现。现在的很多现代艺术，尤其是那些乱七八糟的流派，是一个特别祸国殃民的玩意儿。

有艺术家说：这个人又胡说八道了。注意，艺术有没有价值？我在这里说的就是艺术有价值，艺术非常有价值。它是直接影响人类情感中枢的。因为它特别有价值，所以完全不能容忍它变成一个——专门越丑越好、越爆炸越好、越扯淡越好——那就不行。因为它对人的价值太重要了，它对人的心灵的影响也太严重了。所以它必须做正面的贡献，要帮助人类成长，帮助人群建设，这变成了艺术家的职责，而不是标新立异。

但是西式的现代艺术当中，极其地求新、求怪、求变、求与众不同。它对那些美感的东西、协调的东西，甚至是嫌弃的。你画的东西太顺畅、太丝滑了，它会说你没有创意。这个东西是错的。我喷的就是这玩意儿。你要是觉得你必须画这个玩意儿，你就已经投降了，你已经向西式艺术圈投降了。你这是艺术家吗？你是别人艺术圈儿的代言人，是艺术买办，是艺术走狗，那就不再是艺术家了。

什么是艺术家？艺术家是对人有责任感的群体。他要用他的笔，要用他的音符，来引导人、抚慰人，让一群人走向更加健康、阳光的未来。我们走向星辰大海的路途当中，应该有它的BGM存在。但它的作用是什么？是引导人心、抚慰人心，而不是让人神经错乱。现代化的艺术当中，一大堆是让大家错乱的玩意儿。你做这种艺术的，就不要说自己是艺术家，那妥妥的就是一个艺术买办，艺术走狗，那就是西式艺术圈的鹰犬啊。我喷的就是这么狠。你过来恨我就恨我吧，无所谓。我现在又得罪了一个圈子，嘎嘎有钱的一个圈子。完了，货又卖不出去了。穷？我让你穷，活该啊。那么有钱的圈子你就得罪他。
