# 第843章 汉文明的终极杀招：拒绝丛林法则 VS 当代丛林法则驯化实录！ 

实际上，除了我们中国，世界上的绝大多数国家——包括所谓的欧美文明国家，包括现今那些号称自己站在文明尖端的国家，也包括那些看似彬彬有礼的国家——全都遵循丛林法则。只有中国人不是这样。一定要说的话，可能我们附近几个过去与我们渊源极深的国家，还残留着一定的影响，这些是我们当年推行王道教化后留下的痕迹。除此之外，其他地方全都是丛林法则。你看到他们文明礼貌、讲道理，但这并不妨碍他们奉行丛林法则，他们本质上就是丛林法则的。

你看最近几个月我们与美国的大对仗，再看看美国那边人们的日子——我们姑且不论他们是否水深火热，但他们的那种生活方式，就是典型的丛林法则下的生存方式：赢家通吃，穷人一无所有。而且你会发现，美国人民是最能忍耐的。所有相信丛林法则的人，都是能够忍耐的。世间有很多人群，其实都比中国人能忍。以前很多公知喜欢说“中国人就是能忍，汉人就是软弱无能、特别忍耐、特别苟活”，但实际上，除了汉人以及被汉文明辐射到的汉文明圈，其他群体都特别、异常地能忍，非常能忍。为什么这样说？因为他们相信丛林法则，就会特别能忍——输了就是输了，他们只会臣服于强者。

我刚才说到南宋的时候，是不是提到了蒙古下层的奴隶投靠南宋？为什么会这样？当时的蒙古兵马在欧亚大陆上特别强大，但就是这样强大的蒙古兵马，在对南宋下手时，却有很多底层的奴隶悄悄投靠了南宋，跑了。为什么？因为他们内部是一个奴隶制社会。奴隶领主喊一个口号：你必须跟我一起出去打仗，而且要自费备马、备马鞍、备兵器、备所有食物和水。打完仗后，你可以分一部分好处。蒙古军队集结速度之所以那么快，就是这个原因。但他们为什么能够乖乖跟着去打仗？是因为吃不上饭、主动跟着干吗？不是。是因为奴隶制有非常强的人身依附关系：你家不派男丁出来打仗，家里老弱就别想活。这是一种强人身依附的奴隶制管理方式，所以反而能迅速驱动大批人跟领主出门干架。机动性和后勤补给速度非常之快，集结速度快得不需要预备粮草，直接就出来了，自带马匹、自带兵器。你相信我们这里的军队还能干这事？我们不得把一切都备好了才上路？人家就可以做到。为什么？因为不拿下层当人，弱的人就应该跟着强的人走。这就是丛林法则。奴隶制治下一定会灌输丛林法则，所以下层的人不思反抗。

后来为什么很多奴隶会投靠南宋？因为他们在不断的斗争中，有一部分奴隶发现对面的人活得比自己像人。体会到了这些之后，就有人逃跑，跑到了南宋这边。其实南宋经常性地吃亏，他们难道不知道吗？但还是要跑过来。为什么？因为当他意识到自己也有可能成为一个人的时候，他就不想当奴隶了。后来的蒙元时期，蒙古王公日子过得非常好，被称为“一等人”。但一等人、二等人、三等人、四等人下面还有奴隶，这些奴隶绝大多数都是蒙古底层牧民。在蒙元时期，他们的底层牧民被王公当成奴隶来贩卖，甚至有国际奴隶贩卖。所以后来朱元璋“驱逐胡虏，恢复中华”之后，为什么会说“归我者永安于中华”？因为他把蒙元朝廷赶回草原后，实际上有很多下层牧民留下来了。他们干嘛要留下来？尽管日子也是平民生活，不见得多好，但他是人，不再是奴隶，不再是任人贩卖的奴隶了。所以才会说“归我者永安于中华”，才能收拢这些人回来。

主要的原因是什么？我们这里是一个人的社会。尽管后来满清几百年不断编撰谣言，使你以为宋明是黑暗没落、没用废物的时代，但事实上，我们可以从历史的碎片和角落中印证出来：在这两个时代当平民，日子总归过得尚可。这是什么？拿人当人的时代和拿人当奴隶的时代，是完全不同的思维模式。当一个人生活的环境是奴隶制，他必然要接受这种丛林法则——强的有理，弱的没理。当你坚持奴隶制的丛林法则时，你的忍耐力就会特别强，因为你根本不知道人过的是什么生活。这就是“为人奴婢而不自知”。这种情况在清末也出现了，这就是鲁迅先生哀叹的中国人身上的劣根性。因为他所见证的就是清末民初这个时代，中国人身上的迷茫、麻木、无助。为什么这个时期这样？因为中国人在那个时候经历了200多年的奴隶制洗礼，根本翻身无路。当你压根儿不存在上升可能性的时候，你怎么会有梦想？当你没有这些梦想时，那当然是闭着眼睛当牛马是最好的选择，至少心灵上没那么痛苦。清末的人为什么会麻木？因为你不麻木就会反抗，反抗了就会死。所以鲁迅先生当年哀叹的中国人身上的这些劣根性，来自于哪里？来自于满清200多年奴隶制的奴役和打压。而只要你在这种打压下持续生活并习以为常，觉得打压是正常的、反正天天如此，你就必然会进入丛林法则的框架中去——谁赢了谁就是这里的主人，谁赢了谁就是我的老大。这时候你已经没有正确的是非观了。什么是正确的是非观？正义的那个人才是对的，哪怕失败了，失败的英雄也是英雄，成功的贼子也是贼子。这就是朴素的正义观。如果我们不能坚持这个，那么谁赢了谁就是老大，谁赢了谁就是对的——这就是丛林法则。

这几十年是不是有很多公知就是这样？他为什么天天说美国对，天天说欧美文化先进？因为他们发达，所以就是对的。这是什么？这就是奴隶心态、奴婢心态，没有正确的是非观，只求强弱胜负。只要一个群体是这个样子的，他不可能上进，也不可能发动真正的革命。因此，我们后来为什么会革命成功？因为我们启迪了民众，因为我们唱起了“不愿做奴隶的人”。你不想做奴隶，你才会革命；你不想做奴隶，你不甘心做奴隶，你想要正义，你想要公道，你才会站起来。这就是中国人之所以在今天勇立潮头的根本原因。所以，当有人跟你说“你看人家发达了，你看人家有钱了，你看人家住的房子，你看人家的公司，所以他说的是对的”——只要你心里相信这玩意儿，你就只能成为狗。也有可能戴两块智能手表，但也不是一个人狗能戴两块智能手表的。这事儿你们接住梗了是吧？厉害了，连这个梗都接着了。我下次不说他了，都被你们发现了。
