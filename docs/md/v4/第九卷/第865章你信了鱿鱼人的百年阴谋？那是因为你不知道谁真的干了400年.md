# 第865章 你信了鱿鱼人的百年阴谋？那是因为你不知道谁真的干了400年

这种使用阴谋篡改历史、修改他人记忆、欺骗他人感情、制造一张密密实实的网、愚弄他者、破坏经济的秩序和构成结构、破坏大家的信任感、使人们对自己没有清晰认知的事情，有没有人做呢？那还真有人做。我不想传播这些信息啊——就动不动说什么“由于又怎么怎么地了”，由于是母系认同，然后什么“鱿鱼布局”，布局几百年啊；什么由于“由于”布局了几百年，然后由于给我们这个会、那个会，拉拢了什么什么人，又在哪个地方干了什么事——几百年阴谋啊！几百年的阴谋，我们都被蒙在鼓里了。幸亏我开了天眼告诉你啊，不然你都不知道，你得被他骗到要死。那这个呢，就是标准的阴谋论的部分，标准的阴谋论。

来啊，我们就问你一件事情。就是这个什么鱿鱼人，这群人——他们有几个人？是不是人特别少啊？第一件事情就是人特别少。第二件事情，是不是走到哪被人打到哪？这两件事情总是真的，对吧？这两个东西是真的。一群人积累了很长很长时间，按照他们自己编造的故事，他们有2000年的历史了。这群人，然后呢，他们一直在流浪，一直居无定所，走到哪里被人打到哪里，而且只要被打就毫无还手之力，被打了就只能逃跑。这就是他们的身家履历，毫无疑问。那么这个鱿鱼人这群人，你不管今天是不是真的有很多大老板认同自己是鱿鱼人这个身份，但这群人的数量非常少，而且自古以来居无定所，没有能力掌控自己的土地，没有自己的国家，没有自己稳定的居所，而且走到哪里就挨揍。然后我就问你一件事情：就这样的人群，你说他老牛了，全是阴谋，我们所有人都被他骗了——有合理性吗？我就问你，这个事有合理性吗？你有没有觉得这几样东西之间绑不到一起去？一群人流浪了2000年，几十年前才找到了一个定居的地方，之前就是走到哪里被人揍到哪里，然后他光是埋下这个什么陷害全世界的伏笔，都埋了好几百年，而且没有人知道？逗什么闷子呢？嗯，逗什么闷子呢？这个事儿啊，这个事儿做不到，你知道吗？这是一个实力问题，这是实力问题。物质决定意识，你没有这个实力，这些事情你是做不了的啊。这是第一件事儿啊。

然后第二件事情：这种使用阴谋篡改历史、修改他人记忆、欺骗他人感情、制造一张密密实实的网、愚弄他者、破坏经济的秩序和构成结构、破坏大家的信任感、使人们对自己没有清晰认知的事情，有没有人做呢？那还真有人做啊。这种事情是有人做的啊，这件事情确确实实是有人做的。谁呢？中国人老熟悉了。你们自个儿说啊，就我刚才描述的这样的类型的操作方式——谁啊？谁？你说呢？篡改历史档案，消灭历史印记，制造全民文盲，拖延科学技术。谁使人们从美变成丑？谁做的这一套操作持续了多久？有没有300年？他自己亲自动手的有300年，然后如果说包含他后续时间的漫长操作手法，到今天多少年啊？我算数不好啊，是400年还是有400年了吧？是不是400年阴谋啊？400年啊！400年！然后这个故事被嫁接到了鱿鱼人身上。你说那些人是好人吗？我从来不说他们是好人，我不觉得他们是好人。但是他们的无能程度不足以做这件事情。没有觉得他们是好人，我认为他们是一群无能而且心眼很坏的家伙，但是因为他无能，所以这些事他做不了。然而有人能做，凭什么去做的呢？因为他窃居神器，以落后的身份占据了一个庞然巨物，然后啊，他利用这个国家原有的影响力与框架，借壳上市去进行后续的操作。所以它本体虽然没多大的力气，但是他借的这个壳子足够的大，所以他才能够做后续的这些事情。你说阴谋有没有？有系统性地摧毁一个历史悠久、漫长的文明体的记忆，这不是阴谋是什么呢？当然你也可以说他是阳谋，因为对方烧书的时候是直接烧的，毁书的时候是直接毁的，篡改的时候也是直接篡的。所以有没有人做这些故事？有。是鱿鱼吗？不是。所以老是有人在那里讲啊，鱿鱼人的千年秘密，他的千年阴谋——谁的阴谋？凡是把这个什么鱿鱼人吹得天大地大、他最大的那种，一方面啊，他制造了一种恐怖感，让你觉得这群人好厉害、不可战胜；另外一方面就是在为清洗者开脱：既然坏事都是鱿鱼干的，那清洗者肯定没那么坏嘛。但是这些事儿是谁干的呢？
