# 第869章 全世界都在 认新爹 时 ▶ 为何只有中国坚持了下来？

数典忘祖。  
对，为什么我们对于背弃祖先这件事情特别地看重？你有没有发现，我们自古以来对于背弃祖先这件事情就特别地看重。前几十年的时候，很多的公知啊，把这个东西呢，称为中国人的保守落后，认为呢，老是看祖宗怎么做的，叫厚古薄今，这样则会妨碍大家的进步。而实际上是什么？为什么我们自古以来就非常在意祖先的看法？当你出征之前要说“列祖列宗在上啊”，做了事情的时候，要看自己还有没有颜面去见列祖列宗。为什么要在意祖先的看法？为什么不准你忘记祖先？因为我们这个群体的来源就是依赖于祖先的。祖先崇拜、祖先溯源、同宗共祖——我们就是依赖于这样的信念感而存在的一个群体。它既有血缘上面的归因，也有文化上面的归因。但是血缘归因是指谁是我亲爹，文化归因指的是我认为谁是我亲爹。所以在我们的这个族群构建方面，非常非常在乎的，就叫做敬天法祖、同宗共祖，在乎的是500年前是一家。这个呢，是我们从一开始确立文明开端的时候，就决定了一个底色。  

甚至可以这样说：其实在全世界任何群体当中，最正宗的、最传统的、最古朴的族群凝聚方式，就是祖先认同。在全世界范围内其实都是一样的，没有区别。包括你到漠南、非洲，现在还残存的一些部落当中，你会发现他们也一样，这些部落民也是祖先崇拜的。他们也会去讲一个故事：他们的祖先在什么地方，是什么人。这就是典型的祖先认同。但是为什么在现代社会还有祖先认同的现象？除了原始部落，就剩下我们了呢？因为在人类的发展的过程当中，只要你进行了深度的社会化，彼此出现了征伐，就不可避免地出现失败。只要失败一次，就会被迫地认别人当爹。只要你认错了爹，你就没有办法再坚持祖先崇拜了。所以人类社会从一开端的时候走入到文明，甚至刚刚启蒙的那一刻，全世界的所有的人都是实行祖先崇拜的，一直到它失败。  

而我们之所以到现在还秉持着同样的原则，是因为我们在过去的几千年的、甚至几万年的文明史当中，从来没有彻底败北。在失败的时候都保留了星星之火，最终以燎原之势卷土重来。我们能够做到一以贯之的传承，所以才能够做到今天。无论是分子生物学的溯源，还是在文化上的溯源，都能够跟我们数千年甚至上万年前的老祖宗直接产生共鸣。这个是人类社会当中硕果仅存的一种传统的文明塑造方法。不是我们独创的，实际上所有的部落开端都这样，但是只有我们把这个火把从原始时代举到了今天。
