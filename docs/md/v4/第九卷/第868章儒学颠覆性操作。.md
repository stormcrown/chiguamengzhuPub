# 第868章 儒学颠覆性操作。 

弄清楚君子是谁。

最早的时候，“君子”指的是社会中的贵族管理岗位，就是“君之子”。相当于某一层级的核心管理人员的继承人。

后来，“君子”的用法开始泛化，变成了贵族人士的专称——所有的贵族男子都可以被尊称为君子，这是泛化的第一阶段。

再后来，儒学开创了新的理念：只要你品格合格，通过学习就可以成为君子。阶级隔膜被打破之后，“君子”进一步泛化，扩展为所有有学识、品格端方的人。

因此，“君子”在不同的时间段有不同的含义。比如在“君子六艺”这个阶段，君子的含义是指社会管理阶层的人。他们学习那些技能，是因为那是社会管理的基本技能——不同时代有不同的要求。

今天我们对于君子的要求又有所不同。有学识、有品格、有坚定的个人态度，这样的人就可以称之为君子。那么你想怎么培养自己的孩子？你希望他成长成什么样子？就看你心目当中的君子到底是什么样。按照君子的要求培养自己，而不是用到别人身上。君子就是一个自我修行、自我成长、自我培育的过程。

这个事的确很重要——君子的品格在今天还要不要紧？我认为还是要紧的。不管我们的教育当中已经包含了多少品格教育的内容，也不管今天的社会氛围是不是已经非常好了，我们依然可以对自己提出更高的标准、更严的要求，让自己更加优秀、心态更加端正、意气更加昂扬。追求上进，到什么时候都是生效的。

也许古典时期的那个君子已经过时了，但今天有今天的君子，今天的人也有今天的人所要求的品格。一个人的品格与才能足够高，能够对社会做出贡献和引导，那么他就可以称之为君子，这没有任何问题。

所以今天的人也可以把自己朝向君子的方向去培养。如果你希望自己的孩子成长为这样卓有贡献的人，就对他提出这样高的要求。但请注意：所有家长对孩子的教育都是言传加身教。如果你对孩子提的要求很高，那么请把自己的脊梁打直——无论财产方面是否足够优胜，至少你的榜样作用要非常强大。

**备注：**  
- “当世人”改为“成为君子”（原口语误，结合上下文指通过儒学学习成为君子）  
- “义气”改为“意气”（“意气”指精神状态，更符合语境）  
- “剂量打直”改为“脊梁打直”（“剂量”疑为口语误，结合“把……打直”的常见用法和以身作则的语义，修正为“脊梁打直”）  
- 删除了口语词“嗯、啊、摁一下”等，调整了句子衔接，使内容更连贯易读。
