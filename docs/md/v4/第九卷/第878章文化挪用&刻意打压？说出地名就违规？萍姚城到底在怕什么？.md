# 第878章 文化挪用&刻意打压？说出地名就违规？萍姚 城到底在怕什么？

那个古城啊，那个平遥古城啊，为啥说他们家的名字要这么个说法？刚才不是在聊晋商吗？聊晋商的时候就绕不开这个地方，叫平遥古城。那个地方为什么不能直接顺顺当当地说出来？因为上一次我顺顺当当地说出它的名字，然后我的视频被下架了。还说一个地名居然会被下架了，厉害。

那这个地方当年曾经干过什么？大家听过相关消息的人，还有印象的应该知道一件事——他们去年的时候干过一件事情，发了一个通告，然后说，嗯，呃，本地不允许拍摄汉服以外的摄影。也就是当地规定了所有的这些什么服装租赁、拍摄的机构只可以上架汉服。有一些个别的小伙伴听了这个新闻的时候，可能会觉得，哎呀，这不错呀，居然还有这个地方讲究啊。然而，可是，但她嘴里面的“汉服”指的是什么呢？啊，清民女装以及旗装。对，都是清代的。她嘴里面的所谓的“汉服”全部都是清代的，包括了清民女装。清民女装还可以勉勉强强说，毕竟是当年晋商道少奶奶穿过的。但实际上，除了清民女装之外，正规的旗装也可以在里面。真正被排挤的是什么？真正被排挤的是交领大袖，是齐胸襦裙，是各种真正的汉服以及影楼汉服以及汉元素。

你听着她发通告的时候，还说“我们这里只让汉服上”。你去看他的图，你就发现，除了汉服都能上。所以当时做这个视频，然后就被下了。当然视频下了也无所谓，毕竟他们的通告最后也说，唉，我误会，误会没有，这个事情错了没整。也就是说他们当地的这个通知也被列为非法了。觉得挺好，虽然说把我的视频弄了，但是他们通知不也弄了吗？诶，好，欢迎大家来到直播间啊。

穿汉服他们怎么那么难受啊？大家那个什么完全可以去看一下，现在有很多大品牌的专业的汉服店里面，不写“交领”。有兴趣的小伙伴可以去某宝上面去端详端详，仔细地去检索一下。仔细到什么程度呢？大约就是你先找汉服店，然后到汉服店里面呢，再去专门搜索“交领”，就大体上是这个样子啊。找那些体面的、上了号的，体量不算太小的那种，不是散兵游勇式的那些正规的汉服品牌。然后你再去搜索里面的交领，你会发现有很多的大品牌的、年头较久的传统汉服的店，商家都不写“交领”二字了。不写交领，但是他有没有交领的衣服？那必须有啊！汉服你没有交领你做啥呢？那是主体，那是核心啊，那是最核心、最重中之重的东西。然后他一定有交领，但是他不能够写“交领”二字。你猜为什么？

明晃晃的交领穿在模特儿的身上，然后他的标题写“直领大襟”、“直领斜襟”。他把“交领”给删掉了，不让你说交领。所以谁干的这件事啊？猜啊，咱们只能去猜，毕竟人家又不会站出来说“我干的，我干的”，没有这样的人。所以谁干的这件事呢？也就是当交领重现江湖的时候，谁最难受？交领重现江湖谁最难受？汉服重新穿上大家的身子，谁最害怕汉文化卷土重来？谁最恐惧？那就是谁干的。
