# 第888章 真相一省百姓为何被曾剃头杀光？

为什么清末的时候，华夏火种在湖南？因为曾剃头，因为曾剃头啊！啊，曾国藩，曾国藩呢？曾国藩是一个厚黑学的大家，是表里不一的伪君子。但是这个人很强，很聪明，很有上进心，很有进取心，而且很机灵，随时随地能够把握机会。在任何的场合下面，他都能够找到契机爬上去——这就是曾国藩。所以，当你想要学习这些东西的时候，你可以去研究一下他的生平。但是呢，你靠看他的日记没什么用，里面都是冠冕堂皇的东西。你信那个玩意儿，那就叫上当受骗。看日记没用，但是你可以分析他的行为。他是厚黑学的大家。

那曾国藩之所以叫曾剃头，为什么呢？因为他杀了太多的人。怎么回事？当年太平天国起义，太平天国运动席卷大半个中国。而且这种席卷，因为它是发自于普通百姓的一种革命，很快地，星星之火就燎原了，属于是各个地方的人都积极地参与，最后形成的就是遍地都是革命军的态势，声势浩大。当他实现了这么一个声势浩大、军民一体的环境的时候，曾国藩代表满清朝廷说：“我要来剿匪了，请问谁是匪？”曾国藩当时面临到的局面就是，他说他要剿匪，但是到处都是太平军，每一个人都极有可能是太平军——因为这是一个人民的革命。于是最后他做了什么？嗯，全杀了，遍野无人，一律杀空。曾剃头的名号，就打这来的。

然后呢，他做完了这些事情之后，就获得了朝廷的重用。为什么他做了这些事儿之后，最后没有被卸磨杀驴呢？这功劳就来自于太平军。因为太平军在他行军的过程当中，拔除了南中国的大部分的满城。将这些军事驻点物理拔除了之后，这个朝廷在当地就没有统治的能力了。因为满清朝廷的统治，就是借由这些军事驻点的八旗兵来进行的管控和镇压。离开了这些他们的八旗，他对这些地方是完全失控的，没有哪个百姓认他。相当于说，只要八旗的兵马一消失，这个地方就天然是反叛者了。当这种局面出现了之后，能够掌控当地的，就变成了在当地盘根错节、有势力关系的各路汉臣。而曾国藩就是其中之一。

他的基本盘呢，就是湘军系列。他是一整个的一群人，一根藤上的蚂蚱，上上下下的一票人——这个就是湘军派系。既有军权又有财权，话语权非常巨大。然后呢，这群人你要知道，中国人认乡里啊。当他是湘军体系的时候，他就会回馈湖南人。不见得是好人心，也不见得是好心，但是呢，他回馈乡里的这个动作也确实做了。于是他就搬了大部分的钱财，就挪到了湖南去。而这些东西一挪过去，清廷的控制又较为失控，之前的那些不准办私塾之类的东西是不是就自动解禁了？以前清廷不准地方办私塾，但是现在你管也管不过来了。我这钱多的这么多，我们作为湘军体系是需要后继有人的。后继有人我就得培养自家子弟，所以培养自家子弟要干什么？要办学堂。于是呢，湘军就借由之前所敛的巨额财富，投放到了湖南地界上面，兴办了大量的学堂。有教育就会产生人才。因为推动了教育，所以在湖南走出了我们第一代的革命志士。数量特别多，质量特别好，因为这个地方的教育质量很好。

哎，好，欢迎大家来到直播间。嗯，啥？嗯哼。开始又开始造谣了啊——长毛野人来了，见人就吃。太平军立下的法度包括什么？第一，你要蓄发，之前不剃头就砍头。这事儿大家都知道吗？啊。剃发易服令这个东西，它不是一开始下一个令就完事了，它是坚持了清代的200多年的。实际上这个事儿从关外就开始了，加在一起的时间已经超过了300年。那么，剃发易服这个事儿，从清初坚持到了清末，它意味着什么？意味着所有的人家都知道，不剃头就砍头，必须剃头。而太平军来了之后，蓄发不剃头了啊。清代的时候，妇女只要离满城近，只要在市区就要断足，自小断足，把脚要掰折。然后太平军来了，发放放足令，放脚不断足。然后你说：“哎呀，长毛来了要吃人。”你觉得老百姓傻的吗？他是没文化，是已经被迫成了文盲，但他蠢吗？骗谁的呢？太平军走过的地方，大家都留起了头发，大家都放了脚。后来太平军失败了之后，清军卷土重来，把之前的放足令直接列为非法，说：“太平军居然让女的放脚，违法，违背了我们的祖宗规矩。”然后让大家的脚重新掰折。

哎，好，欢迎来到直播间。嗯，啥？我生活在那个年代，我爹妈会给我断足。对，他为了保我的命会给我断足。但是这是爹妈的责任吗？不是，这是满清皇室的责任，他们强迫汉人断足啊。如果你生在那个时候的话，你爹妈会硬摁着你去剃头。不管你觉得那种样子有多丑，都会给你剃，为什么？她要保你的性命。所以呢？所以呢？幸亏我们活下来了。
