# 第863章 他们在笑你空谈理想｜百年前的毒舌预言正在应验 

横渠四句，它本身是什么？  
为天地立心，为生民立命，为往圣继绝学，为万世开太平。  
这个东西是什么？咱们得说清楚啊——这四句话到底是什么？它在干什么？  
很明确的一点是，不管你管它出现的那个时间里头，这一项东西其实还代表了另外的意识形态的内容。但是呢，即使只看表面文章，就这四句话，它在干什么？它在励志啊！这四句话是在励志，对吧？谁说的？张横渠（张载）说的，所以称之为横渠四句。  
（我现在可以重启了吗？）  

这个话本身是一个励志型的语言。那你说，现在在这个地方来抨击这种东西的人，是为了什么呢？啊，因为它是在立一个伟大的志向。至于说这个志向在你看来是不是荒唐、实现不了、对我有什么好处？没有用，就是虚头巴脑的，他又不干……哦，无论是怎么个情况，无论你心里面是怎么去驱使它的，总之对方是立了一个伟大的志向和伟大的目标，毫无疑问。那我就问你：反对他的人能为了什么？反对他的人只有可能产生一种影响力，就是告诉你说励志这个事儿不重要——一个人立什么志向呢？就应该管吃好喝好。这叫什么？这叫做庸俗化，这叫做庸俗化啊！这个东西就叫做庸俗化。  

而励志，在张横渠那里，他叫做为天地立心、为生民立命。那放到今天，真正这个国家的脊梁带头人有没有励志？哎，大家有没有去看我们经常去宣发的所有的稿件啊？你们看了的话，我们从立国开始，我们要为共产主义的事业而奋斗——听过没有？我们要为共产主义事业而奋斗，听过了吧？我就问你，这玩意儿是不是励志？你确认共产主义一定实现吗？其实在这个领域里面，也会有人用同样的方式去解构它：什么玩意儿啊？什么叫共产主义啊？怎么回事啊？也会有这样的人呀。但是因为有人在里面搞庸俗化，我们就不需要立志了吗？  

所以你看他立的志向是不是正面的。为天地立心是什么？遵守天地自然的基本法则，遵循天道，寻找天道之间的人心啊。为天地立心，为生民立命是什么？约等于为人民服务啊——但是那个时候他们没有这么崇高。但是为生民立命是什么？天下之伤皆是我之伤，天下之责尽是我之责，这是道义一肩挑的表示。为往圣继绝学干什么？传递文化，使之万世不灭、生生不息。怎么了？这是错的事吗？他做不做得到，那是他的能力问题，但是这个事不是正确的目标和梦想吗？为万世开太平——我们今日建设了这个世界，为的是将来能够更好、持续性地好下去。这事儿错了吗？  

所以有什么可以批的？他这个理念所立下的志向，和我们今天——我们今天的价值的标尺之间，十分的兼容，异常的兼容，高度的有内核上面的一致性。所以你觉得那些反对说这玩意儿没用、不应该整这玩意儿、这个东西就不能是什么传播传统文化——这些人是干什么？他打着打倒过去、追随未来的这个借口，打着这个旗号，然后说反对这些东西。最终他反对的是什么？反对的是社会主义价值观。嗯？哎，啥？不应该活在当下吗？这跟活在当下有什么矛盾点吗？我就问你，这跟活在当下有什么矛盾的吗？我们要为共产主义事业而坚定的、永恒的、持续的奋斗，和我今天晚上必须得吃饱饭、不能饿着睡觉，有什么矛盾的吗？啊？我们要追求整个国家的进步与发展，和我们今天要看个电影，有什么矛盾的吗？没有矛盾的。  

**备注：**  
- “张恒驰”修正为“张横渠”（即张载，号横渠先生）。  
- “天下之商皆是我之殇”修正为“天下之伤皆是我之伤”（根据文义，原文“商”应为“伤”的错别字）。  
- 删除了部分口语化的重复语气词（如“唉”“啊”“嗯”等）以提升段落易读性，但未删减实质性内容。  
- 调整了少量语序和标点，使表达更连贯。
