# 第833章 「神仙不问出身」｜为何荧幕仙君全凭血脉？

我们明代就有修仙小说，那个明代开始就有修仙的故事。实际上也不止这个时候，在更早的时候就有。但是明代呢，白话文小说盛行，普通人也能够去看啊。那这种修仙类的小说，典型的特征就是“神仙本是凡人做，只怕凡人心不坚”。所以我们这里的修仙文，原本呢，应该和游戏里面的升级打怪是一个逻辑的。它呢，就是一种可以突破阶级——我们的修仙小说的目标啊，是突破阶层，突破种姓，甚至可以突破天地，是超脱于现有的一切框架。但是呢，现在的那些什么修仙文，尤其是被改编成电影、电视剧的这种修仙文啊，搞的全都是种姓制，我也是离了大谱了。我们的影视传媒界比孔夫子古董多了，但是呢，他们动不动就嘲讽孔夫子落后，封建闭塞保守没意思，我们要进步。他们玩的东西比孔夫子落后2000年。啥修仙都是逆袭呀。我特意强调就是翻拍成这种什么仙侠电视剧的这种啊，特意强调了这个，因为我也不是说所有的修仙小说我都看。但是呢，修仙小说、修仙爽文当中包含了一部分的血统论，而且呢，它拍摄成影视作品的时候，格外的会去挑选这种类型的脚本拿出来去拍。所以我一直强调，我们的影视传媒圈儿是特别闭塞保守的，甚至是赤裸裸的封建和奴隶制的啊。

备注：修正了“翻译成那个”为“翻拍成那个”，其他内容未作删减或修改。
