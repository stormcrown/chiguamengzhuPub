# 第875章 多子多福≠爱生孩子！#传统文化 #冷知识 

先做一个小调查：你们觉得“多子多福”这个事儿，是代表中国人喜欢生孩子，还是代表中国人不喜欢生孩子？认为“多子多福”意味着中国人特别“繁殖癌”、特别喜欢生孩子的，扣“一”；相反就是“二”，不爱生孩子。那么“多子多福”这句传统的口号，它到底代表什么呢？一代表中国人特别喜欢生孩子，二代表中国人不爱生孩子。你们觉得到底是一还是二？嗯嗯，啥字面意思不就是一吗？所以我才要调研呢！哈哈哈哈，直接让你顾名思义我就不问了。啊，“多子多福”是什么？多子多福可以算作一个劝导语。之所以会出现一句劝导语，实际上就是因为不爱生、不爱养。中国人并不是很热衷于生孩子的，不是很爱生孩子。你想多了，诶，谢谢海天一色。而且呢，这个东西不是一个绝对值，它是一个对照值、相对值。它是一种相对值，而不是一个绝对值。你说都盼着自己断子绝孙？没有这样的。人类的本能当中就包含了生育的本能，所以都会有一定的生育意愿，这东西是天生天养都有的，刻在基因里面，谁都逃避不了。这个叫做“食色性”，也客观存在。

但是呢，为啥？因为中国人这里有一个特点：我们的生产力发展较早，文明发展开端较早，最后形成了非常稳定的、可预期式的农耕生产系统。也就是说，你一生当中从事什么职业、拥有什么财富、将来的未来十年你能攒下多少本钱，如果不遇到大的变数，你是能算得出来的。这意味着什么呢？这意味着在中国这里，普通人会在生育这件事情上面盘账。为什么其他的人群不盘账？因为生产力水平低下，意味着他生下来的孩子根本不保证能够养得活，生存的概率不确定，风险极大，搞不清楚明天人还在不在。这种情况下，他没有机会也没有可能性去做这个规划，所以不存在爱或不爱，他用纯自然的方式来面对自己的生育就可以了。

但是在中国这里不一样，中国人在这个事上是有条件有可能去盘账的。所有在现代社会生活的，在座的各位，你但凡身边有年轻人或者你本人就是年轻人，请问在家里面的长辈催生的时候，你有没有在解释的过程当中盘账？都在盘账吧！划不来、生不起、养不起，是这样的吧？这是什么行为？这就是一个盘账计算成本的行为。古人只是古，不是蠢，他们也是收入平均的，对自己的收入是有预期的，对风险也是有预期的，所以他也会盘账。而会盘账的人群，对于生育都不会那么爆棚式的热爱。因此呢，就会产生后续的后果。如果一个群体因为文化水平太好、经济水平太好，反而导致大家不断算账，以至于对生养孩子没兴趣，会造成一个社会的后果，就是你的人口结构失调。

今天现代社会大家已经普遍发现这件事情了，因为现在比较严重。古人也会察觉这种问题，就是你不能够放任这样的思想存在。那么应该如何调整呢？就要制定社会规范。在现代社会的话，前几年的时候，是不是很多人吹嘘说欧美发达国家生一个孩子给多少多少补贴？当然，阿美（美国）那里显然这个话有点谣言的意味，但是欧洲那里确实给了不少补贴。那请问这样的方式有真正本质的促进作用吗？没有啊。这一些拼命发补贴的地方，吸引了大量的外来的第三世界国家的人口跑到那里去进行人口侵占。但是古代能够做到这事吗？做不到。是不是他做不到这事？然后又要提高你的生产生育的积极性，他又发不起那么多的成本。

你要知道当年那个谁勾践，曾经因为他那里民生凋敝，需要催生，于是他就制定一套一套政策：说生个儿子奖个猪还是奖酒？生个儿子奖猪加酒，还是生个女的奖猪加酒？反正就是生孩子给你奖个牲口，然后再给点酒。如果生了双胞胎的话，甚至还可以给你安排乳娘。当年勾践就安排了这种促生的方法。但是这个政策你算算账就知道了，撑不起几年。所以这是一个短期政策。

那么在过去发生过的长期的促生政策，包括了什么呢？明摆着的，包括说你到了年纪不结婚的那种人，课以重税，这就是单身税。现在在国际上也有一些地方在隐性地收取单身税，这个是我们老祖宗用过的招。同时呢，它还包括限制性的条款，就是不允许老少配，因为老少配会影响生育后代。这些是限制性的条款，这些相关的条款属于行政操作的细则。

但是呢，我们还是有一个非常底线的、大的运行框架，其实它也是为了促进生育的，就是要让人在生育当中获利。这个获利是什么？我之前讲过一个视频，就是讲孝道。孝道是什么？就是你只要生了孩子，这个孩子由官府保证，他必须对你好，将来必须给你养老，他的收入当中必须给你生活费。如果他做的事情不好、不孝顺的话，官府代为惩戒。这个就是孝道系统。这个孝道系统你有没有发现？相当于官方不出钱，但是你只要生了，你就有养老金——生一个孩子有一份养老金。它就是起这个作用的。所以实际上中国以前的家庭伦理系统、道德系统也是在促进生育。
