# 第893章 贴脸开大：文化反击录 《文脉醒辞》

最开始的时候我是怎么走起来的？最初怎么走起来的？其实就是因为被某些人“贴脸开大”啊。之前聊过这个事儿，是在零一、零二的时候。零一、零二那会儿，他们搞了一些国际活动，给各国领导人赠送民族传统服装。送的是什么呢？西服剪裁的满式马褂。就是那个东西把大家恶心哭了、恶心吐了，于是才出现了汉服的复兴运动。我们的很多传统思潮，实际上都是在对方不停地“贴脸开大”之后，我们才终于醒过神来，想起来了。例如说我们隔壁的思密达，他在那儿跳着脚，一直说我们要那个什么申遗、端午祭，这玩意儿是我们的，如何如何地喊，嚷嚷制造声浪是吧？国内还有很多营销号跟着上，唉，说我们什么什么这不要那不要，如何如何的是吧。然后怎么着？大家端午节有假放了啊。在那之前端午是没假的，在那之后端午有假放了。放假的节日才是节日。

然后说马面裙怎么开始起来的？不就是因为有人偷了马面裙是吧？不是这样的话，没有人“贴脸开大”。你要去进行传统文化的复兴，要去讲解说我们以前到底是什么什么样的，人家就叫复辟。到今天我的直播间里面，还是时不时就会有人冲出来说：“你在那里说明代的裹脚，不断是复辟啊，是洗地啊，是那个什么想要裹小脚如何如何的，你想骗人干什么呢？”他们对于我们真正的传统被找回来是很恐惧的。但是为什么真正的传统被找回来这件事情很重要呢？因为它是一个人的立身之根本。当一个人觉得自己的民俗都很恶俗、都很落后、都很蒙昧的时候，他会瞧不起自己的祖先。而任何人当他瞧不起自己的祖先的时候，他就会对自己的认知产生错位，他会觉得自己今天所有的出息都是外人帮助的，于是就会认野爹、认洋爹。你会发现，互联网上那些认为中国传统一无是处、凡是说中国传统好的人都是要复辟的人，他同一时间会专门去认一个洋爹，尊崇西方中心主义，一定会这样子操作的。为什么？因为这集的东西啊，它内核上面是一致的。一个人呢，他总要给自己的先进、给自己的发达找一个来路，一定是要有归因的，必然要找一个原因。如果说我们自古以来就很烂，烂到那种闻着味儿都恶心，那你就必须为今天我们的快速成长找一个新的理由。什么理由呢？外头来的呗，必须得是外头来的呀，不就是这种原理吗？所以他一定会这样做。

那什么情况下，我们真实的传统会被我们发掘出来并且不断传播呢？那就是对方又“贴脸开大”了。例如说前些年的时候啊，铺天盖地的各路官媒都训诫我们，那个啥啊，那个端午不能快乐，要安康。甚至有人专门去画官宣的那种海报图的时候，画各民族过年都是各种各样的仪式，然后呢，画到汉民族这个地方开始哭坟，一到拜祖宗的这个节奏的时候就开始哭坟。这个是当年他们画出来的画啊。祭祖在他们的画作当中就是哭坟，这显然是不对的，这是截然不同的两件事情，分明就是在造假。但是呢，诶，人家就贴着你的脸开大。如果不是他贴上来做这个事儿，你能去解释吗？你凭空说这些事情，我们自古就是坟头蹦迪的，人家是不是觉得你又要复辟、你又要造谣、你又要胡说八道？但是当对方把所有的祭祖行为用一幅简笔画归纳为哭坟的时候，就是在指着鼻子侮辱你。那这个时候有人站出来说：“不对，我们的祭祖不是这个样子的，从原理到行为全都是错的，你不能这样做。”这个时候叫什么？叫做有的放矢。所以我们的传统文化的复兴，离不开各路反贼的做法。就好像我的直播间能够人数破千，靠的都是各路黑子抬庄。这个世间有阴有阳、有阳有阴，没有支撑啊，起不来呀，对，这就是作用力与反作用力啊。
