# 第872章 君子六艺冷知识！▶ 数学竟是古代贵族必修课！！！ 

古人有君子六艺，有尚武的习俗。推荐研习这种习俗吗？啊，推荐没有问题啊，可以可以。啊，没有错过，我看见了。可以的可以的，强身健体完全没问题。实际上呢，现在有很多家庭条件稍微好一点的家庭，非常在乎让孩子强身健体，都看得出来。但是呢，受到西方影响比较多的家庭呢，就会让孩子去学习现在流行的一些什么徒步啊等等一些项目，还有一些专门的自行车等等的专项。那么我们传统的项目当中有没有能用的？当然有了呀。啊，你要是条件允许的话，学一学骑马咋的不行吗？啊，完全可以学骑马呀，但是这考验家庭的财力啊。因为骑马这件事情确实还是有点贵啊，实在讲就是有点贵啊。你如果支撑得起的话，确实可以学。唉，然后呢，学射箭行不行？完全可以。然后君子六艺当中还包括了数学的啊，君子六艺是包含了数学的。你会阻拦孩子学数学吗？你恨不得给他报个班，让他好好学数学，对不对？所以呢，这些东西都完全可以学啊。然后君子六艺当中的礼是什么啊？礼呢，是社会制度。注意了，礼是社会制度啊。所以今天的孩子要不要教？这个学校里面的思想品德教育、思想政治课，就教的是这个东西。所以不要觉得我们今天的现代社会已经不搞君子六艺了。古时候的君子六艺是只有社会管理层的才需要学习的，普通人不学的，普通人也没有资格学。但是今天所有接受教育的人，都学习了里面大把的内容啊，知道吗？今天所有的人都学习了里面很核心、很要紧的内容。无论是识文断字的、书写的、算术的，还是国家制度、社会管理层面的这些东西，都纳入到了我们的义务教材当中去。只能说过去那些贵族们去搞的这些军事相关的项目，学校里面确实没教。但是呢家长可以支持一下，学校里面只能学习一些基础的体育教育啊，这些也是强身健体的，但是可能强度没有那么好。完全可以去尝试一下，自己家里面补强一下嘛。练一练武没事啊，可以练一练。然后现在还有一些地方去有那些什么习武的班儿啊，也有这种，也可以报一报。我觉得都挺好的。
