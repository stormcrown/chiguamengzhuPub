# 第892章 警惕这种历史叙事！⚠️正在悄悄摧毁民族自信！ #历史真相

芜湖  
什么“五胡乱华”啊？讲这些东西的人，包括讲少数民族南迁，使用各种各样的中性词汇去描述历史事件的这种方式，是什么呢？这属于是一个标准答案，这是中国人的政治正确啊。我们一定要弄明白一件事情：政治正确的反面不是政治错误，政治正确的反面叫做事实正确。所以在这种政治正确的定义下，也就是说，我给你一个钢印，我给你一个标准答案；不管你识别出来的证据是否支持这个答案，最终你必须要证明这个答案。这相当于是我们出了一道数学题，然后给了答案，再让你去填写运算过程。尽管大家发现怎么运算都算不出来这个数，但是不管你怎么算的，最终你必须给出这个数，只有这个数是标准答案。  

所以你就会发现，很多学界的人、有学术背书的人，反而特别热衷于给你背标准答案。那这个标准答案当中就包含了所谓的融合论。按照今天的分子生物学的溯源来看，这个融合论的说法本身就是有问题的。因为如果真的如他们所说，打过一仗就容易融合，中国人的父系血缘不会那么干净。我们的父系血缘能够那么单纯，就意味着它不是他们所说的婚娶关系。这个所谓的融合与婚娶背道而驰，实际上是战胜与战败之间的关系啊。战败了之后，彻头彻尾的战败，除了顶层签约的极个别的人，其他的人实际上都会面临到社会层面的排挤。而且呢，主流的团队一定会跑路啊，这个才是我们历史上面的基本的规律。  

就像是当年啊，匈奴——南匈奴南归了之后，实际上真正的南匈奴南下的部分，只是它当中整个匈奴本部当中的一小部分。剩下的大部分战败了之后，他会选择什么？西逃。包括契丹也是这个样子，战败了之后，大部就会选择西逃，真正留下来的是其中的极少数。那当它定义这个东西叫融合的时候，那么何处不可容啊？这种言辞、这种标准答案背后是贻害无穷的。毫无疑问，遗患无穷。但是因为它是标准答案，所以你在哪都能听见它。但是为什么会觉得不舒服呢？包括说哪怕没有学习过历史的人也会意识到这一点。尽管他背的是标准答案，说的话呢，好像还挺好听的，还特别和气，还特别正规，而且背后的背书、学术背书头衔一老长排，一张名片印不下，但是我就是不舒服。这个不舒服来自于哪里？这个不适感就来自于：他没有站在你这一边说话。  

这个就叫做历史观啊——站在哪边说话就叫做历史观。历史观，屁股坐在哪个板凳上面，人站在哪个队伍里面，决定了他的是非价值判断。就像宋金之争，宋金之争，为什么居然会觉得，哎呀，那个金国掳掠宋国地基，并且百般蹂躏，这件事情叫做融合？太好了，干得漂亮啊？军事的威压，战无不胜，女真不满万，满万不可敌。为什么他能够这样牛哄哄地开始吹这个东西？因为他把屁股坐到了战胜者的那一边。他只决定啊，谁赢我帮谁。谁赢帮谁，这件事情是对的吗？这是投机行为，这是押宝行为，这是赌徒心态，唯独离正义背道而驰。  

如果说你站在宋的那一边呢，你当然可以批判赵宋皇室的无耻、软弱、无能，但是同时更要批判侵略者的残暴、野性、无德。这个东西就是你的站位问题，你到底站在哪一边？如果站在了渔猎部落的那一边，那抢赢了就是最厉害的事情，而且抢赢了之后还把对方羞辱了，让对方居然跪服了，那就太高兴了，能够欢欣雀跃——意味着他站在了渔猎部落的那一边。但是按照他自己的身份，按照他自己的身份，他不可能是一个渔猎部落。所以干什么？他站在了赢家的那一边，甚至没有站在自己的这一边。一个人要么你选择站在正义的这一边，这叫做顺应天道；要么你站在自己的这一边，这是维护自身利益。但是像这一种，打着政治正确的旗号，站在赢家的那一边，这叫做什么？这叫做奴婢心理：谁赢谁就是最好的主子呀。他是没有自我意识，没有自我认知，也不知道自己是谁。他只选择谁赢我帮谁，谁赢谁是我的同伙，谁赢了谁是我的同类。就是他已经预备好了，随时随地认野爹，谁都可以是他的爹，除了他的亲爹。  

这种行为的方式，即使放到野兽界也是令人不齿的。但是呢，当他愿意甚至自甘为奴的时候，这就很正常。因为对于奴隶来讲，这种心态太正常了。就像你刚才说的那个什么，呃，辽啊，辽人。契丹——辽实际上是剃发易服的祖宗。中国人所遭遇的剃发易服的第一波就是辽人做的。虽然你看墓地的那个墓志里面的绘画的时候，会发现辽国的贵族也会使用汉制的服装，但实际上在辽的内部，是发行过剃发易服的相关政令的，要求剃发，要求服装改左衽。但是呢，他比后来的清要柔和一些。为什么？因为他说的不是砍头，而是不照这样子改要依法惩治你。他依据律法而为，就是依法办事啊——你不按照我的服装改，我要依法惩治你。这个法呢，比后世的砍头要轻一些，就是这样的一个区别。但是剃发易服原本就是辽推行的。那你说，当他都已经开始改变你的服装，改变你的发型，甚至契丹人还专门设计了一款单独的契丹文的时候，他要改变你的文字系统啊；他在官僚的录取上面，用不同的录取方法和标准——结果你硬说，哎，是一样的。因为啊，因为有汉奸家庭是吗？那汉奸家庭在这种时代的汉奸家庭是什么身份？无论是辽、金还是后来的清、元，都是一样的。这种能够在高层获得利益分成的这些家庭，看上去是豪门，实际是奴婢和依附者，只是早早投靠的奴婢。所以呢，给的优待比较多一点点。放到清的话语上面啊，这个就叫做抬旗。  

所以实际上我大部分时候呢，是非常不喜欢政治正确的啊。我不喜欢政治正确，因为它不唯物。唯物主义要求的是：不管这个事儿好不好听，我们都真实地面对他；如果他对我不利，我就想办法改变他——这个才叫做唯物主义。实实在在地面对世界，是什么就是什么，不是什么就不是什么，这就叫做唯物主义。看待历史也要用这样的方式。但是当你规定了标准答案，它必须是什么的时候，这就不唯物了。
