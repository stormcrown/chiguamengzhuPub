# 第836章 【青史棱镜】吕后·武瞾·慈禧 

朋友讨论吕后、武则天、慈禧三人对中国历史的影响，有什么看法吗？嗯，吕后、武则天、慈禧，把这仨放到一起是吧？还挺有等差排列的，也不太等差啊，但是挺有刺激感的。因为最前面的吕后，也是最优秀的一个政治家。这三个人当中，排在最前面的吕后，是这三个人里面最优秀的一个政治家。尽管说她最后啊，尽管吕后最后的战果不是非常好，毕竟吕家基本上都快没人了，但她确实是当时大汉开国时期的一个重要政治家。她基本上延续了刘邦的国策，然后把这个国家维系住了。所以我说她是一个优秀的政治家。至于在历史点评当中，她常常差评满满，毕竟人家当年有那么大的一个吕氏家族，背后也确实存在着篡权的嫌疑嘛，所以点评不高很正常，点评评分比较差一点是很合理的。所以呢，吕后这个地方千万要注意：她和戚夫人之间的恩怨情仇，你不能拿她看作宫斗剧，因为吕后不会给你玩宫斗戏的。她是一个实权在握的、正经的开国元勋级的皇后，她犯不上去跟你玩什么后宫的恩怨情仇。要整，咱们就来政客之间的套路啊。欺负人的问题就在于，她压根儿就不知道什么叫做政客，所以就嗝儿了啊。然后整体上来讲，吕后是一个不错的政治家，延续了大汉开国年间应该有的政治局面。这是第一个人。

第二个人呢，就是武则天啊。武则天这个人呢，我之前曾经在直播当中聊过，她属于一个在从政水平上面平平无奇、权谋之术上面操作不错的一个君王。注意啊，武则天是一个在治国理政方面平平无奇、然后在权谋之术上面操作得比较顺畅的一个君王。她身上的好处就是，她当时确确实实维持住了朝堂内部的安稳。但是啊，她的一个坏处就是，她这个得位的问题，实际上造成了整个国家的内部力量没有办法拧成一股绳，因为始终都有人要惦念着如何抗衡她。第一，她只是一个太后；第二呢，她同时又表现为说，我已经穿上龙袍从政了啊，已经当了皇帝了，于是就有了明确的篡权的可能性，于是就造成了最大的一个问题——朝堂的内部不能够拧成一股绳，也不能够游刃有余。她必须借由权谋之术借力打力，拉一批打一批，她必须用这样的方式才能够管好。所以呢，当她的内政上面难度系数这么高的时候，她在边塞方面犯下的错误就会非常多。由于她的得位的问题，其实刚刚归降的很多部落对她是最为不满的，因为权威性严重不足，权威不足，合法性严重不足。然后呢，她又没有能力把触角延伸到边境地区去，就造成实际上在武后临朝期间，大唐的疆域是严重缩水了的，并且造成了之前很多新归附的部落产生了离心力——这个是武则天当时产生的一个负面影响。然后呢，更严重的一个负面影响，就是由于她自身以太后这种身份最后成了帝，甚至呢，到临终的时候，还曾经试图将她的权柄交给她的侄儿，姓武的。所以呢，就造成了后来的皇后、太后们的社会信用评分全部都下降了，因为之后的皇帝在遇到特别能干的大女主的时候，就开始担心这个媳妇儿会不会背刺我啊。注意了，实际上武则天称帝的行为，阻断了后来大部分的女性在政治生涯上面走到巅峰的可能性。所以这个是武则天，她有一点儿正面性，但是同时呢，产生了很长远的负面影响。这是一块。

然后当然了，前面这两个都还算是在女性政治家当中进行排行，就是有好处也有不好的地方。嗯，吕后好的地方多一些啊，武则天呢，不好的地方稍微多一点，但是她们都算是女性的政治家、女性的政客。然而放到慈禧这个地方，那就大不一样了。我认为慈禧跟她们摆到一张桌上的话，能把那两个女的弄恶心了。为什么呢？因为慈禧是跟全国所有人对着干的一个角色啊。注意了，很多人说慈禧葬送了她的大清，所以呢，对我们来说是有好处。注意啊，我再次再三反复跟大家强调一件事情：就是慈禧是给大清续命的人啊，慈禧是给她大清续命的人，所以她是彻彻底底的敌对势力啊。所以呢，吕后也好，武则天也好，她们呢，只是做得好与做得不好的问题，甚至说你要是觉得啊，这女的点评得不合我的心意也没有关系，她们身上总归都是评分高低的问题、做的好坏的问题。但是安放到慈禧身上的时候，凭什么好坏呀？她是敌人呢！敌人？你为什么要谈什么好坏的问题？谈什么评分？谈什么高低？她不应该跟任何一个中国政客摆在同一张桌上。

嗯，东汉的张德何西能力都不比武后差。嗯，武则天在历朝历代的女性政治家当中的话，排行不会很高的啊。好，欢迎来到直播间。啥？脖子上大金链子很扎眼？你这话说的也很扎眼啊，让我竟然一眼就看着了。这玩意儿叫大金链子吗？我的天，哎呀，你吓死我了啊，我居然有大金链子了。嗯，啥？慈禧可以和宋美龄在一桌？大差不差啊。好，慈禧和宋美龄可以坐一桌啊。什么？宋美龄是坏人？慈禧是敌人？不是，宋美龄也是敌人。你拿她当什么坏人呢？我从来都说她不算坏人，她不是人。你要是对此有意见的话，那就是我对。
