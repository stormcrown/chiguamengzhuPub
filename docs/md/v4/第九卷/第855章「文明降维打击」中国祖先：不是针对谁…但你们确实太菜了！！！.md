# 第855章 「文明降维打击」中国祖先：不是针对谁…但你们确实太菜了！！！

天下皆下品，唯有读书高。因为中国人开启文明太早了，我们的古人常常把我们周围的人统称为“蛮夷”。这不是专门指谁，而是说——我不是特指你们中的哪一位，在座的各位都是中国古人——就是这种姿态。为什么会有这种姿态？中国的古人为什么有这种傲慢、狂放的姿态？因为我们进入文明的时间过于早，太早了，早得有点离谱。因为崛起得太早，所以当我们穿上衣服的时候，发现别人还光着；当我们开始使用精致的耕作工具、每顿饭都能吃上东西的时候，别人还在那里风餐露宿；当我们这里已经开始讲究家庭的秩序，谁可以做什么、谁不可以做什么，走路时都知道要礼让、要分先后秩序，而不会你争我夺的时候，别人那里还在疯抢。这种情形就会导致我们的祖先抬眼一看，遍地蛮荒，所以他就将自己称为文明之后，而将其他一切都统一称为蛮夷，连分都懒得分——就到了这样的地步。

因为发生了这样过于早、太早的文明开端，于是造成了一个问题：在这个文明演进的过程当中，我们在文明这件事情上、在秩序这件事情上获得了很多的好处。因为当你拥有很稳定的秩序、内部很好的分配机制，不再争抢，可以进行细化分工，在这种情况下，你的生产会不断进阶，不断扩大。这就像流水线生意一样，就像我们在企业里面的流水线工作——任何一个工作岗位上面拧螺丝的人，都不知道产品的全貌是什么样子，你让他做他也做不出来。这就是社会分工精细化。你实现了这个社会分工的精细化，会造成什么问题？每一个人都能够发挥自己的长才：跑步跑不过别人的人，在丛林当中会死去；但是在中国这里，你跑步不行，你手灵巧，你就可以过得比别人还要好。

最后产生的一个结果是，我们就会越来越尊重秩序，越来越尊重稳妥，越来越尊重那些能够开创知识与秩序、还有技术的先进的文化人。我们是特别尊重文化人的，因为我们自古以来、几千年以来，就在这事上拿到了无数的好处。所以中国人极其在乎文化人说了什么。而且你想一想，我们2000多年前到处都是贵族，结果出来了一个孔夫子说“有教无类”，谁都能读书，谁都能学习怎么当管理者，以至于后来一代一代的思想迭代之后，最后就呈现出了“王侯将相，宁有种乎”。这些思想都是一点点地萌发、一点点地出现，循序渐进产生的。当中国人这里已经出现了这些东西的时候，我们就会格外格外地尊重、格外格外地在意文化人。

但是过于在意文化人的思想，就在上个世纪的时候，导致中国人在接受了一些文化人的错误思想之后，也受到了最严重的冲击。像这种“混血才好，别人家多高级，你看他白人长得俊男靓女的，老漂亮了”——这些思想你以为他只教给了中国人吗？不是的。这些洋人从头到尾都很好的思想，他教给了全世界，但是中国人受到的影响最剧烈。就是因为我们自古以来就在乎这些文明之崛起，就在乎这些文化人说了什么，就在乎社会上面谁最体面、谁最妥帖，我得向他学。因为我们在乎它传递的讯息，所以受到的影响最严重。而那些文明开化较短暂、较后期、较晚的这些人群，反而没有受到严重的冲击。

为啥？来了一位丝滑的舰长？舰长还有丝滑呢？肯定比我丝滑，对不对？就像这两天大家都在那里看的那个阿三巴拉特那边——有没有发现他们就不在乎这事儿？你以为当年那些殖民者去的时候跟他们客气吗？不客气的。殖民者什么时候跟殖民地的人客气过？没客气过。还不是吹得好像英伦岛上面的人最顶级的人都是这样整的。但是为什么隔壁的人就不是那么在意这件事情呢？他们为什么拥有那么强烈的自信心呢？因为他们自古以来连成文史都不记载，他们压根儿不考虑这个问题。谁的神话故事讲得好，谁就是扛把子——是靠这个东西来的。所以人家压根儿不在意你这个引领谁最强；打了砸了的行，我就跟着你干吧。妥协可以妥协，但是他并不是追随，也并没有想要用别人的思想来改造自己，他们就不存在这样的欲望。压根儿没有这样考虑过，也没有尝试过这样的想法，所以他反而没有受到这些东西的影响。以前就没有考虑过这件事情，因此甚至于学习了别人的语言之后，他还可以嫌弃人家宗主国的语言不正宗——就是因为自信。

而这个自信是什么？这个自信其实是人类在自然情况下必须拥有的一种品性。你不受到任何文明的熏陶的情况下，你天然就会拥有一种盲目的自信。文化的开化程度较低的群体，普遍性拥有近乎于猖獗的自信。大家只管到世界上去看，全这样。因此这些人相较于中国人，他们反而没有受到上世纪那些所谓公共知识分子、那些传递的信息的影响。这种狂信也是对付公知的好方法。就是这样的。

因为公知是什么？公知是一种认为自己很有文化、于是要去教化众生的人。但是如果他们真的很有文化，他们就知道他们教的那些东西是坨屎了。所以这些人都是一瓶子不满半瓶子咣当的东西，他们当中没有真正的大智慧的人。有文化也可能很有文采，但是没有大智慧，也没有大心胸，更没有大格局。然后他才会成为上世纪的那种类型的公知——自我贬低、自我否定，并且想要把这个东西传递给所有其他的人。那么什么才是应对这些人的防护盾呢？要么你要智慧超过他的格局，戳破他的谎言；要么你就变成一个压根儿听不懂他在说什么的、这样的一片白纸，也不会被他欺辱。当你什么都不想，根本对他接受无能的时候，他就无能为力了。
