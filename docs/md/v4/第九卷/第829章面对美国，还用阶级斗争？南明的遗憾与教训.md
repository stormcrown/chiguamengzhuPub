# 第829章 面对美国，还用阶级斗争？南明的遗憾与教训

在抗战期间，国共两党达成的共识当中，就包括了“毋复南明旧事”——也就是以南明为鉴，在面对外敌的时候不能内讧。因为当时最大的矛盾是外敌在前，所以呢，其他的矛盾存不存在？当然存在。土豪劣绅压人的那些贼老板、克扣工人的依然存在；包括那些小资群体啊，光顾着个人享受，把自己的个人私利凌驾一切之上的这种事情，有没有？都有。是不是矛盾？是矛盾。但是大敌当前的时候，这些矛盾都要往后放一放，这些人就变成了可以团结的对象，这就是主次矛盾的转化啊。

为什么能够在抗日战争期间，我们意识到这个主要矛盾与次要矛盾的重要性？就是因为南明打了样——也就是外敌当前的时候，他们竟然跑出来搞什么阶级的斗争啊，这就是南明他身上的这些问题。嗯，好。为什么南明没有南宋坚挺？这玩意儿你拿南明和南宋直接比吗？呵呵呵呵呵呵呵呵，完全不一样啊，这两个没有办法放一起比的。为啥？因为南明严格来讲的话，他没有一个正常的、正统的继承延续，他的那个顺位继承有问题。

不要觉得顺位继承这件事情很无趣啊。皇位继承权这件事情叫什么？叫做“国本”。历朝历代都说皇位继承权这件事情叫做国本。如果哪个皇帝没儿子，臣子们要愁死了；皇帝不立太子，臣子们天天要进谏。为什么？这叫做国本不稳。怎么个国本不稳呢？就是继任者他的身份不明确，就会导致你在权力交接的时候出问题。所以嫡长子继承制这件事情本身，他的目的就是为了让权力交接进行得顺畅，避免国家内耗。所以在一个正常的国家里面，我们的这种权力的顺位继承是特别特别重要的。国本你不要觉得它是飘忽的、是没有意义的事情。要么他有专属的身份，他原本就是太子——这是最简单的；或者说呢，他就剩这一个儿子了也行，那就他了，这个也叫做唯一继承权。

然后到了赵构这个地方啊。赵构他原本呢是康王嘛，很边缘，但是到最后也挑不出旁人来了。然后又通过朝堂审议、太后作保，是通过正规程序上位的啊。你看赵构像个废物似的到处跑啊，被人家追得山里海里的，还把自己追得那个雄风不在了，但是他是正常的方式继承的皇位。简单来说就是他是合法继承人啊。他作为一个合法继承人，可以拥有什么样的权利？就是他下面的臣子，不管心思怎么浮动，反正你总得给他点面子，必须得听他一点话，因为这个是权力构造的核心点。赵构再废物，他是一个合法继承人，正常登基的皇帝，就意味着他可以撬动朝堂资源。没有人在这个事儿上面会犹疑、会飘忽不定。谁要是叛了谁就是叛贼，谁从了那就是从龙之功。在他身上是有这种正当属性的。

但是南明朝廷身上没有。为啥没有？崇祯皇帝吊死了，后继无人。所以南明朝廷的第一个障碍就是：到底选谁当皇帝？候选人好几个，每一个帮派啊，都支持自己的候选人。最后是几个兵头给搞起来的，一位知道吗？所以他本身在这个政权更迭上面就有问题。这会造成一个障碍，就是朝堂当中天然的会有些人不服从这个皇帝的指挥。虽然说皇权地位下降的时候，皇帝常常说话不算数，但是呢，他只有一个继承人的时候，就算是装，你也要装得在这个旗帜下面。大家就能够达成一致性。但是南明朝廷做不到，文臣自己就分了两派，武将那边又有自己支持的对象。所以谁才是合法继承人，这件事情就够他们一年到头打嘴仗的了。

来，那个都讲到南明了，就看一下这个《南明史》啊。《南明史》这本书呢，咋说嘛，这本书反正就是个升压小妙招。上火的小伙伴儿，嗯，一般人看了之后都会觉得心里头添堵不舒服。但是呢，这段故事也很重要啊，尤其是在开篇的时候就分析了李自成的错漏，然后后面呢，又分析了史可法的错误啊。史可法是犯过错的啊，为啥？因为立场不坚定，太过柔软啊，太过摇摆。在他自己权力极盛、应该说话的时候他不说话，以至于南明朝廷的内部没有一个一致性的话语权。在这件事情上面，史可法是有责任的。当然了，到后面他自身的这个脊梁骨非常的直，能够直接殉国，能够弥补他之前所有的错漏。但是呢，在南明朝廷这个问题上面，他是犯下过错误的啊——就是不坚定啊，你不坚定其实就是有问题。

嗯嗯，所以这个……这个书呢……啊，这本书啊，这本书是非常值得入手的。只是说你去看他的时候，可能会有点……可能会有点难过啊，这是难免的事情。因为它确实是一个非常遗憾的一件事情，而且它造成了之后三百年的黑暗史。这三百年的黑暗又造成了后面的百年沦丧，所以中国人在过去几个世纪里面所遭遇到的种种不堪，都跟南明的这一段历史密不可分。所以它值得一看，也值得收藏，但是确确实实啊，看了之后不会特别的开心，可是它的价值不菲。

我们在抗战期间的时候，国共两党达成的共识当中就包括了“毋复南明旧事”，就是南明这个时期打了一个样啊——在面对外敌的时候，不能够内讧。有外敌当前的时候，别的事情都要放一放，而不能够搞内讧啊，算计一下什么流寇我们要先打了，或者是地主我们要先收拾了。这些事情，这些矛盾都要往后面靠一靠，因为外敌当前的时候，最主要的矛盾已经变了。而南明朝廷就长期没有办法掌握主要矛盾。南明朝廷的内部其实一直有一拨人，希望能够利用关外马帮群体来镇压农民军流寇。这种做梦一样的幻想啊，这种梦一样的幻想，其实极大地破坏了内部团结起来的可能性。也就是外敌当前的时候，他们竟然跑出来搞什么阶级的斗争啊，这就是南明他身上的这些问题，这个就是南明身上的臭毛病。

所以你看我们抗战期间的时候，是不是谈了抗日民族统一战线？这个抗日民族统一战线是干啥？就是在民族复兴、民族保全的这个民族主义的大旗之下，我们放下心中的成见，放下阶层的差异，极大地去争取那些革命意志不坚决的中坚群体。这个就是抗日民族统一战线——团结一切可以团结的力量。重音够重吗？所以他的这个口号是干什么？团结一切可以团结的力量。因为当时最大的矛盾是外敌在前。所以呢，其他的矛盾存不存在？当然存在。土豪劣绅压人的那些贼老板、克扣工人的依然存在；包括那些小资群体啊，光顾着个人享受，把自己的个人私利凌驾一切之上的这种事情，有没有？都有。是不是矛盾？是矛盾。但是大敌当前的时候，这些矛盾都要往后放一放，这些人就变成了可以团结的对象，这就是主次矛盾的转化啊。为什么能够在抗日战争期间，我们意识到这个主要矛盾与次要矛盾的重要性？就是因为南明打了样。南明当年就是外寇当前的时候在内部整起了阶级冲突。
