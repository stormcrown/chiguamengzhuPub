# 第852章 731档案首公开｜ 原木实验 触目惊心！最后一条看哭百万网友 

事实可以证明，历史也可以证明：日军侵华是没有什么人性之光的。他们是蛰伏数十年，不断打造自己内部的意识共识，不断缔造对中国人的仇恨，然后才开启的侵华。这是敌我关系，而不是什么“他就打了，战争之中，战争结束了，过去就过去了”——不是的。如果这样子想，那就没有办法以史为鉴；如果这样子想，我们很有可能要重蹈覆辙。

不，我们得知道谁敌谁友，我们得知道敌人出现在我们身边的时候会以什么样的面貌，我们得知道处于敌我关系的时候，那不是简单的说“你一个贼老板，你压迫我”——no！真正的敌我关系是你死我活，不是什么要压榨，不是什么欺负了人，而是那种“你在喘气儿，我就要收拾你”的状态，这才叫做敌我关系。

所以，既然大家都出去看电影了（当然我还没看，我准备过两天再去看，因为现在没有时间），你们都看过电影了，应该感知到了。而且应该感知到一件事情：就是任何的文创作品，第一件事情就是选对立场，选对叙事方法。很多以前的抗日影片我们都看过，以前的很多抗日大片，不是制作不够精良，也不是演员演得不好，也不是导演不懂得镜头艺术，但是你永远都会觉得有点什么味儿不对。什么味儿不对？因为他的立场不对。因为过去很多这些大片的导演都谨小慎微，生怕把敌人塑造得太过邪恶，以至于他非得要在这样的影片当中去挑出对方身上的人性之光。

但是事实可以证明，历史也可以证明：日军侵华是没有什么人性之光的。他们是蛰伏数十年，不断打造自己内部的意识共识，不断缔造对中国人的仇恨才开启的侵华。这是敌我关系，而不是什么“他就打，打着就打了，战争之中战争结束了，过去就过去了”——不是的。如果这样子想，那就没有办法以史为鉴；如果这样子想，我们很有可能要重蹈覆辙。

所以，应该做的是什么？应该做的是要了解什么是敌，什么是我。在敌我关系的情况下，对方会使出无数下作的手段，他不可能有任何的下限。在咱们的731罪证陈列博物馆里面，有专门的解释词，有一个词汇叫做“马路大”。参观过的人应该会看到这个词，它有一个专门的解说：什么叫做“马路大”？它的原意，在日文当中是“原木”衍生，含义是“材料”。而在731这个邪恶的组织内部，就是将中国人称之为“马路大”的。这件事情是为了什么？他为什么专门要用这样的一个词来称呼中国人？因为他要用命名的方式，使得操作实验的各种各样的人觉得他所直面的那个不是一个人类。他用命名的方法，使得这些操作的人能够去罪化，去除负罪感，觉得自己所面对的这个活生生的个体不是一个人类，而是一个材料，是一块木头——这就是作用。

所以当对方视我们为“马路大”的时候，我们过去有很多的导演，却在这样的场景之下去挖掘对方身上的人性之光。这是什么意思？这是你屁股摆歪了，板凳坐错了。一切就都是错的，演得再好也是错的，拍得再好也是错的。最近的这部电影我还没有看，听说它里面纠正了这个问题。即便是它导演得不够完美也没有关系，只要屁股摆正了，问题都不大，因为这个才是根本性的问题。

啥照相馆这一点就做得很好的。我是听说他们这一次的电影把立场摆得很端正，没有把对方视为“我要想办法找出他身上人性之光来，我要想办法找出他的善良来，我要想办法找出他的被迫来”。只要你没有做这件事情，将对方真的视为仇寇，那这件事情这个电影，你总归都是能做得好的。细节的瑕疵不重要，细节的瑕疵没有那么重要。更加根本的是，舆论是一种武器——能够理解吗？舆论是一种武器。而所有的武器，除了这刀、这剑是否锋利之外，还有一个最重要的就是你这个武器是朝谁使的。我们过去的很多相关的电影，就是那些著名的大导演们，他不是没有武器，不是刀剑不锋利，而是他的刀口是向内的。哪怕这个武器不够好，也不要紧，你刺向哪里是最重要的。听说这一波的电影，它的方向是对的，万事OK，所有的瑕疵都可容忍。
