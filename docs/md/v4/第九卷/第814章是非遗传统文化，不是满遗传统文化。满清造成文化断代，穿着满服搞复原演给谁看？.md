# 第814章 是非遗传统文化，不是满遗传统文化。 满清造成文化断代，穿着满服搞复原演给谁看？

为什么他们永远都要传递这个谎言？因为他们试图让中国今日的妇女背叛祖先，认为自己媚外投敌是天经地义的奔向文明。当她们产生这种幻觉、产生这种幻想的时候，中国的妇女当中就会出现叛徒。而一个妇女成为了叛徒，她的子女也会跟着一起成为叛徒。这是在破坏中国人的传统向心力。你看着她只是在说脚，但实际上她是在通过一个谎言拆解我们的内部凝聚力。所以，为什么要在直播间里面、在视频里面不停地辟谣？因为造谣一张嘴，辟谣跑断腿。

我现在这个分享就是我自己的一种分享，就到这里。现在呢，来说一下——嗯，有两位这个盟主的粉丝，他们其实也讲到了这个问题。其中一个是一位动漫的博主，他说他在这个圈子里觉得很烦恼，因为他经常遭受到女权相关组织的威胁。如果他的作品里不包含女权——女权是拳头的那个拳啊，田园女拳那个——然后他就会不断地被骚扰、投诉。他感觉非常无助，内心不想，但又不得不屈从。还有一位粉丝呢，他讲的也是有关漫画。他说现在不少知名的，尤其是国漫里面，本来我们是很想支持国漫的，但现在很多知名的国漫里面，经常让很帅的男主留满清的辫子头、穿蜈蚣扣。他觉得非常忧虑，因为这样的动漫对年轻人的影响很深。他又找了那种画得很帅的男主，发现现在很多非遗传承人，明明是非遗传承人，传播我们的传统文化，但是他们也穿满服。所以他感觉到非常忧虑，就是也希望我跟盟主连麦的时候讲一下这个，嗯，希望能够得到盟主的指导。同时呢，也是想对这些我们这些听直播的小伙伴们做一个提醒，让大家当看到这些文化作品的时候，内心有一个这样子的警惕心啊。帮助我说完了。好的，谢谢谢谢。

刚才福娃娃的分享，那其实呢，不是现在做很多文化复兴努力的人，常常会被清代的服饰、清代的习俗所裹挟。为什么会这个样子？这里头呢，其实又充斥着大量的商业利益。我们知道国际顶流网红，实际上也会大量地穿着清装。他知不知道呢？他也许原来不知道，现在不可能不知道。但是为什么坚持不懈地一定要这样子做？因为这个世间是有老板、是有金主的存在的。事实上，现在整个服装市场当中，都充斥着这样的企业主，专门把相应的元素塞到各种各样的时装当中去。大家现在去到街边的任何实体店铺当中去找，凡是与国风、国潮、新国风、新中式相关的东西，里面通通都有满洲元素。这个东西它不可能是自然而然诞生的，它一定是有专门的资金推流冲进来的。

但是如何应对它？悲观毫无意义。然后我说这个事儿怎么办呢？怎么办呢？是毫无意义的。我们只能从自身做起，要相信人民百姓的眼睛是雪亮的，美与丑之间是有界限的。虽然他们用了各种各样的宣传，甚至努力宣传这个东西是传统、这个东西是漂亮，但是当有真正美丽的东西呈现眼前的时候，他就会发现什么是美、什么是丑、什么是有力量、什么是没有力量。

事实上，在今天很多复兴传统文化的人群的内部，也是有分歧的。因为今天我们所留下来的传统文化当中，确实有很多是经历了满清两百多年系统性改造的产物。所以，它的所谓的传统当中，就是包含满清的元素。因为很多我们过去历史上的传统的东西，其实在清代的时候出现了大断代，包括了文娱项目。宋代的瓦子当中很流行说唱啊，说唱表演非常流行，但是你今天再听到说唱的时候，只会想起外国的乐队。这就是因为我们自己内部的说唱，目前只散落在一些地方的戏曲的边角当中。以前它本来是一个大众娱乐项目，却一点一点地边缘化了。为什么会这个样子？因为清代的时候，对于整个的文娱产业进行了系统性的洗礼。因为文娱产品是特别容易影响他人心思的，所以所有的剧本都会受到专门的审核。因此，清代所编撰出来的戏，清代审核之后产生出来的戏，里面多多少少都掺杂了立场上面的反华倾向。所以这些东西当它已经存在了两百多年，你说这个东西不是中国本来的传统，很多的人就会觉得难受。

直接对抗式的方式能够起到的作用微乎其微，对方只会觉得你好极端——这事都已经过去了吗？过去了吗？当你还觉得蜈蚣扣是正常的传统服饰的时候，它就没过去。只有当你真实地意识到了它是什么，它才过去。所以今天要怎么办？把我们的真相拿出来。就像直播间里面，为什么永远都要有人不停不停地讲裹脚？因为有无数无数的清粉、清蚁，他要千方百计地把清代的断足摁到汉人的传统上面，这样子他们才能够成为解放者，才能够成为高高在上的文明之师。因为他们曾经发布过禁止缠足令，虽然他在断章取义，但是他把它拼起来之后来误导你，就制造了一个“满洲人其实带来了文明，不然你还在断足呢”的错觉。

实际上，禁止缠足令是针对谁的？任何政令都有发布的对象。满洲的禁止缠足令是针对满洲女子的，他根本没有针对过汉家女子推动禁止缠足。事实上，他极大地鼓动了缠足的进程。最核心的项目就是乾隆年间的访华使团，从南方口岸的它禁止了。为什么它禁止了它自己控制最深重的地区，它禁止不掉？而离它老远的山西的地方却又禁止掉了呢？合理吗？所以他在说谎。当谎言不断地被重复的时候，真相就会令人迷惑。因此我们要怎么办？我们要从正面来破解它。

中国人是一个什么样的群体？我们讲究孝道，讲究“身体发肤，受之父母，不敢毁伤，孝之始也”。什么叫做毁伤？破坏你的健康、破坏你的美感、破坏你的社会地位，这个就叫做毁伤。而断足是一个很明确的毁伤行为。历朝历代的裹脚，它都是一个装饰型的裹脚，根本不会掰断你的骨骼。但是在清代的时候，墓葬当中大面积的出现了女性的断脚。所以谁作的这个孽？他们为什么一定要把这件事情翻来覆去地说？因为当这种谎言重复千遍，变成某一种直觉的时候，就会有无数的今天的共和国的女子认为：中国的传统就是一坨屎，幸亏有伟大的满人来解放了我们的妇女，不然的话怎么活呀？于是，她就天然地背弃了自己的祖先。

他为什么要宣传这些谎言？为什么要大肆地宣传这种谎言？尽管在晚清时期，太平天国运动的诏令当中，就明文发布了禁止缠足，并且大规模地让妇女放足。然而太平天国失败了之后，太平天国的放足令被满洲的官方直接界定为非法。所以谁才是支持放足的？谁才是反对放足的？在历史的进程当中，这是一个唯一的选项——就是满洲贵族群体强迫汉家女子断足。这是一个唯一正确的真相。但是为什么他们永远都要传递这个谎言？因为他们试图让中国今日的妇女背叛祖先，认为自己媚外投敌是天经地义的奔向文明。当她们产生这种幻觉、产生这种幻想的时候，中国的妇女当中就会出现叛徒。而一个妇女成为了叛徒，她的子女也会跟着一起成为叛徒。这是在破坏中国人的传统向心力。你看着她只是在说脚，但实际上她是在通过一个谎言拆解我们的内部凝聚力。所以，为什么要在直播间里面、在视频里面不停地辟谣？因为造谣一张嘴，辟谣跑断腿。

造谣的人是故意的，是成组织、成建制的，编造了无数的谎言，甚至假造了无数断章取义的史料。他们是存心的，他们是有组织的，他们是有资金的，他们铺天盖地。但是辟谣的只是一些真诚的正常的人，而正常的普通人是不会做这些事情的。为什么要跑断腿？因为你必须不断地重复。每一张辟谣的嘴，所应对的并不是某一个造谣的嘴，而是千千万万个无名无姓、无良心的网军呐。所以为什么直播也好，视频也好，里面总会有一些重复的东西？因为这些东西很重要。
