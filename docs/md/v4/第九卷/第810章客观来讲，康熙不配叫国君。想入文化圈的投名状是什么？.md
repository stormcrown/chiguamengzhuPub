# 第810章 客观来讲，康熙不配叫国君。想入文化圈的投名状是什么？

呃，客观来说，大清都是明君——这是事实啊。你这句话一说就一点都不客观了。一个正常的国君，哪怕他是昏君，哪怕是晋惠帝这样的傻子，他都不会时时刻刻地认为所有的臣民都想造反。他甚至会觉得说：“你们吃不上饭了？何不食肉糜呢？”这句话听起来荒唐，但他内心上其实是坦荡的，因为他的为君目的是希望百姓吃得上饭，他只是理解不了应该怎么吃得上饭。所以他是一个傻子皇帝，他是一个庸君，也是一个昏君，但他依然是一个君王。可是你们心里头那个伟大的康熙圣天子在上呢，发现天下闹起饥荒的时候说：“闹饥荒都是因为你们汉人吃太累了？一天吃三顿能不闹饥荒吗？从此给朕改了，只吃两顿饭就可以啦！”这个东西，这句话你听起来叫做圣天子在上，叫做如何如何的英明神武，然而他说的这句话在良心、在良知、在为君之道上面，连晋惠帝那个傻子都不如。晋惠帝只是笨，而康熙那叫做坏。一个为君者坏成这个样子，他就不是君王，他只是奴隶头目。

嗯嗯，客观来说，大清都是明君——这是事实啊。你这句话一说就一点都不客观了啊。大清一个明君都没有，因为他们没有君。他们没有君，不是明不明的问题，他连昏君都没有，暴君也没有，因为他们没有君。整个满清没有君。好，我们这节就开始讲君啊。啥玩意儿叫君？不是谁冲上来都可以称之为君的。汉字是一个表意文字，每一个字都有它自己的力量，有它自己的本源。这个字你要用对了地方才能够用。所以什么叫做君？记得吗？我们对于男子的至高描述叫做“君子”，对女子的尊崇式的描述也会称之为“女君子”。君子是什么意思？君是社会顶级管理者。我们在先秦时期的社会顶级管理者称之为君。而君子是什么？君子就是君的儿子。请问君子有什么样的要求？君子六艺。君子六艺要求的是什么？他要懂术啊，算学，他要懂数学，算学他要懂军事啊，射箭啊——箭是军事指挥技能，也包括了这个剑当中的内容，其实也包括了祭祀当中的一部分技能，因为我们在祭祀仪轨当中也包括了射箭。然后要会驾驶战车，相当于今天的会驾驶独立坦克。同时呢，要通礼。礼是什么？礼是整个国家制度建设。我们对于一个君子，这些都只是他的技术上的要求，同时还有品德上的要求啊。品德要求，什么叫做品德要求？因为当你要代天牧民的时候，身为一个君，身为一个国主，你要视自己的国人为自己的子弟，你要把你的国民当做自己人来看待。你的为君目的是让这里的人能够安居乐业，能够为你而战，能够为国而战，这才是为君之道。他是什么？他要顺应天道，顺应民心，这种要求才叫做君。他的底线是能够受到全体国民的集中认可。这个认可当中，你说他是个昏君也认可吗？对，昏君会遭大家的讨厌，但是为什么昏君也是君呢？因为从上到下的所有人盼着的是这个昏君能够改邪归正。你有这种愿望，其实就已经在承认了他是一个君，只是这个君不够好，我们想把他换一换，或者改一改，或者教一教。这些人都叫做君。那么无论是暴君还是昏君，或者是庸君，他们事实上都有得位的正当性。即使他做的很多事情不好的时候，他依然可以成为这个国家唯一的旗帜，能够成为这个国家向心力的象征，这个时候他才可以称之为君。这还是君当中最烂的那一档啊。君当中最烂的那一档，他的底线是什么？底线是能够号召，有号召的能力。所以南明小朝廷他拥立的这个皇帝啊，后来南明小朝廷所拥立的这个君王很好吗？不好，不好啊。南明朝廷拥立的君王不怎么样，而且呢，正当性严重不足，因为他的就任就是靠几个军头拥立的。但是这一个朝廷的旗帜举起来，大半个中国的流寇、农民军都集体响应，这叫做什么？这叫做旗帜有用。这个君的旗帜得有用，它才配成为一个君。而清代的十几个皇帝，就没有一个人可以称之为君的。他除了自己的八旗子弟，号令不了天下的任何一个群体，所以他们日常的感慨就叫做“总有刁民想害朕”。所以他把所有的八旗兵马分散驻扎在整个国家的内地，甚至设立内地的监管的哨卡长城，就是为了割离百姓，避免百姓集结造反。
