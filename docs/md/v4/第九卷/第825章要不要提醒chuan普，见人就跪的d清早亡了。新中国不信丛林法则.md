# 第825章 要不要提醒chuan普，见人就跪的d清早亡了。新中国不信丛林法则

谁赢了谁就是我的老大。这个时候，你已经没有正确的是非观了。什么是正确的是非观？正义的那个人才是对的。哪怕失败了，失败的英雄也是英雄，成功的贼子也是贼子。这个就是朴素的正义观。如果我们不能够坚持这个，那么谁赢了谁就是老大，谁赢了谁就是对的。这是什么？这就是丛林法则。

这几十年，是不是有很多的公知就是这样的？他为什么天天说阿美对，天天说欧美文化先进？因为他们发达。他们发达它就是对的。这是什么？这就是奴隶心态、奴婢心态。他没有正确的是非观，只求强弱胜负。只要一个群体是这个样子的，他是不可能上进的，他也是不可能发动真正的革命的。因此，我们后来为什么会革命成功？因为我们启迪了民众，因为我们说起来“不愿意做奴隶的人”。你不想做奴隶，你才会革命；你不想做奴隶，你不甘心做奴隶，你想要一个正义，你想要一个公道，你才会站起来。这个就是中国人之所以在今天勇立潮头的根本原因。

所以当有人跟你说：“哎呀，你看人家发达了，你看人家才是有钱的，你看人家住的房子，你看人家的公司，所以他说的是对的。”只要你心里头相信这玩意儿，你就只能成为狗狗。狗也有可能带两块智能手表，但是它也不是一个人啊。狗能带两块智能手表的，这事儿你们接住梗了是吧？厉害了啊，连连这个梗都接着了。哈哈哈哈。嗯，我下次不说他了，都被你们发现了啊。那个啥，呵呵呵呵，那我一定是错了。呵呵呵呵。好的，嗯，来到点了啊，到点了，再见再见。

实际上，除了我们，世界上的绝大多数的国家，包括所谓的欧美文明国家都一样，包括说现在的各种各样号称自己文明尖端的国家都一样，包括那些彬彬有礼的国家也都一样，全部都是丛林法则。只有中国人不是啊。一定要说的话，可能我们附近的几个过去跟我们渊源极深的国家，也有一定的影响残留。这些是我们当年王道教化推行之后留下来的一些痕迹。其他的地方全部都是丛林法则。你看着文明礼貌讲道理，丛林法则不耽误，它本质就是丛林法则。你看现在这几个月我们搞的中美大对仗，你看阿美那边的人过的日子。咱们就不说他是不是水深火热的问题，而是他的那种生活方式，就是典型的丛林法则下的生存方式——赢家通吃，穷人啥也没有。嗯，对，完了发现阿美的人民是最能忍的。所有相信丛林法则的人都是能够忍耐的。这个世间你会发现，很多很多的人群其实都比中国人能忍啊。之前呢，很多的公知喜欢说：“哎呀，中国人就是能忍，汉人呢，就是软弱无能，特别忍耐、特别苟活。”实际上，除了汉人以及汉文明圈，除了汉人以及被辐射到的汉文明圈，其他的群体都特别特别的异常的能忍，非常非常的能忍。为什么这样说？因为他相信丛林法则。相信丛林法则的人就会特别的能忍，因为输了就是输了，他只会臣服于强者。

就是我刚才不是说到南宋？说到南宋的时候，是不是讲到了蒙古下层的奴隶？投靠南宋的为什么会这样？当时的蒙古兵马是不是特别强？在整个欧亚大陆上面，蒙古兵马都是特别强大的。但是他这么强大的蒙古兵马，在对南宋下手的时候，却有很多的底层奴隶悄悄的就投靠了南宋，跑了。为什么？因为他们自己的内部是一个奴隶制的。奴隶是什么？奴隶就是奴隶领主喊一个口号：“你必须要跟着我一起出去打仗。”他得自费备马、备马鞍、备兵器、备所有的食物和水，然后跟着一起出来打。打完之后，他可以分一部分的好处。这就是蒙古军队的集结，之所以速度那么快，就是因为这个原因。但是呢，唉，他为什么就能够乖乖的去跟着打这个仗呢？是因为他吃不上饭了，主动的跟着干吗？不是的。是因为他的奴隶制拥有非常强的人身依附关系——你家里面不派男丁出来打仗，你这个家庭老弱就不要活了。他是一个强人身依附的奴隶制的管理方式，所以他反而能够迅速地驱动大批的人跟他一起出门干架。机动性能和这种后勤补给速度非常之快，他集结起来的速度很快，很快的都不需要预备粮草，直接就出来了，自带马匹，自带兵器。你相信我们这里的军队还能干这事儿？你不得把一切都备好了，他才会上路的吗？人家就可以做到。为什么？因为他不拿下层当人，弱的人就应该跟着强的人走。这是什么？这就是丛林法则。奴隶制治下一定会灌输丛林法则的，所以下层的人会不思反抗。后来为什么很多的奴隶会投靠南宋？因为他们在这种不断的斗争当中，反而有一部分的奴隶发现对面的人活得比他像人。他体会到了这些东西之后，就有人逃跑，跑到了南宋这边。其实南宋经常性的吃亏，难道他不知道吗？但是他还是要跑过来。为啥？因为当他意识到自己也有可能成为一个人的时候，他就不想当奴隶了。

后来的蒙元时期，蒙古王公日子过得非常好，叫做一等人。但是一等人、二等人、三等人、四等人，下面还有奴隶。这些奴隶是谁？绝大多数都是蒙古底层牧民。对蒙元时期，他们的底层的牧民是被王公当成奴隶来贩卖的，甚至有国际奴隶贩卖。所以后来朱元璋驱逐胡虏、恢复中华之后，为什么会开始“归我者永安于中华”？因为他把这个蒙元的朝廷赶回到了草原之后，实际上有很多的下层牧民不走，有很多的下层牧民就流下来了。他干嘛要留下来？尽管日子也是平民的，生活不见得多好，但是他是个人，他不再是奴隶，不再是任人贩卖的奴隶了。所以才会说“归我者永安于中华”，他才能够收拢这些人回来。主要的原因是什么？我们这里是一个人的社会。尽管说后来满清的几百年啊，经过不断不断的编撰谣言，使得你以为宋明是黑暗没落的、没用的废物时代，但是事实上，我们可以从历史的碎片和角落当中印证出来，你在这两个时代当平民，日子总归过得尚可。这个是什么？拿人当人的时代和拿人当奴隶的时代，是完全不同的思维模式。

当一个人生活的环境是奴隶制，他必然要接受这种丛林法则——强的有理，弱的没理。当你坚持奴隶制的这种丛林法则的时候，你的忍耐力就会特别的强，因为你根本不知道人过的是什么生活，这就是为人奴婢而不自知。这种情况在清末的时候也出现了，这就是鲁迅先生声哀叹的“中国人身上的劣根性”。因为他所见证的就是清末民初的这个时代，中国人身上的这种迷茫、麻木、无助。为什么这个时期这样？因为中国人在这个时候经历了200多年的奴隶制的洗礼，根本翻身无路。当你压根就不存在上升的可能性的时候，你怎么会有梦想？当你没有这些梦想的时候，那当然是闭着眼睛当牛马是最好的选择，至少说心灵上面没那么痛苦。清末的人为什么会麻木？因为你不麻木，你就会反抗，反抗了就会死。所以鲁迅先生当年所哀叹的中国人身上的这些劣根性，来自于哪里？来自于满清200多年的奴隶制的奴役和打压。而只要你在这种打压之下持续的生活并且习以为常，你觉得这种打压是正常的，反正天天如此，你就会必然进入到丛林法则的框架当中去——谁赢了谁就是这里的主人，谁赢了谁就是我的老大。这个时候你已经没有正确的是非观了。什么是正确的是非观？正义的那个人才是对的。哪怕失败了，失败的英雄也是英雄，成功的贼子也是贼子。这个就是朴素的正义观。如果我们不能够坚持这个，那么谁赢了谁就是老大，谁赢了谁就是对的。这是什么？这就是丛林法则。
