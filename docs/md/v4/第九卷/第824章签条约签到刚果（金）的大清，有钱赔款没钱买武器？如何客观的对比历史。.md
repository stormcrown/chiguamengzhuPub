# 第824章 签条约签到刚果（金）的大清，有钱赔款没钱买武器？ 如何客观的对比历史。

谁觉得满清好？要说出理由，和历史其他朝代比较着说。你这个选择方向错了——为什么？因为要说满清好和坏，它没有资格和其他朝代做比较，没资格与任何时代、任何时期比较。国家强与不强，要进行横向比较。不然的话，拿清朝的生产力和秦朝比较，你会发现满清的生产力竟然强于大秦，然后你说“看，还不是我清最强吗？”他们就敢这样吹。任何国家强与不强，要看你在当时世界上的地位、当时世界上的入口占比、当时世界上别人眼中的形象。这就是好坏，优劣就源于此。

无论是辽、金还是后来的元，我们跟哪一波打的时候——就南宋那个废物的样子——也还打的是有来有回的。蒙古的大汗也是死在宋的土地上，蒙古的军队也在宋的土地上一回又一回被拦下来。我就挑最弱鸡的一个，国际地位上是不是强于那个大清？天天吹自己天朝上国的大清是啥地位？任何一个列强来了之后，我表示一下，我宣战，然后打两枪，我投降——不都是这样的吗？所有人有眼睛都能看出来：你再拖十天，对方自己都能哭得不行，急着必须现在立刻马上投降，一天都不能等了。

为什么我们那么厉害？我们的文明辉煌了上万年。请问为什么在近代史上面连刚果都能骑到我们脖颈子上面拉屎、签不平等条约？解释一下吧。要么一万人都是屎坑，要么你清两百多年是屎坑。你挑一个，你觉得我国会挑哪一个？

啥？上AD钙奶啊？为啥？因为那边小破站的特供啊。今天难得那边有一个特殊的品类，谢天谢地，终于让小小破站的伙伴们拔得了头筹。啊，啥？忘了是带货主播了？那不行，不能够。一定要记得牢牢的、记得清楚：我这里是一个带货直播间，我是认真带货的。

嗯，啥？不要过度贬低近代清？北京都不敢这样说啊。不行的话你去告我。你要是实在忍不下去，你就去告我啊。别说什么“不要贬低近代清”，清就不是近代——我们现在定义的近代史从1840年开始算，所以你清不算近代。第二呢，你要是不贬低清，你就得贬低整个中国。你选一个：要么是中国整体自古就烂，要么是清是烂的。你挑一个吧，你是让我们烂五千年、一万年好，还是烂满清两百六十年好？你自个儿选一个。你觉得我国会认可哪一种说法？简单的选项，就这么简单。

要自信，必须讲清楚我们的来路。世间所有人都这样：一个人想要自信，你就得说“我打哪来的？我以前咋样？”这玩意儿叫基础。我的基础好，我才是自信的。所以，我们国家这些年搞文化自信，看看搞的什么文化自信：我们在不停地考古，不停地发掘古典遗迹、上古文明，然后说我们的文明史已经推进到一万年了，还发现里头有“自古以来”。这就是中国人现在骄傲和自信的来源。然后你说，既然我们以前这么牛，我们近代为什么垃圾成那个样子？为什么连刚果都能骑在我们脖颈子上拉屎？解释一下啊。

我们那么厉害，文明辉煌了上万年。请问为什么在近代史上面连刚果都能骑我们脖颈子上面拉屎、签不平等条约？解释一下吧。要么一万人都是屎坑，要么你清两百多年是屎坑。你挑一个，你觉得我国会挑哪一个？嗯啥？满清是跟刚果分账吧？我管呢！我管这事儿呢！反正你敢签我就敢喷。他连不平等条约都签到非洲大陆上去了，还能不能喷他两句？至于他到底是为了卷款潜逃，还是真的就怂了，那关我什么事？实际上他跟清签的那些条约，我都怀疑他在分账。但是他卷款潜逃这件事情，卷的款不就是我们家祖祖辈辈攒了几千年的家当吗？是你落中国土地上也不一定进我的荷包，但是肉烂了在锅里头。就算是土财主截了胡，他置换成了万顷良田，那也是在我的国土上面，哪一天我家翻了身，那就是我的一份结果。为什么我们建国的时候一穷二白，开始积贫积弱？我们攒了上万年的家当去哪了？他是不是跟刚果金分账？是不是卷款潜逃？我管他呢！无论是哪种方式，该喷的就喷。

嗯，啥？谁觉得满清好的？要说出理由，和历史其他朝代比较着说。你这个选择方向错了——为什么我跟你说选择方向错了呢？因为要说满清好和坏，它没有资格和其他朝代做比较，没资格与任何时代、任何时期比较。国家强与不强，你要进行横向比较。不然的话，拿清朝的生产力和秦朝做比较，你会发现满清的生产力竟然强于大秦，然后你说“看，还不是我清最强吗？”他们就敢这样吹。他们从来都是穿越时光式的对比，为什么敢在那里嚣张地说“我们家那个比大明强”？因为他更改记录就行了。咱们不带这么跨时空、跨时代去追索的。

任何国家强与不强，要看你在当时世界上的地位、当时世界上的人口占比、当时世界上别人眼中的形象。这就是好坏，优劣就源于此。我们其他的时代——包括大家心目中最垃圾的弱宋，咱都不说北宋，就说南宋。很多人心里南宋垃圾得不得了，就光说这个偏安一隅的地方，它的科学，周边的人都要跟我们学文字、学服装，震慑寰宇，那叫一个漂亮。服装、鞋帽、各种各样的装饰品、瓷器都是当时顶流。文化影响力从来没有下降过，脚盆的赏梅文化，这一套赏花系统就是宋时学过去的。我们的文化宗主国地位完全没有流失。包括说南宋为什么喜欢买和平？因为太富有了，财力非常优胜。为什么老喜欢跟其他地方勾连、交岁贡？因为交岁贡便宜——前脚交完岁贡，后脚通过走私贸易能赚得盆满钵满。他不想打仗，亏啊。打仗要付硬成本，而且耽误走私贸易。所以在经济、技术、文化、艺术上面，宋都是当时的顶流。甚至于我们现在南海底下还沉着很多沉船，就是宋代的海贸沉船，上面还有大量的铁锅走私贸易。所以南宋这种垃圾的时代都有环球航海贸易——有没有转整个地球不知道，但一定走了很远，有海上丝绸之路。这还是在弱宋时代，而且挑的是更弱的南宋。就那个时候，我们也没有说所有国家都能来揍我们。

无论是辽、金还是后来的元，我们跟哪一拨打的时候——就南宋那个废物的样子——也还打的是有来有回的。蒙古的大汗也是死在宋的土地上，蒙古的军队也在宋的土地上一回又一回被拦下来。甚至有很多蒙古底层的缺口直接就投降了。这是什么？这就是国际地位。我就挑最弱鸡的一个国际地位，上面是不是强于那个大清？天天吹自己天朝上国的大清是啥地位？任何一个列强来了之后，我表示一下，我宣战，然后打两枪，我投降——不都是这样的吗？我跟你们所有宣战，所有洋人都对不起我，我向列国宣战开两枪，我投降。不都是这样干的吗？为什么要这样干？满清根本不敢打外战，从清初到清末都不敢打外战。包括在那里天天说“哎呀，我们平准噶尔啊，没有我清谁能收复西域啊？”牛皮是这样吹的吧？打准噶尔打了多少年？好几十年，屡战屡败。真的是因为准噶尔特别强吗？没有。当时准噶尔部已经弱到了内部分化、两面受敌的程度了，直接能拖出来战的兵马也就是个三五万。就这个程度，腹背受敌、内部分化、瘟疫横生的准噶尔，能够与清之间打拉锯战打了几十年。为什么？不是说八旗兵特别威武雄壮吗？为什么？因为他不敢调动真正的八旗兵，永远都是收着的、拢着的，一次只能调动一点部队。他不敢大规模调动军队，因为他的八旗兵要镇守内地防造反。他敢大规模调兵遣将，内地就瞬间反清复明。

其他时代老是嘲笑说“你看你修长城，就那么点大的土地”。第一，长城不是边界线，是哨卡。第二，其他游牧民族人少，不定居，没有边界线，所以地图学本身就有问题。但是到了清的时候，清还说不修长城——他不在草原上修长城了，他在内地修长城。为什么？哨卡长城起的就是哨卡、调兵及发信号的功能。清为什么要在内地修哨卡？防造反、防联动。所以为什么他外战那么烂？我宣战第二天我投降，他降得比谁都快。所有人有眼睛都能看出来，你再拖十天，对方自己都能哭得不行，急着必须现在立刻马上投降，一天都不能等了。为什么？因为他只要调动军队、发放军饷，就有可能让汉人的指挥官、汉人的绿营兵马获得权力和饷银。而只要他拥有权力、财力，他就会造反。满清就是这个态度，从建国一开始到最后都是这种态度。所以你说“那时候列强多强，厉害了太狠了，我们都打不赢，强成啥样”——武器之间真的有代差吗？有这么大一点儿？这点代差，就凭清的那些白花花的银子就足以弥补了。你以为西方的商人多有良知多爱国？坚决不能把武器贩卖给清？不存在这件事情。他们那里的商人缺德到底，你只要付银子，啥都能买着。为什么最后没有？因为他根本就不敢打仗。清根本不敢打外战，因为他把他自己的主要兵马都放在内地，而且是分散式地放在了内地。

怎么个分散法？第一，各种大地区、大关键节点的大城里面会有专门的满城。然后，在各种各样的交通要道和主要市镇上面，有小股的八旗兵、满八旗兵协同汉军旗兵，设置一小股一小股的分支兵马——多的几百人，少的几十人都有。就这样逐层逐层分下来，每一个像样的镇子上面都有满军配合汉军旗小股兵马，驻点、定时定点地随时应招，干嘛？防造反。稍有风吹草动，立刻调动起来。但这种格局，你想一想：任何一个国家，不管怎么掘地三尺，能够供养的兵马永远都是有限的。他只要把兵马放在了内地，那么在边陲地带就一定严重欠缺可以调动的兵马。他没有人给他打仗。你以为他可以随地征兵？他敢吗？他不敢。满清去随地征兵，征下来的就是推他的人。为什么这样说？鸦片战争的时候，号称是要抗英，但是满八旗的领军带了兵马出去抗英，打仗时路过汉人为主的城池，先回头把这个城给屠了。为什么？他担心自己在抗英的时候后面汉人抄他的后路。这还是一鸦的时候，已经是清代晚期了。清代晚期都是这种局面，你觉得他敢征兵？他可以就地征兵？他敢发动什么战争？怎么可能呢？他不敢打外战。所以经常就是我宣战，第二天我投降，签约，句号。为什么满清签的不平等条约可以签到非洲去？他感动一下谁呀？非洲那里哪怕是一个欧洲殖民者的傀儡政权，他也得听人家的。
