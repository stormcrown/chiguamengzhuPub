# 第831章 《被曲解的统战智慧》——为什么说团结人根本不懂毛选

你居然夸了汉民族啊？夸了汉民族就是贬低了少数民族？这个逻辑哪来的？所以，虽然谁都不知道这个逻辑哪来的，但他们把这两个东西划等号。然后呢，就会变成最后你破坏了团结。那么这种人是什么呢？这种人就叫做“团结人”。

那么，为什么他们会经常使用“团结”这个字眼？所以你要先从最原始的这个“团结”二字开始理解。啊，那么“团结”这个字，包括说“民族团结”这个理论，当时是在什么背景之下，由什么样的机构提出来？他的目标又是什么？这个事儿你必须先理清楚。就是到底什么叫做“团结一切可以团结的力量”。这个呢，是我们当时在进行抗日民族统一战线的时候推出来的，这种手法叫做“团结一切可以团结的力量”。但是注意了啊，这里头呢，一定要先给你们谈一本书，就是《毛选》。凡是谈到这个事的时候，你必须把《毛选》先放到前面。我竟然没有把货上上来？我这太不负责任了吧。

好，然后这个“团结一切可以团结的力量”，他呢，首先要参照《毛选》的第一篇——《谁是我们的敌人？谁是我们的朋友？——社会各阶层分析》。你必须要把这一篇先吃透它，然后才知道什么叫做“团结一切可以团结的力量”。因为这句话的重心在哪？在“可以”二字。你首先要找到可以团结的那一个人，就是进行社会各阶层人群的分析。分析完了之后，找到可以团结的那个对象，然后去团结他。这个呢，就是联合。那这种联合为什么要这样做？注意了，他是在抗战期间推出的。那这个事是什么呢？就是日军的侵华。我们给他定了一个性质：日军侵华定性的叫做什么呢？叫做“亡国灭种之战”，对吧？既然是“亡国灭种之战”，那么它中间的矛盾是什么？矛盾那就不是阶级矛盾。因为对方是以倾国之力来进行灭国灭种的，他没有考虑我们的阶级，也没有考虑他的阶级，而是举国动员，消灭另一个国家、消灭另一个民族，对吧？他是用的这样的操作的方式。那这样的侵略战争，你要如何反抗？你要如何集结起一个有效的力量呢？那这个时候我们的统一战线的意思就是：以民族主义的旗帜，团结社会各阶级。因为在那之前，我们的社会矛盾当中包含了大量的什么大资产阶级、买办，各色的这些压迫群体，使得普通的百姓难以为生啊——这是我们头顶的大山之一嘛，就是这些压迫阶级，这些反动的压迫的阶级啊，这些是我们要推翻的大山。但是呢，当我们面临着亡国灭种之危的时候，主次矛盾发生了转变。这个时候，某些看上去并不是那么好、不是那么先进的阶级，就变成了一个可以团结的对象。所以呢，它就叫做“去团结一切可以团结的力量”。

所以你会发现，抗日民族统一战线上面，最主要要去谈、要去团结的对象是什么？就是那些什么资产阶级啊、爱国企业家，然后小资产阶级，然后包括了一些民营的相关的这些人士啊。那么这一些人，他本身在那之前是属于我们要革命的对象，因为这些人身上都有一些不良的倾向——实实在在的说，那个时代这些人身上都有不良的倾向。所以呢，我们在一开始的时候，是要把他们作为革命对象来看的。但是倭寇一旦入侵，那不好意思，主次矛盾就变了，这些之前要革命的对象，这个时候我们要有条件的联合起来。所以这个是什么？这个叫做抗日民族统一战线啊。

但是呢，哎，谢谢 RYU 啊。但是这件事情在前一些年的时候，被人混淆进来了一个概念。就是他只谈统一战线，他不谈前面的东西了。他只谈统一战线，只谈“团结”二字，他给你搞了一个节选。然后就变成：只要那个人他跟你不是完全一样的，你就必须要团结他。至于说他的主张合不合理，至于说这种方法需要付出什么代价，他不管；至于说我们现在的主次矛盾是不是这件事情，需不需要付出这么大的成本去做这件事情、造成其他的后果，他也不考虑。他只做一件事情：反正“团结”二字是提出来了的，文章里面都写了的，所以就是要团结。这些人是什么呢？这些人就叫做“团结人”。注意了，我们在抗日民族统一战线上面是不会团结伪军、是不会团结伪满势力、是不会团结汉奸买办群体的，对吧？毫无疑问的事情就是，当时我们团结的第一对象是什么？是占比最大的广大的工农劳动者。所以是怎么团结的？方法、步骤、对象他一概不留，他只把“团结”两个字节选出来。你反对谁谁谁，所以你就不团结；你跟谁谁谁不友好了，所以你就不团结；你把别人做的烂事扒出来了，所以你就不团结；你批判了满洲王朝，所以你就不团结满族同胞了。所以你就要求必须要团结他，你必须得夸，不然的话，满族同胞不高兴，他带着东北跑了怎么办？这就是他们的话啊。这句话的背后的潜台词就是：满族同胞有分裂意识。但是问题有吗？那玩意儿是谁干的？那玩意儿不是遗老遗少干的吗？那不是境外分裂势力干的吗？你的对象弄明白了吗？没有。但是呢，他就搞裹挟，他就搞裹挟，知道吧。

所以什么叫做“团结人”？为什么我们现在说“团结人”的时候总是嘲讽？因为他们不是在真的搞团结。真的搞团结，你第一件事情就是要定义：我们现在首要矛盾是什么？我们最需要团结的群体是谁？但是他们会忽略掉一切的前提条件，忽略掉一切的方法论，单独的揪出“团结”两个字。于是呢，谁跳反跳得越凶，你就必须安抚谁。这就被他们作为团结的所谓的纲领。但是问题是你安抚任何一个群体都要付出代价，代价谁出呢？最终不是团结而出。所以这叫什么？叫慷他人之慨。
