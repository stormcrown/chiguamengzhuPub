# 第894章 资本迷思：从先秦货币到清代桎梏 《资本诰》

嗯，请问明清时期真有所谓资本主义萌芽、对封建的批判和个人自由主义的发展吗？注意，这个就是那啥——明清你不要堆一起，不相干啊，全不相干知道吧？明和清就不能够堆一堆儿去。清是没有这个玩意儿的啊。清不但没有资本主义萌芽，清也没有什么个人自由主义。清连正常的自由都没有，它是一个奴隶制分层管理的国家啊。你连行走的自由都是极大受限的。我记得好像网上有人专门做了一个可视化的图，我也忘了是谁做的了啊。干的是啥呢？其实就是清：他会把他自己核心的兵马，包括了他的八旗和他的绿营啊，这些核心的直属的兵马，分拆到各个满城和关键的路口、节点，乡县之间的道路要塞这些地方啊，去驻军。干嘛？镇压百姓造反，避免百姓联动。所以你以为，在这种网格式、高密度网格式管理的地方，你还能随意迁徙啊？啊，个人自由主义，别想了，嘛玩意儿都没有。嗯嗯，啥，嗯——南宁左侯是他，是他做的图是吧？嗯，网上有小伙伴做过啊，可视化的图片，大家去搜一下，你可以找一下可视化图片，比较直观。我只能跟你说，清对整个的中原进行的是军事化网格管理。

嗯，资本主义、封建主义是欧美提出来的。欧美奴隶制的一大把，那他是奴隶一大把，那他是奴隶制吗？是的呀。他们其实早期一直都是奴隶制。嗯，他们最后的奴隶制好像是二三十年前，最后的奴隶制才真正消亡啊——就是欧美地区，我都不包括其他的地儿，其他的地儿现在都还有奴隶制呢。就欧美天天吹牛的这些地方，他们十几二十年前最后的奴隶制才消失，到现在遗毒仍在啊。

然后，刚才不是有资本主义、什么封建啊什么的这些东西啊。第一呢，中国是没有封建主义的——封建主义是西方人起的名字，中国没有封建主义。封建的时代是什么时候啊？封建时代在中国，这里是先秦。打从秦开始，我们就已经没有封建主义了，只有郡县制、中央集权、郡县制逐层管理，他已经没有封建主义了。然后但是呢，他在起名的时候，把这种古典经济类都叫做封建主义。行，就按照他的定义啊，古典经济类的全部都叫做封建主义。那么，我们从唐代以后开始，中国就已经进入到了一个准自由、准自由经济阶段啊——准自由经济、准自由贸易，而且有大量的跨国国际贸易。你觉得这个阶段有没有资本啊？有没有资本？你自己想一下。所以什么资本主义啊，萌芽是流动的，货币就是资本。然后呢，啊，用货币价值来衡量一切，就叫做资本主义。那么中国人其实在先秦时期，就已经注意到了资本的力量，所以我们提出了重农抑商——就是因为完全货币化，用货币来衡量一切，只考虑货币流通，会导致一个国家陷于崩坏之中。所以我们才会重农抑商，重实体而控制商业啊。呃，不是破坏，而是控制，是掌控的意思。商业不能够无端无序地扩张，不能够由它做主，而必须由实业作为基石。这个是我们先秦时期就总结出来的规律。请问这个时候他有没有资本倾向？如果没有，他控制他干什么？所以实际上我们那个时候，就已经有资本萌芽的倾向了，所以才会控制。而后来的什么所谓的明、清资本主义萌芽——清，清是没有的，他倒退得太狠了，他啥也没有。明有没有呢？有。为什么？是因为他的管理能力到中——其实到中期的时候就已经不行了啊。从土木堡开始，明的中央王权的管理能力就开始趋于失控，这个重实体而控制商业的这件事情，他控不住了。所谓的萌芽，其实就是你摁不住他了，他再怎么样也要拱出来的状态啊——这个就是所谓的萌芽。所以有的时候呢，咱们就不能够直接地照搬几个定义、几个概念，咔咔一背就完事儿了啊。
