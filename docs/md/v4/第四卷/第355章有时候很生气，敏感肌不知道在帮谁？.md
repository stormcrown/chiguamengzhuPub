# 第355章 有时候很生气，敏感肌不知道在帮谁？ 

好的，以下是整理后的易读段落，已修正错误词汇，未删减内容：

---

嗯，谢谢观自在啊。记得以前都是叫“小鬼子”的，啥时候开始叫“小本子”的？奇怪，其实就是在啊，这个事儿观自在肯定是知道的。那其实呢，就是在弱化日本的侵略符号啊，这是在弱化日本的侵略符号，而这件事情已经运行多年了啊。然后呢，认真地介入到了普通人的生活，是到了什么时候呢？大约是在20年前啊。他开始逐步地将“鬼子”这个词，从新生的电影、电视剧的环境当中被撤除了。然后呢，他借用了一种操作的手法啊，这个操作的手法就是大家知道那个什么——我们进入到互联网时代之后，有一种大家很清楚的玩意儿叫做“禁忌词”，知道吧？我们进入互联网时代之后，有个东西叫做“禁忌词”。他早期的时候，这个禁忌词表现为什么样的形式呢？给大家分享一下：就是最早期包括论坛啊，包括一些小说频道啊等等这种地方啊，一旦涉及到禁忌词的时候，它会给你变成一个小方块字。可不可以写呢？呃，可以写。写上了之后读者看不看得到呢？看不到啊。当时呢，我曾经写过一个东西，然后上面写的那个词叫做“中国军队”，你们猜上面哪个地方出现了方块啊？中方块、方块。对啊，就是那个时代。好，从那个时代开始，各种各样的禁忌词就一个又一个地跳上了大家的屏幕，其中就包括了“鬼子”啊。不是从一开始就开始的，但是慢慢地就出现了越来越多的平台，把“鬼子”、“日本鬼子”列入到禁忌词的行列。当他进入到禁忌词的领域当中的时候，出现了什么呢？啊，那就变成了说大家总要找一点替换的东西吧，因为话还是要说的，所以就开始替换。那么慢慢的就开始尝试着如何去替换原本要说的这个词呢？然后就开始出现了所谓的“小本子”。那么“小日子”这个词出现得更晚一些啊，“小日子”这个词出现得更晚，他跟一些在采访之中的口误有点关系啊，那就是所谓的什么“小日子过得不错的”，什么什么什么人之类的。本来是想叫“小鬼子”，但是一看是公开采访，所以呢，就临时地改成了“小日子”什么什么什么什么东西。那这种时候呢，嗯，变成了一种所谓的网络流行词汇，再叠加到前面的所谓的禁忌词，最后“小本子”变成了“小日子”。然后后续呢，就被各种的影视作品不断地叠加这种所谓的政治正确言论进去啊，毕竟说蜜月期不要拉仇恨啊，你又叫“小鬼子”，你羞辱了友邦是吧？很多人有这种想法的，放心啊，20年前有这种想法的人，甚至是舆论界的主流。

那最后造成的现象是什么呢？就是有一部电影叫做《铁道游击队》，还是一个演技很不错、我很喜欢的硬汉演员去出演的，大家可以去搜一下。那么这个电影我一共看了10分钟，然后就看不下去了。为什么？在这10分钟的开场里面，他就已经在接头的时候，在一个完全中国人的场合下面，在地下人员接头的过程当中说“日本则”如何如何——这是一个完全不符合中文语境的生造词汇，很明显是一个在回避“日本鬼子”的一种操作方式。回避“鬼子”这个词，他才会专门地、很硬性地、很刻板地去创造了一个所谓的“日本则”。听了这个之后，我整个人都不好了，所以我就把它关掉了，我就没有再看了这个电影。当时我甚至是怀着很好的期待去瞧的，我也很喜欢那个演员，我觉得它设置的氛围、技术都很不错，效果也很好看，但是10分钟之后我放弃了，我忍不下去。当他把鬼子叫做“日本则”的时候，他已经出现了美化。因为为什么我们会将侵略者叫做“鬼子”？包括清末的时候，我们也叫“洋鬼子”、“红毛鬼子”。为什么要这样叫？这是一个对对手、对敌人的否定性称谓。我对对手予以否定和贬低，于是呢，我的心气儿就会提起来。所有人都是这样的，不止中国人如此，其实其他的所有的人也是如此。在任何的对峙、战乱的期间，都会给对方起一个否定性的名称，然后去强行的称呼他，从而形成一种心理上的优越感。而这种心理上的优越和俯视的感觉，会提升我们的士气，让我们在战斗的时候更加的有勇气，更加的坚决。这就是我们在战争期间，如果喊着“打倒小日本鬼子”，那么我们就可以扛着枪冲锋陷阵；但是如果他喊的是“打倒大日本皇军”，那这个时候你腿都要打颤的，你把对方叫那个名字的时候，你不得跪下来磕三个响头吗？所以为什么要进行这样否定性的称谓？是因为心虚吗？不是，是因为你要构建一个心理优势，你必须要做这件事情。所以当我们的抗战的影片当中，会用含糊的、其实的所谓的“日本则”这样的奇怪的词汇的时候，这说明投降的人可太多了呀。所以我当时非常非常地厌恶这个影片啊。

哎，来，欢迎大家来到直播间啊，别的事情先放到后面，我把你们的那个付费都卡在屏幕上了。我现在，我现在老聪明了，我现在越来越聪明啊。啥是叫《铁道英雄》嘛？片子不记得了，反正讲的是铁道游击队的故事啊。来，那个啥，嗯，我们来整一波那个带货哈。这个是人民的胜利——中国大阅兵纪念章，还没有收藏的小伙伴可以下单一套哈，我觉得还是非常值得的。尤其是当我们看过了直播之后，我真的是觉得，好几十年没有这样的叙事方式了，我们的革命逻辑又一次回来了。为了这个快乐，我我觉得完全可以再一次地推送哈。现在的发货速度比之前快多了，之前那个之前是预购式的啊，现在可以发货的速度比较快了。这个叫做“人民的胜利”，那它的名字起得也好，名字起得也好。我们这一次的阅兵走的就是人民叙事路线。我们的抗日战争是怎么取得胜利的？它就是人民的胜利。谁站在了人民的一边，谁就会取得胜利。国民党反动派为什么失败？是因为他没有拉到国际援助吗？他拉到的援助老多了。是因为他没有精英吗？当时中国最多的精英就在为他服务。他没有装备吗？他没有兵马吗？他什么东西都更好，为什么最后他取得的成效不行？因为他没有站在中国人民的一边。他一面假装抗战，一面在出卖着人民的利益。而老百姓的眼睛是雪亮的，你把它卖了，他不可能永远都不知道。所以当时的反动派，就是一次又一次地在挑战着人民的底线。所以谁取得了胜利？对，人民需要英雄，有英雄的带队，人民才会取得胜利。但是什么叫做英雄？站在人民一边的才能够叫做英雄，因为人民就是英雄的力量。诶，好，欢迎来到直播间啊。哎，我我我我才说好聪明，我把那个什么你们的付费的页面都卡在那儿了，结果我现在就全看不见了，他还给我缓存消了吗？还有这种事儿的吗？调戏啊，这这有点耍流氓了吧，啊，过分了啊。
