# 第324章 啥是抽象的百年大翻译运动 

### 关于百年翻译的问题

阿拉伯人能够收集和翻译西方古典书籍，也能学习中国的造纸术，但他们却不收集和翻译大唐的书籍——难道是盛唐看不上吗？哼哼，你这说的差不多把答案都给怼出来了。啊哈哈，嗯，什么？《中华帝国全志》这本书可以看吗？可以啊，但是得看你用什么样的角度去看。没事，我相信你能够找到适合自己的角度。可以看，可以看哈。

那么，来讲百年翻译。首先得说一下什么是百年翻译。哈哈哈哈哈。嗯。

自从西方人打算给自己安排一个古希腊的老祖宗之后，就出现了一个问题。你说古希腊是你老祖宗，说古希腊特别牛，厉害得不得了，哎呀，人类文明的源头啊！那个高峰期简直了，让人叹为观止，啥都有。结果吧，你还有个黑暗中世纪。黑暗中世纪的时候，那简直连洗澡都不会，上厕所都上不明白，连个痰盂都不知道应该往哪放。你这个中间落差是不是有点过于严重了啊？

所以呢，就出现了一个问题。当他想说2000多年以前的古希腊是他的文化源头的时候，就会出现一个问题：既然你的祖先那么了不起，那怎么你之前都不知道把屎泼哪儿呢？你解释不了这件事情。所以，他就必须要给这个源头找一个来路。就是，“对，那是我祖宗，那是我祖宗发明的。但是那东西都不在我这了啊，离散了。”好吧，离散了也行。然后离散了之后呢，你又得倒腾回来吧？不然我怎么知道我有这么个祖宗呢？对他的解释就是：当年的那个古希腊文明衰落了之后，它就离散了，然后我们就找不着了，没有了；文化呢，就消失了、消散了、不会了。以前的那些干干净净、漂漂亮亮的街道也都没有了，然后遍地都是屎了，是吧？以前的在山科拉里头都能够吃饱喝足，还能盖大学了；现在呢，农垦区里头农民都得饿肚子。这个技术水平一落千丈啊，太遗憾了。那你是怎么发现那玩意儿是你祖宗的呢？

所以他就得给这个祖先的文献找一个来源。但是你要去找一个真正的来源，你得造很多很多的文物古迹，然后呢，还要有文化、文字的书写的载体是吧？一脉相承地传下来。知道的就是老子这一本书，在正常的传续过程当中，知识的损失量都已经叹为观止了，对吧？那你想，这种中间隔了黑暗时代的传递，折损量得多大呀？你怎么能够找得着呢？所以他给自己安排了一个冰箱——他安排了一个冰箱啊。注意，当年他们也给中国人安排了一个冰箱，有人记得吗？他们给中国人安排了一个冰箱。当然了，这个是他们在东亚地区的孝子贤孙、他们的学生，假设他自己是我们的冰箱。今天还有人说“汉唐在日本”，汉唐在日本？有有人喊这玩意的吧？有啊，这个就是冰箱论。

好，这个招谁开始的呢？欧洲人开始的。欧洲人开始给自己找了一个冰箱，因为他不用冰箱的话，他解释不了他这个东西没有办法传承。以前那么老多的东西一无所有，连屎都不知道往哪泼，到了这样的地步，连澡都不知道该往哪洗，太过于离谱的断代了。这个时候他必须得找一个原封不动地保存了古希腊文献的冰箱。于是他就选择了一个地方——他就选择了阿拉伯地区。为什么选这儿呢？因为当时正逢阿拉伯世界的衰落期啊。衰落期的意思就是：不管他说啥，阿拉伯人都“嗯、嗯”，然后继续在家里干仗。所以呢，他才会去选择这么一个地方，交通上面又有可能性。好吧，之前呢，虽然说漫长的时间里头，阿拉伯人都是跑他这里来抓人、抓奴隶，出去贩卖白奴，双方之间的感情并不亲密，甚至是有世仇的。但是这个阿拉伯人莫名其妙地就当了一回冰箱。他们非常热情洋溢地去把古希腊当年的文献逐字逐句地翻译成了阿拉伯文，还把人家原作者的名字写得特别清楚，然后就存在了他们的图书馆里面。但是没用啊——注意，没有应用。好玩吧？他学了很多的知识，但是阿拉伯人自己不用这些东西，就存在图书馆里面，动都不带动的。老厉害了，我就没见过这么高风亮节的人，反正就突然之间这么崇高起来了，特别伟大啊。然后终于到欧洲崛起了，然后呢，他去找到了他祖宗的这个东西。有没有觉得这个路径就特别像游戏里面的挖宝？这个玩意儿它就是游戏里面的挖宝，挖宝项目哈。他就跑去那挖了一个祖先留下来的痕迹，然后啊，他就进行了翻译运动，这个就是百年大翻译。你看我说了一个百年翻译，扯了那么一长篇，这个呢，就是百年翻译运动。

说这是一整个图书馆太大了，然后不知道咋翻译，然后一点儿一点儿地又把它给倒腾回来。所以呢，他所研究的什么文化复兴、文艺复兴，你要复兴，你总得之前有兴过的那个“兴”。就从阿拉伯的这个冰箱里面原样地倒腾出来，然后又翻译回了他们自己的欧洲文字系统里头，又原样地复刻了。然后呢，他就文化复兴了啊。这个就是它大致的一个流程，这个就是百年的翻译运动。他为自己的这个文化的复兴找到了一个理由。但是注意了，之前的啊——之前古希腊时期的那些文献的原文基本上是零，基本上是零。然后呢，这个阿拉伯电冰箱啊，它里面的那些抄本，翻译的初代抄本、初代译本是零啊。所有的后来呢，都是说“我从那个里面翻出来的这个部分”才开始逐渐地有了版本，在那之前没有版本的哈。这个就是他的一个复兴的过程。当你能够按照正常的态度去看待这个过程的时候，就能够理解它有多么的荒诞了。

要知道，光是常凯申这一个人，因为拼音文字这件事情，就导致他的名字这样的一个角色——不管你喜不喜欢他，你都一定认识他的一个角色——竟然在翻译的过程当中就可以失真到这么离谱的程度。然后你说把一套东西——一个你当时早就已经不用了的古希腊文，一个已经失传了的表音的字母文字的古希腊文，翻译成古阿拉伯文的文献，然后你又将它翻译成近现代的欧洲文字，不管是拉丁文也好还是什么文也好，总之呢，你这样子二传手地倒腾回来，然后居然还能够看得懂？我建议他们大学里面就取消那个什么莎士比亚文学这个专业吧。啊，至少说这个专业里头不要再学什么古代英语的这门课了，有什么可学的呀？事隔两三千年他都看得懂，才隔400年学什么课啊？还学一个什么古英文课，弄得好像是个外文似的，还得专门学习，骂谁的呢？隔几千年不都能看明白吗？隔几百年你自个儿的语言你看不懂了呀？开玩笑，这门课取消！他这门课的存在就是对英文的羞辱。
