# 第396章 错怪了圆嘟嘟，没有走私而是公开卖粮？ 

文玩市场上面什么东西值钱？什么东西不值钱？主要看有没有稀缺价值，倒不分好坏。你要是把秦桧当初跪的那块石碑拿出来，作为珍品、孤品，保不齐在文物市场上面也能拍出天价来。但是不代表这个东西或者这个人有什么价值，知道吗？文物市场的价值取决于稀缺性。所以呢，清朝的那些铜板不值钱才是合理的，因为它活的时间太长了，发行的数量有点太大了。大量的存世就会让它不值钱，这个事儿才是合理的。

啥？嗯，说毛文龙是敌后作战，袁崇焕是阵前走私。可人家非得说毛文龙才是走私，请问要如何有力反驳啊？为什么要动不动就是有力反驳呢？好奇怪啊。对方直接给你空口造谣的时候，你不应该直接喷回去吗？啊，对，对方造谣的时候，你不应该直接现场喷回去吗？袁崇焕直接贩粮给敌人，这件事情不算走私是吧？哦对，是明堂正道的，卖的不算走私。哎呀，太棒了啊。对，袁崇焕没有走私，他都是公开卖的。然后你说那个啥啊，毛文龙走私。那毛文龙走私了没有呢？啊，毛文龙保不齐真的走私了。怎么走的私呢？毛文龙确实做了走私生意，他把后金手上的那些什么雕啊、貂啊这些买卖截了一些胡。他们两边都是倒着来的。袁崇焕这玩意儿啊——注意，袁崇焕这个东西，你一定要说他没走私，我是支持的。他没走私，他都是直接卖国家的战略资源的，明着卖啊。嗯，那个什么学派还搁那呢，他这个名字一瞅就像是一个卖麦克的，怎么起这么个名字呢？你说，嗯，袁都督是直接卖的，能算走私吗？对，他确实算不得走私啊。那能怎么办呢？那个啥，袁都督都是通过制度性的方式直接贩粮的。呵呵，发了卖粮的牌照出去。大家都可以直接确定的一点，就是当时建州部非常缺粮，已经很困难了啊。然后呢，说不能够往北边卖粮啊，一定要把关口给我掐死了。关口卡死了，收紧了粮袋子，对方就没招了是吧，就开始困难了嘛。然后是袁都督力排众议，说某一个时降时叛的蒙古部啊，没毛病，铁铁的绝对是我们的死忠啊，把粮给他没事。然后就把粮食卖过去了，人家倒手就直接卖给了建州部。所有人都知道他们之间过从甚密，所有人都知道这一个部落并不是非常的忠诚啊，时降时叛。但是袁都督自己公开地说，这对人家老忠臣了，是自己人，卖卖卖快点啊。

朱祁镇是被宣大总兵在土木堡阴了，嗯，也先是来护驾的，于少保是得利者，是有这种说法。就是那个土木堡之变啊，土木堡之变的话，它本身就是一个有争议的事情，因为它在叙事上面，原本它的叙事上面就有问题。因为呢，土木堡当时，土木堡之变的那个时候，瓦剌部是臣服的状态。而当时朱祁镇率队出去巡边，他巡边的这个动作被人描述成所谓的出征，说他带了是10万还是20万兵马出去。但问题是在当时他的那个条件下，他不可能征到这么多的兵，一次性在那个时机就给带到北边去，这件事情他根本就做不到，所以不可能是这么多的人。然后同时呢，他所巡边走过的这个地方，还是在明王朝的实控区范围内。也就是说，如果没有边军把门打开，也先是进不来的。所以呢，它出现在土木堡这个地方，出现这样的事变，本身就已经极大的存疑了啊。但是呢，现在在史书的内部记载当中，具体情况到底是怎么样的，还不能够确知。我们只知道一件事情：其实不光是在朱祁镇期间，瓦剌部是臣服于大明的，实际上一直到明代的晚期，到了崇祯年间，瓦剌部都还在给明王朝交税。这件事情是北边的大毛那边的探险者去记录下来的。他们当时呢，试图东扩，所以派了一些先遣的人员跑到这边来探底子、踩点儿。然后有一些队伍就行走在草原上面，观察到了各个部落的往来，他就会记录下来。所以实际上，明王朝在蒙古高原、蒙古草原上面的管理与收税的行为，在清代是被系统性地削了档案的。但是呢，在大毛的笔记当中，在大毛的官方的记载当中，却有他们的先遣队伍在草原上面明确地看到了瓦剌部在晚明、明代的末期，实际上都还在给明王朝交税。

呃，一种说法：明建国时因为华北空虚，所以分封藩王种植树木，兴修水利，恢复华北生态。现在的新中国植树造林才达到的。明末这个统计不清楚，但是有一点可以明确的一件事儿啊：就是清代毁林毁得特别严重。明代是否有那么大面积的植树造林，这个事儿不能够那么确知。为什么？因为清代毁林毁得太彻底了啊。清不让烧煤。那为什么不让烧煤呢？啊，清不让用火器啊。第一，清不让用火器；第二呢，清专门控制了矿业的发展啊。清代是严格控制矿业发展的，为什么？因为矿业的矿工是特别有组织的人，是一群极容易发生兵变的群体啊。矿工群体是特别有组织、特别有纪律的潜在的兵员，组织力极强。所以一旦矿工、矿业出问题的时候，他们就是现成的预备役。清王朝的时候，他是一个极大的压制生产力进步的一个阶段，尤其控制火器、热力相关的这些先进的能源技术的发展。谢谢堂皇大道啊！他为什么要这样子做？因为当你的技术、当你的能源运用的技术开始逐渐地迭代，变得越来越先进的时候，所有的老百姓手上都可能会拥有更强力的武器装备。一旦他手上有了这些东西，他第一个要干翻的就是八旗的统治者啊，这个事是毫无疑问的事情。因为什么叫做总有刁民想害朕？就是统治者和他治下的所有的百姓，彼此之间是一个完全敌对的关系啊。整个清代的二百多年局面都是如此，当时的大家无论是上还是下，心知肚明，甚至无论是中还是外，也都心知肚明。老百姓和朝廷之间是一个完全对峙的关系，而不是说什么普通的压迫呀，就不是这事儿啊，是一种敌我关系、敌我矛盾。在这种局面下，八旗的统治者并不希望百姓手上拥有任何先进的工具啊——就不说武器了，他不希望他拥有任何先进的武器。所以他会刻意地控制，甚至刻意地压缩各种各样先进技术的产业。其中最重要的一样就是军事产业啊。所以呢，军火包括铸造、包括火器的生产、包括火铳啊等等这些东西，在清的时候都已陷入了停滞和倒退啊。它比明王朝的时候技术要倒退了很多，不是没有前进的问题，而是倒退啊。所以呢，他在晚清时期与列强作战的时候，需要到库房里面去翻崇祯年间的火炮，因为崇祯年间的火器的铸造比清末的时候要好，就已经是到了这样的程度。所以呢，整个清王朝所做的事情，就是要让技术停下来倒退，普通的百姓手上不要有任何的先进的工具。

那这个时候燃料就变成了一个问题啊。注意了，我们用什么样的能源，其实和你处在一个什么样的生产力发展阶段，是要适配的。例如说，我们在宋的时候，就已经普遍性地使用煤炭了。这是一个什么样的适配的阶段呢？就是宋，我们已经进入到了初级火器阶段，它的火器的质量、广度都远远不如明，但是呢，它已经进入到了初代的火器发展阶段。所以呢，跟它所匹配的就是它在能源方面所使用的呢，是普遍性的煤炭啊，普遍性地使用了煤炭。然后到了清的时候，他遏制了火器的发展，甚至直接把火器都给收到了库房里面去啊，普遍性地开始搞起了刀马作战，进入到了冷兵器时代。那所有的生产都开始控制，都开始压缩和倒退的时候，它的能源产业也必然会倒退。更何况矿工是一群那样的潜在的预备役啊——矿工就是潜在的预备役，有组织的团队，那更是统治者所无法容忍的存在。所以整个清王朝，它在矿产的开发上面，都是一个持续性的倒退和萎缩的状态。严重欠缺矿产、严重欠缺煤炭的情况下，普通人所要使用的燃料就只剩下了木柴。于是呢，你就会发现什么树啊，遮阴？你开玩笑的，你是什么贵人吗？你居然还要树荫？你一个普通人，你只要想办法让自己活下来，还有开水喝就已经——唉，开水都想不到了。能够想到让饭能够做熟，就已经要谢天谢地了好不。那有的时候连烧水的柴火都凑不齐的时候，你开什么玩笑的呢？还说哪个地方要有什么树林子？没有，没有这回事啊。京城的周边所有的山全都光秃秃的，啥也没剩啊。
