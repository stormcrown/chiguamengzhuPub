# 第329章 如何学好近代史，高考拿高分（阿美种姓制） 

嗯，咱们崛起之后怎么解决日本这个祸害？我现在最担心的就是，咱们一崛起，日本直接“咔嚓”一跪。我就希望他能硬气一点，你知道吧？但不太可能。大概率咱们一起身，他就“咔嚓”跪地上，还是那种滑跪出溜一下，不一定出溜到哪去的那种。他们自古以来身段就这么柔软，弱则卑服嘛。墙倒屋塌？他真跪了，这事还不知道怎么办好。但愿他到时候不跪吧。

谢谢轻雾啊。自从知道1644史官，整个人都通畅了。以前读书觉得怪怪的东西都想通了，因为，因为他才合理。我念书的时候对于近代史感到非常痛苦，也不知道有多少人和我有同样的感觉——就是学近代史的时候总是觉得极为屈辱，非常难受，整个人都不好了。不知道有没有这样的啊？你要是学学近代史就特别痛苦，特别难受，恨不得不想看，学不下去，想把书撇了的那种。来扣个一。嗯，一学近代史，糟心烂肺的那个味儿就冲出来了，特别难受，非常不爽。但是你只要确认了1644史官，你学近代史的时候就会非常顺畅。为什么？因为当我们的衰落节点在1644的时候，1840那就叫做触底反弹啊。你那个时候看到的就是我们的触底反弹，你看到的每一次、每一个行动，全部都是我们复兴之前的号角。你这样子一下子，咔嚓一下子，整个学习都能上进，你知道吗？

嗯，诶，谢谢葡萄牙。欢迎大家来到直播间啊。嗯，啥种姓制度？你们又立了什么flag呀？呵呵呵呵呵呵呵呵……你们这是又立了什么flag呀？但是那个，种姓制度这个东西真是一个很厉害的制度。有多少人跟我有共同感的？可以不喜欢，可以痛恨，可以厌弃，但是我觉得种姓制度这玩意儿真是厉害啊。有这种感觉是吧？有这种感觉是吧？有这个感觉就对了。你可以瞧不起这个东西，你也可以蔑视它，甚至可以痛恨它，但是请尊重它。为什么？因为种姓制度这种设计太牛了，这个设计真的是太牛了。它充分地利用了人的从众心理，也充分地利用了人的求胜心理，知道吗？既从众又求胜——就是我背负了，但是我赢了。在整个种姓制度当中，它就是这样的。这套东西的设计好狠，真的是好狠。谁在种姓制度下面生活个一代两代人，什么爹都能忘了。

在阿三那边的种姓制度下面啊，每一个人都有他的一个社会定位，这个社会定位呢，是它社会层级当中的一部分啊。它首先得确认自己处在哪个层级，然后在这个层级上面，下面有一批，上面又有一批。你对上面的那个要无条件的服从，对于下面的那一个，嗯，随便你蹂躏。这个东西呢，就是一个种姓制的表象。那么种姓制之下，你说光踩一个，又捧一个，多累呀？注意了，种姓制，它干了一件事情——他把一个人的社会阶层定位给你锚死了，不准动，严禁改变，不可能改变，除非转世。到了这种时候人没了负担。今天所有人都觉得自己压力大，要卷，为什么？因为你想上进啊。你想上进，所以你才卷，所以你才压力大，有的时候睡不着觉，上火，着急——我今天老板喷我了，明天我可咋办呢？你为什么有这个想法？因为你想上进。为什么你想上进？因为你确实有上进的可能性。中国人自古以来就有上进的可能性，所以你老是忧心忡忡的，就想着我怎么能够更进一步，嗯，有点啥事我可以冲上去。这是中国人的思路。

但是阿三从来不这样想。为什么？因为他的社会定位锚死了就放在那儿了，稳稳当当地搁在那个地方了，不可能变动，这就让他充满了确定性。当一个人充满了确定性的时候，其实内心是非常安稳的。所以咱们看他们就特别难以理解：你过着日子苦成这个样子了，人人都瞧不起你，踹一脚，结果你还踏实的，不要不要的，觉得自己炸了天，骄傲得不得了。为什么？因为它的社会阶层既不可能上升，也不可能下跌，非常稳定地锚死在了一个地方。所以他的内心情绪的稳定性极高，没有任何的负担——没有要上进的负担，也不用担心阶层的下坠，同时也不需要有任何的道德负担。肆无忌惮，想干啥干啥，为啥？因为你改不了，反正也是改不了的，反正也就是这个样了。他永远都能够为自己自圆其说。

谢谢清朝老贼啊。嗯，破罐子破摔。所以这一套东西啊，是认认真真的，我就认为这个玩意儿它太牛了，这个设计它好狠啊。谢谢玛雅套，谢谢旧时代酱油。糖是第一次，嗯，民族复兴明是第二次，现在是第三次。算吧。嗯嗯，警惕，警惕汉人的种姓化。其实这个世界上搞种姓制的人是很多的，有的地方是明显的种姓制，例如我们隔壁；有些地方是隐藏的种姓制，隐藏的种姓制也是种姓制。大家有没有看阿美的电影？阿美是一个天天宣传美国梦啊，宣传什么上进、宣传奋斗的一个地方，但是有看过他们的电影、电视剧吗？就是早些年他们还争气的时候——哈比不是在说他们今天的群魔乱舞，今天的有点变态了，咱们就不看了——就是之前他们还阳光上进的那个时期，他们的电视剧啊，你都会看到阶层的不可逾越。你似乎是在哎呀，追求梦嘛，理想嘛，谁都能成就嘛，你可以发财，但是阶层不能逾越。就在他们的电视剧里，赤裸裸的就是那玩意儿。所以他们那儿也是种姓制，只是隐藏的种姓制。所以为什么动不动就什么玻璃天花板啊、隐形的屏障啊、什么东西啊，就是没有命名的种姓。嗯，好，欢迎来到直播间啊。
