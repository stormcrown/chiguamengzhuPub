# 第318章 别动！动则国灭 ，蒙主分享阅兵观后感 

那啥，阅兵看完之后有分享，看后想说的方面吗？是要考观后感是吧？我跟大家说啊，念书的时候，每一次学校里头帮我们组织看电影、看剧目，都让我们写观后感。每逢这个时候，我就特别的头疼。我觉得观后感比那些只给两三个字的命题作文要难好多。但是这一次真的不难。怎么说呢？我们这些年一直在说我们的目标是什么啊，我们的目标是中华民族的伟大复兴，我们的目标是复兴。来，既然目标是复兴，说明一件事：就是我们之前曾经兴过。那我们之前曾经兴过的时候，是什么样的一个兴法？那以前我们兴盛的时候是啥样的？今天我们的阅兵给我们展示出来了——什么叫做大军已至啊，别动，动则国灭。呵嘿嘿嘿嘿哈哈哈哈，知道吧？啊，就是这个意思啊。我们以前看书的时候，什么叫做别动？动则国灭。现在我们看着了。不用喊那么大的声音，我们只需要在自己家的城门楼子前面走这么一趟，人人都听见了这个声音：别动，动则国灭。做到这样的程度叫什么？这就叫复兴吗？我们上一次兴盛的时候，我们之前兴盛的时候就是这样的。落后就是从清朝开始的，谁也洗不了。对啊，就是这样的嘛。啊，简单来说就是：既然我们已经走到了复兴的道路上，而且看到了曙光，见到了，我们有一部分已经在当时的水平上达到了基本一致。到了这样的技术水平时，请问其他的山头要不要占？其他的山头要不要占？就是舆论的高地，我们要不要拿下？要知道啊，只要相信什么满清也有盛世，给我们开创了什么什么东西，你总结不出来。我们是要复兴的，为什么？因为从清朝入关的那一刻开始，就没有哪一天过过好日子，也没有哪一天取得过进步，也没有哪一天取得过真正的成就啊。所以当它只有这样的档次，当它只要在这个地方霸占的那一刻，就每况愈下，一天比一天差的时候，你说我们的复兴当中包含它吗？嗯，开玩笑吗？所以如果解决不了这个问题，那最后就是像封建王朝一样烂，所有的皇帝都是狗皇帝。我们现在的公屏上还有这样的人呢，说你这娘们就是个帝王粉。在应该有君主制的时候有君主制，那是先进的表现。马克思主义的唯物史观考虑的就是这个问题：不同的时间需要不同的制度，不同的时代需要不同的管理方式。什么叫做先进？符合当时的生产力水平，并且能够促进生产力水平发展的，就叫做先进。什么叫做落后？在大家都已经先进了的情况下，它还倒行逆施的那种，就叫做落后。所以谁是先进呢？在应该拥有皇权的时候，在皇权应该一言堂的时候，拥有了郡县制管理的高度一致的调度能力，这就是先进，所以它是拥有先进性的。然后呢，在生产力极大发展，皇权应该开始退居二线的时候，做出这种尝试，那这是谁呢？这是明朝的皇权并不集中，实际上它的衰落当中有一部分原因就是皇权把握的程度太低了。但是这是什么？这是先进性的表现，因为生产力发展到了一个新的水平，所以这时候它应该尝试变更新的管理方式了。只是在它蜕变的过程当中，被另外一个更加落后、更加野蛮的群体，用更加落后的制度截了胡。这是它倒霉的一种方式。但是无论它怎么样的失败，总之后面来的那个是个垃圾啊——我有点侮辱垃圾了啊。诶，好哦，欢迎来到直播间啊。

嗯，啥宋呢？宋是被唐坑的啊。我知道很多人特别喜欢盛唐，我这里一直在说一件事：唐呢，是有它的优势的，但是唐埋下了巨雷。宋身上所有的短板，实际上是在给唐擦屁股。对，很多时候那个去批判宋啊什么的，你最多也就是说它有点软弱，但是它为什么这个样子啊？它的原因是什么？它的这种决策上为什么总是有一点你看了之后有点哀其不幸、怒其不争的感觉？它为什么会做出软弱的这样的选项？其实是整个宋都在为李唐王朝擦屁股。不是它在那个地方留下来的大坑，宋不至于那么惨。宋是有很先进的东西的啊。今天我还在看一个东西，给大家分享一下。那个家里头有人在那里看那个什么，看视频给他推了一个广告，然后这个广告是干嘛的呢？推行所谓的那个什么宋式美学。我不知道有多少人看过这种类型的广告，就是卖什么家装、卖家具，卖这些玩意儿的，说有宋式美学。大家都知道，宋是我们的审美艺术创作的一个高峰期嘛，艺术作品非常的丰富，而且呢，确确实实这个时候的审美水平和档次都非常高。那他只要给你来一波什么宋式审美，然后你就觉得，哎呀，这好好中国风啊。但是请问它是什么？刷到过是吧？好，然后那个我看了一下，那视频上面的样子是啥呢？是极简风。注意一下，如果你刷到了，它说宋式审美典雅高贵，然后又是极简风，我就跟你说：那玩意儿就不是宋式审美，那是日式审美。来，听听，听明白了没有？听明白了吗？啊，我可以再重复一遍：凡是它在给你推送宋式美学、宋式家装，但是来的全部都是极简风，里面啥玩意儿都没有，摆得溜干净，那是什么东西？注意了，那个东西它起了个名字叫宋式美学，实际上是日式美学，是和猫砂盆同一原理，由于贫困而造成的，学了个半吊子之后产生的日式的简约风美学，或者叫贫困风美学。包括在互联网上很红的所谓的侘寂风也是这种原理。就是因为它整不了那些豪奢的东西，然后呢，它就把它简化、简化、再简化，最后就变成了所谓的这个日式审美的系列。那这个东西它有没有它自己的美学的东西呢？也是有的，尽管有点贫困，但它也是有美学的内容的。可是当你把它冠以宋式的时候，就突然之间把宋式给你掩埋了。为什么呢？因为宋啊，它是一个很典型的——嗯，怎么描述这个词呢？就是大家知道雕梁画栋，我们经常在看跟清式院落有关的建筑的时候，例如说北京有很多什么什么王府啊，类似这样的地方，它呢，就会看到它的很多的梁柱上面会画上很多很多的绘画，什么花鸟什么的东西都有，然后你常常会觉得它很俗艳。那么宋式的审美就很好玩，宋式的审美是艳而雅。它的这个房梁上面、花窗上面那种绘画的反复程度，一点儿都不比清的少，它的画量极大。它的那些残留的宋式的建筑上面，去进行它上面的绘画的复原，还原的时候你会发现，我勒个天呐，它恨不得把花园子都给你贴在了房梁上面，就到了这样的程度。但是非常的雅致，为什么？这个就是绘画的风格、色彩的搭配，还有线条美所带来的一个效果。它和清之间有共同性，就是，艳丽的东西特别的多，非常喜欢艳丽奢华的东西，但是它和清有一个很对立的地方：就是清很俗，但是宋很雅。所以呢，在这个上面咱们就弄明白一件事：在审美这个事情上头，一旦有哪个营销号我跟你说，我是宋式审美，然后一看身上一朵花都没有，屋子里面全是木板子、木线条，放心，这玩意儿是日式的，属于穷困潦倒风的那种。宋式的特征就是阔气、有钱，然后呢，它就想法设法地把各种漂亮的东西都堆进去，但是因为审美的档次很高，所以它虽然堆了，但是堆得很好看。这个就是典型的宋式审美。所以在座的所有的爷们儿——是不是在座的应该爷们儿比例不低？在座的所有的爷们儿，如果穿越到宋，你脑袋上不簪一朵花，不好意思上街。宋人就好这一口，大老爷们儿出门的时候不分老少都要簪花，极艳丽的那种。啊，啥送送这个确实有钱。对，宋有钱这件事情，它其实呢，一方面是优势，谁能说有钱是劣势呢？但是一方面也确实带来了一些负面的影响，就是老想靠花钱来摆平世界。啊，啥你啊？我李逵要插三朵。没问题，没问题。嗯嗯，什么？要顶菊花吗？当然可以，没有任何问题。我们自古以来，菊花都是美丽和雅致的代表。菊花呢，第一，第一件事情就是：菊花不是拿来匹配丧葬的，它们之间没有任何绑定的关系。无论是供奉菊花，还是到了季节赏菊花，都是在欣赏它的美丽。所以不要把它说什么晦气阴间丧气风，什么丧葬风不存在。这是第一件事情，就是中国人自古以来都有赏菊的风尚，不会把它关联到任何的负面观点上面去。这是第一桩。第二桩事情就是：丧葬和晦气在中国是不等同的。来，这句话我可以反复强调，一定要弄明白：把丧葬和晦气绑在一起，这是非常西化的一种思维方式。注意，中国人在丧葬上面一点儿都不会拿什么所谓的晦气这个玩意儿放进去。我们在丧葬之中去给我们的亲眷进行配送的，全部都是给它最喜欢的和它最喜欢的，而不是把我觉得晦气的东西给它。这种东西是大不敬，是非常不礼貌的。它和我们的基本原则叫做事死如事生——我们去招待过世的人，和招待活着的人是同样的原理。我们视它为油然生存，只是去了另一个世界、另一个维度。当我们有这种基本的态度的时候，你就知道了：既然你把它当做你的活人一样的看待，当做你的亲人，当做你的长辈，那么你在给它办宴席，然后并且让它坐在首席上面，实际上就是这个原理。这样大宴宾客，以它的名义大宴宾客的时候，你说你把最晦气的东西穿它身上，最难看的东西放它台子上，你恶心谁呢？谢谢。法无常。所以这两样东西都要 get 到，它就是中国人自己的本源的东西。第一，我们对于菊花是一个完全正面的审美，是不包含有任何晦气的东西在里面的。它代表的风格，无论是坚韧也好、抗寒也好、高洁也好，甚至说它的美艳都可以——因为菊花当中有一些品种真的是十分美艳——这些东西都没问题，但是它都是好的，正面的、阳光的词汇，没有负面词汇。第二件事情：我们在丧葬之中，给我们的亲人，给我们的长辈，不会发放任何废弃的物品。我爱的东西我才给他，他爱的东西我才给他。他活着的时候，喜欢什么我就给他什么。
