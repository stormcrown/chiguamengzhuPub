# 第394章 还好在民国时期，我们保住了汉字 

关于宋育仁提出的“文化辛亥论”和“汉达西三系文化论”，我原本没有详细了解，但经你提及，我可以谈一些相关的历史背景。所谓“文化辛亥”，实际上是辛亥革命后一批文化人主张在推翻满清王朝统治的同时，也要清算其遗留的文化遗毒——比如腐朽、没落的满清垃圾，相当于进行一次文化领域的正本清源和垃圾清运。这些人后来被统称为“文化辛亥”行动者。

辛亥革命不仅包含武装革命，还有文化层面的努力。例如民国初年曾兴起过一次汉服复兴运动，当时复原的汉家衣冠虽带有戏服痕迹，但整体上比21世纪初的尝试更显气度，可能是因为那时还保留了一些传统底子。然而，这次复兴运动被两股力量打断：一是袁世凯复辟时穿上汉家衣冠，导致该运动名声受损；二是蒋介石专门出台国服文案，将旗袍马褂定为“国服”，导致汉服复兴功亏一篑，甚至使此前两百多年坚持抵抗剃发易服、保留服装体系的职业与地区也最终转向旗袍马褂。蒋介石此举很可能是不怀好意的政治操作。

此外，文化辛亥的另一重要方面是保住汉字。当时因满清两百多年带来的耻辱，许多人主张全盘西化、汉字拉丁化，认为表音文字更高级。最终通过《施氏食狮史》这类同音文的论证，我们成功保住了汉字。同时，国史大纲的编纂也是文化革新的一部分，以中国人的视角整理历史，强调历史对于国家和民族的重要性。

然而，民国时期的文化正本清源未能彻底成功。当时虽有“大师林立”，但许多大师要么是带着西方任务回国的（如胡大师），要么虽爱国却因教育中西方内容过多而对中国文化理解不足。真正挽回了中国传统文化的是新中国成立后，树立了自立、自强、自尊、自爱的精神内核，这与远古先民的精神一脉相承。后来公知们所谓的“恢复传统文化”，实际上恢复的是清代和民国初年的糟粕文化——如主子、奴才之类的封建残余，这些在清朝以前的文人时代本应受到批判。

革命尚未成功，同志仍需努力。对于旗袍马褂，如果明确称为满洲贵族服饰，我完全接受；但如果强行定义为全体中国人的传统服饰或汉服的一部分，我将继续批评，直到正本清源完成。
