# 第367章 汉唐宋明，为何公知独骂秦制

那个啥，郡县制跟编户齐民是一样的意思吗？  
不一样啊，不一样。  
呃，这两个概念上面不太一样，但是呢，它们之间是有关联的啊。  
就是最开始的时候的这个“民”啊——注意了，就是“民”的概念其实一直在发生变化。我们今天觉得的“民”呢，可能包括了像你我这样的升斗小民，但是在古时候啊，早期我们设立编户齐民的时候的那个“民”，是高级人。有能力有条件计入户口的，已经可以算作“国人”了，知道吧？已经可以算作“国人”了。那些没有条件被计入户口的叫做“野人”。今天的“野人”呢，你就觉得他是山里的猴儿啊，但是那个时代的“野人”，就是没有纳入到编户齐民的人，就被称之为“野人”。这些人呢，他也要耕作，他也要生活，但是呢，他不受到国君的保护啊，是一个纯纯的——嗯——生活业态上面比较简单，受了盘剥——当然盘剥程度呢，略浅——可是呢，保护的程度也就约等于零的一群人，被称之为“野人”。这个是和“民”相对照的。  

所以我们的编户齐民是一个逐渐的管理程度在逐渐深化的一个过程，我们的“民”的范围也在不断的扩张。最初的时候，编户齐民指的呢，就是我能够给你登记户口，能够给你登记身份，该服役的时候要服役，你要为整个的国家做贡献。相当于说你已经拥有了公民身份，这样的才叫做编户齐民。那么编户齐民的这种方式，在郡县制没有出现的时候，它就已经有了啊。郡县制没有出现的时候，编户齐民这种制度管理法就已经出现了。因为我给你编了户口，给你发了户口本，等于说你有了权利的同时也有了责任啊，你是在我体系内的有编制的人了。编户齐民就是有编制的人，他是一个广义编制啊。  

那么到后来呢，因为管理的深度越来越深，我们开始管理半径扩大，这个呢，就是春秋与战国嘛。管理半径扩大，然后我们说应该怎么管呢？原来的那种承包式的小片区的管理方法有点不适用了，我们就逐渐地转入了郡县制。郡县制最初的试点来自于楚国啊，楚国结果最后是最落后的一波啊——楚国后来特别热衷于分封——但是呢，郡县制的试点也是最初发端于楚国的。然后各个诸侯国都尝试过郡县制的管理，但是践行郡县制最深入的就是秦啊，他就把郡县制深入到了每一个“里”里呢——就相当于一个小村落或者是一个社区，它就是一个“里”。那么当秦把他的郡县制的管理深度已经深入到村镇期间的时候，你想一想什么消失了？野人消失了啊，野人就消失了。就相当于他把他自己管理半径里面的所有的人都给你登了户口，都纳入了编制管理，都有了权利和责任啊。确实该教你的东西交了，但是同时呢，该服的役你也必须要服了。这个时候他就将郡县制和编户齐民啊，结合在了一起。就是这样的一种关系啊。  

所以后来我们会觉得编户齐民和郡县制管理是一致的。它其实呢，是从秦国开始实践出来的。最初的时候谁先出现呢？编户齐民先出现，后来才出现了郡县制。啊。  
啊，啥？你消失了？你是野人吗？哈哈哈哈哈哈哈哈。  
啊，那成本不是高上天了？对啊，秦国啊，秦国实行了一种约等于计划经济的管理方式。他之所以能够最后一统六国，就是因为这种方式能够极大地调动整个国家的国家力量。因为调动的程度更高，所以呢，他的战斗力更强，国家富庶的速度更快。但是同时他也要付出成本啊，那这个成本就是他税收得老高的了，税收得老高的了。你就，你就在那个时代属于离谱的那种高税率啊，相当相当过分的那种税。那为什么要这样子？因为他要整体性地进行管理和分配，它要由国家来进行资源的分配与调度。因为如果说我是一个柔软的姿态，让你的地方性的乡绅也好——各种当时不叫乡绅，当时是贵族啊，各种各样的小贵族、世家贵族这些群体，他还能够管理自己的那种封地、封国等等的势力范围的话，那么中央王权就没有办法深入到这些区域当中去，他就调动不了这方面的资源，就不能全国一盘棋地进行操作。  

所以秦当时呢，为了能够极大地发动力量，他就需要取缔所有的世家贵族身上所拥有的权利。所以那个时候呢，在给他进行商鞅变法之后啊——秦国商鞅为啥死那么惨啊？因为这个，因为他剥夺了所有封君贵族身上天然拥有的权益，然后让他们在没有军功的时候就和普通的黔首、后来的平民百姓一个待遇。那这种时候，那你你能不气人？别人能不恨你？但是他富有，他拥有充分的调动能力，也是来自于这个啊。秦帝国当时他拥有那么强的动员力，也是因为这个缘故。嗯哼，啥？  
商鞅不仅愚民，还剥夺了贵族的特权。对啊，商鞅那个时代啊，剥夺贵族的特权，所以呢，他几乎得罪了所有的贵族群体。这些整个国家有能力说话、有实力说话的人，除了君主一人，其他的全部都被他得罪光了。所以他最后才会死的那么惨啊。但是呢，这是人类社会制度的一次试点性的尝试，这个尝试非常重要。你要知道，没有这样强悍的动员力，你怎么知道你到底能做到什么程度？之前没有人想过这件事情，或者说他假想了，但是没有人敢这样试验过。但是秦国敢，秦国做了一次彻头彻尾的社会性实验，最后发现有用，可以成，可以极大地调动社会的资源，价值很大。所以他才能够一统六国。但是副作用也很大，副作用就是这个社会管理成本极高，高到了畸形的地步啊。其实你想一想也就知道了，你你要把所有的税啊，收到每一个人、每一块田，然后呢，你还要求说啊，你得了军功，有了奖赏之后，你才能够有田，然后一旦违法犯罪，你这个田还要收回来，相当于土地的国有化。你这是从别人嘴里面捞好处啊，亲啊！当整个的中央王朝能够说调动这样深入的资源的时候，各地的地主、小贵族、大贵族、旧时代的封君，他就是你的仇敌。  

所以你看为什么到了《史记》啊，《史记》里面对秦始皇是极为贬低的。贬低到了什么程度呢？贬低到了大家都会觉得他姓赵，说始皇帝嬴姓赵氏——就是司马迁干的事情。这句话本身就是骂人啊！注意了，这句话本身就是在骂人。现代人不那么容易意识到，觉得他赵氏就赵氏呗，有什么大不了？他就是赵氏啊。但实际上在那个时代，姓和氏命名法是非常规范的，它是有规矩的。所以呢，当一个君王已经成为君主的时候，当一个君王已经成为天子的时候，你说他是赵氏，就已经是在骂他了。所以司马迁说他嬴姓赵氏，这件事情是在侮辱始皇帝啊。那为什么呢？因为始皇帝的名声不好。为什么不好呢？因为他开罪了所有旧时代的贵族。然后你再考虑一下司马迁，他姓什么？其实司马是一个氏啊。司马是一个氏，从他的氏上面就可以窥见，一般就是他家以前必然是贵族，那他必然是贵族，知道吧。  

哎，好，欢迎来到直播间啊。啥，赵姓嬴是胡说八道的？嗯，那个什么，那个姓氏这个事我讲过很多次，我今天就概论性地稍微说一下。姓呢，是指你的家族血缘，家族血缘这个东西是固定的，所以呢，在先秦时期的男子基本上不称姓，因为没有必要去叫这个事儿，他只能证明说你的血缘来历是什么。所以日常的称呼的时候用的都是氏。是什么呢？氏是可以经常更改的，氏就是你的社会性的身份。氏是一个人的社会身份啊。包括说从更加古老的时代也是如此：黄帝轩辕氏，轩辕就是他的职业，看到了吧？轩辕是一个人的职业啊。燧人氏，燧人就是烧火嘛，燧人也是他的职业。到后续他也是以职业、以封地来作为一个人的氏的。包括刚才有说司马迁，司马迁他这个“司马”以前呢，就是一个氏。为什么？因为他是官职，司马是一个官职。如果这一家子人出了一个司马，然后呢，这一个司马又刚好开宗立派之后，说我这就形成了一个小宗是吧，于是呢，他这一个后续的子孙就会称为司马氏。所以在先秦时期呢，还有王子氏啊，有公孙氏，这些都是他的身份。还有邯郸氏，这就是他的封地啊。有鲁氏，这也是他的封国啊。这些呢，都是属于他的氏。  

而放到始皇帝身上是这个样子的啊。始皇帝为什么有人将他称之为赵氏？是因为他的祖上曾经和赵国之间的这个赵氏的群体之间有同宗的关系啊，那是好多好多好多年前，他们曾经有同宗的关系，所以呢，才叫做嬴姓赵氏。但是氏是可以不断地改变的，只要你的身家改变了，重新立了宗，你就可以改氏。所以呢，原本的赵国的赵氏下面，其实还分出了其他的小氏，什么小宗，有一个邯郸氏，就是从赵氏里面分割出去的。所以当你知道赵氏里面可以不断地调氏、不断地分割、不断地改氏的时候，你就知道了：当他们将始皇帝称为赵氏、称为赵政的时候，意思是你其实是姓赵的麾下的那个龟孙儿。但实际上始皇帝的身份是什么呢？他是秦的公子，他是秦国的公子。当他的国是秦的时候，你可以说他的氏是秦啊。他的身份啊，他家里面是有封国的——注意了，封国。当他的家里面的封国在秦，他又是秦的公子的时候，他的氏应该是秦。你可以说嬴是他的氏，为什么？因为他后来成了天子，天子可以不称氏，他的姓氏就变成了一体化，直接称姓。姓是嬴，氏也是嬴也没有问题。那么因为他的国呢，国号为秦，所以你把他的氏呢，称之为秦也没有问题，虽然他可能不是很高兴，但是这没有问题，符合基本规则。  

可是当你说他的氏是赵，是他们家几千年前曾经和姓赵的那家子是拜把子，甚至比人家的还低一头——这是什么？这就是在指着鼻子骂他龟孙儿啊。啊，好，欢迎来到直播间啊。
