# 第399章 颛顼的绝招：绝地天通。让中国人很早就大于神 

谢谢西泽颛顼的绝地天通，是不是为后面“人大于神”、“神要服务于人类社会”的思想定下了基调？有点这味儿啊。颛顼有这味儿，哎！有这味儿，有这味儿！就是刚刚才说的那个什么“专需”啊。啊，我一我经常念的是“绝天地通”。哈哈哈哈哈哈哈。嗯，好。

其实呢，中国这里啊，人权、人世间的权利和神明的这个地界儿，一直之间它还是有一个发展的、迭代的过程的，不是从一开始的时候人的权力就足够的大。要知道，人世间的政权与天沟通这件事情本身它意味着什么？它就是神权啊。与天沟通的权利就是一个神权。然后呢，在那之前，神权为什么是分散的？因为我们早期的结构是一个相当于部落联盟的结构啊。注意啊，它主要是一个部落联盟的结构。在后续的时间里面，在我们与周边的其他的人群往来的过程当中，可以有一部分的对照，你又能够察觉一件事情。例如说当年的百越啊，就是始皇帝南征百越的那个百越啊——百越为什么叫百越？因为它就是分了大量的部落，大量的散装的部落。每一个部落都信奉自己的图腾，所以每一个部落都有自己的巫师与天地沟通。所以那个是我们的早期部落联盟的形态之下，也会出现有大量的巫师，也会出现有大量的与天沟通的人群。而这些人群代表了什么概念呢？第一，他的权力是分散的；第二呢，还有一个问题，就是当你与天的沟通的权利这样分散的时候，你的叙事是没有办法一统的。

要知道，人群啊，构建成一个完整的人群，需要一个完整的叙事。这就是我们今天在那里带货的时候会去带那个《国史大纲》。《国史大纲》解决的就是这个问题——它解决了是中国人的叙事问题：我用什么角度去思考过去？这就是我们有一个历史叙事啊。这是我们有历史记载之后，我们用历史来叙事；但是没有历史记载清晰的時代呢？我们用传说来叙事，用神明来叙事。你用一套叙事方法按啥神权呢？当它是分散的时候，掌握在各色各样的人等的手里面，于是最终解释权就不是一个。那么这个群体呢，就掌握在完全不同的各色的神权的大祭司的手里面。只要哪一天他把那个故事一改，大家就可能散黄啊，就可能说各回各家了，这个风险是随时存在的。

那么怎么解决这个问题？所以颛顼的这个绝天地通啊，实际上呢，他是可以做到一件事情：第一，他明显的收拢了神权啊；第二，当散装的巫师的数量越来越少，大家的叙事就会进入到一个共同叙事当中去。当只有一套共同叙事的时候，这是谁掌握了权力呢？啊，他最终就会由特定的人来掌握权力。所以不是说这个时候突然之间人的权利就胜过了天，而是这个时候人开始与天的解释权这件事情做了斗争啊。他在解决的，是解决我们人世间的统治者能否收拢所有与天相关的神权解释权。当他已经可以把所有的神明解释权都囊入自己的怀中的时候，他就拥有了绝对王权。而绝对王权的下一个步骤，必然是让王权胜过神权。为什么会这个样子？因为王权的治理方式和神权的治理方式，它在逻辑上面是不同的。而哪一种更能够让他们开发出更先进的生产力呢？一定是王权。因为在王权的制约之下，每一个人都能够意识得到自己是一个人，而不是被神明放牧的羔羊。一个人只有当他意识得到自己是人，是天地人三才当中的一个，他才能够肆无忌惮地进行创作。所以它是一个开端啊。

我今天，嗯，这么说吧——啊，就是咱们要区分一个东西：就像佛家、佛学和佛教你要区分开啊；道家、道学和道教你也要区分开。为什么说到“教”的时候，它就变成了一个组织？注意了啊，说成了“教”的时候，它就变成了一个组织。而组织的话，里面就会有人——这好像是个废话啊，不重要。组织是由人来构成的，而人是要生活的，所以组织是要牟利的，知道吧。所以呢，注意了，当它是一个教门的时候，它和它的思想之间很有可能会脱钩，因为它要谋取自己的利益。所以你看啊，虽然说道家的思想当中有大量可圈可点的东西，而且我会推荐这套书的原因，就是我们的生活当中很多的传统文化的符号就是从道教当中来的，它跟我们的生活、跟我们的思想密不可分，它很重要，需要我们学习。但是不要因为这个就有什么滤镜，不要因为这个就有什么滤镜啊。

就像现在道士们穿的道袍，上面可都钉上了蜈蚣扣了。有一些人是被迫的，因为上头协会要求。但是这你也就看出来了——嗯，这就看出来了啊——为啥协会会要求这玩意呢？必然协会里头就有问题吗？是不是这就已经是很大的毛病了？然后呢，再想一想，抗战的时候，道门当中有没有英雄？有啊，有英雄参与了抗战的。但是他也有一大批，一大批的汉奸啊。所以呢，学思想啊，了解知识，看它的发展，然后去思考“我是谁”，以及这一些知识、这些文化如何为我所用。但是不要盲目的崇拜，不要对某一个群体擅自地贴上滤镜。嗯，啥道袍已经必须要穿蜈蚣扣了？说了这事儿了，嗯，说了这事儿了。我之前就聊过这件事情。现在它的整个的道门从上到下啊，规范的标准、传统的服饰都已经轻化了啊，全部啊，全部都轻化了。你别看那个上面什么道士给你宣传的他们的什么什么法袍如何如何轻化了啊——两千多年道门都没有轻化，两千多年道门都扛住了，一直在用交领右衽，在用系带隐扣。今天强制推行蜈蚣扣啊！几百年的时间，道门里面的法袍都还在用原有的形制。现在啊，你们自己去看，就那个花花绿绿的那个味儿，你一看那个配色就知道了，味儿不对啊。你自己去瞧，咱武当的还正常？我去的就是武当，交领右衽下面给你加了三个蜈蚣扣。谁家交领右衽的服装需要蜈蚣扣？不需要啊。交领右衽的结构上面就不需要蜈蚣扣这种密封的东西。蜈蚣扣是啥？蜈蚣扣是密封条。它在旗装上为什么需要？因为旗装是个偏襟，是歪的。一个歪的衣裳，你不用密封条粘上那个衣服就破了。所以在旗装上面，蜈蚣扣是刚需，它就能拼到一起去。我就问你交领右衽的袍服，你要那玩意干啥？一个系带子的衣裳，你往上面贴密封条多不多余？但是武当山那边的啊，道馆附近的各种各样的服装供给链条，上面好好的交领右衽的道袍，底下一定在腋下给你塞三个蜈蚣扣。你还觉得正常？你自己去看，我亲自到那边的服装店、服装的供应商那里瞧着。
