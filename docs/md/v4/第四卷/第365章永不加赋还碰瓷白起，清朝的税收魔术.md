汉人王朝也有屠杀，比如白起就坑杀了40万。  
都是封建王朝，没区别？两件事情啊——  
第一，白起杀害老弱妇孺了吗？这是一个很根本的东西啊。  
就是第一：白起杀害老弱妇孺了吗？嗯，白起杀害了老弱妇孺吗？没有。  
他是消灭战争对手当中的有生力量。你能说这玩意儿没区别？都是人没区别吗？真的没有区别吗？你确定吗？  
在战争之中伤害了战俘，固然也不算是什么很有良知的事情，但是杀战俘和屠戮对方城池当中所有的老弱、一切的人口，性质是一回事吗？啊，这是很根本的区别。这是第一桩事情。  

然后第二桩事情呢，有一种那个什么……  
之前呢，对明朝一般都有一个黑暗的滤镜，然后清朝身上呢，有一个很美好的滤镜，叫做“永不加赋”。听过“永不加赋”这个词儿的，似乎还字字泣血、为天下生民立命的啊。听过“永不加赋”这个词的，来，扣个一。来，“永不加赋”啊——听说过啊。圣天子永不加赋啊，来来来。听到这个的啊，扣个一，呵呵。  
那这个“赋”是什么啊？不信“永不加赋”？来来来——赋是什么啊？  
我们的税和赋是不一样的。这个赋和之前那种什么所谓的、我们俗称当中的人头税，关联得更加紧密一点。所以这个赋呢，其实脱胎于最早——它脱胎于一种劳役，就是早期的徭役。相当于说你需要为公共建设去修城墙啊、挖沟渠啊这些工作，最后换算成了一种税务的负担，就会变成叫做赋。  
好，那清王朝喊过一个口号，叫做“永不加赋”。但是注意了啊，他是“永不加赋”吗？确实没加——没加赋啊。但是呢，他加了税。他没有加赋，但是他加了税。收都是照收了的，但是呢，他确实在名义上面没有加赋啊。  
那他把这个“永不加赋”作为清朝的一大善政，反复地推广，然后呢，拉一踩一——一边说大清王朝永不加赋，一边就说明朝的那个什么税收人头税啊，所以呢，负担很重。来，我们来看一下啊——你知道明王朝有多少人吗？没有人知道明王朝末期到底有多少人。为什么不知道？因为整个明王朝的200多年间，只有洪武年间统计过人。只有洪武年间统计过人头，设了户口，设了登记表格。这个东西是我们的征税的基础——无论是税还是赋，都是按照人、按照家庭、按照田亩来进行计算的。但是这个统计只有洪武年间做了一次之后，再也没做过。这是什么意思？就是尽管说你一家子里面呢，需要出一个壮丁的劳役，于是呢你把它换算成白银，这个就是丁银，他就是所谓的赋，也叫做俗称的人头税——就是按你家里的壮劳力来收的。  
好，那既然你家里面有一个壮劳力啊，统计出来了，你需要去交这样的一个人头税——就是徭银嘛，徭役——你要把这个东西交上去。那么将来你要生孩子的呀，你要生孩子的呀，最后孩子是不是又长大了？后来你们家里面已经有五个壮丁了，但是他没有人口统计，他不知道！有意思吧？所以你们家依然只需要交一个人头税。我就问你这玩意儿，他加赋了吗？加了吗？他没有喊过任何一个“永不加赋”，但是他甚至不曾统计过。所以明末到底有多少人，其实没有人知道，从来没有这个统计啊，完全不统计。没人知道明末真实的人口状况，没有官方数据。今天你们所知道的一切数字，都是我们的学者通过侧面的证据——包括说什么卖盐啊、包括卖盐、包括产粮、包括开发田亩等等的周边的其他数据——然后进行了侧面推论。因为是推论，不是什么官方统计，所以呢，这个数误差特别大。收得最紧的一个，认为明末有1.8亿；放得较宽的呢，认为有2.6亿；还有更宽泛的统计啊，还有更宽泛的，但是数量更大的那种呢，认同他的人就没有那么多。但是科学家有主流吗？没有。所以不见得哪个数就一定真，或者哪个数就一定假。但是就算你取一个中间值，明末的时候啊，也有两个亿。注意了啊，这个数不准，我先说这个数不准——因为没统计，都是靠的推测，靠周边数据的推测。而明末由于他的收税的能力特别差，所以我们通过周边的信息去进行推测的时候，这数不准。所以即使是学者的推论，误差也特别大，这个误差很大的数字也不准，靠不住。按照这种统计的方法，就在中间取一个大差不差的中间值啊，就是2.2亿。这种数字呢，比较折中。注意了，这不是数字有没有效，是折不折中的问题。我们选这样的一个折中的数字，2.2亿作为明末的人口。  

清初的时候有多少啊？有没有人知道？有没有人知道清初的时候有多少？那个清初的时候是做了统计的，为什么？因为一个王朝立国了之后呢，他需要收税。为了收——甚至为了抢——他要进行统计。所以清初是有数据的。多少呢？这个数据啊——来，八千万。来来来，清初八千万。明末的数据，较为折中的一个数字是2.2个亿，清初八千万。我就问你这个差值去哪儿了啊？这个差值去哪儿了？然后你再去跟白起比一比，还说白起好残忍？来来来，你把这个数字拼到一起去看的时候，你觉得白起残忍吗？  
我们前两天的时候，大阅兵反复地重复了一下倭寇当年的罪孽。然后当时的中国军民，在14年抗战期间，付出了多少伤亡的代价，取得了最终的胜利。来，大家来报个数嘛——我们14年的抗战，艰苦卓绝的抗战，在二战期间，全球火器时代杀伤力极强，甚至对方由于极度的残忍，还推行了细菌战的情况之下，14年的抗战当中，我们有多少伤亡？3500万。好，来，3500万。然后你再算一算，我们刚才算的那个从明末到清初的这笔账。然后你说，哎呀，都一样？哎！  

好，欢迎来到直播间啊。暴风雨大地讲一讲万户——为何西方在宣传这个故事？我们的正史、野史未见记载。他是他说的万户指的是哪一个故事呀？我咋有点对不上号呢？啊啊，你们谁那个想得起来呢？那啥，发明火箭的那个？哦哦哦，我知道了我知道了，我想起来了。  
嗯，我想起来了。第一件事情啊，这个档案在中国没有很正常哈，知道吧？这个档案在中国没有是很正常的。因为清代的时候销毁的档案当中，除了历史原件、原文之外，销毁的数量最多的就是科技类的档案啊。注意了一件事情，现在在互联网上喷我的人当中，一直有一个咬死了喷的点，就是在喷我去说那个什么郑和下西洋走了全球这件事情。但是很简单的一件事，郑和下西洋的资料全部都被销毁了，这件事情是非常可疑的啊。因为没有任何理由去销毁他的档案资料，即便他劳民伤财——虽然这也是谎言啊——即便他真的劳民伤财，销毁资料、销毁档案、销毁图册，也是一件完全错误、不合逻辑、根本不合理的事情啊。  
啥坤舆万国全图呢？因为我就是打这事上面说的，知道吧。但是呢，他们一般都绕过这个话题。反正呢，这个事情放在这儿啊——就是为什么会销毁那么大量明王朝时期与科学、与技术、与贸易发展相关联的原始档案？就这么一件事儿啊。你既然销毁了那么多的档案，我们国内关于说什么啊，发明什么火箭，然后如何如何失败等等的这些事情，完全有可能之前有档案，但是后来被销毁了。  
清王朝对于火器是特别避讳的，避讳到了什么程度——火器专家可以直接流放关外。已经到了什么样的地步？崇祯年间铸造的火炮，到了清末打仗的时候，还要给人家刨出来接着用，质量还杠杠好。为什么？因为清的时候，他的这个火器的水平已经低到令人发指，比明末都要差了不知道多少条街。所以这是干什么？这就是他在不断地销毁、摧毁啊——所有的工匠、技术人才和技术档案，他反复不断地去做这件事情。所以我们国内没有这种档案是很正常的。  
然后至于说西方人为什么特别喜欢吹这件事情？我看到的他们一般把这个东西讲得呢，比较可笑啊。尽管讲故事的人有的是用很庄重的、甚至是很郑重的方式来讲的，但是听故事的人、传播故事的人都会把它视为一个笑谈。那这里头就有一件事情啊——他不见得真的如他所描述的那种细节的操作方式。为什么？因为中国人只是古，不是蠢。他的那种极为草率的弄了几个火箭筒子，直接把人家给轰上去了，这种操作方式本身是一个不合逻辑的操作方式。  

像啊，火器是马匪的天敌。那个清不是一开始就反对火器的。注意他在关外的时候，没有入关的时候，他是非常积极地招纳火器的设计研究还有铸造的人员。实际上他在关外掳掠奴隶的时候，你以为他是在那里抓花姑娘？不是的，他是在抓大量的工匠和技术人才。所以当年他还在关外还是后金的时候，非常注意火器的技术。甚至在他稍微兴旺起来之后，他跟明王朝之间打仗的时候，都是要动用火器团队的，是可以实现大炮互轰的啊——水平如何另当别论，但是他都有，并且积极地招募相关的人才，积极地联络各种各样的走私商贩，保证他那个地方有火器的供给。所以实际上他不是不知道火器这个东西好，不是不知道这个东西有用，而是知道这个东西好、有用，所以他必须遏制它的发展。  
因为当他进关之后，他面临的局面就和关外不同了。在关外的时候，他是一小撮匪帮，面对着另外的一个庞然巨物。但是进关之后啊，它变成了一小撮匪帮，骑在了广大的、普通的、但是呢却拥有文化技术的人民的脖颈子上面拉屎。他已经住在了大众的中间，这个时候他最重要的事情就是：要让这些老百姓不敢反抗，要让这些老百姓失去组织，要让这些老百姓越来越落后。你想一想，如果说有火器，火器的技术、火器的人才仍在，你能够保证这种装备不会流落民间吗？保证不了的。但是请问他的那些铠甲防得住这玩意儿吗？防不住的。所以只要火器推广，他的统治就一定岌岌可危。为什么？因为总有刁民想害朕。他和其他的时代不一样啊——其他的时代，皇帝最大的危险来自于政争、党争、朝臣的文官武将的压榨啊，皇权最大的风险来自于权力的博弈，这些都是属于社会上层内部的一种斗争。但是清王朝不同，清王朝他的统治阶层所面对到的对手，是有如汪洋大海一样的老百姓，每一个老百姓都恨他。他所面对到的对手不一样，就会决定了他的操作方式不一样。  
所以到了清王朝的时候，他们会非常忌讳拥有火器这件事情。越到后来，火器的地位就越低。清初的时候，工匠还会设计连发火铳，然后就被流放了。那么之后呢，他的火器技术每况愈下，刀枪入库，马放南山。所以才会出现到清末的时候，从库房里面去翻崇祯年间的大炮出来抗敌这种破事。而原理上面，其实在内核上都是高度一致的——那就是因为它本身就是所有老百姓眼中的敌对势力，所以他不能够让武器升级换代。任何一个鸟枪换炮都可能葬送掉他的龙椅啊。所以他为什么会不发展啊？它不光是不发展，实际上是一直在倒退哈。
