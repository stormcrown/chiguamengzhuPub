# 第343章 我以我血荐轩辕，敢叫日月换新天 

我在高中的时候，大概是念文科那会儿，人比较清闲，所以看了些杂书。其中就包括了《鲁迅全集》。我那时看的杂书，一类是古典小说，另一类就是《鲁迅全集》。毕竟高中确实很闲，这些东西都是拿来当课外阅读消遣时间的。但对一个少年来说，其实并不能深刻体验到其中的情感。我只是看了，并且很明确地知道：这是我们的革命斗士。仅此而已，其他的感知不到。我坦诚地承认，在那个时期，一个没遇到什么障碍的少年，确实感知不到他字里行间那些愤懑之情到底从何而来。

但我相信，同样生长在那样环境背景下的教员，他是能深切体会到的。那是什么呢？那就是他们其实有着共同的目标，共同的目标、共同的立场和思维的基础底色。这些东西是能让两个人共鸣起来最重要的东西。尽管他们行进的方向和水平各不相同，但根基是一样的。

包括鲁迅先生，他的文章里有一堆一堆说中国人坏话的内容。实实在在讲，他的文章里确实有一堆一堆说中国人坏话的东西，但你不会觉得他在骂中国人。他在哀其不幸，怒其不争。他希望中国人改变，希望中国人站起来，希望用他的笔警醒世人：看一看吧，睁开眼睛吧，不要再蒙昧了。想一想自己过的什么日子吧，想一想自己还是不是一个人吧。不要再去迷惘于过去，不要再把自己困守在那半亩田之间。去想一想自己宏观的想法到底是什么样子的，自己的过去到底从何而来，今天的苦难又是从哪里来的。有点出息没有？你怎么可以这么堕落呢？这就是鲁迅先生的姿态。

那么教员的姿态是什么？教员的姿态是：是的，我们今日蒙受了无比的苦难，但我们有着极其光明的未来。为什么？因为我要去开创它。所以教员终其一生所做的事情，就是用我们从历史当中获得的经验、教训、文化，结合了从现代哲学思想当中所获得的思想、分析方法与方法论，试图解决一个问题：如何让这一群蒙受300年苦难的中国人，重新走向光明。

所以这两个人在本质上面是一样的。一个是说：“赶紧的，给老子站起来！”一个是说：“跟我走，我们一起去找升起太阳的地方。”

嗯，总觉得鲁迅的作品看起来很压抑。他的那个时代就是那样的压抑。在那个时代还感觉到很幸福、很舒爽的，你知道是什么人吗？大汉奸。在鲁迅先生的那个时代，过得舒服、顺心、畅意的是什么？是汉奸、买办、走狗。所以你看他的文章，感觉到压抑、痛苦，那就对了。因为他是中国人，他和所有受苦、受难、受害的中国人站在了一起。他在为我们而疼痛，他在为所有说不出话的中国人说话。所以才会压抑，因为那个时代的中国就是这么压抑的。在那个时候还觉得开心、舒爽，又觉得这个好、这个美的，多多少少沾点卖国。

嗯，好，欢迎来到直播间啊。
