# 第377章 磕一个邪门CP

哎，好，欢迎来到直播间。有人问：看了清史，为什么感觉康雍乾比明中期的君主个人能力更强？两朝君主的综合能力如何比较？呃，关于这个问题，我们来看清史——无论是看清史还是看清修明史，在进行君王能力比较时，都会觉得满清的皇帝特别能干。不然的话，为什么互联网上那么多人引经据典，都要说“大清十二帝个个是圣君”呢？是吧？啊，为什么会这样呢？因为他自己编的书，他自己编的书就是这样吹的。然后呢，他再去编明的史书时，就会说这个明朝的君主什么“远迈汉唐”这些词儿他都用了；但是讲里面的细节的时候，就讲得那个人跟精神病、跟变态似的。于是呢，你去看的时候，就会觉得明朝的君主很无能，然后清朝的君主很威武啊。

来，咱们对照一件事情。我觉得什么东西呢，都怕比较。例如说，很多的时候呢，这些学者、公知、营销号都会告诉你：清朝的君主特别的勤政。这个东西听过的吧？特别的勤政啊。勤政这个事儿听过哈。清朝的君主特别的勤政——勤政，勤政，哎，极为勤政！动不动就是早上几点钟就起床了呀，晚上睡得比狗还要晚呢之类的啊。这个事儿应该是听过的。好，来，那有没有听过对于朱元璋的点评？说他贪恋权势，什么事都要拢到自己的手上来，不肯放权。听过了没有？同样是什么事都放到自己的手上办啊，但到大清十二帝各个是圣君手上的时候，这叫做“事必躬亲，勤政爱民”；放到朱元璋身上的时候，叫做“贪恋权势，不肯放权”。来来来，听过没有啊？听过的你对照一下啊。什么东西都怕比较知道吧，你要对照来看。同样的是什么事情都抓在自己的手上：“我不决定了，你就不能说我定下来了才行，你得经过我的审核。”同样是这么一件事情，放到清朝那就叫做勤政爱民、事必躬亲；放到明朝，那就叫做贪恋权势、不肯放权、极端极权。所以你按这个标准的时候，你一定会觉得满清的皇帝太优秀了，那能不优秀吗？而揽权的叫做勤政是吧？底下的饥荒闹得稍微少一点，就叫做爱民是吧？敛财的水平高级一点就叫做爱民？然后如果出去打仗，打个百八十年的，最后终于打赢了，就叫做开疆拓土、十全武功。你按这种标准来算，天哪，这世上不是明君的也没有几个了啊！什么？嗯，标准不一样了，那标准差老远了，知道吧。

嗯，啥？没饥荒就好了？大家安居乐业嘛？不是没饥荒就好了。你立的标准太高了啊，你提了太高的要求。不是没饥荒就好了，而是别把饥荒闹大了，不要在史书上面写上“岁大饥，人相食”啊。重点是你不要写上“岁大饥，人相食”。如果说他真的做得到说人没有饥荒，嗯，真的，虽然说中国的老百姓们是非常勇于造反、勇于革命的，但是如果说他真的是能够造成没有饥荒的环境，那大家可能还真的不会造反。但是就以清的那个时代的民变的频次——每年都有，年年都有啊！只分这一年有几次，只分这一年的造反次数的多少，以及造反的县市到底有几个，就这样的区别。没有哪一年是没有革命的。能够让中国人做到这样的程度，按照其他朝代的标准，这就是年年有饥荒。诶，谢谢轻雾啊。康乾下那么多次的江南，放任何一个汉人王朝，都是隋炀帝般的穷奢极侈啊！本来就是，但是吧，但是这里头就有一个东西啊，就是——呵呵，电视剧这玩意儿是老狠的了。大家不要瞧不起电视剧啊。虽然我不怎么看电视剧，但是呢，我从来都不鄙视电视剧。无论这个电视剧拍得有多烂、多像屎坑，但是注意了，电视剧对人的影响是很大的。大家应该看过什么《康熙微服私访记》啊，还有那个什么《戏说乾隆》啊、乾隆下江南跟小美人儿约会的这种故事啊，再包括说像《还珠格格》里面虽然没有那样讲，但是呢，它里面也有微服私访吧？同时呢，还有什么“陛下还记得大明湖畔的夏雨荷吗？”他也是在讲他以前曾经微服私访的这种情形。这种故事啊，这种故事是很多的。这些电视剧、这些小说，包括一些电影，当你把这些东西放进去的时候，不管你明面上觉得他衣服怎么样啊、演技怎么样，这些东西都摆开不谈，他一定会在你的心里头烙下一个烙印。你会觉得他那个时代的皇帝会有微服私访啊，注意了啊，你会觉得他有微服私访。而这个东西就很可怕，因为他不可能微服私访啊！注意了啊，他不可能微服私访，因为他做不到啊。来来来，注意了啊，他是做不到微服私访的。尤其是当他在电视剧里面动不动就表现得说“啊，我暴露身份了是吧”，在当街上说为了惩凶除恶，暴露身份，“我是你们家主子爷，你必须要如何如何”是吧？哎呀，还有时候出现这种现象。我跟你说，就是清朝的那些皇帝，但凡说他暴露出来他的口音的独特性——这个事儿他是一直保持的啊。清朝的时候，他们的统治的顶层的八旗，是一直搞着这种隔离生态的，甚至刻意地保持了自己的口音和行为动作习惯。这个是他为了区别于普通的汉人，所刻意保存的风俗。为什么太平天国去屠满城的时候能够找着人？以为他们不会跑吗？不是，是因为他们的语言、生活的模式、行动的姿态和别人不一样，是他们刻意隔离之下的一个结果。然后你来啊，一个人长得有特殊性，然后说话的口音有独特性，着装上面、动作上面都和你格格不入，你一定会很警惕。然后如果说他哪个地方还表现出了格外的高高在上，你知道会有多少的革命志士就在街边盯着，打算要拿下这一个大元吗？啊，他做不到的，知道吧。所以他根本就不敢微服私访。所谓的下江南是怎么下的？他们走那个全部都是官道啊，然后能走运河的时候都走运河，在运河里面大船并着小船围在一起，能够把河面给你摊平了，然后两岸全部都是禁卫军，是用这样的方式一路平推推下去的。那他这和功利有啥区别？他为什么还要下江南呢？就近啊。他下到江南之后就近就开始索贿啊，就近索贿！你懂吗？他想要什么东西的时候，都已经怼到你脸前面了，我的部队人马都在这儿了的时候，我要什么你给我拿什么。嗯，啥？搜刮？呃，大大差不差啊，就这一个意思。是不是江南太富有了？江南本来就很富有诶。啊，江南本来就是很富有的。要不然的话，他们怎么这一图、二图、三图特别喜欢在江南图？然后办的那个、那个案子叫什么案来着？嗯，奏销案啊。清初的时候奏销案、哭庙案，主要的针对的就是江南。为啥？因为他太有钱了，剃发200多年，然后在清初的时候一顿嘎，找出各种理由啊，就把你给噶了，然后把家财一路收工。

不少国人认为雍正是个标准皇帝。嗯，能说说真是个好皇帝吗？标准皇帝跟好皇帝有什么关系吗？没有什么关系啊。你要说雍正是个标准皇帝，这句话呢，如果放到清的200多年间，我是同意的啊。在清的200多年间，在他的十几个君王中间，雍正算是一个标准的皇帝。因为其他的都是劫匪头子，其他的都是明晃晃的劫匪头子啊。那个雍正是按照一个君王治理国家的原则来进行的治理。但是这代表他是一个好皇帝吗？是皇帝和好皇帝，中间还隔着巨大的鸿沟。你要知道李世民和晋惠帝都是皇帝，不能说是皇帝就好，对吧？很明确的一点就是，雍正的行为在清算得上是一个皇帝了，但是是好皇帝吗？不是啊。他极其善于敛财，极其善于敛财啊。当一个人敛财敛到他那个地步的时候，当一个人敛财到他那个程度的时候，大家对九千岁是怎么评价的啊？来，你们先说一下，你们先说一下你们对九千岁是怎么评价的哈。啊，你们觉得九千岁是啥样？嗯啊，魏公公，对啊，魏忠贤。你们觉得九千岁这个人怎么样啊？好，下面我的评论就来了啊：雍正的水平和九千岁是差不多的。如果说需要分桌吃饭的话，他和九千岁坐一桌。
