# 第331章 如何评价桂桂和洪天王

问：三藩（指吴三桂等）为何失败？如何评价“三桂”叫得这么亲切吗？啊，叫这么亲切？他姓吴，你应该知道的吧。叫这么亲切？叫“萧三桂”啊？那个什么……三藩为什么失败？他没有力量，能理解吗？他没有力量。为什么说他没有力量？难道作为大军阀他没有力量吗？没有。什么叫做力量？一个是生产力，一个是组织力。军阀的组织力是不够的。要知道什么是真正的组织力。在国战的层面上面，组织力不是一个军队的调度，它还包括了后勤，还包括了所有没有参与到战争之中的百业——士、农、工商等等的人群。这是什么？这是所有发生战争的时候，你要求的叫做上下一心。能够理解吗？在战争领域当中，你要求的是上下同心才能做得到。所以放到这三藩身上的时候，他没有办法做到上下同心的。哪怕他的军队管理上面可以有组织度，但是呢，他做不到上下同心，因为老百姓不可能认他，也不可能相信他。要知道他们当年可都是狗汉奸、死叛贼，人人恨不得生啖其肉的玩意儿。你突然之间喊一句“我要改邪归正”，大家就信你？你不需要花个十几二十年证明你是个好人吗？信用折损了就是折损了。所以他们是调动不了真正的力量的，他只能调动自己的麾下兵马。而这种调动，当这种调动失去了后勤、失去了民间的声援的时候，他就只能靠着发军饷、强迫，然后逐层制衡的方式来组织。也就是大家没有共同的认知，没有共同的意识。所以他为什么败？因为他的组织能力非常差。

但这个事儿，我给你们一竿子戳到比较古老的时代来。大家记得吗？秦王之后，秦亡了之后，秦帝国衰亡之后，这些叛军举的谁的旗子？大家还记得吗？谢谢贤人。秦亡了之后，叛军举的是谁的旗子？记得吗？对，扶苏啊。为什么要举扶苏的旗子？为什么要为扶苏公子复仇？真的是因为扶苏公子有什么才能、长君多少多少万，实力特别强横吗？不是，是因为扶苏名声好，是一个仁德君子。扶苏积累了十年的仁德君子的名声，所以哪怕是叛军都要举他的旗子拿出来用。这个就是信用值。一个人在过去的十几二十年间，一直积累的都是他是一个仁德的人、讲信用的人、一个好人。他积累了这个信用分之后，当他举起旗帜来的时候，就有相信他的人愿意跟他走。请问三藩哪个有这个玩意儿？没有啊。三藩都没有这个东西。他们当年都是狗汉奸，都是死叛贼，而且人尽皆知。所以他们是做不了组织这件事的，他干不了。哪怕是肩不能挑、手不能提、在政治领域上面毫无建树的公子扶苏，都比他们强一百八十条街。嗯，啥？但是刘邦和项羽都没打扶苏的旗子吧？他们不算造反的那一拨儿，他们算是六国复辟。六国复辟每一拨都是举自己家的旗子。推翻秦王朝的造反这一拨是要诛暴秦，诛暴秦推送的是谁呢？推的是公子扶苏啊。嗯，来来来，欢迎来到直播间。嗯嗯，那为什么要打扶苏的旗子？人家又不是父亲。对啊，这最开始的一拨是为了推翻暴秦，然后他找的这个理由是你杀了公子扶苏，而公子扶苏是我们心目当中的仁德君子。公子扶苏没了，那我就要反啊。所以呢，就能够拿扶苏出来做理由。为什么我特意强调扶苏这件事情？因为公子扶苏事实上在他的前半生当中，基本上是没有什么政治和军事的建树的，他自身的能力到底到了什么程度，这件事情没有经过实践的检验，不知道。但是呢，他确实没有明确的建树。一个在政治与军事领域当中并无建树的公子，仅仅因为他有仁德之名，就能够成为旗帜。这个就是中国人如何构造组织力最重要的一个东西：你的名义必须要正，否则的话你没有办法号令。

什么玩意儿？这是学而讲。为什么牛顿不产生于中国？我要盟主回答。你？谁啊？你啊？你谁啊？我答不答的？我可以讲，但是你不能听。你知道吧？这种说话的方式属于后续。我可以讲，但是你不能听。一个是刷屏，一个是暴躁，一个是异常的不讲礼貌。嗯，啥？刚才发生什么事了？来了个精神病。嗯。然后我是专业的带货主播，对啊，不然呢？说话的时候我看见了，我也就说了。哎呦，在这个地方我要你回答？答个得儿啊？我就是答了，你也不配听。

我看看啊，讲讲太平天国吧。太平天国的失败是汉人的遗憾吗？太平天国的失败肯定是一个遗憾，但是我们也在太平天国当中吸取了教训。为什么？因为我们看到了什么样的组织形式、什么样的思想纲领会起到什么样的作用。我一直说一件事：所有人类社会当中的行为，你是对是错，必须要经过实践的检验。所以人类社会的一切的尝试都需要试错，需要试错，这是没得选的，没有人在一开始假设的时候就知道他应该如何。其实太平天国是有自己的理论纲领的，虽然他起事的时候叫做拜上帝教，但是注意了，拜上帝教是他圈人的一种手段，而他为什么用这种手段？是因为其他的手段不行。他起事的那个时期，能够拿来组织别人、能够拿来给大家讲故事的这些手段已经严重不足了，很多都被人家试验过了。要知道，在他之前有什么白莲教啊，有什么天理教啊，各种神话故事都讲过了；然后再加上什么？再加上还有什么类型可以拿来号召造反呢？就是“民为水，君为舟”的这一套民本思想，也是可以拿来造反的。这就是“王侯将相宁有种乎”的来历嘛。但是到了清，晚清时期，这套思想也没有用了。为什么？因为掌控了儒学民本思想解释权的已经变成了“苟如都降了”，变成了汉奸奴才。所以这套思想在当时也没有用啊。他能够用得到的东西，能够给大家讲故事并且让大家过来听我讲故事的东西变得非常之少。再加上普通百姓经过了二百多年的蹂躏之后基本上变成了文盲，他可以选的东西太少太少了。所以他选择了拜上帝教。

那这个拜上帝教在晚清时期的中国是有群众基础的，因为有大量的传教士花了漫长的时间在给中国人讲这个故事。而当时的中国人，由于自身是一个文盲，生活又极为的困苦，而一旦跟着传教士混，变成了教民，他又拥有了一部分的治外法权，就不再那么严酷地去遭受清廷的压迫。所以呢，就会出现许许多多的百姓都听，甚至会有一点相信这种上帝的故事。这就有了群众基础。所以为什么当年洪秀全他们造反的时候要找这么一个故事出来讲给大家听？是因为他有一定的基础，大家能够听得明白，而且呢，他是外来的和尚，他不会因为本地的狗儒截胡导致你的革命不能成功。这才是他使用拜上帝教的根本原因，不是因为他信教——他不是信教，他是想要利用这套系统。不然的话他就不会把自己编成了上帝的二儿子，而且还是一个来路清晰的嫡子。对于中国人来说，大老婆生的孩子和外头不明不白的生的孩子是性质完全不同的。所以你看他找到了一个很微妙的夹缝，就去做这件事情为自己证明。

但是注意了，太平天国革命起事之后，什么拜上帝这个东西就变得不是那么重要了。实际上，他在推行的策略当中包含了文化的建设，就是扫盲运动。我不知道有多少人听过这一段，就是太平天国是推行了扫盲运动的，教大家认字。很多人误以为太平天国的统治者全部都是一群文盲，不是的。他们是教了老百姓认字的，是推行了扫盲运动的。这是一块。然后呢，同时他还推行了均田地。均田地这件事情是很明确的，在进行社会财产的再分配。因此呢，他事实上做的是一些非常根本也非常正确的革命选项，还包括了就是放足。是不是那个凡是在互联网上去搜我的都会说“我为什么裹小脚洗白”，觉得裹小脚是保护健康的是对的？来那个啥还想，还想截屏呢，还想断章取义的可以在这个地方弄。那这个里头呢，就是大家所认为的那种残忍的、暴力的、让人变成残疾的这种断足式的缠脚，是清代的时候才出现的。而太平天国的人民的革命决定取缔这一切，所以它不是我们的汉家传统，没有这个东西。汉人起义了之后要做的事情就是解放妇女和给妇女放足，而且放足了之后很快就在各种的乡里乡间推行开来，女子普遍性的就不再给自己家的孩子断脚了。当然了，等到他的革命失败之后，过来围剿革命的这些反革命分子就取缔了他的这个放足令，并且严厉批判，认为他要求女子放足的这件事情是非法的，要求大家必须把脚再重新缠回去，缠断她。然后呢，在一部分的县市的口述历史访谈录当中，表现出来的现象是有一部分的地区为了执行这个缠足令，于是勒令下去：谁家的女子不缠脚，举家砍头。举家砍头代价异常的巨大。所以你看到太平天国其实在革命性上面是非常先进的。但是啊，但是他失败了。为什么失败？还是它本身内部的组织方式不够优胜，同时还有一个问题，我觉得也是很根本的：有的时候人们的失败不见得是你有根本的错误，也有可能是战术性的错误，然后就导致一次败，次次败。整个这个太平天国运动的失败过程当中，其实战败造成的问题更大。这是战术层面的问题，就是他自身缺乏足够优良的将领。
