# 第955章 国足输在哪_钱太多反而毁了荣誉感

【整理后的文本】

「国足输在哪？太扎心了」 ——钱太多反而毁了荣誉感 —— 《风骨辞》 昔有齐地，蹴鞠为戏，履革成球，踏月如飞。 今作动画，邀胡儿为将，卷髯黑面，称祖述宗。 岂知禹贡分疆，自有英杰；奈何楚材晋用，反诩开通？ 观夫绿茵场上，金帛蔽目，失其本心； 而寒门健儿，悬梁刺股，竞逐长风。 盖华夏之胄，耐苦寒而敏思智，均衡如砥，何须外求？ 然利字当头，骨软筋酥；荣誉既弃，溃如流沙。 愿后来者鉴：毋售其璞，毋隳其魂。

自个儿去看，我们哪些项目能够出息，哪个团队有荣耀感，有这种目标，他哪个团队能够出息。有一个地方说自己是蹴鞠之乡，想要搞文旅项目，专门去宣传自己的什么蹴鞠的发展史，是哪来着？你们能记得这个地方吗？我记不住了。是不是山东哪里啊？好像是山东。说什么蹴鞠之乡要宣传我们的蹴鞠文化。啥？淄博是淄博办的吗？我印象当中是山东啊。当年整这个活儿，当地的地方文旅就试图打造这样的一个文化名片。那时候我还看他们好像是搞了一个动画片，专门做了一个动画片，就宣传他的蹴鞠文化。结果在里面安排了一个类似于足球小子的这样的一个角色，竟然是一个卷卷毛的黑人，就想宣传我们自古以来就一家独大，自古以来就神圣，自古以来我们踢球就得靠非洲外援呗。当年他们做的这个动画片，就做的这个东西。这个事儿我当时看了之后，还是觉得很有一点离谱的。这个是严重的伤害感情，你不觉得吗？严重的伤害感情。你拍的这个片子，你打算要传递什么信息？虽然说蹴鞠是中国人发明的，但是中国人踢的不好，啥时候想赢球都得找个老外。任何时候你想赢，你都必须要请外援。什么意思？说中国人运动能力不行呗。说中国人不会打球，不会打比赛，没有外援混不下去。论证说中国必须得是多元的，不多元活不了。这完全不符合历史现实，这跟我们的客观的历史现实之间不吻合呀。啥，还有这事儿？对，还有这事。这是早几年的事情啊，这是早几年的故事。当地专门去做的一个自己的这种文旅类的宣发的动画片。当时我看的是一张图，好像做的那个图，看作图的质量还可以，可见当地的投入不低，但是给里面专门安排了这样的一个足球外援。所以你就说今天我们的国足能出息吗？出息不了。我们什么样的队伍能够出息？在我们所有的竞技体育当中，什么样的类型的能够出息？有家国荣耀感的。你自个儿去看，我们哪些项目能够出息，哪个团队有荣耀感，有这种目标，他哪个团队能够出息。不赚钱的都出息了。呵呵呵呵呵呵呵。跟我讲的逻辑差不多，因为没有商业利益的敦促，他能够让自己上进的动力就来自于荣誉。你没有足够的钞票变现的时候，你的这些获利的方式就来自于荣誉。荣誉的高低决定了你最终的成果的时候，就会力争上游。只要你真的力争上游，一定会发现其实中国人做什么都行。为什么？为什么中国人看上去做什么东西都行？哪怕是无论怎么瞧，我们都有短板的短跑，我们依然能够出很不错的成绩。为什么什么都行？因为中国人的这种体质的特征，身体的体质的特征是非常非常均衡的，是没有明显短板的，非常均衡的。这是我们在进行自然筛选、物竞天择之后的适者生存。我们都是大筛选之下的赢家的后裔。什么样子能够适者生存？食谱足够的广泛，过敏源足够的少，粮食的利用率足够的高，神经的敏感程度足够的高，耐寒耐热的忍耐力，甚至包含了饥饿的忍耐力，足够的强，包括我们的激素分泌足够的稳定，既不会过于疯狂的亢进，也不会突然之间陷入萎顿和低迷。这些东西是在任何领域里面都能用得到的东西，这是基本素质。包括在运动领域当中，它一样非常的有用。最顶流的运动员，他的智力一定不能差的，因为他有很多的东西要进行分析。顶流的运动员并不是四肢发达、头脑简单，他需要有充分的分析的能力。所以我们在那些精巧型的运动当中占据了非常高的水平，非常高的咖位。为什么？因为基础条件、先天条件放在这儿，我们擅长于做这些精细的微操作。那为什么国足不行？赚太多了，赚太多了，太舒服。

【备注】  
修正的错误词汇如下：  
1. "出品"（原"我们哪些项目能够出品"）→ 改为"出息"（指有成就）。  
2. "出席"（多处，如"他哪个团队能够出席""不赚钱的都出席了"）→ 改为"出息"。  
3. "桥"（原"哪怕是无论怎么桥"）→ 改为"瞧"（意为观察）。  
4. "神神然"（原"自古以来就神神然"）→ 改为"神圣"（结合上下文，指古人认为蹴鞠神圣，但被曲解）。  
5. 删除了重复的字词（如原"我印象当中是是山东"中的多余"是"；"这是早早几年的故事"中的多余"早"），但内容未删减。
