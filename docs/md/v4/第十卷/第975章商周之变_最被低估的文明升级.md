# 第975章 商周之变_最被低估的文明升级

### 商周之变：被低估的文明升级——从迁徙到定居，华夏精神内核初定型

——《商周易代鉴》

观夫玄鸟栖檐，周粟盈仓。牧野戈销而礼乐兴，殷民星散而血脉长。昔者纣台倾酒，实为聚族之仪；今见周鼎铭文，尽作劝农之章。岂知耕战相生，农时即天时也；奢俭易位，道德亦术道哉！彼黍离离，非关易姓；其民淳淳，总是故邦。可叹简朴成风，竟掩九鼎之重；规划为美，谁察万民之伤？夫计划虽精，终困阡陌之畴；礼制虽严，难缚天地之纲。

周在处理商的移民之后，将商的国人分散到诸侯之中，继续当国人。所以，周是不是商的延续？呃，我不知道。你特意在意“周是商的延续”指的是什么？它确实存在着政权的更迭，而且这个政权的更迭不是普通的更迭——殷商与姬周之间的政权更迭不是一般性的更迭。因为他们中间生产力发生了跃升，所以形成了一个现象：这一次政权的更迭也同时带来了一套上层建筑的更迭，秩序发生了较为明确的变迁，甚至审美情趣都开始发生了变迁。这些东西都发生了改变。

我先给你们谈一本书——《西周人的日常生活》。其实讲的不是什么普通的西周人，讲的是贵族。咦，只有一个平台上有吗？那可太寂寞了！那另外一个平台就给你放这个吧。好，那个……神器易主？嗯哼，啊，好的好的。

商周易代和其他时代的普通王朝更迭不完全一致，主要原因是这个时候的生产形势发生了较为明确的跃升。周的时候，我们还没有能够形成一个完全稳定的定居——我们确实是定居的生活，但这个定居不够稳定。这可能取决于几个方面：一是生产力的水平，比如农耕的肥料回填系统还不够完善，有可能造成我们在长时间耕作之后需要远距离轮耕（也就是游耕的现象），就会出现迁徙。同时，也有可能是由于殷商的政治制度造成其内部权限的更迭，所以王都需要经常迁徙——在继承权的问题上可能需要迁徙。所以由于多方面的原因，殷商实际上不是一个非常安稳、一直稳扎稳打的时代。但是到了周的时候，它实现了完全的定居生活。这个时代代表了生产力的极大跃升，这种完全的、深度的定居生活带来了整个价值体系的变迁和所有体制上的变化。所以，西周的管理方式和殷商开始发生了转变。

大家都知道，我们今天很在乎的所有体系的建设都源自于周礼。当然，周礼不是突然诞生的，它是在商礼的基础之上诞生的。商也是有礼制规范的，社会制度的配套规则在商的时候就有。但到周的时候，由于生产力发生了改变，上层建筑也需要跟着发生改变。所以，周礼和我们后世的这些各种秩序（包括秦制等）中间有一个非常紧密的衔接关系，它们都建立在相似的生产生存模式之下。这种情况下，它会带来的衍生作用当中，其实包含了一些生活审美情趣，甚至包含了道德标准上的描述，都发生了改变。

所以有些人会觉得商周易代似乎人不一样了。但一样吗？人是一样的人。当年的商的移民也一直在周的时候延续了自己的香火，传承人还在，甚至很多的贵族群体还在。那么，同一时间，商代时候就存在的“交领右衽”，到了西周的时候通过礼制的方式被固定下来了。所以人没有变，基本的审美的东西也没有大变化。但是内部由于生产力的变迁，确实发生了变化，这跟我们书里讲的一些东西有关系。因为在西周的时候，很多贵族开始崇尚简朴——听起来有点奇怪，贵族还简朴？对，西周时的贵族在审美情趣或道德标榜上开始追求简朴，这和殷商时的格局变得不一样了。

殷商的时候，大家注意纣王身上有一个很大的罪名叫做“酒池肉林”。这个事儿肯定是子虚乌有的，但能够形成这个词汇，其实代表了另外一件事：殷商时期的贵族表示自己是贵族的方式，是吃很多的肉、喝很多的酒、粮食有富余——他们是通过炫富的方式来表明自己的贵族身份。但是到了西周的时候，就变了。西周的贵族开始追求这种简朴感和仪式感，不再去整那些很奢靡的东西。其实他还是奢靡的，可是装也要装得自己简朴。这就是西周——它已经是一个非常严密的农业社会，而农业社会是非常讲究计划的。

因为到了西周的时候，农业变成了国家内部的第一产业。农业最重要的事情就是得学会怎么计划、怎么规划：要留多少种子，今年的收成是多少，这些收成要拿来做什么用、怎么分配。所以，西周的时代最重要、最重要的生产管理学的特征，就是懂得如何进行规划、如何开源、如何节流。甚至在西周的时代发动战争，都要去计算自己什么时候种田、什么时候插秧、什么时候收割。西周时代所发生的战争（包括春秋时代的战争，它是延续了西周战争的规则的）都要求符合农时。所以这个时期是一个很明显、非常强调计划、非常强调规划的时代。因此，对于“节约是一种美德”就给他定了调。不同的时代，大家有完全不一样的追求。

所以，可以说，商周易代是在很多方面产生了划时代的进步和改变的。但是两方完全是同一套文明体系，也是同一拨人。说什么换种类了、文明不一样了、不是一拨人类了，这种纯属造谣生事。其实也对，打仗打的是后勤，因为你是纯农业国家。为什么在商代的时候不用搞这个事？因为商代的时候，我们自己也没有达到这种完全定居生产。当你没有完全定居生产的时候，内部的这部分产业就是方便迁徙使用的。当你很容易迁徙、不是必须在这个地方安土重迁的时候，其实打仗不一定要依着农时。至于说什么粮草先行？没有粮草去对家拿啊，知道吧？没有粮草的时候去对家拿。
