# 第957章 长安真相_万邦来朝背后是铁血军靴与户籍制度

【长安真相】｜万邦来朝 背后是铁血军靴与户籍制度！#吃瓜蒙主《霓裳辨》长安幻夜，霓裳非胡。商队络绎，衣冠殊途。胡无人则汉道昌，圆领禁而市井疏。岂曰包容？实为鲸吞之势；妄称多元，不过羁縻之图。移民危厦，终溃于安史；灯塔迷舟，焉渡得重溟？嗟乎！昔人琢唐为镜，今人照镜成唐。

农夫应不应该包容蛇？东郭应不应该包容狼？你用几个关键词去说一下，你觉得唐王朝是一个什么样的时代？你对它的形象描述，你感觉到应该是啥样的？——包容、开放、自信、善战，大气万邦来朝，包容、发达、富强，跟今天的阿美一样，多元、扩张、混乱。也就是说，大家印象当中的盛唐（我不是指客观现象，而是普遍印象），它是一个开放的、包容的、多元的，有如今天阿美一样的地方。为什么会这个样子？因为大家印象当中的盛唐，就是仿照着今天的阿美去创作出来的一个古典军事帝国。今天大家能够在各种媒体上，甚至在自己的直觉当中感应到的那个盛唐，并不是真实的盛唐本身。这倒是很正常的事，历史嘛，也没有监控录像，我哪知道呢？我又没经历过。但是它有一个很典型的特征：当我们把它所有关键的形容词抽出来的时候，高频出现的词汇和阿美所创造的所谓“最顶级的灯塔国家”的词汇是一模一样的——多元、开放与包容。盛唐真的是这样吗？问题就在这儿了，盛唐它就不是这样的。

你说盛唐开不开放？如果盛唐开放的话，我们的历朝历代，除了满清都是开放的。除了清，我们没有任何一个时代不开放。开放是什么？开放是指允许别人来，而且我很热衷于冲出去。盛唐是谁都能来的吗？不是啊。有很多人在描述盛唐的开放与包容时，说“谁来谁都是咱自己家人”，盛唐压根不允许这种事情存在。你来到唐朝做生意，正当的生意人都要登记造册，有人监管；你做买卖时有专门的市场，你和其他人的市场不能混用；注册的执照是分类别的，而且还有服装上的要求：胡人不可以着汉装，有严格的规定，而且胡人不能够娶汉女。你说这个东西叫做开放？怎么开的？历朝历代都是允许通商的，我们能够带着商队、带着护卫队一起出去做买卖，那一定会出去的呀。这个叫不叫开放？提着刀、带着剑、带着商团一起出去做买卖，开门自由贸易，请问这个叫不叫做开放？那么，除了清，我们的任何一个时代都是开放的。汉代的时候，从我们凿穿西域开始，就是商队夹杂着部队一起开门自由贸易；唐也是这样干的；宋也是这样干的；明也是这样干的。你觉得宋比较弱，但他在海上的力量并不弱，一直有大规模的海贸，直接就是各种舰船压阵、护航、开门自由贸易。这个叫不叫开放？如果这个叫做开放，那么除了清，其他所有时代全部都是开放的。那你怎么能说盛唐给你的印象是开放的？应该说所有的中国古典时期都是开放的才对。

然后你说它是多元的。一般说多元的时候，是说“哎呀，这个地方好多民族、好多种族，大家都住在一起”。刚才我没跟你说吗？哪个地方都在一起了呀？生活不可以在一起。除了你有了特殊的身份，否则的话你在司法审判的时候待遇不一样，明文规定的。就是在进行司法审判的时候，你拥有的公民权是不一样的，你没有权利穿汉家衣冠。今天有好多人说“哎呀，唐圆领就是胡人的衣裳”，哼，很遗憾，胡人不许穿这个。因为在唐代，这个算汉装，胡人不能够穿唐制圆领。她要圆领？只能穿他们自己家的那个小袖的、收腰的、对襟的圆领，不能够穿唐制的交领圆领袍，有规定不让穿。他是咋多元的呢？就是我们开放了几个市场，允许贸易。什么时候都会开放市场、允许贸易的，哪怕是清这么闭锁的一个时代，他偷偷摸摸的也要开一个十三行，去进行各种海外贸易。所以啥叫多元？

嗯，李白是什么角色？李白是一个诗人，诗是什么诗？就是那个时代的流行歌曲，就和今天你去唱的那些歌本质上是一样一样的。他是一个流行歌曲的作词人，顶级作词人，他就是那个时代的高配版、顶配版的方文山。他在自己的诗歌里面直接高唱“胡无人，汉道昌”，然后你说这个时代开放又包容？所以唐代是一个有明确的自我认知的时代。它之所以土地那么广大，并不是因为开放，而是因为强大，是因为它有强烈的扩张欲望，想要开疆拓土、征服世界，是因为他有一个极致的扩张的野心。所以他的境内才会拥有各色人等，他不是为了打开国门，纯粹是在帮自己圈地。然后在圈地的这个过程当中，对于新招揽进来的这些人，并没有给出一致性的待遇。这个玩意儿不叫做开放和包容，而是在试图消化对方。要知道，要想实现把别人真正归入自己家的话，步骤是非常漫长的，因为别人也有自己的自我认知。你怎么让这件事情消弭下去，让他的自我认知跟我们一样呢？就是让他有这种羡慕的感觉，然后他才会想要融入。可以看看今天的华人——今天我们就是润出去的那些华人们啊，跑出去了之后，他就想方设法地要融入他人的社会，甚至会痛恨自己的脸、自己的五官，会调整自己的面容、化妆、穿衣服、说话的方式，千方百计地模仿他人。谁说他是中国人，他跟谁翻脸。唐代的那些胡人就是这个样子的。为什么这个样子？因为唐代官方制定的规则就是歧视胡人。你生活在这个地方，但是你的待遇不一样，然后别人告诉你说谁待遇好？汉人待遇好。于是他们就千方百计地让自己变得越来越像汉人，甚至会发生这样的争执：有人说“哎，咱都是胡人，大家一块儿干吧”，谁是胡人？你才是胡人，你全家都是胡人。那个时代就会出现这样的戏剧化的场景。只是说盛唐的这个高峰期太过于短暂了，所以虽然他曾经做出这种尝试，但这个尝试并没有成功，他没有来得及真正消化它，吸纳进来的人就已经开始雪崩了。于是呢，就出现了唐瓦解之后所留下来的种种祸害和遗毒。那些接受了盛唐的招揽、认为自己是唐人，但实际上压根儿没有真正规划、还有自己族群的独特认知的人，在这个时候就变成了“只要大唐死了，我就是盛唐的继承者”。因此在后来的时代里头，这个事儿就变得越来越乱了。

所以你说盛唐是一个什么样的形象？我们今天脑子里面直观性的那个盛唐的印象，实际上是今日之阿美在我们的古典时期的一种映射，它是一种想象，而这个想象并不是一个美好的想象。还是那句话：移民国家没有好下场，移民国家不可能有好下场。阿美、欧洲都是前车之鉴。只要你的人群构成纷繁复杂，你就只能走上坡路，只能富有你；任何一个弯道都不能走，任何一个下坡都不能滑，出现任何挫折的时候，大家就一定会各回各家、各找各妈。为什么？因为你是移民国家，我不是这里的原住民，我跟你之间没有一对一的关系。所有的移民国家面临到的都是这样的困境。但是在今日某一些群体所营销出来的古典主义的盛唐的那个时代，他们把它描述成了这样的一种移民国家的生态。盛唐显然不是——它是试图消化外来人口的，只是没有做到。
