# 第924章 文化话语权与文化定义权

文化话语权与文化定义权  
你认为，官方发布说“只有这种款式才叫做华服”，这件事能不能做到？有没有一小指甲盖儿的可能性？比如规定交领右衽、对襟交领、对称受力结构是我们的传统服饰。这种可能性是否存在？  
对，这就是我今天要说的。现在你必须回答我：你认为它是否存在？因为我们不能讨论一个完全不可能存在的东西。如果脱离民族的维度，什么都不用谈。你就说，现在官方因为强大了，要定义自己的民族符号了，或者要定义我们的文化符号了，所以规定我们的传统服装是交领右衽、对襟交领的对称受力结构。这件事有没有万分之一、十万分之一、百万分之一的可能性会推行？有或无？  
就你刚才定的这个定义，这种衣服无论叫什么都好——叫华夏也好，叫华服也好，我都无所谓。只要这种款式的东西被拿来作为我们文化的包装纸定下来，定义为“这种款式的衣裳就是我们的文化外延”，官方有没有万分之一的可能性说出这种话？你必须先弄清楚这点，因为别的东西都是其他的戏子。这件事说不清楚，其他所有东西都是在做梦。  
就是说，官方不可能这么定义吗？  
绝无可能。  
不定义已经是最好的定义了。如果真的去定义，你怎么知道我们现在内部没有常凯申的死忠粉认为旗袍和马褂才是我们真正的传统服饰？道教的标准道袍都已经加上了蜈蚣扣，某些地方的准官方团体去设计的皇后的嫡衣大礼服已经是厂子金了。所以这件事绝无可能。  
你刚才说的前一个问题和我说的其实是一体的。我不知道你为什么非得把它拆开，但明显这是很一体化的一件事：你把这种东西定义为一个民族服装，意味着它有一个群体——一个庞大的、了不起的、伟大的群体在力挺它、保护它、捍卫它。今天掺沙子的人之所以在街上的名声那么恶臭，就是因为有人在捍卫这些衣裳。当你把它作为一个纯粹的文化符号时，不管谁来都能给它改版，不管谁来都能给它设计。  
我跟你讲，你觉得西服已经扩散了，西服现在就已经失去了话语权。西装的标准制式已经没有了，自己也没有话语权了。西装的发源地欧洲已经不能定义什么叫西装、什么叫不叫西装了，任何一个地方都可以设计自己民族风的西装。当你把它作为一个纯粹的文化包装纸时，就是这样的下场——意味着这个服装没有了它可以依附的人群。衣服是穿在人身上的，这款衣裳没有了穿在人身上的可能性时，就变成了谁穿都行，它只是一个蛮新潮、蛮可爱的款式而已。就像我们今天去穿西装发传单，是因为我崇拜西方文化吗？不是，是因为公司制服这样规定。穿Polo衫去送外卖，是因为他崇尚阿美的文化吗？不是，是因为这是公司制服。当你已经俯视到了这样的程度时，它就消失了。  
很多人误以为西装现在代表着西方人的所谓现代化话语权，全世界到处覆盖。现在穿西装的人并不崇敬西方，为什么？因为西装扩散到这样的程度之后，就已经不再是一个人群的符号了。哪怕是那些崇敬西方人的媚外的人，他都不会觉得西装是一个多么了不起的东西。它已经变成了一个谁都可以设计、谁都可以改版、谁都可以调整的衣裳。一套服装变成这个样子时，实际上它就消失了，文化的内核就不见了。  
好的，明白了。像之前的话，也有人在那里说：“我们要语言覆盖啊，现在大家都学英语，将来我要让大家都学汉语，怎么了？学汉语亏了他吗？”对于我这样的人来说，学汉语、学汉字是让他有机会学习更优秀、更先进的文字和语言系统，他在占我的便宜。我还需要为这件事情高兴、鼓掌？哎呀天呐，我被人家占便宜了，好高兴啊？有这回事情吗？没有这回事情。为什么要这样说？因为所谓的英语影响全球，只有在什么时候？英语代表了他们这种特定的、高维的、所谓的白人昂撒文化。当英语是世界霸主时，它才有这样的地位。当世界霸主的地位沦陷时，所有说英语的地方的人都会去追随下一个领导者，或者热爱他自己的群体，而不会再继续把岛上面的那群人当做老爷。英语当它扩散到全世界之后，就变成了一种大家都能使用的工具，它就失去了原有的这种文化符号的特点。你也从小学英语，凡是上了学、像你这样还能写论文的人，学英语一定学得很好。你会因为学英语就变成了英国人吗？不会的。你会因为学俄文就变成俄国人吗？不会的。  
大面积的扩散之后，它就会变成一个工具。服装也是这样：它什么时候是文化的符号？当它独属于某一个群体，其他人想要拥有这件衣裳、想要学习这件衣裳，都要经过别人的授权；然后发明人还可以指着你鼻子说“你穿的不对，改版重穿。我们的祖先就是这样，我们也会给周边的藩属国、亲近的国家赐服，赏赐过去之后会告诉他，你可以穿什么样的，不可以穿什么样的，要立规矩的”。为什么？因为文化的所有权还在我们手上，我们来定义它可以是什么样子、不能是什么样子；做成什么样子你就违规了。这是拥有了文化的定义权，然后我授权他者使用。这时这个服装是有主体人群的。  
但是无论是西装也好，英语也好，现在都已经失去了它的主体人群。你跑到阿三那里去，听阿三说的英语，它和本土的英语其实已经有很大的偏差了。甚至有一些阿三已经开始觉得英国佬的英语不标准。为什么？因为岛上的那些岛民已经失去了英语的话语权、定义权，他已经不能够指着别人的鼻子说“你口音不标准、你的语法错了、你这个拼写不对”。他没有这个权利了。每一个地方运用英语的人都会为这个英语添砖加瓦，甚至调整他的口音、调整他的语序。现在中国也在调整英文的语序，我们已经改变了它一部分的词汇、一部分的口语上的语法。这时，岛上的人对于这个语言都失去了绝对定义权。  
服装也是如此。当你把它扩散，扩散到到处都用，你就会失去它。
