# 第906章 简易说易_3分钟略懂周易底层逻辑

「吃瓜蒙主」简易说易：3分钟略懂周易底层逻辑。有人问：“能讲一讲周公写的易经是否有那么伟大吗？”回答：周公的话没有说他写易经啊，他是编撰易经，而不是写，不是原创人员。嗯，之前我们认为传说当中的易，最开始来自于河图洛书，当然这个部分里面的神话色彩非常浓厚，所以不保真，但是它是有渊源的。这说明一件事情：它不是周公旦自己去书写的，也不是他自己原创的，而是在那之前，相关跟易关联的这一套东西早就有了，只是他需要重新梳理、重新编撰，有可能还需要新一轮的拓展。周公旦确实做了这些事。实际上到春秋的时候，像孔夫子等国也都参与其中，做了相关工作。所以易这本书我们今天读的时候，常常上面写“周易”，但它实际的源头非常久远，甚至来自于传说时代，具体时间点目前还不确定。

那么易的意义到底是什么？它里面包含的东西其实包含了天文、算学、哲学，还有逻辑学等内容。这些都是在易当中很明确包含的。它还有没有其他的？我相信还有。为什么这样说？因为中国古人有一个特点：他们非常迫切地要把一些档案记下来留给我们，这就是稳定定居生活人群稳定传世之后，试图将知识积累传于后世。但在传播过程中，一定会发生流散。实际上我们的古人几千年前就发现了：知识会迭代、会变化、会流散。写下来的东西不能保证它一直存续，也不能保证后续的人一定能够看得懂。所以在进行文字书写时，有时他们会刻意将它们进行汇总、搞汇编。你会发现我们的先辈非常执着于搞大规模的汇编。在先秦一些时期，史料比较散乱，但是也能看到包括《尚书》在内的书——它的成书年代也有争议（原文“程颐”有误），但它也是各种内容的汇编。易也是多学科的汇编。你会发现我们的祖先非常想把东西汇总成百科全书式的东西，努力将知识传播到后世。这个事到后来就变成了《永乐大典》，这也是搞汇编的一种操作方法，只是时代不同，汇编的方式和效果也开始改变了。

那么易代表的是什么？易本身就代表了变化，它是哲学的归纳，以及对历史上各种各样思想模型的汇总。所以如果数学很好，和我不一样（我数学不行，多年没学过数学了），如果数学很好，会数学模型，知道如何进行数学数字方面的归纳和整理，完全可以从数学角度上去解读易，因为它里面就有数学。所谓的阴阳是什么？阴阳就是二进制。八卦是什么？八卦就是算法模型分类。完全可以按照数学的方式去解析它。所以不是一上来就说“易是什么，我的文化水平不足以充分理解什么是易”，但你可以按照自己的专业去理解它。为什么？因为我们祖先把它汇编成一本书，是把自己所能够得到的各个方面的知识都堆叠到同一本书当中，希望它不要离散，希望它能够触类旁通，希望它能够启迪后世。所以作为后世的子孙，我们拥有任何一个角度的切入点，都可以去读易，而不是必须把它读通才行，那个没有必要。你懂哪一块就读哪一块，我都是支持的。

事实上，之前我认识一个老大爷，他号称自己是研究天文地理的，实际上是一个看那个啥的先生。从小学习特别不好，但看易经能看懂。他就是偏门，别的数理化之类的东西，不管那个课程多么浅显，他不爱学，学不下去，但易经一翻开，他就看明白了。最后自学成才，走上了那个啥先生的道路。这也是一种方式。我所讲的不是鼓励你去走这个路，而是说它有很多方向的解析。包括现在也有很多的人把它拿来当做什么？看山看水、看天看地，所谓精通天文地理的什么先生，帮人看宅子。所以到目前为止，这个行业也存在。能不能用这样的方式来解析？也没有什么问题。用这个方向去读易也可以。但你学数学的，可不可以读？可以。你学哲学的，能不能读？当然必须要读。所以你用什么方式、用哪个角度，我觉得都可以去读它。
