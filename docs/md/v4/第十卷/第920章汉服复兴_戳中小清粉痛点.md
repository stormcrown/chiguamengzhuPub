# 第920章 汉服复兴_戳中小清粉痛点

小清粉越反对，越证明汉服复兴戳中了痛点。说实话，在如今互联网上的各种人群中，真正会造成恶劣影响的就是小清粉，其他所有人群对于汉族拥有自己的传统服饰都是喜闻乐见的。一个很明显的特点就是：全国各民族对于汉族拥有自己美丽的传统服饰都是喜闻乐见的。除了特定那一小撮小清粉和遗老遗少之外，连满族同胞都对汉族的民族复兴和文化复兴喜闻乐见。因为主体民族越强盛，国家越稳定；国家越稳定，小族群生活就越安稳。正常人都能想明白这个道理。那么怎么办？当这个国家里有90%（事实上超过90%的人群）都认可这样的衣服很重要、很传统、很漂亮、很了不起的时候，它不是国服也是国服。
