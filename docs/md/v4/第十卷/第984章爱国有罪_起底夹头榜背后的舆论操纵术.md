# 第984章 爱国有罪_起底夹头榜背后的舆论操纵术

### 整理后的易读段落

**爱国有罪？｜起底“夹头榜”背后的舆论操纵术 #恨国党**  
——《夹头鉴》  

夫舆论沧溟，易沉赤子之心；网络洪炉，惯销金石之志。昔者公知跋扈，恨国器嚣，爱国如履薄冰，颂华似负千钧。司马南临美利坚之险，颈夹电梯；众宵小举“夹头”之帜，口诛笔伐。然则风雨如晦，鸡鸣不已，纵怀投机之疑，犹存振臂之勇。  

观夫夹头榜列，忠佞同讥；键盘剑横，是非共泯。嘲讽如潮，淹没了青史真言；标签似雪，覆盖了赤诚肝胆。嗟乎！曲直难辨之际，且听余言。  

——  

“夹头”这个梗，你们不知道是什么意思吗？好吧，我来分享一下。  

这个梗最早来自最近因税务问题上了新闻的司马南。这位大主播多年来一直在舆论的海洋里沉浮，早期跟过气功热，什么时髦他追什么，始终站在舆论前沿。  

在那个恨国党横行、公知猖獗的年代，爱国反而像是一种耻辱。谁要是说中国一句好话，就会挨骂。但司马南当年却在如此艰难的环境里，肯说出爱国的话。不管他心里怎么想，能在那种压力下发声，本身就是不容易的。  

正因如此，他引发了很多公知和恨国党的痛恨。讨厌一个人，就会想方设法“开盒”他（人肉搜索）。于是有人挖出司马南在美国买了房子、过日子，嘲讽他“赴美是生活，爱国是工作”。更巧的是，他在美国坐电梯时把头伸出外面，结果脖子被夹了，还叫了救护车，上了新闻。那些公知就拿这件事做文章，笑话说：“你天天说爱国，脖子被夹了吧？”从此，“夹头”就成了讽刺口号式爱国的梗。  

后来，这些人还搞了个“夹头榜”，凡是说中国好话的人都被塞进去，变成一个互联网嘲讽的标签。  

所以，这个词到底是讽刺还是什么？说它的人是在讽刺。如果你觉得说中国好话本身很讽刺，那它或许就是讽刺。但上夹头榜的人，绝大多数都是好人。
