# 第931章 满蒙联姻_嫁到草原的闺女很难活过40岁

满蒙联姻，嫁到草原的闺女很难活过40岁？大玉儿是蒙古嫁过来的。嗯，所以呢？所以呢？哦，你是在我谈和亲的时候去想到的大玉儿是吧。啊，对，一直在清的早期，存在着满蒙联姻的传统。为什么有这个传统？因为双方是好安达，就是拜把子的关系，歃血为盟的拜把子兄弟关系。这种拜把子的关系，然后呢，就互相把妹妹嫁给对方，是一个常规操作方式。那么在姻亲中，不光是有草原部落上面的贵女嫁到皇室里面当妃子、当皇后，实际上也有大量满洲的公主嫁到草原上面去啊。有个别的呢，能活过40岁，大部分的呢，都英年早逝啊，没的特别早啊。
