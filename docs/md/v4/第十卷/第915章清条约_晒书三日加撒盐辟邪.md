# 第915章 清条约_晒书三日加撒盐辟邪

唐诗宋词元曲明小说「清条约」，晒书三日+撒盐辟邪：

**清条约全集 三卷遗编血泪涔，金缯割尽九州沉。  
包茅不入周王贡，舆璧空输赵主琛。  
纸上鲸波焚鼎祚，墨间狐媚蠹天心。  
汗青若照瀛台月，犹照当年五跪深！**

我背后的“清条约全集”链接在哪儿啊？  
这个有兴趣？真的吗？  
嗯，来，这个角度不好，不好啊。  
这是三大本啊。  
三大本你们只能看见两大本——这是错了啊，少了一本。三大本儿，这三大本是我斥巨资购买的二手书。你要是有兴趣的话，也想办法去买个二手书。孔夫子上面有啊，某宝上面也有啊。谢谢，每个人都有哈。你自个儿去找，自个儿去找他的那个……嗯哼，嘶。网上好像也有一些免费版，但是这玩意儿我是压在这里镇宅的，所以免费版不好使啊。我，我是专门斥巨资购买的二手书啊。为啥呢？绝版了。所以呢，你问我怎么挂链接？挂不上链接啊，我又不是二手书贩子，你找错人了。你自个儿去找吧，有，有啊，包邮的诶。  
好，欢迎来到直播间啊。  
啥？有时间记得撒盐辟邪，阴气太重。从他回来之后就已经暴晒三天了啊，已经晒过太阳了。

- 将“迷人都有”改为“每个人都有”（“迷人”疑似笔误或方言口音，根据语境应为“每个人”）
- 将“嗯哼。嘶”保留（语气词，未改动）
