# 第988章 脚盆人为何敢挑衅_唐朝打了坏样板

脚盆人为何敢挑衅？「唐朝」打了个坏样板。  
—— 《临江仙·唐鉴》  

烽火曾照长安月，胡尘漫卷朱门。  
强弓铁骑定乾坤，何须夸海纳，孤冢葬黄昏。  
百年祸乱谁人种？东瀛浪涌鲸吞。  
旧时样板刻新痕，兴亡皆自取，莫怨九天云。  

特别热爱唐，而且强调它的开放。其实呢，唐强盛是因为它自身的武力强大，而武力强大跟它开放半毛钱关系都没有。它的开放是它的死因。唐的这种所谓的开放包容，就是唐的死因，是它死得那么惨的原因，也是后世出现百年乱局的原因。  

所以你看公知，你说他吹的东西有没有道理？有道理。但他净挑你身上的缺点吹，这是等着自尽呢。老坏了这些人。  

然后呢，我们现在来看一下那个脚盆。为什么这个事也跟李二凤有关系？因为当年脚盆人也派了很多人跑到李唐王朝来学习，之后呢，他也觉得自己有了正统权。民国的时候，他们叫嚣所谓的“汉唐在日本”，到今天都还有人相信。来，小伙伴们，有没有人听过“汉唐在日本”的？听过的扣个一。  

来，听过“汉唐在日本”的扣个一。纯属开不了玩笑，你知道吧。哎，脚盆人之所以后来能够有那么大的野心，动不动就想搞你一巴掌，你说丰臣秀吉为什么敢的呢？他怎么敢的呢？他什么人呢？他就敢这样。他怎么就不敢了呢？前面多少人给他打了样的呀！小小的部落敢喊号，说自己满万无人敌吗？那这种情况下，他凭什么不行？样板放在前面呢，你还承认了人家的正统，你还觉得谁强谁有理是吧？谁强谁是爹。那这种情况下，不好意思啊，你打了样了，你让别人怎么办？  

嗯，所以，所以那个时候啊，所以脚盆之所以敢，是因为本来就有先例。当然了，我们今天觉得他怎么敢的呢，是因为他当时其实不强。可是在他自己看来，那个时候他的国力已经不错了，有一战之力了。那他当然就要试一试嘛。
