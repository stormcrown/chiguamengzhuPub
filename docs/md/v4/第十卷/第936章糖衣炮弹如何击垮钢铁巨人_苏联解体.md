# 第936章 糖衣炮弹如何击垮钢铁巨人_苏联解体

糖衣炮弹如何击垮钢铁巨人？苏联青年为牛仔裤疯狂背叛祖国？#意识形态斗争 #苏联解体 冷战隐秘战场揭示轻工业缺陷引爆的意识形态灾难。

嗯，思考社会思潮变迁。其实这个事儿呢，在上世纪的时候，中国也经历过。就是当年美国试图要推翻苏联的统治，这个从冷战铁幕开始就已经在不断不断地着手对抗了。当年世界是分两极的，双方之间都有这种对峙与对抗，属于你死我活的阶段。在这种情况下，欧美方提出了一套解决方案，包含了一个东西，就是我们后来称之为“糖衣炮弹”。还有年纪比较长的人，听说过“糖衣炮弹”这个词吧？就是看上去给你的都是好东西，但你一旦接了这个好东西，那里面可就是炮弹了。

那这个糖衣炮弹当中包含的内容有什么呢？其实最重要的就是一种意识形态斗争。而这个斗争如果都是明堂正道的——我站出来你必须要听我的——那不好使啊！这种斗争太流于表面，非常容易激发对方的防卫意识。真正能够深入敌营、刺痛对方的，一定是润物细无声的东西。所以呢，他们最开始接触到的，第一就是所谓的自由。自由这个东西，谁能说它不好呢？谁也不能说自由这件事不好，是吧？无论你生活在哪里，也无论你生活的方式是什么，没有人有权利说自由不好——因为你一旦说自由不好，他就说“我把你拴起来吧”，瞬间就上套了。所以呢，怎么渗透？怎么让你觉得自由很美好？第一，告诉你自由是对的；然后呢，你应该享有生活的一切美丽是正确的。接着提供的是什么东西呢？最开始影响到的是牛仔裤和流行音乐，这是他们当年的一个重要子弹，重要的武器。牛仔裤加上音乐，这些东西在当时对苏联家里的年轻人吸引力非常之大。

你想一想，之前都在那里，最最浪漫的歌就是“有个姑娘在等我”这样的歌，像《喀秋莎》这样的歌，这已经是非常浪漫的歌了，但这是战歌，昂扬向上，背后是抛头颅洒热血的故事。像这样的歌在那个时候已经属于浪漫了。但是新来的歌曲——颓丧的、摇滚的、奢靡的、堕落的——我说了，世间这种倾向于庸俗、倾向于低俗的东西，实际上非常能够激发人的本能。而这些东西本身有一种客观的魅力存在，吸引人放松，吸引人松弛下来，吸引人瘫坐在那里：“算了吧，享受吧，真好啊，快乐呀。”这些东西非常迷人。

于是呢，在那些感觉到自己并不危险的人中——因为当时苏联已经是一个非常非常强壮的巨物了，是一个巨无霸。这种巨无霸的外在形象，这种强大的武力的外在形象，会使得他的国民有一种安全感。这种安全感就是“反正天塌了有高个儿的顶着”。这种安全感、这种底色、这种底线已经保住了。既然没有了生死之忧，那就瘫着吧，快乐吧，开心吧，摇摆吧——对他的吸引力就会指数级地放大。而当时传递进来的就是这些东西。

这个时候咱们要注意一件事：当年的苏联身上是有一个客观缺陷的，这个缺陷也使得它特别容易中招。什么缺陷呢？就是它的轻工业特别糟糕。有人知道苏联的轻工业糟糕到什么程度吗？其实一直到今天，他们那里的轻工业都很烂。他的轻工业烂到做皮鞋，残次品率高到令人发指；合格品出来了之后，你穿上都有可能系两遍鞋带，鞋带就折了。他的轻工业烂得像坨屎一样。但是当一个国家，它的外壳已经很坚硬，钢铁丛林已经打造出来的时候，人们是会有一定的安全感的。有了基本的安全感之后干什么？保暖思淫欲嘛！我都已经不用丢命了，那我当然就琢磨着：这个面包是不是可以更松软一些？这是天然需求啊。天然需求就是：我既然有了飞机大炮，让我不用丢性命，那这个时候我吃饭，我凭什么要吃黑面包？我能不能吃点白面包？但是苏联的轻工业告诉你不能。于是，这些人就会格外受到这个外来的糖衣炮弹的攻击，因为那个甜味儿对他来说太重要了。

而当这些——无论是靡靡之音，还是流行的服装、流行的发型——你看上去细枝末节，其实不重要，只是吃喝拉撒的一点小小东西。这些小小的改变一旦发生，人们心里的向往、人们的梦就开始发生转变了。人群是需要理想和梦想来集结的，没有共同的梦想，没有共同的执念，人和人绑不到一起去。所以那个时代就出现了一个巨大的问题：苏联这个国家看起来非常强大，但是它内部人心思变。

你吃过他家的甜萝卜？呵呵。我们北边的那个邻居，他家里面凡是跟轻工有关系的东西，除了那些特别倚重于原材料的，其他的都不行。到现在他都不行，一贯如此。而他当年为了打造这样一个钢铁帝国，把他所有的力量都用在了建造外壳上面。而外壳的堡垒越强大，就意味着内部的人安全感越强；安全感越强，他就越需要物质和精神上的满足感。但是由于内里的轻工业的稀烂和文化产业的单薄——因为它的文化产业，自身的文化产业非常单薄，局限于一些极小极小的范围——就使得他自身的国民，在拥有了基本安全感之后，想要拥有的幸福感提升，没有办法在自己这里得到满足。而对方就这样子走进来了。

当对方走进来了之后，注意：我们很难直观地体验到我们的国家在钢铁长城上面的强悍之处，但是会非常明显地体验到这件衬衫是否舒适、这个皮鞋是否合脚。也就是当时苏联的普通人，能够直观感觉到幸福感的东西，来自于大洋彼岸的对家。他的个人认知发生扭转，进而形成群体认知，已经发生了扭转。它达到一定的临界点之后，这个趋势就不可逆转了。
