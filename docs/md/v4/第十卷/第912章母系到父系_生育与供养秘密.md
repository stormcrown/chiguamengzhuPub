# 第912章 母系到父系_生育与供养秘密

从母系到父系，社会变迁背后的生育与供养秘密，谁的孩子更易存活

玄鸟生商只认巢，鲛珠泣血育灵苗。  
麟趾空寻千涧鹿，璋瓦独掌九霄潮。  
一脉能开星宿海，万疆终化铜山镣。  
莫道父系天纲固，血亲深处锁龙桥。

原始社会当中为什么是母系，而不是父系？因为找得着母亲，不一定找得着爹。他不一定能找着父亲，但一定能够找得到母亲。她可以明确母亲是谁，所以就依照母亲作为坐标系的原点，围绕她来建立社会关系。所有的这些社会关系是靠这个连接点串起来的。这个连接点有权利，所以是母亲拥有权利。

在后世转进到父系社会的时候，也就是父亲知道哪个孩子是自己的孩子，而是当生育私有化之后，坐标系转移到了父亲这里。父亲周边的所有社会关系形成了他的核心力量。在原理上面大致相同。但是父系相较于母系有一个优点：母系的孩子有母亲不一定有爹，而父系的孩子有爹就一定也有母亲，所以父系的孩子有双方的供养，存活率会更高。同时，父系家庭它可以扩展的自身范围会更大，因为一个女性一生当中能够生育孩子的数量是有上限的，但是一个父亲在理论上没有上限——实际上当然有，但其可扩张的范围就直线提高了。
