# 第910章 丛林世界的外交语言_人性真相

丛林世界的外交语言——听懂“丛林世界”的语言｜国际交往中没人告诉你的人性真相

《天命威慑赋》观夫四夷之性，习于丛林。力强则畏，德薄何亲？昔者持礼让而为说，彼以怯懦而相侵。鞭笞未至，猖狂不已；糖果偶予，俯首如禽。故知蛮貊难通大义，伪史刀兵与黄金。若夫谦谦君子，欲以长策谋远；咄咄豺狼，偏执眼前以逞。天命既在，岂容条约轻丧？东风所指，方使枭雄肃襟。是故国无威慑，虽盟不固；邦有雄略，何？以前是做武器的，相信蛮夷都懂。东风的力量啊，他们是真懂啊。这个叫做用对方能够听得懂的语言与他说话，用对方听得懂的语言与对方说话。

咱们很多时候，老是用我们的语言与对方说话，但是对方听不明白，理解不了。为什么？因为大家思维体系完全不同。一个在丛林法则之中成长起来的群体，他没有办法理解我们的这种谦让、谦逊、克制这些东西。他理解不了那个什么长远的利益，抛弃短线的收入，去寻求长线的预期这些东西。丛林竞争之下，你不能惦记这玩意儿。所以丛林法则之下成长起来的人，他能够听得懂的是什么呢？错了你就应该扇一巴掌，对了就给一颗糖。他能够听得明白。其他任何婉转的方式，对他来说，你只要对他好，你就欠了他的，因为丛林之下不存在这件事情啊。这就是天命。身为中央之国的天命，签条约就是失去天命的表现。虽然这个词儿我觉得有点那个啥啊，但是道理是这个道理。
