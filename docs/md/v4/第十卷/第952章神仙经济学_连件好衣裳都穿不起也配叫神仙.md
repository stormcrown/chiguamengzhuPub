# 第952章 神仙经济学_连件好衣裳都穿不起也配叫神仙

《神仙经济学》  
连件好衣裳都穿不起也配叫神仙？  
绛阙由来重盛装，岂容瘦骨效夷光。  
吴衣当风承汉韵，周昉施朱胜素缃。  
莫道仙家崇简淡，开元粉本自堂皇。  
可怜今饰蚊纱客，错把寒酸作鹤翔。  

中国传统上对神仙形象的创作，大概走的是什么样的路线？尤其是这些年，很多仙侠戏把神仙都弄得好像是刚刚守了寡的死人相，让人很尴尬。所以，我觉得很有必要让大家都能看到，中国的传统上，会把神仙绘制得多么张扬、多么有力量、多么飘逸，又多么美，繁复且富丽堂皇。  

这个逻辑很简单：你都已经当神仙了，却展现出一副寡淡寡淡的形象——一个像鳏夫，一个像寡妇，好像日子马上就要凉透了似的。那看到你的人，都没办法对你生起崇敬之情。人靠衣冠马靠鞍，你都当神仙了，还不肯好看一点，这不是很奇怪吗？  

现代的仙侠剧，就是把那个“飘飘欲仙”整得好像把蚊帐披身上了似的，像买不起布料、裁不起衣裳的样子。那一个又一个所谓的神仙仙女，恨不得肋条骨都要露出来戳我脸上了，一副吃不上饭的样子。这是严重违背了我们传统意义上对神仙的审美价值观的。  

神仙应该脸颊饱满圆润，面容端方大气，衣着富丽繁复。一看就震慑我——我的天哪，有这种震慑感，那力量才能冲击到我的面前来。所以，今天就来介绍一下这个，我们瞧一瞧中国神仙这一套形象创作的传统。

- 将“这种是严重的不严重的”改为“这是严重违背了”（原文口语化重复，修正为更通顺的表述）。
