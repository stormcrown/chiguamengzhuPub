# 第941章 被烂菜叶子砸千年的奸雄_到底冤不冤

被烂菜叶子砸了千年的奸雄，到底冤不冤？  
曹操在徐州屠城，你怎么看？  
哼，又是想啊，把关注点分散开来，分散到五千年。然后某一个地方，它的不文明就显得不那么醒目了，是吧？这就是：虽然他是负分，但是别人也不是满分啊，是吧？虽然他是负分，但是别人也不是满分，大家都不是满分，大家是一样的。不就是干这个事的吗？还能怎么样啊？  

曹操怎么了？嗯，曹操其实对历史是有很大的贡献的。但是所有的戏台子上面，他都是配角，都是反派。被人骂了多久？作为一个赢家，作为一个赢家，曹操被骂了多久？他死了多少年，就被骂了多少年，不是这样的吗？更何况是什么样的局面呢？终其一生，一生被骂了多少年？你自己算数——是不是终其一生就被骂了多少年？然后你说“都一样的，都那么坏”，一样吗？  

曹操是著名的奸雄，没有人说他好话。他做出了那么巨大的历史贡献，但是因为人品不完美，所以一直被人骂。戏台子上面、画本子里面他都是反派。拿他做例子？你可找错例子了。他又不是圣天子，曹操一辈子没有当过圣天子，从来没有接受过被他迫害的人的“谢主隆恩”。他被人骂了几千年，身上全是烂菜叶子。你拿他举例子，嗯？
