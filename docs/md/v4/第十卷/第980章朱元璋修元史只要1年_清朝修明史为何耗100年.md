# 第980章 朱元璋修元史只要1年_清朝修明史为何耗100年

### 整理后的易读段落

《踏莎行·读史》  
墨篡青编，烟销铁证，修书更是焚书令。百年刀笔刻虚名，谁知纸缝孤魂哽。元史匆成，明史细订，从来后朝评前政。若求真相应焚何？灰飞早罩千秋镜。

为什么叫“青石稿”呢？那是民国时期的学者写的一篇马屁文章，属于马屁史料。虽说理论上，民国确实应该为已经没有了的清王朝盖棺定论，写一篇墓志铭——这就是所谓的史书。我一直强调，史书后朝修前朝史书这件事，其实代表的是：给前朝写墓志铭。所以，他用什么样的方式写，更多印证的是后边的朝廷到底对前面那一拨是怎么看待的。

像朱元璋，他去修元史时，实际上约等于啥也没写。他呢，标准就是在行军路上，要求必须先把对方钉死在棺材板里，好让自己成为唯一的统治者的一种操作手法。意思是：这个墓志铭写得好不好，我不在乎，反正只要有这个墓碑立在这里，你死也好不死也好，反正都死了。这就是宣布权力的一种方式。所以他的修编速度特别快，一年都没凑齐，订了个书皮儿就把东西弄出来了。

但是到了清修明史时，就完全不一样了。修了将近100年，修完以后，还把原始的档案资料付之一炬。这是什么呢？这就是（谢谢冬瓜爱情）清王朝在修史这件事情上，不只是要把前面的明王朝给摁在棺材板里，而且一定要让明王朝生前的影像也全部失真。他希望后世对这个时代有一种混沌的滤镜，不知道它到底长什么模样。所以他才需要动用无数的“PS”手段，写一些似是而非的东西。你说里面全假吗？不至于。大致的脉络和事件必须是真的，要不然就离了大谱，圆不起来。但是，事件的具体描述和态度，基本上都是颠倒黑白的。所以他才那么急迫地在修完明史后，将原始档案付诸一炬，销毁到千不存一的程度。这是一个非常非常严重的态度表现：清对明十分警惕，甚至十分恐惧。所以明知明王朝已经消亡，在给它编撰墓志铭时还慎之又慎，甚至在里面动了无数的手脚。

同时，《清史稿》理论上属于民国时期，也应该给清王朝写一个墓志铭。但第一，这时比较混乱，确实没有那么大精力专门去做；第二，《清史稿》看上去还颇有仪式感，似乎也算写了墓志铭，但它本质上是一篇马屁文章（谢谢远方何方）。它实际上是对清王朝进行了一番涂脂抹粉和歌功颂德。所有人只知道他修明史，不知道24史也修过，都修过啊。所以叫做“四库全书出，而古文亡”，就是这个道理。打从他（清朝）把《四库全书》整体修编出来之后，原本的文章、原本的文献就被事实上的销毁了——他是摧毁式地进行图书修编。

### 备注：修改了哪些词汇

1. “观定论”改为“盖棺定论”。  
2. “原始”改为“元史”（指朱元璋修元史）。  
3. “而古文王”改为“而古文亡”（原文引用“四库全书出，古文亡”的常见说法）。  
4. “圆不起来”中“圆”保留，但通常写为“圆”或“圆谎”，此处未改。  
5. “千不存一”保留，未改为常见“十不存一”，因原文如此。  
6. “涂脂抹粉”保留，未修改。  
7. 删除了部分口语重复（如“朱元璋啊”“他呢，这个呢”等），但保留核心内容。  
8. “谢谢冬瓜爱情”和“谢谢远方何方”作为互动保留，未改动，但用括号括起以区分正文。
