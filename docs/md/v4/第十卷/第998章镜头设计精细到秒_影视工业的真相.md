# 第998章 镜头设计精细到秒_影视工业的真相

《鹧鸪天·影视预制》  
镜里春秋套路深，舌尖犹借外邦音。  
三分景别裁情绪，几秒宫商挑客心。  
工易效，意难寻，流水线作宴如林。  
劝君莫笑农家拙，自有真诚胜铄金。

当初《舌尖》在拍摄的时候，不是自己脑子一热就拍了。整个纪录片学习和借鉴了一个非常成熟、非常成功的纪录片——日本那边的作品。那个纪录片本身拍得很好，对日本怎么看是另一回事，但他们的影片拍得好，技术水平高，画面配色好，节奏把握得好。这个东西，咱们有一说一，得承认。

当初最开始版本的《舌尖》，在拍摄和制作时，就学习和借鉴了日本那边的纪录片。什么？《孤独的美食家》？没关系，不是那个。我忘了叫什么名字了，但日本那边有一个很优秀的纪录片。我跟你说，你不能直接关联到美食上面，这不是美食类的，别老跟我纠缠这事。我忘记它叫什么名字了，也不是美食类的。

重点是什么？重点是那种成熟的纪录片有一个东西：你作为观众去看影片时，看到第几秒时你的情绪会到哪个位置，然后你需要一个音乐加上一个镜头让它轰一下子起来。再隔几秒钟，又会进入下一个小高潮，人的情绪又开始低迷，于是你又需要一个动态的镜头、一个音乐，嘭一下再把你的小心肝一挑拨。这就是当初他们借鉴的影视制作手法。

所以我说这不是什么《舌尖》的问题，他在学习别人的设置和制作技术。这种技术就是所谓几分几秒时用大景，几分几秒时用近景，几分几秒时音乐要高挑。这个东西是什么？就是你刚才说的预制菜，知道吗？这玩意儿就是预制菜。对，纪录片拍摄手法如此，所以他不是去学习了什么《孤独的美食家》，不是这个东西。他当时学的也不是这个片子，他学习的是别人的专业技术手法，还有别人把握节奏的方式，以及音乐编撰的节奏感。这些东西都属于技术流。

但是当这个技术已经可以归纳总结为几秒钟一个阶段、几分钟一个大节、几分钟一个小节时，它变成了什么？它变成了一种预制菜。知道吗？这种东西就是影视专业行业当中的预制菜，早就有了。实际上早期的预制影视作品特别优秀，我们早年间看的很多好莱坞大片都是这样的预制产品。它们形成了一个完整的电影制作产业链，形成了生产线。所以在那儿，每一个行业都是一个很小的工作团队，约等于一个人，甚至一两个人。到最后真正形成作品时怎么办？散兵游勇拼起来肯定不行，所以有一个非常清楚的套餐、套票，也就是清晰的创作框架结构：你的故事应该是什么样的人和什么样的人，产生什么样的矛盾冲突，在什么时间开始入手，用什么样的节奏开始入手，当什么时候形成主要矛盾，嘭一下冲击心扉，在什么时候缓缓落下帷幕。它变成了一个成品套餐系列。这个餐单已经精细到了秒，甚至更加细节的程度时，它就是大家所知道的预制菜，也就是电影已经形成了完整的工业标准化产出。

这几年我们也开始学这个东西了，学了工业化，学了技术，但内容没法学，因为内容已经烂得一塌糊涂了。好。

啥？城里套路深，我要回农村？你以为农村套路就不深了吗？谢谢暗送秋波，谢谢清香白莲。嗯，啥？预制菜没有现炒的好吃？这个事儿吧，在它的预制标准很高时还不一定。你想一想当初好莱坞的大片，那些风靡全球的影片，其实绝大多数都是预制菜。这时你说预制的不好吃，跟着工业流程走的不好吃，谁也不信。是今天你看他的电影很烂，你才说预制的不好吃、套路的没意思。但是在它刚刚设计出来的时候，这个玩意儿是很高级、很在行的，知道吧？

那个时代也有一些“小炒苍蝇馆子”，就是那些小制作的、个人团队搞的非主流影片。当时除了好莱坞大片之外，其实美国那边还流行很多非主流影片，这些影片常被冠以“艺术电影”的名头。它们的特点是别致、和主流不同，甚至刻意逆主流而动，但另外还有一个特点：普遍性的难看。啊，标准高就是好吃？对。所以我说预制的产品，预制菜不一定是难吃的，同时预制的电影也不一定是难看的，关键看做工到底怎么样，诚意到底足不足，管理到底是不是精妙。当时的美国预制影片——好莱坞大片系列，其实品质还是很有保证的，反而是那些精致小丑、私人小食堂这类艺术影片，不光恶心你，还难看，知道吗？
