# 第977章 岛国宿命与侵略基因_从地理视角解构倭寇文化

岛国宿命与侵略基因——从地理视角解构倭寇文化·樱花背后的铁蹄

《东瀛鉴》有言：扶桑悬岛，蛇形沧溟。地无三纵之险，陆缺九回之屏。浪涌四面而心悬千仞，山裂八荒而气郁重阴。于是丧文化滋，太宰治文如鸩饮；死亡美祟，浮世绘色似魇侵。观其屋宇低檐，犹困兽蜷缩；赏其光影明灭，实幽魂沉吟。昔者徐福东渡，本传仁义之种；鉴真西行，原播大乘之音。奈何岛瘠难养参天木，水咸不育连理林。宋始乃有船载女，暗行借胎之计；明寇持刃，妄图劫掠之谋。

我们今天要聊一聊倭寇侵华之心不死的话题。为什么会这样？首先得看它的地形。你看看日本的地形，光是看地图都会替它提心吊胆：它不只是一个岛，而且是一条狭长、散碎的岛链。这样的结构会让人在生活中严重缺乏战略纵深。想想当年抗战，中国人为什么能保有最后的底牌？持久战的战略思路，倚仗的正是我们的战略纵深——我们撤退时还有地方可退，有可能保存有生力量。中国人的心里，安全防线跟其他群体不同：后退有路，体量巨大，所以个体也好，群体也好，安全感实际上比别的群体强。这种安全感让庞大的群体很难直接产生危机四伏的意识——没有直面危险，就很难生成危机感。所以中国人普遍在情感上更有安全感，自我认知更稳妥，这也是人格健全的表现。

但日本那个地方就不一样。你去看他们的电影、读他们的书，会觉得人有些变态。之前有位小朋友给我推荐了一本日本作家写的书——那个作家很有名，得过不少大奖，我一时间想不起来名字了，这不重要。我看几页就看不下去了，特别难受。我就觉得那个人心理不健康。文笔、故事先不谈，你听他说话就丧里丧气的。看完几页纸，觉得不想喘气了，太累。他的文学和艺术品带来的直观感受就是丧气、阴暗。即使是描述辉煌的、阳光明媚的场景，描写恋人温馨的互动，听完后你也觉得“算了吧，散了吧，放了吧，别要了吧，离了吧”——不管描述多好的东西，让人感觉“得了吧，没劲”。这就是整个文化氛围和思维的底色——严重缺乏心理安全感，心理防线脆弱。太宰治就是典型。他的文笔先不论，重点是你看了就觉得丧气、灰暗。这种东西从哪儿来？看日本地图就能明白：狭长的海岛，不仅狭长，还是海岛——没有战略纵深，遇到事情没地方退，没有后路。资源禀赋又差，诞生的文化特征中就包含了丧气，于是就有了欣赏死亡之美的倾向。他们的绘画作品中也有这种味道，有一种独特的绘画方式：喜欢画屋子里的美人——日式仕女图，专门画室内的女性，运用独特的光影效果——非常黑暗的屋子，给人物身上打一束光。看上去人挺漂亮，有光晕笼罩，但你一点都不觉得光明，心里刻下最深印记的是背后黑黢黢的黑暗。他们喜欢的就是这种调调。这跟自然特色息息相关：凡是环境狭窄的岛屿，上面的人和动物在生长发育乃至基因进化上都会倾向于变小。这种倾向影响整个群体的心理：一方面是自我放弃、自我放逐的丧文化；另一方面是想要获得安全感——没有安全感的人特别渴望安全。因为土地狭窄，他们特别想要宽阔的土地；因为自身丧气，所以特别向往阳光温暖。这种开阔、阳光、灿烂的生机、蓬勃的文化和人群，他们见过吗？不幸的是，他们见到了——中国。从第一次接触中国人开始，他们就见到了蓬勃如春天、如朝阳、如旭日般的国家。所以他们在自己的文化中特别喜欢强调“旭日东升”的概念，觉得自己在中国的东边，连太阳都比中国升得早。但实际上，真正拥有阳光力量的是中国人。他们特别羡慕这个地方，有条件就想来学习，学了之后试图改造自己的东西。有人声称汉唐文化在日本保留得很好，但实际上，他学没学？学了。能保留那么好吗？保留不了。这不仅是心意的问题，而是做不做得到的问题。汉唐文化最典型的特征不是柱子怎么立、建筑怎么横平竖直，而是磅礴大气的盛世之风，来自慷慨大方、视天下为无物的开阔胸怀。这种开阔感不是岛国能承担的，跟他想不想一点关系都没有。他跟我们学了，也不可能学像，因为我们渴望生命、渴望阳光、渴望绵延万代、渴望延续的文化底色，诞生于民族的构成之初，而日本严重缺乏这个——土地和文化都不支持。所以他学习之后做出来的东西也不像。比如所谓的唐风建筑，非常低矮狭小。原因一是当地材料和技术不支持建造那么庞大的建筑——唐风建筑特别开阔宏大，是大骨架类型；而日本岛上的所谓唐风建筑却非常低矮狭窄。他确实用了唐风元素，但为什么不对劲？因为岛国上身高1米6的男人就算大高个儿、大壮汉了，普遍身高1米4，你盖一个门框两米多高的房子，那人没法住。哪怕借了工匠、砍了最魁梧的树、有条件做成那样的房子，住进去也心慌——身高1米4，看着旁边一根柱子就能把自己压底下，这日子没法过。所以他的房子特别低矮，真正去过古朴民居的人会发现，正常的中国人进老式民居屋子要低头，因为他们以前的平均身高很矮。整体个子矮，门框就没法高。所以他的学习是似学而非学，这是基础条件决定的。

这也决定了另一件事：他老想改变自己。因为客观的先天不足，总想改变自己。身高问题上，他就一直向中国汉人“接种”，这叫“度种”。从认识中国人那一刻就开始了，高峰期在宋代——航海技术变好，出海风险变小，他们运载很多年轻姑娘到中国沿海，停在海上，招揽青壮年男性上船，等姑娘怀上孩子再运回本岛。这种“度种”行为持续不断。中国强大时，他来找汉人接种；后来中国衰落，他又找白人接种。所以现在日本岛上的人面貌特征很复杂，这是他们一直试图改变先天不足的结果。侵略也是调整中不可或缺的一环——想要获得更好的资源、更开阔的土地，只要有机会，就一定要出来占便宜，从没停过。中国很强的时候，他也动过手——唐代被揍过，明代被揍过好多回。为什么会挨揍？因为他动手，侵略之心不死。他总是试探着抢一波、占一波，发现不行、抢不过，一巴掌被拍地上，于是开始谦逊地当学生学习。实际上，他的学习行为是被迫的，是一种试图侵占的行为，野心从一开始就有。现在依然有。前几年有新闻可以看，他们一直这样做，只是没人报道所以你没注意到。在他们的幼儿园和小学教育中，一直有军国文化的内容，从幼儿园开始教。用这种方式教出来的孩子，精神底色中包含什么？别人家也有传帮带，只是传的不是正经东西。所以今天他们依然怀着侵略之心，只是手法变了。以前赌国运，直接杀上来；现在这招用不了了，就搞“文化换家”。

要知道，中日两国普通人的面貌差异不大。虽然情感上觉得对方不如自己好看，但客观上没有明确种族差异，共同生活十几年就认不出区别了。面貌差异不大，姿态差异由生活习惯造成，共同生活十几年就能拧过来。这就造成一个情形：埋伏特别容易。人员一旦迁徙、扎根生活，很难认出来——这是一个客观现实，得接受。因为有这个前提，他们现在搞文化换家：人迁过来，你又发现不了，再把他们的衣服、建筑、饮食、植物、行为方式、礼节、园林等都搬过来，说这些东西就是中国的——谁说不是就不爱国。假设成功，人住过来了、建筑搬过来了、饮食衣着舞蹈植物都搬过来了，你说这还是中国、跟日本格格不入，骗谁呢？不能这样自欺欺人。所以现在他仍在侵略，怎么侵略？搞文化换家。

一到春天，我就讲“撒库拉”（樱花）。为什么？因为樱花是文化换家中非常重要的一环，影响力巨大。现在中国哪个城市在春天不号称满城樱花？搞樱花季，樱花橡皮、樱花笔记本、樱花笔都见过吧？我这儿还有三支樱花签字笔，还有樱花冰淇淋。文具店老板娘送的。各种文具、餐饮，商场促销贴的全是樱花贴纸，地铁有专列，公交有专列，上面全是大幅海报彩绘，出租车公司宣发广告、赏樱路线图——一到春天铺天盖地全是这东西。回想当年倭寇在全世界侵略时做什么？侵略成功一个地方就在那里种上樱花，然后说“樱花就是故乡”——他们自己的电影、书、人都这么说。当年侵略军走到哪里就种到哪里，难道是为了环保？樱花的部队是什么高级部队吗？不就是烧杀掳掠的匪帮？为什么要种树？因为种树本身就是侵略行为。你明知道这树是干这个的，种一两棵铭记历史，我觉得可以——记得这是侵略者种下的。我们的土地上有许多当年列强开埠、通商、殖民的建筑，我们留着欣赏，没问题。但樱花最大的问题是：你一说它是日本之花、军国之花，立刻有一群人乌泱乌泱冲上来给你盖帽子，说你不爱国。你必须说樱花是中国的花，源自中国，是中国人最爱的，自唐以来人人爱。你要不说，你就不爱中国。这叫什么？这叫文化换家，是侵略行为。把别人家富有格外象征意义、有军国特征的植物，强行说成是自己家自古以来欣赏的植物，这比当年倭寇强行种树还要严重。当年他种了，但所有人都知道这是侵略，反侵略成功后树就被砍掉刨掉——倭寇侵华结束后，中国土地上很多强行种植的樱花树被砍掉，改种桃柳。这件事当年做过，中国人应该很清楚。但现在，我们很多大城市大面积推广樱花商业文化、观赏文化。我所在的武汉是重中之重，但可能还不是最悲哀的地方。最悲哀的是南京。南京是什么地方？30万冤魂在那儿。有些地方种了就算了，恶心；而像南京这样的地方，不单单是恶心。
