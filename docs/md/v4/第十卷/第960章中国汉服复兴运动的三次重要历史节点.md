# 第960章 中国汉服复兴运动的三次重要历史节点

中国汉服复兴运动经历了三次重要历史节点，然而每一次都被同一股力量所打断。以下是这三次节点的梳理及其共同特征。

---

### 第一次汉服复兴运动：辛亥革命后至民国初年

很多人误以为汉服复兴始于2003年，实际上早在辛亥革命后的民国初年，第一次汉服复兴运动就已出现。在此之前，太平天国运动可视为一次雏形，但它未能建立完整的服饰体系，只能算作革命的附属产物。如果将其算作第一次，也未尝不可，但不够完整。

民国初年，由文化群体和部分官僚共同推动的汉服复兴运动曾大有希望。理论上，最大的障碍——清朝统治已被推翻，然而新的阻力随即出现。常凯申（蒋介石）将充满轻装元素的旗袍马褂定为国服，以官方手段硬性打断了汉服的复兴。注意，如果常凯申选用的是中山装或现代连衣裙作为礼服，问题还不大；但他偏偏选择旗袍马褂，这不禁令人深思：他究竟是否算真正的革命者？他截胡革命成果、成为反动派的动机何在？这一节点非常耐人寻味。

### 第二次汉服复兴运动：新中国成立初期

许多人忽略了建国初期的第二次汉服复兴运动，甚至在我们内部也常刻意回避这段历史，似乎新中国与古典中国完全割裂。但事实并非如此。

开国大典上，就有汉服团队参与行进。建国后，我们拍摄的革命影片和文艺影片（如早期的《刘三姐》）中，汉服的还原度非常高。可以说，在新中国立国之初，汉服复原工作就已开展，且做得相当不错。此后，直到某个“不可说时期”的“不可说之人”设计了汉服的现代化方案。因此，汉服复兴在新中国的前30年也是一段重要的文化历程。

然而，这次复兴在70年代末、80年代初被打断。打断它的工具——又是旗袍。如果打断时使用的是西装或西式礼裙，尚可归因于国际化进程；但事实是，80年代初旗袍再次登场，阻断了汉服的复兴。这与民国时期常凯申的做法如出一辙，操作手法高度相似。

### 第三次汉服复兴运动：2003年至今

从2003年起，由下至上的汉服复兴运动再次兴起。在全国各类媒体、服装圈层、各路学者乃至学术圈的绞杀之下，它硬生生地长出来，如今已大江南北烈火燎原，不可遏止。

然而，从2023年开始，旗袍和“清民装”又一次跳出来，宣称“这就是汉服，这就是我们的传统服饰”，并指责不认同者“不团结、不爱国、叛国、抛弃传统”。为什么每一次汉服复兴运动，都是旗袍冲出来打头阵？为什么每次犯事的都是同一种服装？为何巧合如此之多？这其中的微妙，值得我们深思。
