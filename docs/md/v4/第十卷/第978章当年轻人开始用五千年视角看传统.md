# 第978章 当年轻人开始用五千年视角看传统

《华夏衣冠鉴》  
五千年光射斗牛，一朝胡骑破衣冠。  
西伯利亚风霜恶，东欧罗刹鬼火残。  
衣冠南渡心犹在，文物东归梦未阑。  
莫道神州无正色，重开日月照琅玕。  

现代社会，我们要设立自身的传统审美。  
在一般的国家里，当它的国家内部构成比较简单的时候，它会设立一整套的纲领。大多数国家都是这样做的——它们自己设计一套新的传统风尚。我们隔壁的，无论是岛国、半岛还是岛国，都有这样的设计化操作。但是这是因为它们体量小，根本原因是它们的体量狭小，所以它们才能这样做。  

而中国体量庞大，在我们这里，只要能够做到一件事情，我们的华夏审美系统就能够回来——就是把我们的历史审美视角转换成五千年视角。只要解决这个问题，其他的东西全部都迎刃而解。因为实际上，现在对于我们的审美系统能够构成破坏的只有一个东西，就是满清审美。其他的，虽然说这些小蛮夷们特别喜欢裹挟各种各样的少数民族，但实际上除了满清刻意引入的这一套西伯利亚和东欧审美之外，其他所有的少数民族在过去的漫长数千年间，与我们之间已经达成了审美美学上的高度一致。在满清之前，我们周边的包括各个部落也好、部族也好，他们所使用的服装与审美都是向中华体系靠拢的——虽然有自己的某些地方性的纹饰特点，但整体的框架上是完全向中华系统靠拢的。只有满清倒行逆施。  

所以，目前我们真正需要解决的，就这一个时代的问题。而这一个时代的问题好解决吗？其实很好解决。只要解决我们的历史断代法就可以了——到底我们是从什么时候开始衰落的？只要弄清楚我们的历史断代法，从清朝开始衰落，后续的事情迎刃而解。它不需要说什么“等到什么时候安定”，我们已经安定了。之所以还有问题，是因为现阶段由某一些群体，裹挟着某一些力量，在试图反攻倒算，为满清的这个罪恶黑暗的时代洗白。解决掉这部分的历史冗余，后续我们顺理成章地就会把我们的传统走向现代化。
