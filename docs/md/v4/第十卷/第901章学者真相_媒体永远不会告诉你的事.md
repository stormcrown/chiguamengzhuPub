# 第901章 学者真相_媒体永远不会告诉你的事

99%人不知道的学者真相——媒体永远不会告诉你的事！  

《学林叹》：  
媒体择贤如选秀，网红学者满堂春。  
真儒埋首青灯下，荆玉沉沙谁问津？  
西学框架成桎梏，东风无力百花贫。  
愿借董狐千秋笔，重开百家争鸣新。  

学者并非都觉得西方很高级。我们的学者当中也有很多有本土意识的，绝对数量不见得特别多，但有自尊自爱精神的学者放到这个范围里就很多了。不是所有的学者都是你看到的样子。因为媒体会筛选符合媒体利益的学者，让他当网红来捧。网红学者和广大的真正学者是有区别的，甚至是有鸿沟的。我不是学者，我是一个带货主播。为什么带货主播可以当网红，而学者一旦成为网红，就意味着他的偏重点不可能放在学术上面？好的学者有很多，但媒体不会报道，不会把人捧出来。谢谢盘珠。当媒体不选择他出来当网红的时候，你就不知道这些优秀的学者，他们的贡献也就会显得不重要。  

还包括那些基础普通教材。教材筛选内容时，是按照某些学者所谓的共识进行的筛选。你要知道，教材也有筛选的学者，不是所有人都有同一个共识。学界通常是有分歧的，自然有不同的认知，要有讨论空间，这是必然存在的。但哪一个部分会被选进教材里面，哪一个人会被挑进媒体里面，这就有差异。所以不是所有学者都跟着西方的指挥棒走，只是说我们现在还没有拿回绝对话语权，做得还不够。  

而且很多很爱我们自己国家与民族的学者，实际上由于长时间的教育，他们也会受到西方中心框架的影响，就是所谓的“标准答案”。标准答案背得多了，学得久了，尽管他们已经在我们自身的叙事道路上努力前进，但还是会被原有的框架绊手绊脚。
