# 第926章 母系社会女性地位_老母亲靠儿子兄弟获权

母系社会女性地位高？真相竟是老母亲靠儿子和兄弟获权！  
啊，母系社会女性的地位有多高？不要在母系社会当中谈女性地位高不高，没有意义。因为母系社会里面地位高的是谁？老母亲啊。母系社会里面地位高的那个是老母亲，而不是女性。那么老母亲她是依据什么获得社会地位的呢？她是靠她的兄弟和儿子们的实力来获得社会地位的。所以母系社会当中拥有实力的人群，还是大体量的男性。但是呢，女性当中男性靠山特别多的人，会成为一个天然的权力中心。这就和后世的那种太后专权、太后干政是有关联的，这个东西是一脉相承的。也就是说，你要说一个地方他的女性强不强，你要看他家的妻子有没有话语权。但是呢，当你去看她是不是母亲有话语权的时候，这不影响女性的问题。就像满清时期，女性地位极端低下，皇后都没有任何后宫的管理权了，就是个装饰品，是个吉祥物了。喊“臣妾做不到”，那是真做不到啊。那这个时候依然会存在太后干政。为什么？那是娘，她是凭借母亲的身份啊。这一种呢，是一种从上至下的权力差，是来自于母亲的身份。虽然母亲是一个女性，但是她不是依靠女性身份获得的，是靠母亲的身份获得的啊。这个事儿你一定要弄清楚。也就是一个女性在母系社会当中没孩子、没兄弟，她没有办法获得这个权力。所以权力的力量来源到底是什么？一个人的社会地位取决于他的社会贡献加上贡献预期。而女性的社会贡献当中，天然地包含了一个东西，就是劳动力再生产，就是生育。女性的生育本身就是一种社会公共贡献。生育越多，生育的成果越显著，孩子的质量越好，那么这个母亲所拥有的权力就越大。
