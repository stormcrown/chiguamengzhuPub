# 第938章 血统纯净度排行榜_父系基因纯到离谱

血统纯净度排行榜：这个民族父系基因纯到离谱 #人类学 #科普  

中国男人的外来现象会不会产生血统污染？  
你觉得什么样子叫做血统污染？  
当你提出血统污染的名词的时候，  
就得先说清楚：  
你觉得什么样子算是血统污染？  
只要有外来的血统，  
就算血统污染吗？  
先得弄清楚这个事儿啊。  
这个说不清楚的时候，  
那所有的外来，  
它当然都会与原有的体系有所不同。  
但是呢，我们历史上面有没有外来啊？  
历史上面的外来是很多的啊。  
大家心里头有点儿底数，  
大家心里头有点儿底数——  
就是中国人这里父系基因非常单纯，  
但是母系基因较为复杂。  
这个是我们目前全世界啊，  
在分子生物学上面进行各种分析，  
得出来的结论都是相似的：  
我们的父系基因极为单纯，  
当世翘楚。  
但是呢，母系基因非常的复杂。  
很多地方呢，和我们刚好不同啊，  
甚至刚好相反——  
就是母系基因较为清晰，  
而父系基因来历复杂。  
这两种情况它代表了什么？  
父系基因单纯，  
其实意味着相同的父系群体在进行基因扩张啊，  
也就是说，  
他在历史上面的战争领域当中，  
取得了绝对优势的长期胜利。  
他取得了长期的胜利，  
于是呢，取代了旧有的其他类型的父系基因，  
成为广大土地上面父系基因的单一来源。  
长期的胜利会带来父系基因的单纯，  
而长期的失败则会带来父系基因的复杂啊。
