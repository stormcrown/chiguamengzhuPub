# 第997章 逆风辨忠魂_危鼎鉴

逆风辨忠魂 【危旌鉴】  
夫观沧海者，必验于逆涛；识忠良者，须证于困局。昔盛唐崩摧之际，安史横流，非无胡将守孤城；赵宋倾覆之时，崖山血浪，犹见孤臣抱幼主。锦上添花之辈，易饰丹心于顺境；雪中送炭之徒，难掩赤胆于绝渊。观今之世，每见鸾凤栖梧，皆言眷眷；及至风雨摧柯，顿作萧萧。霓裳羽衣之客，竞逐尧舜笙歌；板荡疮痍之秋，谁效屈子沉江？故曰：颂声盈耳，未若危旌独擎；蜜语萦梁，何如寒刃同戍？

如何判断自己是爱国，还是爱一个强大的国？能够举一些历史实例吗？如果一定要去做这样的判断的话，我们只能在倒霉的时候看。一定要下定论的时候，去看一个人的定论的时候，你得看他的逆风局，看他的困局。在困境之中还和我们站在一起的，那才是真自己。人到什么时候都是这个标准。包括那个盛唐戛然而止的时候，虽然造反的是胡人，但是汉魏盛唐里面也有大量的胡将，是不是捍卫盛唐的也有胡将？那这个时候你怎么区分？谁在最困难的时候与我们同在，谁就是真正的经过了考验的自己人，而不是在打顺风局的时候过来吃好喝好玩。我爱中国啊，我们今天的中国就很招揽这些人。大家在网上也能看到，生活中也能看到啊。

2. 现代白话部分删除了多余的语气词“啊”（如“啊，但是”改为“但是”），并修正了重复的“得得”为“得”。
