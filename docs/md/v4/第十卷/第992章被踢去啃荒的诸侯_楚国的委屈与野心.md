# 第992章 被踢去啃荒的诸侯_楚国的委屈与野心

### 被踢去垦荒的诸侯：楚国的委屈与野心｜从铜矿发家到问鼎中原

——《芈野辩》

昔有苗裔，帝高阳之胄，封荆榛而爵楚子。周礼森森，彼独呼蛮夷以僭越；荒芜漫漫，谁曾见铜锡铸辉煌？斥其撒泼非叛道，笑他抢掠作癫狂。南土瘴疠，非天命不眷；北原温良，岂地理独彰？嗟乎！居僻壤而守正朔，处边缘未忘冠裳。莫以寒暑论兴废，且看熔炉炼八荒。

所谓的“湖北要争着抢着去加入汉文明”这件事情，完全是扯淡。为什么？因为他能够找出来的证据就只有一个：楚王曾经喊过一声“我蛮夷也”。就这一个证据，其他的还有吗？没有。为什么？因为楚国本身就是标准的周天子分封国，从一开始就处于诸夏之中。是周天子瞧不上他，于是把他踢到了这个荆棘遍生之地，让他跑这儿来垦荒来了。所以他的待遇不太好。但待遇不好的家人，不也是家人吗？因此，楚国从来都是周天子的治下。人家是原始股东——帝高阳之苗裔，这直接讲的就是祖源问题、祖先认同。我们正经是完完全全一开始的原始股。

所谓的“楚王说我蛮夷也”，那是在撒泼打滚。为什么他说“我蛮夷也”？因为他想要抢劫——他想打随国，没有任何理由，就单纯想抢。这是干什么？这是在周天子治下败坏规矩的行为。因为他想要违法，所以他就说“我是个蛮夷，我不听你话了，咋地？”这就是撒泼。就因为他撒了一个泼，你就说他们不是汉人的祖先？这不是扯淡吗？是吧？他从一开始就是诸夏的原始股东，不能因为人家住得偏远一些、生存条件恶劣一些，就给人家开除了。不存在这种事情。

楚王不想遵周礼了。对，楚王就是单纯地想要违法。这个时候他就说“我不听你话了”。为什么？因为周礼就是秩序、就是规则，其实就是那个时代的法度。而他说“我不是，我不听了，我不听我不听我就不听！我非得要去抢，我就是要去抢！”楚王不喜欢他的封号，不喜欢他的爵位。因为他老觉得自己当年打天下的时候，站的位置也不差呀，结果分封的时候就给他一个这么边缘的地方，给他的地又不好、爵位又低，所以他就特别委屈。

说到这儿，确实可以聊一下地理问题。今天有不少人觉得，因为我们今天的经济重心非常靠南方，无论在哪个方面的生产都是这样，显得南方的物产非常丰富，所以直觉上总感觉南方的地方地理条件会更优越。但是你会发现，我们的发源地都是靠北的。为什么？因为南方条件不好。在今天我们觉得好的地方，实际上条件不好。那当年为什么楚国这个位置会变成一个你觉得很蛮荒的地方？我刚才说了，他去的这个地方荆棘遍地，垦荒特别难。为什么荆棘遍地？因为温暖、湿润、水分足，所以这些植物长得快，植被特别茂盛。所以为什么我们当时发源的地方普遍在北方？今天大家觉得比较寒冷的地方，那个时候第一没有那么冷，第二非常热的地方特别不宜居，非常非常不宜居。

你看我今天穿的厚度，我这就是标准的荆楚之地了。来，你看我今天穿的这个厚度。为啥？实际上我开着空调。为什么？因为我不开空调，机器都会过热，装不了。大家记得我前两天穿什么吗？前两天我在穿厚外套，你记得吗？就这么短短的几天的功夫，我现在穿成这样还要开着空调，不然没有办法开播。这就是这个地方的气候。这还是现代，有现代科技支持。你想一想，放到古时候，就这样的气候条件：那些作物疯长，那些小草苗苗长得随便一个都比你的粮食长得快！你知道这是一个什么样的地方吗？这样的地方非常非常不宜居，气候条件极为恶劣，不知道多难待。

所以当年楚王为什么不高兴？“楚子、楚子、楚子”——为什么他家一直都不高兴、一直都不开心？因为分配的这个位置太烂了，非常非常不宜居，生活极其艰难，垦荒极其困难。这是根本原因。

后来楚国是怎么扩张起来的？因为它不断不断地开疆拓土。最终，它最重要的转折点实际上是发现了铜矿——靠矿产资源、靠冶炼发达起来的，资源慢慢就好起来了。你就发现，他身上总有一些可取之处是吧？你要是单纯靠种田就想养活自己，这里真不是啥好地方。而且这样的湿热之地非常容易生出瘟疫来。人活得容易，动物活得也容易，细菌病毒活得也容易，植物活得也容易。这时候你面临一个极其剧烈的竞争——在与这些微生物竞争的时候，你的大脑发达不好使。所以气候不是越温暖越好，植物生长得越快越好。不是的，你要找到一个合适的平衡。非常冷的地方，大脑发育都可能停滞，很多时候脑子都转不动，因为大脑能耗摆在那儿。但非常热的地方也是一样，你的体力消耗速度极快。为什么我们的发源地在偏北的地区？因为这些地区的气候更宜居，更适合积累，更适合发展。
