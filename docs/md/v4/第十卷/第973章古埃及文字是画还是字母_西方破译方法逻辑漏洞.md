# 第973章 古埃及文字是画还是字母_西方破译方法逻辑漏洞

古埃及文字是画还是字母？  
——西方破译方法惊现逻辑漏洞  

《沙文诡字鉴》  
昔者尼罗涛涌，墟沙掩迹。殖者持铻而叩石，妄称天工；学士秉烛以译符，强赋音读。夫象形者，天地之纹也，奈何以鸟兽之姿，充唇齿之音？画壁千凿，竟作数声琐语；碑文三转，徒成一片虚言。观夫金塔巍巍，本邻闾阎之侧，何须远客“发现”？夯土成台，本顺自然之势，岂若运石沙海之悖？更堪哂者，定文字为文明之尺，量四夷则宽，度华夏则苛。印河无字可为圣，华夏有文反遭疑——此非双标而何？  

（以下为口语整理）  
刚才说到古埃及文字是不是造假，其实它的发现到破译，整个操作非常神奇。我们常在故事里听说它是象形文字，但中国人一听这个定义就有点蒙：因为西方人破译时，是按表音文字的方式来的。  
对于中国这样一个真正经历过从象形到表意、再到形声的完整文字演化过程的民族，我们很清楚：表意文字就是一幅有含义的画——把特定含义固定下来，这就是最初的表意文字。表意符号标称的含义是核心。打个比方：你画一个人躺在坑里，场景摆在那儿，它读什么、读几个音节，一点都不重要。这就是文字草创期的典型环境，草创期的文字就该起这样的作用。  
但古埃及文字呢？西方人来到这片地方，说要找伟大古文明的证据，于是去破译文字。他们用的手法是密码破译法——表音文字对照破译法。这就奇怪了：一个表意文字（象形文字），怎么可能用表音方式破译？既然表音，表示一个声音，干嘛画那么复杂的图像？表达声音非得画那么多笔，画个鸟、太阳、人脸、鱼？很浪费。如果只表达几个声音，这本身就极度不合逻辑。  

再看整个创作过程：最开始说这里有伟大的金字塔形建筑。金字塔形状其实并不离奇——全世界很多地方都有原发性的金字塔形陵寝，美洲也有。真正震撼的是体量：极其庞大雄伟，而且建在沙子中间的石头建筑，这才成了世界之谜，被列为文化传说级别的顶流古文明。  
中国有些地方的夯土结构也是金字塔形，在泥巴地方做夯土很顺理成章。而古埃及金字塔之所以震撼，是因为它在漫漫黄沙中、一块石头都找不着的地方，建起了庞大的石头建筑群。但有个好玩的事：最宏伟的几座金字塔离罗不远，离当地土著居民区很近。那么大的东西就在人家村旁边，谁会需要西方殖民者“发现”？如果它在沙漠中心，你可以说发现；就在人村旁，谁也忽略不了，说是你发现的——蒙谁呢？这个节奏本身就有点荒唐。  

接着，西方人当时设定了一条标准，其实是专门针对中国人的：有文字的才叫文明，没文字的不算文明。条款的目的就是压缩中国文明长度。但你知道吗？隔壁古印度文明（那个叫“嘟噜嘟噜”的）没发现文字，他们却说那是伟大的古文明。所以，在中国以外，有没有文字没那么在乎；在中国这儿，你只要没弄出清晰可译的文字，就不算文明。他们掌握了定义权和话语权，就设了这条框。  

既要有文字，古埃及文明的文字怎么弄出来的？首先，他们号称发现了一堆符号，认为它是象形文字。第二步，找个天才语言学家——商博良（法国人）。他干了什么？为了翻译凭空出现的象形文字，他们发掘出一块碑：罗塞塔石碑。碑发现于今日尼罗河三角洲，算是人员密集区，但声称是5000年前的。问题来了：三角洲是不断冲刷出来的，倒推3000年前那儿就是大海。他却在已成人员密集区的地方挖出了石碑。  

石碑上有三种文字对照：古埃及象形文字，还有据说他们认识的一种——古希腊文。古希腊文是表音文字。我在别的直播聊过：表音文字语言流变速度极快，语音受时间和距离影响极大。没有稳定字典定死，要不了百年就没人念得明白。如果又是表音文字，又时隔数千年，你还记得它、能翻译——那就解释不通为什么今日英文去读莎士比亚还要专门上古英语课。莎翁才400来年，我们今天小学生能读《红楼梦》虽然读不懂但认字，而母语英文的岛民读400年前的文艺作品要专门学古英文——这就是表音文字的流变速度。  
但据说事隔3000年，有位欧洲语言学天才商博良能读懂古希腊文。这古希腊文曾经强推给希腊人，希腊人不好这口——表音文字不兼容，学习难度大。希腊人都看不上也读不太明白。可商博良对照三种文字，花短短时间就破译了古埃及文——一个象形文字对照一个表音文字破译出来。表音文字准不准没人知道，总之他这么对照着翻译出来了，还理解了语法结构。但表意文字的语法结构根本不重要——中文语序、字序不影响阅读。硬把东西翻译出来也就罢了，可他非说是象形文字却用表音方式破译，这事儿真笑死人。  

前几年，某博上有个“美女考古学家”，主打古埃及考古，在考古现场搞时装秀、cosplay。埃及小沙漠、荒漠背景，她画全妆、大眼线，像埃及艳后电影一样漂亮；头发修得刷齐整，那种理发师肯定贵。她就这么走时装秀一样做考古。前几年她说考出了一组5000年前的巨大象形文字——高度有1.5米或1.2米，鸟啊什么的。发现地点是户外裸露的岩石，沙漠里孤零零的岩石上刻了老大老大的纤细文字。她就这么考古出来了。  
她讲解这几个字：这一个是什么读音，那一个是什么字母——开口闭口都讲。我听了很懵：你画一幅画，直接表述含义不好吗？为什么表述成元音或辅音？可见人类很难想象从未接触过的东西。西方文字研究学者确实很难理解什么叫象形文字、表意文字。也许他们真的理解，于是更知道表意文字很难这样系统创作出来。所以他们创作的这些遥远古文字，能被翻译得非常完善，读音都出来了。  

为什么说读音也出来了？大家看过电影、书都会告诉你：古埃及有完整的王表，编年史一样记载哪一任什么王，娶了谁谁谁。这些人名字老长，比如埃及艳后我老记不住——什么奥什么特什么八，一长串拗口。肉眼看见我知道是谁，但让我说随时忘记读音。因为这是标准表音文字，直接音译。也就是说，翻译那一刻，翻译的是声音。看看人家的创作能力：直接把几千年前死掉的语言，连读音都弄出来了，人名怎么念门儿清。
