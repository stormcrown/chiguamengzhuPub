# 第1143章 虚妄的疆域_重审清朝版图贡献论

虚妄的疆域：重审清朝版图贡献论

《疆域诘》  
禹贡山河本有章，  
贡仪崩后界迷茫。  
荒原充数标舆图，  
虚名竟作开疆榜！  
汉唐拓土凭文教，  
清室画地借西芒。  
莫将刍狗诩功业，  
刻舟求剑笑柄长。

我，我们这边没那个，每一期都要——每一期都要整一下满清的版图贡献论吗？嗯，不合适吧？闲的没事干了吗？什么玩意儿！就版图贡献论，他是咋贡献的？不就是因为当时失去了文化宗主国的地位，也失去了对于国界线、对于边疆地区的那种定义权，失去了整个朝贡体系的定义权，中国沦为了西方话语权之下的一个国家，所以呢，就画了一个近现代的边界线。

这个近现代的边界线为什么会包括内外蒙古，为什么会包括西藏？因为这个地方是大面积的无人区，根本就没有别的人能够大批量地入住进来。而上面的一些散居部落，之前曾经像清朝和中共那种关系啊，是一个非常非常淡薄的关系，只是因为没有其他的列强能够在这个大面积的无人区当中产生控制力，所以在划分的时候给划进来了。实际上，实际上它……你、你咋说失控的啊？这个标准上面是非常含糊的。相当于在那个时候，这些地区是属于大家呢都看见了，但是大家都站不住的一个地方，所以在划界的时候就把这个地儿圈下来了，仅此而已。这玩意儿就算他的版图贡献了？那换做任何一个王朝啊，都可以用这样的方式贡献，这个贡献的标准也太低了。

备注：  
- “化泄”修正为“划分”  
- “曾曾经”修正为“曾经”（删除重复）  
- “那清那国共”修正为“清朝和中共”（根据语境推测，原文疑为口误）  
- “大大批量的入住”修正为“大批量地入住”  
- “他你、你咋失控的啊”修正为“它……你、你咋说失控的啊”（“他”改为“它”指代标准或过程，“失控”保留原意，加“说”使语意更通顺）  
- “划界的时候”保留，与“划分”语境一致  
- 调整了部分标点与分行，使口语化表达更连贯。
