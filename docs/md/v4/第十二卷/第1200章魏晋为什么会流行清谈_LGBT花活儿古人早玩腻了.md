# 第1200章 魏晋为什么会流行清谈_LGBT花活儿古人早玩腻了

魏晋为什么会流行清谈？LGBT什么的花活儿，古人们早就玩腻了。

魏晋为什么会流行清谈？魏晋时期整个的社会风尚是很坏的，主要是晋朝，魏朝还好，但是晋朝很糟糕。晋朝最大的问题就是它窃国，所谓的“司马昭之心”，在当时相当于是打碎了所有大家的共同道德认知。所以呢，它自己内部属于是德行不正，就出现了一种情况，叫做“上梁不正下梁歪”。因为君主德行不正，是偷的、抢的、是贼子，而且天下都知道他就是贼子，那你用什么方式管？那你就得是放纵下面的人。因为你自己都是一个缺德的玩意，所以你下面的臣子，你要臣子忠诚，你要臣子忠君守义啊，大家都知道你是不义之君的时候，臣子是不可能忠诚的。于是“上梁不正下梁歪”，造成的结果就是朝臣也是一团糟，大家都很乱。只要整个系统都是乱得一塌糊涂的时候，自然而然的就会衍生出一种情况：你进去当官，去搞经世济民，不但难度大，而且没有人配合，而且大家都觉得你像傻子，这事压根就干不了。正经事儿你根本就不能做的时候，那请问该做什么呀？聊天、吹水、闲谈。正事儿没法干了，整个的西晋，由于它立国上面的一些问题，直接造成它整个的这个时间里头正事干不了，从上到下没像样的人混不下去，也待不住。所以这个时候自然而然的就出现了那种大家集体嘴炮、瞎说八道、爱咋地、随便浪荡，整个的社会风气都坏了。那个时候就属于群魔乱舞。今天大家看的那个什么LGBT这些鬼东西，那时候都有，都玩腻了，你知道吗？那些什么嗑药的玩意到处都有，随便那个什么跑到山间林间就开始开裸体派对，到处都有，那简直了。美国玩的所有的那些浪荡的、看不得的、不像人间的那种花活儿，西晋都玩过，想怎么恶心都行。所以那个时候你不清谈，你干啥？你真的到了那个时候，但凡还有点文化，你都得去清谈。咋说好的魏晋风度呢？这就是魏晋风度。非把魏晋绑到一起，我也是绝了。美国再怎么样，它也比晋强一点，还是强几毛钱。

哎，走一波。巧克力很好吃啊，吃过的小伙伴，那公屏上回馈一下，是不是老好吃了？这个巧克力非常干净，配料表很干净，里面全都是天然可可啊，是没有代可可脂的。天然可可口味丝滑，加工的水平也是一级的。简装包装非常的简约，因为它是一个军品，属于粮食口粮级的，所以呢，外面是锡纸包装。我为啥不给你们演示？因为吃没了啊，我这经常空口带货的原因就是吃没了，一般是这样的原因。这个巧克力很好吃，所以样品包装都没有了。外面是一层厚厚的锡纸真空装，里头的油脂装在里头呢，是白色的纸盒子，在里头呢，金色的糖纸，就这种结构里面一小块一小块的巧克力。每一盒呢，就是500克两斤的量，很实在，很适合家里头把它就拆出来，扔进糖盒子里面，一天偶尔叼一块啊，不推荐大家多吃了，不好。理直气壮地吃完了，对我我就这样啊，不然好吃的东西我攒着，它又不能下咽，巧克力就得吃。这个巧克力呢，它分了两个型号，大家选择的时候注意一下。一零型呢，是35%的可可；二零型呢，65%的可可。一零型的偏甜一点点啊，看你个人的口味，喜欢甜、喜欢奶味的，选一零型；喜欢可口味的、喜欢不太甜的，选二零型，就是这种筛选方式。吃过的回馈一下，不吃啊。你真经世济民了，司马家又问你想干啥？上梁不正下梁歪，没有办法啊。它整个的那个国家，它在立身的时候就是乱的，从一开始就是干脏活的。那上头就是在干脏活，你让下面的人怎么样？你出一个像高家出现了那个什么兰陵王，不觉得格格不入吗？整个高家都是疯的，不想说，它疯得让人犯恶心，但是呢，出现了一个正常人，你咋说？这就属于你在那个时候，正常人没有办法呆，体面人、有道德的人、有秩序的人在那个时候很难过。孝文帝海河改制，我老觉得好像你的历史观很多时候跟我不一样。我觉得孝文帝的改制属于一个，如果说对他自己来说，并不是一个非常聪明的决策，但是对我们来说是一个非常好的决策。
