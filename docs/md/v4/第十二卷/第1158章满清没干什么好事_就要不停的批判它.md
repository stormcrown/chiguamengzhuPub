# 第1158章 满清没干什么好事_就要不停的批判它

满清没干什么好事，就要不停地批判它。什么叫做“东三省对于武汉的贡献”？我们作为一个大一统国家，每一个省份都会用自己的资源为整个国家做贡献。你想要干嘛？你觉得我IP在武汉，然后觉得自己IP在东北，你就对我做了什么贡献吗？还是想要我感恩你家大清祖上？我跟你说，你这个东西的味儿一出来都能闻着，味儿太冲，你知道吗？你但凡多蹲个几天呢，你都知道我就是生在东北、长在东北的。那又怎样？我很清楚东北的历史课外小读本讲的是什么玩意儿，我就是念那个东西出来的。所以我很清楚你心里在想些什么——不就是屁股坐歪了之后，记不得自己祖上是哪位了吗？

我当年在义务教育阶段的时候，你知道那个就是我们教历史课啊。课外的话，会有一个专门的地区历史读本。每一个地方都会有那个。我现在在武汉，武汉这边的话也会有。就是义务教育配套的一些有关于这个地区的相应历史的东西。武汉这边的很多学校呢，学的就是楚国的相关信息啊，包括《楚国八百年》这样的纪录片，在很多学校里面都会专门拿来学习，并且借由这个给学生们普及：这片土地上曾经发生过什么样的故事？就是这样的补充历史读本。

在东北的时候，他学的是什么？学的是站在清的历史上所讲述的萨尔浒之战。站在清的历史上去讲萨尔浒之战，你知道那是个什么味儿吗？然后呢，整到今天说啊：“你要讲清楚东三省怎么地？”东三省怎么了？东三省自古以来就是中国的一部分。从哪个股开始？哎，不要扯你大清，跟你大清半毛钱关系都没有。你大清就是跑这来借住的一群人。东三省这块地，是从先秦时期就隶属于中国管辖的。那个时候的“中国”指的是周天子治下的王畿直辖区，就叫做“中国”。其他的地方呢，就属于诸侯。而诸侯当中的燕国统治了东北。我们在汉代的时候，在东北实行郡县制管理。你讲“自古以来”，自古以来，都变成你的了？为啥？就因为流浪的时候过来借住了一下，借住的时候被安排了一个工作，安排工作之后还造了反，于是这片地方就变成了你的祖地了？咋那么会呢？写你名了吗？

东三省自古以来就是中国的一部分，跟你家大清一点关系都没有。明代的时候他就是中国的一部分。我们在上面驻军屯垦、设立卫所，羁縻管理。当时的东北诸部手上都拿着明王朝发放的印绶、印章。这玩意儿是努尔哈赤销毁的，因为他把所有的部落都揍了一遍，然后抢掠了印章，把这些印章统一销毁了。你就靠这个，然后就印证“没有没有没有，这个就没有东三省”是吗？不要耍这些流氓啊，没有用。通上来？哎呀，就是啊，什么东三省怎么怎么地。东三省从来都是中国的一部分，别搁我扯什么地域历史论啊。人是活的，腿长在腿上，会走的，人是不断迁徙的。真是会扯淡！

我们不需要讨好大清才能够拥有东北，因为本来就是我们的土地啊。清王朝没干过什么好事，不需要夸他。他安排的衣服丑，那就是丑；他安排的器皿难看，就是难看；他打击了大量的文人，那就是破坏文化；他销毁了大量的书籍，那就是造成文化倒退；他株连工匠家庭，那就是造成技术倒退。怎么着，批评不得？他是什么了不起的玩意儿，做了错事不让人说啊？说了清朝一点不好，就不团结了？你是清啊？大清王朝在的时候，你是哪位？姓什么？哪家子？什么旗上旗下？主子还是奴婢？分得清楚吗？闲的。

嗯，好，欢迎来到直播间。阿呆，你闭嘴，需要我再给你踢出去吗？我最烦的就是你这首。你以为我不知道你在说什么吗？哎，只要是说：“唉，这哪个王朝不好”，立刻就是“所有王朝都不好”。不是，还真就不是。有一些王朝不好，有些王朝是好的。为什么好？因为待在那个时代，他就需要这样一个角色。他扮演好了自己的角色，做出了自己卓有成效的贡献，带来了进步，它就是好的。我真的是，我怎么可能吃饱了撑的，要求几百年前、一千年前的王朝拥有我们今天的制度？谁疯了吗才会这样要求？不可能啊。他那个时候就应该是那种君主集权制，那就是当时最先进的制度。他就按照那种方式去管理，就是最好的管理方式。但是清代在那个时间，他所做的事情全部都是拖后腿的，全部都是向后转的，全部都是腐朽的、没落的。就这。
