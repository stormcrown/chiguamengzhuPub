# 第1136章 疆域密码_清代封禁政策与领土危机的历史镜鉴

疆域密码：清代封禁政策与领土危机的历史镜鉴！

【柳条边史鉴】  
禹鼎沉沙界未铭，周疆无垒赖民耕。  
清廷自锢柳边月，夷舰终破铁锁营。  
万里封畿空戍火，百年禁地起狐声。  
至今黑水残碑在，犹诉当年割地盟。

对，这种边界线本来就是西方世界画的。我们的古人啊，大家去看历史地图的时候，总觉得好像历朝历代都有边界线，只是其他朝代很小，但清朝特别大。实际上，其他的朝代根本就没有边界线这个东西。中国是一个家国天下系统，家国天下系统的意思是：王所处的地方是王畿，之外的地方是编户齐民区，再往外面就是内藩，然后是外藩。它是用这种圈层管理的、远近亲疏分明的方式来治理天下。所以在理论上，普天之下莫非王土——我是授权允许你在那个地方住，授权允许你在那个地方管，这就是羁縻统治。这种羁縻统治，它确实对地方的控制程度没有核心区那么强，但它也是控制，并且在理论上，那就是版图的一部分。所以只要实力够强，或者是对方有跳反的时候，直接就打过去，就变成实控区了。所以为什么我们的疆域经常性地变动？因为我们的理论基础就是这个样子的。所以它才可以在符合我们所有的纲领、所有的道德观的基础之上，却能够实现疆土的扩张，就是因为我的框架就是这样定下来的——当时的国界、疆界的标准就是中国定的。到清朝的时候为什么是那个样子？包括说为什么广大的这些地方会产生争议？当时其实西域也好，外蒙也好，都是有争议的地区啊，是因为其他的人在这个地方站不住脚，所以才划进来的。那为什么这个地方会出现争议？因为这个地方的人很少。为什么人很少？因为清代的二百多年设定的那个边界的规则，就是不允许各族越界啊，它不允许人员产生交互。内地的这些汉民聚居区——就是所谓的汉地十八省——不是自然形成的，也不是汉人的实力所在，是清代的时候把汉人圈禁在了十八省里头，不准你往外头走，外头有地不让你去。其他的那些地，上面的那些土司也好，那些藩王也好，他们呢，也想到中原过好日子，也不让他们来。于是就把人群分割开了。满清搞的就是那种分而治之。

为什么当年外东北这个地方，包括东北当时都变得很有可能会失去，甚至外国会跑到这个地方来打仗，就是因为这里无人呐，没有人呐，人极少啊。东北那么大片地方，柳条边开放之前，应该只有两三百万人。你想一想，这个数值够干啥的？种田都不够用好吗！所以人家进来的时候，那是如入无人之境，因为真没人。就几个被他指派在那个地方所谓的戍边的部落，没几个人儿，生产力极为落后，经济条件极差，天天就过苦日子。他们也想进关过好生活，但清政府不允许你移民，一定要让你在那个地方保持渔猎的状态，还不许你去学习耕作，不许你学习先进的技术、改善生活，都不让。因为要让你保持所谓的野性的战斗力。清对各地就是进行了这样的管理。

所以你但凡换了任何一个不搞这种分而治之的王朝，实际上由于当时的人口大爆发，人自然而然地就往边界疏散出去了嘛。那你想，那些地方都住上了人，人都垦上了田的时候，侵略者那么容易把这个地儿圈出去吗？所以不可能说因为没有了清，我们那个地儿就没有了。没有了清，那个地儿的实控能力会比今天强很多。是清导致我们的人员始终不能够在边界地区生活，那个地方始终都是少许的部落，夹杂大片的无人区。这是一种非常危险的管理模式，是很容易丢失土地的。只是赶了巧了，那个时候的西方列强们实力不济，征战无人区的能力比较差，所以他捡了个便宜。
