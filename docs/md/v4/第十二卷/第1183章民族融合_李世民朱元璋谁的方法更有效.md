# 第1183章 民族融合_李世民朱元璋谁的方法更有效

民族融合：李世民与朱元璋的方法，谁更有效？#冰酒

李世民让北境的内附（原为“内富”，修正）成为可能。  
除了内附，还有其他办法吗？  
其实，我们历史上采取过很多策略，这个问题并不复杂。  
事实上，我们当时与周边被称为“胡”、“蛮”、“夷”的人群，本是同种的。  
在同种的情况下，按照中原王朝的传统思路，通过王道教化就能解决问题。  
所以历代帝王都曾尝试过。  
汉代也曾招揽胡人内附，把他们用作先锋、前哨、骑兵、雇佣军等，也有用于工农的，让他们参与社会生活并给予保障。  
当时有这样的内附（原为“内服”，修正），但很快五胡乱华就给了教训——整合不好就会出现混乱。  

李世民时期，大批胡人（原为“胡宗”，修正）内附，与后来安史之乱的爆发有直接关系，也与宋朝面临的北方恶劣局面有关。  
所以，内附能否成功，关键在于消化能力——把人内附来了，得能消化。  
消化不了，他们始终记得自己是谁，强盛时还好，衰弱时就会出问题。  
“大唐最伟大，弱的时候呢，我就是大唐”——基本上都是这种模式。  

那么最后该怎么办？我认为朱元璋的尝试是另一种方案。  
很多人对朱元璋的功绩没有客观认知，他有很多功绩被埋没，其中就包括对内附胡人的管理方式，这很重要。  
当年蒙元离开时，带走了士绅、豪强、地主和部分文人等群体；蒙古贵族的上层包括一些宋人也跟着走了。  
但蒙古下层牧民中有许多人坚持留在大明境内，因为他们知道回草原后会重过过去的苦日子。  
“人往高处走”，他们想留下来过好日子。  

蒙元被赶走后，中国境内各族杂居，有大量蒙古中下层（中层可能大部分跑了）和色目人下层。  
这些人在原系统中过得不好，知道在汉人系统中会更好过，因为这边更尊重人。  
但留下他们很简单——立国时很强，他们自然认主；可衰弱时仍有隐患。  
朱元璋立下规矩：永“安于中华”，必须融入。  
融入的标准是：所有蒙古与色目人，不准族内通婚；只要结婚生子，必须找汉人。  
第一波就是强制通婚，建立紧密的血缘关系。要么单身，要么与汉人结婚，严禁族内婚。  
由于少数民族体量极小，问题不大。  

这种策略与汉唐时期的整族内迁、内附完全不同——拆散后统一通婚，大家都进入各自家庭，自然有共同生活方式，后代有共同血脉，按中国人的方式工作、休息、学习，语言、文字也会统一。  
没有单独的部落，其他语言文字就没法用，所以问题很快被消解。  
明代这个问题不凸显，明朝灭亡不是内附人群造成的。  
如果朱元璋没这样做，北元（原为“北原”，修正）逃窜后留下的蒙古和色目人一旦各自集群，也会形成内附胡族的势力。  
但他消解了这种可能，真正造成问题的是一些关外群体，他们当时实行羁縻（原为“机密”，修正）管理，保有部落，情况不同。  

至于“团结警告”——都已经成为一家人，结婚生子、同住一屋，这还不叫团结？  
难道必须各过各的才叫团结？这种理解很奇怪。  
某些人喊团结，偏偏要独树一帜、拥有额外权利、完全不同，才叫团结，这太离谱了。  
朱元璋的方式才是真团结——大家彻底成为一家人。  

（接下来推荐产品）  
给大家推荐一款新产品：野生蓝莓冰酒。  
冰酒是在果实产区，利用自然低温逐渐降温，让果实因低温脱水并经历霜冻，水分减少但未全干，形成低温浓缩果汁，再用它酿造。由于含糖量高，酒体偏甜，有浓郁果香和甜蜜口感。  
这款冰酒是东北长白山野生蓝莓酿造的，保留了蓝莓的营养成分。  
尝过蓝莓的朋友应该知道，它的口味和葡萄酒完全不同。  
那天试酒时，我家那位说：“这啥味儿？不是葡萄味儿啊。”我说：“对了！蓝莓酒喝出葡萄味就坏了。”  
冰酒偏甜口，口感柔和，适合中国人。
