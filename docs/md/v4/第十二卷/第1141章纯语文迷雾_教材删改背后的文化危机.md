# 第1141章 纯语文迷雾_教材删改背后的文化危机

《纯语文迷雾：教材删改背后的文化危机》——关于乾隆诗作入选小学语文的思考

我念书那会儿，课本里还没见过这么“接地气”的东西。可后来，有个人（记不清具体名字了）要在国内推广“纯语文”——就是他把《最可爱的人》踢出教材的那位。他主张的“纯语文”，只让学生认字、学修辞、练造句、写作文，至于文章里蕴含什么样的内核和精神，统统去掉，因为那些不属于“纯语文”。

当初他把《最可爱的人》踢出去，理由是“那都是过去的东西，精神跟今天没关系”。为了搞“纯语文”，他把当时某流行歌曲（好像是周杰伦的某首歌歌词）也塞进了教材——虽然没什么意义，只是寻常话语，但“这是纯语文，只有语文，没别的东西”。当时确实有这么一号人，搬弄过这档子事。现在不好点名，不过这人还在折腾，就是那位王某人。当年他推行了很多糟糕恶劣的教材编撰，甚至教学管理上的做法，都按这种逻辑编造理由，把精神上的、道义上的、文化上的东西踢掉，只剩下表面文章。

这件事当年引起舆论哗然，老百姓当然不同意。后来他被调职了。但要知道，教材不是一个人编的，而是一个小组编的。要改教材，第一件事就是设立小组，并让小组与他形成共识——大家都得认同“纯语文”，才能搭班子。所以就算把王某开掉、调职、甚至免职，也解决不了根本问题。因为他在改掉教材、踢掉优秀内容、塞进垃圾的那一刻，就已经布置好了一整套编研小组。这些人，你能每一个都撤掉吗？每一个人都犯错了吗？没有——他们只是听领导指挥，编撰能力没问题。所以即使这个人被踢掉，那种编撰原则、方式和逻辑，依然会应用在教材中。

为什么会出现这些“low爆了”的诗？如果语文教的就是纯语文，只需要认字、组词、造句、写作文，那乾隆的诗能不能达到这个标准？还是可以的——虽然没文化、没文采、档次低、low爆了，但它有个闪光点：跟皇帝有关系啊。于是就可以编点小故事，让课堂寓教于乐。不管它多烂，当以“纯语文”为标准时，你可以不考虑它多烂，只要它有字就行。

《最可爱的人》倒是又回到教材里了（最近回来了一阵子），回来是应该应分的。

**备注：**  
- “从某个叫叫王什么来着”改为“从某个叫王什么来着”（删除重复的“叫”）。  
- “你能够每一个人都看掉吗”中的“看掉”修正为“撤掉”（根据上下文，意为撤职调离）。  
- “布排好了一整套的啊，编研的小组”改为“布置好了一整套编研小组”（删除语气词“的啊”）。  
- “应应分的啊”改为“应该应分的啊”（保持原意，但修正为更规范的口语表达）。  
- 其余口语化表达（如“嗯”“唉”“啊”等）保留未删，以维持原叙述风格。
