# 第1157章 清朝的钱去哪了_民国穷到全员当买办

清朝的钱去哪了？民国：穷到全员当买办！真相太扎心！

王朝更替本质上是经济清算：中国历史上的改朝换代，核心在于打破财富垄断，通过暴力革命重新分配社会资源，解决贫富分化的僵局。然而，清朝到民国的过渡存在清算缺陷——由于清帝和平逊位，未能彻底查抄府库、追索海外资产，导致巨额财富（尤其是全球转移资产）未被收回。这正是民国贫困的根源：未清算的清朝财富遗存海外，使民国政府极度贫穷，迫使精英阶层沦为买办，无力自主发展。

清朝财富积累的真相是什么？八旗子弟确实有些实力——客观上讲，他们是有实力的，尤其是经济实力比较好。至于为什么经济实力比较好，我昨天直播时聊到过：因为他们并没有经过清算。正常来讲，过去的2000年中，我们经历的改朝换代很多。所谓王朝周期律，其中有相当一部分原因是由贫富分化造成的：经济权力过于集中，使贫民无立锥之地，剥削过度，社会分配陷入僵局。改朝换代就是一切推倒重来——这是一次大清算，把所有的账全平了，重新分配，再立新规矩。而清朝迭代到民国的过程中就有这个缺陷：因为是和平逊位，所以没有经过彻底的推倒重来——府库没有查抄，相关人士的账户没有追索。而且在这个过程中，一部分资产很可能并不在中国国内——当时的民国政府没有能力进行全球追索。要知道晚清时期，资产已经可以开始全球安置了，西方已在不少地方设立了各种银行、票号。所以当时发生了多少资产转移？不清楚，因为没有人查账。这件事已经过去了，但民国有多穷我们是知道的——民国确实很穷，不然这些老爷们怎么会甘当买办呢？整个民国的这帮人中，没有几个肯踏踏实实一步一个脚印做事的，很多都逐渐倾向于买办。除了他们自己骨头软之外，还有一个方面就是民国实实在在的穷。那为什么会这么穷呢？于是有人说：“我们一直都穷，穷了几千年。”开什么玩笑！古时候那些皇帝们开疆拓土，是为了扶贫去的吗？你相信他们有那么有良心吗？我反正是信不过的。他们打出去不是为了博好处，而是为了撒钱？做什么春秋大梦啊！所以我们过去的几千年其实一直在积攒：从汉代的丝绸之路，到宋代开始的海上丝绸之路，再到明代的全球大航行，都是在做买卖，都是在积累财产。当时全球的白银都流向中国。清代时，虽然这个朝廷很拉胯，甚至可以说很垃圾，但它在敛财方面是很能干的——搜刮盘剥、掘地三尺，农民面黄肌瘦。钱去哪了？不都是它收缴去的吗？它的税收比明代高出几倍——具体好像有人算过是三倍还是几倍？这是税收。它的物价也比明代高出几倍——不同物件有不同的物价。就凭这种方式一波又一波地盘剥。而且它与海外做生意全部都是顺差，哪怕是鸦片战争后，鸦片贸易看上去开始逆差了，但短暂的逆差很快被清王朝自产鸦片弥补，然后又开始顺差收割全球。请问这些财产去哪了？这一波一波攒下来的东西哪去了？为啥民国那么穷？既然这波财富没有被清算，就意味着它在全球任何一个角落都有可能有所遗存。想想在现在的市场经济条件下，它手里票子一摞一摞的，海量的——你猜它有没有力量？嗯，主要原因有人爱大清。为啥有人爱大清啊？桃子为啥有人爱？对不对？怎么爱的？闲得没事干去爱那个遍地饥荒的年代吗？不会呀。你咋爱上它的？还不是因为它投了巨资拍电影、拍电视剧、出书，用大明星帮你演戏，每天漂漂亮亮的，爱民如子，无比亲切，万世盛名——于是你才爱上的。不然呢，我们平白无故就会去爱那些受苦受难的时代吗？怎么可能！
