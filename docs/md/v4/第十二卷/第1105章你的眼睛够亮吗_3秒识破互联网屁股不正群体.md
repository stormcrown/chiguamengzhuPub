# 第1105章 你的眼睛够亮吗_3秒识破互联网屁股不正群体

你的眼睛够亮吗？3秒识破互联网屁股不正的群体。

——————————
群虻集矢汉旌残，谁解孤峰立雪寒？
指鹿声喧槐国议，三人市虎豹章冠。
茧房织就霓裳幻，狼燧燃成星火澜。
幼儿园前豕鹿辨，标题阵里剑霜酸。
楚狂歌裂千重幕，秦镜光寒万点丹。
莫道浮云能蔽日，星河不语自凭栏。

互联网上，在中国啊，在中国的互联网上面，汉民族主义者遭到各种小号的围殴，是一个常态现象。这并不意味着这个群体的弱势。我们得看到一件事情啊，就是像这一次的阿美这边的选举事件，我们能够看到的是，在尘埃落定之前，大家在互联网上听到的消息当中，哈哈哈，其实是占据了主流的。大家会觉得阿美那边的人糊里糊涂的都会相信，哈哈哈才是好的。虽然我们呢，有点像看笑话，但是最后得出来的结果呢，啊，这证明了一件事情：就是在互联网上啊，声量最大的那个群体，不见得力量最大，也不见得人数最多。

大家经常会发现，你一旦讲汉民族的东西，你一定会在互联网上招惹到无数的虫子，蜂拥而至啊，而且是秒到。他们的那个速度呢，就仿佛是有人通风报信，有人在群里面散播英雄帖，呵呵，然后接着订单的就冲出来了，开始围剿。你就会觉得，啊，原来难道只有我一个人是这样的吗？难道其他的人全部都是团结人，或者是这些啊，那些什么啊，中华主义者之类的这些人吗？其实不是的。有的时候极个别的群体，由于他在客观上的劣势，他为了争取自己的影响力，反而会去特意的做数据，制造它的声量。

例如说现在世界上特别有名的那些什么无孩爱猫女、阿美，现在天天把这个概念玩得溜，对吧？那无孩爱猫女这个群体数量真的很多吗？他们之前在互联网上的影响力是覆盖级的，谁都不能反对他，谁说他一句话啊，那你就是反人类了，就好像它是人类本体一样的。但是你看最后的结果，它们的数量很少。真正拥有力量的是那些不怎么说话的人，但是他们的眼睛是雪亮的，他看得见，他知道谁是谁非。所以不要觉得互联网上谁的嗓门儿大，谁就厉害，谁就人多，不是的。有的时候正好是有些人特别的错误，因为他知道他自己是歪理，知道自己的屁股不正，知道自己不是好人，知道自己的势单力薄，所以他会特意的做数据，显得他声势浩大，哪儿都是他无处不在。这个就是狼群战术。不是什么东西，看上去人人都这样说，于是呢，他就是对的，于是他就是多数人的想法。有的时候呢，他只是因为他，嗯，太糟糕了，所以呢，他不得不去制造一个庞大的数据，制造一个信息茧房，这个才是互联网上的常态啊。

因此呢，我们说，哎，什么群体，什么爱狗群体是主流了啊，你看你说一句狗不好，那他们都要跟你翻脸。不是啊不是，就像前就这这几天的新闻吧，应该是这几天的哈，诶诶，好，谢谢。金翼之虎啊，啊，就这几天的消息是哪儿，然后有在那个，有，有谁在小区还是哪附近，就灭了一群狗啊。有人那个往草地上丢东西，灭了一群狗。然后呢，就有新闻报道的时候，选择性的报道，用那个新闻的标题转移大家的视线。这标题起的非常的新闻系啊，新闻专业咋起的呢？说这个某某地啊，惊现投毒害狗，附近几十米远就是幼儿园。他给大家那个标题形成的印象是什么？哎，你瞧他们在草地上扔小零食去害狗，但是呢，你看他们都不注意孩子，孩子离得那么近，他都那个在这个地方投放有毒物质啊，他就是要让人形成就是投毒的这些人，连人都不在乎，十分的残忍的这种印象。

但实际上这里头隐含的另外的一个故事是什么？当然，现在的那个，现在的读者啊，被捶打的久了之后，耳光扇得久了之后呢，也学会了这种战斗的模式，于是很快的意识到，你这个叙述明显的是一个拉偏架的行为。因为这同时就意味着，离幼儿园几十米远的地方，长期的有大量不拴绳、没有主人牵制的狗，对吧？不然他怎么可能会吃到这个地方的有毒物质呢？所以幼儿园这么近的地方，不牵绳啊，不戴嘴罩、无人管制的狗，大批量的就在这个地方，哪个东西造成的危害大？所以实际上这个这个世间，就是你听到的声音不一定是真的。很多的时候真的是要擦亮了眼睛，去一点一点地扒开它里面的隐藏的信息。我们现在信息呢，不是缺少，而是太多了，所以你很难去找到里面最核心的一个点啊。
