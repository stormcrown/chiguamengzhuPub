# 第1194章 自然审美的特点_阶级审美的驯化

自然审美的特点与阶级审美的驯化——以《加勒比海盗》为例

关于选美这件事，其实一开始它就是以颜值作为标的物的一种市场运作。搞起这个活动后，可以带动大家的跟风，促进传媒业、影视业的发展，还能带动消费产业。因此，很多企业纷纷投资，把全球各地各种各样的审美大赛都拱了起来。大约十几二十年前，选美这个行当还特别流行，那时每个地方都有自己特色的美人。而全球选美大赛的冠亚季军中，比例最高的是拉丁美人。为什么呢？为什么拉丁美人获奖率那么高？主要原因在于拉丁是一个混血区。混血区有一个特征：这种特定的群体会产生两种倾向——混血优势和混血劣势，两者同时存在。这属于初中生物学的知识，没有超纲，我相信大家都能听明白。在这种远缘杂交的过程中，子代会获得亲代身上的特点，但因为种间杂交，它会随机获取双方的特点，最后拼接起来的个体常常千奇百怪。这就是杂交互补所产生的一种比较混搭的状态。在这种混搭过程中，由于双方基因特征不匹配，比较容易出现遗传病和一些奇葩的个例，这叫做杂交劣势。杂交劣势的比例非常高，这也是为什么我们在选育选种时需要淘汰大量残次品，并进行近亲回交。所以我们看到拉美地区的普通人常常长得有点奇奇怪怪的，包括他们自己制作的卡通片、动画片中的人物面孔差异很大，千奇百怪。这是客观现象，所有杂交群体都是这样。

但同时，杂交也有优势。杂交优势是指选取了双方身上各自优秀的东西，这就像中基因彩票。这种杂交优势下产生的混血美人，恰好获得了双方的优势特点，因此拥有双方的审美点，让观众没有种群屏障。无论什么类型、什么种群的人，都会觉得她很好看。所以，在审美还很市场化的时期，国际选美大赛中经常是拉丁裔美人获奖，就是这个原因。突然想到一些好莱坞美人也非常符合我们的审美。

说到这里，我们可以聊聊关于自然性基本审美的一些基础原理。这并不复杂，都符合正常的生物学基础知识。当你待在自己的群体内部时，任何一个群体首先会认为自身的特征是美的。就像美洲豹会优先认为美洲豹这个类别最美，非洲猎豹则会优先觉得猎豹类型最好看，彼此之间会有审美上的冲突。这是一个普世现象：无论什么种群，最终都会觉得自己的种群最好看。当然，如果你不这么认为，非要觉得自己最难看、别人身上都有优势，那不好意思，你一定经受了审美驯化。因为自然界所有生物的本能都会觉得自身的种群最漂亮、自身的特点最优秀，所有群体都一样。因此，在天然审美范围里，我们会偏重于自身的特点。

那么为什么我们之前会觉得好莱坞推送的顶流美人很好看呢？这是因为每个群体中最美的那些个体身上会出现雷同点，就像黄金比例一样。虽然每个地方有自己的审美特征，但它们之间会有雷同的东西，这属于数理上的美学——比例合适、是否对称、是否健康。这些都是非常天然、自然而然的美学。当这种美达到极致时，可以跨越种族。但这种顶尖的、能跨越种族的美人非常罕见，绝对是千里挑一、万里挑一。所以当年好莱坞选定自己的顶级演员时真的很上心，他们挑选的是真正的顶流美人。为什么要这样做？因为当时还没有实现全球审美的洗脑。如果那时候直接摁着你的脑袋说“小美人鱼是顶级的美人”，你一定会抽他几个大耳刮子，电影也卖不掉。他想要让别人相信他的审美有实力、能产生审美的定义权，就必须拿真美人出来。所以几十年前那些老派的好莱坞大明星，是真正可以跨越种族的绝色美人，任何种族都有这样的个案，但数量极少。他们只是把这些人挑出来了。挑出来之后会产生什么影响？当你多年以来一直看白人的顶级俊男靓女时，时间久了你就会觉得“白种人长得好漂亮”。形成这个印象后，他就可以在里面注水了。我们会注意到，好莱坞下一波的新人就没有这么好看，包括现在大家认为不错的“寡姐”，如果放到更早一代人当中，她很有特色，但不够漂亮；放到后面这一波里，就算很好看了，因为下面更丑。好莱坞逐渐逐渐地把那些不好看的像掺沙子一样掺进去，让大家逐渐适应，去欣赏跨界之美、欣赏并不美的美、欣赏你不喜欢的美。这就是审美的驯化。

经过漫长数十年的审美驯化后，我们目前的审美已经开始趋向多元了。不管你怎么想，事实上你已经能认清楚外国人的脸了，这就是驯化的表现。因为之前中国人普遍对外国人脸盲，无论对方什么肤色。所以，整个审美问题既存在很天然的、属于人类本能上的自然审美，同时也存在可以被社会人为驯化的社会化审美。这种社会化审美怎么进行？它需要高压灌输：不管你喜欢不喜欢，它要求你认为它美的人高高在上，哄抬其地位，让你觉得他很高级范儿，让你觉得跟着他就会发财、会高级、会了不起。慢慢地你就会产生“她也很美”的感觉，这就是社会化审美的训练。我有时把它叫做“阶级型审美”或“科技型审美”，它是通过高压驯化方式，让对方觉得本来不符合自然审美的东西是美的。这在所有人群中都存在，即使在一个族群内部也有。我过去说审美问题时常提到“高级人审美”：所有贵族都会努力与中下层拉开距离，有时通过肤色，有时通过头发，有时通过衣着。无论怎样，他们都想方设法与他人不同。拉开这些差距后，就形成一个社会审美倾向——凡是跟上等人相似的就是美的。因此，审美驯化必然包含这种高压式灌输，它一定是把某些人抬到很高的咖位，然后进行强制性输出，告诉你他很高级，最终你会潜移默化地觉得她很美。

我们就在这两种不同的审美倾向下，能看到一些东西。例如，今年流行什么样的服装？服装也是典型的审美内容。你会发现，当它一定要让你欣赏某种服装时，会通过高压灌输方式走出来。什么是高压灌输？就是铺天盖地：你打开任何一个屏幕，看到的顶流都穿着它，那些上等人都以此为荣，那些大号、顶流美人都会把这种衣服穿在身上。然后你会觉得她好好看，她一定是全场的主咖，她一定是最美的那一件。这就是服装领域的审美驯化。嗯，巴黎世家、香奈儿，对。其实西方大部分奢侈品牌就是用这种方式。你真的看它时，并不会觉得它美。例如你把它的logo去掉，你觉得那种夹脚人字拖、水晶拖鞋是什么美的东西吗？那玩意儿我小时候在公园里踩水时穿过，没有任何区别，就长那样。但为什么人家能卖那么高价？因为它摆在了最豪奢的柜台里，聘请了一些顶流名媛穿着它，还莫名被人家“街拍”了——街拍还是用顶流的相机，我也不知道是哪个路人就用顶流相机把人家随机抓拍到了。于是这东西就变成了爆款单品，大家一起跟风而上，觉得它就是今年流行主咖、帅炸了。但真的是那件东西本身好看吗？不是。甚至有些东西会刻意做得不好看。为什么？你会发现近些年很多奢侈品，包括自己的走秀，甚至是很多流行单品主推款，都刻意不好看。是因为设计师无能吗？不是。因为美是比较局限的，人类能欣赏的美其实是很小范围内的东西——它自然的、协调的、对称的，这种自然美范围不大。如果一个顶流设计师还在设计大家都觉得好看的美丽东西，它很容易就跟中低端品牌撞衫，无法特立独行，无法成为行业翘楚，因为他无法证明与众不同——这已经很重要的东西了。所以你会发现他会设计得格外难看，然后用高级logo和顶流大V出场来给自己抬咖，这就是一种审美驯化的路子。当你连这种东西都能接受时，其他稍微好看一点的东西简直就是主的恩赐。服从性测试，对呀。你但凡这个底线被突破了，这些丑东西你都当高级货、顶流拿出去炫耀，那你什么不能接受呢？他明天给你卖个蛇皮袋子，你也愿意出高价；后天卖一个农民戴的草帽，也可以出高价，还是毛了边的；再后天把穿旧了的解放鞋卖给你，还是高价。一定要首先突破底线，突破了之后才有下一个底线，然后不断不断用这些很糟糕、很恶劣的东西灌输到整个主流平台上。这个东西就是驯化。

富豪们真的喜欢这种吗？不是。他们眼睛没瞎，放心。他们眼睛没有任何问题。他们之所以穿这些东西，是为了印证自己的身份，是为了标榜自己在某一个圈子里面界定身份。而且当他们真的在圈内活动时，也不全穿这个。因为这些“广告衫”更多时候是要站到街上，让路人用高档摄像机、专业的风格进行抓拍，然后驯化路人，让路人觉得他好高级。这是一部分。然后还会有一种专门定制款，在他们额外的场合私下穿着。那些什么名媛、顶咖去穿的、你看上去就很雷的衣裳，都是品牌方安排的。品牌方是为了什么？是为了驯化客户。品牌方让这些名人穿着这些衣裳，是为了驯化客户，最后让那些迷恋这些东西、翘翘脚还可以买到一件两件的人为此买单，这是开拓市场的一种方式。你不要以为他真的觉得这玩意儿美，他们的眼睛没有任何问题。驯化客户，对呀，当然是在驯化客户。要知道那些顶流艺人、名媛、翘楚们之所以穿上它们，是不付账的，他们是广告支架、行走的广告牌、行走的广告位。他们免费把这东西穿到街上，让粉丝出来接机、拍各种各样的照片，说“炸天了”。最后都会有一些爱他的人、相信他的人为此买单，这才是它产生盈利的一种方式。顶流安排的那些什么限定款服饰是免费的，他们不要钞票，直接给。嗯，对，都是很值的。但是，大家还是会上套。你去观察一下自己身边，其实也有人爱这些东西，而且你是不能够指出这件事的，因为到时候别人会很尴尬，甚至跟你撕破脸。这是什么？这就是其实大家都知道皇帝没穿衣裳，但那小孩子在人群里说了“你看他光着呢”，他就变成了罪人。这个世界就是这样的。

什么才是真正的奢侈品？真正的奢侈品都是那些无论你用什么方式都很难拿得到的东西，才是真正的奢侈品。而现在的商业化奢侈品，实际上就是借由某些顶流宣传它是奢侈的，然后以奢侈的价格来收割。通常情况下，无论是审美还是品质，都不是很过关。我也弄过一点，连质量上面都不需要特别考虑的那种。嗯，量产的工业品都不是奢侈品，对，没错。但是，量产的工业品有足够的利润。我们传统上那些真正的奢侈品，例如像云锦这样的，实际上是它的成本太高昂了，每年的出产量就那么一点点。例如小产区的西湖龙井，每年就那一点产量，都不要说品质的问题了，谈都不需要跟你谈的，就靠这个稀缺属性。但是那些服装也好、鞋子也好、包包也好，有哪一个有稀缺属性吗？没有。盗版可以做得比正版还要结实、还要漂亮，是不是？那没有任何稀缺性，它靠的全是品牌叙事和名人背书。能够在市场上被我们普通人摸得到的，都不是真正的奢侈品。真正的奢侈品都藏在一些我们看不到的角落里面，因为过于稀缺，以至于我们拿不到它，这才是常态。

带一波，带一波好啊。带一波啥呀？我天天带一波。嗯，带一锅这个牛肉粒吧。这个牛肉粒很好吃啊。啥玩意儿啊？他说“阴阳怪气”是说我阴阳怪气对吧？那不然呢？我们直播间要讲文明的，我不阴阳怪气，难道直接骂人吗？是不是要讲点道理？直播间是一个文明的场所，直播不开喷，开喷不直播，这是根本性的原则。所以我一定不会开喷的，必然是使用阴阳怪气的方法。既然你听懂了，那真是棒棒的，说明你的阅读理解能力起码超过了小学三年级的水平，比昨天那个人要强啊。棒棒的，给你鼓励一下，一个拉黑。不说完不能拉黑，不然的话我心里头会添堵的。这里是不能够直接那人家拉黑的啊，我必须要说完我的话，不然的话念头不通达。那一天到晚拉黑这么多人，合适吗？合适。
