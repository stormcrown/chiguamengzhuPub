# 第1116章 古代王朝发房子发地_一分钟看懂农民在盛世怎么活

### 古代王朝发房子发地？一分钟看懂农民在“盛世”怎么活。

**《观史叹农》**  
鼎革初定授田忙，  
里闾书声杂稻香。  
岂料朱门吞瘠土，  
终闻白骨泣残阳。  
兴衰非独庙堂事，  
冷暖直关闾巷殇。  
莫道周期天数定，  
人谋岂可卸纲常？

古时候的普通人，也就是农民的一生，其实大部分时候日子过得不是很好。但如果是在王朝的前半段（即从王朝建立之初到兴盛高峰点之间的那段时间，有时是数十年，有时是百年），在这段光景里，中国的普通农民的日子看怎么比较：如果是横向比较，那是非常好的。

例如，以汉代来说。在汉代，乡村里设有“里”——里就是小村子，或者说像乡村里的小区。小村落建有围墙、大门，有专人看门，并登记出入人员。任何人出生后会获得一次专门登记，登记后只要长到一定年龄，就有资格分田。但这种制度仅限于王朝早期以及相对兴盛的时期，因为后来土地兼并出现后，就没有这个项目了。在早期，所有人只要长到一定年龄，就有分地的资格。条件较好的地区，除了分地，还会分一间房子。所以农民的一生，先是生长在父母保护之下；长大后，到了能够自立门户时，就会拥有一块田和一块可以盖房子的地，类似今天的宅基地；有的甚至直接给一间房，都有这种情况。这取决于乡村的贫富。

关于学习资格：在汉、唐、宋、明等时期，如果当地条件较好、经济没有崩溃、国家也不动荡，农民子弟仍拥有受教育的权利。甚至会有一定的强制教育，但门槛很低，只要求认得基本字，目的是在发通知时能够签字画押。这是最早期的做法。所以，如果朝代处于安稳期，普通农民子弟的生活与今天相对贫困地区的贫困农户之间的差异，恐怕不是特别大。

耕作条件方面，从战国开始，铁器的运用就比较多。有了铁器后，耕作效率和产出率提高，日子相对好过一些。税收一般按照田产征收。但除了交正常的农税之外，农民还要服劳役和兵役。早期服兵役通常在农闲时节征召，有时是受训几个月，有时则是打仗时征召。劳役频次较高，有时没有错峰安排，可能耽误自家劳动——这说明当地管理不当。劳役通常是挖水渠、补城墙等活计。所以，普通农民在中国古典时期，日子过得不算糟糕，尤其横向比较时，生活是很好的，因为相对稳定。
