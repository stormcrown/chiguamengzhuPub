# 第1147章 被曲解的文明坐标_聊聊中国历史时期划分合理性

被曲解的文明坐标：聊聊中国历史时期划分合理性

《文脉匡正辞》云：观夫泰西定历代之纲目，犹持矩度圆；华夏承天地之正朔，若秉烛夜行。彼以三器划鸿蒙，岂知攻玉之智早超石髓；妄将五段分今古，焉晓铸鼎之技已贯星虹。昔者周行封建，裂土而治，此诚先王经略之制。秦立郡县，海内为一，实开万世治安之基。奈何西人窥斑见豹，强以领主之规绳我千年；更见竖子数典忘祖，竟把金科作秕糠。夫玉石至坚，雕琢见文明之深；青铜虽利，冶炼显工艺之精。

全世界范围内，真正有着最绵长历史、最辉煌创作，并且能够有始有终地用成文方式记述历史的，就我们独此一家，别无分号。结果呢，这个历史的分段划分方法，却不是中国人定下的，是西方人定下的。但他那边的历史进程和我们完全不一样，尤其是在分段的词上面，比如“封建”这个词，就给我们今天的很多讨论挖了一个很大的坑。

你谈到封建，封建是什么？西方人划分封建时，他的封建是指他们之前领主制的那个阶段——分封而建制，所以叫封建。他这个封建的管理水平，约等于我们的先秦时期。而等到我们秦之后的大一统时期，是郡县制的君主集中制管理。但对这个阶段，他没有真正的阶段划分，因为西方人没有这个阶段——他没有这种标准的郡县化、郡县制中央集权制的管理。这种管理方式和我们近现代的国家管理方式非常非常相似，中间差距极小，只有在经济模式、生产力的模式上略有差别，但在管理原则、原理上是一样的。所以封建这个词在中国就出现了两重含义：一个是分封建制，和西方人一样，但这只限于先秦时期，到秦代开始就没有了。而秦代之后的阶段，他又依据这边的经济生产水平、小农生产水平和经济划分法，把它称为“封建”，但这种封建和以前的封建完全不同，而且和世界上其他地方所谓的封建也完全不同。也就是说，我们在君权管理的2000年时间里，是全世界独树一帜、唯一一个叫封建但根本不封建的时代。所以你起名字搞分段，不是按照我们的历史来进行的划分，这个词汇用起来就很荒唐。

包括石器时代的划分也是这样。他把对石器的利用、利用水平，用这种方式区分开新石器时代和旧石器时代。但哪怕就是中国以外的地方，哪怕是新石器时代的晚期，也没有实现玉石加工，因为玉石是硬度非常非常硬的石头，它的硬度好像是13到16吧，我记得大概是13到16。所以我们才有一个词叫“他山之石可以攻玉”，至坚者玉嘛，《红楼梦》里也说“至坚者玉，至贵者宝”，我们都把它视为最坚硬的石头来选择。既然是最坚硬的石头，你想把它雕琢出来，需要用什么方法、什么工具、什么器材？所以当我们的器材已经足以加工最坚硬的玉石时，你还把这个阶段叫做石器，就已经有点意思了，对吧？你怎么考虑你对这个石材的加工问题？我们能够加工玉石了，但还是叫石器。所以实际上很有可能，中国的历史是完整地经历了所有技术演化的进程，但其他的地方是我们的次生文明，或者是我们传播出去、衍生出来的文明体。所以它们经常是某些事物发展到某一个阶段后，没有继续发展，直接就进入到下一个阶段了。

例如新石器时代就进入到了青铜时代，而我们这里是把石器一直发展到了玉石这样璀璨的巅峰，在此期间开始的冶炼也是一点点发展起来，都能找到痕迹。包括青铜：纯铜比较软，后来发展青铜合金，要有铜合金就需要锡矿、配套的矿藏，要有炉子，要能烧制，因为炉温不一样，烧制合金的标准也不同。我们这里的所有的器材也好，矿藏也好，都是能够找得到的。但在其他的地方，尤其是在跟我们争夺文化源流的西方——西欧地区，他没有这些资源，他也号称走入了青铜时代。他是怎么走进青铜时代的？这是一个问题。就他西方人定的这个分段方法，说石器之后进入到了青铜，但他怎么来的青铜？他没有足够量的矿藏，也没有足够量的可以配套的燃料，而且没有配套的熔炉。相应的东西都不存在的时候，他就说他有青铜时代。他青铜时代哪来的？所以他们这些年就一直在搞考古，说要发掘他的青铜器。然后他的青铜器当中，有一部分是海里捞出来的，有的是农民用锄头刨地的时候从地底下刨出来的。可能他们家的农民以前从来不刨地，随便种两棵小麦就能刨出青铜器，这也是很厉害。我们这里挖青铜器都得挖十几米下去，有的特别深的地方要挖几十米下去，人家那边农民一镐头就能挖出一大堆来。反正它就能刨出各色各样的青铜器，还能从海底下捞出两千多年以前的青铜器。而且还有理由：我们几百年前宋代沉船里的铁器都已经烂成一坨了，要鉴定元素，然后才一点点复原，比如铁锅。但他们在清代的时候还没有办法铸造铁锅，却跟我们差不多同一时期就进入了青铜时代。在没有矿藏、没有燃料、也没有炉子的情况下，就直接进入青铜时代，所以这个时代划分是很有问题的。

可是中国这边，我们的进程就比较清晰明了，每一个阶段都有自己的发展脉络。石器时代和青铜时代之间是交织的，即使是进入了青铜时代，我们也依然在开发那些石器，比如玉器的雕琢，把玉器作为礼器来使用。一直到今天，大家都还是喜欢玉石，这就是一种很明显的痕迹——好的东西最后会被留下来，这是最重要的历史遗迹，放在我们身上的文化传承。但在其他的地方就没有。包括大家说他们喜欢宝石，宝石是什么？宝石是各种各样的矿物晶体。晶体这个东西，虽然到今天技术发达了，才能切割成什么八心八箭——各种切割面让它很璀璨，但晶体在自然界当中是可以孤立存在的。你找到了宝石矿，把旁边的杂石敲掉，它就可以直接出现，可以用一些比较随形的方式直接佩戴，所以它不存在那么大量的、非常艰难的加工。你去看一下玉石的加工流程就知道了，玉石的加工流程和宝石的开采流程是完全不同的。所以他们有可能会找到宝石，因为有一些裸露的宝石矿，你把它其他的石头扒拉掉之后，整块的宝石就会出现。但玉石完全不是这个样子，它需要我们有漫长的时间对这些东西极为了解，并且有切割、加工、研磨、雕琢的所有器具，还有工匠和艺术的水平才能完成。所以你硬是把这些东西绑定在一个什么时代里头，我觉得是一种很将就的划分方式。晶体基本上都是单质，所有的这些宝石基本上在相应的矿藏中就是夹在矿中间的，你想找到它确实也需要点运气，但找到之后不需要巨量的雕琢，敲敲打打的就能弄出来，所以它的加工工艺要小得多。
