# 第1182章 武则天和慈禧只有一个共同点

武则天和慈禧是不能放在一起比较的。她们只有一个共同点：都是女性。仔细想想，这个共同点其实就两个：都是女性。哈哈。如果要比较武则天，更应该把吕雉和刘娥放在一起聊一聊。你可以把武则天、吕雉和刘娥放在一起比一比。我确实觉得，无论是吕雉还是刘娥，在政治角度上比武则天要强。吕氏和刘氏的能力要比武则天强。但是为什么会这样呢？我之前聊武则天时谈到过，武则天身上有一个最大的短板。她之所以能当女皇，本质上是因为皇家的继承人都是她生的。她是以太后的身份，以摄政的身份去当了女皇。中间就差那么临门一脚。当时国家的整个大众也好，所有的氏族群体也好，或者官僚群体也好，对这件事有点介意，但没那么介意。所以，在朝堂之上，她拉了几个自己人开始打嘴仗之后，就把这件事基本上平息下来了。她戴上冠冕，就可以以皇帝的身份执政了。但是她在操办这件事的过程中，实际上犯了一个忌讳。

中国的皇权传承，最要紧的不是皇帝聪慧或昏庸，而是皇帝能不能稳定传递系统的稳定性。稳定性来自于它符合继承的规则。所谓的嫡长子继承制，偶尔会有例外，但大多数时候我们都遵照这个制度。按照这种方式一代代传下去，可能会碰见昏庸无道的，但我们仍秉承这个原则，因为这种方式最稳妥，不容易导致朝堂生乱。而且国家不是靠皇帝一个人说一言堂。皇帝傻了，事情就乱了吗？不是的。我们国家的皇帝是一个象征意义更重的角色。有时候，皇帝哪怕是襁褓中的幼儿也可以。因此，武则天在这件事上犯的最大忌讳就是，她明明可以临朝称制，没有问题的情况下，却非得去当皇帝。当了皇帝也妥协了也行，毕竟她的儿子还是李家的传承人，没有问题。可是她在中间，不管是表演还是真的这么想，曾经试图让武家的人承接国祚，这个毛病可就大了。一旦做了这个动作，就是篡权，会导致整个管理策略出现问题。你想后世的时候，皇帝结婚迎娶皇后时，是不是就不踏实了？当皇后有可能篡权时，还能让她当自己的伴侣吗？要知道，之前胡人政权中，五胡之后的胡人政权普遍有一种风俗：如果某一个儿子被选中当继承人，那么他的母亲就要被拿掉。立子去母。为什么这样？其实就是为了避免母亲专权，尤其是避免外戚篡权。但在武则天身上，我们发现她搞出这个活儿后，实际上就践行了母亲有可能篡权的角色。这严重影响了王朝的继承制，动摇了根基。

所以，因为她做了这些事情，根基不稳，在管理过程中就必须采取制衡之术。制衡之术经常在朝堂上拉一派打一派，大家打嘴仗，彼此不能和睦，这就是一个很大的问题。她到了当皇帝这个阶段，反而把自己的管理弄得名不正言不顺。本来作为一个太后，包括吕雉也是以太后的身份管理朝堂，以太后的身份完全可以名正言顺地管理整个国家的朝政。但是到武则天这里，因为她去称制当了皇帝，反而变得不正当了，人人都像防贼一样防她。于是，她采取的策略自然而然侧重于朝堂内部的争斗和纷争。谢谢福娃娃。这是她的一个必须选项。当你做了这种选项时，对真正国家大事的处理就会出问题。例如，在武则天整个武周一朝之前，打下来的突厥地盘没了，飞了，人跑了。跟吐蕃之间的关系，以前的和睦，现在分分合合，合合分分，已经开始分多于合了。所以，她在国家最根本的事情上做得并不好。但之所以能这么不好，是因为她自己立身出了问题，所以她必须倚重于某些人，甚至某些酷吏，大家都知道的奸臣。吕雉和刘娥是真的强，因为人家真正在当朝理事。但武则天呢，因为权争搞得太多了，她的侧重点完全偏了。
