# 第1180章 杨坚功绩_力挽狂澜结束乱世_汉语是满语变种谣言澄清

杨坚功绩：力挽狂澜结束乱世。汉语是满语的变种？谣言澄清。  
[吃瓜蒙主]  

杨坚是不是被西方夸大了呢？看跟谁比呀。杨坚他的能力已经很强了，没有到夸大这个程度啊。杨坚可以说是力挽狂澜的一个人了。你要知道那个时候的中国的环境，其实是非常恶劣的。  

今天有好多人说，我们穿越到什么魏晋南北朝嘛，是吧？那你喜欢啥不好，你喜欢魏晋南北朝，想过去当大女主，这真是多离谱的事情。在那个时代，中国国土上的混乱程度、黑暗程度，那是令人发指的。虽然说晚期的时候，五胡开始建立标准化的政权，甚至开始逐渐地汉化，就是说已经开始正规化了。但是正规化了就意味着它好了吗？除了孝文帝改革的那个小小的阶段当中，它实实在在的是推行了全面的汉化，其他的时代的所谓的汉化，其实就跟我们今天中国人学英文是一样的操作方式：我学习你先进的一点东西，但是我依然用自己原本的那套理论、原本的那个身份来进行制衡。所以那个时候的统治，对于绝大多数的土著中国人来说是非常不友善的。  

像杨坚这样的是依附于鲜卑的这些世家大族，当时有很多呀。他们之前还不就是乱世之中，仗着自己手上还有点势力，然后就抱了胡人的大腿，然后跟人家去开创伟业，但是心里头依然怀着其他的想法。所以杨坚在这个时候，实际上它结束的并不是什么其他的政权，它结束的是一个乱世。因此他个人的成就是否真的扩展到了开疆拓土、豪横无比的程度，不重要。因为他结束了一个乱世，我认为给他一个很高的评价是不过分的啊。  

啊，带货啊，好的好的。带货啊，啊，来，这个是清酒，国产清酒。清酒这个东西它是从西周的时候开始出现的。西周的时候我们在进行祭祀礼仪的时候，使用的酒水都是清酒。它是一种酿造工艺，也是一种过滤方法。运用这种酿造工艺和方法的呢，都可以称之为清酒。但是你现在看到的这一款清酒，是用大米来酿造的，它的这个酿造和与大家在日本餐馆里面喝到的那种会更相似。因为他们是在唐代的时候跟我们学习的清酒酿造法。但是他们的清酒发展其实是工业革命之后，进行了标准化的推广才开始变好的。在那之前，日本的清酒产量其实极低，因为他们的物资不丰，而酿造酒水需要比较多的粮食的富余，他们古代的时候一直都做不到，所以酒的产量非常的少。最后还是中国这里的酿酒的量是最大的，因为我们的农耕的产业更加的庞大，精细的耕作呢，使我们的物产更加的丰富，就会有各种各样的酒水出现。那清酒呢，就是我们延续下来的祖先的酿造方式。在先秦时期，它常常在北方使用黄米来酿造，后世呢，又开始开发出来糯米和大米的酿造法。现在大家看到的这款就是用大米来酿造的清酒，口味呢，非常的干净清纯。呃，喜欢清酒的小伙伴不要错过。它的产地是江西赣州啊，是一个从原料到设计完全国产的一个品牌。这个酒水呢，性价比还是很好的，度数呢，在15度也很适中。清酒的酿造方式其实跟黄酒有一点点像，因此度数也比较相近。因为酵母菌耐受的酒精度大约到16度左右，所以一般情况下，它的发酵的度数都会在15~17度之间哈。这个清酒呢，就是标准酿造的15度的清酒。喜欢的小伙伴不要错过哦。  

不买汉人的酒？我也可以不卖啊。是吗？你悉听尊便呢。一个冲上来开黑的小号，然后给我威胁说不买汉人的酒。你家主上都买汉人的酒，知道吗？啊，你家主子都买，你愿意不愿意的，反正我也不管这些事。好喝是吧？对对对啊，这个酒本身还是非常干净的。嗯，怎么还用汉字呢？他们怎么可能认这个东西呢？人家说了汉语是满语变的，大家都看过这个谣言吗？在互联网上传播的还很厉害，说现代汉语的普通话就是满语的变种。满语都亡了，然后他说是满语的变种？如果说普通话就是满语的变种，晚清时期的林则徐，为什么还要找一个满人师傅去学习满语？那他学满语得多容易啊，是吧？结果呢，硬是要请一个满人师傅学了好些年，人家还说他学的不好，又追加学费又重新学是吧？结果到今天，普通话就是满语的变种。只要不要脸，什么谣言都能编。谁不买汉酒了？这不是前面那个小黑子，在网络上号称不买汉人的酒，然后在摊儿上的时候照喝不误是吧？吃饭的时候也不分胡人是汉人还是谁是吧？他也不挑。赚钱的时候也不分老板不是汉人是吧？消费的时候呢，也不管对方那个卖货的是不是汉人；说话的时候也不管说话的这个语言是不是汉人创造的；写字的时候呢，也不管这个文字是不是汉人创造的；考试的时候也不管这个考试的系统是不是汉人创作的；甚至去背课文的时候，都不考虑文章到底是不是汉人写的。反正呢，他想喊号的时候，就说我与汉子那个啥啥啥是吧？爱喊什么号喊什么号吧，就好像他在这里喊了个口号，他就能脱离出来了似的。不知...
