# 第1109章 冷知识爆炸_外国人双眼皮其实是眼窝缺脂肪

### 冷知识爆炸·外国人双眼皮其实是眼窝缺脂肪 —— 【水调歌头·容鉴】

秦俑彩痕在，汉月照衣冠。  
谁言单眼证，血统妄分藩。  
本有重眸星汉，偏作胡沙离散，诡计裂轩辕。  
皮下脂深浅，岂可定忠奸。  
眉弓影，窝壑现，误称鸾。  
欧风东渐，偏把沟壑作双鸾。  
三十春秋面相，百代语言肌理，心相刻容颜。  
莫被皮囊惑，青史自昭然。

为什么大家总是会觉得白人的双眼皮比例特别高，说他们没有单眼皮？其实是因为他们的眼窝这个地方欠缺皮下脂肪。所谓的“眼睛深邃”，就是抠进去了。这种结构会导致这里出现一道沟壑，尤其是当他们的年纪超过二十来岁之后，以前还有点婴儿肥的那些皮下脂肪一消失，这里就全部有一道沟。这个沟就会让你误以为它是一个大号的双眼皮，实际上根本不是。白人也有很多的单眼皮，而汉人也有很多双眼皮。包括秦始皇陵兵马俑里面也是既有单眼皮又有双眼皮的，双眼皮的比例并不少。没有人跟你说过什么“汉人都是单眼皮”，这是一个谎言，而且是刻意制造的谎言。为的是啥？为的是把那些双眼皮的人开除出汉籍，让我们内部产生撕裂。实际上，哪怕是老秦人这些正宗的汉人群体，他们也是有单有双的。兵马俑之所以你看到的都是单眼皮，是因为外面的彩绘掉了。有彩绘的时候，他们在眼皮子上面画一道线，那就是双眼皮。兵马俑也一样，是有单有双的。

很多白人的双眼皮是眉弓太高形成的。对，白人也有很多单眼皮，你得去细看。就是他本来是单眼皮，眼睛也不大，但是因为他缺少皮下脂肪，这个地儿抠下去了，形成了阴影，你就会觉得他眼睛有点大，然后还会觉得这个地方有一道沟，就觉得他是一个大号双眼皮。是不是有一个什么整形手术专门就是开双眼皮，叫什么“大号欧双”，对吧？我记得在看到广告的时候瞧到过“大号欧双”，就是所谓的那个双眼皮可宽可宽的，一直宽到上头。实际上欧洲人的双眼皮也是各种各样的，而且也有很多单眼皮。之所以觉得她的双眼皮特宽，是因为他那个地方眼窝深陷，形成了一道沟壑。

单眼皮没什么不好，这跟好坏有啥关系？我们没有谈单眼皮的好坏。人长得好不好看，跟他是单是双没有什么关系。长得好看的人，单眼皮比双眼皮还好看的人有的是。眼皮是单是双并不影响她的美貌。我们现在谈单双，是因为有一种所谓的基因谣言说汉人是单眼皮的，所有双眼皮都要开除出汉籍，他们都是混了胡人血的。这个谣言是刻意地给汉人掺沙子，试图让汉人相信自己没有共同的血脉关系，试图让汉人通过单双眼皮把自己的内部切割成两块。就跟那个什么胡族啊，那个什么族一样——就是人为地把人划切成两块。他是有这样的动机在里头。实际上，正宗的、纯粹的汉人群体当中依然是有单有双，而且双眼皮的比例一点都不低。

说什么话都会影响脸的形状，那肯定的。你的所有语言、所有表情都会导致肌肉的走向发生改变。有些人年纪大了之后，就显得越来越好看了；有的人年纪大了之后，就变得越来越凶。其实就是他过去的这些年所使用的所有表情，最后都堆在脸上面，能够让你看出来。有一些上了年纪、已经三四十岁之后，你还能在他脸上发现凶巴巴的样子，就好像长在脸上一样，就是因为他之前或者日常的生活当中是比较凶的，所以就会把这个东西刻在脸上。同样的，语言也是这样。因为不同的语言，它发音的方法是完全不一样的，运用的肌肉线条、运用的肌肉也是不一样的，最后就会导致你的肌肉的走向发生偏差。确实，你们身边都遇到这样的人是吧？我们在这个相由心生，对，也可以这样说。之前还有个说法说“30岁之前的脸是父母给的，30岁之后的脸是自己给的”，讲的也是这个事儿。就是你在30年之间，经常性的这种——是微笑居多还是愤怒居多，是用什么样的方式说话，是慢声细语的还是嘶吼的——都会影响到你自己的面部的表情纹路，也会影响到肌肉的走向。所以人越到后来，他的面孔就跟他的行为之间关联越深。
