# 第1101章 盲目自信才是人类本能_99的人都被洗脑了

“盲目自信”才是人类本能，99%的人都被洗脑了！  
夷夏殊形，天生我贵；红毛鄙陋，唐韵自雄。玄奘西行，笔诛异貌；晚清虽敝，犹斥鬼戎。殖民遗毒，摧心毁骨；反思成疾，丧志卑躬。迷途当返，重拾天授；华夏精魄，岂付东风？  

在中国人身上也是这样。我们并不是自古以来就觉得自己不好看，我们自古以来觉得世界上我们最好看，别人长得都丑。这是自然行为啊，只要一个群体他有自我意识，这个东西就是基本的东西，这是最基本、最基本的底色。  

有关于说，如果一个人他的生活环境比较闭塞，甚至他这个群体可能也都比较闭塞，也不发达，这种情况他会不会拥有自信？其实呢，我之前忘了在哪个视频当中提到过有一个基本的概念，大家一定要知道：人是一个智慧型的生物，而我们的这个智慧会让我们产生一个bug，就是一个缺陷。这个缺陷就是我们会思考，而人在思考了之后必然会出现的一个疑问就是：我从哪儿来，我往哪去，我是谁？然后呢，还会有一个非常根本性的疑问：我为什么而活着？而只要这个问题被提出来，就会陷入一种不断循环自证的环境里面去，然后精神就会崩溃。所以这个bug就是我们的人类在思考的进程当中，在发达的智力的进展的过程当中，天然地出现了一个缺陷，就是会让我们产生自毁倾向。所有的人身上都有，就是我们会有自毁的倾向。于是呢，为了打上这个补丁，我们就有一项特异功能——甩锅、找理由、找借口。所有人身上都有这个毛病啊，我们都会在遇到事件的时候、遇到挫折的时候，找一个理由把这个原因甩到外面去，反正错了一定不是我的错。但是如果成了，那必然是我成了。这是干啥？这个就是在构建“我为什么而生活”、“我为什么而存在”这个答案。这是人类天然的能力，所有的人全部都拥有这个能力。如果你身上没有，那说明你受到的教育有问题。这是一个能够持续稳定地存在、非常根本性的东西。你也可以叫它“盲目的自信”，就是我什么都没有，但是我自信，叫“盲目的自信”。这是一个根本性的东西。  

所以带着这个理由，我们去看一下全世界的所有的群体，你会意识到一件事情：哪怕是漠南、非洲的那些闭塞地区的、完全没有（或者说很难接触到现代社会的）部落民，也是拥有充分的自信的。哎，谢谢哈拉少啊，都是有充分的自信的。我们去看隔壁的阿三，他的条件很好吗？我们日常看他的新闻的时候，像不像看笑话一样的？每天都在“哈哈哈”，说“这咋想的呢”，哈，奇葩奇葩。我们每天都在这样吐槽，但是他们的人非常非常的自信。你跟他聊过天你就知道了，异乎寻常的自信。他可以什么都没有，但是他可以自信，而且他觉得他可以碾压你。全世界的所有的人其实都有这种能力，无论它生长在什么地方，无论它是否闭塞，因为这个是人类的本能，是存活之基。  

但是在中国人身上，确实有相当相当一部分的中国人欠缺这个东西。这个是我们周边的环境、舆论和教育综合带来的一个缺陷。它不是什么可以骄傲的东西，就是那种动不动就要质疑自己，而从来不质疑别人。别人身上有优点，我一定要看见；但是别人身上有缺点，算了，一定要原谅他。我身上有缺点，一定要深扒，一定要搞清楚到底错在哪，我一定要改；但是呢，我身上的优点不要骄傲，不要自满。这个是近几十年来外来文化对我们造成的影响。我前几天的时候旁听了一耳朵，有人说“哎，这个就是中式教育啊，就是中式家庭，就是中式思想”，错的，这不是中式思想，这跟中式一毛钱关系都没有，这是西式特意灌溉给我们的思想。这种不断的反思、相信自己重大缺陷是从什么时候开始呢？从上世纪80年代左右开始的。那个时候的学者、那个时候的文人开始去撰写西方的文明有多么的伟大，开始盛赞中国以外的所有地方的美好。如果那个地方落后，那么它就是“野性之美”、“自然之美”、“奔放之美”；然后如果那个地方先进、发达，那就是“富有的”、“有秩序的”、“文明的”、“礼貌的”、“有远见的”、“严谨的”。总之呢，就是对外人种种的溢美之词停不下来，但是对中国人呢，就一定要贬低：有秩序那是没自由，没秩序那就叫做野蛮和落后。嘴都长在他身上，任意他去贬低。所以呢，我们的这种所谓的“反复自省、从不骄傲”并不是自古以来的东西。  

我们自古以来是什么样子呢？记得吗？就是清末的时候，中国人已经不太自信了，因为那个时候的中国人营养条件很恶劣，身高比明代的时候下降了平均五公分，普遍性的半饥半饱，然后整体的形象面黄肌瘦，条件非常的恶劣。而且呢，99%以上都是文盲，一字不识。就是在这样的一个很闭塞、很落后的情况下，中国人称外国人是什么？“红毛夷”，对吧？哪怕是在那个时候，他也觉得外来的人是丑的，是不好看的。放到今天的某一些人身上，说“你这个是盲目自大，是过于闭塞、是不懂审美等等的”，其实是错的。全世界的任何人都有责任、有义务相信自己是最好看的，这个是天然而来的，不需要其他的什么人去教你，就是自然而然的。任何人都相信自己的族群是最漂亮的，这个才是自然法则。不然你怎么存续下来？天天觉得自己这不好那不好的还能活下来？开玩笑的吗？不会的啊。每天否认自己存在价值的这个群体，一定会消亡的。所以中国自古就是很自信，很自信，自信到什么程度？就是在我们最糟糕、最糟糕的时候，觉得洋鬼子长得好丑啊，这个是很自然的一个族群自保的精神。那么在我们当年更加强盛的时候呢，我们就说大家经常去吹嘘的“开放又包容的盛唐”。那个时候玄奘法师西行，西行了之后回来，被迫地写了《大唐西域记》（还不是他写的，口述的，然后别人帮他去抄录下来，又重新修缮了一下）。就这本书作为南亚大陆上面的唯一成文史来使用，就是他的游记口述版游记。那在这本游记当中，玄奘法师对于他途经的各个小国是怎么看待的？如果哪个国家还算是比较体面，他就说“颇累我”，就是有点像我，还不错；但凡哪个地方他觉得那个地儿不好，立刻说“丑陋鄙陋”。这还是个和尚呢，口不出恶言的僧侣，得道高僧、文化人儿，谦逊有礼的大和尚，他对外部的人群的描述是“长得难看”。好看的叫什么？长得像我，“累我”啊，“颇累我”。中国人以前都是这样的，这个是自然而然的。所有的族群其实都是这样的，整个世界上面的所有族群都是这样的，没有什么例外。就是只要他还有自己的自我意识，他就会觉得自己这个群体很好，跟我不一样的，就是有问题的。就像当年漠南非洲，第一波看见了白人探险家的时候，他觉得这帮子人得了白化病好可怜，都是病人。但是今天的黑人有好多就觉得黑了不好看，甚至漠南草原上面都有人去做那些什么漂白，这是经过后来驯化才产生的，这不是天然产生的。在中国人身上也是这样，我们并不是自古以来就觉得自己不好看，我们自古以来觉得世界上我们最好看，别人长得都丑。这是自然行为啊。只要一个群体他有自我意识，这个东西就是基本的东西，这是最基本、最基本的底色。不是独属于中国人的，而是属于全世界所有人的，任何人都有。如果一个群体你发现他对自己特别看不上，那么他必然是遭到了殖民主义的侵害和压迫，他的思想被人驯化了，才会否定自己。  

嗯，什么“否定自己的根本不就是为了超越自己吗？”是错的啊。这个想法本身是错的，这是洗脑包的一部分，这绝对属于洗脑包啊。就是一个人相信自己很好，嗯，这样说吧，我们把它替换到一个人身上去。就是在现代的心理学里面，一个人他很不自信，觉得自己不成，心理专家会怎么安慰你？告诉你“你很好，你可以的”，然后呢，说你“相信自己可以，你才能更上层楼”。请问所有的心理专家是不是都这样跟你讲的？所有研究人类的心理专家一定会告诉你“你首先相信你不错，你很好，你拥有爱，你可以获得一切，你才能够获得进步”。结果放到社会行为上面，他却会告诉你“你相信你很烂，你才会超越自己”。骗谁呢？一个群体当然是相信自己很优秀，他才有可能努力地去学习之后，努力地让自己变得更好。他相信自己很烂之后，更大的概率是：他愿意跟在别人的屁股后面当小弟——这是好的？给别人当狗这是好的？更糟糕的情况是躺在地上直接瘫了。首先要相信自己有底牌，相信自己很不错，下一个步骤才能是登峰啊。你想要攀登高峰，然后就相信自己的腿不行？开什么玩笑？任何人不可能这个样子的啊。所谓“你越不自信就越自信”，这就是他在欺骗你。他希望你不自信，因为你不自信，你必然会贪，这是他给你的洗脑包，他在哄你呢。一个人只有相信自己，你才有可能奔向更好的地方。你相信自己能登上珠穆朗玛峰，那你遇到要登月的时候你就不怵，你就觉得可以登月，没毛病。你相信那个河沟你都踏不过去的时候，他说“因为你相信河沟踏不过去，所以你必然能够登上珠穆朗玛”，骗鬼呀！
