# 第1128章 浮华皮相_仙侠古偶对传统文化的背离与资本逻辑

浮华皮相：仙侠古偶对传统文化的背离与资本逻辑 —— 《观仙侠剧有感》

血统论成仙阶梯，白衣散发竟相迷。  
修行千载无人问，恋爱灭世称传奇。  
雪洞岂是神仙府，嫡庶分明悖伦常。  
愿借葛洪丹灶火，照破荧屏万古魑。

不想拍清其他朝代的证据，也都不拍了，因为老板们不投资。老板还是那些老板，他们的目标就是在我们的传统文化当中搅浑水，然后把清给洗白，把它烘托成什么代代明君，烘托成千古盛世，烘托成封建时代的巅峰之作。他们本来就是要干这些活的，但我们又不让他干。那他投什么呢？他投仙侠。就是投那些仙侠、古偶。这几年是不是仙侠古偶特别多？看上去像是传统元素复苏，实际上，仙侠古偶和我们的传统元素是完全背道而驰的。

大家去看，典型的仙侠里面的那些价值观，你去梳理一下——现在很多仙侠类的东西出来，就是为了某个爱人就可以掀翻全世界，这玩意儿哪个地方仙？哪个地方侠？是不是？然后就是什么凤凰生的是凤凰，龙生的是龙，上仙生的是仙儿，大神生的是神。我也没怎么看，这个词我也背不下来，反正是不是纯粹搞血统论？我们传统上面的神仙是干啥的？神仙本是凡人做，只怕凡人心不坚。都是靠你一点一点地修行，一点一点地在人间行善积德、记工分，最后攒下来的，才有可能位列仙班。位列仙班之后，你还得日日考评刷KPI，但凡人对你的香火表示不满意，你都可能失去神格。我们的传统文化是这样的呀。但是你看现在的仙侠古偶，请问有哪个地方跟我们的传统文化有相似性？就是那些装扮上面有一点雷同，我都不敢说它相似。因为啥？包括梳头发，是不是个个都是披头散发的？披头散发的这种形式，中国人有没有呢？也有。啥时候？春天的时候，生发的季节，在院子里遛弯散步的时候披头散发。其他的时候都是要工整地把头发梳上去的，或者是小儿垂髫，头发没留齐的时候可以散发，其他的成年人都是要把头发梳起来的。哎，他所有的仙侠古偶都给你把头发散着。然后服装，那些服装是不是做得都跟纱窗、蚊帐似的？中国人怎么可能喜欢那样的。然后那些神仙一个一个穿得白惨惨的，住的跟雪洞一样的房子。我们在《红楼梦》里头，贾老太太去看薛宝钗的房间的时候说：“看你这一应的器物皆无，这雪洞似的，你一个姑娘家家的住成这个样子，让我们老婆子要住马圈去吗？”是吧？中国人是不喜欢这个样子的。你但凡家里头有点什么，就应该把它摆出来，让它显得有点儿东西，显得你自己有装饰的意愿，有那种追求美好生活的愿景，而不是素得啥也没剩，光秃秃的，那叫做家徒四壁，不是什么正面的辞藻。但是你看那些仙侠剧是不是这样的？都已经当神仙了，就是为了像奔丧一样过日子，那你当神仙干什么呀？是不是它完全背离了我们所有的传统的这些审美也好，价值观也好，甚至神仙妖怪的妖魔系统也好，完全都是背道而驰的。

所以你说这些老板想投资什么？他就是想在中间搅浑水。他只想宣传满清很伟大，其他的朝代他并不想说很伟大。但现在不让他宣传满清很伟大了，怎么办呢？他就把水给你搅浑。除了这些，这些还算是虚拟项目对吧？你可以说仙侠也好，古偶也好，都属于那些不太正经的，不算是正剧。那你哪个东西复原度高一点，总是有的吧？也有复原度高的。前几年的时候，是不是有一部号称服化道复原超级好的剧，叫什么来着？叫……那起的名老长了，嗯，知否是吧？那吹得破了天了。但是请问它传递的是传统价值观吗？不就是嫡庶神教——同一个爹生的孩子，嫡出的怎么样，庶出的怎么样？然后主角必须能生儿子，配角呢？她看不上就必须生女儿是吧？儿子必须能够高娶，然后别人配角生的女儿必须要低嫁。这都是啥玩意儿啊这是？然后动不动就是那个，摔了个跟头，被其他家的公子拉拔了一把，被人看见了，然后你就必须要嫁进去。男女授受不亲，事急从权——中国三千年前的古人都不带这么不要脸的，而现代的这些文案他就敢这么写。然后你说这个就是我们的传统啊？这玩意儿是在传播传统，还是在污蔑传统啊？
