# 第1151章 满服等于汉服吗_满服凭什么不能做自己

满服 = 汉服？满服凭什么不能做自己？  
服章载道，胡越同簪？指鹿混珠，夷夏堪叹。金瓯缺角，欺麟辱鸾；满裳非汉，禹甸何安！  
是什么人呢？敢问满族是不是中华民族的一份子。嗯，是。嗯，如果是，那么满族服饰能不能称为汉服？那肯定不能啊。你是不认字儿吗？你的母语有点什么问题？你，你这个母语是不是不是汉语啊？  
啊，那我就稍微体谅你一下，帮你翻译一下啊。中华民族是一个合集，它下面呢，有56个子集，其中有一个叫做汉族。汉族的传统服饰叫做汉服。满族呢，有它自己的传统服饰叫满服，也可以叫做奇装。OK，包括清民女也是满服的衍生品，满服的一部分，也可以叫做奇装的一部分。  
各个民族都有自己的民族服饰啊。你不能在那里说藏服能不能藏袍，能不能叫做汉服。你敢去问？你有这个脸？你有这个胆儿？你去问，你去问呀，你去问呀！这就是可着汉人一家欺负是吧？啊，满服就要当汉服了？你先把其他的50几个民族都说服了再说。我是无所谓的，你去把其他的各个民族全部都说服了，一下子再讲啊，然后我们再回来谈。  
大家都是中华民族的一份子。然后藏袍是藏袍，蒙古袍是蒙古袍，我都很支持。结果你说满服就是汉服，啥意思啊？满族不存在吗？你欺辱满族的同胞吗？人家有自己的传统服饰，人家有自己的传统节日，人家有自己的传统习俗，你凭什么不许满族同胞穿着他自己的民族服饰？凭什么要说满族的服饰是汉服？你欺负人家满族同胞不敢吭声吗？是不是人呀这是？
