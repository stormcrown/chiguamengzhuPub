# 第1179章 李世民是什么民族_我汉我也是什么意思

李世民是什么民族？“我汉我亦”是什么意思？  
唐朝的皇族本是东北游牧部落改成了汉姓——这事你去跟李世民说。我尊重李世民的态度。李世民说了，他是陇西李氏的后裔。我尊重李家自己的声明。我尊重当事人。我跟你们这些人不一样，你们特别喜欢给古人改户口本、改祖宗、改出生地。我不这样干，我尊重古人自己的意见。李世民说了，他们家祖上有族谱的，有出生地的，有所有的履历表的，列得清清楚楚。人家陇西李氏的后裔，那我就承认这一点。为什么？因为当事人说了他是陇西李氏。结果你这个隔了1000多年、2000年之后的人说他不是，他是东北游牧民族的后代。你自个儿去找唐太宗说，别跟我说，说不着啊。  

努尔哈赤是中国人兄弟？那这个事儿我说了不算啊。你问努尔哈赤去，你让努尔哈赤自个儿说。他又不是没说是不是？你问努尔哈赤，问他的儿子，问他的孙子，这才有准儿啊。你跟我们说这个话有用吗？对吧。他是哪儿人，他认为是不是中国人，这个事儿你得问他本人啊。自己也喊过口号，不行。你去找一找他，他儿子、他孙子都曾经喊过口号。关于是不是的问题，你自个儿去问吧。你说我又不能强行给别人发什么国籍证书，我有那个档次吗？我还给古人安排国籍，还安排不了。  

他是东北、辽宁人，那怎么的？那个英伦岛上面出来的那个赫德是广东人吗？啊，他在广东当官收关税，中国通，他是中国人不？你问问他本人啊，他在哪里出生的，跟我有什么关系吗？我觉得那个英国老师说的挺好：你在马圈里出生不等于你是马。我们今天中国的国土上面出生各种各样的人，有很多旅着游、求着学，就在这生个孩子，是中国人吗？真的就是你在讨论古人是哪人的时候，你能不能尊重一下本人的意愿？人家本人不同意的时候，你硬给他摁着头说，还找出一大堆的理由，就是欺负死人不会说话呗。  

然后还特别喜欢说那个李世民是胡人，你问过李世民了吗？李世民同意吗？上一个说李世民是胡人的那个是吧，已经死半道上了，那还是大大的宽恕之下，死半道上了。结果还是天天就摁着头，李世民是胡人呐。哎呀，你咋那么会呢？欺负人家死了不会说话对吧？人家同意不同意你也不在乎了。爱新觉罗家庭200多年，天天强调自己的身份，然后到你这儿，反正他就是那样呗。人家的身份都由你来决定，不由人家本人决定。反正我不是很能够理解这些人的思路。当事人怎么讲？浑不在意。反正我说是啥就是啥。别说像唐太宗这样的古人早就作古了，努尔哈赤也早就没了是吧，那完全是欲辩无言。就连人家当面锣对面鼓的现场反对，人家都不同意那个谁？翟崇禧吧，白崇禧，人家说了自己是信教的汉人，坚决地确定了自己的身份，他叫做信教汉人。最后呢，哎，给人家定了一个族名，给人家安排了个新祖宗，人家同意了吗？人家不同意啊，是吧。粟裕将军说：“我从来不知道我是那个啥呀，我是汉人呀。”不行，户口本上硬给你编一个新的，一点都不尊重当事人的意见。想怎么给人家安排啊，就怎么安排。人家自个儿怎么想的就不考虑。杨靖宇家里面的后人，为了证明自己家是汉人后裔，每年去给杨靖宇将军上坟的时候，都要带着猪头，带着酒水，大张旗鼓地把整头的猪脸摆在面上啊。结果你问他啥族，哎，人家给你登记一个。这就属于自个儿的意见，他不管时代的背景，他不管，当事人及其家属的意见他也不管，他想说他是哪的就是哪的。硬是说努尔哈赤是中国人，努尔哈赤同意了吗？他儿子同意吗？他孙子同意了吗？他们家后面200多年，子子孙孙那一辈又一辈的啊，一个同意了，这件事情就没有啊。他家是啥时候加盟的中国？你要说今天的吗？你是不是欺负满族人？今天的满族人是啥时候加盟的？最早的一波，1911年，最晚的一波，1949年。我没有说不可以加盟啊，但是你不要挑时间的吗？啥时候加入的呀，你不用考虑一下这个问题的？  

我得带一波货了，老不带货的话不合适。我是一个带货主播，不能整错了身份啊。今天有一波那个酒水，就是这个清酒。我天天就带酒，对，我这边直播间就是卖酒的。卖酒的直播间有的时候唠一唠别的东西。至于说赠品多还是主货多，那是另外的事。随便你冲着什么来都可以来。这个是清酒，国产清酒，江西赣州出产的。配料表非常干净，就是水、大米和米曲。清酒酿造工艺和黄酒是一致的啊。清酒的这种工艺是在西周时候出现的，这个命名也是西周时候出现的。它是一种酿造和过滤的方法，用大米酿造的这种专门称之为清酒。这专门把这个定义为清酒，那是日本整的活儿。但是呢，无论是在配料上面、方式上面、过滤上面、命名上面，它都是纯纯的正宗的中国货。这个产品也是纯纯的正宗的中国货，它是国产清酒。喜欢清酒的小伙伴可以入手了。酒体非常干净，正宗的国产清酒。清酒本来就是中国的，只是说人家学了之后呢，因为他用了工业化的方式进行推广，所以呢，大家呢，常常觉得日本的清酒做得好。但是今天我们中国的酒品工业啊，做的也很好。这个是国产清酒啊，不是日本的。这个是15度啊，酒精度数15度，在清酒当中属于非常适中的一种。口味上面也确实非常适中啊。咋感觉今天破站也在炒呢？对，因为这边出了两三个想要还大清恩情的那种呢，可能比较感恩，我也不知道大清对他家有什么恩德。万一人家真是爱新觉罗氏，占了好大便宜呢。不知道互联网上都是昵称，这事不好说啊。一般情况下呢，这种人呢，在互联网上发言的时候，“我是汉族，但是我认为啊”，我总结这个句型叫做“我汉我亦”。我也反正我理解不了。真正的汉族的话，一般不会说“我是汉族，我不认为怎么样”。因为凡是喊这个口号的，一般他就是“我是汉人”，但是后面就是反汉的内容。他为啥要说是汉人呢？他要是说不是的话，这个反汉就过于赤裸了。因此我一般不查户口本，随便悉听尊便爱咋说咋说吧。这事我赶不来啊。我难，我也我也对。就这种句型，反正我在互联网上待的这些年吧，很明显的一样东西：凡是一个人在说一句话的时候，先讲一下自己的什么特殊的身份，那不好意思，后边儿他要说的内容就是来背叛这个身份呢。他强调这个身份，主要的原因就是背叛这个行为非常可耻，他怕被人喷，他就要专门打一个logo出来。基本上都是这样。真正说你要真是这个身份，你就直说了。例如说我在互联网上怼女权的时候，我就直接怼，我不会说“我是女的，但是我觉得怎么样”。虽然我真是女的，但是这很重要吗？我自己明知道我是女的，我干嘛要说我自己是女的呀？我就直接说你这事说的不对就完事了。但是呢，会有一些人说“我是个女的，但是，哈哈哈哈”，或者“我是个男的，但是我觉得对”。民族主义者吧，对呀，我不光是民族主义者，我还是保守主义者。那你想说啥？要是想录个屏，然后出去开黑，我也支持啊。  

这个按国籍法，当事人说了不算？你说谁呀？你说的这个当事人是哪位？你知道国际法是哪一年出来的吗？国际法出来之前的人你要管是吧？当事人说了不算，为啥呢？因为有国际法。国际法是哪一年的？你自己去问一下度娘，哪一年的时候才出现的国际法。然后你就拿这个玩意儿又去管了一下那二凤，又去管了一下努尔哈赤，还得管一下忽必烈，再得管一下铁木真是吧。你得挨个地谈呗？不行啊。国际法规定了他们是不是？他们从出生到死的时候，都没见过国际法这玩意儿。国际法啥时候出来的呀？真的，你说话做事从来不考虑时间问题，咋琢磨的呢？我就理解不了这种脑回路，知道吧。为什么说某人是中国人？因为我们960万平方公里土地。哎，我们960万平方公里土地是哪一天划下来的？你在讲历朝历代的土地的时候说啊，以前就那么一点儿，一点点长大的。那时候你就知道我们的土地范围是在变化的，人是在迁徙的，不断地动的，对吧？按说以前我们还迁都了，人还移动了。对，你咋都知道呢？明明知道人是会动的，时间是会变化的，然后就说不行，我们今天960万平方公里土地上的，历史上曾经喘过气儿的，曾经经历过的，曾经在这里当过一回的，所有的人全部都是中国人，全部都要依照国际法进行管理。对我来说，你这个话题超纲了啊，聊不下去。
