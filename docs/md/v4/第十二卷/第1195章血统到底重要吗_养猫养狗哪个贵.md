# 第1195章 血统到底重要吗_养猫养狗哪个贵

血统到底重要吗？养猫养狗哪个贵？#宠物  
蒙古和满洲都是一群人，有什么血统之分呢？你要说血统的问题，其实就是看哪个群体在乎血统，哪个群体就能保住它。但首先，你得在乎这件事。而现在的情况很有意思：一方面，政治正确要求你不准在乎血统——你在乎血统，你就是纳粹，你就是小胡子，你就不正经；但另一方面，这些人又天天说你这个人血统不纯。我也不知道他们到底在乎不在乎。他们说“不准在乎血统，在乎血统的都不是好人”，然后又说“你这种汉人，你们都是串儿”。他们自己脑子里在打架，我都理解不了。哈哈哈。对于我这种人来说，你前句话和后句话的好坏，我不分也不管，你起码得逻辑通顺吧。这些人身上最大的毛病就是逻辑不顺畅，这个不顺畅是我理解不了的那种不顺畅。

养猫养狗也不在乎血统？嗯，农村土狗也挺好的。你有没有意识到一件事：农村的土狗其实也是有血统的。我说认真的，没跟你开玩笑。你认识的大黄、大黑都是有血统的，你知道吗？不知道吧。重点就在这儿。他们常常会说我们的那些东西没有经过育种。但你有没有考虑过：我们作为一个四五千年以上、甚至可以说倒推一万年的农耕民族，对育种这事儿是超级在行的。一个特别擅长育种的群体，我们饲养的狗高度相似，你说它是串儿，合理吗？我们去看唐代墓葬里的狗狗陶俑，和今天的大黄长得一模一样——没上颜色，但款样、五官、身体结构、尾巴上翘的形状、毛笔的方向，全都一模一样。一千多年的时间，外貌保持一致。你能说我们的乡村土狗不在乎血统吗？有时候他们跟你说“这个事没有啊，这里都是串儿，我们这里的品种狗才有血统证书”。对，我们的土狗没有血统证书，但不代表它没有血统。

宠物和动物方面，我们的牲畜血统带来的是什么？为什么会进行这方面的育种？为什么说它有血统？因为在育种培育的过程中，它要符合一些非常基本的杂交选育原则。这个原则是：一个品种通过杂交可以选育出你想要的特点，但选到合适的特点之后，必须进行纯化，让它变成新纯种。这样它的特点才能稳定地传递下来。不然的话，杂交品种到第二代时会出现严重的性状分离，和母代或父代之间毫无瓜葛，变得非常混乱。所以，所有能够延续到今天的动物或植物，这些老品种都是有稳定传承的。只要有稳定传承，它就是有血统的。

在城市里养猫狗很烦，开销也大。其实养猫的成本不高，猫不怎么花钱，在城市里也能养。但养狗的成本确实比较高，因为狗需要相当大的运动量，你必须得遛。出去遛的话，必须拴绳，按规矩还得给它嘴上罩个口罩，这才是安全防护。狗毕竟是灰狼亚科动物，它忠于主人是基于自身群居生物的一种服从属性。不是所有人都会训狗的，有些狗根本没训好，对人缺少服从性。结果你拿出来说“我家狗狗多可爱”，那肯定不行。所以养狗需要负担更多的责任。养猫的成本相对就低一点。我自己也养猫，因为猫不需要遛，在家里的一个屋子就够它玩了，需要的场地也不大。但如果你要养得特别精细，那花销当然无上限。但凡你像养一只正常的小土猫、田园猫，像我的话开销很小，不挑食，还很健康。高层建筑不建议养猫？对，如果是高层建筑，一定要把门窗封好。因为猫有个毛病：它不恐高。一般人不会飞，所以恐高，但猫不会飞却不恐高，所以特别容易出事。养猫的一定要关好门窗。城市里尽量别养狗了，养狗成本太高，需要相当一片地来运动。狗每天的运动量必须足够，少一点它必然拆家，因为它就是那么活跃。住别墅就可以养狗了。行，啥时候去整个别墅，你就可以养你的小狗狗了。

有人说猫的智商不如狗？对，它脑子才那么一丁点，想让它跟狗比智力有点难。但猫架不住它可爱呀。我养猫是因为它聪明吗？不是，是因为它萌。它萌是因为聪明吗？不是，是因为它毛茸茸。城市里养猫，猫运动量会不会太少？你在家多设几个猫爬架，每个地方都给它安排登高爬低的小台阶，它日常在屋子里跑酷的量不会少的。但凡它软塌塌地瘫在那儿不动弹，你也没办法，抱起来扔上去让它运动，它转脸又跑了，跑去躺着。那是它自个儿懒，管不了。养两只猫互相追逐，运动量就大了？不是所有猫在追逐时都会跑。动画片《猫和老鼠》里它跑，但有些猫真懒，是真懒，不管干什么都瘫在地上。不过那是少数，大多数猫还是正常的。

“无孩爱猫女”适用于中国的情况吗？中国也存在这样的人。这个定义虽然说叫“无孩爱猫女”，其实跟你养不养猫、有没有小孩没关系。核心点是责任感的问题。“无孩”是不为未来负责任，“爱猫”是不爱人、不为人、不为社会公众负责任。相当于一个自我中心主义者，只为自己好，身边的一切都不作数。既不想将来，也不想群体，也不想社会，完全是自私自利的人。主旨就是这个。在阿美那边，这个定义基本界定为“无孩”。但在中国，问有没有这样的人？那肯定有。她养不养猫，也是这个样子；养不养孩子，也是这个样子。核心点就是自私自利，不看未来，不看身边，只考虑自己。这种类型就可以叫“无孩爱猫女”。有点像那种精英女士？好像通常这类人喜欢说自己精英，是不是真的精英我也不好说。我认识的一些比较精英的女性其实还是很随和的，各种类型都有。但互联网上那些精英——非常自私自利、眼里看不见别人的——我日常生活中没见过，但也许也有吧。我个人觉得他们那种精英是表现出来的。

这个艾草贴，贴肚脐的。这个贴肚脐的是我的小伙伴给我的建议，他说这个产品特别好，他用过，用完之后身上暖洋洋的，很喜欢。这是艾草贴，浑身热腾腾，循环变好。所谓“驱寒湿”，是他们号称的作用。为啥要这样说？因为平台不让介绍功效。我也奇怪为啥不让介绍功效，说不清楚。冬天贴的吗？啥时候都行，看体质，看你是不是需要，祛寒祛湿。看体质，不分时间。
