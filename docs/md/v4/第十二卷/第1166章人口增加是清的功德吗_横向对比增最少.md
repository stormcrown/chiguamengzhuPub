# 第1166章 人口增加是清的功德吗_横向对比增最少

清代的人口增加，就算他的功德了是吧？嗯，好的好的，我们就来再来讲一下清代的人口增加的问题。你在聊清代的人数的时候，首先要聊的是明代的人——明代的人丁到底有多少？实际上，由于明代的管理制度的原因，它并没有进行大规模的人口统计。在其立国之初，曾经梳理过纳税单位，就是所谓的按照丁户来进行计算的纳税单位，做了鱼鳞册进行了统计，各个县级的官衙都有存档。但是在这之后，大家就按照这个账册直接来收税就好了。所以一直到明末，没有进行过再次的人丁统计。因此，对于明代的人丁数额，是没有官方数字的，这是第一件事情。

然后第二件事情，明代有一种东西，大家一般情况下会把它叫做人头税啊，虽然它并不是按人头来收的，它是按照劳丁来收的，是一种丁税。本质上呢，是一种徭役的转化基因。原本应该是说每一个地方的劳工——就是成年男性的这种丁口——每年有一定服劳役的义务。什么大清3000年翻四倍？大清没有3000年，大清的300年翻了四倍。我会讲到的，而且会告诉你四倍是个什么概念。

然后在明代的时候，由于丁税就是这个丁瑶银的收取，使得整个明代并没有真正意义上的人口统计，因为我们实际上既然你要按照丁来收钱了，那么藏匿人丁就变成了一个常见的操作。尤其是在一些大户上面，会出现大量的隐丁啊，这已经是逃税的一种方式。就是引丁、隐匿丁口、隐匿田地都是一种常见的避税法。所以明代的统计数据呢，基本上你就没法看，完全不准。

但是呢，现在的学者依然找到了其他的契机去进行人丁上面的人口上面的统计。明代的人数到底有多少？他会有几种统计的方式：一种是通过田亩数，就是它出产了多少的粮食。粮食这东西，一直都在我们历朝历代的史书当中占据非常重要的地位，他一定有这个科目，就是每年你都要登记各个地方的纳税、各个地方的量产。所以根据田亩数、根据粮食的产收，会进行人数的倒推。同时呢，会根据盐——盐也是统计人口很重要的一个东西——根据盐的消费量来进行统计。这是两个统计的口径，还有一些其他的旁证。通过这些主要证据和旁证的互相交叉印证，有学者对明代的人口数量进行了计算和推演。在这些推演当中，由于它统计的方法和计算的方法不同，会有很多的口径。数额较低的口径是在大概1.6到1.8亿，而比较宽泛的口径达到了2.6到2.8亿。所以呢，比较主流的选用数据的一些人，通常会比较折中，选用比较居间的一种方式，去选取了这种大约2亿多一点的这种口径，来计算明代的人口数量。所以它的人口数量大约是在2.2亿左右，这是一个比较常态的估计法，是有证据的估计，不是在胡说八道的那种猜想。这个是明末的人数。

但是呢，清初的时候也同样是为了收税，清初的时候又一次进行了统计，当时的全国能够算得上人口的人不足9000万，不足9000万。OK，所以目前可以先把这两样东西先对照一下：清初的时候他进行了一次专门的统计，不足9000万；而明末根据多方学者的交叉印证，人数是在2.2亿或者以上，我们也可以说它在2.2亿左右。所以中间那一个漫长的时间的一次大动荡，实际上带来了人口上面过半的折损，这是第一件事情。

然后第二件事情，有一部分的口径当中，认为整个清代人口膨胀了四倍，就是从1亿走到了4亿，然后又说这个是一个大功德，从来没有这么多的人。OK，那么我们来看一下，这个人口是为什么会出现这么多啊？第一呢，就是他在开国的时候，这个人数的数量本身就不算小，无论是9000万也好，还是1亿也好，在当时的世界上都是非常庞大的一个数字。而且那些人是会耕田的、有技术的、会生产的、懂联合的人啊，并不是生番，是一些非常熟练的工匠和农夫。这样的人他有养活自己的能力，这是第一个前提——就是有1亿的优质人口。第二件就是，其实有相当一部分所谓大航海时代带来的高产的作物，实际上在明代的时候就已经开始推广了，不是清代才推广的，是明代就开始推广了。从明代开始，我们就已经开始推广大航海相关的这些高产的作物。至于大航海是谁做的，那是另外一个的话题，我们把它略过去。

然后呢，由于这些高产的作物——尤其是像红薯和玉米这些对土地的挑剔呢没那么严重，相对来说在贫瘠的地方，甚至是零敲碎打的地方也能够生长的作物——提高了大家的可能获得的平均口粮数量。数量上面是有提高的，但是我们一定要注意另外的一点，就是质量上面有所下降。无论是红薯也好，玉米也好，它在提高产量的同时，其实降低了质量，降低了我们摄入的碳水化合物的品质，这是一件事情。然后我们的人为什么会滋生人口？就是你能吃上多少饭，就能够有多少人。所以在清代的时候，出现人口的增长是很正常的一件事情，因为农民可以利用更多的贫瘠的以及边角料的田地，去种植一些高产作物，使得它能够维持一个半饥半饱的状态啊。

然后呢，另外的一个原因就是政策上的原因。清代的时候提出了摊丁入亩。摊丁入亩的这个东西呢，是他的首创啊，明代的时候就有人提出来了，但是没有能够推行下去，清代的时候推行下去了。至于为什么——你反正你不从你的人头就交上来，他什么都推得下去。清代的时候把这摊丁入亩推进下去了。那么摊丁入亩是干什么？就是我们取消了丁瑶银（丁税），同时呢，这笔税呢，折算进了田亩的税收当中，用地产来去交税。但是它造成了另外的几个问题啊。先说好处吧：好处就是你隐匿丁口没有用了。所以在清代的时候，因为丁瑶银的取消、摊丁入亩，所以呢，我们在统计人数的时候，相对来说会更加的真实，因为不需要隐丁了，自然都相对真实。但是另外的一方面，又有一个新的问题，就是它的赋税增加了，所以他的赋税严重地增加了。因为原本的丁瑶银是拿来由当地的民夫是交的地税——丁瑶银原本是地税，本来是民夫，应该去给县太爷牵骡子、牵马，给他修修板凳、修县衙、修城墙、修城门，这种劳役的一种税。等到了清代把摊丁入亩推行下去了之后，要知道田税是直接进国库的，也就是原本是收入地税项目的这个税目就直接进到了国库里面去，这就导致了地方上面严重的欠缺了资金。那怎么办呢？就允许你重收一遍。所以事实上就是他并没有取消某种税啊，而是这个税额呢，事实上就加倍了、增加了。然后说那个有田的才交，没田的不用交。但是没田的也得种田，因为种田也是他就业的一种方式。所以呢，这个过程当中，农民的田租又提高了，负担再一次的提高，这是它的负面影响。那么正面影响的就是我们在统计人口的时候，不会有人藏匿了。所以呢，清代统计的人数会比明代的更多，因为它没有隐丁的现象。

然后另外的一个项目呢，是到了乾隆年间的时候出现的这种现象，应该是之前就有，但是乾隆皇帝呢，就把这个事指出来了——就是有大量的官员虚报人口。因为不收丁税了之后，那个人数啊，就跟你的税收报账呢没有关系了，你不会因为我报了特别多的人，所以就要交特别多的税。这个责任没有了之后，那个当地的人口是否繁茂，就变成了当地的官员的GDP上面很光鲜亮丽的一笔。因此有大量的官员、官吏啊，就喜欢虚报人丁，因为不用交税嘛，想报多少报多少，报得越多显得他的成绩、业绩就越好——这个地方人口能够增长，就说明这个地方生活过得好吗，这是一个直观的KPI。所以呢，乾隆皇帝就发现这个事呢，天天这么虚报，太离谱了，就专门申斥了一下，才把这个不正之风摁下去。所以，清代的人口统计当中，包含了相当一部分的虚报人口。

所以呢，有关于清代人口的增长，主要的原因就是这几项：一个呢，是取消了丁税；一个呢，出现了虚报的人口；另一个呢，是高产作物的引进。这几个因素共同叠加，造成了人口上面的爆发。其中物质上面的原因就是高产作物的推广。那高产作物是指在中国推广吗？不是的，在全球范围内，只要有人经过的地方，其实都推广了高产作物。也就是说，高产作物造成了全球的人口增长，而不是单独的中国人口增长。那么在东南亚地区，我们周边的地区，别的国家是什么样的情况呢？我们可以做一下横向的对比。当时能够辐射到高产作物的这些地区，它的人口增长的比例呢，通常都是在7~10倍左右，这些是有人做过统计的。虽然不是我亲自算的（我数学不好），但是呢，是有相关的数据的。就是在全球范围内，所有接受了大航海恩泽的这些地方的群体，都实现了人口7~10倍的增长。而入关的满洲群体，在他的200多年的统治过程当中，也实现了七倍以上的增长。而这片土地上面，最擅长耕作、最擅长经营、最擅长囤积物资的汉人群体，实现了四倍的人口增长。请问是多了还是少了？请问它是正向的贡献还是负面的贡献？而在这个人数增长的同时，汉民族的平均身高由明末的1米7左右下降到了清末的1米65（男性啊，1米65）。尽管说在老照片当中，你依然看得到他的身高高过于相当一部分的列强侵略者，但是那是因为欧洲的殖民者太矮了，而不是因为中国人太高了。中国人当时的平均男性身高1米65，但是明代的时候平均身高是1米7。经过了200多年的繁盛的盛世啊，中国人的平均身高下降了五公分。只有持续不断的饥饿，才会造成一个群体的平均身高下降。
