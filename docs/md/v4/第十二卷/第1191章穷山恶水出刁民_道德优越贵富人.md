# 第1191章 穷山恶水出刁民_道德优越贵富人

穷山恶水出刁民，道德优越贵富人。#隐入尘烟

嗯，我前两天在其他平台就刷到一个关于《隐入尘烟》这部影片的影评。但是我看完之后，觉得这部片子对我们中国农村这边，感觉是抱有很大恶意去拍的——这是第一个想法。后面又联想到，比如现在我们有些电视剧，它主角配角的人品，是根据经济实力和他们的阶级地位来分配的。像有些男女主，可能他们出身高贵，人品又好，又特别善良；但相反，一些恶毒的配角，可能出身贫穷，素质不高，品德又低，还不择手段。然后我就想，是不是跟我们现在的某种风气有关，比如类似“穷山恶水出刁民”这种说法。唉，这个问题到现在有点积重难返了，而且也不只是在影视这一个行当里。

在我这里蹲得久的人都知道，我其实休息的时候喜欢看点小爽文。不知道有多少人跟我一样有这种臭毛病。我就发现现在的小爽文，里面的角色安排已经和10年前完全不一样了。过去的爽文，会找一些贫穷的、无奈的、身无长物的人，让他升级、打怪、成长，然后大家就觉得：“哎呀，厉害了，我也行了。”但今天不一样，现在的很多爽文已经开始占大小姐的生态位了。我这几天看的短篇小爽文里，已经出现了很多什么“卑劣的贫困生”之类的情节。所以这种丑化穷人的倾向已经非常鲜明。我相信任何一种现象的出现，原因都是复杂的，不会只有一个单面性。

第一呢，这些年客观存在一个现象：既得利益的群体试图要让自己道德高尚，形成这种道德碾压。这种现象不是今天才有的，历史上大部分时期都是这个样子，这不离奇。大部分时期都是这个死相。所以我们古典时期的很多文艺作品里，也经常是公子哥配小姐，经常是王侯将相。（喂喂，你下来啊！这个啥？不是猫咪直播间串场了，串场了。来来来来来，喂我，我怕它挠门，哈哈。）然后这种现象自古就有，因为既得利益群体总是要给自己脸上贴金，地主老爷总是要说自己和善的。实际上，在过去的几十年间，我们强调劳动人民身上光辉的人性，这才是历史进程当中的例外。过去的几十年，才是我们漫长历史岁月当中的例外。实际上，这种“穷山恶水出刁民”的文艺塑造法，自古都有，而且中外皆同。

应该说，如果我们还坚持着自己的社会主义建设方向，那么后续应该做的一件事，就是重启文艺创作座谈会。因为没有一个专门的约束，这些文人也好，创作者也好，其实他们跟我有点相似——我们做这些纯粹的创作流的东西，都需要金主爸爸。全世界所有做创作的人都需要金主爸爸。我做直播的时候，在座的各位就可以称得上是我的衣食父母。郭德纲当年为什么喊这个号？因为他是实话。所有做文创的人都需要金主爸爸，你不是站这一边就是站那一边。而当你进入的行业需要大体量的资金流投入才能运转时，你的金主爸爸就是资本家。那么资本家的大小姐就必须是最漂亮、最善良的那一个，大小姐身边的丫头就必须是居心叵测的那一个——这就变成了一种刚需。现在我们在整个文艺创作界，这种倾向非常鲜明。《隐入尘烟》就是其中的一个典型。它在假设一种中国的农村。你说它里面的那些场景有没有可能存在？如果作为细节来讲，这些场景都有可能出现。但是它把所有的场景全部捏合在一起，拼凑在一个家庭身上的时候，就错了。因为民间，尤其是穷山恶水的群体当中，除了那些扯皮拉筋、鸡毛蒜皮的事情之外，还有一种被迫的乡村底层互助——这个东西是必然存在的，因为乡村底层不互助活不下去。这部片子完全忽略了底层互助的本源的人性光辉，专门强调其中可能存在的勾心斗角，并且把它放大化，这就是一种污蔑和栽赃。这和我们现在看的短篇小说里那个居心叵测的贫困生是一个原理。

我知道在这种连麦的场景下，应该让对方多说话，但这是我的毛病，是我的问题，不太好管，就是有点停不下来的那种。我不是故意的，我属于张开嘴就这样的那种人。嗯，这边带清酒啊。哦，我知道，我看着了，我看着了。现在互联网上说这女的带清酒，她不是好人。哼哼，这些人都有一致的毛病。清酒是周代产生的技术，是中国人一直使用的酿酒技术，是一种酿酒法过滤技术。脚盆（日本）跟我们学过，后来推广的时候也推广过这种清酒的饮用。于是，既然人家脚盆用了，中国就不能用这玩意？那是脚盆的文化代言人啊，是吧？哪怕它是国货，哪怕它是国产的，他们也不在乎。反正你不行，你沾了脚盆的光了，你就是媚日了。但缝到了樱花上面就不一样了。樱花是脚盆人用脚盆本岛的樱桃植株培育出来的专用观赏型花卉，而且凝聚了脚盆人自己的审美情趣，甚至包含了军国主义的态度。然而，既然中国的土地上也种了樱桃花，那么不好意思，它就是中国的。自古以来谁说它是脚盆的，我跟谁急。这些人就喜欢这样颠倒——是中国人自己的东西，但凡别人那里有一点点，那就不是中国的；外国人的东西，但凡中国人这里有一点点，那就是中国的。这是在搞什么？这是在搞文化置换啊！还有裤衩原产地说是中国？哼，就说猕猴桃的原产地，就算按他说的原产地是喜马拉雅山麓，就算这个原产地好了。什么时间？中国的设立是有时间的，你别听他们在那里瞎咧咧。就包括在喜马拉雅山麓的这个地方，什么时间？百万年前？那个时候你搁哪呢呀？中国搁哪呢呀？中国人搁哪呢呀？你知道吗？不知道吧！谁也不知道啊！那个时候有什么中国？几百万年前有中国吗？就没有，好不好！在百万年前原发于此处，然后就扩张到了全世界。在中国出现之前，它就已经扩张到全世界了，然后你说它是中国的？真的是，这才叫做“宇宙中心在中国”呢！这个时候就不说自己是一个“思密达之心”了啊？然后硬要蹭，硬要说这玩意就是中国的。不好使！哪怕是1亿年前，在中国这个地方，它也是中国人？要脸不要啊？胡说八道！凡是做这种主张的人，其实就有一种目的——他要把脚盆的审美嫁接在中国人身上，要把脚盆的文化嫁接在中国人身上，实现文化置换、文化换家。
