# 第1132章 清史之镜_叙事立场与学术困局

# 清史之镜：叙事立场与学术困局

## 《青简悬镜篇》
董狐血溅春秋简，  
司马魂销汗青编。  
内亚巧设割席计，  
中原暗藏断根渊。  
廿载修史立场乱，  
百年辩理逻辑偏。  
愿借秦镜悬紫阙，  
照尽奸邪正坤乾。

我们来看国外的新清史。往好处看，往坏处看呢？先往好处看的话，它的逻辑很顺畅。新清史有一个优点就是逻辑顺畅，有一个缺点就是心眼很坏。这就是新清史的特点。

新清史是一种典型的内亚史观。可以说，它把自己的屁股放在了内亚民族——也就是布林海这些所谓的北方游牧、渔猎民族——的板凳上。它屁股坐在那边，然后来看待中国史，就是站在这种角度。所以，你要说它坏吗？它就这样坏，因为它是站在非中国或中国以外的地点来看待这个事。它只考虑其中的一小部分人，其他大部分的人就不考虑。它站在那一小块来看。

整个历史叙事中，大家都听过“新清史认为满蒙非中国”这个说法。所以，如果你相信它是“满蒙非中国”的人，都是被它骗了。它喜欢整这个活儿。其实说这个话的也不只新清史一家，全世界有相当一部分人说过这话，也不是它的原创，从南到北、从中到外都有人扯过这个犊子。这并不重要，重要性根本不在这儿。关键在于：任何历史叙事都要有主语。主语是谁？你的板凳坐在哪儿？你用谁的身份去看待什么事件？这才是最重要的。

新清史的特点就是站在外国视角看中国，站在内亚视角看中原。它就是用这种叙事方式。虽然中原的人最多，虽然中原发生的故事最多，但它的板凳是坐在中原以外的。因此，在它的叙事当中，中原就变成了一个任人欺凌的物件儿，谁拿到了就算谁的宝贝物件，已经缺失了中国视角——中国人不重要，中国人看法也不重要——因为谁抢到了就算谁的。

新清史的逻辑中包含了很多这种东西。当一个历史叙事包含了这种内容时，它必然会对这个国家有巨大的危害性。既然谁抢到了算谁的，那再来一个列强，我们怎么办？这就是新清史一块非常糟心的东西。

但同时，新清史也有它的优点。它这么糟心，为什么到今天还有人专门把它当做一生之敌？能够配得上当敌人，而不是拿去当擦脚布，原因就是：新清史虽然屁股坐歪了，但它的逻辑很顺畅。它的屁股一直坐在那个地方没动，始终坐在那个地方，所以它的历史叙事逻辑很顺畅。

为什么国内这边在修清史的时候就整得不行？也是因为这个原因。国内的清史研究学者，原因很复杂，有很多方面的原因，其中也包含了利益方面的原因——包含了利益方面的诉求。例如，有些学者希望清史再多修几年，要是修一两百年，他就能拿一两百年的工资，因为他还有徒子徒孙、还有儿子孙子。所以为了把修史这件事“子子孙孙无穷匮也”，他也有这个动机，就拖着整，专门不肯合你的心意。这种动机也是存在的。

但是，更大的问题是：在国内修清史的这些学者，他屁股有问题。这个屁股，还不是说到底是坐在清那边、还是坐在汉民老百姓这边、还是坐在西南少数民族这边，而是他一会儿坐一个板凳。我们现在修的这套清史当中最麻烦的东西，就是它一会儿换一个板凳。你一发现说那些讲述特别残酷、特别让人崩溃、看了就会产生那种“哎呀糟心玩意儿”情绪的内容时，它的叙事就变成了满清朝廷，它站在满清朝廷的立场上来讲，这段历史就变成了“大势所趋”非得这样做不可等等，把受害人的痛苦完全忽视掉了。但是呢，一谈到什么外国人怎么怎么了，它又一下子一屁股调转，坐到了老百姓这边来，什么老百姓要和谁谁谁站在一起。总之，整个国内的清史研究学者根本就没有讨论出一个能够进行历史叙事的整体历史观——没有。

今天有人把这种没有历史观称之为“团结史观”：凡是遇到有可能谁干了坏事，立刻就说那不存在的、没有坏事、大家是融合、是好兄弟，于是这个事就讲不明白了。所有事情只要遇到矛盾，它立刻就把这个屁股左边坐一下、右边坐一下，左边又坐一下、右边又坐一下。所以这个东西修不出来，整个逻辑顺序都是颠倒的。

因此，我们国内的清史研究学者和国外的新清史研究学者在打嘴仗、搞学术辩论时，次次都输，而且输得特别丢人。主要原因是逻辑特别乱。别人是很坏，你也看得出来他很坏，人人都看得出来他很坏，但别人逻辑是顺的。结果咱们这边的学者颠三倒四，次次辩论次次输，简直是脸都丢到地上、丢得没有了。

但凡真看过这段历史……哎呀，他们真看过。但是呢，他们又想要让这个事情一团和气，所以这个东西就讲不明白了。这些学者并不是资料有问题，他们每一个人手上的资料都比我多，他们每一个人背下来的历史事实、历史人名和所谓历史线索都比我多，不知道多多少倍。但是，想把这些事情串起来，是需要有一个主语来进行叙事的。你要把这个事讲明白，得知道板凳应该摆在什么地方、屁股坐在哪个板凳上面，然后才能把这个事串起来讲——不是把所有菜往那一丢就完事的。而这个板凳他坐不清楚，有多少板凳他就每个板凳都搭一个边，所以稍一动作就摔了。他们没有视角来讲，能怎么办呢？确实没有。所有资料往那一丢，是原始档案。对，原始就是一堆原始档案往里头一码就完事了。但是，这个是明代修《元史》时根本就不想修好造成的，就装了一个封面，别的啥都没整。

虽然乱，但真呢。原始档案因为是原始档案，所以比较乱、比较杂。同时，由于元代并没有认真修饰，修得很草率，所以里面的修史东西、自己的原始档案也是一团乱。尤其是修史过程中必须书写蒙古人的人名，但蒙古人名重名特别多，高度重名——动不动就这个人和那个人重名了，同一个朝堂上能有好几个什么铁木耳之类的，乱七八糟一堆全是重名。重名就算了，关键人家重名重的是蒙古语，然后你把它写到史书上时又做了音译，所以一个人可以翻译成好几个名字，同一个名字又有好几个人。所以《元史》是很乱很乱很乱的。但是《元史》没有叙事角度的问题，因为它就是一个原始档案。原始档案的叙事角度就是站在元朝廷的角度上讲的，它的叙事角度非常清楚，所以在里面讲朱元璋时直接归到“贼”里面，讲到朱元璋的队伍时直接归在“贼”这个队列里，连字都没改。它的叙事没有角度问题，叙事视角没有问题。只是我们在做阅读理解时问题特别大，那就归学者管了，我也管不着，我看那玩意儿头疼。

所以之前那个，确实我们现在要修那个清史已经修了好多年了，一版一版出来，又一版一版打回票，反正就是修不出来。我觉得修不出来是件好事。历史观没有调清楚的时候，你修这玩意儿干什么？叙事角度没选定的时候，这个史书修出来就是拿出来恶心人的，最好别修出来，完全不需要它修出来。

其实当年我读书的时候，有一个好朋友，他很喜欢读《清史稿》。要我说，为什么最好清史修不出来呢？因为当你的叙事主体不能确定时，一本史书一旦叙事主体和叙事角度歪了，它对人产生的影响非常大。我这个朋友，他把《清史稿》读完了之后，唉哟，爱满清爱得要死要活的。那些明君贤臣、后妃，简直了，又清廉又明智，聊着聊着天就恨不得要开始山呼万岁。所以你就说，但凡我们修了一个东西，叙事角度就跟《清史稿》似的，那你还不如别修出来。最后修完了，念过它的人都会变成包衣奴才。
