# 第1173章 宋朝虽弱_依然是文化宗主_死于安乐与今不同

宋朝虽然军事上偏弱，但它依然是文化宗主国，经济繁荣，最终死于安乐——这和今天的情况不太一样。  
有人提到晋、宋、清是“耻辱三部曲”。但宋朝不至于如此。晋朝有内乱，自身不正导致乱象严重；但五胡乱华之后，靠同行的衬托，东晋反而成了正统。所以评价一个朝代是否耻辱，要看当时横向对比的水平。宋朝肯定算不上。它的经济、科学、技术、艺术、文化都很繁荣，没有丢掉文化宗主国的地位，周边的藩属国依然在学习我们的文化、艺术和审美，我们仍拥有话语权。宋朝没有跌到那种程度，但到了清朝确实已经什么都没有了。  

（插一句：咦，这个气泡米酒大家不感兴趣吗？挺好喝的呀。这是支持一下，支持一下。这个送酒还是很好的——就是那种气泡米酒，像汽水一样喝的米酒。呵呵呵，这个想法还是蛮可爱的。）  

“哪个宋？西湖歌舞几时休的那个宋。”那个宋，要说它威风八面，谁都不信。但论富庶、论文化昌明、论饮食文化的繁盛、论玩花活的能力、论享受的水平，宋朝一点都不差。所以“送酒”没什么问题，因为确实是宋代就开始流传下来的酿酒技艺和菌种，那个酒曲也是宋代留下来的，现在好像已经登记了非遗目录。反正总之吧，这个系统确实跟宋代有瓜葛——就是西湖歌舞的那个宋。哈哈，嗯呵呵。宋酒很出名呢。嗯。  

说宋朝“打架不行”其实也不对，打架其实是行的，但他们不敢打。为什么不敢打？因为太富有了。光脚的不怕穿鞋的，但穿鞋的很怕光脚的。他们太富庶，条件太优越，生活太好了，所以社会的中上层非常软弱。对他们来说，那点岁贡算什么？那种支出简直洒洒水。到了这种程度，他们就特别容易认怂，因为他们觉得自己是瓷器、是玉瓶，而对方是粗瓷瓦罐。不是战力不行，不是没有技术，不是没有器械，也不是没有兵员，更不是没有良将忠臣——所有要素全都有。甚至说没有马场，但宋朝实现了养殖场的培育方式，虽然马的质量不好，其实能养。为啥不养？拿来养羊了——养好吃。宋代已经有了高密度养殖的技术，但人家不养，你佩服不佩服？  

宋朝之所以在军事上失利，很多时候跟它财力过于丰饶有关。出征一次的军费太高了，但称臣纳贡那点东西——谢谢正常的网民——对它是九牛一毛。而且前脚把岁银交出去，后脚就跟对方进行贸易，全部翻几倍赚回来。当时宋的技术和生产水平非常好，所以能在与北方各国之间进行贸易（包括走私贸易和边境互市），赚得盆满钵满。在这些岁贡的事情上，它完全不在意。这个国家为什么亡？叫做“生于忧患，死于安乐”。所以，国家有强敌在外、有威慑在外，是一件很好的事情，不然醒不了精神，意识不到自己的存在。  

但是，把敌人养肥了——对呀，这就是它自己卖了自己的上吊绳。南宋的做派就是自己出售了自己的上吊绳，对方手上的装备都是它自己卖出去的。嗯，跟现在会不会有点像？不像，我们现在是很有危机感的，而且在国际上有明确的敌人和对手。你看我们直播间里为什么不亲近？因为敌人和对手在这个事情上，和我们的一些内奸勾连起来了。所以，他们在这个地方，看上去是在喊一些政治正确的口号，实际上是在刨我们的祖宗坟，在刨我们的国之根基。  

所以，我们有外敌，也有这些搅浑水的人、左右横跳的人，身边有这些汉奸走狗、卑劣无比的人士——这是一件好事。这些人的存在，我们才能够知道“我是谁”。如果周边全是朋友，每一个人都花团锦簇，你意识不到这个事情。但哪怕在直播间里，你都能看到多少品格卑劣的人在这里颠倒黑白、故意篡改逻辑、刻意扭转他人认知、刻意篡改历史记忆、搞历史虚无。公屏上这几句话就是这样搞起来的。所以有这样的敌人，我们才能意识到我们是正确的——这很重要，非常非常重要。不用害怕敌人的出现，他们出现就意味着我们可以进入战斗模式了。  

而这件事，对今天的中国来说很要紧。我们不会像宋代那样——虽然我们在技术、生产力、经济实力上都起来了，但不可能像宋一样。我们到目前火力不足恐惧症都还没有治愈，依然在感觉周边各种各样的威胁。有威胁感就会自立自强，就不会蝇营狗苟。但我们国内确实已经出现了蝇营狗苟的东西，就像公屏上这些搅浑水的人。有人担心身边都在安享太平，其实我们现在的局面没有恶劣到全民皆兵的程度。你怎么可能在身边看到好多斗志昂扬的人呢？正常和平的时代，周边的人都在想办法继续过日子，这才是一个常态。一个国家在和平期，如果每一个人都天天亢奋，那是不对劲的。绝大多数人都是过日子的普通人。  

但我们这里是有整体性的。所以不用太担心这个问题，中国目前没有这个问题。有和平的时候，我们就过和平的生活；只有敌人出现在面前时，再去抗击敌人。并且心里要有数：这世间是有敌人、有对手的，是有品性卑劣的人的。有人说“在舒适的摇篮里，由成年人变成了婴儿”，但你没那个机会，放心。这世间永远都在变化，不可能永远有这样的安于享乐的摇篮，没有。
