# 第1113章 历史无对错_衣服没有对错_和稀泥就是想颠倒黑白

历史无对错？衣服没有对错？这话你信了吗？和稀泥就是想颠倒黑白。#历史真相 —— 青史斑斑血泪凝，是非岂可作烟澄。赵高指鹿犹能辨，贾谊过秦尚可征。十二冕旒遮暗日，三千铁锁缚明灯。休言胜败皆天数，自有春秋镜似冰。

历史没有对错？衣服也没有对错？哼，那对呀。所以历史没有对错，所以我们战败也是对的，战胜也是对的，是吧？我们贫穷也是对的，我们富有也是对的，我们好看也是对的，我们丑陋也是对的，我们被人看不起也是对的，我们被人崇拜也是对的。这就叫做“你的历史没有对错”。我们到了革命既正确又错误，我们维持高压统治，压迫所有的人，既正确又错误。不然那你什么叫做历史没有对错？什么叫做历史没有对错？你说我是狡辩号。什么叫做历史没有对错？历史的事件当中就是包含了美丑、胜败、前后，这一切都是没有对错的。那么我们过去的一切起承转合，全部都是没有对错的。OK，那么蹲大牢的人是没有对错的，在街上做好事的人也是没有对错的。这些全部都是历史进程当中发生的。什么叫做没有对错？不就是想要和稀泥、想要搅浑水、想要把错的说成是对的吗？那些对的你是不愿意把它说成是错的，但是错的你就很想把它说成是对的，想要洗地吗？他们只对中国历史虚无？不是的，他们只对汉民族历史虚无。你搞错了一件事情：你以为他们是在对中国历史虚无，不是，他们是在对汉民族历史虚无。他们对其他的那些民族的时候，会强调他们独特的、伟大的历史的。就像现在，他为什么要特意替清朝洗地？因为他一定要强调清朝的十二圣君个个是明君，没有昏君这种东西。不管大家都觉得多丑多辣眼睛，反正就是美；不管大家觉得那个时候多么屈辱、多么 low 爆了，那都能够代表中国的形象。
