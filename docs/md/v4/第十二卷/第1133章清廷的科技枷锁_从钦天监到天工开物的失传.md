# 第1133章 清廷的科技枷锁_从钦天监到天工开物的失传

清廷的科技枷锁：从钦天监到《天工开物》的失传！

《或问》  
康乾无耻称盛世？锢术埋薪祸自滋。  
汉匠遭黜器自锈，回回历残书烬时。  
锁钥空悬国门闭，火种谁传五百年？

嗯嗯，欧洲工业革命发生在乾隆年间。我们的落后是从乾隆开始的。嗯嗯，如果你按它实际发生的时间，可以在乾隆年间来算啊。但是呢，它不是突然之间发生的——对方去建设他自己的科学体系、技术体系、生产体系，也是要时间的。所以实际上，从满清入关开始，这个局势就已经发生了逆转了。

康熙年间的时候，我们国内其实还是有一些明代留下来的，包括天文学家啊，也包括军械专家。这些人都还在。当一个王朝管理到稳定了之后，这些专业的技术人员其实还是愿意效力朝廷的，因为他们的这些技能、他们的这些本事，如果不效力朝廷的话，其实就没有用武之地。例如说这些搞天文学研究的人，他不进入到天文台——就是不进入到钦天监的话，他后续的这些事情都没法做呀，是吧？他是很难去当一个所谓的独立学者的吗？因此，这些类型的人，这些技术型、专业型的人才，是愿意归入到朝廷当中去的。但是呢，都被拒绝了——包括天文学。天文学家想要进入钦天监，因为你是汉人就不让你进。当时还设定了一场所谓的笔试，比啥呢？就是说那个汉人的这些天文学家的水平行不行。所以当时安排了一波：还安排了一个什么擅长回回历的色目人，然后安排了一个在明代的钦天监当过实习生的洋人，然后安排了一个汉人。但是呢，最后啊，反正最后他说结果啊，汉人不行。于是呢，从此就把汉人排除在了钦天监体系之外，钦天监就交到了洋人手上，同时呢，让里面的一部分色目人参与进来。实际上我们知道，回回历根本就不准啊，它的那个制定的历法原则也不一样，然后它中间修订历法的能力也没有。所以呢，回回历是时间越久越不准的。但是呢，哪怕是把色目人叫进去，他都禁止汉人进入钦天监这样的核心机构。所以你就想，这个逐渐能够被西方反超的过程，它并不是在乾隆年间咔嚓一下子完成的啊，只是说在那之前呢，这边一直在遏制着所有的科学技术、技术的进步。同时呢，至于其他的地儿，他还干了什么其他的事情，有没有向外输出这件事情，目前都还在推理阶段，是一个有理有据的推理，但确实证据链条还不完整，只能这么说啊。就是在乾隆年间的时候，西方实现了反超，但是在那之前，满清朝廷这里就在不断地遏制我们这里的技术进展。

包括说那个《天工开物》，最后我们是在海外找到的孤本，在中国境内没有了。有一本书失传，这件事情听起来似乎无关紧要，因为书嘛，很脆弱嘛，是不是？但是要知道，《天工开物》是晚明时期的印刷品，印刷品也就是说它不像手抄本一样只有一本，它是有很大量的、一本技术应用类的、大量发行的一个出版物。能够在本土上面一本都找不到，所以你就想，它在整个的自己的历史进程当中，其实都是一直排斥着科学技术的进步和传播的，甚至是原有的那些固有的技术，他都会给你拴住，不让你用。所以我们这里在不断地往后退啊——不是说没前进，是不断地往后退——而西方呢，却在这个时候崛起了。那么下一个问题就是，它是怎么崛起的，对吧？
