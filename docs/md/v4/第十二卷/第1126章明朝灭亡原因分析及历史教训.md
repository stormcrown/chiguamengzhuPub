# 第1126章 明朝灭亡原因分析及历史教训

明朝灭亡原因分析及历史教训

明朝的灭亡具有极大的偶然性，但也存在必然性。它的身上有多重因素叠加在一起，并非单一因素导致。尤其不能说是被清朝灭亡的，因为它的灭亡与清朝没有任何关系。所以，也不是说“我大明亡了”就要追溯清朝——责任不在清朝身上。明朝是自己把自己作死的，它自身出了问题才走向灭亡。但是，清朝身上的所有罪孽，都是自己造成的，因为它本身做了太多罪恶的事，拖了太多后腿，开了历史的倒车，摧毁了无数人的尊严和脊梁骨，也同时摧毁了伟大的文化。批判清朝是因为这个缘故，而不是因为它摧毁了明朝——它没有摧毁明朝，明朝是自己灭亡的。

明朝灭亡的原因很复杂。从外部因素来讲，外因包括了小冰河期导致的气候变迁，这对国家的经济底盘造成了影响。气候变坏，农产就会受影响，农业是经济最基本的民生保障，所以明代的外因在这方面起了作用。良田绝收又带来次生灾难，包括瘟疫和虫害，这是一个自然方面的连锁反应。同时，还有内部秩序层面的问题。

内部问题从明朝一开始的军户制度说起。到了土木堡之变，勋贵几乎被一网打尽，之后军户制度就崩溃了。军户制度本是通过屯垦养军的方式，尽量节省军费并拥有更多可用兵源，但土木堡之后勋贵系统崩溃，军队的话语权丧失，文官开始指挥军事系统，军户制度随之发生变化。军户常常被他们的头人当作奴工、兵丁甚至家丁来使用。年深日久，通过隐瞒丁口、欠饷等手段敛财，军户系统彻底崩坏。

这就是所谓的“明军不满饷，满饷无人敌”——始终拿不到饷银。因为整个军户构建的军事系统，在分配上出了严重问题，监管不良。编制上说有一万人，实际能找出三千活人就不错了，剩下的全是空饷。而这三千人也只是会喘气，根本没有体制考核之类的东西。当明军陷入这种境地，连饭都吃不上，还想有战斗力？军户制度崩到最后，已经到了敲骨吸髓的程度。

所以明朝内部自身问题特别严重。究其根本，还包括税收问题。明代后期，地方上很多税根本收不上来，尤其是有钱人家里，收税越发困难。之所以局面几乎难以扭转，部分原因在于明朝本身是一个很规范的政府、很规范的权力机构，意味着它不能用野蛮或打劫的手法面对自己的子民百姓。但到了清代，不过几十年功夫，过去的税全都收上来了——怎么收的？你不交银子就交头颅。哭庙？交人头。带头说不能与百姓争利？交头颅。用这种简单粗暴直白的方式，税就补上来了。明代能做这种事吗？能这样做的那能是好国家吗？一个正常的国家，若做了这种事，不在税上崩，也会在别处崩。所以明代不能做，因此税始终收不上来。

另外一部分原因是，这个时代大部分管理人员失去了主人翁意识，不知道给谁打工，觉得给谁都一样，皇帝换谁都可以。这是经济转型带来的必然结果。所谓资本主义萌芽时期，大老板们已经崛起，势力庞大，富可敌国。他们想，上面还有一个皇帝，要重农抑商，不能让我随便说话算数，这肯定不行。于是出现了明代的格局：税收不上来，话还要听他们的。皇帝连后宫都不敢出，因为一出去就归文臣管制，命都可能保不住。皇帝在朝廷上说话不好使，经常被架空。整个中层管理势力裹挟在一起，是明代的典型特征。这些人对国家没有归属意识，不在乎皇帝是谁，是中是外，是什么文化体系，都浑不在意。

到这个时候，官僚和地方上的士绅、豪族、富豪彼此勾连很深。他们经常看皇帝不顺眼。明代皇帝有一个核心纲领叫“君王死社稷”——一面是君王要为天下负责，另一面是君王拥有天下的所有权。但按照这种标准，下面那些富可敌国的人一定会把君王当作仇敌。他们向往的时代是元代：蒙古统治者根本不考虑社会治理，只是放羊吃草，派驻官员和小股部队镇压威压之后，剩下就是层层承包转包的包税人，只要税和服务到位，别的一概不理。在这种分包下，豪强、地主、士绅社会地位极高，拥有极强的自主权，根本不用在意皇帝要干什么。但明代截然相反，皇帝什么都要管，打算死社稷就意味着要为整个社稷负责任，就会管事。而这些豪强士绅希望皇帝最好别管事，放鸭子，他们才能成为土皇帝。所以他们希望有一个对天下不负责任的君王摆在那边当旗子，天灾人祸时往他身上甩锅就完事。因此，国在谁手上他们浑不在意。入关群体他们也会三呼万岁，甚至主动剃发易服去争从龙之功。这些社会中层觉得自己无祖国，这是一个很大的毛病。明末包括黄宗羲等人的思想家，已经出现了去君主化的倾向，认为君主不重要，想给谁卖命就给谁卖命，凭智力上位，不考虑家国问题，换个老板无所谓。所以当时有相当一部分人投诚速度飞快。

因此也产生了衍生的东西：明末清初的文艺作品中出现了鲜明的“贬男宠女”倾向。从晚明开始，很多作品看不起男人，常把男性描绘得没有脊梁骨，而把女子描述得非常贞烈。因为那些士绅、大人、豪强等男性群体觉得换老板都一样，想剃头就剃头，头皮太痒总能找理由跑路。但女性系统却非常鲜明地要保护自己的家国。为什么？因为男人投诚满清后依然是一家之主，不管给谁当臣子都能当一家之主。但女人不一样，大明朝女性有比较好的社会地位：女人可以单门立户，出女户，经商，拥有社会地位，还可以从军。所以女性的地位在明代还可以，很多女性有自由的思想，性格奔放。而满清是接近于奴隶制的管理方式，八旗制度建立在人身依附关系之下。一个在自由领域成长的女性，非常依赖于那种文明体系。一旦换到半奴隶制的环境，以体能的弱势必然成为他人附庸。女性的平等常要建立在较高的社会文明指数下，因为女性在体力上确实有客观弱项。所以男人不在乎，想投哪边投哪边，女人却非常坚决，贞烈人士层出不穷。因此明末到清初的文创作品中，有相当多专门抬高女性的地位，认为女性非常有品格，而男子是天地间的浊物。原因是那个时候这种现象太多，个案造成刻板印象，写下来就是这样。明代末年到清初有很多作品都如此。
