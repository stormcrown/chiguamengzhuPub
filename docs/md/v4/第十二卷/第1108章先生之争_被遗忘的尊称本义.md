# 第1108章 先生之争_被遗忘的尊称本义

先生之争：被遗忘的尊称本义  
《正名箴》  
尊称本自杏坛生，岂料今成男女争。  
绛帐昔闻传道统，青衿谁解辨浊清？  
效胡末学移真义，守故愚人訾旧名。  
若使宣文逢此世，可堪竖子妄讥评？

很多人在争论“先生”与“女士”，也就是这几年才对这个事情有所争议的。有关先生、女士这种称谓，过去呢，女士肯定是性别的，但先生是没有性别的。包括在《红楼梦》中，是不是还曾请人来说书时，说请了几个“女仙儿”？这个“女仙儿”就是“女先生”的缩写。“女先生”这个角色，或者说在这个时期，“先生”这个角色，实际上就是指特定的、有一定文化的表演艺人，可以称之为先生。而那些风水师也可以称为先生。总之，在特定文化行业当中工作的人士，都可以被称为先生。

更早的时候，先生可以指代老师，也可以指代老师。所以“先生”这个词本身是对身份的一个界定——先我而生者，对我有指引作用的人，可以提供一定资讯和指导的人，都可以称为先生。后来，先生被普遍性地应用在男性身上之后，而且这个毛病好像是从洋人来了之后才开始出现的——普遍性的所有男士都开始被称为先生之后，所谓的先生就不再指女性了。其实这是一个一点点变化的过程。

但是，在过去，虽然大部分情况下称呼先生都是指男士，但在指代那些年龄较长、尤其是出生于清末或民国时期的女性时，如果她还有比较好的成就，我们就会遵照当时的习惯称她为先生。而现在，任何人只要被称为先生，就开始搞起了男女之争，这件事十足的没有必要。因为所谓的先生被男性霸占，不过就是几十年的光景罢了。在那之前出生的这些女士们，她们自来当老师的时候，就会被人家称为先生。

中国人的习俗一向是拿别人愿意接受的东西去称呼他人，拿别人喜欢的东西去敬奉他人，这都是表现尊重的一种方式。我觉得今天的这场先生女士之争，尤其是当他把时间一下子拉长到过去漫长的历史当中时，实际上就是“拿今朝的剑斩前朝的官”，相当的离谱。
