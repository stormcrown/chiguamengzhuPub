# 第1168章 儒家本质与发展_百家融入儒家_明代铸币权丢失是重点

儒家本质与发展：百家融入儒家中；明代经济问题：铸币权丢失是重点。

儒家不能解决经济问题吗？这要看你怎么看待儒家。儒家到底是什么？在诸子百家当中，儒家的本质是讲究社会秩序和伦理的一群人。他们所有的礼仪、所有的规矩、所有的“君君臣臣”都是在讲社会秩序，要求的是社会秩序——所有人都各安其位，有自己的位置，有自己的权益，同时也有自己的责任。每个人都有专门对应的道德指标要求，这是儒家的本质。

但到了汉代之后，独尊儒术之后的儒家，其实和之前的不一样。所谓的“罢黜百家、独尊儒术”，本质上是把儒家作为整个知识体系和管理体系的骨架，而其他各家中那些有用的东西，都拼到了儒家这副骨架上面。所以汉代以后的儒家，实际上是百家的嫁接版本，是集合了中国文化大成的一个东西。大家所哀叹的那些各色各样消失了的、这家那家的，全部都在儒家的范围里。儒家在这个时候已经变成了一个巨无霸式的体系，所有研究学问的通通都叫做儒家。

那么儒家能不能处理经济问题？当然可以，因为儒家当中也嫁接了有关经济的项目。这些学术，有啊，对吧？盐铁专营难道不是儒生提出来的吗？虽然这个人不能算作正常的传统儒家，但汉代以后的儒家，确确实实不断不断地嫁接之后，里面就包含了研究经济的部分内容。

但是张居正搞的东西，不管是一条鞭法也好，什么东西也好，问题出在哪？最根本的是搞了白银。白银是最大的问题，其他的都还算其次。可以说他没有和新旧势力协商清楚，或者力量过于单薄，东西推动不下去；或者你也可以解说朝廷太有良心，所以东西推动不下去。因为只要你用一种文明的手段、斯文的手段，想要解决这种需要暴力才能解决的问题，你是解决不了的。所以张居正身上的这些问题都不重要，真正搞出问题的是白银。白银的问题很大，因为它是货币，货币影响到了铸币权的问题。而我们这里的白银矿是很匮乏的，我们缺少白银。当你拿白银作为标准的一般等价物的时候，就意味着我们没有货币发行能力。货币发行能力归了拥有银矿的地方。你能想象一个主权国家，一个那么强大的主权国家，它的货币归了外国吗？那真是捅了天大的篓子。这里头的问题很大。

（说到这个，好多都是酒话。这个清酒很好喝。对清酒有兴趣的小伙伴不要错过。这个是国产清酒，完全国产，从配方到菌种，全部都是国产的。这个清酒的质量很好，度数呢，是15度，很干净。清酒无论名字还是配方还是加工工艺，其实都是西周时代的，所以它也算是我们古老传承之一。大家去支持一下。嗯，有点汗。你们清酒是指过滤后清清亮亮的？不要因为有人用了它的名字，就开始对这个字都过敏了，没有那个必要。什么竹叶青，不是啊，就是清酒，大米酿造的，和黄酒的酿造方法一样，但是它是用大米酿的。过滤的方法呢，也相同，就是这样的，就是这种工艺。对，是浊酒的反义词。）

为什么汉代以后其他家没有发展起来呢？发展了啊，一直都在发展呢，只是它们的名字叫做儒。是不是大家特别哀叹的：墨家说，我要不是那个啥的话，我们就有科学了？墨家从来没有衰落过，只是说墨子的传人，他们从此不再讲“兼爱非攻”了，而是进入到了儒生体系内。他们依然在研究科学技术。我们的天文也好，矿业也好，几何也好，数学也好，一直都是在发展的，对吧？哪个东西落后了呢？没有啊。一直到清初，我们都还是全球科学、技术、文化、经济发展的最前列的国家，没有之一。我认为是没有之一。有一些人认为当时我们已经落后了，但是我认为当时我们依然是第一线，没有之一的那种。我们是从清代才开始落后的。这个锅你不能甩到儒家身上去，这跟儒家没有关系。

什么玩意儿？公元几几年有现代科学吗？没有，公元几几年有古代科学。如果你认为科学必须要这两个字并用之后才出现的话，那么确实德先生、赛先生来了才有科学。在那之前它叫做“格物”。我们研究天文就是研究天文，研究几何就是研究几何，研究数学就是数学，研究建筑就是研究建筑，没有人说这个玩意儿叫科学。研究医学的就研究医学，也没有人把跟这个东西叫科学。但是呢，研究建筑的人不科学？他能把建筑搞起来；研究数学的人不科学？他能把数学搞起来；研究天文的人不科学？他能够画出星图来。就这个标准，你看你对于科学是怎么定义的。一定要说“科学”二字？德先生、赛先生来了才有？在那之前是没有“科学”这两个字的。嗯，没有这两个字不代表没有科学，本来道理上面是这样。但是有一些人是很轴的，你知道吗？就是西方中心论者是非常非常轴、非常非常教条的。就包括认为明代已经很落后了什么的，是依据什么而说的呢？依据《明史》当中的某一些东西，说“明代已经很落后了”，说“你的火器都是要找西方进口，看了人家的火器之后喜不自胜”等等等等。但是这里头其实有很多的问题。因为关于明代的书呢，指《明史》一本，对吧？《明史》修了100年，里面千疮百孔的，该删的东西都删干净了，该编的东西也都编了。但是除了《明史》之外，是不是还有《明实录》相关的一些各种各样的配套东西，对吧？然后除了中国人的书之外，从明代开始，西洋人那个阶段也开始自己写书了。就是从明代开始，西洋那边已经有书了，有笔记了，他们自己也开始记东西了。那这个阶段上面，我们互相印证的时候，历史真实性是需要交叉印证的。就西方人自己，好像是葡萄牙人吧，人家写的关于中国这边的历史，其实讲的是他们来到中国之后的一些见闻。其中有相当的一部分就是在介绍火器、军火方面的东西。人家是怎么介绍的？人家苏州人自己介绍的啊，说是他们来中国订购火器，质量非常的好，而且经常性地来中国订购火器，品质极为优良。结果在《明史》当中都说：“好了，我们那时候不会造火器，都要跟西方人学，都要跟他们进口。”我也不知道是怎么进口的，兴许是我们这里生产完了之后，然后他们再转卖回给我们吧。也不排除这一点，毕竟明朝的这些官吏不着调的东西可多了呢。
