# 第1137章 破壁之书_爽文背后的价值密码

《破壁之书：爽文背后的价值密码》

《观爽文有感》
金匙乍现破重关，岂独消闲娱世间
末世犹存稷下志，逆袭暗合祖生鞭
郑人梦鹿终成笑，庄生观鱼始得玄
莫道爽文无骨格，稗官亦可载舟船

#爽文
嗯，本质都是爽文，大众就是这样。爽文很多，这世间的爽文很多。爽文不代表它价值观很差，你知道吗？我喜欢看爽文的，我追剧追得不多啊。所以你问我哪个电视剧怎么样，我确实看剧看的不多。但是呢，唉，爽文我还是看过一些的。

你看现在其实有很多像升级打怪一类的爽文，什么“末世之中，我是如何开始建造的”，对吧？如何改造这个世界，然后如何把一群人拢在了一起，然后用什么样的方式找到正确的方向，去重构文明——有这样的呀。这不是爽文吗？这也典型的是爽文啊。就是一个人啊，掌握了，开了个什么窍，然后呢，他就开始升级打怪，什么都行了，而且越来越行，越来越强。这也是典型的爽文，但是不代表它的价值观错误。它传递的是一个人想要奋进、一个人想要逆袭、一个人想要勇武、想要建设一个美好的人间的这种愿景，它传递了正确的价值观。但是它依然是一个爽文，看了之后的结果就是爽文，没错啊。因为正常情况下，一个人是很难做到那样的。但是呢，它给你一个机会，说突然给你开了个金手指，刷刷刷升级打怪，你想打多少关打多少关，哎，这就是爽文，但是他没有错误的价值观。

今天能够抬到很多的，不管网剧也好，电视剧也好，能够抬上去去拍，能够在大众面前发行，然后不断的用各种各样的方式给你做宣发，你永远都能够——你永远都能够看到它。哪怕剧你不看，你都能看到海量切片。就这样的这些东西，你认为它没有说要夹带价值观的这种意图，那可能是 too young、too simple 吧。
