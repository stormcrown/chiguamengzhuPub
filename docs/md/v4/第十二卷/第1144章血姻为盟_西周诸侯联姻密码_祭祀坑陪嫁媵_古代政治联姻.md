# 第1144章 血姻为盟_西周诸侯联姻密码_祭祀坑陪嫁媵_古代政治联姻

#### 整理后的易读段落

**血姻为盟：西周诸侯联姻密码**  
周室分封以血盟，媵妾陪嫁结连衡。中山虽学中原技，未入诸夏婚程。祭祀坑中辨异种，宗庙殿上认同根。朱明犹续姻亲策，千古权谋一脉承。

中山是大国，并非小国。它向中原学习了技术和治理方式，实为高仿版，却不被接纳。根本原因在于当时的标准：首先必须是诸夏，然后才能谈其他。因此，任何一个国家想加入，第一要务是联姻，而非自称“我学了就属于”。那时，联姻是头等大事：想尽办法结亲，成为打断骨头连着筋的真正亲戚。这种亲缘关系绝非靠中间几个联姻者就能维系，更不是随意放弃便割断的。只有达到全面、不可分割的亲缘关系时，才被接纳为诸夏。当时标准非常严格。

这些更早的分支并非外人，而是同种。我承认，那是同种。若非同种，连这个条件都无从知晓。同种才会开放婚姻通道；不是同种时，连通道都没有，直接否决。看看祭祀坑里摆的是什么？都是跨种的。只有首先同为同种，才有联姻的空间。只有深度联姻、彼此无法割舍，才视作同一体系，进而谈文化之事。这是我们过去的次序，历来如此。诸侯国之间不断通婚，正是怕亲缘疏远后生隔阂。所以诸侯动辄通婚，这边结、那边结。甚至在一个诸侯与另一诸侯联姻时，出嫁女一方会接收其他诸侯国陪嫁的媵（高级妾室），即多个诸侯国联姻时，其他诸侯国把自家女儿一起作为陪嫁的媵送来。其目的是凝聚整个诸侯间的力量，让彼此始终有亲缘关系，不会某一天一拍两散。这是西周时期的整体管理方略。因此，强调一定要通婚，不通婚则其他事情都休提——通婚是一切前提。这一招后来被朱元璋继承。

#### 备注：修正的词汇
- “血姻为盟” → 未改动（原诗标题及正文均使用“血姻”，保留作者用语）
- “应” → 改为“媵”（陪嫁的妾室）
- “连音割不开” → 改为“连姻割不开”（“连音”为口误，应为“连姻”）
- “朱夏” → 改为“诸夏”（“朱夏”系误写，应为“诸夏”指中原华夏诸侯）
- “修题” → 改为“休提”（“修题”为音误，意为“别提”）
- “付好就上了” → 改为“否决了”（根据上下文，原意是“直接否定就结束了”，“付好”系口误，调整为“否决”）
- “连衡” → 保留（诗中“连衡”指连横策略，此处意为结盟，可通）
- “同种” → 保留（正确词汇，指同一族群）
