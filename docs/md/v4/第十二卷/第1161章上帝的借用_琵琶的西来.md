# 第1161章 上帝的借用_琵琶的西来

### 上帝的借用，琵琶的西来

上帝、天使这些词，是明代的时候被人借用的。谁借用的呢？就是当时的传教士。明代的传教士来到中国之后，号称是传教、给我们传播知识，但实际上，他们传播的都是我们有、而西方没有的知识。这事也算是离谱了。你就看他们怎么学的吧。总之，在他们的传教过程中，传教的成果应该不怎么样，没多少人信他们。但是呢，确实把他们当地的神话传说故事传播进来了——这一套故事就是关于玛利亚的那一套。他们传递这些故事，写这些小故事、小话本的时候，在里面使用了“上帝”这个词。这个词很明显是在借用中文词汇，而不是它原本的。原本无论你是叫Gold也好，叫God也好，叫什么也好，反正它压根儿不是中文词，而是在翻译过程中借用了这个词。包括“天使”也是在翻译的时候特意借用的。然而在我们本源上，无论是“上帝”也好，“天使”也好，都有专门的指向性，那就不是那个意思，也跟他们的这一套系统完全不相吻合。

中国人的“上帝”是指我们的这个——到后世就是所谓的老天爷，主要指的是昊天上帝。早期的时候还分东南西北中，就是五方上帝。但它的核心意义，指的都是我们所看待的天庭的掌管者。早期有五方上帝，后来我们汇总起来的时候，有些传说中就称之为昊天上帝。我们所说的“上帝”通常指的就是昊天上帝，也就是俗称的老天爷。“天使”指的是一种使者的身份。通常情况下，如果君王向地方上传旨，那么传旨的人就会被称为“天使”，就是天子派来的使臣，主要是这种意义。因此，天使是什么样子？就是主观派下来的。包括后来《西游记》里面，上帝是不是指玉皇大帝？玉皇大帝这个名字形成的时间有点晚，是在宋明时期才出现的。早期没有叫玉皇大帝的。玉皇大帝的身份就比较明显了，他走向了世俗化路线，把天庭的结构模拟得比较世俗、比较官僚，是这个阶段出现的。所谓的天使本来指的就是——无论是天朝的、天子的、天庭的都行——他的使臣会被称为天使。但是被借用了之后，现在你一提起那个什么天使，就觉得是扑棱翅膀的小孩，就是另外一回事了。这也算得上是一种文化挪用吧。但是因为时间比较久，大家已经习惯了，也可以说它产生了语义上的演变。

### 冰酒推荐

有个新产品给大家推荐一下——这瓶儿啊，来演示一下。这字印的，我估计拿到你们面前你都看不清楚，还是给你看看瓶子，然后我来念好了。它叫做野生蓝莓冰酒，喜欢冰酒的小伙伴别错过了。冰酒是啥？先介绍一下冰酒吧。冰酒呢，是在果实的产区，由于自然的低温，它逐渐降温，然后让果实因为低温而逐渐脱水，经历了霜冻。就是在这种情况下，它里面的水分消失得比较多一点之后，但没有全干——它可不是葡萄干，不是果干啊。在这种情况下，它出现了这种低温浓缩果汁，就是用这种低温浓缩果汁来进行的酒体的酿造。由于它里面的含糖量比较高，所以一般情况下，在酒水的最后依然偏甜，有比较浓郁的果香和甜蜜的口感。这个类型的叫做冰酒，有冰葡萄酒。而我今天带的这个冰酒，是野生蓝莓的冰酒，就是东北产的那种蓝莓。这个酒是东北长白山的——长白山的蓝莓冰酒。蓝莓这个口味，蓝莓是浆果，大家吃过蓝莓的应该有印象吧？蓝莓的口味里面自然而然就保留了蓝莓自身的那些营养成分。想试冰酒的小伙伴不要错过，这个冰酒的品质是非常好的。他家是正经的老酒厂，酿造的工艺非常过关，所以这家的酒质量都很好，里面的那个果浆本身的香气也是很清晰的。那天去试酒的时候，我们家那位同志在那里喊着说：“啊，这啥味儿啊？这也不是葡萄味儿啊。”我说：“那不就对了吗？这一蓝莓酒你要是喝出葡萄味来，那就坏了。”它确实是蓝莓酿造的，和葡萄酒的味道是不一样的，大家可以试一下。而且那个冰酒相对来说是甜口的，比较适口。我觉得中国人应该都会比较喜欢吃这些口感更加柔和的东西。

### 琵琶的起源争议

琵琶一直以来，有人说它是胡人的乐器。那么他的理由是什么呢？因为你说的这些事情总得给人家一个理由，对吧？你为什么要说琵琶是胡人的乐器呢？你又找不到源头——哪个胡人弹了呢？其实也没有，没有找到任何源头，没有说是哪个地方发源了琵琶，然后通过谁带过来。这些东西都没有，那些证据是没有的。但是啊，虽然没有证据，可是却由官科信誓旦旦地公开发表，而且在传媒系统上面去公开发布，接受采访就说琵琶是外来的。笑死人的证据是：琵琶的这个命名方式是拟声的方式，而汉语不用拟声的方式给乐器命名——就这个证据你知道吗？他的证据是“拟声”，因为它的命名方式是拟声，所以这个乐器不是中原乐器。这咋整啊？就好像是，为啥说你不是中国人？因为你的名字叫哈利哈里，这个名字一听就不是中国人，所以你就不是中国人？这个奇葩事就是这样推论出来的。这是前期的。那在西方中心论之下，我们这里一切都是西来的，就是在这种基础之上推出来的玩意儿。“两个字的都是外来乐器”之类的理由多了去了。我每次看他们编出来的理由的时候，都为他们感到汗颜——他是怎么有脸说出来的？你要说什么东西是外来的也不是不行，咱们又不是跟外界隔绝，也不是不跟外人学习。但你总得有个证据吧？哪个地方先发源了？别人在哪个地方演奏了什么乐曲？通过什么方式、什么样的商路传过来的？你总得有个道路吧？啥也没有，直接就是“这名儿起得不对，所以不是”。
