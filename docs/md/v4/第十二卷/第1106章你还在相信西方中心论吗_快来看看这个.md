# 第1106章 你还在相信西方中心论吗_快来看看这个

你还在相信西方中心论吗？快来看看这个！  
西学东渐，崇洋渐深；史观偏执，自抑日沉。考据求真，当破迷障；逻辑为刃，可斩妄尊。华夏正道，源流自证；春秋笔法，虚实须分。祛魅正本，重光汉统；鉴往知来，魂铸昆仑。

呃，关于西方中心论这个东西，其实很有意思。因为过去一些年，我们知道有很多人，或者说有相当一部分知识比较丰富的人，持有明确的西方中心论思想。那么它是怎么诞生的呢？一定要弄清楚一件事，就是西方中心论是怎么根植到中国人脑海当中的？它基于什么？

其实，它利用了中国人的一个短板，或者说利用了中国人的一个长项。中国人最大的优点是什么？我们有历史沿革，我们有历史记忆，我们敬天法祖，我们真的知道过去发生了什么。我们有世界上最长的日记本，这是中国最大的长项。但也正因为这个，给我们制造了一个短板、一个漏洞。中国人尊重并且看重历史，我们特别要求对历史的信任——一个信任历史的民族，会导致一个后果：当别人信誓旦旦，并且用非常宏大的手段，制造了一个非常梦幻、非常伟大的历史世界时，我们就认为他真的很厉害，并且真的相信他有成文史。为什么会相信？因为我们真有。所以，当时西方人开始传播自己祖先的伟大和光荣时，中国这边的很多学者是发自内心相信的。这才是我们出现西方中心论的根源——相信他们从来都是伟大又光荣的，相信他们以前也是世界文明的源头，至少是之一，甚至相信他们常常能够吊打我们，我们只是偶尔强盛，而这个强盛也不很强。这是今天很多人都还保有的印象，这些都是西方中心论产生的影响。

只要你相信他自古以来就辉煌又伟大，那么自然而然地衍生出来的思路，其实就会变成：你看我们家里有臭豆腐，是受他影响的；哎呀，我今天吃上肉了，是受他影响的；我今天个子高了，是受他影响的；我今天种地一如何能够除草，也是受他影响的。所以，这是一个归因习惯的问题。当你心里有西方中心论，相信他伟大又正确时，你在遇到一些事情时，就会把原因找到他头上。然后一次又一次地找原因，只要自己有毛病就找自己毛病，有好处就在别人身上找好处。这些事反复不断地做，就把自己的思维驯化出来了。于是，大量学者在反复学习这些东西，并不断反思自己历史上的错误之后，就形成了根深蒂固的西方中心论。

我们自己在考古场地中发现了世界上最早的陶罐、车马、纹样等，然后他第一时间想到的是：西方人的哪个地方果然影响到了我们。当时的很多学者，包括现在也有很多这样的学者，这就是西方中心论入骨之后的一个产物。所以，为什么学习历史、以自己的视角来看待历史很重要？就是因为当你的屁股摆正之后，你就会很快意识到对方没有那么完美，而且对方的东西里头可能有问题。所以，认知历史，甚至今天质疑西方的历史，是非常要紧的。至于西方历史到底是什么样子？你天天质疑，那难道人家就没有了吗？这不重要。因为它的历史到底什么样子，应该归它自己来证明——它来举出证据：它的原文在什么地方，怎么传递的，中间有多少个版本、遗址，发现在什么地方，它的后代在哪里。这些问题都是西方人应该回答的，不是我们的问题。我们只需要干什么？找出对方的毛病，告诉你这事儿你撒谎了，你重新找证据，你重新提，你要证据。不要跟我说闲话。这，就是破除西方中心论最重要的一件事情。

现在推的这套书，是何青老师的《以图证史》。这套书一共上下两册：一本叫《言不必称希腊》，一本叫《光从中华来》，就是在质疑西方历史造假的问题。西方历史造假，其实源远流长，但过去我们对他过于信服，以至于很多人想到这件事时就直接认定了：你要是觉得西方历史造假，你就是没上过学。这确实也有道理，因为只要你专业地去学习历史，你一定绕不开辉煌又伟大的西方历史。包括说，哪怕我学习的项目当中，也包含了西方在法治方面的历史——那所谓的汉谟拉比法典嘛，那个由神学而推导出来的法制项目。这些东西全部都是学过的呀。但是合理吗？一定要考虑一下，合理吗？汉谟拉比法典是在石碑上面的那块石头，它是怎么刻的？怎么出现那么小的字？而且时隔千年，你是怎么认得那个表音文字，把它翻译出来，还把每条每条的法律都弄得清清楚楚的？合理吗？一个根本不合理的东西，但我们首先要相信它的存在，然后再去盛赞它的伟大，这叫“因信称义”。这是科学吗？这是神学呀，亲。所以怎么破除迷信？破除迷信需要的是科学的武器。什么是科学的武器？找证据，找出对方逻辑的漏洞，然后推翻它。它的地基倒了，你就知道它的大厦摇摇欲坠，必然崩塌。你知道它不是世界的中心了，就会开始去思考：我们的祖先有可能没那么糟糕，有可能真的很伟大。那我们为什么吃过苦头？伟大的人难道不能摔跟头吗？聪明的人难道不能栽跟头吗？投资圣手不会亏本吗？都会的呀。人是会倒霉的，倒霉的时候喝凉水也会塞牙缝。所以，虽然过去的几百年当中，我们吃了很多苦、很多亏，而且确实从外观上、从行为上看都显得卑鄙又猥琐，但它是什么原因造成的？是自古以来吗？如果我们自古以来就这样衰弱，我们为什么会存在数千年绵延不绝的文明？我不相信在这个残酷的丛林竞争的国际之间，一个国家积贫积弱数千年，却依然能够绵延存在，而且形成了最庞大的人群。这是不科学的，是不合逻辑的。所以真相是什么？首先要做的就是破除迷信。这套书就是拿来破除迷信用的。凡是还能够读一读书的小伙伴，完全可以把这一套入手一下。
