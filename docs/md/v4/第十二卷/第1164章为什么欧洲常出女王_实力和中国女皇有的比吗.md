# 第1164章 为什么欧洲常出女王_实力和中国女皇有的比吗

为什么欧洲常出女王？实力和中国女皇有得比吗？[吃瓜蒙主]  

为什么欧洲常出女王？这是他们的继承制度问题。这个继承制度来自于他们的婚姻制度，而他们的婚姻制度和我们古时候不一样。我们古时候是一夫一妻多妾制，早期还有“媵”的身份，和妾有点不同。但无论是媵还是妾，除了配偶的本职工作之外，其实还兼具一个功能，叫做生育保险。婚姻的目的中包含生育继承人这个工作，如果只有一个妻子，很容易面临生育继承人无法完成的情况。例如只生了一个女儿就没有了，例如压根生不出来，例如身体状况不好——这些可能性都存在，而且概率不低。因此，通常情况下，在我们这种世俗的婚姻领域中，就会考虑到一些很现实的问题。既然婚姻是结两姓之好，我不能因为只生了一个女儿就离婚，这会引发两家族甚至两国之间的动荡。既然婚姻很稳定，就必须上一个生育的保证，让我们之间一定会有继承人，并且继承人的数量不能太少，因为孩子也不能保证一定能健康长大，风险都存在。因此就安排了一个补充策略，就是一夫一妻多妾。  

在西方对照中，你会发现不一样。他们的婚姻制度是标准的一夫一妻制。一夫一妻不代表他们真的忠贞，这是两回事。他是真正的一夫一妻，没有妾室，但有情人，而且是公开的情人。这个情人就只是情人而已，可以生育，但生育的私生子不进入继承系统。所以这种标准的一夫一妻制带来了巨大的风险：如果你只有女儿，就只能让女儿继承。所以整个继承规则就建立在这种充满不确定性和不稳定性、神权主张之下的一夫一妻制基础上。因为这种系统不保证生育条件，因此继承关系就会按照血缘的方式来继承地位，于是就会出现所谓的女王这样的角色。  

但实际上，女王在古典时期必然是一个高风险的存在。因为女性的生理原因，会导致她一生中只要想要继承人就必须生育，而生育一定会面临风险，而且在生育过程中一定会面临执政能力下降的问题。风险又大，执政能力很有可能逢几年就低一次、逢几年就低一次，这样的不稳定情况其实不利于古典国家的统治。所以实际上，古典时期的女王绝大多数都没有办法掌握实权。她是通过继承制而获得的王位，获得了自己独特的领主身份，但真正去掌管这些领地的人通常是她的配偶。极个别的人确实通过自己的一些政治博弈掌握了权力，但那是硬拼能力了。实质上，这些女王绝大多数就是因为财产继承而拥有了领主身份。  

叶卡捷琳娜有实权吗？有实权——她通过政变把她丈夫都给收拾了好吗？要不然怎么称她为大帝呢？能够被尊称为大帝的，那是实实在在有实权的人啊。  

有开国女王吗？你指的是哪？西方吗？没听说西方有开国的女王。但是中国这里有开国的女皇帝，不过她当皇帝的时间非常短，是真正的女皇——她不是通过什么皇后、太后的身份转变，而是真正自己打下了一片地，然后自称女皇。有一个，当了二十几天后就灭了。唉，对对对，陈硕真。我不是在说武则天，武则天我不至于记不住她。武则天的话，她是通过自己的太后身份，实际上是太后的临朝称制，但是换了个壳子，大家觉得无所谓，所以就让她上位了。武则天最后是被逼宫下来的，对，神龙政变。武则天是神龙政变下来的，因为晚年乱政了嘛。当然本质上的原因是，确确实实应该还政于李氏了，李隆基这一派都已经成长起来了，而她还拒绝还政，实际上就有了篡权的风险，因此被人逼下来是很正常的一件事情。她有了乱传位的想法，是呀，乱政嘛。本来让她当皇帝，大家也没有反对，随便打打嘴炮就上去了，搞搞政变就上去了。因为太后临朝称制这件事，在我们这里是无所谓的，本来就有以孝治天下，让太后临朝问题不大。而且武则天本身虽然说能力不至于非常好，但也并不差，武则天的能力尚可，平平无奇，守城之君还是当得了的。就是在这种程度上面，所以朝臣们没有激烈反对，国家内部也没有混乱，她就当了皇帝了。但是她确实存在乱政，授权给她的面首，授权给她的子侄，实际上带来的都是一些篡权的风险啊。  

我一直强调一件事：中国古代的皇帝大部分情况下都不是一言堂，也没有什么绝对的专制，其实是与官僚、与士族、与各色人等共天下。皇帝是这个国家的一个招牌、一面旗帜，很多时候我们允许虚君的存在。就是说这个皇帝可以不会说话，可以很笨，他只要在那里就行。你在这儿，我们就可以围绕着你运行我们这一摊班子；但如果你不在，那很可能为了上面那个人，咱们就开始打得头破血流，这个活就没办法干了。因此，我们古时候的皇帝要起到定海神针的作用，他可以不会，可以无能，但他得有。而武则天带来的风险就是，当她女主临朝，并且她不想传给她儿子的时候，就会引发真正的继承权上的动荡，而这个动荡是真正动摇国本的。皇帝可以是一个庸庸无为的昏君，这个不会真的动摇国本，因为做事最终还是朝臣在做，还是可以维持住的。但是如果你导致继承的顺序发生混乱，大家都会因为下一个上来的皇帝得位不正，而开始去争谁才是正统、谁才有资格当皇帝，那就一定会分帮立派，一定会打得头破血流，然后才是排位赛。排位赛是损耗最大的东西，所以宁可忍耐一个昏君，都不肯让你发生这种变故。这就是为什么我们要搞嫡长子继承制啊。  

武则天是否对后来的君王有影响？有。所以后来宋朝的刘娥，实际上政治能力是比武则天要强的，但她想要去穿一下冕服的时候，所有朝臣都反对啊。大家不反对她临朝称制，不反对她管事，也愿意听她的，认为她管得也不错，但是没有任何人同意她穿上冕服，就是因为要避免这种皇权传递不稳的情况出现啊。武则天开了一个很坏的头，她败坏了太后的政治信誉，所以之后大家再去交权与妻子、交权与太后的时候会很慎重。  

哦，现在带货酒。嗯，你们说得对，我这是一个带货直播间，所以大家要喝果酒的话，可以试一试这种。这是多东北的长白山酒业出品的甜白葡萄酒，真的是非常非常好喝。这个甜白葡萄酒呢，是在发酵到一半的时候终止了发酵，终止发酵就会导致里面保留有更多果汁原味的成分，还保留了一部分的糖，所以它是甜的。甜白葡萄酒是透明的酒体，葡萄的果香、葡萄果汁的味道非常浓郁，同时呢，是一个微甜的口味，很适口，适合大多数人饮用。度数呢，也不高，只有八度而已。而且这个甜白葡萄酒酿造的工艺就非常稳定了。这家酒厂是一个资历非常非常老的酒厂，是建国之前就成立了的老酒厂，整个它的酿造的水、果、酒的工艺都是很成熟、很稳定的，所以酒的品质也非常的可靠。
