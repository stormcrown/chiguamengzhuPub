# 第1162章 中国哲学_毛选可实践接地气

中国哲学：《毛选》（毛泽东选集）可实践、接地气。  
哲学不都是在经济发达的地方才能学吗？  
那你以为《毛选》是什么呢？  
《毛选》就是哲学。  
《毛选》是最接地气的哲学。  
首先，你要弄清楚哲学是什么。  
哲学就是对事物规律的总结，对事物根本规律的总结就是哲学。  
按照近现代一些学者的说法、哲学家的说法，他们特别喜欢把西方的哲学定义为哲学，而中国的哲学都不算。  
但如果按照这种说辞的话，那么西方哲学不管是黑格尔也好，还是谁也好，总之他们特别喜欢把一句话说得非常绕，提出不同的定义，然后在定义中间绕逻辑，以纸面上的逻辑套逻辑再套逻辑。所以研究哲学的人，你经常觉得他说话不顺畅，并且有点颠三倒四的，或者心理状态不稳定，就是由于这种原因——每天在研究的东西都是怎么钻牛角尖。  
但是根本性的问题是什么？哲学是拿来解决问题的。哲学要梳理社会当中一切根本的原因、路径、逻辑、道理，然后去寻求解决方案，解决实践当中的问题。这才是哲学的本意。如果哲学就是把东西都抽离出来，纸上谈兵，其他的事情一概皆不管，最后就会变成西方哲学的那种——学深了就精神有问题。  
《毛选》就是一个典型的哲学著作，因为它在整理事件背后的原因，梳理背后的逻辑和道理，并且进行归纳与总结，然后再把它转化到实际的操作当中去，解决实践当中的问题。这个就是哲学的根本要义。不能够因为《毛选》写得过于朴素，你就认为它不是哲学——这才是真正的哲学。哲学不是给你玩空中楼阁式的，那玩意儿叫炫技。  
老子的《道德经》不是哲学吗？《庄子》不是哲学吗？是的呀。是因为近代的哲学的定义被西方人掌控了之后，他们就声称中国没有哲学。并不是真的中国没有哲学，而是西方人声称中国没有哲学。我是在他们的这种话语权下面来讲解哲学的定义，并不是真的认为中国没有哲学。中国哲学诞生的时间当然很早，我们的诸子百家当年其实就是在讨论社会问题，讨论人性，讨论人类的存在逻辑。无论是字面上的逻辑，还是实践当中的逻辑，在诸子百家当中都是有的，全部都存在。大家在这个时候激烈地探讨，专门成立学社去研究，彼此之间还要进行各种辩论是为了什么？为的是在春秋这种大争之世里面，寻求一个社会制度的解决方案。哲学必须要解决问题。不能解决问题的东西，你要它何用？如果说哲学是不需要解决问题的，我只要吹水就行了，那你请把它叫做小说，而且是枯燥的、难以阅读的、屁用没有的、特别不好读的、谁都不想看的小说。  
既然它是一门学问，既然是哲学，讲的是终极的道理，那么这个道理必须能够被实践所证明。所以如果认为哲学不需要实践，那么马克思主义是什么？马克思主义要求的就是哲学需要在实践当中获得印证，否则你怎么知道你编出来那个东西是你做梦来的还是真的？所有的学术、所有的研究，它本身应该是为人服务的，而不是为这个学者本人服务的。所有的这些研究的成果，不是为了让这个学者著作等身成为大家，而是为了让这些事情能够造福世人。这是一个根本性的东西。  
所以我说，在所有的哲学著作当中，《毛选》是最好的，因为它试图让每一个普通人去阅读，让每一个普通人掌握这个人世间的规律。学会了这些之后，我们再去看待这个世界，再去解决从身边的小事以及更加遥远的大事。这些都是一个人应该掌握的东西。  
但是知识过去一直是有壁垒的，包括说哲学家为什么总是高高在上？他是为了形成知识的壁垒。这个事儿倒是不分中外，全世界所有的地方都是这个样子的。学术研究好了的、对社会管理研究清楚了的人，基本上都是上车就锁门，对下面的进行一些识字的、开心的、普及型的教育，能够打工就可以了。其他的那些真正的高级的知识、那些真正的道理，他们就在自己的内部分享。这个事儿古今中外概莫能外。古时候中国的识字率——除了清朝——古时候中国的识字率不低，很多的政府都有强迫教育，要求大家认字，不然的话连公文都看不着：我发一个通告，你不知道在讲什么对吧？所以呢，一直都有强迫识字。然而关于真正的道理、真正的学说，他是如何理解的？那些诸子百家的学术著作、那些经史子集的典范，他是不让百姓学习的。哪怕说你能看见，你也需要老师来专门引导，否则你看不懂。这个就是知识的屏障，历朝历代都有。孔子当年想要学周礼，但孔子作为殷商后裔，勉强还跟贵族搭了一点边儿的这种关系，他学不到。最后他去求教于老子。为什么要找老子？因为老子是图书管理员，他知道周礼里面的内容。所以即使是在中国的古典时期也是一样的，上车的人都会试图锁门。西周的时候通过一个图书馆来锁门，那么到了宋明时期的时候，就通过学术——大家掌握了这解释的垄断权、释经权——来锁门。事情都是大差不差的。所以能够有机会去学习真正的、很有意义的、很有价值的内核的东西，是非常宝贵的。  
因此我一直都会盛情地推荐《毛选》，因为它讲的就是这些东西。那个叫什么123呢？你要是闲的没事干的，找一个能跟你聊得来的人好不好？你在这刷屏干啥呢？我看着闹心烦。什么叫做“不能证明的才是哲学”？比如上帝是否存在。你知道那玩意儿叫神学。不，不能证明的那个东西叫神学。虚无缥缈，以信称义的，先信了再说，信了之后就说他是真的，这个玩意儿叫神学。哲学是需要能够指导实践的，科学是直接面对物质世界的。要是理解不了这些正常的社会基础的概念的逻辑啊，你去找一个比较神叨的人，跟你聊点开心的事情。大晚上的，而且是大周末的，你跑到这儿来给你自己添堵干啥呢？我又不可能认同你那个东西，你刷屏刷多少遍也没有用啊。刷完了之后我只会说你这个人吧，要念一下小学三年级是吧？基本上给你的回馈就这样。你特别喜欢这个吗？那你就回去重新学一下小学好吧。基本的阅读理解需要了解一下，基本的人世间的这种逻辑需要了解一下。我也建议你可以从《三字经》开始学起，我觉得《三字经》多背两遍还是大有好处的，虽然我没背下来。如果说你能够超出于《三字经》的水平的话，也可以去阅读一下《论语》，多读两天也能够感受一下，知道一下什么叫做人世间的道理和逻辑。这些东西都有了之后，你才有可能能听得懂我说话。要不然呢，我只会车轱辘话满天转。在那个地方不停的刷屏，刷屏这件事情会显得一个人的智力水平堪忧，需要治疗。OK。
