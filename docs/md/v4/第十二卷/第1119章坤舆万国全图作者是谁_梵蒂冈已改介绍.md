# 第1119章 坤舆万国全图作者是谁_梵蒂冈已改介绍

关于《坤舆万国全图》的作者，目前存在争议。梵蒂冈已修改官方介绍，不再标明作者是利玛窦。有诗云：“昆舆遗墨渡沧溟，利氏虚名梵蒂瞑。郑和帆影破瘴疠，欧罗舟楫困膻腥。凿空欲掩羲和迹，铸铁难销禹鼎形。莫道昆山无片玉，海西犹自拜占庭。”

网上看到的明朝《坤舆万国全图》只包含亚洲和半个欧洲，因为全稿不在国内，而是空运到了海外。关于此图，推荐李兆良老师的研究，他对此钻研已久，并在全球学术论坛发表演说，系统论述了作者问题。自他发表演说并出书后，梵蒂冈在其官方网站上删除了“利玛窦是作者”的介绍。因此，如今除中国外，再无地方声称《坤舆万国全图》是利玛窦所作。然而，在国内仍有中国人坚定地认为这是外国人传授给我们的。

李兆良的一些观点不够严谨，但不仅是他，所有质疑西方历史的人的观点都不够严谨。为什么不严谨？因为证据不足。为什么证据不足？因为大量证据被西方人自己掩埋了，连他们自己都找不到。为了构建西方中心论，他们不断编造伟大历史，这一过程中事实上遮蔽了真实的历史。如今他们想找回真正历史很困难，不仅我们困难，他们自己也是如此。他们自己的历史被自己玩没了，我们不能为此负责，他们必须为自己的行为负责。

关于《坤舆万国全图》的解析书籍已经绝版，想看的读者无法获得正版，只能到孔夫子旧书网寻找；如果不在意影印版，某宝上也有。这本书值得一看，虽然有些啰嗦，但写书很正常。书中主要内容是对坤舆万国全图进行地图学的分析与对证，这项工作非常重要。因为如果地图学是欧洲人发明的，那它的理论前身是什么？不可能没有任何基础，突然从只会画两根线就发展出完整的地图学，进而开始全球探索。要画全球地图，航海能力必须足够，但大航海时代欧洲人明显是草台班子，否则怎会不知道如何解决坏血病？明代中国已解决此问题，远航没有这种困扰，而欧洲人却因毫无防备而大面积受害。这说明他们的航海很莽撞，不了解风险。他们的整套逻辑叙事顺序存在问题，正因为自身有问题，我们才提出质疑。至于真相，并不归我们管。
