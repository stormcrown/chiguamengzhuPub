# 第1165章 五胡乱华成五胡入华_谁在偷摸篡改历史

五胡乱华变成五胡入华，谁在偷偷摸摸篡改历史？[吃瓜蒙主]  
又有人编出“五胡归华”这一说法，是个历史博主对吧？  
我还当是什么大家呢，  
难道就这么定了性？  
是不是还有“规划”？  
什么叫“规划”？  
如果当年他是华人出去了，  
然后回来才叫“归华”。  
那些人算吗？  

在面对历史这个问题上，  
古人比今天的人要客观。  
因为他们身上没有那些政治正确的枷锁，  
不会因为10年前才出现的伙伴，就去说  
1000年前的祖先是假的。  
古人比我们今天的人诚恳、坦白。  
所以“五胡乱华”这个定义没有任何问题，  
因为他们当时不是“入华”，  
也不是“归华”。  
他们早就已经内附生活了。  
只是在这个时候，  
由于整个西晋的动荡，  
自己把自己的核心群体折腾得差不多了。  
人数、驻军的能力、管理的能力都一团稀糟，  
脆弱得不堪一击。  
于是这些之前就已经内附的部落开始造反，  
想要反客为主，  
并且采用了非常残暴、非常野蛮的手段。  
所以这个阶段，  
不管是谁下的这个定义，  
“五胡乱华”是毫无问题的。  

那个“五胡入华”是这种中性的、含糊其辞的说法，  
而且它很不客观。  
因为这些人不是这个时候来的，  
他们早就来了——在汉代、东汉的时候就已经来了。  
他们作为雇佣兵、雇农而存在着，  
长时间以部落的形式在边陲地带生活，  
是被招揽进来的人群，  
不是没有往来的人群。  
所以他们不是“入华”，  
不是这个时候才来的，  
他们早就在这儿。  
但是在这个时候，  
他们以客犯主，  
制造了天下动荡、生民涂炭。  

用常识和逻辑去看历史、推测历史会出问题吗？  
会。但是呢，你不能把这种推理的方式绕过去——它也是很重要的一个东西。  
但是你要是说你只用这种方式，是会出问题的。  
因为在历史的进程当中，  
也有很多东西确实是属于意外，  
一些偶然事件。  
你现在是不是有些人把它叫什么“黑天鹅”什么的？  
不管它叫什么名字吧，  
反正一些突发性的事件，  
也会出现在这些很重要的位置上。  
这些事情呢，不能够完全运用逻辑来推理。  
但是整体的事情——  
你去判断它这个书写上面有没有注水，  
有没有在里面被人家篡改，  
那些野史是不是其实是政治的有效补充——  
在这种情况下，  
是可以运用到一部分的推理和推测的。  
事实上，在一些证据链条不是很完整的那种历史段落当中，  
正规的研究学者也会使用这种旁证、推理、猜测这些方式。  
尤其是在中华上古史当中，  
运用到的推理和猜测就会特别的多，  
因为证据链条呢，比较分散。

备注：
