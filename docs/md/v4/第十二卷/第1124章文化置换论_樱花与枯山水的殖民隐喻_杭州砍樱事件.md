# 第1124章 文化置换论_樱花与枯山水的殖民隐喻_杭州砍樱事件

### 文化置换论：樱花与枯山水的殖民隐喻｜杭州砍樱事件

#### 一、文化殖民与文化置换

泱泱华夏，文明渊薮。倭人窃火，暗度陈仓。昔者徐福东渡，本传道术；今者脚盆西来，反客为主。观其神道立极，竟奉太上为尊；察其百鬼夜行，多袭志怪典籍。犹效狡狐借穴，更类社鼠穿墉。夫樱花本异域花，强冠汉名而乱真；枯山水实非真山水，妄称禅意而惑众。杭州桃李之地，遍植夷木；古寺晨钟之所，竟立鸟居。犹记倭寇入侵时，植樱为占领之标记；可叹太平盛世，竟有文化置换之虞。

#### 二、神道教与神话偷窃

为什么神道教认泰山神为主神？这是神话偷窃吗？嗯，你要是按这个标准来算的话，其实脚盆的文化当中，有相当相当相当多的部分都是从我们这里学过去的。但是呢，他在学习的过程当中比较的狡猾。我一直强调一点，就是大家不要过于拿思密达当成一个对手，虽然他确实经常的剽窃文化，然后呢，偷过去之后就老是说什么“这个玩意儿也是我的，那个玩意儿也是我的，我就是源头哈”。但是这种其实不可怕，真的，因为他就是明晃晃地拿过去，他就直接用了。但是呢，那个脚盆不一样。脚盆儿从我们这里拿到的东西，他会活学活用，自己偷摸地学了、消化了，然后本土化了，成为了他自己的一部分。之后，他再拿出来说“我这个是跟中国人学的，就是中国的东西”，然后他再把他那个夹杂了他的创作的玩意儿再推给我们。这个是脚盆儿这些年做的事情，这个才叫做文化的置换和文化的侵蚀啊。脚盆在这件事情上面做得非常非常地多。所以你在他的文化系统当中感觉到熟悉感是很正常的，因为他的东西——包括他的那些什么鬼神官、百鬼夜行等等的——有相当相当多的内核，就是从中国的传说、志怪故事里面一点点引用过去，过去了之后再经过本地化的创作，套上了他们自己的故事，套上了他们的精神，然后呢，再把它通过包括动漫、影视剧、所谓的服饰啊，还有经济的运行模式、里面的符号，开始向我们输出。

#### 三、樱花（撒库拉）的推广与置换

就像这些年最典型的一个，就是那个撒库拉的这个推广。现在我们在任何地方一说起撒库拉的时候，那就是“哎呀，这个玩意儿本来就是中国的，你不能把它推给脚盆是吧”。你只要说这个玩意儿是脚盆培育的，是脚盆儿的审美，是脚盆的这种精神的表现，然后他立刻就说“你这是为脚盆来偷我们的文化啊，这本来就是中国本家的”，然后如何如何如何。实际上这个就是脚盆的阴险所在。他呢，能够把他自己的东西借上一个中国的壳子，然后呢，塞到中国的家里面来，最后就会导致中国的东西变得越来越像脚盆儿。现在这种东西已经在建筑行业、服装行业，甚至植物审美上面渗透得非常非常地深了啊。典型的东西就是现在我们的一些传统寺院在进行改造了之后，经常会在前面装他的那个门——那个叫鸟居，对，鸟居。然后还有那个破风，建筑上面那个破风。这种形状的那个东西，在中国这里的话根本就不做成那样的。他那个破风只有一个装饰的效果，它下面没有连廊，对吧。所以你说他跟我们学了没有呢？学了。但是学完了之后，他其实消化吸收之后变成了他自己的，并且赋予了他的精神，然后呢，他再把这个东西说“哎，我是跟你学的，这就是你家的东西”，又塞给你。最后就导致我们变得越来越像他。

后来也是这个样子的。中国人历史上根本不存在赏樱文化，没有赏樱文化这个东西啊。我们所谓的赏樱看的是果实啊，我们喜欢的是樱桃。我们所欣赏的都是樱桃，会结果的那种。但是他给我们推过来的是什么？是不会结果的、他们培养的、用他们本地的樱花品种——尤其是以大岛樱为核心的副本去培养的、他们当地的观赏型纯观赏的撒库拉品种。然后呢，他推过来就借用了“樱”这个名字，让很多的中国人误以为这个东西和我们欣赏的樱桃一样，但实际上压根不同啊，亲缘关系也特别的遥远，审美的逻辑更是八竿子打不着。而且呢，有果实和没果实的差距是特别巨大的。因为“樱”这个名字就源于璎珞啊，它本身就是在描述樱桃这个果子长得像一个一个的小宝石一样，不灵不灵的，成串成串的，真好看呀。“樱”的名字就得源于它的果实。但是 SAKURA 结果吗？不结果，它是完全背道而驰的东西。可是呢，他就把这个东西强行地推过来，并且通过文化的包装告诉你“这东西就是中国的”。除了中国人认为这玩意儿是中国的，全世界所有的地方都认为它是脚盆的。所以你你看他的这种操作的手法，是一种渗透式的，一点一点地参与进来，然后逐渐置换的一种操作方式啊。

#### 四、枯山水与园林文化的置换

杭州有。其实现在好多地方都有，尤其是像那个什么枯山水啊。现在有不少的地方搞那个园林建设的时候，就会在里面掺杂枯山水。大家都见过吧，经常网友就会嘲笑它叫做“猫砂盆儿”，大型猫砂盆儿就是长那个样子，但是他就是把它建设成那个样了啊，到处都给你整猫砂盆。那枯山水这个玩意儿，你真的是觉得它是中国原生的吗？山水确实是中国原生的，但是枯山水是吗？中国人是非常讲究自然而然的。我们看过《红楼梦》的应该知道一件事情，就是那个贾宝玉跟他的父亲去逛大观园的时候——园子刚建成，他去写对联的那段日子，那一场戏当中，贾宝玉专门去描述了这个稻香居里面的东西，认为它非常的缺乏自然感，它的水是没有源的，它的山是没有脉的，在整个富丽堂皇的这一片中间突然来了一个茅棚草舍，是一个极度不自然的状态。其实这个里面蕴含了一部分中式园林的基础思想，就是我们要求它自然。所以你去建设任何一个园林的时候，考虑到的东西就是它的全局观，以及呢，能够让它模拟出自然的生态，和整个你在行走的过程当中、蜿蜒的过程当中，都能够看到各种各样的自然的景致。那么如果摆一块石头要它像山的话，它必须真的像山；如果有一道水要象征河流，那你要真的有一道水。所以中国园林当中的山水讲的是自然和天真，以及确实在自然界中存在啊。

但是呢，日本那种——脚盆的那种，很明显是一个极度简配版。他整了一堆沙子，画两条花纹，就说这玩意儿就是水波荡漾了，需要你去想象。然后他把这个想象呢，归拢到了这个禅意当中去，最后就是你要是想不出来，你就是缺乏禅宗的这种顿悟的思想啊，就是能力不足的表现，他还 PUA 你。但实际上他为什么要搞这种简配版？他但凡是真弄点水不好看吗？资源不行，资源禀赋太差了，他做不出来。所以呢，他就什么玩意都搞简配版。减配了就减配是吧，你地儿少，你把屋子盖得小一点我也不介意。但是你硬是要说，你那个东西才是好的、才是对的、才是我们的本源，然后你就给我塞回来了啊，这个东西就不行哈。这是两边儿，它实际上是有审美上面，以及整个意象层面上、文化内核层面上、思想层面上的偏差的。但是他通过这种切入的方法，就直接给你一股脑地塞过来。他的威胁要大得多哈。鸠占鹊巢，我觉得这个描述还是比较准确的。他是有那种寄生型的这种关系的，他是依附于一个群体存在，但是呢，很快的他又希望这个宿主能够把所有的养分都给他，最后他做大了，然后把宿主吃掉了。他喜欢搞这一套东西。

#### 五、类似手法：清汉女服装与历史记忆

嗯，实际上这种操作的手法，我们这几天都在互联网上看见了，这个所谓的“清民女”改名“清汉女”，和这个操作手法如出一辙。嗯，哪怕是在民国的时候，这个概念也是很清晰的，就是清代是没有汉装的是吧，剃发易服不就是玩的这个事吗？就是不允许你有汉装，不允许你有自家的衣装啊。不光是汉族，只要是他能够实控的地区，全部都这样，所有的民族都不可以有自己的服装，全部都要照他的版式来。这个就是他专门创作出来的东西。但是你看现在他硬是冲上来说，这玩意儿就是汉服的一部分，就是把他自己的东西然后说这个东西就是你的，然后塞给你。那 SAKURA 明明不是我们欣赏的东西，他一定要塞过来说“这本来就是你的”，你要是抛弃撒库拉的话，你就是割舍掉了你自己的植物欣赏文化。我们本来就没有这个项目呀。你像那个，清民女装，本来就是历史进程当中那个阶段大家被压迫、刻意地被摁头穿上的东西啊，而且是用杀头的方式穿上的东西，然后你非得说这就是你自愿的，你非得说你自愿——你不是自愿的，你是你是不是要不团结了？非得说谎才能团结？为什么要背叛自己的祖先才能团结呢？是吧。

嗯，背后那哥们儿你被放出来了？我背后哪个哥们儿啊？谁把你关了呀？我这里没有人关人儿啊，你想多了，我这里压根儿没有管理员，我这里也没有背后的那个人。你如果是你被关起来了的话，那不是我关的。阿呆还在吗？不在啊，阿呆不在啊。谁在说撒库拉是中国的？没听过。那等到那个下一季，大概是明年的四五月份吧，我印象中，不是四月吗？三月呀，反正大概就是这个时候吧，春天刚刚温暖起来，春暖花开那个季节。你在那些包含了像南京、武汉这些每年都会办樱花季的城市啊，你就在这些城市的旅游的相关话题里面去找，你就能看到每年都会办专门的樱花活动，而且是海量的各种各样相关的宣传片都会有。你在这些里头，你但凡在他的视频底下提问说“这个玩意儿不是脚盆培育的吗”，立刻就会有一大堆人冲上来说“你是脚盆派过来的间谍，你想把我们祖传的 SAKURA 送给脚盆”。你只管去，你试一试，捅一捅马蜂窝。我每年都捅一次，所以我身上马蜂可多了。

#### 六、杭州砍樱事件的现实案例

嗯，杭州整个城市都变成了樱花，甚至把桃李、桃花、杏花挖了种樱花。现在在全国有很多地方这个样子啊，不只是杭州啊，好多地方是这个样子。但其实当年我们赶走了倭寇之后，杭州那边曾经发起过一次大规模的市民活动，就是把那个樱花树砍掉去种桃李。杭州的市民主动地拿起自己家的工具，去把这个樱花树砍掉，去种了桃李。包括了像苏白堤这样的地方啊，以前的那些什么柳树什么的，其实就是在那个时期种上的。而为什么在那个时候种？因为在那之前倭寇入侵的时候，他们种了很多的 SAKURA。在倭寇的自己的概念当中，传播撒库拉就是在传播他战胜了，他在哪个地方种下了撒库拉，哪个地方就是倭寇的故乡。所以它本身就是把这个玩意儿当作他战胜的一个标志物，当作他圈地的一种标志物、文化标志物的。它自身就附带有军国主义的倾向，而且这个文化也不是我们给他的，是他们自己打包上去的。他们的勋章也好啊，战争纪念品也好，上面都是撒库拉。倭人他自己就是这样搞的呀。结果现在你说这玩意儿有军国主义倾向，人家说你这个是好战是吧，真相不允许你说，说了就是不和平。那凭什么和平要用谎言的方式呢？如果都靠谎言就可以和平的话，那世界早就和平了，多少人会撒谎啊。
