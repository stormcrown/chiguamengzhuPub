# 第1145章 衣冠明志_从汉服礼服看华夏文明的高度

《衣冠明志：从汉服礼服看华夏文明的高度》

《观服制有感》中写道："广袖曾昭万廪丰，短衣岂解黻衣功？蠹蝝妄议云霓色，不见仓颉造字工。"

汉服当中有很多款式，也有相当多的适合劳动的款式。但是，为什么在我们进行复原的时候，专门强调了礼服？而那些媒体进行攻击的时候，也特意强调礼服？因为礼服昭告了我们的历史、现实啊。劳动的这些服装确实是我们服装体系的一部分，但是礼服昭告了一个非常重要的现实——就是我们拥有极其先进的生产力和极其发达的文化系统。没有这些东西支撑，你是不可能使用宽袍大袖而且放量极大的这种礼服的，你压根儿就不能穿。所以任何时候，我们做传统服饰复原（全世界所有的传统服饰复原）都是奔向华美的、精致的、贵族的、礼服式的、不便利的，就是这种原因。

所以当有人进行这种攻击的时候，你就直接告诉他：我就是喜欢礼服，你管得着吗？为什么喜欢礼服？因为我吃饱了。为什么我的祖先会穿这些不方便的衣服？因为他们也吃饱了。你的祖宗饿着肚子不归我负责，他之所以一直那么勤俭，一直穿小袖，一直穿小裤脚，是因为他每天都需要打猎；而我的祖先不需要——就是一个富。

嗯，他在批评我们的时候是没有讲任何逻辑的，对吧？我，我也是看到有人说：这些年我们基本上，他的那些攻击我们都有理由可以驳斥回去，但是人家不看。因为他的目的并不是说跟我们讨论问题——讨论问题不会用他这种智障的方式的。他不是来讨论问题，他是过来栽赃陷害的，知道吗？他说，哎，你的这个衣服如何如何如何——他是过来栽赃陷害的，不是来听你讲道理，也不想跟你讲道理。所以我们所辩解的一切的道理都没有用。所以我觉得咱们就该打的打回去啊——他说你这个东西根本就没有用，你这个东西就是什么封建、复辟、保守、落后、愚昧，然后什么不利于劳动，然后不利于啥啥啥的。就说什么没有这个……就是宽袍大袖的礼服之所以穿着，是因为吃饱了撑的；我的祖先也吃饱了撑的，就这样！因为饭特别多，粮食特别充足，所以我们闲的没事干，就喜欢把自己弄得老漂亮老漂亮了，就想要用衣服表现出我不需要干活哦，就希望用衣服表现出来我四体不勤、五谷不分，就这么得瑟，咋了？嗯，啥啊？哈哈哈哈哈哈！

他要讲道理你才能讲道理啊，他们做的这些事情不是来讲道理的。铺天盖地的胡说八道、这些毫无因果联系的东西——你以为他在讲道理吗？他没讲道理。你跟他讲什么道理？就主打一个互怼嘛。怎么互怼啊？你说我不便利，我说你穷，对吧？你我大袖不方便——我大袖我富裕啊！你为啥小袖？你为啥短袖？你吃不上饭，你可不就小袖短袖吗？武力对轰让他疼啊——他怎么驯化我们呢？是不是我们说两句正经话，他就开始让你疼，天天围攻，围攻起来的，让你感觉到痛感。他在对我们进行驯化，怎么着？我们这里不能动？

嗯，也没见有人穿和服干农活呀。全世界所有复原的古典服饰、传统服饰，就没有拿来干活的，知道吗？因为全世界干活的衣裳都高度相似——衣裳布料这东西不管放到哪儿都挺贵啊，放到中原以外的地方更贵啊。因为我们这里呢，起码说男耕女织，家里面还能支二尺布；别的地儿更糟心，他们连家里头连个灶台都没有地方，你还指望他能有纺织机啊？他那里更难，好吗？所以更早时候，那些农民下地的时候，连个汗衫都舍不得穿，能光膀子尽量光膀子，因为被汗浸湿的衣裳有盐渍，你就得洗，经常洗它就容易破，这玩意儿是硬损耗啊，这是财产损失。所以但凡是干活的，都是能少则少。拿什么所谓的干活来扯犊子呢？那就是往身上贴一个所谓的左翼进步、热爱工农的旗，然后本质上是干啥？本质上是攻击中国的传统文化。你就不用理会他们啊，直接打过去——我为啥穿宽袍大袖啊？吃多了，我富裕，我宽敞了啊，家庭条件好，我就爱穿礼服。
