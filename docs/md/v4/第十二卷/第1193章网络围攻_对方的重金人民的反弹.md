# 第1193章 网络围攻_对方的重金人民的反弹

网络围攻：对方的重金，人民的反弹 [吃瓜蒙主]  
放心，只要有点什么事，评论区必然沦陷，私信区也必然沦陷。所以为了保证精神健康和心理的稳定，我从来不看这些地方。辛苦我的小伙伴们，你们肯定是在各种评论区、各种点评的地方都会跟他们遇上。而这些人呢，是撒泼打滚，根本不讲道理的、颠倒黑白的，而且是特别擅长于断章取义、移花接木的。基本上满嘴谎言，然后呢，直接说别人在说谎。你去面对这些人的时候一定会很难受，所以如果不想战斗就不战斗，不要在意这些事情。要保证自己的身体和心理的健康，这才是最重要的。因为这些人不会永恒地待在这儿，他们不是发自本心的。  

你要知道，如果真的是某位小姐的粉丝，看到我的视频的话，她不会做这样的决断。因为我整篇的视频从来没有攻击过任何一个人。之所以有人会出来这样子跳脚，是因为他们被某些势力裹挟，甚至其中有一部分带队的人，是被某些势力收买、刻意为之。所以咱们就慷慨地围观一下，他们的表演不就完事了吗？大家不用特别地在意这一波的什么所谓的网络围攻啊。  

为什么他们会进行这样的网络围攻呢？是因为他们无能为力。要知道这一波还不光是在我们国内的互联网，是在全球范围内进行的重金推广，结果居然能够出现反弹的声音。这一点是让他们非常痛苦的事情。因为他突然意识到，原来中国人的，哪怕是对这些事情一无所知的人，在看到的时候都会感觉到不太对劲。能够普遍性地让完全对此事无知的路人都觉得不太对劲儿，实际上是这20年中国人在崛起的道路上面不断追溯传统（守护）的一个产物。可以说，虽然革命尚未成功，同志仍需努力，但是已经有了阶段性的成功了——能够让那么多那么多的人意识到这个东西不对味儿，这是伟大的成就。  

所以这些人为什么那么跳脚？你们居然醒了！何等的气急败坏！我们看他气急败坏，我们应该开心，应该抚掌大笑，应该举杯畅饮来（欢）庆。你想一想，这样重（资）的出击，结果引来这样的力量反弹，我都要觉得吐血了。我天哪，我这些功夫——花了几年的功夫培养的这一记重拳，居然打完了之后刺还扎手了。
