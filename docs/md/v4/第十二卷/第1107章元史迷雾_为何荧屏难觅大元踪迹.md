# 第1107章 元史迷雾_为何荧屏难觅大元踪迹

**元史迷雾：为何荧屏难觅大元踪迹**  
《元鉴》  
铁骑曾惊欧亚风，文明未筑九霄宫。  
权争暗涌金帐里，税蠹明摧黔首中。  
西去王孙融异俗，南来帝子滞华嵩。  
百年霸业如朝露，留与荧屏话塞鸿。  

为什么元朝的影视剧很少？  
那个敏敏郡主的故事叫什么来着？啊，金庸笔下的敏敏郡主出自《倚天屠龙记》。有人问：“那集叫啥来着？”对，对，就是《倚天屠龙记》。这不就是蒙古背景吗？不然呢？还有人提到“赤兔崛起”，哎，蒙古背景的作品其实不算少。前几年还有一部正剧，讲谁的来着？应该是铁木真吧。前些年确实有一部《成吉思汗》（剧名我记得不是特别清楚），相关的剧是有的。  

但你觉得元朝存在感不强也很正常。因为无论讲什么，在那个时代，蒙古人除了骁勇善战之外，其他方面的表现确实不多。实实在在的说，就是很少。除了善战这个部分，其他元素真的很少。你让他们怎么演？总不能翻来覆去只讲骁勇善战吧？  

元朝的统治不足百年，在这不足百年的时间里，蒙古王公们基本上啥也没干，内讧倒不少。元代一直有权臣试图架空皇帝，皇帝与蒙古王公在朝堂上互相掐架，就折腾这些事。对于整个国家来说，他们的管理约等于无——直接承包、转包给那些色目包税人、豪强、地主、士绅。这些人对农民、工匠敲骨吸髓。元代的大致进程就是如此。蒙古人在整个社会活动中的参与度不高。哪怕讲这个时代的故事，你也会发现只有到一些特定场景才会出现几个蒙古人。因为他们的社会参与度确实不高，实际控制能力弱，人也没多少。  

当年蒙古西征、南下扩张，整体上在欧亚大陆进行快速扩张，以极少的骑兵用闪电战方式推进。这带来了一个醒目的效果：扩张速度极快，范围极广。但同时伴随另一个现状：实控力在所有地方都很差。包括那些王公贵族和蒙古骑兵，核心圈层到哪儿就与当地人通婚、学习当地文化、采用当地习俗。这种症状到处都有。后来有一波人——当然不完全是蒙古人，是突厥化的蒙古人——建立了莫卧儿帝国，去到印度那片土地后，基本与波斯女性反复通婚。留下的绘画作品里一点蒙古人的痕迹都看不出来。这些西征的蒙古人跑到中亚后还改信了宗教，到哪儿就和当地系统兼容，主要原因是自己缺乏管理系统，甚至在宗教领域也缺乏清晰的系统。所以到哪儿就变成当地的样子。  

相对来说，保存蒙古特征较好的反而是在中国。至于是刻意为之还是学习有困难，不好说。但确实在中国这块的蒙古人，蒙古特征保存得比较好，也可能是因为统治时间比较短。刚才瞟了一眼，有人说蒙古王公都不会汉语。其实不是的，他们学了。包括蒙古皇帝，有些皇帝学习了汉语，还学古诗文，有儒生当老师。当时有个儒生写了反诗被人举报到皇帝面前，皇帝说：“不管，无所谓。”人家看得懂。皇帝都看那种诗歌了，你说呢？他们还是有一定学习的。在着装、习俗等方面，也接受了相当一部分汉文化。所谓“蒙古人完全没有汉化”，实际上是后来清代编出来的故事。在蒙元期间，他们学习了一部分文化和语言，也包括服装——元早期的服装和中后期其实不一样，是学习了的。蒙古皇帝还穿棉服呢，系统性地学习了一些东西。  

我个人认为，蒙古当时处于草创阶段。今天的蒙古人所信奉的比较清晰的祖先，是从成吉思汗这个阶段才开始把族源梳理清楚的。可以说成吉思汗是一个缔造者。在那之前，蒙古划分了太多部落，彼此之间有姻亲关系，也有仇怨，甚至来历都不太一致。所以是从成吉思汗那时候才开始族群构建。这个时间进程比较短暂，自身拥有的可达成共识的文化内容并不多。包括蒙古文字也是借由回鹘文进行创作的——表音文字，比较方便挪用。连文字都是二次创作的东西。许多习俗在当时自己都没统合清楚，还没有核心内容的时候，去到哪儿很容易被当地成型的文化系统所吸引。所以你会看到，西征的蒙古大军到某个地方实施管理后，很多行为方式、甚至基因、面貌就与当地人越来越像。就是因为自身体量小，且没有形成清晰的文化边界线，所以很容易被其他人吸引过去。
