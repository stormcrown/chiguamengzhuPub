# 第1110章 列强无不怀念他鞑大清_无不害怕中国复制明朝模式

列强无不怀念他大清！无不害怕中国复制明朝模式！  

「沁园春·衣鉴」  
霓裳非裳，衣冠非冠，暗涌惊涛。看西风卷幕，强裁华服；东流遗韵，妄断旌旄。辫锁重楼，尘蒙汉月，忍见残妆饰旧朝。凭谁问，这绮罗经纬，可缚龙韬？  
须知服色昭昭，是青史凝成血未消。念赵武胡服，图强变法；苏秦纵佩，终破连交。衣载春秋，纹藏兴替，岂容他人代刻雕？凝眸处，待重理云锦，再织今朝。  

你知道“列强无不怀念我大清”吗？阿美家的女领导人到现在都还在喊，中国要是回到清朝就好了，我们就可以追上她了。你自个儿去看一下新闻啊，全世界都盼着中国到清朝去。然后中国人穿上了清朝时候被蹂躏、被虐待的衣裳，然后你说洋人不满意了？骗鬼呢！洋人自己都没有脸说这个话。嗯，洋人就盼着我们赶紧穿上清代的衣裳，然后你说，哎呀，你和BBC一样。BBC啥时候说了？你把原文弄出来，说BBC说了她穿的清民女装，那个对中国不好，这个衣裳不能穿。BBC啥时候说了这个话？你给我把原文翻出来。你们这些人就是专注在那里整一些谣言，然后开始“你给我交代一下”。阁下哪位啊？让我交代一下。你要是觉得我反华了，你就去告一下国安啊；你要是觉得我违法了，你就去报一下警；你要是觉得我侵犯了你的私人权利，你就去法院。不要跑到我的直播间里来，你给我解释一下，你给我交代一下？你谁呀？你也配得上让我给你交代？  

洋人巴不得给中国人安上辫子。何止啊！洋人每一次做中国人的那种负面形象宣传的时候，外国人所有的媒体、全球所有的媒体，包括大量的反华媒体，在做中国人的负面形象宣传的时候，采用的全都是清代的服装、清代的发型、清代的妆容、清代的滤镜。结果呢？啊，他说，哎呀，你不能说那是清代的，不能代表中国。谁说的？这话是谁说的？你敢不敢把正主拿出来？一个简单的衣服，让你上升到戴高帽的程度，是对的吗？对不对，不归你来判断啊！人民群众的眼睛是雪亮的。一件衣服丑还是美，能不能代表中国人的精气神儿？把一件衣服穿出去之后，外国人认为我们是梦回大清、想要当他们的奴隶了，还是开始梦回汉唐宋明、想要当世间的文化宗主国了？这是我一眼都能看得见的东西。你自己装瞎，不要把别人也一起拖下水。你可以没有审美情操，觉得那些丑陋的东西是美的，并且中国人都应该穿上，以展示我们的卑劣。但是中国人不同意，不同意就可以说出来。至于说谁的声量更大，谁背后的资金更强大，谁能够拥有最大体量的推流、全球发布说这玩意儿就是中国形象——是的，我们普通的平头老百姓斗不过这些大老板。他们可以买无数的广告位，然后让所有的人都穿上这件衣服，说，哎呀，这个衣服好看好看。但是人民群众的眼睛依然是雪亮的东西，好不好看，丑不丑，放得明明白白。全世界只要看到这件衣裳，就能想起来中国人当年为奴为婢的时代。结果你说这个东西就是可以代表中国，你凭什么？  

哼，怎么了？那为什么前苏联号称他们穿了牛仔裤就亡国了呢？牛仔裤算什么呀？九块九包邮呀，怎么就能欺负到前苏联那样的一个庞然巨物了呢？衣服突然就不重要了。哎，前苏联为什么灭亡？因为T恤、牛仔，因为那个收音机是吧？然后到了中国这里，衣服算什么？穿球服算什么是吧？穿清女装算什么？搞奇装异服算什么？什么都不是事儿了。能不能不要这么双标？一个人用一样的标准对待全世界所有的人行不行？能不能做到？衣服穿到别人身上的时候，那是别人的文化传播，那叫做能够继承自己的历史、继承自己的祖宗。然后穿到中国人身上，就是啊，刻板了，复辟了。衣服能代表什么呀？穿得好看啥也不是是吧？然后说凡是指出这件事情的人啊，那个指破皇帝的新衣的那个人是反华的，是跟境外势力联动的。那境外势力啥时候指出了这个皇帝的新衣呢？从来没有过。但是呢，他有谁说的呢？我也不知道谁说的。大概是一些很爱国的人说的吧。
