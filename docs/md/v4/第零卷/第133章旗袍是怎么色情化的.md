# 第133章 旗袍是怎么色情化的

旗袍是如何沾染“风尘味”的？

这个问题一抛出来，许多人都会炸毛。比如有人会说：“哎，我这种端庄女性也喜欢穿旗袍，它怎么就有风尘味儿了呢？”再比如：“我们的国母宋庆龄也喜欢穿旗袍，她怎么就风尘了？”唉，请不要偷换概念。旗袍上会出现风尘味儿，这与过去有没有许多革命女性穿过旗袍毫无关系。

旗袍是否有风尘味儿，这事儿需要由普通人来判断。只要提到旗袍，你难免会想到高开叉、露大腿、紧掐腰身、扭臀上楼梯的那般风情万种。看到某部电影时，是不是就想起这类印象？那电影叫什么来着？诶，我忘记了。据说女主角在里面换了无数件旗袍，每一件都十分美艳。然后呢，导演的镜头永远跟着这位女士的腰和臀，不停地摆荡。这时候你若跟我说旗袍非常正经、一点情色化倾向都没有，那电影讲的不就是偷人吗？你就说有没有吧！连创作影视作品的人都知道，当你使用这类镜头时，自然而然会对人产生什么样的精神映射。咱们就别狡辩了。

那么，旗袍为什么会有色情化的倾向呢？这事儿还真不是旗袍的错，而是服装演进史中无法避免、甚至可能是必然发生的情形。它与旗袍的发展史颇有关联。之前我们聊过：旗袍怎么出现的？它本身是想要去八旗化，却又被迫保留了一定的八旗元素，最终形成一种西式连衣裙。它带有革命的属性，想要推翻旧有的形制，却没有建立起一套独具匠心的自己的形制，于是便出现了这种夹生的、带有嫁接感和拼接感的服装。它只是一种阶段性的女式时装。

本来，这种衣服设计出来后，在城市女性尤其是上层社会中流行，也很正常。而且其中绝大多数穿着的都是正经女人，她们不可能把大腿一划就露给你看——不存在这种事儿。所以早期的旗袍确实没有任何风尘味：它只是一款长长的连衣裙，裙摆的开叉并不高，很多时候开叉不会超过膝盖。而且穿着旗袍的习惯，也和中国人传统穿袍子、裙子的习惯一样，就是裙袍下面是要有裤子的——一定要有裤子！记住啊，那些宣称汉服下面没裤子的人，都是无知的。中国人穿裤子的传统源远流长，而且一直穿裤子，男人女人平等地都穿裤子。旗袍和裙子下面也穿裤子。既然穿了裤子，开叉又不高，而且还是平面裁剪，其实它就是一件袍子——一件带有连衣裙剪裁方式的袍子，不可能有任何色情倾向。

但是它后来怎么就变味儿了呢？变味儿的过程其实有点无可奈何，或者说它必然会变味儿。这里我先提出一个小概念：人类社会，尤其是中国传统社会里，社会的时尚潮流很多时候是由娼妓业带领的。这是不分时代的，所有时代都如此——只要存在娼妓业，那么妓女、伶人这些人，她们的日常着装就会成为妇女们效法的对象，因为她们真的站在时尚前沿。本质上，这与现在那些小网红、服装流小明星给你秀最新时装款的性质，大差不差。她们有一定的社交属性、表演属性和社会公共属性，所以就会带动时尚潮流的改变。比如，某个娼妓喜欢在领口单独绣一朵花，之后那些贵妇们就也会开始在领口绣一朵花。这种风尚在历朝历代都有，不独只是清代，也不独只是民国。这个常识一定要先讲清楚：这不是在进行职业歧视，也不是时代歧视，而是说只要是娼妓行业，通常都会对整个女性时装的潮流起到引领作用。

为什么娼妓会引领贵妇的时尚？贵妇不嫌弃吗？还真不嫌弃，哈哈，真的不嫌弃。为什么呢？因为所谓的瞧不上娼妓，说她们是下九流等等，实际上是一种职业上的歧视。歧视的是什么？歧视的是“以色示人”这一点——她们脱离了自身的独立人格，用自己的美色去服务他人。但这种歧视并不针对她们身上的衣裳、打扮和妆容。美丽本身是没有罪的，谁不想要漂亮、不想好看呢？好看没有问题，但拿好看去讨好别人、出卖色相去消遣，这种情形才需要被贬低。所以，虽然娼妓受到人家看不起，贵妇们却依然会向娼妓学习穿衣打扮，因为美丽本身没有罪。

那么说到这里，就牵扯到旗袍的问题了。为什么旗袍最后会出现色情化倾向？为什么它会和娼妓产业绑定得那么深？主要原因其实与民国时期进行的那一次大革命有关。在对“剃发易服”进行洗地的那些人中，一直有一个话术叫做“男降女不降”。之前我聊过，不存在这件事——事实上是不存在这件事的。但这个口号在民国的辛亥革命中是真的正式提出来了的：我们在对旗人、对满城进行反攻的时候，大体量地放过了满人妇女，提出的口号就是“男降女不降”，是可以放过一部分女性的。既然对女性手比较松，就意味着有更多的旗人女性在这场变革中存留下来了。

但是存留下来的旗人，面临到一种很糟糕的景象。我们想一下，过去八旗子弟是骑在所有人脖子上的，无论他们是否已经落魄，身上都有一根“铁杆庄稼”保证他们优于其他人、优先享受待遇，在社会公正和价值评判上总是高别人一头。既然产生了巨大的、长期的、绵延式的阶级压迫，那么有压迫就有反抗。革命成功后，这些旗人自然而然地遭到了社会大众的白眼。因此产生了一个后果：虽然真正被革命清算的是那些负有罪孽、罪孽深重的旗人中上层，但大体量的旗人下层以及很多被放过的旗人女性，事实上也被大家看不上了——这是他们过去两百多年统治所留下的罪孽感。

在这种局面下，就会出现大规模的失业。要知道在民国时期，本来经济条件就不好，普通人就业和生计都很困难，但旗人会更困难，因为只要别人知道你是旗人，别人就会横着眼睛看你。事实上这种情况一直到解放后都还没有得到非常彻底的解决，这种心理惯性上的反感长期存在。在民国时期，它表现得非常尖锐：旗人群体的就业非常困难。既然就业困难，人又要活下去，怎么办？失业就要找工作，找工作受困，那就只有向下寻找工作。所以旗人的工作相较于其他群体格外下沉：男性很多都去拉车当车夫了——这种纯粹的重体力劳动，属于各种劳工阶层的底层，只要出力气就能吃上饭，不需要技术，也没有任何优待，人人都看不上，这种岗位里就会出现大量旗人。

同样，旗人女性呢？过去在家庭生活中不需要就业，可是在这种经济极端困顿的时候，女性也必须出来贴补家用。怎么办？在这种严峻的就业和生活压力之下，出现大量旗人女性沦落风尘——这是客观事实造成的结果，不是我今天又污蔑谁。当年就是这种局面，咱们在讲历史。在那段时期，有大量的旗人女性沦落风尘，娼妓行业当中出现了非常多过去的格格贵女们。这种现象在民国时期普遍存在，而这种情形的出现，就会影响到整体的服装。

当旗人女性穿着带有旗人元素的衣裳，在娼妓行业中普遍存在时，就会引领一次新的关于时尚的风潮。娼妓行业对旗袍的改造变得非常有趣。这里面倒不完全是旗人与非旗人的问题，实际上还包含了一部分西化的元素。前面说过，按照中国传统，无论是旗人还是非旗人，实际上在长袍下面都是穿裤子的。可是西方人不穿。西方女性搞女权运动的第一次解放，就是女人可以穿裤子——因为在那之前女性没有权利穿裤子。当时的西化风尚中，就包含了在裙子下面不再穿长长的衬裤。这是一个西化的风潮，并不完全由娼妓带来，但它确实先在娼妓业出现。

娼妓业中穿着旗袍的人，首先在旗袍身上进行了大体量的改良：包括收腰、显示身材，也包括高开叉展示大腿，还包括穿丝袜而不穿衬裤。这是一种西化的时尚风潮，带来的是整个服装向更加西化、更加贴身、也更加富有性意识的表现方式发展。然后这种风潮就从娼妓行业一点一点扩散出来。我们说过，娼妓业会引领妇女的时尚风。从娼妓业开始的旗袍改良之后，就流行到了社会普罗大众的层面，有很多追潮流的时尚女青年就会学习这种更加时髦、更加西式、更加有魅力的方式来穿着改良版的贴身旗袍。

不过这种时期并不长，那已经是民国中后期的事情了，早期并不是这个样子。它流行了一段时间，而且非常局限——基本只流行于娼妓行业以及一部分大型城市的社会中上层、非常时髦的女士当中。所以今天我们看一些影视作品，把民国时期的旗袍全部演绎得那么收腰，说什么“我又胖了一寸，这衣裳穿不了了，要去改”——这种其实很不客观。实际上有很多良家女子，是不愿意把腰身收到那么紧的。尤其像那些富有革命意识形态的革命妇女，以及女学生，她们穿着的旗袍是一种平板剪裁的、比较长且保守的旗袍，很多人一直到后来都还在穿着衬裤。那种大家印象中非常风尘的旗袍，其实流行于时尚圈以及姨太太圈。

所以在这点上，有些小说写得还是有点到位的：既然你想当少帅的第十八房姨太太，那么穿着这种性感旗袍就是天经地义的事情了。它是在特定群体中流行的一种特殊服装。所以，旗袍身上有没有风尘味儿呢？有的。既然今天流行的旗袍当中，有一部分非常着重于宣传这种特殊款式，那么它就是有的——因为这种款式的服装原本就诞生于风尘，原本就为风尘服务，身上出现风尘味儿简直是天经地义的。

所以，旗袍的风尘味儿，咱们先要怪谁？第一要责怪那些专注于拍摄风尘味儿旗袍的影视作品，它们没有把旗袍的全方位形象展示出来，而专门把那些收腰、紧身、露大腿的旗袍秀给大家看，而且专门拍人家的腰臀比。另一个方向，咱们要责怪一些境外势力进行的各种文创——所有的游戏、影视作品，只要里面涉及中国元素，就一定要把它穿成春丽装。大家去看一下春丽装什么样子：是不是收腰、高开叉、露大腿？对，就是那个模样。当旗袍的整体形象被刻板化地定死在这个位置时，人们的直觉难免会觉得旗袍有风尘味。

所以，旗袍怎么来的风尘？这样来的风尘。旗袍有没有风尘味？可以有，也可以没有。怎么样有？怎么样没有？要凭个人的选择。但是你不能一边穿着收腰、露大腿、高开叉的旗袍，一边说你不可以说它有风尘味。说真的，一件衣服是这种模样的时候，它是不是旗袍都有风尘味。你爱这个就爱，我也不反对，穿衣自由嘛。我去看这事儿的时候也不介意，虽然有时候会有点嫌长针眼，但没关系呀——你穿嘛，你愿意露我也可以看。可是你硬要说旗袍一定没有风尘味，显然也是不对的。
