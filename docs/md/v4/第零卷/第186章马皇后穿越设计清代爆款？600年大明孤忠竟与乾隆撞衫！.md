# 第186章 马皇后穿越设计清代爆款？600年大明孤忠竟与乾隆撞衫！

一个好消息，一个坏消息。好消息是，大明开国皇后马皇后搭上了时代的顺风车，穿越了——fashion啊！那他穿越到哪儿了呢？这就是那个坏消息：他穿越到了乾隆年间，并且做出了一个违背祖先的决定——为他数百年前的同乡设计一套来自遥远的西伯利亚地区的服装。啧，你听我说的这段话就已经足够鬼畜了。

那这是咋回事呢？咱们必须要讲一下事件的大致背景。这是一个来自于河北青龙满族自治县、据说曾经自称为格格的一位小姑娘。他呢，找到了一个新工作——恭喜他！这位新工作呢，就是到贵州地区去宣传贵州的屯堡服饰。然后在这个宣传的过程当中，他就变成了当地的形象宣传大使啊，化身为“贵州屯堡小娘娘”，去宣传贵州的屯堡服饰，并且号称这是来自600年前的大明遗风啊。说屯堡的人保留了600年前的大明风华，所以他们的衣装就变成了凤阳汉装的一个延续啊。凤阳都牵扯进来了，那当然要两地联动一下。于是呢，这位曾经的格格就带领团队到凤阳去参加传统服装的大展销。在这个传统服装的活动当中呢，他就给大家宣传了一下贵州的屯堡服饰，说她身上所穿着的这一身衣裳呀，就是当年马娘娘在凤阳设计出来的服装啊。这一套的衣冠呢，被当年迁徙的屯堡的军户们一直穿着，到了今天。600年大明遗风依然存在，大明孤忠就在贵州呀。

听起来之后，这个故事是不是特别的荡气回肠？这故事放哪儿讲，那都太漂亮了呀。这么好的故事当然是要拿来做文宣的嘛。所以呢，各地的文旅都对这个事很上心，很积极地推广，并且觉得太好了——这个故事一讲起来，我们这里就超有文化的啦。

然而吧，这个梗呢，是永远都过不去了：搞文宣的人呢，你得先有文化。一个来自于河北青龙满族自治县、从前的格格、今日的汉人姑娘，可以代表贵州的屯堡地区的传统劳动妇女，宣传他们以前可能也并没有穿着过的衣裳，并且代表他们去到凤阳宣传马皇后当年的服装设计。这个蜿蜒曲折的路程，就很容易导致中间出现信息的失真。而这个丢失掉的关键信息，就变成了某一种文宣上的短板，也就是今天我们不得不拿出来聊一下的：马皇后她当年到底有没有设计凤阳衣冠？有没有设计凤阳汉装？以及屯堡的这些女性，有没有传承大明遗风？有没有传承凤阳的衣冠、凤阳的衣装呢？啊，这里头就是“如有”。什么东西叫“如有”呢？就是你说有没有贵州屯堡？那是有的。贵州屯堡有没有劳动妇女？那也是有的。贵州屯堡的劳动妇女有没有传承凤阳汉装？那还是有的。但是问题在哪儿了呢？问题就在于，他传承下来的东西，跟这个来自于河北青龙满族自治县、从前的格格、今天的汉人身上所穿着的这件衣裳，就没有半毛钱的关系。因为他身上穿着的衣裳，是清代乾隆年间之后才开始草创成型，到道光年间才开始落地生根的一款专门的清代民女装。所以你看我为什么说马皇后穿越了呢？一个正常的马皇后，她只能设计出来明初的服装。但是吧，这个穿越版的马皇后，他可以直接在明初为他的故乡设计出来几百年之后才会出现的衣服。这个事儿就变得很漂亮了。

所以这里头呢，牵扯到了几个概念，咱们必须把它挨着个儿地梳理出来啊。关键词太多了，有点一头浆糊。咱们就从屯堡说起吧。

贵州屯堡的这个“屯堡”指的是什么？它其实呢，约等于今天的生产建设兵团。就是当一个中央王权开始抵定中原了之后，他会试图在边疆地区迁徙一部分的人口过去，从而对当地实施更加深刻的管理和统治。这样的生产建设兵团呢，担任了几种职责：一种呢是生产，所以要迁徙干活的人过去；一种是建设，所以会有所谓的屯堡；同时呢，他还是一个兵团，所以这些人呢，还是一个军户的身份。所做的事情呢，就是既要驻守边疆，又要传递经济建设与文化。所以贵州屯堡的这些人，从来都是非常地坚持自我，也非常地有文化自保的意识。这产生了一个后果，就是他说：“哎，他们传承了600年的大明衣冠。”这件事情有没有点影子呢？那还真有，真的有。因为贵州屯堡地区的男女，在这个剃发易服之后，他其实借由当地的特殊环境，以及与少数民族杂居的这样的氛围，所以他没有完全践行剃发易服。他的身上保留了很多明代的遗风和遗俗。因此你说他有没有啊？有的。

所以后面一个话题就来了：他保留了什么样的遗风遗俗呢？啊，这就是后来所说的凤阳汉装。凤阳汉装有没有？哎，这个词儿有，这个东西也有，而且在屯堡人身上也确实存在。所以咱还得弄清楚啥叫做凤阳汉装。实际上，凤阳汉装的“妆”是妆容的“妆”。那这个词它所代表的是凤阳当地的一种发型和鞋子的设计方案啊。这样的东西呢，叫做“妆造”——这就是我们拍电影、电视剧里头的时候，服化道那个产业的“妆造”。所以凤阳汉装它本身存在，而且确实也被带到了贵州屯堡。那么凤阳汉装指的是什么？这就很好玩了啊。真正的凤阳汉装，它指的是特殊的发型，还有包头巾的缠头巾的方法，还有绣绣花鞋的鞋子的款式。这个才是凤阳汉装。那他是怎么诞生的呢？以及为什么传递凤阳汉装这事儿很重要呢？啊，那这就不得不谈到鞋子和发型这件事情了。因为到了清代剃发易服之后，实际上鞋子和发型是被严重摧毁了的。

为什么这样？很多人对这个女装不会有非常激烈的印象，是因为他们认为“男降女不降，男从女不从”。所以很多人都没有意识到一件事情是：其实清代的汉家女子的发型，是发生了颠覆性的改变的啊。为什么这样？因为当时的汉家女子也遭受到了剃发。这个剃发有人说主动，有人说被动，但是我个人认为这是一个被动的。原因在于，当时的汉家女子流行的发型，从原来的“三绺梳头”，把前面高高隆起一个发包，变成了清代的那种很奇怪的发型。什么样呢？有的地区会把后面的发辫搭到前额来，变成一个额头上的小辫子；有的地方呢，会设计非常宽的抹额，把他的秃掉的前额遮起来；有的地方呢，就喜欢梳一种整整齐齐、有点齐得过度了、像是在遮蔽地中海一样的那种齐刘海，挡住她前额的秃发。这样特殊的发型，变成了清代女子剃发之后流行的发型款式。那这个显然就和明代的妆造完全不同了，因为明代不剃头嘛。

所以贵州的屯堡有没有保留一部分的大明遗风？有什么呢？发型保留了！这个发型呢，就是她不剃头。于是呢，发辫很粗，发髻很大，同时会包一个头巾。这是什么？这是标准的巾帼之治啊。所谓巾帼就是妇女的头巾嘛。所以明代的妇女保留下来的梳发辫、盘发髻、缠头巾的方式，被屯堡的女子传承下来了。这确确实实是凤阳汉装的一部分。还有一部分的凤阳汉装就是来自于鞋子。明代的时候呢，在贵族女子和时尚女子这里流行一种缠足法，把脚缠得细细又尖尖的，方便他们去穿那些很漂亮的时装鞋子。但是这种缠脚的方式，并不适用于明代的普通劳动妇女。而普通的劳动妇女会干嘛呢？她们就是大脚天足，然后会穿着一种鞋头上面有翘翘尖尖的款样的绣花鞋。这个也是凤阳汉装当中的一部分。而屯堡的女子就留下来了这种鞋子，因为他们从来不缠脚。那么这种缠脚之风到了清代的时候，急转直下，从原来的“修饰脚型”变成了“集体缠足”。而屯堡的女子拒绝缠足，也拒绝了过去的那一种贵族式的修脚型的缠脚。于是他一直保留了明代的劳动妇女的天足，以及特定的款式绣花鞋。这个也是凤阳汉装当中的一部分。

因此我们就会看到：关键词拿到面前，对不对呢？对。可是跟他宣传的东西就没什么关系。因为这一波他所宣传的所谓的凤阳汉装，是一直由那位刚刚的小姑娘在宣传，她身上的既短且窄的袖子叫做“宽袍大袖”？一直在宣传他身上穿的极窄的服装是凤阳汉装。然而，他身上的这件衣裳是什么时候诞生，又是什么时候流行的呢？这就真的很微妙了啊。因为他身上的这件衣裳是一个标准的窄版“厂字襟”搭配“蜈蚣扣”的清代民女服饰。这套服饰的诞生的历史不算太短啊。他呢，是乾隆年间初见雏形，道光年间完成设计，并且进行了大规模的推广。那这样的一套服装，为什么会在这个时候被设计出来？它和以前的传统服装有什么样的区别呢？最重要的区别就是它是一个“全密封”服装。“厂字襟”这样的结构，它有一个特点：一边的襟口极小，挂不住；另外的一边的襟口极大，会拖地。所以你要穿上它，必须用高密度的扣子把它钉起来。所以厂字襟的衣服就会搭配密密实实的蜈蚣扣。这种特殊款样的衣服会诞生在哪里？一个密封性能非常好、天热了你也不能解开的衣裳，会出现在哪儿？对，它会出现在极寒之地。所以这种款样的衣裳，实际上是诞生于西伯利亚地区的苦寒之地。既可以省布料，又可以有密封条。这样的衣裳，有没有可能是贵州或者是安徽人设计的呢？绝无可能。因为这两个地方的气候，热的时候热，冷的时候冷。他需要他的衣裳有通风换气，同时保暖可调节的功能。所以它非常需要设计上面的受力结构的对称性。所以它可以敞开穿，也可以系起来穿，可以叠穿，也可以单穿。这样的衣服才是符合这种气候条件的。所以马皇后当年有没有在凤阳的时候设计衣服？很有可能设计了。设计的是什么样的衣服呢？当然是明代初年的时候，专门推广的明制衣冠服饰。而当时的明制衣冠服饰，无论哪一种款式，它都有一个共同的特点，就是受力结构对称。它会是交襟结构、对襟结构和交领结构。但是“厂字襟”这种绝对不对称结构，完全无法出现。这种结构是在清代的时候才专门设计出来，同时进行了强制推广的。

那么请问屯堡人当年有没有穿这件衣裳呢？这也是其中的疑问之一啊。假设说屯堡人当初他真的就穿了这个衣裳，那虽然年代有点不对劲，但好歹说确实是屯堡服装，这事没毛病吧？嗯，这事儿又出毛病了。因为屯堡人当年没穿这个衣裳。为啥没穿？因为屯堡所处的这个地区是一个地处边陲，并且多民族杂居的地方。这样的地方由于地形和人员的复杂，以至于呢，八旗在这里非常难以实现控制。所以在当地确确实实保留了很多的传统服饰的款样，并没有直接被厂字襟蜈蚣扣全覆盖。实际上一直到清末的时期，我们知道啊，清末其实有很多外来的各种列强家里面的人在中国这里呢，玩探险呐，玩探索呀，搞地形测绘呀。这些人当中有不少人不安好心眼儿，但是呢，他们用自己的画笔和相机，留下了当时的很多的形象记载。因此我们可以说，你动机啥样我先不管，总之，一直到晚清时期，在西南地区的摄影作品当中，当地的妇女穿着的依然是交领交襟的结构。也就是说，贵州的屯堡女子，他真的在200多年间，扛过了剃发易服的残酷压榨，而真实地保留了大明遗风。唯一的问题就在于，今天的贵州屯堡宣发的小娘娘，身上穿的衣裳跟屯堡服饰不一样啊。你倒是穿着人家真穿的衣裳呀！为什么不能？这就不得不谈到说，这种厂字襟蜈蚣扣的衣裳，连人家八旗的剃发易服都没有能够推进贵州，最后怎么就会变成了贵州风俗的呢？谁干的这个事儿啊？这口锅啊，八旗背不起。因为他们当年实控能力确实力有未逮，没做到啊。谁推进进来的？这个锅就不得不甩到民国的常凯申身上了。

嗯，这个活儿吧，是这样的一个流程啊。其实呢，在孙中山辛亥革命驱逐鞑虏之后，中国人是有一个共识的：我们终于推翻了那个满洲贵族的群体，我们终于不用再接受双重压榨了。这个时候我们干什么？我们要恢复汉家衣冠嘛。所以在民国初年的时候，由社会的文化层群体和社会的中上层群体，联合发动了一次恢复衣冠的活动。在当时的很多摄影照相馆里面，都留下了当时的人亲自设计的交领交襟的服装。你说这些服装好看吗？我觉得设计得不太完美啊，但确确实实已经是在试图恢复汉家衣冠了。然而呢，这个风潮啥时候突然之间就断了呢？虽然在诱因上面很多，但是真正完全断绝，它是来自于1929年国民政府的一道政令。1929年的时候，蒋介石相关的这个团队发布了一套跟服装相关的政令，然后制定了民国标准礼服。在汉服复兴运动刚刚开始的这个时期，他界定了民国的标准礼服是旗袍加马褂。就很哇塞啊！在这个我们终于推翻了满清统治、终于不用再穿着旗装加马褂的这个时代，由蒋介石官方政府直接强行推广覆盖要求，说我们的官方服饰标准——旗袍马褂来了，走起！所以当年的八旗，由于兵马力有未逮而漏过的边陲地区的屯堡服饰，在这个时候就被蒋介石的近现代化的强行推广给覆盖进去了。所以旗装的厂字襟配蜈蚣扣的元素，就从这个时刻开始，逐渐地蔓延到了当年满洲贵族都没有能够解决的边陲地区，从此成为了所谓的传统符号。

而到了新中国的时候，我们试图解决这个问题。方案呢，一开头的时候是解决阶级矛盾的问题。我们说一切都走阶级矛盾的解决路线，于是呢，大家一起来推翻所有头顶的三座大山，其他的事情我们就先姑且放在那里。这个时候我们确实阶段性地解决了这个问题，因为当我们一起走向近现代化的时候，之前的这些什么服装啊、纽扣啊，就变得不那么重要了。然而呢，由北边的大体老师带队而来的很多专业的学者，在进行了民族识别之后呢，要求说你都识别出来了，你总得给他安排个服装吧。于是就产生了自50年代到70年代，绵延数十年的民族服装创作运动。在这个创作运动当中，有大量原本其实穿着交领的少数民族，就被专门设计出来了一套以厂字襟蜈蚣扣为底本的、崭新的传统民族服装。那在这个阶段上面，很多的西南地区的原本的少数民族以及屯堡居民，就这样被强行的推送了一波他们祖上没穿、但是此时此刻开始要穿了的新的传统民族服饰。所以在今天，当我们再一次地去进行宣传的时候，你说有一些地方呢，去找出了一定的服装的印证，说他们穿过厂字襟蜈蚣扣，可不可以？我觉得可以，因为过去几十年，他们确实曾经穿着过这种衣裳。你说这是他的传统服装行不行呢？如果说100年的传统也叫做传统的话，那我觉得也没有什么问题。传统服装你又穿过，你也接受，OK，可以可以，过关。但是吧，当你在宣传的时候，说这玩意儿是600年大明遗风，这就有点过分了。因为这套衣服能够被推送到屯堡，加在一起也不过百年的功夫。实际上强行的商业化、强行的推送，让大家信以为真的历史不过几十年而已。一定要说它有，可以有。但是一定要说是600年？这个牛吹得有点大。而且如果说600年前，马皇后在安徽的那种地方，设计出来了西伯利亚风的服装，那我觉得他在服装设计上面是挺失败的，因为大家很不方便使用啊。

所以今天的这件事情，我觉得再一次地给我们印证了一个很重要的真理：搞文宣的人要有文化。如果没有，就要找有文化的人问一问，了解一下：到底马皇后的600年的大明遗风应该长什么模样？官科不成你找民科，民科不成你找官科，总归不能去找一个随随便便出来的形象代言人吧。作为一个model，他是没有这方面的责任感的。没有人说model必须知道身上的衣服的每一个纽扣历史来源是什么，他没有这个责任。结果你就要求他给你讲历史课，让模特来讲历史课这件事情？嗯，挺荒唐的啊。所以历史要按照时间、要按照地点来进行逐层的分析，而不是张嘴一来：“这就是你祖宗的衣裳呀！”——我祖宗当年就是为了抗击这个衣裳，才特意蹲在那个地方几百年不改的。
