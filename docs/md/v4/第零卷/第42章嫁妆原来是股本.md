# 第42章 嫁妆原来是股本

我之前讲了一期关于彩礼的内容，于是大家就说我不公平。我说好像也对，是不太公平。所以呢，我们今天来公平一下，我们来讲一下嫁妆是什么。嫁妆这个东西也是自古以来啊，特别特别酷，哈哈哈。从有彩礼的时候呢，就有嫁妆。那么这种嫁妆，它到底是以什么样的形式去存在的？它起到的作用是什么？是不是它永远都是私产啊？离婚的时候我必须要带走，谁都不许碰啊？我们很多的影视剧啊什么的，传递出来的都是这个信息。那么请问是不是这样呢？哎，我们今天就来研讨一下这个问题。  

这个碎嘴子啊，永远都是重复同样的一句话：婚者何？两姓之好也。呵呵。那么两姓之好，我们说了，在先秦时期，两姓之好就是贵族之间的联姻，跟老百姓是没有关系的。因为那个时候“百姓”就是有姓的人啊。在后世呢，姓氏已经下沉到普罗大众的身上的时候，老百姓才是今天我们所认知的老百姓。在早期的时候呢，这些贵族之间的联姻，通常对对方的家族、对方的身份、对方的继承，将来会占家族财产的百分之几，都是非常清楚的。因为所谓的礼制当中最重要的一个很核心很核心的东西呢，就叫做嫡长子继承制。继承是所有家族财富累积当中核心又核心的东西。那么婚姻当然也就跟继承之间关系特别的紧密。  

既然婚者结两姓之好，那么是咋结的呢？我给诸位打个比方。你在今天所想象的“我爱你呀，你爱我呀，我们一起领个证儿啊”——啊，不好意思，那个时候不是这个意思啊。那个时候通常我们说，互相之间是没有这个什么谈恋爱的轨迹的。会因为是两姓之好，不是俩人之好。那么两姓之好咋谈呢？你按今天的想象可能很困难。我们来放大一点点啊。如果在今天的社会里面，所谓的两姓之好是一种什么样的谈判方式？你可以想象成这个样子：有一家上市公司和另外一家上市公司，每家各出一份股本，成立了一家子公司。这个就叫做婚者结两姓之好。而那个新生的小家庭呢，就是这个两边注资的子公司。  

这个子公司里面，当然啦，那个企业法人代表是谁呢？啊，那个男的这个事儿是法定的啊。因为那个男的他的财产呢，是固定了的。从他的家族当中，按照嫡长子继承制的分成方法，分到他头上的分到多少是多少。如果他正好还是那个嫡长子，他会继承爵位啊、土地呀、绝大部分的财产。那么在这种情况下，他要去跟另外一个家世匹配的女子进行婚姻联合的时候，你有没有考虑到，我们成立一个子公司，那么这个公司是不是有股本的问题？男人的股本是非常清楚的，他的继承权占了百分之多少就是百分之多少，今天有多少钱，未来能够从他老爹手上捞到多少，都很明白啊。这个东西就是男方的股本。  

然而女方这边的股本呢，就有点含糊。因为女方不能够继承她爹那边的什么爵位啊、什么国土啊、什么这些乱七八糟的核心财产，她继承不到，她继承到的都是一些浮财。那这个时候就好说了：人家出了这么大一笔固定资产，你这里呢，带来的是一笔流动资产。然后你说我们两个合了资，然后这个女的要担任什么岗位呢？合资了之后，她要担任的是后勤总管家、财务总监，很核心的岗位啊，非常核心。如果她是嫡长子妻的时候，她还要当宗妇，还要掌管家族祭祀。这么重要的一个岗位，这么重要的一个权责，在这家子公司当中担任一个核心顶层的高管，你说我就带两百块的流动资金进来入股，行不行呢？类比清楚了没有？哼哼哼哼哼，不行对吧？  

放到今天谁敢跟我这么谈合作啊？大耳刮子呱呱的就抽上去了，什么合同现场就给你撕了，谁跟你干这个活儿啊？要不要点脸啊？所以什么东西是嫁妆？早期的嫁妆就是在弥补这个女性外嫁的过程当中，她身上所携带的资产的不足，就对它进行一个补强，这个东西就叫做嫁妆。为什么很多的书里面强调的一件事情是说，女人带的嫁妆越多，她腰杆子越硬呢？你想你都投资公司了，你是不是股本占比越高，你在公司里头说话的时候越硬气？“我才是投资人，你那么多话，管是谁登记的，给老娘待着！”这个东西就叫做嫁妆的力量。  

所以在这种情况下，你就会注意到一件事情啊。所谓的“嫁妆永远都是女性的私产”，这件事情在宋明时期的法律里面确实有相关的规定，然而它实际上是没有办法百分之一百兑现的。因为任何情况下，当你股本合资，钱已经挪进来了，家庭生活要有消耗啊，然后要有共同的支出，要有各种各样的损耗。你们共同经营的情况下，共负盈亏的情况下，怎么保证你的嫁妆进来啥样，出去还是啥样？凭啥呀？你这是啥投资方法啊？盈亏都归人家的，我这里永远都保本？我跟你讲，在现代的法律当中，这都是违法的。投资是不能够要求保本儿的。一旦你企业投资，你说你要保本儿，这叫做非法放贷。放在古代也是一样的。当我们已经合资成为一个公司的时候，我这里的家庭继承财产和你那里所带来的嫁妆之间，就是彼此之间的股本放在了一起，共同经营、共同管理、共负盈亏。  

尤其是当古人的离婚率超乎你想象的低啊，没有人在结婚的时候，就先想好我们离婚的时候要怎么分，离婚的时候我的嫁妆应该带上多少多少银两走。那个时候没有人想这个问题。几乎我们在结婚的时候都是一结定终身的。那在这种情况下，这个投资是必然会有损耗，也必然会在不断的经营过程当中互相交融，成为真正的共同财产。所以你说啊，它就是女方的绝对的婚前财产，谁都不能碰，那是不存在的。哼哼，理论上可以有，但实际上不存在。一定有损耗，一定不可能原模原样地拿回去啊，只能说基本给一个保障。  

那么问题就来了：先秦时期的逻辑的确如此，但是当我们社会走向平民化之后，人都能结婚了，还是这个样子吗？哼，不要想了。我们说过了，在彩礼当中啊，我们专门讲过这一条：当我们的婚姻走向平民化之后，对于这些经济条件不怎么样的家庭，钱就是最重要的诚意。所以彩礼在这个阶段呢，就常常表现为买卖婚姻和买断人生。那这种时候你还指望说“我陪嫁给你什么很海量的东西”？大部分的时候就不会了。所以呢，收了很高的彩礼，但是只陪送仨瓜俩枣的那个日用品，这个情况在非常非常平头百姓的这个地方呢，就常常会存在。这个时候妻子在夫家的地位呢，就显得比较岌岌可危，然后一辈子的辛苦劳作，这个情况呢，就比较普遍。  

但是如果这个家庭她比较看重这个女儿，她会做什么呢？她会做到让她的嫁妆更加丰厚一些，腰杆子就硬一些嘛。这就说明什么？它至少说明她的父亲、母亲这边的家庭，对她有一个很重要的期待，对她有一个很重要的认可，将来会给她撑腰。这个才是它的一个核心价值。  

所以今天当我们看法律规定、法律解释里面的时候，其实你会注意到，我们会强调嫁妆是女方的婚前财产啊，是女方的个人财产。理论上是这样的，没错啊。但是所有的婚姻，真正的婚姻是一定会产生财产交融的。如果你说你进来，你结婚的那一刻就算计着说“我们离婚的时候怎么剥离”，我建议你那嫁妆就别拿了。装成这个样子干啥呢？多不多余啊？费不费劲呀？何苦来哉？是不是？我一贯对婚姻的态度是这个样子的：要么然真心相爱，人人都祝福；要么然我们势均力敌，互相合作，共度今生。但是一旦进入到买卖婚姻——“我亏了呀，你要给我钱呀”——到了这个阶段的时候，哎，痛苦就随之而来了。因为你只能端着算盘算账，从结婚的那一秒开始就不断地算计：离了之后什么归我，什么归他？我怎么能够捞到更多的钱？这种日子何苦来哉？  

所以嫁妆是什么？嫁妆本来应该是对一个家庭的共同投资。后来出现了卖姑娘的情况的时候，嫁妆变成了一个空有名号的名义上的符号。而今天的嫁妆是什么呢？今天的嫁妆，你期望它是什么，它就是什么。我们依然可以选择投入到家庭当中去，成为共同建设的一个资金。但是如果从结婚的那一刻起，就在谋算着离婚的时候可以拿多少，那你就抠一点啊，把你那个嫁妆锁得死死的，千万别让人摸着，最好让人家都不知道你还有嫁妆这件事情。到时候你想拿走就拿走，法律规定了那个东西永远都是你的，多好呀。但是当你做这个打算的时候，请问你结婚的对象知不知道？
