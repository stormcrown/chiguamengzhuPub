# 第25章 日本侵华，是怎么做到理直气壮的

有没有想过一个问题：日本这个国家，做了很多次侵华的尝试，已经是熟练工种了。我们今天回顾一下，当年日军侵华的时候，在文化方面做了些什么。很多人都知道，他们编了一套历史观。基本上遇到的时候，就会有人把它翻出来骂一遍。我的主张是可以骂，但同时要向对方学习和借鉴，要了解他的思维底线到底是什么，知道他为什么这样做，他到底抓住了什么弱点、什么软肋，它才会起作用。从所有的教训当中去学习，这是很重要的一件事。

当年日军侵华之前，在文化进攻的前哨上做了哪些铺垫呢？大家很熟悉的，有一样叫做“崖山之后无中国，明亡之后无华夏”。这件事情在网络上面被批驳得臭不可闻，所有人都说：“不对啊，他说的就是放屁，贼子之心不死啊，他完全是在胡说八道。元清肯定也是中国，那是中华正统。”东西就倒向了这个方向。日本人表示很欣慰，他达成了目标。

今天我们来梳理一下他当时所做的事情，他可不只说了这一句话。在文化侵略的前哨上，他一共立了三条最重要的纲领。其中的第一条就是“中国可以灭亡”，这个前提就叫做“崖山之后无中国，明亡之后无华夏”——你曾经亡过，那么今天再亡很正常，中国可以亡，这是第一项。

第二项就很有意思了，也是日本人积极主张的东西，却是今天的互联网爱国人士非常非常推崇，甚至觉得是中国人立身之本、中国团结之纲领、中国伟大千秋伟业的基础——夏入夷则夷，夷入夏则夏。大家应该都听过吧。

第三板斧是什么呢？第三板斧就是“日本是中华正统”。这三样东西结合在一起，才成为日本人能够入主中原的一个前提条件。我们看他是怎么配合的：第一，中国是可以灭亡的，所以今天你又衰落了，又灭亡了，很正常。那么你灭亡之后，我能不能来呢？如果说中国必须是本族人民才能建立的国家，那么我来就是不正确的，对吧？但是夏入夷则夷，夷入夏则夏——唉，谁来谁是中国人。这就是今天很多爱国人士的主张啊：谁来谁是中国人，我爱中国就是中国人。日本人也是这样主张的。日本人当初非常非常爱中国，非常努力地学习中国文化。联军入侵的时候，别人抢古董、抢金银、抢宝石，抢得一溜够，日本人抢了大规模的古籍过去。唉，他爱中国，爱得彻头彻尾，于是他就觉得自己很有资格。发现了没有？中国可以灭亡，谁来谁是中国人，配合战一打，日本人就拥有了来中国的权利。

第三项就是“我是中国人的学生，所以我是中华正统”。听起来很有道理嘛。今天是不是有很多人说“中国是一个伪装成国家的文明”？那既然是一个文明，文明就可以通过教育来传播，学生当然可以继承老师的衣钵。于是在这种逻辑之下，日本人作为中国人的学生，继承了华夏衣钵，于是他就是华夏道统的正宗继承人。三板斧旗下，日本侵华就拥有了本质上的正义属性，这也是他建立大东亚共荣圈的文化基础。

所以有没有觉得很有趣？日本人当时主张了三个项目：一个是“崖山之后无中国”，一个是“夏入夷则夷，夷入夏则夏”，一个是“日本是华夏正统”。而今天在中国的互联网界、学界，甚至很多所谓的文化人的心里，除了“崖山之后无中国”是假的之外，后面两条都是很正当的——中国是个文明嘛，学了中国文明的当然就是中国的正统了，所以很应该；夷入夏则夏嘛，谁来谁是中国人嘛。除了“中国可亡”这件事情被否定之外，基本上日本人想要侵华的文化基底已经完全达成了。也就是说，今天如果日本人再高举“中华正统”的旗帜、传承中华文明，再来到中国，它拥有了绝对的正义性——因为今天的中国人已经认可了日本人的侵华主张。

那么我们现在来解析一下他这个三板斧。

第一个就是“中国可以灭亡”——“崖山之后无中国，明亡之后无华夏”。这件事情其实是在否定元和清的正统地位，而这件事情在过去其实是一个共识，它并不是日本人带进来的。这句话是真的，中国人说的是真的中国人的主张——虽然不是原话，但是中国人是这样想的。因为所谓的“中国”“中国人”或者“华夏”，其实并不是一个很单纯的国土概念。今天的很多中国人把中国的概念理解为某一片土地，其实这是错误的。中国有土地的概念，但这个土地的概念并不绝对，因为土地可以扩张也可以缩小，它不是一个固定地区的概念。它其实是杂糅了民族、国土、文化三位一体的一个概念，是一个综合的概念。所以实际上，它是中国人带着中国文化、运用中国制度所建立的国家，叫做中国。至于这片国土，河南失守了可以移到湖南，湖南失守了可以移到湖北，湖北失守了可以移到江西——这个国土是可以换的，但是人得是这儿的人，文化和制度得是这儿的文化和制度，这个东西就叫做中国。所以中国人过去是确实认为，宋和明的灭亡是非常严重的“亡天下”。

在中国的古代观念当中，“亡国”根本不是一件很重要的事情，因为中国人所谓的“国”是一个比较小的概念。像周代的时候，最上面的统治者叫周天子，下面叫做列国，所以任何一个有王侯的地方的驻地都可以称为“国”，它带有行政的属性、区域的属性，甚至说哪一个省拥有了某一定的自治权之后都可以成为“国”。所以“国”这个概念并不是一个非常高的概念，它是一个低级概念。同时“国”能不能亡呢？可以亡。例如秦是不是亡了？秦二世而亡嘛，作为一个国家它灭亡了，灭亡了又怎样呢？大家也没怎样，换成汉不就完事了吗？包括汉最后是不是也灭亡了？灭亡了之后也分了三个国家，三个国家之后又崛起了其中一帮势力，又成为新的国家——不断亡国的过程，不断的亡国，不断地重造，司空见惯，稀松平常。中国人对亡国没有任何的意见。

为什么到元和清的时候，我们会发现在王朝末年，任何一个王朝的末年都是稀烂的，别说今天大家都知道，说宋末那一团糟，不像人样，一帮子怂货骂得够惨了吧。但是宋代末年的时候，出现了大规模的殉国现象；明代也是这样，明末的时候何其黑暗，民不聊生，真的是民不聊生，但是明亡的时候大规模殉国。这和其他的任何朝代改朝换代都不一样，之前的每一次亡国大家也就是换个皇帝，但是到了这两个朝代的时候，很明显就不同了。那么糟糕的王朝末年，居然成批量、成批量地殉国，是因为这两个朝代它不是单纯的“亡国”，而是在中国传统意义上面的“亡天下”。

什么是“亡天下”？中国人所谓的“天下”，也是一个杂糅的概念，有的时候它指的是非常非常广阔的范围，但是大部分的时候，它所指的其实是我们的文化系统、政治系统和一整套的规制，这个东西叫做“天下”。所以到元的时候，它出现了一次“亡天下”——虽然说看上去是换了个国，实际上是这个国家的统治者和这个国家的人民不是一伙人，不是一个族群，不是一个系统，大家的族群、血缘、文化认同、祖先认同全部都不一样，这种情况叫做“亡天下”。明末转清的时候也是一样的，为什么那么大规模的殉国？甚至有很多的士大夫特意地躲到山里面，就专门躲着剃发易服，宁可当野人都不出山，也是因为那意味着“亡天下”——整个文化系统、民族的认同、所有的骄傲、所有的尊严都磨灭了。这个东西才是后来他要搞剃发易服、必须杀得人头滚滚的前提。所以所谓的“崖山之后无中国，明亡之后无华夏”，它本身是以前中国人的固有信念。

至于说这导致我们今天的国土没有合法性，我觉得这些说话的人吧，脑子有病。国土的合法性只有一个来源：你打赢了，并且占据了你这个国土就有合法性了。你占完了之后当然就可以说“自古以来”，那就可以显得比较体面一点是吧。但是你想，我们“自古以来”的地方那多了呀，俄罗斯西伯利亚平原上至少有一半是我们“自古以来”，我们可从来没有叫唤过——为啥？没打下来你叫唤什么呀。打下来了就可以“自古以来”，全世界任何地方，我们如果哪一天打下来了，占据了，有我们自己的人管理那个地方，用我们自己的系统来运作，那个地方我们都可以叫它“自古以来”，至少我们还有一本《山海经》嘛，又不比人家以色列人的什么《圣经》来得低级，是不是？全面得很，从山川河流到物种全都介绍了，“自古以来”妥妥的，地图都画好了，凭啥不行啊？咱又不是编的故事。所以所谓的“必须要从上一代继承下来”，那都是扯淡。没有哪个国土是从一个亡国的地方继承下来的，继承不了。满清亡国的时候就剩一个紫禁城了，你跟他气成什么呢？他啥也传不给你，他东西早就已经割出去了，有的割给了德国，有的割给了法国，都被人家分了，咋能落到我们手上？解放军先来的，解放军不打。那些地方签的合同全是满清政府签的合法的合同，皇帝押了大印的。我们是说它不平等条约，不平等。是不是合法条约？就是合法条约呀。那为什么有的就认了呢？例如我们跟俄罗斯之间的界限，我们就认了；有一些我们就没认，打下来的，哎，那就是我们的“自古以来”，你那个合同不作数；没打下来的，咱们曾经有过合同，那是你的。就这样嘛。我这个人说话比较赤裸一点，但事实社会、现实世界就是这个样子。所以什么“我必须承认元清合法性，要不然中国没有合法性”都是放屁。中国合法性的唯一原因就是：中国人的军队把土地占下来了。

所以日本人的第一板斧，无关紧要又挺紧要的——“崖山之后无中国”被今天的中国人给否定掉了。

日本人的第二板斧，就叫做“夏入夷则夷，夷入夏则夏”。哎，这句话很有意思，因为这句话的渊源来历很有趣。很多人说“这是孔子说的”，我觉得你要是敢去孔子的坟前的话，你不怕他把棺材盖给掀了吗？我跟你说，一米九的壮汉天天持着剑，驾着战车四处奔跑，这种人你真敢跟他说这个事儿？孔子最在乎华夷之辩，你跟他说“谁来谁是中国人”，他现场把你脑壳给削了，信不信？你绝计保不住天灵盖儿。所以孔子120%没说过这句话。

这句话最开始的一个源头是谁呢？是委屈巴巴的韩愈。为啥说他委屈？因为这话不是他本意，他原本不是这个意思。他本来是这样说的：“孔子作《春秋》，诸侯用夷礼则夷之，进于中国则中国之。”这句话在很多人的嘴里面被篡改过，就是“进于中国则中国之”被改成了“夷入夏则夏”。所以我说韩愈非常委屈啊，因为人家本来说的是：诸侯，你好好的践行中国人的礼仪和制度，咱们就算你是自己人；如果你居然敢学习夷人的礼仪和制度，就把你赶出去。唉，这句话本身的意思是什么？夷人开除国籍啊——我比较简单直白，他就是这个意思：夷人开除国籍。请问今天我们说“夷人开除国籍”是不是照样皆大欢喜？大家都欢欣雀跃，可以把所有夷人开除国籍。唉，韩愈就是这个主张。但是呢，当他被篡改了之后，就变成了“中国人用外国的礼仪就算外国人，外国人用中国的礼仪就算中国人”。说老实话，中国人自古都没这么干过啊。哪怕是今天的中国人吹得吊炸天的那个大唐王朝，万邦来朝，开放又包容，外国人可以当高官，就这个时代里头有一个典范人物叫做高仙芝。这个人他不光是文化水平很高，打仗水平很高，而且全身上下每一个地方都完成了彻彻底底的汉化。如果汉化有进度条的话，他绝对被拉到了100%。如果按照民族文化论来说的话，他进度条都100%了，他是汉人呢？他严格践行汉文化，而且他对于大唐天子之忠诚，番邦到处造反的时候他还起来平叛呢，多了不起的一个人呐——100%的忠诚，100%的虔诚，100%的汉化。不好意思，他也是归在胡将那一派里面的。中国从来没有出现过“谁来谁是中国人”这种情况，只有当你真的完全融入到已经找不见自己格外的祖先和血源的时候，你才算是中国人，这是一个很严格的界限。

好，这句话，韩愈委屈巴巴地当了一个所谓源头之后，到底是谁把他篡改成这样的意思的呢？最开始出现这种含义的人叫做许衡。这个人的人生履历就很有趣：他出生在金国，是个汉人儒生，但是他出生在金国，一出生的时候就不在汉家王朝管理之下。然后他在金国科举读书，成了大儒。他想要保护在这个地方当二等公民、下等公民、当贱民的汉人，于是他想推行儒学去改造这个胡人的政府。在这种情况下，他对儒学进行了大规模的修改、修正和修订。然后当蒙古崛起，他又成了蒙古人治下的一个儒生的时候，他成为了忽必烈的重臣，然后他觉得他要保护蒙古治下那些当了贱民的汉人，想要改造这个蒙古的统治者。于是呢，他提出的主张就叫做“行中国之理，则为中国之主”——你只要按照中国的规矩来办事，哪怕你是个外国人，也认你当中国的主子。有没有意识到，这种说法和这种理由本身和汪精卫是一模一样的：反正也没有办法了，我认怂了，必然是你统治了，那么你想办法用更好的方式让底下的中国人、原本的土著过得不那么难受吧。是这样的一种折中的思路，产生了许衡的这一套“行中国之法，则为中国之主”。这是这个阶段。

然后到下一个阶段，这种想法什么时候又践行了一次呢？就是到了清代的时候，康熙皇帝又推广了一次。然后再下一次推广，就是日本人在东北推广了一次。哇哦，这个玩意儿和前面两个好呼应啊——反正每一次都是入侵者、殖民者的统治者非常喜欢推崇这句话，就是“夏入夷则夷，夷入夏则夏”。而今天的中国的爱国人是坚定地认为“谁来谁是中国人”，嗯，这是中国五千年未有之怪现状，但是现在是政治正确，我也是无能为力啊。咱们大势所趋，看看什么东西是大势，不知道。

然后第三个东西就是“日本人是华夏正统”，这就涉及到所谓的正统到底是什么东西。“中国是一个伪装成国家的文明”，这个是西方的汉学家主张的啊——西方人啊，也是列强，也是侵略殖民我们的人哦，哎，他们的学者主张“中国人是文化来的，跟其他东西没有关系，文化认同、文化继承就可以继承这片土地”。唉，日本人也是这样想的，日本人觉得“嗯，我是正经中国人的学生”，说老实话，我也承认他向中国学习了很多东西，基本上算是中国的学生，虽然是有点编外啊，但他是中国的学生。那么对于他来说，他只要说“中国是一个文化的共同体”，那么我既然是传承了你的衣钵，我就是天然的文化继承人，我就是华夏正统。在历史上也出现过很多这样的主张，例如当年的契丹，他也主张自己是华夏正统，因为他曾经臣服于唐，接受唐的统治和唐的官爵赏赐，在这种情况下他认为“我本来就是唐的一部分，那么唐灭亡了之后，我当然是华夏正统”。发现了没有？大家都喜欢用这种方式。然后后来越南也是在中国走向衰亡的时期跳出来说“我们这里继承了华夏正统，我们是小中华呀，你看我们写的字，你看我们穿的衣服，总比你那个什么金钱鼠尾辫像中国人吧，是吧？我们华夏正统”。清代的时候，朝鲜也好，日本也好，越南也好，纷纷举起旗子，认为自己才是华夏正统，而且说老实话，他们看起来确实更像一点，因为穿的也真的是好看很多倍。

总之呢，最喜欢主张“中国是文化继承者”的，其实就是这些想要找个理由来占领中国土地的人。但是今天的中国人也非常支持这一点，哎呀，文化决定论多么的高级啊——只要学了这个文化的就是中国人，文化学得好的就可以当中国的统治者。唉，我就觉得这些人怎么可以那么二极管呢？“二极管”这个词很好啊，很明显的就描述出了这些人的特征：就是这些人的生活当中，和我能够谈得拢的叫做手足兄弟，谈不拢的那叫敌对势力，除此之外就没有别人了。而在正常中国人的概念里面，手足兄弟那是一个妈生的，谈得拢的那种人叫做朋友。二极管那里没有朋友——要不然你认可中国文化，咱们是一奶同胞；也不知道吃的什么奶，反正就只要谈得拢，唉，一奶同胞；谈不拢的，即使是一个妈生的，我不认识她，她是敌对势力。这个完全违背人类的人类学常识，你懂吗？正常人家，谁家里面不可能会出一个败类呢？有可能的呀。人说，“这个妈生了八个儿子，七个都是很好的孩子，然后有一个真是活脱脱的败类”，大家群起而攻之，把他赶出家门去，唉，不认他了，割席了。是的，再怎么割席，那是不是你家儿子呢？还是得认。唉，确实他还是这家的儿子，只是我也很烦他，我也很恨他，我也想宰了他——就这种区别啊。至于说隔壁家有一个谁跟我关系贼拉好，特别谈得拢，天天在我们家吃、在我们家住，对好朋友特别特别好的朋友，他是你亲儿子吗？不是啊。不就这种区别吗？那文化决定论者就在这个地方，显得像一个智障一样的——100%的二极管是吧，既违背历史，也违背现实，而且违背正常的基础的逻辑：凡是谈得拢的就死活非要说大家是一奶同胞。一个肚子里生出来的才叫一奶同胞，一奶同胞也可以有犯罪分子，这是很难理解的事吗？

总之呢，日本人的三板斧，在他狼子野心不死的情况下，它会一直不断地持续地向中国输出。而它持续输出的过程当中，最重要的两个斧头已经打得非常结实，已经达成了所谓的社会共识，已经成为了很刻板的概念。我可以说，如果下一个阶段日本人又一次侵华的话，他的理论基础比过去要扎实得多。因为“谁来谁是中国人”，日本人要乐死了。我没法乐，我也不知道大势所趋在哪。肉食者鄙，指望不上，我们看看下里巴人是怎么想的吧。
