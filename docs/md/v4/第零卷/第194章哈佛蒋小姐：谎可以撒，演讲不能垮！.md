# 第194章 哈佛蒋小姐：谎可以撒，演讲不能垮！

哈佛蒋小姐的黑料一天比一天更多。我们会在她身上发现一个很有趣的迹象：她天天都在撒谎，然后日常被人揭穿，揭穿之后就删帖。但是呢，我们会发现，在她所有的话术当中，她非常在乎一件事情。例如，之前有人扒出她的履历当中大概率存在造假的嫌疑，因为她一会儿是全职，一会儿是兼职，一会儿已经入学了，一会儿又好像还没有入学。所以当她的人生进程表上有大量缺陷的时候，自然就有人问你那时候到底在干什么。然后蒋小姐自己就把这段履历给删了——她已经删过好几条这样的相关信息了。可见她并没有想在这个事儿上面跟网友吵嘴：你揭穿了，我就换一个谎来撒；你再揭穿，我再换一个谎来撒。唯一的一个好处就是她撒谎的水平不太高，所以每次都不圆。

但是有一个点，你戳了它一戳一蹦的什么点呢？就是你不能说她演讲不好——发现了没有？你可以说她撒谎，你可以说她是表演，你可以说她完全没信用，但是你不能说她的演说做得不好。她真较劲啊，真的开始跟你反驳呀。您说那个演说太次了，她说怎么可能呢？我们哈佛都很喜欢我的演说，旁边的人都围观，都跟我说话，明显我演说就演说得特别好。你看，人家开始辩驳了啊。其他的谎言她从来没有辩驳过，但是这个实锤的、全世界都拿她当笑柄的演说，她不允许你反驳。为什么会这样？她为什么会那么在意“演说必须成功”这件事情？这就是今天我们要聊的。诶，演讲这个事儿有那么重要吗？啊，对于中国人来说，她确实不一定重要，但是对于老外来说，这还真的挺重要的。

大家有没有去回看一些以前的电影、电视剧？在很多的影视也好，美剧也好当中——尤其是美剧——你会看到里面的一些人物，有的时候是小人物的成长啊，有的时候呢，是女性的破茧重生。这样的影片常常在她取得成就了之后，就要开一个小型的发布会。这种发布会的演说呢，常常就是从一个故事开始：我什么时候发生了什么事儿，于是最后怎么样。然后底下的人哐哐哐地就开始鼓掌。对。所以蒋小姐身上的这个对于演说、对于演讲的成功非常执着的念想，不是她身上的毛病，也不是中国人身上的毛病，也不是姓蒋的毛病，而是美国人身上的毛病。他们一贯如此，极度地在乎演说的成功。

为什么会这样？大家去看一些招聘启事啊。10年前在我们的互联网上，有一些国际化的招聘启事，里面呢，还包括有一些专门贩卖这种课程的培养，这种适合你出国、适合你找到跨国高薪工作的培养集训营。这种招募启事当中都有一个要素，叫做“培养青年领导力”这样的词儿，反反复复地出现。那什么叫做领导力啊？领导力这个东西就是：我能够率领一个队伍做事，大家都听我的，我说的话他们都相信，他们愿意跟着我走，这玩意儿就叫做领导力啊。其实呢，它是什么？它是构建一个组织的能力。所以领导力这个事儿，说老实话还确实挺重要的。然而问题就出在这儿了：当他强调的是一个构建组织的能力的时候，西方人的这个领导力被局限在了一个很小的范围里面——他们特别执着于你要当有领导力的人，你必须得有充分的演讲能力。所以呢，你会看到当年的这些集训的课程当中，也包括了持续不断的演说：如何用演说吸引他人，怎样的演说的结构才能够让别人受到感动、受到感召、跟着你走。这个是这些课程当中非常重要的事儿。放到10年前，这一门课往往少了说要20个W，你才能够真正地登堂入室。所以还算是高端大气上档次。

那问题就来了：为什么西方人的领导力居然就是演说呢？哎，说到这儿的时候，我们就得说一说中国人。既然说领导力是一个组织能力的表现，那中国人——这个全世界特别特别执着于集体主义的群体——很明显是要求有强组织力的。那既然要求有组织力，你就必须要有引领组织的能力，那对中国人这里也有领导力的相关要求。但是我们不叫这个名字，我们叫什么呢？我们自古以来跟领导力这个东西叫做“君子风度”。是的，就是你以为的那种扇着摇扇、晃来晃去的翩翩君子，其实他在一开始的时候指的就是领导力核心。为啥？因为君子的雏形就是“君之子”，君是什么？君就是行政管理者。所以呢，君子指的是行政管理者的继承人，也就是下一任的管理者啊。因此呢，他身上要求的就是今天美式的领导力当中特意强调的“青年领导力”的核心，这就是标准的君子风度。

然而，当我们去看到君子风度的时候，中国人对君子风度的要求是什么？这就不得不提到君子六艺啊。君子六艺当中包含了什么呢？你要会算账——这个是“数”。数是干什么？是包括了计算，包括了工程。也就是说你至少得是一个理工类的人才，对于基本的科学技术要有领会的能力。然后呢，你要会“御”，御就是开车啊，驾驶战车。放到今天，你约等于可以独立地驾驶主战坦克，拥有军事才能。还要有“射”，射是啥？那个射就是射箭啊。射箭呢，它有双重职能：一方面呢，是军事职能；另一方面呢，它会涉及到祭祀礼仪当中会使用到射这个仪式。然后还有那个“礼”。礼呢，你常容易理解为礼仪规范——不是啊。礼是社会制度，礼仪规范是社会制度的外在表现。他实际上学的是社会制度管理，所以这属于高端行政岗啊，国家级的高端行政岗：社会制度变迁、社会层级管理全部都要会。然后其他的就是什么？ “书”，它就是书法，然后文字系统、命名系统，这些东西全部都是君子必须要有的东西啊。甚至还包括了音乐啊。当然了，先秦时期的音乐和你所想的可能不太一样，因为音乐呢，是我们礼制当中很重要的，甚至是社会度量衡当中非常重要的东西。所以它确实是国家管理纲要当中的一部分。

因此我们会看到一件事啊：君子六艺也是青年领导力的一个表现。但是我们提出来的东西当中压根儿就不包括演说。甚至我们会专门对君子的领导能力提出了一个要求，要求他“敏于行而讷于言”——就是少说话、多干事儿。这件事情一直到几十年前，我们的共和国成立之后，对于所有的社会的管理层还是一样的，提出这个要求，叫做“少说话、多干事儿”。所有你要当领导的人，闭嘴。你学会闭嘴，先学会听别人说话，然后你学会怎么做事情。这个叫做领导力。

但是你发现了吗？我们和西方人的这种领导力的定义和对领导力的指标要求截然相反：我们这里要求你闭嘴，那边说要你赶紧把话说漂亮。为什么会有这样完全背道而驰的系统呢？它其实来自于我们在社会管理、社会统治学相关的结构上面的根本不同，甚至可以说它是背道而驰的。这是因为中国自古以来，至少说自商代灭亡之后，我们就已经完全进入到了一个世俗化的阶段。而世俗是什么？世俗是我今天吃得上饭，是因为我昨天干了活儿；我的粮食能够收割，是因为我之前撒了种子；我想要拥有一件衣裳，我就得先去织布。这个东西就叫做务实的精神。当中国人这个时候摆脱了神权，进入了世俗化之后，世俗要求的就是：干了多少活儿就能吃得上饭，做了多少事就能穿得上衣。这个东西就叫做世俗化。

但是呢，在更加古老之前，它的一种结构叫做神权制。人类社会普遍性地曾经经历过神权，因为它是一个相对早期也相对容易的一种组织管理形式。它用的管理方法呢，就和我们世俗化的管理方法不一样了。要知道，世俗化的管理系统当中，每一个人都在想：你说这事儿对不对啊？你不是蒙我的吧？你怎么能说那粮食是你送我的呢？那明明是昨天我才去收割的嘞。世俗化的人群，你是很难用这样的迷梦一样的故事去蒙骗他的。所以在世俗化的制度之下，领导者要做的事情就是身先士卒，表现出自己的高风亮节，表现出自己这也能干、那也能干的指导的能力。这是你必须要有的东西，没有这个别人不跟你走。没有信用、说话不算数的人，别人不听你的。这个就是世俗管理系统当中非常困难的东西，也就是我们的祖先为什么要求君子六艺这么难。

但是神权系统就不太一样。而现在的西方，尤其是美国，他是真的神权系统，非常非常至上的一个地方。当然了，有不少的西吹可能不太承认这一点，但是啊，前几年的疫情期间，人家是正经地给我们打了样儿的。在人家的政府的核心机构内部，用电视台直播和转播的形式公告天下，说怎么克敌啊，怎么解决瘟疫的问题，跳大神儿念经，保佑驱逐——用这样的方式进行国家指导，这是一个典型的神权的神学的操作方法。对今天的美国，依然保有着神权管理之下的一些模式。而神权管理是用什么样的方式进行管理呢？讲故事。神权的管理的基本手法就是讲故事：我讲一个能够震撼你心灵的故事，并且反复地宣传，不断地重复，不断地演绎，也可以制造一些场景去进行某一些兑现，然后让你真的相信他，最后你就跟着他的旗帜行走了。这是一种组织力的构建方法，就是神权构建法。而这种神权的构建方法当中，最重要的能力是什么啊？讲故事的能力。你会不会种稻子、会不会种玉米不重要，但是呢，你一定要讲一个充分的故事，说明玉米之神长什么样子，在什么情况下会把这个玉米赠送给你，让你吃得上玉米。神权的管理，要求他的领导者必须能够把这个玉米之神讲得玄之又玄，信以为真。但是世俗的领导者就必须要解释：这个玉米得上什么肥？那虫子到底得拿什么驱虫？你这个问题解决不了，农民不跟你干。可是神权系统当中，他要的就是讲故事。

当我们理解了这个基本的逻辑之后，就更加容易去理解今天蒋小姐身上的这种一身的反骨啊。她对于你揭露她撒谎这件事情浑不在意：有人按照她的时间表捋出来说，你那个时候不可能全职，你到底念书了没有？你要是念书了就不能全职，你要是全职了你就不可能念书，你总得二选一吧？她把帖子删了。为什么呢？因为说谎在神权讲故事的系统当中不重要。神学故事有几个不说谎的呀？重要的是您能把这个故事讲得漂亮、有魅力、吸引人，这才是核心要点。至于说的是不是真的，谁在乎这件事情？神权系统不在乎谎言。于是呢，当你用这样的方式去理解她的时候，就能明白一件事情：我们的互联网的内部，这些各色各样的能人已经不断地在帮她扒皮了。她的家族履历、她的求学履历、她的每一张申请表，在什么地方、住了什么样的房子、住了什么样的宿舍、应该会在哪个超市里面买菜，大家都已经扒出来了。然后和她自己的自述对不上，但是蒋小姐浑不在意。你说我撒谎了，我就给你换一条啊，我下一次撒别的。揭露了一条，她就改一条，完全不介意丢人现眼。可是呢，你不能说她演说讲得不好，你不能说她的故事不迷人，你不能说她的东西没有制造全场泪崩的效果。因为全场泪崩就是美式的青年领导力核心。

那么这种神学式的组织方法，中国人学了没有呢？呃，学了但没全学啊。在今天的中国，用这样的演说的方式来展示自身的形式存在，但是我不知道你有没有经历过啊——就是呢，有一些社会上面的企业，长期地会搞一种会议营销。有的时候这个会议会开得很大，甚至是好几千人的大会场。他会不断地找他的明星业务员，然后开始讲自己的发家史。一般情况下，这个履历啊跟那个哈佛小姐姐有点像：就是我以前可苦了，也不管多富裕，反正家里面父不慈母不爱的啊，日子过得特别苦，如何如何地艰苦奋斗，然后进入到了我们这个公司的平台之后，终于找到了契机，飞黄腾达。现在已经有多少套房、多少辆车，家里面的孩子已经进了什么样的学校里面去念了什么样的书，马上就要拿到什么样的学历了，我现在这个日子过得爽歪歪，我很想把这种生活的经历分享给你，让你跟我一起干。哎，这个呢，就叫做会议营销法。这在中国的很多的会场当中都客观地存在。这种销售的管理方式，就是中国人去学习了领导力管理法的一种结果，我们把它应用在了商务的场合上面。

好在这种时候我们就知道说，你要让他人产生追随感，去用你的演说去感动他人，形成一个组织的追随力量的时候，要怎么做呢？要把听众纳入到一个氛围场里面来。这个氛围场一开始的时候是谁在干的？为什么我强调说西方的演说这个玩意儿脱胎于神权，就是在这个位置上了。因为他的这一套运行法则就是跟当年的教堂学的。大家听过教堂的音乐吧？唱诗班、管风琴辉煌宏大，气势磅礴，十分动人。有的时候听着那些优秀的唱诗班，我压根儿听不懂他在说什么，却有一种潸然泪下的感觉。因为啥呢？音乐可以直击人的心灵，它是挑动你的情绪的。而原本的传教的系统、宗教的系统里面搞的是什么呢？就叫这种氛围感：高大的门厅里面，璀璨的灯火飘渺，如同仙乐，你都不知道那个乐器在哪里响的管风琴，还有那些童声演绎的十分动人的歌曲。这些东西都会让人进入到一种状态：震撼、情绪、共鸣。然后别人就开始讲，从小到大的各种各样的故事，最后来到这里听了谁的解说之后，他就突然之间悟了，就明白了，就幸福了的这种感受。你就突然一下觉得自己好像被打开了——至于咋打开的，那就另当别论——但是在感知的层面上面就豁然开朗。那这个是什么呢？这个就是一种精神营销法则。那这个呢，是一个神学基本的要素啊。所以所有的神权机构里面，一定有这些传播神权内容的东西。而神学内容的东西，大部分就是讲解这些小故事，然后让你感同身受，进入到这种能量场之中，与大家一起共鸣共枕，然后你就会觉得身边所有的人都是兄弟姐妹，我们才是一起的呀。至于实际上有什么利益冲突啊，被害方是不太清楚这件事情的，加害方就比较明白一点。但是呢，今天我们要谈的不是这个，就忽略过去。

所以组织力是什么？组织力是感召他人的能力。而西方人的组织力，长期数百年间一直是以神学为基础的，于是呢，他最侧重于的能力就叫做讲故事。故事讲得好，队伍就带得好。相信你的故事的人越多，跟着你行走的人越多。所以他的努力就会着重于腔调、气质、身段、声音、仪态，还有能量场。但是中国人这里我们不太讲究这个事情。是的，中国人在遇到这样的故事的时候，通常会很紧张：你干啥？你讲故事给我听是做什么？你为什么不讲道理？为什么中国人在这个事上会有这种警惕的情绪陡然出现？因为我们过去的几千年都是世俗文明。而在世俗文明当中，一个人有无领导力，一个人有无根本的引领能力，在于什么呢？身体力行。大禹能够成为领导者，是因为他知道如何疏通河流，跟他带队挖完了之后，咱的洪水能泄——这是凭实力上的。李世民能够带队伍，固然因为他出身高贵，但是也因为他能打呀。朱元璋能够以平民之身扫荡中原，是因为什么？是因为他格局高，能战斗，能文能武。所以一个人能做事，一个人能够带来利益，一个人能够让大家的生活改善，他才会成为一个领导力的核心。所以你看今天的中国，我们去筛选领导、提拔干部的时候是什么标准？你以前管过的那个地方，GDP有没有改变啊？文化的氛围有没有提升？受教育的水平够不够优秀？你那里有什么样的典型的文化的符号的特征？让别人欣赏你，总得有点政绩吧。这是什么？这叫做实干精神、务实精神。其实它也是我们自古以来，由于我们是一个世俗社会，要求他有客观真实的表现，它必然产生的一个结果。

所以为什么中国人要求的和美国人要求的总是那么不一样？就是因为我们两个社会本质上面是不同的。所以为什么蒋小姐在中国的舆论场上面显得那么的格格不入？因为只要中国人不再用仰望的视角去看待美国人的时候，就会迅速地发现：你天天老讲故事，老画PPT，老跟我们吹那些大牛干什么呢？你这么厉害，你演给我看看不就行了吗？是骡子是马，拉出来遛遛呀。而蒋小姐可能是在西方、在美国待的时间有点久，她很难理解中国人的思维方式，总是觉得我已经讲得这么动人了，我都潸然泪下了，我都快哭死了，你怎么还能够不感动呢？但是对中国人来说，不敢。我们真的不敢。凡是那些全场泪崩的，除了讲的是切肤之痛，否则的话你到底打算干什么呀？

所以时代变了。中国人只要用平视的眼光去看世界，就会发现我们跟我们数百年以前以及更以前的祖先达成了精神上面的共鸣。对于中国人来说，只要有公平竞争的机会，我们很快地就会意识到一件事情：这个世间会生产、会建设、会积攒的人才是最优秀的人才。为什么蒋小姐会突然之间水土不服？没办法，如果说你连一个人、一个村、一个镇的发展都解决不了，光靠讲故事就想搞社会的、国际的发展，发展个锤子哟。总而言之，当一件事情落地到中国人的这里的时候，我们会发现中国人有自己的审查指标。如果说领导力，如果说发展的引领者可以称之为君子的话，那么在中国人这里，君子可以分成两个类别：一种叫做道貌岸然、集会表演、极会煽动的伪君子，以及敏于行而讷于言的真君子。是骡子是马，拉出来遛一遛。光靠言辞，除非你和我一样是一个带货主播呀，亲。

作为一个带货主播呢，我们一定是要感谢我们的金主的。那今天的这个金主确实是非常慷慨大方，因为他特意地叮嘱了咱们：这期广告放片尾，观众的观感最重要，流量差没有关系的，不要强求。哎呀，这是第一次啊，第一次见到居然有这么良心的甲方跟我说广告放片尾看不完没关系，慷慨大气。所以今天真的是要冲出来介绍一下这个良心的甲方到底是谁啊。我们来推广一下这个产品啊，确实不错。它是一个手机搭子之王，最强手机搭子。哎，这个“手机搭子”的意思就是：你想要走哪儿到哪儿都能看手机吗？不太健康，但是确实没有办法忍不住。像我这种行动力比较匮乏的人，那就属于走哪躺哪儿。那躺下来了，又想刷手机，然后又想让自己的脊椎健康，我可以怎么办呢？啊，那这个就是现在销量也非常优秀的“温眠人体工学手机枕”。这个产品呢，近一个月以来已经有7000家购，上一周到货了之后呢，就直接销售了300个，两天之内就完全卖光了。唉呀，这个产品啊，确实非常值得你去试一下。想要躺沙发、打游戏的啊，一刷手机就停不下来的，躺在那儿躺俩小时的脖子会僵、肩膀会酸、头晕眼花、浑身不得劲儿的，就可以来试一试这款温眠人体工学手机枕。那他强在哪儿呢？它可以给你一个支撑：躺、靠、趴、抱枕，每一个角度都可以啊，维护着你所有的腰、背、颈部，全都给你拖住了。这种情况下啊，它的效果就能够让你的骨骼感觉到舒适，躺在那儿的时候呢，也不会损伤自己的脊椎和后背了哈。正躺、侧躺都完美贴合，支撑脖子，保护颈椎。一枕多用，又省地儿又省钱。你喜欢久坐、喜欢刷手机都可以试啊。那质量上面好不好呢？近三个月店铺的好评接近了百分之百。为什么能够做到这样啊？就是因为呢，它引用的是进口的60D太空记忆棉，而市面上常见的都是40D。那这样的高投入，使得抗它的能力直接提升了50%，压一年压两年，一点儿事儿都没有，十分耐用。耐用只是基本功，而且呢，它还经过了六道质检，测试了53项，甲醛和有害物一点都不存在。上架一个月爆卖3200个，想多要一个都没有货。那价格呢？唉，我们来压了一波：原价511，店铺409。但是今天我们申请到了80元的大额优惠券，现在只要329。量不多，大家抓紧时间。33天试用，不爽的就可以直接退，两年之内都有质保，随时换。手机党、久坐族、颈椎不好的、想送礼的都可以冲它。
