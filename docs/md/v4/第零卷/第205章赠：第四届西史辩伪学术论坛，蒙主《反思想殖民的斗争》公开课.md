# 第205章 赠：第四届西史辩伪学术论坛，蒙主《反思想殖民的斗争》公开课

今天讲到这个题目，其实目的是抛砖引玉。因为在座的各位老师，在很多领域当中，都有比我更深远的耕耘。所以今天，我们就从更加外围的传媒以及受众影响的方向上，来解析一下反殖民的反思与反殖民的斗争。

首先，看四个字：中华文化博大精深。我不知道大家看到这句话时有什么感想。二十多年前我读书的时候，有一位老师，他在讲中华法制史。每一次在上课开篇时，或者讲到某个段落时，他就会说：“中华文化博大精深。”那时，我们的反应是什么呢？教室里就是这么多的人，大家哈哈大笑，就好像孔乙己走进酒馆时，到处都洋溢着欢乐的笑声。我也是其中之一。

后来，局面是怎么发生变化的呢？就是我开始做这位老师的课题，最初只是一个泛泛而谈的主题。结果我发现了一个问题：在进行古今法制对比的过程中，我注意到，现代社会所需要、所感知到的这些规则，在我们的先民的规律当中全部存在。当然了，他们没有股票市场，没有手机，所以这些规范是没有的。但是，所有跟人类社会行为、跟人类日常生活相关的规范，远古都有。

然而，那时为什么我们的教室里洋溢着快活的声音呢？因为我们其他所有课堂上的老师，都在给我们传递另外的信息，叫做“汉谟拉比崇拜”。当时我们传递的所有制度、我们今天现行规则的起点和延续、谁给我们开启了现代秩序文明、现代社会的基础又是从哪里构建的、人类的理性又是从何而来——答案都是：从汉谟拉比而来。所以，当时我在做这篇论文时，感受到了一种前所未有的困扰。你说汉谟拉比最优秀，没有他就没有现代社会的一切，但我分明在我们的古籍中找到了所有的东西，找到了everything。那么，它什么时候消失的呢？我的老师会告诉我：“因为我们落后了，因为那个东西没有用了，所以被淘汰了。”那是真的吗？

多年之后，我看了一本书，相信很多人也看过。我们日常生活中，通常说一个人有“狼子野心”，这妥妥的是一个负面词汇。可是在那之后，开始吹嘘“狼性文化”：你愿意去争夺，你去撕咬，你列队去狩猎——这是什么？这就是最先进的野性，这是人类的本源，这是文明的高级形式。然而，狼会种田吗？民以食为天。不会种田的是高级的，种田的是低级的；面朝黄土背朝天的是没意思的，有狼子野心的是开启现代文明的。这个东西是什么？所以，我们现在谈思想殖民。

那么，什么是思想殖民？思想殖民最终会表现为什么形式？它一定直通一样东西，就是种姓制。我们两千年前的革命者说：“王侯将相宁有种乎？”这个“种”，就是种姓制的“种”。中国人最反对的就是种姓制——天生贵种：你活下来就比我高贵，你会喘气就比我高级。但是，什么是种姓制？为什么殖民一定会推向种姓制？因为殖民就是要证明一件事情：“我比你高级。”怎么证明呢？就是种姓制。我特别想展开来讲，因为我觉得这个制度太了不起了。我不知道在座的各位怎么看，但对我来说，我意识到：种姓制，它本身是禽兽制度当中的顶流，它太符合人类心理上的需要了。我们需要什么东西？我们要满足精神上高高在上的优越感，几乎所有人都需要这个。包括现在，在互联网上，大家看各位老师的文章时，也会总结说：“你不就是想满足你的心理优越感吗？你不就想说你的东西比别人的更好吗？”他为什么这样说？因为确实，人类普遍有这种需求。而种姓制就在满足这种需求，它是顺势而为，顺流而下，和人类成长的逆流而上完全不一样。它太顺了，所以太容易推广，力量极为强大、极为顽固。只要有殖民，就会产生种姓制。种姓制就是我比你高，你比他高，它比另外的那个更高——它就是整个世界的排序。

而现在，我们世界上所谓的殖民心态、现代社会的思想殖民，它的表现形式就是种群优劣论、所谓的西方中心主义。它最终表现出来的姿态就是：西方人是最高级的；然后，游猎、渔猎的游牧群体是第二高级的；最底层的，就是种田的、面朝黄土背朝天的“土克拉”们。从而，将中国人排在了整个世界的种姓等级的最下面。

所以，当我们说反殖民的时候，反殖民应该怎么走？它一定要调整一件事情：叙事主体——谁来说话？我发现一件事：很多在座的老师们都在追求客观、准确、科学地去描述我们所知的一切。这是绝对科学的态度，也是唯物主义的态度。但它非常容易陷入一种局面，就是你会失去主语。我们说一句话是要有主语的：“我去做什么事情”、“他做了什么事情”——这个东西就是叙事主体。而我们非常非常容易在极微妙的地方失去这个主语。一旦失去了主语，事儿还是那个事儿，但它为谁发声、对谁产生影响，就开始发生转变。所以，这些年以来，精神上的、世界的思想殖民，他们不全都是在说谎，他们也经常客观、科学地去讲述一个事件。但因为讲述的主体和中国人的本体不同，所以，他讲着故事，既科学又理性，却把中国人又放到了种姓制的最低端。

因此，反殖民就必须改变我们的叙事主语。中国人必须要在世界上有发言权，必须要能够说话。而历史叙事这件事情，本身就是我们站在自己的角度，说我们自己的故事。它会决定一个群体的群体记忆，就是：我是谁？我从哪里来？K老师是搞哲学的，一定知道所有哲学首先必须解决的就是“我是谁”的问题，否则其他所有的讨论就变成了空中楼阁，没有办法落地，没有办法应用。怎么去用我们的知识？怎么去用我们得出来的结论？在座所有老师考据出来的这些科学结果，放到哪个战场上，能不能起到反殖民战争的作用？就取决于我们能不能通过叙事的方式来调整我们的群体记忆。

而我为什么要强调群体记忆呢？因为我们的群体记忆被改变了。中国人这几百年来受到的肉体折磨固然是一个部分，但更重要的一个部分，是我们的精神折磨来自于我们的叙事被别人改了。从清开始一直到现在，始终都有人在改变我们想问题的角度和方向。以至于哪怕是清代的那些逃过了种种约束、有了学术知识、有了文化水平的人，他也不知道自己要为谁说话。他会在纸上写文章：“圣天子在上啊……”那么，你的哲学思想优秀吗？黄宗羲的哲学思想是优秀的，他的哲学内容哪个地方不好吗？很好，很好，很有用，放到今天可圈可点、很有价值。但是当他说出“圣天子在上啊”，不好意思，你的主语变了，你所有的东西在为谁服务就开始发生转变。

这个时候，谁的记忆才会涉及得到我们民族的存亡？我们要影响的到底是谁在怎么样想问题？所以，有的时候我就觉得，现在的自媒体、现代科技能够带来的广泛讨论空间，非常重要。它会让很多优秀的学者研究出来的内容，不仅仅只在书斋之中。这些优秀的学术成果，能够最终进入到群体叙事，进入到群体记忆之中，让所有的人都知道：“我是谁。”这个是什么？这就叫做民族存亡。

所以，在民国的时候，钱老师为什么要写《国史大纲》？因为他觉得中国可能要亡了，我们可能要当亡国奴了。那么，当一个中国人要成为亡国奴的时候，我们怎么能够让自己最终还有机会活过来呢？就是记忆。《国史大纲》，就是我们的群体记忆。群体记忆决定了群体的存亡。

所以，西史辨伪，它不仅仅是一个单纯的学术之争。除了学术之外，它还是一个叙事权的竞争。我们研究出来：我们的冶铁需要多少温度，所以就能够说：“嗨，你们英国人不可能做出来这个东西”；因为我们知道炼一个透明的玻璃需要多少温度，所以我知道：“你们不可能做出来那个透镜”。但是，谁说了算？谁讲的故事有人听？谁说的话算作权威？这个东西就是在争夺叙事权。而今天的中国人，迫切地需要去争夺叙事权。叙事权则必须争夺人民群众的关注度，要让大众都在意，所有人都知道：那些数学的、几何的、天文的、冶炼的，看上去高高在上、跟我一日三餐毫无瓜葛的东西，是我的立身之本，是我在这个世界上存在的本源之一。那么，这个时候，我们就拥有了力量。

就像这一次何新老师跟唐继根老师的学术争端，看上去是学术的争端，但最终甚至不需要很多学者下场，评论区就可以把唐老师的很多讨论直接推平了。为什么？因为越来越多的普通人都意识到一件事情：“你在替谁说话？为什么你说的每一个字我都听得懂，但我就是不高兴呢？”这个东西，它就是人民的趋势。

所以，我们今天在座的所有老师都一定用得上的武器，就是唯物主义。就像刚刚那一篇论文，什么是唯物？一切事情的发生都是有源头的，一切事件的发展都是有背景的，这个东西就叫唯物主义。我活着，所以我才能吃饭，这个就叫唯物主义；因为我有纺织机，所以我能够穿上衣服，这就是唯物主义；因为我有炉子，所以我才有金属，这也是唯物主义；因为有金属，所以才能有蒸汽机。当你只讲“我发明了蒸汽机”，那请问前面的呢？为什么前面什么都没有？为什么会出现无根之木？为什么会出现无源之水？有越来越多的学者在解决这个问题，于是，通过唯物主义的叙事，帮助所有中国人挣脱国际种姓制的枷锁。

从今年的九三阅兵之后，越来越多的人发现：“天哪，我们中国人花了才几十年的时间，就追上了你们几百年说的遥遥领先的东西。然后你说我们几千年都是落后的，不可能进步？你在骗谁？”当现实的、唯物的重拳挥在所有人面前的时候，叙事权就开始逐渐转移阵地了。

所以为什么要强调“农村包围城市”？因为它是我们力量的来源。我们从哪儿来？包括所有学者所做的所有努力，最终为谁而服务？当被服务的对象得利，并且意识到它很重要、这些武器对它极为有用的时候，我们就实现了“农村包围城市”。

当然了，我在最初提出这种想法的时候，是学习了西方人的。西方人当初来搞颠倒叙事的时候，做的就是这项工作。不知道大家注意到了没有？几十年前，中国的平均受教育水平并不好，绝大多数人没有受过高等教育，受到一个职业教育就已经是社会精英阶层了，哪里知道什么这个历史那个历史？亚里士多德念完之后，舌头要打卷的呀。后来，怎么越来越多的人越来越相信西方历史的辉煌？地摊文学大家都看过，也曾经瞧不起过、蔑视过。但是，这是西方人当年进行舆论围剿战时使用的“农村包围城市”，只是那个时候我们是被包围的对象。而今天我们希望用这样的方式反制回去，找到我们自己的叙事道路，然后走向真正的历史正本清源。

二十年了，我亲自关注这个领域二十年时间。当然，这不全都是西史辨伪发展的二十年。但是从一开始，我最初听到的就是何新老师的质疑。当时他那一声质疑，突然之间解答了我在学校里读书所遇到的困境：凭什么中华法系不好啊？我们的系统很完善，而且我们的系统跟我们的社会道德、公共伦理是配套的，这种完全一致、一点都不割裂的东西竟然是不好的？而西方人传进来的那种完全孤立的制度，要求你“必须抽离出理性来才能够去理解的东西”，竟然是好的？那么，何新老师当时提出的疑问，一下子让我认同：“对！不然呢？还能有什么其他的解释？”

但是最终能够把这些思想落地生根的到底是什么？我意识到一件事：在西史辨伪产生的早期，很多的时候是在论坛上、在一些很草根的地方讨论它。而这种讨论的过程当中，我觉得最难能可贵的东西，就是质疑西方历史的所有的从业者、参与者、讨论者，甚至旁观者，都非常冷静，都愿意好好说话，愿意慢慢说。我觉得这就是那个时候最可贵、最可贵的一个东西。因为西史的信徒们特别着急，他们特别喜欢给人戴帽子：“因为你不相信，所以你是九漏鱼；因为你不相信，所以你是文盲嘛；你都不相信西方哲学的伟大，那你肯定烂透了。”他在不停地攻击，而我们在不停地解释。这种憋屈的、时不时就要受委屈的局面，使得我们在整个斗争过程中获得了群众的支持。因为围观的人看得清楚：谁心虚、谁胆怯？谁可以讨论、谁是一言堂？

他的可贵之处就在于还是接地气的，是愿意讲给最普通的人听的。而今天我们延续的，就应该是这样的路子。什么路子呢？叫做“你打你的，我打我的”。我们只管讲自己的东西，把我们的内容用最细致的、最接地气的、最符合普通人认知的、最符合朴素唯物主义的方式传播出去，让所有人都听见。至于那些污蔑的，那些泼脏水的，说你是九漏鱼、你这个臭文盲、你就是在割韭菜——欣然接纳之后，局面的转变其实来得很快。就像今天，我们已经开始攻守易形。继续这样走下去，直到最终胜利的来临。而谁会胜利？有来源的会胜利，有人民基础的会胜利。

好，谢谢各位老师。
