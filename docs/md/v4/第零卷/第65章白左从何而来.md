# 第65章 白左从何而来

只有经得住时间考验的，才是世间的真理。而现在呢，西方的白左终于被彻底拉下了神坛。大家现在听到白左这个群体的时候，第一印象就是这群傻缺——这和他们诞生之初那种高级的、伟大的、富有牺牲主义精神的形象，是完全背道而驰的。所以今天越来越多的人提出了一个疑问：白左这个玩意儿是怎么生出来的怪胎啊？是的，人类的群体里面为什么会出现白左圣母这帮子人呢？就理性主义而言的话，他们是很奇特的一群人。这种行为，就是割自己的肉，养育别人，然后由别人来侵犯自己。白左所干的各种各样的事情，都在践行这样的一条道路啊。听起来就是很奇葩的一种选项，但它却似乎理所当然地就诞生了。这涉及到人类普遍存在的某些心理上的要求，还有对于痛觉的一些要求。

第一件就是心理上的诉求。所有的人心里头都有两个维度：第一，是利己的，就是维护自身的利益；另一个维度呢，就是利他——一定要让自己觉得自己是一个好人，不然的话，这个人生很难完满。哪怕一个人他是大恶人，也会想方设法的去做点好事，让自己觉得自己有利他的精神。这个对于一个人的精神稳定非常重要。所以不管是在哪啊，只要他是一个人，他就会努力的让自己表现为利他、善良、有牺牲的奉献精神。

那么白左这群人，他在诞生的时候，诶，那是在一个西方世界——白人世界在攫取了全世界200多年的财富之后，进入到了一个和平的享乐的时期。而在这个时候呢，他的生活条件很优越，对于其他的地方呢，充满了这种俯视的视角。他相信别人都没有我好，别人都没有我先进，别人都没有发达。这种自身的站位又很高，同时呢，又觉得日子过得很不错的人，就容易产生一种圣母的情怀。这个情怀就是诞生白左的土壤。它们表现为富有牺牲精神，表现为关心全世界，表现为关心跨物种的一切，表现为对每一朵花、对每一个小动物、对每一次风的吹动，都潸然泪下。

这种行为啊，你要说它好理解呢，其实也好理解。你就看这些人所拍摄的很多跟环境有关系的、跟动物保护有关系的纪录片，你就知道了。你去看这些纪录片的时候，看到里面的一些山河破碎啊，看到各种各样的落叶，看到各种各样的石头、各种各样的冰川融化，以及那些小动物受到伤害，那些皮毛、那些灭绝等等的事件的时候，是不是很快的就会感同身受，觉得痛彻心扉？这个就是人类身上的这种共情能力的体现啊。他去看到别人痛苦的时候，非常容易感同身受。这种机制是刻在DNA里面的——就是当我们看到同类受伤的时候，会感觉到自身的痛觉，于是呢，就会去积极的救援。这种情况下呢，这个族群才能够更好地存在，并且壮大下去。所以在基因层面上面，我们对其他人的痛苦感同身受，这件事情对于我们来说很重要啊。人人身上都有这种能力，就是感知他人的痛苦。而白左很多的行为，就是他夸张了对其他人、甚至其他物种、甚至对于无生命体的痛觉感知。这种放大是在自己的经济条件很优渥、自己的生活很平稳的情况下，他不需要太多的关注自身了。于是呢，他就把所有的这种共情的、察觉痛苦的能力，就灌注到了世界上其他的物种、其他的人类以及其他的非生物上面去。

而同时呢，我们又说到了人类的基因当中还有一个短板。我们对于其他的生物的痛觉，有非常强烈的感同身受的能力，这是一块；但是同时呢，我们还有另外一块的能力，就是对于自身的痛苦，有非常强烈的埋没的欲望。我们去看很多的那个新闻事件的时候，你就会发现一个现象：有些人在遭遇了重创之后，你去询问他当时的感受、当时发生了什么，你一旦询问到了细节，你就会发现他什么都不记得了。他不记得最痛苦的那个时间段发生的事情。这个是刻在我们基因里面的一种自我保护机制——就是当我们的身体自身察觉到极度的痛苦的时候，会进入到休克，会进入到昏迷，会迅速地来否定这一段记忆，甚至短暂性的失忆。总之呢，我们的记忆就一直在告诉我们：那种极端的痛苦要把它忘记。有没有发现这个中间有一个bug？我们对于其他东西的痛觉，很容易复刻到自己身上，感同身受；但是我们对于自身身上的真正的重创，却严重的缺乏想象力，甚至会刻意的遗忘。

所以在这种时候你就更好理解啊，白左这群人，他就是在生活条件很好的情况下，完全忽略了自身的危机和自身的痛感，而把所有的情感投射都放到了外界，去观察其他的生物、其他的物种。对于其他的物种掉根毛，它都能哭半天；但是对于自身很可能在几年之后就会发生的巨大危机，毫无察觉。所以我们今天为什么很多的人会觉得他们很蠢呢？其实就是因为，诶，他们……（此处原文有省略号或中断，但结合上下文应是省略）所以我们会注意到啊，白左这群人，他自己的内部的逻辑是很自洽的，非常的单一标准——就是无条件地帮助弱者，谁弱就帮谁。但是在围观的群众眼里面，他们是非常的双标的——就是他们对于特定的群体极其的关照，但是对于另外的群体呢，却极其的冷淡。其实呢，这个现象普遍性地存在，就是因为他们觉得某一个群体没有危险，于是呢，他就相信这些没有危险、实际上马上就要受害的群体是安全的、是强大的；而对另外的一个群体，他只要想象到他有痛感，他就立刻去共情过去了。所以这些人呢，既双标又单一标准，每一次的选择都是愚蠢的选择，每一次的选项都永远站在让自己的族群破灭的这条道路上面。

而今天我是很高兴的一点，就是大家普遍性地看到了白左的双标，也看到了白左的下场。这是一件最好的事情啊。我希望他们能够走得越快越好，走得越早越好——至少说能够给国内的很多正在成长之中的幻想型白左，打上一个警戒线，让他知道白左会发生什么样的结局。你像现在的欧洲，那些白左圣母心的人，相信穷人是很可怜的，于是呢，就去把贫困地区的人大规模的拉到欧洲来享受福利；而同时呢，对于欧洲的原住民，他认为你只要在这里了，你是原住民了，你就是高等人，你就是压迫者，你就是一个占便宜的人。于是，哪怕这里的原住民已经领不到退休金了，哪怕说社会的福利已经摊薄到了生活很困难的情况下了，他依然把它定义为压迫者，然后想方设法的从他们身上压榨利润，去补贴给那些从外面招揽过来的所谓的穷人。这种自我伤害的倾向啊，只要是一群人，他进入到了这种狂热的自我奉献的感觉之中，不能自拔的时候，就不可避免的会出现。

所以我是希望说，诶，我们国内的人能够引以为戒，能够看到他了之后知道：哦，原来前车是这个样子的；原来往那个方向上去跑，是会车毁人亡的。有这样的结局，那才是更好的结局啊。向大体老师致敬。
