# 第108章 高考只是预备役

每年的高考呢，都是国之大事。到了这个时候，各种配套的措施全都为他而服务啊。可以看得出来，高考真的是国家大事当中何其重要的一环。既然这么重视，所以每年到了高考季的时候，都会有风起云涌的声音啊，给大家传递关于高考的不好，有关于高考的方式让人僵化啊，高考的方式是八股啊，教出来的孩子没有创意，呃，不能够综合录取，导致这个孩子呢，就只是会分分分啊，其他的都不会，都是高分低能儿。这些话题其实已经炒作了二三十年了，但是到今天依然没到尽头，换汤不换药的说辞年年都有。所以今年呢，还是来说一下高考到底是什么。

在很多人的这个描述当中啊，高考呢，是一个人才选拔机制。那么既然是选人才嘛，叫做不拘一格降人才，什么样的人对社会有用，我们就应该选什么。所以，如果你只是去挑分数高的话，那你岂不是挑书呆子了吗？那不行。人家给出来的理由呢，常常都是如此。但是最重要的一个错误性的前提就是，高考并不是一个人才选拔机制。嗯，如果你是高考学生的家长，这个时候不要着急，我不是说你的孩子不是人才哦。高考它本质上面是一个人才预备役的筛选机制啊。我们真正的人才是什么？人才是对社会有用的人。那么也就是说，真正的人才选拔机制应该是像省考、像国考，像大中型的企业进行招募、考试一样的，这种挑出来的人是能够直接为社会做贡献的，这种才叫做人才选拔机制。而高考是这样的吗？不是。高考呢，我们是选一些聪明的、老实的、稳定的、愿意在学习系统当中打磨的孩子，选完了之后出来进入高等学府进修。他们有的呢，学习科学，有的学习文学，有的学习艺术，最后他成为了一个人才，然后再去接受人才的选拔，为社会而工作，这才是正常的一个流程。因此，高考啊，它是一个预备役的筛选方法，它并不是一个人才筛选方法。

既然它筛选的是预备人才，也就是说他的标准不是这个人有多强的能力啊——蹦得好高啊，我的天，它会飞，那不得了，立刻就有用。这种呢，叫挑人才。但是挑人才预备役，我们不需要这样。我们有一套完整的标准，就是这个人呢，脑子不能太坏啊，必须在一定的智力基准线之上。这个智力呢，指的是逻辑智力，就是他能够理解因果关系，能够排除 A、B、C、D 啊，按照这种智力筛选筛一道。同时呢，还有一部分的学科，进行的是这种秩序性的筛选，就是虽然你可能没那么聪明，但是没关系，你老老实实按部就班地学，也能够过线。这些科目啊，就是为了这些肯踏实、肯刻苦的人去准备的。所以你瞧我们的整个高考，筛选的就是这两类人：聪明的以及稳定的。这些人是最有可能被培养成人才的。所以高考干的就是这个活儿，而做这个活儿不需要你是一个全能手，不需要你会骑马、打马球，不需要你高尔夫一杆进洞啊，也不需要你有飞机驾驶执照。OK，他只需要你的脑子好、情绪好就可以了。而这个有多少重要的意义呢？前面我跟你们说的那几个项目，你有没有算过它背后需要多少的资金流？对，这就是意义的本身。

如果一个人他只需要智力好、情绪好，他对他的家庭经济条件是没有要求的。尤其是当我们的九年义务教育普及下去，公办的高中价格也极其便宜，而且还有津贴的时候，一个普通家庭甚至贫寒家庭的孩子，只要他脑子没问题，依然可以接受完整的教育。但是如果我们的所谓人才筛选标志当中，说有飞机驾驶执照的加100分啊，会打马球、拿过马球竞赛金奖的，唉，加100分是吧，高尔夫能够一杆进洞的加100分，我就问你，你就别说贫寒家庭了，小康家庭你干不干得来？不要闹了。前几年的时候，曾经跟一个家长聊过一件事情啊，他呢，觉得自己家境不错，其实也就是小康家庭而已，但是他相当的自信啊，他觉得自己家境挺好，他就希望说那个中考里面应该加上美术课啊，这才叫做综合评选，怎么可以只考语数外呢？啊，应该加上美术，艺术是很重要的人文精神。我勒个天，为啥呢？因为艺术，美术这个东西，如果你一年愿意掏10万块钱的培训费的话，一个手残也起码能够拿到60分，他就为了那60分啊，他就想要在中考当中加上美术筛选。要知道其他的那些交不起一年10万学费的孩子呢，哪怕他说他很有天分，美术可能也只能拿30分。这就是他想要的东西。这叫做啥呀？这叫做上车了的人想把车锁死。不就是这吗？这叫做阶层隔离。您就瞧瞧，只是一个中产之家、小康之家，勉强一年能够交得起10万块钱的美术培训费，仅此而已，他就想把车门锁上，让别人上不来车。你想一想那种家里面又薅好几个小目标，好多个小目标，每年都进账好几个小目标，分分钟捐你一栋楼，分分钟捐你一个学院，分分钟捐你几十年的那个什么奖学金项目的，这些大老板，家里面要是万一有十几二十个傻儿子，你觉得他会干什么？

高考的价值是什么呀？我勒个天哪，今天有很多的营销号，拼命地去宣传发达国家的美好生活。人家什么是登记制的录取呀？开放又包容。人家是什么综合型的综合素质录取呀？招揽多方面人才。唉，人家是介绍信式的录取呀，多么看重人脉关系和社会活动能力，要看你的社会背书的。例如说你想要去一个名校啊，然后你看那个名校里头那个刚需的分，兴许能够上，但是我不一定能上得了，因为它还有附加的考核项目，就是所谓的活动型背书。例如说你跑到非洲去扶个贫，然后很好，你获得了一个项目背书，然后你去进名校的时候，别人500分才能录取，你200分就能进了，还是排在前列的。这是干啥？这个叫做推荐制。你就想一想这个里头的投资项目，社会中层以下的人玩不玩得起？砸锅卖铁也许玩得起，你找得到中间的介绍人吗？所以你看这些营销号，把那些美好的大学录取的方式啊，描述得天花乱坠的时候，一定要想一想，我们的国家专门给大家普及过一个知识：在今天的中国还有6亿人月入1000以下。所以在这种情况，我们要照顾谁？谁才是多数？多数是什么多数？多数就是百姓，多数就是人民。我们这样的国家，不站在人民的视角上面考虑问题，我们居然跟着那些媒体走，去站在人家大老板、小老板的身边去考虑问题。天哪，那个公子啊，马球打得那么好，凭什么高考不让上北大呢？我的天，太不公平了，应该把马球加进去嘛，让那些打不起马球的人下课。连马球都不会打，凭什么上北大？你配吗？他的潜台词其实就是这个样子的。

所以我们要什么？高考是什么？高考只有一种原则，其他的东西一切都可以放开。它不是招揽人才的，它是筛选有可能成为人才的人的这个池子。越客观、越公正，对社会越有利。地主家的傻儿子，你自有晋升通道，不要跟普通人来争抢这种智力赛道。有智力你就上，没智力你就下。既然家里有 money，你去拼 money 不好吗？为什么要拿 money 去跟别人拼脑子呢？所以，高考只有一件事情：凡是符合这个原则的就是对的，不符合这个原则的就是错的。作为全世界自古以来最公道的一场考试，我们只要三件事——公平、公平还是公平。
