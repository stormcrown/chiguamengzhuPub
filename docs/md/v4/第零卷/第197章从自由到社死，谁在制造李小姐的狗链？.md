# 第197章 从自由到社死，谁在制造李小姐的狗链？

大连李小姐的事故，想必大家都听说了吧？这些瓜你们肯定比我吃得早、吃得透，说不定连视频都看过了，看得比我还完整。所以那个事件本身不是很重要，但事件背后的东西就变得非常微妙了，因为它成为我们整个历史进程中不可或缺的一个转折点、风向标。那么，呃，我们就可以谈一谈这个事情的背后到底有一些什么样的文化符号和社会思想的变迁，以及我为什么要把李小姐这件事情称为一场“事故”，而不是“故事”。

要知道，她身上有一些很典型的文化特征，是我之前在一年间的很多视频中不停强调的东西——就是今天的中国人身上依然保存有一部分逆向民族主义的思潮，以及奴婢心态的残余。即使是在我们飞黄腾达的今天，这些东西依然潜藏在一些人的心里。那么崇洋媚外这种事情真的司空见惯，不要太多哦。实际上这几年甚至好了很多。然而，既然称之为“事故”，那就不是崇洋媚外这么简单了——这种货色有什么好谈的？崇个洋、媚个外，好多人还幻想说“我要生一个混血，聪明又漂亮”，这种完全违背科学知识的幻想怎么来的？不就是崇洋媚外嘛。所以既然这种思想本身并不离奇，实际上现在也在逐渐扭转之中，那为什么她算事故呢？

因为李小姐她太年轻，完全没有经历过社会的捶打，甚至很有可能在她的思想茧房里迷了路——在她接受的信息茧房中迷失了方向，以至于她做出的选择匪夷所思。她出去浪荡了一下，给别人当了狗、当了奴婢之后，竟然还在别人的镜头前面主动地say hi、主动地表演、主动地录像。她这种跟别人粉丝say hi的过程，就是没有拒绝对方将视频公之于众的一种表现，也就是说她的内心深处毫不设防——这就是事故了。你要知道，出门浪一下，因为嘛，因为背负嘛，想出去给人家当舔狗嘛，这种事情很有可能不止这一桩。但为什么她闹那么大？因为她演啊！而为什么演呢？因为她觉得演了没毛病啊——这就是问题所在。她之所以成为了一起事故，就是因为她觉得演了这件事情没毛病。来，李小姐，以及各种各样的李小姐们、李先生们，向我看齐：来瞧一瞧，看一看——你为啥会觉得拴个狗链子、在大家面前say hi、吐舌头这件事情没毛病？这就是毛病本身，好不啦！

于是我们就得来探讨一下：这样一起在过去的20年间司空见惯、在近期已慢慢消弭的崇洋媚外事件，是怎样变成了一起事故的。所以在这事上，除了她自身的这种心理建设、自我构建之外，还有其他的动因——就是她竟然觉得这事儿没所谓。咋回事儿呢？这不可能是来自家庭教育——首先排除这一点。因为中国的家长嘛，记得吗？怎么说？浑身爹味：“不许做这不许做那的”，天天要我们要面子——“要面子！要面子！要什么面子呀？面子值几斤几两钱呀？”——是不是中国式家长？既然是这样充满爹味的家长，就不可能让自己的姑娘出去说“去浪吧，演给大家看，让全世界都知道你有多浪”——这不可能是中国家长规训出来的。

那谁在做这件事情？事情发生之后，在互联网上当然引发了各种各样的讨论和声浪。大部分的声浪是：“天呐，你弄得我都变得没脸没皮了，我出门都不好意思说我是中国人了。喂，你把我的信用值都打低了，好吗？我非得喷你180刀不可！”这样的声音体量最大。但同一时间实际上还有另一种声音，显得更加尖锐：“他只是违背了一点点道德而已，你不能批评他。”虽然这句话本身很荒唐，但很显然，李小姐当年所经受的舆论风浪中，她自己的思维茧房里必然有这样的声音。如果她思想的声浪中没有这种声音存在，不管她有多媚外，她都知道这事儿丢人、不能被人家知道，而不是说“录像吧！快点，千万要让大家知道我怎么当狗！”这种操作背后必然有助推力。什么样的助推力？这就是助推力。所以完全可以说，今天在互联网上试图挽救李小姐的这些人，就是将她推入深渊的推手，毫无疑问。

这些人带来的声音是什么呢？当我们总结了他们的各种言论之后，大体可以分为两个阵营，但这两个阵营常常混淆，甚至有的时候会一体化。什么阵营呢？一个是女权主义阵营，另外一个就是自由主义阵营。它们经常搞混和双打，一起来攻击我们大众的潜意识。做的事情基本上就是把女性权益和自由、权益和自由主义、自由至上绑定在一起，然后它所给我们传递的公共信息里头，就和这回洗地的内容一毛一样——没什么区别。

第一个：你不能给女性带上枷锁，否则你是不是要当塔利班？唉，这次是这样洗地的吧。来，那什么叫做“给女性带上枷锁”呢？就是女性要有道德伦理观，你不能那么浪荡——穿了小短裙出去之后很容易被人家欺负的，你不要这样子，君子不立危墙之下呀，亲。然后他说：“我可以骚，你不可以扰。你凭什么不去规训那些小流氓，凭什么要来规训我漂亮的小姑娘？”哒哒，不好意思——人家小流氓不犯事儿的时候我真拴不住，但是我想保护你。保护你的这个过程相当于什么呢？我看见你拿着手机在台阶旁边走路了，我就说：“停一停啊，你千万不能看手机下台阶儿，小心摔着。”然后你说：“你是不是觉得我是你的私有财产？”就是这样的操作方式——人家在劝你保持安全距离，你觉得他把你看成他的私产。我是你家私产吗？不好意思，我家私产我从来不劝，我家私产要是干了什么坏事，一巴掌就呼过去了，谁跟你搞劝说这件事啊？这是拿你当人。但是他们扭曲了这个概念，混淆了这个概念，然后拿来灌输。

这是过去很多年，我们的女权主义者混搭了自由主义之后，去教给小姑娘们的思想：你要想当一个人，你就得想干啥就干啥啊，你不要在乎别人怎么说——别人怎么说跟你什么事？道德跟你有什么关系？伦理与你有什么关系？只要不违法，你就是好好的一个大女主。基本上这个自由与女权主义的混合双打走的就是这个路子。而李小姐身上的这起事故很明显就有这个痕迹——她能够这样肆无忌惮，必然是受到了这些人的思想影响，以至于她出现了一些认知上的偏差，甚至真的觉得道德是一种枷锁。

所以，道德到底是啥？这个话听起来吧，从上世纪80年代开始，中国的传统道德其实就一直在被批评着，已经批评了好几十年了。大家已经习惯了，甚至有的时候一想起传统道德，都会觉得闭塞、压抑，“你这个东西不好，糟粕”。我们要考虑一下，新时代要摩登化嘛，要国际化嘛，要接轨嘛，是吧？但是呢，我们的传统道德到底是什么？传统道德的“道德”到底是什么？道德其实是一种社会规范，它和法律一样，也是一种社会规范，是一种社会共识。那为什么要有社会规范？你闲着没事干都有法律，你还要这东西干什么？因为社会是什么？社会是好多好多人聚到一起，而且是陌生人——谁都不认识谁。陌生人彼此聚合，然后相隔很近，抬头不见低头见，最大的问题就是信任成本。因为人类在原始阶段不存在这个事——反正谁都不信，谁见了面先偷袭，赢了再说是吧。但我们进入到社会化阶段之后，就不能这么干了，就要有一个公共的信任。我说你好，你也得跟我说你好——这就叫共同的认知，这就是信任感。那这种信任感怎么构造？我们要构造信任感的时候，就得大家都有一个底线。那这个底线是啥？这个底线就是道德。

为啥要有道德？为什么要觉得“我们出门的时候不能够去偷别人的东西”？你听起来这不是废话吗？不好意思啊——你有没有去看外国博主自己的海外经历？说明他们的外卖、快递什么的，要是放在门口了（都不说马路上，就放在自己家门口），3分钟没拿，它就消失了，分分钟消失。不管你有多少监控都没用，找不着它，反正撒丫子就跑路了。这个是什么？这个就是没有共同社会底线共识。但中国这里是这样吗？那偷快递是多丢人的事情啊！所以我们的底线就是：你不能随便去偷别人的东西，放马路上你也不能拿。这个东西就叫底线。因为有这个底线，所以我可以跟外卖小哥说：“我这会儿没空，你把它放门口。”这是信任度。但是如果说有人打破这个信任了——你放一个我偷一个，你放一个我偷一个——下一步呢？再也不敢做这件事情了。这就是信任成本提高了啊。

那么放到李小姐这个案例当中，其实原理是一样的：为什么出现千夫所指？这玩意儿干你什么事？她一个人出去浪，与你何干？你那么多事儿！——信任成本变了呀。这个事儿上面，最近是不是有人洗地的时候说：“你要恢复古代的沉塘吗？你要浸猪笼吗？”来，我们来说一下浸猪笼这件事情的基本原理：为什么古人要浸猪笼？你以为爹不爱女儿，你以为老族长就喜欢欺负村里的姑娘吗？不是这样的。而是一个偷情背德的事件已经曝光于天下了，大家都知道了的情况下，如果你不做出惩处，那就说明你们村儿放纵这种行为，我高度怀疑你们村的男男女女都不干净——婚丧嫁娶绝对不找你们家。为什么？信任成本过于高，我信不过你们家。这就叫做成本问题。所以为什么要大张旗鼓地办仪式、搞沉塘、浸猪笼（尽管这是私刑，在古代也是非法的），而且昭告出去让隔壁的村落都知道？因为他得让别人晓得：这个事儿在我们这里是不被容忍的，所以我们这里所有活着的人都是有底线的，你可以放心嫁娶。

所以，干嘛要千夫所指？干嘛要让一个背德的人付出社会性死亡的代价？这其实就是在洗清自己。如果说一个人出去浪荡了，还昭告天下了，所有人都知道她拴狗链子了、汪汪汪了，然后大家说“哎呀，干得好！干得漂亮！干得呱呱！叫她自由了！她了不起了！”那在外部的目光看来的话，你们所有鼓掌的人都跟她一样——你也是这样的，明天要把狗链子拴你脖子上面。而你愿意负担这样的成本吗？不愿意。不愿意你要怎么办？拴狗链子的那条就是狗。这个叫做什么？这叫做千夫所指，这叫做违背道德的成本。这也是为什么我们要有道德，为什么道德背后要有行为的制约和牵制。尽管你说浸猪笼这种私刑很明显是不好的、落后的、伤害了生命的，但它的行为逻辑本身内在是自洽的，它有逻辑行为内核。所以放到今天的这起事故当中，当一部分的自由女权主义者混合双打冲出来说“她自己的事儿跟你合适”的时候，那不好意思啊——真干我事儿，而且与所有的中国人、与所有的中国女性都有干系，它严重地影响到了大家的公共社会评分。那这个时候她面临到这种成本、付出这种代价，就变成了天经地义的。这不是将其视为私有物品，而是将她视为社会当中的一份子。

那这事儿是专门针对女人的吗？呃，不是啊，亲，不是的。古时候的浸猪笼也不是只浸女人，但讲故事的人只告诉你浸女人——实际上浸猪笼是进去两个，成双成对的——所有背德的人都要付出代价，这个地方才值得信赖。所以你看我们前几天的时候，是不是有一个老爷们儿又跪到了洋人的面前？尽管说他的这个行为有点喝醉了之后在那里故意挑衅的表现，但是他竟然跪了，他也面临到了千夫所指。只是有一些人觉得他这个有点怄气的行为，所以不是那么严重，可是影响依然存在，依然是骂声一片。这是什么？这是拿你当个人儿。所以你只要跪了我就开喷，否则的话，我们整个社会都跟着一起跪了，这种方式显然不行。

因此，当我们能够理解什么是道德的时候，那去指摘道德的这群人就很有意思了。他干什么要指摘道德呢？从上世纪的80年代开始到本世纪初，大约有30年左右的时间，社会上面和舆论场上面是不断地指责传统道德的，把它批得体无完肤，甚至不考虑当时的时代的背景、生产力的条件，反正批就对了——只要是过去的、只要是中国的，就是落后的。那这些人是在干什么？他实际上不是在批你的道德，他是在搞道德置换。什么样的道德置换？它是用西方式的道德、西方式的伦理和教化来置换中国人的道德伦理的思维方式。所以你看，在旧有的道德被批判了之后，中国社会上不是没有其他共识的，它出现了新的共识。什么样的共识呢？外国的月亮是圆的，外国的空气是香的，外国的人是美的，外国的食物虽然不好吃但是是健康的——这些东西是那种西方至上、西方中心主义的庞大舆论战中不可或缺的一环。就是道德，其实也是舆论战当中理所应当会存在的一个环节：他的道德是好的，我们的道德是坏的，这也是一种中外博弈。

而今天呢，因为中国人又开始撑起腰来了，于是我们开始去思考我们原本的道德指征是怎样的，而西方人的那个道德在今天又暴露出来什么死相。于是大家的心态开始发生了一些转变，但是有些人转变的速度有点慢——尤其是当她出现了信息茧房，被自由主义女权思想裹挟进去的时候，她就误以为今天的世界依然是西方人说了算、白人说了算的世界：只要是个白人，他就长得好看；只要是个白人，他就高级；只要是个白人，他就是正确；只要是个白人，他就叫做浪漫、叫做美好、叫做文明、摩登、国际化。这就是今天李小姐身上所犯下的茧房之下的错误。而这个茧房才是罪魁祸首。

什么样的茧房？那就不得不说到刚才说的“自由女权主义的混合双打”。那么女权主义在之前我曾经聊过很多期：女性权益本身没有任何的问题，但是当它变成女权主义的时候，这个事儿问题就大了。因为女权作为一个“主义”，那就是“女性权益最重要”的意思。一个社会，当你把女性权益提到最重要的位置上，你只能对标一件事情，就是男的。所以你会发现女性不一定要拿男人怎么样，但是女权主义一定会把矛头的尖尖指向所有有关联以及无关联的男性，因为只要是女性至上、女性视角，性别只有两种。所以为什么西方人在对抗女权主义的时候会搞出98种性别？因为他发现性别只有两种的时候，你打的全是男人——“不好意思，我现在不承认我是男人了，我是流水一般的性别，我今天是个塑料袋儿，你打不到我”——这是一种对抗的方法。但是呢，它背后又有另外的推手，咱们今天先不谈它。总之呢，当你是一个女权主义者的时候，你的对标对象就变成了各种各样本来是合作方的男性，就产生了社会的撕裂。

同时，自由也是好东西——毫无疑问的，自由是多么美好的一个词汇：我想做什么就做什么，我不想做什么就不做什么。然而自由是好东西，到了自由主义的时候就不一定了。因为自由主义就是“自由就是最重要的事情”。然而我们说了，那么多人一起聚合成了一个社会，大家抬头不见低头见，明明是陌生人却需要互相say hi的，这个生活当中要有共同的认知、共同的信用成本。每一个人都需要交付出去一定的成本。是什么呢？我的自由不能触到他人的边界——这个事儿按理来说吧，属于小学思想政治的范围，但是我不知道今天课本里面教不教。然而呢，咱们得知道一件事：所有的自由不能触及到他人的利益。那自由主义是啥呢？“你管我干啥呢？你管不着我，你就是管不着我，我就是自由的。”这个玩意儿叫做自由主义。它把自由抬到了一个过高的位置上面去，以至于它必然伤害到他人的利益，甚至公共的利益。像今天的李小姐的事故，为什么变成事故？就是因为她在宣传的这个过程当中，已经伤及了公共权益。而她自己做这件事情的时候，在对着镜头说“粉丝们，你们好”的时候，她一定没有意识到这一点，因为她的茧房告诉她：你就是一个自由的女性，你想干什么就干什么，谁都不能指责你——否则我们帮你喷他，他个老古董，让他为此付出代价吧。因为她天天听这些胡搅蛮缠的言论，以至于她误以为这就是世界的真相，最后她付出了代价。

所以这件事情当中，李小姐本人毫无疑问是一个受害者——无论是那个境外的渣男，同时还有国内的各种各样的推手，在不断地对她耳语说“没事儿没事儿，这个代价你不用付”。这些人都是促成她出现事故、付出代价的凶手、刽子手。然而最后承担代价的必然是李小姐本人。而在这件事情的背后，社会公众层面上又起到了什么样的作用？就怎么说呢，这叫做福祸相依。尽管说这是一个灾难性的事故，所有的中国人都觉得面上无光，然而它确确实实表现出了一个转折点——一个攻守异行的转折点。在过去的几十年间，这种自由主义至上的思潮的宣传始终都是在传媒领域当中最最崇高的方向——谁去批判一个人的私德，就好像自己都败坏了一样。而今天，整个互联网上面的声浪发生了转变，那些洗地的声音已经没有那么大声了。为什么？因为这个事儿形成了一个社会公共痛点。我们要知道，一件事要获得解决，不是这个事儿出现了就可以，而是这个事儿触动了绝大多数的利益，能够让很多人感觉到疼痛，这件事情才会获得解决。所以，很遗憾的是，李小姐受害这件事情本身，却会让崇洋媚外这件事情的风潮发生一次逆转——因为大家都会意识到：这事儿啊要处置了，在做这事儿的人要付出代价了。你不能再像过去20年一样，崇洋媚外还觉得自己时尚、与国际接轨。中国人的规矩要由中国人说了算，中国人的道德标准就是比老外的高——我们就是高，你要是不接受高道德，你出去。而这样的社会氛围，它本身对于我们自己的内部社会构建是大有好处的，对于提升我们的社会认知也是大有好处的。所以再遇到什么凭空跪的、拴狗绳的、汪汪叫的、摇尾巴的，喷就是了——放心大胆地喷，总归得让他付出成本。
