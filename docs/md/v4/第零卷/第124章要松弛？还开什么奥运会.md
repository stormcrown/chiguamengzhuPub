# 第124章 要松弛？还开什么奥运会

潘展乐小帅哥的采访，大家都看了没有？是不是特别提振士气？超好看的哈！我们的运动员在游泳项目上摘得金牌之后，下来接受采访时说：“哼，让他们那些人都不讲礼貌，跟他打招呼都不理我，还一副瞧不起人的样子。现在我在这么难熬的游泳池里面，都把他们游过去了。”唉，我很喜欢这一段非常坦荡的采访，直接就把前几天在我们中国互联网上十分活跃的那些又土又直的大侄子们的言论，啪啪地扇了好几巴掌。

大家记得前几天巴黎奥运开幕式上面，搞出那些群魔乱舞之象之后，国内的大侄子是怎么说的？那是人家特有的松弛感。这只是一个运动会，又不是生死之战，对金牌毫不在乎，对一切毫不在意，这才是优雅，这才是高贵，这才是松弛。一个运动会，你整出最大特长是松弛，都松弛了，你开party不好吗？为什么要开运动会呢？所以你瞧，真正的运动员对于运动会是什么态度？运动咱就是得赢，赢了才有面儿；如果不赢，就下次再卷土重来，这叫做志气。

所以今天要聊运动会到底是什么。这世界上如果要松弛的话，大家要玩的东西很多，不会去搞运动会的。运动会是干什么的？我们现代的奥林匹克运动会，是19世纪的时候开始出现的。那么当它开始说要办这种运动会的时候，当然了，安插了一个非常非常久远的历史，说那几千年前，我们就已经每隔4年办一次运动会。它代表了什么？代表了什么？代表了什么？所以呢，在今天这样的时代里头，已经走向了国际化了，每一个国家都应该参与到这项盛会当中来，然后彼此之间交流，彼此之间了解，用和平的方式达成全世界所有人类的握手言和，友谊第一，比赛第二。是的，他们在开创这项运动会的时候，确实说得很好听，办的也好像是这个样子。但是这段话是什么呢？这段话其实是一个运动会的广告，它本身并不是说真正的源流。要知道你真的很难印证古希腊雅典城到底有没有这种运动会，甚至雅典城那个时候到底是个城还是个村儿，实际上都是查无实据的。那么它其实就是为了自己当时想要设这样的一个运动会，而专门去找了一个老祖宗。当然，这事儿本身不是很重要，毕竟我们这里卖一个瓜子儿，都要在包装袋上去写“当年清朝的时候，有一个什么，人家一不小心做出了什么样”的卤瓜子儿，这是广告宣发的正常操作手段啊，咱们就姑且接受它。

总之呢，虽然这个运动会在操办的时候，确实一直说的是大家一起来活动一下，然后呢，说明一下我们彼此之间的友谊。但是我们看一下它诞生的时代背景：19世纪。19世纪的意思就是一八几几年啊，一八开头的这个年代。大家去想一下，那是个什么时代？从工业革命西方开始支棱起来了之后，他们一直在做的事情，就是大张旗鼓的全球劫掠以及全球殖民。所以当时所推崇的这个“我们实现了国际化”是什么？国际化呢，是西方白人劫掠世界的国际化。所以那个时代，他搞全球所有的国家都应该参加运动会，是干啥呢？他的目的到底是什么？这就要看运动这件事情是怎么样决定输赢的。

我们知道在现代运动会上面，我们有什么田径类，然后有什么球儿，然后还有一些什么游泳啊等等的相关的竞技项目嘛。这些东西都有，我们去比较一下所有的这些项目当中的一些特点啊。什么人有可能会获胜呢？营养良好的人。吃不饱饭的人，你想跑步跑赢那是不可能的；有时间专门训练的人，白天还在种地，然后晚上呢，不睡觉就腰酸背疼的人也是不可能的。同时呢，能够跑到这个会场里面来去参加运动会，要有配套的专业设备。哪怕最早期的时候，大家设备都不太专业，但是起码你的鞋子轻不轻便、摩擦力好不好，衣服是不是透气，是不是会裹在身上，这些都是当时最先进技术的一个表现。今天我们知道，我们在进行运动会比赛的时候，所有的运动员身上的东西都是高科技产品。那个时代看上去有点落后，但是那也是当时最高科技的产品了。所以你想，作为那样一个全球绝大多数国家都在被侵略、都在被殖民、被奴役、被盘剥的时代，谁能够在运动会上面独领风骚，展示自己的能力和才华？对，吃饱了饭、有闲钱、有功夫、有技术、有生产能力——谁呀？哦，殖民者。诶，是的。在19世纪那样的一个特殊时期，全球真正能够在运动会上面崭露头角的，除了各地那些天赋异禀的天选之子之外，只有殖民国家的那些专业运动员。所以那个时候办这样的运动盛会，它的结果是非常好预期的：就是那些富庶的、发达的、先进的国家，就是那些当时的殖民国家，他们会包揽大部分的奖牌；而其他的被殖民的地方，有一些天赋异禀的运动员可能会崭露头角，但是大多数的时候都是去陪跑的。

然后同样呢，我们想一下我们国家在那个时代的遭遇。记不记得那个时候，我们的脑袋上面还有这样的一个隐形的牌匾，叫做“东亚病夫”？为什么非得要把这四个字弄在所有中国人的脸上，说我们弱呢？因为那个时候我们是被侵略、被殖民的对象，所以在整个国际舆论市场上面，中国人的形象就是那种弱的、没有抱负的、起不来身的、苦哈哈的农民的形象。这样的人有可能在运动会上崭露头角吗？不太能够。所以那个时候的中国官方政府，为什么会在自己内部纷争非常剧烈、经济条件十分糟糕、而且战争频发的时期，还非得要扑腾着选出几个运动员出来？我们记得那个时候对这些艰苦努力训练、出去打比赛的运动员是怎么形容的吗？他们叫做“为国争光”。“为国争光”这四个字，一直到几年前，咱们还在普遍地运用它。就是因为参加这种国际性的赛事，就是要洗清我们脑门子上那个“东亚病夫”四个字，我们得在国际的舞台上面去证明自己身强力壮，有竞争的能力、有聪慧、有体能。所以你说这个玩意儿叫做“松弛”？这跟松弛有什么关系啊？运动会是什么？运动会就是强者展示自己肌肉的地方，只不过就是不打仗而已。所以本质上面来讲，我们去看运动会它本身的诞生历史，你就知道了：它其实是列强炫耀肌肉、耀武扬威的一个场所。虽然表现的形式是大家比一下吧，玩一下吧，搞个party吧。形式不重要，实质才重要。即使是当年昏庸腐败的民国政府，他们也是很清楚这一点的，所以无论如何都要派员参加。如果我们连派员参加的能力都没有，那就更拉胯了。这也是为什么啊，从新中国建立以来，我们一直以举国之力去培养这些运动员，让他们越来越优秀，努力地奔向国际赛场上，这也是为国争光。而且不是中国人这么矫情，不是中国人走哪卷哪，而是全世界都是这样的，包括你心中那些非常松弛的列强们，更是这样的。

他们在一开始设运动会的时候，是因为他们是全世界吃最饱的饭、用最先进的科技的人，而且又是主办方，规矩他来定，能够占到最大的便宜。但是呢，随着世界格局的改变，我们就看见了有很多的国家也安稳下来了，大家发展起来，慢慢地在运动会上面崭露头角。尤其是像中国这样的发展中国家，当我们走到了一定阶段之后，我们在金牌榜上总是霸榜的。那这个时候那些西方列强们真的就很松弛地认为“友谊第一，比赛第二”无所谓了吗？如果真的是这样的话，请问我们的游泳运动员，为什么一天要尿检那么多次？检查的次数都已经超过一个人类一天能够尿尿的次数了。然后你跟我说这是西方人的松弛感？不准竞争对手睡觉就叫做松弛？棒棒的。

我们再看一下我们的优秀项目：乒乓球啊，国球啊，全世界我们的乒乓球打得最好。乒乓球是怎么进入奥林匹克运动赛场的呢？要知道这个球可不是中国人发明的，它本来是法国人发明的，以前呢，是一种室内网球的性质啊，相当于是一些办公室的人，一些小贵族们，想要把那些暴晒场所下面暴力运行的网球项目，搬到自己的卧室里面、客厅里面大家进行一下娱乐比斗。然后呢，端起茶水、端起咖啡，大家就爽一下。就这个活动硬是被搬到了奥林匹克赛场上干嘛？因为他们打了好多年，他们觉得自己好娴熟，这是他们的特长项目，他们发明了这个规则，是他们定的，设备标准是他们定的。于是呢，他们把把这个项目主动立下来了。但是中国人在这个方面天赋异禀，我们最擅长的东西就是精细微操作。中国人这个品种啊，他的特长就是我们每一块肌肉都能用得妙到毫巅。也许不是力量最强的，但是呢，用微妙来解释的话，一点都不离谱。所以很快的，我们就通过科学的训练，实现了在乒乓球这个专业上面的霸榜。实现了之后发生的情况是什么？大家去看一下历年的新闻就知道了。当我们几乎包揽了所有全球的乒乓球的最高奖牌之后，人家改了乒乓球的大小，然后我们还是拿了奖牌；他们改了拍子，后来我们还是拿了奖牌；他们改了场地，但是我们还是拿奖牌。于是他们就开始商量说，这个运动展现不出各国竞争的风貌，应该予以取消。这玩意儿就叫松弛是吧？拿不着奖就要撤项目，这个东西叫做松弛？这不就是耀武扬威不成，被人家糊了一脸之后，决定说这个项目立错了，应该撤下来。

所以什么是运动会？如果说我们要松弛，我们为什么要办运动会？我们办服装走秀不好吗？是不是？我们要松弛，我们办大型酒会不好吗？时代发展了，大家都能够买得起机票了，想要松弛还不容易？找个场地寻一点歌舞项目，搞个全球联欢，这才叫做松弛呢，这才叫做友谊第一呢。你都已经弄到赛场上面了，金牌、银牌、铜牌含金量都不一样了，领奖的台子站的高低都不一样了，放谁家的国歌，升谁家的国旗，都要立一个规矩了，居然跟我谈松弛？运动会它就不是一个松弛的地方，它就是来竞技的，它就是来扬我国威的。所以我就支持中国的运动健儿，在遇到这些委屈的时候，直接刚回去，直接扬我国威，直接说我不同意。凭什么欺负我？你跟我没礼貌，我就让你站不上讲台。这就是最好的运动会，这也是真正的运动精神。

然后同样呢，我们想一下我们国家在那个时代的遭遇。记不记得那个时候，我们的脑袋上面还有这样的一个隐形的牌匾，叫做“东亚病夫”？为什么非得要把这四个字弄在所有中国人的脸上，说我们弱呢？因为那个时候我们是被侵略、被殖民的对象，所以在整个国际舆论市场上面，中国人的形象就是那种弱的、没有抱负的、起不来身的、苦哈哈的农民的形象。这样的人有可能在运动会上崭露头角吗？不太能够。所以那个时候的中国官方政府，为什么会在自己内部纷争非常剧烈、经济条件十分糟糕、而且战争频发的时期，还非得要扑腾着选出几个运动员出来？我们记得那个时候对这些艰苦努力训练、出去打比赛的运动员是怎么形容的吗？他们叫做“为国争光”。“为国争光”这四个字，一直到几年前，咱们还在普遍地运用它。就是因为参加这种国际性的赛事，就是要洗清我们脑门子上那个“东亚病夫”四个字，我们得在国际的舞台上面去证明自己身强力壮，有竞争的能力、有聪慧、有体能。所以你说这个玩意儿叫做“松弛”？这跟松弛有什么关系啊？运动会是什么？运动会就是强者展示自己肌肉的地方，只不过就是不打仗而已。所以本质上面来讲，我们去看运动会它本身的诞生历史，你就知道了：它其实是列强炫耀肌肉、耀武扬威的一个场所。虽然表现的形式是大家比一下吧，玩一下吧，搞个party吧。形式不重要，实质才重要。即使是当年昏庸腐败的民国政府，他们也是很清楚这一点的，所以无论如何都要派员参加。如果我们连派员参加的能力都没有，那就更拉胯了。这也是为什么啊，从新中国建立以来，我们一直以举国之力去培养这些运动员，让他们越来越优秀，努力地奔向国际赛场上，这也是为国争光。而且不是中国人这么矫情，不是中国人走哪卷哪，而是全世界都是这样的，包括你心中那些非常松弛的列强们，更是这样的。

他们在一开始设运动会的时候，是因为他们是全世界吃最饱的饭、用最先进的科技的人，而且又是主办方，规矩他来定，能够占到最大的便宜。但是呢，随着世界格局的改变，我们就看见了有很多的国家也安稳下来了，大家发展起来，慢慢地在运动会上面崭露头角。尤其是像中国这样的发展中国家，当我们走到了一定阶段之后，我们在金牌榜上总是霸榜的。那这个时候那些西方列强们真的就很松弛地认为“友谊第一，比赛第二”无所谓了吗？如果真的是这样的话，请问我们的游泳运动员，为什么一天要尿检那么多次？检查的次数都已经超过一个人类一天能够尿尿的次数了。然后你跟我说这是西方人的松弛感？不准竞争对手睡觉就叫做松弛？棒棒的。

我们再看一下我们的优秀项目：乒乓球啊，国球啊，全世界我们的乒乓球打得最好。乒乓球是怎么进入奥林匹克运动赛场的呢？要知道这个球可不是中国人发明的，它本来是法国人发明的，以前呢，是一种室内网球的性质啊，相当于是一些办公室的人，一些小贵族们，想要把那些暴晒场所下面暴力运行的网球项目，搬到自己的卧室里面、客厅里面大家进行一下娱乐比斗。然后呢，端起茶水、端起咖啡，大家就爽一下。就这个活动硬是被搬到了奥林匹克赛场上干嘛？因为他们打了好多年，他们觉得自己好娴熟，这是他们的特长项目，他们发明了这个规则，是他们定的，设备标准是他们定的。于是呢，他们把把这个项目主动立下来了。但是中国人在这个方面天赋异禀，我们最擅长的东西就是精细微操作。中国人这个品种啊，他的特长就是我们每一块肌肉都能用得妙到毫巅。也许不是力量最强的，但是呢，用微妙来解释的话，一点都不离谱。所以很快的，我们就通过科学的训练，实现了在乒乓球这个专业上面的霸榜。实现了之后发生的情况是什么？大家去看一下历年的新闻就知道了。当我们几乎包揽了所有全球的乒乓球的最高奖牌之后，人家改了乒乓球的大小，然后我们还是拿了奖牌；他们改了拍子，后来我们还是拿了奖牌；他们改了场地，但是我们还是拿奖牌。于是他们就开始商量说，这个运动展现不出各国竞争的风貌，应该予以取消。这玩意儿就叫松弛是吧？拿不着奖就要撤项目，这个东西叫做松弛？这不就是耀武扬威不成，被人家糊了一脸之后，决定说这个项目立错了，应该撤下来。

所以什么是运动会？如果说我们要松弛，我们为什么要办运动会？我们办服装走秀不好吗？是不是？我们要松弛，我们办大型酒会不好吗？时代发展了，大家都能够买得起机票了，想要松弛还不容易？找个场地寻一点歌舞项目，搞个全球联欢，这才叫做松弛呢，这才叫做友谊第一呢。你都已经弄到赛场上面了，金牌、银牌、铜牌含金量都不一样了，领奖的台子站的高低都不一样了，放谁家的国歌，升谁家的国旗，都要立一个规矩了，居然跟我谈松弛？运动会它就不是一个松弛的地方，它就是来竞技的，它就是来扬我国威的。所以我就支持中国的运动健儿，在遇到这些委屈的时候，直接刚回去，直接扬我国威，直接说我不同意。凭什么欺负我？你跟我没礼貌，我就让你站不上讲台。这就是最好的运动会，这也是真正的运动精神。
