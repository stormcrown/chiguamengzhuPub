# 第120章 像人一样活着

有个小伙伴跟我提出了一个问题：在革命年代，有无数的仁人志士抛头颅、洒热血，抛家舍业地为了整个国家和民族的兴盛而奋斗。然而，在现代人类行为学和人类心理学的科学研究范畴里，有一个非常著名的理论叫做马斯洛需求理论。这套理论认为，人类会首先满足自己的生存和生理需求，以及爱与被爱、安全感的需求，然后才会去追求理想、梦想、未来和自我成就。那么，这套行为心理学的判断，是不是就否定了过去先烈们的努力？毕竟，他们当时所面临的局面，为什么没有首先保证自己的安全、保证自己活着呢？于是，他提出了一个疑问：这种理论的出现，是不是就是为了解构我们的英雄？这个猜想本身有一点点阴谋论，所以我们姑且不从角度去解析它的动机——我无法解释一个学者为什么要这样分析。只能说，我们从理论开始，也到理论结束，看看马斯洛的需求理论与我们过去英雄辈出的时代之间到底有什么关联和矛盾。

在这套人类需求层次理论里，我们首先要注意一点：西方人很奇特，他们的古典著作留下来的确实很罕见，基本上没找着，但他们却热衷于构建各种理论体系。今天我们会发现，无论是自然科学还是社科人文，都有大量基础理论的信息是西方学者贡献的。而这些理论都有一个共性：它们在用自然科学的1+1=2、2+2=4的逻辑分析方法去解析人类行为和社会构建。我觉得这种科研方式有点机械，没有我们的老祖宗来得灵活。但我们要习惯，现在的社会科学就是这样研究的。马斯洛的需求层次体系也如出一辙，是对人类行为的一种解析。这种解析方式并非没有合适的应用场景——它在现代企业管理中非常有用。比如，你在招工时，要让员工吃饱饭，老板该给多少薪水？这叫生理需求；然后要让员工觉得安全、不会被辞退，这叫安全需求；还要让员工有被需要的感觉，有爱与被爱、融入团体的感觉，比如进行团建，这就是第三层次的需求；最后就是自我实现，理想不就是所谓的走上人生巅峰、迎娶白富美吗？看到了吗？他的需求层次是非常个体化、非常私人的，讲究个人体会。一旦放到现代社会普通个人身上，它可以完美印证。这也是为什么这套理论在各种营销号、鸡汤、杂志上被频繁描述——只要讲一个人的成长，他就会把这个需求层次体系咔嚓咔嚓地讲一遍，然后说“我们理论联系实践了，兑现了，是不是很合理？”通常都会这样操作。但当我们反复聆听它时，有时会被这个理论裹挟住，因为实际上它解释的是现代普通人的行为和心理需求，而不是人类社会整个历史进程中都普世的真理。例如，在那些动荡的岁月里，它其实是满纸荒唐言。为什么这么说？放到这里，我们可以绕开一点：为什么这种生存层次的描述一拿出来给中国人宣传时，特别容易打开市场？从一开始，我们的杂志、营销号推出它时，很多人很快就接受了。我们那么容易被新知识吗？不是，是因为它太像我们的旧知识了。我们古人也说过类似的话：仓廪实则知礼节，衣食足则知荣辱。发现了吧，一模一样！所谓的五个层次，说的就是仓廪实则知礼节、衣食足则知荣辱。所以中国人为什么那么快接受它？因为这跟中国人想的一模一样，只是换了一套话术，so easy，我很快就接纳了。但要知道，我们老祖先说话从来是两头堵的，它从来不会只说一个方面，必然还有另一个方面。什么方面呢？举个例子：“由俭入奢易，由奢入俭难。”这句话和前面有什么关系？我来翻译一下：由俭入奢易——之前过的是苦日子，现在每顿饭都有二两肉、一两酒，这种好生活有什么享受不得的？但如果你每天都有肥鸡大鸭子，后来开始十日一锅粥，那日子肯定苦得不要不要的，这就叫由俭入奢易、由奢入俭难。这个解释是物质层面的，但同样适用于精神层面。它表现为：在革命年代，我们的革命先驱曾说过：“生命诚可贵，爱情价更高。若为自由故，二者皆可抛。”生命是什么？爱情是什么？这是低层次需求；自由是什么？这是高层次需求。为什么前两者宝贵，却可以为后者抛掉？另一位革命先驱给出了答案，他写道：“为人进出的门紧锁着，为狗爬出的洞敞开着。”为什么自由最重要？因为当过了人，真的很难再去做一条狗——这就是由奢入俭难。

那么革命年代的中国人，包括历朝历代发动过革命、做出过积极反抗的中国人，他们的思想里是什么？为什么他们没有先考虑吃饱饭、跪在地上讨完饭之后再考虑自我成长（这才是马斯洛生存体系的顺序），而是颠倒过来，在自己没饭吃时拼命革命战斗，甚至明知自己享受不到战果，也宁可让战果交给其他人，也要冲上前去？这件事是为什么呢？我前面说了，因为马斯洛的体系是放在一个人身上的，而任何人类社会的革命或斗争从来不是个人的，而是面向集体、群体、国家、民族、血脉，面向的是子子孙孙无穷匮也。这就是一个根本性的差别。当你用个人行为的心理去分析一个民族命运时，永远解析不通。因为在革命年代，我们所面临的局面是：作为个体，他可能优先考虑吃饭问题，再考虑理想；但作为整个国家民族，当时就是生死问题——这是第一层次的需要，即存在。而一个国家、民族的存在，它仅仅只是肉身上的存活吗？明显不是。这就是人和狗的区别：狗活着是肉体的存活，去哪里讨饭都可以；但人活着不只是肉体存活，更是精神存活。所以革命时期，人们首先去满足国家民族的生存，而这种生存绝不是跪地乞降式的延续。有没有这种想法的人呢？有的。要知道，当时也有一些人说，为了国家民族的存续，只要存续就行了。怎么存续？有一种非常easy的办法，叫“降欧美不如降日本”。这是谁？哎，汪精卫曲线救国棒棒哒。所以你看，无论哪条道路，都在试图满足国家民族的整体生存。但在这种生存道路选择时，去印证马斯洛需求体系就会错对层次——在个体上，它表现为一个至高的需求泯灭了所有底层需求；但对于国家民族的存续而言，它进行的就是最底层的需求：活着，并且像一个人一样活着。

所以接下来，我们来聊聊如何像一个人一样活着。这其实不太容易，甚至在人类社会中也并不普及，需要很多支持。我们先从一些较早的例证说起。例如，当年满洲贵族从白山黑水之间崛起时，他们是一个很小很小的团体，通过军事方式崛起，之后需要拉帮结伙。拉帮结伙的过程是怎么操作的？在他们的第一代首领那里，方式远不是什么“团结一切可以团结的力量”，而是大杀四方——先往死里揍，揍得你家快要咽气了，然后把家里的孩子牵出来，为我打仗、为我养马。当时东北土地上的各个部族，绝大部分都是这样被满洲军事贵族收服的。这个过程非常血腥残酷，甚至发生了很多屠族事件。最后，被收拢的这些部族对整个满洲贵族体系忠心耿耿。但我们会发现，当同样的满洲贵族军事群体崛起、终于扫荡南明小朝廷并控制整个中原大地时，他们面临的困境和当年在满洲地区崛起时完全不一样——抗清运动此起彼伏，两百多年从未停歇。这又是为什么？为什么就是不肯忠诚、不肯投诚？他们使了那么多招数，甚至让你忘记祖先的衣冠长什么样，甚至让你忘记老祖母的脚到底是几寸大，都到了这种地步，为什么你还是要不停地造反？这一点，曾经有很多任皇帝都提出过疑问：“朕已经是这样的明君了，但总有刁民想害朕。”为什么呢？这是因为中原大地上的人群和其他地方的部族不太一样。这里的人祖传有一种“记日记”的恶劣习惯，而这个“日记”就是我们的祖辈如何生存、如何生长、如何斗争的历史。在这些所有成长和斗争的历史中，有无数印记都在告诉我们：我们从奴隶开始站起来，开始说“子子孙孙无穷匮也”，开始说“王侯将相宁有种乎”，开始说所有的人生而为人。所以，在我们的典籍里，本来就包含了人、爱、爱有等差、兼爱非攻等各种信息。这些信息通过一本本史书、一个个顺口溜、耳熟能详的童谣，不断传递到每一个人身上、耳朵里、记忆深处，于是造就了一种印象：虽然今天有人称我们为“民人”，与其他人产生了区隔，有人要求我们成为奴婢、进入奴籍，但永远有一种潜在的声音告诉我们，我们曾经站直过，当过人。所以，为什么清朝两百多年革命从未停歇？不是什么真的缅怀朱明——朱明王朝到末年一团稀糟，谁会缅怀它？缅怀的是我们曾经当过人的日子，曾经不用跪在地上说自己是奴才的日子。所以由俭入奢易，由奢入俭难。这就是我们从奴隶解放为人，走了一千年的岁月，等我们已经当人当了两千年之后，再被人摁在地上说“你要跪地称奴才”，那是不行的——这叫做由奢入俭难。

同样，在革命时期，我们的先烈为什么要那样说？为什么说“为人进出的门关闭了，而为狗爬出的洞敞开着”？这是在告诉大家，他为什么而革命。不是为了活着——当狗也能活着，狗粮没有吗？今天还有很多人在吃狗粮，吃得哇哇香，还对别人不吃狗粮的行为嗤之以鼻。但是，咱们吃过人的饭呀，红烧肉嘎嘎香，为什么要去吃狗食呢？这个就是由奢入俭难。这也是当年革命志士在艰苦条件下依然愿意抛头颅洒热血、用青春年华换取子子孙孙无穷匮也的原因。因为第一，他不是站在个人立场上思考自己的成就（升职加薪、迎娶白富美），而是站在整个国家民族的立场上，去研究子子孙孙无穷匮也，以及我们如何屹立于世界民族之林。同样的，也因为由俭入奢易、由奢入俭难——当过人，真的很难再去当狗啊。
