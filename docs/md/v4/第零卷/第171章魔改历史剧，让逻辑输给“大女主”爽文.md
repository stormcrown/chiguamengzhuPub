# 第171章 魔改历史剧，让逻辑输给“大女主”爽文

最近看了个电视剧的切片，嗯，有点颠覆我的三观，所以给大家分享一下。新出了个剧，说是武则天女帝在朝堂上宣布：“我们要有一定的建设性，要有一定的开拓性。”那怎么开拓呢？她要派王子出去和亲。这确实是一个富有历史开创性的创举，但是呢，很明显，这个事在历史的过程中有点离谱。所以今天来分析一下，为什么派王子出门和亲这件事情很离谱。

我其实有点能理解今天的编剧、创作者们为什么要整这样的电视剧。因为现在不是强调“大女主”、女性做主、女性安全嘛？呵呵，因为喊出了这种口号，于是大量的女性观众非常执着于要看到女性当家作主的场面。在这种场面之下，当然就不能够放弃中国历史上唯一一个能够被大家铭记的女皇帝。既然不能忘记女皇帝，那女皇帝到底应该干什么呢？于是他们在这种剧目的创作中说：“开天辟地似的开创了女官系统。”其实在那之前就有女官系统，他们不在乎啊。然后又说：“开天辟地似的安排了男子嫁出去。”呵呵。

所以第一件问题就是：恐怕得让某一些人去理解一下，为什么婚姻这件事情会存在一个“嫁”和一个“娶”。要知道，婚姻嘛，两个人凑一堆过日子。按照今天这种小门小户的理解来说，你家出个男的，我家出个女的，然后凑一起付首付、买个房，你们俩就生活在一起了，就是一个小家庭了，这不是很好吗？结婚嘛。但是呢，古人不是这样想的。为什么古人会非常强调嫁与娶的关系？嫁，就是把人送出去；娶，就是把人接进来。接进来的这边叫添丁进口，嫁出去的那边呢——哎呀，我娇养的孩子呀，不在我家了呀，要哭。嫁为什么要这个样子？这是因为“婚姻者，合两姓之好也”，它就不是俩人的事，它是两个家族的事情。

为什么要谈到家族的问题？因为这是一个财产关系。古时候的任何一个家族，它实际上就是一个大的财产的、职业的体系，而每一个个体都是这个体系下面的一个细小的分支。你结婚不是你一个人结婚，而是我们选择合作伙伴。同时呢，任何一个小家庭，它都是一个最小的生产单元：如果你家是打工的、种田的，那就是最小的生产单元；如果你家是经商的，你就是最小的财务单元；如果你家里是从政的，还是顶流的政客，那不好意思，你就是一个最小的政治单元。所以，当你身上有这么多的责任，和你整个的家族、姓氏、荣耀都绑定在一起的时候，嫁与娶的这种关系就必须非常明确。因为一方获得了一个人，而另一方失去了一个人，那么到底是谁归入到了谁的系统当中去？谁做主决定另一个人在这个岗位上怎么工作、怎么生活？就变成了非常重要的事情。因此，不是说“唉，男女平等，反正结婚男的嫁出去、女的嫁出去都一样的”，真不一样啊。

就是因为这个原因，所以在古典时期，为什么大部分出嫁的都是女性出嫁呢？那为什么很少有男性出嫁呢？其实也有，就是入赘嘛。入赘这个行为就是娶一个男人进来的行为。那么为什么很少有男性出嫁？主要是没什么人肯娶。我们考虑一下，我刚才说了，娶是一个什么动作？是给家里面添丁进口的动作：我把一个外面的人加盟到我的家里面来。这个时候你要考虑什么？你就想你是一个小企业的小老板，我要招一个员工进来当我的合伙人，我要分他40%的股份——就这样的一个情况。请问你招的这个合伙人应该是一个什么样的人？第一，跟你得匹配得上，那不能是拽着你的裤脚、天天给你扯后腿的人，不行，要有一定的匹配度。然后第二，你不能够让他功高震主，你一定会这样想的吧？你一看，哎哟，这个人能干到什么程度？我一个注资3000万的小公司，我把他的合伙人招进来了之后，一年之后他就会变成持股300亿的那个大咖，你敢把他招进来？不，不敢呐。

所以为什么在古典时期基本上都是男娶女嫁？因为古典时期，体力确实是非常非常重要的核心资产之一。体力、野心，他背后可能牵扯到的家族的势力，都是他身上的本源力量。于是当我去迎娶一个角色进来，结果这个角色的背后拥有的力量可以吓死我，那我打死也不娶她。所以为什么在男婚女嫁的这件事情上面，会有这样的专门的一种倾向？就是，唉，女子呢，有可能高嫁，男子大概率会低娶，就是这样的一个缘故。因为要不然的话，你就会特别像一个赘婿，你整个的经济生态都会颠倒过来。而且呢，不光是你自己感觉到委屈，对方还感觉到心惊胆战，时刻就怕你颠覆了我。这个时候你们没有办法达成一个非常良好的合作关系。

所以你看像和亲这种特殊的政治联姻，你说“哎呀，派男的去、派女的去都行”，那为啥大部分都是派女的？因为人家不敢要男的。一个男性的继承人代表了什么？如果你家里面的其他的兄弟都没了，你就是那个铁杆的唯一的继承人了。好嘛，我把对家的皇帝放我自己这儿来了。他个人可能是一个手无缚鸡之力的弱鸡，但是它的背后代表了什么？他有继承权，他有正统性，就意味着他自己的家族、他自己的势力，那边有很多的能人会想借这杆旗子。于是就有无数的人因为这个理由就会冲过来追随他。哪怕他自己再怎么肩不能挑、手不能提，他也拥有了一个极为强大的力量。我家的赘婿的力量比我这个老岳丈还要牛，分分钟都可以把我弄死的那种，我还敢招这样的女婿呢？我有什么样的女儿能够镇得住她呀？

所以为什么在这种联姻当中，大家都不愿意招赘婿？一旦招赘婿的时候，一定要看那个赘婿，他是不是政治资本特别的差，是不是个人的修养特别的差。很多这种有赘婿倾向的质子，古时候也有质子嘛，男子送到别人家里面去，然后安排什么婚娶，这种情况也有。那么这一种被招揽进去、目测有点像是赘婿的这种质子，都会去挑选那些政治资本不是很良好的角色，就是因为这样的一个原因，不然的话容易出问题。

当我们不能够随意地招揽赘婿的时候，你说我派一个王子出去和亲，对面的那边都会怀疑你的动机到底是什么。这个中间可能捅出来的篓子那可是太大了。所以它不是一个什么性别平等这种简单的问题，而是当一个人身上他天然地拥有某一种正义的、正统的继承权的时候，你把他投入到敌营之中，他并不是一个弃子，而是很有可能瞬间在所有人的心目当中变成一个完美受害人。请注意一件事情：在政治领域当中，完美受害人是很可怕的。因为完美受害人这个身份本身，就是政客身上不可或缺的一项巨大的政治光环和政治财富。这个事放之古今中外，全部都是一样的道理。

一个有政治身份、有政治继承权的人，一个流亡的王子，然后被发现了：他没有犯任何的错，他拥有所有的正统的权利，这个人身上道德并无瑕疵，并且他受到了他人的迫害，完全地离开了政治的中心，开始流亡外地。你知道这个时候会有多少人想要在他身上押宝吗？他的名声是这样的完美，柔弱意味着它是一个非常完美的棋子，就会招揽无数的人想要在他身上押宝。因为只要你押中了这个玩意儿，叫做“从龙之功”。从龙之功这种本事啊，是完全值得你一把梭哈、把我自己的九族都搭进去的。我们的历史当中有无数做这种押宝的人，而且通常情况下他们都是自视甚高，觉得自己的本事可以炸了天，在现有的朝堂体系之下配不上我，我一定要像窜天猴一样直接奔上天际。这种人就会去博这样的从龙之功。

所以在真正的这种政治场合下面，是真的没有人会做这样一件事情——就是制造完美受害者。但是我们回看一下，像这个电视剧里面，如果说武则天女帝真的就派了一个有继承权的王子送出去给人家和亲、给人家当赘婿去了，且不论对面的那个王敢不敢收留他，就算敢吧，这个时候这个王子的身上就变成了一个完美受害者。那他是完美受害者其实都还在其次，因为我们历史当中北方的人群非常有野心的时候，也干过这样类似的事情——就是把有继承权的，甚至把皇帝直接一撸就拉走了，当人质去了。当人质这件事情嘛，我掌握了你的核心继承权的时候啊，我就可以震慑你，你不管要做什么事情都要提点小心。所以你看靖康之变的时候，直接把徽钦二帝给掳走了是吧？土木堡的时候直接把英宗就给掳走了。问题是干什么？他要拥有这样的话事权，挟天子以令诸侯嘛。天子在我手上的时候，我这边干什么事都好干。所以是有这种情形的。

但是当你发生这种情形的时候，请问中原王朝是怎么解决问题的？他立刻就立一个新的皇帝，表示“你那个不重要了，你那个已经是先帝了，我不听你的，我不受你的要挟”。也就是说，能够对抗正统继承权的，只有另外的一个正统继承权。

于是我们就谈到了这部电视剧。因为她要拍一个大女主的电视剧，她就选了一个我们历史上最典型的大女主——武则天。但是呢，武则天是有正统继承权的人吗？他能够对抗一个有正统继承权的完美受害者的王子吗？不能，因为武则天身上最大的短板就是他得位不正。当然了，很明显的，现在的这些编剧试图在洗白，他想要证明说武则天没有得位不正的问题。但是很遗憾的是，他确确实实得位不正。他获得皇位的方式是通过朝堂制衡的办法，让大家去接受太后临朝的这种客观的事实，然后黄袍加身。他并不是一个真正意义上的开疆拓土的、开天辟地式的设立王朝式的皇帝。这个王朝不是他打下来的，而是通过政治博弈，通过婚姻的方式去当别人的小老婆、当别人的儿媳妇、然后当别人的正妻，最后呢，当了别人的妈，然后再去窃权，获得了这样的身份。那这种身份他严格来讲，大家称他为皇帝，本质依然把她视为一个太后。

当你没有正统权，而对方的流亡的王子身上有正统权的时候，这个棋子真好用啊，谁不用谁傻子。这种时候他可以怎么样用？要知道，不管那个王子在长安城内是多么的没有用，多么的受人压制，每天都好像屠刀架脖颈子上似的，卑微得不得了，那也没用。只要他踏出了长安城，他的身边就会跟随着一大票仕途从容的能臣良将，这些押宝的人就会跟着他一起走。而当他真的走到了草原上面去，见到了那个要把他娶进门的岳父的时候啊，岳父的手上有兵马，女婿的身上有正统权，请问这个时候的最优解是什么？我欺负欺负他，让他为我的女儿洗脚？那他一定是傻透了。这个时候双方的最优解就是：我出兵马，你出名义，然后咱们携手杀进长安去，我获得你麾下的某些权益，而你获得那个王位。这是最简单、最直接的处事方法。只要是一个政治家、是一个政客、是一个有野心的王者，就一定会做出这种选项。

所以你说怎么办啊？我也不知道怎么办。我理解不了他的这种非凡的创举啊。甚至在某些层面上，我觉得呢，我们可以去考虑一下武则天女帝的历史评价。我曾不止一次地说过，他并不是一个非常好的皇帝，但是呢，也不是一个非常坏的皇帝。因为她是一个女性君主，所以大家对于她的评价呢，通常会比较宽容，给他提一下分数。可是呢，无论在历史上面对他是如何进行褒贬，或如何进行贬斥的，但是他从来没有背过这么沉重的一口黑锅。他有可能是一个好色之徒，有可能是一个弄权之辈，也有可能呢，有点糊涂，有点草包，试图传位给武家公子。但是无论怎样，这些都还是在政客的职权范围之内所做的事情。而这种把继承人送到对方的手里面去，直接动摇自己王位的这样一种神奇的操作，实在是很明显地违背正常逻辑的操作方法。这已经不是历史当中是否发生过这件事情的问题，而是如果你让一个有为之君去做这样一个完全不合情理、完全没有政治的下限、完全没有智慧的行为，它产生的后果是，让很多的观众也变得单线条起来了。

这是这几年很多的观众身上都开始陆陆续续爆发出来的问题：就是大量这些违背了现实主义、违背了逻辑的文学创作，单纯地去讨好观众，让她觉得心里好爽——“哎呀，我是大女主，我也可以怎么样”。这种思路本身就会导致它缺乏逻辑性，缺乏客观性，对现实世界没有感应，也不知道别人会怎么样想，也不知道这些事件当中会产生什么样的利害冲突，最后就变成“只要我爽就可以了”。如果说每一个人在这些文创当中得出来的结果就是“我开心”，那它跟嗑药也没有什么两样，只会让大家变成糊涂蛋。

所以什么样的东西能够创作？如果说你的这个创作当中讲的是，武则天女帝在登基的过程当中，“哎呀，我用了什么样的阴谋诡计，怎样的权力制衡，甚至有威逼、有色诱”，我觉得都是合情合理的。无论他中间怎么样的机灵巧变，但是很明显，它有可能会发生，它是合乎逻辑的，它是合乎根本利益诉求的。可是如果说你去编造的这个历史故事，是说“我是女的，所以我让女人当宰相，棒棒的；我是女的，所以我让男人嫁出去，棒棒的；我是女的，所以我让男人到厨房里面去吃饭，棒棒的”，最后得出来的结果就仅仅只是“哎呀，我好开心”。除了这个之外，你什么都想象不到。因为一个严重违背逻辑的文创产品，实际上会导致人的思想扁平化。当你的思想扁平化了，你在世间任何一个地方去处事的时候，都会扁平化。我们这几年互联网上的很多的讨论是不是也有这个味儿？就因为这种无孔不入的扁平化，“我是女的啊，所以我50了，你也应该追着来舔我”，这种东西在互联网上就开始出现了。你说你这样子心里想行不行？行，很开心，但是它不合逻辑，也不符合现实。文艺的创作品也是要考虑现实世界的。所以你要说，你说武则天忒好色了，天天就讲他的艳情史，我觉得没有问题。但是你要说武则天让女人当家做主，武则天让男人滚出去嫁人，我觉得你最好别这样颠倒黑白。
