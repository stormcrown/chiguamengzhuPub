# 第64章 闭关锁国图什么

自从我们跟美国人之间的竞争进入白热化阶段之后，中国人消费国内产品的意愿开始越来越强。这本来是一个国家、一个民族进行自我保护的正常方式，但有不少媒体和公知突然受不了了，直接跳起来训斥大家：“怎么着？你们又要闭关锁国吗？还没有落后够吗？还没有困苦够吗？”

好，这里就涉及到一个概念——闭关锁国。闭关锁国是什么？闭关锁国的影响是什么？我们的衰落是不是因为闭关锁国？一般大家直觉所认知的闭关锁国是所谓的明清时期，它连带两个时代。那么这两个时代我们有没有闭关锁国呢？我们先看明代。明代时，我们其实一直都有海外贸易，包括郑和下西洋也是进行海外贸易。但为什么又说闭关锁国呢？因为我们曾经在短暂的数十年时间里，有了一项政策——海禁政策，禁止民间的私营外贸。这个私营外贸行为涉及很多层面：第一，如果你是零散的，出行危险系数很高，而且当时海上的强盗数量相当可观，出事的比例太大；第二，国家本来就收不上税，一看民间赚得盆满钵满，结果还是收不上来税。所以曾经有一段时间，我们在明代践行过这种锁国政策，就是不让民间的私营势力下海经商。但在此期间，我们是否真的闭关锁国了？没有。因为没有民间私营贸易，实际上是收归国有，当时有大规模的国有海外贸易。

至于说明代从中期开始、中后期一直都有的所谓海盗问题，比如当时所谓的沿海有大量倭寇犯边，其实里面不全部都是海盗，还包含有一部分人，其实就是那些外贸商人，试图用海盗的方式，将本国的货物据为己有，造成国家的折损，最后装进自己的荷包。这种行为就是所谓的监守自盗，在那个时候变得很多。所以明代有没有闭关锁国？有过短短一段时间禁止民营势力下海经商，但从来没有真正的闭关锁国。因为第一，有官办的贸易；第二，我们会注意到，很多人一边说明朝闭关锁国，一边说明朝武器落后，要找洋人买武器。你说你天天找洋人买武器，然后又闭关锁国，这锁的是哪门子的国？他们自己都会打自己的脸。所以明代并没有真正的闭关锁国，我们跟海外之间的联系非常紧密。甚至在明代有大量传教士，数以万计的传教士来到中国传教。虽然他们好像没有什么传教成果，但有很强的翻译成果——他们翻译了很多从西方来的、在西方也没有的伟大科学著作，给我们教授了地图学、几何学、数学、代数等等，还教我们如何种桑、养蚕、种植水稻、育种、看天文地理。当时的传教士充满了无私奉献的精神，把欧洲都没有的这些知识全部带给了中国的普通老百姓。我们不能一边说那时候西方给我们传递了先进的科学知识，一边又说那个时期闭关锁国、封闭自守，这两件事是矛盾的，你总要选一边。如果那时候我们闭关锁国，那么明代的那些科学著作就应该是明代人自己写的；而如果有大量洋人来华传播科学技术，我们承袭了别人很多知识，那么我们就必然是边境开放的。

好，当我们进入到清代的时候就更有意思了。因为清代的闭关锁国非常明确，别说是民间，官方上面也没有明确记载的海外贸易。可是这时候我们会注意到一件事：虽然普通人都没有感受到海外贸易的好处，但皇家从来都不缺海外贸易的好处——皇室的仓库里有从西方来的各种所谓的奇技淫巧的装备，还有先进的科学书籍，从来都不少。这些东西从哪来的？反正也不知道。哎，一方面闭关锁国，一方面又获得大量西方资源，这也算是一件奇葩的事。总之后来我们说，我们衰落就是因为闭关锁国，反正他就是找到了这么一个理由，说“闭关锁国嘛”。好的。那么清朝到后期，我们闭关锁国了吗？事实上，当时清朝因为有大量战败赔款要缴纳，国库里没有这些银两，清朝后期已经把自己的海关——收关税的位置——都交给了洋人来管理。也就是说，别说闭关，连关都已经送给别人了，你上哪里去闭呢？因此你说清代闭关？哎，没有，连关都不存在了，关已经在洋人手上。那么后来我们又闭关锁国了吗？哎，你说列强入侵、大家围剿，全世界都知道我们是一块肥肉，人人都想到这里来分一杯羹。德国和日本之间打仗，都能打到我们的土地上，来决定我们土地的归属。这种情况下，你说这个国家太落后了，因为他闭关锁国？你要不要想一下，人家都在我们土地上打仗了，我们是怎么闭的？关怎么锁的？国是不是闭也没闭住，锁也没锁住？如果我们真的闭关锁国锁住了，还有他们什么事儿啊？

所以闭关锁国是什么意思呢？如果我们单纯从字面含义考虑，闭关锁国就是自己在家里面关门。对于中国这样一个土地非常广阔、资源非常丰富、人口数量足够多、生产能力足够强的国家，闭关锁国无非就是完成一个彻底的内循环，不与外界交互而已。在这个过程中，能够产生的损耗是非常有限的。至于技术进步什么的，外面真的一直都比我们进步吗？这涉及科学史造假的问题，是另一个领域了。但很显然，保护自己的国土不受侵犯，关闭自己的海关，不让别人倾销，这是天经地义的事。包括现在的美国、欧洲都在干这样的事啊——对我们输出的产品收取高额关税，这不是壁垒吗？他们对我们很多先进产品采取拒绝入境的方式，请问这不算壁垒吗？拒绝在很多科技领域里交流，请问这不算壁垒吗？所以谁在闭关锁国呢？闭关锁国是不是对的呢？为什么欧洲人做这个事叫保护自己的经济利益，而我们做相关的事就叫闭关锁国、必然落后呢？什么标准都行，就是不能双标。所以我们就去看一下，哪怕是在最受批评的、我们不应该闭关锁国的那个时代，全世界被批评为闭关锁国的其实也就是我们而已。而其他很多国家，尤其是那些小国家，他们会闭关锁国吗？不会的，他们也没有能力闭关锁国。这些国家什么下场呢？敞开国门，整个国家都沦为殖民地。我们是半殖民地，我们已经哭了200年，而他们完完全全是殖民地，到现在连哭的资格都没有。那个就是你又弱、你又开放、你又无能、你又敞开国门，这种情况下那是必死无疑的呀。所以闭关锁国真的就是错的吗？什么情况下需要关上门？防盗的时候就应该关上门呀。你明知道外面有贼子，你还要把大门敞开，高唱“我家大门常打开”，欢迎八方来客吗？那个时候的八方来客是谁？有哪一个八方来客是怀着好心的吗？都没有。

所以第一，当时并没有真正的闭关锁国；第二，即使在那个时候，闭关锁国也不是一个错误的选项，因为群狼环伺。所以在今天，我们看到很多公知又急了：“哎，你看你又要闭关锁国了，不是，吓死人了，他在替谁叫屈呢？”如果我们真的是闭关锁国，搞起内循环，排斥外来商品的话，谁会比较吃亏呢？他在为谁着急？这确确实实是我们需要掂量的问题了。
