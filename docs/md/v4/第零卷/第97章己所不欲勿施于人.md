# 第97章 己所不欲 勿施于人

己所不欲，勿施于人——这当然是一个君子之风的正规道德要求。然而，我今天为什么提到它？是因为前几天遇到一个小伙伴，他在那里说：“你这个人呀，老是搞内外分别，这不好。你要知道，我们国家的主导就是己所不欲，勿施于人。所以呢，你既然不希望我们的国民在海外被人家驱逐，你就应该欢迎别人在我们家居住呀。”嗯，他感觉也知道我在说内外有别哈。好的，我们今天就来聊一聊这句话的应用场景。

首先，道德这个东西是在哪用的？很多人，尤其是像中国人这种高道德感的族群，喜欢把道德运用在方方面面，到哪里都讲道德。所以中国人在世界上的口碑有几种：一种就是“中国这个国家太可怕了”，大家都知道的，这叫“中国威胁论”；而另外一方面，就是中国人呀，是小肥羊。同时，其实还有第三种对中国人的评价，主要针对的是华人，就是海外华人——他们在所有的移民国家里面，都堪称当地社会遵守秩序的翘楚，都是模范公民。所以你看，中国人哪怕是传了几代，跑到了海外去生活，在任何地方，他表现出来的形象永远都是高道德感的、己所不欲勿施于人的。

然而，其实我们有的时候错判了一件事情：道德这个东西，它到底在哪里生效，在哪里会失效？道德伦理其实都需要一个强权的保护。你要知道，如果说你跑到丛林里面去跟狮子老讲道德、讲伦理，说我们在干架之前要先互相鞠一躬，不然的话，你这个家伙不讲武德。然后呢，东北金渐层那一个巴掌呼过来之后，我估计你的天灵盖儿是一定保不住的。这就是道德、伦理、礼貌、秩序能够用在哪儿呢？能够用在我们自己的文明圈以内。它一旦离开了文明圈就会失效。因为所有的道德，包括所有的法律，都需要彼此之间有共识——我们遵守着同一套价值观、同一套伦理观、同一套法律规范。这种情况下，我们才能够互相礼让，遵守道德。而一旦跨出了这个圈，我们该干的是啥呢？哎，老祖宗教过你的，叫做“入乡随俗”。

今天很多人把“入乡随俗”讲得有点太过温情脉脉了，看上去就好像是别人喜欢早上起来跳舞，到那儿我也得早上起来跳舞；别人喜欢晚上的时候走一杯，我到那儿了，我也得晚上走一杯。这种朴素的、近乎于游客式的、单单纯的小可爱一样的小绵羊一样的入乡随俗，和我们老祖宗所讲的入乡随俗完全不同啊。真正意义上的入乡随俗，是当你走出国门，发现对方的社会秩序其实是一个丛林法则，能占一点便宜就要占一点便宜，能抢到一点好处就要抢一点好处的时候，你也得让自己变成一个能抢就抢、能夺就夺的人，不然活不下去呀。

所以我们在这个事情上面牵扯一下：为什么古典时期有很多的公主、贵女们不愿意和亲？按照今天很多人的想象，哎呀，嫁到贵族不高级的呀？人家是不是拿银托盘给我拖WIFI密码咯？然而实际上，古时候的贵人死命地拒绝和亲，并不是因为生活条件有多差，而是因为她跨出了文明圈。对方的游牧民族里面的社会秩序和道德感、伦理观是跟我们截然不同的。她从我们这里进到他那里，就被迫要遵守对方的习俗，而这种习俗对于我们的道德观来说，是一种幻灭式的，是一种灭顶之灾。因此她不愿意去。所有去的那些姑娘们，都怀揣着一种伟大的政治理想，是那种“风萧萧兮易水寒”的姿态，冲到敌营里面去做内奸的，或者叫外交工作。

总之呢，实际上所有的道德也好，伦理也好，我们都是在文明内部才能够应用，并不能够跨圈使用。于是我们问题就来了：开篇的那位小伙伴说，我们今天的国家非常主张的一件事情，叫做“己所不欲，勿施于人”，我们在国际社会上从来都是这样说的。所以呢，你要懂得理解什么叫做外交语言。

我们来模拟几个应用的场景哈。什么情况下我们会用到“己所不欲，勿施于人”？比如说，时间倒推回去，我们放到100年前，当时呢，列强围拢了我们，整个中国在国际社会上面的地位，那简直就是不可描述。在那种时候，谁都来欺负我们一下，谁都来这里薅一把羊毛。当时我们有没有政治家站出来说：“己所不欲，勿施于人，你不能这么欺负我，不然要有后果。”有后果吗？大家一看——勿施于人，有后果吗？没后果呀。你这个小弱鸡，我怕什么呢？所以当时我们的外交人员也会高呼“己所不欲，勿施于人”，这是什么姿态？这是一个讨饶的姿态，以一种君子之风，实际上别人听见的声音是“求求你放过我”。所以当时我们所提出的每一次“己所不欲，勿施于人”都没有任何的回应，别人想怎么瓜分怎么瓜分，想怎么占便宜怎么占便宜。有用吗？没用。

今天我们在国际社会上依然会说“己所不欲，勿施于人”，这是干什么？这是在说“你不要跳脚啊，你不想遭遇什么的话，你就不要跳脚”。然后你就会看到对方真的就开始乖了。这是干什么？今天我们变成了国际社会上的庞然大物，于是我们依然君子之风、彬彬有礼地说“己所不欲，勿施于人”，而对方所听到的是威胁。是的，我们的国家一两百年都在对国际社会说“己所不欲，勿施于人”，可是它的含义是完全不一样的。为什么会不一样？因为听众的地位不同。

所以今天我们是不是因为“己所不欲，勿施于人”，所以我们应该对全世界敞开国门？开什么玩笑，这个世界上就没有这种规矩。国家所说的“己所不欲，勿施于人”，和你的那种人口大交换，它就不是一档子事情。我们强大起来是为了让我们的人民过上好日子，不是为了招揽八方来客在我们这里过好日子，这是一个根本性的差别。

所以学一学啊，如何用文质彬彬、斯斯文文、十分有礼的君子气概说出“勿谓言之不预也”。
