# 第135章 代孕没有新鲜事

代孕这件事其实被曝光不是一天两天了。  
这一次突然上了那么一两天热搜，本质上无非是某一个窝点被曝光后，陷入了一次舆论的小高潮。  
其实这件事在暗地里已经存在了相当长的时间。即使不说国内，在国外很多地方，代孕本来就是合法的。之前也有很多人跑出去干代孕这个活儿。  

那么，代孕这个东西到底是什么？今天的互联网上为什么会产生讨论？为什么还是有很多人觉得代孕是一门正当的生意？甚至我会看到一些女性站出来说：“就应该支持代孕啊，不然我这种缺乏生育能力的人可怎么办呢？”  
这些问题在历史上都有答案。中国的历史从来不缺乏记载。代孕这件事，我们过去不叫这个名字，但这种事儿一直都有。无非是技术的进步，导致它可以和原本的人群之间产生区隔罢了。而在过去，这种行为叫做什么？叫做“典妻”，叫做“质妻”，叫做“买卖人口”，叫做“借腹生子”。这种事儿在过去的几千年里一直都有，但从来都处于法律的模糊地带，从来都处于社会价值评判的负面。  

而今天，我觉得最可悲的不是这种事情的存在，而是这件事简直快要被洗白了。代孕这件事，竟然在今天的社会主义中国，在某些人的嘴中，快要成了造福人民的一种东西。它到底是什么时候开始的？应该说，自打有了婚姻，里面就存在代孕这种可能性。因为婚姻本来就包含了孕育共同继承人这件事。而如果你的配偶没有孕育的能力，就有可能产生需要纳一个妾、找一个肚子来生另一个孩子、然后带回来抚养的方式。这本质上也是一种代孕。  

但是之前，它好歹是纳入到一个家庭的生活系统里面来——虽然地位低一级，但好歹也算是一家人了，好歹算是养了终身了。再怎么说，哪怕是庶母，也算是母的一种了，没有完全剥离孩子和母亲的关系。但后来就开始出现那种母子分离式的代孕。这是什么时候出现的？它第一次大规模出现是在南北朝时期。  

很好，我们来回顾一下南北朝是一个什么样的时期。那个时期看上去有了建制，有了国家，不太乱了，还有了规则，所以大家有婚姻、有登记，家里好歹还可以单门立户了。但同时，经济、政治都一团混乱，非常崩溃。每一个政权都是走马观花一样，今天管了明天崩，明天管了后天崩。而且很多政权从立身那一刻起就立身不正。就是在这种非常混乱的时期，出现了大规模的典妻现象。  

什么是典妻？就是贫苦人家把自己家的妻子，用典当质押的方式，送到其他人家去帮别人生孩子，生完了再把妻子领回家来。通常情况下，不会只押给一家。收回家里养了几天身体后，又要质押到下一家去。有没有发现这种情形和今天的代孕女子没有什么本质区别？今天给东家生孩子，明天给西家生孩子，生完之后母子分离，把孩子交给别人家去养。从南北朝时期就已经开始出现典妻现象了。  

而到了唐代，愈演愈烈。这是因为唐代我们很清楚它的状态：国家看上去很兴盛，可是世家门阀对底层的压榨极其严峻。这就是为什么黄巢后来需要“天街踏尽公卿骨”。这种严峻的两极分化式社会管理，带来了很多典妻现象。而到了后世，什么时候出现了典妻的集大成呢？就是在宋代的时候。宋代也是一个经济繁荣、自由体制非常繁荣的时期。理论上讲，那个时代甚至提高了妾室的家庭地位。但是提高妾室的家庭地位，反而导致了中人之家不愿意纳妾了。纳妾需要养一个妾室一辈子，他们想要降低成本怎么办？就叫做“临时纳妾”。  

曾经大家在互联网上看到过这样的小说，叫做“典妾”。其实“典妻”和“典妾”是一对。对于穷人来说，付出妻子这个代价叫做“典妻”；对于有一点薄产的人来说，去招募一个临时的妾室，这就叫做“典妾”。本质上是一件事情。总之在宋代的时候，虽然伦理上或者说在法理上认为这件事情非法，但事实上在民间它大规模地存在。在明代的时候，它倒是受到了比较多的打击。可是进入清朝之后，由于经济的原因，典妻的现象更加多了。到了民国的时候，卖儿卖女卖老婆，简直就是一件天经地义、到处都是的事儿。  

所以我们就会看到，代孕这件事不是今天的人才想出的花活儿，也不是技术进步带来的造福民生。想要借别人的肚子生自家的孩子，这件事古已有之。只要有人，只要有贪婪，它就必然存在。  

所以今天，我们在一个社会主义的国家里去考虑说，用妇女的人身自由来牟利，然后你说这是一件自由的事情。自由是什么？向上的自由是自由——我可以自由地学习，自由地工作，自由地奋斗，自由地上进，这叫做自由。那么向下的自由是自由吗？你说她想要赚取自己的收入，这件事天经地义，她是可以自己选择的。看上去对——可以自己选择。  

要知道，我们民国的时候取消了买卖人口，然后娼妓业里面就不再能出现说“把某人卖进了妓馆”这种事情。所有娼妓的许可证上面的登记都变成了“自愿沦落风尘”“因为贪图享乐而自愿当妓女”。这种自愿真的是自愿的吗？向下的自由从来都不是自由。而社会主义中国要解决的就是这种非自由、假装的自由。我们的解放妇女，就是让妇女能够用劳动来养活自己，而不必用身体去养活自己。  

而今天，走到今天了，突然之间有一群人跳出来，说“天大的经济利益，我可以靠这个东西赚钱”。没得选的。只要说这个世界上允许代孕合法，那么月薪3000却被营销号忽悠着去买300块钱一支的口红、1000块钱一双的高跟鞋的女孩子，就会被卖卵和代孕的广告勾引过去。这个玩意儿不叫做自由。  

然后还有人说：“你这个样子就会让那些不能够生育的人没有办法获得自己的孩子。”我们禁止领养了吗？我们禁止领养了吗？你为什么不去领养？不能生育就去领养孩子。连古人都知道这个道理——这叫做收养，这叫做过继。自古以来就有的东西，怎么到你这里就突然说“没有生育能力还非得要去生一个自己的孩子”？谁教给你的道理？生老病死，命运无常。一个人有可能出生的时候就不健全，他依然要好好地活下去。没有生育能力的人，也依然要好好地活下去。凭什么就要借由他人的身体？  

要知道，只要你开发出一个合法的代孕或者变相合法的代孕，受害的就永远是那些低阶层的人。当然了，你可以幻想自己是高阶。那不好意思，月薪3000可能是月薪8000的低阶，月薪8000就是月薪2万的低阶，月薪2万就是那些“我不爱钞票，钞票是什么”的低阶，而那些不知道钞票是什么的人，又会是“朕独有这个天下”的低阶。谁还不是低阶呢？这个世界上你想要人人平等——平等是什么？平等是人格的平等。但是当你有了出卖人身自由的权利的时候，你就会发现这个人格的平等保护不了你。  

所以我们为什么要禁止娼妓？为什么要禁止人口的买卖（哪怕他是自愿的）？为什么要禁止代孕？为什么要禁止商业化的器官买卖？因为只要这些东西存在，你以为你有薪水，你以为你是社会的中层，你以为你是一个小资、小布尔乔亚，每天下午可以喝咖啡，你就是上等人就可以拿别人的肚皮了吗？不好意思，你上面还有一个三流艺人，仗着自己那两个糟钱儿想要去找别人借肚皮呢。而在他的上头，还有另外的高门贵女想要找他借肚皮呢。大鱼吃小鱼，小鱼吃虾米。而社会主义要解决的就是不让这种“吃”的行为发生。结果很有可能被吃的人居然感到了不满：“你凭什么不让我吃虾米？”  

我们的国家在保护谁呀？我们在保护每一个人。结果被保护的人站出来说“不要，放我出去，外面有那个丛林竞争的世界”。你去看一看，你手上那三两个钱儿到底够干什么的？到底够不够保护你的人身自由？到底能不能保护自己体体面面地赚钱，而不去当别人试药的小白鼠？无数的“润人”在前面给你打了样儿了，还想去体验？悉听尊便。但是在国内，我永远坚持反对任何对于人身的买卖关系。
