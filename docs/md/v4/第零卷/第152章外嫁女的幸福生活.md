# 第152章 外嫁女的幸福生活

人红是非多啊，我这个不红的是非都多，那非常红的是非就特别多。这两天吃了一个瓜——有一个做旅游记录类栏目的超级大V，这几天陷入了一场口水战。其实也就是他自己跑出去旅了个行，常规地跑到了欧洲的立陶宛（大家知道的波罗的海小国）。去那旅游之后呢，发现当地有一些情色产业，于是他在当地上面进行采访的时候，就比较侧重于采访“你们这有没有失足妇女”这种问题，做了一档片子。最后起题目的时候呢，也略有一点博眼球，就是“这地儿莫不是欧洲的淫窟啊”——大概就这么个事。结果吧，本来是平平无奇，大家一起去吃外国西瓜的这种小破事情，但是呢，居然引发了一场口水战。为啥呢？就是因为有人跳出来不满了。是一个互联网大V，也是两口子，跑出来说：“哎呀，你这个侮辱了立陶宛，你造谣事实啊！然后你侮辱了女性，你这个人不负责任，是个下头男！”呵呵呵呵呵，反正看得我很欢乐。

所以在这个地儿呢，还得介绍一下后边的这个人啊——为什么他要出来咬前面的旅游大V呢？因为她自个儿就住在立陶宛。这个人的身份是一个中国女性，然后嫁到了立陶宛去。并且呢，我也不知道她日子过得好不好，反正她协同她的立陶宛的丈夫一起做了一个自媒体号，然后在中国的自媒体上面赚钱。换句话说，她现在的工作形式就是：虽然是个外嫁女，但是在吃娘家饭。然后吃着娘家饭呢，又开始骂娘家人。这个行为就引发了很多互联网网友的围观啊——大家其实现在还是在吃瓜看热闹，就是：“你嫁出去了，还吃娘家饭，还一边吃饭一边砸锅是吧？可带劲了是吧？”然后呢，他其实就让大家衍生出来一些想法，说这个外嫁女的群体怎么老是这样——因为这也不是第一起了。好多的外嫁女，她在互联网上面就展示什么“给公公婆婆一全家老小做饭”，然后人家说：“你在国内都不做，你跑到人家去做这些饭，然后说你自己生活好幸福，你是不是有点不对劲？”她就开骂中国啊。这种性质的事情特别的多。于是呢，海量的个案最后就会积累成大家的一个群体潜意识，形成一个所谓的思想钢印，或者叫做刻板印象——其实就是成见。你天天这样演，那自然而然成见就出现了嘛，这叫快捷方式，不然我还专门认识你一下吗？所以这种成见既然已经出现了，就会有人在上边说：“唉，她出去的时候就是为了显示她好；那么如果她过得不好，还要吃娘家饭是不是？那她一定要说娘家不对嘛？”哎，我们是在揣测她的心态。但其实呢，嗯……有关于这些事情，我就觉得吧，我们把这个帽子直接盖到外嫁女一群人身上其实是不公的。这不是这一群人的问题。虽然她表现得格外的跳脱，个案的数量呢也特别多、特别醒目，但实际上这种思维的方式或者这种行为的方式，并不是因为结婚，也不是因为跟外国人结婚，所以跟外嫁本身是没有关系的。她跟我们选择婚姻的目的更加有关系。

那关系在哪儿啊？我们可能就要稍微绕一点点去聊一下有关于婚姻建立的问题。我们人类早期在原始社会的时候啊，最开始的所谓的婚姻形态跟我们今天认知到的一切的婚姻都不一样。那个时候呢，有一个阶段可以叫做群婚制——就是我也不知道昨天的丈夫是哪位，今天呢再选一个，明儿呢明儿再说。这种环境就叫做“不知有父”，母系氏族的时候就是这个样子的。所以它是一种群婚的关系，大家没有固定的配偶，不晓得对方是哪个，随机配偶的模式。这个阶段呢就比较乱，什么家庭啊、伦理啊，你论不着，只能认得你的娘。所以这个阶段就算是母系氏族社会里面了。但是等到后来我们慢慢地构建了家庭关系之后，逐渐地就出现了我们今天的人所熟悉的这种婚姻的模式——就是知道谁是丈夫、知道谁是妻子，彼此之间呢有分工、有合作、有责任，然后呢有权益。到了这个阶段，它才算真正完成了婚姻制度的一个建设。我们现在所比较熟知的这一套婚姻逻辑，其实是建立在周礼的基础之上的，也就是它的雏形呢在周代的时候才完成了一个完整的建设——它是逐渐逐渐来的啊，到周代彻底构建起来了。这就是我们今天所了解的婚姻。

那这种婚姻它里面包含了什么内容？我们自己去看，包括也可以去看今天的婚姻法也是一样的——我们今天的婚姻保护的是什么？保护的是繁衍后代的权益，保护了继承的权益，保护了共有财产的权益，保护了财产分配的权益，以及夫妻之间相互抚养的义务。这一系列就是我们婚姻的基本的内核。古时候呢跟这个有差别，但是差别也不是很大，这就是婚姻的一个本质嘛。那么现代婚姻既然是这个模样的，这就叫规矩啊，这个玩意叫规矩。但是规矩是死的，人是活的啊，这里头就会出现很多的分叉。分叉就出在这了——虽然我们立了这一套婚姻的规则，但是你为什么而去订立婚姻是个人的选择，这不一样的。我为各种原因都可能会结婚，那结婚的理由、结婚的目标就会决定我们婚后的一些行为的方式了。

比如说比较典型的一种，也是比较传统、在全世界范围内都普遍存在的一种婚姻的方式，就是联姻或者是合作性的婚姻。这种结婚呢，通常建立在一种门当户对或者资源互补的前提条件下，大家在搭伙儿，再想办法在一起干活儿，一起构建一个家族、一个家庭，共同繁育后代，然后呢积累财富。有的是为了实现阶级的跃升，有的是为了保住自己的阶层不跌落。不管是哪一种目标，反正他们是搭伙的、配合的，彼此之间呢是有牵连关系的啊，捆绑得比较紧。这个是合作型的婚姻。古时候这样的情况很多，门当户对嘛。然后现代呢，其实也很多，尤其是在基层的劳动者这里，合作型的婚姻的数量是非常多的——开夫妻店呐，开夫妻的车呀，然后一起在一个工地里面打工啊，这些都是属于典型的合作型的婚姻。这是婚姻形态当中比较友善的一种，因为彼此的牵连很深，不太容易离散，互相之间会有比较深的帮扶关系。

然后还有一种呢，是受到全世界广泛赞誉的婚姻模式，它的目标就是喜爱——发自内心的热爱，看了之后眼睛里面bling bling就放光的那种。这种婚姻呢，是在全世界任何地方都会获得祝福的。有的时候哪怕说这两个人门不当户不对，但是因为这种炙热的爱情，也会受到所有人的祝福。如果说他们没有好的结局，我们就会把它写到书里面去，让它拥有一个幻想中的美好结局，例如梁祝。这就是典型的一种爱情、友情、共同的志向、共同的理想，通过感情而缔结的婚姻。这种婚姻到哪里都是备受赞美的啊，从古至今它都有，现代一样有。但是呢，它能够成为主流价值观并且被所有人认为它是婚姻的默认条件，这个历史并不太长啊。我们不考虑任何条件而单纯的以爱情为前提去缔结婚姻，其实是有要求的——就是每一个个体能够单独地养育自己。这个是建立在工业革命之后，人们呢开始逐渐经济自主化，开始呢都成为了平凡的打工人都自己养活自己的时候，自己为自己负责的时候，那么其他的选项就不太重要，于是呢感情就会被放到了非常核心的位置，很醒目。因此到这个阶段的时候，爱情至上就开始一点一点地出现了。今天呢，我们对于婚姻观，很多时候就是“没有爱情的婚姻那就是坟墓”啊，这已经成为默认的前提了，这是现代婚姻制度所构建出来的一个东西。

但是除了这些大家都很默认、很接受的一种婚姻目的之外，其实还有另外的婚姻目的，就是交易型的婚姻。这玩意儿从古至今都有，原始社会也有啊。啥叫交易？就是我用我的一些青春也好、美貌也好、生育的资源也好，去换取你手上的东西。这是一种买卖型的婚姻。这种买卖型的婚姻从原始社会就有了，很可能那些不知有父的女子，她就是为了一个露水而选择了这个男人作为今晚的丈夫。那么到了后来，我们构建了婚姻的这种缔结形式之后呢？唉，她为了对方的家财万贯，于是呢嫁到对方的家里面去，这种情况完全存在。所以从古至今，这种买卖的交易型的婚姻都有啊。它里面有可能是用青春来置换财富，有可能呢是用美貌来置换对方的贵族头衔，有可能呢是用我的生育资源来置换对方的这种阶级的岗位。这种交换一直都是存在的。但是为什么到今天大家似乎就觉得，诶，这个交易型的婚姻好像对我们的生活产生了比较大的影响？这是一个很重要的问题。就是交易一直都存在，为什么在过去似乎它影响不大？其实是因为古典时期，我们的婚姻制度是考虑到了交易婚姻的啊。古典时期的婚姻制度，它并不认为人们必须要因为爱情而结婚，它默认你有可能会产生交易，所以它去进行婚姻制度的约束的时候，是有很多限制的条件的。最典型的限制的条件就是离婚很难。它导致你整个的这个交易变成了一个一次性的——一个人一辈子你想要卖自己，卖一次可以，卖两次兴许也行，卖到第三次的时候就没有人接手了。这就是古时候对于整个的婚姻，它认为你有可能甚至很有可能会有交易买卖，所以呢它考虑的制约条件就是：我不能让你随意买卖。于是呢，极大地限制了这个婚姻当中的条款。例如婚姻当中的这些责任和义务，你必须要兑现啊——就是所谓的三从四德，我之前也讲过：女子应该负什么责，男子应该负什么责。这些责任如果你没有尽到的话，有的严重的国法会来管你啊；国法管不到的地方，家法、家族的律法会来管你；家法也不管的时候呢，隔壁左右邻里街坊的口水也能够管你。所以他是有一套严密的制约机制的，就会导致虽然你确实是买卖婚姻，你确实是在用你的美貌来置换对方的家产和贵族头衔、阶级跃升，但是你只能卖一回，而且卖完了之后你所有的婚姻当中的责任——例如生育、例如家务、例如三从四德——你必须要遵守。于是他就变成了一种一次性的并且条款明晰的买卖，他有格式条款。

然后现代之所以买卖婚姻会影响比较大，主要的原因是我们今天的所有的规则、社会制度的建立，都是建立在我们爱情至上、合作为底线的一种婚姻观下面的。他默认每个人的婚姻缔结要么是因为感情，要么是因为志向，要么是因为合作，他排除掉了买卖婚姻的这种存在性。所以在我们的条款当中，根本就没有制约条款。如果你是买卖呢，离婚是自由的，不生育是自由的，生别人家孩子是自由的，背叛是理论上不自由但实际上没有约束条款的。所以在这种情况下，虽然是买卖婚姻，但是你可以不兑现买卖之前谈的所有的条件，而且可以翻身，把这个合同一撕就可以卖下一家了，完全行动畅通无阻。这是最大的一个问题。也就是我们今天的秩序，它在事实上会导致那些偷鸡摸狗的、试图用买卖婚姻的方式赚取私利的人、破坏默认规则的人，他不用付出代价。当海量的人都在这种买卖的交易的婚姻当中获得了好处，但是却没有付出代价的时候，他就觉得这个玩意是他天经地义应该获得的利益啊，这是人正常的心态。那么今天是不是这种事情特别多？例如这几年天天在炒彩礼。我们几十年前根本就没有彩礼这个项目呀，为啥今天要炒呢？因为每一个获得彩礼的人，之后呢迅速的离婚，然后把彩礼套现之后呢又不用负责任，又很快就可以转入下一家，而且谁都不能批评他，因为他是自由的，他是正常的。那这种情况下，他不赚白不赚呢，这就变成了他的一个既得利益，那就谁都不能动他，这变成了他天经地义应该拥有的东西了。明明是在交易，但是他觉得这个买卖是合理、合法、合乎天伦、合乎道德的，谁不让他赚这个黑钱，谁就是错的。那这两天是不是又有新闻，那个胖东来又要求他的员工不能够有那个买卖婚姻是吧？说你要买卖婚姻你就不能拿福利，要福利你就不能拿买卖婚姻，看上去很公平公道，但是是不是在互联网上一片骂声？为啥呢？因为本来人家就默认的，不管我拿多少福利，不管我拿多少好处，反正我就是应该得的，我凭这本事拿来的钱，你凭什么不让我拿？是因为我们整个的社会现在对于婚姻是欠缺约束的。这其实会衍生出很多衍生的问题，但是这不是今天的主题，所以就先把它绕过去。

总之呢，就是我们在婚姻的建立目标上面存在这三种：合作型、爱情型以及交易型。那么交易型婚姻很明显是有巨大的问题的，尤其是在现代社会。那么就是今天我们讲的这个外嫁女的群体，她其实是可以归类到交易性的婚姻当中的。不是所有的外嫁女都是这个死德行，有一部分是因为正常的恋爱关系而嫁出去的。例如她在国外留学、在国外工作是吧，他就刚刚好跟那个外国男人建立了很好的感情，甚至是共同的工作，有着共同的理想、共同的梦，于是他们结婚了。这种方式它和正常的婚姻没有任何区别。今天能够在互联网上跳脱、能够刺激我们的神经、能够让我们作呕的这些外嫁女，她本质上面是交易性的婚姻。这个交易型的婚姻是不分国内国外的，国内也一样有啊。国内的这些交易型的、买卖型的婚姻，翻脸就不认人的。这些交易型的婚姻的这些人不也是这样的吗？是吧，他翻头把自己的那个获利都套现捞走了之后，是不是直接就开始开喷，说你们都欺压了我了啊，你们对不起我了，这就是我应该拿出来的东西啊，你凭什么让我还？还是这样干的吧。你把她的身份置换到外嫁女，操作方法没有什么区别。这不是因为她外嫁，而是因为她交易。

那请问外嫁当中是不是就交易特别多呢？那还真的是啊。我承认外嫁当中呢有一部分确实是由于感情的存在，但是外嫁的这个群体当中，尤其是在过去的几十年的过程当中，其实交易的比例太高了。哪怕说我们今天去看他的时候说，你看你不就是嫁了个外国街溜子嘛，什么档次啊，我都瞧不起你，但是他是交易的。为什么？因为他当初嫁出去的那一刻，她是因为崇拜对方是洋人，不管那个人有多穷、不管那个人有多丑、多老、多么猥琐、多么不堪，但是他是外国人，外国人这个身份在她的眼睛里面是熠熠生辉的。她本身是为了崇拜对方，而把自己的青春、美貌、声誉的资源拿来置换一个外国身份，他是在卖出自己的资源，换取对方的身份。所以他是一个典型的买卖婚姻呢。你都已经是买卖婚姻了，就是已经在做生意了，生意你这个时候就有问题，知道吧？因为生意是有赚有赔的。赚的时候咱啥也不说，早几十年的时候有好多的那个外嫁女啊，她回娘家的时候大包小裹的，林林总总就把国外的那些小东西都拿过来打赏，这种咱见过的——给你这个糖果，给你这个衣裳，给你这个收音机。当年他们就是这样做的。在这样做的过程当中，你看他是不是其实跟家里人好像很亲密，往来的频次很高，而且呢姿态也很不错，甚至有的时候彬彬有礼。但这是什么？这是他的买卖，赚了，至少说在那个阶段他赚了。但是问题并不出在这个投资上面，问题出在中国发展太快了。对，这是中国的错啊——中国发展太快了。她嫁出去的那一刻还是崇拜对方的洋人身份，崇拜对方的什么贵族头衔，哪怕是落魄贵族、死穷鬼也行啊，贵族头衔高级。但是架不住中国策马奔腾的就冲上去了，于是呢，无论是外国身份、护照、绿卡，还是那些什么所谓的贵族头衔、贵族姓氏，都变成了一坨shit。这意味着什么？投资失败，折了本儿了，亏透了，裤衩亏没了。关键是因为这个行为中间间隔了10年甚至15年的时间，这就是中国发展的时间啊，以至于青春不再，二次创业做不到了。那这个时候怎么办？任何一个做生意的人去做这种投资的时候，没有办法沉没，资本已经大到这样的程度了，跑也跑不了，套牢了，你怎么整呢？挽尊嘛，挽回损失嘛。聪明一点呢，就会去协同她的外籍的丈夫啊，然后赚中国的钱——因为中国发展了呀，这就是强行地挽回损失。然后还有一些呢，就是不断地抨击中国不好，这就是强行挽回尊严。有的呢，是一边赚钱一边说中国不好，这就是既想要挽回损失又想要挽回尊严。所以这就是买卖失败之后的一个产物。

当然，这些其实都是个人能力平平无奇，或者是沉没资本太过庞大，所以确实套牢了、跑不了的。还有一些呢，就是调转船头比较快，就是轻资上阵的，这种呢就有可能逃逸。那么这些人就相对灵巧、相对精明一些。他当年媚外啊，嫁出去了，投诚做了一个亏本买卖，发现亏了，发现这边有可能赚之后调转船头——我离了国，但是我爱国，而且因为我离了国，所以我更加爱国，我比你们所有的中国人都知道什么叫做爱国，来，我教你什么叫爱国。所以呢，就有一批人特意在这个时候开始交投名状，决定在对方的船可能会沉没的时候呢，调转船头又回到我们的娘家来啊。这也是一种选项。

所以本质的原因是什么？为什么他们的身上普遍地表现出了这种让我们感觉到尴尬的、感觉到不安的、真的感觉到你在干什么呢的荒唐行为？其实主要的原因是，他构建婚姻的初始的目的不单纯。即使在现在的互联网上，其实也是存在那些正常的婚恋的关系的，就是涉外的关系当中有正常的人。那这些正常的人的特点是什么？他不拿自己外嫁说事是吧？他做什么生意就做什么生意，而不会去把这个什么外国的身份、什么我在外国的生活、我在外国的这种丈夫等等的东西拿出来。因为这个东西不重要，他只是结了个婚而已，又不是卖掉了。那这件事就不重要。但是他如果是卖掉了，那这个事对他来说就最重要，那是他的核心资产。这才是最大的一个差别。

所以我们今天去批评外嫁女的时候呢，一定要注意的一点就是：不是所有的外嫁女都这样，而且呢她之所以这样的不堪，也并不是因为她是外嫁女，而是因为她出卖了自己。所以这里头真正的教训应该是——不要采用买卖婚姻的方式。任何时候用买卖，都会让我们陷入到一场生意当中，而生意的投资一定是有成有败的。如果你连日常的投资、小卖部的生意都做不好的话，为什么能够去幻想自己投入一生幸福去投资一个婚姻这样的大事业就会成功呢？而且我们知道，在投资领域、在做生意领域里面有一个非常非常重要的原则，叫做“做熟不做生”。同样是做买卖型的婚姻，你在国内的时候你知道规则，你知道如何应对，大家在同一个文化圈里面，甚至你更好伪装成是一个爱情的婚姻对吧？其实它的条件要比外嫁优越得多、容易得多，相当于你升级打怪的难度都要降低很多，而且你身边有亲戚朋友拉把一把，后退的余地也会比较大。但是当你用买卖的方式去结婚的时候，又选择了外嫁，你就是一把将所有的资产全部都梭哈投进去了，而且投进了一个完全陌生的领域，举目无亲，后退无门。所以对于这些外嫁女，她为什么那么狰狞？就是这样——她做了一个极大的、付出一生代价的买卖，但是亏了，以至于她没有办法，她必须回娘家讨饭。回娘家讨饭这个事儿会让她尊严尽丧，所以她就必须要挺直了腰杆子，一边吃饭一边骂娘，这样她才有体面。所以我们在互联网上看到的很多荒唐的东西，其实就源自于此。

至于说外嫁女能有几个是买卖婚姻的？你要是放到几十年前的话，那个量恐怕特别大，超出大家的想象。有一个小数据分享一下吧——就是在本世纪初的时候做出的一个汇总，大概是在90年代到2000年初这个阶段啊，在我国北方的某一个城市，她发现呢有大量的女性外嫁到了日本，于是呢进行了一下汇总性的统计，然后发现这些外嫁的女性当中大约有七成、接近八成的人完全不会日语。在你语言都完全不通的情况下就嫁到了海外去，这你不是买卖婚姻是啥呀？呵呵，任何的条件都是没有办法谈的呀——谈合作不能谈，谈感情不能谈，谈志向也不可能谈，连嘴都张不开的情况下，只有一种可能：买卖。所以比例有多大？就有这么大。那既然有这么大比例的买卖婚姻，然后再加上我们现在中国的飞速的发展，就一定有这么多的人已经亏本亏到了无以挽回的地步了。那么他们会怎么做，可想而知。所以今天的环境啊，能够出现这样的氛围，很正常很正常很正常。我们中国人怎么看？吃瓜子看吧，还能怎么看呢？他还他能干啥？他在别人家的船上，你只管看他跳舞乐就完了。
