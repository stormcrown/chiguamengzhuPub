# 第69章 谁在给导盲犬引路

财帛动人心，真的是现代社会颠扑不破的真理。我这两天又在刷视频的时候，刷到了一种小生物。我对动物本身是没有什么成见的，但是对于跟它配套的人，常常会很有成见。今天谈的是什么呢？就是导盲犬。狗狗当然是一个很正常的生物，我对它是没有什么特殊的意见的。然而当它作为导盲犬而存在的时候，问题就来了。它真的有导盲的能力吗？导盲犬这个行业的存在真的有合理性吗？它为什么会存在？它背后到底是什么样的利益链条？这些都是问题。

我们曾经在互联网上看过很多相关的视频。什么什么地方居然不让导盲犬进来，什么什么地方居然不让导盲犬带着盲人进去。很多人在看到这种视频的时候，觉得我们对于盲人是不是太不公道了？我们对于残疾人是不是太不关照了？我们怎么可以不宽容这些正在工作当中的狗狗呢？它们是工作犬呀！但事实上，导盲犬这个群体本身就是有巨大的问题的。我们从头开始：全中国有多少视力障碍人士呢？数以千万计，能够登记在册的已经超过了1500万。这样庞大数量的视力障碍群体，能够匹配的导盲犬是多少呢？加在一起，累计培育出来的200多条，目前能够使用的、正在服役的不到100条。请问100条狗对1500万以上的残疾人能够起到多大的帮助呢？我觉得在我手上已经很难比较出来了，是不是可能是我整个人身上的一根汗毛那么大的量？所以，第一，导盲犬帮不到大多数的人，帮到的只是极少数的人。第二，导盲犬的培育成本非常高。你去看导盲犬这些基地提出来的各种自己的吹牛广告就知道了——它要求选种，要求长时间的训练，然后要求各种各样的筛选，要求各种各样的高淘汰，这些通通都是成本。所以目前所估计的任何一条导盲犬，它的培育成本大约是在20万左右。这样的一个金额，是任何一个残疾人都能够承担的吗？很显然不能够。同时，很多的导盲犬基地说，我们是通通过各种捐赠活下来的，然后我们会把这个导盲犬免费赠予盲人来使用。好，你是免费赠予的，听起来是不是像一个慈善事业一样干得漂亮？好，那么第二个问题就来了：导盲犬不光是培育的成本高，它的保育的成本一样高，甚至更高。

我们考虑一件事情：大家有没有去养过狗？有没有去养过猫？有没有养过任何一个小宠物？你对于一只动物的关照，要给它食物，给它水，照顾它各种排泄的需求，然后生理上的需求，治疗疾病、安抚情绪、梳毛、整理、卫生——这一系列的工作其实是相当沉重的。当然，因为喜欢它，倒也无所谓。可是导盲犬是工作犬来的哟，它不是因为它特别喜欢狗狗所以养的狗狗，是因为我的生活很不便利，我的眼睛看不见，所以带进来的一只狗狗。然而我依然要给它喂食、喂水、铲屎、遛狗、洗澡、梳毛——这些工作当中，相当的一部分是一个视力障碍人士很难完成的。也就是说，一个残疾人要为了一个照顾残疾人的生物，而专门聘请一个人类去照顾这个小动物。有那个功夫，我雇的那个人难道不能给我导盲吗？好，第三个问题就来了：导盲犬真的可以导盲吗？我们知道狗狗可以认路。我小的时候，家里人跟我说，猫认千，狗认万，老母猪认二里半。听起来狗可会认路了，对不对？但是第一，狗是红绿色盲，也就是说它分不清楚红绿灯。第二，一只狗狗能够记住的道路是非常有限的那么几条，去了任何一个新的地方，它不认得路牌，不认得方向，也听不懂完整的地名，也无法向路人询问。所以这种情况也就意味着一个盲人只有在什么样的情况下能够使用导盲犬呢？这事儿有一个电影可以给我们一个很好的解析。前几年的时候有一个专门的电影就是讲导盲犬的，有一个厨师视力障碍了之后，他获得了一条导盲犬，然后这条狗狗常年进行陪伴，给他付出了很好很好的服务。什么样的服务？就是从他的家到他的餐厅，从他的餐厅到他的家，日复一日，年复一年。如果一个盲人的生活真的是这个样子的，千篇一律，我想的话，一个导盲犬大概也可以给他进行服务吧。何况他是名厨，很有钱，所以照顾这个狗狗他也可以专门雇一个人——我也觉得很多余，但是起码说这条路狗是可以认得的。可是对于我们一般普通的人来说呢？我们是不是永远都是这一条道路呢？对于绝大多数的人来讲，不是的。我但凡说早上去上班，晚上想去公园遛个弯，很可能那个狗狗就不认得路了。所以它能够起到的作用微乎其微。

整个导盲犬，本质上来讲，它离宠物的距离并不遥远，离工作犬的距离非常遥远。这也是为什么有很多人在说到导盲犬你要关照的时候，经常去拿那些防爆犬啊、搜救犬啊去做对照。有人说，哎，我们这也是工作犬，人家也是工作犬，我们要有统一的待遇，我要能够上地铁，我要能够进公园，我要能够进餐厅，诸如此类。但是所有这些视频里面都有一个巨大的瑕疵：一个视力障碍、行动困难的盲人去到一个陌生的地方，由一个根本不可能认识地图的狗狗带路，同时旁边还有一个摄影师拍他受到了多么不公正的待遇——有那个功夫，你让摄影师给你带过去不行吗？所以很明显他们都是来挑事儿的，只是来证明导盲犬是工作犬，仅此而已。然而那些搜救犬也好，那些防爆犬也好，它们作为真正的工作犬，受到的是什么样的工作待遇呢？第一，它们在进行训练的时候非常严苛。第二，这些犬类有明确的上下班的制度，它们在工作的时有非常严明的监管制度，身边进行监管的人员不光是健全的人，而且是受到了完备训练的工作人员。也就是说，严格来讲，工作犬是工作人员手上的一个工具，工作的核心是这个工作人员。而导盲犬是什么？导盲犬陪伴的是一个基本上在日常生活当中有部分失能的盲人，一个残疾人。所以导盲犬在这个配对当中变成了主导，而作为一个狗，在人类社会里面，它没有主导的能力。说老实话，任何正规的工作犬，它的负担都是很重的，但是导盲犬就算它做得非常完美，也只是服务了一个人而已，而它的这项服务实际上做得并不好，也有大量的电子设备可以去替代它，偏偏成本还非常高昂。所以工作能力也好，工作成本也好，导盲犬都是不能够胜任的。

那么问题就来了：这样一个起到的作用非常少、数量又非常稀少、帮不到很多人、成本又极其高昂、绝大多数人压根就负担不起的导盲犬，为什么会这么肆无忌惮地、成年累月地在我们的互联网上进行宣传？任何人都不可以说它不是，甚至有很多人去拍专业的视频来说我们普通人对导盲犬不公，我们不爱它，我们不礼貌，我们错了——为什么会这样呢？利益的诉求才是根本的诉求。所以这个背后有什么样的利益团体的需求？如果说就是一个什么导盲犬培训基地，说老实话，这个利益的链条不够大。你想一想，我们国内最大的导盲犬培训基地在大连，这个基地成立十几二十年来，一共培育出了100多条导盲犬——这种工作效率，它背后的资金量就算大，能够大到什么样的天花乱坠的程度呢？所以单纯一个导盲犬的产业，它还不足以说让我们为了这个金钱而迷醉。它实际上衍生的是它背后一个更加庞大的产业，就是宠物及其配套的产业。原本导盲犬这些狗，它们提供的工作、提供的服务和一个正常的狗之间差异并不大——认得的路和普通的家犬没什么区别，需要的帮扶也和普通的家犬没什么区别。但是呢，它却能够塑造一个人身边都有工作犬、工作犬是在为人类服务、狗狗是人类最好的朋友这样的一个社会氛围。所以呢，它能够带来很多的衍生影响。第一，你会对狗狗产生一个极大的信赖感，你会对它普遍性地产生喜爱的感觉，你会希望狗狗可以进入到更多的公共空间，因为它有服务的能力。所以一旦这种社会氛围形成，那么养狗的、给狗狗服务配套的，相关的服装的、拴绳的、饲料的、零食的、玩具的，一切产业全部都跟着一起来了；再加上疫苗的、医院的、绝育的，诸如此类一切一切的东西。你想想它是一个多么庞大的数量，而且背后所牵扯到的利益链条又是多么的复杂，甚至完全可以说百万曹公衣食所系。这样的一个系统，才是为什么始终有人在为宠物这样的一个角色贴金，试图让它能够更上层楼，拥有更多的精神价值，让更多人相信我们的生活当中离不开它，很需要它，它很不得了。这样的社会氛围、这种形式的路人缘，才能够让这个产业继续维系下去。

然而它其实是一个谎言。因为本质上，宠物有功能的，我们会拼死地去用；没有功能的，实际上就是人类的个人享受而已。它和我去买一个包包、和我去买一台电子游戏机没有什么本质上的区别。但是当我们为它赋予了更多牺牲、奉献的特点的时候，突然之间它的附加值、它的政治正确的属性就加大了，于是这个产业就变成了一个更加庞大的、更加不能够被人质疑的产业。哎，政治正确常常会对人产生压迫。如果一个产业涉及到政治正确，那这个产业就很可疑了。
