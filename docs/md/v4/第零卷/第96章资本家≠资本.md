# 第96章 资本家≠资本

其实我们常常有一种迷思：认为资本家也是人，资本家呢，也很聪明，还格外地聪明，所以他们也是一些有远见、有卓识的人。那么，既然作为一个资本社会，它其实需要庞大的人口作为消费者、作为市场，同时呢，也需要劳动者。它当然是对生育有要求的嘛，它对人口有那么强烈的渴求，是不是？这个话说的肯定是没错的。只是中间我们需要调整的一点就是：资本不等于资本家。资本家确确实实是人，而且还是人尖子，但是资本可不是哦。资本那是钞票成了精，只有一个诉求，就叫做利益。

于是，建立在这个基础之上，我们就可以来讨论一下：资本对于人的需求，和资本家对于人的需求，中间其实有一个很微妙的差异。首先要谈的一件事情就是，不同的社会形势之下，我们对于人的需求其实是不一样的。之前我曾经讲过，我们的人类社会当中呢，其实有几种不同的权柄：神权、君权和金权。这三种权柄，既然它要成全，它怎么可能不依赖人口呢？而所有的权都需要人作为基石，这个是一定的。但是他们需要的人可不同哟。

要知道，我们看一下神权需要什么人？它肯定不要我，因为我是一个无神论者。神权要的是信众，所以呢，它要求大家在信仰上面、信念上面、思想上面有高度的统一性。所以它的人口繁育的方式是生育加传教，所以它通常呢，都会非常重视生育。当我们来到了君权社会的时候，更务实一些。它最需要的是它整个的国民能够一体化，能够有高度的相似性，它可以避免这个国家在非常动荡、非常低谷的时期分崩离析。因此，一个君权型的社会呢，它是要求这个社会的内部，大家有一个血缘上面、至少是种族上面的同一性。这样子的情况它比较不容易分崩离析。因此，传统的君权社会，它通常都会有民族国家的倾向。也就是说，无论是神权还是君权这样的社会体系下，大家都要求同类的存在。

然而，进入到金权社会的时候，它突然之间发生了变化。资本要的是利益，而利益是不在乎信仰、不在乎种族、不在乎民族、不在乎血缘，也不在乎你是不是仇恨金权，只在乎一件事：你是个人就行了。这种截然不同的人的筛选方式，就会决定了他们去获得人口的方式是完全不同的。

那么，这三种模式之下，大家是怎么获得人口的？神权模式之下，它要么是传教，要么是生育。总之呢，它要做这两件事情。而在君权社会里面，因为它对于民族、国家有一个很强烈的诉求，至少说它需要有一个主体民族很强大，所以呢，它需要客观上的生育——就是我得生出来的是我家的孩子，要不然怎么能够保证将来要分家的时候，咱打断了骨头还能连着筋呢？这是一个底线的诉求。因此，君权国家对于生育上面有非常强烈的需求。在最传统的君权国家、古典中国里面，我们就曾经采取过这方面的策略。大家都很厌弃的那个系统，其实就在为这件事情而服务。我常常谈到这个东西，它好多作用哦，对它在这儿也有用。

你想一想，一个人啊，她生育的时候其实是很困难的，她要付出很多的成本和精力。而且这个孩子呢，一般要长到三岁以上，才能够脱离母亲，单独地在地上奔跑；到六岁的时候才能帮家里面捡捡柴火、喂喂鸡。所以这样的一个漫长的投资期，对于所有人来说都是一个巨大的压力。今天你觉得有压力，1000年前的古人一样觉得有压力，他们只是古，又不是蠢。所以你看我们的传统社会，一方面用精神上面告诉你“不孝有三，无后为大”、“多子多福”，天天的洗脑；另外一方面呢，就告诉你：哎，我们有孝道系统，生下来的孩子之后要伺候你哦，生下来的孩子之后有义务养你哦，不然的话，官方会代替奶奶收拾他哟。所以呢，它其实给父母以一个很远期的回报的目标，这个叫做利益回馈。这是在君权社会体系之下，我们对于社会的生育进行的激励。

但是，金权社会就和前面的方式完全不一样。因为他对人口的诉求就是没有诉求，他不在乎这些人口的素质，不在乎他们的血缘，也不在乎他们的信仰，甚至不在乎他是不是丑，不是自己——就好像是你的老板，根本不在乎你在厕所里面吐槽那个狗头老板一样。所以我们去看一下，最早崛起的那个庞大的资本国家，就是英国。他当初崛起的时候是第一次工业革命，那是一次科技的大爆发、生产力的大爆发，直接带来了资本家群体的崛起。崛起了之后，事实上他当然需要扩张，对吧？它需要劳动力，它也需要庞大的市场。按照正常的大家所理解的，要劳动力怎么办呢？给福利，让大家生孩子才会有劳动力咯。要市场怎么办呢？哎，让大家拥有更好的收入，你就有市场咯。但是，显然资本家们等不及。

我们去看那个时期的英国，是怎么缔造自己的人口和市场的。在英国的本土上面，事实上他压榨到了每一个人的劳动力，所以当时才会有雾都孤儿啊，才会有六岁的烟囱工，看不出他有想要保护妇女儿童的倾向。他当时获得人口和市场的方式，其实就是创造一个伟大的日不落帝国。他想要劳动力，搞三角贸易贩奴——毕竟广袤的草原上到处都有人在奔跑，你只要把它圈进来就可以了，他足够干活了。同样的，它需要市场，唉，培育市场需要多么漫长的岁月呀，于是他直接把大炮架到了我们的国境线上面，然后威胁我们打开国门进行商品倾销。所以你瞧，资本他要不要人口？他要的，但是他不在乎这个人从哪来。

岁月的流转，后来呢，贩奴当然被取消了啊，这种长枪短炮逼人家打开国门的方式也取消了，大家进入到了一个相对文明而讲礼貌的阶段。但是到了这个阶段，资本的手段发生了本质的变化吗？呃，并没有。我们去看一下，美国人发现自己的境内的劳工的价格提高了之后，他们直接把工厂搬到了劳工更加平价的地区，这个就是如何获得劳动力。然后他发现自己的国内市场，其实呢，再扩张也很有限了，之后他干了什么呢？他进行国际化的贸易，要求大家全面敞开国门，包括我们也是他们要求敞开国门的一个部分。但是如果大家都工业化了，是不是他就倾销不出去了呢？所以它同时还有另外的一个方式，就是卡住你的脖子，不让你发展科技，不让你发展工业，然后呢，他好把高端的货物卖给你。我们这些年是不是常常地发现，我们有很多的行业被美国卡着脖子？干嘛？他不卡脖子，他怎么卖给我们货？所以这就是资本所选择的获取人口的方式。无论是几百年前还是几百年后，它其实都是一模一样的：开拓式的、劫掠式的、收割式的获得人口，而不是用传统的方式繁育人口。

为啥不这样做？因为你要去繁育人口，做的工作可就太多了：基础的建设、公共的福利、漫长的等待岁月。对于资本来说，这种放长线钓大鱼的买卖，他看不见收益，他等不起18年。所以他需要立竿见影，他需要编造一个伟大的美国梦，然后让你自费购买一张飞机票、一张船票，来到他的土地上面打工。一个月的活儿，给你一个月的钱；干到退休年龄干不动，再克扣一点退休金，让你流落街头去吧；吃一点底线的福利，你还要谢主隆恩。这就是今天大家所看到的美国街头的景象，是不是如此？对，他之前曾经有几十年，有过一些福利的高光时刻，然而那个时候是啥时候呢？冷战时期。冷战时期的特点就是君权凌驾于金权之上，那个阶段的美国践行的是一个大政府主义啊，所以在那个时候呢，他能够把社会福利全面地覆盖下去，缔造一个何其美妙的美国，缔造一个何其美妙的美国梦，来吸引全球的移民。然而，等到冷战结束啊，这些东西都悄悄地走向了末路。今天的美国那个社会福利呀，但凡是你的那个镇子小一点，连根电话线都舍不得给你牵。劳动妇女在很多的地方，压根儿没有所谓的强制产假这件事情。所以呢，很多的美国劳动妇女刚刚出院就要上班，因为如果他在家里面去歇了产假，一方面是没薪水，另一方面是很可能会失业。这就是资本主导之后的这个美帝国主义，就变成了今天的这副模样。

所以你说资本要不要人口？要的，但是他要短平快的人口，他要现成的人口。所以这种收割的方式在过去呢，它是一种大扩张式的收割，因为外面是广袤的草原，随便它收，这个就是所谓的蓝海时代。蓝海时代大家都比较愉快啊，大家不断地扩张的时候，有庞大的市场、广袤的天地的时候，大家都一团和气。那个时候资本家看上去都是好朋友，哎，叫做什么资本家俱乐部。当然，在国内的话，他文雅一点点叫学会啊，其实都是一回事。总之呢，他们可以聚在一起喝喝茶、喝喝酒，谈一下明天的买卖怎么做。然而，等到这个市场、这个蛋糕的探索到达了边缘的时候，直接就进入到了红海。啥叫红海呢？看上去这个红海跟我们好像有关系，实际上跟咱没啥关系啊。红海呢，就是这一个资本家和那一个资本家之间抢市场。即使到了这个时候，依然没有人说：哎，我放点福利吧，让大家生孩子吧，拓展一下市场吧？没有！想得美！他从来没有考虑过这件事情。因为对于资本家来说，等18年不划算，他干倒对面的那个资本家，那家的市场就直接归他了，短平快。这个就是一个资本的本能的诉求。

当然啦，虽然说资本是这样一个短视的、逐利的群体，但是资本家是活体呀，是吧？人当然是可以掌控 money 的嘛，非常的有道理。所以事实上资本家是有促进生育的欲望的。你有觉得我在打脸吗？啊，没有啊？我们来继续。实际上，资本家是有促进生育的欲望的，而且在现实生活当中，他有很多的实例。国际化的大资本家马斯克，是不是天天在自媒体上跟你宣传：生孩子好啊，生孩子妙啊，生孩子呱呱叫啊？他还跟不同的女人生育很多孩子，还把宝宝抱在怀里面给你看，你看，小宝宝多么的可爱呀，又是你生孩子。而且也有很多的大资本家呢，会把他自己的家庭的什么愉快的照片啊、愉快的旅游啊，都放给你看：你瞧，我们都是多子女家庭，好好的教育，彬彬有礼，未来充满了希望。他都在诱使大家生孩子。包括我们的国内，其实也有资本家在不断地说的：生孩子好，生孩子妙！谁呢？前两天我们刚聊过的徐老板嘛，百子千孙计划已经进行到了一半，非常的身体力行。所以资本家是想要让大家生孩子的，他们想方设法的鼓励大家要生啊，要生啊，那是我们的未来啊。没错，他们一直这样做的。

然而，资本家说话，他只是说，你要让他真的帮你开拓生育的道路，他可就不开了啊。所以，一个人怎么说的不重要，我们要看他怎么做。像今天的很多年轻人，你去问他说：诶，大家都鼓励你生孩子，说生孩子可好了，你要不要生？年轻人会告诉你：见了鬼的，生孩子好贵呀。包括说是不是有很多年轻人说：生育嘛，其实就是促进消费，我只要生育了，必然会消费指数级的提高。所以资本家也是鼓励生育的。这件事情当中呢，错乱了一个情节，就是资本家确实鼓励生育，他也喊了这个口号。然而在现实生活当中，你是怎么样发现一生育，消费就指数级地提高了呢？我们把时间轴倒拉回几十年之前，你去问一下祖辈，那个时候我们对于生育的说法叫做：生一个孩子，添一双筷子，加一个碗嘛。没有人觉得他的成本非常的高昂。但是现在放到任何人的身上，他都会觉得生育是一件很奢侈的事情。干嘛呢？因为这几十年来，一直都有不断的媒体的热搜、各种专业的宣传、各种各样的科普，在不断地告诉你一些信息：生养一个孩子，哎，生育的时候呢，你一定要考虑哦，婆媳关系是肯定不和的，所以你得有月嫂；生下来之后呢，得有营养品添加，得有奶粉；而且呢，开裆裤不利身心健康，必须从早到晚穿上纸尿裤；孩子长大了一点之后，你必须要送到学前培育的里面去，搞一搞感统训练，搞一搞音乐启蒙、绘画启蒙；然后呢，读书了之后，你得有一些才艺方面的训练进行辅助；念完了9年义务，他会告诉你教育水平不太好的，你必须要参加课后辅导班去进行培优；如果这个孩子在学习上面不是顶尖的，他还会告诉你，出国留学经济又划算。于是，按照这种标准一算下来，20年前就曾经有专业人士给大家进行过这方面的统计，叫做养大一个孩子在中国需要120万。20年前2000块月薪就可以在街上腰杆挺直的时代，他要你用120万养一个孩子，听了一下子就觉得小腿肚子的筋都开始转了。这个玩意儿谁推动的呢？显然不是官方推动的，这就是资本在推动。官方推动的是什么？官方推动的是廉租房，是生育保险，是社会福利保障。虽然做的没有那么完美，但是左一样右一样的，都在不断地推进过程当中，总量上面其实也是一个天价的投资。然而资本要干什么？资本需要你在人生当中的每一个步骤，都尽可能的消费，因为每一个消费都会利好他的收入。

这个具象到一个你非常容易探究到的场景，就是你可以直接去问任何一个母婴店的老板，说你支不支持生育啊？他双手双脚的赞成说，那肯定的呀，多生孩子是最好的，生孩子可好了，一定要生孩子。然后你说生生孩子成本太高了，你把你店里面的奶粉都打对折，纸尿裤都打两折好不好啦？你猜下一个步骤会发生什么？是的，前面一个是什么？前面一个是资本家，后面一个是什么？后面一个就是资本。因为利益真的直接会给他伤筋动骨。资本和资本家是两回事。虽然说作为人，他知道需要人口，也知道人呢，是从婴儿逐渐长大的。但是作为资本，他实实在在的没有时间去等待你18年后成为一条好汉。作为资本，他只需要在你人生的每一个步骤上面，都想方设法地获利。而想方设法获利的行为，就直接提高了你的生育和养育的成本，直接降低了生育的指数。所以这就是为什么很多的时候，他一面说着应该生孩子，一面去帮助大家绝育。真的是资本很坏吗？不是，他并不希望你断子绝孙，他甚至希望你能够多子多福，然而他不想为这件事情出一毛钱，因为出钱是伤了他的筋，动了他的骨。他却想要依附于你的多子多福，多多获益。因此这种时候你要是说他的动机是什么，嗯，反正对于资本来说，他在乎你活不活，但是不在乎你生不生。
