# 第137章 金鹰奖是要黄了吗

看了个娱乐热搜啊，就是说那个刘亦菲入选了金鹰奖。结果呢，在第二轮投票的时候，主办方悄悄就把她的名字给撤下去了。这一下子，是不是炸了粉圈的锅了呀？都说怎么回事。唉，记者去采访，结果主办方拒不接受采访，不讲原因。于是大家就只好靠业内人士来猜测。目前来说的推测呢，是认为金鹰奖在今年设了一个新规则，就是所有的参评人员和团队都要是中国国籍。但是大家知道的啊，刘亦菲并没有中国国籍哈。目前推测是这样。反正就这么个破事儿吧。

这件事情上面，第一呢，表现了主办方无能。因为你在初筛的时候、在入围的时候，就没管这档子事儿。自己立的规矩自己给忘了吗？还是专门出来蹭热度？当然这个事儿也不是很重要啊，重心也并不在刘亦菲身上。我今天想提的问题就是：作为一个电视剧的评审单位，作为一个评审奖项的主办方，为什么非得要设一个规矩，说必须得是中国国籍呢？是因为金鹰奖是纯粹的一个国内奖项吗？它一点儿往国际化发展的野心都没有？一点儿这种心愿都没有吗？

因为其实这个世界上所有评奖的东西，评奖是什么？就是我可以点评你是几分，我可以说你好或者不好。这种评审的权力是很大的权利，这种权利其实就是文化话语权当中非常核心的一部分。我们就可以注意到，全世界各大影视奖项、音乐奖项，包括美学奖项，都是老外的那些奖。就是他们在过去的一两百年间，掌握了文化的话语霸权嘛。然后他的霸权怎么体现呢？就是你得在我这里过关，在我这里过关合格了的，你才是一个优秀的艺术人才，才是一个优秀的影视人才。包括说我们中国国内啊，前几代的一些电影导演，是不是都跑到海外去拿奖？拿完了奖之后回来直接就升咖了啊，就升级了，拍的片子就值钱了。那这件事是干啥？其实就是他们在把握着这个评审权，然后用这个评审权去筛选符合西方价值观的一些导演、艺人、剧本。从这样的评审权之后，那其他的人想要红、想要赚、想要拥有圈内的咖位，他就需要符合西方人的审美标准和西方人的政治正确。要不然的话，人家不给你讲，你就红不起来啊。

那这个里头问题就来了。过去人家设的这些奖项，之所以在明明看不上中国、甚至反对中国的前提下，依然让中国的导演和演员去参加，就是因为他想要筛选这些当中愿意服从西方领导的人呐。那我们的奖项结果自己给自己立了个规矩，说非中国的不让参与。我觉得这件事情本身是有点放弃自己文化话语权的一个表现——我们是没有资格评价外国人干得怎么样了吗？至于说，哎，我们现在打开了这个渠道，依然评不到外国人，因为很多有咖位的外国人啊，不肯来，他来了之后会被自己的圈子封杀的。但是他来不来是他的事儿，我评不评是我的事儿啊。直接在规矩里面就直接立下了“只有中国人能参与”，那这就是关起门来自己乐自己嗨，直接就把整个世界的这个文化斗争的潮流就给抛诸脑后了。你扔开了它就不存在了吗？要有这种评审的价值和权力在那，这个就是我们文化出海和文化话语权争夺非常重要的一块阵地。

希望啊，只有这一个奖是这么怂、这么没出息的。希望不是所有的奖项都一点野心、一点野望都没有。我希望的是中国的评审人员，就应该把眼界打开了，评价全世界的各种各样的人士、各种各样的艺术品，并且是按照中国标准来评的。只要说我们的市场够大，我们的格局够远，最后我们评价的标准就是世界的标准，那么就是大家向我看齐。这个才是我们最终应该达成的一个目标。光是关门，没有什么大用。
