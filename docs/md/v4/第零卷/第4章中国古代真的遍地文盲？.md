# 第4章 中国古代真的遍地文盲？

一直都有一种说法，认为中国古代遍地是文盲。最重要的依据是在清末的时候，我们进行过一次统计，差不多中国的识字率只有1%左右。大部分的人，或者说绝大部分的人都不认字。比如去签订文书时，只能找中间人帮忙按手印，连签卖身契也是按手印的方式，自己的名字都不会写。应该说，这个局面肯定是真实的，因为当时确实如此，甚至留下了大量的影像资料，这应该是没有造假空间的。

但是，据此就说中国古代遍地文盲，那就是把别的时期也拖下水了。清末是这个局面，并不意味着其他时期也是这个局面。这种假设又是建立在历史只能上升、不能后退的前提之下的。我们在其他时候也讲过这件事：历史并不是永远上升，文盲率并不会只降低不升高。

我这个人喜欢从侧面来论证一件事情，因为正面刚的时候，你总是要提出一些非常具体的证据，而很多时候这些否定的事情，要去找一个很具体的证据并不容易。我们从侧面来看一下中国古代文盲率的问题。很多时期我们是不统计文盲率的，对吧？所以你要我说拿出数据来，唐代的文盲率是多少你知道吗？抱歉，我真的不知道，因为没有人统计过这件事情，官方也没出过这样的数据。正面刚的话，没有，真的没有。但是我们可以从侧面来论证它。

首先，我们来看一本书，叫做《东京梦华录》。这本书讲的是北宋都城的市井生活是怎样运行的，里面写了各种各样的行业、怎么经营、人们怎么买东西、商店怎么给你送外卖，还有很多市井民俗、商业业态，包括衙门哪一天当值，都尽可能地写进来了，算是对这座繁华大都市的一个汇总性描述。在这部书里，它讲到了一个很重要的行业：东京城里有大量的脚店和酒楼，规模大小不同，但性质一样，都是大饭店。大家可以吃饭，也可以点歌女进来唱歌。这样的中型大饭店场所，在东京城里数量非常庞大，有上千家大大小小的。人们很喜欢到馆子里去吃东西，而所有的餐馆都一定会有服务员，因为你要去点菜的话，那个时候又没有支付宝自动点餐，肯定要交代服务员今天吃什么。在《东京梦华录》里就写了这样一个细节：客人要点餐的时候，店小二是用笔来记录客人点的菜谱的。有没有发现点什么？数量庞大的酒店，里面的店小二数量更为庞大，因为一家酒店再怎么也得有上十个店小二吧？这些店小儿属于士大夫阶层吗？文人阶层吗？不可能，他们就是普通的底层人，只有下层人民才会去做店小二这种服务业。也就是说，当时的宋代东京城里，作为下层劳动人民的店小二、服务员，普遍都是会写字的。如果一个服务生都会写字，一个城市里上千家的酒店，就意味着有上万的底层服务人员会写字。而且他会写字，能够写餐单，就说明他的书写速度不会很慢；他写的餐单能让后厨看清楚，就说明他的书写质量不会鬼画符；同时也意味着后厨里的厨师、传菜员、配菜员也识字，能够看得懂。单单只是酒店业这一个行业，就发现大量的底层普通劳动者是认字甚至会写字的。这和晚清时期我们所统计的状况明显背道而驰。我们记得在很多纪录片里，包括当时留下的客观影像资料里，晚清时期的饭店服务员，最优秀的服务员是什么样的呢？记性特别好，嘴巴特别溜，可以像唱歌一样把客人点的菜单直接唱给后厨听。也就是说，到晚清时期，我们酒店里最优秀的服务员也是不认字的，他像一个传声筒一样唱歌。而宋代京城里的服务员都会写字，你能说这是历史进步吗？我认为是倒退。

第二个例证。明代中后期，中国的出版业发生了一个技术的飞跃，从而带动了出版业的发展。当时出现了大量的小说话本。我们可以去看看，明代的时候这些小说话本的体例非常复杂，差不多涵盖了今天大部分的创作领域，例如言情小说、官场小说、幻想小说、修仙小说、三俗小说，都有，量很大，恨不得浩如烟海。当时这些小说的出版业养活了很多出版商，甚至还有好几个很著名的大出版商的老板是女的。我觉得这可能跟当时言情小说的流行有关系，这些女老板大概在选稿子的时候有优势。一个这么庞大的、能够养活一个出版行业的发行数量，你可以稍微想一想，它其实意味着量非常大。这么大一个量，像我刚才说的，里面包含了言情、修仙、三俗，良莠不齐，有一些讲得很粗俗，有一些稍微好一点。但共同的特点是：它们都是白话文小说。当时的白话文小说非常盛行，量很大，品质错综复杂，涵盖的话题非常多。这意味着什么？我们可以考虑一下：这些措辞并不优美、文化素养并不高、内容没有什么深刻内涵、基本上是拿来杀时间的文章，是给谁看的？给那些达官显贵、文人士族看？他们真看了都不敢让人家知道，因为丢人，从此以后就抬不起头来了。里面只有少数非常优秀的作品才会走到这些人的书桌上面。而这些小说里相当多的书根本就不怎么样，只是非常通俗。这些书给谁看呢？如果中国的识字率只有1%那么多，这么大量的通俗甚至有点三俗的小说，给那1%的人看吗？他们消费不来，也没功夫看这个东西。真正的社会精英怎么会一天到晚刷网文呢？只有我这种没什么用的闲人才会干这种事情。也就是说，明代的时候，中国很多市井小民是会买小说回家看的。这就意味着中国古代很多普通的市民都认字，至少认到了可以念小说的程度。这和晚清时期的局面截然不同。晚清时期，西方人在中国也推动了出版业，当时出版的都是什么呢？画报里面都没字的，里头都是一些图案，还有很多是拍美女照片，因为那个时候贩夫走卒不识字，他们想看个东西解闷的时候就会去看这些画报。而明代的时候，这些贩夫走卒看的是小说。意识不是永远上升的。

第三个方面。我们都知道，在民国的时候，中国的文人界推动了一次白话文革命，包括我们大家都认识的鲁迅先生，还有当时的胡适，都参与了白话文革命。他们开始主张文人要放下身段，不要老是去写文言文，要让文学走向大众，于是开始用白话文来撰写文章。当时出现了大量白话文写作并不容易，事实上，有些作家在白话文写作上挺有障碍的，因为那时的文人很明显刚刚接触到白话文写作，很不适应，有很多词在用的时候甚至没有达成共识。我们所有人念书的时候都说，哎呀，鲁迅这人写了错别字，当做通假字全让我们背，就是这个原因。因为他刚刚推行白话文，所以有一些通俗的字和词的用法还不熟练、不规范，就会出现大量的通假字问题。问题又来了：记得吗？明代中后期的时候，中国已经诞生了非常庞大的白话文小说，也就是说，那个时候我们的出版业已经可以非常成熟地刊登大量的白话文文章了。在白话文写作已经在明代都这么成熟的情况下，为什么到民国初年的时候，居然还要由文人来推动一场白话文革命？而且这个时候进行白话文写作的作家们，对白话文如此陌生？你觉得有不有趣？这意味着在明代的时候，普通人可以阅读一些白话文文章，而到了清末的时候，已经不存在这个事情了，以至于所有的文人都对白话文非常陌生，他们要重新开创白话文的体例。

我们永远不要假设历史是一直上升的，要去客观地认识它。不要因为今天是这个样子，就认为昨天一定更惨。我们得去看一看，那个时候真实的状态是怎么样的。中国人要理清楚中国人自己的事情，而不是一竿子把过去都打倒。因为今天我们的努力带来了我们的进步，所以过去的东西都抛掉它们，它们很稀烂、很糟糕？不能够客观认识自己，不能够以自己的祖先为荣，不能够客观地判定祖先曾经对历史、对文化、对我们的今天的文明所产生的巨大贡献，中国人是会找不到自己的根，也找不到自己的自信的。要想找到自己，就要看见自己从哪里来；知道从哪里来，才知道我们要往哪里去。

今天的网络上，时不时就批判韩国人又偷了我们什么东西。韩国人为什么要偷？因为他需要凝聚自己的民族自信心。我们也经常嘲讽印度人，怎么那么盲目自信，印度人动不动说我们的英语口音是最正宗的，英国人的口音不行。我们看到他们的自信时会嘲讽他，哎呀，他什么都相信了。但是，对于任何一个国家来说，它的国民相信自己是优秀的，相信自己的历史是强大的，对这个国家力量的凝聚非常非常重要。有没有发现中国的氛围不一样？我们动不动就去反思，哎呀，我们的祖先不行：封建迷信、落后、积贫积弱。这些词就像铆钉一样打在中国人的脑门上，可那些并不是真的呀。我们甚至没有去想我们的祖先到底是什么样的，他们过的什么样的日子。虽然我们手里的证据比印度人、比韩国人多得多得多，我们甚至不去看，也不去思考他们以前是怎样的，就直接给他们下了一个定义，打上了标签：封建迷信、落后、积贫积弱、短命。这些标签都是打在中国人身上的。所以，正本清源，拨开迷雾，看看真正的中国的祖先是什么模样，对今天的我们非常重要。我们不能只是去嘲讽韩国人抢我们的东西。在这个领域，我们真的是要反思一下，我们自己为什么那么不珍惜祖先的东西？一直要等到韩国人偷，才觉得“你怎么偷我家的东西”？在那之前呢？他偷之前，你有没有在说这个东西是我家的，这个东西特别好，这个东西是我尊严的一部分？我没有看到。大部分的时候，在韩国人把这个东西偷走之前，它们都代表着封建迷信和落后。结果人家偷走了你的封建迷信又落后的东西，你还感到愤慨不满，为什么？矛盾吗？矛盾这个时候就要改。我们自己先要把自己的身子立正了，自己先要客观地、以尊重的态度来看待自己的祖先、自己的历史，我们才能把后面的东西梳理清楚。
