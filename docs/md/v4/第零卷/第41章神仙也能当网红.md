# 第41章 神仙也能当网红

从今年春节开始，“神仙也能当网红”成了一个社会热点。好多的景区里面也开始跟风，但人红是非多，神红也会是非多。所以呢，它现在招惹了一身的是非——你这是迷信，你这是落后，你这是开历史的倒车，你这是**淫祀**。哇塞，这么大的一顶帽子直接砸下来。那我们是不是应该辩解一下，到底什么东西是**淫祀**？**这**指的是这一波小神仙啊。神仙流是不是**淫祀**呢？说到**淫祀**，首先我们要知道什么是**祀**。国之大事，在祀与戎。一个国家最重要的事情，除了军事之外，就是祭祀。嗯，说起来好像是不是有点奇怪啊？什么东西是祭祀？对，我们现在聊清楚什么东西是祭祀。因为在古典的时期，我们进行社会管理，最根本的一个东西就是进行权力的划分。最顶层的权利就是君王可以祭祀天地，然后其他的呢，就是君子可以祭祀祖先。所以呢，我们的祭祀是分层级的。庶民啊，不好意思，那个庶民连姓都没有，嗯，所以他没有祭祀权。没有祭祀权是没有管理权的一种表现。因此你就能够想象得到，所谓的**淫祀**是什么呢？就是扩大化了的、泛滥的、多余的、未经许可的祭祀，就叫做**淫祀**。

这里面大家在课本里面都学过一项东西啊，叫做《西门豹**治邺**》。这篇文章当中所讲的那个河伯娶亲，就是一个典型的**淫祀**。因为官方没有许可有河伯，官方没有同意河伯娶老婆，所以你就不能娶。唉呀，这个当神仙属实是不太容易啊。虽然当地的百姓因为恐惧于河流的泛滥，所以呢，自觉自愿地哈哈，去相信了有一个河伯的存在，并且去帮他供奉各种各样的东西，还献出财产啊，过得倒是挺苦逼的。但是呢，事实上官方并没有承认这样的一个神仙的存在，于是这种方式就叫做**淫祀**。所以西门豹**治邺**最终的处理方案就是，嗯，唉，要娶亲那个神婆、巫婆先下去吧，去跟河伯汇报一声好，把当地的宗教势力直接给丢到了河里面去，解决了一大半的问题。剩下的呢，就通过正规的政治治理来解决大家的生计的问题以及河水泛滥的问题。这个就是我们传统上对于**淫祀**的一个典型的定义和解决的方案啊，比较好理解。

好，那么我们来了赵世子这些啊，网红流神仙算不算**淫祀**呢？如果按照宋代以前的标准，呃，确实算。但是从宋代开始，这个东西发生了变化啊。我们一直强调说啊，宋明之后呢，我们开始走向了一个更加平民化的社会。在这个时候呢，我们的这个神仙流啊，就开始泛滥了。神怪小说、神怪故事、神话传说，这些东西呢，越来越多。神话系统发生了一次翻天覆地的变化，他从单线的变成多线条的了。为什么到清代的时候我们会产生一个《聊斋志异》？其实就是在这几百年的岁月流变当中呢，我们产生了大量的神鬼传说啊，复杂再复杂，变成了神仙，成了漫天的星河。在这种情况下啊，按照先秦的标准，那就叫做**淫祀**泛滥，太可怕了。但是到了这个阶段，唉，官府说不管了。这个时候就变成了一个什么呢？官府没有打击，官府没有治理，官府没有说你不行，你就不算是**淫祀**。所以既然我们是宋以后的这样的时期，那么按照这个标准来说的话，嗯，网红神仙流不算**淫祀**，只能算作**地方性的**神仙和民俗习惯传统。按照这个标准来说的话，大家在逢年过节的时候把他请出来游一游，然后拜一拜，然后求一求——啊，我明年到底能不能发财啊，我那个男朋友到底可不可以谈呢，唉呀，这个婚能不能结，结了之后会不会出卦——好，这些问题呢，都是当地的这些小神仙要解决的一个问题。这就是为什么啊，会出现这样的一些神仙，也为什么他们能够那么红起来。那当然更重要的原因，我觉得是因为好看啊，颜值真是抗打呀，我都觉得漂亮。基本上说他的那个外观设计当然也是它很核心的一个部分啊，能够当网红的，没有一个颜值差的。OK，我们就延续到下一个话题去。

为什么会出现这样浩如烟云的神仙？中国人是一个务实的民族。没有需要的东西，不实用的东西，不好用的东西，我们就会弃如敝履。例如**活鸡**吹多大的牛我都不吃，知道吧？被我们嫌弃的那些呢，就是没用的东西。凡是被我们留下来的，特别是被我们追捧的，那一定是很有用的东西啊。你迷信没错，嗯，你可以说他是一个迷信，对我。那你莫名其妙的去相信一个莫名其妙的神啊，怎么的啊，我还不能说你了吗？那你可以说呀，你说又不耽误我信，因为信了有用啊。好，我们看我们的这些浩如烟云的神仙系统，它是怎么产生的。为什么老百姓会有这样的神仙信仰的诉求呢？在我们的过去的很长很长的岁月里面啊，普通的人所面临的生活实际上是很严峻的。例如说你家里面那个妇人要生产，对，有产婆保活吗？不保活，真的不保。那这个时候这些家庭会干什么？真的有钱能够请得起名医，在大概率上保活的家庭凤毛麟角。更多的普通的家庭只请得起普通的产婆来帮助。那么这个时候他会做什么事情？就是求母子平安嘛。那么他去找谁求呢？找官府求，官府管不了，没有那些医疗资源分给你是吧。那我去找医生求？但凡他要是求得上医生，他也不会去想其他的招了，是不是。所以对于大部分的这些百姓，当他有这方面的诉求的时候，他就会去听信啊，旁边的人就告诉他说，某个山上有个什么姥姥庙啊，他们家那个保母子平安特别灵，他就一定是一步一叩首地冲上去了。为什么呢？他没得选。这是他最后的一个救命稻草。

所以今天如果诸位愿意到乡下去走一走啊，到国道上面去开车四处遛一遛，往各个小山村里面去钻一钻，你就会发现，绝大多数的村落的门口都会有一个很大很古老的树，然后那个树上面会缠上各种各样的丝带，遇到一些特定的时节的时候，那个下面还会摆上香烛。大部分的村落都是这个样子的。放在先秦时期，这叫什么？这叫做**淫祀**。但是在今天为什么就普遍性的它都有？干嘛呢？我就是去求一些很普通的诉求啊，例如平安啊，例如吉祥，例如明年风调雨顺啊，例如福泰安康。他就求这些东西，虚无缥缈，谁都无法给我们承诺。但是呢，我又需要这样的一个心理安慰。我去找谁呢？别人说门口那个大树已经千年成精了，可靠。这叫做信则有，不信则无。百姓知不知道这事儿呢？他也知道。可是他需要谁能够满足他这种需要，他就信；谁不能够满足这个需要，这个庙的香火就会没有。所以我们会看到啊，所有的这些我们去拜的各种各样的地方，各种各样的这样的宗教场所，实际上都是香火最旺的，是那些什么有求必应啊，啊那些什么财神啊、那些送子——这些目标非常明确的地方，香火是最好的。在这里头就能够看出来一个很典型的人权社会，而非神权社会的特点，就是人们去拜拜的时候其实是有求的，甚至有的时候是交易的。它是一种人的地位很高，并不是完全匍匐于神灵脚下的一个状态。这本质上来说是人类文明的高阶，而不是低阶。所以你直接冲上来说“你迷信”，有点耍流氓了。人家想要心灵的抚慰是吧，想要人家摸摸头说你明年一定会发财的，你看你这个光脑门儿一定可聪明，没毛病。他就想要这一句话，然后你死活不告诉他，你非得说智力是天生的，你要理智接受自己的现实，检讨自己的错误呀，不能迷信。然后人家回家就抑郁了。你觉得他想要什么呢？

我们的神仙来自于哪里？我们的神仙来自于每一个人心里头的愿望。这个愿望能够实现，能够通过现实生活中的东西实现，他就不去求了。我跟大家讲，过去那些保母子平安的、那些小小的、各种各样的奇奇怪怪的庙啊，那有些地方大妈庙，有些地方姥姥庙，有些地方奶奶庙——这些地方现在基本上就没有了。为啥呢？现在大家都知道了，医院可以保母子平安，是不是？所以什么情况下你能够让这些东西消于无形？什么时候我能够反封建、反迷信、打倒神权？什么时候呢？问题解决了就可以让它消失了。今天解决了吗？呵呵，有相当部分的东西还没有能够解决。例如说最近很流行说啊，大学生啊，哈，在上进和上班之间选择了上香。他为什么要上香呢？因为他知道自己上学应该干什么，也知道考学该干什么，也知道上班他该干什么，他都明白。但是他累，他卷，他心里不安，他不平静。他想要获得这种平静和安抚，他去上香，去求一下我明年的平安。求完了之后花了十块钱的香火钱，他回来他就平安了，他浑身都舒服了，他就觉得获得了安慰。有没有其他的更科学的替代方式？有的，每一家医院里面都有一个心理科啊，还有专门的心理医院、心理医师、心理咨询师，有执照的，特别科学。起步价200块钱一个小时，那还是不入门的。贵一点的分分钟来，一来**一次**啊，1000块。然后呢，这个心理专家还会告诉你啥呢？啊，你这个是心理上面有什么问题，你的原生家庭有什么问题，你要检讨自己的思路啊，你要检讨如何和别人交流。对，我花1000块钱出去，然后让别人训我一脑门子包，检讨一下我们家祖宗十八代，是不是都已经生活得十分的不良好啊，有些什么恶劣的传统需要去推翻，需要去破坏。但是你一看我到庙里面去啊，十块钱的香，20块钱的解签的钱，人家的那个和尚道士就会说：这只是人生当中最小的一个坎儿，后面一片坦途，你这个人前途无量！我一出去之后心胸开朗，啥抑郁症都没有了。你就说便不便宜吧？咱都不谈科不科学这件事情，你就说便不便宜吧？啥东西能吸引人？这个东西就能够吸引人。

如果我们能够通过科学的方式、管理的方式、社会学的方式，就能够让人心灵上得到抚慰，哎，背后感觉到有人撑腰，老是感觉自己前途无量，好有希望啊，大踏步向前，星河大海呀！你做到了这个之后，你看他还去不去烧香拜佛去？他有那个闲工夫吗？人人都可以站出来说啊，你封建，你迷信。但是这世上的所有的人都需要这种心灵的安慰，也需要生活当中各种各样的鸡零狗碎，有地方吐槽，有地方获得支持，有地方获得心理上的支撑。这是一种客观上的社会诉求。所以在这种社会诉求之下，神仙流他就是要起到这样一个基层的、安抚的作用。所以如果这些神仙他没有对社会、对人的心灵、然后对人的价值导向产生非常错误的影响，让他开始变坏，我们没有干预它的一个必要。因为社会里面需要他们的存在啊。你不让他存在，你不让赵世子存在，他就有李世子；你不让李世子存在，就会有三太子；不让三太子存在，他就会孙悟空；没有孙悟空就会上猪八戒；大家都不让上了，他牵一头骡子出来，都要说我要拜一下白龙马。这个是客观需求，谁都否定不了的。如果我们真的把这些东西都推倒了，一概不许，那么“信我者得永生”，顺道发两板鸡蛋，就会冲进到每一个社区里面去。而在我们的神仙流系统里面，我们都是很散碎的，很没有约束力的，爱信信，不信走，想信就给我仨瓜俩枣的，不信你就回家去，老婆孩子热炕头，我不管你。如果连这样的一套散碎的系统我们都不能够容忍，要千方百计地去批评他啊——你迷信，你封建，你要倒退——那么请问“信我者得永生”怎么办？哒哒哒哒哒，那些像密码一样、我都不敢说的东西怎么办？一个地方他有这样的需求，有这样的诉求就得去满足他。我们用这种方式满足，如果不满意，那么其他人就用另外的方式去满足，我们会更不满意。有问题就解决问题，**叫号**是没有用的啊，高呼打倒封建迷信是没有用的。能够构建社会，能够让人得到安抚，它才是有用的东西。
