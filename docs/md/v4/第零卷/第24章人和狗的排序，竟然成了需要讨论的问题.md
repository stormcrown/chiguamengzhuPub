# 第24章 人和狗的排序，竟然成了需要讨论的问题

听说你们眼神都特别好，真的特别好的小伙伴，之前应该看到了一些东西，就是发现我有一只猫。对，我有一只长得挺丑的猫，就是一个普通的田园猫。但是呢，我很喜欢它，毛茸茸的，性格也特别好。撒起泼来的时候也是满屋子跑酷，从天到地地乱飞。猫嘛，就是这样的一个生物。

中国人自古呢，其实就蛮喜欢猫咪的，包括我们很多的诗歌里面都有猫的存在。我们把它称为狸，称为狸奴，哎呀，都是小可爱啊。“我与狸奴不出门”，抱着一起烤火，听起来就很温暖啊，确实也是猫猫喜欢的东西。所以养小动物、养宠物，确确实实是全世界人类的传统，所以才会有很多很多的动物，莫名其妙的就被我们驯化了。也不算莫名其妙，因为古人在驯化它们的时候，其实都是有目的性的。例如猫嘛，在真正吃饱喝足以前，它们是有工作的——不抓老鼠的猫给爷爬。只有当我真的吃撑了的时候，它才是好可爱的小狸奴。狗也是一样，人类养狗是为了什么？是为了守卫，是为了狩猎。它们是非常功能性地进行驯化的一群生物。

而今天好像发生了一点变化，可能真的是大家集体都吃撑了，所以它们的职责啊、能力呀，都变得毫不重要。只要它长得毛茸茸的，它就是我的毛孩子。这个东西就变得奇怪了。听到这儿你应该都知道了我要讲什么，因为这几天的新闻铺天盖地，触目惊心——居然可以在城市里面养那么大的烈性犬，还可以不牵绳、不绑嘴，单独在外面跑，然后把人家孩子咬到半死。这还不是最可怕的，我觉得真正可怕的是，孩子已经受伤到那个程度了之后，居然还有很多人在为那条狗辩解。甚至今天已经开始出现有人说：那个狗咬人是因为应激反应，一个幼儿的存在会让一条狗应激咬人，于是那个幼儿的存在是错的。从他的话里面，我只能总结出这样的东西：就是人不应该生孩子，不应该活着，人类不应该有幼崽，应该把空间腾给可爱的毛孩子们嘛。平息一下啊，这些东西真的是让人不知道应该愤怒还是应该悲伤。总之我如果是匿名的话，绝对不是这么客气地说话，但是今天我是露脸的，听说未来还要实现前台的实名制，所以我说话必须保持克制和温和。我尽量温和地来谈论一下，今天这种率兽食人的局面到底是什么模样，是怎么发生的，以及我们应该怎么办。

说到狗对人类社会的危害，中国是世界上第二受害人群——就是每年被狗咬的人数、受到伤害的人，我们排世界number two。哇哦，我们真是万年老二！哎，我们好多项目都是排行第二诶。大家来猜一猜，这一次排行第一的是谁呢？我猜你肯定猜得到——隔壁的三哥，这一次他不排老三了，他排老大。我们在这个事情上居然能够跟印度一较高下，也是让人非常赞叹的一件事情。那么我们来看一下，为什么中国和印度这样的国家会有狗患，而其他的很多地方没有？甚至很多非常非常落后、落后到一塌糊涂的国家，也没有那么严重的狗患呢？如果说发达国家没有狗患的原因，是因为人家法制健全，嗯，狗狗都有什么什么样的制度规范啊——对，它们还有安乐死规范。但是中国的狗妈、狗爹们都不同意这件事情，就是当一个狗被人遗弃之后，延续多长时间就有权利对它进行直接的合法安乐死。中国的爱狗人士每次遇到这样的事情的时候，都群起而攻之，所以这个东西在中国居然推行不下去。诶，我们不向先进国家学习了嘞？总之，先进的国家通过这样制度性的规范，实现了它们的国民保护，我们暂时还没学。你可以说，哎，我们没有那么先进，我们学不了吗？好，那么我们来向落后国家来学习。除了三哥之外，世界上其他的落后国家总有100个吧，而且这些落后国家当中大部分非常喜欢生孩子，所以人口密度也非常的大，但是它们的受害却没有我们严重。请问为什么呢？因为这些国家它的秩序实在是不咋地，所以不但没人保护狗，也没人保护人，它进入到了一个正常的丛林竞争的环境。这种竞争环境里面，那就很有意思了嘛——那狗敢咬人，打死完事了呗。不好意思，那些什么爱狗爹、狗妈，“狗是我的小宝宝”的，这些群体在那个地方活不下去。它实现不了这种说吃饱了撑的、我要弄个狗当小可爱，这种事情做不到。饿了的时候看着街头的狗，那不是吃它还挑嘴啊？连蝙蝠都吃的地方哈，连蚂蚱都要吃，连个蜘蛛都要烤熟了的地方，狗是多好的蛋白质来源呢。所以那些地方也没有狗患。

现在中国就处在这样一个很夹生的位置上面。对于发达国家的那种高度秩序化，然后谁说话也不好使，只要定了这个规矩咱就照着办，什么舆论反抗啊、什么跳起脚来说“狗狗是我们家的小可爱啊”，这些事情人家压根不理——这一点中国做不到。所以中国对于犬只饲养的一系列的规范，形同虚设。我们不是没有规定，但是执行不了。但凡哪一次执行了，突然之间就有一群人跳起来。我就觉得这些人组织性那么强啊，背后肯定有人，但是他们都不承认，这我也没有办法，因为我也查不了人家的账号是吧。但是每一次出现这个事情，他们都可以拉帮结伙了，就把这个事情啊弄得特别特别的大。于是呢，每一次地方政府发现狗患严重需要群体打狗的时候，他们跳出来了；然后等到那个狗患严重咬了人的时候，他们不存在了。所以对狗只的管理就一直陷入停滞。那我们有没有规范？我们规范可多了。我们之前定了专门的规范：城市禁止养犬，一刀切就是不准养。后来呢，被人把这个给拱破了啊，说我们可以养温顺的那个小可爱吗？好，他们又拉帮结伙的。总之呢，就特别有秩序，特别团结。所以我们所有的这些什么狗必须牵绳、狗必须戴嘴罩、狗必须打疫苗、狗必须领狗证，然后肩高多少的狗不许养、什么品种的狗不许养——这些规则约等于没用。在这一点上面，唯一能够跟我们比肩的就剩下印度了。但是人家印度真的是有原因的呀。因为我们知道印度是一个印度教的社会，印度教，那人家不光是满街流浪狗，人家还有满街流浪的耗子呢，大家还要到庙里面去施舍牛奶去养耗子啊，要施舍东西，养那个漫山的猴儿，那养满街的流浪狗也是很寻常的一件事情，知道吧？咱跟他们是一样的吗？咱也是那种集体养耗子的社会？要求说毛茸茸的都是小可爱呀，说不许灭鼠、不许灭蟑螂，那都是万物的生灵啊——这个时候你们咋不保护了呢？到狗的时候，突然之间一堆人都跳出来了。

总之呢，中国现在我们面临的问题是：秩序推行不下去。为什么推行不下去？是因为有一群人坚定不移地保护所有的潜在的威胁。那么他们为什么居然能够有理有据地去保护这些对人类社会产生潜在威胁的群体呢？因为他们有思想的支撑。他们思想的支撑是什么呢？“狗狗是人类最好的朋友”这句话，在我们很多很多的地方，包括电影、电视剧、报纸、杂志，所有的自媒体，也包括很多蓝V的官方号上面，都会不断地重复这个信号，就是“狗狗是人类最好的朋友”。这玩意儿好像已经不证自明是一个真理了，但是这个东西从哪来的呢？它从一个美国的律师的嘴里面说出来，像一个广告语一样的向全球推广。这个广告语的覆盖率就约等于“钻石恒久远，一颗永流传”。至于钻石是不是恒久远？它一个碳，它怎么恒久远啊？项羽烧一个阿房宫，它绝对就是燃料，它还能恒久远？但是不行，结婚必须要买钻，恒久远那代表了爱情。那倒挺像的啊，毕竟这俩都不咋恒久远。总之呢，“狗狗是人类最好的朋友”这件事情本身就是一个广告语的强行推广。但是它到中国的时候，这个思想钢印吧，就打得比较深刻一点。然后这些人呢，也特别特别的团结，就形成了这种社会潮流，然后你说它一句坏话都不行了。就像今天这一次这么恶劣的咬人事件，依然有人站出来说：哎，是那个孩子惹着狗了；以及旁边的那个很明显跃跃欲试的那个白色的拉布拉多，也有人说：哎呀，它是想上去拉架的，但是它没有拉成，哎呀，小可爱，好委屈，好无辜。这个是什么呢？其实这种症状我特别能够理解，就是面对一个动物的时候，它进行脑补。事实上我也每天进行脑补，我也会跟我们家猫进行聊天。那个聊天的过程就是：“哎呀，你今天小饼干吃得脆吧？”“还喵哦啊。”“今天小饼干塞牙是吧？”它真的知道我在说什么吗？很明显，它根本就不知道。所谓的聊天的这个过程，其实就是我自己在跟自己说话。但是呢，很多的狗爸狗妈不承认这一点，他们认为他们的毛孩子真的是啥都懂，只是不会说话。

在这一点上面，我们就要来聊一下关于狗到底是什么。狗是什么？请百度一下：狗是属于灰狼科的，它和狼是完全同种的东西。同时你可以百度一下澳大利亚野狗——就是那些流浪狗成群结队的，在野外放肆地生存，几代之后你会发现它们的面貌都发生了改变，再也不是以前那种宠物狗萌萌哒的那个模样了。它们长得、行为、结团的方式都变得和狼没有什么区别。唯一的区别就是它们跟人类社会特别的熟，它们知道人类的短板在哪里，而且它们不恐惧人类。狼对于人类的集体社区是有畏惧的，它会躲开，但是这种野狗群不躲。但是其他的属性却和狼是一模一样的，这就是一个典型的返祖现象。为什么会返祖？因为它们的基因本来就是那样的。今天的狗就是过去的狼，而人类对它的驯化，仅仅只是在里面挑出了几个极少的基因变异——就是亲近和驯服。而这几个简单的变异，经过一段时间的放纵，自然就回去了。而除了这简单的几处变异之外，其他的地方狗和狼是一模一样的，没有任何区别。有人说“它是我养的，我了解它”——你不知道人类和野兽之间到底有什么区别。我觉得这里面很重要的一样区别就是人。人类出生的时候其实是一张白卡纸，所以我们会听到有狼孩儿的传说、豹孩儿的传说、熊孩儿的传说。为什么？因为人类的婴孩，它是一个零代码的状态，它是空白的，于是被任何的野兽抚养了之后，它都会向抚养它的那个野兽靠拢。你会发现，被狼养大的孩子，他所有的行为举止，甚至身体上的很多结构，都开始像狼靠拢。但是被人养大的狼会向人靠拢吗？它只会向人有点亲近感，却根本不会像人。它不会学习人用前肢吃饭，也不会学习人来说话，也不会学习人行走的方式。除了略有亲近之外，它和狼没有任何区别。因为它捕猎的代码、它发声的代码、它和同类交流的代码、它在族群当中排定先后次序的代码，是已经写进基因里面的。它不像人类一样，人类是白纸出身，可以重新书写，而这些动物根本不能。所以我的猫是我从一个月大，用手捧着丁点儿大的时候就直接开始抚养了，但是它所有的行为都是一只猫，它没有任何地方向人类学习。狗也是一样的。所以在这个时候你跟我说“毛孩他什么都懂，他根本就不会咬人的”，他说“我的小可爱和我的亲儿子一样”——你的亲儿子再笨，保底能够念完9年义务制教育，你家的毛孩子能吗？啊，动不动啊？它有六岁智商、八岁智商、九岁智商，你12岁智商够小学毕业对吧？你让它参加一下小学的入学考试，确保小学毕业，我就承认它是你的毛孩子。这个时候就要翻脸了吧？因为事实上它做不到。

所以人类和这些小动物之间，我们存在的这种合作关系、盟约关系、共生关系，其实是建立在有约束的前提下的。我们是怎么约束的？中国人和欧洲人不一样，他们以前呢，有很多游牧的传统，甚至渔猎的传统，因为他们部族比较复杂，经济生态也很复杂，农业又很落后。那这种情况下，狗对它们的经济价值会更高。但是即便是这个样子，在欧洲其实也有很多国家有吃狗肉的传统，甚至有狗肉节，只不过说因为经济生态的原因，狗在他们的经济生活当中起到的作用更大一点点而已。而在中国，它可以对应到猎人这样的一个群体，所以中国也有这样的一群人，那不是没有。那么中国人是怎么对待狗的？虽然我们也玩儿什么“左牵黄，右擎苍”是吧，也也在家里头养狗，让它们有的看门，有的打猎，有的遛弯儿，但是我们只挑其中聪明的狗留着作伴儿，其他的呢？啊，什么叫做“仗义每多屠狗辈”？它为什么不说“仗义每多屠猪辈”呢？就是因为古时候很多的屠夫最重要的工作，其实就是杀狗和卖狗肉。怎么来的？因为中国人是一个非常非常清楚自己要干什么的人群，目标的明确性，导致我们对周围所有的事物都有一个明确的分配。对狗、对其他所有牲畜都是一样的。狗是六畜之一，它和其他的动物之间并没有格外特殊的待遇。基本上一只狗它生了一窝小狗，我们会在这些小狗当中进行筛选：最聪明的，特别特别灵巧、特别特别敏锐的，可以作为猎犬来培养；然后其次的那种很聪明、很稳定的，可以作为看门狗。但是龙生九子还各有不同，狗生了一窝，自然也是有好有坏，就是有那些傻不愣登的、愣头愣脑的。那它们是怎么个待遇呢？捕狗嘛，不然那些酒肉朋友的肉哪来呀？它们的这些肉很多时候其实就是吃狗肉，因为狗肉是提供的量最大、价格最便宜的。中国人对待狗就是这样的一个务实的态度。所以虽然对于跟着我打了一辈子猎的老伙伴，可能会有一些格外的优待，例如说它死了我不吃，而是把它埋起来，但是对于狗的整体却不会说“哎呀，我养了狗，所以天下的狗都是我的好朋友啊”。这种混乱的思维在中国古时候是不存在的。

而在今天，这种奇怪的想法出现了。我觉得很多时候是因为社会生活条件好了，所以呢，思想就会变得非常的复杂，它就开始去想“哎呀，小动物疼了，唉，小动物冷了”，而不再产生以人类为中心的一个思想结构。这种思想的方式，其实和古时候率兽食人的那些王公贵族是非常相似的。那些王公贵族那里，最亲的是他自己的这帮子人，其次呢，是他所养的动物，再次才是那些农奴一般的人。而在今天，很多普通人那里，却突然之间产生了这种贵族感，就是除了我自己，就是我的小可爱，其他所有人类请靠边站。这和中国人以前正常的次序是不同的——就是远近亲疏的关系怎么排序？很明显，我们今天的社会里面，对于这个排序出现了一个错乱，他把很多阿猫阿狗、鸟鸟蛇蛇这些小动物排到了不太熟悉的人类前面。人类把人类排到了动物的后面，这是只有在极端乱世的时候才会出现的。而在今天的朗朗乾坤之下，它们变成了一种非常政治正确的东西。其实这也是某一种程度上的乱世，这是一种思想上的混乱，就是次序乱了。

很多人都知道一句话叫做“兴百姓苦，亡百姓苦”。今天的很多人把它理解为“哎呦，王朝盛世我们也是当韭菜，王朝破灭我也是当韭菜，所以随便王朝在还是不在，皇帝有还是没有，我只管当韭菜，或者我不想当韭菜了，把自己一噶了事”，一种躺平主义。但实际上这句话它本身代表的是：秩序在的时候人民在当韭菜，而秩序不在的时候人民会死。所以虽然“兴百姓苦，亡百姓苦”，但这两种苦是截然不同的。人类社会最重要的东西就是秩序，坏秩序好过没有秩序。所以你才会在历史当中看到，有很多王朝在走向末路的时候，已经那么糟糕了，已经是一个那么昏庸的狗皇帝了，为什么还是有很多人追着赶着的，非得要奉一个天下之主呢？因为狗皇帝好过没皇帝，坏秩序好过没秩序。当没有秩序的时候，社会中下层的人就会陷入到丛林法则的最底层，那比什么狗皇帝的秩序性压迫要惨烈无数倍。

而今天自由主义这种大潮袭来，其实给我们带来的就是思想上的无序开始走来了。人类的思想也开始进入了丛林法则的时代，就是谁说了算，谁的思想就是对的。于是虽然说人类的繁衍是自然本能，但是当有人说“生孩子就是有罪，LGBT就是王道，丁克就是了不起，同性恋好帅呀”——当他拥有绝对话语权的时候，这些话就变成了正确的。谁有权谁说的话就正确，谁的思想就是正确，这就是思想界的丛林法则。而至于什么东西才是客观性的正确，什么东西才能维持一个客观上的秩序，让每个人从中获利，或者每个人能够最大程度地不受害，变得不重要。为什么今天的中国人居然有人说“实在不行我就去信教”？我最看不起一神教，但是哎，你再搞下去，我就去信教。为什么？坏秩序好过没秩序。再落后、再糟糕的一神教也好过思想上面无端混乱的杂糅的政治正确，这是人类必然会追随的一种道路。所以为什么今天的中国有人开始复古，想要把宗族制度都翻出来了？像我这种想要把周礼翻出来的，说实话已经是温和派了——有些人想要把极端宗族主义都给翻出来，你知道吗？为什么？因为坏秩序好过没秩序。而很多人都已经意识到了，虽然他没有总结，但是他知道这种无序非常可怕，哪怕是皈依最落后的从不进步的一神教，都比这种没有秩序要好。所以思想上的无序靠什么解决？曾经我们去重整山河的时候，运用了共产主义思想。但是共产主义思想也是一个需要循序渐进、不断宣传、不断教育、不断在社会上通过秩序来进行巩固的东西。而今天这个巩固还有吗？谈共产主义差不多都已经变成老古董行为了。于是混乱来临，因为新的秩序没有出现，而旧的秩序却被打破了。

在无序的情况下，有些人高举着freedom的旗帜，然后把各种各样颠三倒四、彼此矛盾、彼此冲突的东西引入到一起，把这个东西叫做“多元”。但是寻常人等可能不知道你这个多元、你这个自由到底哪好哪不好，但是他本能的会知道，这个东西会伤害到我的根本利益和长远利益。于是他一定会选择走向旧时代，因为那个时候虽然落后、虽然保守，但是它有一个秩序。当我们不想走向坏秩序的时候，就要找到一个好的秩序，而不是拿无秩序来对抗，因为没有什么无秩序能够战胜有秩序，哪怕那个秩序很糟糕——好绕口的一段话，但是局面就是这么个局面。

思想上的混乱会带来很多东西衍生的混乱，而今天很多人潜意识里面已经意识到了。所以有人信东，有人信西，有人总结这个，有人总结那个，每个人自身都想方设法的完成自己的逻辑自洽，因为如果没有这个，他自己都会站不住脚。所以将来我们会走到哪里去？我希望所有的中国人能够保持中国人自己的特长，就是理性，就是实用主义——什么东西能够对我们好，我们就用哪一个。很有可能下一步，我们会从后现代主义的这种混乱之中挣脱出来的方式，是走向前现代主义。到时候真的这样了，你可别哭，因为哎，没啥大事儿——坏秩序好过没秩序嘛。
