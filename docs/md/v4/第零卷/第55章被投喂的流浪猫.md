# 第55章 被投喂的流浪猫

今天看到一条新闻：有一位可怜的仁兄跑去羽毛球馆里打球，正在战况激烈的时候，他脚底下出现了一个移动的障碍物，一下子把他绊倒了。摔倒在地之后，经过鉴定造成了十级伤残。十级伤残虽然是伤残中最轻微的一级，但事实上已经是很重的伤了。所以，他理所当然地控告了场馆以及那只猫——那个移动障碍物是一只移动的猫。这起官司后来判下来，结果找到了这只猫的饲主作为主要责任人，同时馆场的负责人作为次要责任人，共同进行了一次赔付。受害人获得了24万的赔偿金，这个赔偿金当然是很合理的。唉，伤残这个级别啊，24万是起步价了。

严格来讲，这条新闻里面传递出来的信息是喜大普奔的。因为在过去我们的印象当中，涉及到猫狗的很多判案，通常要么找不着主，要么对方赖皮不负责任。但是这一次案件，我们不但为这个乱窜的猫咪找到了主人家，而且还清晰地让他承担了这样一个事件的后果，受害人也得到了应有的赔偿。本来是一件好事，但是这个新闻专门这样爆出来，我跟你讲，报道这件事情，它每一个字说的都是真的，但是这个新闻放上去之后，在评论区里面，你就能看出来它到底想引发什么样的讨论——就是：我喂了一个猫，我还要为它负这样的责任吗？为什么？因为那只猫原本是一只流浪猫，是有人在那个馆场附近专门安放了一个投食点，每天定时在那里进行投喂，这个猫才经常性地跑到这个场馆里面来。

在这个投喂人的心目当中，这个猫是无主的，是野放的，跟我毫无瓜葛，我只是来释放了一下我纯洁的小爱心，帮助了一下那个流浪的、可怜的、可爱的小动物。所以这个投喂人事实上感觉自己纯洁又无辜，十分善良，根本就不应该为这件事情负责。而这个情形啊，在很多在户外进行野放投喂的人心目当中常常存在，他们普遍都有这样的心理。请问诸位的小区里面有没有这样的人？我就问有没有吧，我都见过好多啊，爱得不得了。

那么这些投喂的人，事实上在自己的本心里面，他们千方百计地相信自己是在行善，是在照顾流浪的无辜的小动物，他当然觉得那是一个野动物，我不需要为它负担责任。这是什么？我满足了我爱心的诉求。你可千万别觉得你履行了社会义务啊，这是一个爱心的诉求。你获得的是利益——就是“我好有爱心呀”这样的一个利益，并不是什么满足社会要求。因为社会上面并没有要求你去喂流浪猫，流浪猫也没有举裁判说“快来喂我，你不喂我，我要没了”。都没有，你是为了满足自己的爱心去喂了一只流浪猫的。既然你从流浪猫身上获得了这种情感反馈——哇，我好有爱心，哇，我好善良——那么同时就必须要为这件事情承担后果，就是一旦这些流浪猫狗做了什么事情，饲主就应该负责任。

其实这个案子给我们传递出来的正向消息是什么呢？它清楚地传递了一个信息：在我们的系统里面，在我们的社会规范系统里面，猫和狗这些所谓的小宠物，它到底是一个什么样的身份定位？什么定位呢？猫、狗、宠物、牲畜是主人的财产，一旦它发生了什么问题，都是由主人来负责承担后果的。如果是你在我家里面被我的猫挠伤了，我是不是应该出医药费、出疫苗的钱？虽然我觉得我们家猫从来都是这样圈养的，你没有打疫苗的必要，但是你觉得不安全，你要打，那我当然就应该付账，必须要负责任，对吧？那么在外面那些你说它是流浪的，这就涉及到一个定义了：什么是流浪？动物会流浪，为什么？因为我们默认人是有一个家的，如果他没有这个家，我们就说他是一个流浪者。那么动物它是流浪的，就意味着你默认这个动物应该有一个家。

那么野外的猫是不是需要人来喂养呢？它都野外了，为什么要人来养？一个野外能够生存、并且没有危机的东西，你为什么要去喂它？还有很多的三有动物，你的每一个小区里有大量的三有动物，例如黄鼠狼，例如麻雀，例如灰喜鹊，例如斑鸠，你喂了没有？喂不喂啊？都是小动物，明明是小动物，你为什么非得要去喂猫呢？不就是因为它毛茸茸的好可爱，可以安抚我的内心吗？我为什么养猫，我也是这个原因呢。亲，对，它能够安抚你的内心，它能够萌萌哒，它能够让你觉得十分舒服，所以你为什么不把它带回家里面养？

流浪猫的存在，本身就是猫主人不负责任的一种表现。它既然已经在外面，它能够生存就生存，不能够生存它就不生存，这个就是我们对于普通的生灵的一个态度。外面有很多的麻雀啊，有很多的灰喜鹊，在过冬的时候，日子非常非常难过，我们并没有去干涉它，因为这就是大自然的法则：能活下来的就活，活不下来的就不活，除非它已经面临到灭绝的危机了。我们对于野外的动物都是这样看的，这是唯一的准则，也是大自然的法则。至于说那个特别可爱的、生存能力很受限的流浪猫是啥呢？那是被人抛弃的。所以怎么对它呢：要么你把它带回家去好好的抚养，要么你就让它面临到大自然的优胜劣汰。这个才是一个人在面对自然生态的时候应该采取的策略，也是对自然生态最好的一种选项。而不是说我们选择性地去干预其中，那些特别可爱、特别漂亮的，让它大肆的繁殖，然后在外面占据生态链的主流。你要知道，一群野猫在这个地方，麻雀和灰喜鹊、斑鸠就遭了大殃了呀！

总之呢，就是我们应该如何对小动物，以及我们应该如何对流浪动物，都是同样的准则。因为我们所有的规则，我们所有的秩序，我们所有的行为，都应该是为了人类的发展而来的。一直有一些人在那里鬼叫“我们应该设小动物保护法、宠物保护法”，这个思路本身就是很过分的一个思路。你看我们有没有动物保护法？有的。什么样的动物呢？野生动物、珍稀动物，对吧。对这些动物进行保护，对它们保护的原理是什么呢？如果我们的自然生态链受到了破坏，人类的生活就会恶化——看到了没有？这个保护的规定，保护的是谁？保护的是人类。我们的所有的社会规则、法律规则、一切的制度，是为了人而存在的。为什么？因为我们是人呢，亲，没有其他的理由。人定的规则是为了人的存在而安排的。所以我们去保护野生动物、保护珍稀动物也好，为的是人类的生存环境和人类族群的永续发展，而不是为了小动物本身。对，这个世界就是这么残酷。

事实上，那些所谓的“我很有爱心，我去投喂了”这些人，他也是为了人类的私利。他在嘴上说的叫做“我爱小动物，我在保护那个猫咪”，猫咪没你的保护，它也一样能活啊。活不下去的那些是有问题的，有问题呢，你真想让它活，你就抱回家去。所以本质上他是干嘛？他是在安慰自己的心灵，他想让自己感到幸福，他希望在不付出巨大代价的情况下当一个大善人，这是一种精神的满足感。他说他在为自然，事实上他也是在为私利。这个是投喂流浪猫狗事实上的一个矛盾。

那么在这种情况下，我们如何认定那个流浪猫狗的归属呢？哎，我们前面说了啊，所有的动物伤人，我们先要找的是谁？动物的主人。所以在这起案子当中，为什么会把那个饲主揪出来？因为那个猫啊，它虽然是个流浪猫，但是是有人专门定点的去喂的。大家有目共睹，人人都知道那个猫除了不住在他家里之外，就是它的猫，呼之即来，挥之即去。那这种情况下，你能够不为它负责任吗？要负责任。说老实话啊，我觉得这个案子真的很好，它拿出来广为宣传，这些出去擅自去喂其他流浪猫狗的人才会警醒：我为了它是要喂它负责任的。因为什么？因为这些生物它没有自制的能力，它没有充足的理性，它不知道应该做什么，不应该做什么，也不知道社会规则是怎么制定的。如果出了事儿，谁负责？谁养谁负责。不然岂不是都找不着主了吗？

如果说这个人是在野外遇到了一个确实无主的生灵，他受到了伤害怎么办呢？嗯，有这种伤害的人很多啊，自己认栽能怎么办呢？难不成让那个小动物给它赔点什么吗？拔两根毛给你了不起了好吗？已经十分抱歉了，还不可以吗？可以了，就到这里。所以小动物是没有办法为自己的行为负责任的。但是在城市生活里面，在城市的秩序里面，我们人类为什么要构建城市？是在构建一个完全以人类为中心的一个运行体系。它才是我们从一开始构建乡村，到后来构建村镇，再到后来构建城市，都是在为人而服务。所以在一个城市的运行当中，无论它的秩序也好、设置也好，甚至包括这里面的一切生灵，可存在或者不可存在也好，统统都是为人而服务的，不是在为其他的生灵。如果你说生存权，谁都生存权，万物皆平等，那不好意思，老虎也是有平等的生存权，那咱干嘛不让它上村呢？老虎很想进村啊，狐狸也很想进鸡圈呐，众生皆平等，你凭啥不让人家进来？为什么呢？啊，不，为什么？就是因为无论是村镇也好，城市也好，我们都是为人而建的，只要你会伤害到人就不行。所以，流浪可怜，要么人能生存，要么人不能生存，只要威胁到人类的生存，它就不可以在这个地方维系下去，这就是标准。

然后我在评论区里面看到了一个特别可笑的评论，我一定要给大家分享一下。就是有一个人说：“那我就是一个有爱心的人，我还给流浪汉盒饭呢，那是不是我养了那个流浪汉，他将来去偷去抢，我还要负责任啊？我还要为他负责吗？”非常非常有趣。为啥呢？第一，他把流浪汉和流浪猫狗放在了同一个生态位上面。流浪猫狗可以决定自己咬不咬人吗？它知道咬人是对的还是不咬人是对的吗？它不知道，但是流浪汉知道。你甚至忘记了流浪汉，他只是流浪，但他是个人。如果你真是这样想的啊，我建议你下一次去给流浪汉盒饭的时候，你就跟他说：“你看我不光给你盒饭，我还给流浪狗喂食呢。”你看那个流浪汉对你什么态度。这段话说起来爱心满满，但本质上是在说什么？嗟来之食。
