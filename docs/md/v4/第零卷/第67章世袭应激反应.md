# 第67章 世袭应激反应

前两天有一个视频就很火，一个小孩子说：“嗯，我要继承家业，当农发行的行长啊。”天哪，我终于知道了，还有一个农发行干得漂亮啊。总之呢，这件事情发生了之后，在网络上面就引发了很多很多的种讨论，一时激起千层浪啊。普遍性的网友直接跳出来啊，家里面发生了什么，连这个都可以继承了吗？要搞世袭制了吗？为什么在中国的互联网上，会形成这样的一种争吵？唉，我觉得这个可能比小孩子口无遮拦这件事情还要重要一些。小孩子嘛，哎，他很可能就是做了点小梦，胡说八道，这个概率也是存在的。而且已经有调查组下来了，所以没有关系，事儿不是什么大事啊。那现在呢，我们就来看一看我们互联网上的小伙伴天天都在怎么想。问题在我们这里。

你说，哎呀，我要继承家业，继承他的官职。嗯，我们立刻想起来了，你要搞什么？是不是要有事儿？有什么问题？这代表了什么呢？这代表了我们的历史厚重感啊。经过了漫长的历史教育和历史教训，中国人可是太清楚什么叫做世袭了。在这件事情上，我们可以傲视全球，因为世界上其他国家的人民绝对没有我们这么敏锐。你要知道那个学渣小布什诶，他如果在教室里面发表一个演说，说我立志要上哈佛当高材生，然后要当美国总统，这绝对是一个励志演说。哎，有人会去说，哎，你你你爹不就是总统吗？你要干啥？哎，没有这个问题。这代表了美国梦啊，人人都可以当总统嘛，这叫做梦想。但是放到中国人身上，这就不一样了啊。

中国人到美国去之后是干什么呢？啊，虎爸虎妈拼着死，要让孩子去念书、考学、搞奖学金、升级加薪，挤进专业的行业，太可怕了。你看中国人跑那边去拼死了，洗盘子、刷盘子，周末都不休息，开了餐馆之后一年365天不关门啊。你是要卷死大家吗？你是不是一个正经做买卖的？你是不是违背了劳动法的工作制度？过去一直都是这样批评我们的对吧。但是你看中国人无论走到哪里，我们表现的是什么呢？努力的赚钱，努力的攒钱，努力的考学，努力的上进。这其实代表了一个潜台词，就是我爹是谁不重要，我是谁更重要，我往哪走最要紧。我只要埋头去走，总能走得通。而对于大部分其他国家的人来说，其实这个希望是不存在的。例如说小布什的同学，你才考20分，我每次都考100分，但是不好意思，只有你才能够做梦去当美国总统。这个叫什么呢？这就是用金钱和努力都不可能打破的真正的阶级隔阂，这个就叫做世袭制。

而中国人对这个事儿太熟了。我们过去啊，曾经听过谁是富可敌国的，最豪横的是什么人。烟花三月，为什么要下扬州？为什么要去苏杭干嘛呢？哎，到那个地方浪才最美，是为什么呢？因为顶流的大盐商啊，都在那做生意，豪掷千金建的那个园林，那叫一个美不胜收哦。但是请问这些盐商是真的不想进长安城，不想进京都吗？不好意思，老爷们不让他进，你腰缠万贯不好使。真正的阶层隔离，真正的世袭制，真正能够卡脖子的老爷们，不在乎你能攒多少钱。这个才是真正的亘古不变的隔阂。

而今天我们就能看到了，中国的人民不愧是革命的人民，不愧是先进的人民，不愧是敏感的人民啊。革命老区的人民了不起，就在一个小孩子的口无遮拦当中，我们都能够看到世袭制有可能会卷土重来的味道。我们随时随地准备好了锤子和镰刀，向所有的特权头上砍去。咱们就是不能够让他们再回来。老爷，老爷是什么？人民才是永恒的。
