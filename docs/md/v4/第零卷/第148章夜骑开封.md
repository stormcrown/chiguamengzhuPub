# 第148章 夜骑开封

其实就“夜袭开封”这事儿，你但凡把视角放到不同的人身上，会觉得特别有趣味性。  
如果放到大学生那边，这玩意儿就是青春无价——有人一声呼喝，然后热血沸腾，直接上头就去了。头一天还在那儿感叹自己“脆皮大学生”，吃饭吃不饱，摔跤就骨折，好像个小废物似的；结果夜骑几十公里，毛事没有。这就是一个非常宝贵的青春记忆，代表了阳光、朝气蓬勃和力量感。这事儿是客观存在，就放在这儿了。  

但是呢，视角稍微一转，转到那些被他们第二天堵路之后耽误了上学、耽误了上班、耽误了看医生、耽误了考试的人——那就是一种天都塌了一样的欲哭无泪。  

再把视角放到开封当地文旅那儿。当地文旅其实高兴得很，原本以为这泼天的富贵终于降到我开封府头上了，呵呵；结果没想到家里的盆不够结实，金元宝砸下来，把盆砸穿了。实际上他们已经做出很多努力了，然而幻想着这一波热度要来，兴许得来个十万吧，好狠哪！唉，我预留了二十万人的空间档位都留好，预案全部拉好，之后我就成网红城市了。没想到三四十万都挡不住——别说开封了，你换别的城市其实也兜不住这个数。  

其实开头的时候，就是几个大学生在那儿折腾，喊了个号子，在社交媒体上喊了个号子之后，别人觉得有意思，直接就跟风上了，根本不考虑什么前因后果。年轻人琢磨这事儿干嘛呢？他只要想起来“哎呀，热血汤包”，这两样东西能直接连起来，然后直接就出门冲出去了。大家都觉得这事儿朝气蓬勃得很可爱，但架不住就是可爱；这个东西多了之后就变成了“可爱多”，就不一样了，是吧？  

所以想到几十年前也有这种问题。当年我们发展还不行的时候，其实有很多公知，老是拿我们和新加坡做对比，说“你看人家新加坡也是华人地区、华人国家，但是采用了西方人的管理方式，人做得多好”。问题在于，你管一个县城——新加坡顶多算个大县吧？它也就是一个县城的尺寸，那么大的体量，算它是个超级大县城好了。就那么大个地儿，你管一个县城和管一个数百万平方公里的庞然大物，完全不是一个量级，这就叫量变引发质变。  

这波事儿也是一样的。你要说是有几百个大学生，热情洋溢，我沿途跟拍，爽翻了，嗨翻天，烧烤好吃，汤包好吃——一开始是个VLOG，几个人的事儿；然后变成了一个新闻，几百个人的事儿；再然后变成了一个社会性事件，大家认为会有几万人的事儿；然后就做好了二十万人的预算；再然后就变成了，嗯，四十万赵军夜袭开封府——就，就就这么档子事儿。  

关键这个东西你再转换一下视角，你知道吗？就又变得有意思起来了。你考虑没考虑过外国人的视角？对于外国人而言，这是一个什么场景？年轻人据说非常迷恋于奶头乐，新时代的年轻人，没有强而有力的组织，就是一句口号就能集结起数十万人之多；使用遍地都有的共享单车，一夜行程几十公里，达成一个共同的目标。就这种组织力度，而且中间没有出现什么零元购，没有出现什么打砸抢，没有混混、没有无赖——就这样一种全面的、阳光的、朝气的、有序的、乖乖的，真的是乖乖的。因为中间有一些地方路堵了之后，官方要求他们回去，他们就回去了。就这样有序的年轻人，都是大学生，二十上下，这放到任何一个国家里面，都属于军队级的素质。能够做到这样令行禁止的部队，在全球范围内就那几个大国能做到，很多小国连这都做不到——军旅做不到。我们这里的脆皮大学生，天天吐槽自己是个小废物，清澈又愚蠢，但他们就可以做到这个。对于外国人来说，天都要塌了——我们在这里玩，但这已经产生了军事的震慑力啊。  

当然了，下一次最好别玩这么大，哈哈。我觉得能兜得住这么大量级的地方真的是不多，基本上兜不住啊。这个就是当你达到了一定的水平之后，它就是这样一种局面。所以你要是问我个人什么观感——我作为一个中年妇女，我只要看见年轻人我就开心。
