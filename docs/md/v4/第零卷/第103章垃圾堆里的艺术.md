# 第103章 垃圾堆里的艺术

央美又来上分了啊。上一波呢，是讲的央美的国画啊什么什么的。我说这个东西吧，它里头其实有一些市场供应比呀等等的问题。这一次可没得洗了啊，因为央美的毕业设计展上面出现了垃圾堆。大家有没有去看这个热搜的乔木？其实就是一个央美毕业的女孩子，做了一个毕业设计。做的设计是啥玩意呢？总之就是你们知道的，那些捡破烂的老太太们，会把那个纸壳子一层一层一层摞起来。对，她就整了一摞子纸壳子、真纸壳子啊，然后就做了一个毕业的设计。大家说，哎，你这玩意儿做的不就是一个垃圾堆吗？于是呢，作者本人出来哭唧唧地解释了一通，说我这是取材于我自己的老祖母啊，她在家里头就整这些玩意儿。我为了这个东西，每一块那个纸板子都是我自己亲自切的，我还切了自己的腿呢。这真是不得了了！而且这个东西它的造价有2万块钱之多呀，里头还专门买了一个电机，它会转。你瞅瞅厉不厉害？这垃圾堆还会转，一般的老太太做的垃圾堆不会啊。总之呢，她就做了一个特别高大、特别威猛的一个会转的垃圾堆，然后作为了一个毕业设计作品。这属于你不解释还好，越解释越黑的一个项目啊。你这是把老祖母都给埋进去了呀，是吧？哪家老太太摞那个纸盒子，能够摞得像你这么不工整呢？总之呢，她既没有真正的致敬到劳动者，也没有致敬到真正的艺术，但这是他个人选择。他但凡能毕业，我能怎么着呢？啊，我又不是他老师。这个背后影射出来的一个问题，并不在于这个作品本身，而在于什么叫做现代艺术。

在现代艺术这个领域里面啊，我们常常说相对应的叫做经典艺术。经典艺术大家都好理解，包括今天的任何一个人，其实都能够理解什么叫做经典艺术——就是那种一幅画摆到面前，哇塞，我的天哪，他那个手是不是人的手啊？他怎么画得出来的呢？这种东西就叫做经典艺术，就是肉眼可见的美丽、肉眼可见的震撼、肉眼可见的辉煌。这些东西呢，就叫做经典。而现代艺术就比较奇特了啊，这些年以来，有大量的现代艺术震瞎我们的双眼。我也不知道咋震瞎呢，总之我的眼睛已经瞎了很多次了。给大家分享一下：例如说，开现代艺术先河的第一个作品，是把一个马桶拎到了展厅里面，往那儿一放。对，它就是一个现代艺术，就是一马桶，没有别的啊，装置也没有，也没有电机也不会转，也没有勾搭。但是呢，它是现代艺术，为啥呢？因为它是艺术家做的。它还有一种啊，就是把一个香蕉用一块胶带粘在了墙上面，也是现代艺术。后来呢，被人偷吃了，第二天展厅方面呢，又放了一个新的香蕉，又粘上去了。对，它是一个现代艺术。据说还是几千美金啊，忘了还有标价呢，这香蕉有点贵啊。再例如说，还有那个什么——这这能不能说呀，是吧？总之呢，还有人把什么卫生巾贴到什么什么地方，我都不想唠这个话题。总之是个现代艺术这个玩意儿啊，它就永远都跟正常人的思维绑定不到一起。

近两年来呢，有一些人玩梗开玩笑，就说在现代艺术的展厅里面，我吃汉堡吃了一半儿，然后呢，有事儿，我就把汉堡放在了地上的角落里面，去上了一个厕所，回来的时候发现大家都围着我那半块汉堡拍照片。这就是今天展厅里最杰出的作品。虽然是一个梗，虽然是个乐子，但是这确实凸显了现代艺术的特色——就是正常人看不懂，脑子清楚的人看着像个笑话，脑子糊涂的人看着迷糊。总之天晓得他是什么现代艺术。艺术与非艺术的界限呢，就只有一条了，叫做艺术家的创作。之前还有一个伟大的艺术家，天天蹲在家里不知道吃什么，反正就放屁，然后放了屁之后自己拿塑料袋装起来，然后说这个是艺术品，拿出来拍卖。唉，这都是什么跟什么呀！总之我们来对比一下吧。经典艺术领域里面，一个艺术品是不是艺术，取决于它好不好看、震不震撼、迷不迷人。而在现代艺术里面，艺术与非艺术之间的界限呢，就在于它是艺术家创作的，或者不是艺术家创作的。有没有注意到这个中间的问题所在？当它是靠人的身份来划分艺术与非艺术的时候，就意味着这个东西它形成了一个产业，而这个产业变成了没门槛的东西。它的门槛儿叫做艺术家，而一个人能不能成为艺术家，就意味着他有没有其他的艺术家给他背书——承认这叫做什么？这叫做圈子。在现代艺术的领域里面，一个人他进入到了一个圈子里面，拥有了艺术家的身份，拥有了艺术家的背书，于是他不管做什么都成为艺术啊。对，有些人会说，你看人家是有理念的，人家能把理念说出来。说真的，要不然你做一个艺术品，然后让我来给你编一个理念，我可以编的，没有关系。我还可以现场给你编理由，这东西谁编不出来呀？一个风扇破了啊，这代表了那个什么后现代主义的颓丧之美啊，对于80年代那个黄昏岁月的回忆是吧？对于幼小时期那种嘎吱嘎吱响的摇扇的追索啊？这代表了心灵的创伤，这代表了灵魂的升华。现编呗，啊。那个至于风扇到底是老奶奶用坏了忘了修，还是艺术家刚刚放在那儿的，那谁知道呀？艺术的重点就变成了你是不是艺术家，就像今天我们看到的这个垃圾堆是一样的。这个垃圾堆要是真的是一个捡垃圾的老太太放在哪个墙角的，那显然就属于妨害了那个，呃，妨害了消防安全啊，然后物业呢，还要去清除，物业还要去规劝说你收拾收拾吧，这不安全、不卫生，养虫子。但是它是一个艺术专业出来的、顶级艺术学生的毕业作品，于是它就变成了一个艺术品。这就是一个典型的圈子路线。而任何一个产业，当它形成了圈子路线的时候，它就变成了一种变现的通道。怎么变现？按那个一包屁能卖600美金，这种东西就叫做变现。你放的行吗？我放的肯定不行，但是艺术家放的可以。那问题就在于，什么样的人是艺术家？谁来决定谁是艺术家？什么标准决定一个人是艺术家？艺术家变成了什么玩意儿？当放屁的、粘香蕉的、买香蕉的、卖胶布的都是艺术家的时候，那些画画的、雕塑的、研究美的变成了啥呢？
