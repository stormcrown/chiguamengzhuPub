# 第50章 tiktok为什么被围剿

美国目前初步通过了一项法案，要求TikTok必须被收购或分割。给出的理由非常简单直白：你危害了我们的国家安全，你不是本国企业，不能干这个活。看看这营商环境对外资多么“友好”。总之，从这件事情上我们可以看出几点问题。

第一，我们天天说美国是一个自由又民主的国家。民主吗？半数美国人反对这件事，但国会直接就通过了——不用跟你打招呼，也不在乎你打了多少电话、有多么不满意、断了多少人的财路。Who cares？这充分展现了什么是自由、什么是民主。民主一人一票，好像还不如“为民做主”是吧？这是第一点。

第二，我们还得关注几件事：为什么老美非得跟TikTok过不去？这不是今天才开始的，他老早就开始收拾了，一轮又一轮，骂个不停。要说TikTok有什么攻击性，我们能感觉到吗？这是很重要的事。今天我看到有人说，这就叫“非我族类，其心必异”。他们内部非常强调这一点——昂撒白人就是昂撒白人，从来不跟你当一伙的。虽然这个观念存在，但他们并不是对所有非我族类都往死里踩，因为人家现在是世界霸主，丢不起这个人，也会损失无数小弟。全世界大部分小弟都跟他非我族类，他不能把这事摆到台面上。所以他对TikTok这么狠，除了非我族类之外，一定还有更根本的原因。

过去他们围攻非我族类的企业也不是第一轮了，但主要围攻对象是什么？比如把法国的那个高管抓过来，原因是人家的专利权、市场挤到了自己的势头。这是有攻击性的，确实夺了咱家财路，所以抢一下。前几年打了中国这边的高科技企业，重点在科技领域。当时我们说打贸易战、科技战，打的是我们这边可能挑战他们科技领先的核心企业，比如华为这样的出头鸟。因为你看你技术都敢不用人家配件了，每年注册的专利数都敢超过美帝了，你咋那么能耐？这是核心问题——确实产生了核心竞争。这时候下黑手，虽然很流氓，但比较好理解：竞争嘛，惨无人道、毫无底线、臭不要脸的竞争，起码有个理由。但当他针对TikTok的时候，就要揣摩清楚：他为什么非得针对一个具有传媒经验、传媒特点的普通视频平台？

TikTok作为技术，并没有高级到什么地方去。老美自己也有视频平台，短视频、长视频、社交平台都有，软件开发什么的，说老实话没难到那个份上。你看人家AI构建、写软件、搞创意没啥大毛病，虽然在工业上差一点，但在艺术创作上走在前面。在媒体软件这个领域，别人是有实力的。在他的优势主场，他非要跟TikTok过不去，那一定是TikTok还有点什么别的事。TikTok到底有什么长项，能让美帝感到恐惧、非收拾不可，甚至要专门立法？说老实话，他当年针对华为都没有下这么重的手，从立法根上要把它弄没了。所以TikTok到底触及了美帝的什么？

一般情况下有人说，你看加沙冲突，就靠TikTok才能传播到美国的大街小巷。其实认真看，美国人自己本土的平台上也有相关信息，并不是没有。但为什么要针对TikTok？这里涉及几个问题。第一，开头说了，美国没有真正的民主，是标准的精英政治。半数美国人抵死不同意收拾TikTok，但国会几十个人投票就落定了，谁说话都不好使。精英政治，这是典型的表现。这种精英政治延续了古典时期的贵族政治、神权政治、领主政治等遗毒——极少数人说了算，下面的人听听就完了，不要插嘴。在这种情况下，我们要考虑政治生态、人文生态和整个社会结构高度保持一致。没有谁家的政体跟人民结构之间没关系。政体是精英政治，社会结构就不可能完全平等。虽然他喜欢讲平等，天天吹自由平等，真自由真平等怎么会有这样完全隔离式的精英政治？那是不存在的。

首先，美国社会精英和平民阶层之间有一道天堑，一道爬不过去的鸿沟。怎么维持？一方面让底层享受快乐教育，舒舒服服的，当服务生都能幸福一生，长着一张没被欺负过的脸。你想晋升中产、中产以上、资本家阶层？不可能，你没本事，没敲门砖。甚至想靠苦读从贫民窟爬出来？公立学校不教你，高中还在学加减乘除四则运算，想考大学得是绝顶天才，普通人没希望。这叫阶层隔离。美国是一个明确有阶层隔离的社会。这种阶层隔离面临同样的传媒生态，就必然是传媒隔离的状态。

怎么传媒隔离？几十年前，美国跟我们宣传要有言论自由，媒体是言论自由最重要的工具，不可以是任何人的喉舌，得是自由的媒体。我们当时完全接受这个说辞，所以早就把自己的媒体去中心化了。以前我们的媒体是喉舌，后来听说违背言论自由，就非常自由了。你看我们今天媒体多自由，想说什么说什么，老百姓反对？老百姓是谁？不在乎。但美国是典型的说一套做一套。他告诉我们的叫媒体自由、第四权、无冕皇帝，但放他自己那是不是这样？我们看他们的媒体结构：传统媒体、报纸、杂志、电视台、广播，把背后股权扒拉扒拉，最后总能找到个位数的那几家老板。所以事实上，美国多少媒体、多少报纸、多少电视台，背后的核心老板就那几个，而且互相是连襟、亲家，在同一个俱乐部抽雪茄。这种关系下，遇到重大事件，整个美国所有媒体可以一起发通稿——这叫言论自由，非常自由。几百家媒体出得天翻地覆的同一篇稿子，厉不厉害？这是传统媒体。

放到网络媒体上，是不是自由了？国内很多人羡慕美国网络多自由，什么上面都有，全是我们的负面。但美国自由的网络平台真的自由吗？这就涉及一个点：它是IP中心制。上面有大量顶流IP，备受关注，这些顶流制作能力很强。你仔细看，虽然表面教你吃喝玩乐，但片子做得那个好的程度，不是寻常人等玩得起的，需要巨大精力和钞票。也就是说，在互联网平台上成为有发声权的顶流，需要巨大投入和机缘。虽然平台公开、全球化，但顶流IP数量很少，它们形成的资讯矩阵覆盖了绝大多数互联网。作为美国这样一个FBI可以随时破门而入检查你上一条帖子的国家，数量有限的顶流IP非常好掌控。所以，你说自由平台有多少？顶流那部分有没有被管？不知道，被管的人也不会告诉你。总体上是中心分发制，只是多中心。

但到了TikTok这件事上，我们发现TikTok跟这个系统完全不兼容。TikTok的中文版叫抖音，大家都知道。抖音这个平台，有人又爱又恨。爱的是拿到手就离不开，看到的全是想看的；烦的是，明明关注了几个最想看的IP，但虽然推送频率高一点，可那些相似视频依然不断推送，雨露均沾。这叫去IP化。抖音是很典型的去IP化运营，非常分散，不让搭IP行程。你想记住一个大号只听他的？不好意思，抖音平台不支持，滑过去就推另外一个人。一旦去IP化，就实现了非常彻底的去中心化。而对于美国那套传媒系统，这是颠覆性的破坏。

过去，我们都觉得美国人喜欢玩花活儿，但普通人随便浪一下就能红的几率非常小。可是一进TikTok就不一样了。比如今年特别红的“科目三”，原创跳得可帅，但你别指望刷到一条机械舞科目三，再一条舞蹈家科目三，再一条牛哄哄的科目三——不存在。科目三红的时候，上一条是专业班子，下一条就是隔壁王老二，动不动的服务生跳的辩手科目三就在你面前轰炸。对我们来说稀松平常甚至厌倦，但对老美来说，这简直是颠覆性的。它意味着，虽然我是个小瘪三，从小接受快乐教育，地图都认不全，走出小区就分不清东南西北，只能当服务员，这辈子就这样了，但我在TikTok上跟大家一起跳科目三，人人都能看见我，而且刷到我跳科目三的概率跟刷到川普跳科目三差不了多少。这叫“王侯将相宁有种乎”？有没有发现，这种阶级鸿沟式的社会之间产生了一个天然隔绝？这个小小的侧面印证了一件事：TikTok帮助美国人树立的潜意识里的价值观，跟美国官方打上的价值观是完全对立的两套系统。

美国官方打上的思想烙印，就像美国电影里那样：大灾大难来临时，普通人等着钢铁侠、蜘蛛侠、蝙蝠侠降临，等大救星。但TikTok里不一样，大舅出现的频率和隔壁汤姆、杰瑞一样，直接提升了普通人的认知水平、认知档次和自我概念。这是颠覆性的，而且跟个人收入绑定了，更吓人。断人财路如同杀人父母。别人不光从心灵上得到“我原来也行”的享受，还有钞票进账，干些不咸不淡的事就有钱，不需要牛逼哄哄。这对绝大多数接受快乐教育长大的无能普通人而言，是石破天惊的事。所以为什么这次TikTok发大弹窗，叫大家出去干？大量人真的去打电话冲国会，因为触及到了他们的利益——不光是精神利益，还有客观的金钱利益。冲击力特别勇猛。

但即使冲击到这个程度，美国依然通过几十个人的投票强行走过法案。下一步就是参议院、总统签字。最后能不能通过，要看舆论风暴能到什么程度，后面团体分赃能分到什么程度。但至少初步概念已经到这儿了，充分印证了言论自由——扯淡；民主投票——扯淡；人民心声——扯淡。半数美国人群情激愤地反对不好使。总之，这件事的利害关系，中国人看热闹就行了。这个法案不管通过不通过，对我们没有本质影响。如果通过了，TikTok在美国的公司可能会出问题，但它本来也不给我们纳税，看热闹就行了。如果没通过，美国的意识形态会受到颠覆性破坏；如果通过了，美国的形象、社会公共信用、国际信用会受到颠覆性破坏。这玩意儿对我们来说也是喜闻乐见。这一次，虽然不排除有被迫倾向，但TikTok站得还是挺直的。这种情况属于“今亡亦死，举大计亦死，等死，死国可乎？”。
