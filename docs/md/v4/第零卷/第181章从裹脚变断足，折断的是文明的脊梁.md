# 第181章 从裹脚变断足，折断的是文明的脊梁

你那么喜欢古代，你那么喜欢汉族，你是不是想裹小脚？这是我一天到晚都要经历的一些质问。所以裹脚永远都是直播间里长盛不衰的话题。那么今天我们就再一次谈一谈有关于裹脚的话题，这就是为什么会裹脚。

前年的时候，我曾经做过有关于裹脚演变的视频。但是呢，那讲的仅仅只是缠脚的发展史。在这里简单地梳理一下，就是大家心目当中的那个小的、有点吓人、拆开了裹脚布之后脚没有办法看、行走都一摇一摆的——这种断足式的裹脚，并不是裹脚的本体。它是在清代的时候才大面积推开的一种裹脚形式。而其他的时代，裹脚就是一个正常修饰脚型的功能。所以呢，首先弄清楚的就是：裹脚与裹脚是完全两种东西。

在清代以前，裹脚的形式就是把脚缠瘦，有的是缠弯，有的是缠翘。总而言之，它并不伤害骨骼。这一点从墓葬当中可以获得相应的证据——就是女尸的脚骨没有断，骨骼没有坏。意味着无论他缠或者没缠，他的健康没有受到根本性的损坏。事实上，在宋明两代的贵族女子当中，虽然普遍性缠脚，但是他们热爱打马球。因此呢，那个时代的缠脚很明显地与大家想象中的那个很可怕的缠脚不同。

而这种断足式的缠脚，却萌发于号称禁止缠脚的满清时期。因此我们必须弄清楚它到底是怎么出现的，为什么会发生这样奇怪的情形。那如果说要讲原因的话，最初的那种简易版的、朴素版的、正常版的裹脚，它的原因就比较简单。因为在南唐时期，最初的一波流行性的裹脚来自于舞蹈家的创作。舞蹈家把自己的脚缠得尖尖又翘翘的，然后去跳了足尖舞，从此缠足之风开始在女性当中盛行。这种缠脚和芭蕾舞演员的缠足非常相似，它呢，既有功能性，又有美观的属性。这种缠脚是一种有审美意义的缠脚。

但是到了清代，大家所熟知的、脑子里面一想就认为缠脚就是那个样子的——断足式的、残疾式的、破坏式的缠脚，很明显它不可能美。你放到什么地方，当你的脚已经变成一个残障状态的时候，它离美都一定是南辕北辙。那既然没有和过去一样的美感诉求，那么清代的这种断足是从何而来的呢？如果你去典籍当中找，那其实没有人告诉你，没有人去专门跟你说“哎，这个时候我们把脚掰断了到底是为了什么”，没有这个具体的说法。但是呢，我们能够从侧面去进行一些合理性的推论。

第一件事情呢，就是据说满清入关之后，发行了一套规则制度，说这套制度呢，勒令不准缠足，就是禁止缠足令。那么这套禁止缠足令有没有呢？有啊，存在。但是咱们必须得弄清楚它的对象是谁。这一套禁止缠足令发布给哪位的呢？唉，它是发布给满洲女子的。对了，有趣啊，这套禁止缠足令发布给了满洲女子。在皇后令当中发布出来的东西，叫做“凡是缠脚的不准入宫”。你想，满清时期的所谓“入宫”，其实是不包括汉家女子的。汉家女子进去，那是被关进去了啊，不属于入宫的项目。入宫的都是旗人女子。而它限定说缠脚不准入宫，就是在通知旗人女子不得缠脚。所以清代的这个，无论是“禁止缠足、兼职入宫、缠足了父兄要受关联”，全部都是针对旗人女子的啊。

那旗人女子禁止缠脚，意味着什么？意味着当时旗人女子想缠脚啊。亲，如果旗人女子压根不想缠脚的话，你下这种禁令是不是闲得没事干、吃饱了撑的、吃咸了齁的？所以在清初的时候，很明显地出现了一波极大的风潮，就是旗人女子在入关了之后疯了一样地想缠足。你有没有想象当中“哎呀，我那个脚吓死人的残疾了呀”？我的天，来的那些青壮年的女子，一个个的脚37码起步、39码封顶，然后她要把自己弄成这么小的一截截。天哪，吓死人了。你觉得旗人的女子脑子有问题吗？没有。不管他们生在哪里、长在哪里，他们的大脑都是健康的人，智力都是正常的智力。这就意味着，他不可能在自己的脚长到了36、37、38之后，然后想要把自己的脚咔嚓一刀剁断它、掰折它——太可怕了，这不可能是一个正常的选项。这从另一个角度印证了：在清初那个时代，所谓的禁止缠脚，并不是禁止你把脚打折，这件事情是不需要发公告的啊。也就是那个时代的缠足，依然是明代的新月式的缠足——把脚缠瘦、缠翘、缠弯，它不影响健康，但是却可以让人的脚穿鞋子之后非常漂亮。因此很明显地证明了一点：在满清入关之后，满洲人的女子的内部，开始学习汉人女子的时装风范，从而开始学习了时尚漂亮的缠脚。而满洲的统治者认为，你连这都学，那你再换换衣服、再改改发型，我能认出你是谁来吗？没有区别了。我怎么证明我是满清贵胄、我是关外人士、我是高大的满大人呢？于是他就规定了说满洲女子禁止缠脚。所以呢，这个阶段：第一，汉人女子的缠脚方式是健康的缠脚方式；第二，满洲人女子在积极地向汉人女子学习；第三，满洲的统治者禁止满洲女子向汉人女子学习。这是一种隔离法。

好，既然他没有禁止缠足，结果汉人的女子当中普遍性地出现了断脚式的缠足，那怎么发生呢？是不是因为汉家开始搞了父权压迫？想多了啊。那个时代的性别没有对立。为什么没有对立？因为矛盾分主要矛盾与次要矛盾。满清入关了之后，剃发易服，留发不留头，主要的矛盾就是如何应对八旗，大家没有闲工夫去对抗什么性别之争。所以这个时期的断脚，我们得找到它的一些很可能被隐藏了的原因。所以有一部分合理性的推论，其中的一个推论就来自于入关群体入关之后所采取的一些策略和它产生的后果。

当时满人入关的时候，其实他并没有想象到自己可以千秋万代。因为他是以一个极小的体量、以落后的管理方式去管理了一个庞大而且先进的国家，这个国家的人数又极多。那这种情况下，他并没有充分的自信，觉得自己在这里可以长长久久地统治下去。没有自信的时候，他会做什么呢？啊，作为关外的以劫掠为主要工作的人群，他在没有打算长久占据的情况下，他会做的事情就是抢。因为抢到的才是自己的，建设的不一定是谁家的。所以在清初的时候，他从进关开始就开始做了一件事情：抢与分赃。那这个时期呢，他发布了圈地、投充、逃人各大法案。这些东西是干什么的？就是凡是满清马帮圈下来的、跑下来的地都算他的，至于这个地上有没有其他的人不重要，反正是征服之地，我抢了就是我的——这就是圈地。然后呢，唉，你圈了地，他是一个马上人群，他不会种地怎么办呢？就要招人、招奴婢、招奴工进来干活。这就是投充。那同时呢，还会有一个部分，就是他们在关外的时候，就曾经屡次进关劫掠人丁，因为他那里缺人、缺劳动力。劫掠过去的人丁呢，在他那里变成了奴隶。而汉人并不是很喜欢当奴隶的，大概你也不喜欢，对不对？于是呢，在这些人随同入关、临近故乡的时候，就有很多的奴婢撒丫子就跑路了。因此他就发布了一个法案，说凡是满洲八旗群体，家里面有逃奴的，可以肆意地出去追索逃人，这个就是逃人法。

那么在这一种以圈地和分赃以及抓捕奴隶为主要目标的清初的政策当中，它就产生了一种客观性的操作。我去抓逃奴，那我哪知道哪个是我逃的呀？抓到的就是我的嘛。于是呢，他就会想，抓谁就抓谁，抓过来谁就是谁。甚至普通的平民也抓，地主家里面的也抓，只要说你没拦住我抓进来了，那就是我的奴隶。而且在奴隶制的管理之下，一日为奴，终身为奴，世代为奴，不可赎身。这种情形非常的可悲。于是就会出现抵制抓奴的人。谁有权利、谁有资格、谁有可能性抵制抓奴呢？只有地主阶层才有这种希望。于是就会这样的一种行为：我们会注意到，在清初的这种抓奴的操作当中，事实上有很多的汉家地主家庭的女子，直接当作奴婢就拉进了别人家的后院儿里去了。地主家的姑娘保养得好、漂亮，那被抓进去当了奴婢怎么办？你要把他要回来，你得证明这孩子不是你家的，是我家的。但是满人汉人之间，他长得一样啊，你怎么区别？你怎么证明这个院儿里的不是我们八旗家的包衣，一定是你家的姑娘呢？如何印证这一点？那如果说汉家女子的脚只有半截儿，你有本事证明说你满家的女子和我一样，你们也搞半截儿？你要是不能的话，不好意思，半截脚的就是我家的姑娘。这是一种自我保护措施，这是其中的一条合理性的推论。

那么还有第二种合理性的推论，就来自于另外的一项政令。大家都知道剃发易服令啊。当然了，这几年大家天天经受的都是一些洗地的洗礼啊，就说“剃发易服男降女不降，男从女不从，女子依然着旧时衣冠”。别闹了，不可能啊。人家剃发易服令明确地说了“无论男女俱照满洲式样”，剃发衣服是没有放过任何一个女子的。女子也要剃发，女子也要易服，大家都是一模一样的操作。所以在这个过程当中，我们就得知道剃发易服到底都干了些什么。那么不管他政令上面说的啊，是什么样一些具体的条文，但是他在实行的时候，他要把一些具体的实操的口令发布到各个执行的小团队里面去。那么当时他们去执行剃发易服令的这些小团队，所领到的任务口令是什么口令呢？一共有三句话，叫做“留发不留头、留袖不留手”，还有一条叫做“留裙不留脚”。这三条加在一起才叫做完整的剃发易服令。

因为清初的时候，并没有一个真正的满洲制式的服装，没有。又要你改，怎么办呢？他就找出了几个汉家衣冠身上典型的特征。第一个是袖口的结构，与关外的马帮群体的极窄的小袖口不一样，所以他会设定这样的服装制式：就是你但凡袖子宽了、袖子长了，那就要砍手。那你要保住手吗？你就把袖子裁短、把袖子裁细，这种方式就叫做“留袖不留手”。那么请问我们过去的先民，汉人在清代的时候有没有手呢？有的。既然有手，那就说明大家改了袖子。没错，清代的剃发易服令当中，首先调整的就是袖口的大小和袖子的长短。那么自然就说到了第三条，它叫做“留裙不留脚”。裙子是什么？黄帝垂裳而天下治，裙子就是典型的长，它是下衣，这个是汉家衣冠服饰当中非常重要的一个要素，就是下衣下面不能够只穿裤子，你还有一个长裙。那么女子身上所穿的就是在裤子外面要专门系一条裙子，这个是我们的标准穿着方式。那在这种穿着方式里面，关外群体是怎么来的呢？它是一整条的袍子，从上到下，袍子里面直接就是裤子，没有裙子。所以请问当他要给你改变服装的时候，他会做什么？他会去除掉你身上最典型的特征，就是汉家女子的两截穿衣法：上衣下裳。既然他不让你两截穿衣，要让你把裙子卸掉，但是裙子又是我们服装当中不可或缺的一个部分，最后就会产生什么效果呢？留裙不留脚。裙子留下来了吗？尽管清代的马面裙已经变得挺难看，在结构上面也有了一些变化，但是很明显，清代的女子是穿了裙子的。我们留住了手，于是袖子就变了。那么我们留住了裙子，于是脚就变了。所以剃发易服令当中，尽管目前找到的东西没有直接的说明这一点，但是它的执行方案当中很明确地有“留裙不留脚”的这个项目。既然裙子留下来了，那么脚不见了，是不是完全吻合了这一道政令呢？因此，为什么在清代的时候出现了大面积的断足式的裹脚？为什么在清代的断足式的裹脚当中，距离满城越近，裹脚的比例就越高？为什么其他时代的缠足既不破坏骨骼，而且呢，都是社会中上层进行装点用的东西？而到了清代的时候，它跨越了阶层的界限，只要是个人儿，只要还住在靠近满城的城市周边的地区，就会断足？是为了什么？它已经与那些客观的、与生活相关的、与美感相关的、与文化相关的东西都没有了关联。这是来自于满清剃发易服和高压统治之下的残酷的压迫。

你说证据全吗？对，证据不全。但是它的逻辑是对的。为什么在清代的时候脚就断了？身体发肤受之父母不敢毁伤，孝之始也。中国人是不愿意摧毁破坏自己的身体的，因为你的身体上的所有的伤痛都会让父母伤心流泪，这就是大不孝。而让一整个民族在200多年间持续不断地、坚持不懈地每一家从上到下全部都进入到大不孝之中，没有一种外来的强制的力量推行，是不可能实现的。因此为什么会出现断足式的裹脚？这是一个猜想，但这个猜想比什么都可靠。
