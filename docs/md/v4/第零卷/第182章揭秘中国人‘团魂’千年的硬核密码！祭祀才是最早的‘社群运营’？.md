# 第182章 揭秘中国人‘团魂’千年的硬核密码！祭祀才是最早的‘社群运营’？

清明节又要到了，该上香的上香，该烧纸的烧纸，该上坟的上坟，该找姥爷的找姥爷。大家的活儿可都来了啊。在这样的一个日子里头，我们就来聊一聊有关于祭祀的问题。祭祀这个东西，一听起来就古老、神秘、玄幻。“哎呦，我的天哪，你在琢磨些什么事情啊？又搞封建迷信的嘛？”但是呢，当我们去回看我们的历史典籍的时候，就会发现有一句话叫做“国之大事，在祀与戎”。这句话讲的是治国之道，讲的是国君手上有什么东西坚决不能撒手——别的事情可以代理，这个事情不准找代理，亲自干。诶，为什么它会成为国之根本？所以今天我们在讲祭祀的时候，不是在教你如何上香、如何上坟，而是要去理解祭祀这个行为它自己的根本要义到底是什么：人类为什么要发明祭祀这样的一件事情？它的总体的价值是什么？它在今天还有价值吗？啊，它能够跟军事直接并列，这个事情就不得了了啊。要知道军事是啥玩意儿啊？军事是可以搞耕战频道的啊，军事和我们吃饭是同等重要的事情，而祭祀又居然和军事可以并驾齐驱。所以今天我们看祭祀的本质的时候，得看它到底在干什么事情。

那祭祀不是中国人独有的，全世界包括原始部落民都有祭祀的仪式。所以祭祀是干嘛？祭祀就是咱们找一个神仙啊，有的时候找一座山、一条河、一颗石头，甚至一头羊，然后呢，大家在那个地方跳跳舞、唱唱歌，歌颂一下，赞美一下，有的时候是崇拜一下、恐惧一下，然后完成了一套仪轨。诶，这个祭祀就完成了。这是最古老的祭祀的方式。之后呢，随着生产力的进阶，这种祭祀就会开始逐渐地升级换代，换到了大家众所周知的商——一天到晚地祭祀，隔三差五地祭祀，那祭祀的日记本排得一摞加一摞，简直是早中晚餐各有不同的花样祭祀，而且祭祀的内容那叫一个血刺呼啦的啊。所以一想到它血腥祭祀，吓死人啦！那是一个黑暗又蒙昧的时代呀。但是说老实话，商的这个祭祀已经非常非常地升级换代了。所以今天我们来聊祭祀它到底是怎么发展的，首先就要把它的根要揪出来。

祭祀是干什么？它其实是人类社会最重要的一种组织方法。我们说人怎么聚在一起形成一种力量啊？人类原本就是一个群居型的生物，群居型的生物，你要发挥你的能力，发挥你的才干，最重要的是什么？团起来吗？那怎么团起来呢？一堆人闲的没事干，就聚在一起嗑把瓜子儿，然后我们就歃血为盟了？想太宽了啊，太平洋都不得这么宽的。所以实际上我们需要让一群人团起来，有组织、有节奏，一起干大事。咱们得找到一种共同点啊，这个共同点是啥呢？在原始社会的时候，大部分情况下，恐惧就是共同点。因为人们呢，害怕大自然的风雨雷电，害怕山上滚落的石头，害怕那个滔滔的洪流——他什么都害怕，因为随时随地都有可能饿肚子，随时随地有可能失去生命。恐惧就是大家最直接的共同点。然后怎么说？我们好害怕那条河呀，于是呢，共同去祭拜那条河的人，就结成了一个组织。那么我们怎么证明我恐惧那条河、想要供奉那条河呢？诶，我就摆一个盆子，在里面放上水，水里面养一条鱼，然后我们每一个人滴几滴血放进去喂那个鱼，然后再把它放到水里面去。你看，我直接就开始供养这条河了。虽然这个仪式是我们假想出来的，但是对于河水的恐惧是真实的，于是人们就用这样的方式凝结在了一起，来共同治理水患。这就是一种组织的方法，而这种仪轨就叫做祭祀。人类文明的早期祭祀就是从此而来的。于是你会发现，在原始的图腾当中，有人拜火，有人拜太阳，有人拜猛兽，有人拜山川，这些呢，都是来自于对大自然的敬畏，这是原始图腾的崇拜。

后来呢，人们把它拟人化了，就进入到了一个新的时期，就是神明祭祀。神明祭祀这个东西吧，说真的，到现代社会其实还有很多人在坚持。为什么？因为它太像人了，和我们的生活特别容易关联在一起。神明祭祀是什么呢？神明祭祀就是你把一件大自然当中隐秘的事件拟人化，形成一个似乎跟你有点相像的、有人格的角色，然后你去崇拜他，让他拥有一种伟大的能力，然后你去给他供奉。这种方式对人来说，它更容易共情。你想要直接理解山是怎么想的，你确实很难想象；但是你想要理解山神是怎么想的，就觉得这个事儿好像突然之间门槛都降低了呀。所以你瞧，我们就会进入到一个神明祭祀的阶段。那这个阶段我们崇敬同一个神明的人，哎，就团起来了。请问现代社会有没有这样的人？乌泱泱的、一片一片的，一国一国的，甚至一洲一洲的。要知道这个世界上几大宗教当中，有几个非常重要的，其实都是精神的，敬唯一的神。为什么那些比较成熟的宗教会要求说，你敬唯一的真神？唯一这件事情就意味着你专注，你只拜这一个，就意味着所有人都崇敬这一个，你就不容易产生分歧，于是你就可以凝结起更多的人来，组织力就会更强。所以你看我们去搞这些祭祀，搞这些仪式活动是干嘛？就是为了强调我们之间的共同点。所以你会注意到什么？呃，教堂里面去搞礼拜活动啊，每一周都会去做，然后有的时候呢，隔一段时间就搞一波大的啊，搞一波大活动。在中国呢，它一般要发点鸡蛋，在其他的地方呢，有的时候发点小蛋糕什么的。这些东西都是为了招揽你，来参加我家的祭祀活动——虽然它可能不叫祭祀，形式上面略有差别，但本质上是没有区别的。是干嘛？因为你在不在这个组织里面，我哪知道？鬼晓得你在家里头的时候，有没有乖乖地听我的话呀。所以怎么样？我要强化训练。每一周你为了那个小蛋糕，为了那杯免费的饮料，为了那板鸡蛋，你不得不过来给我重复一下我们的仪轨，对不对？祈祷的手势对不对？句型对不对？这就是强调我们之间的共同点，这个行为就是祭祀活动啊。那这就是神明祭祀阶段。中国人有没有呢？嗯，中国人也有啊，哈哈，中国人也有——就是我们也曾经恐惧过大自然，所以呢，我们有水神呐、火神呐、太阳神呐，但是后来在我们的传说当中，他们似乎都不那么强大。发现了吗？为什么会这样？为什么在中国的神明系统当中，这个神明老是显得不够高大上、不够威严，甚至动不动就被人给欺负了呢？对，这就是中国人的祭祀有一些非常本源的特异性。

中国人崇拜了一个东西：我们崇拜祖先。听起来好像是特别微不足道啊，谁还没个爹呢，是不是？谁还没个爷爷呢？这世上所有活着的人不都有祖先吗？啊，因此呢，中国人的这种崇拜感啊，常常显得没什么崇拜感，因为太朴素了，过于接地气，以至于你觉得它不高级。但是我们今天就得弄清楚一点，祖先祭祀这件事情，它的微妙和强大之处。我们先说它的强大之处：你就说你崇拜一个山神啊，我恐惧那一座巍峨的高山呐，于是我拜那个山神。后来有一天你发现你有氧气瓶了，咔咔咔的就奔上去了，你还怎么崇拜这个山神？你都知道山尖尖上那个雪多厚了，拜不了呀，是不是？然后你说，哎，我就崇拜那个唯一的真神，结果突然发生了大瘟疫啊，欧洲的黑死病就这样了啊，发生了大瘟疫。然后神父说，神明说了，你喝这个水就能好，死了1/3。于是大量根本就不认字的老百姓就蒙圈了：不是说好了全知全能吗？啊，你要是说我犯了罪，他把我收走了也就算了，我们家刚出生那个娃，他能犯什么罪？过不过分？为什么男女老少公平公道的都给带走了？心里的困惑就会使他产生信念的动摇。所以神明观其实也是在遇到冲击的时候，咔嚓就很容易崩坏的。技术生产力的进步啊，很快地就会摧毁它的信念。例如说一个优秀的医生一来就说：“你信神，不给你喝药；你不信神了是吧？好，给你药。”然后咔好了——你只要不信神，你就能好，完犊子了，这个神明信念感刷刷的就往地狱里面坠啊。所以这些虚拟的东西在我们生产力进步之后，是非常容易幻境破灭的。

但是中国人这里就不一样了：我不管你生产力进步到什么样的程度，我也不管你是不是念完了博士后，你就是念完了博士后之后，还进了什么科研小组，你已经是全世界最顶流的科学家了，你敢不认你这个文盲的爹？不，不敢。很遗憾，没有办法，你就是再牛，你也得认你爹，因为事实上他就是你爹。这是什么？这叫做唯物主义啊，亲。所以中国人的这一套信仰系统，中国人的敬天法祖系统里面的法祖，是很好玩的一件事情。因为他唯物，因为唯物，所以颠扑不破；因为唯物，所以你推不倒它。你再怎么说，他腐朽了，他封建了，他再怎么封建，他也是我爸爸；你再怎么恨他，他也是我爷爷。我给他早晚三炷香怎么了呢？这就是为什么这种唯物主义的东西是最难以被推翻、最难以被颠覆的。之所以今天的中国人，即使在社会原子化的阶段，还会想起来我们有祖先崇拜，就是因为这事儿它本身是一个客观现象。那么中国人因为祖先崇拜，于是就产生了一种紧密的血缘连接，用这样的方式就给组织起来了。我们都是同宗的，我们500年前是一家啊，我们同一个姓，以前肯定有关系，咱们就格外的亲厚一些。这个是什么？这个就是社会组织关系。所以你瞧，祭祀是闲的没事干吗？不是的。在我们凝结成组织的时候，我们就决定了找出什么样的共同点让人们聚在一起，而中国人找出来的这个共同点非常的唯物：咱有共同的父亲，这个事儿你推翻不了。于是呢，我们就坚持不懈地一直撑到了今天啊。这是我们的系统组织方式能够绵延千年不绝，非常根本的要义就是祖先崇拜这事儿——不管你爱不爱，反正它是真的。

那么当你遇到了这种情况的时候啊，就会思考一个问题：我们去搞这个祖先崇拜、祖先祭祀，拜着拜着，万一忘了怎么办呢？说实话啊，人类忘记祖先太容易了。来，在座的各位开始回想，你的祖父叫什么名字？如果万一你还记得，你的曾祖叫什么名字？你居然还记得是吧？来，你的高祖叫什么名字？记得吗？忘了吧。最后你最终能够搞清楚的就是肯定一个姓啊，剩下的我去回家翻一翻，去查一查，问一问，打听打听。这就是人类的正常传承。你脱离三五代之后，你没有办法牢固地记得你的祖先到底姓甚名谁、生活在哪里、职业是什么，因为他跟你的现实生活已经脱钩了。所以你想要一直记得我们有共同的祖先，这太难了。看上去最唯物、最简单、最朴素，人人都有爹，但是你真的追溯起来就会发现，你必须持之以恒地记一套本子。这是什么？这就是后来的族谱。早些时候呢，也有啊，但是只有贵族才会有。这是什么？这就是祭祀权。祭祀是一种权力的象征。权力是什么？权力是你有能力把这群人组织起来，你有能力去办这个祭祀，你有能力去号召这些人，这才是一个权力。所以在我们的文明体系的早期，实际上只有贵族家庭才有姓氏，只有贵族家庭才有宗祠，只有贵族家庭才有权利进行祭祀。这是啥？这是你有权利建立组织了。平民百姓你干啥？你还要组织起来，你是不是要造反？所以祭祀权这个东西，并不是那么随随便便就会产生的，它需要你有稳定的传续系统，有稳定的文字记载系统，而且呢，在这个过程当中，持之以恒地不停地办仪式，不停地办仪轨。这就是每年清明我们都得烧纸，绝对不能错过啊，不然的话，你忘了怎么办呢？之所以今天我们还有这种传承，是因为在过去的时代里面，我们一代又一代的都把它保留下来了，才能够让它活着生长到现在。所以，当你说到我们都是炎黄子孙的时候，这句话本身就是一个组织方法。

说到这儿你就明白了，为什么我们先秦时期的政治家会说“祀与戎不可以假人”。戎是军事能力，而祀是什么？祀是号召力与组织力。我有权利祭祀，我有权利举祖宗的牌位，我有权利给上天供奉。这意味着凡是要给上天供香的，都要站到我面前来；凡是认这个祖宗的，都要站到我面前来。这就是组织的号召能力。你就说对于一个国家来说，这事儿重不重要吧？让大家都听你的话，相信他应该跟你走，这事儿要不要紧？这都就是国之根本呐。所以祭祀就是组织方法。它呢，会随着时代的变迁而不断地升级换代和改变，但是它不会消失，因为任何一个人群都需要组织起来。只要你需要组织，你就一定会有祭祀，最多你就是把格式改变了，把崇拜的对象改变了，但是组织本身是没有办法变的。所以我们是怎么变的呢？啊，我们早期的时候也有神明的崇拜，甚至现在有一些地方上也有一些神明的崇拜。后来呢，我们进入到了祖先崇拜，到了宋明之后呢，市民化社会，大家的家里面呢，但凡家有薄产，都开始供祖先了，家家户户都祭拜自己的祖先。这些事情呢，就一点一点地扩大化，全面地传承下来了。

而走入到了近现代社会之后，我们面临到了一次新的冲击。因为当时的世界上所谓文明体列强们，他们不记祖宗，于是中国的好多人开始反思自己：我祭祖宗这事是不是错了呀？我比别人差劲，我比别人穷，他又不记祖宗，而我记祖宗，那肯定是我有问题。所以这时候就有一些人呢，开始重蹈覆辙，开始搞起了神明祭祀，开始遗忘自己的祖宗，甚至专门特意地背弃自己的祖先。但是吧，我们的历史是在向前走的，你只要想把人组织起来，你总要选择一种形式啊。当这个祖宗这件事情开始动摇了的时候，我们就选择了新的组织方式。哪怕是今日之共和国，我们不叫它祭祀了，但是我们有没有组织方法？我们是不是在义务的教科书当中，会给大家讲解我们国家的伟大的历史？有没有讲解我们国家的政治纲领与革命的目标？我们在追求一个共产主义社会，我们正在经历社会主义社会的建设，我们应该如何做，我们应该坚持怎样的理想和价值观？这些都是我们读书的时候反复强调的吧。同样的啊，凡是在组织内部接受调度管理的人，是不是每个月都要去开开会，要去交一个心得报告啊，是不是要表示一下“我现在依然在听党的指挥”？这是在干什么？这个方式和方法，和我们古典时期的早晚三炷香啊，初一、十五要进香，在原理上面是一样的，就是重复地强调我们之间的共识。只是共识从原来的拜祖先，变成了现在我们追求伟大的理想。它依然是一种组织的方法，只是因为生产力和思想的迭代，于是组织的方法也升级换代了。而从原来的那种上香啊、祷告，转换成现在的上课、开会，它终归最终的目标是让人团起来，拥有共同的意志，在有命令的时候能够站起身来，我听令。这个有没有成效呢？前几年的时候，大家都看到了旗子举出来，就有人说：“我来我来，我一起来。”这个就是组织力。当我们发现组织力这个东西从远古到今天，一直是人类孜孜不倦地去追求的东西，我们也一直努力地运用最先进的方式来诠释我们的共同点和共同的理想和目标，然后我们也用坚决的、重复的仪式去强化我们之间的共同点。

那么祭祀到底应该贡献些什么东西啊？这就是经常被人批判的：中国人的血腥祭祀啊、迷信呐、啊、浪费呀，这都是经常被批判的东西。这就是祭祀的内容啊。如何供奉这个事儿吧，它其实也有延续的一些基本的理论纲要啊。实际上呢，就是祭祀的祭品、贡品，这些东西在全世界范围内，它是有一个共性的。共性是啥？就是所有的祭祀，所有的崇拜贡品都是“请客加送礼”，就这两样。全世界范围内啊，这不是中国独有的。所有的祭祀，一个是请客，一个是送礼。大家来观摩一下啊：我们在远古的各种各样的部落民的部落群体当中，他崇拜大自然，他觉得和想要一条鱼，他就给他一条鱼；他觉得和想要鲜血，他就给他鲜血。这是什么？请客、送礼、吃饭。我们在先秦的故事当中，有河伯娶亲的故事，这当然是民间的淫祀——淫祀也是祀啊，也是一种祭祀方法。那河伯娶亲提出来的要求是什么？河里的神仙想要娶媳妇儿，你要给我少女。这是干什么？请客、送礼、吃饭。残酷吧？对，请客、送礼、吃饭，原则上面是一样的。包括说你嘲讽的商代的血腥祭祀，但是在那个时代，他的这个血腥祭祀当中的，无论是猪、牛、羊、狗，这是什么？请客、送礼、吃饭。然后你说，哎呀，他居然还供奉一些各种部落的人头，唉，吓死人了，他居然杀人。商代是一个什么时代？商代是一个各个部落之间还没有完全达成一统的时代啊，周边的一些部落群经常会发出侵扰，经常会互相斗殴，这样的一个博弈期。这种时候那些所谓的啊，异族，那些遥远的邦国，那些有可能侵犯我们的群体，那是啥？那是敌军呐。在那个时代，对于商人来说，那些迁徙型的、打劫型的部落，那叫做敌军。我就问你，到了武松打虎的时候，他去给他的大哥上坟，他供了什么？是不是供了潘金莲上去？为什么要供潘金莲上去？因为潘金莲害死了他大哥，他要把他大哥的仇人给他送过去。原理上面一脉相承。所以先秦时候为什么要搞战俘的祭祀？那是把你的仇家送到你的祖宗那里去，告诉你我打赢了。这种行为在后世有没有？我就给你假设这样的一个场景：你家在某一个山脚底下有一片很好的肥沃的土地，结果有一天被野猪给霍霍了，野猪就住上头了。然后呢，你的老父亲因为损失了这片地啊，一口气没上来，人就没了，驾鹤西去。那你最后应该如何告慰你的老父亲？把野猪群咔咔一锅端，把猪头供到坟前头。这个叫什么？家祭无忘告乃翁。所以祭祀的原理是什么？满足祖先的需求：请客、送礼、吃饭，就这几件套。所以你为什么先秦时期的祭祀那么血腥？因为在那个时代，我请你吃肉，这就是最豪奢的宴席；我请你吃敌军的头颅，这就是最高档的供奉啊。绝顶高档的供奉是干啥？是把我自个儿送过去。这种时候是干什么？我有求于祖先，我极大地有求于祖先，这个时候我会把自己送到祖先那里去，亲自跟他聊一聊。这就是请客、送礼、吃饭，然后顶级的牺牲就叫做我自个儿送货上门。原理上面高度一致。

而且呢，今天的中国人其实也秉承着这个原理。之前有那个我们抗战期间的娃娃兵，他们有自己的陵园，有自己的雕塑，你会发现在这些娃娃兵的雕塑的口袋里面、包包里面，塞满了糖果。这是什么？这是今天的后人给他们的祭祀供奉品。为什么要给这个？因为那些可怜的孩子，在那个苦难的时代里面，他需要这个却没有办法获得，于是我把这个拿给他。我们去供奉先烈的时候，应该给他什么？他喜欢酒就给他酒，他喜欢烟就给他烟，他想要吃饱一顿饭，我就给他米饭，他想要看今天的风和日丽，我就给他一幅地图，给他一幅照片，我就让他看一看我们今天的军机和军舰，就满足了他的心愿。这个就是祭祀，这个就是贡品，这就是我们组织力的根本来源。所以你说，哎呀，这个东西过时啦，找什么姥爷呀，供什么祖先呀，什么玩意儿！就早晚三炷香呀，什么东西就上一碗米饭呀。事死如事生，你面对任何的先人，都视同他依然健在，视同他的理想与梦依然存在，我们依然踏着他的脚步继续前行。不忘来路，不忘初心，方得始终。中国人就应该秉持着这样的原则。我们从远古时期的呼号开始，一直走到了今天。我们不再搞那些花哨的东西了，但是我们的心还是没有变。为什么？因为不能遗忘。因为忘记了我们不知道从何而来，因为忘记了我们就不知道为什么要站在一起。这叫做什么？国之大事，在祀与戎。
