# 第168章 六畜与萌宠的差距：至爱亲朋，得加钱

前两天的时候，我吃了这么一个不咸不淡的小瓜。就是在我们的热搜榜上面，出现了一个很奇怪的热搜，说某某人以前有刑事案底。哎呦，这厉害了啊！这是什么事呢？我一瞧，哦，原来有一个网红开始遭到了网络围剿，说他有刑事案底，不是个好人啊。哦，他不是好人。然后呢，于是我就去看了一下，这网红是干啥的。原来是一个训狗师。训狗师以前多年以前曾经有过刑事案底，这事儿耽误他训狗吗？所以这两件事情是怎么关联起来的？我就开始纳闷。所以我就说，我去吃一下这口瓜，到底是啥意思啊？然后发现，哦，原来事情的源头呢，起源于一个很小很小的事件，就是这个训狗师呢，给狗洗澡的时候，那狗死了。按说这最多是一个财产纠纷的事件，但是很快地引发了舆论的声浪，就说这个训狗师不是好人，他虐狗啊，他对狗不好，他的训狗方法是错的，是一个暴力的训狗法啊。我们应该像教育孩童一样的，用正向训狗的方式，去把狗养育成我们心目当中的那个可爱的样子，而不是用你犯了错误我就开始揍的这种暴力方式啊。这个方法是错的，不科学的。好，那个训狗师的方法不科学，OK，我接到你的信号了。请问训狗师的训狗方法不科学，这件事情和他以前曾经有过违法犯罪史，这件事情有啥关系吗？啊，没关系啊。没有谁说过说训狗师这个执照，甚至好像都不需要执照，其实就可以训狗。那么也就是说，当训狗师这件事情，和他以前曾经有过案底，这件事情毛关系都没有，居然能够挂在热搜的榜一上面啊。那这个背后就一定有戏了。

咱们要知道啊，热搜榜一不便宜，一个东西当它很贵。这么贵的东西就是为了给某一个网红泼脏水，然后呢，这个泼的这盆脏水吧，还跟他犯的所谓的错误没有半毛关系的时候，这就意味着，这不好意思了啊，那一定是这个网红触及到了某一种利益的关键点。所以今天我们就来聊聊这个瓜背后的故事。它到底触及到了什么利益呢？我就去瞧了一下这网红。他训狗吗？训狗当然，无论是宠物也好，牲畜也好，它在产业链条里头，他自己就是吃这口饭的，他还能砸这个锅不成？我就去看了一下，发现他真的砸锅了。怎么砸的这个锅？就是他的这个训狗场吧，他搞的这个项目呢，就是专门接别人家不要的那种恶犬，什么护食的、咬人的、有进攻性的、拆家拆不停的啊，这种摁不下去的恶犬就会送到他那儿去。送过去之后呢，他的训练方式呢，也是很暴躁的啊，就是你护食是吧？你冲我呲牙？咔一顿揍啊，有的时候揍的简直都见血。然后还护食？还不听话是吧？再一顿揍。揍乖了呢，我就把这个狗还给你。好，这狗训好了啊。如果呢，揍了千奇百怪的了，它还是不乖不好使，那行吧，我训狗失败，我给你一条小狗，你重新去培养啊。所以他的这个方式呢，就是反正最后我会还你一条狗，是不是原来的那只我不管啊，我会还你一条听话的、不咬人的狗。大概就是这么一个操作的流程。

那这件事情本身，它触及到了什么样的根本利益？在这事上面呢，我们就得聊一聊有关于我们身边的人类的共生、伴生动物的问题。要知道，我们去驯化动物这个事儿，从我们的史前时代就开始了。哎呀，我这个爱讲自古以来的啊，爱讲老八股的人，这一趟一下子一竿子戳到了原始社会去。我们考虑一下，我们当年去驯化狗是为了干啥？打猎嘛，看家嘛。你一瞧这个狼居然夹起尾巴，听我的话了，是不是开心的不得了？把家里面剩下的骨头给它啃，它就可以帮我去打猎，唉，给我维持治安，我这赚大发了呀。于是呢，原始的人类就开始让狗和我们一起生活了。那么猫其实也差不多的。就是你看我们现在终于进入农业社会了，我们积攒了粮食，但是粮食的克星是谁？小老鼠嘛。我攒下了这么多的粮食，等到吃的时候发现一半儿没了，是不是亏大了？所以呢，这个时候我发现猫咪可以吃老鼠，于是呢，我每天吃饭的时候就给它剩小半碗，让它留在我的家里面，顺带着帮我捉老鼠。这个就是我们过去对于动物最基本的一个关系啊，一个呢，是合作的伙伴，一个呢，就是牲畜。

所以呢，当我们对他有这种工作岗位的需求的时候，对于他的态度就是：我家里剩一口，我就拿这个东西勾搭着你，然后呢，你再去干别的好活的时候呢，我再给你发点奖金。这个呢，其实就是一个恶毒老板和那个悲催的打工仔之间不可不说的故事。按人类社会与这些伴生动物之间的原始感情，就是从这里而来的。当然，在这个过程当中，有一部分的雇主和他的雇员之间建立了非常深厚的情谊，但是这是完全个体化的事件啊，他就特别特别爱他们家那个，可能呢，彼此之间还曾经有过救命之恩，但是那是个例。整体上来讲，它是这种合作伙伴和雇员的关系。后来啊，随着我们的经济、文化的水平的昌明，我们进入到了真正的文明时代，开始出现了王权，然后开始出现了贵族。那这个时候时代呢，就发生了变化，这个产业链条也发生了变化。以前的狗是拿来看门的，不会看门儿或者汪汪乱叫，或者咬了家里人怎么办呢？哎呀，吃肉嘛，狗也是肉啊，甚至还有专门的屠狗啊。仗义每多屠狗辈啊，人家都没说屠猪，你看屠狗是一个正经行当啊。所以到了这个时候，唉，那么这些我们的伴生动物是怎么开始慢慢升咖的呢？啊，这就是我们的阶级社会开始出现了大量冗余资源之后，就出了这个事儿了。为啥呢？就是吃饱了撑的啊，饭太多了没处放了，肉太多了容易馊了，钱太多了没处花了。然后这些贵族们就干了一些事情啊。

我们要知道，贵族是很注重一件事情的，他要与其他的人产生阶级隔离啊，他要有明确的区分度。例如说你种田的人长得黑秋秋的，那么贵族呢，就一定要很白净啊；那么种田的人穿的破衣烂衫的，那么贵族就一定要穿的非常的华丽和宏大；同样的，如果说平民家里面也有一条狗，但是贵族家里面也有一条狗，这两条狗都是吃剩饭的，还都是阿黄，请问我们之间有什么区别吗？这就不对，这就错了。我们怎么整？我们就要给贵族家里面的狗狗猫猫升咖。那么如何升咖呢？唉，在唐朝的时候啊，有一个东西。大家记得我们在街面上行走的时候，如果看到了那些带什么小人儿的那种车标啊，小翅膀的车标，是不是不能够往下掰？掰掉了之后是不是亏得很大啊？你掰一个小纪念品回来，要给人家赔一台普通的车回去。这个东西叫做车标、logo，高级的logo，价格不菲。那么在唐代的时候，有一个东西，就是在唐代的坐骑啊，那个马脸前头有个牌牌，然后这个牌牌呢，上面会印上和刻上各种各样的方式啊，使用上这个家族的族徽、标志、logo、符号，啥玩意都有，有的呢是银的，有的是黄金的啊，他愿意做啥形状做啥形状，愿意做啥款式做啥款式。从此你那个农夫家里头拿来拉车的那匹老马，和我贵族家里面拿出去、出门浪、拿去泡妞的这种高头大马，就有了本质的区别，因为他有了马标。区分度就在这个时候出现了啊。那么在狗在猫身上也是以此类比的。

于是呢，当我们出现了这些吃饱了撑的啊，穿暖了之后热得都炸汗的贵族的时候，他们呢，就把自己身上的绫罗绸缎也好呀，那些机灵巧购的这些玩具、玩偶也好呀，啊，那些精舍啊，那些美食、那些佳肴们，甚至一些专用的奴仆，都给他们家的阿猫阿狗准备上了。所以今天的人自称猫奴、狗奴，那个时候啊，是真的有猫奴、狗奴的，就叫做抱猫的那个丫头，抱狗的那个丫头啊。一个猫一个狗，嗯，平常的那些小贵族家里头可能配一个奴婢，要人家豪门，我那个顶流的狗是不是叫什么？唉，我得给他配三五个奴才专门伺候他。这种时候呢，唉，他就直接实现了一个产业的升咖，同时呢，也就带动了整体的这个行业的发展。那么在这种情况下面，我就会发现啊，牲畜和萌宠看上去是同种，长得也大差不差，但是它们之间是完全不同的两种东西，而且呢，能够带来的产业、能够带来的身份、能够带来的精神感受是截然不同的。所以印证到我们现代社会，其实呢，也就出现了相同的问题啊。我们在现代社会里面，猫狗的地位是如何发生转变的？要知道过去的很多年里面，贵人养的狗啊，那和那些什么普通平民百姓家的看家狗，就是永远都不一样的啊。但是当我们解放了之后，我们有一个要求，就是大家人人都是平等的，而且劳动最宝贵。因此这个时候你说我在家里面养个狗，专门捋毛的，这叫什么？这是不是资产阶级复辟、封建主义复辟呀？你是不是又想当贵族了？那个粮食应该优先给人吃，你把你们家的那个狗喂得比别人家的孩子都肥，你什么意思呀？所以在这个时代里头，如果谁家里面去给狗买了十几二十件衣裳，那意味着你们家犯了政治性的错误。所以这个时代呢，它是没有什么所谓的那个宠物产业的啊，这个行当起不来，因为整体上来讲，狗啊、猫啊什么的，他就是有自己的职业属性，回归到它原来六畜兴旺的牲畜的这个岗位上了。

但是这个局面呢，在上世纪90年代左右开始发生了转变。就是我们又开始出现了这样的一个，嗯，自由商贸之后，它自然产生出来的这种，嗯，产业链条的变化。有一部分人呢，就把这个牲畜产业当中的高端部分，转化成了这个宠物产业啊。因为这个时候我们不再批判这种伺候自家猫狗的行为了，于是呢，它自然就把这个产业呢，开始逐渐地升级。利益会驱动着人们不断地为产品推陈出新，最后这个产业呢，就一点一点地扩大开了。那么产业是出现了，可是这个产业只要出现了，并且想要印证自己是一个朝阳产业，它就需要向大家展示一个东西，就是我这个行当啊，是不断扩大的趋势，这个玩意儿非常重要。在商业领域里面，你觉得这个行当很好，是一个朝阳的、蓬勃的、向上的，是不断扩散的，那么你就会注资投入进来；如果觉得唉，不行了，有人撤资，你也会卷款跑路。所以呢，他一定要给你缔造一种感受，这个行当是在越来越庞大的。那怎么能够让这个产业变得庞大呢？上世纪90年代的时候，它开始的第一个扩展运动，它是从哪儿来的呢？就是从城市养狗的这个问题里面来的。要知道，过去为了保证人类的生存安全和人类的生活环境，实际上呢，在上世纪绝大多数的时间里面，我们是禁止城市养狗的，有专门的条例规定了这个东西。但是到上世纪90年代的时候，这个东西在各个地区开始逐渐地被打破了啊，城市里面也开始逐渐地允许你养狗。既然说城市允许养狗了，这意味着什么？这意味着你整个的产业链条当中，扩大了一大片的受众。于是这是第一波的产业扩张。

产业扩张了之后啊，你说我再接着继续地让更多的人去养狗，更多的人去养猫，这个行吗？其实扩散到一定的程度的时候，它就不可再扩张了。要知道，虽然确实有很多的人喜欢猫猫狗狗，但是也同样的有很多人不喜欢猫猫狗狗，甚至厌恶猫猫狗狗，这都是客观存在的。所以这个方向上的扩张啊，它是有极限的。到了极限之后就没有办法再成长了。可是产业必须要成长，它才能够催动大家一起投入进来，才能够让大家赚更多。于是呢，它就必须在这种客观的数量很难增长的情况下，提高它的价格。那么怎么提高一个产品的价格？养狗嘛，你去隔壁家抱个狗崽儿回来嘛，这么大点个事情能够扩展到什么程度呢？所以你瞧，是不是后来他们就开始不断地推广和吹嘘品种狗？大家都听过这些东西吗？哪一种狗特别适合看门儿，哪一种狗特别适合跟小朋友往来，哪一种狗特别的听话，哪一种狗特别的小巧玲珑和可爱。无论是电影里面，电视剧里面还是杂志里面去讲的小故事，这个时候开始给大家宣传狗的品种。然后呢，就会滋生出来一个新的产业，就是育种产业。那这个育种产业，当他去培育了一些新的、稳定的品种的时候，是不是这个产业量级就提高了？你以前一个阿黄啊，你找别人50块钱就要了一条回来，但是因为我现在的这个啊，叫做阿柴了，它是一个高级的进口的纯种狗，它叫做阿柴，于是你给我2000嘛，限购，2000块还要限购。请问这个产业升级了多少啊？这呢，就是在一个品种的行当当中升级。但是这个玩意儿嘛，你说这个品种我们可以无限度地不停地到处都是新品种吗？它也有极限。这个玩意儿成了极限之后，就会在其他的地方再扩张。怎么扩张呢？他就会告诉你一个新的故事啊，这也是洋人告诉我们，狗是人类最好的朋友。大家开始听到这样的故事了。所以呢，他就会拍很多很多的电影、电视剧，然后里面的狗狗呢，跟人之间的那个甜蜜的互动，甚至呢，在危难的时刻的帮扶，然后就让你产生共情和代入的心理，最后呢，就觉得，哎呀，看着狗狗我就觉得他太好了，我得对得起他。虽然他其实根本没帮过你，但是因为电视剧里头有一个狗帮了它的主人，于是呢，你就感觉到我得对它加倍地好才行。

那这种感情的投入，就会使得大家对于他的那个投资量，直接指数级地提高。以前啊，我们说剩一碗饭，泡一点肉汤，那个狗狗就要冲你摇尾巴，但是现在呢，啊，因为他已经是你的好朋友了，你怎么能够给他吃这个呢？什么整鸡的腿儿是不是就该给上了呀？啊，然后呢，整块的牛肉是不是应该剁不剁给它喂上了呀？啊，那么到狗粮的时候，以前给狗粮的时候五块钱一斤，后来你说他是我的朋友了，那么这个狗粮50块钱一斤，你也得买，朋友嘛，应该应分的。所以升级换代就是在这个过程当中，一点一点地根植进了我们的心里头。然后这几年还有一个另外的产业，就是所谓的兽医产业。那么这个兽医产业也是自古就有啊，因为人们想要豢养牲畜，为了维持自己家的财产嘛，我也得让它健康起来，生病了我得治，实在治不了了，拉了吃肉啊。但是现在不一样了啊，现在我们来看一下啊，正常的一个兽医站，给一个小型的动物去做一个阉割手术，需要50块钱，这是他的药品钱和手术费啊。但是如果你去你们家楼下的专业的宠物医院的话，噶一个蛋啊，保底500吧，甚至贵的呢，可能要1000。那中间的这个价差是什么？这个就是超额的利润预期。所以现在互联网上还有很多很多这样的视频啊，就是你看我们家的猫又捣乱了，我骂不骂他呢？我打不打他呢？然后呢，他就给你打一个字幕在上面：妈妈对我凶了，我明天要生一个几千块的小病。有看到这样可爱的视频吗？看这个视频的时候，你想到的是什么啊？这个猫猫好可爱，心里头有好多的内心戏，他像一个孩子一样，像一个我的小朋友一样的。但是他给你根植的另外的一个信息，就是我给猫猫看病的时候，花几千块钱很正常。这是在干什么？这是在提升产业价值。

所以大家就会发现啊，当我们试图提升某一个产业的产业价值和它的产业利润率的时候，实际上是要为这个产业包装上一个特定的形象的。那么这个形象是什么呢？啊，这个形象就是萌宠，是我的挚爱亲朋。要知道这种行为，人类呢，打从有了贵族群体之后，实际上就有了的啊。当年贵人们手里面的那个有奴仆，有拥抱的阿猫阿狗，其实就是他们的挚爱亲朋，他低于他的主家，但是高于其他所有的平民。可是呢，这种行为在历史上面一直都是备受批判的，因为它叫做率兽食人。在现代社会当中，我们说人和人都是平等的，于是他就不能这么干了。因此我们今天的这个产业，他想要提升宠物地位，就只剩下了一种选项，他没有方法去贬低其他人的地位的时候，他就只能去提升宠物本身的地位。于是你瞧他们天天地在互联网上去攻击那些降低宠物咖位的人，然后努力的呢，哎，让这个萌宠、挚爱、亲朋、手足同胞，他是我的好朋友，我离不开他。甚至在很多的热搜，或者是一些灾害的那种场景下面，会出现这样的网络推断，就是如果发生了灾害，在我的猫和一个不认识的小孩儿之间，我应该先救谁？他甚至会在视频当中鼓励你去救那个你自己的猫。你瞧这是什么？这其实就是在运用媒体的手段，传媒的手段，帮大家在脑子里面打下一个烙印：我的猫、我的狗是高过于我不认识的人的。这是一个错误的想法，但是呢，他用这种看上去无害的、很温情的方式刻到了你的脑子里面。然后他就把某一些动物提升了档次。

那么当整个的产业都依附于这种人类的情感的时候，它就会产生一个反面，就是什么会破坏这个产业的这种形象呢？这个产业建立在猫狗是人类挚爱亲朋的前提之下，于是任何人戳破了这个泡泡，谁就断送了他们的饭碗。那么这个时候我们就要说回来了啊，这个网红为什么会在训狗这件事情上面触及到那么多人的利益，以至于被发动网军进行围剿呢？就是因为在他的训狗场里面，狗就是牲畜，训不出来了，就可以拿了吃肉。这是对牲畜、对食材的态度，而不是对挚爱亲朋的态度。如果说这样的人可以成为大网红、正向的，如果说那些狗肉厂啊、狗肉养殖场能够成为正向的、推广的、打广告的，如果说那些狗肉馆儿啊，炖锅的人能够成为正向的、推广的、打广告的，那么阿猫阿狗们就会在人们的心目当中成为牲畜和食材。而对于牲畜和食材而言的话，你不会有那么庞大的产业链条给它配给，因为你不会给你们家的猪带两块智能手表，但是你可以给你的兄弟狗狗带两块智能手表，就是这样的一种区别。所以它根本性地触及到了宠物产业的链条上面的根本利益。

那么在这个领域里面，你说我怎么看？我哪知道我怎么看呀？因为我家里面也有小宠物，哎，我也觉得他跟我很好呀。但是呢，无论我有多喜欢他，在他和一个陌生的人面前，我永远选择那个人，除非那个是我的仇人，我也许会犹豫一下，但是犹豫之后的结果，很有可能我连仇人都是要救的，但是那只猫永远都排在人类的后面。所以你说我应该如何去看待这种产业的问题？我觉得如果说我们真的支持经济上面的自由，舆论上面的自由，产业上面的自由，那么爱猫爱狗的人可以把它视为自己的挚爱亲朋，但是不能够强迫其他的人也视你的猫狗为挚爱亲朋。同样的，把猫狗视为食材和牲畜的人，也有权利去展示它们，训练猫狗去工作，以及将它们用作食材去烹饪的过程。大家都能够在自己的领域里发言，这个效果才是一个自由市场最后得出来的成效，这才叫做大家所吹嘘的啊，用脚投票，用嘴投票，用手来投票，这才是正常的一个环境氛围。而不是说出现了谁表现出啊，其实它就是牲畜，然后立刻冲上来大规模的网军围剿，不行，这个人是个坏人，他以前是有黑料的，为什么他昨天晚上啊，吃夜宵没给钱？当你用这种方式去攻击一个人的时候，意味着恐惧。当你恐惧的时候，意味着这不是是非问题，而是利益问题。当它变成了利益问题的时候，这意味着你已经选定了一个板凳，你的屁股已经坐稳了，那这个时候你做出来的判断，你说它是客观的，是正常的，是人性的，NO，不可能了。所以吃瓜怎么吃？就硬吃。但是如果说我们的互联网上真的可以有各种各样的说法，各种各样的立场，我相信它还是会更好的。有碰撞才有火花吗。
