# 第114章 汉唐在日本？

我曾经在视频和直播里都聊到过一个话题，就是和服。凡是聊到和服相关的问题时，通常都会遇到这样一套话术，很有意思，跟大家分享一下。他会说：“你这种反和服的心态吧，就是狭隘的民族主义，就是盲目排外的精神。因为你看，我们的爱国人士鲁迅先生和秋瑾女士，当年都曾经非常喜欢和服，专门穿和服拍登记照片，也非常喜欢日本文化。所以实际上，我们完全应该把日本文化和日本的军国主义分割开来。文化的东西呢，跟我们之间有千丝万缕的关系，毕竟汉唐在日本嘛。”这一套大家应该都听过。所以今天呢，我觉得可以来聊一聊，到底为什么会有人相信汉唐在日本，为什么鲁迅先生、秋瑾女士都曾非常喜欢和服，并且穿着和服去拍郑重其事的登记照片。

说到这些事情的时候，我们得先聊一聊衣服这件事到底有多重要。前两天还听人说：“衣服不就是一件衣服吗？实质才是最要紧的。你天天纠缠着衣服，我觉得那些人就是吃饱了撑的。”嗯，这种想法也有合理性，叫做“文胜质还是质胜文”的讨论。这种东西从来都有人聊过。但今天当我们单独讲衣服的时候，它不光是“质”是否必须要胜于“文”，或者“文”是否很重要这样一件事，更多的是服装本身——服装到底是什么？要知道，我们从来都有一句话叫“黄帝垂裳治天下”。这个黄帝呢，是更加远古的君王。“黄帝垂裳治天下”，为什么一件衣裳跟治天下之间会有关联？这里就表现出衣服的重要性来了。

服装这个东西，如果说它最早的作用是保暖和遮蔽的话，那么到了人类真正进入文明社会，有秩序、有管理、有规则的情况下，服装其实就是拿来区分你我、展示文化独特性的一种存在。你穿什么衣服，不光是衣服本身好不好看，更重要的是代表了你是谁、你站在哪一边、要按照什么样的方式生活。它具象到我们的生活里面，细节当中：你穿T恤衫的时候可以随便遛弯儿，但一定要正装出席才能签署正式的合约；穿泳装的时候，你总应该站在水池子旁边；穿运动服的时候，倒是可以待在运动场旁边去奔跑。所以服装和身份、和人的定位之间，其实一直都有联系。

那么“垂裳治天下”所讲的就是：天下子民当中，我的这一批人应该穿什么衣服，这种衣服又代表了什么样的文化内核。所以我们才会观察到，全世界所有自主走进文明体系当中的族群，都会有自己的传统服饰。这个服饰有的很华丽、很漂亮，表现了它的生产力和它曾经拥有的一些审美水平、技术水平；也有一些群体的衣服非常草率，甚至看上去有点粗陋，但即便如此，他也会保有自己独有的特点和外观。这是因为哪怕是一件很草率的衣服，它也彰显出了我与他的不同，这是在回答一个问题：我是谁。而所有的人，无论作为个体还是作为群体，在做任何事情的时候，第一大前提就是“我是谁”。服装所回答的就是“我是谁”“我们是谁”这个巨大的社会问题，当然它也是哲学问题。有些哲学家一提到这个问题就“嘎嘣”一下神经断了，这种事也会发生。但所有的人，第一问题必须答清楚“我是谁”。

那么“我是谁”呢？民国时期的革命先驱面临了非常艰巨的问题，就是这个“我是谁”，我应该是什么样子。这里头牵扯到一个更加久远的话题，叫做“剃发易服”。我们都知道，清代的时候推行了剃发易服。有些人拿它轻描淡写，有些人视之为理所当然，还有些人认为百姓自己都接受了——革命军进城的时候，要大家去剃辫子，大家哭着喊着死也不剃，“祖宗之法不可变”。所以服装是怎样的，有什么关系吗？清代的服装也是我们的服装。但在那个时代的百姓还是学者，统治阶级还是被统治阶级，其实大家心目当中有一个更加深刻的共识：清代的这些服装体系，它本身与中国漫长数千年延续的服装体系并不兼容。剃发易服真的不是一个普普通通的换衣裳而已。这件事情当我们去溯源的时候，其实在清代的中期就已经表现出非常明显的特点了。

清朝中期的时候，朝鲜作为属国，派了使臣来觐见。而这边接待他的人是以几个汉臣。这些臣子在遇见了朝鲜人的时候，感觉到了“若故人来”。这种感受非常微妙：他看看自己的衣裳，再看看对面的衣裳，突然觉得对面的人才是和自己的祖先关联更深的一些人。他为什么会有这种感受呢？要知道，中国这个文化群体能够延续至今，搞了几千年、上万年不停，是因为我们喜欢写日记。大家可以去看一下，我曾经还专门做了这个视频。我们做日记这件事里头，就有一些很微妙的习惯：就是老记档案，吃了什么、穿了什么、做了什么、在哪里发生了什么事情，这就是我们过去的档案。这个档案有国家级的、地方级的，其实也有家庭级的，就是我们都知道的那些宗祠、族谱，对吧？那么宗祠里面除了摆放族谱之外，还会摆放一些很重要的、族群当中的人物的画像，这也代表了家族的荣耀和传承。这就相当于我们在自己家里面摆放了老祖父、老祖母的照片一样。那么你能想象得到吗？你的老祖父、老祖母和你长得一毛钱关系都没有，然后你看了他之后还觉得好亲切哦，这是我的祖父诶——抱歉，你是做不到的。而清代的人面临的就是这种局面：他的祠堂里面摆放的是衣冠楚楚的祖先，但是呢，他看到自己身上的服装，看到自己头上的头发之后，就会觉得自己和祖先之间产生了剧烈的割裂感。祖先在哪里？祖先的衣冠在哪里？在那个时候的人的心目当中变得越来越恍惚、越来越遥远。如果说没有朝鲜的使臣站到他的面前，他很可能察觉不到，但是当有一个使用故国衣冠的人真的出现在他面前的时候，这种潸然泪下的感受就猛然之间出现了。

如果说清代的时候，我们是在面临朝鲜的使臣时产生了这种“若故人”的感受，那么在民国时期的很多中国社会中上层人士心里，他们就是在看到日本人的传统服装的时候，感受到了“若故人来”。那个时候的日本人形象很好吗？说实话，实际上日本人当时属于列强，所以我们的革命志士、我们的先驱都很清楚这一点——他们对我们没安好心。而且呢，当时的日本人长得也一点都不好看，和你们今天所看到的所有的影视作品完全不一样。那个时候的日本人平均身高非常矮，那种特别特别小的个子，然后那种侵略者身上的志得意满、张扬的样子，是不可能真的在个人魅力上面吸引到中国人的。吸引到我们的，远非他的个人品性或者外貌，而是他们身上那些“若故人”的服装。所以这就是为什么鲁迅也好，秋瑾也好，他们都很喜欢和服，也喜欢日本人的仪态、喜欢他们的器皿、喜欢他们的建筑、喜欢他们的文化——为的就是“交领右衽”，为的就是“若故人来”。他们想要在这些更加遥远的群体身上找到自己的根由，然后去回答一个问题：我是谁？我为什么而战？我是一群什么样的人的结合体？我们聚起来之后应该是什么模样？我们战斗之后要建设一个什么样的国家？这些问题，一切问题的前提都在于“我是谁”。而这个问题，文人们在日本人身上去寻找答案，那就叫做“汉唐在日本”。

这句话在今天的很多人的嘴里也存在。有一些人动不动就说“汉唐在日本”，我要跟他学这学那，“学过来的其实都是祖先的东西”。在今天我们详细分析之后，你知道他们是很荒谬的，他们在搞文化置换。但是在那个时代，那个时代的先驱也认为汉唐在日本，但他们是真诚的、是爱国的，并且是真的想要在这些海外遗存的残迹当中，去找到自己祖先的脚印，然后去把它接续上来，找到本我，然后用这种方式拉起一个队伍来，去跟列强相抗争。所以当时的“汉唐在日本”是拥有革命属性的，是拥有爱国属性的，也是有民族主义属性的。这玩意儿真的是随着岁月的变迁，它总是会不停地变，有的时候是这样，有的时候就是那样。

好，那么就进到第二个问题了：这种“汉唐在日本”、这种到日本去追溯文化的行径，是不是一种资产阶级的行径？要知道，无论是鲁迅也好，秋瑾也好，尽管他们是革命者，但他们同时也是小有产者，家里面多多少少有点家底，不然他们怎么能够远渡重洋呢，是吧？那么我们作为下里巴人，我们这些普通的老百姓，怎么可能在乎这种事情？爱穿什么穿什么，who cares？这句话也相当的小资了。那么真的如此吗？恐怕不是。对于服装的需求，对于服装的需求，所有人都是一样的；对于“我是谁”的疑问，所有人都是一样的，他不需要有文化。所以我们会看到一个现象：在清朝晚期的时候，我们爆发了太平天国革命。太平天国革命虽然它的领袖不完全是农民，但是它下面的革命者绝大多数都是很纯很纯的农民，没有什么文化。要知道清代的识字率特别差，他们真的没文化，真的没家底儿，穷困潦倒，只能拎起镐头出来闹革命、要造反。就是这样的一群人，他们在聚齐队伍之后干了什么呢？他们去抢了戏班子，抢了寺院和道观，不劫财——嗯，也可能顺道截了一点啊——但是他们主要的抢掠的东西就是服装。因为在当时的民间的诉求和传说当中，戏服是不祥的，僧道也是不祥的，所以这些人的服装呢，并没有经历剃发易服的改制。所以那个时候闹革命的这些人，尽管他们没文化，也不知道，甚至家里面没有一个祠堂可以看他们伟大的祖先穿了什么样的衣服，但是他们有耳闻，他们听说了：戏服是老祖宗穿的，僧服、道服是老祖宗穿的。于是呢，他们做出了和这些远渡东洋的学者、文人们一样的选项，就是更换自己的服饰。他们抢来了衣裳，然后花花绿绿、五颜六色，同时搭配上他们那蓬头垢面的底色，搭配上他们粗糙的皮肤，搭配上他们简陋的农具，浩浩荡荡地就出发去革命了。所以衣服不只是衣服而已，衣服也不仅仅是有钱人的专属。有钱没钱，有文化没文化，当他想要拉队伍站起来说“我是谁，我要干什么”的时候，一定要选择一套衣服。而这套衣服在当时的中国人心目当中，它不叫做汉服，但它就是我们今天所说的汉服。大家都在找它，怎么找，各有道路。这就是为什么那个时代会出现这么大规模的一些在服饰上面的各种各样的做派。

当然，后来的革命军用了更加彻底、更加决裂的方式去制定了一套我们新的、民国时期的特定民国服饰，就是中山装配旗袍。它里面加了大量的西方服装的底色，在这套服装设计当中，使用了大量西装的剪裁方式，包括了旗袍，也包括了中山装，其实它的底色都是西式服装。但是这个不要紧，重点是我们此时急于拥有一套自己的衣裳，而革命军选择了用一种“算了，过去什么样，我先不找，我们先把它割舍掉。过去既然我们落后，说明一切都不好，我们重新来过”。这个时候他构建了一套新的体系，这是我们今天所熟悉的民国时期的革命服饰体系。但是这并不意味着当时晚清时期很多的人曾经做出的努力是无效的。可以说他们最后没有达成那样最好的结果，可是他们的努力很有意义，也很有价值。服装是什么？服装是文化与政治的外延，没有人能够脱离它。

而今天啊，又出现了一个问题：就是有些人说，哎，只要是汉人穿的都是汉服，清代的汉人也没有光着，也穿着衣裳，当然也是汉服。这些人都是多余的，他们为什么非得要去折腾这样东西？是呀，为什么呢？为什么生在清代的人不觉得清代的民装是汉服，一定要去找两三百年前的衣裳，而今天的人却要翻旧账，说那就是汉服？我觉得今天的人如果喊了这个口号的话，非常对不起当年抛头颅洒热血的老祖先们。他们花了大量的努力，付出生命的代价，想要找到自己的故国衣冠，而我们今天告诉你：哪有什么我们的故国衣冠，根本就没丢——把革命者当年的努力付之一炬，也不过如此吧。

在这个地方，道士也可以顺带提一句：清代的服制系统为什么不可以把它称之为汉服？因为清代的服制它并不是一个民族性的服装体系，它真的是一个阶级性的服装体系。因为清代的服装系统是分了两个层级的：一个圈层呢，叫做旗装。旗装它并不是纯粹由满人来穿着的，它是由旗人穿着的。旗人当中啊，包含了满、蒙、汉还有其他的民族，它本身是一个阶级属性，而不是民族属性。里面还包含了很多被抬旗的包衣奴婢，也都是可以穿着旗装的，它更多是一种阶级群体的概念。而与旗装相对应的也并不是汉服，因为中国当时有大量的少数民族和汉族一样，穿的都是民装。所以其实当时的清代服饰系统，它是分成了旗装与民装两个体系，而不是所谓的满服与汉服。它不是民族服装，而是阶级划分的服装。在这一点上面，很多西南地区的少数民族，他们的那个更久远时期的照片、图画，你可以看到端倪：就是在解放之前，这些地方真正他们流传下来的老祖先穿的那些传统的衣服，在服装的形制、剪裁、结构上面，其实和汉女所穿的这些平民女装是没有本质区别的。所以它并不是一个民族服饰——民族服饰不可能不区分汉族、苗族、藏族等等，这样的区分才叫做民族服饰。但是在清代的时候，他是把旗与民做了划分，所以清代的服装体系应该算作阶级服装体系。在阶级体系里面，咱们就不能去强调说“你是什么族啊”，那个时候不强调这个东西。旗人的高阶统治者，一视同仁地高压统治所有的其他民族，这才是清代的管理真相。所以今天如果你硬要说汉服从没断代，那是十分不公道的，因为清代的人、民国的人，无数的有识之士，也包括了无数不识字的农夫，都曾经为这件事情努力过。今天我们要是真的就把这个事儿含糊过去了的话，如果民装可以当汉服，那“汉唐在日本”很可能就会变成真的。

那么“汉唐在日本”，在当年的那种情况下，它富有合理性。可是今天我们依然可以谈吗？矛盾是在不断变化的。时代的变迁，决定了我们今天不可能再谈“汉唐在日本”这种事情。因为今天的中国，我们已经在自立自强的道路上面找到了“我是谁”这个问题的答案。我们已经知道了：我们祖先穿什么、用什么、戴什么、写什么样的文章，这些不再被掩埋。这20年来，我们做中华文明探源工程，挖掘了无数的文化遗址，在里面找寻祖先的印记，然后不断地去追溯、不断地去推导我们的历史进程，实际上已经构建起了我们新的自我认知的体系。同时呢，那些研究汉服复原的小伙伴们也非常的优秀，目前不但是复原了各种类型的服装，而且还把它做成了真正的生意，推广到了全中国，也推广到了世界上面。当我们自己已经回答了问题“我是谁”，我知道我是谁了的时候，如果还有人高喊着“我的祖先在日本”，那很可能他的祖先真的是在日本。那么我们对于那些祖先在日本的人，应该是个什么态度啊？咱们就斟酌一下哈。总之呢，总不能当做真正的手足同胞看吧，毕竟他还没认错呢。
