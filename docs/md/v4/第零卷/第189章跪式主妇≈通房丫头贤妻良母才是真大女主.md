# 第189章 跪式主妇≈通房丫头  贤妻良母才是真大女主

当你听到“贤妻良母”这个词的时候，会想到什么样的形象？很有可能是那种旧时代唯唯诺诺的、犹如附庸一样的女性——哎呀，不独立了，没自主了，没想法了。或者按照王女士的说法，这叫“结婚了被男人吃，生了孩子被孩子吃”。但是，这个古老的词汇是怎么变成今天这种下场的？它原本到底是什么含义？“贤妻良母”真的是一个社会附庸、真的是唯唯诺诺吗？我们得知道它的历史进程，也得知道它的本质真相。所以今天我们来聊一聊“贤妻良母”。

首先，来看一下它是怎么变成今天这种奇怪局面的。当然，在历史的开端，“贤妻良母”绝对是对女性的至高评价。你得到什么档次才够资格当贤妻良母呢？甭说了吧，当一个马皇后啊，可以当贤妻；你或者是当孟母，你可以当良母；不行的话，你当岳飞的母亲，给她刺一个字，鼓励儿子保家卫国，这也可以。所以“贤妻良母”这样的角色，在历史上是可以写进史册里的伟大女性。但是为什么后来变了呢？要知道，这个演变还真的不完全是说因为人的思想如何如何就直接让它变了，而是因为我们的社会生产方式发生了转变。从工业革命席卷全球之后，事实上逐渐逐渐地出现了一种风潮：女性开始走出家门工作。工作之后会产生的一个结果就是，大家频频都是工人，你凭什么给我指手画脚？这样的态势一旦出现，家庭内部的分工就开始发生转变，于是之前那种“贤妻良母”到这里就开始不适用了。

较为典型的一个例证，就是在我们解放之后的妇女解放运动，女性能够开始顶半边天了，大家都要出门务工，甚至有专门的妇女干部坐到你家炕头，说“出来打工吧”。当大家都已经开始出来进行社会公共劳动之后，我们的社会就不会专门强调你是什么样的妻、你是什么样的母，而是强调家庭和睦、夫妻合作，男女搭配干活不累嘛。这个时候强调男女平等，强调每一个人都是社会公共劳动的一份子，强调的是家庭活动当中的互帮互助和互让。这种家庭关系当中，自然就不会专门去强调“贤妻良母”这样的角色，因为这不对等嘛。于是，当这个历史的潮流逐渐逐渐开始演进，到了上世纪80年代，这个风尚又开始发生改变了。因为之前的劳动妇女把腰杆子挺得笔直，使得很多以前还有一点大男子主义残留的男性开始越来越觉得憋屈。而这个时候，他们看见了一线曙光——什么曙光呢？就是这个国门“咔嚓”打开了一下之后，好多杂志开始给他们写小故事，说日本的全职太太回家的时候跪在地上给丈夫拿拖鞋，一天三顿饭，精致的小便当给你做了一盒又一盒，每天在丈夫起床之前她先起床梳头、洗脸、把妆都化好，连丢垃圾都要保持自己老漂亮的样子了。这个时候，那个丈夫那叫一个舒心，那叫一个顺畅。听到了这种故事的中国男性开始在心里又一次萌发了当年民国时候某一些大师的幻想——要娶一个日本太太。因此，从这个时候开始，有一些人心目当中的“贤妻良母”变成了那种样子。但是那种样子真的是中国人所说的“贤妻良母”吗？即使放到中国的古代，这种角色那也叫“通房大丫头”，贤妻是排不上的，良母也排不上。因为“贤妻良母”就不是一个伺候人的角色，是给儿子背后刺字的角色啊。所以放到这儿的时候，我们就知道了为什么这种角色开始逐渐转变——因为社会当中的公共信息和语境发生了变化，人们想象当中的“贤妻良母”的形象就随之发生了转变。

但是吧，任何一个潮流，当你往这边儿“哐哐哐”地开了一顿之后，瞬间就会开始有人不对劲了，给我拽回来吧。于是，当有很多人不断不断地强调所谓的日式太太，时间久了之后，就开始有人生出反骨来了——那叫什么玩意儿啊？我们要进入大女主时代。当大女主心态开始逐渐萌生的时候，人家说要“贤妻良母”，怎么可能呢？连出个门儿她都觉得男人是个负担的时候，她不可能在家里面琢磨着“贤妻良母”这个事儿，听见这个词儿她都觉得侮辱了她。所以，“贤妻良母”在今天为什么变成了一个纯纯的负面词汇？因为社会定位发生了改变，社会风尚发生了改变。尤其是在传媒业，各种各样稀奇古怪的大女主电影纷纷上线。尽管不知道她的收入来源是什么，也不知道她到底在家务上面有什么本事可以自立自强，但是哪怕说肩不能挑、手不能提、薪水没多少，人家可以住城中心大平层，而且过日子的时候爽到飞起——只要没有男人，我就是大女主。可能不吃不喝也能够当大女主，这个事儿我理解不了。但是无论如何，当整个的社会舆论风潮发生转向的时候，人们的想法就会随之而发生改变。于是，“贤妻良母”就开始变成了一个污名词汇——你说我是贤妻良母？你是不是骂我呢？我像是一个无业人员吗？我好歹还有月薪3000呢，凭什么让我当贤妻良母？这种心态油然而生。

但是呢，我们不得不将这个词汇的本身的本源挖出来——我们的祖先为什么要总结出“贤妻良母”这样的褒扬之词？是因为它特别歧视女性吗？是因为它特别打压女性的这种独立自主意识吗？真的是这样子的吗？“贤妻良母”也要注意家人的健康啊。如果你坐久了之后，肩颈脖子又酸又僵，千万要留意了。不要以为只有坐姿会影响肩颈，睡姿也很重要。想一想以前有没有翻来覆去睡不着，睡醒了之后脖子又难受的呢？如果有，就得小心啦。跟我一起试用一下这个温眠人体工学枕包——有用的——它能够让你睡得又快又沉，睡醒之后肩颈不酸不胀。这个枕头呢，我亲身试用过，对于肩颈脖子确实能够舒缓不少。很多的小伙伴可能不是特别在意这个，但是换了这个人体的工学护颈枕之后，它的效果确实不错。你看它前面凸起的颈托，把这个脖子往上面一靠，就会托住脖颈本身的自然曲度。也许你会觉得这个造型有点奇怪，但是别担心，看这本儿，还有那本，这些教材都有说到这样能够舒缓你的肩颈酸胀，让你睡得更快更沉。这也难怪他们家仅仅两年的时间，销量就做到了40万，连回头客都有2万多人，都是复购给家中长辈的。所以小伙伴啊，为了家人的健康，也为了自己的健康，很有必要在家里面买一个。那么更重要的呢，是温眠的售后是有保障的：两年之内坏了能够免费换新，两年以后坏了半价以旧换新。可以去评论区领一张优惠券，抓紧去购买，券后的价格非常的优惠，买贵了随时可以退差价。

所以在历史上，“贤妻良母”到底是怎么诞生的，以及“贤妻良母”的评价指标到底是如何的？在这个领域上面，必须先弄清楚的一件事情就是“妻”是怎么诞生的。“妻”是什么？妻呢，是夫的伴侣。什么情况下会结成夫妻呢？这叫“缔结婚姻”嘛，我说了好多遍。什么叫做“缔结婚姻”？“婚者，合两姓之好”——两性是什么？两个家族，不同姓氏的家族，然后缔结了一场婚姻。这个时候咱们就得弄清楚了：在过去漫长的数千年的时间里面，家庭、夫妻不是简单的爱情关系，绝对不是看对眼儿了之后我们两个就在一起，你侬我侬过日子，早餐互相喂一下，晚上互相安慰一下的这种关系。自古以来，婚姻家庭它是社会的最小生产单元。所以我们自古以来制定的无数的与婚姻家庭相关的法律规范、道德规范，都是为了保护这个生产单元能够稳定。所以我们现代人不能够因为家庭和古时候的家庭是一个词儿，你就觉得是一回事儿。古时候的家庭，你得把它理解成一个企业：它叫做夫家和妻家共同出资，营造的一个新生的子公司。所以当你把它理解成企业的时候，夫和妻之间的社会分工就比较好理解了。

怎么分的工呢？一个企业设了一个新的商号，注册了一个新的执照，你那个执照上面要不要写法人的代表？要的吧。你去做一个登记，你自己就看见了——法人代表是谁？户主喽。谁呢？如果是正常的婚姻，那就是丈夫作为户主；如果是入赘，那就是妻子作为户主。所以户主是谁？户主就是那个夫。那么“妻”是什么角色？任何一个企业，你光有法人代表就完事了吗？法人代表是负责干什么的？你自个儿悄悄地去问一下那些专门注册公司的人，他会告诉你：法人代表是负责背锅的，是负责背债的，是负责出事的时候去蹲班房的。那么实际公司运营当中，重要的核心角色是谁呢？叫做经理人。对，妻子就是那个职业经理人的角色。所以在古典时期，真正的婚姻缔结的方案，实际上就是“夫”作为法人代表——当然了，他也必须得是大股东——然后以“妻”作为职业经理人以及财务总监，来进行整个企业的运营。那么妻方也要带入他的股本，通常情况下，这个股本不会超过夫家，但如果超过的话，就容易攻守异势啊——那家里面谁说了算，这个事就不好说了。总之呢，它是一种企业缔结方式，是一个商号缔结方式，而不是单纯的两人看对眼儿的关系。所以为什么要父母之命、媒妁之言？你听起来很腐朽，你都不自由恋爱，但是之所以这样，是因为人家出资了的父母是谁？父母是原来出资方啊，那是出资方。出资方你能不听他的话？你今天要设立企业，你们家出资的那个人啊，你不听他的话，甩开他，不理他。然后媒妁之言呢？媒妁是干啥？居中作保的人啊。你设立一个商号、一个企业，又不听原始出资方的，然后呢，又要把中间作保的人踢走——你这商号怎么设立？所以古时候为什么这一堆规矩？因为它的性质和今天不同。所以放到这个框架里面，咱们就能弄明白了：如何界定一个古人的贤妻是不是贤妻、良母是不是良母？那你就要按照这个家庭、这个企业的考核KPI，来对这个职业经理人进行针对性的考核。考核合格的那叫做“贤妻良母”，考核不合格的，你就是占了个岗位而已。

那么什么叫做“贤妻良母”？刚才我为什么说日式太太不算“贤妻良母”，最多算是通房大丫鬟？你就看她的那个工作岗位：优秀小厨娘啊，卧室内的优秀服务生，进门的时候递拖鞋、擦地板的那个了不起的服务生。这种服务生的岗位，这玩意儿就是丫头和小厮干的活儿，它压根儿就不是职业经理人的工作。所以在这个范围内，日式太太的工作内容与之毫不相干。那么“妻”应该干什么？在这个领域当中，我们就不得不去看一下《红楼梦》了。那《红楼梦》里面谁是“贤妻良母”？《红楼梦》里面的各色人等非常多，而且普遍性的都结了婚或者想结婚。所以在这种传统的大家族的内部，什么样的人算是“贤妻良母”呢？王熙凤算不算？来，在座的男性小伙伴来说一下：你觉得王熙凤给你……你觉得她是不是贤妻良母？不是吧？别闹了——这种母老虎、这种泼妇，偷个人她恨不得把我皮都揭了的，这种能够叫做贤妻吗？很遗憾啊，这种类型她还真的可以叫贤妻——是不是良母不好说，因为她只生了一个女儿，并且那孩子还很小，不知道能不能培养好。但是呢，在“贤妻”这个岗位上面，打小三儿是贤妻的表现。为什么这样说？这就不得不将王熙凤和她的婆婆进行一下对照了。她的婆婆是谁呢？她的婆婆姓邢，叫邢夫人。而邢夫人干了什么呢？她的丈夫想找小老婆，不管是三五百两银子还是三五千两银子，他都想方设法的都划了出来，把人就给请进来了。无论她想要什么，她都想办法地把银子掏出来，让丈夫去满足这样的私欲。但是你有没有发现，到王熙凤这个地方，就叫做一个精打细算。贾琏想从他身上抠出点银子出来，那简直要想破头，那个借口一定要找得明明白白的，甚至有的时候啊，不惜去说“哎，你到哪里哪里再去弄一点出来”，甚至不惜说“你去把那个什么什么东西当了它”……已经到了这样的一个地步。这固然是因为贾府内部的财政本身就有问题，但是呢，在“贤妻”这一件事情上面，我们可以看到一个表征：就是王熙凤是有一定的理财的能力的，至少说她会划了钱。还记得吗？我们以前有一句俗语说“那个丈夫是爬前的耙子，妻子是装钱的匣子”，这样子才叫做一个和睦的家庭，这才叫做贤妻干啥？这强调的是理财的专属能力，就是你得会攒钱，你得知道怎么把钱弄到我们家里面来，怎么能够让大家能够穿得上衣、吃得上饭，这个也是贤妻的工作范围。所以在这个领域里面，你看，王熙凤一直在试图东墙补西墙的工作，她在内部进行财务调度；但是邢夫人干啥？花钱，花钱又花钱了——虽然她这个花钱不怎么满足自己的私欲，但是你把家里的钱都花出去了呀，这叫奢靡，这叫浪费，这个叫做动摇家庭根本。所以不管邢夫人有多么的顺从，她都不是贤妻，甚至她彻头彻尾就是贤妻的反面。而王熙凤呢，尽管又是泼妇、又是母老虎，没有人说她的好话，然而她在这些领域当中都属于贤妻——惊不惊喜？意不意外？对于“贤妻”的点评，其实压根儿不那样。为什么会这个样子啊？因为“贤良”二字本来就是中性的。亲爱的小伙伴，你注意到了吗？当我把“贤妻良母”当中的核心点评词汇一抽出来，你就突然发现不对劲啊——这玩意是专门拿来形容女人的吗？显然不是。“贤良”二字，通常情况下会拿来形容朝中重臣、人中典范，例如“贤臣”、“良将”。所以“贤良”从来不是拿来形容女性的专属名词，也不是一个顺从的专属名词，它是对它是有技术岗位分工的要求的。“贤”是什么职位？“贤”呢，是有能之士的意思。“贤”的意思是有能力、能掌控，同时有没有注意到“贤”字下面有个“贝”？“贝”是什么？贝是钞票，是钱。所以“贤”是有能力、并且会理财、会聚财的人才叫做“贤”。而“良”是什么呢？“良”呢，既可以是善良，也可以是良好。所以“良”的意思是指它的美与它的好，也就是说一个人能够把一件事情办好，办得柔和漂亮，同时呢，又有能力让它顺畅，而且还能够聚财，他才叫做“贤良”。你就说吧，就这样的能力放到男人那里，不也是凤毛麟角吗？所以当我们把一个这么高的点评放到了“妻”与“母”这样的身份上的时候，其实意味着我们古时候的人们对于家庭当中的妻子与母亲这样的岗位是有多么的看重。甚至于会把用于君子身上的至高点评，放到了一个妻子、一个母亲的身上，因为我们知道他是社会最小生产单元，是社会未来的希望的来源。所以我们极为看重妻子与母亲，对顶尖的妻子与母亲赋予至高的点评，就叫做“贤良”。

那什么样的人可以用到“贤良”这样的词汇？什么样的岗位可以用“贤良”来点评呢？那就是辅助岗啊。我们知道这个社会上面呢，其实有几种社会的分工。例如很多人会质疑：刚才我说的这一整段话当中，丈夫似乎隐形了，丈夫不见了吗？为什么什么事都是妻子干的？哎，注意到了没有：我们刚才说了，家庭是一个企业、一个商号。一个国家，你也可以把它想象成一个企业、一个商号，这个里头的法定代表人、法人代表是谁？是户主。在家庭当中，她就是那个丈夫；在朝堂之上，她就是那个君王。这个人是干什么的？他是负责负责任的，负责背锅的，出事的时候捅娄子的时候负责蹲班房的，他是负责背负骂名的那一个，也是负责决策的那一个。这个东西呢，就叫做“君为臣纲，夫为妻纲”。它代表的是纲领，代表人的表率、责任，作出决断，判断将来下一步到底往哪儿走。然后呢，后面的贤臣良将、贤妻良母负责干什么？负责支持这个目标，把它实现，把它做好。做得好了，你就是贤臣良将、贤妻良母；做得不好，那就不算啊。那么“君”的这个部分以及“夫”的这个部分，作为纲要、作为纲领、作为负责人，他干什么呢？他做出了决断，决定往这儿走，然后呢，立成一个标杆，作为一个榜样，带队往前走。但是如果最后发现这个方向是错的，锅就归他呗——这种叫做社会责任分工。有人负责订立目标并且身先士卒的表率，站到前面去，承担所有背后的责任；然后有人呢，就负责进行协调协作，把它实现。这种协同的方式，不光是古代这样，现代也这样。我们现在的企业当中，是不是也是有人拍板儿、有人盖章、有人签字，同时有人在下面不停地跑程序？对这个东西就是标准的啊——做主以及打辅助。包括你在打游戏的时候，有人在前面“哐哐哐”的一路猛冲，做你们的带队人；但是后面是不是得有人加能量？是不是得有人断后？如果连这些都没有，光有一个人冲冲，几遍死几遍。所以实际上，“贤良之士”、“贤良之妻母”从来都是至高点评，而且他身上绝对不是为了顺从而存在的。他的职位、他的岗位要求就是你得把这事办成。把一件事情办成，就意味着他有充分的权利，有充分的自主，而且你必须有自主意识，你得知道目标在哪儿。你光是听话，光是顺从，他说撒钱你就撒钱，最后家里面毛都没剩了，这叫什么贤良？所以在前面突击猛进的带队人非常冒进的时候，你还得在后面蹬缰绳拽着：“不许跑了！不许跑了！再跑，跑不动了！”你就会发现这种角色在朝堂之上叫做“诤臣”，这是魏征一样的角色；而在在家庭当中的话，那就是贤妻呀——我有点怕你。所以你会注意到一件事情：贤妻良母、贤臣良将，他们都是执行者与辅助方，但是也是不可或缺的角色。中国人对于性别并不是那么看重，而对于社会的分工岗位更加看重。

好，那刚才你都讲了这么一大堆的老八股了，这事儿跟我们到底有什么关系吗？嗯，其实还是有那么一点点的关系啊。现在呢，很多人都会意识到我们结婚很困难，甚至呢，相亲都挺困难的。那么我们在相亲的时候，到底能不能提“贤妻良母”这种要求？你敢提吗？反正我不敢提。谁要是再敢在相亲的时候说“我要找一个贤妻良母”，你这个单子直接被所有人都pass掉了，连中介都不敢收。为什么？因为这个词已经烂了，一听了之后就觉得你要找一个跪式服务的日本太太。但是呢，我们应该如何提出这种指标？这个事儿无分男女啊。如果说你是一个男人，你想要找一个贤妻良母；或者说你是一个大女主，你想要找一个贤夫良父——都可以这样去操作。干什么？我提出的要求是：我们双方来共同建设一个家庭，你必须要拥有良好的打辅助的能力，拥有良好的经济配合的能力，拥有良好的组织能力，拥有良好的理财能力，拥有良好的教育能力。这个东西就是典型的贤良的标准。你说能不能提？能提。提具体的这个具体的指标一拿出去，凡是听了之后就觉得崩了的，那就是只想要权利、不想担责任的人——绕他远一点。正常的人其实是希望互为辅助的，愿意互相打辅助，愿意互相协同作战的，那就叫做贤良啊。至于其他的那种，让他们自个儿玩儿去吧。
