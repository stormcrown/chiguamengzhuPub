# 第173章 《玉盘》为什么像战歌

今年的春晚给我们上了个大分啊！春晚过去了那么久之后，居然还有他的歌曲在互联网上不断地流传，而且二创、三创、四创生生不息。这说明这活儿干得好呀，有进步呀！什么歌呢？《玉盘》！预判？预判你为何悬于屋顶上——相信大家现在都已经听过了，哪怕不看春晚的人，也都已经听过这首歌了。诶，为什么会这个样子？当你去打开这些二创视频的时候，就会感觉到：哎，这是出啥事了？又是“招摇”又是“草”，有时瘦来有时胖。就是因为所有的二创作品，基本上都把一些战争场景和这个童声小合唱拼在了一起，而且只要一拼在一起，底下的评论区咔咔的：“哎呀，这个味儿对了，没错啊！我听到它的时候就是这个调调啊！听到它的时候，我脑子里面想到的就是这个感觉——封狼居胥啊！马踏瀚海，赶紧的！”唉，于是呢，很多人说《玉盘》这首歌，它看上去是一个儿歌，然而呢，却是一个战歌。那么今天我们就来聊一聊，儿歌是什么？嗯，战歌是什么？

当然了，当我们在这里说“儿歌”的时候，原创作者肯定不高兴啊，因为原创作者人家很郑重地说了：“不要说我的歌是儿歌。”我觉得是因为在大多数人印象里，儿歌指某一类特定的风格和表达方式，面向特定的人群，比如说低幼儿童。嗯，我更常用的是“童声音乐”这个词。对于音乐人的执着，我们就来解析一下：那么儿歌和童声之间到底有些什么样的界限？嗯，其实也比较好理解。因为儿歌是什么呢？儿歌就是“小兔子乖乖，把门儿开开”。哎，这个是什么？这个叫做儿歌。它唱的是一个孩子可以直接理解的、孩子能够直接明白的东西，这种歌曲叫做儿歌。我们日常生活当中会听到很多的儿歌，什么《种太阳》啊、《黄鹂鸟》啊，这些都是属于儿歌系列的。但是呢，童声歌曲那就不一定了。要知道，你用儿童的嗓子来唱，就可以叫做童声歌曲，但是唱的内容如果儿童根本就不懂，也压根儿不可能理解，那这个时候你就不能叫它儿歌，必须称它为童声音乐。啊，这就是一个界限。

那到这个界限上面就有一个问题：儿童又不理解，你非得让他唱干什么呢？你换个人唱，换一个懂的人唱，不是很好吗？啊，这时候我们就不得不转进一下，转进到一个自古以来的苍茫的时代。干嘛？因为呢，用童声来唱儿童压根儿就不懂的东西，是历史源远流长的东西。在我们的历史长河当中，它还有一个名字叫做“童谣”。说到童谣你就熟了吧？童谣是拿来干啥的？当你翻开历史课本，你就发现所有的革命军的开篇之前，都会出现一个童谣，各式各样的童谣啊。当我们前面说“小兔子乖乖，把门儿开开”它是一个儿歌的时候，那么“开了城门迎闯王，闯王来了不纳粮”是什么？童谣！儿童知道什么叫做“纳粮”？儿童知道什么叫做“闯王”？儿童知道开了城门后果是什么？他都不知道。但是他唱得大街小巷人尽皆知，这就叫做童谣。当我们弄清楚这个的时候就发现了：哎呀，原来啊，还有这回事呢！哎，原来童谣可以制造这样的影响力啊！

对，童谣是一种很特殊的音乐形式——你说它是音乐都有点儿不太对味儿的那种音乐形式，因为当我们去传唱童谣的时候，并不在乎它是不是音乐。这就是童谣。当我们要理解它的时候，先要知道什么是“谣”。谣呢，我们在互联网上也常说“你造谣啊”“你传谣、造谣、传谣500条”，呵呵。当我们去传这个话的时候，你就知道什么是谣——就是未经实证，但是传播甚广的言论，这种东西就可以叫做谣。那么童谣呢，就是用儿童的声音去传播这些信息，它常常是传播一些预言性质的啊——“闯王来了不纳粮”，或者是“山那头有个大王当了皇帝，咱们就不闹灾荒”。哎，这些东西呢，都可以算作一种预言。

你就有意思了，对吧？你让它放预言，你找个大仙儿放预言不好吗？啊，那个大仙儿七老八十的，一看仙风道骨，一说将来必然是谁当皇帝，大家不都听了吗？但是这里头啊，咱们要弄清楚一个界限：为什么我们去发预言的时候，不找一个仙风道骨的老神仙在那个地方座谈讲法？因为它效果没有童谣来得好，真的，它的效果严重地比不上童谣。它们中间的区别在哪？你要知道，当这个人是一个——不管他长什么样啊，仙风道骨的老神仙也好，是什么也好——总之呢，人群当中一定还是会有大体量的人觉得“说的是不是真的呀？他不是个骗子吧？”甚至可能有人认识他说：“不对，她昨天还在我们家隔壁编草鞋呢，他哪懂这个呀，是吧？”这个信息就会开始出现筛选，那么就会有很多人不相信。这个中间的差异是什么呢？我们得理解一下人类在心理上面的一些行为的逻辑。

人类从我们诞生的那一刻开始，就有一个强而有力的威压垂在我们的头顶上。这是什么？中国以外的人把那个玩意儿叫做“神”，那中国人嘛，把它叫做“天”什么的。啊，它本质上是大自然。我们知道，哪怕是今天，我们所有的科学也好，技术也好，都是为了征服和掌控、理解自然。我们做到“我即自然”了吗？没做到。到今天我们都相信某一个地方肯定还有一些我们没有探索到的神秘之地，连科学家都承认这一点。也就是我们对于自然的探索，压根儿没有走到尽头的时刻。你再回溯到过去的几千年，那就是刚刚开始尝试征服自然的时刻。那个时候自然是什么？是一种不可名状的、拥有无限威压的角色。你想到它，你就觉得什么都有可能发生，什么都不难理解。你只要说这玩意儿是自然的属性，那就是了吧。这种情绪是刻画在每一个人类的基因里面的，很难挣脱它。今天的现代人也没有能够挣脱它，古人更加不可能。那么，童声就是在利用这个。

而我们再倒述一下：我们知道啊，所有闹革命的前面都要进行舆论的宣发，让大家相信前面一个王朝气数已尽，后边一个王朝要起来了。你这个宣发的活动怎么推广才能有用呢？你总不能运用前面一个朝廷的喉舌吧，人家也不跟你干呀——谁给他发饷银，还是要分得清楚的。所以这个宣发你必须自己做。自己做怎么推广这些玩意儿，还让很多人都相信呢？还让人家觉得你不是骗子呢？一般情况下，你就要利用自然之声。诶，为啥刚才说“自然”？因为自然之声就是可以直接影响人类心理的东西。我们回顾过去的历史，革命的前夜是不是都有这种东西？有没有记得有一位革命者在造反之前，在营地的周围假装自己是一只狐狸，开始学狐狸叫？有没有记得有一个王朝在他兴起之前，传说在某一座山上面有凤凰叫？自然之声！它需要让大自然来告诉你：“大楚兴，陈胜王”——不然谁知道陈胜是谁呀？那不就是我们隔壁家阿二吗？啥玩意儿，我看过他穿开裆裤。但是当有狐狸说“大楚兴，陈胜王”的时候，效果就不一样了啊，性质完全不同。这就是自然之声。

那么说到这儿的时候，你就能够理解为什么大家会用童声。因为童声在人类的心理或者潜意识或者直觉当中，它就是一个无意识的群体——儿童没有实现社会化，他没有学会是是非非，他甚至不知道什么叫做“放粮”，他不知道什么叫做“闯王”，所以他嘴里面说出来的，那就可以称之为自然之音啊。这种直接影响你潜意识的这种行为，才是我们为什么会在历朝历代都运用童声、运用童谣来传递信息。这是一种本质的力量啊。尤其一点啊，它比那些狐狸叫好用。你要知道，学狐狸叫你得躲起来，不能让人看见你本体；但是让小孩儿说话，你在家里面给两颗糖教教他就行了。这成本指数级地降低，非常非常的低廉。而且呢，儿童喜欢到外面瞎跑，跑着跑着大家一起唱，哎呦，这个传播的效率啊，一传十、十传百，指数级的增加，扩散效率极好。而且家家户户都有孩子——我说的是古代啊，家家户户都有孩子——那传播起来深入人心。你就想，当你要造反的时候，你会不会用这个手法呢？你放到那个时代，你也这么干。这就是为什么会有童谣啊。

当我们理解了这件事情的时候，就能弄清楚一件事啊：那童声、童谣这些东西为什么能够直接地打动人心？因为在我们任何一个人的潜意识当中，都存在着一个符号——童言无忌。儿童呢，是没有坏心的；儿童是质朴的；儿童的眼睛是特殊的，非常明亮，能够看清善恶。这是我们所有人脑子里面，不管你说不说都有的潜意识。这种潜意识一旦被激发出来，哎呀，那就不得了了。所以为什么要用童声的方式传递信息？诶，话就放在这了。

那么现实生活当中，有没有这样的实证的例子啊？你自己去设想这个场景去。你说你手机上面收到了一条短信，说“今天晚上八点有地震”，你会干什么？这号码对不对？是不是诈骗电话？然后再去找人问问：“现在是不是地震监测已经到了这么精准的程度，可以预报时间点了？”然后再去反诈APP上查询一下：“现在有没有这种信号被截胡的风险？”你是一定要筛选这个东西的。但是如果这句话是一个儿童说给你的，效果一样吗？我们可以回看啊，我们过去几年间在互联网上一直流传着这样的一个小故事：说有人家里面有一个小孩儿，在家里面玩玩具，只要是坐在那个地方抬抬头，四处一望，诶，人家就说了：“哎呀，这里有两个小哥哥！哎呀，妈妈，那里有两个小哥哥！”他每天这样说，最后造成这个家里面无论老少人心惶惶，甚至开始到外面去找大仙儿了，四面求助，就觉得自己家里面有问题，甚至已经计划好了搬家。直到某一天，这个孩子终于具体地指到了冰箱上面的招贴，然后大家发现：“哦，这个小哥哥指的是海尔兄弟。”所以你瞧，即使是在现代化的社会，有用吗？没有用。当一个孩子懵懵懂懂地就跟你说了一个完全超乎于他个人所处的场景的语言的时候，你瞬间毛就炸了：发生了啥？他看见了啥？他听见了啥？他是不是听见了我所不知道的自然之音？现代社会也是这个样子。那你想，这个玩意儿多么的牢不可破，因为它刻在每一个人的基因本能当中，我们就会直觉性地相信大自然发给我们的预言信号。而所有的预言信号当中，最容易理解的就是小孩说话——因为我真听得懂，虽然他啥事不懂，但他会说人话呀。这就是童谣、童声对于所有的人类群体会起到的重大影响力。

因此，当我们听到一首儿童唱的歌曲，里面的内容能够唤起我的某一种畅想的时候，哎，这个东西那就不得了了。于是后面就说：为什么这个歌会让我觉得它像一个战歌儿呢？啊，战歌儿它有什么特征？战歌一定是“冲吧，兄弟，上吧，兄弟”？这不一定啊。要知道，那个战歌最主要的作用是激发你的战斗欲望。什么样的情况能够激发一个人的战斗的欲望？要知道，一个人不会平白无故地就想出去跟人家干架的。所有的战争都有成本，所有的人都知道这一点：只要打仗就会有巨大的损失，甚至会有生命之危。这么大成本的一个东西，你要用什么才能调动起他心里头的这篝火啊，让他能够上头，觉得性命无所谓，什么价更高呢？啊，这就是战歌的本源。

所以我们记得啊，上世纪非常流行的什么《喀秋莎》战歌，唱的是什么？那个漂亮的姑娘在河边等我。它有说喀秋莎火箭筒吗？没有啊。喀秋莎是不是火箭筒？谁在乎这事儿啊。当真有一个漂亮的喀秋莎站在河边等我的时候，那我不光要打，我还得活着回来。这个就是战斗的欲望。同样的，在我们国内有没有这样的？有啊！最典型的，我们被称为“复国歌”的那一首——《我的祖国》。很多人不记得这个歌，当我跟你说它叫“一条大河”的时候，就大家都记得了。这个歌唱的是什么？还有一条河，河上有波浪，波浪上面有船，船上面有帆，帆下面有人唱号子。他就唱这玩意儿啊，河边还有稻米。啥呀？这是谁？还没见过这个吗？但是这首歌是一个战歌，所有人听到了之后会流眼泪，会觉得我得干点什么。为什么？因为这首歌在歌唱你最爱的家乡，它在歌唱你的国家，你希望过上什么样美好的生活，你希望阳光多么灿烂，能够照在谁的身上，你希望孩子能够好好地上学。就这种愿望，这个愿望是什么？就是一种美好的保家卫国的愿望，可以让你踏出国门去，一直到我们取得最终的胜利。这种促进的力量就叫做战争的动员。

那么放到《玉盘》身上，你说它有没有？“玉盘”呢，来自于一首诗啊——“儿时不识月，呼作白玉盘”。所以呢，它其实唱的是月亮。它说：“哎呀，那个月亮啊，真亮；月亮啊，真圆；月亮啊，真漂亮。我喜欢，咱啥时候去，啥时候把她带回来？”这个歌啊，虽然人家词写得非常好啊，但本质上唱的就是这个：“我去，我把她捎回家来。”哎，基本上它就是在干这个事儿。那么像这样的一首歌，它不停地在歌唱这个月亮的美好，歌唱着我们星辰大海的梦，歌唱着我们要去、要把它带回来。这个是干什么？它在号召你：你想要吗？想要却拿到它。你喜欢吗？喜欢把它带回家。这个就是心里的愿望。它用儿童的方式来唤醒我们心底深处对于星辰大海的愿望、愿景，哪怕它遥不可及呢，咱都想试一试。这和歌唱家里的景、歌唱岸边的姑娘、歌唱大河、歌唱水稻是一样的事情。任何一首歌里面，当它用这样的方式唤醒你对这些美好东西的渴求的时候，它就有可能激发你内心战斗的欲望。不战斗，你凭啥拿到这些好东西？不战斗，你凭什么保护你自己家的女孩子呢？这就是一个很根本的一种本源的力量，所有的战争的动机都来源于此。所以你说，小孩去唱一下月亮，居然大家就热血沸腾地说走出门干。这是啥意思？孩子都要了，给他带回来呗！你好！
