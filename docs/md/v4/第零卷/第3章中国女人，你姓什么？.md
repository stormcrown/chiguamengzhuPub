# 第3章 中国女人，你姓什么？

上期视频发布之后，有网友问我，虽然说得都很在理，但我们该怎么办呢？在此做一个简短的答复。

关于我们被话语霸权控制这件事，也不要过于悲观。中国之所以陷入四面楚歌的局面，本质上还是因为我们没输。那些已经输掉的国家，文化产业看上去非常繁荣、发达、国际化。而中国因为有本土文化，有自己强而有力的反抗力量，所以才显得格外四面楚歌。如果没了自己，自然也就没了敌人。如何扭转这个局面？说起来理论上不难，但实际上却不容易。有两个路线：

第一个路线是从上至下，就像秦始皇的“书同文，行同伦”。这个方法简单，只要有一个强而有力的中央政府都能实现，速度快，后续效果也很绵长。中国两千年的统一就是秦始皇奠定的。但它的副作用也很明显——反弹激烈，对中央权威的破坏很强，所遭受的反抗也会非常强大。

第二条路线也是中国人现在正在走的路线，就是人民的力量。在这个领域里，我们已经取得了很大的成绩。上一期我说中国的时尚界在慢慢扭转，其实就是本土文化、普通人一点一点靠自己的力量圈起了地盘，然后跟外来时尚领域的领军人物竞争，迫使对方不断扭转局面，已经开始发生变化了。所以现在中国走向世界的国际超模已经不丑了。另一个领域是汉服，它纯粹是从底层草根自己生长出来的。从诞生开始，它就面临境内境外各行各业的嘲讽、围殴和否定，但今天它依靠最底层最微小的力量已经生长起来了。如果没有汉服，我们可以回顾一下二十年前中国人的传统服装是什么样子——它被定义为旗袍马褂。而今天这个局面已经完全不同，这就是人民的力量。包括今天我能够坐在这里说话，也属于人民力量的一部分。我开口之后有人赞同、有人反对，这就是这个力量已经发挥作用的结果。如果是过去，你说了这话只有骂声。

好，问题回答完毕。今天我要谈的话题，是因为前两天我跟人拌了个嘴。事很小，大家也没真的吵起来。有个嫁到外国的中国女人发消息在国内说：“今天登记结婚了，冠了夫姓，感觉到我们真的是一家人，同一个姓氏凝聚在一起，非常幸福，非常有力。”这种事其实频频发生。我们也知道，外国的女人基本上结婚后就要冠夫姓，例如希拉里·克林顿，她自己根本不信克林顿，是因为她丈夫姓克林顿。这个现象在很多西方国家几乎是半法定的状态，大家结了婚就直接登记了。当时就是这样的背景，结果我跟一个小姐妹拌起嘴来了，因为她认为中国女人现在不冠夫姓是由妇女解放带来的，如果没有妇女解放，我们只会比西方更惨。我说你这话好像不太对劲，中国女人自古就没有冠夫姓啊。她就不高兴了，说你这不是扯吗？你懂什么？我的奶奶、我的太奶奶都是冠夫姓的。你有没有看过香港的电视剧？那些太太、张太太、王太太、李太太，都是因为她们丈夫叫张某、王某、李某。我说居然还有人会这样想。我说外国人是否冠夫姓跟中国人没关系，我也不是很在乎，但中国人自己如果理不清楚自己的局面，这事儿就不合适。

所以今天要说的是：中国人自古是不冠夫姓的。首先说那个小姐妹拿来反驳的例证——我的太奶奶、我奶奶全部冠夫姓，宋美龄后来被称为蒋宋美龄，香港电视剧里全是这太太那太太。这一套论证本质上是在默认人类历史始终是向上的，认为我们的今天一定比昨天好，昨天一定比前天好，是不断向上发展的过程，才会形成这个推论。既然我的奶奶是冠夫姓、没有独立人格，那我奶奶的奶奶的奶奶一定更恶劣。但人类历史并不是一直上升的，它是一个有起有伏的过程。中国几千年历史中，有向非常文明领域急军猛进的时候，也有突然被打断、倒退回来的时候。而她所说的这个阶段的女性称谓，基本上是从民国时期传下来的传统。我们很多时候说“中国传统”，这些传统常常是从民国下来的，而民国下来的传统大部分并不是真正的中国传统，而是民国文人把西方人的东西杂糅到中国人的规矩里形成的一种传统。如果你假设历史只能向上，就容易默认今天一定比过去更美好。但如果没有这个成见，我们就可以认真讨论一下：中国古代女性到底是不是冠夫姓？

首先，很好证实的一个领域是族谱。中国大部分家族以前都有族谱，今天剩下的可能不多，但很多还是有的，尤其是南方族谱系统保持得很好。你可以随意找一两个家族的族谱看。不是每个家族的族谱都会记载女儿，有些记载，有些不记载，但所有家族都会记载他们家娶了谁当儿媳妇，因为要记录自己子孙的母亲是谁。所以妻子这个角色一定会出现在族谱上。你去任何一家的族谱里，都会看到“陈某某娶妻王氏”“张某某娶妻李氏”然后生谁谁谁，绝不会看到“张某某妻张氏”——那族谱完全没意义，因为你翻遍了也看不出来谁是谁的妈，因为所有人都是张氏。中国人非常讲究家族伦理关系，一定要分清楚谁是谁的，所以族谱上每个人的名字和妻子都会写清楚，不可能让妻子和丈夫同姓，那就全乱套了。

第二个方向：很多人默认中国古代女人是从夫姓的，这其实跟几十年前一些小品、相声里讲这个事也有关系。我二十多年前听过传统相声演员讲笑话：说古时候女人一结了婚就叫做某某氏，万一我丈夫姓西、我姓洪怎么办？叫西红柿。这是一个曾经笑遍南北的梗。像这种段子在很多相声、小品、影视剧的角落出现，会形成人们的潜意识，认为以前的人肯定自称“某某氏”，什么张王氏、赵李氏、蒋臣氏等等。这是文艺界创作段子不够严谨带来的后遗症。事实上古人并不是这样称呼的。“蒋王氏”这样的称呼存在吗？存在，但不是这样用的。“蒋王氏”的意思是“蒋门王氏”，即姓蒋的那家的姓王的媳妇，通常用在祭祀或某些大型交际的重要场合，相当于“蒋家的王太太”。这并不是说结了婚就没自己的名、没自己的姓了，只是身份发生了变化，我们更改了对她的称谓。

第三个方向：中国人是怎么认知婚姻的？中国人对婚姻的定义叫“婚者结两姓之好”，结婚本质上是两个家族之间的联合。怎么证明是两个家族的联合呢？姓张的一家跟姓王的一家联合了，结了婚之后妻子改姓张了，那还怎么体现两个家族的联合？虽然结婚时会说“你就是他家的人了”，但这只是说从属关系调到了那里，就像工作岗位从你家调到了这个单位。但如果要离婚——比如遭了家暴，妻子第一个步骤绝对不是告官，而是回家找兄弟、找爹来撑腰。街坊邻居会看到大舅哥打上门去的场景，邻居怎么议论？“你看陈家的那个王挨了揍，王家都打上门来了。”如果他改姓了，这事儿就麻烦了：“你看陈家的那个陈挨了揍，王家打上门来了”——王家为什么要为一个姓陈的人出头？什么关系？要不要解释？中国人很在乎伦理，不会让这种情形发生。我们甚至主张同姓不婚。今天说只要超出一定亲缘范围就可以结婚，但古时候如果姓张的跟姓张的联姻，是要遭到巨大非议的，完全践行同姓不婚的原则。所以根本不会出现“张张氏”这种角色。“婚者结两姓之好”，一结婚就变一个姓，矛盾。

然后从第四个侧面看：之前那个小姐妹跟我论证时会说“你看香港电视剧里那些阔太太，张太太、张先生，王太太、王先生，你懂什么？你是不是不看？连小说都不看？”一个中国人想了解古代人怎么生活，其实不是特别困难，因为历史延续做得很好，传承到位，资料都写在历史书里。但阅读史书难度可能大，看文言文很累；文人笔记文绉绉也不耐看；大学教授的论文论证什么也很枯燥。但有一个很简单的、也会觉得很舒服的领域去了解，就是世情小说。从明代开始，中国的世情小说变得发达，出现大量通俗甚至三俗小说，数量和品种跟现在的网文覆盖率差不多。这些小说中有相当多的部分写得很通俗，跟讲故事差不多，你稍微调整语言习惯很快就能看下去，里面的故事也很有趣。看世情小说时不要去想事情是不是真的——所有小说故事都是编的，但里面的语言习惯、器皿、衣裳、风俗一定是真的，如果这些不真，读者没有代入感，小说就没人看。所以我们要看那个时候的人怎么称谓，就要看世情小说怎么描写。中国有一部非常著名的世情小说，人人知道，但千万不要把它当成古典名著、学术高峰，那样就会夹住看不进去了。看一个人说话要琢磨背后的东西，很累。拿它当故事看很简单。这是什么书呢？《红楼梦》，讲世家豪门的故事。你不看书看电视剧也行。红楼梦里当家主母，贾政的夫人叫什么？按照香港电视剧的规矩，贾政的夫人应该叫“贾夫人”“贾太太”，那不好意思，整个园子全是“贾”（假）——贾太太、贾奶奶、贾姑奶奶，当丫鬟的脑袋如果不是超级计算机都活不下去。我们看《红楼梦》就会发现：贾政的妻子叫王夫人，贾政他哥的妻子叫邢夫人，贾政的小老婆叫赵姨娘，贾政的妈叫史太君。王熙凤在里头就夹上了，因为王夫人既是当家主母又是她姑妈，作为侄女必须避尊者讳，不能说那个人叫王夫人你也叫王夫人，于是她被称作“琏二奶奶”，认真的全称应该是“琏二奶奶王氏王熙凤”。在小说的开篇，茶棚里交流四大家族格局时，一个外男都知道他家当家主母姓王，现在天天干事儿的人是他侄女也姓王叫王熙凤。这和我们想象的完全不一样。不光是他们家内部的丫头会叫王夫人、史太君，外面消息灵通的外人也知道哪个夫人姓什么，名字响亮的还会知道她的名字。这才是真正中国人的传统：不管你嫁了谁，你都是你自己。你一面为丈夫家管理财政、相夫教子，做妻子的工作，同时你是自己家族派来的代表，你不可以抛弃自己的姓氏和名字。这和今天西方所说的“既然结婚了，必须是同一个姓氏”完全不同。中国从来就没有进行过冠夫姓这件事。

说实话，我不是特别理解外国人为什么搞冠夫姓。一个女人嫁过去，连自己的姓都没了，她爸爸到底要不要认她？哪天过不下去离婚了，她到底是姓丈夫的姓还是姓她爹的姓？要是还姓丈夫的，后来的太太受不受得了？想想都觉得好混乱。每个人的身份应该既是个体，又从属于某个集体。我觉得中国的传统在这个领域做得很好，把每一个人的身份都标记得很清楚。

今天主要就是排除这样的误会。在我们的日常默认的潜意识和共识中，这样的误会其实有很多，也许以后可以继续找到并把它撕开。
