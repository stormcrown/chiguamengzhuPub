# 第155章 合作婚姻的投资与风险

一定有很多人对我说：“你去看一下《再见爱人》。”然后我去看了一部分之后，还特意去看了大家对它的解析和点评。接着我意识到一件事：它确实可以拿出来聊一聊。

关于《再见爱人》，先铺垫一下：它是一档综艺类的真人秀节目。故事背景设置很简单，就是有几对恋人或夫妻想要分手、离婚，剧组把他们聚到一起，搞点活动，让大家相互了解，再请一些专家和嘉宾帮忙分析问题，看看能不能和好；实在不能和好，就看看怎么和平分手。大致就是这样一档综艺，已经办了很多季。之前没那么火，但效果也不错，基本靠名人效应，看名人的家长里短——吃瓜嘛，都是这样。

但这一季有点破圈了。怎么破圈的？不靠名人，靠素人。这东西就是：努力永远比不上别人的天降神姿。这一季出现了一对夫妻：丈夫是个不太有名的小歌手，叫李行亮；妻子是个素人，大家都不认识，叫麦琳。这对夫妻号称要离婚，参加了这档节目。本来按剧本走，今天冲突一下，明天冲突一下，最后握手言和——稳定结婚率、降低离婚率，这就是社会公益类栏目嘛。结果素人不受导演掌控，中间出现了太多幺蛾子，以至于观众觉得自己吃瓜的小手已经握不住瓜子了，认为应该参与到活动中来。所以很多普通观众开始参与讨论：这个麦琳到底怎么回事？她怎么能够雷人到这种地步，让我觉得世上有这样的人会让我崩溃？同时，也有人觉得她做的某些事，就是自己日常中碰到过的。甚至有些常年批判女权主义、批判“小仙女”的账号专门出来说：“麦琳集中了我想象中所有小仙女的元素，她应该就是个小仙女吧。”同时也出现很多女权主义账号，不断强调麦琳是社会性、结构性的男权压迫的受害者，如何如何开始为女性招募，就这样引发了一大票社会讨论，变成一股热浪。

那在这件事中，我怎么看呢？第一，肯定是吃瓜看。第二，我的观点是：麦琳不是女权，麦琳不是小仙女，麦琳确实是受害者。你说我是暴论，那我就是暴论好了，但后面我会做出自己的分析。为什么我要强调麦琳不是女权，更不是极端女权中的小仙女？因为她做出的所有匪夷所思、荒唐、气人、让人抓心挠肝的言行，并不是建立在“我是女的，所以我要怎样”的前提之下。麦琳的一切言行，前置条件都是“我是李行亮的妻子，所以我要如何如何”——它是建立在这种社会关系下的，而不是女性身份上的。所以我们不能因为有女权者利用麦琳的言论去为女权招募，就说她是女权者。她的行为模式有点像小仙女的自私自利，但原因却不是要当小仙女。

为什么要这样说？因为在麦琳和李行亮的一些对话中，能看得出来：麦琳会跟李行亮强调“我每天做那么多事情，难道这不重要吗？”看到这段对话的人都会意识到一件事：麦琳是做事的人，她在家想办法参与家务，照顾家人。问题在于她确实照顾得不好，但这不重要。重要的是，一个真正的小仙女是要求别人都来照顾她的，但麦琳强调的是“我对你的这些照顾，你为什么不领情？”这是完全不同的心态。所以她做的事情很招人烦，但她不是小仙女。至于女权主义，那纯粹是“我是一个女的，你凭什么让我做家务？凭什么让我生孩子？”但麦琳不光生孩子，而且拼命生孩子，还好好照顾孩子和家庭。所以她也不是一个女权。这是她身上的独特之处。

因此，当我们今天要去分析麦琳的行为方式时，得看她自己怎么定义自己。她给自己安排的社会岗位是“李行亮的太太”，而在这个岗位上出了篓子，所以她才变成了我们今天眼中那个有点荒谬、有点极致、有点匪夷所思的麦琳。为了了解她的心态，我去看了两个人的婚恋史。在其中，我总结出一条规律：麦琳其实是有点追求合作型婚姻的人。我之前聊过婚姻的几种设立目标，其中我很支持的就是恋爱型婚姻和合作型婚姻。而麦琳所追求的，其实就是合作型婚姻。

合作型婚姻的典型特征是：两个人要么并驾齐驱，要么优势互补，总之是要搭伙一起过日子，一起创造生活。合作型婚姻的典型特点是夫妻各自对这场婚姻、这个家庭有所贡献、有所注资、有所投入。麦琳身上就有这种特征，她试图在婚姻中进行投入。

我们来看看他们两人的婚恋历程是怎样转变到今天这个模样的。今天我们看到的这俩，一个这样，一个那样——一个有才华又有家当，似乎不太平等，没有门当户对。但他们刚开始时是平等的。最初阶段，李行亮有学历、有才能，但进入艺术圈这个东西要等人赏识，否则一切归零。麦琳虽然没什么学历和才能，但也进入艺术圈的公司打工。所以两人作为初代打工者时，社会地位可以找平；同时麦琳颜值高于李行亮，在这方面还略胜一筹。所以两人基本达成平衡。

达成平衡后，下一步，他们慢慢打造自己的事业和家庭。这个阶段，麦琳会陪伴李行亮参加歌手比赛，陪他去应别人的邀约唱歌、表演，并提供后援支持——这些都是注资、投入，是合作型婚姻和合作型恋爱的典型表现。这个阶段双方都有投资，而且李行亮的入资并不大，因为刚步入事业正轨的歌手收入很寒酸，支出反而很大；而麦琳的投入非常大，她要照顾家庭、省吃俭用，同时还生育了两个孩子。这个入资对李行亮来说是天文股本，因为李行亮父母双亡，对家人的渴求极其重大，而麦琳在这时给他提供了包括自己在内的三个家人，这离救命之恩也就是一脚油门的事。所以在这个阶段，麦琳由于自己的大股东身份，获得了家庭的绝对话语权。

但这个进程走到后面时，发生了一次重要变化：李行亮起飞了。演艺行业一旦起飞，收入会指数级提高，数量绝对不是普通打工仔三年升一级的那种，而是一起来直接跳上去。这种跳跃式晋升，会让两个人的代差拉开得特别快。这可以类比古代情景：他们之间的关系有点像生于贫贱的一对夫妻，丈夫中举了——这种起飞速度才和李行亮今天的速度相似。昨天大家还是在朝为田舍郎，吃野菜糊糊，你都要谢谢娘子煮得好吃；但第二天暮登天子堂，已经吃过国宴了，你再回来看野菜糊糊，是不是就不太对劲了？代差迅速拉大后，问题就出现了：李行亮对家庭的注资已经是千万级，但麦琳依然是照顾普通一日三餐，且没有大厨级别或营养师级别，甚至还在考虑菜市场每天能不能省下20块钱菜钱——这种注资和配合度完全不匹配。

在这件事上，我们记得古人有一句话叫“糟糠之妻不下堂”，它其实是定在法律条文中的，叫“三不去”。大家都知道古人有“七出”，经常被女权拿出来说随便就可以休弃妻子，那简直不要脸。但实际上“七出”有补充条款叫“三不去”。三不去的主要内容是：几种情况下，妻子绝不可以抛弃、休弃：第一，“有所取无所归”——我娶她时她有娘家，但我想休她时她已经没有家人了，抛弃后她就没处去了，这种情况严禁休妻。第二，“与更三年丧”——男人出去赶考，父母不在了，妻子在家为父母养老送终并服了三年丧，这种对家庭的入股是重金入股，且入股机会你错过就没了，属于约等于救命之恩的入股，严禁休妻。第三，我们娶的时候生于贫贱，之后飞黄腾达了，想把她扔了——这种情况也属于三不去之一。所以你看，古人打了这样一个补丁叫“三不去”。对照麦琳，她其实就是典型的“三不去”中生的贫贱的夫妻：我们是在草根、草莽时结盟的，结果你起飞了，你想把我扔下来？你起于草莽时我是怎么跟着你的？这是很重要的一点。这就是古典婚姻合作制度中打上的合作补丁，禁止撕毁条约。

到这时，我们要考虑一个问题：当设立了三不去的规矩后，就意味对于任何人都有可能产生一种意识——投资，甚至是风险投资。我去找一个才子，认为他有可能中举，这叫天使轮投资，投资量不大，赔了也就赔了一辈子，但赚了就是天赚、爆发期。所以古时候常有这种生于草莽、生于微末时的天使轮投资。典型的天使轮投资故事就是王宝钏。王宝钏这笔投资其实是成功的，但中间出现了18年的套牢期。所以在传统戏曲中，王宝钏一直强调“我觉得这个人行，他肯定能成，我就准备投这个项目了，一把梭哈”，而且她娘家的父母千方百计要她出局，她不同意，一定要守着，不肯割肉。唱戏唱到最后她终于封后那一刻，是很耀武扬威的——我守了这些年，终于得到了这个位置，没有任何不满意，因为这个行为本身就是风险投资。

所以合作型婚姻本身就包含了这种风投类的项目。麦琳当初有没有风投意愿不清楚，也许她慧眼识珠，看出李行亮能红，所以有风投；也许她认为就是彼此搭伙，无所谓。无论哪种，总归是一种投资行为，双方都在家庭中注资。到了李行亮起飞的这一刻，他们的注资开始不平衡了。一旦不平衡，第一时间要调整自己的身份和定位。例如古时候，那些朝为田舍郎、暮登天子堂的官僚贵人，中举后，家里的糟糠妻依然是只会种田喂鸡的农妇，这时她需要太太圈社交，但这个女人做不到，又不能休弃糟糠之妻，通常会打个补丁：娶一个更能做这些活儿的如夫人来替代社交功能——其实就是妾，但因为担任社交功能，别人会尊称她为如夫人。那是当时的补丁。

现代社会不能那样做了，但现代社会给了我们另一个补丁：如果发生这种情况，你可以套现。古时候女人没法套现，说离婚把家当分出来多少多少，做不到。现在允许套现了，所以会出现丈夫飞黄腾达后与原配妻子离婚——你看很多创业期的大富豪，一旦成为行业扛把子，就和原配切割财产离婚，这时他一定要割肉。这就是现代婚姻保障妻子权益的补充条款：你可以套现出局了。

但如果这时不套现出局，两个人继续维持婚姻，同时又没有如夫人这种角色参与，就会进入麦琳现在的场景：李行亮的身家和咖位已经到了储蓄千万的级别，他需要一个全职太太。全职太太也可以在这种家庭中注资，但需要的全职太太是什么样？营养师、厨师、育儿师，还要理财师、社交达人——这几项功能全面都行，那是一个完美级的全职太太，这就是对家庭的注资。如果做不到，哪怕其中一项有特殊专长也行。但这几项麦琳全都不行，导致这个时间她完全无法胜任全职太太的岗位了。

这个阶段，如果李行亮像那些抛弃糟糠之妻的人一样，转头说“不干了我换人”，事情会变得比较简单：分割财产直接出局，套现出场。她就变成一个身家还不错的单亲母亲，同时还是恶劣婚姻中的受害者，标签非常完美，是大家都会同情的对象，你今天绝对不会骂她。但麦琳所面临的困局是：李行亮飞黄腾达后，依然顾及之前所有感情，甚至不断说“我爱你，我想好好过日子，我们要接着过以前的生活”。这时问题来了：麦琳已经做不到这个全职太太需要的岗位能力，但李行亮依然把她安排在这个位置，却对她做的工作不满意。

我为什么一定要强调他们的合作属性？因为他们两口子曾在车内有一段对话：麦琳一直说“我每天做这么多事情，你看不见吗？”李行亮说：“你做了很多事情，但那是你想做，而不是我需要。”他们之间的落差就出在这里。麦琳并不是出来当吸血虫的，她想要在婚姻中注资，并真的做了，李行亮也承认她做了投资，但这个投资起不到作用、没有价值。这才是他们真实的矛盾点。这种矛盾一旦出现，两个人很难在语言、生活、精神上达成共识，因为彼此已完全不平衡。但基于过去，麦琳作为原始大股东、占据最多投资百分比的大股东，拥有话语权。一个拥有话语权的人，这时却变成了弱势者。她需要干什么？有几种选项：一种，突破现有格局，升咖、升级，让自己有能力投资，但暂时做不到。另一种，转变心态，从大股东变成小股东、从主导者变成附庸者——这也可以，作为附庸者，她提供精神抚慰：“你好棒，你好优秀，我好爱你，我好崇拜你。”这种妻子也有，生活可以很和睦。她虽然不会做事，但会哄人。但麦琳过去一直是主导者：“你得按这个做，你听这个的，你必须把这个穿上。”让她直接转为附庸者，转不过来——调岗调不了，追加投资追不上。这时她怎么办？关键她又不方便套现出局，也不想套现，李行亮也不想让她套现。于是矛盾点变成：她身上的可选项非常非常少。

因此，她选择了一个让大家非常厌恶的方式：既然我没办法追加投资、显示我的能干，那我就要把你拽下来，让李行亮的投资减少。怎么减少？一个简单方式是把他家的事业拆了——一旦李行亮从明星岗位跌下来，大家直接平等。但她能这样做吗？不能，这是他的身家性命、家庭真正的基本盘。家庭事业不能拆。所以她要把他从云端拽下来，但最重要的东西又不能拽。那哪些东西可以拽？精神上的投入。婚姻中除了经济投入、劳动投入，还有精神投入。例如一个温婉的女性每天说“你辛苦了，你好帅，你好可爱”，这些都是精神投入。这种情况下，如果李行亮还对家庭有很多精神投入，麦琳就更难受了，因为她更追不上了。李行亮有没有精神投入？在节目中能看到他精神很稳定、平静，不争吵，会照顾她：夜里出去聊天时拿三块毯子，一块给妻子坐、一块给妻子披；吃东西时优先把东西递到她面前。这些都是精神投资。你连这些东西都投资了，请问让我这个完美的全职太太干什么？于是麦琳有一个刚需：李行亮必须是错的。你给我毯子，为什么不给我两条？你给我画画，为什么不给我包包？你给我包包，为什么不给我红色？你给我递咖啡，为什么不给我茶？因为李行亮所有家务上的精神抚慰、服务必须是错的，甚至李行亮必须不爱她。这样，李行亮做错了，麦琳才是对的，她才有正向贡献。

在这件事上，常有人说麦琳精神不稳定，一天哭八百回。哭八百回是真的，但这是投资——证明我受到了伤害，我是受害者，我难受。这件事是我在家庭中的奉献。她为什么哭？不是精神不稳定。我们看他们夫妻在车内的单独对话：李行亮谈到孩子分配问题时哭了、崩溃了，因为他父母双亡、缺乏亲人，一听说所有孩子都要跟妻子走了，直接泪崩。这时他精神不稳定了，但麦琳反应非常冷静，措施及时、敏锐、正确：“别哭了，别哭了，没有事情的，我就是那么一说，过去了，你好好开车。”她不稳定吗？没有掉眼泪，没有歇斯底里，没有任何词不达意。为什么？因为李行亮崩了，精神不稳定了，那么她就可以提供精神抚慰的投资了，就可以来摸摸毛了。麦琳要的就是这个：她需要为家庭注入能量，需要是家庭中的贡献者。她跟其他嘉宾聊天时永远在说“我为家里做了什么，我可会做这些事了”，至于会不会另当别论，但她一定要说。她特别会做这些事，做了好多事，省了好多钱——多少不知道，但她一定要强调这一点。甚至在社交场合，她一定要显示自己是对的、很能干，让所有人都围着她转。这是干什么？她在展示自己是一个很有能力的社交达人，意味着对家庭有投资能力。

麦琳一直在强调这一点。所以我们为什么讨厌她？因为她的个人实质能力与想要的东西之间存在巨大落差，这个落差恐怕不是一两天就能追上的，所以她才会表现出这种非常扭曲、难以理解的姿态。在这件事上，如果她找到一个机会做正向的转变——打破表面的壳子，突破自己，找到真正的长项，发挥能力，最终找到自己的精神锚点，她就不光是李行亮的妻子，还是麦琳本身，是某个家、学者、研究员或岗位上的女强人——那么她就不会再表现成这样荒谬的样子。但暂时她还做不到，就会继续荒唐。这是她可能发生蜕变、变好的方向。如果没有，她很有可能继续用这种扭曲的方式让自己显得很好。

所以在这件事上，要让我猜，我觉得他们两口子可以过很多年，不会离。他俩怎么离？多么适配，多么登对！要知道，在婚姻关系中，一个想要当照顾人的老母亲，另一个就想要听老母亲话当乖儿子，这种婚姻关系可遇不可求。你说李行亮现在很优秀，出来后分分钟能找小迷妹，也许能找到几个小迷妹，但她上哪去找一个妈？他需要婚姻中有这种关爱，像母亲一样带有控制欲的爱，别人给不了。他们两个人在这一点上完全匹配。至于麦琳生活习惯不好、不讲卫生，李行亮都已经忍了十几年了。他们两口子配合得很好，彼此毫不在意，咱们旁人不要插嘴了。总之，我觉得两个人很登对，根本分不开。旁人去看时，他们之间的吵嘴是他们内讧的一种方式——越吵嘴，李行亮受到的打击越多，他们的婚姻就越稳定。除非哪一天李行亮的弦绷断了，但麦琳很冷静，不会让它真的崩断。最后就是在皮筋上跳舞，两口子越绑越紧，棒棒的！我很欢迎这种局面出现，当然更希望他们真的能往好的方向走。
