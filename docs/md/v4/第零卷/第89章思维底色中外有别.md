# 第89章 思维底色中外有别

中外思维为什么有那么大的差异？这是一个小伙伴给我提出的问题。故事的背景大致是，他发给我一个视频切片，讲的是潮妈和她的粉丝之间的一次互动。潮妈说，她在海外生活了这么多年，发现外国人和中国人（包括海外的华人）之间有很明显的思维差异。最典型的就是对“得寸进尺”这个行为的评价完全不同：在中国，我们普遍认为“得寸进尺”是一个负面评价；而在西方人那里，“得寸当然要进尺”，不然呢？你都给我了，我不是天经地义地拿着吗？有便宜不占是错的。所以这种思维底色的不同，引起了大家的好奇：为什么会有这种奇怪的差异呢？

其实我们过去常说，“仓廪实而知礼节”。如果放到几十年前，我相信没有人会提这样的问题，因为那个时候我们这里也真的是你争我抢的。现在很多小年轻可能没有体验过：曾经在前几十年，我们的资源不够好，甚至购物、买吃的、买菜、买肉、坐公交、过马路等等事情，常常需要你争我抢。到今天你去公交站的时候，也常常会觉得一些老年人好不讲规矩、好没有礼貌，追着赶着就往前面冲。为什么？就是因为那个时候资源实在不充足，如果他不抢，他可能会上不去车，最后就形成了一个行为习惯：他总是要往前面挤，总是要去插队。你听起来是不是觉得，我们这里也是你争我抢的？但是今天，年轻人就不同了。

我们的问题来了：明明都是在“仓廪足”的环境下，西方人已经“仓廪足”了三、四百年了，而我们这里才三、四十年，初步的“仓廪足”，对吧？两相对照，我们的思维依然有很大差异。怎么回事呢？可见这不完全是生产力和经济水平进步的产物，即使经济都进步了，大家的想法还是不一样。

其实在这件事中，我们得为那些老年人抱个屈：他并不是不知道什么叫做文明，也不是不知道什么叫做礼让，他是实在不抢不行。一旦脱离了那样极端的环境之后，大部分人很快就扭转了观念，开始有序地排队了。因为他很清楚地知道：礼让才是有礼貌的，谦逊才是有礼貌的，秩序才是有礼貌的，退一步才是海阔天空的。他过去是被迫地抢，并不是心里觉得抢是对的。这才是一个很重要的文明底色。

所以中国这里虽然也有过争抢，可是我们的道德、价值观和评判标准从来没变过。而这种思维的底色来自于什么呢？它来自于我们更加漫长的历史上“仓廪实而知礼节”的积淀。所有的秩序、所有人类对于文明的体验，其实跟生活环境之间一定是有挂钩关系的。我们这里常说“仓廪足而知礼节”，其实是我们已经搞清楚了礼节是什么，等到“仓廪足”的时候，我就可以去兑现了，并不是那时候才出现这种礼节。所以这个礼节是啥？恐怕要倒推回更加古早的过去，才能找到它从哪里开始。

我觉得可以从“愚公移山”开始。愚公移山里面最重要的一句话，也是我常常在给别人聊这个故事时讲到的，就是“子子孙孙无穷匮也”。这句话的背景牵扯到一些不太容易理解的点：为什么会子子孙孙无穷匮也？你怎么知道子孙会听你的话？山神又为什么会相信你说的子子孙孙无穷匮也？这太奇怪了。愚公老爷子不在了之后，后世怎么会真的就一直挖下去了呢？其实这个疑问就代表了我们的文明本身。因为我们过去的几千年甚至上万年的历史中，我们的祖先一直在做这件事情：通过有秩序的、团结的、无数人、无数代的人一起来共同做一件事情，让它产生强大而且长久的生产力。

就好比我们的都江堰。很多人说都江堰有2000多年历史，很厉害。是的，但是2000多年以前，李冰父子把它建成了之后，就放在那儿了吗？不是。事实上，在这2000年里，我们不断地修缮，不断地根据水文条件的改变去进行调整，所以到今天都江堰依然能够发挥作用。你看，即使是一个建筑，在漫长的2000年里，我们依然实现了一件事情，叫做“子子孙孙无穷匮也”。我们的各种功臣、各种王朝更迭、制度建设，所谓的科举、考试、学习、四书五经，方方面面的所有东西，我们其实都延续了很多很多年、很多很多代。这些全部都是“子子孙孙无穷匮也”。这叫做什么？这叫做远见。而远见来自于什么？远见来自于对未来的稳定预期。这种稳定预期来自于什么？来自于有规划、有秩序、有建设。这就是我们文明的底色。

我们在过去漫长的岁月里，无数代祖先都在进行一个统合型的建设，想方设法的让所有人拧成一股绳，朝向一个方向走。这个过程当中，我们还会进行一些社会性的筛选：淘汰掉那些不服从管理的、那些跳脱的、那些博取私利不顾公众利益的群体，把他们踢出去了。所以我们的制度建设也是围绕着它来的。因此我们今天说“一旦资源禀赋足了，就可以仓廪足而知礼节”，礼节是怎么来的呢？就是因为礼节是在我们漫长的岁月当中共同缔造的一个社会公共秩序。这个公共秩序是为了集中力量办大事，是为了让我们团结一心，是为了让我们这个族群“子子孙孙无穷匮也”而产生的。

这才是我们中国人——无论是生活在本土还是跑到海外去，只要他是中国家庭出来的、华人家庭产生的——就会在心里有一种合作的意识，有一种配合的意识。那种“我给你一点什么，然后我再获得一点什么”的交换意识。这就是为什么中国人很反对得寸进尺，因为得寸是对方让的，这个时候我们应该合作，而不是冲上去把剩下的都抢过来。我们不支持这种劫匪的行为。

这个时候我们就转过来看中国以外的世界。我觉得这个词很有趣：我们在谈这个问题的时候，说中外的思想差距。但是中国我们知道是一个中国，而外国是200多个国家，我们直接把人家一锅端了、一锅烩了，认为他们都是这个样子的。那还真是如此。所以他们那里为什么会是这个样子的呢？如果说我们这里是由于数千年、上万年的高秩序、高组织性的社会所产生的礼节，那么中国以外的社会其实就是由于漫长的岁月里始终都没有超脱出丛林竞争法则，所产生的零和博弈和社会达尔文主义。

你想一想，一个人为什么会觉得从别人那里拿到东西是天经地义的？什么情况下，我从你那里拿了一个桃之后，还得把你的一筐桃都拿走呢？是的，这是一种非常野兽的思维方式。就像我们家里养的阿猫阿狗，就是这种想法：你一旦对它特别好，从来不限制它的行为，从来不惩戒它的错误，它自然而然地就觉得你欠它的，你要为它服务。这就是一个很正常的趋利避害的本能：有好处我就占，如果有更多的好处，我就去占更多的好处。这是一个比较原始的思维方式，在自然界普遍存在，在中国以外的人类社会也普遍存在，高度一致。

所以我们会注意到一件事：几十年前，我们这里有很多公知盛赞西方人的进取精神——所谓的冒险、自信、独立、勇往无前。这些精神建立在什么基础之上？就是他们所鼓吹的海洋文明。这个海洋文明是怎么体现他的冒险和奋勇争先的精神的呢？就是驾着船出门打劫。是的，这种“我必须要得寸进尺”的精神，其实本质上面就是这种劫匪精神。在这一点上，他们也是一以贯之的。它源自于历史维度上的文明进程不同。

虽然这些年以来，他们一直给自己编撰伟大又辉煌的古代文明，甚至把自己打造成了世界的文明源头，然而真正有据可查的历史上面能够找到的，其实是中世纪快要结束的时候才开始的。在这段历史当中，他们真正进入到“仓廪足”是在工业革命之后。在工业革命之前、文艺复兴之后的这段时间里，那里还践行着一种贵族政治和农奴经济体制，这种管理方式实际上是上不知下、下不知上。这种情况，你想让他去打什么配合战是不太可能的。所以我们在他那里也可以很明显的看到这种倾向：我得把好处占住，抓不住的东西我很可能就失去。当一个社会长长久久都处在这种社会制度下面的时候，所有的人的精神底色必然是“得到的就是天经地义的，没得到的要想办法去争取”。过去辉煌的时候，这叫做勇往无前的进取精神；而现在，西方中心主义沦落，我们开始反思他们的错误的时候，就直接挑出来了：你这玩意儿也太不像样子了吧！

这就是今天为什么很多年轻人意识到中外思维的差异——其实并不完全是今天才知道差异，而是今天才知道，原来我们才是文明的那一个。在这点上，我们再次为所有敏感地察觉到这种差别的年轻人点赞。知道我们强大的根由，是很重要的事儿。秩序、文明、谦让、礼貌，大部分的时候都是有用的。
