# 第176章 士农工商：谁是国之根基

嗯，士农工商啊。好，士农，士农工商。呵呵，工排第三是吧？你觉得为什么要这样排呢？哈哈。

古人在进行这些阶层划分的排序时，讲的是什么？讲的是什么？你得弄清楚这件事。它不是一个什么所谓的客观描述，啊，这个排序不是一个所谓的客观描述。这是一种政治权力的等级划分，是强行规定的，是一个硬性规定，并不是一个自然而然产生的东西。

那为什么士要排第一？士是什么？士是社会管理者。一个国家把社会管理者放到最前面，相当于是给他提供了国家级的保障，就是一个铁饭碗的意思。今天你去考公，去参加国考，就是古时候的那个士。士这个群体为什么要有这个？你今天为什么参加国考？你不就想要那个铁饭碗吗？你不就要那个保障吗？士的这一套待遇就是属于保障型的待遇。所以为什么把他排到前面去？因为他是社会管理层。

然后为什么要把农排在第二个？不是因为农民本身经济或者技术能力有多么强大，而是因为农为国家的立身之本。我就问你，今天是不是这样？不管今天我们把中国吹成一个什么顶破天的工业国，我们确确实实实力越来越强了，但是不管我们怎么说，我们什么工业大国，我们每年的开国第一——每年啊，每年的第一个头号文件是干什么？农业。你自己去看，我们每年的第一号文件全部都是农业。为什么？农为立身之本，农为立国之本。没饭吃，什么都是归零。所以哪怕农民没有多少的利润，农民没有多少的经济实力，他依然要在社会地位上面对他进行一个强行的拔高定义。如果经济的实力不行，我就在政治实力上面给你补强，要你成为国家的真正的建设者，让你拥有回报。那你说能不能在农民兄弟身上提供经济回报呢？不能，因为你要他用农耕的方式获得超额利润，就意味着粮食必然涨价，那就意味着农业系统崩盘，意味着国家崩盘。所以农业是什么？农业是那种很辛苦、获利微薄，但是对这个国家至关重要、是性命之忧的这一种行业。为什么要给他至高的政治地位？因为它真的是国家根本。所以这个是我们治理方略的一部分，而不是所谓的客观地位评价。

然后那个工啊，工是什么？它包括了科学家、技术人员、工匠、各色人等。然后其中的顶级大工、顶级研究者，它实际上是会被划分到士那个阶层的，就是社会管理者。和今天其实是一样的，今天的顶流的科学家是不是拿了津贴的？有院士补贴的，是这样的吧。那么那些普通的技术工作人员呢？他个人的能力能够创造的经济效益，能够获得的收入比较好，于是我们给他定了第三等的政治地位。就像今天的码农，你的收入已经很好了。那这个时候怎么确定这个社会的重要性？就把它排在最根本的农业的旁边。这个就是工。

至于为什么要让商垫底？是因为商比较弱吗？不是，是因为商比较强。商的这种贸易形式，这种买低卖高的形式，非常符合个人的利益的需求。你只要抬高他的地位，大家都会去做买卖，为啥？来钱快、获利快、容易。而如果全民都倾向于行商的话，那么这个国家就会走向极度商业化。一个国家极度商业化，死的要多快有多快。就是今天的阿美。对啊，就是今天的阿美啊。我们古代的重农抑商，反映了商强农弱的事实。其实无论你放到什么样的时代里面，商必然强于农。商强农弱是一定会出现的客观事实。你希望农强是不可能的。你想让一个人靠种田就发大财，那你那个米粒子得变成黄金做的，那还让人吃吗？那就直接动摇了国之根本啊。所以它是必须利润低微的，农业是一定要利润低微的。农业的利润一高起来一飙升，那个什么阿美一刀一个的鸡蛋，奇一颗怎么吃？是不是那个天价的米、天价的面让人活吗？农业啊，农业一定是一个低利润的行当。但是你低利润就代表了他低收入，那怎么补充它？所以你要为他提供专项的保障。

如果说清晰的、有理性的商人，他会知道他自己的根在什么地方。什么意思呢？就是所有的商人，他做的是一个流通与贸易的工作。这个事儿重不重要？重要啊。有商人进行流通的话，物资会更加的顺畅，叫做无商不富。有商业的推动，有资金流的推动，才能够让它运转得更加丝滑、更加有效，这是一个客观贡献。但是也同样的，因为它这个样子的运转方式，使得它聚力的能力特别的强。而一旦它聚集到了过大的力，太强大了，叫什么？我们古人把它叫做“富可敌国”。一个商人一年的利润如果和国库的收入是一样的，那这个商人能够调动社会资源的能力就和国家的官方是一样的，这个就叫做富可敌国。当它已经可以随时随地地让这个国家可生可死的时候，你要求他必须要有仁爱之心啊，必须要有贡献于万民的这种愿望？做梦的吧？不可能的。因此，当他富可敌国的时候，他就会呈现出一种敌国的心态，他自己就变成了那个敌国。所以为什么要重农而抑商？因为商太容易聚力了。力聚到了一定的程度，它就可以驱使其他的力量，这叫做“有钱能使鬼推磨”。而当它能够驱使的力量越来越多、越来越大，它会变成什么？他自个儿就是一个国。而一个纯粹的以商业利益为诉求的国，是个什么呢？就是今日之阿美啊。那老百姓就别想过好日子了。

所以为什么我们今日的管理是合宜的？为什么我们今日的方案是合宜的？因为我们今天所订立的目标叫做为人民服务，叫做共产主义。它本身是包含了经济分配与再分配的，它要保障的是这个社会运行的基本的利益，而不是纯粹的经济利益。当一个社会按照纯粹的经济利益的方向去工作的时候，就是今日之阿美。你按照纯粹的经济利益，偏远山区迁什么店？搞什么公路？亏不亏啊？我把公路建到山上面去，我把电缆迁到山上面去，那个庞大的成本，就为了你那三瓜俩枣的，你交了几块钱电费？这个就是一个商业思路。当你用商业思路去运转它的时候，就会跟今天的阿美一样。他很多的小城市、小村镇里面，没有网络，那个电不保供，网不保供，很多的东西都运不进去。为什么？成本高啊。所有的这些东西，它不排都是有成本的，成本高昂了。作为一个以经济利益为第一诉求的人群来说，他就不要这个。嗯，啥？他们人口密度不一样，整体的人口密度一定是不一样的。但是啊，咱们又要说到但是了。中国也有人口密度很低的地区，我们人口密度很低的地方，我们也是实现了三通的水电，全部都是保供的。这些很偏远的地方全都有这个东西。我之前去到那个高原上面的时候啊，有一些真的是茫茫草场，你知道吧？隔着老远就看见不知道多远的地方，我也搞不清楚距离了，反正就孤零零的一栋房子。你开上百公里，旁边都找不着人的那种地方，老远的位置就一栋房子，有电线啊，旁边就竖着电线杆子呢，有电线牵过去。你知道这是什么成本吗？相当于路上所有的那些线缆全都是为他一家铺的。商业单位会干这种事情吗？商业单位是不可以干这种事情的，那真的是亏到底裤都没有。只有以人的利益为导向，你才会做这种事情。以商为导向，这种事情是坚决不能干的。公路都铺上去了，对对对。公路都在，有公路，有铁路。不然的话怎么过去呢，是吧？就我这种啊，难不成你让我走过去吗？不可能的是吧？就我这种体能的，我也不能走过去啊。有公路啊，有公路，有水有电。所以这个就是整个在管理目标上面的不同。

嗯，啥？为一家花那么多钱，也是国家的钱，也是全体老百姓的钱。你又委屈了？你又委屈了？你知道人家待在什么地方吗？为什么苦寒之地要让他在那个地方生活？只有有人居住的土地才是我们的土地。不要觉得你画一个边境线就完事了啊。土地不由你在地图上面画边境线就决定的。满清就喜欢在地图上面标边界线，标完了之后，这地儿都是他的了？无人区你画什么边界线都不好使。简单来讲，就那个东三省，在满清的时期，除了辽东一少部分的熟田有种植、有人口之外，其他的地方被他迁成了一个无人区和半无人区，只有少少的几个部落流荡在那个上面，而且他不允许别人过好日子，不允许别人驻扎下来。你知道这种操作造成了什么样的恶果吗？就我们旁边的棒国啊，就那个棒子国，在那个时候不断地迁徙人口到我们的东北的土地上面来。人家迁了人，种了田，设了村庄，设了管理，记了档案，画了地图，设了路标，安放了界碑。然后你大清划一条界，说东三省是我的龙兴之地，那里是我的？自古以来好使吗？半拉吉林都已经被人家棒国的老百姓住上了，住得踏踏实实的，上面还有官僚管理，人家还记档案本，人家跟我们一样是有记载历史的啊，虽然是半路学的，那也是有传统的。你知道那个时候大毛跑到东三省进来的时候，曾经有相当长的一个阶段，把吉林省的大部分地方都划给了棒国吗？你以为是为什么？因为人家在那个地儿住了人呢。所以为什么我们要把电线牵到那些鸟不拉屎的山旮旯里面去？有人住的，那才是我们的地。地上面的人是我们的人，这片地才是我们的地。你光画地图不好使。就因为我们把路修过去了，电线架过去了，那个地方有老百姓住着，那些老百姓还每天按早上晚上呢，一天三顿饭的烧着，有炊烟升起，因此那个地方才是稳稳妥妥的我们的国土啊。你这会儿还觉得亏了啊？如果说你觉得亏了，你觉得不应该出这个钱，不应该在那个地方搞这个建设，你以为人家就非得住那儿吗？别人也是长了脚的。没公路，没水没电，日子过得苦，他会干什么？他会向内地迁徙，然后那个地方变成无人区。没有人居住的地方，你凭啥说是你的？隔壁人家人口大爆炸，直接就迁了人过来，一天迁一百，一年要迁多少人进来？然后村子设了，学校设了，工厂设了，你说这个地儿就是你的？那么大脸的吗？啊，戍边。我们历朝历代任何一个时代，你要想真正的掌控这个国家，必须要做的一件事情，就叫做移民实边。所有的边境地区都不是什么很宜居的地方，因此呢，他就要设定一套规则，一部分是强制迁徙，一部分呢是优待政策。当然了，我不是说我们现在的这套规则毫无瑕疵，它做的就是对的。之所以我们现在在很多的边陲地方只有少数民族，也是因为在规则制定上面有偏向性。我们正在调整这个，我们正在调整。就是所有在那的人，只要是在相应地区居住的人，都拥有相应的特殊的保护、特殊的福利，这个就是一个专门的实边政策。历朝历代都有，它对于保家卫国非常要紧。这个世界上没有任何一个国家可以保护它的无人区。

嗯，啥？给再多钱也没用。你，你别闹了啊。我们现在啊，我们现在去的人一点都不少。你不爱去是你的事儿，但是我们现在为了这些政策而去的人一点儿都不少。嗯，你自个儿不爱去可以不去，也没有人强迫你。但是我们每年都有人去，比例肯定不高。要多高的比例啊？你要把那个边境线上面造成像珠三角一样的密度吗？啊，不可能啊，人家那个地方的环境也不支持。嗯，啥啊？老家有1/5去了新疆？好的啊，很好啊，很好。我们现在在那个边疆省份都有专项的扶持政策，每一个地方呢不太一样。然后目前而言啊，有一些地区设立的那个政策还不错，所以确实还是有一些吸引力的。我们之前几年的话啊，大概十几二十年前的话，还有一波去广西的啊。你们以为广西的密度已经很高了是吧？但实际上十几二十年前还专门移了一批广西到新疆的。那是从建国之后就开始有了，是吧？新疆生产建设兵团，其实就是这个有代表意义的一个，呃，属于这种性质的实编。实编的性质。而且我觉得有的时候呢，就是在想的时候啊，你觉得你考虑的那个问题呢，可能是吃太饱了。就是可能生活条件比较优越的时候，你老觉得那个就叫做苦生活。实际上呢，如果说你的条件没有那么优越，你会发现在今日之中国，边陲地区的生活条件并不差。
