# 第188章 印度赢学、美国MAGA，谁在悄无声息改变这个世界

新一轮的印巴冲突给中国的军事产业打了一波国际广告。在这之后，我们的生产线就可以全力开动，万邦来财了。

当然啦，在这个事件当中，我们就会发现，印度人这边的飞机连国境线都还没有飞出去，就已经被对方轰下来了。但是呢，在印度人那里，这个叫做英勇作战，并且完成了战术的指标啊。可能他们的指标和正常人不太一样。

于是，当这一些看上去像笑话一样的反馈信息出现的时候，很多中国人就产生了这样一种想法：说这个国家真是又菜又爱玩，真的是赢学入脑，每天都要给自己找借口，死鸭子嘴硬。当我们产生了这样的想法的时候，很难免地就会产生轻慢之心。所以今天呢，我们就得来分析一下，印度人的这些奇特的思想到底什么模样，到底怎么产生的，它会产生什么样的后果。我们真的可以这样傲慢地去看待这个邻居吗？

说到这儿的时候，我们不得不把时间轴倒推一下。在上世纪的60年代，教员——那个可以拨开一切迷雾、直指本源的教员——他说：“我没想明白印度为什么要打我们。”是的，连教员都搞不清楚这个国家他到底在想什么。所以他的思维的鬼畜之处，压根不是你简简单单一句“赢了”就可以解决的，也不是简简单单的一句“又菜又爱玩”、“死鸭子嘴硬”可以直接弥补抹平的。咱们想要真实地理解一个印度，得从他自古以来去说起来。

那么在近几个月，很多的赢学分析专家这里啊，对印度的解析呢，就是：这是全世界赢学的发源地，赢学思想的集大成者，赢学国家的宗主国，万赢之赢之国啊。基本上是这样的一个论调。为啥呢？因为这个国家他就喜欢说自己赢了又赢了，飞机掉下来是赢了，饿肚子还是赢了，恒河水拉了肚子还是我赢了，不管怎么着他都赢，表现出来的叫做赢学。以至于呢，很多人就会把今天的美国那里的MAGA和这个印度的赢学画了一个等号，觉得都是赢学嘛，大赢小赢这区别而已。其实这两种赢学是完全不同的。所以我们今天要解析的不是阿美的MAGA、赢学，而是印度的这种赢学本源、赢学本赢，他的“赢”到底是怎么回事，能够在一切的失败当中看到胜利。这是一个极其了不起的本事。怎么做到的呢？这就不得不说到印度的发展史。

严格来讲，我们说不清印度的发展史。为什么？因为作为一个号称的文明古国，他并没有真正的成文信史的记载。印度历史上面真正的成文信史就一本《大唐西域记》，这是玄奘法师赴印度河流域留洋归来之后，被迫口述、由他人书写的一本游记啊。这就是印度整个的古典历史当中最可靠的一本有信源的历史书了。为啥？这也跟他的英国之国有关系啊。就是虽然没信史，但是呢，这个地方的大致发展脉络居然能够梳理出来，和其他的任何地方都不一样。为什么？因为他们有非常清晰的种姓隔离制，而这种种姓隔离制会导致上下不能通婚，彼此之间甚至人种来历都完全不一样。这样的形式，我们就可以用分子生物学的溯源法去解析，诶，你的历史进程是怎么演进的。所以细节虽然不知道，都是传说，但是大略那还真科学。

这个科学路径是怎么走的呢？在印度的种姓制内部，我们知道最高等级——婆罗门，说是他们以前的神权阶层；刹帝利，是他们以前的王权阶层。其实呢，当你去观察印度人内部的姓氏的时候，你就会发现一个迹象了：在他的高种姓人群当中，姓通常很短。例如说我们都知道的一个著名的电影明星叫阿米尔·汗，这个姓是什么？其实它就是王的意思。记得吗？成吉思汗。唉，这些游牧群体的王经常被称之为汗王、汗。所以印度那里这些短的啊，跟过去的军事、跟神权、跟王权相关的姓，就是他的高种姓。而那些复杂的，一嘟噜、一串老长的，而且内容涉及到工匠、涉及到干活、涉及到扫地的这种类型啊，那就是低种姓。所以他的这些种姓划分跟他的职位划分之间也有关系。不同的时期，不同的职位，最后呢，就会在今天留下他们的姓氏。

种姓制啊，他是一成不变的。放在那个地方之后，就对人直接产生了影响。理论上说啊，一个人想要向往自由，那自然而然地就会觉得种姓制不太好。但是呢，在印度那里适应了种姓制的人群，心中种姓制是天经地义的事儿。像今天之所以会出现印巴冲突，其实就是因为多多少少总是有一批人，在听说人可以和人平等的时候，感觉到心向往之，于是呢，就会受到伊斯兰教的感召。那么在这片土地上面就出现了种姓制的群体，和信了伊斯兰教、于是跳脱出种姓制的群体发生了对峙。那么当年英国的殖民者就在中间画上了一条国界线，然后让你们这些信种姓的和不信种姓的开始对轰。他们原本是一家人，就用这样信念的方式就被撕裂开了。

但是呢，我们就会发现一个小故事。在前几年的时候，印度这里曾经有一个印度教的男性，号称自己想要追求人人平等，于是去皈依了伊斯兰教。伊斯兰社区呢，很开放式的、宽容地邀请他入住之后，他带着家里面的七大姑八大姨就搬过去了。过去了之后，他开始推行伊斯兰种姓制：高等级伊斯兰、中等级伊斯兰、低等级伊斯兰。嗯，所以你看种姓制这种东西是一个虽然看上去很离谱，但是对当地实际处于制度之中的人却极有魅力的一项制度。因为它可以让人非常地舒适，随时随地转换身份，随时随地自圆其说，随时随地觉得今天吃的苦爽歪歪。为什么全世界没有几个地方的人能够理解他们，就是这个原因。因为能够实现一个社会，像婚礼蛋糕一样的，18层不停地切割，这在世界上任何地方都没有。而之所以产生这样的局面，跟南亚大陆这个地方独特的地形有关系。它是一个半封闭的地形，按说很安全。然而在它的北部却有一个山口可以走进来，而由于这个地方呢，既宜居又不太宜居，以至于没有形成一个统一的政权，所以从来没有人去堵那个山口，山口始终都是开放着的。最后就造成了一个结果：永远都由崭崭新的殖民者进来，而殖民者来了之后，就会进入到这一片既宜居又不太宜居的土地上面，从此堕落成了一个不太善战的群体，于是就感召来了下一波的殖民者，开始压起了蛋糕，压起了地层。

这样独特的地理环境形成了独特的社会结构，而这种社会结构又形成了独特的人文心理。这个人文心理啊，千万不要用中国人自己的思想方式进行揣摩，你揣摩不到的。为什么这个样子？我们来看一下。中国人也经历过奴隶统治，包括说在满清的200多年，中国绝大多数的人都是奴隶，然后被八旗直接高压统治着。但是那个时候的中国人的选项是什么呢？我们蛰伏下来，有机会的时候革命，扛不住的时候再蛰伏下来，我们等机会。因为这个社会上面只有两层，只有两层意味着当我是奴隶的时候，我可以选择苟着，也可以选择革命，只有两条道路，这就很单纯，非常好选。

但是放到印度那里就完全不一样了。一个人，他的下面有好几个等级，它的上面同样也有好几个等级。最后形成的局面就是任何一个人，你看上去他只是一个人，然而他的灵魂分成了两面：他既是奴隶，又是奴隶主。而且这两个身份在他的身上有机地交融，完全密不可分。对他来说，这个事儿天经地义。你跟我笑，你一定是我的奴隶，我给你一份300块一个月的工作，你赶紧把我们家的屋子收拾好呀。你扇我巴掌，你一定是奴隶主，你是不是很爱我？我干活不错的哟，给我一份工作吧。对他来说，这件事情是无缝切换的。这意味着什么？这意味着一个印度人，他拥有超强的弹性。弹性到了什么程度呢？就是他到哪里都能活下去，他面对到什么环境，他都可以快乐。这种快乐感是我们不太容易接受的一种快乐感。

例如说，之前有很多去印度旅行的人发现，说他们当地的人会去那个很可怕的恒河里面喝水、办仪式，而且觉得这里很神圣，反正百毒不侵了已经。一般其他地方的人看了之后，都会感觉到精神上面有点崩溃。但是对于当地人来讲的话，这很正常，很合理。然后呢，还有一些公知说：“你看我去印度参观了的，人家虽然有贫民窟，但是他贫民窟里面的人非常地平和啊。日子过得不好，但是非常地平和，非常地自信，非常地快乐。”说他们有讨饭的人，但是他们讨饭的人坐在街头的时候，只是静静地等待赏赐，等待施舍，既不愤怒，也不急迫。怎么样？最后一句话：平和。

平和的印度人，非常擅长于在生活的细枝末节当中去寻找快乐。例如说他们喜欢用那种超级浓香型的洗发水，因为可以节省用量的同时，全身都是喷喷香的。当然，作为中国人，我们对此大可不必。如果你头发出油严重，头屑多，头上起痘又痒又疼，用手一挠，衣领、肩膀都是白屑，不妨试一试我最近在用的这款赫恩发力洗发水。它拥有去屑加控油的报告，去屑率高达99%，搭配了OTC去屑成分，像是头屑吸尘器一样，把头屑、头油打扫得干干净净。平时掉发严重、发际线还越来越高的，它就专门加了姜根、何首乌、人参等五重本草植萃，有效防止断发，洗头时也不用担心掉一手头发了。而且它是用贵价氨基酸来替代硫酸盐，零硅油零硫酸盐，添加pH显示6.0的弱酸性配方，洗得干净还不伤头发，不像市面上其他加了硅油、硫酸盐的洗发水，容易堵塞毛囊，导致头油掉发。香型是高级好闻的木质调，清新香柠檬与温暖的柏木交织扑面而来，阳光干净的少年感，让他一闻就爱上。赫恩是十多年的老国货了，现在品牌还出了礼盒装，评论区领取专属大额券，买一套更划算啊。

印度人为什么平和？你去想一下它的社会分层结构，就会比较好地理解这个问题。因为当他的社会分层已经变成了这样的垒蛋糕，你要革命的时候，怎么个革法？一个人说我要革命，你上面要推翻17层楼，绝对不是一个阶层。而且这17层楼层层各有不同，每一层的软肋短板都不一样。你一场革命要推翻17层楼，你就觉得你革不革得了吧。而对于稍微二层、三层往上面走的呢，尽管他上面有一些人在压榨他，但是他下面有很多的人任他压榨，他压根儿不想革命。于是产生的效果就是：革啥呀？不是挺好的吗？我被奴隶主扇了巴掌，我回来就扇奴隶的巴掌，自圆其说，完美自洽。于是就平和而且快乐。

那么，为了适配这样一套社会的管理规范，印度人呢，就为此而开发出了一整套的宗教信仰系统。在给关注过印度的人这里啊，你一定知道印度的宗教信仰纷繁复杂，他们的系统内部充斥着各色各样的神明，你数都数不清楚。犹如天上的星辰一样的一个街区信猴，下一个街区就可能信蛇，再下一个街区信的就是牛串。一个地方，你的信奉，你的忌讳就完全不一样。这些看上去是一致的信仰逻辑，但是内部却纷繁复杂的这一套东西，干嘛用的？社会管理用的。为什么要用这样的方式来进行管理？因为人在他的本能当中，总有一种想要诉诸于自由的愿望，所有的人都有。但是当你想要这些东西的时候，又遭遇到了残酷的压榨，你就有可能产生反抗之心。而印度呢，由于它的多层次的压迫管理，使得这样的压迫变得非常非常地残酷。那这个时候怎么能够消弭这个？怎么能够解释我们这种压蛋糕、压地层的社会？你总能够在理论上面自圆其说。于是他采取的方式就是：我们是一整套的神仙系统，但是呢，唉，你在不同的社会档次，你是这个伟大神明身体的不同部位演化的。头顶演化出来的就是高等人，脚皮演化出来的就是低等人，天经地义。然后如果说你想要翻身，你该吃的苦就必须吃透它，你把你这个脚皮上的苦全吃透了，下一次轮回的时候，你就可以到波棱盖儿了。它的基本的原理是用这样的方式去解释：他的社会为什么会变成压地层的模式，你为什么会吃那么多的苦，并且告诉你如何翻身呢？熬过去，快乐地享受你所有的痛苦，然后你就可以享受高等级的快乐了。

当他的这一整套行为逻辑已经落地生根之后，是不是你就更能够理解：为什么他在历史当中没有真正的成文史了？这样的国家，你记载成文史，是不是要等着别人造反呢？让别人记得自己以前曾经是绝顶的上等人，你一开发出这种记忆来，他就要开始反水了呀。所以它不能够有真正的成文史，而只能够有经文复述之下的故事。先是讲神仙，然后再讲人的故事。所以在他们的历史传说当中，有一些王分分钟就可以活500年，然后过了几百年之后，他又活了500年。这是什么？这就是轮回。

当他的整个社会系统内部都是这种模样的时候，他就会培养出来犹如弹簧、犹如变色龙一样的印度人。我们就会发现一个现象：今天很多的中国人说印度又菜又爱玩，敷衍啊，已经看待别人的时候产生了轻慢之心。我再次提醒，这种轻慢之心要不得。他是我们的邻居的时候，他确实又菜又爱玩。但是人是有脚的，脚是会走的，人可以迁徙的。一旦发生迁徙，那就不是又菜又爱玩的问题了，你就要考虑印度人身上独有的特色：弹性高级。弹性高级的适应性，他因为既是奴隶，又是奴隶主，所以他身上能够把这两种东西兼容得非常完美，让任何正常人都理解不了的完美。以至于他在任何社会当中都可以生活得如鱼得水。顺利：我一定是神赐的，我一定是这里的上等人；不顺利：你一定是我的奴隶主，你必须要照顾我。所以他对自己没有任何道德上面的要求，没有任何道德上面的约束，他只有既是奴隶又是奴隶主的兼容型的心态。进而造成了他到哪里生活都可以，生活得很开心、很快乐，并且把他的这种快乐辐射出去。幸福感是会传播的。当你永远都看着他身上跳跃的、快乐的、舞蹈的、歌舞的——演什么苦情戏都可以在里头跳上一个小时的——你一定会感觉到，哎呀，平和，太开心了，我想向你学习。你学着学着你就和他一样了。最后你就会发生一个现象：一个公司里面进了两个印度人之后，这个企业就会变成一个印度人的企业，这个企业里面的文化就会变成印度的文化；一个国家进了一批印度人，之后这个国家就会变成一个印度人的国家。尤其当有一批印度人进入到了管理阶层的时候，它整体的社会氛围和行为的方式就会越来越印度。所有的这些影响力、这些感染力都来自于他在文化底色当中的那种：受多大的苦都开心，欺负人开心，被欺负也开心的这种兼容属性。

所以尽管说作为一个国家的印度，他组织力欠缺，生产力落后，而且呢做事敷衍，以至于在现代科学上面实在是有点浮皮潦草啊。修一个飞机，修完了下来剩一大把螺丝，不知道拧哪。所以不修的飞机还可以在天上飞，凡是经过维修的，基本上都要往地上掉。当他在现代社会的科学技术面前显得如此地格格不入的时候，他却能够在世界顶级的企业当中独占鳌头，在世界的发达国家的内部生活得如鱼得水。这是什么？这就是虽然菜，虽然又菜又爱玩，但是他确确实实是一个不可小觑的力量。作为整体的印度人，可能确实没什么大用；但是作为个体的印度人，只要他出现在你的身边，你的家庭就有可能变成一个印度味的、咖喱味的家庭。如果说它出现在你的公司，你的公司就会变成一个印度味的、咖喱味的公司。如果它出现在了你的国家的管理阶层，你这个国家就会变成一个印度味的、咖喱味的国家。他是一个可以随时随地影响到世界上面各个群体的人群，这个也算是独有的特色。

之前曾经有很多中国这里的公知也好，学者也好，无论是左还是右，都曾经盛赞过说中国的文化有极强的包容性，有极强的感染力，谁来了都会变成中国人。这个事情有先例吗？实际上并无先例。没有发生过“谁来就变成中国人”这样的先例。除非当时中国极强，把对方暴揍在地上，然后让对方与自己妥协。而蒙古人南下打到中国之后，中国这里沾染上了非常强烈的草原风俗，以至于朱元璋卷土重来的时候，需要花几十年的时间去除胡风、胡俗。关外的马帮进入到中原了之后号称汉化，但实际上让整个中国的社会、风俗、人文的习惯都开始向奴隶制靠拢。我们今天所知道的大量的陋俗，就是从清代来的。所以你说中国文化特别能够感染别人，谁来谁就会像我们一样——错。中国的文化只有在极强的时候，才会让别人向我们学习；在实力较弱的时候，是很容易被他人切入和影响的。但是印度人不一样。他即使被人欺压，即使他是弱方，即使他失败了，他依然能够用他的方式来感染到其他人的文化系统，甚至颠覆他人的文化系统。

所以，当我们去看他们的空战，当我们去看他们的赢学，当我们去看他们自己死鸭子嘴硬的样子的时候，难免地会很傲慢地觉得：“什么龙象之争，你也配？”是的，如果那个“象”是住在我们的隔壁，那确实没有什么问题。但是如果像他家的小象住到了我们家里面来，那这个问题就会变得非常地严重。做邻居永远都不用怕他，但是做家人——不管这个家人关系有多么地疏远——你迟早都要吃上咖喱的。
