# 第153章 擦边启示录

体操运动员跳擦边舞

我看到热搜标题的时候愣了一下，心想：体操运动员现在这么惨了吗？然后在想，擦边舞那么赚的吗？当然，这个有点过于私人的问题，最后我还是去吃了那么一口瓜。大致的流程是这样的：有一个体操运动员已经退役了之后，就开始跳舞蹈，有点儿小性感的那种舞蹈。然后借了一个号，这个号发的视频也不是很多，就那么几十条嘛，就是展现一下自己美好的身材和容貌。就这么个事儿。

结果呢，这一点事情中间就出现了一个偏差。因为他有一个同行，似乎是个师妹之类的，就冲上来说：“哎呀，姐姐你怎么搞起擦边来的？这有点影响那个运动员的形象。”通过这种方式，两边就撕扯起来了，有点阴阳怪气的互喷的意思。结果引发了大家的一次大讨论。为啥会引发大讨论？真的是他俩很红吗？没有，是因为有媒体推波助澜，开始说起了运动员擦边、运动员擦边的话题。最后把大家都给裹挟进去了，都去盯着到底哪擦边了。于是我也去看了一下哪擦边了，于是今天核心的问题就出来了：到底什么是擦边？

擦边这个概念，那可真叫一个含糊。擦边是啥呢？它在最早的时候，是我们在乒乓球运动当中出现的一个专业名词，就是擦边球。正常的球要从台面上啪嗒打过去，但是有一些球擦着那个台子的边蹭过去了。你说他出界了吧，他也没出；你说他没出去吧，他又压在线上面了。所以到底算不算违规呢？就不知道，所以这个行为叫做擦边球。因此，我们在互联网上说谁擦边的时候，其实就是说他似乎有点违规的意思，但是我又没抓包，于是这个行为就叫做擦边。

那么现在呢，由于我们国家是禁止色情传播的，写小黄文的被抓起来了，大家都知道。所以在这个范围之内就出现了一种倾向：就是我呢，要挑战一下你的边界线，分分钟地去试探一下到底什么是色情。在边界线上反复试探的这个脚步，就叫做擦边。因此呢，我们有的时候也可以把它解释为轻色情。就是色情是违法的，是严重违规的行为，但是当我的这个东西介于色情与不色情的边界线的时候，你不知道它是还是不是的时候，这个玩意儿就要擦边了。所以擦边这件事情，它本身就是一个刻意的模糊的界限。

闲的没事干的时候，他不会说别人擦边的。我在这捂的这个样子，我又长的这个样子，没有人会说我擦边。但是呢，如果说很性感的一个女孩子，其实她穿的也是正常衣服，但是就因为她很迷人，于是可能就会出现有人说：“哎，你擦边。”为什么擦边呢？因为你勾搭到我了。这个就是边界模糊所造成的一个结果。所以你真的需要批评一个人擦边的时候，首先就要弄明白到底什么是擦边，以及你如果有这个擦边的标准，你可以惩处。没有擦边的标准的时候，擦边就是一个含糊的词语，你没有办法用这种方式来惩处一个人。

有关于擦边这件事情，其实前几年的时候就已经出现过这样的案例了，最典型的就是椰树。椰树在直播间里面找了一群俊男靓女，跳体操，做健身运动，有的时候也跳一点网络舞蹈，然后穿的呢，其实规规整整的，就是紧身的短款运动装。然后就有很多人举报，哎呀，擦边啦、色情啦、表演啦，最后把人家直播间给干停了。然后呢，直播间人家就开始给自己整活啊，人家那个俊男靓女在那里搞健身的时候，穿个厚外套，怎么的，我捂上了。捂上了你看不出来我身材好吗？还是看得出来。所以他就经常在这种边缘上面进行试探。所以边缘上试探这个行为就叫做擦边。

但是呢，因为有媒体的介入，而媒体有一个权利叫做话语权。媒体就特别擅长于起标题，特别擅长于描述一件事情。当他描述一件事情的时候，使用了擦边，大家就觉得啊，轻色情，这是一个直觉上的关联。至于对方是不是真的在轻色情，我不知道呀，我哪里晓得这件事情，媒体说了就是呗。但是媒体它真的是有标准的吗？没有。尤其是在这种含糊的话题上面，没有任何的标准。我们回忆一下过去的几年当中，是不是有很多人在互联网上很真实地去吐槽一个现象？就是我们现在卖的那种什么鲨鱼裤啊、瑜伽裤啊，太紧身了，而且有好多的人穿着这种瑜伽裤夜跑，就在马路上面跑，跑的时候就觉得路过的人看他一眼都会觉得自己很尴尬。那有人吐槽呀，结果呢，请问媒体怎么说的？媒体说：“哎呀，你们这些老古董不能这个样子，人家穿衣自由。”同样的是穿紧身运动衣、紧身运动裤、运动小背心儿，一露腰三尺半，都到了这种地步了，然后你说这叫做穿衣自由。但是在互联网上，人家穿的规规整整的，只是有点紧身，跳了个小舞蹈，你把它叫做擦边、轻色情，合适吗？

所以谁在描述这件事情？这件事情的描述它是不是精准的？而用了这样含糊的词汇去描述之后，我们对它就进行惩处，说爱封号，不能让他说话了，这合适不合适？甚至包含这一次撕扯的双方，这个被禁言的这一方，固然她确实穿了小短款呀，穿了紧身衣呀，去跳了舞，但是撕她的那位，她也穿了小短款，也穿了紧身衣，她也跳了舞呀。为啥封这个不封那个呢？所以这些话头你只要这么一牵扯进来，那就是一场大乱战，简直了。就是谁站在了话题的风口浪尖儿上，谁成为了话题的中心人物，谁就被封禁。当你的管理方式是用这种方式的时候，那就是要么擦边要么红，你既擦边又红，那就完了。准则到底是啥？反正我是弄不明白的。这叫做枪打出头鸟，万万莫出头。但是互联网大家就是要红，所以这件事情呢，就是很含糊的一套标准。

如果你的标准是一致的，你说所有的紧身衣全部都不能出境，全都不能出现在大庭广众之下，因为这东西涉嫌轻色情，行不行？我支持的，大家都知道，我是一个保守主义者，我是一个保守派。我是支持你穿衣服的时候宽松一点，不要绷在身上的。你不能一边说绷在身上是穿衣自由，一边说绷在身上叫做轻色情，这个东西不行。但是吧，这件事情当中，我依然还看出来一点好的东西。就是你瞧我们在互联网上有这么多在那里穿着比较露一点，然后穿的比较性感一点，然后又跳性感舞蹈的，无论男女都有。那么这些人能够自由自在地在上面去跳这些舞，表演自己的身材，表演自己的热辣，并且从中获得自己的收入，他其实印证了另外的一件事情，就是我们的社会环境真的还是挺好的，真的还是挺安全的。

要知道我们为什么今天这么多的人可以用这样的方式在互联网上，甚至在大街上面就是穿着黑丝小短裤，就在大街上满大街地走？为什么能够做到这样子？其实是因为我们从49年建国之后就严厉地打击了所有的色情行业，我们严格禁止真正的色情产业，最后造成的就是我们国家是没有真正的色情产业的。于是呢，这些有点露的、有点性感、甚至有点过于性感的人行走在街上和公共场合的时候，没有人过来跟你说多少钱一晚。但凡是一个色情业合法的地方，你想在街上穿着这个遛弯儿，不要闹了，你走100米，能有30个客户冲上来说多少钱一晚。就是因为我们禁绝了色情行业，所以我们有这样安全的社会环境，所以才能够有人用这样的方式来表现自己，他才能够用这样擦边的方式来获利。整个的这个擦边行业，都要感激我们对于色情产业的打击。

就是我们中国啊，去打击色情行业，其实是从古至今的，并不是今天。今天我们能够把它刨了根，是因为我们能够把社区的治理管理到很深入的程度，所以才能够刨根。但是在过去，我们也是治理色情行业的，所有的卖淫嫖娼的这个产业都是在下九流里面，而且进行的是一种高压式的管理，是一定要严厉打击他的。为什么要打击他？他会对一个国家产生什么样的影响？有关于说那个身体的买卖，这个东西哪怕在原始社会当中也会存在的，它属于人类的本能的部分，叫做食色性也。而食色性也，你一旦把这个部分拿出去交易，那天然地就会进入到色情产业里面去。但是后来我们为什么就不断地贬低他，不承认他，不许他好好的存在？所有进入这个行业的人都归到下九流，不管你有多少钱，因为我们整体的社会需要团结起来，一起搞大生产、大建设。而出卖身体这个方式是一个非常非常简单、非常非常低成本，同时对于个人的这种身体、对于个人的精神却造成了不可逆的损伤的一种行业。
