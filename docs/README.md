# 吃瓜萌主

吃瓜盟主本人制作的视频，转为文本内容。

- 视频制作
    - 吃瓜蒙主
- 声音剥离
    - 软件 [ffmpeg](https://ffmpeg.org/)
- 语音识别
    - [软件 AsrTools]( https://github.com/WEIFENG2333/AsrTools )
- 文本整理
    - [Deepseek](https://chat.deepseek.com)
- epub电子书制作工具
    - [honkit](https://github.com/honkit/honkit)
    - [calibre](https://calibre-ebook.com/download)
