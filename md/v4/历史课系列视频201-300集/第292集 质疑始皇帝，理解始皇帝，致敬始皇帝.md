# 第292集  质疑始皇帝，理解始皇帝，致敬始皇帝

嗯，我们来看一下。  
在秦律里面有“杀夫无罪”的规定，前提是男人出轨。秦律对出轨的管制特别严格，有时候我都觉得这事管得好。但是，它一定就是好的吗？反正心里头畅快，你就说它好呗。至于待在别的地方也有其他的副作用，那是副作用的事。不过，秦律当中确实包含了对于出轨的惩戒。  

目前有一种推测：始皇帝这个人吧，他受出轨事件的伤害比较严重。而且别人的出轨，一般都是妻子、丈夫劈个腿，这种问题好解决——不管你多怄气，了不起撕破脸，离婚把他轰走。但始皇帝不一样，因为出轨的人是他亲母亲。亲妈出轨，你也不能不认娘啊，于是就出现了这种问题。我目前的推测是，始皇帝在这个事上受的心理创伤比较严重，所以他非常痛恨出轨的行为，而且公平地痛恨出轨的男女双方。于是，在始皇帝的这个时期，秦律对于出轨的约束非常非常严，严到了什么程度呢？就是你要是发现捉奸在床，把人堵那儿了，现场打杀而无罪。已经到了这样的程度。  

怪不得他不立后，可不是嘛，伤着了。始皇帝是受了心理创伤的，这个事毫无疑问。哈哈哈哈哈。嗯，这条法律挺好，现代人肯定觉得挺好，因为现代人对这个事的约束实在有点......这个约束跟鼓励的差不多。我也觉得有点过分。很认真地说，你既然制定了婚姻这套社会规则，就需要保护它的存在。你要么就干脆取消这项制度，但凡说你还认为应该有婚姻制度、保证小家庭的传续、保证家里孩子在父母双全的环境下长大——你既然有这个目标，就应该保护婚姻的存在，这是基本的社会管理学原理啊。你要么别设立婚姻家庭，你只要设立了，就要保护它的存续。怎么保护呢？谁破坏了这个婚姻的稳定，就让他受到惩罚，这个是必须要有的。你既然说婚姻是夫妻双方缔结的一项契约，那这个契约当中当然包含了忠诚义务，就是不违约的义务，一定要有。那么违约的人就应该受到惩罚，不然的话，这个规矩不是形同虚设吗？所以现在的这种出轨不光无罪，甚至在财产分割上面的偏向性都很少，实在是有点离谱。我对这项规矩完全不认可。为什么呢？因为我觉得我们现在的法律制定体系当中，掺杂了太多西方中心主义的人，包括互联网上各种各样的法律爱好者，也包括很多法律从业者，都充斥着西方中心主义思想的人。甚至其中有一些人还是教徒，这个就更要牵扯出来了。  

西方人的这一套思想——西方人的法律思想，是和他们的宗教思想之间高度一致的。你会意识到一件事：为什么今天有很多法律工作者，包括互联网上很有名的法律大咖，你细看他的言论，就会发现他是一个一神教的教徒。为什么？为什么要信这个？因为一神教的思想和现行的西方法律的法哲学思想、法理思想之间有极大的共通性。于是就会出现：你学习法理，结果学着学着学着了教义，两边一通，你突然觉得这说得有道理，这不是一回事吗？直接就出来了。为什么会这个样子？你去学西方的法哲学史就会发现，西方的法治变迁过程当中，宗教系统起到了极大的作用。事实上，在相当漫长的岁月里，宗教的内部规则和社会的法律制度之间是高度一致的，甚至可以说，世俗的规则需要为宗教规则而服务。所以它的思想体系当中包含了大量的神教思维，这是天经地义的事。但是当中国人的司法体系也向它这一套东西学习的时候，麻烦就出来了。因为中国是一个世俗社会。世俗社会是什么意思？就是每一个普通人都会说：你这人要讲道理的嘛。世俗社会是要讲道理的，而不是要讲神权的，不是要讲政治正确的。为什么西方人那么容易接受政治正确？尽管他们现在也受苦，受得一塌糊涂了，但是他们很容易接受政治正确，很容易接受教条的东西。因为他摆脱神权的时间非常短暂，绝大多数的西方人到现在都没有能够摆脱神权思想。神权思想是什么？神说什么就是什么，这是教条，这是标准的教条，也是标准的政治正确。神说的就是对的，就是有道理的——不一定，但只要是他说的，那就是对的。这是什么？过去叫做神权思想，今天叫做政治正确，这是一回事。所以为什么西方那里特别多的政治正确？为什么立下了一个规矩，他立刻就照办？只要听起来还像回事，他就非得照着办不可，特别的刻板。因为他没有那种真正的唯物精神的道理存在，没有道理，没有规律，他就只能去听规则。而规则是什么呢？一开始是神权，现在是政治正确。然后法学思想当中也不可避免地充斥着神权的思想。  

欢迎来到直播间。嗯，想问一下：我是唯物主义者，与对所有宗教信仰存有敬畏之心冲突吗？  
——你所谓的敬畏之心指的是什么？我怀疑是那种“万一那玩意儿真的呢？不行的话还是烧个香吧”的敬畏之心。如果是这种敬畏之心的话，你不是一个唯物主义者。不理解但尊重？尊重和敬畏不是一回事。注意了，尊重和敬畏不是一回事。尊重是：“你信这个啊？好的好的，我尊重你的喜好，尊重你的偏好，我不否定你的选择。”这个东西叫尊重。但敬畏不是这个东西。敬畏是：“虽然我是个唯物主义者，但是我见庙就烧香。”这能干什么？这——尽管我认为他大概率不存在，但是万一呢？这是什么？这是唯物不坚定的表现啊。  

那啥，男足啊？这啥时候谈到男足了？我看一看有一个什么东西——“满清时期为什么把足球禁止了”？哦，不是吧，足球禁止了？他是把结社禁止了啊。注意了，满清不是禁止的足球，它禁止的是结社。至于它为什么禁止结社，你不好理解吗？你想一想，结社是干什么？一群人设立一个组织机构，斩鸡头、烧香、敬酒，愿同年同月如何如何，要立誓的，立契约的，然后拜了把子，在一起，今天做这个活动，明天做那个活动——这个玩意儿，我跟你说，在清的时候，所有的结社都是被禁止的。为啥？你都斩鸡头、拜把子了，你下一步不得反清啊？这是一块。然后另外一块为什么会格外去打击足球？因为除了结社这一个痛点之外，足球还有第二个痛点：足球的这种组织训练方法当中是包含了兵法的。注意了，足球的训练方法当中是包含了兵法的——谁在前头，谁在后面，谁在侧翼，怎么包抄，怎么制造假象，谁在那个地方突一下就让你进门了——这是啥？这是兵法演兵训练啊。你都结社了，我都怕你推翻我，然后你还进行练兵？开玩笑的吗？
