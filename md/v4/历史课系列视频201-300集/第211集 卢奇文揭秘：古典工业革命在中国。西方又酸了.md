# 第211集  卢奇文揭秘：古典工业革命在中国。西方又酸了

CA伪史论最典型的证明，不就是西方对于二战历史的肆意篡改吗？我们之前也聊过这个事。西方人没有那么尊重历史。他们有没有历史学家？有没有考古学家？有的，甚至今天的很多中国人还说，我们的考古学就是跟西方人学的。你凭什么说人家不尊重历史呢？那这事咱们得弄明白。西方人研究考古学是为了什么？你们觉得他们研究考古学是为了什么？有人知道的吗？哎，money！对对对对，他们是为了money，是正正经经的，就是为了money。

为什么？因为他们发现这个东西可以形成市场之后，就把它运作成了一个所谓的文物市场。它形成了一个文物市场之后，大家会发现，再早两三百年的时间，在他们所谓的航海、大探险、大殖民的那个时间里头，西方人在全世界到处游走的那段岁月中，他们一直试图开发各种各样的古典遗迹。然后，他们会把这个地方的雕塑也好，各种各样的物品也好，有文物价值的东西，打包上一个非常绚烂的历史故事，再卖到某个博物馆里。这些东西已经形成了一个产业，而且形成了产业链条。

那么西方也有很多富豪，非常热衷于给自己贴金，就是贴这个历史文化的金。谁家里收藏的古董文物很多，说明你们家源远流长，不是一个暴发户。所以在西方早一两百年之前，有相当长一段时间里，他们的博物馆、他们富有的当地有钱人，都非常热衷于搜集文物古董，而且不限地域，不限文物的性质。只要是一个有历史渊源的东西，就会成为这个产业链条中的一环。一件事，当买卖干成了之后，他们就会想办法拓展货源。然后他们发现，中国这里拥有全世界最丰富的货源。所以所谓的“考古学家”来中国考古，实际上最初就是来中国盗墓。他们最初的这些“考古学家”，就是一些文物贩子和盗墓专家。

嗯，啥？他们收藏的全是我们的古董，极少收藏自己的——还是有的。而且这个事儿有一个历史的演进过程，并不是一开始他们就只收藏中国的文物。以前他们对其他地区的文物没那么嫌弃。是近些年，主要是西方人对于中国以外地区的文物变得入手比较谨慎。为什么呢？因为造假率太高了。他们的博物馆、他们的有钱人经常被骗。之前他们哪儿的都收藏，但是后来发现，就近几十年的事，他们在中国以外的地方收集的那些古董，上当受骗的比例特别高。虽然在古中国这里也经常受骗，因为中国文物多，但文物贩子和造假古董的人也格外专业。可是中国这边的假文物，假的程度有限，不至于让他丢脸。但他后来发现，在其他地方收集的那些——就是各种各样的探险家卖给他们的那些大石头之类的文物——造假率太高了。他们的博物馆和有钱人经常上当受骗，次数多了之后，他们就长了教训。后来，他们就把相关的内容开始降低比例了。

呃，如果第一次工业革命发生在明中后期，那明是不是也算反动呢？不是洗满清。呃，我看不出来你这话啥意思。如果第一次工业革命发生在明中后期，你指的是欧洲在明中后期的时候开发了第一次工业革命，是这个意思吗？你是想说工业革命如果在明的时候出现，也会出现在欧洲的意思吗？你这句话里面潜藏了一个台词，就是工业革命无法发生在中国。你这句话里面的潜台词是：工业革命无法发生在中国。而这句话是真的吗？这句话不是真的。所以你没有办法这样子去直接定义。嗯嗯，啥？大一统王朝搞工业革命不搞笑吗？这有什么搞笑的？你以为工业革命是啥？工业革命和你的那种拉人头的革命性质不一样。工业革命本质上是一个术的高速迭代，划时代的技术的高速迭代。这个东西自古以来我们就有工业，百工一直都是一个专门的行业，是一种专门的职业——就是百工，技术人员、工程人员。这些东西都属于百工类。所以我们一直都是有工业的，只是说这个工业发展到什么样的程度时，你会把它定义为工业的革命，就是有颠覆意义的工业式进步。什么样的程度，你会认为它是有颠覆意义的？工业进步先定义这个东西，知道吧？你得先定义这件事情。

而在一个正常的中国王朝上面，它是不会遏制技术的进步的。我们的技术其实始终都在进步，除了战乱时期，一直都在进步。然后一直到清开始倒退，不是停滞，是倒退。我一直强调这事儿，从清开始，我们的工业技术的水平是倒退的。

工业革命也可以叫做产业升级吗？如果它是一个划时代的进步，我认为它就可以叫做工业革命了。实际上前天的时候，我刚刚看了有关同志给我推广的一个，嗯，卢清源老师的一个文章，应该是文章还是他的一个演说？里面的内容就是讲到了古典工业革命这个概念。我还蛮喜欢这个词的，分享一下。在春秋战国期间，我们的生产工具从青铜转向了铁器。铁器时代的来临，意味着我们的生产力水平出现了一个急速的转折点，高速的进步。这一种划时代的变革——由青铜转向铁器的变革，影响了后续2000年的技术发展。那么它能不能算作那个时代的古典工业革命呢？要知道，冶炼术绝对属于工业的范畴。而在春秋战国时期，中国人已经实现了从青铜到铁器的普遍应用，出现了这样的大规模转折。那么这样的转折算不算是一种工业革命？

那么大家都很在意火器这种武器的问题。火器的迭代——好，火器在宋代的时候已经开始应用于战场。包括它使用了辅助的、助燃性质的火器，也包括了一部分类似于爆竹这样的相关方式的火器。虽然是初代的，也没能实现枪炮，因为它的铸造术不足以让它形成枪管、炮管。但是，它确实已经开始使用火药，使用了助燃的石油。对，当时它是使用石油及其衍生品进行助燃。所以在宋代的时候，火器的雏形已经开始出现。而到了明的时候，火器已经间接成为了枪炮。那么这种算不算是一个高速的进阶呢？
