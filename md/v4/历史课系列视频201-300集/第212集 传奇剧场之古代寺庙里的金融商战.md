# 第212集  传奇剧场之古代寺庙里的金融商战

欢迎来到直播间。好，我们啊，差不多就可以开始啦。嗯，好的好的啊。想听听和尚的故事吗？我们可以听一听历史上和尚们的故事啊。注意一件事情哈，我们得把佛学和佛教区分开来，能够理解吗？

嗯，佛学呢，是一套哲学思想。你说我不完全相信它，没有任何问题，因为思想上面总是允许辩论、允许争执的。那佛学的思想呢，它是通过不同的时间，各种各样的思想家累进叠加进行创作的一套思想体系。这一套思想体系当中还颇有一些可取之处。但是，当你把它列成了一个教门的时候，就不一样了。因为教是什么啊？它呢，这个含义本身就包含了教化的意思。也就是说，它要有信息的传播，要有信众。诶，所以呢，当一个学问转化成了一个教门的时候，性质上面就会发生改变，因为它变成了一种组织机构。因为它变成了一种组织机构，而一个组织机构会变成什么？组织机构是干什么的？是一个人的集合啊。所有的组织机构都是一个人的集合体，而人这个东西不确定性就会非常的强。因为人类的行为方式，在很多的时候是趋利避害的，你放到哪儿去谈都是一样。人类的基本的行为模式其实是趋利避害啊。

所以呢，人类文明的早期阶段，普遍会经历奴隶制和神权管理。为什么会这样？因为它用了一种非常高压的手段：你不服从集体，那你就要受害。这也是利用了人身自然而衍生的一种趋利避害的倾向，从而进行组织建设啊。人类文明的早期把人凑起来，就是用了类似这样的手法。因为一旦超出家族的范围的时候，你很难用血缘为纽带，你就需要其他的管理方式。所以全人类几乎在我们的早期的时候，都经历过神权阶段，都经历过奴隶制的阶段。这个就是因为呢，你有的时候啊，要用这种“你不从就害”的方式来形成一种组织结构。

但是当我们走到今天的时候，当然我们可以说，哎，那都已经过时了啊，那就已经是过气的、过去那些不适合的操作手法了。因为大家普遍都受了教育，并且普遍都有了一定的文明的共识（谢谢字母简）。当我们有了这些东西的时候呢，那就不需要用过去几千年前的那种强制性的手段了啊。但是呢，我们从中依然可以窥见一斑：所有的人类集合体，你要构成一个组织机构、团体的时候，第一呢，你要有自己的纲领啊；第二呢，你就是要善用趋利避害的倾向，从而让大家按照这个纲领来行事啊。那么这个纲领的东西就是我刚才说的。佛教的纲领是什么呢？它其实就是佛学的思想。那这个思想本身，不管你喜欢或者不喜欢，总归它里面是大有可取之处的。可是当它形成一个组织机构的时候，它为了让它的组织机构存在，就需要做几件事情：一个呢，是要求它的人越来越多，这就是影响力啊，需要更多的人。然后第二呢，你要让这么多的人聚起来，一方面是要让他产生恐惧感，这就是要它避害而向你屈从；那另一方面就要诱之以利：你要和我在一起有好处。于是就会出现，唉，我用趋利避害的方式，使得越来越多的人聚在这个框架之内。这个方式呢，就是营造一个组织、建构你必然存在的一些操作方法。

那佛教它自打诞生开始，它成为一个教门，开始就一直在做这件事情。你要把这件事情办好啊，那恐惧这个部分，它是通过地狱观啊，通过这些故事讲解、威慑，使人产生恐惧感：你不信这个东西，你可能就会如何。这种它叫做恐惧感，这就是要求你避害。为了避害你也说那个什么，嗯，不行的话我就去试试呗，是吧？万一呢，是吧？万一他说的那个是真的，那我倒霉了可怎么办呢？那这个呢，就是希望你能够避害它，用这样的方式来驱使你进入到它的圈层。但是另外一个方面呢，就是要诱之以利啊，你得有好处。好处当中包含了口头的，例如说你信了之后就会如何如何获得解脱，就会如何获得幸福，这些呢，是口头的。但是要知道啊，你要形成一个真正的组织体系，光靠口头许诺是不行的。所以里面啊，一定会出现经济分配，这就是为了用趋利避害的方式营造自己的组织架构。所以它内部需要有经济利益上的分配。而你要想分配，你首先得有，你要有就得有聚财的方式。但是他又说自己是出家人的，这种聚财的时候呢，一个呢，是谋求信众的打赏；另外一个方向呢，自古以来啊，古典时期的寺院、教门，通常情况下会涉及到一种放贷啊。古典时期，各大寺院都会经营一个它的支柱产业，就是放贷啊。一个是放贷，还有一个类型呢，就是专门的啊，它利用自己的税收减免政策会圈地啊。这两个方向都是古典时期教门始终握在手上的、非常重要的生财之道。有了这些财源，然后呢，它利用这些财源进行财富的分配，使得越来越多的人依附投到它的教门之下啊。这就形成了一个组织建构。

但是呢，当我把这个东西给你梳理出来之后，你立刻就会意识到一件事情：如果它这个教门做得大了，就会威胁到整个国家的社会治理，这是毫无疑问的事情。事实上，在我们的大一统社会管理方略当中，你别说这样类型的教门了，就连你的家族稍微多子多孙了一点，做得势大了一点，朝廷上面都要给你来一波推恩令是吧？推恩令啊，或者是汉代的时候，还有专门的啊，迁徙世家大族去修皇陵啊。这些呢，也是一个专门的、对于基层过大的组织单位进行消耗的一种方法啊。那就是为了让整个的上传下达能够更加通畅，能够让整个国家能够走向一体化，而不是由其中的某一些组织把自身做大，然后与朝廷分庭抗礼。那这种方式呢，是非常可怕的啊，那就出现了组织力的分权。

所以历史上为什么会出现一次又一次的灭佛运动？并不是在消灭佛学思想啊。灭佛运动从来消灭的不是思想，消灭的是组织形式。因为当它的组织框架做得过大的时候，它天然的就会损耗整个国家的制度建设和基本盘，甚至会影响到国家的土地与财产的分配、再分配。为什么？这是消灭有威胁的组织啊。对啊，我们历代呢，都存在这样的灭佛运动啊。那它呢，就是去消灭其中会影响到整个社会治理的不安定的组织因素。所以呢，放到今天啊，这件事情你说它复杂呢，其实，嗯，没有那么复杂，但是肯定有一部分的内容也不是那么好说。可是只有一句话可以讲：就是你只要违背了我们现行的制度建设，违背了现行的法律，那无论你是什么样的身份，该收拾的就要收拾，该整顿的就要整顿啊。该收拾的收拾，该清理垃圾的都一定要清理出去。那既然能够上面出来说已经开始发布公告、做公开的调查，那就说明里面的证据已经实锤了。

嗯，好，欢迎来到直播间啊。什么少林也是这个思路？少林什么思路啊？不知道啊。但是呢，就是我们历史上相当多的寺院都曾经做过这件事情啊，或者可以说那些稍微大一点的寺院都做过这些事情。因为你要想把这个寺建大，那相当于建一个经营机构啊。今天呢，很多人呢，把那里当做一个景点，叠加一个宗教场所来看。但实际上啊，它能够营造起来，能够运行起来，你得把它当做一个经营型的机构来看待，才能够更好地去理解它。一个经营型的机构，你想要运转起来需要的是什么呢？啊，财源需要你有人力，需要你有人脉啊。那这些东西都具备的寺院，就可以把自己经营得很大。

所以当你去看到说哪个地方那个寺院历史源远流长的时候，你可以放心大胆地去推论下面的东西啊：它历史上一定是经营过圈地和放贷业务的，没有例外啊，一定是干过这事的。至于现在当然都已经经过了改制了啊，历史上这些机构都干过这事儿。
