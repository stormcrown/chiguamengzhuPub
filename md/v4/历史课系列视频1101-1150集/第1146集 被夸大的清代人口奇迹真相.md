# 第1146集 被夸大的清代人口奇迹真相

### 被夸大的清代人口奇迹真相

关于清代的人口问题，确实值得我们深入探讨。首先需要厘清明代的人口基数——明代实际并未进行过大规模的人口统计。立国之初，官府曾以“丁户”为单位编制《黄册》（即赋役册），作为征税依据，但此后便沿袭旧册征税，直至明末未再重新统计人丁，因此明代没有官方人口数字。此外，明代征收的“丁税”（实为丁徭银，即劳役折算的税金）导致民间普遍隐匿丁口以避税，尤其大户人家更是如此，这进一步扭曲了统计数据。

不过，现代学者通过其他途径推算了明代人口：一是根据田亩数与粮食产量进行倒推，二是依据食盐消费量等指标，再结合多种旁证交叉印证，得出了不同口径的估算值——低值约1.6–1.8亿，高值约2.6–2.8亿。主流观点折中采用约2.2亿作为明末人口常态估计，这一数据是有依据的，并非凭空猜想。

相比之下，清初在开国时曾进行过专门统计，当时全国人口不足9000万（或可称“不足一亿”）。从明末约2.2亿降至清初不足9000万，经历了漫长动荡，人口折损过半。

清代的“人口奇迹”常被形容为从1亿膨胀至4亿，翻了四倍。但这一增长背后有多重因素：

1. **优质人口基础**：清初虽然人口锐减，但剩下来的人大多是熟练的农夫和工匠，具备自给自足的生产能力，这是后续增长的先天优势。

2. **高产作物推广**：大航海时代带来的红薯、玉米等高产作物，早在明代就已开始推广，并非清代专利。这些作物对土地要求低，能在贫瘠或零散地块生长，大幅提高了人均口粮总量，但同时也导致口粮质量下降（如碳水化合物品质降低），民众常处于半饥半饱状态。

3. **摊丁入亩政策**：清代推行“摊丁入亩”，取消丁徭银，将其折算进田亩税收中。这一政策取消了隐丁的必要，使得人口统计趋于真实。但副作用是地方官府失去原有丁税收入后，又以“火耗”等名义加征，实际税负加倍，农民的地租也随之提高，加重了负担。

4. **官员虚报人口**：由于不再征收丁税，虚报人口不会增加税负，反而能彰显政绩，导致大量官员虚报数字。乾隆皇帝曾专门下旨斥责此不正之风，但虚报现象仍部分存在于统计中。

综上，清代人口增长是取消丁税、高产作物推广、虚报数据等多重因素叠加的结果。需要特别指出的是，高产作物的推广是全球性现象，并非中国独有。在受大航海影响的其他地区（如东南亚），人口增长比例通常达到7–10倍。而在中国，入关后的满洲群体（统治阶层）在其200多年统治中实现了7倍以上的人口增长；但最擅长农耕、经营与囤积的汉人群体，增长仅为4倍。这究竟是正向贡献还是负面表现，值得深思。

更令人警醒的是人口质量的变化：明末汉族男性平均身高约1.7米，到清末降至约1.65米。尽管在老照片中中国人身高仍不输部分欧洲侵略者（因欧洲当时殖民者身材也较矮小），但200多年的“盛世”却使平均身高下降5厘米，这背后是持续不断的饥饿对群体健康造成的侵蚀。
