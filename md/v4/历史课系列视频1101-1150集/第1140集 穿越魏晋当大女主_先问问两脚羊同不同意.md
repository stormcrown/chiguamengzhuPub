# 第1140集 穿越魏晋当大女主_先问问两脚羊同不同意

### 整理后的易读文本

穿越魏晋当大女主？先问问“两脚羊”同不同意！ #五胡乱华 《读史感魏晋风流之谬》  
清谈误国犹堪训，粉饰风流岂足夸。  
洛水魂惊胡骑尘，铜驼泪洒汉宫沙。  
岂因贾后权倾日，便作钗裙盛世花？  
莫效楚宫迷细腰，青编血泪鉴天涯。  

要不要了解五胡乱华？呵呵，这个啊，我今儿看到的那个，哪儿的来着？是有人分享的，还是我自个儿平白刷着的？我也没注意啊。反正就是某大学的一个教授——哎呀，我的天哪，我的教授档次好高啊！做客电视台、官媒哦，文化类的节目。然后呢，说姑娘们如果有机会穿越的话，最好到魏晋南北朝，因为有很多大帅哥可以看，而且可以上演大女主剧本。我就想知道，这个教授是咋评上的呀？我就感觉到，哎呀，十分的无语，你知道吗？就这一小段就让我异常的无语。你搞穿越啊，我个人从来都是说的——以前也有人问我说穿越到哪个朝代，我说穿越你离得越近越好，除了清朝，离得越近越好。因为你语言不通是吧，行为模式完全不一样，甚至社会的公共道德的逻辑都不一样，你在那儿很难活下去的。结果人家交了一个，唉，魏晋南北朝太狠了。

魏晋南北朝为什么会有人觉得它风流啊？为什么觉得帅哥多？为什么觉得浪漫？为什么觉得多情？你要说这些元素有没有呢，那是真有，但是为什么有，不用考虑一下呢？啊，就那个时候为什么会有这些元素？哎，我是纳了闷了，我简直了！你说莫名其妙呢，你就想到穿越魏晋南北朝，那是啥日子呀？帅哥是很多，浪漫是很多，但是为啥这个样子？不就是因为那个时候大家都不干正事吗？啊，都不干正事，然后就开始琢磨什么啊，比美啊，烧蜡烛啊、豆腐啊，比美擦胭脂，然后去比谁后面出街一趟收到的香囊更多啊，比谁的那个宽袍大袖，比谁的皮肤更娇嫩，天天斗这个。为啥呀？闲的。为什么闲呢？没正事给他做。为啥没有正事做呢？因为朝堂乌烟瘴气。为啥乌烟瘴气呢？因为皇帝得国不正，上梁不正下梁歪，直接导致整个王朝一团混乱。

然后更离谱的是那种什么号称说“到时候可以上演大女主剧本”。啥叫大女主啊？我勒个去，我都不知道该怎么说她大女主。她是不是看了一下贾南风，然后就觉得可以大女主了？什么玩意儿叫大女主剧本啊？贾南风确实啊，也算是大女主，但是她能够以又丑又疯又笨的这种档次，能够成为大女主，那不是欺负她丈夫是个傻子吗？啊，就是因为她丈夫是个傻子，所以她才能够长得又不好看，然后人品又拉胯，智力又不高级，水平又糟糕，性格又很恶劣，品德又很败坏的这种前提条件下，她还能够指点江山当大女主。那前提条件是那个皇帝不争气，那个皇帝是个傻子，是一个智力残障儿童啊！结果他们在那里幻想说什么“这玩意儿可以上演大女主剧本”。她是假设这个人长得也不行，脑子也不行的情况下，怎么上演大女主吗？我不是很懂这些人到底想宣传一些什么东西。

整个魏晋南北朝，那魏都还好说啊。魏呢，虽然也有点毛病，但是基本上它还是一个有纲领、有操守的一个时代，保底的一个秩序还在。从晋开始，那就已经是群魔乱舞了啊。西晋结束了之后，直接就是来了一波五胡乱华。五胡乱华那是什么好日子？对那个女性倒确实挺生猛的，毕竟和男人很平等，平平的都摆在桌子上面，该红烧红烧，该清蒸清蒸，是吧？所以我是觉得有一些人，他是不是跟那个什么，跟电视剧，甚至是跟言情小说学习的这些东西，以至于去发表言论的时候，感觉到十分的荒谬。就是你可以不知道这些历史的事实啊，因为不保证每一个人都去读这些事情，这个是无所谓的。但是呢，我们在整个的初高中的整体的历史教学当中，其实还是包含了很多我们的整个的国史的纲要的内容，该有的内容本身应该是有的才对，是不是啊？这不得了啊不得了！想要去魏晋南北朝？我支持他。嗯，谁想去谁去，反正我不去。

### 备注：修改的词汇

1. “平白扫着” → “平白刷着”（修正为更通顺的日常说法）
2. “档次好高” → 保留原词，但原文中“档次”应为“档次”的笔误，实际为同一词，未作改动。
3. “逗这玩意儿” → “斗这个”（“逗”改为“斗”，更符合上下文“比美、斗富”的意思）
4. “生咖” → “生猛”（根据上下文，五胡乱华时女性被当作“两脚羊”吃掉，此处应为“生猛”或“生冷”，修正为“生猛”以表达惨烈）
5. “青编” → 保留，原指史书（“青编”即竹简编成的史书，正确）
6. 其他少量口语重复及标点调整为规范表述，如添加问号、逗号等，整体未删减内容。
