# 第1138集 破笼_清末求生录

《破笼：清末求生录》  
《戍荒赋》云：昔者清廷画地为牢，汉裔禁足十八疆。海陲封帆棹，塞北断蹄痕。沃野千里成囹圄，亿兆黎庶作困兽。官牒剥脂膏，闾阎匮米薪。男儿俯首矮五寸，妇孺缫丝夜未央。及至王纲解纽日，铁锁铿然落九霄。云帆直指南洋浪，筚路斩棘关东雪。自梳青丝为生计，徙民白骨拓荒原。三百万众守东北，百载血泪固金瓯。嗟乎！封疆岂靠戍卒戟，守土终须黎庶耕。今观史册斑驳处，犹闻父老踏歌声。

感觉他们也挺了不起的啊，没有错。他们是在那种特定的时代下，挣脱原有的生活方式，寻求自己的人生。一群想要自己决定人生的女性。当时，在那种特定的环境之下，很多人的谋生都有问题。女性当中出现了这种自梳女：她们有的下南洋去务工，有的则在附近务工，什么样的形态都有。同时，也有很多男性出门务工，大家在这方面并没有本质的区别，其实都是求生而已。

本质还是因为整个清代，有一个所谓“汉地十八省”的划分——清代把汉人圈禁在十八省范围之内，严禁汉人外出：沿海的不许下南洋，北方的不许进东北，也不许出各个关口。所有十八省范围就是汉人允许居住的范围，超出就不许出去。所以，它形成了非常严重的内卷。这是一个没有办法的内卷：资源非常有限，人口已经滋生，官府又盘剥过剩，以至于普遍性的贫困。清末时，中国男性的平均身高相较于明代下降了5厘米之多。当经济演进到了食物、生活水平，甚至直接影响到了人的平均身高时，说明这种持续性的贫困已经绵延了很久，否则不会直接影响到人的身高。

所以，在清末，当这种管制开始出现放松——也不是主动的，而是清廷的约束能力下降了——无论是海禁还是关口，其实都管不住了。这时出现了大量的人外出自谋生路，所以出现了自梳女、南洋打工人、闯关东、走西口，都是这个时期开始出现的。这其实就是人在自己能力范围之内，竭尽所能地去探索生存空间的一种自然方式。在过去的任何时代，这都是合理合法、非常正当、甚至会获得朝廷鼓励的，因为百姓主动去开辟疆土，会使得边疆安稳，让边疆更好地掌握在自己手里。但清代由于防汉、制汉的心态，持续两百多年，一直稳稳地占据其核心纲领，所以一直要求汉人不可出关、不可下海。这些人就被死死地圈在了汉地十八省。

今天又有一些人喜欢说“你的土地就那汉地十八省”，怎么可能？这是故意被圈禁的。但凡没有那些界限，早就走出去了。也不可能到了晚清闯关东之前，整个东三省（包括一部分外东北）这么庞大的土地上，累计人口不足三百万。这么大面积，累计人口不足三百万，其实约等于无人区——除了少数的聚居区和熟田之外，其他地方几乎就是无人区。为什么当时的侵略者来到这个地方时那么容易？大规模地派先遣队进来，然后大队伍就开进来了，是因为这片没有人。所谓的戍边的人，就是一些游移居住的、没有定居的部落：人又少，装备又落后，而且是散居的、游荡型的，不知道今天住在哪，也不知道明天住在哪，每年要迁徙两到三次。这样的人，你说他们可以守边？他们怎么守边？在现代化火器之下，这些零零散散的边军根本没有守边的能力。真正能够占据一片领土的，必然是其人民普遍居住在这个地方，建设了城市，建设了乡村，开垦了土地。要知道，有土地的人会捍卫自己的家乡，天然的就会成为坚定的战斗者。

所以，为什么到晚清后期，它开始开放这些边界？因为不开放也不行了——不开放的话，所有地方都要失去了。于是开始出现大面积的汉人闯关东、走西口这种大面积的迁徙行为。正是通过这样的方式，我们才能够保得住今天的东北土地以及内蒙古大部分的土地。
