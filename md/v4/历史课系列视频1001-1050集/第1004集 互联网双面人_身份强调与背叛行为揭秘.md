# 第1004集 互联网双面人_身份强调与背叛行为揭秘

互联网“双面人”：身份强调与背叛行为揭秘。一个人在说一句话的时候，先讲一下自己的什么特殊的身份，那不好意思，后边他要说的内容，就是来背叛这个身份的。

咋感觉今天破站也在吵呢？对，因为这边出了两三个吧。嗯，我也没细数到底是谁啊，反正出了两三个那个就是想要还大清恩情的那种吧，可能比较感恩。嗯，我也不知道大清对他家有什么恩德，万一人家真是爱新觉罗氏，占了好大便宜呢。不知道。互联网上都是昵称，这事不好说啊。一般情况下呢，这种人会在互联网上发言的时候说：“我是汉族，但是我认为啊”，我总结这个句型叫做“我汉我X”。我也啊，反正我理解不了。真正的汉族的话，一般不会说“我是汉族，我就认为怎么样”。因为凡是喊这个口号的，一般就是“我是汉人，但是后面就是反汉的内容”。他为什么要说是汉人呢？他要是说不是的话，这个反汉就过于赤裸了。因此我一般不查户口本，随便啊，悉听尊便，爱咋说咋说吧，这事儿我管不过来。

嗯，啥“我男我也”“我女我也”对，就是这种句型。嗯，反正我是在互联网上待的这些年吧，很明显的一样东西：凡是一个人在说一句话的时候，先讲一下自己的什么特殊的身份，那不好意思，后边儿他要说的内容就是来背叛这个身份的。他强调这个身份，主要原因就是背叛这个行为非常可耻，他怕被人喷，他就要专门打个logo出来啊。基本上都是这样，这个百试不爽啊。真正说你要真是这个身份，你就直说了。例如说我在互联网上怼女权的时候，我就直接怼，我不会说“我是女的，但是我觉得怎么样”，虽然我真是女的，但这很重要吗？我明知道我是女的，我干嘛要说我是女的呀？我就直接说“你这事儿说得不对”就完事儿了。但是呢，就会有一些人说“我是个女的，但是，哈哈哈哈，我是个男的，但是我觉得”，呵呵呵呵哎呀。

**备注：修改了以下词汇**
1. “破绽”改为“破站”
2. “炒”改为“吵”
3. “我赶不赶不来”改为“管不过来”
4. “那个百试不爽”改为“这个百试不爽”
5. “我我我就认为”改为“我就认为”，删除重复的“我”
6. “我自己明知道”改为“我明知道”，删除“自己”（原文“我自己”不冗余，但为修正口语重复，保留可选，此处按易读处理）
7. “你这事儿说的不对”改为“说得不对”，调整“的”为“得”
