# 第1033集 泰山封禅的皇帝为什么那么少_宋朝因何花钱买和平

泰山封禅的皇帝为什么那么少？宋朝为何花钱买和平？

泰山封禅的皇帝为什么那么少？后面的皇帝好像都不去了。一开始的时候，泰山封禅的地位是很高的，当然也不是最开始就高。因为所谓的泰山封禅这件事本身，讲的是天子祭天之权。以前，天子是在他自己的首都那里去祭天的。但是，从秦始皇时期开始，就有方士给秦始皇编造了一整套关于东海海上仙岛的神话传说。从那之后，泰山就开始被抬高了地位，因为向往着海上的仙山。这一整套系统把泰山的地位提升之后，就开始出现去追求在泰山进行封禅，寻求天地互通。

但是，要去泰山封禅，需要这个人拥有天下一统的资质，也就是你得配得上当一个天子，你才有资格去做泰山封禅。当时是有准入门槛的——就是如果你不够这种天下的一统，不是真正的皇帝、真正的君王，没到那个咖位的时候，你是没有资格去泰山封禅的。可这件事又在宋朝这个地方出了问题。因为宋朝那皇帝一统，一统个鬼啊，他打仗打得那么差劲，地都没守住，关卡都没守住，结果呢，人跑泰山封禅去了。一个这种档次的君王，都敢跑到泰山去封禅，那么后世再去泰山封禅，你这个段位是不是就没有印证了？这就相当于你这个广告牌上从来只有奢侈品，结果第二天的时候，一个修脚房在上面打了广告，在那之后，请问奢侈品还会在这个广告位上面打广告吗？就是这个意思。跟“落水之事”异曲同工——就是打从司马氏之后，落水就臭了，大家再也不指着它去发誓了。不是落水犯错，当然也不是泰山犯了错，主要就是因为有一些糟心的人利用了他之后，直接把这个地方的招牌给搞坏了，所以没法用了。

宋朝穷？怎么可能。你可以说宋朝弱，你可以说宋朝缺骨气，但是宋朝不缺文化、不缺技术、也不缺钱。宋朝是很富有、非常富庶的。宋朝但凡不富的话，他交得起岁币？他但凡要是穷一点的话，早就去打江山了。就是因为太富了，条件过得太好了，所以他对打仗毫无兴趣。因为对他来说，打仗这事儿亏本不划算。你看他年年交岁币，跟这边称臣，跟那边称臣，对他来说交的那点岁币，小意思啦。宋朝的经济来源，除了他自己内部科学技术的发展所带来的内部产业之外，大部分来自于外贸。这个外贸往哪儿都有，实际上宋跟北边做的贸易特别多。就是那些你号称说“你应该要岁币”，是他收了岁币，好像他赚了似的——前脚收的岁币，后脚就得跟宋去买各种各样的货，无论是铁锅、丝绸、茶叶，啥玩意儿都有，各种稀奇物件都要从宋这边去购买。所以前脚送出去的岁币，后脚就双倍、三倍地赚回来了。你想，对于宋来说，他为什么要打仗？所以那个宋之所以能够骨头软成那个样子，过于富裕也是很重要的一点。就是对他来说，交岁币这事儿太划算了，这买卖简直了，分分钟就给你赚回来。

同时，除了往北边去进行贸易之外，它还有海外贸易。宋的海贸是非常非常发达的。当时在那个泉州港那边，有专门的中东商人整家整族地居住在那里。当然了，后来这些人都变成了祸害，那是后话。但是在整个的宋代，他通过海贸还是帮宋的皇室赚了特别多的钱财。他的那个富裕程度呢，就是我们现在去发现的宋代的沉船，里面有大量的那些陶瓷——这些东西咱们就不说了，那是铁定有的——里面还包含了很多走私的铁锅。这个铁锅贸易是全球铁锅贸易，持续多年获利不菲。就这样的收入水平，再加上他自己的制造业的水平、科技的水平和文化艺术品的水平，他在经济和文化等等的相关方面，完全全面碾压周围的所有的经济体。所以对他来说，花钱买和平是最划算的生意。于是，这个也作为了他的一个诱因，就是他为什么那么怂。但凡他要是真穷困的话，他也就豁出去了，你知道吗？都已经穷得底儿掉了，反正也是饿肚子，那还不如出去抢一票。其实北边很多的政权都是这样的，目的就是反正日子过得也不行，干脆出去抢一票，对面有肥羊。但是宋所面临的邻居就是大家都是饿狼，只有他自己是肥羊。那这个时候他用什么方式最合适？就是买和平的方式啊。有钱这事儿拖了宋的后腿啊。
