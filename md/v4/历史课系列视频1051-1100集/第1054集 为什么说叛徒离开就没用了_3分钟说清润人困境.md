# 第1054集 为什么说叛徒离开就没用了_3分钟说清润人困境

为什么说叛徒离开就没用了？3分钟说清润人困境！

《七律·斥叛徒行》  
背主求荣作跳梁，一朝出走现原形。  
虱虫失寄终枯槁，犬马无功自断粮。  
柴氏容颜成鬼相，游牧梦话付荒唐。  
劝君莫效东门啸，故国青山日月长。

叛徒离开就没啥用了。  
很有道理。  
哈哈哈哈哈。  
我也觉得很有道理。  

为什么那些在国内非常跳脱的公知啊，那些润人，真的跑出去了之后，日子都过得一个比一个苦？因为在国内的时候，他可以祸害我们，外国老板就会给他发钱。出去了之后他还有什么用啊？他在那儿能祸害着我们什么呀？他当不了我们身上的虱子和跳蚤的时候，它就没价值了。

那之前的那个，那个谁啊，柴大记者啊。柴大记者当年在国内的时候还是挺好看的吧。呃，咱们就不说她表情什么哈，但是至少说她长得还行，还挺好看的，周正也有亲和力。个人的才华其实也是有的，对吧？但是你瞧瞧那个，前些时她又自己出镜了嘛，你看她现在啥样——头发都快没了，那个脸瘦得一副寡相。哎呦我的天哪，真真是已经开始没法看了啊。

我就觉得应润尽润，我特别鼓励这一点。你觉得这个不好那个不好，你觉得哪个地方是你的精神祖国，你就赶紧去。我特别支持这一点啊。您那个什么523喜欢什么游牧计划啊，天下游牧是一家——这世界上有一群特别喜欢吹这个东西的，你去啊，你倒是出去啊，你跟他们一起呀。他又不肯啊。你们都跟那个，你们都对那个柴记者特别特别了解是吧？一说柴记者，哎呦，我这滚屏的那个速度我都快要看不过来了，哈哈哈。

**备注：**  
- “半离开了就没啥用了”改为“叛徒离开就没啥用了”（“半”改为“叛”）  
- “前些时他又自己又出相了嘛”改为“前些时她又自己出镜了嘛”（删除多余“又”，并将“出相”改为“出镜”，以符合常见表达；同时“他”改为“她”以统一指代柴静）  
- “那个脸瘦的那一副寡相”改为“那个脸瘦得一副寡相”（“的那”改为“得”）  
- 其他少量口语化表述调整为更通顺的句子，如添加标点、删除重复词，但未改变原意。
