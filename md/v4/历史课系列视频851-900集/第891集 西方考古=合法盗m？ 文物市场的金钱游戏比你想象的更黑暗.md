# 第891集 西方考古=合法盗m？ 文物市场的金钱游戏比你想象的更黑暗

C 维维史论最典型的证明，不就是西方对于二战历史的肆意篡改吗？西方人没有那么尊重历史。他们有没有历史学家呢？有的。有没有考古学家呢？也是有的。甚至今天很多中国人还说，我们的考古学就是跟西方人学的，你凭什么说人家不尊重历史？那这事儿咱们得弄明白。西方人研究考古学是为了什么？你们觉得他们研究考古学是为了什么？他们研究考古学是为了钱，这是正经的，就是为了钱。为什么？因为他们发现这个东西可以形成市场之后，就把它运作成了一个所谓的文物市场。它形成了文物市场之后，大家会发现，在早两三百年的时间——再早两三百年的时间，在他们所谓的航海、大探险、大殖民的那个时间里头——西方人在全世界到处游走的那段岁月当中，他们一直试图开发各种各样的古典遗迹。然后，他们会把这个地方，无论是雕塑也好，各种各样的物品也好，只要有文物价值的东西，都会给它打包上一个非常绚烂的历史故事，然后就把它卖到某个博物馆里面去。这些东西已经形成了一个产业，而且形成了产业链条。

那么，西方也有很多富豪非常热衷于给自己贴金——就是贴这个历史文化的金。谁家里面要收藏的古董文物很多，就说明你们家里面的源远流长，不是一个暴发户。所以在西方早一两百年之前，有相当相当长的一段时间里面，他们的博物馆，那些富有的当地有钱人，都是非常热衷于搜集文物、古董的。而且他们不限地域，不限这个文物的性质，只要它是一个有历史渊源的东西，就会形成这个产业链条当中的一环。一件事情，当它的买卖干成了之后，他们就会想办法拓展自己的货源。然后他们就发现了一件事情：中国这里拥有全世界最丰富的货源。所以所谓的他们的考古学家来中国这里考古，实际上最初就是来中国这里盗墓。他们最初的那些所谓的考古学家，就是一些文物贩子和盗墓专家。你看，他们收藏的全是我们的古董，极少收藏自己的——当然还是有的，还是有的。

而且，这个事儿也是有一个历史的演进过程：并不是说一开始的时候他们就只收藏中国的文物。收藏中国文物的这个历史一定是非常久远的，没错。但是以前他们对于西方文物、对于其他地区的文物，没那么嫌弃。之前他们对其他地区的文物没有那么嫌弃的。是近些年，主要是近三四十年以来，近三四十年以来，西方人对于中国以外地区的文物就变得入手比较谨慎了。为什么呢？因为造假率太高了。他们的博物馆、他们的有钱人经常被骗。这件事就是说，之前他们哪里的都收藏，但是后来就发现一件事情——就近几十年的事——他们发现在中国以外的地方，他们去收集的那些古董，上当受骗的比例太高了，特别高。虽然在古老的中国这里也经常受骗，因为中国这里虽然文物多，但是文物贩子和造假古董的人也格外的专业。可是呢，就是中国这边的假文物，它假的程度有限，不至于让他丢脸。但是呢，他们后来发现，他们在其他地方收集的那些——就是各种各样的探险家卖给他们的那些什么石头、这石头那石头的文物——造假率太高了。他们的博物馆、有钱人经常上当受骗。上当受骗的次数多了之后，他们长了教训。后来呢，他们就把相关的内容开始降低比例了。
