# 第874集 基因战争爆发 ▶ 你的审美早被重置：为什么总感觉老外更浪漫 

过去几十年，从教育到传媒，铺天盖地地都在说外国人好，都在说人家的人种优秀。不负责任就叫做浪漫，皮肤不健康就叫做白皙——缺乏角质层、黑色素代谢异常就叫做白皙，这算什么呢？连身上长毛都叫做性感。然后天天跟你讲混血聪明又漂亮。你以为混血聪明又漂亮，相信了之后会怎样？觉得特别想生混血，生下来的孩子跟你不像，你就觉得好看。这本质上是痛恨自己的一种表现。

你以为混血聪明又漂亮，只是单纯在说混血儿好不好吗？不是的。“混血聪明又漂亮”这句话有一个潜台词：就是中国人好丑、好丑、好笨。只要你相信混血聪明又漂亮，就一定会相信自己的基因很烂，因为这两个东西是绑在一起的。不然怎么会随便混一个什么玩意儿，就突然之间聪明又漂亮了呢？对面的那个人又不比你聪明，又不比你漂亮，单纯一混，你就高级了？那说明什么？说明自己不行。

所以这些各式各样的传言、各式各样的教育，甚至包括科普推广，甚至包括很多官媒的一些宣传的段落当中，都有这些言辞。而这些言辞会在潜意识当中植入到社会公众的思维当中，最后形成的直觉就是：外国人好好，外国人好优秀。哪怕他那个地方穷，他也很优秀啊——他穷困无秩序，那就叫做自由。要不然如果他居然讲礼貌，那就是我们不讲礼貌、他好讲礼貌；如果说他没有我们讲礼貌，那就是我们这里太刻板了、事儿太多了，你看人家多洒脱。永远都能找得到理由。

所以这件事情啊，崇洋媚外这件事情不分性别。男的难道不崇洋媚外吗？来，公屏上面的爷们儿是不是比较多？来，公屏上面的爷们儿数量很多的啊，我知道你们不要假装。公屏上面的男性同胞们，现在开始考虑一个问题：喜不喜欢金发碧眼的洋妞？喜欢的扣个一。来你们自己看，那你们自己看就知道了哈。所以冲上来说“哎呀，女的怎么崇洋媚外，看着外国的比中国的好”，来，你自己看公屏上的一啊，自个儿数。所以这种事情不要选择什么性别啊。人的心理的倾向会受到很多东西的影响，性别是其中较弱的一个因素，不太重要。之所以在你面前觉得很醒目，是因为你的观察的侧重点不一样。而且呢，由于很多的女性在特别喜欢外来人群的时候，会选择用婚嫁的方式解决这个问题，所以你看起来就好像女的特别喜欢找外国男朋友。但实际上啊，男性当中也有很多这样的人，数量不少啊，数量不少，都一样啊。

观自在很有经验，他说“扎人”。呵呵呵呵呵，诶，你这个词儿一用出来，我就觉得有故事啊，下次讲给我听。大家都差不多的。当你接收的信息当中，出现了大量的外国的、跨种族的俊男美女的时候；当你接收到的故事是他们的爱情特别的真挚、特别的诚恳、特别不追求利益的时候，一定会产生这种心情的。这种心态就必然会出现，它就不是一个性别问题，不是性别问题。
