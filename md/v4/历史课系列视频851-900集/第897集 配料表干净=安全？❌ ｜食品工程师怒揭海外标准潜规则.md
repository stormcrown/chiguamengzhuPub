# 第897集 配料表干净=安全？❌ ｜食品工程师怒揭海外标准潜规则

开口闭口就是食品安全，弄得好像民不聊生一样的这种说辞。我不知道心里怎么想的，反正你要是真的诚心诚意去做，自给自足——小农经济是祖先留给我们的本钱，这个东西又不是做不到。如果说还需要在社会公共的大系统当中生活，那我们就尽量去购买合规的产品，多多观察、了解法律规范，这些都是很重要的。但是跑到互联网上，不断地刷屏，说这也不行、那也不行，还使用了很多自己推测的东西。如果真的觉得身体受到了伤害，根据商品的配料表和上面提供的厂家信息，直接提告啊，我们国家是有这些保护政策的。

嗯，好，欢迎来到直播间。嗯，啥？去国外待一段时间就老实了？有一些人去国外了之后，就开始觉得人家的食安特别好。为啥呢？他们的食品配料表特别安全、特别干净。出了国的人，一开始出去的时候都会有这种感觉：说你看人家这个配料表多干净，啥也没有。那就对了，因为人家规定了可以不写——人家就没有强制要求你必须写。为什么看着我们的东西配料表那么吓人呢？里面但凡加了的，无论它的含量多么微少，你都必须得写。然后人家那个地方，嗯，随意啊，随意，根本就可以不写。而且它在正常的生产指标当中，就是有容错指标的，甚至允许有什么老鼠、蟑螂这些东西的残余。

那个，而且我建议大家少看点营销号那些危言耸听的东西——什么“我们的大米、玉米胚芽都抽走了”，你知道那样操作需要费多大的功夫吗？你知道那样会提高多少的生产成本吗？啊，建议你要是真的想了解这个事，认认真真地去深入了解一下食品工业现代化。食品工业是怎么操作的？小作坊当中有一些不正规的东西肯定存在，这个也是我们打击的对象。但是你冲上来就说“哎呀，我们现在什么这个东西，什么狠活儿，那个东西，什么科技”——没有科技，这么多的人怎么能够干干净净地每家每户都吃饱饭？你一栋楼里面住了上千号人，这样高的密度，你想要过那种田园牧歌的小农生活，有可能吗？而我们能够享受的这些便利，比如说下楼就有便利店，几步路远就可以上学，出门就可以搭车，不舒服了可以直接叫车去医院——这样近的距离，这样高的密度，这样集成式的生活方法，一点代价都不用付的吗？你这样子生活就必须使用工厂的这样的产品，才能够让大家维系运转。如果说你按照这样的方式生活，然后还要搞小农经济，动不动地——能理解吗？所以你真的对这个事情特别在意，要么人住到乡下去自给自足（这个事儿可以尝试），要么就好好的筛选一下自己的吃穿用度，好好地学习一下现代的食品工业是怎样操作的，挑选正规的合规的大厂，好好地筛选你自己日常使用的所有东西。这个才是我们能够做的，而不是跑到这个地方来危言耸听，什么大米没胚芽啊、小麦没胚芽啊，要不然就是在那里讲什么牛奶如何如何。证据呢？有没有证据？有证据，这些东西都是可以举报的，知道吗？严重伤害身体的这样的食安问题是可以直接举报的。国家提供了多种的手段、多种的通道，但凡有证据可以直接提告啊。跑到直播间里面来刷屏说这些事情——你是有证据还是无证据？是否真的有这件事情？如果真的有，为何不走正当的渠道去举报？

嗯，啥？添加剂不标全？海关都进不来。对，那个，我们国内啊，我们国内是要求标识所有的添加剂的，包括你看我们现在是不是有很多地方打广告的时候，喜欢说什么“哎呀，我们这里是出口欧盟的”，就好像显得他的那个标准特别高。有没有这个问题？有。为什么？因为对方会设一个专门的、相当于关税壁垒一样的这种防护机制——他对这种外来的、进口的东西提出特别细节的要求，特别高的要求。这不代表他们当地的食安好，他是为了营造一个壁垒，保护他们本地的本土产业。不是他真的标准高，而是他双标啊！他对于自己国内的那个标准，和对于什么进口商品，尤其是针对性的生产地区的进口商品，它提出了完全不同的标准。所以你听说什么什么东西就是欧盟标准，于是认为人家欧盟的那个食安就会特别好，这就是纯属一个幻想。

还有很多人觉得脚盆那边的食安做得特别好。脚盆那边他们自己的奶粉里面可以——头那啥玩意儿，生化物对吧？之前造成了多少问题。自来水里面可以有，有问题啊。大家自己去搜那个物件叫什么名来着。所以很多这种事情的——为什么你到他那里的时候，看他的标识总觉得特别的干净？因为他们可以不写。前两年的时候吧，那个生产酱油的那个厂家，不是说“哎呀，你那生产到那边去的那个酱油配料表就干净，在我们这里酱油就有添加”——因为他去了之后就不用标称了，按照当地的法律法规就可以不用写了，知道吧。每个国家都按照他们自身的这个市场的保护机制，会设一些，甚至是阶段性地设一些针对的门槛儿。所以所谓的“达成了他的标准就特别高”，于是就幻想那个市场里面的条件特别好，这纯属于想象。你得按照当地的标准看待当地的企业，你才知道当地的水准到底是什么样子的。
