# 第965集 五胡乱华真相调查_秩序崩坏时新移民为何0元购

五胡乱华真相调查——秩序崩坏时，新移民为什么选择“0元购”？

《胡风诔》云：  
“昔匈奴之归附兮，效衣冠而习礼。表面从文明教化兮，内里藏射雕之技。叹史家不察虚实兮，妄断胡羌已华矣。犹如留学英伦二十年兮，炸鱼薯条非英裔。忽晋室之崩摧兮，引狼烟而裂疆畿。方知文化融合之艰兮，非形似所能及。愿后来者鉴此史兮，辨魂形而后立。”

如何看待已经“汉化”的南匈奴首领？第一个起来五胡乱华啊，看见了。你得先好好理解“汉化”这两个字，对吧？因为我们一般情况下，在书里面特别容易说哪些群体已经汉化了。那什么是汉化了？因为呢，我们无论是学者也好，科普号也好，还是各种普及读物也好，里面经常会在各种段落当中说“某某群体已经汉化了”。这是一个非常常见的说法——一旦他的生活模式、语言模式开始接近，就说对方汉化了。我觉得呢，最重要的就是你这个定义本身有问题，所以才会出现你提问当中的这种疑问：不是都汉化了吗？他怎么第一个就跳反，而且迅速地把关外、就是把境外的其他匈奴接进来，就开始搞事呢？哎，咋回事呢？是不是说好了汉化，他为啥这个认知似乎不太对劲儿？这是因为它的这个“汉化”的定义有问题。

一般情况下，它只是说这个群体在行为、或者起名字的方式、生活方式上面、语言方式上面、应用的文字上面差不多，然后他就觉得已经汉化了。来，放到这儿的时候，我们就可以直接回顾一下我们自己。你来假设一下你自己的生活：例如说今天，你获得了一个留学的机会，去了小英（英国）留学。留学完了之后呢，正好有一个不错的企业给你伸出了橄榄枝，你觉得这买卖划算，于是你就到那个地方去工作下来，收入不错，买了房子。你周围的所有的邻居都是标准的伦敦腔，于是你也会说标准的伦敦腔；你周围的邻居要穿标准的西式套装，那你也穿标准的西式套装；他们吃炸鱼薯条，你也吃炸鱼薯条；他们在参加婚礼的时候给别人送手绢儿，你也给别人送手绢儿。我就问你：你“鹰化”了没有？来，我就问你，你鹰化了没有？就这个样子，你在那个地方已经生活很多年了，你鹰化了吗？假设，假设好不好？你鹰化了吗？能理解吗？

所以你达成了这些标准，按照古典时期的、包括南匈奴部的归附，它实际上达成了一个什么样的指标呢？其实就是和你去小英留学加工作20年，所掌握的语言、文字、生活习俗等等的这些技能是一样一样的。但是请问你从此就已经是英人了吗？不是，不光你还知道自己是海外华人，实际上呢，当地小英的居民看你也是华人、华裔。是不是可以理解？其实就是这个概念。它的这个归附只能做到这样的程度。

什么时候才能够说算作真正的汉化？所以我说你光用这个名字，但是没有给它界定一个清楚的标准，这是一件挺麻烦的事情。所以你就会觉得：他都已经汉化了，咋还会弄成这个样子呢？不就是这回事吗？因为你的这个归附的程度就到这个程度，更深的认同感还没有建立。他还没有认为自己是炎黄子孙，他也没有认为他跟我们之间同宗同源、完全一致、完全一体，这个事情他还没有做到呀。他只做到了中间的这一部分。而做到了这个程度的时候，你就想一想：你在小英工作了20年，做的买卖很不错，周围的邻居都对你很友善，你的口音、你的习惯、你的着装，包括化妆的方式、生活的模式一毛一样了。然后这个时候，小英咔嚓一下子骨折式地衰落，来来来，这个时候小英骨折式地衰落，然后你的祖国蹭一下子就蹿高了，你回不回？然后这边给你抛出了橄榄枝，给你发了offer，待遇很好，你回不回？是不是这样的？其实在道理上面没有什么非常本质的区别，性质上面是一样的。

我一直说我不喜欢他们新定义的这个词，为什么？因为这个词不唯物。他们老是说那个什么“五胡入华”，显得更和平、更客观，但是他不唯物啊。因为他们不是这个时候来的啊，五胡不是这个时候进来的。他们是在东汉的时候就来了，他们是作为归附群体，是作为外来打工者、作为新移民进来的，知道吗？他是从汉代开始就陆陆续续地来了。所以他使用“五胡入华”这个事情，根本不是说你的态度如何的问题，而是你用了这个词，它不唯物，你就显得这个时候他突然之间来了。“突然之间来了”是啥意思呀？他是外头的人咔嚓一下冲进来了吗？然后你又拒绝说这是侵略，你又说人家是外人，又拒绝承认他入境的行为是侵略，你尴尬不尴尬？所以实际上，他们是这些新移民群体，发现本土出现了秩序崩坏了之后，搞起了一波“零元购”。零元购搞上头了之后，他就觉得可以把他那个什么在境外的那些同族也拉拢进来，一起搞一波大的，就进入到了五胡乱华时代啊。所以，我不是说他的这个东西态度和目标对不对，而是你这种命名方式实在是不唯物，这个道理是没有办法讲得很顺畅的。
