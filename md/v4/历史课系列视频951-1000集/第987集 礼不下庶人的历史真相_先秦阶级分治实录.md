# 第987集 礼不下庶人的历史真相_先秦阶级分治实录

### 整理后的易读段落

**礼不下庶人的历史真相：先秦阶级分治实录**

《鉴古箴》有云：“礼器森森耀九重，刑书隐隐锢氓僮。簪缨世胄联姻盟，瓮牖黔首泣朔风。徒慕先贤多妻妾，焉知古制本双轨？愿君拭目察春秋，莫将糟粕奉圭翁。”我在谈论先秦时期的风俗时，这些内容与咱们当下大概率没什么关系。先秦时期我们能找到的所有与礼制相关的规则，叫做“礼不下庶人”。请大家记住这句话——有知道如何打字的朋友，请为我们打出来：“刑不上大夫，礼不下庶人”。关于“礼不下庶人”中的“礼”，指的是社会规则，其中就包括婚姻关系，因为婚姻是礼制的一部分。所以，“礼不下庶人”意味着先秦时期的婚姻与庶人、与黔首毫无关系——这完全是贵族家庭内部的事务。贵族家庭内部的事宜与平民无关，不要幻想说老祖宗的优秀传统消失了。你自己琢磨一下：如果两家缔结婚盟，把所有姑娘都归你一个人，那社会上会出现多少单身汉？你一听就知道这是扯淡。

虽然出生率上男性确实比女性略高，但也就高一个百分点而已，而且男性的死亡率更高，所以实际上男女比例大致持平。在这种持平状态下，如果一个男人娶多个女人，你觉得剩下的是谁？喂，兄弟，剩下的就是你啊！心里有点数，别动不动就说“老祖宗的优秀传统没有了”——这套传统跟你没什么关系。“礼不下庶人”中的“庶人”才是你。

有人问：“这句话没有被曲解过吗？”哪句话？ “礼不下庶人”当然被曲解过，而且历朝历代都有曲解。常常有人以为“礼不下庶人”是指不用对平民讲礼貌——实际上当然不是。“礼”是一种制度，诞生于贵族制度的基础之上。所以，“礼不下庶人”是指贵族的这套规则不要运用于平民。而“刑”是什么？刑是用来干涉平民犯罪行为、违法行为的。这些行为不要上到“大夫”级别（即“士”这个级别），这部分人不用参照适用于平民的法律。这是古时候在贵族政治前提下产生的制度性分化——一部分上层人士借由“礼”来管理，一部分基层人士用“刑”来管理，这是两套纲领。

礼制太繁琐，平民学不来——本来就是学不来的。我之前做过关于“礼不下庶人”的视频，专门讲过这件事：礼为什么不下庶人？因为礼制中大量内容庶人根本践行不了。比如你走到什么地方必须用什么步法，去往特定里程的远足必须用什么车，参与宴席必须穿什么衣裳、喝什么酒、配什么舞蹈——这些都是庶人根本做不到的。而这些正是礼制的一部分，在当时贵族秩序中非常重要，用来标称社会身份和社会分工。但这些东西在庶民阶层能用吗？不能用。所以，“礼不下庶人”确实是因为当时的社会管理条件下，必须是双轨运行。

（直播互动：有网友提到“朱子发现礼不下庶人导致民族消亡危机，所以推行理学让所有人参与礼的实践，因此他是圣人”。）这事儿扯不到一起去，这位叫“李嘉诚”的小伙伴，你把时代搞错了。“礼不下庶人”这个阶段是在先秦时期，那时才是礼制与刑制分野的时期。双轨制管理：管理庶人用刑，而且相当长时期内刑制都不是公开的，你不知道有哪些条款属于刑——这就是所谓的“刑不可知则威不可测”。后来法家的诞生就建立在这个基础上——法家将一切规则明文化，其中相当一部分努力就用在了明文化上。但先秦时期，在大部分和平有序的时间里，刑根本没有明文化，而是藏于暗室，用作威慑——威慑平民、庶人、黔首。所以，你说“礼不下庶人是朱子发现有问题”是不对的。实际上从法家这里就已经对社会制度进行了明文规定，之后大家践行的就是同一套法案了。这事儿跟朱子没什么关系，跟理学也没什么关系，而且我也不觉得朱子的理学够格称得上什么圣人。

### 备注：修正的词汇列表
1. "行不上大夫" → "刑不上大夫"（根据历史语境修正）
2. "树人" → "庶人"（原文中多次将“庶人”误写为“树人”）
3. "前首" → "黔首"（原文中“前首”应为“黔首”，指平民）
4. "珠子" → "朱子"（原文中“珠子”指朱熹，修正为“朱子”）
5. "氓僮" → 保留原诗句，未改；但若上下文指“民僮”则可能为笔误，此处维持原貌。
6. 其他口语化表达如“扯犊子”改为“扯淡”以符合书面易读风格，但未删减原意。
