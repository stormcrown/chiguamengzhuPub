# 第131集  郑和巨轮，近海航行？打脸西方论 

宋人身上美好的东西非常多，尤其在享乐方面。有人说，你不应该享乐。时隔千年，我们说宋人不应该享乐，但请问如果你生活在那个时代，你是希望过吃好吃、穿好穿的日子，还是时时刻刻都蛰伏起来准备战斗？那个时期的百姓，和今天的你一样，都是过日子的人。大家看着眼前的吃吃喝喝，炒菜源自宋代。实际上，这种烹饪方法是因为技术进步带来的，不是灵光一现的想象，而是科技进步的结果。宋代经济发展迅速，技术进步也非常快，极为先进。所以，炒菜的出现是因为铁锅普及了——铁锅不是这时才刚出现，但此时的生产工艺已经实现大规模生产，得以普及。带来的效果就是快炒、高温烹饪的方式得以出现。

宋代的经济来源中，也有相当一部分来自铁锅的对外贸易，因为其他地方生产不出来，没有这种生产力，没有这种技术水平。所以他们会将铁锅卖出去赚外汇。铁锅在那个时代的技术水平就相当于芯片，而且是别人都没有、只此一家、只有这里才能生产的芯片，是拥有独特生产技术屏障的产品。到处都卖？实际上，你若去查账单，可能没多少，查不出多少货。为什么铁锅贸易如此？因为这玩意儿都是走私的私铁。这玩意儿很容易被加工成兵器，所以原则上铁锅是不允许进行大规模贸易的。但是，这玩意儿一本万利啊！所以我们在南海发现的宋代沉船上面，有大堆大堆的铁锅。你去查他的账本，都不怎么提铁锅这件事，因为不准卖。但你一翻沉船，一摞一摞的，半船塞的都是铁锅。没想到？宋代的铁锅出口到世界。实际上，铁锅的走私贸易从宋明到清代乾隆之前都有，连清代都有，因为西方人掌握铁锅的铸造工艺是在乾隆年间。技术上唐朝就有了，但是到宋代，我们的生产水平才能够达到通货的程度，才能进行大体量的外贸。

有人问海船上装铁锅不怕生锈吗？人家是专业贩运的，专业贩运又不是泡在水里头。古时候的海贸有自己的防护水平，例如金属器件用油保养就不易生锈。然后瓷器是怎么运输的？所有的瓷碗、瓷盘子中间撒上种子，一摞一摞摞起来捆结实，往上面浇水，种子就会发芽，变成天然的减震层。有人说瓷器塞麦秆和稻梗，原理差不多。

有人问远洋的水怎么解决？我们往水里加醋。远洋航行这种事，要印证一个群体有没有远洋航行能力，就要看它能否解决船上吃喝拉撒的问题，这是根本性的问题。西方人说他们进行了最大规模的全球探险和远洋航行，但在前期一直不知道船员为什么生病，也不能解决船员被迫酗酒的问题——真的是被迫。因为淡水非常容易腐败、滋生细菌，不是烧了水就能解决，烧开了也解决不了已经被污染的水中的有毒物质。所以西方的船员，你看电影电视剧里，也都是酒鬼。因为只有发酵的酿造品才能维持基本质量，喝下去不生病。所以他们的水手普遍是酒蒙子。中国人显然不是那个路子。远洋航行要工作，团队里有科学家、天文工作者、工匠等，这些人不适合经常饮酒。所以我们的船队运行方式不适合喝嗨了。我们的解决方式是在水里添加醋。醋里面有醋酸菌，上面起的那层膜，有了这个，醋酸菌维持不错的活跃度。水体有醋酸菌均衡制衡，作为主体菌群，一个没有危险的主体菌群在里面活着，其他菌就繁衍不起来，水就可以正常饮用，不会滋生有毒物质。所以远洋航行时水里用醋。你们那里叫醋蛾子或别的名字，很多地方都有。喝酸的能解渴吗？可以，没有问题，不是直接喝醋，只是水里含有醋的成分。所以我们的航行，我们知道该如何操作，这意味着我们有经验，确实想出过解决方案。所以出海的人会吃各种芽苗菜，不会生坏血病；会带鸡等动物，就有蛋白质；粮食以种子形式带出去，保鲜期长。有了这些食物来源，才能长时间在海上生活，才有可能推行远洋航行。

之前一直有学者认为中国人没有远洋航行能力，甚至规定中国人不可以有，因为假设远洋航行得是西方人开创的。所以一些专业学者给郑和航道绘制图案时，让他捋着岸边近海航行。很多年里都有学者说郑和团队是近海航行，没有真正到远洋航行的程度。但是从考古近几年发现的桅杆可以倒推出郑和宝船尺寸，100多米，不到200米。这种体量的海船贴着岸边能开吗？在近海不是开几步卡几步？做不到。这种超巨型海船必须进入远洋区段，绝对不能在近海航行。所以那些说郑和只是近海航行、遇到非洲就绕回来的人，是不是专业人士？是专业人士。有没有权威性？有权威性。所以你相信他没有问题。但我这里只需要留一个气口：你能不能也容忍其他人说话？能不能等一等技术的发展？能不能等待其他人的研究成果？认知在不断刷新，认知是有时效性的。从清末开始，中国人非常自卑，自卑到认为自己的长相都不如别人。这种自卑感会带来什么？就是会觉得自己什么都没有。于是当有人说西方人有这个、有那个，你就对对对。因为你觉得这一切都得向对方学习才合理。所以当学者看到郑和下西洋，既然我们技术不行，那航海能去哪儿？必须捋着岸边走，就必须是近海航行。走到非洲沿线时，肯定要遇上坎，好望角难开，技术不行，到这必须回来。实际上原理是：如果他没回来，就不会有七次、八次下西洋。于是推论说郑和是近海航行到非洲触底反弹就回来了。但这合理吗？郑和宝船体量100多米不到200米，放到今天都是巨轮，船上还带着2万正规海军，还有商船等。你说这也没干那也没干，出去浪了一把就回来了，对于正常有理性的人来说，合理吗？
