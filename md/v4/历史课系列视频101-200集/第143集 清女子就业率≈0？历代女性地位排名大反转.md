# 第143集  清女子就业率≈0？历代女性地位排名大反转

然后那个啥啊，清朝的女子地位甚至不如三皇五帝时期。嗯，这事不好比啊，这确实不好比。为啥呢？因为三皇五帝时期母系氏族的残留还存在，所以两性的地位是不能够直接进行一比一的对照的，那没有办法直接比。所以呢，怎么去比较说这个女性地位的高低的问题，咱们最好是在同一个社会的阶段范围之内进行一下比较。一竿子戳到三皇五帝的时候，因为他存有这种母系氏族的残留，所以那个时期的女性的社会评价和社会的贡献值，包括工作的参与度，都与后来的时期是有差异的。这个跟生产力的水平也有关系。当我们进入到了铁器时代，也就是深入的农耕时期，从这个时期开始，我们可以算作它是一个相对来说在生产力的基本指标上面保持了一致的时代。在这种范围之内呢，那女性的地位是在不断的波动的变化的。

什么情况下女性的地位会比较高？第一呢，就是女性的社会地位要高，第一件事情你得做到和平啊，至少要相对和平，一定要相对和平。第二呢，就是它内部的这种社会结构上面相对来说还比较的活跃、活泼、自由，拥有一定的自由度。这两样东西要同时存在，女性的社会地位才会相对较高。所以你看说秦汉时期女性地位高不高啊？你要是看贵族女子的话，你会发现社会地位挺好的呀。但是呢，因为社会的整体的活跃度，尤其是上下的这种活跃——就是上升通道实际上是约等于没有的，有一点儿但是约等于没有——这种情况下的女性的社会地位就不会很高。

然后到了唐的时候，很多人说开放又包容的盛唐啊——我特别讨厌这个词儿啊，所有在直播间里面蹲久了都知道，我特别讨厌这个描述方式，叫做开放又包容的盛唐呀。盛唐是因为强大所以表现为开放和包容，这是因为我强了所以我很宽厚，你随便捶我一个小拳拳，我还当你是打情骂俏是吧？那本质是因为强大，而不是因为开放啊。那盛唐的衰落的原因反而是开放与包容啊——它衰落的原因是开放与包容，而开放与包容是因为它的强大，所以它中间是有一个这样的逻辑线索的关联的，所以我不喜欢这个词儿啊。那么唐代的时候，那女性地位呢，就比之前的那些时期要略好一点点了，因为呢，平民的女子也拥有了经商贸易的权利。这个呢，就是社会自由度发生改变之后，女性的社会地位就会有所提高。但是这个地方啊，我不是在单独指贵族女性，我很少单独的去指贵族女性这个群体，因为无论什么时期，贵族女性相对而言都是拥有一些特殊的权利的。我们得把普通女性拿到台面上面来评点一下女性的社会相对地位。那么在唐代的时候，社会地位就提高了一点点，但是这个时候依然存在着世家门阀的问题啊，所以上升通道存在，活跃度存在，但是呢，并没有足够的宽敞。所以可以说这个时期的女性地位有点高，但是远远不像电视剧里给你传播的这个时候女性地位最高了，错。

实际上在后面，我们更加进入到了商品化自由的社会、市场经济的范围之中的时候，女性的社会地位是更高了的。从宋代开始，包括妻妾制度当中的妾，就开始拥有了一定的家庭权利。在那之前叫做“契同买卖”，就是视为商品附庸，可以任意交易。但是到了宋的时期，妾室是需要进行相应登记的，并且获得了较多的家庭关系的保障。能不能每一家都做到啊？那确实也不一定，因为这个时候很多的文人、士大夫等等的这些富人群体，他不一定正规的纳妾。有的时候包括说啊，经常有人传说说宋代的文人经常把妾室送人，其实呢，这些妾都不是正规的妾，因为在宋代的时候的规范当中，妾室是不能够任意的买卖送人的。他的这些买卖送人的其实是他所豢养的“妓”，他没有登记，没有把他真正的当做妾，他才能够做这件事情，是把它作为仆庸一类的人而存在。那么在宋的时候，在明文的规定上面，妾室的地位已经提高了。那么仆庸的地位呢，相较于之前的时代也提高了很多。大家去看的电视剧里面，动辄告诉你说“打杀了他吧”，嗯，在宋代的时候，这个东西是非法的。宋代的时候这些行为都是非法的。有相当一部分的家庭当中的仆庸，在宋明两朝的时候，其实是一个雇佣聘任制的打工仔。那你就不存在说我随便打杀了他吧，这个事不允许啊，是一个明确的非法行为。

所以呢，什么时期女性地位会高呢？一个就是社会的通达程度、自由程度、商品化的程度，然后市民化的程度都会影响女性的社会地位。所以北宋的女性的社会地位就相对较好，尽管说存在由所谓的理学家，但是理学家定的是什么？定的是规则，而不是社会地位的高低。不要觉得理学家规定了女的不许爬墙，所以女的社会地位就低了。谁跟你说的社会地位取决于你能不能公开出轨啊？就是今天你也不能说女性的社会地位跟女性是不是可以肆意出轨有关系吧？这两件事情风马牛不相及啊，千万不要把这玩意绑到一起去。那社会地位，你要去看他的经济地位和社会的公共活动的参与度。而在宋代的时候啊，宋明两代的时候，实际上女性的社会公共经济体系当中的参与度，都比之前的时代要更好，包括说家庭的财产继承的分配，也都比之前的时代要更好。所以女性的社会地位在这两个时期其实是上升了的。

但是啊，但是到了清的时候，这玩意儿急转直下啊，这叫做急转直下啊。清朝的女性几乎无就业，也没有办法就业了。凡是那个条件较好一点的地区啊，就是整体的管控力度较强的地区，实际上都是女性普遍性的缠脚。那种把你的脚都已经掰成那个样子的时候，都已经残疾的时候，很多的工作都没有办法参与啊。之前有人还专门说崇祯的贵妃还还不是缠脚，对，崇祯的贵妃缠脚出了名了，她缠了脚，那他出了名的爱打马球，你怎么不说呢？是吧。所以其他的时代那个所谓的啊，什么贵妃缠脚、公主缠脚——对，缠了没缠了，能跑步，能打马球，能踢、能蹴鞠啊，人家可爱这一口了。这属于贵族女子的独特爱好啊，打起马球来飞起一样的。所以呢，这些女子可以骑马，可以打仗，可以做工，可以出门啊，可以找工作。但是到清的时候，这个项目不存在了啊，这个项目没有了。
